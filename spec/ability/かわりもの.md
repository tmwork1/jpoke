# 仕様書: かわりもの

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%8F%E3%82%8A%E3%82%82%E3%81%AE

## 2. 基本効果（1段落）
場に出たとき、正面の相手ポケモンに自動的にへんしんし、そのポケモンの姿・タイプ・能力ランク・技構成をコピーする。

## 3. 詳細仕様

### フォルムチェンジ関連
- 入場時に即座にへんしん（フォルムチェンジ）
- コピー対象：
  - 種族値（実数値として反映）
  - タイプ
  - 能力ランク（攻撃・防御などのランク）
  - 技（最大4つ、PPは各技5に固定）
- コピーされない要素：
  - HP（自身の最大HP・現在HPを維持）
  - レベル
  - 個体値・努力値（結果としての実数値のみ反映）

### 発動条件・基本挙動
- 場に出たときに自動発動（強制）
- 対象は基本的に「正面の相手」
- ダブルバトルでは正面が優先対象
- 対象が不在の場合は発動しない

### 発動しない条件
- 対象がへんしん状態などでコピー不可の場合
- 対象が特定のフォルム（例：一部の特殊状態）でコピー不可の場合
- 自身または対象が特性無効化状態で処理が阻害される場合

### 無効化・上書き
- かがくへんかガス中は発動しない
- 無効化解除後でも自動再発動はしない
- 特性のコピー・交換は不可（この特性自体は特殊扱い）

### 交代・入場時
- 入場時に1回のみ発動
- 交代すると元の姿に戻る
- 再登場時に再び発動

### その他の特性との相互作用
- へんしん後は対象ポケモンの特性になる
- いかく等の入場時特性はコピー後には発動しない（すでに入場処理済みのため）
- ばけのかわ等の状態もコピー対象外（フォルム依存）

## 4. 処理順・優先度
- 発動タイミング：入場直後（他の入場時特性と同タイミング）
- 処理順：
  1. 入場
  2. 他の入場時特性処理
  3. かわりもの発動（へんしん）
- 同時発動時の優先度：
  - 行動順（すばやさ）依存
- すばやさは発動順に影響

## 5. テスト項目（実装検証用）

### 発動ケース
- 入場時に相手ポケモンへ自動変身する
- 能力ランク・タイプ・技が正しくコピーされる
- 技のPPがすべて5になる

### 非発動ケース
- 対象不在時に発動しない
- 特性無効化状態で発動しない

### 境界ケース
- HPがコピーされず自身の値を維持する
- へんしん後に特性が対象のものに変化する
- 交代で元に戻り、再登場で再発動する
