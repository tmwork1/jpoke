# 仕様書: さいせいりょく

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%95%E3%81%84%E3%81%9B%E3%81%84%E3%82%8A%E3%82%87%E3%81%8F

さいせいりょく は、ポケモンのとくせいの一種。
## 効果
他のポケモンに 交代 すると、自身の最大 HP の1/3が回復する (小数点以下は切り捨て)。
## 説明文
- 第五世代 ・ 第六世代: ひっこめると HPが すこし かいふくする。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは せんとうから ひっこむと すこし だけど たいりょくが かいふく するんだ
- 第七世代 ・ 第八世代 ・ 第九世代: てもちに ひっこむと HPが すこし かいふくする。
## 特性の仕様
- 使うと交代する技 / 相手を交代させる技 や、 だっしゅつボタン / だっしゅつパック / レッドカード の効果で 交代 したときも発動する。
  - しっぽきり で交代する場合、技の効果でHPが減少した後にさいせいりょくの発動判定がある。途中で きのみ の発動圏内に入った場合、きのみの発動が挟まれる。
- さいせいりょくによる回復は、交代先のポケモンを選択するタイミングですでに完了している。
- 本来他の特性を持つポケモンが、 特性を変える技 / 特性 の効果で一時的にさいせいりょくを得ているときに交代した場合、さいせいりょくの効果は発動する。
- 本来さいせいりょくを持つポケモンが、この特性を失った状態で交代した場合、さいせいりょくは発動しない。 とくせいなし 状態のときに交代した場合でも同様に発動しない。
- かがくへんかガス が発動しているときは交代しても回復しない。かがくへんかガスのポケモンと同時に交代した場合、後に交代した方のポケモンの特性が優先される。
- かいふくふうじ 状態でもこの特性の効果で回復することはできる。 バトンタッチ でかいふくふうじ状態を引き継いだときでも同様となる。
- かたやぶり の効果がある技を受けて だっしゅつボタン で交代したときや、かたやぶりの効果がある 相手を交代させる技 を受けた場合であってもさいせいりょくは発動する。
- しぜんかいふく とは異なり、 第七世代 以前でも戦闘が終了したときに発動することは無い。
## 備考
- 第六世代 と 第七世代 では、 いのちのたま / かいがらのすず を持つ相手から 攻撃技 を受けて特性所持者の だっしゅつボタン が発動したとき、いのちのたま/かいがらのすずによるHP変動と同時にさいせいりょくのポケモンのHPが回復していく様が映し出される。
  - 全体攻撃技を受けてさいせいりょくのだっしゅつボタンが発動し、その際の攻撃で他のポケモンが きのみ による回復を発動させたときも、同様の現象が起きる。
  - 第五世代 と 第八世代 以降ではだっしゅつボタンの発動させたポケモンのHPバーは表示されず、回復する様子は映し出されない。
