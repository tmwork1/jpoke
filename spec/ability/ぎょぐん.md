# 仕様書: ぎょぐん

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8E%E3%82%87%E3%81%90%E3%82%93

## 2. 基本効果（1段落）
HPの割合に応じてフォルムチェンジし、能力値が変化する。HPが一定以上で「むれたすがた」、それ未満で「たんどくのすがた」となる。

## 3. 詳細仕様

### フォルムチェンジ関連
- 対象ポケモン専用特性
- フォルム：
  - HPが最大の1/4以上 → むれたすがた
  - HPが最大の1/4未満 → たんどくのすがた
- フォルムによって能力値が変化（種族値差し替え）
- タイプは変化しない

### 発動条件・基本挙動
- 入場時およびHP変動時に自動判定
- 条件を満たすたびに即時フォルムチェンジ
- ダメージ・回復の結果に応じてリアルタイムで変化

### フォルム変化タイミング
- ダメージ計算後にHPが閾値を跨いだ場合：
  - その直後にフォルムチェンジ
- ターン終了時回復・ダメージでも判定される
- 行動中の途中でHPが変化した場合も即時適用

### 無効化・上書き
- 無効化・上書きされない

### 交代・入場時
- 入場時にHPに応じたフォルムで登場
- 交代してもHP条件により再判定される

### その他の特性との相互作用
- フォルムに依存する能力値が即時更新される
- かたやぶりの影響は受けない（自己状態変化のため）
- フォルム変化は能力ランク変化とは別扱い

## 4. 処理順・優先度
- 発動タイミング：
  - 入場時
  - HP変動直後
- 処理順：
  1. HP変動確定
  2. 閾値判定
  3. フォルムチェンジ
  4. 能力値更新
- 同時発動時の優先度：
  - 他の状態変化処理後に評価
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- HPが1/4以上でむれたすがたになる
- HPが1/4未満でたんどくのすがたになる
- 回復により閾値を超えた場合に元のフォルムに戻る

### 非発動ケース
- 特性無効化中にフォルムが変化しない

### 境界ケース
- HPがちょうど1/4のときにむれたすがたになる
- ダメージ・回復の直後に即時フォルムが更新される
- 無効化解除後に即時再判定される
