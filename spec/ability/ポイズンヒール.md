# 仕様書: ポイズンヒール

調査日: 2026-05-02

## 2. 基本効果（1段落）
どく・もうどく状態のとき、ターン終了時のどく/もうどくダメージを受けるかわりにHPを最大HPの1/8回復する

## 3. 詳細仕様
- 発動条件: どく・もうどく状態で毎ターン終了時に発動し、どくダメージの代わりに最大HPの1/8を回復する。
- もうどく状態でも回復量は最大HPの1/8固定（もうどくのターン経過による増加はない）。
- どく・もうどく状態でもたたりめ・ひゃっきやこうによるダメージは通常通り受ける（ダメージ耐性ではない）。
- かいふくふうじ状態では回復できない。かいふくふうじが終了するターンも回復できない。
- 無効化・上書き: かがくへんかガスで無効化され、通常のどく/もうどくダメージを受けるようになる。

## 4. 処理順・優先度
- 発動タイミング: ターン終了時のどく/もうどくダメージ処理のタイミングで、ダメージの代わりにHP回復を行う。

## 5. テスト項目（実装検証用）
- 発動ケース: どく状態でHPが最大HPの1/8回復すること。もうどく状態でもHPが1/8回復すること（増えないことも確認）。
- 非発動ケース: かいふくふうじ状態では回復しないこと。かがくへんかガス下では通常のどくダメージを受けること。
- 境界ケース: どく状態でたたりめを受けたとき通常のダメージを受けること。
