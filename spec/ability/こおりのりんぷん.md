# 仕様書: こおりのりんぷん

参照URL: https://wiki.pokemonwiki.com/wiki/こおりのりんぷん

## 基本効果
特殊技で受けるダメージを半減する。

## 詳細仕様

### 軽減対象
- 技分類が 特殊 の攻撃にのみ適用する。
- ダメージ固定技には適用しない。

### 分類参照技との相互作用
- サイコショック / サイコブレイク / しんぴのつるぎ のように、与ダメージ計算でぼうぎょ側を参照しても、技分類が特殊なら半減する。
- 分類が変わる技は、最終的な技分類が特殊のときのみ半減する。

### 場状態との相互作用
- ワンダールーム中でも、特殊技半減の効果はそのまま有効。

## 処理順・優先度
- ダメージ補正段階で、特殊技の最終ダメージに 0.5 倍補正を適用。

## テスト項目

- [ ] 通常の特殊技ダメージが半減される
- [ ] 物理技ダメージは半減されない
- [ ] ダメージ固定技は半減されない
- [ ] サイコショック系が半減される
- [ ] 分類が変わる技で、最終分類が特殊なら半減・物理なら非半減になる
- [ ] ワンダールーム中でも効果が維持される
