# 仕様書: マジックガード

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%B8%E3%83%83%E3%82%AF%E3%82%AC%E3%83%BC%E3%83%89

マジックガード は、ポケモンの とくせい の一種。
## 効果
相手の 攻撃技 以外による ダメージ を無効化する。具体的には以下のダメージを受けない。
- わざ の補助効果によるダメージ
  - 反動技 による反動ダメージ : ビックリヘッド / てっていこうせん / クロロブラスト の反動、 とびげり / とびひざげり / かかとおとし / サンダーダイブ の失敗時の反動も受けない。
    - 例外として、 わるあがき の反動ダメージは受ける。
  - ニードルガード
  - はじけるほのお の火花
- 状態異常 によるダメージ
  - どく / もうどく
  - やけど ( 物理技 のダメージは通常どおり半減する)
- 状態変化 によるダメージ : ただし、 こんらん / ほろびのうた / みちづれ 状態によるダメージは防げない。
  - あくむ / のろい / しおづけ
  - やどりぎのタネ : ダメージを無効化するため、使用者の位置にいるポケモンを回復させることもない。
  - バインド : ダメージのみを防ぐ。 にげる / 交代 はできなくなる。
  - ふんじん : ほのお タイプの技を使おうとすると失敗するが、ダメージは受けない。
- 場の状態 によるダメージ
  - 自分の場の ステルスロック / まきびし / キョダイコウジン
  - ひのうみ / キョダイベンタツ / キョダイゴクエン / キョダイホウゲキ / キョダイフンセキ : 戦闘アニメが再生される環境で、無効化できるタイプを持たない場合は、ターン終了時にこれらの状態のエフェクトだけが発生する。
  - てんき の すなあらし / あられ
- もちもの によるダメージ : 道具の効果が発動するときのメッセージも流れなくなる。
  - 自分の いのちのたま / くっつきバリ / くろいヘドロ
  - 相手の ゴツゴツメット
  - 相手の ジャポのみ / レンブのみ : これらのきのみは消費もされない。
- 相手が持つ ダメージを与える特性 によるダメージ
  - うのミサイル / さめはだ / てつのトゲ / とびだすなかみ / ナイトメア / ヘドロえき / ゆうばく : 第五世代 以降では、相手の特性バーだけは表示される。特性発動時のメッセージは流れない。
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代 ・ 第九世代: こうげき いがいでは ダメージを うけない。
- ブラック2・ホワイト2 ( ライブキャスター の チェレン ): この とくせいの ポケモンは あいての こうげき いがいでは ダメージを うけないんだ
## 特性の仕様
- 攻撃技 のダメージでなくても、以下のダメージは受ける。
  - ひんしになるわざ / はらだいこ / みがわり / のろい (呪い) / ソウルビート / しっぽきり / みをけずる 使用時のHP消費
  - こんらん 状態/ 命令無視 による自傷ダメージ
  - わるあがき の反動ダメージ
  - いたみわけ の使用時、または受けた時
  - ほろびのうた / みちづれ 状態が発動したことによるひんし
- みらいよち / はめつのねがい によるダメージは受ける。
- マジックガードのポケモンが特性 ミイラ / とれないにおい / さまようたましい のポケモンに 直接攻撃 を使用した場合、マジックガードが上書き/交換された後に、相手の ゴツゴツメット / ジャポのみ / レンブのみ のダメージや、自分の反動技のダメージと いのちのたま の反動ダメージを受ける。
- 特性 かがくへんかガス のポケモンを倒したことでかがくへんかガスが解除される場合、相手のゴツゴツメット/ジャポのみ/レンブのみは発動してダメージは受けるが、反動技のダメージといのちのたまの反動ダメージは受けない。
  - 攻撃技に対して、かがくへんかガスが だっしゅつボタン の効果で交代したことでガスが解除される場合、反動技のダメージは受けるが、いのちのたまのダメージは受けない。
- かたやぶり の効果がある技を受けてもマジックガードは無視されない。
- マジックガードの効果が発動しても、それを明示するメッセージは流れない。第五世代以降でも、発動時にマジックガードの特性バーは表示されない。
- マジックガードにより もうどく のダメージを防いでいる間も、もうどく状態でいるターンは記録されている。例えばもうどく状態になってから5ターン目に特性を失った場合、そのターン終了時に最大HPの5/16分のダメージを受ける。
### 第四世代のみ
- フィールド上での どく ダメージは受ける。
- 第四世代のみ、ダメージを防ぐ以外にも以下のような効果があった。
  - まひ 状態でも行動不能にならない。ただし、 すばやさ は通常どおり1/4になる。
  - どくびし で どく 、 もうどく にならない。
## 備考
- 初登場の 第四世代 では ピィ 系統の 専用特性 だった。
- いしあたま の上位互換と言える効果を持つ。
  - 第五世代 でのみ、 ぼうじん の上位互換とも言える効果だった。第六世代以降ではぼうじんの仕様が変更されたため、上位互換ではなくなっている。
