# 仕様書: あめふらし

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%82%E3%82%81%E3%81%B5%E3%82%89%E3%81%97
調査日: 2026-04-30

## 2. 基本効果（1段落）
場に出たとき、天気をあめ状態にする。第6世代以降は通常5ターン（しめったいわで8ターン）継続する。

## 3. 詳細仕様
- 発動条件: 入場時に発動する。特性が後からあめふらしへ書き換わった場合も即時発動する。
- 持続: 発動者が交代・ひんしになっても天気は継続する。
- 上書き関係: すでに通常のあめ状態なら再発動しても継続ターンは更新しない。
- 強天候との関係: おおひでり/おおあめ/らんきりゅうの場では特性表示のみで天気変更は失敗する。
- 無効化: かがくへんかガスで特性が無効化されている間は発動しない。

## 4. 処理順・優先度
- 発動タイミング: 入場時特性処理。
- 同時発動時の優先度: 天候始動特性同士は行動順比較ルール（実効すばやさ準拠、同値はランダム）で後の発動が最終天気になる。
- 継続ターン: 第6世代以降はターン終了時に天気ターンを消費する。

## 5. テスト項目（実装検証用）
- 発動ケース: 入場であめ状態になり、継続ターンが正しく設定される。
- 非発動ケース: 強天候中は表示のみで天気が変化しない。
- 境界ケース: 既存あめ状態での再入場時にターンがリセットされない。
