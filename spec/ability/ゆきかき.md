# 仕様書: ゆきかき

調査日: 2026-04-30

## 2. 基本効果（1段落）
天気がゆきのとき、すばやさが2倍になる

## 3. 詳細仕様

### 効果
天気があられ/ゆき状態のとき、すばやさが2倍になる。

### 説明文
天気が ゆき のとき 素早さが 上がる。

### 特性の仕様
- すばやさ補正はランク補正の後に計算する。まひの効果はゆきかきやその他の補正の効果を考慮した後に計算する。
- この特性を受け取ったり失ったりした場合、第七世代まではそのターンの行動順には影響ない。第八世代からはそのターンから行動順に影響が出る。
- 天気があられ/ゆき状態でも特性ノーてんき/エアロックのポケモンが場にいるときは発動しない。

### 備考
- すなかきのあられ/ゆき版の効果がある。
- 第九世代現在では、特性ゆきかきを所有するポケモンは全てこおりタイプを持つ。

## 4. 処理順・優先度
- 発動タイミング: すばやさ計算フェーズで評価する（ランク補正の後）。
- 同時発動時の優先度: 同種の効果が重なる場合はゲーム内の一般優先順（行動順・速度・ランダム同値）に従う。
- すばやさ補正との関係: ゆきかきによるすばやさ2倍はランク補正後に適用し、行動順計算に反映する。

## 5. テスト項目（実装検証用）
- 発動ケース: ゆき/あられ状態でのすばやさが2倍になること。
- 非発動ケース: ゆき/あられ以外の天気では発動しないこと。ノーてんき/エアロック存在下では発動しないこと。
- 境界ケース: まひとの複合適用、第七/八世代の行動順への影響タイミング差異。
