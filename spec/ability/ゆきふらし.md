# 仕様書: ゆきふらし

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%86%E3%81%8D%E3%81%B5%E3%82%89%E3%81%97
調査日: 2026-04-30

## 2. 基本効果（1段落）
場に出たとき、天気をあられ（第8世代以前）/ゆき（第9世代）状態にする。第6世代以降は通常5ターン（つめたいいわで8ターン）継続する。

## 3. 詳細仕様
- 発動条件: 入場時に発動。特性書き換えでも即時発動する。
- 持続: 発動者が場を離れても継続。
- 上書き関係: 既に同じ天候（あられ/ゆき）なら継続ターンは更新しない。
- 強天候との関係: おおひでり/おおあめ/らんきりゅう中は変更失敗。
- 無効化: かがくへんかガス中は発動しない。

## 4. 処理順・優先度
- 発動タイミング: 入場時特性処理。
- 同時発動時の優先度: 天候始動特性同士は後処理側が残る。
- 継続ターン: 第6世代以降はターン終了時に消費。

## 5. テスト項目（実装検証用）
- 発動ケース: 入場で対応世代の天候へ変化。
- 非発動ケース: 強天候中で変更失敗。
- 境界ケース: 同一天候中の再入場でターン更新なし。
