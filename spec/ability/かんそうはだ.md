# 仕様書: かんそうはだ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%93%E3%81%9D%E3%81%86%E3%81%AF%E3%81%A0

## 2. 基本効果（1段落）
みずタイプの技を受けるとダメージを受けずに最大HPの1/4回復し、ほのおタイプの技で受けるダメージが増加する。また、天候によって毎ターンHPが増減する。

## 3. 詳細仕様

### みず技への反応
- みずタイプの攻撃技を受けた場合：
  - ダメージを無効化
  - 最大HPの1/4回復
- 変化技には反応しない

### ほのお技への補正
- ほのおタイプの技で受けるダメージが1.25倍になる

### 天候による影響
- あめ状態：
  - ターン終了時に最大HPの1/8回復
- ひざしがつよい状態：
  - ターン終了時に最大HPの1/8ダメージ

### その他のダメージとの関係
- 天候ダメージとは別枠で処理される
- 状態異常（どく・やけど等）のダメージは通常通り受ける

### 無効化・上書き
- 特性が無効化されている場合（例：かがくへんかガス）：
  - みず技回復・ほのお弱点増加・天候効果すべて無効
- 無効化解除後は通常通り機能
- 特性のコピー・交換は可能

### 交代・入場時
- 常時効果
- 入場時の即時効果はなし
- 天候効果は場に出ている間のみ適用

### その他の特性との相互作用
- ちょすい等の水無効特性とは別系統（回復量が異なる）
- かたやぶりの影響は受ける：
  - この場合、みず技でダメージを受ける
- 天候変更特性と組み合わせることで回復・ダメージ状況が変化

## 4. 処理順・優先度
- 発動タイミング：
  - みず技被弾時（即時）
  - ターン終了時（天候処理）
- 処理順：
  1. 技命中
  2. タイプ相性判定前後で無効化処理
  3. 回復適用
  4. ターン終了時に天候効果処理
- 同時発動時の優先度：
  - 天候処理は通常の終了時処理順に従う
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- みず技でダメージを受けずHPが1/4回復する
- あめ時にターン終了でHPが回復する
- ひざしがつよい時にターン終了でダメージを受ける
- ほのお技のダメージが増加する

### 非発動ケース
- みずタイプの変化技では回復しない
- 特性無効化状態で各効果が発動しない

### 境界ケース
- HP満タン時にみず技を受けても回復は上限で止まる
- 複数の終了時効果と正しく順序処理される
- かたやぶりでみず技ダメージが通る
