# 仕様書: いたずらごころ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%84%E3%81%9F%E3%81%9A%E3%82%89%E3%81%94%E3%81%93%E3%82%8D

## 基本効果
使用者の変化技の優先度を+1する。第7世代以降、この特性で優先度が上がった技は相手のあくタイプに無効化される（PPは消費）。

## 詳細仕様
### 優先度上昇の対象
- 変化技のみ優先度+1の対象。
- 他の技が出る技で攻撃技を出した場合でも、1ターン目の行動が変化技として処理されるケースでは優先度上昇の影響を受ける。
- 溜め技は原則として1ターン目のみ優先度上昇の対象。ジオコントロールは2ターン目にも優先度上昇の影響を受ける。
- 優先度補正後も優先度0以下の技（例: ほえる）は、あくタイプ無効の対象外。

### 無効化条件
優先度が上がった技は、以下の順で無効化判定される。
1. じょおうのいげん / ビビッドボディ / テイルアーマー
2. サイコフィールド（対象が接地時）
3. ファストガード
4. あくタイプ（相手対象時のみ）

- 無効化された場合でもPPは消費される。
- 使用者自身または味方を対象にした技は、相手にあくタイプがいても有効。

### 対象範囲ごとの挙動
- 自分含む全員対象技（例: ほろびのうた）では、無効化要因ごとに影響範囲が異なる。
- じょおうのいげん / ビビッドボディ / テイルアーマーで無効化された場合は、全体が技の効果を受けない。
- サイコフィールド / あくタイプで無効化された場合は、無効化対象のみ効果を受けず、それ以外には適用される。
- ファストガードは、全体対象技については基本的に無効化しない。

### かがくへんかガスとの相互作用
- かがくへんかガスでいたずらごころが無効化されている間は、優先度上昇は発生しない。
- ガス解除後は、その時点から再び優先度上昇が有効になる。
- ガス発動中は、いたずらごころ由来のあくタイプ無効判定も発生しない。

### 世代差分（実装要点）
- 第6世代以降、いたずらごころで優先度が上がった技はファストガードで無効化される。
- 特性変更による同ターン反映は第8世代以降で即時反映、第7世代以前は原則として次ターン反映。

## 処理順・優先度
- 優先度補正は技選択後の行動順決定時に参照する。
- 無効化判定は「先制技阻害系特性 → サイコフィールド → ファストガード → あくタイプ」の順で行う。
- 同ターン中に特性が変化した場合の反映は世代差分に従う（第8世代以降は即時、第7世代以前は即時反映しない）。

- [ ] 相手あくタイプに対して、優先度上昇した技が無効化される（PP消費あり）。
- [ ] じょおうのいげん / ビビッドボディ / テイルアーマーで無効化される。
- [ ] サイコフィールド（接地）で無効化される。
- [ ] ファストガードで無効化される（第6世代以降想定）。
- [ ] 自分・味方対象技は相手あくタイプがいても有効。
- [ ] 全体対象技で、無効化要因ごとの影響範囲差分が再現される。
- [ ] かがくへんかガス中は優先度上昇しない。
- [ ] かがくへんかガス解除後に優先度上昇が復帰する。
- [ ] 溜め技は原則1ターン目のみ優先度上昇し、ジオコントロールは2ターン目にも適用される。
