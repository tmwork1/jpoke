# 仕様書: クリアボディ

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%AF%E3%83%AA%E3%82%A2%E3%83%9C%E3%83%87%E3%82%A3
調査日: 2026-04-30

## 2. 基本効果（1段落）
他のポケモンによる能力ランク低下を受けない。

## 3. 詳細仕様
- 防止対象: 相手または味方から受ける「ランク低下効果」を無効化する。
- 防止対象外: 自分の技・反動・持ち物・状態異常など、ランク補正以外の能力低下は防げない。
- 既に下がったランク: 特性取得後に自動回復しない。
- かたやぶり相互作用: かたやぶり効果付き技では無視されて低下を受ける。
- リセット系: くろいきり/クリアスモッグ等のランクリセットは「低下」ではないため別処理。

## 4. 処理順・優先度
- 適用タイミング: ランク変動適用直前。
- 優先関係: しろいきり/クリアチャーム等と競合時は特性側優先表示ルールに従う。
- 同時効果: 追加効果・入場時効果・変化技による低下を同一判定系で遮断。

## 5. テスト項目（実装検証用）
- 発動ケース: いかく、能力低下追加効果、変化技の低下を防ぐ。
- 非発動ケース: 自己能力低下技やまひ等による能力低下は防げない。
- 境界ケース: かたやぶり、くろいきり、複数低下同時適用。
