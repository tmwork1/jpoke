# 仕様書: きもったま

参照URL: https://wiki.pokemonwiki.com/wiki/きもったま

## 基本効果
ゴーストタイプに対して、ノーマル技・かくとう技を無効ではなく等倍で当てられる。第8世代以降は いかく を受けない。

## 詳細仕様

### 相性変更
- きもったまの効果は、ゴーストへの無効化だけを打ち消す。
- 他タイプとの複合相性は通常どおり適用する。
  - 例: はがね/ゴーストへの かくとう技 は抜群。

### いかく無効
- 第8世代以降、入場時の いかく による こうげき下降を受けない。

### 特性変化との相互作用
- 直接攻撃で ミイラ / さまようたましい などにより特性が変化した場合、連続攻撃中の後続ヒットは新しい特性で判定する。

## 処理順・優先度
- 相性変更は命中・ダメージ判定時に適用。
- いかく無効は入場時の能力変化処理時に適用。

## テスト項目

- [ ] ゴースト相手に ノーマル技 が等倍で通る
- [ ] ゴースト相手に かくとう技 が等倍で通る
- [ ] 複合タイプ相手で、ゴースト以外の相性が正しく残る
- [ ] 第8世代以降想定で いかく を受けても こうげき が下がらない
- [ ] 特性変化が起きる連続攻撃で、後続ヒットの判定が変わる
