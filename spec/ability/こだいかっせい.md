# 仕様書: こだいかっせい

## 2. 基本効果
にほんばれ状態、またはブーストエナジーを持っているとき、自身の最も高い能力が上昇する。

## 3. 詳細仕様

### 発動条件
- 以下のいずれかで発動：
  - にほんばれ状態
  - ブーストエナジー所持（発動時に消費）
- 条件成立時に一度だけ発動

### 上昇対象能力
- 自身の実数値で最も高い能力を対象とする
- 対象：
  - 攻撃 / 防御 / 特攻 / 特防 / 素早さ
- 同値の場合は内部優先順位で決定

### 上昇量
- 攻撃・防御・特攻・特防：1.3倍
- 素早さ：1.5倍
- ランク変化ではなく実数値補正

### 効果の維持
- にほんばれによる発動：
  - 天候が終了すると効果も終了
- ブーストエナジーによる発動：
  - 場を離れるまで持続

### 再発動
- 天候による場合：
  - 再度にほんばれになれば再発動
- ブーストエナジーは1回限り

### 無効化・上書き
- 特性無効化状態では発動しない／効果が消失する
- 無効化解除後、条件を満たせば再発動する
- 特性のコピー・交換は可能

### 交代・入場時
- 入場時に条件を満たしていれば発動
- 交代すると効果はリセットされる

## 5. テスト項目（実装検証用）

### 発動ケース
- にほんばれ状態で入場時に能力が上昇する
- ブーストエナジー所持で能力が上昇する
- 最も高い能力が正しく選択される

### 非発動ケース
- 条件を満たさない場合は発動しない
- 特性無効化状態で発動しない

### 境界ケース
- 同値能力時に正しく優先順位が適用される
- 天候終了で効果が解除される
- 再度天候成立で再発動する
