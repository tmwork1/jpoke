# 仕様書: ノーガード

調査日: 2026-04-30

## 2. 基本効果（1段落）
この特性を持つポケモンが使う技と、この特性を持つポケモンに向けられた技は、命中判定を経ず必中扱いになる。

## 3. 詳細仕様
- 発動範囲: 「使用者がノーガード」または「対象がノーガード」のいずれかで命中判定を必中化。
- 影響外: タイプ相性無効、まもる系、技不発、対象不在などの失敗理由は無視できない。
- 半無敵状態: そらをとぶ/あなをほる等で対象不在のターンは必中化しても当たらない。
- 一撃必殺技: 命中判定部分には適用されるが、レベル条件など別失敗条件は残る。
- かたやぶり相互作用: かたやぶりの有無に関わらずノーガードの必中効果は適用される。

## 4. 処理順・優先度
- 適用タイミング: 命中率計算直前で必中フラグ化。
- 同時効果: 命中・回避ランク、すながくれ等の命中補正より優先して命中確定。
- 範囲技: ノーガード対象への命中判定のみ必中化。

## 5. テスト項目（実装検証用）
- 発動ケース: 低命中技が必中になる。
- 非発動ケース: まもる系、半無敵状態などは通常通り防がれる。
- 境界ケース: 一撃必殺技、範囲技、相互ノーガードでの判定順。
