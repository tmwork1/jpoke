# 仕様書: ちからずく

参照URL: https://wiki.pokemonwiki.com/wiki/ちからずく

## 基本効果
追加効果のある攻撃技を使うと、追加効果を失う代わりに技威力が 1.3 倍になる。

## 詳細仕様

### 対象技
- 基本は「追加効果を持つ攻撃技」。
- 追加効果が複数あっても威力補正は 1.3 倍のまま。
- 一部の技は世代・内部フラグにより追加効果がなくても対象になる場合がある。

### 無効化される効果
- ひるみ・状態異常・能力変化などの追加効果は発動しない。
- 追加効果以外の本体効果（優先度・吸収・反動・タイプ変化など）は通常どおり。

### 持ち物・特性との相互作用
- ちからずくで威力上昇した技は、いのちのたま の反動を受けない（与ダメージ増加は有効）。
- かいがらのすず の回復は発生しない。
- ちからずくで無効化される追加効果に依存する受動発動（一部特性/持ち物）は発動しない。

### 特性喪失時の扱い
- 技の解決中に ミイラ などで特性が変化しても、すでに確定した技威力補正は維持される。

## 処理順・優先度
- 技選択後、威力計算時に 1.3 倍を適用。
- 追加効果判定時は該当効果をスキップ。

## テスト項目

- [ ] 追加効果つき攻撃技の威力が 1.3 倍になる
- [ ] その技の追加効果（状態異常/能力変化/ひるみ）が発生しない
- [ ] 追加効果以外の本体効果は維持される
- [ ] いのちのたま反動が発生しない
- [ ] かいがらのすず回復が発生しない
- [ ] 技中に特性が変化しても当該技の補正挙動が破綻しない
