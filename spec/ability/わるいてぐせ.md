# 仕様書: わるいてぐせ

参照URL: https://wiki.pokemonwiki.com/wiki/わるいてぐせ

## 基本効果
| 項目 | 内容 |
|---|---|
| 効果 | 直接攻撃を受けたとき、攻撃してきた相手の持ち物を奪う |
| 発動制限 | 自身が道具を所持している場合は発動しない |
| 発動タイミング | ダメージ計算終了後 |

---

## 詳細仕様
### 発動条件
| 条件 | 詳細 |
|---|---|
| 攻撃種別 | 直接攻撃を受けた場合 |
| 所持条件 | 自身が道具未所持 |
| 相手状態 | 相手が道具を所持している |

### 無効条件
| 分類 | 内容 |
|---|---|
| 自身条件 | 自分が道具を持っている |
| 戦闘不能 | 攻撃でひんしになった場合は発動しない（ただし反動等で相手がひんしなら発動） |
| 攻撃種別 | 非接触技では発動しない |
| 接触無効化 | 特性「えんかく」、持ち物「パンチグローブ」により非接触化された場合 |
| 身代わり | 自分がみがわりで防いだ場合は発動しない |

---

## 処理順・優先度
| 順序 | 内容 |
|---|---|
| 1 | ダメージ計算 |
| 2 | 相手の反動・回復処理（いのちのたま・かいがらのすず等） |
| 3 | わるいてぐせ発動 |
| 4 | 一部アイテムの発動判定（しろいハーブ等） |

### 詳細処理
| ケース | 処理 |
|---|---|
| 連続攻撃 | 最終ヒット後に発動 |
| いのちのたま | 反動ダメージ後に奪取 |
| かいがらのすず | 回復後に奪取 |
| パワフルハーブ / ジュエル | 先に消費されるため奪えない |
| きあいのタスキ等消費 | 消費直後に発動 |
| だっしゅつボタン | 交代が優先され発動しない |
| とんぼがえり等 | 交代と同時に発動し、退場した相手から奪う |

---

## 競合・上書き・例外
### 特性・状態との競合
| 対象 | 挙動 |
|---|---|
| ねんちゃく | 発動はするが奪えない |
| ねんちゃく（相手ひんし） | 奪取可能 |
| かがくへんかガス | 無効中は発動せず、解除後に発動 |
| さまようたましい | 特性交換後、相手側のわるいてぐせが発動 |

### アイテム競合
| ケース | 挙動 |
|---|---|
| どろぼう / ほしがる / マジシャン | 奪われた直後に奪い返す |
| はたきおとす / ついばむ / むしくい | 自分の道具消失後に即発動 |
| くっつきバリ | 相手に渡った直後に奪い返す |

### 能力変化アイテム
| アイテム | 挙動 |
|---|---|
| しろいハーブ | 判定はわるいてぐせ後 |
| だっしゅつパック | 同上 |
| ものまねハーブ | 同上 |

## テスト項目
- [ ] 接触技被弾時に自分が道具未所持なら奪取する
- [ ] 自分が道具所持時は発動しない
- [ ] 非接触攻撃では発動しない
- [ ] ねんちゃく相手には発動しても奪えない

---

## 相互作用
### 接触判定関連
| 条件 | 結果 |
|---|---|
| ぼうごパット | 接触扱いのため発動し奪取可能 |
| 非接触化（えんかく等） | 発動しない |

### 行動制約
| 条件 | 結果 |
|---|---|
| こだわり系取得 | そのターンの行動で技固定 |
| とつげきチョッキ取得 | そのターンのみ変化技使用可能 |
