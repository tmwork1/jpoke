# 仕様書: へんげんじざい

## 1. 基本情報
- 参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B8%E3%82%93%E3%81%92%E3%82%93%E3%81%98%E3%81%96%E3%81%84
- 種別: 技タイプ追従の自己タイプ変更

## 2. 基本効果（1段落）
技を使用する直前に、自分のタイプをその技のタイプへ変更する。交代すると元のタイプへ戻る。第九世代仕様として、1回の場滞在につき1回だけ発動する制約を持つ。

## 3. 詳細仕様
### 発動条件
- 行動可能で技が実行段階に入ること。
- 未発動フラグ（その場滞在で未使用）を満たすこと。

### 非発動条件
- 行動不能（ひるみ・麻痺行動不能等）で技が出なかったとき。
- テラスタル中（SV仕様）で発動不可とする場合。

### 無効化・上書き
- 特性無効化中は発動しない。
- 特性変更後は新特性の仕様を適用。

### 交代・入場時
- 交代時にタイプを初期化し、発動回数フラグをリセットする。

### その他の特性との相互作用
- リベロと同系統。共通実装（タイプ変更ロジック）を利用する。

## 4. 処理順・優先度
- 発動タイミング: 技効果解決前（タイプ一致補正計算より前）。
- 処理順: 行動成立判定 -> へんげんじざい判定 -> タイプ変更 -> 技威力/補正計算。
- すばやさは参照しない。

## 5. テスト項目（実装検証用）
### 発動ケース
- 異なるタイプの攻撃技を選ぶと、先にタイプが変わり一致補正が乗る。
- 変化技でもタイプ変更が発生する。

### 非発動ケース
- 行動不能時はタイプが変わらない。
- 同一場滞在で2回目以降は発動しない（SV仕様）。

### 境界ケース
- 交代後は再び1回発動できる。
