# 仕様書: ノーマルスキン

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%83%8E%E3%83%BC%E3%83%9E%E3%83%AB%E3%82%B9%E3%82%AD%E3%83%B3
調査日: 2026-05-02

## 2. 基本効果（1段落）
全てのわざがノーマルタイプとなり、タイプを変えた技の威力が1.2倍になる

## 3. 詳細仕様
- 発動条件: 技を使用するとき、使用する技のタイプをノーマルタイプに変換する。元々ノーマルタイプ以外の技を変換した場合は威力が4915/4096倍になる。
- 元々ノーマルタイプの技を使ったときはタイプ変換が起きないため、威力は上昇しない。
- わるあがきを使用したときは発動しない。
- 変化技にも発動する。でんじはをノーマルタイプに変えた場合、ゴーストタイプに無効になる（相性の影響を受ける変化技はでんじはのみ）。
- 無効化・上書き: かがくへんかガスで無効化される。特性変更を受けた場合は変更後の特性を優先する。

## 4. 処理順・優先度
- 発動タイミング: ダメージ計算・命中判定の前に技のタイプと威力を変換する。
- 同時発動時の優先度: 他の威力補正（タイプ一致ボーナス等）と乗算で適用する。

## 5. テスト項目（実装検証用）
- 発動ケース: ほのお技などがノーマルタイプ相性で当たり、威力が4915/4096倍されること。
- 非発動ケース: 元々ノーマルタイプの技を使ったとき威力が上昇しないこと。わるあがきには発動しないこと。
- 境界ケース: かがくへんかガス下では技のタイプ変換が起きないこと。
