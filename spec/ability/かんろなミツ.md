# 仕様書: かんろなミツ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%93%E3%82%8D%E3%81%AA%E3%83%9F%E3%83%84

## 2. 基本効果（1段落）
戦闘中に初めて場に出たとき、相手全体の回避率を1段階下げる。発動は戦闘中1回のみで、再登場しても再発動しない。 :contentReference[oaicite:0]{index=0}

## 3. 詳細仕様

### 発動条件・基本挙動
- 「その戦闘で初めて場に出たとき」に発動 :contentReference[oaicite:1]{index=1}
- 効果：
  - 相手全体の回避率を1段階低下
- ダブル・トリプルバトルでは相手全員が対象
- 発動時に相手が存在しない場合は発動しない
  - その後相手が出ても発動しない :contentReference[oaicite:2]{index=2}

### 発動回数制限
- 戦闘中1回のみ発動可能 :contentReference[oaicite:3]{index=3}
- 一度発動したポケモンは：
  - 交代・ひんし後に再登場しても発動しない :contentReference[oaicite:4]{index=4}
- 発動して効果が無効化された場合（例：クリアボディ）でも：
  - 「発動済み」として扱われる :contentReference[oaicite:5]{index=5}

### 無効化・上書き
- かがくへんかガス等で特性が無効化されている場合：
  - 発動しない
- 特性を後から得た場合：
  - 発動条件を満たせばその時点で発動
- 特性のコピー・交換で取得した場合も発動可能
- かたやぶりの影響は受けない（自己発動型）

### 効果の適用対象
- 回避率低下は能力ランク変化として扱う
- みがわり状態の相手には無効 :contentReference[oaicite:6]{index=6}
- 姿を隠している相手（例：そらをとぶ中など）にも適用 :contentReference[oaicite:7]{index=7}

### 交代・入場時
- 入場時に即時発動
- 他の「場に出たとき発動特性」と同様に処理
- 発動順はすばやさ順 :contentReference[oaicite:8]{index=8}

### その他の特性との相互作用
- いかくと同様の発動枠（能力低下対象が異なるだけ） :contentReference[oaicite:9]{index=9}
- クリアボディ等：
  - 回避率低下が無効化される
  - ただし発動自体は消費される
- かちき等：
  - 回避率低下が成功した場合に発動条件を満たす

## 4. 処理順・優先度
- 発動タイミング：入場直後
- 処理順：
  1. 入場
  2. 発動判定
  3. 回避率ランク低下適用
- 同時発動時の優先度：
  - 補正込みすばやさ順（いかくと同様） :contentReference[oaicite:10]{index=10}
- すばやさは発動順に影響

## 5. テスト項目（実装検証用）

### 発動ケース
- 初登場時に相手全体の回避率が-1される
- ダブルバトルで複数対象に適用される
- 特性を後から得た場合に即時発動する

### 非発動ケース
- 戦闘中2回目以降の登場では発動しない
- 相手がいない状態で発動しない
- 特性無効化状態で発動しない
- みがわり状態の相手に効果が適用されない

### 境界ケース
- クリアボディ等で無効化されても再発動しない
- 発動順がすばやさ順になる
- 姿を隠している相手にも適用される
- 発動機会を逃した場合（相手不在）は次回登場で発動可能
