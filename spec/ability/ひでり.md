# 仕様書: ひでり

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%B2%E3%81%A7%E3%82%8A
調査日: 2026-04-30

## 2. 基本効果（1段落）
場に出たとき、天気をにほんばれ状態にする。第6世代以降は通常5ターン（あついいわで8ターン）継続する。

## 3. 詳細仕様
- 発動条件: 入場時に発動し、特性が後からひでりに変化した場合も即時発動する。
- 持続: 発動者が場を離れても天気は維持される。
- 上書き関係: 既ににほんばれ状態なら継続ターン更新は行わない。
- 強天候との関係: おおひでり/おおあめ/らんきりゅう中は特性表示のみで晴れ化しない。
- 無効化: かがくへんかガス中は発動しない。

## 4. 処理順・優先度
- 発動タイミング: 入場時特性処理。
- 同時発動時の優先度: 天候始動特性同士は後に処理された側の天気が最終的に残る。
- 継続ターン: 第6世代以降はターン終了時に消費。

## 5. テスト項目（実装検証用）
- 発動ケース: 入場でにほんばれ状態になる。
- 非発動ケース: 強天候中では天気変更が発生しない。
- 境界ケース: 既存晴れ状態で再発動してもターン更新されない。
