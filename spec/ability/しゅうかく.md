# 仕様書: しゅうかく

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%97%E3%82%85%E3%81%86%E3%81%8B%E3%81%8F

## 基本効果
ターン終了時に、自分が最後に消費したきのみを復活させる。基本確率は 50% で、晴れ（にほんばれ / おおひでり）では 100% になる。

## jpoke実装スコープ
- 対象: バトル中の効果のみ
- 非対象: 第八世代のフィールド遭遇効果（野生ポケモン出現率補正）

## 詳細仕様

### 発動条件
- ターン終了時に判定する。
- 持ち物を持っていると発動しない。
- 判定は毎ターン1回のみ。
- 場にいるときのみ有効。控えでは発動しない。
- 姿を隠しているターンでも発動する。
- きのみ消費時点の特性は問わず、判定時点で現在の特性がしゅうかくなら発動可能。

### 発動率
- 通常は 50%。
- にほんばれ / おおひでり中は 100%。
- ターン経過で晴れが切れるターンは、判定時点で晴れでないため 50%。
- ノーてんき / エアロックが場にいる場合、晴れ状態でも 50%。

### 復活対象（最後に消費したきのみ）
- 自身が消費したきのみのみ対象。
- 自身のなげつける / しぜんのめぐみで消費したきのみは対象。
- ほおばる / おちゃかいで食べたきのみは対象。
- 相手のやきつくす / はたきおとす / むしくい / ついばむ / ふしょくガスで失われたきのみは対象外。
- 相手から受けたなげつけるで消費したきのみは対象外。
- 自身がむしくい / ついばむで食べたきのみは対象外。
- トリック / すりかえ / どろぼう / ほしがる / ギフトパス、またはマジシャン / わるいてぐせ / ものひろい / きょうせい経由で移動したきのみは対象外。

### 復活後のきのみ使用
- 復活したきのみは、使用条件を満たしていれば即座に使用する。
- 使用条件を満たさない場合は復活のみ行い、消費しない。
- さしおさえ / マジックルーム / きんちょうかん下でも復活自体は起こるが、きのみ使用はできない。

### 交代・瀕死との関係
- きのみ消費後に交代しても、再登場後は発動可能。
- きのみ消費後に瀕死になり、復活して再登場した場合も発動可能。

### 複数回消費時の対象更新
- 最後に消費したきのみのみを対象にする。
- 途中で別の道具を入手し、その道具を消費した場合は対象を更新する。
- 途中で入手した道具を消費せずに手放した場合、以前の消費きのみを対象にできる。
- 途中で入手した道具をはたきおとす等で失った場合は「消費扱い」ではない。

### 発動順
- しゅうかく / ものひろい / ナイトメア等の同タイミング効果は、補正込み素早さ順で処理される。
- トリックルーム中は遅い順。
- トリックルームが解除されるターンは、解除後の素早さ順で判定される。

## 実装時の優先確認項目
1. ターン終了フェーズ内での判定位置が、天候カウント減算後になっていること。
2. 復活対象が「消費」と「喪失」を区別して追跡されること。
3. 復活直後の即時使用が既存きのみ発動ルールと矛盾しないこと。
4. かがくへんかガス / とくせいなしによる無効化中は不発になること。

## テスト観点
- 通常天候で 50% 成功 / 失敗の両ケース。
- 晴れ中は 100% 発動。
- 晴れが切れるターンは 50% 発動。
- ノーてんき / エアロック下では晴れ中でも 50% 発動。
- 持ち物あり / 控え / 復活候補なしで非発動。
- 復活後に使用条件を満たすきのみの即時使用。
- 喪失経路（やきつくす、はたきおとす等）で誤復活しない。
- 交代・瀕死復帰後に再発動できる。

## 補足
- 隠れ特性限定。
- この特性を持つポケモンは全てくさタイプを含む。
- フィールド遭遇効果は SV では発動しない。
