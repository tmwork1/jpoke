# 仕様書: おみとおし

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8A%E3%81%BF%E3%81%A8%E3%81%8A%E3%81%97

## 基本効果
場に出たときに敵全体の持ち物を確認できる。

## 詳細仕様

### 発動条件
- 相手の場に持ち物を持つポケモンがいるときにのみ発動
- 持ち物がないポケモンが交代してきた場合は発動しない

###複数対象の判定
- ダブルバトル：相手全員の持ち物が判明（Gen6以降）
- トリプルバトル：自分と対角の位置の敵の持ち物も判定対象
- みがわり / 姿を隠している 状態でも持ち物を読み取れる

### 他の特性との優先度関係（Gen5以降）
テラスチェンジ → かがくへんかガス → きんちょうかん → じんばいったい → **おみとおし** → ぎょぐん / リミットシールド

**かがくへんかガス発動中はおみとおしは発動しない**

## 処理順・優先度
- 入場時に自動発動（他特性との優先度あり）

## テスト項目

- [ ] 敵の持ち物を確認できる
- [ ] 敵が持ち物を持っていない場合は発動しない
- [ ] ダブルバトルで複数敵の持ち物を同時確認
- [ ] かがくへんかガス 発動中は発動しない
- [ ] テラスチェンジ より後に発動
