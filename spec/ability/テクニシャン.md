# 仕様書: テクニシャン

調査日: 2026-04-30

## 2. 基本効果（1段落）
元の威力が60以下の技の威力を1.5倍にする。

## 3. 詳細仕様
- 判定基準: 基本は「技の元威力 <= 60」を基準に適用判定する。
- 可変威力技: 可変前提の技は個別仕様に従う（例: ソーラービーム天候半減時は対象外）。
- 他補正との関係: 持ち物・味方補助・フィールド等で最終威力が60超になっても、元威力条件を満たせば効果は維持。
- 連続技: 1発あたりの威力で判定。
- 例外技: ふくろだたき・わるあがき等は世代差/個別仕様を実装時に分岐管理する。

## 4. 処理順・優先度
- 適用タイミング: 威力補正レイヤーで乗算。
- 同時補正: 他の威力補正（オーラ系、バッテリー等）と合成して最終威力化。
- 分類: ダメージ補正ではなく威力補正として扱う。

## 5. テスト項目（実装検証用）
- 発動ケース: 威力60技のダメージ上昇。
- 非発動ケース: 威力61以上の固定威力技で不発。
- 境界ケース: 可変威力技、連続技、例外技で期待値一致。
