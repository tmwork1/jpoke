# 仕様書: いろめがね

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%84%E3%82%8D%E3%82%81%E3%81%8C%E3%81%AD

## 基本効果
使用した技が「効果いまひとつ」だったとき、与えるダメージが2倍になる。相性判定そのものは変化しない。

## 詳細仕様

### 相性判定
- 複合タイプ相手は最終的な相性が「今ひとつ」のときに発動
  - 例：むしタイプの技をはがね/ひこうに使う → 半減+2倍=等倍 → 発動しない
  - 複合タイプの片方が「今ひとつ」でも、もう片方が「抜群」なら最終的に等倍以上 → 発動しない

### 全体攻撃技
- 複数対象技を使用したとき、効果が「今ひとつ」だった相手に対してのみダメージが増える
- 「今ひとつ」でない相手へはダメージ増加なし

### 他の相性変更との相互作用
- もりののろい / ハロウィン で追加されたタイプが影響
- タールショット で追加された弱点が影響
- フリーズドライ / フライングプレス による相性変更が影響
- テラスシェル の効果で等倍以上を「今ひとつ」にされたときも発動
- らんきりゅう の効果で等倍を「今ひとつ」にされたときも発動

### 半減効果との区別
- あついしぼう / たいねつ 等、相性判定が変わらない半減効果は発動に影響しない
- ホズのみ 等半減きのみも相性判定を変えないため影響しない

### 遅発ダメージ技
- みらいよち / はめつのねがい のダメージ発生時、使用者が場にいていろめがねを持つなら発動

### 技無効化時
- 技が無効化された場合、特性も発動しない

## 処理順・優先度
- ダメージ計算時に相性補正として適用
- 相性補正の位置により、相性で与えるダメージが2倍になったのと異なる計算結果になることあり

## テスト項目

- [ ] 技が効果いまひとつのときダメージが2倍になる
- [ ] 複合タイプで最終相性が等倍以上なら発動しない
- [ ] 全体技で「いまひとつ」相手のみダメージ増加
- [ ] みらいよち 使用者がいろめがねなら、ダメージ発生時に発動
- [ ] フリーズドライ で相性が変わった影響を受ける
- [ ] テラスシェル で等倍以上を「いまひとつ」にされても発動
