# 仕様書: くいしんぼう

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8F%E3%81%84%E3%81%97%E3%82%93%E3%81%BC%E3%81%86

## 2. 基本効果
HPが通常より高い割合のときでも、回復きのみを早めに使用できる。

## 3. 詳細仕様

### きのみ発動条件変更
- 対象：HP回復系のきのみ（オレンのみ等）
- 通常：
  - HPが最大の1/2以下で発動
- 本特性時：
  - HPが最大の1/2以上でも発動可能（発動ラインが緩和される）
- 発動タイミングは通常のきのみと同様（被弾後など）

### 対象外
- 能力上昇きのみ（フィラのみ等）は対象外
- 状態異常回復きのみも対象外

### 無効化・上書き
- 特性が無効化されている場合（例：かがくへんかガス）：
  - 通常の発動条件に戻る
- 特性のコピー・交換は可能

### 交代・入場時
- 常時効果
- 入場時の即時効果なし
- 場にいる間のみ有効

### その他の特性との相互作用
- きのみの発動条件変更のみで、回復量には影響しない
- きのみ使用制限系（きんちょうかん）には従う（使用できない）

## 5. テスト項目（実装検証用）

### 発動ケース
- 通常より高いHP割合で回復きのみが発動する
- 被弾後すぐにきのみが使用される

### 非発動ケース
- 対象外きのみで発動条件が変わらない
- 特性無効化状態で通常条件に戻る

### 境界ケース
- HPちょうど閾値で正しく発動する
- きんちょうかん下では発動しない
- 回復量が通常と同じである
