# 仕様書: ゆきがくれ

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%86%E3%81%8D%E3%81%8C%E3%81%8F%E3%82%8C
調査日: 2026-04-30

## 2. 基本効果（1段落）
天気がゆきのとき、回避率が1.25倍になる

## 3. 詳細仕様

### 効果
- 天気があられ/ゆきのとき、受ける技の命中率が0.8倍になる。
- タイプに関係なくあられのダメージを受けなくなる。
- フィールド効果: 手持ちの先頭にこのとくせいを持つポケモンがいると、マップのあられが降っている場所での野生ポケモンの出現率が0.5倍になる。

### 説明文
天気が ゆきのとき 回避率が 上がる。

### 特性の仕様
- 第五世代以降での詳細な倍率は3277/4096倍となる。
- 天気があられ/ゆき状態でも、ノーてんき・エアロックのポケモンが場にいるときは発動しない。
- 自身があなをほる・ダイビング状態のときでもゆきがくれは発動する。
- 必中技・一撃必殺技の命中率は変化しない。
- フィールド効果: きよめのおふだ・きよめのおこうを持っている場合はフィールド上での効果はなくなる。

### 備考
- 同じ効果のすなあらし版の特性としてすながくれがある。

## 4. 処理順・優先度
- 発動タイミング: 命中判定フェーズで評価する（相手の命中率に0.8倍補正）。
- 同時発動時の優先度: 同種の効果が重なる場合はゲーム内の一般優先順（行動順・速度・ランダム同値）に従う。
- すばやさ補正との関係: 速度を変更しない特性では行動順に直接影響しない。

## 5. テスト項目（実装検証用）
- 発動ケース: ゆき/あられ状態で受ける技の命中率が3277/4096倍になること。あられダメージを受けないこと。
- 非発動ケース: ゆき/あられ以外の天気では発動しないこと。必中技・一撃必殺技の命中率は変化しないこと。
- 境界ケース: ノーてんき/エアロック存在下、あなをほる/ダイビング状態中の発動確認。
