# かいふくふうじ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | かいふくふうじ（状態変化） |
| 分類 | 状態変化（技「かいふくふうじ」や「サイコノイズ」で付与） |
| 効果概要 | HP回復を伴う処理を無効化する |
| 代表的な持続 | 5ターン（かいふくふうじ起点）/ 2ターン（サイコノイズ起点） |
| 効果対象 | 付与されたポケモン |
| 重ね掛け | 同一状態の重複は不可 |

## 発生・解除・予防

### 発生

- 技「かいふくふうじ」が命中したとき付与される。
- 技「サイコノイズ」でも同種の状態が付与される（持続は2ターン）。

### 解除

| 条件 | 結果 |
|---|---|
| ターン経過（発生源に応じた規定ターン） | 解除 |
| 交代 | 解除 |
| メンタルハーブ（第6世代以降） | 解除 |
| 戦闘不能 | 解除 |

### 予防

- 特性「アロマベール」で予防できる。

## 制限される回復

- 回復技によるHP回復（じこさいせい、ねむる、はねやすめ、ねがいごと回復など）。
- 状態/場による定期回復（ねをはる、アクアリング、グラスフィールド、やどりぎのタネなど）。
- 第5世代以降は特性・持ち物による回復（たべのこし、くろいヘドロ、各種回復きのみ、あめうけざら、アイスボディ、ポイズンヒール、かいがらのすず など）。
- 吸収技の扱いは世代差がある（下記参照）。

## 制限対象外・代表的な例外

- いたみわけによるHP変動は阻害しない。
- 特性「さいせいりょく」による交代時回復は阻害しない（状態をバトンタッチで引き継いだ場合も同様）。
- HP以外の回復（状態異常回復、PP回復など）は通常どおり処理される。
- バッグから使用する回復アイテムによる回復は別系統で、かいふくふうじの対象外。

## 世代差分の要点

| 世代 | 主な差分 |
|---|---|
| 第4世代 | 主に「技によるHP回復」を封じる |
| 第5世代以降 | 特性・持ち物由来のHP回復も封じる |
| 第6世代以降 | 吸収技は「回復しない」ではなく「技自体を選べない/使用できない」扱いに変更 |

## ターン処理メモ

- かいふくふうじ終了ターンでも、一部の定期回復（ねをはる、アクアリング、やどりぎのタネ、グラスフィールド等）がそのターンは回復しない挙動があるため、終了タイミングと回復判定順を分けて管理する。

## 実装メモ（jpoke）

- `ON_BEFORE_HEAL` で回復量を0にすることで回復阻止を実現する。
- ログは `HEAL_BLOCKED` を使用する。
- 持続カウントは `ON_TURN_END_3` の tick で進行する。
