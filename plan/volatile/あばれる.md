# あばれる — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

2〜3ターン間、強制的に「あばれる」を使い続け、終了後にこんらん状態になる。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_MODIFY_MOVE | source:self | 技をあばれるに固定 |
| ON_DAMAGE | source:self | ターン終了後にこんらん付与 |

## テスト状況

## 実装メモ

- 強制行動化は `ON_MODIFY_MOVE` で行い、カウント進行と終了時こんらん付与は `ON_DAMAGE` に寄せている。
- 強制行動ターンの PP 追加消費を起こさない前提で実装されている。

## テスト要件

- [x] あばれる状態では強制行動になる
- [x] 強制行動ターンで PP を追加消費しない
- [x] カウント進行後にこんらんが付与される
- [x] 2ターン継続と3ターン継続の両ケースを確認する
