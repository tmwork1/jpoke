# おんねん — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

おんねん状態の相手を倒すと、相手の持つ技のPPをゼロにして倒したポケモンのPPも残り量に従って消費させる。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_CHECK_ACTION | source:foe | 行動前に状態確認 |
| ON_DAMAGE | source:self | ひんし時にPP消耗処理 |

## テスト状況

## 実装メモ

- `ON_DAMAGE` で被弾側のひんしを拾い、攻撃側が使った技の PP を 0 にする。
- 倒しきれなかった場合は何も起きない設計。

## テスト要件

- [x] おんねん状態の相手を倒すと攻撃技の PP が 0 になる
- [x] PP 消費ログが出る
- [ ] 倒しきれなかった場合は PP が減らない
- [ ] 非通常ケースでも不要発動しないことを確認する
