# さわぐ — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

3ターン間さわぐを使い続け、その間は自分も含め全ポケモンが眠れない。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_MODIFY_MOVE | source:self | さわぐに技を固定 |
| ON_BEFORE_APPLY_AILMENT | | ねむり付与を妨害 |
| ON_BEFORE_APPLY_VOLATILE | | ねむけ付与を妨害 |
| ON_TURN_END_3 | source:self | 3ターン tick → 解除 |
