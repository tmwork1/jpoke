# かなしばり — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

相手が直前に使った技を4ターン間封じる。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_MODIFY_COMMAND_OPTIONS | source:foe | 封じた技をコマンドから除外 |
| ON_CHECK_ACTION | ttacker:self | 封じた技の実行をブロック |
| ON_TURN_END_3 | source:self | 4ターン tick → 解除 |

## テスト要件

- [x] コマンド選択時に封じた技が除外される
- [x] 封じた技を選択しようとすると実行がブロックされる
- [ ] 4ターン後に自動解除される
- [ ] 交代で解除される
