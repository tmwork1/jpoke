# アンコール — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

3ターン間、直前に使った技を使い続けなければならない。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_MODIFY_COMMAND_OPTIONS | source:self | アンコール技以外を除外 |
| ON_MODIFY_MOVE | source:self | 技をアンコール対象に固定 |
| ON_TURN_END_3 | source:self | 3ターン tick → 解除 |

## テスト状況

## 実装メモ

- コマンド選択時に対象技以外を除外し、必要なら `ON_MODIFY_MOVE` で実行技も固定する。
- 持続管理は `tick_volatile` に委譲している。
