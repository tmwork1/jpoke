# あくむ — 実装プラン

## 実装状況

❌ **未実装**

## 概要

- 効果: ねむり状態の対象に毎ターン最大HPの1/4ダメージ
- 付与条件: 対象がねむり状態であること
- 持続: ねむり状態の間継続。ねむり解除と同時に解除

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| `ON_TURN_END_3` | `source:self` | ねむり中なら最大HPの1/4ダメージ。解除判定も実施 |

## 実装概要

### data/volatile.py

```python
"あくむ": VolatileData(
    handlers={
        Event.ON_TURN_END_3: h.VolatileHandler(
            h.あくむ_tick,
            subject_spec="source:self",
            priority=70,
        ),
    }
),
```

### handlers/volatile.py

```python
def あくむ_tick(ctx: BattleContext) -> HandlerReturn:
    mon = ctx.source
    if not mon.has_ailment("ねむり"):
        # ねむり解除済みなら状態も解除
        ctx.battle.remove_volatile(mon, "あくむ")
        return HandlerReturn()
    dmg = max(1, mon.max_hp // 4)
    ctx.battle.modify_hp(mon, -dmg)
    return HandlerReturn()
```
