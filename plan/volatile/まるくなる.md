# まるくなる — 実装プラン

## 実装状況

🐛 **BUG: ハンドラが data/volatile.py に未接続**

## 概要

- 効果: 使用者が硬球状態になり、ころがる・アイスボールの威力が2倍になる
- 持続: 交代・戦闘不能まで

## 現状の問題

`data/volatile.py` の `"まるくなる"` エントリが `handlers={}` となっており、
`handlers/volatile.py` に存在する `まるくなる_power_modifier` 関数が呼ばれていない。

```python
# data/volatile.py — 現状 (誤)
"まるくなる": VolatileData(
    handlers={}
),
```

## 修正方針

`data/volatile.py` の `まるくなる` エントリに `ON_CALC_POWER_MODIFIER` ハンドラを追加する。

```python
# data/volatile.py — 修正後
"まるくなる": VolatileData(
    handlers={
        Event.ON_CALC_POWER_MODIFIER: h.VolatileHandler(
            h.まるくなる_power_modifier,
            subject_spec="attacker:self",
        ),
    }
),
```

## ハンドラ仕様

| イベント | subject_spec | 効果 |
|---|---|---|
| `ON_CALC_POWER_MODIFIER` | `attacker:self` | ころがる・アイスボール使用時に威力2倍 |
