# じごくづき — 実装プラン

## 実装状況

🐛 **BUG: VOLATILES キー名に typo あり（づ → ず）**

## 概要

- 効果: 対象は2ターンの間、音の技を使えなくなる
- 持続: 2ターン、または交代で解除

## 現状の問題

`data/volatile.py` のキーが `"じごくずき"` (ず) になっているが、
正しくは `"じごくづき"` (づ) のはず。技名のローマ字は *Hell Roll* → じごくづき。

加えて、ハンドラ関数名が混在している：
- `h.じごくずき_restrict_commands` (ず) — `tick_volatile` の name 引数も `"じごくずき"` (ず)
- `h.じごくづき_check_action` (づ) — 正しい表記

```
data/volatile.py:
  "じごくずき"  ← 誤
  tick_volatile(name="じごくずき")  ← 誤
  h.じごくずき_restrict_commands   ← 誤

handlers/volatile.py:
  じごくずき_restrict_commands   ← 誤
  じごくづき_check_action        ← 正
```

## 修正方針

1. `data/volatile.py`: キー `"じごくずき"` → `"じごくづき"` に変更
2. `data/volatile.py`: `tick_volatile(name="じごくずき")` → `tick_volatile(name="じごくづき")` に変更
3. `data/volatile.py`: `h.じごくずき_restrict_commands` → `h.じごくづき_restrict_commands` に変更
4. `handlers/volatile.py`: 関数名 `じごくずき_restrict_commands` → `じごくづき_restrict_commands` に変更

## ハンドラ仕様（修正後）

| イベント | subject_spec | 効果 |
|---|---|---|
| `ON_MODIFY_COMMAND_OPTIONS` | `source:self` | コマンド選択肢から音技を除外 |
| `ON_CHECK_ACTION` | `attacker:self` | 選択された音技を実行不可にする |
| `ON_TURN_END_3` | `source:self` | 2ターン経過で状態解除 |
