# まもる — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

そのターン、相手の技をすべて無効化する。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_CHECK_MOVE | defender:self | 技をブロック（まもる効果） |
| ON_TURN_END_1 | source:self | ターン終了で解除 |

## テスト状況

## 実装メモ

- `ON_CHECK_MOVE` で相手の攻撃技を遮断し、1ターン後に解除される典型的な保護状態。
- 自分対象の変化技は通す分岐を持つ。

## テスト要件

- [x] 相手の攻撃技を防ぐ
- [x] 保護成功ログが出る
- [x] 自分対象の変化技は防がない
- [x] ターン終了で状態が解除される

## 残課題

- 保護状態の解除自体は個別未テスト。
