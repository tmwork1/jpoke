# かいふくふうじ — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

5ターン間、HP回復技・アイテムによる回復を封じる。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_BEFORE_HEAL | source:self | 回復イベントをブロック |
| ON_TURN_END_3 | source:self | 5ターン tick → 解除 |

## テスト状況

## 実装メモ

- 回復直前フックを一律に止める設計で、回復技・定期回復・アイテム回復が同じ入口を通る前提になっている。
- 持続管理は `tick_volatile` に委譲している。

## テスト要件

- [x] HP回復がブロックされる
- [x] `HEAL_BLOCKED` ログが出る
- [ ] 5ターン経過で解除される
- [ ] アクアリング等の定期回復も止まる
- [ ] `いたみわけ` による HP 変動はブロックしない
