# 特性実装計画: はじまりのうみ

更新日: 2026-05-01

## 仕様参照

- `spec/ability/はじまりのうみ.md`
- `spec/global_field/おおあめ.md`

## 実装ゴール

- `Event.ON_SWITCH_IN` で おおあめ を発動する。
- おおあめ 中は ほのおタイプ攻撃技を失敗扱いにする。
- 通常天候では上書き不可、特殊天候（おおひでり / らんきりゅう）では上書き可を満たす。

## 実装方針

- あめふらし実装をベースに、強天候タイプへ切り替える。
- 技失敗判定は おわりのだいち と対称な実装にそろえる。
- 天候依存技（かみなり / ぼうふう / ウェザーボール / ソーラー系）は既存の雨効果導線へ乗せる。

## テスト観点

1. 登場時に おおあめ が発動する。
2. ほのおタイプ攻撃技が失敗する。
3. 通常天候で上書きできない。
4. おわりのだいち との競合で後発天候が残る。
5. ノーてんき / エアロック同居時に状態は残り効果は止まる。

## 保留・論点

- 強天候名だけで分岐が増えないよう、天候効果判定の共通化を先行する。
