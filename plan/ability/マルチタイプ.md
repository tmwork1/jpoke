# 特性実装計画: マルチタイプ

更新日: 2026-05-01

## 仕様参照

- `spec/ability/マルチタイプ.md`

## 実装ゴール

- アルセウスがプレートに応じてタイプ・フォルムを変更する。
- 専用道具の奪取・交換を防ぐ。
- 特性が無効化されない仕様を反映する。

## 実装方針

- `Pokemon.types` の TODO を解消し、現在タイプ解決へ専用道具ロジックを組み込む。
- フォルム変更は `Pokemon.set_form()` で統一する。
- 道具保護は `ItemManager.can_change_item()` に条件追加する。

## 変更候補

- `src/jpoke/model/pokemon.py`
- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `src/jpoke/core/item_manager.py`
- `src/jpoke/data/move.py`
- `tests/test_ability.py`
- `tests/test_item.py`
- `tests/test_move.py`

## テスト観点

1. プレートでタイプとフォルムが変わる。
2. プレートの奪取・交換不可。
3. かがくへんかガス などで無効化されない。
4. さばきのつぶて のタイプ連動。

## 保留・論点

- さばきのつぶて を同時実装するか別タスク分離するかを着手時に確定する。
