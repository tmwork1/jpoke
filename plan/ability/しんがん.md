# しんがん 実装計画

## スコープ
- 優先度: -
- 仕様書: `spec/ability/しんがん.md`
- 実装方針: 仕様の発動条件と例外条件を先に固定し、イベント単位で段階実装する。

## 実装対象イベント（第一候補）
- `Event.ON_BEFORE_MODIFY_STAT`
- `Event.ON_MODIFY_STAT`
- `Event.ON_CALC_POWER_MODIFIER`
- `Event.ON_CALC_ATK_MODIFIER`

## 仕様要点（実装に反映）
- ゴースト タイプに対して使う ノーマル ・ かくとう タイプの技の 相性 が、無効ではなく等倍になる。
- 他のポケモンから 命中率 の ランク補正 を下げられない。相手の 回避率 ランクを無視して攻撃できる。
- きもったま と するどいめ を複合したような効果がある。ただし、きもったまと異なり いかく を防ぐ効果はない。
- ノーマル / かくとう 技の 相性 に関する効果については、 きもったま#特性の仕様 を参照とする。
- 命中率 / 回避率 に関する効果については、 するどいめ#特性の仕様 を参照とする。

## 実装フロー
1. `src/jpoke/data/ability.py` の `しんがん` に上記イベントの `handlers` を登録する。
2. `src/jpoke/handlers/ability.py` に `しんがん` の本体処理を追加する。
3. 既存の共通ロジックで再利用できる処理は `src/jpoke/handlers/common.py` に分離する。
4. 特性無効化（`かがくへんかガス` / `とくせいなし`）と交代時リセットの相互作用を確認する。

## テスト観点
- 発動条件を満たすケースでのみ効果が出る。
- 仕様の例外条件（無効化・対象外条件）で誤発動しない。
- 解除条件を満たしたタイミングで効果が停止する。
- 既存システム（場状態・交代・ダメージ計算）と競合しない。
