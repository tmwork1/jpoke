# かんそうはだ 実装計画

## スコープ
- 優先度: -
- 仕様書: `spec/ability/かんそうはだ.md`
- 実装方針: 仕様の発動条件と例外条件を先に固定し、イベント単位で段階実装する。

## 実装対象イベント（第一候補）
- `Event.ON_FIELD_CHANGE`
- `Event.ON_CHECK_HIT_SUBSTITUTE`
- `Event.ON_CHECK_SUBSTITUTE`
- `Event.ON_CHECK_DEF_ABILITY`

## 仕様要点（実装に反映）
- みず タイプのわざを受けても無効化し、代わりに最大 HP の1/4を回復する (小数点以下切り捨て)。
- ほのお タイプの 攻撃技 を受けたときの 威力 が5/4倍になる。
- 天気 が あめ ・ おおあめ 状態のとき、毎 ターン 終了時に最大HPの1/8を回復する (小数点以下切り捨て)。
- 天気が にほんばれ ・ おおひでり 状態のとき、毎ターン終了時に最大HPの1/8のダメージを受ける (小数点以下切り捨て)。
- 以下のように、 みず 技を無効化する効果は ちょすい と共通している。詳細な仕様は ちょすい#特性の仕様 を参照とする。

## 実装フロー
1. `src/jpoke/data/ability.py` の `かんそうはだ` に上記イベントの `handlers` を登録する。
2. `src/jpoke/handlers/ability.py` に `かんそうはだ` の本体処理を追加する。
3. 既存の共通ロジックで再利用できる処理は `src/jpoke/handlers/common.py` に分離する。
4. 特性無効化（`かがくへんかガス` / `とくせいなし`）と交代時リセットの相互作用を確認する。

## テスト観点
- 発動条件を満たすケースでのみ効果が出る。
- 仕様の例外条件（無効化・対象外条件）で誤発動しない。
- 解除条件を満たしたタイミングで効果が停止する。
- 既存システム（場状態・交代・ダメージ計算）と競合しない。
