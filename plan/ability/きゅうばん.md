# 特性実装計画: きゅうばん

更新日: 2026-05-03
実装状況: **✅ 完了**

## 仕様参照

- `spec/ability/きゅうばん.md`

## 実装ゴール

✅ 相手の技やレッドカードによる強制交代を防ぐ。
✅ 自分からの交代（だっしゅつボタン・使うと交代する技）は制限しない。
✅ かたやぶりの効果がある技には無視される。

## 実装方針

✅ **特性側でのハンドラ実装を優先**
- ON_HIT イベントで強制交代技の効果をキャンセル
- 防御側がきゅうばんを持っていれば、技のヒット値をFalseに変更
- 優先度=200 で他のハンドラより先に実行される

## 実装完了内容

### src/jpoke/handlers/ability.py
```python
def きゅうばん_check_blow(battle: Battle, ctx: BattleContext, value: Any) -> HandlerReturn:
    """きゅうばん特性: 強制交代技の効果を防ぐ。"""
    if value and ctx.check_def_ability_enabled(battle):
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)
```

### src/jpoke/data/ability.py
```python
"きゅうばん": AbilityData(
    flags=["mold_breaker_ignorable"],
    handlers={
        Event.ON_HIT: h.AbilityHandler(
            h.きゅうばん_check_blow,
            subject_spec="defender:self",
            priority=200,
        ),
    },
),
```
