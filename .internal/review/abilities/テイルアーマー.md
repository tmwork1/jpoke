# テイルアーマー レビュー結果

## 仕様書
- 変更なし。`.internal/spec/abilities/テイルアーマー.md` は一次情報（.internal/wiki/abilities/テイルアーマー.html）の
  「効果」「特性の仕様」「備考」節と一致しており、誤り・欠落はなかった。
  - 効果: この特性を持つポケモンが場にいる間、自分を含む味方全員は敵から優先度+1以上の技を受け付けない。
  - 特性の仕様: じょおうのいげん / ビビッドボディと同じ効果（じょおうのいげん#特性の仕様を参照）。
  - 備考: リキキリンの専用特性。

## 実装計画書
- 変更なし。`.internal/plan/abilities/テイルアーマー.md` は仕様書と整合しており、
  じょおうのいげんと同一ハンドラを共有する方針・登録内容も実装と一致していた。

## 実装（handlers/ / data/）
- 変更なし。`src/jpoke/handlers/ability.py` の `じょおうのいげん_block_priority`
  （じょおうのいげん/テイルアーマー/ビビッドボディが共有）は、
  `ctx.move.target != "foe"` で自己・味方対象技を除外し、
  `battle.speed_calculator.calc_move_priority` で実効優先度を判定して
  0以下なら通す実装になっており、仕様（優先度+1以上の相手単体対象技のみ無効化）と一致している。
- `src/jpoke/data/ability.py` の `テイルアーマー` エントリは
  `flags={"mold_breaker_ignorable"}`、`Event.ON_TRY_MOVE_1` / `subject_spec="defender:self"` /
  `priority=40` で、`じょおうのいげん` / `ビビッドボディ` のエントリと完全に一致していた。
