# アシッドボム レビュー結果

## 仕様書
- 新規作成（`.internal/spec/moves/アシッドボム.md`）。ポケモンWiki一次情報（どく/特殊/威力40/
  命中100/PP20/相手1体/優先度0/非接触・弾のわざ、追加効果100%でとくぼう-2段階）を反映。
  King's Rock/するどいキバが第五世代以降このような既存追加効果ありの技にも発動する点、
  ちからずくとの関係も記載。

## 実装計画書
- 新規作成（`.internal/plan/moves/アシッドボム.md`）。既存実装（`data/moves/move_a.py` の
  `アシッドボム` エントリ、`handlers/move_attack.py` の
  `アシッドボム_sharply_lower_defender_spd`）を仕様と照合し、一致していることを確認。
  修正は不要だった。

## 実装（handlers/ / data/）
- 変更なし。`data/moves/move_a.py` の型（どく）・威力（40）・命中（100）・PP（20）・
  `flags={"bullet"}`・`Event.ON_DAMAGE_HIT` への登録は仕様・`src/jpoke/data/ps-champ-ja/moves.json`
  と一致していることを確認した。`handlers/move_attack.py` の
  `アシッドボム_sharply_lower_defender_spd` は共通ヘルパー `modify_defender_stats` を
  `chance` 省略（デフォルト100%）で呼び出しており、仕様と一致。

## リーサル計算
- 変更なし。`handlers/lethal.py` の `アシッドボム_reduce_spd`（`LethalEvent.ON_HIT` 登録）は
  命中後に無条件でとくぼうを2段階下げており、100%発動の仕様と一致することを確認した。

## テスト
- `tests/moves_attack/test_move_a.py` に
  `test_アシッドボム_とくぼう2段階低下が発動する` を追加（実際のバトル実行で
  相手のとくぼうランクが-2になることを確認）。

  ランク低下によって2発目のダメージが増加することを確認しており、修正不要と判断した。
