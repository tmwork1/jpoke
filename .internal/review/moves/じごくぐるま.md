# じごくぐるま レビュー結果

## 仕様書
- 個別の仕様書ファイルは存在しないが、一次情報（`.internal/wiki/moves/じごくぐるま.html`）と照合した結果、
  効果内容（相手に与えたダメージの1/4を自分も受ける反動技）は既に
  `.internal/spec/moves/_move_secondary_2.md`（17行目: `じごくぐるま 0 0.25 0 0`）に記載済みで、内容は正しい。
  同種の反動技グループ（アフロブレイク・とっしん・ボルテッカー・ワイルドボルト）も同様に
  個別仕様書を持たず共通表で管理する既存方針のため、個別ファイルの新規作成は不要と判断した。
- 一次情報との照合の結果、**PP に誤りを発見**（詳細は「実装」節を参照）。

## 実装計画書
- 個別の計画書ファイルは存在しないが、`.internal/plan/moves/反動技_025.md` に
  じごくぐるまを含む反動量1/4グループ（アフロブレイク・じごくぐるま・とっしん・ボルテッカー・ワイルドボルト）の
  実装計画が記載されており、内容は現状の実装（`ON_HIT` で `_recoil(battle, ctx, value, 1/4)` を呼ぶ）と一致している。
  同グループの他技にも個別計画書は作成されていないため、既存方針との整合を優先し新規作成は行わなかった。

## 実装（handlers/ / data/）
- `src/jpoke/handlers/move_attack.py` の `じごくぐるま_recoil` は共通ヘルパー `_recoil(battle, ctx, value, 1/4)` を
  呼び出しており、`max(1, int(与ダメ * 1/4))` を `reason="recoil"` で攻撃者に与える実装で仕様と一致していた。変更なし。
- `src/jpoke/data/move.py` の `じごくぐるま` エントリで **PP が誤って `10` になっていた**ため `20` に修正した。
  一次情報（ポケモンWiki）によると、じごくぐるまのPPは第五世代までは25、第六世代以降は20であり、
  現行世代の実装対象としては20が正しい。タイプ（かくとう）・分類（物理）・威力（80）・命中率（80）・
  フラグ（`contact`, `recoil`）は一次情報と一致しており変更なし。
  `python scripts/sort_data/sort_moves.py` を実行したが、じごくぐるまの並び順自体に変更はなかった
  （なお同スクリプト実行により本タスクと無関係な既存の未コミット差分（しんぴのちから・じごくづきのflags変更）が
  そのまま維持されていることを確認した。これらは本レビュー開始前から作業ツリーに存在していた変更であり、
  じごくぐるまのレビューとは無関係のため関知しない）。
