# あくまのキッス レビュー結果

## 仕様書
- 既存の `.internal/spec/moves/あくまのキッス.md` を一次情報（`.internal/wiki/moves/あくまのキッス.html`）と照合。
- 世代表記の誤りを修正: 「第二世代」→「第一世代」（Lovely Kiss は第一世代由来の技）。
- 「ねむり継続ターン数は2～4ターン（実質1～3ターン）」という第1〜8世代基準の記述を削除し、
  Pokémon Champions 共通仕様（`.internal/spec/ailments/ねむり.md`。通常技付与は1/3でカウント2、
  2/3でカウント3）を参照する形に修正。
- Zワザ付加効果・コンテスト効果は非対象と明記する節を追加。
- タイプ/分類/威力/命中率(75)/PP(10)/範囲/優先度/直接攻撃/判定（急所×・命中判定○・追加効果×・
  まもる○・マジックコート○・よこどり×）は一次情報と一致しており変更なし。
- なお `src/jpoke/data/ps-champ-ja/moves.json` に「あくまのキッス」自体の項目が存在しない
  （Champions で覚えられるポケモンがいないため技リストに未掲載と推測される）。
  そのため Champions 版PPの正式値は確認できず、現行のGen9基準値（PP10・命中75）を維持した。

## 実装計画書
- `.internal/plan/moves/あくまのキッス.md` の仕様サマリの表記を仕様書の修正内容に合わせて更新
  （2～4ターン表記の削除、`ねむり.md` 参照への差し替え）。
- コードサンプルの型注釈の誤りを修正: `ctx: EventContext` → `ctx: AttackContext`
  （`ON_STATUS_HIT` は `AttackContext` を渡すイベントであり、CLAUDE.md の役割対応表と実際の実装
  （`handlers/move_status.py`）に合わせた）。
- Priority根拠（`turn.md` に `ON_STATUS_HIT` の記載なし、`おにび`・`でんじは` と同様に既定値100を採用）
  は確認済みで変更なし。

## 実装（handlers/ / data/）
- `handlers/move_status.py` の `あくまのキッス_apply_ailment_to_defender` は共通ヘルパー
  `apply_ailment_to_defender(battle, ctx, value, ailment="ねむり")` を呼ぶのみで、`おにび`・`でんじは`
  と同一パターン。`AttackContext` 型注釈も適正。
- `data/moves/move_a.py` の `MoveData`（type=ノーマル, category=status, pp=10, accuracy=75,
  `ON_STATUS_HIT` ハンドラのみ）は仕様と整合しており、直接攻撃フラグ（`contact`）や音技フラグ
  （`sound`）が付与されていない点も一次情報（直接攻撃×・非音技）と一致。変更なし。
