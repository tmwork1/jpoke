# かたきうち レビュー結果

## 仕様書

- 新規作成（`.internal/spec/moves/かたきうち.md`）。
- `.internal/wiki/moves/かたきうち.html` を参照し、基本情報（ノーマル/物理/威力70/命中100/PP5/接触あり）と
  「味方がひんしになった次のターンに使用すると威力が2倍になる」効果、および
  「ひんしになったそのターン中の使用では威力が上がらない」という技の仕様上の注記を記載した。

## 実装計画書

- 新規作成（`.internal/plan/moves/かたきうち.md`）。
- 「味方がひんしになったターン」をプレイヤー単位で記録する必要があること、既存の
  `Pokemon` の `_this_turn` 系フラグでは表現できないため `PlayerState` に新規フィールドを
  追加する方針、ハンドラ構成（`Event.ON_CALC_POWER_MODIFIER` / デフォルトpriority・subject_spec）
  を明記した。

## 実装（handlers/ / data/）

- `src/jpoke/data/move.py`: `かたきうち` エントリが `handlers={}`（追加効果なしのコメント付き）に
  なっており、本技の核心である威力2倍効果が全く未実装だったバグを修正。
  `Event.ON_CALC_POWER_MODIFIER` に `かたきうち_calc_power` を登録した。
- `src/jpoke/handlers/move_attack.py`: `かたきうち_calc_power` を新規実装。
  攻撃者の所属プレイヤーの `ally_fainted_turn` が `現在のターン - 1` と一致する場合、
  威力に8192（2倍）の固定小数点修正を適用する。
- `src/jpoke/core/player_state.py`: `ally_fainted_turn: int | None = None` を追加。
  味方が直近にひんしになったターンを記録するための新規フィールド。
- `src/jpoke/core/status_manager.py`: `modify_hp` 内、対象がひんしになった瞬間に
  そのポケモンが所属するプレイヤーの `ally_fainted_turn` へ現在のターンを記録する処理を追加。
  既存の `stat_lowered_this_turn` 等と同様、特定技専用の状態更新をこの階層で行う既存パターンを踏襲。
- `python scripts/sort_handlers.py src/jpoke/handlers/move_attack.py` と
  `python scripts/sort_data/sort_moves.py` を実行済み。

## テスト
