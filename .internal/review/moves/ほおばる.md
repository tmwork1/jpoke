## 実装（handlers/ / data/）
- `src/jpoke/handlers/move_status.py` の `ほおばる_check_defense_max` にあった不具合を修正した:
  - 修正前は常に `mon.rank["def"] >= 6` のみで判定しており、あまのじゃく持ちのポケモンに対して
    仕様（`.internal/spec/abilities/あまのじゃく.md`: 「ほおばるは防御ランクが+6のときも成功して
    きのみを消費する。-6のときに使うと失敗してきのみを消費しない。」）と逆の挙動になっていた
    （+6で誤って失敗、-6で誤って成功していた）。
  - `mon.ability.name == "あまのじゃく"` の場合は `rank["def"] <= -6` を、それ以外は
    `rank["def"] >= 6` を上限判定に使うよう修正した。
- `src/jpoke/data/moves/move_ha.py` の `pp=10` を `pp=12` に修正した
  （`src/jpoke/data/ps-champ-ja/moves.json` L812 準拠。Gen9本家基準の値が移行漏れとなっていた）。
- それ以外の実装（`ほおばる_check_has_berry` / `ほおばる_consume_berry_and_boost` /
  `battle.item_manager.force_trigger_berry` の利用）は仕様書と整合しており変更なし。
