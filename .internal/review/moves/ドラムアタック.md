# ドラムアタック レビュー結果

## 仕様書
- 新規作成（`.internal/spec/moves/ドラムアタック.md`）。
- 一次情報（`.internal/wiki/moves/ドラムアタック.html`）を確認し、以下を明記した。
  - タイプ: くさ / 分類: 物理 / 威力: 80 / 命中: 100 / PP: 10（Gen9基準） / 対象: 相手1体 / 優先度: 0
  - 直接攻撃: なし（音技にも非該当）
  - 追加効果: 100%の確率で**相手（防御側）**の『すばやさ』ランクを1段階下げる
  - ゴリランダーの専用技
- 依頼文には「ドラムアタックは命中後に自分の『すばやさ』ランクを下げる技」と記載されていたが、
  一次情報を確認したところ誤りだった（説明文「こうげき することで あいての すばやさを さげる」の通り、
  対象は相手であり自分ではない）。実装・既存の `_move_secondary_1.md`（`対象=defender`, S列=-1）は
  正しく相手対象で実装されていたため、その仕様のまま仕様書を作成した。
