# いしずえのめん レビュー結果

## 仕様書
- 変更なし（`.internal/spec/items/_おめん.md` は一次情報（Wiki）と一致していることを確認済み。
  「使用する攻撃技の威力が、タイプに関係なく1.2倍になる」の記述は物理・特殊いずれの
  攻撃技にも適用される旨を正しく表している）。

## 実装計画書
- `.internal/plan/items/いしずえのめん.md` の誤りを修正した：
  - 「物理技の攻撃補正を1.2倍にする」「実装状態: 実装済み（物理技のみ。特殊技は未対応）」
    という記述は仕様書と矛盾していたため、「使用する攻撃技の威力をタイプ・カテゴリ
    （物理/特殊）に関係なく1.2倍にする」に修正。
  - ハンドラロジック例を `ctx.move.category == "物理"` から `ctx.move.is_attack` に修正。
  - テスト観点に「特殊技ダメージも1.2倍になること」を追加。

## 実装（handlers/ / data/）
- `src/jpoke/handlers/item.py` の `オーガポンのめん_boost_atk` が
  `ctx.move.category == "physical"` のみを1.2倍にしており、特殊技（例: エナジーボール）が
  補正対象外になっていたバグを発見。仕様上は物理・特殊を問わず攻撃技全てが対象のため、
  `ctx.move.is_attack` を条件とするよう修正（`いのちのたま_boost_atk` と同様のパターン）。
  ※この関数は `いどのめん`・`かまどのめん` と共有のため、3アイテム全てに影響する修正。
  なお、本修正は並行レビュー中の別タスクによって同時に着手・反映されており、
  本タスク実施時点で `handlers/item.py` 側は既に修正済みだった（`data/item.py` の
  `いしずえのめん` エントリ自体に変更は不要）。
