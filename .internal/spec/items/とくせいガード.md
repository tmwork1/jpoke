# 仕様書: とくせいガード

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A8%E3%81%8F%E3%81%9B%E3%81%84%E3%82%AC%E3%83%BC%E3%83%89

## 効果
- 効果:持たせたポケモンは、戦闘中に特性が変わることや、無効化されること、かたやぶり系の効果で無視されることが無くなる。
- なげつける:威力30

## 詳細な仕様
- 特性を変える技、特性を変える特性の効果で特性が書き変わることや、無効化されることがない。効果が発動した際は「<所持者>は とくせいガードの効果で 特性が 守られている!」というメッセージが出る。具体的に守った特性の名前が表示されることはない。
 - 所持者はスキルスワップ/なりきり/うつしえなどを使用して自分の特性を変えることもできない。所持者のトレース/レシーバーなども発動しない。
 - トレースのポケモンがとくせいガードを持って場に出た瞬間、とくせいガードが発動して特性の変更が防がれたときは、それ以上トレースが発動することはなくなる。このときとくせいガードを失った後でもトレースは発動することはない。
 - トレースのポケモンがとくせいガードを持って場に出た瞬間では、相手の場にいるポケモンがトレースできない特性だったか、相手が不在だっためとくせいガードが発動しなかった場合、トレースは保留状態になる。その状態でとくせいガードを失ってもトレースが即座に発動することはないが、相手の場にトレース可能なポケモンが繰り出されたならトレースは発動する。
 - 所持者がいえきを受けてもとくせいなし状態にならない。
 - テラレイドバトルで敵テラスタルポケモンから特性を消されることも防げる。
 - ただし、味方からバトンタッチでとくせいなし状態を引き継いだときは、とくせいガードは発動せずに特性が消える。
 - 所持者はかがくへんかガスで特性の効果を消されることがない。
 - 所持者が場にいるときにかがくへんかガスが発動したときは、「<所持者>は とくせいガードの効果で 特性が 守られている!」というメッセージが出る。
 - かがくへんかガスが発動している場に所持者が出てきたときは、メッセージは出ないものの、特性は守られている。
 - かがくへんかガスが発動中にとくせいガードを失った場合は、即座に特性の効果も消されることになる。逆に、とくせいガードを入手したポケモンは、即座に特性の効果が復活する。
 - 所持者の特性がぶきようであるときでも道具の効果が発動し、特性の変更や無効化を防ぐ。
- とくせいガードを持っているときしか特性の変更を防ぐ効果は無い。
 - すでに特性が書き変わったポケモンがとくせいガードを手に入れても、元の特性に戻ることは無い。
 - すでにとくせいなし状態のポケモンがとくせいガードを手に入れても、とくせいなし状態は消えない。
- 以下のように、所持者の姿が変わったことで特性が変わるときは、とくせいガードの効果で防がれることはない。
 - 所持者がへんしん状態になったときは、相手の特性をコピーして元の特性から書き換わる。
 - 所持者がシェイミ (スカイフォルム) であり、こおり状態になったときは特性がしぜんかいふくに変化する。
 - 所持者がオーガポン (みどりのめん) であり、テラスタルしたときは特性がおもかげやどしに変化する。
 - 所持者がテラパゴスであるときは、フォルムチェンジに伴い特性が変化する。
- 所持者の特性はかたやぶり系の効果 (特性かたやぶり/ターボブレイズ/テラボルテージ/きんしのちから、特性を無視する技) で無視されなくなる。このときとくせいガードが発動したことを直接明示するメッセージは出ず、かたやぶり系の効果が発動しなかったことになる。
 - アロマベール/じょおうのいげんなど複数のポケモンに効果がかかる特性の場合、その特性を所有するポケモンがとくせいガードを持っていれば、効果範囲内にいる全てのポケモンがかたやぶり系の効果をとくせいガードで防げる。その特性を所有するポケモン以外にとくせいガードを持たせても効果は無い。
 - かたやぶり系の効果がある技を受けたときを除けば、とくせいガードを持ったポケモンの特性も無効化されることはある。
 - ちからずくで一部の特性が発動しなくなる効果はとくせいガードでも防げない。
 - かたやぶり側にとくせいガードを持たせても、相手のとくせいガードで自分の特性が無視されることを防ぐことはできない。
- 所持者に対するなりきり/うつしえ/トレースなどで、他のポケモンに特性をコピーされることはとくせいガードでは防がれない。
 - 所持者に対するスキルスワップ、または所持者が使用するスキルスワップは失敗するため、もう片方に特性がコピーされることは無い。
