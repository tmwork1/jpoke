# 仕様書: じゅうでんち

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%98%E3%82%85%E3%81%86%E3%81%A7%E3%82%93%E3%81%A1

## 効果
持たせたポケモンがでんきタイプの攻撃技を受けると、こうげきが1段階上がる。1度使うとなくなる（なげつける威力30）。

## 詳細な仕様
- ダメージが無かった場合は発動しない。以下の状況においてじゅうでんちは発動しない。
 - でんじはなど変化技を受けたとき。
 - 技が外れたとき。
 - 技が無効化されたとき。
 - まもる状態などで防いだとき。
 - みがわり状態で防いだとき。
- ばけのかわ・アイスフェイスでダメージを肩代わりしたときは発動する。
- でんきタイプの連続攻撃技を受けた場合、1発目が当たった時点で発動する。
- 所持者のこうげきランクがすでに最大まで上がっている場合、じゅうでんちは発動しない。
 - 所持者の特性があまのじゃくである場合、こうげきランクが最小まで下がっているなら発動しない。
- 所持者がたんじゅんである場合、こうげきランクが2段階上がる。
- 所持者があまのじゃくである場合、こうげきランクが1段階下がる。
- かたやぶりの効果があるでんき技に対してはたんじゅん・あまのじゃくの効果は発動せず、いずれもじゅうでんちでこうげきランクが1段階上がる。
- 特性マジシャンのでんき技を受けた場合、じゅうでんちの効果が先に発動するため奪われない。
- タイプが変わる技は、変化後のタイプがでんきタイプであるときに発動する。
- そうでんやプラズマシャワー状態の効果で、でんきタイプに変わった技を受けたときも発動する。
- でんきタイプのダメージ固定技を受けたときも発動する。

## 備考
- でんき技のダメージを無効化する効果はない。
