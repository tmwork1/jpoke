# とくせいなし (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

とくせいなしとは、ポケモンの揮発性状態の一つ。
第六世代まではいえきじょうたいと呼ばれていた。

## 効果

- とくせいなし状態のポケモンは特性の効果が消える。
- 特性そのものは消えるわけではなく、トレースやスキルスワップで他のポケモンに特性をコピーすることは可能。

## 付与条件

### 対象に発生する技

- **いえき**: 命中した相手にとくせいなし状態を付与する。
- **コアパニッシャー**: 前ターン同技を使った相手にとくせいなし状態を付与する（Championsスコープ外）。

## 消滅条件

- **交代**: 交代時に全揮発状態が自動解除される。

テラレイドバトルでのレイドボス追加行動（1ターン経過で解除）はChampionsシングルバトルのスコープ外。

## 予防条件

以下の特性を持つポケモンにはとくせいなし状態が付与されない:
ARシステム / アイスフェイス / うのミサイル / きずなへんげ / ぎょぐん / じんばいったい /
スワームチェンジ / ぜったいねむり / テラスチェンジ / ばけのかわ / バトルスイッチ /
マイティチェンジ / マルチタイプ / リミットシールド / ダルマモード（第七世代以降）

また、とくせいガードによっても予防される。

## ターン処理の優先度

`.internal/spec/turn.md` には ON_VOLATILE_START / ON_VOLATILE_END の記載なし。
他の同種ハンドラー（さわぐ等）に合わせて priority=100（デフォルト）を使用する。

| イベント | Priority | 処理 |
|---------|----------|------|
| ON_VOLATILE_START | 100 | とくせいなし状態付与時に特性を無効化する |
| ON_VOLATILE_END | 100 | とくせいなし状態終了時に特性の無効化を解除する |

## 詳細な仕様（Champions版）

- 効果が消えるだけで特性そのものがなくなるわけではない。
- とくせいなし状態のポケモンにトレース/スキルスワップをすれば、そのポケモンが持っていた特性を発動できる。
- バトンタッチで引き継がれる（他のポケモンがとくせいなしにならない特性を持つ場合は消える）。
  ただし、Championsではバトンタッチ未実装のためスコープ外。
- 内部実装: `AbilityDisabledReason("とくせいなし")` を追加/削除することで特性有効状態を制御する。

## 本実装での対象外（スコープ外）

- コアパニッシャーによるとくせいなし付与: 技コアパニッシャー実装時に対応。
- バトンタッチによる引き継ぎ: バトンタッチ実装時に対応。
- テラレイドバトルでの1ターン経過消滅: Championsシングルバトルのスコープ外。
