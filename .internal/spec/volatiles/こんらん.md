# こんらん (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

こんらんとは、ポケモンの揮発性状態の一つ。
こんらん状態のポケモンは行動前に確率で自傷ダメージを受けて行動不能になる。

## 効果

- こんらん状態のポケモンは、行動（ON_TRY_ACTION）前に 1/3 の確率で自傷する。
- 自傷した場合は行動が中断され、相手にはダメージが与えられない。
- 自傷ダメージは `battle.roll_damage(attacker, attacker, "こんらん")` で計算される。
  - こんらん自傷技の仕様: タイプ=""、分類=物理、威力=40、PP=99999。
  - 自分が自分を攻撃するため、タイプ一致・急所・天候補正なし。

## 付与条件

- あやしいひかり・いばる・おだてる・てんしのキッス・フラフラダンス 等の技を受けたとき
- あばれる・げきりん・はなびらのまい・だいふんげき が正常に終了したとき
- 特性どくくぐつ でどく/もうどく状態になったとき
- その他こんらん状態を付与する効果を受けたとき

付与時のターン数: 2〜5 ターンのランダム（`battle.random.randint(2, 5)`）

## 消滅条件

- カウントが 0 になる（ターン経過、ON_TRY_ACTION で tick）
- 交代（揮発状態は交代時に自動解除）
- フルヒール・きずぐすり系のアイテム使用（将来対応）

## 免疫

- 特性マイペース を持つポケモンは ON_BEFORE_APPLY_VOLATILE で無効化される（priority=140）
- ミストフィールド状態のポケモンはこんらんにならない（Champions対応範囲外）
- すでにこんらん状態の場合は付与されない（volatile_manager.apply の既存チェック）

## ターン処理の優先度

`.internal/spec/turn.md` の `Event.ON_TRY_ACTION` より:

| Priority | 処理 |
|----------|------|
| 110 | こんらんの自傷の判定 |

## 詳細な仕様

- カウントのデクリメントは ON_TRY_ACTION（priority=110）内で tick を呼ぶことで行う。
  カウントが 0 になった時点で揮発状態が解除され、その解除されたターンは自傷確率の抽選を
  行わずに通常通り行動できる（tick 直後に `has_volatile("こんらん")` を確認し、解除済みなら
  自傷判定をスキップする）。
- 自傷確率は Champions 仕様で 1/3（約33%）固定。
- 自傷ダメージは `battle.modify_hp` で適用され、reason="self_attack"。
- 行動中断時は `HandlerReturn(value=False, stop_event=True)` を返す。
