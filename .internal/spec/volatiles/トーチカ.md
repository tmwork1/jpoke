# トーチカ (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

トーチカとは揮発性状態の一つ。
技「トーチカ」を使用することで付与される（自分対象）。
そのターンに受ける攻撃技・変化技を無効化し、直接攻撃をしてきた相手をどく状態にする。
第七世代以降の技。BDSPでは使用不可。

## 技の基本情報

| 項目 | 値 |
|------|-----|
| タイプ | どく |
| 分類 | 変化 |
| PP | 10 |
| 優先度 | +4 |
| 範囲 | 自分 |

| 急所 | なし |

## 効果

- 使用したターンに受ける**攻撃技・変化技**を無効化する（まもると同様に変化技も防ぐ）
- 直接攻撃技を防いだ場合、攻撃者をどく状態にする

## 付与条件

- 技「トーチカ」を使用したとき（自分に付与、count=1）

## 消滅条件

- ターン終了時に解除される（count=1 のため、使用したターンのみ有効）
- **交代**によって解除される

## 詳細仕様

### 攻撃技・変化技の無効化

- そのターンに受ける攻撃技（`is_attack=True`）および変化技（`is_attack=False`）を無効化する
- **変化技も防げる**（`protect_non_attack=True`）
  - まもる / みきり / ニードルガード / トーチカ が変化技を防ぐ
  - キングシールド / かえんのまもり / スレッドトラップ / ブロッキング とは異なる
- まもる系と同じ `ON_TRY_MOVE_1` イベントで処理される（priority=100）

### どくの付与

- 直接攻撃（`is_contact=True`）を防いだ場合のみ攻撃者をどく状態にする
- 非接触の攻撃技（れいとうビーム等）を防んでもどく状態にならない
- タイプ相性・特性でダメージが無効でも直接攻撃を防げばどく状態にする（wiki仕様）
- 連続攻撃技かつ直接攻撃でも、どく判定は1回だけ行われる（`ailment_manager.apply` が重複付与を防ぐ）
- `ぼうごパット` を持つ攻撃者は接触判定が False になるためどく状態にならない
  - `query.is_contact` が `ON_CHECK_CONTACT` を発火し、`ぼうごパット` ハンドラが False を返す
- どくタイプ・はがねタイプのポケモンは通常どく状態にならない
  - 例外: 特性「ふしょく」のトーチカ使用者が防いだ場合はどく状態にできる（本実装スコープ外）

### 防御が無視されるケース（今世代スコープ外）

- `ふかしのこぶし` 特性の直接攻撃では守りを貫通し、どく状態にもならない
  - `ON_CHECK_PROTECT` ハンドラで `ふかしのこぶし` が False を返す
- `ハイパードリル` も守りを貫通（技ハンドラ側で処理）
- `フェイント` / `ゴーストダイブ` / `シャドーダイブ` / `いじげんホール` / `いじげんラッシュ` を受けた場合
  - トーチカ状態が取り除かれてダメージを受け、どく状態にはならない
  - 攻撃がタイプ相性で無効な場合はトーチカ状態が取り除かれない

### 連続使用による失敗率上昇

- まもる/みきり/こらえる/ニードルガード/トーチカ/スレッドトラップ/かえんのまもり/キングシールド を連続使用すると成功率が低下する
  - 成功するごとに次ターンの成功率が 1/3 になる（下限 1/729）
  - 連続使用しなかった場合・失敗した場合はリセットされる
  - **注意**: 本実装では連続使用失敗チェックは `ON_TRY_MOVE_1` の priority=30 で行われる

## ターン処理の優先度

`.internal/spec/turn.md` の `Event.ON_TRY_MOVE_1` より:

| Priority | 処理 |
|----------|------|
| 30 | まもる系: 連続使用により失敗 |
| 100 | トーチカ状態による攻撃技・変化技の無効化 |

`Event.ON_TURN_END`:
- priority 未指定（デフォルト）で source:self に対して解除する

## 関連インタラクション

- **ぼうごパット**: 攻撃者のis_contactをFalseにするため、どく状態にならない
- **ふかしのこぶし**: `ON_CHECK_PROTECT` でFalseを返し、守りを貫通する（どく状態も付与されない）
- **まもる / みきり / ニードルガード / キングシールド / スレッドトラップ / かえんのまもり**: 連続使用成功率共有
- **フェイント / ゴーストダイブ / シャドーダイブ / いじげんホール / いじげんラッシュ**: トーチカ状態を取り除く

## 専用ポケモン
