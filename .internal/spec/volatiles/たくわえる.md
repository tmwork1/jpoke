# たくわえる（揮発性状態）仕様書

対象: Pokémon Champions シングルバトル

## 概要

たくわえるとは揮発性状態の一つ。
技「たくわえる」を使用することで自分自身に付与される。
カウント（1〜3）を持ち、「はきだす」「のみこむ」の効果量に影響する。

## 付与条件

- 技「たくわえる」を使用したとき（自分自身に付与）

## カウント管理

- 初回付与時は count=1 で付与される。
- 既にたくわえる状態のとき「たくわえる」を使うと count が +1 される（最大3）。
- count がすでに 3 のときに「たくわえる」を使用すると技が失敗する。

## たくわえるの効果（間接効果）

たくわえる状態は単独では直接的な効果を持たず、以下の技の効果量に影響する。

### はきだす（特殊攻撃）

| たくわえた回数 | 威力 |
|---|---|
| 未使用 | 失敗する |
| 1回 | 100 |
| 2回 | 200 |
| 3回 | 300 |

### のみこむ（HP回復）

| たくわえた回数 | 回復量 |
|---|---|
| 未使用 | 失敗する |
| 1回 | 最大HPの1/4 |
| 2回 | 最大HPの1/2 |
| 3回 | 最大HPと同じだけ回復（全回復） |

## 副次効果（たくわえる使用時）

「たくわえる」を使うたびに自分の **ぼうぎょ・とくぼう** が各1段階上昇する。

## 消滅条件

- **はきだす** または **のみこむ** を使用したとき（成功・失敗にかかわらず使用自体で消滅するわけではなく、効果発動後に消滅する）
  - 第4世代以降: 消滅時に上昇していた分だけぼうぎょ・とくぼうのランクが下降する
- **交代**によって解除される（バトンタッチによる引き継ぎなし）

## 詳細仕様

- バトンタッチで引き継がれない。
  - ただし、たくわえるで上昇したぼうぎょ・とくぼうのランク変化はバトンタッチで引き継がれる。
- のみこむ・はきだす使用後のランク低下は、クリアボディなどの特性やしろいきり状態でも防げない（自分源の変化として処理する）。
- のみこむ・はきだすによるランク低下がだっしゅつパックを発動させる（スコープ外）。

## ターン処理の優先度

「たくわえる」揮発状態自体にはターン処理イベントはない（ハンドラなし）。
カウントの管理は技のハンドラ（`handlers/move_status.py`）側で行う。

## 実装スコープ

| 機能 | スコープ |
|---|---|
| たくわえる：カウント管理・ぼうぎょとくぼう上昇 | 実装対象 |
| はきだす：カウント依存の威力設定・ランク戻し | 実装対象 |
| のみこむ：カウント依存のHP回復・ランク戻し | 実装対象 |
| はきだす外れ・無効時のランク低下 | スコープ外 |
| だっしゅつパック発動 | スコープ外（アイテム実装時に対応） |
