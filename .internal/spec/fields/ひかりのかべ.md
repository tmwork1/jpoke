# ひかりのかべ（場の状態）仕様書

## 概要

「ひかりのかべ」は味方の場に発生するサイドフィールド。
5ターンの間、場のポケモンが**特殊技**で受けるダメージを1/2に軽減する。

## 効果

- 場にいるポケモンが特殊技で受けるダメージを1/2に軽減（×2048/4096）
- ダブルバトルでは約2/3に軽減（×2732/4096）※本プロジェクト対象外

## 詳細な仕様

- 急所に当たったときはダメージを軽減できない
- 特性「すりぬけ」の敵ポケモンからダメージを受けるときはダメージを軽減できない（`ctx.can_bypass_screen` で判定）
- サイコショック・サイコブレイク・しんぴのつるぎによるダメージも軽減できる（move.category == "special" で判定）
- 分類が変わる技は、最終的に特殊技になったときのみダメージを軽減できる
- ワンダールーム中でも特殊技のダメージを軽減するという効果は変わらない（物理技は軽減しない）
- こんらん状態で自分を攻撃したときのダメージは軽減できない（こんらんによるダメージは攻撃ではない）
- オーロラベールとの重複はしない（両方が有効でも軽減率は1/2のまま）

## 発生

- 技「ひかりのかべ」を使用する
- どばどばオーラ（未実装）
- マックスレイドバトル中のおうえん（対象外）

## 継続ターン

- 通常5ターン（発動させたターンを含む）
- ひかりのねんどを持ったポケモンが発動した場合は8ターン（ON_MODIFY_DURATION イベントで延長）

## 消滅

- ターン経過（カウント0になったとき）
- きりばらい
- かわらわり・サイコファング・レイジングブル（これらを受けた場合、ダメージ後に解除）
- キョダイフウゲキ（ダメージ軽減後に解除）
- バリアフリー
- ダークリムーブ
