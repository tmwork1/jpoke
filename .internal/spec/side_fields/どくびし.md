# どくびし（場の状態）仕様書

## 概要

- **種別**: サイドフィールド（相手の場）
- **最大層数**: 2 層
- **発生源**: 技「どくびし」 / 特性「どくげしょう」（物理技を受けたとき）

## どくびしの効果

交代により **地面に着地した** ポケモンが場に出てきたとき、層数に応じた毒状態になる。

| 層数 | 付与される状態異常 |
|------|--------------------|
| 1 層 | どく |
| 2 層 | もうどく |

## 無効条件

以下のポケモンには **毒状態が付与されない**（どくびしは消滅しない）。

| 条件 | 詳細 |
|------|------|
| 浮いているポケモン | ひこうタイプ・特性ふゆう等で地面に接地していないポケモン |
| はがねタイプ | タイプによる毒無効 |
| あつぞこブーツ所持 | ハザード無効（フィールドは消滅しない） |
| めんえき・リーフガード(日照り)・ぜったいねむり・きよめのしお 等 | ailment_manager の ON_BEFORE_APPLY_AILMENT で処理 |
| フラワーベール・リミットシールド・パステルベール の保護下 | 同上 |
| ミストフィールド | 同上 |

## どくびしの消滅

| 消滅条件 | 対象の場 |
|----------|----------|
| 地面に接地している **どくタイプ** のポケモンが着地する | 味方の場のみ（あつぞこブーツ所持でも消滅） |
| こうそくスピン・キラースピン を使用 | 使用者の場のみ |
| きりばらい を使用（第 6 世代以降） | 両陣営 |
| キョダイフウゲキ・おかたづけ | 両陣営 |

## 実装上の注意

- 浮いているどくタイプのポケモンはどくびしを消滅させられない
- どくタイプのポケモンはあつぞこブーツを持っていてもどくびしを消滅させる
- 毒無効の判定順序: 浮遊チェック → どくタイプチェック（消滅）→ ハザード免疫チェック → 毒付与
- ミストメイカー特性はどくびし処理の後に発動するため、その特性持ちはどく状態になる
