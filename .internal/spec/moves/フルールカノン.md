# フルールカノン（Fleur Cannon）仕様書

参照: .internal/wiki/moves/フルールカノン.html（ポケモンWiki）

## 基本情報

* タイプ: フェアリー
* 分類: 特殊
* 威力: 130
* 命中: 90
* PP: 8（ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されている。
  Gen9本家基準の5ではなくこの値を採用する。`src/jpoke/data/ps-champ-ja/moves.json` には
  フルールカノン自体のエントリが存在しないが、同系統の反動技（りゅうせいぐん・
  オーバーヒート・リーフストーム・サイコブースト）はいずれも本家PP5→ChampionsPP8で
  統一されており〔`src/jpoke/data/ps-champ-ja/moves.json:942`(りゅうせいぐん) / `:134`(オーバーヒート)
  / `:934`(リーフストーム)、いずれもPP8〕、フルールカノンも同じ規則でPP8になると判断する)
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 命中して技が成功した後、反動として自分の『とくこう』ランクが2段階下がる（確定効果）。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率90）
* 追加効果: なし（自分自身への確定効果であり、相手への「追加効果」には該当しない）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は発動する（追加効果の有無に関係なく命中判定を伴う技として扱われる）。
  本プロジェクトの対象世代では無関係
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能（本プロジェクトでは対象外の古い技のため実装しない）
* ちからずく: 非対象技（自分への確定ランク低下は「追加効果」扱いではないため、
  ちからずくによる威力上昇・追加効果無効化の対象にならない）

## 備考

* マギアナの専用技。りゅうせいぐんやリーフストームなどのフェアリータイプ版。
* この技で相手を倒した場合、特性ソウルハートは技の効果の後に発動する。すでにとくこうが
  最低ランクである場合、ソウルハートによる上昇のみを受ける（ソウルハート自体は
  本プロジェクトの対象範囲内だが、本仕様書ではフルールカノン側の効果順序のみ記載する）。
* 反動でとくこうが下がる代表的な高火力特殊技であり、りゅうせいぐん・オーバーヒート・
  リーフストーム・サイコブースト等と同様の「反動技#能力が低下する」仕様に従う。
* `.internal/spec/moves/_move_secondary_1.md` の該当行では確率1・対象attackerと記載されているが
  C列（とくこう）が誤って0になっている（りゅうせいぐん等の同系統技はいずれもC列-2）。
  本仕様書ではポケモンWikiの一次情報（自分のとくこうが2段階下がる）を正とする。
