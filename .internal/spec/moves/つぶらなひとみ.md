# つぶらなひとみ 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | フェアリー（第五世代までノーマル） |
| 分類 | 変化 |
| PP | 20（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。本家 Gen6-9 は 30） |
| 命中率 | 100 |
| 範囲 | 相手1体選択 |
| 優先度 | +1（必ず先制できる） |
| 直接攻撃 | × |

## 効果

相手の **こうげき** ランクを **1段階** 下げる。

## 判定

- 急所: なし（変化技）
- 追加効果: なし（本体の効果がランクダウンそのもの）
- まもるで防がれる: ○
- マジックコートで跳ね返される: ○
- オウムがえしでコピーされる: ○
- よこどりで奪われる: ×（相手を対象とする変化技のため無効）

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- バトルパレスの判定基準（「ずるい」技）は実装しない
