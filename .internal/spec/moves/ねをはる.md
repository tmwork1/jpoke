# ねをはる（Ingrain）仕様書

参照: .internal/wiki/moves/ねをはる.html（ポケモンWiki）
関連: .internal/spec/volatiles/ねをはる.md（揮発性状態側の仕様）

## 基本情報

* タイプ: くさ
* 分類: 変化
* 威力: -
* 命中: なし（必ず自分に成功する）
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` 準拠）
* 対象: 自分
* 優先度: 0
* 接触: なし

## 効果

* 自分を『ねをはる』状態にする（揮発性状態。詳細は `.internal/spec/volatiles/ねをはる.md` を参照）。
  * 使用後、毎ターン終了時に最大HPの1/16ずつ回復する（切り捨て、最低1）。
  * 交代・にげるができなくなる（ゴーストタイプは例外）。
  * 自分が『ひこう』タイプ、または特性『ふゆう』の場合でも『じめん』タイプの技が当たるようになる
    （地面にいる扱いになるため）。
* すでにねをはる状態になっているときに使うと失敗する
  （揮発性状態の重複付与不可という汎用ルールで処理される。個別のハンドラは不要）。
* データ上はHP回復技（`heal` フラグ）には分類されない。そのためかいふくふうじ状態でも
  使用すること自体は可能。ただしねをはる状態による毎ターンの回復はできない
  （`ON_MODIFY_HEAL` 経由でブロックされる）。

## 判定フラグ

* 急所: なし（変化技）
* 命中判定: なし（自分対象のため必ず命中）
* まもる: 防がれない（`src/jpoke/data/ps-champ-ja/moves.json` の「守×」、対象が自分のため）
* マジックコート・マジックミラー: 反射されない（対象が自分のため）

## 実装上の注意

* `MoveData` には `target="self"` を必ず指定する
  （欠落すると `Move.is_blocked_by_protect` / `Move.is_reflectable` が
  `target == "foe"` を基準に判定するため、まもる・マジックコートに誤反応する）。
* `heal` フラグは付与しない（かいふくふうじによる使用禁止の対象外のため）。
