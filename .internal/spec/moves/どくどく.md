# どくどく（Toxic）仕様書

参照: .internal/wiki/moves/どくどく.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 変化
* 威力: -
* 命中: 90
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。Gen9本家は10だが本プロジェクトは
  champions基準のためPP12）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 相手を『もうどく』状態にする。
* 使用者が『どく』タイプの場合、この技は必中になる（第六世代以降）。
  * 命中判定そのものをスキップする（命中率90を無視して必ず命中する）。
  * 使用者の特性『へんげんじざい』『リベロ』の効果で技使用時にどくタイプへ変化した場合も、
    命中判定前にタイプ変化が完了しているため必中になる。
  * どくタイプでない使用者が使ったどくどくをどくタイプの相手がマジックコートで跳ね返した場合、
    必中にはならない（跳ね返した側のタイプは無関係で、あくまで最初の使用者のタイプで判定する）。
    一方、どくタイプの使用者が使ったどくどくをどくタイプでない相手がマジックコートで跳ね返した
    場合は必中のまま。
  * 必中判定は技本体にのみ作用し、同ターン中に使う他の技には影響しない。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: あり（命中率90。ただし使用者がどくタイプの場合は必中）
* 追加効果: なし（本技の状態異常付与自体が主効果であり、追加効果扱いではない）
* まもる: 防がれる
* みがわり: 防がれる
* おうじゃのしるし: 対象外
* マジックコート: 反射可能
* よこどり: 対象外（本技はよこどりで奪えない）
* オウムがえし: 反射可能
