## 効果

自分の最大HPの1/2（小数点以下切り捨て）を消費して、自分の『こうげき』ランクを最大（+6）まで上げる。
こうげきランクが-6段階のときに使用しても最大の+6段階まで上がる
（実質的にこうげきランクを12段階分上げる効果として扱われる）。

## 失敗条件

- こうげきランクがすでに最大（+6）の場合は失敗し、HPは減らない。
- 現在のHPが最大HPの半分以下の場合は失敗し、HPは減らない
  （`現在HP <= 最大HP // 2` が失敗条件。境界値である「ちょうど半分」のときも失敗する）。

## あまのじゃく（Contrary）との相互作用

- 使用者の特性が**あまのじゃく**である場合、こうげきランクを上げる効果が反転し、
  こうげきランクが最低（-6）まで下がる。このときHPは通常通り消費され、失敗しない。
- あまのじゃくであっても、こうげきランクがすでに最大（+6）まで上がっているときに使用すると失敗する
  （失敗条件の判定は反転前の実際のランク値 `rank["atk"] >= 6` に対して行われるため、
  あまのじゃく特性の有無にかかわらず同一の判定になる）。
- ランクの反転処理自体は `battle.modify_stats` が特性を考慮して汎用的に行うため、
  本技専用の反転ロジックは実装不要。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない） |
| おうじゃのしるし | × |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |
| よこどり | ○ |
| オウムがえし | × |

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。

## ポケモンチャンピオンズ向けの簡略化

- 第二世代（ポケモンスタジアム金銀を除く）における「失敗と表示されるが実際にはランクが2段階上がる」
  というバグ挙動は実装しない（現行世代の正規挙動のみを実装する）。
- 第四世代限定の「はらだいこ使用後に受けるダメおしの威力が2倍になる」仕様は実装しない
  （第五世代以降廃止済みのため）。
