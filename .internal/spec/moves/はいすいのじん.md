## 効果

自分の こうげき・ぼうぎょ・とくこう・とくぼう・すばやさ ランクを **1段階ずつ** 上げる。
さらに、自分が『にげられない』状態になる（交代・逃げることができなくなる）。

この技の効果で『にげられない』状態になっているとき（＝はいすいのじん起因のにげられない状態を
すでに保持しているとき）に使用した場合は失敗する。

すべての能力ランクがすでに+6で1つも上げられない場合も失敗する（一般的な複合ランク変化技と
同様のルール。例: めいそう・ビルドアップ等の単一/複合ランク変化技が「上げられる能力が1つもない
場合は失敗する」のと同じ）。

## にげられない状態の特記事項（はいすいのじん固有）

- 自分自身をにげられない状態にする技のため、詳細は `.internal/spec/volatiles/にげられない.md` を参照。
- 使用者がゴーストタイプの場合、にげられない状態になっても交代・逃げることができる
  （ゴーストタイプはにげられない状態の効果を無視できるという一般ルールがそのまま適用される）。
- 使用者の持ち物が『きれいなぬけがら』のときは、にげられない状態になっても交代できる
  （逃げることはできない）。
- 使用者の持ち物が『けむりだま』のときは、にげられない状態になっても逃げることができる
  （交代はできない）。
- **使用時に相手の場にいたポケモンが場から去っても、この技によるにげられない状態は解除されない**
  （とおせんぼう等の「相手を対象とするにげられない付与技」とは異なり、はいすいのじんは
  自分自身が発生源かつ対象であるため、「発生源が交代したら解除される」という一般消滅条件が
  そもそも成立しない）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない） |
| おうじゃのしるし | ×（第五世代以降） |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って `いたずらごころ`（あくタイプ相手への優先度技無効化）や `おうごんのからだ`
（他のポケモンからの変化技無効化）の判定対象になってしまう）。

## 失敗条件のまとめ

1. すでにはいすいのじん起因でにげられない状態になっている場合（2回目以降の使用）
2. すべての能力ランクがすでに+6の場合（1つも上げられる能力がない）

上記いずれの場合も、PPは消費される。

## 備考

- タイレーツの専用技。
- 使用者がゴーストタイプであっても、にげられない状態の付与自体は成功する（相手を対象とする
  トラップ技のような「相手がゴーストタイプなら技自体が無効化される」仕組みはない。あくまで
  『にげられない状態になった後、その効果を無視して交代できるかどうか』の話であり、状態異常の
  付与そのものは常に成功する）。
