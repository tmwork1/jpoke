## 効果

使用者自身の こうげき・とくこう ランクを1段階ずつ上げる。

天候が **にほんばれ・おおひでり** の場合は、こうげき・とくこうランクを2段階ずつ上げる。

## ばんのうがさ・ノーてんき・エアロックとの相互作用

- 使用者が **ばんのうがさ** を持つ場合、にほんばれ・おおひでりによる補正を受けず、
  通常時と同じ1段階上昇になる（`battle.weather_for(mon)` が使用者視点の有効天候を返す）。
- 場に **ノーてんき**・**エアロック** 持ちのポケモンがいる場合、天候効果自体が無効化されるため
  常に1段階上昇になる（`battle.weather` / `battle.weather_for` が天候無効時は「なし」を返す）。

## たんじゅんとの相互作用

- 使用者の特性が **たんじゅん** の場合、ランク変化量が2倍になる。にほんばれ中に使用すると
  こうげき・とくこうが4段階ずつ上昇する（`battle.modify_stats` が特性による倍率を汎用的に処理する）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない） |
| おうじゃのしるし | × |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |
| よこどり | ○ |
| オウムがえし | × |

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。
