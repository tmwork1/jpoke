# エレキフィールド 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | でんき |
| 分類 | 変化 |
| PP | 12 |
| 命中率 | - |
| 範囲 | 全体の場 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

**5ターン** の間、地形を「エレキフィールド」にする。エレキフィールドの間、「ひこう」タイプや特性「ふゆう」など
地面にいないポケモンを除くすべてのポケモンは「ねむり」「ねむけ」（あくび経由の眠り）状態にならず、また
「でんき」タイプの技の威力が1.3倍になる。すでにエレキフィールドが発動している場合に使用すると失敗する
（グラスフィールド・ミストフィールド・サイコフィールドが発動している場合は上書きする）。

エレキフィールドそのものの詳細仕様（威力補正・ねむり／ねむけ防止・カウントダウン・グランドコートによる
延長など）は `.internal/spec/fields/エレキフィールド.md` を参照。

## 判定

- 急所: なし（変化技）
- 命中判定: なし（命中率 `-` の技）
- 追加効果: なし（本体の効果がフィールド設定そのもの）
- まもるで防がれる: ×（範囲が「全体の場」のため対象を取らず、まもるの影響を受けない）
- マジックコートで跳ね返される: ×
- よこどりで奪われる: ×（フィールド系の変化技は例外的によこどり対象外）
- オウムがえしでコピーされる: ×（相手に向けて撃つ技ではないため）

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- Zワザによる付加効果（自分のすばやさが1段階上がる）は実装しない（Zワザ非対応）
- ポケモン不思議のダンジョン・ポケモンクエスト・ポケモンマスターズ固有の仕様は対象外
- PPは `src/jpoke/data/ps-champ-ja/moves.json` の値（12）を採用する
  （Gen9本家の10から8/12/16/20の4段階統一により変更されている）
