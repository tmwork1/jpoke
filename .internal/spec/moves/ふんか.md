# ふんか（Eruption）仕様書

参照: .internal/wiki/moves/ふんか.html（ポケモンWiki）
参照: .internal/spec/moves/_variable_power.md（変動威力技共通仕様、第3節）

## 基本情報

* タイプ: ほのお
* 分類: 特殊
* 威力: 可変（1〜150）
* 命中: 100
* PP: 5
* 対象: 相手全員（本プロジェクトはシングルバトルのみ対象のため、対象は相手1体として扱う）
* 優先度: 0
* 接触: なし

## 効果

* 自分（使用者）の残りHPが多いほど、威力が大きくなる。
* 威力は以下の式で算出する（小数点以下切り捨て）。ただし、最小値は1。

  ```
  威力 = floor(150 × 残りHP ÷ 最大HP)
  ```

* しおふき・ドラゴンエナジーと全く同じ威力計算式を使用する。

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: ひるみ効果あり（本プロジェクトは対応不要、通常の非接触技として扱う）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能（本プロジェクトはオウムがえし自体が未実装のため対応不要）

## リーサル計算

* 威力が自分自身の残りHPに依存するが、ダメージ計算対象の変化ではなく攻撃力側の可変要素であるため、
  `lethal.py` の想定する「相手HPの分布」（`StateDist`）には直接影響しない。
  リーサル計算は使用時点の実HPで威力を確定させる通常のダメージ計算ロジックで扱われるため、
  専用のリーサルハンドラは不要（`.internal/progress/move.md` のリーサル実装/リーサルテスト列は n/a）。
