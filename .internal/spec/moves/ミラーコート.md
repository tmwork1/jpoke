# ミラーコート（Mirror Coat）仕様書

参照: .internal/wiki/moves/ミラーコート.html（ポケモンWiki）

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: -（固定ダメージ技）
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: -5
* 接触: なし（直接攻撃ではない）

## 効果

* そのターンに相手から受けた特殊技のダメージを2倍にして、同じ相手に返す（ダメージ固定技）。
* 使用したターンに相手から特殊技によるダメージを受けていない場合は失敗する。
* 複数回特殊ダメージを受けていた場合（連続技など）は、**最後の1回分のダメージのみ**を2倍にする
  （合計ダメージではない）。
* 相手から特殊技を受けたが、無効化された場合や命中しなかった場合は跳ね返す対象にならない
  （＝そのターン他に特殊ダメージを受けていなければ失敗する）。
* みがわりが身代わりに受けたダメージは跳ね返せない（みがわりへのダメージは本体の
  被弾記録を更新しないため）。
* 味方から受けた特殊技には反撃しない（本プロジェクトはシングルバトル限定のため、通常は考慮不要）。
* カウンター（物理版）とは異なり、あくタイプには効果がある／ないの区別はなく、エスパータイプの
  技として通常のタイプ相性判定を受ける（あくタイプには無効）。

## 判定フラグ

* 急所: 発生しない（ダメージ固定技のため）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる（第五世代以降）
* おうじゃのしるし: 発動する（第五世代以降。ひるみ効果は追加効果扱いのため本来は対象外だが、
  本プロジェクトでは おうじゃのしるし 自体が「攻撃技でひるみ追加」の一般規則として実装されるべきもので、
  ミラーコート固有の実装は不要）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし・さきどり: コピー不可（第四世代以降）
* まねっこ: 第八世代以前はコピー不可だが、**スカーレット・バイオレット（本プロジェクトの対象）では
  コピー可能**（カウンターとは異なる）。したがって `non_copycat` フラグは付与しない。
* ゆびをふる・ねこのて: 選ばれない（両技とも未実装のため対応不要）
* ちょうはつ状態でも使用できる（変化技ではないため、ちょうはつの影響を受けない。特別な実装は不要）
* とつげきチョッキを持っていても使用できる（特殊技のため、特別な実装は不要）
* ミラーコート自身は直接攻撃ではない（さめはだ・ゴツゴツメット等の直接攻撃反応効果は発生しない）
* 特性ふしぎなまもりのポケモンに対しては無効化される（一般規則）

## 優先度根拠

`.internal/spec/turn.md` の `Event.ON_TRY_MOVE_1` テーブルに以下の記述がある：

```
| 30 | カウンター・ミラーコート: 適格ダメージ無 |
```

したがって、使用可否判定ハンドラは `Event.ON_TRY_MOVE_1` の priority=30 で実装する。

ダメージ計算の固定化（`Event.ON_MODIFY_MOVE_DAMAGE`）は turn.md に個別記載が無い内部イベントのため、
既存の同種ハンドラ（カウンター等）に合わせてデフォルト priority=100 を採用する。

## 既知の未対応エッジケース（対象外）

以下は第五世代以降の公式仕様として存在するが、いずれも「反撃対象の元HPがすでに1で、
きあいのハチマキ等の効果でダメージが0に補正されたことにより、実際にはHPが減少しなかった」という
極めて限定的な状況でのみ発生する。この場合、本来は「無効化された攻撃」とは異なりミラーコートは
失敗せず固定1ダメージを返す仕様だが、本実装ではダメージ量が0になった時点でその特殊攻撃自体を
「受けていない」ものとして扱うため、この極小エッジケースは未対応（対象外）とする（カウンターと同様）。

* 相手からの特殊攻撃でHPが0まで減る予定だったが、きあいのハチマキ等の効果でHP1で耐えた場合
  （耐えた結果HPが変化しなかったとき）→ 本来は1ダメージを返す
* 相手からの特殊攻撃を『ばけのかわ』の発動により無効化した場合 → 本来は1ダメージを返す
