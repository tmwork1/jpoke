# パラボラチャージ（Parabolic Charge）仕様書

参照: .internal/wiki/moves/パラボラチャージ.html（ポケモンWiki）
参照: src/jpoke/data/ps-champ-ja/moves.json（718行目、PP=20。champions基準でPPは8/12/16/20の4段階に統一）

参照: .internal/plan/moves/HP吸収技.md（HP吸収技グループの共通計画書）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 50（第六世代）→ 65（第七世代以降）。本プロジェクトは第七世代以降の値である65を採用する。
* 命中: 100
* PP: 20（champions基準。Gen9本家と同値）
* 対象: 自分以外の全員（本プロジェクトはシングルバトルのみが対象のため、対象は相手1体として扱う）
* 優先度: 0
* 接触: なし
* 弾（bullet）: 該当しない

## 効果

* 相手に与えたダメージの半分だけ、自分のHPを回復する（`_drain_hp` の `heal_ratio=0.5`。
  端数は四捨五入（round_half_up）、最低1回復を保証する）。
* 追加効果はなし。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率100）
* 追加効果判定: なし
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 特性・道具との相互作用（HP吸収技グループ共通）

* 使用者が『おおきなねっこ』を持つ場合、回復量が1.3倍になる
  （`Event.ON_CALC_DRAIN` 経由、`おおきなねっこ_boost_drain`）。
* 相手の特性が『ヘドロえき』の場合、回復ではなく同量のダメージを使用者が受ける
  （`ヘドロえき_reverse_drain`、`ctx.hp_change_reason == "drain"` を判定条件とする汎用実装）。
* 『かいふくふうじ』状態のポケモンは `flags` に `"heal"` を持つ本技を選択できない
  （`かいふくふうじ_try_action`、`Event.ON_TRY_ACTION`）。

## 技の仕様（本プロジェクトのスコープ外）

* 複数の相手に当てた場合の吸収順序（左のポケモンから順に吸収）は、ダブルバトル以上を
  前提とした仕様のため、シングルバトルのみを対象とする本プロジェクトでは対象外とする
  （CLAUDE.md「対象はポケモンチャンピオンズシングルバトルのみ」）。
