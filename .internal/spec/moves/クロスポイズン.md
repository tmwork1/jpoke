# クロスポイズン（Cross Poison）仕様書

参照: .internal/wiki/moves/クロスポイズン.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 物理
* 威力: 70
* 命中: 100
* PP: 20
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`, `slash`（切る技）, `secondary_effect`

## 効果

* 急所に当たりやすい（急所ランク+1）。
* 追加効果として、10%の確率で相手を『どく』状態にする。

## 判定フラグ

* 急所: 急所ランク+1（通常より高い確率で急所に当たる）
* 命中判定: あり（命中率100）
* 追加効果: あり（10%でどく状態）
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 発動する（追加効果を持つ攻撃技のため）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 技の仕様（きれあじとの相互作用）

* 「切る技」の1つ。特性きれあじのポケモンが使用すると威力が1.5倍になる
  （`flags` に `"slash"` を含めることで既存の `きれあじ_modify_power` ハンドラが適用される）。

## イベント / 優先度

* 追加効果（どく付与）は `ON_DAMAGE_HIT` で処理する。ダメージ計算内部のイベントで
  `.internal/spec/turn.md` には未掲載のため、既存の同種ハンドラ（どくづき・どくどくのキバ 等）と
  同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）に合わせる。
* `critical_rank=1` を `MoveData` に設定するのみで、既存の急所判定ロジック
  （`core/move_executor.py` の `CRIT_RATES` 参照処理）が自動的に適用される。

## リーサル計算

対象外（どく状態異常の付与は確定ダメージではなくターン経過依存の追加効果であり、
本プロジェクトのリーサル計算対象に含めない方針。`.internal/progress/move.md` の
「リーサル実装」列は `n/a`）。
