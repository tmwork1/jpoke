# すなかけ 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | じめん（第一世代のみノーマル） |
| 分類 | 変化 |
| PP | 15 |
| 命中率 | 100 |
| 範囲 | 相手1体選択 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

相手の **命中率** ランクを **1段階** 下げる。

第二世代以降はじめんタイプの技だが、ひこうタイプなど「地面にいない」ポケモンに対しても効果がある
（命中率ランクダウンは地面接地の有無を問わない）。

特性「するどいめ」「クリアボディ」など、命中率のランク補正を下げられない相手には効果が無い
（`battle.modify_stats` 側の既存処理で共通対応済み）。

## 判定

- 急所: なし（変化技）
- 追加効果: なし（本体の効果がランクダウンそのもの）
- まもるで防がれる: ○
- マジックコートで跳ね返される: ○
- オウムがえしでコピーされる: ○
- よこどりで奪われる: ×（対象を持たない変化技ではないため通常は奪われるが、Wiki記載どおり無効）

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- バトルパレスの判定基準（「ずるい」技）は実装しない
- Zワザによる付加効果（自分の回避率+1段階）は本プロジェクトの対象外（Zワザ非対応）のため実装しない
