## 効果

使用者自身の HP を、最大HPの 1/2 分回復する（`じこさいせい`・`タマゴうみ`と同一の効果）。

HP が満タンの場合は失敗する。

天候・特性等による回復量補正は存在しない（`あさのひざし`・`こうごうせい`・`つきのひかり`と異なり一律1/2固定）。

## かいふくふうじとの相互作用

- 使用者が **かいふくふうじ** 状態の場合、`heal` フラグを持つ技として選択できない
  （`かいふくふうじ_try_action` が汎用的に処理するため、本技専用の実装は不要）。
- `サイコノイズ` のかいふくふうじ対象技リストにも明記されている
  （`src/jpoke/data/ps-champ-ja/moves.json` 301行目）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない） |
| おうじゃのしるし | × |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |
| よこどり | ○ |
| オウムがえし | × |

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。
