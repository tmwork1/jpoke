# ブレイブバード（Brave Bird）仕様書

参照: .internal/wiki/moves/ブレイブバード.html（ポケモンWiki）
参照: src/jpoke/data/ps-champ-ja/moves.json（790行目、PP=16）

## 基本情報

* タイプ: ひこう
* 分類: 物理
* 威力: 120
* 命中: 100
* PP: 16（一次情報の値は15だが、`src/jpoke/data/ps-champ-ja/moves.json` の
  Champions側PP統一（8/12/16/20の4段階）により16が正）
* 対象: 相手1体（遠隔）
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 相手に与えたダメージの1/3を自分も受ける（反動技）。
  一次情報では「第四世代では与ダメージ×1/3(小数点以下切り捨て)、第五世代以降では
  与ダメージ×0.33(小数点以下四捨五入)」とあるが、本プロジェクトの反動技共通仕様
  （`.internal/spec/moves/_move_secondary_2.md`）に基づき `round_half_down`（五捨五超入）で丸める。
* 追加効果なし。

## 反動ダメージの計算

* 反動量 = `max(1, round_half_down(与えたダメージ * 1/3))`
  （端数は五捨五超入。参照: `.internal/spec/moves/_move_secondary_2.md`）
* 反動は `reason="recoil"` で処理する（いしあたま特性・マジックガード特性による無効化のため）

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代・第五世代以降とも対象。追加効果自体がないため実質関係しない）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* 同系統の反動技（1/3反動）: フレアドライブ、ウェーブタックル、ウッドハンマー、
  すてみタックル、ボルテッカー
* みがわりに命中した場合も反動ダメージは発生する。この場合の反動量は、
  みがわりへの与ダメージを基準に算出する。
