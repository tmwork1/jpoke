# くさむすび（Grass Knot）仕様書

参照: .internal/wiki/moves/くさむすび.html（ポケモンWiki）
参照: .internal/spec/moves/_variable_power.md（変動威力技共通仕様、第8節「体重依存威力技」）

## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 可変（20〜120）
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 相手（対象）の体重が重いほど、威力が大きくなる。
* 対象の体重（`weight` プロパティ。ライトメタル/ヘヴィメタル/かるいし等の補正込み）から
  以下の表で威力を決定する（第5世代以降の基準。本プロジェクトは第9世代基準）。

  | 対象体重 | 威力 |
  |---|---|
  | 〜9.9kg | 20 |
  | 10.0〜24.9kg | 40 |
  | 25.0〜49.9kg | 60 |
  | 50.0〜99.9kg | 80 |
  | 100.0〜199.9kg | 100 |
  | 200.0kg〜 | 120 |

* けたぐりと全く同じ威力計算式を使用する（対象が使用者ではなく相手である点のみ異なる）。

## 判定フラグ

* 急所: 通常通り発生する。
* 命中判定: あり（命中率100）。
* 追加効果: なし。
* まもる: 防がれる。
* おうじゃのしるし: ひるみ効果あり（通常の直接攻撃技として扱う）。
* マジックコート: 反射不可（変化技ではないため対象外）。
* よこどり: 対象外（変化技ではないため）。
* オウムがえし: コピー可能（本プロジェクトはオウムがえし自体が未実装のため対応不要）。

## リーサル計算

* 威力は相手（`defender`）の体重という固定パラメータにのみ依存し、相手のHP状態
  （`StateDist`）そのものを変化させる要素ではないため、ダメージ計算時に威力を確定させる
  通常のロジックで扱われる。専用のリーサルハンドラは不要
  （`.internal/progress/move.md` のリーサル実装/リーサルテスト列は n/a）。
