# 仕様書: トラップシェル

## 対象範囲外（実装しない）

本技は第七世代から存在する技だが、**ポケットモンスター ブリリアントダイヤモンド・シャイニングパール／
スカーレット・バイオレットで使用不可**（`.internal/wiki/moves/トラップシェル.html` のインフォボックスに
赤字で「BDSPで使用不可」「SVで使用不可」と明記されている）。`src/jpoke/data/ps-champ-ja/moves.json`
（ポケモンチャンピオンズの技一覧）にも本技の項目は存在せず、ポケモンチャンピオンズ側でも
採用されていないことが二重に確認できる。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」
に該当するため、本技は実装対象外とする。

過去のレビューにて `data/moves/move_ta.py` に本技の `MoveData`（タイプ「はがね」・威力150・PP5・
優先度-5、追加効果なし）が誤って登録されていたことが判明したため、本レビューで削除した。
詳細は `.internal/review/moves/トラップシェル.md` を参照。
なおタイプ「はがね」は一次情報（ほのおタイプ）と一致しない値であり、五十音順で隣接する
「トラバサミ」（チャンピオンズでくさ→はがねタイプに変更）との混同が原因と推測される。

### 技の仕様（参考: 第七・第八世代）

- 選択したターンの最初に「トラップシェルを仕掛けた!」というメッセージが出て発動待機状態
  （設置）になる。優先度+6（てだすけ5+いたずらごころ1）よりも前のタイミングで、
  きあいパンチの集中・くちばしキャノンの加熱と同じタイミングで行われる。
- 設置後、敵から物理攻撃を受けると起爆し、他のポケモンの行動に割り込んで即座に攻撃する
  （行動順の繰り上がり。さきおくりの効果を無視する）。優先度の高い物理技で起爆した場合ほど
  発動が早くなる。
- 物理攻撃により使用者がひんしになった場合は攻撃が行われない。
- 設置後、自身の行動順（優先度-3）を迎えるまでに起爆できなければ「しかしトラップシェルは
  不発に終わった!」となり失敗する（PPは消費される）。
- 起爆条件: 敵からの物理攻撃（直接・非直接を問わない）。味方の物理攻撃では起爆しない。
  ちからずくの効果が発動した物理技では起爆できない。物理攻撃を特性・タイプ相性・みがわりで
  無効化した場合は起爆しない。
- ねごと・オウムがえし・まねっこ・さいはいの対象外（non_negoto 相当）。
