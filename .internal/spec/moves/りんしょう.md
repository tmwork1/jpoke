# りんしょう（Round）仕様書

参照: .internal/wiki/moves/りんしょう.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 特殊
* 威力: 60（可変。効果参照）
* 命中: 100
* PP: 16（ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されており、
  `src/jpoke/data/ps-champ-ja/moves.json:950` 記載の値を採用する）
* 対象: 相手1体
* 優先度: 0
* 直接攻撃: なし
* 音技（sound）: あり

## 効果

* 同じターン中に複数体のポケモンがこの技を選ぶと、1体目が使った直後に2体目がこの技を
  使用する（『すばやさ』に関係なく最初に使用したポケモンのすぐ後に割り込む）。
  後で使った方の威力は120になる（2倍固定。3回目以降が使われても120のまま増えない）。
* 先に使われたりんしょうが相性・ぼうおん・まもる状態などに防がれた場合や、対象不在で
  失敗した場合であっても、他のりんしょうの行動順の繰り上がり・威力上昇は発生する。
* 音のわざの1つのため、みがわり状態を貫通する（本プロジェクトでは `sound` フラグにより
  汎用ルールとして既に処理される。本技専用のハンドラは不要）。
* テラレイドバトルでは他のプレイヤーに続けてりんしょうを使用しても威力は上がらない
  （本プロジェクト対象外）。

## 本プロジェクトでの実装方針（シングルバトル用スコープ）

本プロジェクトの対象はシングルバトルのみであり、場に出ているポケモンは常に自分・相手の
2体のみである。そのため、複数体のりんしょうが関わる公式仕様のうち、本プロジェクトで
意味を持つのは以下の1点のみである。

* **威力の上昇**: 同じターン中に自分・相手の両方がりんしょうを選択した場合、後で
  命中判定に進んだ方（＝2体目）の威力が120になる。

行動順の繰り上げ（「すばやさに関係なく最初に使用したポケモンのすぐ後に割り込む」）は、
シングルバトルでは行動主体が2体しかいないため、通常の優先度・すばやさによる行動順決定と
常に同じ結果になる（りんしょう同士の間に割り込む3体目以降が存在しないため）。
ただし、せんせいのツメ・クイックドロウ等の「行動順を変える効果」が一方にのみ発動して
両者のりんしょうの順序が入れ替わるという極めて稀なケースでは、公式ルール上はりんしょうの
繰り上げ効果がそれらより優先されるが、本プロジェクトではこの相互作用は実装しない
（既知の未実装事項として扱う）。

威力上昇の判定は、先に選択されたりんしょうが後から相性・まもる等で無効化された場合でも
成立する（先に使われた技が「使われたこと」自体は覆らないため）。本プロジェクトでは
技実行開始時点（`Event.ON_BEGIN_MOVE`、失敗判定より前）で「このターン、りんしょうが
既に使われたか」を記録することで、この挙動を再現する。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: 反射可能
