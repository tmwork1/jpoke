# わたほうし 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | くさ |
| 分類 | 変化 |
| PP | 20（ポケモンチャンピオンズ準拠。champions/move_list.txt = 20） |
| 命中率 | 100 |
| 範囲 | 相手全員（シングルバトルでは相手1体） |
| 優先度 | 0 |
| 直接攻撃 | × |
| ラベル | powder（粉技） |

## 効果

相手の **すばやさ** ランクを **2段階** 下げる（第五世代までは1段階）。

粉・胞子技（powder）のため、`くさ` タイプの相手や特性『ぼうじん』、道具『ぼうじんゴーグル』を
持っているポケモンには無効。

## 判定

- 急所: なし（変化技）
- 命中判定: ○
- 追加効果: なし（本体の効果がランクダウンそのもの）
- まもるで防がれる: ○
- マジックコートで跳ね返される: ○
- おうじゃのしるし・こんじょう等の追加効果系特性: 対象外（追加効果扱いではないため）
- オウムがえし: ○

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- バトルパレスの判定基準（「ずるい」技）は実装しない
- Zワザによる付加効果（自分の下がった能力を元に戻す）は本プロジェクトの対象範囲外のため実装しない
- 粉技のくさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』への無効化は
  `powder` ラベルによる汎用処理（`core/move_executor.py` の
  `_check_grass_type_powder_immunity` および特性・アイテムハンドラ）で対応済みのため、
  本技固有の実装は不要
