# チャージビーム（Charge Beam）仕様書

参照: .internal/wiki/moves/チャージビーム.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 50
* 命中: 90
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 基準。一次情報のGen9基準値は10だが、
  `.internal/champions/changes_from_sv.md` の全技PP統一（8/12/16/20）により12に変更されている）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、70%の確率で自分の『とくこう』ランクを1段階上げる。

## 判定フラグ

* 急所: 通常通り発生する。
* 命中判定: あり（命中率90）
* 追加効果: あり（ちからずくで発動率100%・威力1.3倍の対象。`secondary_effect`フラグを持つ）
* まもる: 防がれる
* おうじゃのしるし: ひるみ効果あり（直接攻撃技ではないが、一次情報上は対象。
  第四世代・第五世代以降とも対象）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能（本プロジェクトはオウムがえし自体が未実装のため対応不要）

## リーサル計算

* 命中後、追加効果が発動した場合に攻撃側の『とくこう』ランクが1段階上がるため、
  以降の攻撃のダメージ分布に影響する。しんぴのちから・フレアソング・ほのおのまい等の
  同種の「自分のとくこうが上がる追加効果」を持つ技と同様、専用のリーサルハンドラ
  （`LethalEvent.ON_HIT`, subject="attacker"）で対応する。
