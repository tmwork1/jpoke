# からをやぶる（Shell Smash）仕様書

参照: .internal/wiki/moves/からをやぶる.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -
* PP: 16（champions 基準。src/jpoke/data/ps-champ-ja/moves.json 参照）
* 対象: 自分
* 優先度: 0

## 効果

* 自分の『ぼうぎょ』『とくぼう』ランクを1段階ずつ下げ、『こうげき』『とくこう』『すばやさ』
  ランクを2段階ずつ上げる。

## 技の仕様（一次情報より）

* 『ぼうぎょ』『とくぼう』ランクの低下は、使用者自身が引き起こしたものとして扱われる。
  そのため以下が成立する。
  - しろいきり状態や、特性クリアボディ／しろいけむり／はとむね／フラワーベール／
    メタルプロテクト／ミラーアーマー、持ち物クリアチャームの効果では防げない。
  - 使用者自身の特性まけんき／かちきが発動することはない（能力低下の原因が自分自身のため）。
* 特性あまのじゃくを持つポケモンが使うと、逆に『ぼうぎょ』『とくぼう』ランクが1段階上がり、
  『こうげき』『とくこう』『すばやさ』ランクが2段階下がる。
* 特性たんじゅんを持つポケモンが使うと、『ぼうぎょ』『とくぼう』ランクが2段階下がり、
  『こうげき』『とくこう』『すばやさ』ランクが4段階上がる。
* すでに『ぼうぎょ』『とくぼう』ランクが最低（-6）のポケモンが使用しても、
  『こうげき』『とくこう』『すばやさ』ランクの上昇は通常通り発生する。
* すでに『こうげき』『とくこう』『すばやさ』ランクが最高（+6）のポケモンが使用しても、
  『ぼうぎょ』『とくぼう』ランクの低下は通常通り発生する。
* 上記が両方成立する場合（下げ側が最低かつ上げ側が最高）でも、ランクは変化しないが技自体は
  失敗しない。
* 使用者の持ち物しろいハーブ／だっしゅつパックの効果は、技を最後まで出し終えてから発動する
  （＝5つのランク変化がすべて適用された後に判定する）。
* 相手の場の特性びんじょうや持ち物ものまねハーブの効果も、技を最後まで出し終えてから発動する。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化がそのまま効果）
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* よこどり: 対象外（自分を対象とするため奪えない）
* オウムがえし: 反射不可（相手を対象としないため対象外）
