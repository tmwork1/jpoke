# みずのはどう（Water Pulse）仕様書

参照: .internal/wiki/moves/みずのはどう.html（ポケモンWiki）
src/jpoke/data/ps-champ-ja/moves.json:878（みず / 特殊 / 60 / 100 / 20 / 接触なし / まもる有効）

## 基本情報

* タイプ: みず
* 分類: 特殊
* 威力: 60
* 命中: 100
* PP: 20
* 対象: 相手1体（遠隔）
* 優先度: 0
* 接触: なし
* フラグ: `pulse`（波動技。メガランチャー等の対象）、`secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、20％の確率で相手をこんらん状態にする。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（20%・こんらん付与）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象外（本追加効果はこんらん付与効果のため、第四世代までは発動率上昇の対象だったが第五世代以降は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* `.internal/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。

## リーサル計算

* HP依存の追加効果・威力変動を持たない通常のダメージ技であり、リーサル計算上の特別な考慮は不要（対象外）。
