# どくのこな（Poison Powder）仕様書

参照: .internal/wiki/moves/どくのこな.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 変化
* 威力: -
* 命中: 75
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。Gen9本家は35だが本プロジェクトは
  champions基準のためPP20）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 相手を『どく』状態にする。
* 『どく』タイプ・『はがね』タイプには通常の毒付与と同様に無効（`ailment_manager` が
  一律に判定するため技側の実装は不要）。
* 粉・胞子技（`powder` フラグ）のため、以下の場合は無効になる（第六世代以降）。
  * 相手が『くさ』タイプを持つ
  * 相手の特性が『ぼうじん』
  * 相手の持ち物が『ぼうじんゴーグル』
  * 上記はいずれも `powder` フラグから導出される汎用ルールとして
    `core/move_executor.py` の `_check_grass_type_powder_immunity`、
    `ability.py` の `ぼうじん_block_powder`、`item.py` の
    `ぼうじんゴーグル_block_powder_move` が共通処理する。技側の実装は不要。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: あり（命中率75）
* 追加効果: なし（本技の状態異常付与自体が主効果であり、追加効果扱いではない）
* まもる: 防がれる
* みがわり: 防がれる（`move_executor.check_hit_substitute` が音技等の例外を除き一律に判定するため技側の実装は不要）
* おうじゃのしるし: 対象外
* マジックコート: 反射可能
* よこどり: 対象外（本技はよこどりで奪えない）
* オウムがえし: 反射可能
