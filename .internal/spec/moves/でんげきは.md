# でんげきは（Shock Wave）仕様書

参照: .internal/wiki/moves/でんげきは.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 60
* 命中: - （必中）
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 自分の攻撃は必ず命中する（自分の命中率、相手の回避率に関係なく必ず当たる）。
* 追加効果なし。

## 判定フラグ

* 急所: あり
* 命中判定: 必中（命中判定自体が行われない）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
