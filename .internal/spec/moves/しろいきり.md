# しろいきり 仕様書

参照: .internal/wiki/moves/しろいきり.html（ポケモンWiki）

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | こおり |
| 分類 | 変化 |
| 威力 | - |
| 命中率 | - （必中） |
| PP | 30 |
| 範囲 | 味方の場（第三世代以降。第二世代までは自分のみだが本プロジェクトは対象外） |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

自陣営の場を **5ターン** の間「しろいきり」状態にする。しろいきり状態の間、自陣営に出ている
ポケモンは、相手（または第三者）の技・特性による能力ランクの低下を防ぐ。すでに自陣営が
しろいきり状態のときに使用すると失敗する。

しろいきり状態そのものの詳細仕様（防げる例・防げない例、解除条件など）は
`.internal/spec/fields/しろいきり.md` を参照。

## 判定

- 急所: なし（変化技）
- 命中判定: なし（命中率 `-` の技、必中）
- 追加効果: なし（本体の効果がフィールド設定そのもの）
- まもるで防がれる: ×（範囲が「味方の場」のため対象を取らず、まもるの影響を受けない）
- マジックコートで跳ね返される: ×（範囲が「味方の場」のため反射対象外）
- よこどりで奪われる: ○（本プロジェクトではよこどり自体が未実装のため対象外）
- オウムがえしでコピーされる: ×（相手に向けて撃つ技ではないため）

## 発動タイミング

| イベント | Priority | 根拠 |
|---------|----------|------|
| Event.ON_STATUS_HIT | 100（デフォルト） | `.internal/spec/turn.md` に未掲載のイベント。`move_executor.py` が変化技の命中処理として発火する内部イベントであり、既存の同種ハンドラ（リフレクター・ひかりのかべ・オーロラベール・おいかぜ等）と同様にデフォルトpriorityを使用する |

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- バトルパレスの判定基準（「じぶん」技）は実装しない
- Zワザによる付加効果（自分のHPを全回復する）は実装しない（Zワザ非対応）
- PPは `src/jpoke/data/ps-champ-ja/moves.json` に未掲載（本技を習得できるチャンピオンズ対象ポケモンが
  存在しないため）。そのため本家準拠のPP30を採用する
  （リフレクター・ひかりのかべ・オーロラベールなど他の「味方の場」5ターン系変化技と異なり、
  8/12/16/20への変換値が確認できないための暫定対応）

## 備考

- 似た名前の技にくろいきりがあるが、こちらは能力ランクを下げられなくする「場の状態」であり、
  くろいきりの「使用したポケモン全員のランクを±0にリセットする」効果とは全く異なる。
  くろいきりはしろいきりの効果を無視してランクをリセットできる。
- きりばらい等の技によって相手の場のしろいきりを解除できる
  （`src/jpoke/data/ps-champ-ja/moves.json` の「きりばらい」の項を参照）。
