## 効果

使用者自身の HP を回復する。回復量は天候により変化する。

| 天候 | 回復量（最大HPに対する割合） |
|------|------------------------------|
| すなあらし | 2732/4096（≒2/3） |
| それ以外（天候なし・はれ・おおひでり・あめ・おおあめ・ゆき・らんきりゅう等） | 1/2 |

小数点以下は五捨五超入（`round_half_down`）で丸める。HP が満タンの場合は失敗する。

`あさのひざし`/`こうごうせい`と似た「天候依存の自己回復技」だが、以下の点が異なる:

- すなあらし以外の天候（はれ/おおひでり/あめ/おおあめ/ゆき/らんきりゅう）でも回復量は下がらない（常に1/2）。
  `あさのひざし`系は不利天候で1/4に低下するが、`すなあつめ`にはそのような低下がない。
- すなあらし状態のときのみ2732/4096（≒2/3）に増加する。

## ばんのうがさ・ノーてんき・エアロックとの相互作用

- 特性 **ノーてんき**・**エアロック** のポケモンが場にいる場合、すなあらし状態でも
  天候効果自体が無効化されるため回復量は上がらない（常に1/2）
  （`battle.weather_for(mon)` が天候無効時は「なし」を返すことで自動的に考慮される）。
- `ばんのうがさ`はすなあらしを弱化させる効果を持たないため、本技には影響しない。

## じめんタイプ・地面にいるかの影響

- じめんタイプの技だが、地面にいない（ふゆう等）ポケモンが使用しても効果がある
  （使用者自身が対象の技であり、相手への地面攻撃判定を経由しないため）。

## かいふくふうじとの相互作用

- 使用者が **かいふくふうじ** 状態の場合、`heal` フラグを持つ技として選択できない
  （`かいふくふうじ_try_action` が汎用的に処理するため、本技専用の実装は不要）。

## ヒーリングシフトとの相互作用

- 特性 **ヒーリングシフト** のポケモンが使用した場合、優先度が+3される
  （`.internal/spec/abilities/ヒーリングシフト.md` に本技が対象技として掲載済み。
  `heal` フラグから汎用的に判定されるため本技専用の実装は不要）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない） |
| おうじゃのしるし | ×（第五世代以降） |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |
| よこどり | ○ |
| オウムがえし | × |

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。

## 備考

- スナバァ系統（スナバァ・シロデスナ）の専用技。
- 端数処理はこうごうせい等と同じ五捨五超入で、じこさいせい等（小数点以下切り上げ）とは異なる。
