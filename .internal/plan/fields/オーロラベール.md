## ハンドラ一覧

### 1. オーロラベール_reduce_damage（ダメージ軽減）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_CALC_DAMAGE_MODIFIER |
| subject_spec | `defender:self`（AttackContext: 攻撃フェーズ） |
| priority | 100（デフォルト） |

**ロジック:**
1. `ctx.critical` が True → スキップ（急所では軽減しない）
2. `ctx.can_bypass_screen(battle)` が True → スキップ（すりぬけ等）
3. `ctx.move.name == "_こんらん"` → スキップ（こんらんの自傷ダメージは攻撃技扱いではない）
4. `ctx.move.category == "physical"` かつ `side.get("リフレクター").is_active` → スキップ（重複しない）
5. `ctx.move.category == "special"` かつ `side.get("ひかりのかべ").is_active` → スキップ（重複しない）
6. `apply_fixed_modifier(value, 2048)`（×0.5）

### 3. オーロラベール_check_weather（天候チェック）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_TRY_MOVE_1 |
| subject_spec | `attacker:self`（AttackContext: 攻撃フェーズ） |
| priority | 30 |
| 根拠 | .internal/spec/turn.md: 「オーロラベール: あられ状態でない」Priority 30 |

**ロジック:** `battle.weather.name != "ゆき"` なら失敗ログを出して `HandlerReturn(value=False, stop_event=True)`

### 4. オーロラベール_set_side_field（発動）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_STATUS_HIT |
| subject_spec | `attacker:self`（AttackContext: 攻撃フェーズ） |

**ロジック:** 自陣営に「オーロラベール」を5ターン設定する。すでに発動中の場合は `stop_event=True` で失敗。
