# 特性実装計画: うるおいボディ

更新日: 2026-06-05

## 仕様参照

- `.internal/spec/abilities/うるおいボディ.md`

## 実装ゴール

- 天気が雨（あめ・おおあめ）のとき、ターン終了時に状態異常を治す。
- 晴れ等の雨以外では発動しない。
- 状態異常がないときは発動しない。

## 実装方針

- `Event.ON_TURN_END`、`subject_spec="source:self"`、priority=60 で実装。
- priority は `turn.md` ON_TURN_END_2 priority 60「うるおいボディ / だっぴ」に従う。
- `battle.weather.rainy` で雨判定、`mon.ailment.is_active` で状態異常確認。
- **だっぴ**（30% 状態異常回復）と同じターン終了タイミング。

## 変更候補

- `src/jpoke/data/ability.py` — `うるおいボディ` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `うるおいボディ_on_turn_end` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. あめ中で状態異常あり → ターン終了時に回復する。
2. 晴れ中では発動しない。
3. 状態異常なしでは発動しない。

## 疑似コード

```python
def うるおいボディ_on_turn_end(battle, ctx, value):
    if not battle.weather.rainy:
        return HandlerReturn(value=value)
    mon = ctx.source
    if not mon.ailment.is_active:
        return HandlerReturn(value=value)
    battle.ailment_manager.cure(mon)
    announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
```
