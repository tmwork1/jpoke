# 特性実装計画: すいほう

更新日: 2026-07-11

## 仕様参照

- `.internal/spec/abilities/すいほう.md`

## 実装ゴール

- ほのおタイプの攻撃技を受けるとき、攻撃側のこうげき・とくこう（技の分類に応じて
  ダメージ計算で参照される方）を半減してダメージを計算する。タイプ相性の判定には影響しない。
- みずタイプの攻撃技を使うとき、自分のこうげき・とくこうを2倍にしてダメージを計算する。
- やけど状態にならない（付与をブロックする）。
- 特性が有効化された時点ですでにやけど状態なら即座に回復する
  （かがくへんかガス・かたやぶりの効果が終わって特性が再び有効になった場合や、
  なかまづくり等で特性が書き換わった場合が対象）。
- かたやぶりの効果がある技でほのお技の軽減が無視され、同時にやけど状態にされた場合も、
  技終了直後（特性が再び有効になったタイミング）で回復する。ラムのみ/チーゴのみを
  同時に持っている場合はアイテムが先に消費されてやけどが治るため、特性側の回復処理は
  何もしない（タイミングの自然な帰結であり追加実装は不要）。
- かえんだまを持っていても発動しない（ターン終了時の付与も `ON_BEFORE_APPLY_AILMENT` で
  同様にブロックされる）。
- みず技を使うときの補正はかたやぶりに無視されない（相手の特性ではなく自分の特性であり、
  `かたやぶり_disable_foe_ability` は `ctx.defender` の特性のみを対象とするため、攻撃側が
  すいほうの場合は対象外）。

## 実装方針

- 攻撃側補正（みず技強化）: `Event.ON_CALC_ATK_MODIFIER`、`subject_spec="attacker:self"` で
  `ctx.move.type == "みず"` のとき `apply_fixed_modifier(value, 8192)`（4096基準で2.0倍）を
  適用する `すいほう_boost_water` を実装する。
- 防御側補正（ほのお技軽減）: 同イベント `Event.ON_CALC_ATK_MODIFIER`、
  `subject_spec="defender:self"` で `ctx.move.type == "ほのお"` のとき
  `apply_fixed_modifier(value, 2048)`（4096基準で0.5倍）を適用する `すいほう_reduce_fire`
  を実装する。あついしぼうの `あついしぼう_reduce_fire_ice` と同一イベント・同一priorityの
  ため、`AbilityHandler` の既定値 `priority=100` を使用する。
- `flags={"mold_breaker_ignorable"}` を付与する。かたやぶりは `ctx.defender`（攻撃を受ける側）
  の特性のみを無効化するため、この付与によりほのお技軽減とやけど付与ブロックのみが影響を
  受け、みず技強化（attacker側）は影響を受けない。
- やけど付与ブロックは `Event.ON_BEFORE_APPLY_AILMENT`（`target:self`）で、既存の状態異常
  無効系特性と同じ `_prevent_ailment` 共通ヘルパーを `blocked_ailments=["やけど"]` で呼び出す
  `prevent_burn_ailment` を用いる（既存関数の再利用）。
- 特性有効化時点で既にやけど状態なら回復する処理は `Event.ON_ABILITY_ENABLED`
  （`source:self`）で `_cure_ailment_on_enable` 共通ヘルパーを `blocked_ailments=["やけど"]`
  で呼び出す `すいほう_cure_burn_on_enable` で実装する。
- `ON_CALC_ATK_MODIFIER` / `ON_BEFORE_APPLY_AILMENT` / `ON_ABILITY_ENABLED` はいずれも
  `.internal/spec/turn.md` に priority が明記されていない内部イベントのため、既存の同種ハンドラ
  （あついしぼう・めんえき・パステルベール・じゅうなん等）に倣い priority は指定せず既定値
  （100）を使う。

## テスト観点

1. ほのおタイプの技を受けたとき `atk_modifier` が 2048（0.5倍）になる
   （`test_すいほう_相手のほのお技を弱化`）。
2. みずタイプの技を使ったとき `atk_modifier` が 8192（2.0倍）になる
   （`test_すいほう_自分のみず技を強化`）。
3. やけど状態の付与を防ぐ（`test_状態異常無効`）。
4. かたやぶりの技でやけどにされても技終了直後に回復する
   （`test_状態異常無効_かたやぶりで無効`）。
