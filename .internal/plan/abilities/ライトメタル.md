# 特性実装計画: ライトメタル

更新日: 2026-06-05

## 仕様参照

- `.internal/spec/abilities/ライトメタル.md`

## 実装ゴール

- 自分の重さが半分になる（けたぐり・くさむすびのダメージ減少等）。

## 実装方針

**モデルレベルで実装済み。**

`src/jpoke/model/pokemon.py` の `Pokemon.weight` プロパティがすでに対応している：

```python
@property
def weight(self) -> float:
    w = self.data.weight
    match self.ability.name:
        case 'ライトメタル':
            w = int(w*0.5*10)/10
    ...
```

`data/ability.py` の `flags=["mold_breaker_ignorable"]` も設定済み。

**追加の実装作業は不要。テストの追加のみ行う。**

## 変更候補

- `tests/abilities/test_ability_yarawa.py` — テストを追加する

## テスト観点

1. `pokemon.weight` が基本値の 0.5 倍になる（0.1kg 未満は 0.1kg に下限）。
2. くさむすび等おもさ参照技の威力が、おもさの階級変化に応じて変わる。
3. かたやぶり系技受け時は ライトメタル が無視され基本値の重さ（威力）になる。
4. かるいしを持っている場合、特性の効果と累積しておもさが 1/4 になる。
