# 特性実装計画: いかりのこうら

更新日: 2026-06-05

## 仕様参照

- `.internal/spec/abilities/いかりのこうら.md`

## 実装ゴール

- HPが半分以下になったとき、こうげき・とくこう・すばやさが1段階上がり、ぼうぎょ・とくぼうが1段階下がる。
- HP が 50% 超から 50% 以下へ跨いだ瞬間に発動する（一度跨いだら再発動しない）。
- ひんしになった場合は発動しない（ON_DAMAGE_HIT は KO 後にも発火するため、`mon.alive` を明示的に確認する）。
- 第七世代からSVまでのちからずく無効化は、Championsでは撤廃されている（ぎゃくじょうと同様）。本実装は
  ちからずくの効果発動有無を判定していないため、対応は不要（Champions仕様に合致）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=100（デフォルト値、明示不要） で実装。
- ハンドラ先頭で `mon.alive` を確認し、瀕死のときは発動しない。
- `_crossed_half_hp(hp_before, hp_after, mon.max_hp)` で閾値跨ぎを確認（**ぎゃくじょう**と同ヘルパー）。
- B↓・D↓を先に適用し、A・C・S↑ が成立したときにアナウンスを出す。
- `value` は `ON_DAMAGE_HIT` 時の damage 量（正値）。
- 連続攻撃技対応: `ctx.hit_index == 1` のときの HP（`mon.hp + value`）を `ctx._angershell_hp_before`
  に保持し、最終ヒット（`ctx.hit_index == ctx.hit_count` または `ctx.attacker.fainted`）でのみ、
  その保持した「1発目を受ける前のHP」と現在HPで `_crossed_half_hp` を判定する
  （`かいがらのすず`の合計ダメージ集計と同じ idiom。`item.py` の
  `_shell_bell_total_damage` を参照）。

## テスト観点

1. HP 50% 超から 50% 以下へ跨いだとき、A・C・S↑1、B・D↓1 が発動する。
2. もともと HP が半分以下の状態で攻撃を受けても再発動しない。
3. ひんしになった場合は発動しない。
4. 連続攻撃技で2発目のヒットでHPが半分を下回っても、3発目のダメージ計算には
   反映されない（3発目終了後にまとめてランクが変動する）。

## 疑似コード

```python
def いかりのこうら_boost_on_half_hp(battle, ctx, value):
    mon = ctx.defender
    if not mon.alive:
        return HandlerReturn(value=value)

    if ctx.hit_index == 1:
        ctx._angershell_hp_before = mon.hp + value

    is_last_hit = ctx.hit_index == ctx.hit_count or ctx.attacker.fainted
    if not is_last_hit:
        return HandlerReturn(value=value)

    hp_before = ctx._angershell_hp_before
    hp_after = mon.hp
    if not _crossed_half_hp(hp_before, hp_after, mon.max_hp):
        return HandlerReturn(value=value)
    battle.modify_stats(mon, {"def": -1, "spd": -1}, source=ctx.attacker)
    if battle.modify_stats(mon, {"atk": +1, "spa": +1, "spe": +1}, source=ctx.attacker):
        announce_ability_triggered(battle, mon)
    return HandlerReturn(value=value)
```
