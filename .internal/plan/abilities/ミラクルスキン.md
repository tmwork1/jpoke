# 特性実装計画: ミラクルスキン

更新日: 2026-06-05

## 仕様参照

- `.internal/spec/abilities/ミラクルスキン.md`

## 実装ゴール

- 相手の変化技の命中率を 50% に固定する（命中率がすでに 50% 以下なら変化なし）。
- 攻撃技には適用されない。
- 必中の変化技には適用されない。
- かたやぶり系には無効。

## 実装方針

- `Event.ON_MODIFY_ACCURACY`、`subject_spec="defender:self"` で実装。
- `ctx.move.is_attack`（攻撃技は対象外）と `value is None`（必中技は対象外）を確認。
- `move_executor._check_hit` の命中率スケールはパーセント（0〜100）表記のため、
  `min(value, 50)` で50%を上限とする（4096スケールではない）。
- `かたやぶり` の無効化はハンドラ側では判定しない。`かたやぶり_disable_foe_ability`
  （`Event.ON_BEGIN_MOVE`）が `mold_breaker_ignorable` フラグを持つ特性を技実行中
  無効化する既存の仕組みに乗るため、`flags={"mold_breaker_ignorable"}` を
  data/ability.py に設定するだけでよい。

## テスト観点

1. 変化技（どくどく等）の命中率が 50% になる。
2. 攻撃技には適用されない。
3. 必中の変化技には適用されない。
4. かたやぶり系では無効。

def ミラクルスキン_reduce_accuracy(battle, ctx, value):
