# てんねん 実装計画

## スコープ
- 優先度: -
- 仕様書: `.internal/spec/abilities/てんねん.md`
- 実装方針: ダメージ計算の攻撃力・防御力ランク補正と、命中判定の命中率・回避率ランク補正の
  双方を無視する。既存の類似特性（するどいめ・せいなるつるぎ）と同じイベント構成に合わせる。

## 実装対象イベント
- `Event.ON_CALC_ATK_RANK_MODIFIER`（`.internal/spec/turn.md` 未掲載。ダメージ計算内部で
  `core/damage.py` の `_calc_final_attack` から発火。攻撃側実数値のランク補正計算に使う既存の
  同種特性〔いかさま等が絡む攻撃力算出〕に合わせて priority は指定しない＝デフォルト値を使う）
  - `subject_spec="defender:self"`: てんねんのポケモンが技を受けるとき、
    攻撃側の こうげき/とくこう ランク補正（イカサマ・ボディプレスで参照元が変わる場合を含め、
    実際に final_attack に使われる側のランク）を無視する（`value=1` 固定）
- `Event.ON_CALC_DEF_RANK_MODIFIER`（同上、`_calc_final_defense` から発火）
  - `subject_spec="attacker:self"`: てんねんのポケモンが技を使用するとき、
    防御側の ぼうぎょ/とくぼう ランク補正を無視する（`value=1` 固定）
- `Event.ON_GET_STAT_RANK`（`.internal/spec/turn.md` 未掲載。`core/move_executor.py` の
  `_check_hit` から命中判定直前に発火。命中率・回避率ランクを辞書で受け渡す。
  同種特性 するどいめ・技 せいなるつるぎ と同じ priority 未指定＝デフォルト値を使う）
  - `subject_spec="defender:self"`: てんねんのポケモンが技を受けるとき、
    相手（攻撃側）の 命中率 ランク補正を無視する（`value["accuracy"] = 0`）
  - `subject_spec="attacker:self"`: てんねんのポケモンが技を使用するとき、
    相手（防御側）の 回避率 ランク補正を無視する（`value["evasion"] = 0`）

## 仕様要点（実装に反映）
- このポケモンが技を使用するときは、相手の 回避率 ・ ぼうぎょ ・ とくぼう の ランク補正 を無視する。
- 他のポケモンから技を受けるときは、相手の 命中率 ・ こうげき ・ とくこう のランク補正を無視する。
- 相手の低下した ランク補正 も無視する（上昇・下降どちらも無視＝一律 `value=1` / `0` 固定でよい）。
- 素早さ のランク補正・急所ランクに関しては影響がない（対応イベントを登録しないことで自動的に対象外）。
- この特性はポケモンの行動順に影響しない（素早さ関連イベントは対象外のため触れない）。
- イカサマ・ボディプレスのように参照する実数値が入れ替わる技でも、
  `_calc_final_attack` / `_calc_final_defense` が実際に使う側のランクをそのまま無視すればよいため、
  技名で分岐する追加実装は不要（`core/damage.py` 側の分岐にすでに乗っている）。
- かたやぶり・特性を無視する技との関係は、`ability.flags = {"mold_breaker_ignorable"}` を付与すれば
  既存の `かたやぶり_disable_foe_ability`（`subject_spec="attacker:self"`、対象=foeの特性を無効化）が
  「てんねんのポケモンが攻撃を受ける側」のときのみ自動的にてんねんの全ハンドラを無効化する。
  「てんねんのポケモンが攻撃する側」のときは無効化されない（かたやぶりは攻撃側効果のため）。
  これは spec の「防御面においてかたやぶりがてんねんを無効化することはできない」と整合する。
  → 特性ハンドラ側でかたやぶり・かがくへんかガスの個別チェックは書かない（既存の仕組みに一任）。

   （`ON_CALC_ATK_RANK_MODIFIER` / `ON_CALC_DEF_RANK_MODIFIER` は実装済み）。

## テスト観点
- 攻撃側てんねん: 相手の ぼうぎょ/とくぼう ランクを無視する（実装済み・テスト済み）。
- 防御側てんねん: 相手の こうげき/とくこう ランクを無視する（実装済み・テスト済み）。
- 攻撃側てんねん: 相手の 回避率 ランク（上昇・下降）を無視する（新規追加）。
- 防御側てんねん: 相手の 命中率 ランク（上昇・下降）を無視する（新規追加）。
- 自分自身の素早さ・急所ランクには影響しない。
- ボディプレスをてんねん相手に使う場合、使用者自身のぼうぎょランクも無視される
  （既存テスト `test_ボディプレス_てんねん相手には自分のぼうぎょランクが無視される` で確認済み）。
- かがくへんかガス下では無効化される（既存テスト `test_かがくへんかガス_特性無効化` で確認済み）。
