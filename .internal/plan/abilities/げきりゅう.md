# 特性実装計画: げきりゅう

更新日: 2026-07-11

## 仕様参照

- `.internal/spec/abilities/げきりゅう.md`

## 実装ゴール

- 自分のHPが最大HPの1/3以下のとき、みずタイプの技で攻撃するときのこうげき・とくこうを1.5倍にする。
- タイプが変わる技を使用した場合、変更後のタイプがみずタイプであれば効果を適用する。
- みがわり状態でも本体のHP残量で発動判定する（みがわりのHPは無関係）。
- しんりょく（くさ）・もうか（ほのお）・むしのしらせ（むし）と同一ロジックのため、共通ハンドラを流用する。

## 実装方針

- `ON_CALC_ATK_MODIFIER` で `ctx.attacker.hp * 3 <= ctx.attacker.max_hp` かつ
  `ctx.move.type` が対応タイプ（みず）のとき、攻撃補正に 6144/4096 倍を適用する。
- `ctx.move.type` はダメージ計算前段（`ON_MODIFY_MOVE_TYPE` 等）で解決済みのタイプを参照するため、
  タイプが変わる技・そうでんによるタイプ変更にも自然に追従する（追加実装不要）。
- HP判定は `ON_CALC_ATK_MODIFIER` 発火時点の実数値を都度参照するため、連続攻撃技の途中でHPが変動した
  場合も次のヒットから即座に反映される（追加実装不要）。
- しんりょく・もうか・げきりゅう・むしのしらせで共通の
  `しんりょくもうかげきりゅうむしのしらせ_modify_atk` ハンドラを利用し、
  対応タイプは `PINCH_TYPE_BOOST_ABILITIES` 辞書（特性名→タイプ）で解決する。

## テスト観点

1. HPが最大HPの1/3以下のとき、みずタイプの技のこうげき・とくこう補正が1.5倍（6144/4096）になる。
2. しんりょく（くさ）・もうか（ほのお）・むしのしらせ（むし）でも同様にそれぞれのタイプで補正がかかる。
3. HPが最大HPの1/3を超えるときは補正がかからない。
4. みずタイプ以外の技には補正がかからない。
