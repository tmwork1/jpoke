# 特性実装計画: ちからずく

更新日: 2026-07-12

## 仕様参照

- `.internal/spec/abilities/ちからずく.md`

## 実装ゴール

- 追加効果つき攻撃技（`secondary_effect` フラグを持つ技）の威力を 1.3 倍にする。
- 追加効果を無効化する（`resolve_secondary_chance` を経由する効果は確率を 0 にすることで無効化）。
- `resolve_secondary_chance` を経由しない「常時発動する追加効果扱いの効果」（まきびし設置・
  ステルスロック設置・PP減少・にげられない付与など）は、各ハンドラ側で個別に
  `_is_sheer_force_blocked(ctx)`（`handlers/move_attack.py`）を呼んで無効化する。
- 適用時、使用者自身の いのちのたま反動 / かいがらのすず回復を無効化する（`handlers/item.py`）。
- 相手の だっしゅつボタン/レッドカード/アッキのみ/タラプのみ の発動を無効化する
  （`handlers/item.py`, `handlers/lethal.py`）。Champions でもこれらは無効化されたままだが、
  相手特性 ぎゃくじょう/わるいてぐせ は Champions では ちからずくの技に対しても発動する
  （＝これら2特性はちからずくの効果対象から除外＝現状の実装通りチェックを入れない）。

## 実装方針

- `Event.ON_CALC_POWER_MODIFIER`（`h.ちからずく_boost`, subject_spec="attacker:self"）で
  威力補正（`apply_fixed_modifier(value, 5325)`）を適用する。
- `Event.ON_MODIFY_SECONDARY_CHANCE`（`h.ちからずく_disable_secondary_effect`,
  subject_spec="attacker:self"）で追加効果確率を 0 にし `stop_event=True` を返す。
- 両ハンドラとも `ctx.move.has_flag("secondary_effect")` のみを判定条件とする。
- 対象技は「実際に確率つき追加効果を持つ技」に加え、第七世代以降ちからずくの対象になる
  フラグ自体が技データに設定されている技（かげぬい・ぶきみなじゅもん・がんせきアックス等、
  常時発動効果だが `secondary_effect` フラグを個別に付与）も含む。
  いっちょうあがりも同様（`.internal/spec/moves/いっちょうあがり.md` 参照。Championsでも対象のまま）。
  エレクトロビームは Champions で対象外のためフラグを付与しない。

## テスト観点

1. 威力 1.3 倍（`power_modifier == 5325`）。
2. 追加効果無効化（確率つき効果・常時発動効果の両方）。
3. 反動・吸収など「追加効果ではない」本体効果は維持される。
4. いのちのたま反動なし。
5. かいがらのすず回復なし。
6. ハイドロスチーム等 `secondary_effect` フラグを持たない技では通常通り発動する（対照テスト）。
