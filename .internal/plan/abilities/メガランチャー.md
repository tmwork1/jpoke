# 特性実装計画: メガランチャー

更新日: 2026-07-13

## 仕様参照

- `.internal/spec/abilities/メガランチャー.md`

## 実装ゴール

- 「波動技」（`MoveData.flags` に `"pulse"` を持つ技）の威力を1.5倍にする。
  対象: あくのはどう/こんげんのはどう/だいちのはどう/はどうだん/みずのはどう/りゅうのはどう。
- いやしのはどうを使用したときの回復量を、通常の最大HPの1/2(小数点切り上げ)から
  最大HPの3/4(小数点五捨五超入)に変更する。

## 実装方針

- `ON_CALC_POWER_MODIFIER` で攻撃側として `move.has_flag("pulse")` の技に6144/4096倍を適用する。
  かたいツメ（直接攻撃技1.3倍）・きれあじ（切る技1.5倍）・がんじょうあご（かみつき技1.5倍）と同一パターン。
- いやしのはどうの回復処理（`いやしのはどう_heal_defender`、`handlers/move_status.py`）内で
  `ctx.attacker.ability.name == "メガランチャー"` を判定し、回復量を
  `round_half_down(mon.max_hp * 3 / 4)` に変更する（通常時は `(mon.max_hp + 1) // 2` で切り上げ）。
- だいちのはどうの威力2倍（`だいちのはどう_power_modifier`）は `ON_CALC_POWER_MODIFIER` の乗算基準値
  4096 から始まる固定小数点補正のため、他の乗算的補正（メガランチャーの1.5倍・フィールドのタイプ一致
  1.3倍等）との適用順序に関わらず最終結果は一致する（×2倍が常に厳密値になるため、丸め誤差が
  他の補正の順序に影響しない）。したがって priority による明示的な順序制御は不要。
- priority は指定しない（`.internal/spec/turn.md` に `ON_CALC_POWER_MODIFIER` の記載はないため、既存の同種
  威力上昇特性ハンドラに合わせデフォルト順序とする）。

## テスト観点

1. はどうだん等の波動技に威力1.5倍（6144/4096）が適用される。
2. 波動技でない技には補正がかからない。
3. いやしのはどうの回復量が対象の最大HPの3/4（五捨五超入）になる。
4. だいちのはどう＋フィールド＋メガランチャーの複合補正が正しい値になる
   （`battle.damage_calculator.power_modifier` で検証）。
