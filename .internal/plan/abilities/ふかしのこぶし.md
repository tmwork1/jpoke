## 実装ゴール（完了定義）

- [x] 接触技でまもる系揮発性状態（まもる・トーチカ・キングシールド・スレッドトラップ・かえんのまもり・ニードルガード・ファストガード）を貫通し、1/4 のダメージを与える。
  - 仕様書に記載のブロッキング/ワイドガード/たたみがえしは本プロジェクトで未実装（ワイドガードはスプレッド技対象範囲を
    モデル化していないため実質ノーガード状態）のため対象外。
- [x] 非接触技はまもるを貫通しない。
- [x] パンチグローブ持ちのパンチ技は `is_contact` が False になるため貫通しない（`battle.query.is_contact` 経由で自動対応）。
- [x] 既存テストを壊さない。

## 実装対象イベント

- `Event.ON_CHECK_PROTECT` — 接触技のとき False を返しまもる貫通を実現
- `Event.ON_CALC_PROTECT_MODIFIER` — まもる系揮発性状態を持つ相手への接触技に対し 1024（1/4）を返す

## 実装メモ

- まもる貫通は `ON_CHECK_PROTECT` で False を返すことで実現（`_run_protect` が PROTECT_FAILED ログを出しつつ move を通す）。
- 1/4 ダメージは `ON_CALC_PROTECT_MODIFIER` で 1024 を返すことで実現（4096 の 1/4）。
- まもる状態は貫通後も揮発性状態として残る（ON_TURN_END で解除される通常フロー）。
- キングシールドの接触時A下げ等は現状未対応（`_run_protect` が PROTECT_FAILED 時に contact effects をスキップするため）。
