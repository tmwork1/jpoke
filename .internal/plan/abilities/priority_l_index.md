## 共通メモ

- `ON_DAMAGE_HIT` 発火時は damage > 0 が担保されているため damage > 0 ガード不要
- 接触判定は `battle.query.is_contact(ctx)` を使う（`move.has_label("contact")` の直呼び不可）
- 物理技判定は `battle.resolve_move_category(ctx.attacker, ctx.move) == "物理"`
- `ON_DAMAGE_HIT` は `turn.md` に記載のある priority（20 等）が明示された場合はそれに従い、未記載はデフォルト 100
- `turn.md` ON_DAMAGE priority 20 に列挙: のろわれボディ / くだけるよろい / みずがため / ほろびのボディ / いかりのつぼ / どくげしょう
- ランク変化は `battle.modify_stats(mon, {stat: n}, source=...)` を使う
- per_battle_once は `mon.ability.state == "used"` でフラグ管理
- `battle.get_side(mon)` でその Pokemon のサイドの `SideFieldManager` を取得
