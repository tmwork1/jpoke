# 特性実装計画: じんばいったい

更新日: 2026-06-05

## 仕様参照

- `.internal/spec/abilities/じんばいったい.md`

## 実装ゴール

- きんちょうかんとしろのいななき（またはくろのいななき）の効果を持つ。
- シングルバトルでは「味方がいない」ため実質的な固有効果はきんちょうかん相当のみ。
- `flags=["uncopyable", "protected", "gas_proof"]` は data/ability.py に既定済み
  （とくせいなし状態にもならず、かがくへんかガスによって無効化されることもない）。

## 実装方針

- `Event.ON_SWITCH_IN` でアナウンスのみ（**かたやぶり**と同様）。
- `Event.ON_CHECK_NERVOUS`（きのみ制限）は **きんちょうかん_check_nervous** を共有する。
- しろ/くろのいななき効果（自分の攻撃技でKOしたとき自身の能力が上がる）は
  `Event.ON_MOVE_KO` の `attacker:self` で発動する自己対象の効果であり、
  「相手が持っている場合の発動」ではないため実装が必要。
  はくばじょうのすがた（`バドレックス(はくば)`）はこうげき、こくばじょうのすがた
  （`バドレックス(こくば)`）はとくこうが1段階上がる。専用ハンドラ
  `じんばいったい_boost` を追加し、`ctx.attacker.name` でフォルムを判定して
  `_boost_on_move_ko`（しろのいななき/くろのいななきと共有するヘルパー）を呼ぶ。

## 変更候補

- `src/jpoke/handlers/ability.py` — `じんばいったい_boost` を実装する
- `src/jpoke/data/ability.py` — `じんばいったい` に handlers を登録する
- `tests/abilities/` — テストを追加する

## テスト観点

1. 入場時にアナウンスが出る。
2. 相手がきのみを使えなくなる（ON_CHECK_NERVOUS 経由）。
3. はくばじょうのすがたが攻撃技で相手を倒すとこうげきが1段階上がる。
4. こくばじょうのすがたが攻撃技で相手を倒すととくこうが1段階上がる。

## 検証コマンド

```powershell
python -m pytest tests/abilities/ -k "じんばいったい" -v
python -m pytest tests/ -v
```

## data/ability.py 登録（予定）

```python
"じんばいったい": AbilityData(
    flags=["uncopyable", "protected", "gas_proof"],
    handlers={
        Event.ON_SWITCH_IN: h.AbilityHandler(
            h.announce_ability_triggered,
            subject_spec="source:self",
        ),
        Event.ON_CHECK_NERVOUS: h.AbilityHandler(
            h.きんちょうかん_check_nervous,
            subject_spec="source:foe",
        ),
        Event.ON_MOVE_KO: h.AbilityHandler(
            h.じんばいったい_boost,
            subject_spec="attacker:self",
        ),
    }
),
```
