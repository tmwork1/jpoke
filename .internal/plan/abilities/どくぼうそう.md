# 特性実装計画: どくぼうそう

更新日: 2026-07-12

## スコープ

- 仕様書: `.internal/spec/abilities/どくぼうそう.md`
- 実装状態: 実装済み（レビューの結果、判定条件の誤りを修正）
- 実装方針: `Event.ON_CALC_POWER_MODIFIER` で攻撃者が「どく」「もうどく」状態かつ物理技を使用しているときに威力を1.5倍にする。

## 実装ゴール（完了定義）

- どく・もうどく状態のとき、使用する物理技の威力が1.5倍になる
- どく・もうどく状態でも特殊技・変化技には効果が発動しない
- どく・もうどく状態でないときは物理技でも効果が発動しない
- 既存テストを壊さない

## 実装対象イベント

- `Event.ON_CALC_POWER_MODIFIER` — 威力計算時の補正（priorityは `.internal/spec/turn.md` に個別記載なし。既存の同種ハンドラ `ねつぼうそう_modify_power` に合わせデフォルト優先度とする）

## 変更対象（ファイル/関数単位）

- `src/jpoke/handlers/ability.py` — `どくぼうそう_modify_power` を実装する
- `tests/abilities/test_ability_ta.py` — テストを追加する

## 実装ステップ

3. `src/jpoke/data/ability.py` の `どくぼうそう` に `Event.ON_CALC_POWER_MODIFIER` の handler を登録する。
4. `tests/abilities/test_ability_ta.py` にテストを追加し、全テストが通ることを確認する。

## テスト観点

1. どく・もうどく状態で物理技を使用したとき、威力が1.5倍になる。
2. どく・もうどく状態で特殊技を使用したとき、威力は上がらない。
3. どく・もうどく状態でないとき、物理技を使用しても威力は上がらない。
4. こんらんによる自傷行動（変化技扱い、category対象外）では威力が上がらない。
