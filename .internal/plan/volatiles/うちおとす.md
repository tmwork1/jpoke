# 実装計画書: うちおとす（揮発状態）

作成日: 2026-06-19
参照仕様書: .internal/spec/volatiles/うちおとす.md

## 実装概要

うちおとす揮発状態のハンドラを実装する。

## 実装済み内容

以下のハンドラは実装・登録済み:

| イベント | ハンドラ | subject_spec | priority | 説明 |
|---------|---------|-------------|----------|------|
| ON_CHECK_FLOATING | うちおとす_check_floating | source:self | 100 | 浮遊判定を無効化（stop_event=True で False を返す） |

- `ON_CHECK_FLOATING` は `.internal/spec/turn.md` に未掲載のため、既存の同種ハンドラ（くろいてっきゅう、ねをはる）の priority=100 を参照して決定。
- `subject_spec="source:self"` は浮遊判定の対象ポケモン自身がこの状態を持つパターン。

## 動作確認

- ひこうタイプ + うちおとす状態 → is_floating = False
- ふゆう特性 + うちおとす状態 → is_floating = False（うちおとすのstop_event=Trueが後続を防ぐ）
- ふうせん持ち + うちおとす状態 → is_floating = False
- でんじふゆう状態 + うちおとす状態 → is_floating = False（stop_event=True が優先される）
- 交代でうちおとす状態は解除される（揮発状態の共通仕様）
