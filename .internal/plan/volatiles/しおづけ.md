# しおづけ 実装計画書

## 概要

しおづけ揮発性状態の実装。ターン終了時に最大HPの1/16（みず・はがねタイプは1/8）のダメージを与える。

## 実装対象

### 1. しおづけ_damage（Event.ON_TURN_END）

**ファイル**: `handlers/volatile.py`
**登録**: `data/volatile.py`

- しおづけ状態のポケモンのターン終了時にダメージを与える
- ダメージ計算:
  - 通常: `r = -1/16`
  - みず・はがねタイプ: `r *= 2`（= -1/8）
  - 端数は切り捨て（`int(max_hp * r)`）
- `battle.modify_hp(mon, r=r)` で呼び出す
  - `ON_MODIFY_NON_MOVE_DAMAGE` が発火され、マジックガード特性が正しく機能する
- subject_spec: `"source:self"` （EventContext）
- priority: 100
  - 根拠: `.internal/spec/turn.md` の `Event.ON_TURN_END` より「しおづけダメージ」は Priority 100

## テスト方針

- ピカチュウ（通常タイプ）でターン終了後 `max_hp // 16` ダメージを受けることを確認
- みずタイプ（ゼニガメ）でターン終了後 `max_hp // 8` ダメージを受けることを確認
- はがねタイプ（コイル）でターン終了後 `max_hp // 8` ダメージを受けることを確認
- みず・はがね両方（エンペルト）でターン終了後 `max_hp // 8` ダメージを受けることを確認（片方と同じ）
- 交代でしおづけ状態が解除されることを確認
- マジックガード特性ポケモンがダメージを受けないことを確認

## 注意点

- SV（スカーレット・バイオレット）の1/8・1/4レートと混同しないこと。Champions仕様は1/16・1/8。
- 交代による解除は `switch_manager._switch_out` の `remove_all_volatiles` で自動的に処理される。
