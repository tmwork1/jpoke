# 実装計画書: しっぽをふる

## 仕様

### 技データ

- タイプ: ノーマル
- カテゴリ: 変化
- PP: 30
- 命中率: 100
- 対象: 相手1体（デフォルトの `"foe"`）

### 効果

相手の **ぼうぎょ** ランクを1段階下げる。

### 参照仕様書

- `.internal/spec/moves/しっぽをふる.md`

## 既存実装の確認結果

`src/jpoke/data/moves/move_sa.py` に以下の通り登録済みで、仕様と一致していることを確認した。

```python
"しっぽをふる": MoveData(
    type="ノーマル",
    category="status",
    pp=30,
    accuracy=100,
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.しっぽをふる_modify_defender_stats,
        )
    }
),
```

`target` は指定されていないため `MoveData` のデフォルト値 `"foe"`（相手1体）が適用され、
仕様の「対象: 相手1体」と一致する。

## subject_spec と priority

### イベント: `ON_STATUS_HIT`

- **subject_spec**: `"attacker:self"`（`MoveHandler` のデフォルト値を使用）
- **priority**: 100

**根拠**: `.internal/spec/turn.md` に `ON_STATUS_HIT` の独立エントリは無いが、同種の防御側ランクダウン
変化技ハンドラ（`あまえる_modify_defender_stats`, `にらみつける_modify_defender_stats`,
`なきごえ_modify_defender_stats` など）がいずれも `h.MoveHandler(...)` の priority デフォルト値
（100）で登録されているため、これと同一パターンとして priority=100 を採用する。

## ハンドラ構成

### 1. `src/jpoke/handlers/move_status.py`

共通ユーティリティ `handlers/move.py` の `modify_defender_stats` を利用する（既存実装済み）。

```python
def しっぽをふる_modify_defender_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"def": -1})
```

五十音順の挿入位置: `し` 行内の既存位置（`しめつける` 等の後、`しびれごな` の前）に配置済み。

### 2. `src/jpoke/data/moves/move_sa.py`

```python
"しっぽをふる": MoveData(
    type="ノーマル",
    category="status",
    pp=30,
    accuracy=100,
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.しっぽをふる_modify_defender_stats,
        )
    }
),
```

## リーサル計算

`.internal/progress/move.md` の該当行は「リーサル実装: n/a」のため対象外。
相手の能力ランクを下げるだけの変化技であり、ダメージを与えないため専用のリーサルハンドラは不要。

## 注意事項

- `modify_defender_stats` は `battle.modify_stats(ctx.defender, stats, source=ctx.attacker)` を
  呼び出す共通関数であり、まもる・マジックコート・特性による無効化などは既存の
  `battle.modify_stats` / まもる判定 / `ON_STATUS_HIT` 発火経路側で共通処理されるため、
  本ハンドラ側で個別対応は不要
- 確率発動ではない（`chance` 引数はデフォルト1のまま呼び出す）ため、`resolve_secondary_chance`
  は常に1を返し無条件でランクダウンする
