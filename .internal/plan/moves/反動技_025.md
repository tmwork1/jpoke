# 計画書: 反動技（反動量 1/4） — 反動技_025

## 仕様要約

`.internal/spec/moves/_move_secondary_2.md` より：与えたダメージの **1/4** を自分が反動として受ける技。

- 反動量: `max(1, round_half_down(damage / 4))`
  （2026-07 の すてみタックル レビューで、共通ヘルパー `_recoil` の端数処理を `int()` 切り捨てから
  `round_half_down`（五捨五超入、第五世代以降の反動ダメージ計算仕様）に修正した。本グループの技にも適用される）
- 反動は `reason="recoil"` で処理する（いしあたま特性による無効化のため）
- ボルテッカーは既存の追加効果ハンドラ（10%まひ、`ON_DAMAGE_HIT`）を維持したまま `ON_HIT` に反動ハンドラを追加する

## handlers/move_attack.py に追加する関数

五十音順に挿入。既存の `わるあがき_self_damage`（r=-1/4 で最大HP比）とは異なり、
**与えたダメージの絶対値**で計算する（`v=` パラメータ使用）。

```python
def アフロブレイク_recoil(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    recoil = max(1, round_half_down(value / 4))
    battle.modify_hp(ctx.attacker, v=-recoil, reason="recoil", source=ctx.attacker)
    return HandlerReturn(value=value)
```

## 注意点・エッジケース

- `reason="recoil"` を指定することで、いしあたま特性が実装された際に自動的に無効化される
- `わるあがき_self_damage` は最大HP比（`r=-1/4`）、反動技は与ダメ比（`v=-recoil`）で計算する点が異なる
- ボルテッカーの追加効果（まひ）は `ON_DAMAGE_HIT`、反動は `ON_HIT` で発火。`ON_DAMAGE_HIT` が `ON_HIT` より先に発火される
- 仕様書確認: `_move_secondary_2.md` ではボルテッカー反動比率 = 0.25（1/4）で正しい
