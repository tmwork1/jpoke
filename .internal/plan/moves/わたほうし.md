## 仕様要約

- タイプ: くさ / カテゴリ: 変化 / PP: 20（champions/move_list.txt準拠） / 命中: 100 / 範囲: 相手全員（シングルでは相手1体）
- 相手の「すばやさ」ランクを 2 段階下げる
- 粉技（powder ラベルあり）。くさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』には無効
  （汎用の powder 処理で対応済みのため個別実装不要）

## 実装箇所

### `src/jpoke/handlers/move_status.py`

五十音順の挿入位置: 「わ」行 — `ロックカット_modify_attacker_stats` の後、`わるだくみ_modify_attacker_stats` の前。

```python
def わたほうし_modify_defender_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """わたほうしの効果: 相手のすばやさを2段階下げる。粉技のためくさタイプ・ぼうじん無効。"""
    return modify_defender_stats(battle, ctx, value, stats={"spe": -2})
```

### `src/jpoke/data/moves/move_wa.py`

```python
"わたほうし": MoveData(
    type="くさ",
    category="status",
    pp=20,
    accuracy=100,
    flags={"powder"},
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.わたほうし_modify_defender_stats,
        ),
    }

```

## 注意点・エッジケース

- 粉技（powder）ラベルはすでに `data/moves/move_wa.py` に設定済み。くさタイプ・ぼうじん特性・
  ぼうじんゴーグルへの無効化は `core/move_executor.py` の粉技免疫処理・特性/アイテムハンドラで
  処理されるため、本ハンドラ側では不要
- S（すばやさ）ランク下限チェック（-6 で効果なし）は `battle.modify_stats` 内部で処理されるため、
  ハンドラ側での追加チェックは不要
- リーサル計算: 対象外（ダメージを与えない変化技のため `.internal/progress/move.md` は n/a）
