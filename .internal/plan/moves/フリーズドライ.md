# 計画書: フリーズドライ

## 仕様要約

- こおりタイプ / 特殊 / 威力70 / 命中100 / PP20 / 対象: 相手1体 / 直接攻撃なし
- みずタイプに対してのみタイプ相性が特例的に効果抜群（2倍）になる
  （こおり×みず本来の1/2倍を打ち消して2倍にする。他のタイプとの相性は通常のこおりタイプと同じ）
- **Champions ではSVまであった「10%でこおり状態にする」追加効果が削除されている**
  （`.internal/champions/changes_from_sv.md` 参照）。よって本プロジェクトでは追加効果ハンドラを持たない
- 詳細仕様は `.internal/spec/moves/フリーズドライ.md` を参照

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_CALC_DEF_TYPE_MODIFIER` | デフォルト | attacker:self（省略時デフォルト） | みずタイプへのタイプ相性を4倍補正し、1/2倍→2倍にする |

## Priority 根拠

`ON_CALC_DEF_TYPE_MODIFIER` は `.internal/spec/turn.md` に未掲載のダメージ計算内部イベント（タイプ相性計算時に発火）。
同種のタイプ相性上書きハンドラである `フライングプレス_add_flying_type`（`ON_CALC_DEF_TYPE_MODIFIER`、priority省略）に倣い、
デフォルト priority を使用する。

## 実装箇所（実装済み）

### handlers/move_attack.py

```python
def フリーズドライ_water_effectiveness(battle: Battle, ctx: AttackContext, value: int) -> HandlerReturn:
    """フリーズドライ: みずタイプに対して効果抜群（2倍）になる。
    こおりタイプはみずに0.5倍のため、最終2倍になるよう4倍補正（16384）をかける。
    """
    if "みず" in ctx.defender.types:
        value = apply_fixed_modifier(value, 16384)
    return HandlerReturn(value=value)
```

### data/moves/move_ha.py

```python
    "フリーズドライ": MoveData(

        category="special",
        pp=20,

```

## 注意点・エッジケース

- `value * 16384 // 4096 = value * 4` により、防御側の元のタイプ相性計算結果（複数タイプの積）に対して
  みずタイプ由来の 1/2 倍成分だけを 2 倍に置き換える効果になる。タイプ相性計算が各タイプ倍率の積で
  構成されているため、全体を4倍することは「みず成分のみを1/2倍→2倍に置換する」ことと数学的に等価であり、
  みず/ひこう（4倍弱点扱い）等の複合タイプでも正しく動作する。
- Champions では追加効果（こおり状態付与）が削除されているため、`secondary_effect` フラグを付与しない。
  これにより特性ちからずくも本技に対しては何の効果も持たない（元々発動する効果自体が存在しないため）。
- 直接攻撃技ではないため `contact` フラグは付与しない。
- リーサル計算: 追加効果が存在しないため対象外（`n/a`）。通常のダメージ計算のみで完結する。
