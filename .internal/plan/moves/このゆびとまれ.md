# 実装計画書: このゆびとまれ

参照: .internal/spec/moves/このゆびとまれ.md

## 仕様要約

- タイプ: ノーマル / 分類: 変化 / PP: 20 / 命中: - / 優先度: +3（第五世代以前）/ +2（第六世代以降）
- 効果: **ちゅうもくのまと状態**になり、そのターンの相手の技をすべて自分が受ける。対象を選択する技にしか効果はない。
- **ダブル/トリプルバトル専有の効果**。本プロジェクト（シングルバトル専用）では
  `battle.actives` が常に2体以下であり、対象リダイレクト自体に意味がない。
- いかりのこなと異なり粉・胞子技には分類されず、くさタイプ・特性「ぼうじん」・道具
  「ぼうじんゴーグル」による無効化手段も存在しない。

## 実装方針の判断（対象外）

- `.internal/progress/move.md` の このゆびとまれ 行は「仕様書」「計画書」「実装」「テスト」
  「リーサル実装」「リーサルテスト」列すべてが「ダブル専用」であり、本プロジェクトの
  対象外として扱う。
- 類似技「いかりのこな」も同じ理由で全列「ダブル専用」として扱われており
  （`.internal/progress/move.md` 該当行、`.internal/plan/moves/いかりのこな.md`）、一貫した方針とする。
- `.internal/spec/turn.md` の `Event.ON_TRY_MOVE_1` 表には「このゆびとまれ: シングルバトルである」
  （priority=30）という失敗条件の記載が存在するが、これは将来ダブルバトル対応を行う場合の
  参考情報として残されているものであり、本プロジェクトの `Battle` は常に
  `len(battle.actives) <= 2`（各プレイヤー1体ずつ）という構造的制約を持つため、この失敗条件を
  実装すると「このゆびとまれは常に必ず失敗する」という自明で分岐のないハンドラになる。
  この技だけを特別扱いして実装することは、同条件を満たす他の全ダブル専用技
  （いかりのこな・おさきにどうぞ・てだすけ等）との一貫性を欠くため見送る。

## 注意点・エッジケース（将来ダブルバトル対応時のために記録）

1. **このゆびとまれ自体に `powder` フラグを付与してはならない**:
   いかりのこなと異なり、このゆびとまれは第六世代以降も粉・胞子技に分類されないため、
   くさタイプ・特性「ぼうじん」・道具「ぼうじんゴーグル」による無効化を一切受けない。
2. **いかりのこなとの関係**: 第五世代では両技は完全に同一効果だったが、第六世代以降は
   いかりのこなのみ粉技に分類され、無効化手段が追加された。このゆびとまれは無印のまま。
