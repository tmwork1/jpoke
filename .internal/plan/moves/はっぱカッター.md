# 計画書: はっぱカッター

## 仕様要約（.internal/spec/moves/はっぱカッター.md 参照）

- くさタイプ / 物理 / 威力55 / 命中95 / PP25 / 対象: 相手（シングル戦のため相手1体） / 優先度0 / 接触なし
- 追加効果なし。急所ランク+1（急所に当たりやすい）。

## 既存実装との差分

既存実装（`src/jpoke/data/moves/move_ha.py` の `"はっぱカッター"` エントリ）は仕様と一致しており、
修正は不要。

1. **基本情報は仕様通り**
   `type="くさ", category="physical", pp=25, power=55, accuracy=95` で一致。

2. **急所ランクは `critical_rank=1` で仕様通り**
   追加効果を持たない技のため専用ハンドラは不要。`critical_rank` フィールドのみで
   `core/move_executor.py` の急所判定ロジック（`CRIT_RATES[critical_rank]`）が
   自動的に適用される。

3. **`flags` は `{"slash"}` で仕様通り**
   接触技ではない（`contact` フラグなし）が、「切る」技に分類される
   （きれあじ特性の威力1.5倍判定に影響するフラグ）。

## 注意点・エッジケース

- 急所ランクは通常のダメージ計算の急所判定に組み込まれるため、まもる等で防がれた場合は
  ダメージ自体が発生せず急所判定も行われない（既存の共通ロジックに準拠）。
- 追加効果を持たないため、ちからずくの影響を受けない（威力・効果とも変化しない）。
- 原作では相手全員が対象の拡散技だが、本プロジェクトはシングルバトルのみ対象のため
  `target` は既定値 `"foe"`（相手1体）のままでよい。
