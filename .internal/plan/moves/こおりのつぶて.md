# こおりのつぶて 実装計画書

## 仕様要約（.internal/spec/moves/こおりのつぶて.md 参照）

* タイプ: こおり / 分類: 物理 / 威力: 40 / 命中: 100 / PP: 30
* 対象: 相手1体 / 優先度: +1 / 非接触
* 効果: 追加効果なし。優先度+1により必ず先制攻撃できる。

## 現状の実装状況

`src/jpoke/data/move.py` に既に登録済みで、仕様と一致している（変更不要）。

```python
"こおりのつぶて": MoveData(
    type="こおり",
    category="physical",
    pp=30,
    power=40,
    accuracy=100,
    priority=1,
    handlers={},  # 追加効果なし
),
```

## ハンドラ構成

追加効果を持たないため、`handlers/move_attack.py` への新規実装は不要。
優先度（`priority=1`）は `MoveData` の宣言のみで `TurnController` の行動順決定に反映されるため、
専用ハンドラは不要。

## 注意点・エッジケース（review-test 向け）

1. **優先度+1の確認**
   - すばやさに関係なく先制できることをテストで確認する
     （`test_こおりのつぶて_優先度により素早さが低い側が先制する`）。
2. **ダメージ確認**
   - 通常のダメージ計算のみであることを確認する
     （`test_こおりのつぶて_相手にダメージを与える`）。
3. **まもるとの相互作用**
   - 優先度技だが通常通り「まもる」で防がれる（一次情報の判定表より）。
