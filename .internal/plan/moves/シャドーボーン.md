# 計画書: シャドーボーン

## 結論: 実装しない

`.internal/spec/moves/シャドーボーン.md` の通り、本技は第八世代までは使用可能だったが
ブリリアントダイヤモンド・シャイニングパール／スカーレット・バイオレットでは使用不可であり、
README.md の対象範囲定義（第9世代で実装されていない技は実装しない）に反するため実装計画は策定しない。

## 経緯

過去の一括実装計画 `.internal/plan/moves/攻撃技_防御側ランク変化系.md`（防御側の能力ランクを下げる追加効果を
まとめて実装するバッチ計画）で、対象範囲の確認が行われないまま以下の内容で登録されていた:

- `src/jpoke/data/moves/move_sa.py` の `MoveData(type="ゴースト", category="physical", pp=10, power=85,
  accuracy=100, flags={"contact", "secondary_effect"}, ...)`
  - 一次情報（`.internal/wiki/moves/シャドーボーン.html`）によれば `直接攻撃: ×` であり、
    `flags` に `"contact"` を含めていたのは誤りだった
- `src/jpoke/handlers/move_attack.py` の `シャドーボーン_lower_defender_def`
  （`modify_defender_stats(battle, ctx, value, stats={"def": -1}, chance=0.2)`。
  20%の確率で相手のぼうぎょを1段階下げる効果自体は一次情報と一致していた）
- `src/jpoke/types/move.py` の `MoveName` Literal への登録
- `tests/moves_attack/test_move_sa.py` の `test_シャドーボーン_ぼうぎょ1段階低下が発動する`

本レビュー（`.internal/review/moves/シャドーボーン.md`）にて対象範囲外と判断し、上記実装・テストを削除した。
今後、本技を再実装する必要が生じた場合は、まず README.md の対象範囲定義とポケモンチャンピオンズでの
使用可否を再確認すること。
