# エナジーボール 実装計画書

## 仕様要約（.internal/spec/moves/エナジーボール.md 参照）

* タイプ: くさ / 分類: 特殊 / 威力: 90 / 命中: 100 / PP: 12
* 対象: 相手1体 / 優先度: 0 / 非接触
* 効果: 追加効果として、10%の確率で相手の『とくぼう』ランクを1段階下げる。
* 弾（bullet）技に分類される（一次情報の「弾のわざ」カテゴリより）。

## 現状の実装状況

`src/jpoke/data/moves/move_a.py` に既に登録済み。レビューにより以下2点の誤りを修正した：

1. PP が `10`（Gen9本家基準）になっていたが、`src/jpoke/data/ps-champ-ja/moves.json` の値である `12` に修正。
2. `flags` に `"secondary_effect"` が付与されておらず、ちからずく（追加効果技の威力1.3倍化・
   追加効果無効化）が正しく機能しない状態だったため追加。

修正後:

```python
"エナジーボール": MoveData(
    type="くさ",
    category="special",
    pp=12,
    power=90,
    accuracy=100,
    flags={"bullet", "secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.エナジーボール_lower_defender_spd,
        )
    }
),
```

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 10%の確率で相手の『とくぼう』ランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・修正不要）:

```python
def エナジーボール_lower_defender_spd(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"spd": -1}, chance=0.1)
```

`modify_defender_stats`（`handlers/move.py`）が `battle.resolve_secondary_chance` を経由するため、
ちからずくによる追加効果無効化（確率0化）・威力1.3倍化は `flags={"secondary_effect"}` さえ
付与されていれば汎用的に処理される。本技側に個別のガードは不要。

## Priority 根拠

ダメージを与えた技の追加効果（能力ランク低下）は既存の同種ハンドラ（あわ・いにしえのうた・
きあいだま 等）と同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）のパターンに従う。
`ON_DAMAGE_HIT` は `.internal/spec/turn.md` に未掲載のイベント（ダメージ計算内部）だが、
既存の同種ハンドラの実装パターンを踏襲するため個別の priority 検討は不要。

## リーサル計算

対象外（`.internal/progress/move.md` のリーサル実装列は n/a）。確率発動の相手ランク低下効果であり、
ダメージ量そのものを変化させないため、リーサル計算（`core/lethal.py`）の対象としない。

## 注意点・エッジケース（review-test 向け）

1. **ちからずくとの相互作用**
   - `flags={"secondary_effect"}` が付与されているため、ちからずく所持時は威力1.3倍（90→117）になり、
     とくぼう低下の追加効果は発動しなくなる。
2. **弾（bullet）フラグ**
   - 特性『ぼうじん』の無効化対象となる。個別のガードは不要（`ぼうじん` 側で判定される）。
3. **テスト方針**
   - `secondary_chance` を0/1に固定し、とくぼうランクの変化有無を確認するテストを追加する
     （`test_エナジーボール_相手にダメージを与える` / `test_エナジーボール_とくぼう1段階低下が発動する` /
     `test_エナジーボール_とくぼう1段階低下が発動しない`）。
