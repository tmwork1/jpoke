# だいちのちから 実装計画書

## 仕様要約（.internal/spec/moves/だいちのちから.md 参照）

* タイプ: じめん / 分類: 特殊 / 威力: 90 / 命中: 100 / PP: 10
* 対象: 相手1体 / 優先度: 0 / 接触なし
* 効果: 追加効果として、10%の確率で相手の『とくぼう』ランクを1段階下げる。

## 現状の実装状況

`src/jpoke/data/move.py` に既に登録済み（`flags={"secondary_effect"}`）。

```python
"だいちのちから": MoveData(
    type="じめん",
    category="special",
    pp=10,
    power=90,
    accuracy=100,
    flags={"secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.だいちのちから_lower_defender_spd,
        )
    }
),
```

内容は仕様書と一致しており、修正不要。

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 10%の確率で相手の『とくぼう』ランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・修正不要）:

```python
def だいちのちから_lower_defender_spd(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"spd": -1}, chance=0.1)
```

`modify_defender_stats`（`handlers/move.py`）が `battle.resolve_secondary_chance` を経由するため、
ちからずくによる追加効果無効化（確率0化）・威力1.3倍化は `flags={"secondary_effect"}` さえ
付与されていれば汎用的に処理される。本技側に個別のガードは不要。

## Priority 根拠

ダメージを与えた技の追加効果（能力ランク低下）は既存の同種ハンドラ（あわ・いわくだき・エナジーボール 等）と
同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）のパターンに従う。個別の priority 検討は不要。

## 注意点・エッジケース（review-test 向け）

1. **ちからずくとの相互作用**
   - `flags={"secondary_effect"}` により、ちからずく所持時は威力1.3倍（90→117）になり、
     とくぼう低下の追加効果は発動しなくなる。
2. **テスト方針**
   - `secondary_chance` を0/1に固定し、とくぼうランクの変化有無を確認するテストを追加する
     （相手にダメージを与える／とくぼう低下が発動する／とくぼう低下が発動しない、の3パターン）。
