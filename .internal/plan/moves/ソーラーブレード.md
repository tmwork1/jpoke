# 計画書: ソーラーブレード

## 仕様要約（.internal/spec/moves/ソーラーブレード.md 参照）

- くさタイプ・物理・威力125・命中100・PP12（champions基準。src/jpoke/data/ps-champ-ja/moves.json により全技PPは8/12/16/20の4段階に統一されているため、
  wiki記載のPP10（Gen9基準）ではなく12を採用する）
- 2ターン技。1ターン目は攻撃せず溜め、2ターン目に攻撃する。
- 天気が「はれ」「おおひでり」状態のとき、1ターン目に溜めず即座に攻撃する。
- 攻撃時の天気が「あめ」「おおあめ」「すなあらし」「ゆき」のとき威力が半減する（らんきりゅうは影響なし）。
- ばんのうがさ所持時: にほんばれでも溜めるようになり、あめでは威力が下がらなくなる（すなあらし/ゆき/おおあめでは半減のまま）。
  → `battle.weather_for(attacker)` が ON_CHECK_WEATHER_IMMUNE 経由でばんのうがさを判定するため、専用実装は不要。
- パワフルハーブ所持時: 溜めをスキップして即攻撃する（にほんばれ下でも同時成立する場合は天気が優先されアイテムは消費されない）。
  → 既存の汎用ハンドラ `パワフルハーブ_skip_charge`（`data/item.py`, priority=100）がそのまま適用される。
- flags: `contact`（直接攻撃）, `slash`（切る技。きれあじで威力1.5倍）。
- ソーラービームの物理版で威力が5高いが、揮発状態・チャージ処理はソーラービームと共有せず専用（`ソーラーブレード` という別名の揮発状態を使う）。

## Priority 根拠

### ON_MOVE_CHARGE（weather_skip: priority=90 / charge: priority=100）

`.internal/spec/turn.md` の `Event.ON_MOVE_CHARGE` セクションには「隠れる技: 状態変化」等priority=100の例のみ記載され、ソーラー系の天候スキップ判定については明記がない（未掲載のイベント）。
そのため、既に実装済みで動作実績のある `ソーラービーム` の実装パターンをそのまま踏襲する：

- `ソーラービーム系_weather_skip` を priority=90 とし、`パワフルハーブ_skip_charge`（ItemHandler, priority=100）より先に判定させる。
  これにより、天気による自動スキップの判定が確定してから、まだ溜め状態でない場合に限りパワフルハーブの消費判定に進む
  （天気優先でパワフルハーブが消費されない仕様を保証するため）。
- `ソーラービーム系_charge` はデフォルトpriority=100とし、パワフルハーブと同一priorityで実行順は不定だが、
  weather_skipが先に`stop_event=True`を返すケース（にほんばれ下）では到達しないため問題にならない。

### ON_CALC_POWER_MODIFIER（デフォルトpriority=100）

turn.mdに掲載のないイベント（ダメージ計算内部）。きれあじ（`ON_CALC_POWER_MODIFIER`, デフォルトpriority）等、他の威力補正ハンドラと同様にデフォルトpriorityで登録する
（乗算的な補正のため、実行順序が結果に影響しない）。

def ソーラーブレード_charge(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:

## 注意点・エッジケース

### PPの修正

- `src/jpoke/data/ps-champ-ja/moves.json` (552行目) では ソーラーブレード の PP は 12。
- 修正前の実装は `pp=10`（wiki記載のGen9基準の値。移行漏れのバグ）だったため、本レビューで `pp=12` に修正した。
- `.internal/progress/move.md` のPP列は既に12と記載されており（進捗表とmove_list.txtは一致）、実装側のみが誤っていた。

### ばんのうがさ・パワフルハーブとの相互作用

- 両アイテムとも技固有の実装ではなく、`battle.weather_for()` および汎用ハンドラ `パワフルハーブ_skip_charge` で処理されるため、
  ソーラーブレード側で個別対応は不要（ソーラービームと共通の仕組みをそのまま利用できている）。

### 揮発状態の解除タイミング

- ON_HIT で解除するため、まもるで防がれた場合（ON_HITが発火しない）は揮発状態が残り、次ターンも強制コマンドで再攻撃を試みる
  （ソーラービームと同じ挙動。既存の関連実装を踏襲）。

### テスト一覧（実施済み・.internal/tests/参照）

1. 通常2ターン攻撃: 1ターン目はダメージなし+揮発付与、2ターン目でダメージあり+揮発解除
2. にほんばれ下: 1ターンで即攻撃（揮発状態を経由しない）
3. あめ/すなあらし/ゆき下: 威力半減（ダメージ量比較）
4. きれあじ: slashフラグにより威力1.5倍
5. パワフルハーブ: 溜めをスキップして即攻撃、消費される
6. パワフルハーブ + にほんばれ同時成立: 天候が優先されアイテムは消費されない
