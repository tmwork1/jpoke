# 実装計画書: ねむりごな

## 仕様要約

- タイプ: くさ / カテゴリ: 変化 / PP: 16（champions基準） / 命中: 75 / 対象: 1 体選択
- 相手を「ねむり」状態にする
- 粉・胞子技（`powder` フラグ）: くさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』には無効
  （`move_executor` / `ability.py` / `item.py` が汎用的に処理するため技側実装は不要）

`どくのこな_apply_ailment_to_defender`）が `ON_STATUS_HIT` にデフォルト優先度で実装されているパターンを踏襲する。

## 注意点・エッジケース

- PPはchampions基準の`src/jpoke/data/ps-champ-ja/moves.json`の値（16）を正とする。Gen9本家値（15）に
  戻さないこと
- `powder`フラグにより、くさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』への無効化は
  技側実装不要（フラグを付与するだけで汎用処理が効く）
- 対象は相手（`target`のデフォルト値`"foe"`のまま）。自分対象ではないため`target="self"`の
  指定は不要
- リーサル計算は対象外（`.internal/progress/move.md` の該当行で `n/a`。状態異常付与技のためダメージ
  リーサルには寄与しない）
