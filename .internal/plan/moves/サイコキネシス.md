# サイコキネシス 実装計画書

## 仕様要約（.internal/spec/moves/サイコキネシス.md 参照）

* タイプ: エスパー / 分類: 特殊 / 威力: 90 / 命中: 100 / PP: 12（champions基準。move_list.txt準拠）
* 対象: 相手1体 / 優先度: 0 / 接触なし
* 効果: 追加効果として、10%の確率で相手の『とくぼう』ランクを1段階下げる。

## 現状の実装状況

`src/jpoke/data/moves/move_sa.py` に登録済みだが、以下2点に誤りがある。

```python
"サイコキネシス": MoveData(
    type="エスパー",
    category="special",
    pp=10,                      # ← 誤り。champions基準では12（move_list.txt準拠）
    power=90,
    accuracy=100,
    # flags に "secondary_effect" が欠落している    # ← 誤り
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.サイコキネシス_lower_defender_spd,
        )
    }
),
```

### 誤り1: PP

pp=10 はGen9本家基準の値であり、championsへの移行漏れ。
src/jpoke/data/ps-champ-ja/moves.json（296行目）および.internal/progress/move.md（510行目）はいずれも
PP=12であり、pp=12 に修正する。

### 誤り2: secondary_effect フラグの欠落

だいちのちから等、同系統（追加効果でとくぼうを1段階下げる特殊技）は
`flags={"secondary_effect"}` を持つが、サイコキネシスには付与されていない。
このフラグが無いと以下が正しく機能しない：

* `handlers/ability.py` の `ちからずく_boost`（威力1.3倍化）
* `handlers/ability.py` の `ちからずく_disable_secondary_effect`（追加効果無効化）
* `handlers/item.py` 内の各アイテム判定（りんぷん等、`has_flag("secondary_effect")` を参照する箇所）

`flags={"secondary_effect"}` を追加する。

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 10%の確率で相手の『とくぼう』ランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・ロジックは修正不要）:

```python
def サイコキネシス_lower_defender_spd(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"spd": -1}, chance=0.1)
```

`modify_defender_stats`（`handlers/move.py`）が `battle.resolve_secondary_chance` を経由するため、
ちからずくによる追加効果無効化（確率0化）・威力1.3倍化は `flags={"secondary_effect"}` さえ
付与されていれば汎用的に処理される。ハンドラ本体への個別ガード追加は不要（`data/moves/move_sa.py`
側の flags 修正のみで解決する）。

## Priority 根拠

ダメージを与えた技の追加効果（能力ランク低下）は既存の同種ハンドラ（だいちのちから・あわ・
いわくだき 等）と同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）のパターンに従う。
`.internal/spec/turn.md` に `ON_DAMAGE_HIT` の掲載はないため、既存の同種ハンドラに合わせてデフォルト
priority を採用する（だいちのちから計画書と同じ根拠）。

## 注意点・エッジケース（review-test 向け）

1. **PP修正**
   - 既存テストでPP=10を前提にした箇所がないか確認する（tests/moves_attack/test_move_sa.py 等）。
2. **secondary_effect フラグ追加によるちからずくとの相互作用**
   - フラグ追加後、ちからずく所持時は威力1.3倍（90→117）になり、とくぼう低下の追加効果は
     発動しなくなる。この相互作用を確認するテストがあれば併せて追加する（既存のちからずく
     テストパターン参照）。
3. **テスト方針**
   - `secondary_chance` を0/1に固定し、とくぼうランクの変化有無を確認するテストを追加する
     （相手にダメージを与える／とくぼう低下が発動する／とくぼう低下が発動しない、の3パターン）。
