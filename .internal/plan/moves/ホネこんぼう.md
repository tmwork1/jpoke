# 計画書: ホネこんぼう

## 結論: 実装しない

`.internal/spec/moves/ホネこんぼう.md` の通り、本技はソード・シールド以降（BDSP・SVを含む）で使用不可であり、
README.md の対象範囲定義（第9世代で実装されていない技は実装しない）に反するため実装計画は策定しない。

## 経緯

過去の一括実装計画 `.internal/plan/moves/攻撃技_ひるみ系.md` で「スペック要確認」のまま実装され、
以下の誤りを含んだ状態で登録されていた:

- `src/jpoke/data/moves/move_ha.py` の `MoveData(power=100, pp=10, flags={"contact", "secondary_effect"}, ...)`
  - 正しい参考値（第1〜7世代時点）は威力65・PP20であり、また直接攻撃技ではないため `contact` フラグも誤り
- `src/jpoke/handlers/move_attack.py` の `ホネこんぼう_apply_flinch`
- `src/jpoke/handlers/item.py` の `_INNATE_FLINCH_MOVES`（おうじゃのしるし・するどいキバの重複判定用リスト）への登録
- `src/jpoke/types/move.py` の `MoveName` Literal への登録
- `tests/moves_attack/test_move_ha.py` と `tests/moves_status/test_move_ha.py` の双方に
  同名テストが重複登録（後者は物理技であるにもかかわらず status 側のテストファイルに混入）

本レビュー（`.internal/review/moves/ホネこんぼう.md`）にて対象範囲外と判断し、上記実装・テストを削除した。
今後、本技を再実装する必要が生じた場合は、まず README.md の対象範囲定義とポケモンチャンピオンズでの
使用可否を再確認すること。
