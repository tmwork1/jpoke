# 実装計画書: あまいかおり

## 仕様

- タイプ: ノーマル / カテゴリ: 変化 / PP: 20 / 命中率: 100
- 効果: 相手の**回避率**を2段階下げる
- 無効条件・反射:
  - まもる → 対象が`foe`かつ`unprotectable`フラグなしのため汎用処理で自動ブロック
  - マジックコート/オウムがえし → 汎用処理で対応済み（個別実装不要）
  - 特性等によるランク変化無効化 → `battle.modify_stats` が内部で保証

## イベント・Priority

| イベント | Priority | subject_spec | 役割 |
|----------|----------|-------------|------|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（MoveHandler デフォルト） | 相手の回避率を2段階下げる |

**根拠**: `ON_STATUS_HIT` は `turn.md` に独立エントリなし。`あまえる`・`あやしいひかり` と同パターン (priority=100 デフォルト)。

## 実装

### handlers/move_status.py

配置位置: `あさのひざし_heal_self` の直後、`あまえる_modify_defender_stats` の直前（五十音順）

```python
def あまいかおり_modify_defender_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"evasion": -2})
```

`modify_defender_stats` は既存の共通ヘルパー（`あまえる`等で使用済み）を再利用する。

### data/moves/move_a.py

```python
"あまいかおり": MoveData(
    type="ノーマル",
    category="status",
    pp=20,
    accuracy=100,
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.あまいかおり_modify_defender_stats,
        )
    }
),
```

## 注意事項

- 対象は`MoveData.target`のデフォルト値`"foe"`のままでよい（シングル専用のためスプレッド技の区別は不要）。
- かがくへんかガス/かたやぶり無効化チェックは書かない。
- リーサル計算は対象外（n/a）: ダメージを与えない変化技のため。
