# オーラウイング 実装計画書

## 仕様要約

`.internal/spec/moves/オーラウイング.md` を参照。
エスパータイプ・特殊・威力80・命中100・PP12（Championsルールにより本家PP10から変換）・急所ランク+1の
非接触攻撃技。追加効果として100%の確率で自分の「すばやさ」ランクを1段階上げる。

## ハンドラ構成

`Event.ON_HIT` に `MoveHandler` デフォルト設定（`subject_spec="attacker:self"`, `priority=100`）で登録する。
`modify_attacker_stats` は `handlers/move.py` の既存共通関数を使用する。

| 技 | event | priority | subject_spec | 役割 |
|---|---|---|---|---|
| オーラウイング | ON_HIT | 100 | attacker:self | S+1（100%） |

## Priority 根拠

既実装の自分ランク上昇技（`はがねのつばさ_boost_attacker_B` 等）と同様に `MoveHandler` デフォルト
（priority=100）で登録する。`.internal/spec/turn.md` に ON_HIT イベント内の個別優先度の明示的な記載はないが、
既存の同種ハンドラ（自分ランク上昇の追加効果）はすべてデフォルト priority で統一されているため、
それに揃える。

## 実装箇所

### `src/jpoke/handlers/move_attack.py`

```python
def オーラウイング_boost_attacker_spe(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_attacker_stats(battle, ctx, value, stats={"spe": 1})
```

`chance` を省略した場合のデフォルトは1（100%）のため明示不要。

### `src/jpoke/data/moves/move_a.py`

## リーサル計算

- 対象外（`n/a`）。自分自身のすばやさランク上昇はダメージ量計算そのものには直接影響しないため、
  専用のリーサルハンドラは不要（`.internal/progress/move.md` の当該行も `n/a` 指定）。

## エッジケース

- 発動確率は100%のため、`modify_attacker_stats` のデフォルト `chance=1` をそのまま使用する。
- ランク上限（+6）は `battle.modify_stats` 側で処理されるため、ハンドラ側に境界チェックは不要。
- ON_HIT はダメージ発生後に発火するため、みがわりに命中した場合でも自分自身への効果は発動する。
- 直接攻撃ではない（非接触技）ため、ゴツゴツメット等の接触判定系の効果は対象外。
