## 仕様要約

- ゴーストタイプ・物理・威力 120・命中率 100・PP 5（`src/jpoke/data/ps-champ-ja/moves.json` に該当行なし。後述のとおり Gen9/Wiki 準拠の値をそのまま採用）・優先度 0・直接攻撃
- ギラティナの専用技（配信のアルセウス（えいがかん）も習得）
- 2ターン技
  - 1ターン目: 姿を消す（`シャドーダイブ` 揮発状態が付与される）。攻撃は行わない。PPは消費しない。
  - 2ターン目: 相手を攻撃する。攻撃終了後に揮発状態が解除される。
- 姿を消している間の回避・交代禁止・コマンド固定などの仕様は `.internal/spec/volatiles/シャドーダイブ.md` に準拠。
- まもる・みきり・たたみがえし・トーチカ・キングシールド・ニードルガード・ブロッキング・スレッドトラップ・かえんのまもりの効果を無視して攻撃する（`unprotectable` フラグ）。ダイウォールは除く。

## PPについて（champions/move_list.txt に該当行なし）

`src/jpoke/data/ps-champ-ja/moves.json` を全文検索したが `シャドーダイブ` の行は存在しない
（シャドークロー・シャドーパンチ・シャドーボールはTM経由で習得可能なため掲載されているが、
シャドーダイブ・シャドーボーン・シャドーレイのようなTMの存在しない専用技は掲載対象外になっている模様）。
`.internal/champions/changes_from_sv.md` の「技のPPが変更」ルール（8/12/16/20の4段階に統一）は
`move_list.txtを参照`とあるが、参照先に本技の記載がないため champions 側の意図した値を確認できない。

README.md の対象範囲定義「技・アイテム・特性・ポケモンなどの仕様ソースは第9世代を基本とし、
チャンピオンズ側でルールが異なる場合はそちらを優先する」に従い、champions 側の情報が存在しない
本技については Gen9本家/Wiki準拠の値（PP5）をそのまま採用する（値を推測で8等に変更しない）。
既存実装 `pp=5` は変更不要。

## テストのレビュー結果

### 既存テスト

- `tests/volatiles/test_volatile_satana.py::test_シャドーダイブ_まもる中の相手にダメージを与えられる`
  （まもる貫通・`unprotectable` フラグの確認）は実装済み。

  `"シャドーダイブ"` が含まれており、回避・コマンド固定・交代禁止などの揮発状態共通仕様は
  そちらでカバーされている。

### 追加したテスト（`tests/moves_attack/test_move_sa.py`）

ゴーストダイブのレビューで追加された `test_ゴーストダイブ_2ターンで攻撃する` に相当する
「技自体が2ターンで攻撃する」ことを確認する move レベルのテストが、シャドーダイブには
存在しなかったため追加する。`test_シャカシャカほう_使用後に攻撃者のHPが回復する` と
`test_シャドーボーン_ぼうぎょ1段階低下が発動する` の間に追加する（五十音順の暫定位置）。

```python
def test_シャドーダイブ_2ターンで攻撃する():
    """1ターン目はダメージを与えず、2ターン目にダメージを与えて揮発状態が解除される"""
    battle = t.start_battle(
        # ノーマルタイプはゴースト無効なのでどくタイプを使用
        team0=[Pokemon("ゲンガー", move_names=["シャドーダイブ"])],
        team1=[Pokemon("ドガース")],
        accuracy=100,
    )
    defender = battle.actives[1]
    hp_before = defender.hp

    # 1ターン目: 揮発状態付与のみ、ダメージなし
    t.run_move(battle, 0)
    assert defender.hp == hp_before
    assert battle.actives[0].has_volatile("シャドーダイブ")

    # 2ターン目: ダメージあり、揮発状態解除
    t.run_move(battle, 0)
    assert defender.hp < hp_before
    assert not battle.actives[0].has_volatile("シャドーダイブ")
```
