# 計画書: つぼをつく

## 仕様要約

- ランダムで「こうげき」「ぼうぎょ」「とくこう」「とくぼう」「すばやさ」「回避率」「命中率」のいずれか1つのランクを2段階上げる
- タイプ: ノーマル、変化、PP: 20（champions 基準）、対象: 自分（シングルでは自分のみ）
- 命中率なし（自分対象のため常に命中）
- 「ダイウォール」の効果も受けない（ダイウォール未実装のため現時点では追加対応不要）
- すでに最大（+6）の能力はランダム選択の候補から除外する。全能力が最大の場合は失敗する

出典: `.internal/spec/moves/つぼをつく.md`

## 実装箇所

### `src/jpoke/handlers/move_status.py`

挿入位置: `つぶらなひとみ_modify_defender_stats` の後（つ〜て の間、五十音順）

```python
def つぼをつく_modify_attacker_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """つぼをつく: ランクが最大でない能力からランダムに1つ選んでランクを2段階上げる。
    候補が存在しない（全能力が最大）場合は失敗する。
    """
    stats: list[Stat] = ["atk", "def", "spa", "spd", "spe", "accuracy", "evasion"]
    candidates = [s for s in stats if ctx.attacker.rank[s] < 6]
    if not candidates:
        return HandlerReturn(value=False)
    stat = cast(Stat, battle.random.choice(candidates))
    return HandlerReturn(value=battle.modify_stats(ctx.attacker, {stat: 2}, source=ctx.attacker))
```

### `src/jpoke/data/moves/move_ta.py`


## 注意点・エッジケース

- 選択肢は `["atk", "def", "spa", "spd", "spe", "accuracy", "evasion"]` の7種類。ランクがすでに最大（+6）の能力は候補から除外し、全能力が最大の場合のみ失敗する（一次情報準拠）
- `target="self"` の技のため、まもる・ダイウォールの影響を受けない（仕様上の「ダイウォール無視」フラグは現時点では未実装のため追加対応不要）
- シングルバトルのみ対象のため「自分か味方」の対象選択ロジックは不要。`target="self"` で実装する
- よこどりの対象外（第五世代以降はよこどりで奪われない。自分を対象とするため）
- さいはいの対象技。さいはいからの呼び出しでもランダム再選択が行われる
