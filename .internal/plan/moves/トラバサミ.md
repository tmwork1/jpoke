# 実装計画: トラバサミ

## 仕様サマリ

- タイプ: はがね（第八世代くさ→Championsではがねに変更） / 分類: 物理 / 威力: 35 / 命中率: 100 / PP: 16 / 接触: あり
- 効果: 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける
- バインド状態の詳細仕様は `.internal/spec/volatiles/バインド.md` を参照
- まもる: ○ / マジックコート: ×（変化技ではない） / 追加効果: なし
- マッギョ（ガラルのすがた）の専用技
- タイプがはがねに変更されたため、そうしょくによる無効化は発生しない（第八世代くさタイプ時代の特有挙動）

## 対象イベント

`ON_DAMAGE_HIT` — ダメージ命中時に発火。既存のバインド系技（うずしお・しめつける・すなじごく・
まきつく・ほのおのうず・からではさむ・まとわりつく・サンダープリズン・マグマストーム）と共通の
`apply_bind_to_defender` ハンドラを流用する。

## Priority

`.internal/spec/turn.md` に `ON_DAMAGE_HIT` の記載なし。
既存のバインド系技がすべて `MoveHandler(apply_bind_to_defender)` のみ（priority=100デフォルト）
で実装されているため、同様にデフォルト priority を採用する。

## 実装状況（実装済み・変更なし）

### handlers/move_attack.py

`apply_bind_to_defender` を流用（新規実装不要）。

### data/moves/move_ta.py

既存実装は仕様と一致している（レビューにより変更なし）:

```python
"トラバサミ": MoveData(
    type="はがね",
    category="physical",
    pp=16,
    power=35,
    accuracy=100,
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(ha.apply_bind_to_defender)
    },
    lethal_handlers={
        LethalEvent.ON_HIT: LethalHandler(l._apply_bind)
    }
),
```

## リーサル計算

`_apply_bind`（`handlers/lethal.py`）を流用（新規実装不要、既に `lethal_handlers` に登録済み）。
バインドの毎ターンダメージはバインド揮発性状態の `ON_TURN_END` リーサルハンドラ側で処理される。

## テスト方針

- 命中時に相手がバインド揮発状態になる確認
- バインド中は相手が毎ターン最大HPの1/8ダメージを受ける確認
- バインド中は相手が交代できない確認
- 攻撃者が交代するとバインドが解除される確認
- （`うずしお`・`すなじごく`・`まとわりつく`・`サンダープリズン` の既存テストと同パターン）
- `tests/test_lethal.py` に、バインド付与（技経由）とバインド事前付与の結果が一致することを確認する
  リーサルテストを追加する（`サンダープリズン`・`すなじごく` と同パターン）
