# 実装計画: まとわりつく

## 仕様サマリ

- タイプ: むし / 分類: 特殊 / 威力: 20 / 命中率: 100 / PP: 20 / 接触: あり
- 効果: 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける
- バインド状態の詳細仕様は `.internal/spec/volatiles/バインド.md` を参照
- まもる: ○ / マジックコート: ×（変化技ではない） / 追加効果: なし
- むしタイプの技のためノーマル技のようなゴーストタイプ無効化は発生しない

## 対象イベント

`ON_DAMAGE_HIT` — ダメージ命中時に発火。既存のバインド系技（うずしお・しめつける・まきつく・
ほのおのうず・からではさむ・すなじごく・トラバサミ・サンダープリズン・マグマストーム）と共通の
`apply_bind_to_defender` ハンドラを流用する。

## Priority

`.internal/spec/turn.md` に `ON_DAMAGE_HIT` の記載なし。
既存のバインド系技がすべて `MoveHandler(apply_bind_to_defender)` のみ（priority=100デフォルト）
で実装されているため、同様にデフォルト priority を採用する。

## 実装状況（実装済み）

### handlers/move_attack.py

`apply_bind_to_defender` を流用（新規実装不要）。

### data/moves/move_ma.py

```python
"まとわりつく": MoveData(
    type="むし",
    category="special",
    pp=20,
    power=20,
    accuracy=100,
    flags={"contact"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(ha.apply_bind_to_defender)
    },
    lethal_handlers={
        LethalEvent.ON_HIT: LethalHandler(l._apply_bind)
    }
),
```

## リーサル計算

`_apply_bind`（`handlers/lethal.py`）を流用（新規実装不要）。バインドの毎ターンダメージは
バインド揮発性状態の `ON_TURN_END` リーサルハンドラ側で処理される。

## テスト方針

- 命中時に相手にダメージを与える確認
- 命中時に相手がバインド揮発状態になる確認
- バインド中は相手が毎ターン最大HPの1/8ダメージを受ける確認
- バインド中は相手が交代できない確認
- （`まきつく`・`うずしお` の既存テストと同パターン）
- リーサル計算: 技命中によるバインド付与とバインド事前付与で結果が一致する確認（`まきつく` の既存リーサルテストと同パターン）

## 注意事項

- むしタイプの技のため、ノーマルタイプ技（まきつく・しめつける）とは異なりゴーストタイプに対する
  無効化は発生しない（むし対ゴーストは効果いまひとつの通常の相性判定に従う）。
- かがくへんかガス/かたやぶり無効化チェックは書かない
