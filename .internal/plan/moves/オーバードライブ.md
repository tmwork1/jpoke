# オーバードライブ 実装計画書

## 仕様要約（.internal/spec/moves/オーバードライブ.md 参照）

* タイプ: でんき / 分類: 特殊 / 威力: 80 / 命中: 100 / PP: 12
* 対象: 相手1体（シングル対象範囲） / 優先度: 0 / 非接触
* 効果: 通常攻撃。追加効果はない。
* 音（sound）技に分類される。

## 現状の実装状況

`src/jpoke/data/moves/move_a.py` に既に登録済み。レビューにより以下の誤りを修正した：

   ないため、同じ本家基準PP=10を持つハイパーボイス・むしのさざめき・フレアソング（いずれも
   champions側でPP=12に変換されている）から類推し、`12` に修正した。

修正後:

```python
"オーバードライブ": MoveData(
    type="でんき",
    category="special",
    pp=12,
    power=80,
    accuracy=100,
    flags={"sound"},
    handlers={},  # 追加効果なし
),
```

（ぼうおん・パンクロック・うるさがた等）は各特性側の汎用処理で対応される。
