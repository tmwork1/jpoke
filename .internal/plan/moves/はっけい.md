# はっけい 実装計画書

## 仕様要約

`.internal/spec/moves/はっけい.md` を参照。
かくとうタイプ・物理・威力60・命中100・PP10の直接攻撃技。
追加効果として30%の確率で相手を「まひ」状態にする。

## ハンドラ構成

`Event.ON_DAMAGE_HIT` に `MoveHandler` デフォルト設定（`subject_spec="attacker:self"`）で登録する。
`apply_ailment_to_defender` は `handlers/move.py` の既存共通関数を使用する。

| 技 | event | subject_spec | 役割 |
|---|---|---|---|
| はっけい | ON_DAMAGE_HIT | attacker:self | まひ 30% |

## Priority 根拠

`.internal/spec/turn.md` には `Event.ON_DAMAGE_HIT` 自体の記載はないが、
既実装の `かみなりパンチ`・`フリーズボルト`・`のしかかり` 等、直接攻撃技のまひ追加効果と同一パターンの登録
（`h.MoveHandler` のデフォルト priority）であり、これらと揃えるため妥当と判断する。

## 実装箇所

### `src/jpoke/handlers/move_attack.py`

```python
def はっけい_apply_paralysis_to_defender(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return apply_ailment_to_defender(battle, ctx, value, ailment="まひ", chance=0.3)
```

### `src/jpoke/data/moves/move_ha.py`

```python
"はっけい": MoveData(
    type="かくとう",
    category="physical",
    pp=10,
    power=60,
    accuracy=100,
    flags={"contact", "secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.はっけい_apply_paralysis_to_defender,
        )
    }
),
```

## リーサル計算

- 対象外（`n/a`）。まひは行動阻害・素早さ低下効果であり、ダメージ量には影響しないため専用のリーサルハンドラは不要。

## エッジケース

- でんきタイプの相手はまひ無効（`ailment_manager` 側の共通処理で判定されるため、本技のハンドラに追加ガードは不要）。
