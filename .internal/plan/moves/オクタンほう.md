# 計画書: オクタンほう

## 結論: 実装しない

`.internal/spec/moves/オクタンほう.md` の通り、本技はソード・シールド以降（BDSP・SVを含む）で使用不可であり、
README.md の対象範囲定義（第9世代で実装されていない技は実装しない）に反するため実装計画は策定しない。

## 経緯

過去の一括実装計画 `.internal/plan/moves/攻撃技_防御側ランク変化系.md`（防御側の能力ランクを下げる追加効果を
まとめて実装するバッチ計画）で、対象範囲の確認が行われないまま以下の内容で登録されていた:

- `src/jpoke/data/moves/move_a.py` の `MoveData(power=110, pp=10, accuracy=85, ...)`
  - 一次情報（`.internal/wiki/moves/オクタンほう.html`）による正しい基礎威力は65であり、
    110という値の出典・根拠は不明
- `src/jpoke/handlers/move_attack.py` の `オクタンほう_lower_acc`
  （`modify_defender_stats(..., stats={"accuracy": -1}, chance=0.5)`）
- `src/jpoke/types/move.py` の `MoveName` Literal への登録
- `tests/moves_attack/test_move_a.py` の `test_オクタンほう_命中率1段階低下が発動する`

本レビュー（`.internal/review/moves/オクタンほう.md`）にて対象範囲外と判断し、上記実装・テストを削除した。
今後、本技を再実装する必要が生じた場合は、まず README.md の対象範囲定義とポケモンチャンピオンズでの
使用可否を再確認すること。
