# ラスターパージ 実装計画書

## 仕様要約（.internal/spec/moves/ラスターパージ.md 参照）

* タイプ: エスパー / 分類: 特殊 / 威力: 95 / 命中: 100 / PP: 8
* 対象: 相手1体 / 優先度: 0 / 接触なし
* 効果: 追加効果として、50%の確率で相手の『とくぼう』ランクを1段階下げる。

## 現状の実装状況

`src/jpoke/data/moves/move_ra.py` に登録済みだが、以下2点が仕様と乖離している。

1. `pp=5`（本家基準の値のまま）になっている。champions ではPPが8/12/16/20の4段階に
   統一されており（`.internal/champions/changes_from_sv.md`）、本家PP5の技はPP8に変換される
   （りゅうせいぐん・じばく・だいばくはつ等が move_list.txt 上で確認できる）。
   ラスターパージ自体は move_list.txt に記載がないが、同じ本家PP5の技の変換規則に
   合わせて `pp=8` に修正する。
2. `flags` に `secondary_effect` が付与されていない。50%の確率で発動する追加効果
   （とくぼう1段階低下）を持つため、ちからずく（`ability.py` の `ちからずく_boost` /
   `ちからずく_disable_secondary_effect`）と正しく連動させるには `flags={"secondary_effect"}`
   が必須。同種の技（10まんボルト・だいちのちから 等）はすべてこのフラグを持っている。

修正後:

```python
"ラスターパージ": MoveData(
    type="エスパー",
    category="special",
    pp=8,
    power=95,
    accuracy=100,
    flags={"secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.ラスターパージ_lower_defender_spd,
        )
    }
),
```

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 50%の確率で相手の『とくぼう』ランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・ロジックは仕様と一致・修正不要）:

```python
def ラスターパージ_lower_defender_spd(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"spd": -1}, chance=0.5)
```

`modify_defender_stats`（`handlers/move.py`）が `battle.resolve_secondary_chance` を経由するため、
ちからずくによる追加効果無効化（確率0化）・威力1.3倍化は `flags={"secondary_effect"}` さえ
付与されていれば汎用的に処理される。本技側に個別のガードは不要。

## Priority 根拠

ダメージを与えた技の追加効果（能力ランク低下）は既存の同種ハンドラ（だいちのちから・
10まんボルト・ラスターカノン 等）と同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）の
パターンに従う。個別の priority 検討は不要。

## 注意点・エッジケース（review-test 向け）

1. **ちからずくとの相互作用**
   - `flags={"secondary_effect"}` を追加したことで、ちからずく所持時は威力1.3倍
     （95→123、端数切り捨て）になり、とくぼう低下の追加効果は発動しなくなる。
2. **テスト方針**
   - `secondary_chance` を0/1に固定し、とくぼうランクの変化有無を確認するテストを追加する
     （相手にダメージを与える／とくぼう低下が発動する／とくぼう低下が発動しない、の3パターン）。
