# 計画書: ふわふわフォール

## 結論: 実装しない

`.internal/spec/moves/ふわふわフォール.md` の通り、本技は第七世代（Let's Go! ピカチュウ・イーブイ）の
相棒わざ専用技であり、ソード・シールド以降（BDSP・SVを含む）で一度も実装されていない。
README.md の対象範囲定義（第9世代で実装されていない技は実装しない）に反するうえ、
`src/jpoke/data/ps-champ-ja/moves.json`（ポケモンチャンピオンズの技一覧）にも項目が存在しないため、
実装計画は策定しない。

## 経緯

過去の一括実装計画 `.internal/plan/moves/攻撃技_ひるみ系.md` で「スペック要確認」のまま実装され、
以下の誤りを含んだ状態で登録されていた:

- `src/jpoke/data/moves/move_ha.py` の `MoveData(type="フェアリー", power=70, pp=15, accuracy=95, ...)`
  - 第七世代時点の一次情報ではタイプ:ひこう、威力90、命中95、PP15であり、タイプ・威力とも一致しない
- `src/jpoke/handlers/move_attack.py` の `ふわふわフォール_apply_flinch`
- `src/jpoke/handlers/item.py` の `_INNATE_FLINCH_MOVES`（おうじゃのしるし・するどいキバの重複判定用集合）への登録
- `tests/moves_attack/test_move_ha.py` の `test_ふわふわフォール_ひるみが発動する`

本レビュー（`.internal/review/moves/ふわふわフォール.md`）にて対象範囲外と判断し、上記実装・テストを削除した。
今後、本技を再実装する必要が生じた場合は、まず README.md の対象範囲定義とポケモンチャンピオンズでの
使用可否（`src/jpoke/data/ps-champ-ja/moves.json`）を再確認すること。
