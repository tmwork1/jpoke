# うらみつらみ 実装計画書

## 仕様要約（.internal/spec/moves/うらみつらみ.md 参照）

* タイプ: ゴースト / 分類: 特殊 / 威力: 75 / 命中: 100 / PP: 10
* 対象: 相手1体 / 優先度: 0 / 非接触
* 効果: 100%の確率で相手の『こうげき』ランクを1段階下げる（追加効果）。
* Pokémon LEGENDS アルセウス固有の「状態異常時威力2倍」「しもやけ付与」効果は、
  本プロジェクト（第九世代・ポケモンチャンピオンズシングルバトル対象）の対象外。

## 現状の実装状況

`src/jpoke/data/move.py` に既に登録済み。

```python
"うらみつらみ": MoveData(
    type="ゴースト",
    category="special",
    pp=10,
    power=75,
    accuracy=100,
    flags={"secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.うらみつらみ_lower_defender_atk,
        )
    }
),
```

内容は仕様書と一致しており、修正不要。

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 100%の確率で相手の『こうげき』ランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・修正不要）:

```python
def うらみつらみ_lower_defender_atk(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"atk": -1})
```

`modify_defender_stats`（`handlers/move.py`）は `chance` 省略時に100%として扱い、
`battle.resolve_secondary_chance` を経由するため、ちからずくによる追加効果無効化（確率0化）は
汎用的に処理される。本技側に個別のガードは不要。

## Priority 根拠

ダメージを与えた技の追加効果（ランク変化）は既存の同種ハンドラ（エナジーボール・エレキネット 等）と
同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）のパターンに従う。`.internal/spec/turn.md` にも
`ON_DAMAGE_HIT` は他の追加効果技と共通のタイミングとして扱われており、個別の priority 検討は不要。

## 注意点・エッジケース（review-test 向け）

1. **ちからずくとの相互作用**
   - `flags={"secondary_effect"}` があるため、ちからずく所持時は `resolve_secondary_chance` が確率を0にし、
     こうげきランク低下が発動しなくなる（威力1.3倍化は `ちからずく_boost` 側で処理）。
2. **Pokémon LEGENDS アルセウス固有効果は対象外**
   - 一次情報には状態異常時威力2倍・しもやけ付与効果があるが、第九世代（本プロジェクト対象）には
     存在しない効果のため実装しない。
3. **テスト方針**
   - `battle.actives[1].rank["atk"] == -1` を確認する既存テスト
     （`test_うらみつらみ_こうげき1段階低下が発動する`）で十分。100%発動のため乱数固定は不要。
