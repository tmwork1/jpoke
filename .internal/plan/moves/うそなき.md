# 実装計画: うそなき

## 仕様サマリ

- タイプ: あく / 分類: 変化 / PP: 20 / 命中率: 100
- 効果: 相手の **とくぼう** ランクを 2段階下げる
- まもる: ○ / マジックコート: ○ / 直接攻撃: ×
- 対象: 相手1体選択

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（デフォルト） | 相手のとくぼうを2段階下げる |

## Priority 根拠

`.internal/spec/turn.md` に `ON_STATUS_HIT` の記載なし。
既存の同種ハンドラ（`あまえる_modify_defender_stats`、`きんぞくおん_modify_defender_stats` など）が
すべて `MoveHandler(func)` のみ（priority=100 デフォルト）で実装されているため、同様に priority=100 を採用する。

## 実装箇所

### handlers/move_status.py への追加

五十音順での位置: `うずしお` 系の直後、`うたう_*` の直前。

```python
def うそなき_modify_defender_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """うそなきの効果: 相手のとくぼうを2段階下げる。"""
    return modify_defender_stats(battle, ctx, value, stats={"spd": -2})
```

### data/moves/move_a.py の更新

```python
"うそなき": MoveData(
    type="あく",
    category="status",
    pp=20,
    accuracy=100,
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.うそなき_modify_defender_stats,
        )
    }
),
```

## 注意点・エッジケース

- とくぼうが最速で-6段階のときは失敗する（ランク変化共通のガードで処理済み、ハンドラ側の重複ガード不要）
- 音系技ではないため、みがわり状態には無効（技フレームワーク側の対象判定で処理済み）
- リーサル計算: 対象外（`n/a`）。とくぼうダウンは相手の次ターン以降の被ダメージに影響するのみで、
  本ターンの命中側の生存判定・与ダメージ判定には関与しないため
