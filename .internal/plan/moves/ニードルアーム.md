# 計画書: ニードルアーム

## 結論: 実装しない

`.internal/spec/moves/ニードルアーム.md` の通り、本技はソード・シールド以降（BDSP・SVを含む）で使用不可であり、
README.md の対象範囲定義（第9世代で実装されていない技は実装しない）に反するため実装計画は策定しない。
`src/jpoke/data/ps-champ-ja/moves.json` にも本技の項目は存在しない。

## 経緯

過去の一括実装計画 `.internal/plan/moves/攻撃技_ひるみ系.md` で「スペック要確認」のまま実装され、
以下の誤りを含んだ状態で登録されていた:

- `src/jpoke/data/moves/move_na.py` の `MoveData(pp=15, power=60, ...)`
  - champions のPPは8/12/16/20の4段階に統一されており、15という値自体が誤り
    （そもそも本技はchampions未収録のため登録が誤り）
- `src/jpoke/handlers/move_attack.py` の `ニードルアーム_apply_flinch`
- `src/jpoke/types/move.py` の `MoveName` Literal への登録
- `src/jpoke/handlers/item.py` の `_INNATE_FLINCH_MOVES`（おうじゃのしるし・するどいキバの
  重複判定用リスト）への登録
- `tests/moves_attack/test_move_na.py` と `tests/moves_status/test_move_na.py` の双方に
  同名テストが重複登録（後者は物理技であるにもかかわらず status 側のテストファイルに混入）

本レビュー（`.internal/review/moves/ニードルアーム.md`）にて対象範囲外と判断し、上記実装・テストを削除した。
今後、本技を再実装する必要が生じた場合は、まず README.md の対象範囲定義とポケモンチャンピオンズでの
使用可否を再確認すること。
