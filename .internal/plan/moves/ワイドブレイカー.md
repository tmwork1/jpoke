# ワイドブレイカー 実装計画書

## 仕様要約（.internal/spec/moves/ワイドブレイカー.md 参照）

* タイプ: ドラゴン / 分類: 物理 / 威力: 60 / 命中: 100 / PP: 16（`src/jpoke/data/ps-champ-ja/moves.json` 準拠）
* 対象: 相手全員（シングルバトルのみ対象のため実質相手1体） / 優先度: 0 / 接触あり
* 効果: 追加効果として、100%の確率で相手の『こうげき』ランクを1段階下げる。

## 現状の実装状況（修正前）

`src/jpoke/data/moves/move_wa.py` に登録済みだが、以下2点に誤りがある。

1. `pp=15` になっているが、`src/jpoke/data/ps-champ-ja/moves.json` では16であり、
   チャンピオンズの全技PP統一ルール（8/12/16/20の4段階）に反する移行漏れのバグ。
2. `flags` に `"secondary_effect"` が欠落しているため、ちからずく特性による
   威力1.3倍化・追加効果無効化が適用されない誤りがある。同じ「100%相手ランク低下」
   パターンの `うらみつらみ` `ほのおのムチ` `らいめいげり` `りんごさん` は
   `secondary_effect` を保持しており、`ワイドブレイカー` も対象技として扱うべきである。

## 修正後

```python
"ワイドブレイカー": MoveData(
    type="ドラゴン",
    category="physical",
    pp=16,
    power=60,
    accuracy=100,
    flags={"contact", "secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.ワイドブレイカー_lower_defender_atk,
        )
    }
),
```

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 100%の確率で相手の『こうげき』ランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・修正不要）:

```python
def ワイドブレイカー_lower_defender_atk(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"atk": -1})
```

`modify_defender_stats`（`handlers/move.py`）の `chance` 既定値は1のため、
`battle.resolve_secondary_chance(ctx, 1)` を経由してちからずくのポケモンが使用した場合は
確率が0化され効果が発動しなくなる。本技側に個別のガードは不要。

## Priority 根拠

ダメージを与えた技の追加効果（能力ランク低下）は既存の同種ハンドラ（うらみつらみ・
ほのおのムチ・らいめいげり・りんごさん 等）と同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）の
パターンに従う。`ON_DAMAGE_HIT` は `.internal/spec/turn.md` に未掲載のイベント
（ダメージ計算内部）だが、既存の同種ハンドラの実装パターンを踏襲するため
個別の priority 検討は不要。

## 注意点・エッジケース（review-test 向け）

1. **ちからずくとの相互作用**
   - `flags={"secondary_effect"}` を追加したことで、ちからずく所持時は威力1.3倍（60→78）になり、
     こうげき低下の追加効果は発動しなくなる。
2. **接触フラグ**
   - `flags={"contact"}` により、接触時発動効果・特性（『さめはだ』『てつのとげ』等）の対象となる。
     個別のガードは不要（各特性・アイテム側で判定される）。
3. **シングルバトル限定の範囲仕様**
   - 一次情報では「相手全員」を対象とする範囲技だが、本プロジェクトはシングルバトルのみを
     対象とするため、通常の単体攻撃技と同様に扱ってよい（`target` フィールドの既定値
     `"foe"` のままで問題ない）。
4. **テスト方針**
   - 100%固定効果のため `secondary_chance` の調整は不要。こうげきランクの変化を確認する
     テスト（`test_ワイドブレイカー_こうげき1段階低下が発動する`）を追加する。
   - ちからずく所持時に追加効果が発動しないことを確認するテストも追加する。
