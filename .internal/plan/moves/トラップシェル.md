# 実装計画書: トラップシェル

## 結論: 実装しない（対象範囲外）

`.internal/spec/moves/トラップシェル.md` の通り、本技はブリリアントダイヤモンド・シャイニングパール／
スカーレット・バイオレットで使用不可（一次情報の `.internal/wiki/moves/トラップシェル.html` インフォボックスに
明記）であり、`src/jpoke/data/ps-champ-ja/moves.json` にも項目が存在しない。README.md の対象範囲定義
「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に該当するため、実装計画を策定しない。

## 過去の誤登録について

過去のレビュー（本レビュー以前）で `src/jpoke/data/moves/move_ta.py` に以下の内容で
`MoveData` が対象範囲確認なしに登録されていた:

```python
"トラップシェル": MoveData(
    type="はがね",
    category="special",
    pp=5,
    power=150,
    accuracy=100,
    priority=-5,
    flags={"non_negoto"},
    handlers={},  # 追加効果なし
),
```

一次情報（タイプ「ほのお」・威力150・命中100・PP5・優先度-3）と比較すると、タイプが誤り
（「はがね」ではなく「ほのお」が正しい。五十音順で隣接する「トラバサミ」がチャンピオンズで
くさ→はがねタイプに変更されていることとの混同が原因と推測される）、優先度も誤り
（-5ではなく-3が正しい。`.internal/spec/action_order.md` の -5 ブロックにも誤って掲載されていたため
併せて修正した）。また、本技の核心的な仕様（設置→物理攻撃を受けて起爆→行動順繰り上げで即座に反撃、
自身の優先度-3の行動順までに起爆できなければ不発）は一切実装されておらず、単なる無条件の
特殊150威力技として登録されていた。

本レビューでこの誤登録を削除した。詳細は `.internal/review/moves/トラップシェル.md` を参照。
