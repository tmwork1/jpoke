# 計画書: かえんだん

## 仕様要約

- ほのおタイプ / 特殊 / 威力100 / 命中100 / PP5 / 対象: 相手1体（本プロジェクトはシングル専用） / 優先度0 / 非接触 / 弾のわざ
- 追加効果として、30%の確率で相手をやけど状態にする。

## ハンドラ構成

既存の `かえんだん_apply_burn_to_defender`（`handlers/move_attack.py`）をそのまま利用する。
`apply_ailment_to_defender` ヘルパーを使い、`ailment="やけど"`, `chance=0.3` を指定する。
かえんぐるま・かえんほうしゃ・かえんボールと同型の実装であり、新規ハンドラ実装は不要。

```python
def かえんだん_apply_burn_to_defender(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return apply_ailment_to_defender(battle, ctx, value, ailment="やけど", chance=0.3)
```

## Priority 根拠

`ON_DAMAGE_HIT` は追加効果付与の標準イベントであり、他のやけど付与技
（かえんぐるま・かえんほうしゃ・かえんボール等）と同じ priority（デフォルト）を使用する。
本技独自の priority 検討は不要。

## 実装箇所

### data/move.py への更新

```python
    "かえんだん": MoveData(
        type="ほのお",
        category="special",
        pp=5,
        power=100,
        accuracy=100,
        flags={"bullet", "secondary_effect"},
        handlers={
            Event.ON_DAMAGE_HIT: h.MoveHandler(
                ha.かえんだん_apply_burn_to_defender,
            )
        }
    ),
```

- 修正前は `pp=15, power=75` かつ `flags={"secondary_effect"}`（bullet フラグ欠落）という誤りがあったため、
  一次情報（Gen5基本値: 威力100・PP5）に合わせて修正し、弾のわざとして `bullet` フラグを追加する。
- `target` 省略時のデフォルトは `"foe"`（相手1体）。本プロジェクトはシングル専用のため
  「自分以外の全員」という範囲仕様は実質「相手1体」と等価であり、追加のtarget指定は不要。
- `priority` 省略時のデフォルトは `0` → 仕様と一致。
- `critical_rank` 省略時のデフォルトは `0`（通常の急所判定）→ 仕様と一致。

### handlers/move_attack.py

変更不要。既存の `かえんだん_apply_burn_to_defender` をそのまま使用する。

## 注意点・エッジケース

- 弾のわざ（`bullet` フラグ）を持つため、特性ぼうだんを持つ相手には無効化される。
  この無効化ロジックは `handlers/ability.py` の `ぼうだん_block_bullet` が汎用的に処理するため、
  本技側での追加実装は不要。
- 追加効果（やけど付与）の確率変動（ちからずく・てんのめぐみ・りんぷん・おんみつマント等）は
  `apply_ailment_to_defender` 内の `battle.resolve_secondary_chance` が汎用的に処理するため、
  本技側での追加実装は不要。
