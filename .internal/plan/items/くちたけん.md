# くちたけん 実装計画

更新日: 2026-06-12

## スコープ

- 仕様書: `.internal/spec/items/くちたけん.md`
- 実装状態: 未実装
- 実装方針: `ON_SWITCH_IN` / `ON_ITEM_GAINED` でザシアン(れきせん)をザシアン(けんのおう)にフォルムチェンジする

## 実装ゴール（完了定義）

- ザシアン(れきせん)がくちたけんを持って登場すると、ザシアン(けんのおう)にフォルムチェンジする
- ザシアン以外が持っても追加効果はない（フォルムチェンジしない）
- `removable=False` および `fling_power=0` はすでに ItemData に設定済み

## 実装対象イベント

- `Event.ON_SWITCH_IN` — 登場時にフォルムチェンジを発動する
- `Event.ON_ITEM_GAINED` — バトル中にアイテムを入手したとき（消費後に再取得したケース等）

## 変更対象（ファイル/関数単位）

- `src/jpoke/data/item.py` — `くちたけん` に `handlers` を登録する
- `src/jpoke/handlers/item.py` — `くちたけん_form_change` を実装する
- `tests/test_item.py` — テストを追加する

## 実装ステップ

1. `src/jpoke/handlers/item.py` に `くちたけん_form_change` を実装する。
2. `src/jpoke/data/item.py` の `くちたけん` に `ON_SWITCH_IN`・`ON_ITEM_GAINED` handler を登録する。
3. テストを追加し、全テストが通ることを確認する。

## テスト観点

1. ザシアン(れきせん)がくちたけんを持って登場するとザシアン(けんのおう)になる。
2. ザシアン(けんのおう)がすでに場にいるときは再度フォルムチェンジしない。
3. ザシアン以外のポケモンが持っても変化しない。
4. フォルムチェンジ後の種族値が正しく反映される（こうげき高い等）。

## 疑似コード

```python
def くちたけん_form_change(battle, ctx, value):
    mon = ctx.source
    if mon.name != "ザシアン(れきせん)":
        return HandlerReturn(value=value)
    mon.set_form("ザシアン(けんのおう)")
    return HandlerReturn(value=value)
```

登録:
```python
"くちたけん": ItemData(
    consumable=False,
    removable=False,
    fling_power=0,
    handlers={
        Event.ON_SWITCH_IN: h.ItemHandler(
            h.くちたけん_form_change,
            subject_spec="source:self",
            priority=100,
        ),
        Event.ON_ITEM_GAINED: h.ItemHandler(
            h.くちたけん_form_change,
            subject_spec="source:self",
            priority=100,
        ),
    }
),
```
