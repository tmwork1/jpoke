# でんきだま 実装計画

更新日: 2026-06-12

## スコープ

- 仕様書: `.internal/spec/items/でんきだま.md`
- 実装状態: 未実装
- 実装方針: `ON_CALC_ATK_MODIFIER` でピカチュウのこうげき・とくこうを 2 倍（8192/4096）にする

## 実装ゴール（完了定義）

- ピカチュウが物理技を使うとこうげきが 2 倍になる
- ピカチュウが特殊技を使うととくこうが 2 倍になる
- ピチュー・ライチュウが持っても効果がない
- キョダイマックスするピカチュウ（`ピカチュウ(キョダイ)`）も有効
- こんらんの自傷ダメージ（`_こんらん`）には影響しない（Champions仕様＝第五世代以降準拠）

## 実装対象イベント

- `Event.ON_CALC_ATK_MODIFIER` — 攻撃側の実数値＋ランク補正後の倍率を計算する

## テスト観点

2. ピカチュウが特殊技を使ったとき特攻実数値が 2 倍相当になる。
3. ピチュー・ライチュウが持っても効果がない。
4. ピカチュウ(キョダイ)が持っても効果がある。

```python
def でんきだま_boost_atk(battle, ctx, value):
    # ピカチュウ・ピカチュウ(キョダイ)のみ対象（ピチュー・ライチュウは除外）
    # こんらんの自傷ダメージ（"_こんらん"）は対象外（Champions仕様＝第五世代以降準拠）
    if (
        ctx.attacker.name in {"ピカチュウ", "ピカチュウ(キョダイ)"}
        and ctx.move.name != "_こんらん"
    ):
        return HandlerReturn(value=apply_fixed_modifier(value, 8192))
    return HandlerReturn(value=value)
```
