# ハバンのみ 実装計画

更新日: 2026-07-05

## スコープ

- 仕様書: `.internal/spec/items/_タイプ半減きのみ.md`（ハバンのみ節）
- 実装状態: 実装済み

## 効果概要

- 効果バツグンの`ドラゴン`タイプの技を受けたとき、ダメージを1/2にして消費する（1度きり）。
- ウィキ上に特記される特殊な相互作用（他のタイプ半減きのみのような専用の追加効果技等）はない。

## 実装方針

- 通常戦闘: `Event.ON_CALC_DAMAGE_MODIFIER`（`subject_spec="defender:self"`）で発火。
  タイプ半減きのみ共通処理 `handlers/item.py` の `_modify_super_effective_damage` を
  `type_="ドラゴン"`, `modifier=2048/4096` で呼び出す。
  - priority: 他のタイプ半減きのみ・ダメージ補正系アイテムと同じ既定値（`.internal/spec/turn.md` に
    `ON_CALC_DAMAGE_MODIFIER` 個別の優先度記載なし。ダメージ計算内部イベントのため、
    既存の同種きのみハンドラの priority に合わせてデフォルト値を使用）。
  - `ctx.move.type == "ドラゴン"` かつ `battle.damage_calculator.calc_def_type_modifier(ctx) > 4096`
    で効果バツグン判定を行い、条件成立時に `_announce_and_consume_item` でログ記録と
    アイテム消費を行う。
- リーサル計算（簡易KO判定用の `LethalEvent`）: `LethalEvent.ON_BEFORE_HIT`（`subject="defender"`）で
  `handlers/lethal.py` の `ハバンのみ_resist_dragon` → 共通処理 `_type_resist_berry(..., "ドラゴン")` を呼び出す。
  通常ハンドラ側で既にダメージ半減済みのため、リーサル側ではアイテム消費と `item_enabled` の更新のみ行う。

## 登録

- `src/jpoke/data/item.py` の `"ハバンのみ"` に `handlers`（`ON_CALC_DAMAGE_MODIFIER`）と
  `lethal_handlers`（`LethalEvent.ON_BEFORE_HIT`）を登録済み。
- `fling_power=10`。

## テスト

- `tests/test_item.py` の `test_タイプ半減実`（パラメータ化テスト）で
  `("ハバンのみ", "ドラゴン", "ミニリュウ")` を検証。
  - `damage_modifier == 2048`（1/2倍）であること。
  - 使用後にアイテムが消費されていること（`not has_item()`）。
- リーサル側は `tests/test_lethal.py` の `test_タイプ半減きのみ_代表種`（オッカのみ・ヤチェのみ・
  ロゼルのみを代表種として検証、共通処理 `_type_resist_berry` を使うため個別テストは不要）でカバーされる。
