# こだわりメガネ 実装計画

## 効果概要
特殊技のとくこうを1.5倍にする。最初に選んだ技しか使用できなくなる（こだわりロック）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `.internal/spec/items/_こだわり系.md` |
| 実装状態 | **実装済み** |
| イベント① | `Event.ON_CALC_ATK_MODIFIER` — 特殊技のATK補正を1.5倍 |
| subject_spec① | `"attacker:self"` |
| イベント② | `Event.ON_MOVE_END` — 技使用後にこだわりロックを付与 |
| subject_spec② | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック
```python
def こだわりメガネ_boost_special(battle, ctx, value):
    if ctx.move.category == "特殊":
        value = apply_fixed_modifier(value, 6144)  # ×1.5
    return HandlerReturn(value=value)

def こだわり_lock_move(battle, ctx, value):  # こだわりスカーフと共通
    mon = ctx.attacker
    if not mon.has_volatile("こだわり"):
        battle.volatile_manager.apply(mon, "こだわり", source=mon, move_name=ctx.move.name)
    return HandlerReturn(value=value)
```

## テスト観点
- こだわりメガネ所持時、特殊技のダメージが1.5倍になること
- 物理技には補正がないこと
- 技を使用した後、同じ技しか選択できなくなること
