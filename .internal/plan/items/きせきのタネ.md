# きせきのタネ 実装計画

## 効果概要
持たせると、くさタイプの技の威力を1.2倍にする（4915/4096倍、固定小数点補正）。
タイプ強化アイテムの一種（`.internal/spec/items/_タイプ強化アイテム.md`参照）。
なげつける威力は30。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `.internal/spec/items/きせきのタネ.md` |
| 実装状態 | **実装済み** |
| イベント | `Event.ON_CALC_POWER_MODIFIER`（`AttackContext`、`.internal/spec/turn.md`未掲載イベント。`core/damage.py`のダメージ計算内部で発火し、`じしゃく`等の既存タイプ強化アイテムハンドラと同じ扱い） |
| subject_spec | `"attacker:self"` |
| priority | 100（既定値。既存の同種アイテムハンドラ（じしゃく・かたいいし等）に合わせる） |
| once | False |
| fling_power | 30 |
| 消費条件 | なし（非消費アイテム） |

### 発動条件の詳細
- `ctx.move.type == "くさ"`の場合のみ`apply_fixed_modifier(value, 4915)`で威力を1.2倍にする。
- タイプが変わる技の効果でタイプが変わった場合、変更後のタイプがくさタイプであれば効果がある
  （`ctx.move.type`は判定時点のタイプを参照するため、既存の共通実装`_modify_power_by_type`で自然に対応済み）。

### ハンドラロジック（疑似コード）
```python
def きせきのタネ_modify_power_by_type(battle, ctx, value):
    return _modify_power_by_type(ctx.move, value, type_="くさ", modifier=4915)
```
（既存の共通ヘルパー`_modify_power_by_type`をそのまま利用する。じしゃく・かたいいし等の
タイプ強化アイテムと同一の実装パターン）

## テスト観点
- くさタイプの技の威力が1.2倍になること
- くさタイプ以外の技では威力が変化しないこと
- なげつけるの威力が30であること
