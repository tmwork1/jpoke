# つめたいいわ 実装計画

## 効果概要
ゆき の継続ターンを +3 する（5 ターン → 8 ターン）。さらさらいわと同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def つめたいいわ_resolve_field_count(battle, ctx, value):
    return _resolve_field_count(value, "ゆき", additonal_count=3)
```

`ON_MODIFY_DURATION` は `field_manager.py` の天候発動処理（技・特性ゆきふらし問わず）で共通して
発火されるため、技によるゆきと特性によるゆきの両方で自動的に延長される。

## 実装状態
- 実装済み: `src/jpoke/data/item.py` の `つめたいいわ` に `ON_MODIFY_DURATION` ハンドラを登録済み。
- `src/jpoke/handlers/item.py` の `つめたいいわ_resolve_field_count` で実装済み。

## テスト観点
- つめたいいわ所持時、ゆきが 8 ターン継続すること（技発動時）
- つめたいいわ所持時、特性ゆきふらしで発動したゆきも 8 ターン継続すること
- つめたいいわがない場合は 5 ターンで終了すること
- ゆき以外の天候は延長されないこと
