# ラムのみ 実装計画

## 効果概要
以下のいずれかの状態になった瞬間に自動発動し、状態異常およびこんらんをまとめて回復して消費する。

- 状態異常（どく・もうどく・まひ・やけど・こおり・ねむり）のいずれか
- こんらん（揮発性状態）

発動時点で状態異常とこんらんが重複していた場合、両方を同時に回復する（消費は1回のみ）。
参照: `.internal/spec/items/_状態回復きのみ.md`

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント1 | `Event.ON_APPLY_AILMENT`（状態異常付与直後の即時回復。重複するこんらんも同時に回復） |
| subject_spec1 | `"target:self"` |
| priority1 | 50（モモンのみ等、他の状態異常回復きのみのON_APPLY_AILMENTに合わせる） |
| イベント2 | `Event.ON_VOLATILE_START`（こんらん付与直後の即時回復。重複する状態異常も同時に回復。`value == "こんらん"` の場合のみ処理） |
| subject_spec2 | `"source:self"` |
| priority2 | 未指定（キーのみのON_VOLATILE_STARTに合わせる） |
| イベント3 | `Event.ON_ITEM_ENABLED`（アイテム無効化解除時、既に状態異常/こんらんがあれば回復） |
| subject_spec3 | `"source:self"` |
| priority3 | 50（他の状態異常回復きのみに合わせる） |
| イベント4 | `Event.ON_FORCE_BERRY_TRIGGER`（ほおばる等での強制発動） |
| subject_spec4 | `"source:self"` |
| priority4 | 50（同上） |
| イベント5 | `Event.ON_TURN_END`（保険としてのターン終了時チェック） |
| subject_spec5 | `"source:self"` |
| priority5 | 50（他の状態異常回復きのみ（クラボのみ等）に合わせる） |
| 消費条件 | 発動時に `_announce_and_consume_item` で消費 |

いずれのイベントも `.internal/spec/turn.md` に未掲載のため、既存の同種きのみ（カゴのみ・クラボのみ・チーゴのみ・ナナシのみ・モモンのみ・キーのみ）の実装に準拠してpriorityを決定した。

### ハンドラロジック（疑似コード）

共通ヘルパー:
```python
def _cure_ailment_and_confusion(battle, mon) -> bool:
    """状態異常・こんらんの両方をチェックし、該当するものをまとめて回復する。"""
    cured_ailment = mon.ailment.is_active and battle.ailment_manager.remove(mon)
    cured_confusion = mon.has_volatile("こんらん") and battle.volatile_manager.remove(mon, "こんらん")
    return bool(cured_ailment or cured_confusion)
```

イベント別ハンドラ:
```python
def ラムのみ_cure_ailment_and_confusion(battle, ctx, value):
    """ON_TURN_END / ON_ITEM_ENABLED / ON_FORCE_BERRY_TRIGGER 用"""
    mon = ctx.source
    if _cure_ailment_and_confusion(battle, mon):
        _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)

def ラムのみ_cure_ailment_and_confusion_on_apply_ailment(battle, ctx, value):
    """ON_APPLY_AILMENT 用"""
    mon = ctx.target
    if _cure_ailment_and_confusion(battle, mon):
        _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)

def ラムのみ_cure_ailment_and_confusion_on_confuse(battle, ctx, value):
    """ON_VOLATILE_START 用（こんらん以外は無視）"""
    mon = ctx.source
    if value != "こんらん":
        return HandlerReturn(value=value)
    if _cure_ailment_and_confusion(battle, mon):
        _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- 状態異常付与直後に即座に回復・消費されること（ON_APPLY_AILMENT）
- こんらん付与直後に即座に回復・消費されること（ON_VOLATILE_START）
- 状態異常のみターン終了時に残っていた場合に回復・消費されること
- 状態異常とこんらんが同時に発生している状態で発動したとき、両方が同時に回復し消費が1回のみであること
- 状態異常・こんらんいずれもない場合は発動しないこと（アイテムを保持したまま）
- 消費後は次に状態異常/こんらんになっても回復しないこと（アイテムを失っているため）
