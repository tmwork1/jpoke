# ものまねハーブ 実装計画

## 効果概要
相手のランク補正が上昇したとき、実際に上昇した分だけ自分も同じランクを上げる（1度きり、消費アイテム）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_STAT` |
| subject_spec | `"target:foe"`（ハンドラの主体＝所持者、`ctx.target` が主体の相手側のときに発動） |
| priority | 100（デフォルト） |
| 消費条件 | コピーにより実際にランクが変化した場合のみ `battle.item_manager.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ものまねハーブ_copy_stat_boost(battle, ctx, value):
    # 相手のびんじょう・ものまねハーブによるコピーで上がった分は対象外
    if ctx.stat_change_reason in ("びんじょう", "ものまねハーブ"):
        return HandlerReturn(value=value)

    mon = battle.foe(ctx.target)
    # 実際に上昇した分（value は ON_MODIFY_STAT が渡す actual_changes）のみが対象
    boosts = {s: v for s, v in value.items() if v > 0}
    if not boosts:
        return HandlerReturn(value=value)

    # 自分のランクが既に最大などでコピーしても変化しない場合は消費しない
    changed = battle.modify_stats(mon, boosts, source=ctx.target, reason="ものまねハーブ")
    if changed:
        _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)
```

## 実装済みの要点

- `value` は `StatusManager.modify_stats` が計算した `actual_changes`（技・特性の本来の効果量ではなく、実際に反映された変化量）であるため、
  「技や特性などの効果自体ではなく、実際に上昇した分がコピーの対象となる」という仕様を追加実装なしで満たす。
- `subject_spec="target:foe"` により、味方（自分自身）のランク上昇では発動しない。
- `value` を `v > 0` でフィルタするため、相手のランク低下では発動しない。
- コピー先の `mon` が既に最大ランクの場合、`battle.modify_stats` が返す `changed` が空になるため、
  発動メッセージも出さず、アイテムも消費しない（「所持者のすでに最大になっている能力を相手が上げたときは発動しない」に対応）。
- `once=True` は使用しない。アイテム自体が `consume_item` で失われることで「1度発動すると無くなる」を実現する
  （消費後は `item.enabled_ignoring` が False になりハンドラが無効化される）。
- 所持者の特性がびんじょうである場合、`ON_MODIFY_STAT` の同一イベントに対して
  びんじょうハンドラ（`source:foe` 相当）とものまねハーブハンドラが両方登録されており、
  `switch_manager` の登録順（特性→アイテムの順で `register_handlers` を呼ぶ）により
  びんじょう→ものまねハーブの順で発動し、同じ能力が2回上昇する。
  びんじょうのコピーで既に最大まで上がった場合は、ものまねハーブのコピーが actual_changes 0 となり発動しない。
- 相手がびんじょう・ものまねハーブでランクを上げたときは自分のびんじょう・ものまねハーブは発動しない。
  これは `StatusManager.modify_stats` の `reason` 引数を経由して `ctx.stat_change_reason` に
  `"びんじょう"` / `"ものまねハーブ"` をマークし、両ハンドラの先頭でこれらの reason を弾くことで実現する
  （`types/literals.py` の `StatChangeReason` に追加）。

## テスト観点
- 相手のランクが上昇したとき同じ能力・同じ量が自分にも上昇すること（複数段階の上昇も含む）
- 自分自身のランク上昇ではコピーしないこと
- 相手のランク低下ではコピーしないこと
- 発動後にアイテムが消費されること
- 自分のランクが既に最大のときは発動せず、アイテムも消費されないこと
- 所持者の特性がびんじょうである場合、同じ能力が2回上昇すること
- びんじょうのコピーで既に最大まで上がった場合はものまねハーブが発動しないこと
- 相手のびんじょう・ものまねハーブによるコピーで上がった分は再度コピーしないこと
