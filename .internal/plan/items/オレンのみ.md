# オレンのみ 実装計画

## 効果概要
HP が最大値の 50% 以下になった瞬間に HP を +10 回復する（消費）。
オボンのみの回復量違いバージョン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HP_CHANGED` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def オレンのみ_heal_on_half_hp(battle, ctx, value):
    mon = ctx.target
    hp_after = mon.hp
    hp_before = hp_after + value
    if hp_before * 2 > mon.max_hp and hp_after * 2 <= mon.max_hp:
        battle.modify_hp(mon, v=10)
        battle.consume_item(mon)
    return HandlerReturn(value=value)
```

## テスト観点
- HP が 50% 超 → 50% 以下になった瞬間に +10 HP 回復すること
- オボンのみと同じ閾値トリガーであること
- 発動後にアイテムが消費されること
- 消費後に再度ダメージを受けても発動しないこと
