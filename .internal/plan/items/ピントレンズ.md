# ピントレンズ 実装計画

## 効果概要
急所ランクを +1 する。攻撃技の急所ランク計算時に常時発動する。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CALC_CRITICAL_RANK` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def ピントレンズ_boost_critical_rank(battle, ctx, value):
    return HandlerReturn(value=value + 1)
```

## テスト観点
- ピントレンズ所持時、急所ランクが +1 されること
- するどいツメと重複して +2 になること
- ピントレンズがない場合は急所ランクが変化しないこと
