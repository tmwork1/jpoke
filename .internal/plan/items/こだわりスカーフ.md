# こだわりスカーフ 実装計画

## 効果概要
素早さを1.5倍にする。最初に選んだ技しか使用できなくなる（こだわりロック）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `.internal/spec/items/_こだわり系.md` |
| 実装状態 | **実装済み** |
| イベント① | `DomainEvent.ON_CALC_SPEED` — 素早さ計算時に1.5倍 |
| subject_spec① | `"source:self"` |
| イベント② | `Event.ON_MOVE_END` — 技使用後にこだわりロックを付与 |
| subject_spec② | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック
```python
def こだわりスカーフ_boost_speed(battle, ctx, value):
    value = apply_fixed_modifier(value, 6144)  # ×1.5 (6144/4096)
    return HandlerReturn(value=value)

def こだわり_lock_move(battle, ctx, value):
    mon = ctx.attacker
    if not mon.has_volatile("こだわり"):
        battle.volatile_manager.apply(mon, "こだわり", source=mon, move_name=ctx.move.name)
    return HandlerReturn(value=value)
```

## テスト観点
- こだわりスカーフ所持時、素早さ実数値が1.5倍になること
- 技を使用した後、同じ技しか選択できなくなること
- 交代するとこだわりロックが解除されること
