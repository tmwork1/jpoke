# ばんのうがさ 実装計画

## 効果概要
持たせたポケモンはにほんばれ・あめ状態の影響を受けなくなる。
すなあらし・ゆきには無効。天候を自分で発動させることは可能。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `.internal/spec/items/ばんのうがさ.md` |
| 実装状態 | **実装済み** |
| 消費条件 | なし（非消費アイテム） |

### 実装方針（採用済み）
`battle.weather_for(mon)` を中心的な参照 API とし、天候を参照する側（技威力補正・命中率・
特性・回復技等）はすべて `battle.weather` の代わりに `battle.weather_for(mon)` を使うことで
ばんのうがさの効果を自動的に反映する。`ON_CHECK_WEATHER_IMMUNE` ハンドラ（`ばんのうがさ_weather_immune`）が
はれ/あめ/おおひでり/おおあめのときのみ True を返し、`weather_for` はその場合「なし」天候を返す。
すなあらし・ゆき・らんきりゅうは対象外のため、`weather_for` を経由してもこれらの天候はそのまま返る。

### 影響する効果と対応イベント（実装済み）

| 効果カテゴリ | 詳細 | 対応イベント・実装箇所 |
|---|---|---|
| 天候免除の判定 | ばんのうがさ所持者に対しては `weather_for` が「なし」を返す | `Event.ON_CHECK_WEATHER_IMMUNE`（`item.py: ばんのうがさ_weather_immune`） |
| 技威力補正（防御側） | 晴れ/雨によるほのお・みず技ダメージ補正を無視 | `Event.ON_CALC_POWER_MODIFIER`（`field.py: はれ_power_modifier` / `あめ_power_modifier`） |
| 命中率（晴れ時かみなり/ぼうふう） | 攻撃側が持つと晴れの命中率低下（50%）を無視 | `move_attack.py: かみなり_accuracy` / `ぼうふう_accuracy` |
| 命中率（雨時かみなり/ぼうふう/あらし系） | 防御側が持つと雨の必中効果を無視 | `move_attack.py: かみなり_accuracy` / `ぼうふう_accuracy` / `かみなりあらし_accuracy` / `こがらしあらし_accuracy` / `ねっさのあらし_accuracy` |
| ソーラービーム・ソーラーブレード | 攻撃側が持つと晴れでも溜めが必要、雨でも威力半減を回避 | `move_attack.py: ソーラービーム系_weather_skip` / `ソーラービーム_halve_power` |
| エレクトロビーム | 攻撃側が持つと雨でも溜めが必要 | `move_attack.py: エレクトロビーム_weather_skip` |
| ウェザーボール | 攻撃側が持つと晴れ/雨でもノーマルタイプ・威力据え置き | `move_attack.py: ウェザーボール_modify_move_type` / `ウェザーボール_power_modifier` |
| ハイドロスチーム | 攻撃側が持つと晴れの1.5倍化が発動しない、防御側が持つと1.5倍化する | `move_attack.py: ハイドロスチーム_power_modifier` |
| せいちょう | 攻撃側が持つと晴れでも1段階のみ上昇 | `move_status.py: せいちょう_boost_attacker_atk_spa` |
| こうごうせい・あさのひざし・つきのひかり | 使用者が持つと晴れ/雨でも回復量が1/2 | `move_status.py: あさのひざし_heal_self` |
| こおり状態 | 対象が持つと晴れ中でもこおり状態になる | `field.py: はれ_prevent_freeze` |
| 特性抑制（サンパワー等） | サンパワー・リーフガード・ようりょくそ・あめうけざら・うるおいボディ・すいすい・かんそうはだ・ひひいろのこどう・フラワーギフト | `ability.py` 各特性ハンドラで `weather_for` を参照 |
| こだいかっせい | ばんのうがさの影響を受けない（`battle.weather` を直接参照） | `ability_paradox.py: refresh_paradox_charge_state` |

なげつける威力60は `ItemData(fling_power=60, ...)` として実装済み。

## 実装ステップ（完了済み）

1. 仕様書 `.internal/spec/items/ばんのうがさ.md` の各項目をイベント単位に整理する。
2. `battle.weather_for(mon)` を中心 API として `core/battle.py` に実装し、`ON_CHECK_WEATHER_IMMUNE`
   ハンドラを `src/jpoke/handlers/item.py` に実装、`src/jpoke/data/item.py` の `ばんのうがさ` に登録。
3. 天候を参照する各所（技威力補正・命中率・回復技・特性・フォルムチェンジ等）を
   `battle.weather` から `battle.weather_for(mon)` に置き換える。
4. `tests/items/test_item_ha.py` を中心に各効果のテストを追加。

## テスト観点

### 実装済み
- 防御側がばんのうがさを持つ場合、晴れ/雨の技威力補正（ほのお/みず）が無視されること
- 攻撃側がばんのうがさを持っていても、晴れ/雨の技威力補正は通常通り適用されること（防御側の効果のため）
- 攻撃側がばんのうがさを持つ場合、晴れのかみなり命中率低下（50%）が無視されること
- 防御側がばんのうがさを持つ場合、雨のかみなり必中効果が無視されること
- 攻撃側がばんのうがさを持つ場合、晴れでもソーラービームを溜めること
- 使用者がばんのうがさを持つ場合、晴れでもあさのひざしの回復量が1/2になること
- 使用者がばんのうがさを持つ場合、雨のすいすい・晴れのようりょくその素早さ上昇が発動しないこと
- 対象がばんのうがさを持つ場合、晴れ中でもこおり状態になること（`はれ_prevent_freeze` が `weather_for` を無視していたバグを修正）
- せいちょう・ぼうふう・かみなりあらし系・ハイドロスチーム・ウェザーボール等の個別テストは
  `tests/moves_attack/`・`tests/moves_status/` 側にも分散して存在する

### 対象外（実装不要）
- すなあらし・あられ・ゆき・らんきりゅう関連の効果はばんのうがさで防げないため対応不要
- こだいかっせいはばんのうがさの影響を受けないため `weather_for` を使わない実装のままでよい
