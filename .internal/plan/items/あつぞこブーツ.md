# あつぞこブーツ 実装計画

## 効果概要
エントリーハザード（まきびし・どくびし・ステルスロック・ねばねばネット）の効果を受けない（非消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_HAZARD_IMMUNE` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

## 発火側の改修

`is_floating()` と同じパターンで `ON_CHECK_HAZARD_IMMUNE` を追加する。

### 1. `enums/event.py` に追加
```python
ON_CHECK_HAZARD_IMMUNE = auto()
# emit: core/pokemon_query.py (is_hazard_immune)
# handle: data/item.py (あつぞこブーツ)
```

### 2. `core/pokemon_query.py` に追加
```python
def is_hazard_immune(self, pokemon: Pokemon) -> bool:
    return self._events.emit(
        Event.ON_CHECK_HAZARD_IMMUNE,
        EventContext(source=pokemon),
        False
    )
```

### 3. `handlers/field.py` の各ハザードハンドラ先頭に追加
```python
# まきびし_damage, どくびし_poison, ステルスロック_damage, ねばねばネット_speed_drop の4関数
if battle.query.is_hazard_immune(ctx.source):
    return HandlerReturn(value=value)
```

### ハンドラロジック（アイテム側）
```python
def あつぞこブーツ_immune_hazard(battle, ctx, value):
    return HandlerReturn(value=True)
```

## テスト観点
- あつぞこブーツ持ちがまきびし・ステルスロック・どくびし・ねばねばネットの効果を受けないこと
- あつぞこブーツを持っていない場合は通常通りハザードが適用されること
- ハザード以外の登場時効果（天候・地形ダメージ等）には影響しないこと
