# たつじんのおび 実装計画

## 効果概要
効果抜群の技のダメージを 1.2 倍にする（非消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CALC_DAMAGE_MODIFIER` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def たつじんのおび_boost_super_effective(battle, ctx, value):
    if battle.damage_calculator.calc_def_type_modifier(ctx) > 1:
        value = apply_fixed_modifier(value, 4915)  # 1.2倍
    return HandlerReturn(value=value)
```

## テスト観点
- 効果抜群の技でダメージが 1.2 倍になること
- 等倍・いまひとつではダメージが変化しないこと
- 変化技には影響しないこと（ON_CALC_DAMAGE_MODIFIER の発火条件による）
