# どくどくだま 実装計画

## 効果概要
ターン終了時に持ち主にもうどくを付与する。すでに状態異常がある場合は付与しない。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 150（`.internal/spec/turn.md` の `Event.ON_TURN_END` 表 644行目に明記。どく・もうどくダメージ(90)・ねむけ(110)・はねやすめのひこうタイプ復帰(120)・場の状態やフィールドの継続終了(130-140)より後に発動することで、「発動するターンにどくダメージは受けない」「ねむけと重なった場合ねむり優先」「ミストフィールド解除ターンから発動」という仕様を満たす） |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def どくどくだま_apply_poison(battle, ctx, value):
    mon = ctx.source
    if battle.ailment_manager.apply(mon, "もうどく", source=mon):
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- ターン終了時、どくどくだま持ちにもうどくが付与されること
- すでに状態異常がある場合は付与されないこと
- どくタイプ・はがねタイプには付与されないこと（ailment_manager の通常チェックで防ぐ）
- ねむけと重なった場合はねむり状態が優先されること（priority110 < 150）
- ミストフィールドが解除されたターンから発動すること（priority140 < 150）
- 発動するターンにもうどくダメージを受けないこと（priority90 < 150）
- かえんだまとの比較: もうどく（蓄積ダメージ増加）vs やけど（攻撃半減）の違い
