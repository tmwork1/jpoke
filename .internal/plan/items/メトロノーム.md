# メトロノーム 実装計画

## 効果概要
同じ技を連続使用するたびに与えるダメージが+20%ずつ増加（最大+100%、すなわち2倍）。
別の技に切り替えるとカウントリセット。交代でもリセット。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `.internal/spec/items/メトロノーム.md` |
| 実装状態 | **実装済み** |
| イベント① | `Event.ON_CALC_POWER_MODIFIER` — 威力補正を適用 |
| subject_spec① | `"attacker:self"` |
| イベント② | `Event.ON_MOVE_END` — 連続カウントを更新 |
| subject_spec② | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### 状態管理
`item.count`（連続使用数、0リセット〜5最大）と `item.move_name`（前回の技名）を `ItemData` に保持。

### ハンドラロジック
```python
def メトロノーム_boost_power(battle, ctx, value):
    item = ctx.attacker.item
    if item.count > 0 and item.move_name == ctx.move.name:
        value = apply_fixed_modifier(value, 4096 + item.count * 819)  # ×(1.0 + count×0.2)
    return HandlerReturn(value=value)

def メトロノーム_update_count(battle, ctx, value):
    item = ctx.attacker.item
    if item.move_name == ctx.move.name:
        item.count = min(item.count + 1, 5)
    else:
        item.move_name = ctx.move.name
        item.count = 1
    return HandlerReturn(value=value)
```

## テスト観点
- 初回使用は補正なし（×1.0）であること
- 2回目で×1.2、3回目で×1.4、6回目以降で×2.0になること
- 別の技を使うとカウントがリセットされること
- 固定ダメージ技にはダメージ増加が適用されないこと
