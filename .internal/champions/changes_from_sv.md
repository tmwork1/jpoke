# SVからChampionsに移行したときの変更点

## レベルの固定
50で固定

## 個体値の廃止
31で固定

## 努力値システムの修正
SV : 0~252
Champions : 0~32

変換式
if eff_chmp == 0:
    eff_sv = eff_chmp
else:
    eff_sv = 8*eff_chmp - 4
