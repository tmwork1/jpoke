## 説明

### テーブルの見方

- **☐ 列**: チェックボックス。実装完了時に `☐` → `✓` に変更
- **仕様書**: `○` = spec/ability/ に仕様ファイルあり / `-` = なし
- **優先度**: `H`（高優先度8個）、`L`（低優先度1個）、`-`（未定）
- **ハンドラ実装**（実装済み表のみ）: 実装済み / 部分実装
- **テスト**: `○` = 専用テストあり / 関連テストあり / `-` = テスト未実装

### 実装済み表の詳細情報

詳しい実装状況は以下を参照：
- `src/jpoke/data/ability.py` - 特性データ定義
- `src/jpoke/handlers/ability.py` - イベントハンドラ実装
- tests/test_ability.py - テストコード
