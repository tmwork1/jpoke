# こおり (FRZ) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

こおりとは状態異常の一つ。こおりタイプには付与できない。

## 効果

- わざを使用できなくなる（「凍って動けない！」）。PPも消費しない。
- わざを出す直前に **25%** の確率で回復する（Champions固有。SV以前は20%）。
- 解凍した場合はそのターン行動できる。
- 治らなかった場合でも、**行動不能になって2回目の後（3回目の行動時）に必ず治る**（Champions固有）。
- 一部の技（下記「自分を回復する技」参照）を選択した場合は、上記の確率判定を行わず
  必ず解凍したうえで行動できる。

### Champions 説明文

> 一部の例外を除いて技が使えない。 技を使う時に25%の確率で回復する。
> こおり状態になって3度目は回復する。 ほのおタイプ等の技のダメージを受けると回復する。

## タイプ免疫

- こおりタイプは**こおりにならない**。

## こおりの回復

### 自動回復（ターン開始）
- わざを出す直前に25%の確率で回復（Champions）
- 行動不能2回の後（3回目の行動時）に必ず回復（Champions）

### 自分を回復する技（self_thaw）
以下の技を選択した場合、通常の確率判定を行わずに必ず自分のこおりを解凍したうえで
技を実行できる（「凍って動けない！」のメッセージは出ず、「〜のこおりがとけた!」と表示される）。

- かえんぐるま・かえんボール・クロスフレイム・せいなるほのお・フレアドライブ・もえつきる
  （ほのおタイプ技）
- シャカシャカほう・スチームバースト・ねっさのだいち・ねっとう・ハイドロスチーム
  （ほのお以外のタイプだがこおりを解凍する技）
- めらめらバーンも該当するが本プロジェクトでは未実装。

補足:
- もえつきるは自分がほのおタイプでない場合は技自体が失敗し、この場合こおりは解凍されない。
- 使用者の特性がちからずくの効果が発動した技であっても、自分のこおりは解凍できる。
- とくせいこんじょうを持つポケモンが自分のこおりを回復する物理技を使った場合、こんじょうは
  発動しない（第五世代以降）。
- 同時にこんらん等の行動不能を伴う状態変化にかかっている場合、Champions（第五世代以降相当）では
  **状態変化の判定の後にこおりが治る**。例えばこんらんの自傷判定で技が出せなかった場合、
  こおりは解凍されない（詳細は「ターン処理の優先度」参照）。

### ほのお技被弾による回復
- **ほのおタイプの攻撃技（第三世代以降）を受けると解凍する**。タイプ由来の効果であり
  「追加効果」には該当しないため、使用者がちからずくでも発動する。ウェザーボール等、
  命中時点でほのおタイプに変化している技も対象になる。逆にそうでん状態・ノーマルスキン等で
  元がほのおタイプの技が別タイプに変わった場合は解凍しない。
- ほのお以外のタイプで同様にこおりを解凍する技（シャカシャカほう・スチームバースト・
  ねっさのだいち・ねっとう・ハイドロスチーム、第六世代以降）のうち、「追加効果」フラグを
  持つ技（ちからずく対象技）は、使用者がちからずくの場合は発動しない。ハイドロスチームは
  「追加効果」扱いではないため、ちからずくでも常に発動する
  （docs/spec/moves/ハイドロスチーム.md参照）。
  防御側の特性がりんぷんであっても、この解凍効果は防がれない。
- へんかわざであるおにびは相手のこおりを回復しない（ほのおタイプの追加効果扱いの技のため）。

## こおりの予防

- こおりタイプは付与不可
- 特性: きよめのしお、ぜったいねむり、マグマのよろい、りんぷん（追加効果経由のみ防ぐ）、フラワーベール（くさタイプのみ）、リミットシールド
- 場の状態: にほんばれ（はれ状態では付与不可。ただし付与後にはれになってもこおりは治らない）、しんぴのまもり、ミストフィールド（地面にいるポケモンのみ）

## 関連インタラクション

- **はやあし**特性: こおり状態ですばやさ1.5倍
- **こんじょう**特性: こおり状態でこうげき1.5倍
- **ふしぎなうろこ**特性: こおり状態でぼうぎょ1.5倍
- **たたりめ・ひゃっきやこう**: こおり状態の相手に使うと威力2倍

## ターン処理の優先度

`docs/spec/turn.md` の `Event.ON_TRY_ACTION` より:

| Priority | 処理 |
|----------|------|
| 10 | こおりの回復判定（通常の25%確率判定・3回目強制解凍） |
| 15 | ほのおタイプでないもえつきるの失敗（こおりの回復判定より後・へんげんじざい等のタイプ変化より前） |
| 110 | こんらんの自傷の判定 |
| 120 | まひして技が出せない |
| 130 | メロメロの判定 |
| 170 | こおりを回復するわざでこおり回復（self_thaw技による確定解凍。こんらん等の判定より後） |

self_thaw技を選択した場合、priority=10の確率判定はスキップされ、priority=170まで
到達できた場合（＝それまでに行動が阻害されなかった場合）にのみ確定的に解凍される。

`Event.ON_DAMAGE_HIT` より:

| Priority | 処理 |
|----------|------|
| 100 | ほのお技受けてこおり回復 |
| 100 | ねっとう等受けてこおり回復 |

## 詳細仕様（Champions）

- こおり状態になって1回目・2回目の行動時: 25%で回復を試みる
- こおり状態になって3回目の行動時: 必ず回復し行動できる
- 行動不能回数のカウントは `Ailment.elapsed_turns` で管理する
  - 行動不能になるたびに `elapsed_turns` を1増加させる
  - `elapsed_turns >= 2` の場合は強制解凍（3回目の行動時）
  - self_thaw技を選択した場合はこのカウント判定自体を経由しない
