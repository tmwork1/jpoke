# ねむり (SLP) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

ねむりとは、状態異常の一つ。いびき・ねごと以外の技を使えない。
数ターン（カウントに依存）行動不能になり、カウントが0になると回復する。

## 効果

- いびき・ねごと以外の技を使えない（ON_TRY_ACTION で行動不能）
- ねむり状態で「こんらん」になっても、いびき・ねごと以外を選んでいる間は自傷することはなく、
  眠りカウントも消費しない（はやおきの追加消費も含めて消費されない）。
  いびき・ねごとを使用したときはこんらんの自傷判定が行われ、眠りカウントも通常通り消費される
- ねむり中にねごとで出した技の追加効果は「からげんき」威力2倍・「リフレッシュ」ねむり回復は発動しない

## ターン経過（カウント）

### Champions のカウント

| 付与方法 | カウント |
|---------|---------|
| 通常技（ねむりごな・さいみんじゅつ等） | 2 or 3（1/3確率でcount=2、2/3確率でcount=3） |
| ねむけ状態からの移行 | 2 or 3（同上） |
| ねむる（技） | 3 固定 |

### カウント消費の挙動

カウントは技を使う直前（ON_TRY_ACTION）に1ずつ消費される（はやおきは追加で1消費）。
カウントが 0 になったとき、ねむりが解除されて技を出せる。

```
count=2 の場合:
  1ターン目: tick → count=1, 行動不能
  2ターン目: tick → count=0 → remove（ねむり解除） → 行動成功

count=3 の場合:
  1ターン目: tick → count=2, 行動不能
  2ターン目: tick → count=1, 行動不能
  3ターン目: tick → count=0 → remove（ねむり解除） → 行動成功
```

Champions 説明文:
「いびき ねごと以外の技が使えない。ねむり状態になってから 1度目は技が 使えない。
2度目は 1/3の確率で 回復する。3度目は 回復する。技ねむるでは 3度目まで回復しない。」

- 「2度目は1/3の確率で回復」= count=2が1/3確率で選ばれるため（そのときは2度目で回復）
- 「3度目は回復」= count=3が2/3確率で選ばれるため（3度目で回復）
- 「ねむるでは3度目まで回復しない」= count=3固定（必ず3度目に回復）

## 付与条件

ねむり状態は他の状態異常と同様に重ね掛け不可。
タイプ免疫によるねむり無効は存在しない（ノーマルタイプも含め全タイプに付与可能）。

### ねむりを付与できない特性・場の状態

- **ふみん・やるき・ぜったいねむり・きよめのしお**: ねむりを無効化
- **リーフガード**: 天気が晴れのとき、状態異常（ねむり含む）を無効化
- **フラワーベール**: くさタイプのポケモンの状態異常（ねむり含む）を無効化
- **リミットシールド**: メテノがりゅうせいのすがたのとき、状態異常（ねむり含む）を無効化
- **スイートベール**: 味方のねむりを無効化
- **エレキフィールド**: 接地しているポケモンのねむりを無効化
- **ミストフィールド**: 接地しているポケモンへの状態異常を無効化
- **さわぐ状態**: ねむりを無効化（さわぐ中はねむれない）
- **しんぴのまもり**: 技による状態異常を無効化

## 回復

### ターン経過による自然回復

カウントが 0 になると自動的にねむりが解除される。

### その他の回復手段（実装対象外・参考情報）

カゴのみ・ラムのみ・ねむけざまし等のアイテム、
しぜんかいふく・だっぴ等の特性でも回復する。

## 関連インタラクション

- **はやおき特性**: ねむりカウントを毎ターン2消費（通常より早く覚める）。
  こんらんによってねむりカウントが消費されないターンは、はやおきの追加消費も行われない
- **こんじょう特性**: ねむり状態でこうげき1.5倍
- **ふしぎなうろこ特性**: ねむり状態でぼうぎょ1.5倍
- **はやあし特性**: ねむり状態でもすばやさ補正なし、さらに1.5倍
- **ナイトメア特性**: 相手がねむり状態のとき毎ターン最大HPの1/8を削る
- **ゆめくい・あくむ**: 相手がねむり状態のときのみ効果あり
- **たたりめ・ひゃっきやこう**: 相手がねむり状態のとき威力2倍
- **めざましビンタ**: 相手がねむり状態のとき威力2倍かつねむり回復
