---

# 基本式

ダメージ =
floor(レベル × 2 / 5 + 2)
× 威力 × タイプ相性 × 攻撃 / 防御 → floor
÷ 50 + 2 → floor
× 乱数（0.85〜1.00） → floor


---

# 小数点処理

| 処理 | 内容 |
|------|------|
| 切り捨て | 小数点以下を削除 |
| 切り上げ | 小数点以下があれば+1 |
| 四捨五入 | 0.5以上切り上げ |
| 五捨五超入 | 0.5以下切り捨て、超過で切り上げ |
