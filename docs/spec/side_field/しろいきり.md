# しろいきり（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | しろいきり（場の状態） |
| 発生方法 | 技「しろいきり」成功 |
| 効果対象 | 味方側のポケモン全体 |
| 効果内容 | **5ターンの間** 他ポケモン（主に相手）による能力ランクの低下を **防ぐ** |
| 既存ランク低下 | すでに下がっているランクは **回復されない** |
| 重ね掛け | **延長不可**（同状態の再付与でもターン数は増えない） |
| 対象処理 | 能力低下を受けないが、ランク以外の影響（状態異常など）は別処理 |

## 発動条件 / 無効条件

### 発動

- 技「しろいきり」を使用し命中した場合に発生。  
- 効果はフィールド状態として設定。

### 無効／解除

| 条件 | 結果 |
|---|---|
| **ターン経過（5ターン経過）** | しろいきり状態が解除 |
| **技「きりばらい」** | しろいきり状態が解除 |
| **技「コートチェンジ」** | 味方・相手の場の状態と入れ替え可能 |

## 効果詳細（能力ランクの低下防止）

### 防止されるもの

- **他ポケモンの技効果によるランク低下**（例：いかく、こごえるかぜ等）  
  → ランク降下を受けずに無効化される。:contentReference[oaicite:0]{index=0}

### 防止されないもの（例外）

| 影響源 | 結果 |
|---|---|
| 自分が自身でランクを下げる技（例：「ばかぢから」等） | ランク低下は **防がれない** |
| ランクをリセット系の技・効果（例：くろいきり（Gen1）、クリアスモッグ等） | **防がれない／影響する場合あり**（仕様次第） |
| 特性によるランク低下 | しろいきり状態で **防げるものと防げないものが存在**（状況依存） |
| 特性「すりぬけ」など、効果無視系 | しろいきり状態を無視してランク低下を発生させることがある |

## 競合・相互作用

### 特性との相互作用

| 特性 | 作用 |
|---|---|
| すりぬけ | 相手の特性すりぬけのポケモンによる能力低下は防げない
| いかく | 通常のランク低下は防がれる（しろいきり優先） |

### 他の場状態との関係

- 他の場状態と **共存可能**。  
- 「コートチェンジ」でしろいきり状態を他の場状態と入れ替え可能。:contentReference[oaicite:1]{index=1}

## ターン処理順（概要）

1. 技「しろいきり」が命中 → 味方側場に状態設定  
2. カウンタを 5 に設定  
3. ターン経過ごとにカウンタ減少  
4. カウンタ 0 で状態解除  
5. 技「きりばらい」等で途中解除可能

## 補足

- 既に下がった能力ランクを **自動回復する効果はない**。:contentReference[oaicite:2]{index=2}  
- 「しろいきり」は **味方全体の防御用場状態**として扱われるため、自身の行動でもランク低下が発生する場合は弾かれない。:contentReference[oaicite:3]{index=3}
