# おいかぜ（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | おいかぜ（場の状態） |
| 発生方法 | 技「おいかぜ」の成功 |
| 効果対象 | 味方側の全ポケモン |
| 効果内容 | 味方側ポケモンの **すばやさを2倍にする** |
| 効果持続 | **4ターン**（技使用ターンを含む） |
| 重ね掛け | **重ねがけ不可**（同じ状態を重ねても延長しない） |
| 他の場条件との共存 | リフレクター等 他の場の状態と共存可能 |
| コートチェンジ | 味方・相手の場状態と入れ替える |

## 発動条件 / 無効条件

### 発動

- 技「おいかぜ」を使用して成功した場合に発生。 :contentReference[oaicite:0]{index=0}

### 無効／解除

- 特定の技や効果によって解除されない（きりばらい等では解除されない）。 :contentReference[oaicite:1]{index=1}

## 効果詳細（速度補正）

| 状態 | 効果 |
|---|---|
| 素早さ補正 | 計算上のすばやさステータスを **2倍** として扱う（行動順等に反映） |
| 補正計算順序 | 第9世代ではランク補正等の後においかぜによる補正を計算する処理順になる場合あり（詳細は内部仕様） | :contentReference[oaicite:2]{index=2} |

## 競合・相互作用

### 特性との相互作用

| 特性 | 効果 |
|---|---|
| かぜのり | おいかぜ効果時に **こうげきが1段階上昇** する（効果が乗る） |
| ふうりょくでんき | おいかぜ発動時に **じゅうでん状態になる** | :contentReference[oaicite:3]{index=3}

## 例外・注意点

- **ターン効果範囲**  
  効果は発動したターンを1ターン目として数える。 :contentReference[oaicite:4]{index=4}

- **すばやさ変動との影響**  
  おいかぜ中のすばやさ変動は、ジャイロボール/エレキボールの威力算出にも影響する。 :contentReference[oaicite:5]{index=5}

- **トリックルームとの挙動**  
  トリックルームと同時に成立している場合、すばやさ補正の順位処理により逆効果となる場合がある。 :contentReference[oaicite:6]{index=6}

## ターン処理順（概要）

1. 技「おいかぜ」成功 → 味方側場に状態を設定  
2. 有効期間カウンタを `4` に設定  
3. ターン進行ごとにカウンタ減少  
4. カウンタが `0` になると状態解除
