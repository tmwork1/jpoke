# ねばねばネット（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ねばねばネット（場の状態） |
| 発生方法 | 技「ねばねばネット」成功 → 相手側の場に状態付与 |
| 効果対象 | 相手側に新たに場に出たポケモン |
| 効果内容 | 出てきたポケモンの **すばやさランクを1段階低下（–1）** させる |
| 効果持続 | **場に存在する限り無期限（ターン制限なし）** |
| 重ね掛け | **効果は重ねられない**（既に状態があると失敗） |
| 他の場条件との共存 | 他の場の状態（ステルスロック等）と共存可能 |
| 設置対象 | **相手側の場**（味方側には適用されない） |

## 発動条件 / 無効条件

### 発動

- 技「ねばねばネット」を使用し命中した場合、相手側の場がねばねばネット状態になる。  
  技自体が失敗した場合や場がすでに状態になっている場合には設置されない。 :contentReference[oaicite:0]{index=0}

### 無効／解除条件

| 条件・技 | 結果 |
|---|---|
| 相手側の場がすでにねばねばネット状態 | 技は失敗（再設定されない） |
| こうそくスピン・キラースピン | 使用者側のねばねばネットを解除可能 |
| きりばらい | 両側のねばねばネット状態を解除 |
| キョダイフウゲキ | 両側のねばねばネット状態を解除 |
| おかたづけ | 両側のねばねばネット状態を解除 | :contentReference[oaicite:1]{index=1}

## 効果詳細（すばやさ低下）

| 条件 | 仕様 |
|---|---|
| 対象 | **交代で出てきたポケモン** |
| 効果 | 出てきたポケモンの **すばやさランクを –1** |
| 多重交代 | 1体ずつ個別に効果判定 |
| ランク下限 | ランク下限（–6）まで低下可能 |
| 地面にいないポケモン | 地面にいない状態で出てきたポケモンは **影響を受けない** | :contentReference[oaicite:2]{index=2}

## 無効化・保護条件

以下の条件・特性・状態の場合、ねばねばネットの効果は **適用されない**：

| 条件・状態 | 影響 |
|---|---|
| 飛行タイプ | すばやさ低下効果を受けない |
| 特性「ふゆう」 | すばやさ低下効果を受けない |
| 特性「クリアボディ」 | ランク低下無効化 |
| 特性「しろいけむり」 | ランク低下無効化 |
| 特性「メタルプロテクト」 | ランク低下無効化 |
| 特性「ミラーアーマー」 | ランク低下無効化（跳ね返しも） |
| しろいきり状態 | ランク低下無効化 |
| 特性「フラワーベール」 | ランク低下無効化 |
| 持ち物「クリアチャーム」 | ランク低下無効化 |
| 持ち物「あつぞこブーツ」 | ランク低下無効化 | :contentReference[oaicite:3]{index=3}

## 競合・特殊相互作用

| 相互作用 | 仕様 |
|---|---|
| 特性「かちき」「まけんき」 | ねばねばネットでランク低下を受けた場合、発動条件を満たし **特性効果が発動する** |
| 特性「あまのじゃく」 | 逆にすばやさが **1段階上昇** する |
| 特性「たんじゅん」 | 下がったランクが **–2** 扱いになる場合あり | :contentReference[oaicite:4]{index=4}

## ターン処理順（概要）

1. 技「ねばねばネット」が成功 → **相手側の場に状態設定**  
2. その後の処理（行動・交代等）を解決  
3. 交代したポケモンが場に出るたびに **すばやさランク低下判定**  
4. 状態は解除技によって明示的に解除されるまで保持

## 補足

- 技「ねばねばネット」は **場に設置するタイプの技**であり、設置状態そのものに持続ターンは存在しない。 :contentReference[oaicite:5]{index=5}
