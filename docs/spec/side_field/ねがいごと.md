# ねがいごと（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ねがいごと（場の状態） |
| 発生方法 | 技「ねがいごと」の成功 |
| 効果対象 | その場にいるポケモン |
| 効果内容 | 次のターン終了時に対象ポケモンのHPを回復 |
| 回復量 | 最大HPの **1/2（小数点以下切り捨て）** |
| 効果持続 | **1ターン（1回のみ）** |
| 重ね掛け | **同位置に重ねて発生しない** |
| 発動場所 | 技使用時の **その場のポケモン位置** に適用 |

## 発動条件 / 無効条件

### 発動

- 技「ねがいごと」を使用し命中後、場の状態が設定される

### 無効／発動しない場合

| 条件 | 結果 |
|---|---|
| ターン終了時に対象ポケモンが HP満タン | 回復効果は発動するがHPは増加しない |
| 発動ターンに使用者が戦闘不能 | 対象が不在の場合は効果消失 |
| 「かいふくふうじ」などの効果 | 技自体が使用不可／発動できない |

## 効果詳細（HP回復）

| 効果対象 | 回復対象 |
|---|---|
| 次のターン終了時 | 場にいるポケモン1体（交代で出ているポケモン） |
| 回復量 | 対象の **最大HP × 1/2**（小数点以下切り捨て） |

## 競合・相互作用

| 相互作用 | 仕様 |
|---|---|
| 技「ねがいごと」複数回使用 | 同位置では上書きや重複発動はしない（1回分のみ） |
| 交代時の扱い | 回復対象は **そのターン終了時に場にいるポケモン** |

## ターン処理順（概要）

1. 技「ねがいごと」が成功 → 場状態を設定  
2. その後の行動処理・ダメージ等を解決  
3. 次のターン終了時に回復処理を実行  
4. 回復後、場の状態（ねがいごと）は消滅

## 補足

- ねがいごとの回復は **その位置にいるポケモン1体のみ** であり、交代によって回復対象が変わることがある
