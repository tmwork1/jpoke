# しんぴのまもり（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | しんぴのまもり（場の状態） |
| 発生方法 | 技「しんぴのまもり」の成功（味方側） |
| 効果対象 | 味方側の全ポケモン |
| 効果内容 | 状態異常・こんらん・あくび状態を **新たに付与されない** ようにする |
| 効果持続 | 原則 **5ターン**（開始ターン含む） |
| 既存状態への影響 | 既に付与済みの状態は **解除しない** |
| 重ね掛け | 継続ターンが **延長されない** |
| 他の場条件との共存 | 重複可能（例: ほかの場の状態と共存） |

## 発動条件 / 無効条件

### 発動

- 技「しんぴのまもり」によって場がこの状態になる

### 無効／解除

| 条件 | 結果 |
|---|---|
| 技「きりばらい」を命中 | しんぴのまもり状態は解除される |
| 過去世代の特定技（例: キョダイフウゲキ） | 第9世代では適用なし |

## 効果詳細（状態防止条件）

### 防止されるもの

| 状態 | 防止されるか |
|---|---|
| 主要状態異常（やけど／どく／もうどく／ねむり／まひ／こおり） | **防止される** |
| こんらん | **防止される** |
| あくび状態（ねむけ） | **防止される** |

### 防止されない／例外

| 状態・効果 | 結果 |
|---|---|
| すでに状態異常・こんらん・あくび  | **解除されない** |
| 自分自身が技「ねむる」使用時のねむり | 防止対象外 |
| 一部の状態変化系効果（メロメロ等） | 状態変化には影響しない場合あり |

## 競合・特殊相互作用

### 特性との相互作用

| 特性 | 作用 |
|---|---|
| すりぬけ | 敵側の「しんぴのまもり」を **無視**して状態変化を付与可能（自味方側は無視しない） |

### 解除系

| 技 | 効果 |
|---|---|
| コートチェンジ | 自・相手の場状態と入れ替えることが可能 |

## 処理フロー

1. 「しんぴのまもり」技が成功 → 味方側場に状態を設置  
2. しんぴのまもり有効ターン（最大5ターン）  
3. 他の場状態と共存可能  
4. きりばらい等で解除可能  
5. ターン経過で消滅

## 対象外の効果例

- 既に付与されているやけど/どく等の状態は解除されない  
- 自身の「ねむる」等で生じる状態は防止されない
