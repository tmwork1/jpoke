# まきびし（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | まきびし（場の状態） |
| 発生方法 | 技「まきびし」成功 |
| 設置対象 | **相手側の場** |
| 効果対象 | 相手側に **交代で場に出てきたポケモン** |
| 効果内容 | ダメージを与える（交代時のダメージ） |
| 重ね掛け | **最大3層まで重ねられる** |
| 他場状態との共存 | 他の場状態（ステルスロック・どくびし等）と共存可能 |
| 効果持続 | 設置状態が解除されるまで継続（ターン制限なし） |

## 発動条件 / 無効条件

### 発動

- 技「まきびし」を使用して成功した場合、相手側の場にまきびし状態が追加される。  
- 同じ場側に状態が既にある場合は **層数が1増える（最大3層）**。 :contentReference[oaicite:0]{index=0}

### 無効／解除条件

| 条件・技 | 仕様 |
|---|---|
| すでに3層まで設置済み | **追加層は増えない**（状態は維持） |
| こうそくスピン / キラースピン | 使用者側の場から **まきびしを解除** |
| きりばらい / おかたづけ | **両側**のまきびし状態を解除 | :contentReference[oaicite:1]{index=1}

## ダメージ条件・処理

| 層数 | ダメージ |
|---|---|
| 1層 | 最大HP × 1/8 |
| 2層 | 最大HP × 1/6 |
| 3層 | 最大HP × 1/4 |

- ダメージは **交代で場に出てきた時点で発生**する。  
- HP計算は小数点以下を切り捨て、**切り捨て結果が0でも1ダメージ発生**する場合がある。  
- これらの割合は第3世代以降の標準仕様。 :contentReference[oaicite:2]{index=2}

## 影響対象条件

まきびしのダメージは以下の条件により **判定されない / 受けない**：

| 条件 | 理由 |
|---|---|
| ひこうタイプ | 地面にいない扱いでダメージを受けない |
| 特性「ふゆう」など | 地面にいない扱いでダメージを受けない |
| 持ち物「あつぞこブーツ」 | 地面判定を無効化してダメージを受けない |
| 特性「マジックガード」 | 場ダメージを無効化 | :contentReference[oaicite:3]{index=3}

## 競合・相互作用

| 相互作用 | 仕様 |
|---|---|
| 特性「マジックガード」 | まきびしのダメージを **完全に無効化** |
| 持ち物「あつぞこブーツ」 | ダメージを **受けない** |
| ふゆう / ひこうタイプ | 地面判定を回避し、ダメージを受けない |
| 他の場状態 | ステルスロック/どくびし/ねばねばネット等と **同時共存** |

## ターン処理順（概要）

1. 技「まきびし」成功 → 相手側の場に1層追加（最大3層まで）  
2. 設置された状態は解除技が使われるまで保持  
3. ポケモンが場に出る → **層数に応じたダメージ判定処理**  
4. 攻撃対象ポケモンのHPを減少  
5. 次の交代・行動等へ処理継続
