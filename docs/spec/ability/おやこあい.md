# 仕様書: おやこあい

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%8A%E3%82%84%E3%81%93%E3%81%82%E3%81%84
- 種別: 連続ヒット化

## 基本効果
単体対象の攻撃技を使うと、同じ技による2回攻撃として処理する。1回目は通常、2回目は低威力（実装方針では1回目の最終ダメージに対する倍率）で判定し、追加効果や接触判定はヒットごとに個別に処理する。

## 詳細仕様
### 発動条件
- 単体対象の攻撃技であること。
- 特性が有効であること。

### 非発動条件
- 対象外の技種別（全体技、もともと連続攻撃扱いの技など）のとき。
- 1回目が命中せず不発のとき。

### 無効化・上書き
- かがくへんかガス等で無効化中は2回攻撃化しない。
- 特性書き換え後は新特性に従う。

### 交代・入場時
- 場にいる間だけ有効。交代時に専用状態は持ち越さない。

### その他の特性との相互作用
- ばけのかわ、きあいのタスキ、接触反応（さめはだ等）はヒットごとに判定する。

## 処理順・優先度
- 発動タイミング: 技解決中（1ヒット目の解決後に2ヒット目を連続実行）。
- 処理順: 1回目ダメージ/追加効果 -> 状態更新 -> 2回目ダメージ/追加効果。
- すばやさは参照しない。

## テスト項目
### 発動ケース
- 単体攻撃で2ヒットする。
- 2ヒット目が低威力で計算される。
- 接触反応が2回発生する。

### 非発動ケース
- 技が外れたとき2ヒット目が出ない。
- 対象外技では1ヒットのまま。

### 境界ケース
- 1ヒット目でタスキ発動後に2ヒット目で突破できる。
- 1ヒット目で相手がひんしなら2ヒット目を行わない。
