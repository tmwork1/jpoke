# 仕様書: フェアリースキン

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%83%95%E3%82%A7%E3%82%A2%E3%83%AA%E3%83%BC%E3%82%B9%E3%82%AD%E3%83%B3
調査日: 2026-05-02

## 2. 基本効果（1段落）
ノーマルタイプのわざがフェアリータイプになり、威力が1.2倍になる

## 3. 詳細仕様
- 発動条件: 使用する技がノーマルタイプであるとき発動し、技のタイプをフェアリータイプに変換したうえで威力を4915/4096倍にする。
- 元からフェアリータイプの技を使っても威力は上昇しない（タイプ変換が起きないため）。
- わるあがきを使用したときは発動しない。
- 変化技にも発動するが、タイプ変換がバトルに影響するのは相性が関わるケースに限られる。
- タイプ変換後のフェアリータイプとして処理される（タイプ一致判定、相性計算、アイテム補正含む）。
- 無効化・上書き: かがくへんかガスで無効化される。特性変更を受けた場合は変更後の特性を優先する。

## 4. 処理順・優先度
- 発動タイミング: ダメージ計算・命中判定の前に技のタイプと威力を変換する。
- 同時発動時の優先度: 他の威力補正（タイプ一致ボーナス等）と乗算で適用する。

## 5. テスト項目（実装検証用）
- 発動ケース: ノーマル技がフェアリータイプ相性で当たり、威力が4915/4096倍されること。
- 非発動ケース: 元々フェアリータイプの技を使ったとき威力が上昇しないこと。わるあがきには発動しないこと。
- 境界ケース: かがくへんかガス下では通常のノーマルタイプのまま処理されること。
