# 仕様書: ごりむちゅう

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%94%E3%82%8A%E3%82%80%E3%81%A1%E3%82%85%E3%81%86

## 2. 基本効果
最初に選んだ技しか使用できなくなるが、こうげきが1.5倍になる。

## 3. 詳細仕様

### 行動制限
- 技選択時に最初に選んだ技のみ使用可能
- 他の技を選択しようとすると選択不可になる
- 技変更は交代でリセットされる

### ダメージ補正
- 物理技のこうげきが1.5倍になる
- ランク変化ではなく実数値補正として扱う

### 交代・入場時
- 交代すると技制限は解除される
- 再度場に出ると新たに最初の技を選び直す

### 無効化・上書き
- 特性が無効化されている場合：
  - 技制限・こうげき上昇ともに無効
- かがくへんかガス等で無効化可能
- 特性のコピー・交換は可能

### その他の特性との相互作用
- こだわり系アイテムと重複可能
  - こうげき補正が乗算される
- へんしん・スキルスワップ等で特性変化した場合：
  - 状態がリセットされる場合がある

## 5. テスト項目（実装検証用）

### 発動ケース
- 最初に選んだ技しか使用できない
- 物理技の火力が1.5倍になる

### 非発動ケース
- 特性無効化状態では効果が発動しない
- 交代後は技制限が解除される

### 境界ケース
- こだわり系アイテムと同時に効果が乗算される
- 技変更系効果（スキルスワップ等）で状態が変化する
