# 仕様書: こぼれダネ

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%93%E3%81%BC%E3%82%8C%E3%83%80%E3%83%8D

## 2. 基本効果
攻撃技でダメージを受けたとき、グラスフィールドを展開する。

## 3. 詳細仕様

### 発動条件
- 攻撃技でダメージを受けたときに発動
- 接触・非接触は問わない
- ダメージが発生した場合のみ発動（無効化・みがわり等では発動しない）

### 効果内容
- グラスフィールドを展開する（通常5ターン）
- フィールドが既に存在する場合は発動しない

### 発動タイミング
- ダメージ処理後に発動
- 同一ターン中の複数回攻撃では、初回ダメージ後に判定される

### 無効化・上書き
- 特性無効化状態では発動しない
- 特性のコピー・交換は可能

### 交代・入場時
- 常時効果（被弾トリガー）
- 入場時の即時効果なし

## 5. テスト項目（実装検証用）

### 発動ケース
- 攻撃技でダメージを受けたときグラスフィールドが展開される

### 非発動ケース
- ダメージを受けない場合は発動しない
- 既にグラスフィールド状態では発動しない
- 特性無効化状態では発動しない

### 境界ケース
- 連続技では最初のヒット後にフィールドが展開される
- ダメージ0の場合は発動しない
