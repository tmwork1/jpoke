# 仕様書: とびだすなかみ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A8%E3%81%B3%E3%81%A0%E3%81%99%E3%81%AA%E3%81%8B%E3%81%BF

とびだすなかみ はポケモンのとくせいの一種。
## 効果
相手の 攻撃技 で ひんし になったとき、技を受ける直前に残っていた HP の分だけ、その相手にダメージを与える。
## 説明文
- 第七世代 ・ 第八世代 ・ 第九世代: あいてに たおされたとき HPの のこりの ぶんだけ あいてに ダメージを あたえる。
## 所有ポケモン : そのポケモンの別のとくせい
|  | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| ナマコブシ | とびだすなかみ | - | てんねん |
## こんなときに使おう
死に際で相手に反撃ができる。しかし、ナマコブシ自身は ぼうぎょ ・ とくぼう こそ高いがHP自体は低いため、耐久戦で使用しても与えるダメージはあまり期待できない。HPに 努力値 を振って実数値を増やし、弱点を突いてくる相手や積み技で能力を上げた相手に出すなど、HP満タンの状態から一撃で倒されるように立ち回ると相打ちを狙いやすくなる。
## 特性の仕様
- 直接攻撃 でなくても、自分が 攻撃技 を受けて ひんし になれば発動する。
- 天候 や 状態異常 のダメージなど、攻撃技以外のダメージでひんしになった場合は発動しない。
- この特性の効果でお互いのポケモンのHPが0になった場合、とびだすなかみ側が先にひんしになる。このときお互いのポケモンが全滅した場合はとびだすなかみ側の負けになる。
- 味方の攻撃に対しても発動する。
  - 味方の範囲攻撃などでこの特性が発動し、攻撃した仲間をひんしにさせて全滅した場合、同時に相手も全滅させたとしてもとびだすなかみ側の負けとなる。
- 相手の みがわり 状態を無視してダメージを与える。
- 連続攻撃技 が複数回当たってひんしになった場合、連続攻撃技が当たり始めた時点での残りHPの分だけ相手にダメージを与える。
  - 途中 きのみ が発動してHPが回復したときや、途中まではみがわり状態で攻撃を防いでいたときなど、最初のHP量と受けたダメージの総量が異なる場合でも、最初のHP量がとびだすなかみで与えるダメージになる。
- ダイマックス している間に倒されて発動したときは、残っていたHPをダイマックスレベルによる補正率で割った値が与えるダメージになる。
  - ダイマックスした相手に倒されたときは、残っていたHPそのままが相手に与えるダメージになる。
- じばく / だいばくはつ による攻撃を受けたときなど、攻撃者がすでにひんしになって場にいない場合、とびだすなかみが発動することもない。
- みらいよち / はめつのねがい の攻撃を受けた場合、技を出したポケモンが場にいれば発動する。場にいない場合は発動しない。
- 後攻の コアパニッシャー を受けてひんしになった場合、コアパニッシャーの効果で とくせいなし 状態になってから倒れるためとびだすなかみは発動しない。
- HPを吸収するわざ / はじけるほのお / やきつくす / ダイマックスわざ を受けてひんしになった場合、技の効果が発動してからとびだすなかみが発動する。
- こうそくスピン / サウザンアロー / サウザンウェーブ / プラズマフィスト / とんぼがえり / ボルトチェンジ / クイックターン / オリジンズスーパーノヴァ / ラジアルエッジストーム / コンビネーションわざ を受けて発動し、攻撃側もひんしになった場合は技の効果は発動しない。
- アイアンローラー を受けて発動したとき、攻撃してきた相手をひんしにしたとしても技の効果は発動する。
## 備考
- ナマコブシの 専用特性 。
- 同じく自分が ひんし になったときに攻撃しててきた相手にダメージを与える特性として ゆうばく がある。ゆうばくはとびだすなかみとは異なり 直接攻撃 に対してしか反応しない。
