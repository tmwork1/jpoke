# 仕様書: すながくれ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%99%E3%81%AA%E3%81%8C%E3%81%8F%E3%82%8C

## 2. 基本効果
天気がすなあらしのとき、相手の技の命中率を低下させ、さらにすなあらしのダメージを受けない。

## 3. 詳細仕様

### 命中率補正
- 天気が「すなあらし」のとき、相手の技の命中率を0.8倍にする
- 回避率ランクとは別に独立した命中補正として扱う

### ダメージ耐性
- すなあらしによる定数ダメージを受けない
- それ以外の天候ダメージには影響しない

### フィールド効果（任意仕様）
- 手持ち先頭時、すなあらし環境下で野生ポケモンの出現率が低下する

### 発動条件
- 場にいる間、常時発動するパッシブ特性
- 天候がすなあらしであることが必須条件

### 無効化・上書き
- 特性が無効化されている場合は全効果が発動しない
- 特性のコピー・交換は可能

### 交代・入場時
- 場にいる間のみ有効
- 天候条件を満たさない場合は即時無効化される

## 5. テスト項目（実装検証用）

### 発動ケース
- すなあらし時に相手の命中率が低下する
- すなあらしダメージを受けない

### 非発動ケース
- 通常天候では効果なし
- 特性無効化状態では発動しない

### 境界ケース
- 命中補正は他の命中・回避補正と乗算で処理される
- 天候変化により効果は即時切り替わる
