# 仕様書: じょおうのいげん

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%98%E3%82%87%E3%81%86%E3%81%AE%E3%81%84%E3%81%92%E3%82%93

## 2. 基本効果
相手は先制技を使用できない。

## 3. 詳細仕様

### 対象効果
- 相手が使用する「先制技（優先度+1以上の技）」を対象とする
- 優先度+2以上の技も含めて使用不可にする

### 効果内容
- 相手は優先度+1以上の技を選択・実行できない
- 該当する技を選択した場合、その技は不発となる

### 発動条件
- この特性を持つポケモンが場にいる間、常時有効

### 発動タイミング
- 技選択・行動実行の両段階で制限がかかるパッシブ効果

### 無効化・上書き
- 特性が無効化されている場合は効果が発動しない
- かがくへんかガス等で特性が無効化されると制限も失われる
- 特性のコピー・交換は可能

### 交代・入場時
- 場にいる限り継続的に有効
- 交代で場を離れると効果は消失する

## 5. テスト項目（実装検証用）

### 発動ケース
- 相手が先制技を選択できない
- 優先度+1以上の技が使用不可になる

### 非発動ケース
- 特性無効化状態では先制技制限がかからない
- 優先度0以下の技には影響しない

### 境界ケース
- 優先度が変化する効果と併用時は最終優先度で判定する
- 技選択時点で制限されるため行動前に弾かれる
