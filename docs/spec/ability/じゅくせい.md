# 仕様書: じゅくせい

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%98%E3%82%85%E3%81%8F%E3%81%9B%E3%81%84

じゅくせい は、ポケモンのとくせいの一種。
## 効果
このポケモンが発動させた きのみ の効果が2倍になる。
- HP ・ PP を回復するきのみの場合は、回復量が2倍になる。
- ランク補正 が上昇するきのみの場合は、上昇量が2倍になる。
- 半減系きのみの場合は、受ける ダメージ を1/4にする。
- ジャポのみ ・ レンブのみ の場合は、与えるダメージが相手の最大HPの1/4になる。
## 説明文
- 第八世代 ・ 第九世代: じゅくせい させることで きのみの こうかが ばいに なる。
## 所有ポケモン：そのポケモンの別のとくせい

| カジッチュ | じゅくせい | くいしんぼう | ぼうだん |
| アップリュー | じゅくせい | くいしんぼう | はりきり |
| タルップル | じゅくせい | くいしんぼう | あついしぼう |
## こんなときに使おう
耐久型の タルップル なら消費したきのみを リサイクル で復活させることができる。
耐久が低いアップリューにはあまり採用されない。
## 特性の仕様
- じゅくせいのポケモンが なげつける / おちゃかい を受けてきのみを消費させられたときも特性バーが現れ、きのみの効果が2倍になる。じゅくせいのポケモンがなげつける/おちゃかいを使用しただけでは2倍にならない。
- じゅくせいのポケモンが むしくい / ついばむ を使用して相手のきのみを食べたときも特性バーが現れ、きのみの効果が2倍になる。じゅくせいのポケモンがむしくい/ついばむを受けただけでは2倍にならない。
- ほおばる で消費したきのみの効果は2倍になるが、特性バーは現れない。ほおばるの技自体によるランク変動量は変わらない。
- 最大HP割合で決まるきのみの回復量は小数点以下の端数を切り捨てるが、じゅくせいの効果は切り捨てた後の値を2倍する。「1/4回復」が「1/2回復」になるわけではない。
  - 例えばHP150のポケモンが オボンのみ で回復する場合、150/4=37.5なので通常は37回復するが、この特性によって2倍になると75ではなく74回復になる。
- ジャポのみ / レンブのみ で与えるダメージは「最大HPの1/4 (切り捨て) 」になる。「最大HPの1/8 (切り捨て) の2倍」ではない。
- 半減系きのみが発動したとき、じゅくせいの特性バーは現れないが、受ける ダメージ は1/4になる。1/2の補正が2回掛けられるわけではない。
- 以下のきのみを消費したときもじゅくせいの特性バーは現れるが、他の特性のポケモンが消費したときと効果は変わらない。
  - ラムのみ など 状態異常 を回復するきのみは、1度発動することで通常どおり無くなる。
  - サンのみ による 急所 ランク上昇は+2のままで、確定急所にはならない。
  - ミクルのみ による 命中 補正は1.2倍のまま変わらない。
  - イバンのみ による行動順が、他のイバンのみ/ せんせいのツメ / クイックドロウ の効果より優先されることは無い。
- かがくへんかガス のポケモンが ひんしになる攻撃技 を使用してかがくへんかガスが解除されるとき、その攻撃で発動した半減系きのみにじゅくせいの効果は無い。 ナゾのみ /オボンのみ/ タラプのみ などが発動するときはじゅくせいの効果がある。
- コアパニッシャー を受けて とくせいなし 状態になるとき、半減系きのみにじゅくせいの効果はあるが、ナゾのみ/レンブのみ/オボンのみ/タラプのみなどにじゅくせいの効果は無くなる。
- プレイヤーが バッグ から使用したきのみにじゅくせいの効果は無い。
- バトル外で 基礎ポイント を減少させるきのみを使用したときの効果は2倍にならない。
- 以下の道具はきのみではないため効果は変わらないが、発動するとじゅくせいの特性バーが現れる。
  - たべのこし / きのみジュース / しろいハーブ / メンタルハーブ
  - からぶりほけん / のどスプレー / じゃくてんほけん / じゅうでんち / きゅうこん / ひかりごけ / ゆきだま / ビビリだま / エレキシード / サイコシード / ミストシード / グラスシード / ルームサービス / ものまねハーブ
  - きあいのハチマキ / せんせいのツメ / パワフルハーブ / だっしゅつボタン / だっしゅつパック
  - なげつける で受けて 追加効果 が発動した でんきだま / かえんだま / どくどくだま / どくバリ / おうじゃのしるし / するどいキバ /しろいハーブ/メンタルハーブ
