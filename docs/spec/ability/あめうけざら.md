## 2. 基本効果（1段落）
天気が雨のとき、毎ターン最大HPの1/16を回復する

## 効果

あめうけざらは、ポケモンのとくせいの一種。

天気があめ・おおあめ状態のとき、ターン終了時に最大HPの1/16を回復する（小数点以下切り捨て）。

## 説明文

- **第七世代・第八世代・第九世代** 天気が 雨のとき 少しずつ HPを 回復する。

## 特性の仕様

- あめが終了したターンでは回復しない。
- 第五世代以降では、かいふくふうじ状態では回復できない。かいふくふうじ状態が終了するターンでも回復できない。
- あなをほる/ダイビング状態により隠れている間は回復しない。そらをとぶ/シャドーダイブ状態で隠れているならば回復する。
- 特性ノーてんき/エアロックのポケモンが場にいる間は発動しない。
- ばんのうがさを持っているときは発動しない。
- 状態異常や状態変化のダメージやたべのこし/くろいヘドロ等の効果と同じタイミングで発動する。

## 備考

- たべのこしを併用すると、あめ状態の間は毎ターン最大HPの1/8を回復できる。
- 第四世代まではハスボー系統の専用特性だった。それ以外のポケモンは隠れ特性限定。

## 関連項目

- アイスボディ（あられ/ゆきのとき1/16回復）
- かんそうはだ（あめのとき1/8回復）

- 非発動ケース: 条件を満たさない場合や無効化条件下で発動しないこと。
- 境界ケース: 交代直後、特性無効化、同時処理競合時に破綻しないこと。
