# 仕様書: サンパワー

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%B5%E3%83%B3%E3%83%91%E3%83%AF%E3%83%BC

## 2. 基本効果
天気がにほんばれまたはおおひでりのとき、とくこうが1.5倍になるが、毎ターン終了時に最大HPの1/8のダメージを受ける。

## 3. 詳細仕様

### 発動条件
- 場の天候が「にほんばれ」または「おおひでり」のときに有効

### 効果内容
- とくこうが1.5倍になる
- 毎ターン終了時に最大HPの1/8のダメージを受ける（小数点以下切り捨て）

### ダメージ処理
- ターン終了時に発動する定期ダメージとして処理される
- 回復効果（たべのこし等）との順序に依存する

### 無効化・上書き
- 特性が無効化されている場合は発動しない
- 天候が書き換えられた場合、条件を満たさなくなれば即時無効化される

### 交代・入場時
- 場にいる間のみ有効
- 交代で場を離れると効果は消失する

## 5. テスト項目（実装検証用）

### 発動ケース
- にほんばれ状態でとくこうが1.5倍になる
- おおひでり状態でも同様に発動する

### 非発動ケース
- 通常天候では発動しない
- 特性無効化状態では発動しない

### 境界ケース
- 天候が途中で変化すると効果倍率が即時変わる
- ターン終了時ダメージと回復処理の順序を正しく処理する
