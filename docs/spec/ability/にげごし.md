# 仕様書: にげごし

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AB%E3%81%92%E3%81%94%E3%81%97

にげごし は、ポケモンの とくせい の一種。
## 効果
自分の HP が最大HPの1/2超から1/2以下に減ったとき、 トレーナー戦 では 交代 する。 野生ポケモン 戦では にげる 。
## 説明文
- 第七世代 ・ 第八世代 ・ 第九世代: HPが はんぶんに なると あわてて にげだして てもちに ひっこんで しまう。
## 所有ポケモン：そのポケモンの別のとくせい
|  | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| コソクムシ | にげごし | - | - |
## こんなときに使おう
コソクムシを捕獲する際も みねうち などでHPを削るとこの特性により逃げられる。特性を書き換えるか とくせいなし 状態にしてから削れば逃げられない。 クイックボール ・ ネットボール も有効だが、コソクムシに対してもネットボールの補正率は3.5のままである。
## 特性の仕様
詳細な仕様は ききかいひ#特性の仕様 を参照とする。以下に、捕獲の際にも影響が出やすい効果のみ抜粋する。
- 状態異常 / 状態変化 / 場の状態 などのスリップダメージにより半分以下になったときも発動する。
- こんらん による自傷ダメージや、 いたみわけ を受けてHPが半分以下になったときは発動しない。
- 特性 かげふみ / ありじごく や、 にげられない / バインド / フェアリーロック 状態の効果を無視して逃げる。
## 備考
- コソクムシ の 専用特性 。 グソクムシャ に進化すると特性が ききかいひ に変わるが、名前が変わるだけで効果は変わらない。
