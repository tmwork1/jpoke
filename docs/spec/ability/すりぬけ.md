# 仕様書: すりぬけ

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%99%E3%82%8A%E3%81%AC%E3%81%91

## 2. 基本効果（1段落）
相手の場や相手自身にかかっている一部の防御効果（リフレクター・ひかりのかべ・みがわり等）を無視して攻撃や効果を適用する。

## 3. 詳細仕様

### 防御効果無視の対象
- 以下の効果を無視して攻撃可能
  - リフレクター（物理ダメージ軽減）
  - ひかりのかべ（特殊ダメージ軽減）
  - オーロラベール
  - しんぴのまもり（状態異常防止）
  - みがわり
- 「みがわり」に対しては本来遮られる技も本体へ直接命中する

### 防御効果無視の挙動
- 壁系効果は存在していてもダメージ計算時に無視される
- 状態異常技も「しんぴのまもり」を無視して付与可能
- 「みがわり」は存在していても攻撃対象として扱わず、本体に直接作用

### 無効化・上書き
- 特性が無効化されている場合（例：かがくへんかガス）、防御無視効果は発動しない
- 「かたやぶり」等と異なり、特性自体を無視されることは基本的にない（＝攻撃側の能力として機能）

### 交代・入場時
- 場に出ている間のみ有効
- 発動タイミングは常時（攻撃処理時に参照）

### その他の特性との相互作用
- 「マジックミラー」等の反射系は無視できない（通常通り反射される）
- 「ぼうおん」など技自体を無効化する特性は無視できない
- 「がんじょう」「きあいのタスキ」などHP関連の耐久効果は無視できない
- 「みがわり」を無視できる点が最大の特徴

## 4. 処理順・優先度
- 発動タイミング：技の効果適用時（命中後〜ダメージ計算時）
- 処理順：
  1. 命中判定
  2. すりぬけの適用判定
  3. 防御効果を無視してダメージ・効果適用
- 同時発動の概念なし（常時効果）
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- リフレクター下で通常ダメージになる
- ひかりのかべ下で通常ダメージになる
- しんぴのまもり下で状態異常が付与される
- みがわりを無視して本体にダメージが入る

### 非発動ケース
- 特性無効化状態で壁軽減が適用される
- みがわりが通常通り機能する（特性無効時）

### 境界ケース
- みがわり＋きあいのタスキの組み合わせで本体に直接攻撃→タスキが発動する
- マジックミラー持ちに変化技を使用した場合、通常通り反射される
- 壁が存在していても表示上は消えず、内部的にのみ無視される
