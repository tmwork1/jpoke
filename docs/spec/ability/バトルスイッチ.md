# 仕様書: バトルスイッチ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%90%E3%83%88%E3%83%AB%E3%82%B9%E3%82%A4%E3%83%83%E3%83%81

バトルスイッチ は、ポケモンのとくせいの一種。
## 効果
- シールド フォルム のとき 攻撃技 を使うとブレードフォルムに フォルムチェンジ してから攻撃する。
- ブレードフォルムのとき キングシールド を使うとシールドフォルムにフォルムチェンジする。
- 交代 するとシールドフォルムになる。
## 説明文
- 第六世代: せんとうモードで すがたが かわる。
- 第七世代 ・ 第八世代 ・ 第九世代: こうげきわざを だすと ブレードフォルムに わざ キングシールドを だすと シールドフォルムに へんかする。
## 所有ポケモン：そのポケモンの別のとくせい

| ギルガルド | バトルスイッチ | - | - |
## こんなときに使おう
単純なアタッカーのギルガルドは、相手が自分より早いときは キングシールド の 優先度 のおかげで相手の攻撃より早くシールドフォルムになれるため隙が無いが、相手が自分より遅いときは攻撃技でブレードフォルムになった後に攻撃を受けることになるため苦手とする。
キングシールドは変化技を無効化できないこともあり、行動を制限しうる 状態異常 ・ 状態変化 にも弱い。状態異常などで行動ができない場合、 第六世代 ではバトルスイッチが発動してから行動不能になるが、 第七世代 以降ではバトルスイッチは発動しない。シールドフォルムのとき攻撃技が出せなくても安全になったが、ブレードフォルムのときキングシールドが出せないと危険になった。
## 特性の仕様
- 場に登場したときはシールドフォルムだが、 攻撃技 を選択するとブレードフォルムにチェンジしてから技を繰り出す。シールドフォルムに戻る場合はギルガルドを 控えに戻す か技 キングシールド を使用する必要がある。
  - シールドフォルムで 変化技 を使用した場合、バトルスイッチは発動しない。 他の技が出る技 を使用したときについては 後述 。
  - ブレードフォルムでキングシールド以外の技を使用した場合、バトルスイッチは発動しない。 まもる / ダイウォール 等、身を守る技であってもキングシールドでなければバトルスイッチは発動しない。
  - 発動すると特性バーが現れ、ブレードフォルムに変わった際は「ブレードフォルム チェンジ!」、シールドフォルムに変わった際は「シールドフォルム チェンジ!」とメッセージが流れる。
  - シールドフォルムの状態で選んだ攻撃技が、相手の 相性 / 特性 / まもる 状態などによって無効化される場合であっても、バトルスイッチは発動してブレードフォルムになる。
    - ソーラーブレード などの 溜め技 を使用した場合、溜めるターンでバトルスイッチが発動する。
    - わるあがき を出すときもバトルスイッチは発動してブレードフォルムになる。変化技しか覚えていないときであっても同様となる。
  - その ターン で最後に行動したため、あるいは まもる 系統の技を連続使用したためにキングシールドがうまく決まらなかった場合でも、バトルスイッチは発動してシールドフォルムになる。
- この特性は 上書きできない 。 とくせいなし 状態になることや、 かがくへんかガス で無効化されることも無い。
- この特性は コピーできない 。
- この特性を スキルスワップ / さまようたましい で交換することはできない。
- フォルムチェンジ 特性の1つであり、 みずびたし / もりののろい などで書き換えられたタイプや、 パワートリック 状態や スピードスワップ などで変えられたステータスもフォルムチェンジすれば本来のものに上書きされる。
  - シールドフォルムでタイプ/ステータスが書き換えられている状態で攻撃技を出した場合、 タイプ一致ボーナス や ダメージ は元に戻ったものを参照して計算される。
  - ボディパージ で おもさ を変えてもフォルムチェンジすれば元のおもさに戻る。
- 他のポケモンがギルガルドに へんしん しても、バトルスイッチの効果は発動しない。シールドフォルムのギルガルドにへんしんした場合は常にシールドフォルム、ブレードフォルムのギルガルドにへんしんした場合は常にブレードフォルムのままであり、いかなる技を使用してもフォルムは変更できない。
  - しかし、このときもバトルスイッチは上記の技や特性による変更やコピーを受け付けない。
### 第六世代のみ
- 「特性を発動してから行動の判定を決める」ため、 まひ / ねむり 等で行動できなかった場合でもこの特性は発動する。攻撃技を選択し行動できなかった場合でもシールドフォルムからブレードフォルムに変わり、キングシールドを選択して攻撃できなかった場合でもシールドフォルムに戻る。
- ねごと で出た技ではバトルスイッチは発動しない。このときシールドフォルムで攻撃技を使うことや、ブレードフォルムでキングシールドを使うことがある。
### 第七世代以降
- 「行動の判定を決めてから特性を発動する」ため、 まひ / ねむり 等で行動できなかった場合、この特性は発動しない。
- ねごと を使用した場合、出た技に応じてバトルスイッチは発動する。
- ダイマックス するときは現在のフォルムのままダイマックスする。攻撃系の ダイマックスわざ を使えばシールドフォルムからブレードフォルムに変わることはできるが、ダイマックス中はキングシールドを出せないため、ブレードフォルムからシールドフォルムに変わる手段は無い。
### Pokémon LEGENDS Z-A
- LEGENDS Z-A には特性は存在しないがバトルスイッチは行われる。
- 技を出す指示をした後にバトルスイッチするため、それだけ技を出すのが遅くなる。リアルタイム戦闘において不利にもなり得る。
- 非戦闘時でも控えに戻すか、該当の技を出さなければフォルムは維持される。
