# 仕様書: ふゆう

調査日: 2026-04-30

## 2. 基本効果（1段落）
地面にいない状態として扱われ、じめんタイプの攻撃技を無効化する。

## 3. 詳細仕様
- 攻撃無効: じめんタイプ攻撃技をタイプ相性判定前に無効化する。
- 例外技: すなかけ（変化技）やサウザンアロー等、ふゆうで防げない例外がある。
- 地面判定派生: まきびし/どくびし/ねばねばネット、ありじごく、フィールド接地条件などに影響。
- 無効化条件: じゅうりょく、くろいてっきゅう、うちおとす状態、ねをはる状態、かがくへんかガス等で効果喪失。
- かたやぶり相互作用: かたやぶり効果付き攻撃ではふゆう無視で地面技が通る。

## 4. 処理順・優先度
- 適用タイミング: 接地判定/タイプ無効判定時。
- 優先関係: かたやぶり・特性無効化の判定後にふゆう有効性を確定。
- 状態更新: うちおとす等で接地状態が変化したら即時反映。

## 5. テスト項目（実装検証用）
- 発動ケース: じめん攻撃技を無効化。
- 非発動ケース: じゅうりょく中やうちおとす後は被弾。
- 境界ケース: かたやぶり地震、サウザンアロー、設置技接触判定。
