## 2. 基本効果（1段落）
相手の特性の影響を無視して技を使用できる。これにより、本来特性によって無効化・軽減・制限される技の効果を、その特性を無視して適用することができる。ただし「攻撃後に発動する特性」には影響しない。

## 3. 詳細仕様

### 特性無視の基本挙動
- 攻撃技・変化技のいずれにも適用
- 相手の特性による以下の効果を無視
  - 技の無効化（例：ふゆうによるじめん無効）
  - ダメージ軽減・無効化（例：あついしぼう、カブトアーマー）
  - 能力変化耐性（例：クリアボディ）
- 相手の特性が存在していても、ダメージ計算・効果適用時には参照しない :contentReference[oaicite:0]{index=0}

### 無視できない特性
- 「攻撃後に発動する特性」は無視できない
  - 例：てつのトゲ、さめはだ（接触ダメージ） :contentReference[oaicite:1]{index=1}
- 一部の状態異常防止特性は無視可能だが、その後の処理で回復する場合あり :contentReference[oaicite:2]{index=2}

### 状態異常・変化との相互作用
- 状態異常無効特性（例：めんえき等）を無視して付与可能
- ただし特性側の後処理により即時回復するケースがある
- 一部特性（例：リーフガード等）は無視しても回復されない例外あり :contentReference[oaicite:3]{index=3}

### 無効化・上書き
- 自身の特性が無効化されている場合（例：かがくへんかガス）、効果は発動しない
- 特性書き換え・交換・コピーは可能
- 相手が「とくせいガード」を持つ場合、その特性は無視できない :contentReference[oaicite:4]{index=4}

### 交代・入場時
- 場に出た際に発動メッセージが表示される
- 常時効果として存在し、攻撃時に参照される
- 交代すると状態はリセット（特性は保持）

### その他の特性との相互作用
- 「カブトアーマー」など急所無効特性を無視して急所を発生させられる :contentReference[oaicite:5]{index=5}
- 「がんじょう」などの耐久系特性も無視可能
- 「ふゆう」無視によりじめん技が命中する
- 「てつのトゲ」などの反動特性は無視不可

## 4. 処理順・優先度
- 発動タイミング：技の効果適用時（命中後〜ダメージ計算時）
- 処理順：
  1. 命中判定
  2. かたやぶりの適用判定
  3. 相手特性を無視して効果・ダメージ計算
  4. 攻撃後特性（反動など）は通常通り発動
- 同時発動時の優先度：行動順（すばやさ）に依存
- すばやさは行動順決定のみに影響し、特性処理自体には影響しない

## 5. テスト項目（実装検証用）

### 発動ケース
- ふゆう持ちにじめん技が命中する
- カブトアーマー持ちに急所が発生する
- クリアボディ持ちに能力低下が入る
- がんじょうを無視して一撃で倒せる

### 非発動ケース
- 特性無効化状態で通常通り特性が適用される
- とくせいガード持ち相手に特性が無視できない

### 境界ケース
- 接触攻撃時、てつのトゲのダメージは受ける
- 状態異常付与後に即回復する特性の挙動を確認
- 変化技でも特性無視が適用される
