# 仕様書: カブトアーマー

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%AB%E3%83%96%E3%83%88%E3%82%A2%E3%83%BC%E3%83%9E%E3%83%BC
調査日: 2026-04-30

## 2. 基本効果（1段落）
相手から受ける攻撃技が急所に当たらなくなる。急所ランク上昇や確定急所技であっても、特性が有効な限り急所にならない。

## 3. 詳細仕様

### 急所無効化の基本挙動
- 相手の攻撃は急所判定が発生しない
- 急所ランク上昇（+1〜+3以上）でも急所にならない
- 急所時のダメージ補正・能力ランク無視は発生しない

### 確定急所技との相互作用
- 以下のような「必ず急所に当たる技」も急所にならない
  - こおりのいぶき
  - やまあらし
  - あんこくきょうだ
  - すいりゅうれんだ
  - トリックフラワー

### 状態異常・他特性との相互作用
- 自身がどく／もうどく状態でも、特性「ひとでなし」による確定急所は無効化される
- リフレクター・ひかりのかべの急所無視効果も発生しない（＝壁が有効のまま）

### 無効化・上書き
- 「かたやぶり」系効果を持つ攻撃に対しては無視される
  - この場合、急所が通常通り発生する
- 特性無効化状態（例：かがくへんかガス）では効果を発揮しない
- 特性書き換え・交換・コピーは可能（通常特性と同様の扱い）

### 交代・入場時
- 場に存在している間のみ有効
- 交代すると状態はリセット（特性自体は保持される）

### その他の特性との相互作用
- 「スナイパー」など急所強化特性は無効化される（急所が発生しないため）
- 同効果の特性「シェルアーマー」と同一仕様

## 4. 処理順・優先度
- 発動タイミング：ダメージ計算時（急所判定フェーズ）
- 処理順：
  1. 急所判定前に本特性を確認
  2. 急所判定を強制的に無効化
- 同時発動の概念なし（常時効果）
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- 通常攻撃で急所が発生しない
- 急所ランク+3状態でも急所が発生しない
- 確定急所技が急所にならない
- 壁（リフレクター等）が急所無視されずに適用される

### 非発動ケース
- かたやぶり状態の攻撃で急所が発生する
- 特性無効化状態で急所が発生する

### 境界ケース
- どく状態＋ひとでなし攻撃でも急所にならない
- スナイパー持ち相手でも急所にならない
- シェルアーマーと同一挙動であること
