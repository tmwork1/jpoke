# 仕様書: ビーストブースト

参照URL: https://wiki.pokemonwiki.com/wiki/ビーストブースト

## 基本効果
攻撃技で相手をひんしにしたとき、自分の最も高い能力のランクを 1 段階上げる。

## 詳細仕様

### 発動条件
- 発動契機は「攻撃技で相手をひんしにする」こと。
- 発動条件の細部は じしんかじょう と同系統として扱う。

### 上昇能力の選択
- HPを除く5能力（こうげき/ぼうぎょ/とくこう/とくぼう/すばやさ）から選ぶ。
- 参照値は実数値。ランク補正・持ち物・場補正による一時的変動は参照しない。
- 同値の場合は こうげき → ぼうぎょ → とくこう → とくぼう → すばやさ の順で決定する。

### 上限時の挙動
- 選ばれた能力がすでに最大ランクなら不発。
- 2番目に高い能力へ繰り下げはしない。

## 処理順・優先度
- ひんし確定後、場離れ処理の同系統タイミングで能力上昇判定を行う。
- 同時に複数の「倒したとき発動」特性がある場合は既存イベント順に従う。

## テスト項目

- [ ] 相手を倒したときに能力が1段階上がる
- [ ] 最も高い実数値の能力が選ばれる
- [ ] 同値時に優先順位どおりの能力が選ばれる
- [ ] 最大ランク時に不発となり、他能力へは繰り下がらない
- [ ] 参照値がランク補正や場補正で変わらない
