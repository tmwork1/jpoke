# 仕様書: りんぷん

調査日: 2026-04-30

## 2. 基本効果（1段落）
技の追加効果を受けない

## 3. 詳細仕様

### 効果
相手の攻撃技による追加効果を受けない。

### 説明文
りんぷんに 守られて 技の 追加効果を 受けなくなる。

### 特性の仕様
- オーバーヒートなど反動で使用者の能力が下がる技の効果は発動する。
- チャージビームなど、追加効果で使用者の能力が変化する技の効果は、自分を使用したときも相手から受けたときも発動する。
- ダイマックスわざによる効果は防げない。
- 特定条件で威力やダメージが変動する効果や、タイプ・分類などが変化する効果は追加効果ではないので防げない。
- りんぷんで追加効果を防いだことがメッセージで明示されることは無い。
- かたやぶりの効果がある技による追加効果は防げない。
- 特性ちからずくのポケモンから追加効果がある技を受けたとき、ちからずくによる追加効果の消失が優先され、技の威力は1.3倍になる。
- 特性どくしゅ/どくのくさりによるどく/もうどく状態も防げる。
- なげつけるにより追加効果がある道具を投げられたときの効果も防ぐ。
- オボンのみやアッキのみなどによる防御側にとってメリットになる効果も全て防ぐ。

### 備考
- 第九世代現在でこの特性を持つポケモンはむしタイプのみ（蝶や蛾をモチーフにしたポケモン）。

## 4. 処理順・優先度
- 発動タイミング: 攻撃技の追加効果適用フェーズで評価する。
- 同時発動時の優先度: ちからずくとの競合時はちからずくが優先され追加効果は消失する（りんぷんを持っていても同様）。
- すばやさ補正との関係: 速度を変更しない特性では行動順に直接影響しない。

## 5. テスト項目（実装検証用）
- 発動ケース: 追加効果付き技を受けたとき追加効果が発生しないこと。なげつけるの追加効果も防げること。
- 非発動ケース: かたやぶりの技では追加効果を防げないこと。ダイマックスわざの追加効果は防げないこと。
- 境界ケース: ちからずくとの競合、オボンのみ等の防御側メリット効果の無効化確認。
