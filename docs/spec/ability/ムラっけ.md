# 仕様書: ムラっけ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%A0%E3%83%A9%E3%81%A3%E3%81%91

ムラっけ は、ポケモンのとくせいの一種。
## 効果
ターン 終了時、自分の こうげき ・ ぼうぎょ ・ とくこう ・ とくぼう ・ すばやさ の内、いずれか1つの ステータス が 2段階 上がり、別のいずれか1つのステータスが1段階下がる。
第七世代 以前では、 命中率 ・ 回避率 も上下するステータスとして選ばれることがある。
## 説明文
- 第五世代 ・ 第六世代: のうりょくが あがったり さがったりする。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは ターンの おわりに のうりょくの 1つが ぐーんとあがり のうりょくの 1つが さがるんだ
- 第七世代 ・ 第八世代 ・ 第九世代: まいターン のうりょくの どれかが ぐーんと あがって どれかが さがる。
## 特性の仕様
- すでに+6の能力を上げたり、-6の能力を下げたりする変化は選ばれない。
  - 上げる能力に+5の能力が選ばれ、結果1段階しか上がらないことは起こりうる。
- 下がる能力はそのターン上がらなかった能力から選ばれる。1ターンに同じ能力が上下することはない。
- 対象となる全能力が+6まで上がっている場合、能力が下がる効果だけ発動する。ムラっけだけで、対象となる能力を全て最大まで上げることはできない。
- 対象となる全能力が-6まで下がっている場合、能力が上がる効果だけ発動する。
- しろいきり / フラワーベール / クリアチャーム の効果では、能力の下降を防ぐことはできない。
- 発動は ターン の終了時となる ( ターン#5.ターン終了時の処理 も参照)。
  - 他のムラっけや、特性 かそく / ナイトメア / スロースタート の解除、 ものひろい / しゅうかく / たまひろい / はんすう 、持ち物 かえんだま / どくどくだま / くっつきバリ とは、 補正を考慮したすばやさ が高い順に発動する。すばやさが変動するときでも、変動前のすばやさを比較する。
  - トリックルーム 状態の効果も考慮する。トリックルーム状態が解除されるターンでは、トリックルームが解除された後にムラっけの発動判定があるため、トリックルームの影響は無い。
- 場に出てから一度も行動していないときでもムラっけは発動する ( かそく の仕様とは異なる)。
  - 交代 によりムラっけのポケモンを繰り出した場合や、 使うと交代する技 、 相手を交代させる技 、 にげごし / ききかいひ 、 だっしゅつボタン / レッドカード / だっしゅつパック の効果でムラっけのポケモンが出てきた場合、そのターンからムラっけは発動する。
  - ひんし になったポケモンの代わりのポケモンを繰り出すタイミングは、ムラっけの発動タイミングの後になる。
## 備考
- 隠れ特性 限定の特性となる。
