# 仕様書: ぜったいねむり

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%9C%E3%81%A3%E3%81%9F%E3%81%84%E3%81%AD%E3%82%80%E3%82%8A

## 2. 基本効果
このポケモンは常に「夢うつつ」の状態となり、状態異常・ねむけ状態を無効化する。ただし、内部的には「ねむり状態として扱われる挙動」を持つ特殊な状態である。

## 3. 詳細仕様

### 状態異常・ねむけ無効化
- すべての状態異常を受けない（ねむり・どく・まひ・やけど・こおり）
- ねむけ状態にもならない
- どくどくだま・かえんだまは発動しない
- どくびしによるどく状態も無効

### ねむり扱いの内部状態
- 内部的には「ねむり状態として扱われる」ケースが存在する
- 一部の技・特性・アイテムはねむり状態を参照して効果を発揮する

#### ねむり参照の例
- いびき・ねごと：使用可能
- めざましビンタ：威力2倍
- たたりめ・ひゃっきやこう：ねむり時と同様の倍率（条件一致時）
- ドリームボール：捕獲補正4倍

### 行動制約の特殊仕様
- 通常のねむりと異なり、技選択・行動は可能
- 「ぐうぐう眠っている」等の行動不能状態にはならない
- ただし「ねごとで引いたあばれる・ころがる」は行動制限を発生させる

### 無効化不能
- この特性は無効化されない
- かがくへんかガス等でも無効化されない
- スキルスワップ・なかまづくり・トレースなどでコピー不可
- いえき・コアパニッシャー等でも特性なし状態にならない

### 状態異常回復との関係
- カゴのみ・ラムのみは発動しない
- 回復系技（アロマセラピー等）で「ねむり扱い部分」を解除できない
- ただし状態異常治療効果は通常通り適用される場合がある

### へんしんとの関係
- へんしんしても効果は維持される
- ねむけ状態は引き継がれるが、ねむり化は防がれる

### 天候・特性相互作用
- かたやぶり等の特性貫通効果でも無視できない

### 交代・入場時
- 場に出た瞬間に「夢うつつ状態」メッセージが表示される
- 常時発動型パッシブ特性

## 5. テスト項目（実装検証用）

### 発動ケース
- 状態異常・ねむけを完全に無効化する
- ねむり参照技（いびき・ねごと等）が使用可能
- 一部技のねむり依存効果が発動する

### 非発動ケース
- カゴのみ・ラムのみは発動しない
- 特性コピー・無効化は一切できない
- 状態異常としてのねむりは成立しない

### 境界ケース
- 「ねむり扱い」と「状態異常としてのねむり」が分離して処理される
- 変身・コピー時に内部状態の扱いが変化する
- 一部技のみねむり参照として動作する
