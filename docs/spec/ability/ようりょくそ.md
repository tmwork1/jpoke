## 2. 基本効果（1段落）
天気がにほんばれ/おおひでりのとき、すばやさが2倍になる。

## 3. 詳細仕様
- 発動条件: 晴れ系天候かつ特性が有効なときに適用。
- 補正対象: 実効すばやさに乗算され、行動順へ反映される。
- 抑制要因: ノーてんき/エアロックで天候効果が無効化中は発動しない。
- アイテム相互作用: ばんのうがさ所持時は晴れ依存効果を受けない。
- 行動順更新: 第8世代以降はターン途中の天候・特性変化を同ターンに反映。

## 4. 処理順・優先度
- 適用タイミング: 行動順計算時。
- 同時発動時: すばやさ補正計算に統合。
- 状態異常との関係: まひ等を含む最終すばやさ計算で処理。

## 5. テスト項目（実装検証用）
- 発動ケース: 晴れで先手が変化する。
- 非発動ケース: ノーてんき/エアロック下で発動しない。
- 境界ケース: ターン中の晴れ化で行動順再評価が入る。
