# 仕様書: もらいび

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%82%E3%82%89%E3%81%84%E3%81%B3
調査日: 2026-04-30

## 2. 基本効果（1段落）
ほのおタイプ技を無効化し、以後そのポケモンが使うほのおタイプ技の威力を1.5倍にする（交代で解除）。

## 3. 詳細仕様
- 発動条件: 対象に向けられたほのおタイプ技に対して判定。
- 効果内容: 被弾時は無効化し、内部の「もらいび状態」を付与。
- 強化持続: もらいび状態は交代で解除。再発動しても重複増加しない。
- 対象外: にほんばれ等の場効果には反応しない。
- 無効化条件: かがくへんかガス中は無効化・強化とも発生しない。かたやぶり技は無効化できない。

## 4. 処理順・優先度
- 適用タイミング: 被弾時のタイプ無効判定。
- 強化適用: 攻撃側がほのお技を選んだダメージ計算時に威力補正として適用。
- 防御技との関係: まもる等で技を防いだ場合は発動しない。

## 5. テスト項目（実装検証用）
- 発動ケース: ほのお技を無効化して次回ほのお技が1.5倍。
- 非発動ケース: かたやぶりほのお技、かがくへんかガス中。
- 境界ケース: 再発動で倍率据え置き、交代で解除。
