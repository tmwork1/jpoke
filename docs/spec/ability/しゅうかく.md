# 仕様書: しゅうかく

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%97%E3%82%85%E3%81%86%E3%81%8B%E3%81%8F

## 2. 基本効果
消費したきのみが、ターン終了時に再び手元に戻ることがある。

## 3. 詳細仕様

### 発動条件
- きのみを消費した後のターン終了時に判定される
- 場にいる状態である必要がある

### 発動確率
- 通常時：50%の確率で発動する
- 天候が「ひざしがつよい」状態：100%の確率で発動する

### 効果内容
- その戦闘で消費したきのみを再生する
- 再生されたきのみは再び使用可能状態になる

### 発動タイミング
- ターン終了時に判定される常時効果

### 無効化・上書き
- 特性が無効化されている場合は発動しない
- 特性のコピー・交換は可能

### 交代・入場時
- 場にいる間のみ判定される
- 交代で場を離れた場合はその時点で効果判定対象外

## 5. テスト項目（実装検証用）

### 発動ケース
- きのみ消費後、一定確率で再生する
- ひざしがつよい状態では必ず再生する

### 非発動ケース
- きのみを消費していない場合は発動しない
- 特性無効化状態では発動しない

### 境界ケース
- 同一ターン内で複数きのみはそれぞれ個別判定される
- 天候変化により確率が即時変動する
