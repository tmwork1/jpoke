# 仕様書: すいすい

調査日: 2026-04-30

## 2. 基本効果（1段落）
天気があめ/おおあめのとき、すばやさが2倍になる。

## 3. 詳細仕様
- 発動条件: 天気があめ系で、特性が有効なときに適用。
- 補正対象: 行動順判定に使う実効すばやさに乗算される。
- 抑制要因: ノーてんき/エアロックで天候効果が無効化される間は発動しない。
- アイテム相互作用: ばんのうがさ所持時は天候依存の特性効果を受けない。
- 行動順更新: 第8世代以降はターン途中の天候変化・特性変化が同ターンの行動順に反映される。

## 4. 処理順・優先度
- 適用タイミング: 行動順計算時に有効判定する。
- 同時発動時: すばやさ補正の一部として他補正と合成して評価。
- 状態異常との関係: まひ等によるすばやさ補正と同一計算系で処理。

## 5. テスト項目（実装検証用）
- 発動ケース: 雨で行動順が逆転する。
- 非発動ケース: ノーてんき/エアロック存在下で2倍化しない。
- 境界ケース: ターン途中で雨化した際の行動順更新（第8世代以降想定）。
