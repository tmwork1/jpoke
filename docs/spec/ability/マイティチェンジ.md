# 仕様書: マイティチェンジ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%A4%E3%83%86%E3%82%A3%E3%83%81%E3%82%A7%E3%83%B3%E3%82%B8

マイティチェンジ は、ポケモンのとくせいの一種。
## 効果
「ナイーブ フォルム 」の イルカマン が他のポケモンに 交代 したとき、「マイティフォルム」に フォルムチェンジ する。
## 説明文
- 第九世代: 手持ちに ひっこむと マイティフォルムに 変化する。
## 所有ポケモン：そのポケモンの別のとくせい
| ポケモン | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| イルカマン | マイティチェンジ | - | - |
## こんなときに使おう
苦手な相手と対面した時に交代したり、交代効果のある技・道具と組み合わせれば条件を発動しやすい。
## 特性の仕様
- 使うと交代する技 / 相手を交代させる技 や、 だっしゅつボタン / だっしゅつパック / レッドカード の効果で交代したときも発動する。
- 手持ちに戻った時点で フォルムチェンジ は行われる。
- ナイーブフォルムのイルカマンが ひんし になって場から去ったときは発動しない。
- マイティフォルムのイルカマンはひんしになってもナイーブフォルムに戻らない。戦闘中に復活した場合、次に場に出たときもマイティフォルムのままとなる。
- マイティフォルムのイルカマンが交代してもフォルムチェンジしない。
- その戦闘で初めてマイティフォルムの姿で場に出るときのみ、特性バーと「<ポケモン名>は 変身して 帰ってきた!」というメッセージが出る。
  - 2回以上マイティフォルムの姿で場に出ても、特性バーやメッセージは現れなくなる。また、登場時のポーズとエフェクトも最初に場に出たときと異なったものになる。
  - 複数のポケモンと同時に場に出たとき、 いかく などの多くの 場に出たときに発動する特性 とは現在の すばやさ が高い順に発動する。
  - 設置技 の効果より後にメッセージの発動判定がある。設置技のダメージを受けて「変身して 帰ってきた!」のメッセージが出る前にひんしになった場合、次に復活して場に出たときでも最初に出たときと同じ扱いになり、特性バーとメッセージが出る。
- この特性は 上書きできない 。 とくせいなし 状態になることや、 かがくへんかガス で無効化されることも無い。
- この特性は コピーできない 。
- この特性を スキルスワップ で交換することはできない。
## 備考
- イルカマン の 専用特性 。
- 手持ちがイルカマン1匹だけの場合、手持ちに戻せないため発動できない。
