# 仕様書: てきおうりょく

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A6%E3%81%8D%E3%81%8A%E3%81%86%E3%82%8A%E3%82%87%E3%81%8F

てきおうりょく は、ポケモンのとくせいの一種。
## 効果
タイプ一致ボーナス による ダメージ 補正率が2.0倍になる。
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代: タイプが おなじ わざの いりょくが あがる。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは じぶんと おなじ タイプのわざを だしたときの あがる ダメージが さらに たかくなるんだ
- 第七世代 ・ 第八世代 ・ 第九世代: じぶんと おなじ タイプの わざの いりょくが あがる。
## 特性の仕様
- タイプ一致ボーナスが発動する状況は タイプ一致ボーナス#細かい仕様 を参照とする。
- 基本的に、タイプ一致ボーナスが2.0倍になる効果と考えてよい。
- テラスタル 状態では、基本的にはテラスタイプの技を使ったときのみ特性の効果が発動する ( ステラ にテラスタルしたときを除く)。
  - テラスタイプが元のタイプとも一致している場合、その技のタイプ一致ボーナスは2.25倍になる。
  - テラスタイプが元のタイプとは一致していない場合、その技のタイプ一致ボーナスは2.0倍になる。
  - テラスタルにより失ったタイプの技にてきおうりょくの効果は発動しない。その技のタイプ一致ボーナスは、テラスタル前は2.0倍だったところが、テラスタル中は1.5倍に下がることになる。
  - ステラ タイプにテラスタルした場合、てきおうりょくの効果は発動しなくなる。ステラ補正の有無や元のタイプに関係なく、特性の無いポケモンと同じダメージしか出せなくなる。
## 備考
- 通常時では、タイプ一致技のダメージが4/3倍 (1.33倍) になる効果ともいえる。内部処理上のタイプ一致ボーナスは、使用者の特性がてきおうりょくなら8192/4096倍 (2.0倍) で、そうでなければ6144/4096倍 (1.5倍) になるという処理がされている。
  - テラスタル 時では、テラスタイプと元のタイプの両方と一致した技のダメージが9/8倍 (1.125倍) になる効果ともいえる。
