# 仕様書: ダルマモード

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%80%E3%83%AB%E3%83%9E%E3%83%A2%E3%83%BC%E3%83%89

ダルマモード は、ポケモンの とくせい の一種。
## 効果
自分の HP が最大HPの1/2以下になると、その ターン 終了時に ダルマモード へ変化する。回復してHPが1/2を超過すると、ターン終了時に元のすがたに戻る。
## 説明文
- 第五世代 ・ 第六世代: ピンチに なると すがたが へんかする。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ヒヒダルマは たいりょくが はんぶんいかに なると ダルマモードに へんか するんだ ダルマモードに なると もとの ほのおタイプに くわえ エスパータイプが ついか されるんだ さらに! こうげきと すばやさの のうりょくが さがる かわりに ぼうぎょ とくこう とくぼうの のうりょくが あがるんだ
- 第七世代 ・ 第八世代 ・ 第九世代: HPが はんぶん いかに なると すがたが へんかする。

## こんなときに使おう
通常の姿のヒヒダルマとガラルヒヒダルマではダルマモード時の種族値が異なる。通常の姿では こうげき と とくこう が入れ替わり、 すばやさ の低下と引き換えに耐久面が上昇する。ガラルヒヒダルマはこうげきとすばやさが上昇するが、他の能力に変化はない。いずれも HP には変化がなく、合計種族値は通常時よりも高い。
ヒヒダルマの通常特性はいずれもデメリットと引き換えに火力を上げる特性であり、ダルマモードを選ぶと通常特性のときよりも火力を落とすことになる。その為、通常特性にはない他の能力の変化やデメリットの有無で差別化を図りたい。
## 特性の仕様
- フォルムチェンジ を発動させる特性の1つ。
  - フォルムチェンジ前で タイプ や ステータス の実数値が変更されていた場合、フォルムチェンジと同時にタイプとステータスがフォルム本来のものに再設定される。
- フォルムチェンジするタイミングはターン終了時、 場の状態 / 状態異常 / 状態変化 / もちもの の効果が発動した後になる。( ターン#5.ターン終了時の処理 を参照)
  - みらいよち / はめつのねがい はフォルムチェンジ前の姿で受ける。
  - すばやさに関係なく、特性の発動は かそく / ムラっけ / しゅうかく / ものひろい / ナイトメア より遅い。 はらぺこスイッチ よりは早い。
  - スワームチェンジ / リミットシールド / ぎょぐん とは同じ優先順位で発動する。 すばやさ 補正は考慮せず、 ボール から出るときのフォルムのすばやさ実数値が発動順に影響する。
- ダルマモードの状態で 交代 するとダルマモードは解除され通常の姿に戻る。
  - HP1/2以下の状態で場に出ても直後にダルマモードにはならない。発動にはターンの終了まで待つ必要がある。
- ダルマモードに対して トレース は発動しない。
- へんしん / かわりもの により、他のポケモンが特性ダルマモードのヒヒダルマに変身することは可能だがフォルムチェンジはできず、変身した際のフォルムを維持する。
- ダイマックス しているときも、ダイマックス後のHPの割合によって特性が発動する。
### 第五世代・第六世代
- この特性は 上書きできる 。 とくせいなし 状態にもなる。
- とくせいなし 状態にされたときや、他の特性に書き換えられると フォルムチェンジ できなくなる。すでにフォルムチェンジしダルマモードになっているときは解除され通常のフォルムに戻る。
- この特性は コピーできる 。
- この特性は スキルスワップ で交換することができる。
- 他のポケモンがこの特性を手に入れてもフォルムチェンジは発動しない。
### 第七世代以降
- この特性は 上書きできない 。 とくせいなし 状態になることや、 かがくへんかガス で無効化されることもない。
  - とくせいなし状態を バトンタッチ により引き継いだ場合、とくせいなし状態が消える。
- この特性は コピーできない 。
- この特性を スキルスワップ / さまようたましい で交換することはできない。
## 備考
- ヒヒダルマ の 専用特性 。
- 隠れ特性 せいしんりょく を持つ ダルマッカ が特性ダルマモードを持つヒヒダルマに進化する。通常特性のヒヒダルマからせいしんりょくを持つダルマッカは生まれない。
  - 入手方法については ダルマッカ#隠れ特性の入手方法 、 ヒヒダルマ#隠れ特性の入手方法 を参照。
