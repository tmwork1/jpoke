# 仕様書: ダウンロード

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%80%E3%82%A6%E3%83%B3%E3%83%AD%E3%83%BC%E3%83%89

ダウンロード は、ポケモンの とくせい の一種。
## 効果
場に出たときの相手の 防御 と 特防 の値を比較して、防御＜特防なら自分の 攻撃 が、防御≧特防なら 特攻 が 1段階 上がる。
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代: あいての のうりょくを みて つよさを かえる
- ブラック2・ホワイト2 （ チェレン ）: この とくせいの ポケモンは せんとうに でたときに あいての ぼうぎょと とくぼうを くらべて ぼうぎょが ひくければ こうげきの とくぼうが ひくければ とくこうの どちらかの のうりょくが あがるんだ
- 第七世代 ・ 第八世代 ・ 第九世代: あいての ぼうぎょと とくぼうを くらべて ひくい ほうの のうりょくに あわせて じぶんの こうげきか とくこうを あげる。
## 所有ポケモン：そのポケモンの別のとくせい
|  | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| ポリゴン | トレース | ダウンロード | アナライズ |
| ポリゴン2 | トレース | ダウンロード | アナライズ |
| ポリゴンZ | てきおうりょく | ダウンロード | アナライズ |
| ゲノセクト | ダウンロード | - | - |
## こんなときに使おう
相手に依存する効果だが、覚えている技に合う能力が上昇すれば アナライズ や てきおうりょく より与えるダメージは高くなる。 特殊技 しか覚えていないと攻撃が上がったとき特性が無意味になるため、 おんがえし ・ からげんき などの 物理技 も覚えた両刀型も視野に入る。
## 特性の仕様
- 値を比較する際、 防御 ・ 特防 補正の効果については、以下のように扱う。
  - ランク補正 / パワートリック / ガードシェア の効果を考慮する
  - 持ち物 / 特性 / すなあらし / ゆき の効果を除外する。
  - ワンダールーム 状態のときは、相手の防御と特防のランク補正のみ入れ替えた能力を比較する。(防御×特防ランク補正率)＜(特防×防御ランク補正率)なら攻撃が、そうでなければ特攻が上がる。ワンダールーム状態では有効でない方の能力が上がることになる。
- ダブルバトル などで相手の場に複数のポケモンがいる場合、相手全員の防御と特防をそれぞれ足し上げた値を比較する。
  - トリプルバトル の場合、自分から対角の位置にいる相手も含めた全員の能力を合計して比較する。
- みがわり 状態の相手は、 第四世代 のみダウンロードの対象にならない。 第五世代 以降はダウンロードの対象になる。
- スキルスワップ などでダウンロードの特性を手に入れた場合、その瞬間にダウンロードが発動する。
  - 第四世代のみ、場に出て一度ダウンロードを発動させたことがある場合、再度入手しても発動しない。第五世代以降では、すでに発動させたことがあるときでも発動する。
- 特性が発動する時点ですでに上がる能力が最大である場合、特性は発動しない。例えば バトンタッチ で最大まで上がった能力を引き継いだときや、能力を最大まで上げたポケモンがスキルスワップなどでダウンロードを手に入れたときなどに起こり得る。
- 相手の場にポケモンがおらず特性が発動できなかった場合、次のポケモンが繰りだされた後もダウンロードは発動しない (特性 トレース の仕様とは異なる)。例えば とんぼがえり で相手を倒して、技の交代先としてダウンロードのポケモンを出したときなどに起こり得る。
  - こちらと相手がお互い死に出しでポケモンを同時に繰りだす場合は、繰り出された順番に関係なくダウンロードは発動する。
- 他の 場に出たときに発動する特性 を持つポケモンと同時に場に出た場合、 第四世代 では素早さに関わらず発動順が決められているが、 第五世代 以降では多くの特性と同じ優先順位で、素早さ順に発動する ( ターン#1.ポケモンを繰り出す も参照)。
  - 相手の かわりもの の メタモン と同時に出た場合、相手が遅ければダウンロードはメタモンの能力を比較して能力を上げ、メタモンがダウンロードのポケモンに 変身 すると、上がった能力をコピーされつつダウンロードを発動させる。ダウンロード側が遅ければ、変身先のポケモンの能力を比較する。
  - 味方の きみょうなくすり と同時に出た場合、きみょうなくすり側が遅ければダウンロードで上がった能力は即座にリセットされる。ダウンロード側が遅ければ上がった能力がリセットされることは無い。
- かがくへんかガス が発動している間は発動しない。
  - かがくへんかガスのポケモンと同時に場に出た場合、素早さに関わらずかがくへんかガスが先に発動してダウンロードは発動しない。
  - 相手の場のかがくへんかガスが 交代 してガスが解除される場合、かがくへんかガスのポケモンの能力を比較して能力を上げる。
  - 相手の場のかがくへんかガスが ひんし になってガスが解除される場合、その時点で相手の場にいるポケモンの能力を比較して能力を上げる。相手の場にポケモンがいなければ発動しない。
- しっとのほのお / みわくのボイス による 追加効果 は、バトル開始時に能力を上げて1ターン目に受けた場合や、他のポケモンから交代して出したことで特性を発動させたターンに受けた場合に発動する。死に出しで攻撃を上げた直後に受けた場合は発動しない。
## 備考
- 防御 と 特防 の 種族値 が近い伝説ポケモン等を 厳選 する場合、ダウンロードの対象にすることで 個体値 や 性格 をある程度推測することができることがある。
- 対人戦において、防御と特防の種族値が同程度のポケモンなどは、対戦相手のダウンロードで特攻を上げられることを警戒して、防御より特防が少しだけでも高くなるように調整されることがある。
- 第九世代現在、この特性を持つポケモンは 性別不明 のみである。
  - 特性名にちなんでか、この特性を所有するポケモンは人工的に開発・改造された経緯があるものとなっている。
