# 仕様書: きょううん

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%8D%E3%82%87%E3%81%86%E3%81%86%E3%82%93
調査日: 2026-04-30

## 効果

きょううんは、ポケモンのとくせいの一種。

使う技の急所ランクが+1される。

## 説明文

- **第七世代・第八世代・第九世代** 強運を 持っているため 相手の 急所に 攻撃が 当たりやすい。

## 特性の仕様

- 急所率の計算については急所#急所率の計算を参照。
- 第八世代でのみ、手持ちの先頭にこのとくせいを持つポケモンがいると、道具を持った野生ポケモンが出やすくなる（50%→ 60%、5%→ 20%）。

## 備考

- 第八世代でのみ同じフィールド効果を持つ特性にふくがんがある。

## 関連項目

- スナイパー（急所のダメージ倍率幸が上がる）
- ぴんとレンズ/するどいツメ（急所ランクを上げる道具）
- 発動ケース: 「技の急所ランクを+1する」が成立する条件で期待どおりに発動すること。
- 非発動ケース: 条件を満たさない場合や無効化条件下で発動しないこと。
- 境界ケース: 交代直後、特性無効化、同時処理競合時に破綻しないこと。
