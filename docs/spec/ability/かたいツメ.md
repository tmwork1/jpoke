# 仕様書: かたいツメ

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%8B%E3%81%9F%E3%81%84%E3%83%84%E3%83%A1

## 2. 基本効果（1段落）
接触判定を持つ技を使用したとき、その技の威力が約1.3倍に上昇する。ダメージ計算時の威力補正として適用される。

## 3. 詳細仕様

### 威力補正
- 接触技にのみ適用
- 補正値：5325/4096（約1.3倍）
- ダメージ計算の威力補正として処理

### 接触判定
- 「接触技」であることが条件
- 物理技に限らず、接触判定を持つ技すべてが対象
- 技がみがわりに防がれても補正は適用される

### アイテムとの相互作用
- ぼうごパット所持時：
  - 接触による追加効果は発生しないが、本特性の威力上昇は適用される
- パンチグローブ所持時：
  - パンチ技は非接触扱いとなるため補正が適用されない
  - パンチ技以外の接触技には適用される

### 特性書き換え系との相互作用
- ミイラ／とれないにおい／さまようたましい：
  - ダメージ計算後に特性が書き換えられるため、その攻撃には補正が適用される
- 連続攻撃技の場合：
  - 書き換え後のヒットには補正が適用されない
- 全体攻撃技でも各対象へのダメージに補正が適用される

### 無効化・上書き
- 特性が無効化されている場合（例：かがくへんかガス）は威力上昇しない
- かたやぶり等の影響は受けない（自己能力補正のため）
- 特性のコピー・交換は可能

### 交代・入場時
- 常時発動型特性
- 入場時の即時効果なし
- 場にいる間のみ有効

### その他の特性との相互作用
- 技威力補正系特性（てつのこぶし等）と同様にダメージ計算へ乗算で適用される
- タイプ一致補正や持ち物補正などと組み合わせて最終威力が決定される

## 4. 処理順・優先度
- 発動タイミング：ダメージ計算時（威力決定段階）
- 処理順：
  1. 技威力決定
  2. 接触判定確認
  3. かたいツメ補正適用
  4. 他威力補正と合算
- 同時発動時の優先度：
  - 他の威力補正と同様に同一段階で乗算処理
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- 接触技でダメージが約1.3倍になる
- みがわり越しでも補正が適用される
- 全体攻撃でも各対象に補正がかかる

### 非発動ケース
- 非接触技では補正がかからない
- パンチグローブ装備時のパンチ技で補正がかからない
- 特性無効化状態で補正がかからない

### 境界ケース
- ミイラ等で特性が途中で変化しても初撃に補正がかかる
- 連続攻撃中に特性変化後のヒットに補正がかからない
- 他威力補正と正しく乗算される
