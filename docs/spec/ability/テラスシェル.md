# 仕様書: テラスシェル

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%83%86%E3%83%A9%E3%82%B9%E3%82%B7%E3%82%A7%E3%83%AB
調査日: 2026-05-02

## 2. 基本効果（1段落）
HPが満タンのとき、等倍以上でダメージを受ける攻撃技の相性がすべて今ひとつ（1/2）になる

## 3. 詳細仕様
- 発動条件: 自分のHPが最大値のとき、等倍以上の相性でダメージを受ける攻撃技を受けると、相性判定を1/2に書き換えて処理する。ダメージ乗算ではなく相性そのものを変更する。
- 相性で無効となる技（0倍）を受けたときは発動しない（そのまま無効扱い）。
- ダメージ固定技（一撃必殺技含む）には発動しない（演出は出るが実際のダメージは減算されない）。
- みがわり状態でも本体のHPが満タンであれば発動する。
- 一度HPが減少しても、回復してHP満タンになれば何度でも再発動する。
- 連続攻撃技に対しては、HPが満タンであれば全ての攻撃を今ひとつで受ける（相性は1発目でしか判定しないため）。
- かたやぶり・特性を無視する技には発動しない。
- ステルスロックで受けるダメージは減少しない。
- こんらんの自滅ダメージには発動しない。
- 無効化・上書き: かがくへんかガスで無効化される。スキルスワップによるコピー・移譲は不可。

## 4. 処理順・優先度
- 発動タイミング: ダメージ計算時の相性判定フェーズで相性を1/2に書き換える。
- マルチスケイル/ファントムガードと異なり、相性補正を直接変更するため、たつじんのおびや弱点保険などの「相性抜群」トリガーも無効になる。

## 5. テスト項目（実装検証用）
- 発動ケース: HP満タンで攻撃技を受けたとき、相性が1/2として計算されダメージが半減すること。
- 非発動ケース: HP満タンで本来無効な技を受けたとき今ひとつにならず無効のままであること。ダメージ固定技では減算されないこと。かたやぶりの技には発動しないこと。
- 境界ケース: HP1でも回復して満タンになれば再発動すること。かがくへんかガス下では発動しないこと。
