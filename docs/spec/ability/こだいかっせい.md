# 仕様書: こだいかっせい

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%93%E3%81%A0%E3%81%84%E3%81%8B%E3%81%A3%E3%81%9B%E3%81%84

こだいかっせい は、ポケモンのとくせいの一種。
## 効果
場が にほんばれ 状態のときか、 ブーストエナジー を持っているとき、自分の ランク補正 込みで最も高い能力が上がる。
- 攻撃 ・ 防御 ・ 特攻 ・ 特防 の場合は1.3倍になる。
- 素早さ の場合は1.5倍になる。
最も高い能力が複数ある場合は、攻撃・防御・特攻・特防・素早さの順で最初にあるものが選ばれる。
## 説明文
- 第九世代: ブーストエナジーを 持たせるか 天気が 晴れのとき いちばん 高い能力が 上がる。

## こんなときに使おう
ステータス 次第で上げたい能力を変えられる。アタッカーは こうげき ・ とくこう ・ すばやさ 、耐久型は ぼうぎょ ・ とくぼう のどれかを上げるように調整されることが多い。似た特性の ビーストブースト とは違って ランク補正 の影響を受けて上がる能力が変化する。
発動条件は2つあるがどちらも一長一短となる。 ブーストエナジー を持てばほぼ確実に発動できるが、一回しか発動できない上に持ち物が固定される。 にほんばれ 状態でなら持ち物は自由だが、場の準備がいる上に天候が変わると効果も消える。

## 特性の仕様
- ブーストエナジー#詳細な仕様 も参照。
- この特性は コピーできない 。
  - 使用者か対象のどちらかがこの特性であるとき、 スキルスワップ は失敗する。
- 他のポケモンが へんしん してこの特性を得たとしても、特性は発動しない。
  - 特性の書き変えやコピーを防ぐ効果はへんしん状態のポケモンでも発動する。
  - ブーストエナジーの譲渡を防ぐ効果はへんしん状態のポケモンには無い。
- スカーレット・バイオレット (Ver.3.0.0) 以降では、この特性を 上書きする ことはできる。 とくせいなし 状態や かがくへんかガス によって無効化することもできる。
  - かがくへんかガスの効果が切れたときは、交代先のポケモンが出てきた後に特性の発動判定がある ( #発動条件 節も参照) 。
  - 特性が変更されたり、無効化されたりしているときでも、 ウガツホムラ / タケルライコ を除く パラドックスポケモン に対して ブーストエナジー が渡る効果や、奪う効果は無効となる。
  - Ver.2.0.1まで、この特性は 上書きできない 特性だった。 とくせいなし 状態や かがくへんかガス の効果でも無効化されなかった。
### 発動条件
- にほんばれ 状態で特性が発動したときは「<ポケモン>は にほんばれで こだいかっせいを 発動した!」というメッセージが出て能力が上がる。にほんばれ状態の効果が切れると「<ポケモンは> こだいかっせいの 効果が 切れた!」とメッセージが出て補正が消える。
- ブーストエナジー で発動したときは「<ポケモン>は ブーストエナジーで こだいかっせいを 発動した!」というメッセージが出る。この効果は 交代 で消える。
- この特性が既に発動している場合、もう1つの発動条件を満たしても重複して発動することはない。
  - にほんばれ状態の場にブーストエナジーを持って出た場合、基本的にはにほんばれ状態による特性の発動が優先されてブーストエナジーは消費しない。この後にほんばれ状態が消滅すると一旦特性の効果が消えるが、即座にブーストエナジーを消費して特性を再発動させる。再発動の際上がる能力はそのとき一番高い能力になる。
  - ブーストエナジーを消費して発動した場合、その後にほんばれが発生してもさらに特性が発動することはない。
  - Ver.3.0.0で、ブーストエナジーを発動させた ウガツホムラ / タケルライコ が再びブーストエナジーを手に入れても、場にいる限りは持ち物を再発動させることはない。
- 設置技 の効果より後にこだいかっせいの発動判定がある。
  - ねばねばネット 状態の効果を受けたときは、変動後の能力を比較して上がる能力が決まる。
  - Ver.2.0.2以前では、特定の条件で設置技を受けたとき、こだいかっせいの発動順が繰り上がることがあった。 #備考 節参照。
- にほんばれ状態の場に出たときや、にほんばれ状態でない場にブーストエナジーを持って出たときは、 いかく など多くの 場に出たときに発動する特性 より遅い優先順位でこだいかっせいが発動する。
  - 複数のポケモンと同時に場に出た場合、 アイスフェイス / きょうえん / しれいとう / クォークチャージ とは補正抜きの 素早さ 実数値が高い順に発動する。発動順に こだわりスカーフ などの持ち物や トリックルーム 状態などの 場の状態 の効果は考慮しない。
  - 持たせた シード系アイテム / ルームサービス に対応した場に出たときは、持ち物が先に発動する。
  - 場に出てから特性発動までの間に いかく 、持ち物などの効果でランクが変動した場合、特性発動時点の能力とランク補正を参照して上がる能力が決まる。
  - 特性が発動した後に しろいハーブ / ものまねハーブ の発動判定がある。
  - 場に出てた後で天候がにほんばれに変わった場合、他の処理に割り込んで即座にこだいかっせいが発動する。ブーストエナジーを持っていれば、にほんばれ以外の天候に変わったときも同様にして発動が割り込む。
    - 複数のこだいかっせいのポケモンがいる場合、補正抜きの素早さ実数値が高い順に発動する。
    - ひひいろのこどう で天気が変わった場合、素早さに関係なくこだいかっせいが発動した後に「<ひひいろのこどう所持ポケモン>は ひざしを 強め 古代の鼓動が 暴れだす!!」のメッセージが流れる。
- 姿を隠している 間でも、にほんばれでこだいかっせいは発動する。
- 特性 ノーてんき / エアロック の効果が発動しているとき、にほんばれ状態でこだいかっせいは発動しない。
  - このときブーストエナジーを持っている場合は、にほんばれ状態でもブーストエナジーを消費して能力を上げる。
  - ノーてんき/エアロックのポケモンを含む複数のポケモンと同時に場に出た場合、ノーてんき/エアロックより先に ひでり / ひひいろのこどう のメッセージが出てにほんばれになったとしてもこだいかっせいは発動しない。
  - 一度こだいかっせいが発動すれば、後から場にノーてんき/エアロックのポケモンが出てきても上がった能力が無効化されることはない。
  - にほんばれ状態のためブーストエナジーを所持しているとき、場にノーてんき/エアロックのポケモンが出てきても能力は落ちないのでブーストエナジーは消費しない。実際ににほんばれ状態が消えたときに初めて消費する。
- Ver.3.0.0以降では、 かがくへんかガス の効果が発動しているときにこだいかっせいは発動しない。ブーストエナジーを持っていても消費されない。
  - こだいかっせいで能力が上がった後でも、かがくへんかガスの発動中は特性による能力補正は無効化される。この場合ガスが解除されれば能力補正は即座に復活する。ガスの発動中に最も高いステータスが変わっていたとしても、補正が掛かる能力はガス発動時と同じ箇所となる。
  - テラレイドバトル において、特性発動後に1ターン とくせいなし 状態にされたときも同様にして、とくせいなし状態が解除されれば能力補正は復活する。
- こだいかっせいが未発動の状態でノーてんき/エアロック/かがくへんかガスが発動した場合、これらの特性が解除されることでこだいかっせいの発動条件が満たされたとしても、即座に効果が発動することはない。次にポケモンが繰り出された後でこだいかっせいの発動判定がある。
  - ノーてんき/エアロック/かがくへんかガスのポケモンが 交代 して場から去った場合は交代先が出てきた後、 ひんし になって場から去った場合はそのターンが終了し交代先が出てきた後にこだいかっせいが発動する。
  - ノーてんき/エアロック/かがくへんかガスが、 とくせいなし 状態や他の特性に上書きされることで効果を失った場合、次にポケモンの交代やひんしが発生するまでの間、こだいかっせいは発動できないままとなる。
  - 交代先のポケモンが 場に出たときに発動する特性 を持っていた場合、こだいかっせいとは同時に場に出たときと同様の優先順位で特性が発動する。
  - ダブルバトル で、ひんしになったポケモンの交代先がいなかった場合、次のターンの戦闘コマンドが表示される前にこだいかっせいが発動する。
  - ノーてんき/エアロック/かがくへんかガスが解除されても、特性の発動判定までに再びこだいかっせいの発動条件が満たされなくなると、特性が発動することはない。
- ばんのうがさ を持っている場合でもにほんばれ状態でこだいかっせいは発動し、能力が上がる。
  - ばんのうがさを持つポケモンから攻撃を受けたときでも、こだいかっせいで上がった防御/特防は無視されない。
- 場が マジックルーム 状態である場合、ブーストエナジーは発動しない。
  - マジックルーム状態になった後に場に出た場合も、マジックルームが解除されるとその時点でブーストエナジーが発動する。
### 能力上昇
- 発動条件を問わず、特性が発動すると「<ポケモン>の <攻撃/防御/特攻/特防/素早さ>が 高まった!」というメッセージが流れ、どの能力が上がったのかが分かる。
- 上がる能力に HP は考慮しない。
- 上がる能力を決める際に参照にする数値は以下のように扱う。
  - ランク補正 の効果は考慮する ( ビーストブースト の仕様とは異なる)。
  - パワーシェア / ガードシェア / パワートリック / スピードスワップ / ワンダールーム による実数値変動も考慮する。
  - こだわり系アイテム / とつげきチョッキ / パワー系アイテム / くろいてっきゅう などの持ち物や、 おいかぜ など場の状態の効果は考慮しない。
  - 実数値の変動はステータスの比較より先に考慮する。特攻と特防が同じで通常は特防が上昇するポケモンなら、ワンダールーム状態では防御が上がる。
  - バトンタッチ で引き継がれたランク補正/パワートリック状態は、場に出た瞬間から考慮される。
- 特性が発動した後なら、ランクや実数値が変動し一番高い能力が変わったとしても補正が掛かっている能力は変わらない。
  - 防御か特防が上昇しているときにワンダールーム状態が発生した場合や、解除された場合では、その度に補正が掛かっている能力も入れ替わる。
- 特性の効果はランク補正上昇とは異なる。
  - 攻撃が上昇しても、 イカサマ を受けたときのダメージは変わらない。
  - 攻撃が上昇したとき使用する ボディプレス は防御の数値を1.3倍にしてダメージを与える。防御が上がってもダメージは変わらない。
  - テラスタル 時の テラバースト の分類の決定に特性の効果は考慮されない。ほとんどのケースで攻撃/特攻の低い方が用いられることはないが、特性の発動からテラバーストの使用までにランク/実数値変動が起きていた場合は特性で上がっていない方の能力を用いてテラバーストが使われることがある。
  - 上昇した能力を バトンタッチ で引き継げない。
  - 上昇した能力は てんねん で無視できない。
  - 上がった能力はYボタンの情報画面で確認できない。
- かたやぶり の効果がある技でも、こだいかっせいによる防御/特防補正は無視できない。
- 特性で攻撃/防御が上がっても こんらん のダメージには影響しない。
- 交代 すると特性の効果はリセットされる。
## 備考
- 古代の パラドックスポケモン の 専用特性 ともいえる。
- 英語を筆頭に外国語名称はProto(原始)とPhotosynthesis(光合成)を組み合わせた造語となっている。
  - 名前の由来となった同名の技 こうごうせい は にほんばれ に関連する効果である点と共通している。
- スカーレット・バイオレット Ver.2.0.2までは かがくへんかガス に無効化されなかったが、Ver.3.0.0で無効化されるようになった 。
- 基本的に、特性の発動条件を満たした同じ特性のポケモンが同時に場に出たときはすばやさ順に発動するが、 スカーレット・バイオレット (Ver.2.0.2以前) では特定条件で 設置技 によりダメージや 状態異常 を受けたポケモンは、他の処理に割り込んで即座に特性を発動させることがあった。
  - スカーレット・バイオレット (Ver.1.1.0以前) ではブーストエナジーによる特性の発動しか割り込まない。Ver.1.2.0 - 2.0.2ではにほんばれによる特性の発動も割り込むが、持ち物が無いポケモンの場合は特性の発動は割り込まない。
    - にほんばれ状態でブーストエナジーを持って場に出た場合は、Ver.1.1.0以前ではブーストエナジーを消費して特性が発動する。Ver.1.2.0-2.0.2ではブーストエナジーは消費せず、にほんばれで特性が発動する。
    - Ver.1.2.0 - 2.0.2において、設置技の効果で きのみ / シード系アイテム / ルームサービス などの持ち物が発動するときは、特性が発動した後に持ち物が発動する。 マジックルーム 状態でも持ち物があるなら特性の発動が割り込む。
  - 複数の設置技の効果を受けるときは、1つ目の効果を受けた直後に割り込んで特性が発動する。
  - しんぴのまもり 状態や フラワーベール 、タイプの効果で設置技の効果を防いだときは、特性の発動が割り込むことはない。
  - ねばねばネット 状態の効果を受けたことで特性の発動が割り込むことはない。
  - Ver.3.0.0以降は持ち物の有無に関係無く、設置技により特性の発動が割り込むことは無くなった。
