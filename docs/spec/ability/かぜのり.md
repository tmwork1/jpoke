# 仕様書: かぜのり

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%81%9C%E3%81%AE%E3%82%8A

## 基本効果
おいかぜ状態の場にいるとき、こうげきが1段階上がる。また、風の技を受けても無効化し、こうげきが上がる。

## 詳細仕様

### 発動タイミング
- 味方の場が おいかぜ になったとき
- おいかぜ状態の場に出たとき
- 風の技を受けたとき

### 発動条件
- 敵の おいかぜ では発動しない
- コートチェンジ で味方に おいかぜ が移動してきても発動しない

### 技との相互作用
- ふきとばし を受けたときも発動し、交代を防ぐ
- かたやぶり の技では発動しない
- 命中判定より前に発動判定されるため、技を回避しても特性は発動する

### その他
- みがわり 状態でも発動
- データ上は すなあらし も風に含まれるが、発動しない

## 処理順・優先度
- おいかぜ状態の場に出たとき、いかく など他の入場特性とはすばやさ補正込みで判定

## テスト項目

- [ ] おいかぜ状態でこうげきが上がる
- [ ] 風の技を受けてこうげきが上がる
- [ ] 敵のおいかぜでは発動しない
- [ ] ふきとばしで交代が防がれる
- [ ] かたやぶり の技では発動しない
