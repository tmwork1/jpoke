# 仕様書: くだけるよろい

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%8F%E3%81%A0%E3%81%91%E3%82%8B%E3%82%88%E3%82%8D%E3%81%84

## 2. 基本効果
物理技のダメージを受けたとき、防御が下がり素早さが上がる。

## 3. 詳細仕様

### 発動条件
- 対象：物理攻撃技によるダメージを受けたとき
- ダメージが発生した場合のみ発動（無効・失敗時は発動しない）

### 能力変化
- 防御：1段階下降
- 素早さ：2段階上昇
- ランク変化として処理

### 発動タイミング
- ダメージ計算後に発動
- 連続技の場合：
  - 各ヒットごとに発動判定がある

### 無効化・上書き
- 特性が無効化されている場合（例：かがくへんかガス）：
  - 発動しない
- 能力変化無効状態（しろいきり等）では：
  - 防御低下が無効化されるが、素早さ上昇は適用される
- 特性のコピー・交換は可能

### 交代・入場時
- 常時効果（被弾時トリガー）
- 入場時の即時効果なし

### その他の特性との相互作用
- かたやぶりの影響を受ける（特性を無視される）
- ランク変化補正（シンプル等）と連動する
- 能力変化反転（あまのじゃく）では：
  - 防御上昇・素早さ下降に反転する

## 5. テスト項目（実装検証用）

### 発動ケース
- 物理技被弾時に防御↓・素早さ↑が発生する
- 連続技で複数回発動する

### 非発動ケース
- 特殊技では発動しない
- ダメージを受けない場合（無効・みがわり等）発動しない
- 特性無効化状態で発動しない

### 境界ケース
- 能力変化無効で防御低下のみ無効になる
- あまのじゃくで効果が反転する
- かたやぶりで無視される
