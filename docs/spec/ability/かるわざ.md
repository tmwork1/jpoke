# 仕様書: かるわざ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%8B%E3%82%8F%E3%81%96

かるわざ は、ポケモンの とくせい の一種。
## 効果
場に出た後に自分の 持ち物 が無くなると すばやさ が2倍になる。再び持ち物を手にするとすばやさは元に戻る。場に出たときから持ち物がない場合は発動しない。
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代: どうぐが なくなると すばやさが あがる。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは もっていた どうぐが なくなると すばやさが ばいに なるんだ はじめから もっていないと こうかは はつどう しないから ちゅうい すること
- 第七世代 ・ 第八世代 ・ 第九世代: もっていた どうぐが なくなると すばやさが あがる。

## こんなときに使おう
能動的に消費できる もちもの としては、 第四世代 では パワフルハーブ くらいしかなかったが、 第五世代 では ジュエル が、 第七世代 では シード系アイテム が登場した。特に ダブルバトル において メイカー系特性 を持つポケモンと同時にかるわざを出す戦法が用いられることが多い。
## 特性の仕様
- 一度 持ち物 を消費して、かるわざを発動させてから 交代 で引っ込んだ場合、再び場に出てきても「初めから何も持っていなかった」扱いになり、かるわざは発動しない。
- 場に出たときと同時に持ち物を消費したときはかるわざは発動する。
- 道具を消費する手段は問わない。以下の状況でかるわざは発動する。
  - 普通に道具を消費した場合 ( なげつける / しぜんのめぐみ / ほおばる を使用したときも有効)。
  - どろぼう / ほしがる / トリック / すりかえ 、特性 マジシャン / わるいてぐせ により道具を失った場合。
  - ついばむ / むしくい / はたきおとす / やきつくす / おちゃかい / ふしょくガス を受けて道具が失われた場合。
  - 第四世代 では、はたきおとすの効果は はたきおとす 状態にするだけで持ち物欄から道具は消えないが、かるわざは発動する。
- 発動中に次のように道具を手に入れた場合、素早さは元に戻る。再び消費できれば発動する。
  - どろぼう/ほしがる/トリック/すりかえ/ リサイクル / キョダイサイセイ で道具を持った場合。
  - 他者から ギフトパス 、特性 きょうせい で道具を受け取った場合。
- 道具が無い状態で、相手から なげつける を受けて きのみ や道具を消費してもかるわざは発動しない。
- 道具が無い状態で、 ついばむ / むしくい を使用して相手のきのみを消費してもかるわざは発動しない。
- さしおさえ / マジックルーム 状態では道具を失ったとは見なされず、かるわざは発動しない。
- 持ち物が発動しても、消費されない場合はかるわざは発動しない。
- かるわざが他の特性に書き換わった場合、素早さは元に戻る。
  - 他の特性を持つポケモンが道具を消費した後に、特性がかるわざになった場合、かるわざは発動しない。
  - かるわざが発動中に、かるわざが他の特性に書き換わり、再度かるわざに書き換わった場合、かるわざは発動しない。
  - かるわざが発動中に、かるわざ同士で スキルスワップ をした場合、かるわざは発動しなくなる。
- かるわざが発動後に かがくへんかガス が発動した場合、その間かるわざの効果は無くなる。かがくへんかガスが解除されればかるわざは再度発動する。
  - かるわざが発動していないときにかがくへんかガスが発動し、その間に持ち物を消費してからかがくへんかガスが解除された場合、かるわざは発動しない。
- かるわざによって上昇した素早さは ランク補正 とも 状態変化 とも異なる。この素早さ上昇は バトンタッチ では引き継げない。
  - 素早さがランクで変動している場合、変動後のステータスを2倍にする。 まひ の効果はかるわざやその他の補正の効果を考慮した後に計算する( すばやさ#すばやさ補正 も参照)。
- 他のポケモンの行動によって道具が失われかるわざが発動する場合や、道具を持たされかるわざの効果が消える場合、 第七世代 まではそのターンの行動順は変化しない。 第八世代 からはそのターンから行動順に影響が出る。
## ポケモン不思議のダンジョンにおけるかるわざ
- ポケモン不思議のダンジョン では初めから持たせていなくてもアイテムを持ってさえいなければ発動し、敵に2連続攻撃をするという効果に変わっている。
## ポケモンカードゲームにおけるかるわざ
フワンテLV.16 (DPs-B) の ポケボディー として登場。「「ポケモンのどうぐ」がついているなら、このポケモンのにげるエネルギーは、2個ぶん多くなる」というデメリットに近い効果となっている。
## 備考
- 第四世代 では フワンテ 系統の 専用特性 だった。
