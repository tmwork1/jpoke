# 仕様書: うのミサイル

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%86%E3%81%AE%E3%83%9F%E3%82%B5%E3%82%A4%E3%83%AB

うのミサイル は、ポケモンの とくせい の一種。
## 効果
- この特性のポケモンが使用した なみのり / ダイビング が成功すると、HPが半分超のときは サシカマス を咥えた「うのみのすがた」に、HP半分以下のときは ピカチュウ を咥えた「まるのみのすがた」になる。
- 「うのみのすがた」「まるのみのすがた」のときに 攻撃技 で ダメージ を受けると、獲物を吐き出して通常の姿に戻る。このとき攻撃してきた相手に、相手の最大HPの1/4分のダメージを与える (小数点以下切り捨て。ただし最低でも1以上) 。さらに「うのみのすがた」なら ぼうぎょ を 1段階 下げ、「まるのみのすがた」なら まひ 状態にする効果が付く。
| フォルム名 | フォルムチェンジ時のHP残量 | 咥えるポケモン | 反撃のダメージに付く効果 |
| --- | --- | --- | --- |
| うのみのすがた | 最大HPの1/2超 | サシカマス | ぼうぎょ 1段階 低下 |
| まるのみのすがた | 最大HPの1/2以下 | ピカチュウ | まひ 状態 |
## 説明文
- 第八世代 ・ 第九世代: なみのりか ダイビングを すると えものを くわえてくる。 ダメージを うけると えものを はきだして こうげき。
## 所有ポケモン：そのポケモンの別のとくせい
|  | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| ウッウ | うのミサイル | - | - |
## こんなときに使おう
フォルムチェンジさえできれば、次の攻撃で倒されたとしても相手に負担をかけることができる。
## Pokémon UNITEにおけるうのミサイル
Pokémon UNITE では、 ウッウ の特性として登場する。
## 特性の仕様
- この特性は 上書きできない 。 いえき / コアパニッシャー を受けても とくせいなし 状態にならず、 かがくへんかガス でも無効化されない。 第九世代 でも同様。
- この特性は 第八世代 と スカーレット・バイオレット (Ver.2.0.1 - 2.0.2) では コピーできない 。 スカーレット・バイオレット (Ver.3.0.0 - 3.0.1) ではコピーできる。
- 第八世代 と スカーレット・バイオレット (Ver.2.0.1 - 2.0.2) では、この特性を スキルスワップ / さまようたましい で交換することはできない。 スカーレット・バイオレット (Ver.3.0.0 - 3.0.1) では交換できる。
  - ウッウがこの特性を失ったときは、その時点のフォルムを維持してフォルムチェンジできなくなる。
- この特性は、特性の所有者が ウッウ でなければ発動しない。ウッウ以外のポケモンがこの特性を得てダイビング/なみのりを使用しても特性は発動しない。
  - ウッウ以外のポケモンがウッウに へんしん した場合、変身した時点のフォルムを維持してフォルムチェンジできない。うのみのすがた/まるのみのすがたに変身したときに攻撃を受けても獲物を吐き出さない。
### フォルムチェンジ条件
- 「うのみのすがた」「まるのみのすがた」のときに以下のことが起きると通常の姿に戻る。
  - 自身の 交代
  - 自身の ダイマックス
    - ダイマックス中ダイビングやなみのりを元にした ダイストリーム を使用してもフォルムチェンジはできない。 マックスレイドバトル で敵ダイマックスウッウがなみのりやダイビングを使用してもフォルムチェンジしない。
  - 自身の ひんし
  - 戦闘終了
- 通常の姿から「うのみのすがた」「まるのみのすがた」へフォルムチェンジするタイミングは攻撃が成功したときになる。
  - ダイビング は、1ターン目の溜める行動ができればフォルムチェンジできる。2ターン目の攻撃が まもる 状態や よびみず などで無効化されたときや、レベルアップでダイビングを忘れたときでも関係ない。
    - パワフルハーブ でダイビングを1ターンで出したときも、溜める動作は入るのでまもるやよびみずで無効化されたとしてもフォルムチェンジできる。
  - なみのり がまもる状態やよびみずなどで無効化されたときはフォルムチェンジできない。
    - ダブルバトル などでは場の全てのポケモンになみのりを無効化されるとフォルムチェンジできない。1体にでも有効だった場合はフォルムチェンジできる。
  - 状態異常 / 状態変化 などでダイビングの溜め動作や、なみのりの使用ができなかったときはフォルムチェンジできない。
  - 戦闘アニメが再生されるときは、技のアニメーションが終わったときにすでにフォルムチェンジしている。 第八世代 で戦闘アニメが再生されないときは、姿を隠したりダメージが発生したりする前にフォルムチェンジする。
- 獲物を咥えてからHPが変動しても、咥えている獲物が変わることはない。
- 自身が みがわり 状態であるときでも獲物を咥えてくることはできる。
### 獲物を吐き出すときの効果
- 攻撃技を受けても 相性 / まもる 状態などでダメージを無効化したときや、 命中 しなかったときは獲物を吐き出さない。
  - HP1のときに こらえる を使用したときなど、0のダメージを受けたときは発動して獲物を吐き出す。
- 自身の みがわり 状態で攻撃を防いだときは、みがわりが壊れたかどうかに関係なく獲物を吐き出さない。みがわり状態を無視する攻撃で本体がダメージを受けたときは獲物を吐き出す。
- ダイビング / そらをとぶ 状態などで 姿を隠している 間は、姿を隠しているポケモンにも当たる攻撃技でダメージを受けても獲物を吐き出さない。
- じばく による攻撃を受けたときなど、攻撃者がすでにひんしになっているときは獲物を吐き出さない。
- みらいよち / はめつのねがい の攻撃を受けた場合、攻撃者が場に残っているときは発動して獲物を吐き出す。攻撃者が場にいないときは獲物を吐き出さない。
- 連続攻撃技 を受けた場合、1発目を受けた直後に獲物を吐き出す。
  - 反撃のダメージで攻撃側のHPが0になった場合、その時点で連続攻撃は中断する。
  - 相手をまひ状態にしても連続攻撃が中断されることは無い。
- 攻撃を受けてウッウが ひんし になったときも獲物を吐き出してダメージと効果を与える。
  - 獲物の反撃で相手のHPも0になった場合、ウッウが先にひんしになる。両者が同時に全滅した場合はウッウ側の負けとなる。
- フォルムチェンジにより書き換えられた タイプ や ステータス の実数値は元に戻る。
  - 通常のすがたのウッウのタイプやステータスが変更されている状態でなみのり/ダイビングを使用してフォルムチェンジした場合、 タイプ一致ボーナス や ダメージ は元に戻ったものを参照して計算される。
  - うのみ/まるのみのすがたのウッウのタイプやステータスが変更されている状態で連続攻撃技を受けてフォルムチェンジした場合、 相性 判定は最初の攻撃のものが技が終わるまで継続する。ステータス変更の解除は2発目から即座にダメージに影響する。
- 獲物による反撃は、攻撃ではなく てつのトゲ や せいでんき などと同類の効果であり、以下の特徴がある。
  - 相手の みがわり 状態を貫通してダメージと効果を与える。
  - 特性 マジックガード のポケモンに対してはダメージを与えられないが、ランク低下やまひの効果は発動する。
  - うのミサイルの反撃に対して、うのミサイルや アイスフェイス / ばけのかわ / ふしぎなまもり など攻撃技にしか反応しない特性は発動しない。
- 「うのみのすがた」による反撃には以下の特徴がある。
  - 特性 はとむね / クリアボディ / しろいけむり / メタルプロテクト 、持ち物 クリアチャーム を持つポケモンの防御ランクは下がらない。 ミラーアーマー によりランク低下は跳ね返される。
  - しろいきり 状態に守られているポケモンの防御ランクは下がらない。
- 「まるのみのすがた」による反撃には以下の特徴がある。
  - でんき タイプ、特性 じゅうなん のポケモンはまひ状態にならない ( じめん タイプにはまひ状態に対する耐性は無いため、まひにできる)。
  - しんぴのまもり / ミストフィールド 状態に守られているポケモンはまひ状態にならない。
  - 特性 シンクロ のポケモンをまひさせると、うのミサイルで反撃したポケモンもまひ状態になる。
## 備考
- ウッウ の 専用特性 となる。
- ウッウが 色違い であっても、この特性によってウッウが咥える獲物は色違いにはならない。
- 特性名は「鵜呑み」が由来と思われる。
