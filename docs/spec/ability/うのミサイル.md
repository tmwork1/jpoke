# 仕様書: うのミサイル

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%86%E3%81%AE%E3%83%9F%E3%82%B5%E3%82%A4%E3%83%AB

## 2. 基本効果（1段落）
「なみのり」または「ダイビング」使用時に獲物をくわえてフォルムチェンジし、その後ダメージを受けると獲物を吐き出して相手に反撃ダメージ（最大HPの1/4）と追加効果を与え、元の姿に戻る。 :contentReference[oaicite:0]{index=0}

## 3. 詳細仕様

### フォルムチェンジ関連
- 技「なみのり」「ダイビング」成功時に発動
- HPに応じてフォルム分岐： :contentReference[oaicite:1]{index=1}  
  - HP > 1/2 → 「うのみのすがた」（サシカマス）
  - HP ≤ 1/2 → 「まるのみのすがた」（ピカチュウ）
- フォルム中にダメージを受けると：
  - 元の姿に戻る
  - 反撃効果発動

### 反撃効果
- ダメージ：相手の最大HPの1/4（最低1） :contentReference[oaicite:2]{index=2}  
- フォルムごとの追加効果：
  - うのみのすがた → 防御1段階低下
  - まるのみのすがた → まひ付与 :contentReference[oaicite:3]{index=3}  
- 反撃は攻撃技ではなく「追加効果扱い」
  - みがわりを貫通
  - マジックガードにはダメージ無効（追加効果は有効） :contentReference[oaicite:4]{index=4}  

### 発動条件・制約
- ダメージを伴う攻撃（物理・特殊問わず）で発動
- 自身がひんしになった場合でも反撃は発動する :contentReference[oaicite:5]{index=5}  
- 反撃後は必ず通常フォルムへ戻る

### 無効化・上書き
- 特性は以下の影響を受けない：
  - かがくへんかガス（無効化されない）
  - いえき・コアパニッシャー（特性消失しない） :contentReference[oaicite:6]{index=6}  
- 特性は世代によってコピー・交換可否が変化
- 特性を失った場合：
  - フォルムは維持される
  - 以降フォルムチェンジ・反撃は発動しない :contentReference[oaicite:7]{index=7}  

### 交代・入場時
- 場に出た時点では通常フォルム
- 技使用によってのみフォルムチェンジ
- 交代で状態はリセット

### その他の特性との相互作用
- 攻撃ではないため以下は発動しない：
  - ばけのかわ
  - アイスフェイス
  - ふしぎなまもり :contentReference[oaicite:8]{index=8}  
- ランク低下・状態異常は通常の耐性で防がれる：
  - クリアボディ等で防御低下無効
  - でんきタイプはまひ無効 :contentReference[oaicite:9]{index=9}  
- シンクロによりまひが反射される :contentReference[oaicite:10]{index=10}  

## 4. 処理順・優先度
- 発動タイミング：
  - フォルムチェンジ：技成功時
  - 反撃：被ダメージ直後
- 処理順：
  1. 技成功（なみのり／ダイビング）
  2. フォルムチェンジ
  3. 被弾
  4. 反撃処理（ダメージ＋追加効果）
  5. フォルム解除
- 同時発動の優先度：被弾イベント内で即時処理
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- なみのり使用後、HPに応じたフォルムに変化する
- 被弾時に最大HPの1/4ダメージが発生する
- フォルムごとの追加効果が正しく発動する
- ひんし時でも反撃が発動する

### 非発動ケース
- なみのり／ダイビング未使用で発動しない
- 特性を失った後は反撃しない
- マジックガード相手にダメージが発生しない

### 境界ケース
- みがわり状態の相手に反撃が貫通する
- 連続攻撃中でも1回の被弾で1回のみ反撃する
- フォルム中に特性消失→反撃が発生しない
- でんきタイプにまひが入らない
- 防御低下が耐性特性で防がれる
