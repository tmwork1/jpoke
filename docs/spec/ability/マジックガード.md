# 仕様書: マジックガード

## 1. 基本情報
- 参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%B8%E3%83%83%E3%82%AF%E3%82%AC%E3%83%BC%E3%83%89
- 種別: 間接ダメージ無効化

## 2. 基本効果（1段落）
相手の攻撃技そのもの以外で受けるダメージを無効化する。状態異常・天候・設置技・一部反動/道具反応などの間接ダメージが対象で、直接攻撃ダメージは通常どおり受ける。

## 3. 詳細仕様
### 発動条件
- ダメージ原因が「間接ダメージ」に分類されること。

### 非発動条件
- 攻撃技の本体ダメージ。
- HP消費技（はらだいこ、みがわり等）やわるあがき反動など、仕様上除外する自己消費。

### 無効化・上書き
- 特性無効化中は間接ダメージを受ける。
- 特性変更後は新特性に従う。

### 交代・入場時
- 常時効果。場にいる間のみ適用。

### その他の特性との相互作用
- かたやぶりで貫通されない。

## 4. 処理順・優先度
- 発動タイミング: ダメージ適用直前の原因判定。
- 処理: 間接ダメージに分類される場合、そのダメージを受けない。
- すばやさは参照しない。

## 5. テスト項目（実装検証用）
### 無効化されるケース
- どく・やけどによる定数ダメージを受けない。
- すなあらしによる定数ダメージを受けない。
- ステルスロックなどの設置技による入場時ダメージを受けない。
- いのちのたま反動を受けない。

### 無効化されないケース
- 攻撃技の本体ダメージは通常どおり受ける。
- こんらん自傷ダメージは通常どおり受ける。
- いたみわけによるHP変動は通常どおり発生する。
- みがわり等の自己HP消費は通常どおり発生する。

### 境界ケース
- 同一条件で、かがくへんかガス有り/無しの被ダメが切り替わる。
- マジックガード持ちが特性変更された後は、新しい特性の仕様で被ダメ判定される。
