# 仕様書: どくくぐつ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A9%E3%81%8F%E3%81%8F%E3%81%90%E3%81%A4

どくくぐつ は、ポケモンのとくせいの一種。
## 効果
この特性のポケモンが使用した技で どく ・ もうどく 状態になったポケモンは、 こんらん 状態にもなる。
## 説明文
- 第九世代: モモワロウの 技によって どく状態に なった 相手は こんらん状態にも なってしまう。
## 所有ポケモン：そのポケモンの別のとくせい
|  | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| モモワロウ | どくくぐつ | - | - |
## こんなときに使おう
どく か もうどく を入れた後、 こんらん 状態で自傷をすれば、さらにどくダメージを稼げる。 相性の良い技として、攻撃しつつ50%の確率でもうどくを狙える 専用技 の じゃどくのくさり や、確定でもうどく状態にできる どくどく などがある。
## 特性の仕様
- この特性は コピーできない 。
- この特性に対する スキルスワップ は失敗する。
- この特性を他の特性に上書きすることはできる。 とくせいなし 状態や かがくへんかガス により無効される。
- モモワロウ に へんしん したポケモンが使用した技でもどくくぐつは発動する。
- 攻撃技 と 変化技 のどちらを使用したときも特性は発動する。
- 場にどくくぐつのポケモンがいるときでも、 どくびし 状態や どくどくだま で自分から どく / もうどく 状態になった相手や、この特性以外のポケモンの技でどく/もうどく状態になった相手は 混乱 状態にならない。
- この特性による 混乱 状態が発動したときはどくくぐつの特性バーが表示される。
  - 相手がすでに混乱状態である場合、どくくぐつの特性バーは表示されない。
  - 相手が特性 マイペース で混乱状態を無効化する場合、どくくぐつとマイペースのどちらも特性バーも表示されない。
- 相手が ラムのみ / モモンのみ を持つ場合、どくくぐつの発動後にきのみが発動してどく/もうどく状態を治す。ラムのみならどくくぐつで発生した混乱状態も治せる。
- 相手の特性が シンクロ である場合、お互いの特性の発動順は 補正抜きのすばやさ が高い順になる。
  - モモワロウがどく状態を無効化する場合、シンクロの特性バーの代わりに「モモワロウには 効果が ないようだ…」というメッセージが流れる。
  - モモワロウのタイプが変わっていたためシンクロのどく状態が有効になったとしても、それによりモモワロウが混乱状態になることはない。
  - シンクロ側の方がすばやさが低く、さらにラムのみを持っている場合、どくくぐつで混乱になった直後にラムのみが発動してシンクロ側のどくと混乱状態が治るが、このときもシンクロは発動する。
- どくガス / ヘドロウェーブ などを使用して1度に複数の相手をどく状態にした場合、1体ごとにどく状態と混乱状態の付与を行う。
- 変化技でどく/もうどくにした相手が りんぷん / おんみつマント を持つ場合でも混乱状態は発動する。
## 備考
- モモワロウ の 専用特性 。
  - スカーレット・バイオレット では、この特性は ボックス の検索リストには表示されない。 Pokémon HOME では検索できる。
