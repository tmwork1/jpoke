# 仕様書: がんじょう

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8C%E3%82%93%E3%81%98%E3%82%87%E3%81%86

## 2. 基本効果（1段落）
HPが満タンのとき、ひんしになる攻撃を受けてもHPが1残る（きあいのタスキと同様の効果）。また、一撃必殺技を無効化する。

## 3. 詳細仕様

### 耐久効果（HP1で耐える）
- 条件：被弾前のHPが最大値であること
- 効果：ひんしになるダメージを受けてもHP1で耐える
- 連続攻撃技の場合：
  - 最初のヒットでHP1になり、以降のヒットでひんしになる
- 1回耐えた後、HPが満タンでない限り再発動しない

### 一撃必殺技無効
- 対象：ぜったいれいど、じわれ等
- 効果：命中しても無効化し、ダメージを受けない

### 発動しない条件
- HPが満タンでない場合
- 定数ダメージ（どく、やけど等）には適用されない
- 天候ダメージや反動ダメージにも適用されない

### 無効化・上書き
- 特性が無効化されている場合（例：かがくへんかガス）は効果が発動しない
- かたやぶり等により無視される
  - この場合、通常通りひんしになる
- 特性のコピー・交換は可能

### 交代・入場時
- 常時効果
- HPが回復して満タンになれば再度発動可能
- 交代しても状態はHP依存で管理される

### その他の特性との相互作用
- マルチスケイル等のダメージ軽減特性と独立して処理
- きあいのタスキとの関係：
  - 両方ある場合でも1回のみHP1で耐える（重複しない）
- ばけのかわ等のダメージ無効特性とは別処理

## 4. 処理順・優先度
- 発動タイミング：ダメージ計算後（ひんし判定直前）
- 処理順：
  1. ダメージ計算
  2. HP減少
  3. ひんし判定前にがんじょう判定
  4. HPを1に補正
- 同時発動時の優先度：
  - 他の耐久系効果（きあいのタスキ等）と同様に1回のみ適用
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- HP満タン時にひんしダメージを受けてもHP1で耐える
- 一撃必殺技を無効化する
- HP回復後に再び耐えることができる

### 非発動ケース
- HPが満タンでない場合に耐えない
- 定数ダメージで耐えない
- 特性無効化状態で耐えない
- かたやぶりで貫通される

### 境界ケース
- 連続攻撃で1ヒット目のみ耐え、後続でひんしになる
- きあいのタスキと重複せず1回のみ耐える
- 天候ダメージでひんしになる
