# 仕様書: がんじょう

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8C%E3%82%93%E3%81%98%E3%82%87%E3%81%86

がんじょう は、ポケモンのとくせいの一種。
## 効果
- 一撃必殺技 を受けても無効化する。
- 第五世代 以降では、HPが満タンのとき一撃で ひんし になるダメージを受けても、HPを1残して持ちこたえる。
## 説明文
- 第三世代: いちげきひっさつが きかない
- 第四世代: いちげきひっさつが きかない。
- 第五世代 ・ 第六世代: いちげきで たおされない。
- ブラック2・ホワイト2 （ チェレン ）: この とくせいの ポケモンは いちげきで ひんしにされる わざの こうかを うけないんだ さらに たいりょくが まんたんだと ひんしになる ダメージを うけても たいりょくが ギリギリで のこるんだ
- 第七世代 ・ 第八世代: あいての わざを うけても いちげきで たおされることが ない。 いちげきひっさつわざも きかない。
- 第九世代: HPが 満タンのとき 技を 受けても 一撃で 倒されることが ない。 一撃必殺技も 効かない。

## こんなときに使おう
一撃必殺技 の対策になる。
第五世代 以降ではほとんどの 攻撃技 を一発は耐えるため、行動回数を増やす特性としても使える。 イバンのみ や じゃくてんほけん の効果が発動しやすくなる他、 メタルバースト や ミラーコート なども成功しやすい。
最大HPが低いポケモンほどHP1の状態から満タンまで回復する手段が多いため、これを利用した レベル1戦法 もある。
## ポケモン+ノブナガの野望におけるがんじょう
ポケモン+ノブナガの野望 では本編と同様の効果の とくせい として登場する。
- 効果: HP が満タンの時にHPが0になるようなダメージを受けてもHPが1残る。
- 説明文: いちげきできぜつする ダメージを受けても HPが1のこる

## 特性の仕様
- 特性 かたやぶり の効果がある一撃必殺技は防げない。
- 一撃必殺技 ががんじょうで無効化されるとき、 第四世代 までは「<ポケモン>は がんじょうで きかなかった!」というメッセージが出る。 第五世代 以降はがんじょうの特性バーと「<ポケモン>には 効果が ないようだ…」というメッセージが出る。
- 一撃必殺技を受けたときは、 命中 判定が行われる前にがんじょうで無効化される。
- 自身より レベル が低いポケモンが使う一撃必殺技を受けた場合、第四世代まではがんじょうによる無効化が優先され、「ぜんぜんきいてない!」のメッセージは出ない。第五世代以降はレベル差による無効化が優先され、がんじょうのメッセージは出ない。
- 一撃必殺技が ダイマックス により無効化される場合はそのメッセージが優先されて「全然 効いてない!」と表示され、がんじょうに関するメッセージは出ない。
  - マックスレイドバトル でふしぎなバリアに対して一撃必殺技を使うときでも、相手ががんじょうのときは効果が無い。
### 第五世代以降
- 攻撃技 をHP1で耐えるとき、がんじょうの特性バーと「<ポケモン>は 攻撃を こらえた!」というメッセージが出る。
- HPが減少した後や、特性が発動した後であっても、再びHPが満タンまで回復すれば攻撃を耐えられるようになる。
- かたやぶり の効果がある技や、 特性を無視する技 は耐えることはできない。
- こんらん の自傷ダメージも耐えることができる。
- だいばくはつ など ひんしになる技 の自滅効果、 もろはのずつき など 反動技 の反動ダメージ、 みちづれ / ほろびのうた 状態によるひんし効果に対してがんじょうは発動しない。
  - ヌケニン がこの特性を持つ場合、 状態異常 、こんらん以外の 状態変化 、 場の状態 によるスリップダメージを受けるときもがんじょうは発動しない。
- みがわり 状態であってもみがわりのHPが1残ることはない。みがわりのHPが減っていても、本体のHPが満タンならみがわりを無視する攻撃を耐えることができる。
- 連続攻撃技 を受けた場合、がんじょうが発動できるかは各攻撃ごとに判定される。1発目を受けてHPが減少しているとき、2発目を受けるとがんじょうは発動しない。
  - HPが満タンのとき1発目をみがわりで防いだときや、1発目を受けた後 きのみジュース で満タンまで回復した場合など、2発目を受ける時点でもHPが満タンである場合は2発目の攻撃も耐えることができる。
- きあいのタスキ を持っているときは、がんじょうが先に発動してきあいのタスキは消費されない。
- こらえる を使用したときは、こらえるの効果が先に発動してがんじょうの特性バーは現れない。
- 後攻の コアパニッシャー を受けて とくせいなし 状態になるとき、がんじょうで攻撃を耐えてから特性が消される。
- かがくへんかガス が発動している間は発動しない。かがくへんかガスのポケモンが ひんしになる攻撃技 を使いひんしになる場合でも、その攻撃に対してがんじょうは発動しない。
## 備考
- この特性を所有するポケモンは いわ か はがね タイプを持っていることが多い。いわ・はがねをどちらも持たないポケモンとしては クヌギダマ ・ ドンファン ・ ダゲキ ・ クレベース 系統・ コスモウム がいる。
