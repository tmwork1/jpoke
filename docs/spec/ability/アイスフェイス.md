# 仕様書: アイスフェイス

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%A2%E3%82%A4%E3%82%B9%E3%83%95%E3%82%A7%E3%82%A4%E3%82%B9

## 2. 基本効果（1段落）
物理攻撃を1回だけ無効化し、その後フォルムが変化する。無効化後は通常状態となり、あられ（ゆき）状態でターン終了時に元のフォルムへ復元される。

## 3. 詳細仕様

### フォルムチェンジ関連
- 初期状態は「アイスフェイス（こおりのかお）」
- 物理攻撃を受けた際：
  - ダメージを0にする（完全無効化）
  - フォルムが「ナイスフェイス」に変化
- フォルム変化後：
  - 防御・特防が低下
  - 素早さが上昇（種族値変更）

### 無効化対象
- 直接攻撃・非接触を問わず「物理技」であれば無効化
- 特殊技・変化技は対象外
- 連続攻撃技の場合：
  - 1ヒット目のみ無効化
  - 以降は通常ダメージ

### 再生条件
- 天候が「ゆき（あられ）」のとき、ターン終了時に復元
- 復元時：
  - 再び物理攻撃を1回無効化可能
  - フォルムも元に戻る

### 無効化・上書き
- 特性が無効化されている場合：
  - 物理攻撃無効化は発動しない
  - フォルム変化も発生しない
- 既にナイスフェイス状態で特性無効化された場合：
  - 状態はそのまま維持される（巻き戻らない）
- 特性コピー・交換は可能

### 交代・入場時
- 交代すると初期フォルム（アイスフェイス）に戻る
- 再入場時は再び1回無効化が有効

### その他の特性との相互作用
- かたやぶり等で無視される：
  - この場合、物理攻撃は無効化されず通常ダメージ
  - フォルム変化も発生しない
- マルチヒット技との相互作用：
  - 1ヒット目のみ無効、以降ヒットは通常処理
- きあいのタスキ等とは独立して処理

## 4. 処理順・優先度
- 発動タイミング：被弾時（ダメージ計算前）
- 処理順：
  1. 攻撃命中
  2. アイスフェイス判定
  3. ダメージ無効化
  4. フォルム変化
- 復元処理：
  - ターン終了時に天候判定→復元
- 同時発動の優先度：行動順に依存しない（被弾時処理）
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- 物理攻撃を受けた際、ダメージが0になる
- 無効化後にナイスフェイスへ変化する
- ゆき状態のターン終了時に元へ戻る
- 連続攻撃技の1ヒット目のみ無効化される

### 非発動ケース
- 特殊技で無効化されない
- 特性無効化状態で無効化が発生しない
- かたやぶり攻撃で無効化されない

### 境界ケース
- ナイスフェイス状態でゆき終了時に復元しない
- 特性無効化中にゆきでも復元しない
- 復元後に再び1回無効化が可能
- 交代後に初期状態へ戻る
