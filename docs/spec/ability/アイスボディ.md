# 仕様書: アイスボディ

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%A2%E3%82%A4%E3%82%B9%E3%83%9C%E3%83%87%E3%82%A3
調査日: 2026-04-30

## 2. 基本効果（1段落）
天気がゆきのとき、毎ターン最大HPの1/16を回復する

## 3. 詳細仕様

### 効果
- 天気があられ/ゆき状態のとき、ターン終了時に最大HPの1/16を回復する（小数点以下切り捨て）。
- こおりタイプ以外のポケモンでも、あられによるダメージを受けない。

### 説明文
天気が ゆきのとき HPを 少しずつ 回復 する。

### 特性の仕様
- あられ/ゆき状態が終了するターンでは回復できない。
- 第五世代以降では、かいふくふうじ状態のときは回復できない（かいふくふうじ状態が終了するターンでも回復できない）。
- ぼうじんゴーグルを持つポケモンでも回復できる。
- あなをほる/ダイビング状態で姿を隠している間は、あられ/ゆき状態でもアイスボディが発動しない。そらをとぶ/シャドーダイブ状態で隠れている間ならば発動して回復できる。
- 特性ノーてんき/エアロックのポケモンが場にいる間は発動しない。

### 備考
- 第九世代現在では、こおりタイプ以外でアイスボディを所有するポケモンはパウワウのみ。

## 4. 処理順・優先度
- 発動タイミング: ターン終了時の回復処理フェーズで評価する。
- 同時発動時の優先度: 同種の効果が重なる場合はゲーム内の一般優先順（行動順・速度・ランダム同値）に従う。
- すばやさ補正との関係: 速度を変更しない特性では行動順に直接影響しない。

## 5. テスト項目（実装検証用）
- 発動ケース: ゆき/あられ状態でターン終了時に最大HPの1/16回復すること。あられダメージを受けないこと。
- 非発動ケース: ゆき/あられ状態終了ターンでは回復しないこと。かいふくふうじ中は回復しないこと。
- 境界ケース: あなをほる/ダイビング中は発動しないこと。そらをとぶ/シャドーダイブ中は発動すること。
