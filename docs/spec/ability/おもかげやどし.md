# 仕様書: おもかげやどし

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%8A%E3%82%82%E3%81%8B%E3%81%92%E3%82%84%E3%81%A9%E3%81%97

## 2. 基本効果（1段落）
テラスタル時、またはこの特性を持った状態で場に出たときに発動し、自身のフォルム（仮面）に応じた特定の能力を1段階上昇させる。発動は1回のみで、同一滞在中に再発動しない。

## 3. 詳細仕様

### フォルムチェンジ関連
- 対象はオーガポン専用特性
- テラスタル時に本特性へ変化する
- フォルム（仮面）ごとに上昇能力が固定：
  - みどりのめん → 素早さ +1
  - いどのめん → 特防 +1
  - かまどのめん → 攻撃 +1
  - いしずえのめん → 防御 +1

### 発動条件・基本挙動
- 以下のいずれかで発動：
  - 特性に変化したとき（テラスタル時）
  - この特性を持った状態で場に出たとき
- 能力上昇はランク変化として適用
- 対象能力がすでに最大の場合：
  - 上昇は発生しないが「発動済み」として扱われる

### 発動回数制限
- 1回の場への滞在につき1回のみ発動
- すでに発動済みの場合：
  - かがくへんかガスの解除後でも再発動しない

### 無効化・上書き
- かがくへんかガス発動中は発動しない
- ガス中に発動機会を逃した場合：
  - ガス解除直後に発動可能（未発動に限る）
- いえき等で特性なし状態の場合：
  - 発動しない
- 特性はコピー不可・スキルスワップ不可
- 特性の上書きは可能

### 交代・入場時
- 場に出たときに発動判定
- 交代すると再度発動可能（未発動扱いになる）
- ひんし時にテラスタル解除されると元特性に戻る

### その他の特性との相互作用
- 能力上昇は通常のランク変化として扱う：
  - クリアボディ等で無効化される可能性あり
- かたやぶりの影響は受けない（自己対象効果）
- とくせいガード所持でもテラスタルにより特性変化は発生する

## 4. 処理順・優先度
- 発動タイミング：
  - テラスタル直後
  - 入場直後
- 処理順：
  1. 特性変化 or 入場
  2. 発動判定
  3. 能力ランク上昇
- 同時発動時の優先度：
  - 他の入場時特性と同様、行動順（すばやさ）依存
- すばやさは発動順に影響

## 5. テスト項目（実装検証用）

### 発動ケース
- テラスタル時に対応能力が+1される
- 入場時に未発動なら能力が+1される
- フォルムごとに正しい能力が上昇する
- ガス解除後に未発動なら発動する

### 非発動ケース
- 特性無効化状態で発動しない
- 特性なし状態で発動しない
- 能力が最大時に上昇しない

### 境界ケース
- 能力最大時でも発動済み扱いになる
- 一度発動後はガス解除でも再発動しない
- 交代後に再発動する
- テラスタル解除で特性が元に戻る
