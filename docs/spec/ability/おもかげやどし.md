# 仕様書: おもかげやどし

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8A%E3%82%82%E3%81%8B%E3%81%92%E3%82%84%E3%81%A9%E3%81%97

おもかげやどし は、ポケモンのとくせいの一種。
## 効果
オーガポン が テラスタル するとこの特性に変化する。
この特性に変化したときか、この特性になって場に出たとき、自分の フォルム ごとに決められた 能力 1つが 1段階 上がる。
- みどりのめん : 素早さ
- いどのめん : 特防
- かまどのめん : 攻撃
- いしずえのめん : 防御
## 説明文
- 第九世代: みどりのめん
## 所有ポケモン：そのポケモンの別のとくせい
|  | 特性 | 特性2 | 隠れ特性 | テラスタル 特性 |
| --- | --- | --- | --- | --- |
| オーガポン (みどりのめん) | まけんき | - | - | おもかげやどし |
| オーガポン (いどのめん) | ちょすい | - | - | おもかげやどし |
| オーガポン (かまどのめん) | かたやぶり | - | - | おもかげやどし |
| オーガポン (いしずえのめん) | がんじょう | - | - | おもかげやどし |
## こんなときに使おう
テラスタル をすることで、一致技のダメージが上がるだけでなく、 こうげき や ぼうぎょ 、 とくぼう 、 すばやさ が上昇するため、全抜き性能が高くなる。
ただし、どの仮面の オーガポン も、テラスタル前から強力な特性を持っているため、特性と能力アップのどちらを切るが重要である。
## 特性の仕様
- オーガポン#固有の仕様 も参照とする。
- 発動時、特性バーと共に「オーガポンは <碧の仮面/井戸の仮面/竈の仮面/礎の仮面>を かがやかせ <素早さ/特防/攻撃/防御>が 上がった!」とメッセージが出る。
- すでに対象能力が最大まで上昇している場合、特性バーやメッセージは出現しない。
- この特性は コピーできない 。この特性に対する スキルスワップ は失敗する。
- この特性を他の特性に 上書き することはできる。
  - かがくへんかガス が発動している間、おもかげやどしは発動しない。
    - かがくへんかガスによりおもかげやどしが発動できなかった場合、ガスが解除された直後におもかげやどしを発動できる。
    - おもかげやどしは場に出るたびに1回しか発動しない。場に出てからおもかげやどしが発動したことがある場合、かがくへんかガスの発動と解除が起きてもおもかげやどしが再発動することはない。上がる能力がすでに最大だったため特性バーが表れなかったときも発動済みと見なされ、その後かがくへんかガスの発動と解除が起きる間に対象能力が下がっていたときしてもおもかげやどしは発動しない。
  - いえき を受けたときは とくせいなし 状態になる。とくせいなし状態で テラスタル したときも特性は発動しない。
    - ポケットモンスタースカーレット・バイオレット追加コンテンツ碧の仮面のストーリー中において、 オーガポン に対して いえき で特性を無効化すると、 オーガポン が仮面を被りなおした際に オーガポン のこの特性は再び発動するようになる
- みどりのめんのオーガポンが とくせいガード を持っているときでも、テラスタルにより特性は変わる。
- ひんし になってテラスタルが解除されたときは特性も元に戻る。
## 備考
- テラスタル した オーガポン の 専用特性 となる。
  - ボックスにこの特性のポケモンがいることは無いため、検索機能のリストには含まれていない。 Pokémon HOME のリストにも無い。
