# 仕様書: マルチタイプ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%83%AB%E3%83%81%E3%82%BF%E3%82%A4%E3%83%97

マルチタイプ は、ポケモンの とくせい の一種。
## 効果
- 持って いる プレート に応じて自身の フォルム と タイプ が変わる（ プレート#一覧 を参照）。
- 第七世代 のみ、持っている Zクリスタル に応じて自身のフォルムとタイプが変わる。
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代 ・ ブリリアントダイヤモンド・シャイニングパール: もっている プレートで タイプが かわる。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの アルセウスは なまえに プレートと ついている どうぐを もっていると それぞれの プレートに たいおうした タイプに へんか するんだ
- 第七世代 ・ ソード・シールド: もっている プレートや Zクリスタルの タイプによって じぶんの タイプが かわる。
- 第九世代: 持っている プレートによって 自分の タイプが 変わる。

## こんなときに使おう
パーティの補完として任意のタイプに変更できる。 プレート を持たせたときは さばきのつぶて のタイプも自身のタイプと同様に変化するため、 タイプ一致技 として使用できる。
## 特性の仕様
- この特性により アルセウス のタイプが変わると、それに合わせて フォルム も変わる。
  - 手持ち画面に映される小アイコンの姿に持たせた プレート が反映されるかは、世代によって異なる。
    - 第八世代 以前では持たせたプレートに関係無く通常時のフォルムが表示される。 通信対戦 の対戦相手は、見せ合い時点ではプレートの有無と種類を判別できない。
    - スカーレット・バイオレット では持たせたプレートがアイコンに反映される。通信対戦の対戦相手は、見せ合い時点でフォルムからプレートの有無と種類を判別できる。
  - 第六世代 以前・ BDSP では、相手の場に出てきたアルセウスのタイプは見た目や相性などから見分けるしかない。 第七世代 では下画面、 第九世代 では「様子を見る」画面に表示されるタイプから判別できる。
- マルチタイプを無効化することや、他のポケモンにマルチタイプを付与することはできない。
  - この特性は 上書きできない 。 とくせいなし 状態にもならず、 バトンタッチ で引き継がれるときはとくせいなし状態が消える。 かがくへんかガス の効果はアルセウスやさばきのつぶてのタイプ・威力に影響を与えない。
  - この特性は コピーできない 。この特性のポケモンが なかまづくり を マジックコート 状態で跳ね返して使用しても失敗する。
  - この特性は スキルスワップ で交換することができない。
  - アルセウスに へんしん したポケモンに対しても、マルチタイプを書き換えたり無効化したりすることはできない。
- かたやぶり の効果で無効化できない。アルセウスは特性がなければ ノーマル タイプだが、かたやぶりの効果がある技がアルセウスをノーマルタイプとみなすことはない。かたやぶりの効果がある技でもマルチタイプの特性を書き換えたり無効化したりすることはできない。
- アルセウスのタイプは みずびたし / まほうのこな で変更できないが、これは特性の効果ではなくアルセウスという 種族固有の効果 となる。
  - アルセウスに へんしん したポケモンに対しては、みずびたし/まほうのこなでタイプを変更できる。
  - アルセウスが テラスタル したときは、持たせたプレートとは別のタイプに変化できる。
### 第四世代のみ
- アルセウスの持ち物に関係なく、アルセウスの持ち物に影響を与える技の効果は無効になる。
  - トリック / すりかえ は、アルセウスに対して使った場合とアルセウスが使った場合、共に必ず失敗する。
  - はたきおとす / どろぼう / ほしがる はダメージだけ受ける。
  - さしおさえ 状態にならない。
  - ついばむ / むしくい など、影響を与える道具の種類が限定されており、プレート各種に影響を与えることがない技は普通に受ける。
- アルセウスに へんしん したポケモンは、一旦相手のアルセウスと同じ色の姿になったあと、変身使用者が持っているプレートの種類・有無に応じた体の色に変化する。持ち物や特性に対する技が一切効かないアルセウスの体の色が戦闘中に変わるときは、このケースのみである。
### 第五世代以降
- アルセウスがプレートを持っている場合か、アルセウスにプレートを渡そうとした場合、アルセウスの持ち物に影響を与える技の効果は無効になる。
  - アルセウスに対する トリック / すりかえ / ギフトパス 、またはアルセウスが使うトリックは自分か相手のどちらかがプレートを持っているときのみ失敗する。どちらもプレートを持っていなければ成功する。
  - アルセウスの持つプレートを はたきおとす / どろぼう / ほしがる で無効化したり奪ったりすることはできない。プレートを持っていなければ技の効果は発動する。
    - 第六世代 以降では、プレートを持つアルセウスに対するはたきおとすの威力は上がらない。 プレート以外の道具を持っていれば威力が上がり、道具を落とすこともできる。
  - プレートを持っているアルセウスに対して マジシャン / わるいてぐせ の効果は発動しない。アルセウスに対して きょうせい でプレートを渡すことはできない。
- さしおさえ / マジックルーム 状態の効果を受けてもアルセウスのタイプ・姿は変わらないが、 さばきのつぶて を使うときはプレートの効果を受けずに ノーマル タイプになり、威力も上がらない。
- アルセウスに へんしん したポケモンは、マルチタイプをコピーすることができる。
  - 変身使用者が持っているプレート/Zクリスタルに関わらず、対象のアルセウスと同じフォルムになる。ただし さばきのつぶて は変身したポケモンが持っているプレートの種類・有無によりタイプが決まる。
  - アルセウスにへんしんしたポケモンに対しては、プレートを無効化させたり奪ったりすることや、プレートを渡すことができる。第六世代以降でははたきおとすの威力も上がる。
## 備考
- アルセウス の 専用特性 。
- 似たような効果の特性として、持たせた メモリ でタイプが変化する シルヴァディ の ARシステム がある。
- ソード・シールド では所持者がいないが、 ボックス の検索機能により名前と説明を見ることができる。
