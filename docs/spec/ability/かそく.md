# 仕様書: かそく

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%8B%E3%81%9D%E3%81%8F

## 2. 基本効果（1段落）
場に出て行動を選択したターン以降、毎ターン終了時に自身の素早さが1段階上昇する。交代直後のターンなど、まだ行動していない場合は発動しない。

## 3. 詳細仕様

### 発動条件・基本挙動
- 毎ターン終了時に発動
- 「一度でも行動を選択した後」であることが必要
- 行動が失敗しても（ひるみ・ねむり等）発動条件は満たす
- 素早さランクが最大（+6）の場合は発動しない

### 非発動条件
- 場に出た直後のターンは発動しない
- 行動選択前に交代・退場した場合は発動しない
- 行動不能状態でも「行動選択していない」場合は発動しない

### 再登場時の扱い
- 一度交代すると「未行動状態」にリセットされる
- 再登場後は再び行動選択しないと発動しない

### 特性付与時の挙動
- 後から特性を得た場合：
  - 既にその場で行動済みなら、そのターン終了時から発動
- 登場と同時に特性を得た場合：
  - そのターンは発動しない

### 無効化・上書き
- 特性が無効化されている間（例：かがくへんかガス）は発動しない
- 無効化解除後は条件を満たしていれば再び発動
- 特性のコピー・交換は可能

### 交代・入場時
- 入場時に即時効果なし
- ターン終了時処理でのみ発動判定

### その他の特性との相互作用
- ターン終了時に発動する他特性とは独立して処理される
- 同時に複数の「かそく」持ちがいる場合は、補正込み素早さ順で発動

## 4. 処理順・優先度
- 発動タイミング：ターン終了時
- 処理順：
  1. ターン終了時の各種処理
  2. かそく発動
  3. 素早さランク上昇
- 同時発動時の優先度：
  - 補正込み素早さ順
- すばやさは発動順に影響

## 5. テスト項目（実装検証用）

### 発動ケース
- 行動後のターン終了時に素早さが+1される
- 行動失敗でも素早さが上昇する
- 特性を後から得た場合でも条件を満たせば発動する

### 非発動ケース
- 交代直後のターンでは発動しない
- 行動選択前に退場した場合発動しない
- 素早さが最大時は発動しない
- 特性無効化中は発動しない

### 境界ケース
- 交代後に未行動状態にリセットされる
- 行動不能状態でも行動選択済みなら発動する
- 複数かそく持ちで素早さ順に発動する
- 特性付与タイミングによる発動有無が正しい
