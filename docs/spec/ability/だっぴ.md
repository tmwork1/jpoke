# 仕様書: だっぴ

調査日: 2026-02-07

だっぴ は、ポケモンのとくせいの一種。
## 効果
ターン 終了時に、自分の 状態異常 が30%の確率で回復する。
- 第三世代 のみ、発動率は1/3となる。
## 説明文
- 第三世代: だっぴして からだを なおす
- 第四世代 ・ 第五世代 ・ 第六世代: じょうたい いじょうを なおす ことがある。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは せんとうちゅうに じょうたいいじょうを かいふく することが あるよ
- 第七世代 ・ 第八世代 ・ 第九世代: からだの かわを ぬぎすてることで じょうたい いじょうを なおすことが ある。

## こんなときに使おう
場に居座る耐久型のポケモンであれば発動の試行回数を稼ぎやすく、 もうどく の対策にもなる。 ねむる を使うときも通常より早く起きれる可能性がある。
## ポケモン+ノブナガの野望におけるだっぴ
ポケモン+ノブナガの野望 では本編と同様の効果の とくせい として登場する。
- 効果: 自軍のターン開始時に50%の確率で自分の 状態異常 が治る。 こんらん 状態・ ひるみ 状態は対象外。
- 説明文: じょうたいいじょうが しぜんにかいふくする
## Pokémon UNITEにおけるだっぴ
Pokémon UNITE では、 サナギラス の特性として登場する。

## 特性の仕様
- こんらん などの 状態変化 は治せない。
- 発動のタイミングは ターン の終了時になる。詳細なタイミングは以下のようになる。
  - どく / もうどく / やけど / あくむ 状態や、特性 ナイトメア のダメージ判定より前なので、だっぴが発動するターンはこれらのダメージを受けない。
    - ルビー・サファイア においてのみ、 ねむり をだっぴで治してもあくむ状態が回復しないバグがある ( 白昼夢 を参照)。
  - ねむけ による ねむり の発生よりも前なので、これによりねむり状態になったターンにだっぴは発動しない。ねむけ状態になった後に状態異常になったが、ねむけになった次のターンにだっぴで状態異常が治ったときはねむり状態になる。
  - どくどくだま / かえんだま の発動判定より前なので、だっぴで状態異常が治ったターンからこれらの持ち物が発動する。また、これらの持ち物が発動したターンにだっぴで状態異常を治すことはない。
  - あばれる 状態のねむり状態による中断判定より前に発動する。あばれる状態の途中でねむり状態にさせられたときや、 ねごと で出た技であばれる状態になったとき、だっぴが発動しなければあばれる状態は中断されて技を選べるようになるが、だっぴが発動した場合はあばれる状態は継続して技を選べない。
- 味方の場に特性 いやしのこころ のポケモンがいるときは、状態異常の回復判定を2回受けられることになる。
## 備考
- 似たような効果がある特性として うるおいボディ / いやしのこころ がある。
- 本特性を持つ ズルッグ は、 分類 も「だっぴポケモン」である。
