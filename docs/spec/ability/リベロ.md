# 仕様書: リベロ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%AA%E3%83%99%E3%83%AD
- 種別: 技タイプ追従の自己タイプ変更

## 基本効果
技を使用する直前に、自分のタイプをその技タイプへ変更する。交代で元タイプへ戻る。SV仕様では1回の場滞在につき1回のみ発動する。

## 詳細仕様
### 発動条件
- 技が実行段階に入ること。
- その場滞在で未発動であること。

### 非発動条件
- 行動不能等で技が出ないとき。
- テラスタル中で発動不可とする実装方針の場合。

### 無効化・上書き
- 特性無効化中は発動しない。
- 特性変更後は新特性に従う。

### 交代・入場時
- 交代でタイプと発動回数フラグを初期化する。

### その他の特性との相互作用
- へんげんじざいと同仕様として共通ロジックを使用する。

## 処理順・優先度
- 発動タイミング: 技補正計算前。
- 処理順: 行動成立判定 -> リベロ判定 -> タイプ変更 -> ダメージ/効果計算。
- すばやさは参照しない。

## テスト項目
### 発動ケース
- 異なるタイプの技で先にタイプが変わる。
- 変化技でもタイプ変更する。

### 非発動ケース
- 行動不能時はタイプ変更しない。
- 同一場滞在で2回目は発動しない。

### 境界ケース
- 交代後は再び1回だけ発動できる。
- へんげんじざいと同じ挙動で一致する。
