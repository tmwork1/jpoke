## 効果

## 説明文
- 第七世代 ・ 第八世代 ・ 第九世代: だれかが おどりわざを つかうと じぶんも それにつづいて おどりわざを だすことが できる。

## こんなときに使おう
ダブルバトル では味方が使う踊りの技を真似して使える。特殊型のオドリドリなら ちょうのまい を使うポケモンと相性が良い。
シングルバトルでは、相手にちょうのまいや りゅうのまい を使われてもすばやさ関係を逆転されにくくなる。
## 特性の仕様
- テラレイドバトル では発動しない。
- 発動すると、おどりこのポケモンが追加行動をする形でおどりの技をコピーする。おどりこによる行動の後でも、自身の行動順が来たときはそのターンに選択していた技が使える。使おうとした技を置き換えるわけではない。
  - 「おどりこの効果で技を出したが、自身が選択した行動は終えていないポケモン」に対しては、技の効果上そのターンでの行動を終えたとみなされることがある。このポケモンに対して しっぺがえし の威力は2倍になる。 コアパニッシャー を受けると特性が消える。このポケモンに対する ふいうち / おさきにどうぞ / さきおくり / さきどり / そうでん は失敗する。
  - 特性 アナライズ の発動におどりこによる行動は考慮されない。他に「おどりこの効果で技を出したが、自身が選択した行動は終えていないポケモン」がいた場合、アナライズは発動しない。
- おどりの技が使われれば、おどりこは1ターンに何度でも発動できる。
  - ただし、おどりこでコピーされた技が、さらにおどりこでコピーされることはない。
- 同じ踊りの技に対して複数匹のおどりこが発動する場合、世代により発動順が異なる。
  - 第七世代 では補正抜きの すばやさ 実数値が 低い順 に発動する。 ランク補正 / トリックルーム / おいかぜ / こだわりスカーフ / くろいてっきゅう などのすばやさ補正は考慮しない。
    - スピードスワップ による実数値変動は考慮しない。
    - へんしん 状態のポケモンは、変身 後 のすばやさを考慮する。
    - こうこうのしっぽ / まんぷくおこう の効果は影響しない。
  - 第九世代 では補正込みのすばやさが 高い順 に発動する。ランク補正/トリックルームなどの補正も考慮する。
    - こうこうのしっぽ を持つポケモンはすばやさランク補正がないものとして扱われる。
- おどりこによる行動は、元の技による効果や きのみ の発動などが完了してから行われる。
  - 元の技の対象が持ち物 だっしゅつボタン / レッドカード / だっしゅつパック 、特性 にげごし / ききかいひ で交代した場合、交代先のポケモンが出てきてからおどりこが発動する。
- 元の使用者が ひんし になった後も踊りの技をコピーできる。
- 状態異常 / 状態変化 になっているときは、おどりこによる行動時でも失敗判定や回復判定が行われる。
  - ねむり / こおり 状態のときは回復判定が行われる。失敗したときは行動できない。ねむりのカウントは消費される。
  - こんらん 状態のとき判定に失敗すると自傷する。こんらんのカウントは消費される。
  - まひ で動けなくなることや、こんらんで自滅ダメージを受けることもある。
- 状態変化は、おどりこによる行動で解除されるものと、おどりこの行動と無関係にターン経過で解除されるものがある。 状態変化#状態変化の回復 も参照。
  - ひるんだ 場合は動けない。本来の行動とおどりこの行動のどちらが先であれ、ひるみ状態はターンの終了まで続く。
  - ターン経過で回復する状態変化に関しては、おどりこで追加行動をしたときも持続ターンが減ることは無い。
  - まもる / こらえる 状態は、おどりこの追加行動に関係なくそのターンが終了するまで続く。
  - ふんじん 状態の場合、おどりこでほのお技を出そうとしたときもふんじんのダメージを受ける。このときもふんじん状態はターンが終わるまで続く。
  - そうでん 状態の場合、おどりこで出す技も でんき タイプに変わる。このときもそうでん状態はターンが終わるまで続く。
  - ロックオン / とぎすます 状態の場合、おどりこで出す技も効果を受ける。おどりこによる行動ではロックオン/とぎすます状態は解除されない。
  - いかり / みちづれ / おんねん 状態などは、おどりこによる行動で解除される。
  - じゅうでん 状態の場合、おどりこで出すでんき技のダメージは上がる。おどりこによる行動でじゅうでん状態は解除される。
- 他のポケモンが使った踊りの技が、以下のように失敗したときおどりこは発動しない。
  - 命中 しなかった。
  - 相性 や 特性 で無効化された。
  - まもる 状態で防がれた。
  - 技の仕様によりうまく決まらなかった。例えばすでに攻撃ランクが最大のときに つるぎのまい を使ったときなど。
  - よこどり で奪われた。
  - マジックコート 状態で反射された。反射される前と後いずれもおどりこの対象とならない。
- そらをとぶ 状態などで 姿を隠している ときは発動しない。
  - 姿を隠していなければ、 溜め技 を溜めている最中でもおどりこは発動し、技も成功する。
- 以下の状況でもおどりこは発動するが、出せる技は制限される。
  - アンコール 状態である場合、アンコールされた技のみ使用できる。それ以外の技をコピーしようとしたときは失敗する。
  - あばれる / ころがる / がまん など 数ターン行動する技 の使用中である場合、コピーしようとした技は失敗する。おどりこによる失敗が、継続中の技に影響することはない。
    - ただし、 はなびらのまい の使用中なら、はなびらのまいだけは成功する。このときも あばれる 状態の継続ターンには影響しない。
  - ちょうはつ 状態のポケモンが変化技をコピーしようとした場合、失敗する。
  - かいふくふうじ 状態のポケモンが みかづきのまい をコピーしようとした場合、失敗する。
  - かなしばり / ふういん 状態で出せない技は、おどりこで出すこともできない。
- こだわり系アイテム を持っているが技をこだわっていない状態のとき、自分も覚えている技をおどりこで出した場合はその技でこだわる。
  - 自分は覚えていないをおどりこで出した場合はこだわったことにならない。ただし、自分の行動より先に特性が発動したときは、そのターンの自分の行動はPPが消費されることなく必ず失敗する。
- こだわり系アイテムの効果で技をこだわっている場合、おどりこで成功する技はこだわっている技のみになる。それ以外の技をコピーしようとしたときは失敗する。
- とつげきチョッキ を持っていても 変化技 をコピーして使用できる。
- 反動で動けない ときでも、おどりこで技を使用できる。
- いちゃもん 状態でも、おどりこにより同じ技を連続で出すことはできる。
- きあいパンチ の準備中に攻撃を受け、集中力が途切れたときでもおどりこで技を出すことはできる。
  - ひるみ の 追加効果 がある攻撃技を受けてきあいパンチの集中力が途切れたときは、きあいパンチの失敗が優先されてひるみ状態にならないため、おどりこで技を出すことができる。
- Z変化技 による踊りの技もコピーできるが、Zワザの付加効果は発動されない。
- さいはい で繰り出された踊りの技はおどりこでコピーできる。さいはいの使用者自身がおどりこである場合、さいはいで出された技は自分以外が出した技なのでコピーできる。
- 他の技が出る技 で繰り出された踊りの技も、おどりこでコピーできる。
- おどりこでコピーした技はさいはい/オウムがえしで出ない。おどりこで技を出した後にこれらの技を受けた場合は、その前に自分がPPを消費した最後の技が繰り出される。
- おどりこで技を出した後に アンコール / うらみ / かなしばり / ものまね / スケッチ を受けた場合、自分がPPを消費した最後の技が対象になる。
- もちもの の効果は、基本的におどりこによる技にも反映される。
  - おどりこの行動でも対応するタイプの ジュエル は発動する。
  - おどりこで行動したとき メトロノーム の使用回数は維持される。同じ技を出し続けている場合は、おどりこの技のダメージにもメトロノームの効果がある。
  - ミクルのみ の効果はおどりこの行動にも適用され、行動を終えると効果が切れる。
### おどりこで出る技の対象
- 1体が対象の技 をコピーした場合、おどりこの攻撃対象には以下のポケモンが優先される。
  - おどりこと元の使用者が敵対している場合、攻撃対象は元の使用者となる。
  - 元の使用者が味方である場合、味方が攻撃対象にしていた相手を優先して対象にする。味方の攻撃対象がおどりこのポケモン自身だった場合は、味方を優先して対象にする。
- ランダム1体が対象の技 をコピーした場合は、相手からランダムに1体が選ばれる。
- おどりこのポケモンが ちゅうもくのまと の影響を受けているときは、おどりこの攻撃もちゅうもくのまとに引き寄せられる。
  - 味方の技が いかりのこな で引き寄せられたが、おどりこのポケモンは ぼうじんゴーグル などでいかりのこなを無視できるとき、おどりこの攻撃対象には味方が本来選択していたポケモンが優先される。
- めざめるダンス のタイプは、元の使用者ではなくおどりこ自身のタイプに依存する。
- はなびらのまい をコピーした場合でも、 あばれる 状態にはならずに次のターンは自由に技を選択できる。
