# 仕様書: さいせいりょく

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%95%E3%81%84%E3%81%9B%E3%81%84%E3%82%8A%E3%82%87%E3%81%8F

## 2. 基本効果
他のポケモンに交代したとき、最大HPの1/3を回復する。

## 3. 詳細仕様

### 発動条件
- 自分が場から交代して控えに戻るときに発動する
- 交代手段は問わない（通常交代・技・道具による交代を含む）

### 効果内容
- 最大HPの1/3（小数点以下切り捨て）を回復する
- 回復は交代先ではなく、控えに戻った時点で発生する

### 発動タイミング
- 交代処理の完了時に発動する
- 交代を伴わない状態変化では発動しない

### 無効化・上書き
- 特性が無効化されている場合は発動しない
- 特性のコピー・交換は可能

### 交代・入場時
- 交代で場を離れるときにのみ発動する
- 場に出る側では発動しない

## 5. テスト項目（実装検証用）

### 発動ケース
- 交代時にHPが1/3回復する
- 技による強制交代でも発動する

### 非発動ケース
- 場に出るときは発動しない
- 特性無効化状態では発動しない

### 境界ケース
- 小数点以下は切り捨てで処理される
- 連続交代でも毎回発動する
