# 仕様書: ちくでん

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%A1%E3%81%8F%E3%81%A7%E3%82%93
調査日: 2026-04-30

## 2. 基本効果（1段落）
でんきタイプの技を無効化し、最大HPの1/4を回復する（小数点以下切り捨て）。

## 3. 詳細仕様
- 発動条件: 対象に向けられたでんきタイプ技に対して判定。
- 効果内容: ダメージ・追加効果を無効化して回復。
- 回復制限: HP満タン時は無効化のみ成立。
- 対象外: 場全体効果の技（エレキフィールド等）には発動しない。
- 無効化条件: かがくへんかガス中は発動不可。かたやぶり効果付きでんき技には無効化される。

## 4. 処理順・優先度
- 適用タイミング: ダメージ計算前のタイプ無効判定段階。
- 同時効果: 連続技でも回復は1回。
- 防御技との関係: まもる等で防いだ場合は発動しない。

## 5. テスト項目（実装検証用）
- 発動ケース: 単体でんき技を無効化して1/4回復。
- 非発動ケース: かたやぶり技、かがくへんかガス中。
- 境界ケース: HP満タン、連続攻撃、みがわり時。
