## 2. 基本効果（1段落）
場に出たとき、相手が自身に対して効果抜群となる攻撃技、または一撃必殺技を覚えている場合に発動し、「危険な技を持っている」ことを通知する（実際のダメージや能力変化は発生しない）。 :contentReference[oaicite:0]{index=0}

## 3. 詳細仕様

### 発動条件・基本挙動
- 入場時に1回だけ発動
- 判定対象：
  - 自身に対して効果抜群となる攻撃技
  - 一撃必殺技
- 条件を満たす場合のみメッセージ表示（みぶるい）
- 技名・タイプ・対象ポケモンは特定できない :contentReference[oaicite:1]{index=1}

### 判定仕様（技の扱い）
- 一部の可変タイプ技は以下のように扱う：
  - ウェザーボール等 → 常にノーマル扱い
  - さばきのつぶて等 → 持ち物に関係なくノーマル扱い
  - テラバースト等 → テラスタル状態に関係なくノーマル扱い :contentReference[oaicite:2]{index=2}
- そのため、これらの技は弱点判定に使用されない場合がある

### 発動しない条件
- 相手が条件を満たす技を持っていない場合
- 自身が場に出ている後に、相手が後出しで対象技持ちを出しても発動しない :contentReference[oaicite:3]{index=3}
- 特性が無効化されている場合は発動しない

### 無効化・上書き
- かがくへんかガス中は発動しない
- 無効化解除後も自動再発動はしない（入場時限定）
- 特性のコピー・交換は可能

### 交代・入場時
- 入場時にのみ発動
- 交代すると再度発動可能
- ダブルバトル等では相手全体を参照
  - どのポケモンが該当するかは判別不可 :contentReference[oaicite:4]{index=4}

### その他の特性との相互作用
- 能力変化やダメージに影響を与えないため、他特性との直接的な競合は基本なし
- 姿を隠している相手（そらをとぶ等）にも反応する :contentReference[oaicite:5]{index=5}

## 4. 処理順・優先度
- 発動タイミング：入場直後
- 処理順：
  1. 入場
  2. 発動判定
  3. メッセージ表示（効果なし）
- 同時発動時の優先度：
  - 他の入場時特性と同様に行動順（すばやさ）依存
- すばやさは発動順に影響

## 5. テスト項目（実装検証用）

### 発動ケース
- 相手が弱点技を持っている場合にメッセージが表示される
- 一撃必殺技所持時に発動する
- ダブルバトルで複数相手のうち誰かが条件を満たすと発動する

### 非発動ケース
- 条件を満たす技が存在しない場合発動しない
- 特性無効化状態で発動しない
- 後出しで条件を満たす相手が出ても発動しない

### 境界ケース
- 技の詳細情報（名前・タイプ）が表示されない
- 可変タイプ技がノーマル扱いとして判定される
- 姿を隠している相手の技でも検知される
- 交代後に再度発動する
