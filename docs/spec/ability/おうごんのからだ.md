# 仕様書: おうごんのからだ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8A%E3%81%86%E3%81%94%E3%82%93%E3%81%AE%E3%81%8B%E3%82%89%E3%81%A0

おうごんのからだ は、ポケモンのとくせいの一種。
## 効果
他のポケモンから受ける 変化技 を無効化する。
## 説明文
- 第九世代: 酸化せず 丈夫な 黄金の体は 相手からの 変化技を 受けない。
## 所有ポケモン：そのポケモンの別のとくせい
| ポケモン | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| サーフゴー | おうごんのからだ | - | - |
## こんなときに使おう
ほとんどの変化技を無効化するため、相手に自分の行動を妨害されにくくなる。効果対象は変化技に限るので、 攻撃技 の 追加効果 、 特性 、 持ち物 などの影響は受ける。
## 特性の仕様
- 自分が対象の変化技 は防がない。
- 味方から受ける変化技も無効化する。
- 場を対象とした変化技は防がない ( ステルスロック などの 相手の場が対象の技 、 くろいきり などの 全体の場が対象の技 、 リフレクター など 味方の場が対象の技 が挙げられる) 。
- 複数のポケモンを対象とする変化技に巻き込まれた場合、この特性のポケモンのみが技を防ぎ、他のポケモンは技を受ける。
- 相手の マジックコート 状態で跳ね返ってきた変化技も無効化する。
- かたやぶり ・ きんしのちから の効果が発動した変化技は防げない。
- トリックガード や マジックコート 状態とは異なる効果範囲を持つ。他のポケモンから以下の技を受けたときは無効化する。ただし、いずれの場合もこの特性のポケモン自身が使用する技が自身の特性で防がれることはない。
  - 自分含む全員が対象の技 ( おちゃかい ・ ほろびのうた )
  - 味方全員が対象の技 ( いのちのしずく ・ とおぼえ )
  - 味方1体が対象の技 ( アロマミスト ・ てだすけ )
  - へんしん ・ なりきり など、対象の状態をコピーする技
  - スキルスワップ ・ トリック ・ スピードスワップ など、対象と能力などを入れ替える技
  - おさきにどうぞ ・ さいはい ・ さきおくり など、行動順に関する技
  - おきみやげ : 使用者は ひんし にならない。
  - のろい (呪い) : 使用者のHPは減少しない。
  - ちからをすいとる : 使用者のHPは回復しない。
- 変化技の対象に直接選ばれていない場合は、変化技により発動した効果でも防がない。
  - みちづれ 状態の相手を倒したときは、みちづれの効果で自分もひんしにされる。
  - スレッドトラップ ・ トーチカ ・ ニードルガード 状態の相手に直接攻撃したときに発動する効果は防げない。
  - 相手の ふういん 状態により技は出せなくなる。
  - 相手の ちゅうもくのまと 状態により技は引き寄せられる。
  - 他のポケモンの おかたづけ により、自分の みがわり 状態は消滅する。
  - 味方の うつしえ により、自分の特性は書き換えられる。
  - 味方の ねがいごと ・ いやしのねがい ・ やどりぎのタネ の効果で回復することはできる。
  - 味方の さいきのいのり で復活することはできる。
  - 味方の バトンタッチ により ランク補正 や 状態変化 は引き継がれる。
  - 味方の サイドチェンジ で位置を入れ替えることはできる。
## 備考
- サーフゴー の 専用特性 。
- ほぼ全ての変化技を無効化するものの、 上書きできない特性 や コピーできない特性 ではない。特性の効果や、かたやぶりの効果がある技により、おうごんのからだを消されることや、他のポケモンがおうごんのからだを手にすることはできる。
