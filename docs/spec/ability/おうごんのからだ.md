# 仕様書: おうごんのからだ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8A%E3%81%86%E3%81%94%E3%82%93%E3%81%AE%E3%81%8B%E3%82%89%E3%81%A0

## 基本効果
他のポケモンから受ける変化技を無効化する。

## 詳細仕様

## 詳細仕様

### 無効化される技
以下の他のポケモンから受ける変化技は無効化：
- へんしん / なりきり（状態コピー）
- スキルスワップ / トリック / スピードスワップ（能力交換）
- おさきにどうぞ / さいはい / さきおくり（行動順）
- おきみやげ（すべての能力を下げる）
- のろい（呪い）
- ちからをすいとる

### 無効化されない技
- ステルスロック、くろいきり、リフレクター など**場を対象とした技**
- 自分自身を対象とした技
- かたやぶり / きんしのちから で発動した変化技

### 特殊な相互作用
- マジックコート で跳ね返された技も無効化
- 複数対象技で巻き込まれた場合、このポケモンのみ防ぐ
- みちづれ / スレッドトラップ など「対象に直接選ばれていない」効果は防がない

## 処理順・優先度
- 入場時に自動判定

## テスト項目

- [ ] 敵の変化技が無効化される
- [ ] 自分を対象とした技は無効化されない
- [ ] ステルスロック など場対象技は防がない
- [ ] かたやぶり で無効化できない
- [ ] マジックコート で跳ね返された技も無効化
