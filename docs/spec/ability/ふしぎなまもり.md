# 仕様書: ふしぎなまもり

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B5%E3%81%97%E3%81%8E%E3%81%AA%E3%81%BE%E3%82%82%E3%82%8A

ふしぎなまもり は、ポケモンの とくせい の一種。
## 効果
効果バツグン ではない タイプ の 攻撃技 による ダメージ を無効化する。
## 説明文
- 第三世代: こうかばつぐん しか あたらない
- 第四世代 ・ 第五世代 ・ 第六世代: こうかばつぐん しか あたらない。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは こうかばつぐん になる タイプの わざ いがい では ダメージを うけることが ないんだ きょうりょくな とくせいだけど…… じょうたいいじょうや てんきによる ダメージは うけて しまうので ちゅうい すること
- 第七世代 ・ 第八世代 ・ 第九世代: こうかばつぐんの わざしか あたらない ふしぎな ちから。

## こんなときに使おう
特定のタイプ以外の攻撃技すべてを無効化する効果により、このポケモンに対抗できる相手を限定できる。 スキルスワップ 以外の 変化技 ・相手の 特性 や 状態異常 によるダメージなどは通してしまう。 サーナイト や ポリゴン2 などに トレース されると、こちらからダメージを与える手段が限定されることにも注意が必要である。
## ポケモンカードゲームにおけるふしぎなまもり
- ヌケニン (ADV3-B) の ポケボディー として登場。
## 特性の仕様
### 防げる技について
- 変化技 は無効化できない。その技のタイプとの 相性 は関係しない。
  - 相性の影響を受ける変化技の でんじは であっても、ふしぎなまもりで無効化することはできない。
- 攻撃技 であれば、 ダメージ固定技 / 一撃必殺技 であっても効果抜群ではないタイプの技なら無効化できる。効果抜群になるタイプの技は無効化できない。
- 状態異常 / 状態変化 / 場の状態 によるダメージや、 特性によるダメージ など、攻撃技以外による ダメージ は受ける。
  - 自分の持ち物 いのちのたま / くろいヘドロ / くっつきバリ や、相手の持ち物 ゴツゴツメット / ジャポのみ / レンブのみ によるダメージも受ける。
  - 反動技 や、 いのちがけ など ひんしになる技 を使用したことによるダメージも受ける。
  - ニードルガード によるダメージも受ける。
  - はじけるほのお の火花よるダメージも受ける。
- こんらん による自傷ダメージは無効化できない。
- わるあがき は無効化できない。
- かたやぶり の効果が発動した技、 特性を無視する攻撃技 によりふしぎなまもりは無視され、抜群でなくてもダメージを受ける。
- 特性所有者の タイプ が変わった場合や、 さかさバトル などで相性判定そのものが変わった場合は、それに応じてふしぎなまもりで防げるタイプも変化する。
  - もえつきる などにより タイプなし になった場合、わるあがきとかたやぶり効果がある技以外の全ての攻撃技を無効化する。
  - ひこう タイプである場合、 らんきりゅう の効果で等倍になったひこうタイプの弱点技は無効化する。
  - みず タイプである場合、抜群になった フリーズドライ のダメージは受ける。
  - タールショット 状態により ほのお タイプが弱点になった場合、ほのお技を無効化できなくなる。
  - じめん タイプを弱点とするタイプになった場合、じめん技を無効化できなくなるが、 ふうせん や でんじふゆう により 地面にいない 状態になると、まるでふしぎなまもりの効果で防いだかのように特性バーを表示しながらじめん技を無効化する。
  - 抜群技に対して オッカのみ などの半減系きのみが発動するときも相性自体は変わらないため、特性で防ぐことはできない。
- ポケモンコロシアム や ポケモンXD に登場する ダークわざ はコロシアムではタイプがなく、XDではダークポケモン以外に効果は抜群であるため無効化できない。
- 第四世代 のみ、効果抜群にならなくとも ほのおのキバ のダメージは受ける。
- 第四世代までの ふくろだたき / みらいよち / はめつのねがい は無効化できない。 第五世代 以降は効果抜群であれば無効化できないが、効果抜群でなければ無効化する。
- タイプなし のポケモンが使用する めざめるダンス は無効化できる。
### 特性の変更
- この特性を上書きすることはできる。
  - なやみのタネ / シンプルビーム / なかまづくり / ミイラ により 特性 は上書きされる。 かがくへんかガス により特性は無効化される。
  - かがくへんかガスのポケモンが ひんしになる技 を使用したときも、その攻撃に対してふしぎなまもりは発動しない。
  - いえき を受けると とくせいなし 状態になる。
  - コアパニッシャー を抜群で受けて技の効果が発動すると特性を消される。しかし特性の効果で攻撃を無効にしたときはとくせいなしにもならない。
  - 特性所有者が使う なりきり は成功し、ふしぎなまもりを他の特性に上書きできる。
- この特性は スキルスワップ / さまようたましい で交換することができない。
  - 特性所有者がとくせいなし状態のときや、かがくへんかガスが発動している間でも入れ替えはできない。
  - かたやぶりの効果があるスキルスワップでも特性を入れ替えることはできない。
  - 特性所有者が使うスキルスワップも失敗する。
- この特性をコピーする効果は、手段によって成否が異なる。
  - 以下の方法でコピーすることはできる。
    - この特性と対面した相手は、特性 トレース でコピーすることができる。
    - 特性所有者が使う なかまづくり で相手に特性をコピーすることも可能となる。
  - 以下の方法でコピーすることはできない。
    - この特性に対する なりきり は失敗する。
    - この特性に対して レシーバー / かがくのちから の効果は発動しない。
## 備考
- ヌケニン の 専用特性 。
- 以下の条件でヌケニン以外のポケモンにこの特性を持たせることはできる。
1. 特性 トレース を持つポケモンなら、相手の場にいるヌケニンのふしぎなまもりをコピーできる。
2. トレースをスキルスワップで他のポケモンに移し、移した相手がトレースでふしぎなまもりをコピーすれば、スキルスワップが有効な任意のポケモンの特性をふしぎなまもりにできる。以下 第五世代 まで弱点が1つも無かった ミカルゲ や ヤミラミ に持たせることを考える。第五世代でのみトレースをスキルスワップで移しても発動しないため、 第四世代 か 第六世代 以降である必要がある。
  - 「 いえき が使えるポケモン」「 バトンタッチ が使えるポケモン」「特性 トレース を持ち、 スキルスワップ が使えるポケモン(以下 サーナイト とする)」の3体がミカルゲ(ヤミラミ)自身の他に自分の手持ちにいること、 ダブルバトル 以上であり、相手が ヌケニン を場に出していること。
  - まずいえき持ちとバトンタッチ持ちを出し、いえきをバトンタッチ持ちに使い、 とくせいなし 状態にする。
  - バトンタッチ持ちはとくせいなし状態のままサーナイトにバトンタッチする(こうする事で相手の特性をトレースしないまま場に出られる)。同時にいえき持ちをミカルゲ(ヤミラミ)と交代させる。
  - サーナイトはミカルゲ(ヤミラミ)を対象にしてスキルスワップをする。そうするとミカルゲ(ヤミラミ)が特性トレースを得る。
  - 自動的に、とくせいなし状態でないミカルゲ(ヤミラミ)は特性トレースの効果を発揮し、相手のヌケニンのふしぎなまもりをトレースする。相手の場にヌケニン以外のポケモンがいても、そのポケモンがトレースできない特性であれば確実にふしぎなまもりをトレースできる。
3. 第三世代 の教え技で ものまね を覚えたヌケニンを 第五世代 以降に連れて来て、他のポケモンが使った なかまづくり を真似すれば、なかまづくりが有効な任意のポケモンの特性をふしぎなまもりにできる。
- ポケウッド の ゴーストイレイザー4 にはゴースト・あくタイプで特性がふしぎなまもりのセットが登場するが、こちらは デスカーン の ミイラ か オクタン の みずびたし で特性かタイプを変えて対処するよう台本にも書かれている。
