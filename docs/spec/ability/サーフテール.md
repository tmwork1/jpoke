# 仕様書: サーフテール

参照URL: https://wiki.pokemonwiki.com/wiki/サーフテール

## 基本効果
エレキフィールド状態のとき、すばやさが 2 倍になる。

## 詳細仕様

### 発動条件
- 場が エレキフィールド の間のみ有効。
- 所有者が非接地でも、サーフテールの速度補正は有効。

### すばやさ計算
- ランク補正後のすばやさ実数値に 2 倍補正を適用する。
- まひによる速度低下など、他の補正との合成順は既存の速度計算ルールに従う。

### 世代差分メモ
- 第8世代以降は、特性を得失したターンから行動順へ反映される。

## 処理順・優先度
- 行動順決定前の速度計算で適用。
- フィールド変更が同ターン内で起きた場合は、行動順の再評価タイミングに従う。

## テスト項目

- [ ] エレキフィールドで すばやさ が 2 倍になる
- [ ] エレキフィールドでないときは補正されない
- [ ] 非接地状態でも補正が有効
- [ ] まひ時の速度計算が既存ルールどおりになる
- [ ] フィールド変化直後の行動順更新が正しく行われる
