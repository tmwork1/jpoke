# 仕様書: ありじごく

参照URL: https://wiki.pokemonwiki.com/wiki/ありじごく

## 基本効果
相手を交代・逃走しにくくする拘束系特性。場にいる間、条件を満たす相手は通常の交代を選べない。

## 詳細仕様
- 交代禁止は場にいる間のみ有効。
- ゴーストタイプや、拘束無効条件を持つ相手には適用しない。
- とんぼがえり / ボルトチェンジ など技による交代は技仕様を優先する。

## 処理順・優先度
- 交代可否判定時に適用する。

## テスト項目
- [ ] 条件を満たす相手が通常交代できない
- [ ] ゴースト等の無効対象は交代できる
- [ ] ありじごく側が退場すると交代制限が解除される
