# 仕様書: いろめがね

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%84%E3%82%8D%E3%82%81%E3%81%8C%E3%81%AD

いろめがね は、ポケモンの とくせい の一種。
## 効果
使用した技が「 効果は今ひとつ 」だったとき、与える ダメージ が2倍になる。相性判定そのものは変化しない。
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代: こうかいまひとつ のとき わざを つよめる。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは こうかはいまひとつの ときに あたえる ダメージが ばいになるんだ
- 第七世代 ・ 第八世代 ・ 第九世代: こうかが いまひとつの わざを つうじょうの いりょくで だすことが できる。

## こんなときに使おう
相手はタイプ 相性 で技を受けることが難しくなる。ただし最大でも等倍のダメージにしかならないので、ステータス数値の高さで技を受けられることはある。相手の弱点を突けるなら特性の発動にこだわる必要はない。半減されることが多くても無効化されることはないタイプの技を使うときに特性が活きやすいといえる。
## 特性の仕様
- 技が無効化されたときは特性の効果も発動しない。
- 相手が複合タイプを持つときは、最終的な相性が今ひとつのときに効果が発動する。片方のタイプに対して半減でも、もう片方に対して抜群のため最終的な相性が等倍になったときは特性は発動しない。
- 全体攻撃技を使用した場合、効果が今ひとつだった相手に対してのみ与えるダメージが増える。
- 第五世代 以降は みらいよち / はめつのねがい の ダメージ 発生時に使用者の特性がいろめがねで場に出ているならば、いろめがねの効果は発動する。
- もりののろい / ハロウィン で追加されたタイプや、 タールショット 状態で追加された弱点もいろめがねの発動に影響する。
- らんきりゅう の効果で等倍の技を今ひとつにされたときでもいろめがねは発動する。
- テラスシェル の効果で等倍以上の技を今ひとつにされたときでもいろめがねは発動する。
- フリーズドライ / フライングプレス による相性自体が変わる効果は、いろめがねの発動に影響する。
  - フライングプレスは最終的な相性が今ひとつのときに特性の効果が発動する。かくとう/ひこうのどちらかが半減でも、もう片方が抜群のため等倍になったときは発動しない。
- 特性 あついしぼう / たいねつ や ホズのみ 等半減系きのみのように、相性判定が変わらない効果はいろめがねの発動に影響を与えない。
## 備考
- 単純計算では相性が0.5倍のときは1倍に、0.25倍のときは0.5倍になる効果だが、 ダメージ 計算において相性補正の位置といろめがねの補正位置は違うため、相性で与えるダメージが2倍になったときと比べてダメージは異なることがある。
- 第九世代 現在では、この特性を持つポケモンは むし タイプか ひこう タイプを持っている。
  - 通常特性で持つポケモンはむしタイプのみ。
