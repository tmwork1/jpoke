# 仕様書: てんきや

調査日: 2026-02-07

てんきや は、ポケモンの とくせい の一種。
## 効果
天気 に対応した姿に フォルムチェンジ し、 タイプ が変わる。
| フォルム名 | タイプ | 天気 |
| --- | --- | --- |
| ポワルンのすがた | ノーマル | 通常 下記以外の天気 |
| たいようのすがた | ほのお | にほんばれ おおひでり |
| あまみずのすがた | みず | あめ おおあめ |
| ゆきぐものすがた | こおり | あられ |
## 説明文
- 第三世代: てんきで ポワルンが へんかする
- 第四世代 ・ 第五世代 ・ 第六世代: てんきで ポワルンが へんかする。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポワルンは てんきで タイプが へんかするんだ ひざしがつよいと ほのおタイプ あめだと みずタイプ あられなら こおりタイプ ……って かんじにね
- 第七世代 ・ 第八世代 ・ 第九世代: てんきの えいきょうを うけて みずタイプ ほのおタイプ こおりタイプの どれかに へんかする。

## 特性の仕様
- すなあらし / くらやみ / きり / らんきりゅう 状態に対応した フォルム は無い。
- フォルムチェンジ 特性であり、特性が発動して姿が変わるたびに以下の効果がある。
  - みずびたし / もりののろい / ハロウィン / ミラータイプ などによりタイプが変化していた場合、これらの効果を無くす形で天気に対応した姿/タイプに変わる。
  - パワーシェア / ガードシェア / パワートリック / スピードスワップ により ステータス が変化していた場合、元のステータスに戻る。
- 他の多くの フォルムチェンジ特性 とは異なり、この特性を上書きすることや交換することはできる。技 なかまづくり / スキルスワップ / なやみのタネ / シンプルビーム ・特性 ミイラ の効果でてんきやは失われる。 いえき / コアパニッシャー を受けると とくせいなし 状態となる。
  - フォルムチェンジ後にこの特性を失った場合の挙動は世代により異なる。コピーが可能であるかもは世代により異なる (下記参照)。
  - この特性のポケモンが なりきり を使用したときは成功し、てんきやは失われる。
  - かがくへんかガス が発動している間は、ポワルンはフォルムチェンジできなくなる。天候が変動中にかがくへんかガスの効果が切れたときは、即座にフォルムチェンジを発動できる。
- 天候が変わった瞬間に特性が発動する。回数制限はないため、特性や技で1ターンの間に天候を何度も変えた場合、1ターンの間に何度も特性が発動することになる。
- ステルスロック のダメージはてんきやの発動前に受ける。たいよう/ゆきぐものすがたにフォルムチェンジするときも、最大HPの1/8のダメージで済む。
- 場に出たとき発動する特性 の効果で天気を発動させるポケモンが同時に複数場に出てきた場合、最終的な天気が決まるまでてんきやは発動しない。
- 場にいる複数のポケモンのてんきや/ フラワーギフト が同時に発動する場合、 補正込みのすばやさ が高い順に特性が発動する。
- フォルムチェンジ中、場に ノーてんき / エアロック のポケモンが出てきて天候が無効化された場合、即座にポワルンは通常の姿に戻る。ノーてんき/エアロックの効果が続く限り、天気が変わってもフォルムチェンジできない。
- かたやぶり 効果を持つ技でも、この特性を無視することはできない。
### 第三世代・第四世代のみ
- へんしん で他のポケモンがポワルンに へんしん した場合も特性の効果でフォルムチェンジできる。
- ポワルンがてんきやを失った場合、ポワルンはそのままの姿を維持する。ポワルンにへんしんしたポケモンも同様。
  - 再び特性がてんきやに戻った場合、てんきやが発動して対応した姿に変わる。天候に沿った姿である場合でも発動してその姿になる。
- この特性に対する なりきり は成功する。 トレース は 第三世代 では発動するが、 第四世代 では発動しない。
### 第五世代以降
- へんしん / かわりもの で他のポケモンがポワルンに へんしん した場合、天気が変わってもコピーした姿のままでフォルムチェンジできない。
- ポワルンがてんきやを失った場合、ポワルンは通常の姿に戻り、フォルムチェンジできなくなる。
  - 再び特性がてんきやに戻った場合は、またフォルムチェンジ可能になる。
- この特性は コピーできない 。
  - この特性に対する なりきり は失敗し、 トレース も発動しない。
  - この特性が使用する なかまづくり は失敗する。この特性がなかまづくりを受けたときはてんきやは失われる。
  - この特性に対して かがくのちから / レシーバー は発動しない。
- 第六世代 以降では、フィールドに自然発生した永続天候を技や特性で一時的に書き換えていた場合、一時的な天候のターンが切れた時点で一旦天候なし状態となり、その後改めて永続天候が再発生する。このため、天候が切れた時点で一旦てんきやが発動して通常フォルムに戻り、永続天候が再発生した時点で再度てんきやが発動してフォルムが変わる。
  - 例えばたいようのすがたのとき、 おわりのだいち の特性を持つポケモンが他のポケモンと スキルスワップ で特性を入れ替えた場合、特性が入れ替わり一瞬天候が戻った際にポワルンは通常フォルムに戻る。もう片方のポケモンが特性を再発動することで天気がおおひでりに戻り、ポワルンのフォルムもまたたいようのすがたに戻る。
## 備考
- ポワルン の 専用特性 。
- 名前の元ネタは、その時々で気分が変わりやすい人を表す俗語「お天気屋」か。
