# 仕様書: すいほう

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%99%E3%81%84%E3%81%BB%E3%81%86

すいほう は、ポケモンのとくせいの一種。
## 効果
- ほのお タイプの攻撃技を受けるとき、相手の こうげき ・ とくこう が半減して ダメージ を計算する。
- みず タイプの攻撃技を使うとき、自分のこうげき・とくこうを2倍にしてダメージを計算する。
- やけど 状態にならない。やけど状態になっても回復する。
## 説明文
- 第七世代 ・ 第八世代 ・ 第九世代: じぶんに たいする ほのおタイプの わざの いりょくを さげる。 やけど しない。
## 所有ポケモン：そのポケモンの別のとくせい
|  | 特性1 | 特性2 | 隠れ特性 |
| --- | --- | --- | --- |
| シズクモ | すいほう | - | ちょすい |
| オニシズクモ | すいほう | - | ちょすい |
## こんなときに使おう
攻撃・特攻自体はあまり高くない オニシズクモ だが、みず技に限れば高ダメージが期待できる。本来等倍のほのお技も半減相当のダメージに抑えて受けることができる。物理アタッカーでもやけどになる心配が無くなる。
ダブルバトル では なかまづくり で味方の特性をすいほうに上書きすることもできる。
## 特性の仕様
- ほのお 技を受けるときの 相性 判定自体は変わらない。
- かたやぶり の効果が発動した技に無視され、ほのお技を半減することができない。
  - このときすいほうのポケモンも やけど 状態にすることができるが、すいほうの効果により即座に回復する。同時に ラムのみ / チーゴのみ を持っていた場合、道具が優先して消費される。
  - すいほうのポケモンが、かたやぶりのポケモンに対してみず技で攻撃する場合は、すいほうの効果が無視されることはない。
- すでにやけど状態になっているポケモンの特性がすいほうに書き換わった場合もすいほうの効果によりやけどは即座に回復する。
- かえんだま を持っていても発動しない。
- みず 技を使用したときの効果については、タイプや補正率が異なるが はがねつかい と似通った使用がある。発動条件の詳細は はがねつかい#特性の仕様 を参照とする。
## 備考

- みずのベール の上位互換とも言える特性。
