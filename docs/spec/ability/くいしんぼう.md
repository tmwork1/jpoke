# 仕様書: くいしんぼう

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8F%E3%81%84%E3%81%97%E3%82%93%E3%81%BC%E3%81%86

くいしんぼう は、ポケモンの とくせい の一種。
## 効果
本来は残りHPが最大 HP の1/4以下になったときしか発動しない きのみ を、最大HPの1/2以下になったときに発動できる。
もとから最大HPの1/2以下で発動するきのみ（ オボンのみ など）に対しては効果は無い。発動条件が残りHPでないきのみ（ ラムのみ など）も同様に効果は無い。
対象となるきのみは以下。
- フィラのみ ( 第七世代 以降)
- ウイのみ (第七世代以降)
- マゴのみ (第七世代以降)
- バンジのみ (第七世代以降)
- イアのみ (第七世代以降)
- チイラのみ
- リュガのみ
- カムラのみ
- ヤタピのみ
- ズアのみ
- サンのみ
- スターのみ
- ミクルのみ
- イバンのみ
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代: きのみを いつもより はやく つかう。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは たいりょくが はんぶんいかに なると もっている きのみを たべるんだ
- 第七世代 ・ 第八世代 ・ 第九世代: HPが すくなくなったら たべる きのみを HP はんぶんの ときに たべてしまう。

## こんなときに使おう
カムラのみ や、 第七世代 以降での 混乱木の実 をより早く使うことができる。
## ポケモン不思議のダンジョンにおけるくいしんぼう
- ポケモン不思議のダンジョンでは本編とは異なり、お腹の減りが通常より早くなる。
- マグナゲート 以降では道具を食べたり、飲んだりすると、HPが回復する仕様になった。

## 詳細な仕様
- 他の特性を持つポケモンのHPが1/4超-1/2以下のときに、特性がくいしんぼうに書き換わると、その瞬間に対応するきのみを食べることができる。
- この特性のポケモンが場に出た時点ですでにHPが1/4超-1/2以下である場合、その瞬間に対応するきのみを食べることができる。
- かたやぶり の効果がある技を受けてHPが1/4超-1/2以下になったときもきのみを発動できる。
- HPが1/4超、1/2以下になったが きんちょうかん の効果できのみを食べられなかった場合、きんちょうかんの効果が切れた瞬間にきのみを食べることができる。
- HPが1/4超-1/2以下になったが かがくへんかガス の効果できのみを食べられなかった場合、かがくへんかガスの効果が切れても イバンのみ 以外の きのみを食べることはできない。この状態になっても以下の要因できのみを発動できる。
  - ミクルのみ 以外は回復したり 変化技 を受けたりしても発動しないが、定数ダメージを含めた何かしらのダメージを受けると発動する (HPが1/4以下にならなくてもよい)。
  - ミクルのみは自身が行動を終えた後に発動する。 状態異常 ・ 状態変化 などにより行動ができなかったときでも発動する。
- HPが1/4超-1/2以下のとき、かがくへんかガスのポケモンが 交代 して場から去ったターンにイバンのみは発動しない。逆に、交代してかがくへんかガスのポケモンが出てくるターンにイバンのみは発動できる。
- 特性 ミイラ / さまようたましい / とれないにおい のポケモンを 直接攻撃 したとき、相手の ゴツゴツメット / ジャポのみ / レンブのみ や、 反動技 のダメージでHPが1/4超-1/2以下になった場合、特性の変更が優先されてきのみは発動しない。
