## 2. 基本効果（1段落）
みずタイプの技を無効化し、最大HPの1/4を回復する（小数点以下切り捨て）。

## 3. 詳細仕様
- 発動条件: 対象に向けられたみずタイプ技が有効対象であること。
- 効果内容: ダメージ・追加効果を受けず、回復処理のみ行う。
- 回復制限: HP満タン時は無効化のみ成立し、回復量は0。
- 対象外: 場全体対象の天候技等には発動しない。
- 無効化条件: かがくへんかガス中は発動せず、かたやぶり効果付き技では無視される。

## 4. 処理順・優先度
- 適用タイミング: 被弾判定時、ダメージ計算より前の無効判定。
- 同時効果: 連続技でも回復は1回のみ。
- 防御技との関係: まもる等で技自体を防いだ場合は発動しない。

## 5. テスト項目（実装検証用）
- 発動ケース: 単体みず技を無効化し1/4回復。
- 非発動ケース: かたやぶり由来みず技、かがくへんかガス中。
- 境界ケース: HP満タン時、連続技、みがわり時の挙動。
