# 仕様書: ミラーアーマー

## 1. 基本情報
- 参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9F%E3%83%A9%E3%83%BC%E3%82%A2%E3%83%BC%E3%83%9E%E3%83%BC
- 種別: 能力ランク低下反射

## 2. 基本効果（1段落）
相手から受ける自分の能力ランク低下効果を、対象を相手へ反転して跳ね返す。ランク低下以外の付随効果は通常どおり受ける。

## 3. 詳細仕様
### 発動条件
- 低下元が相手由来で、自分の能力ランクを下げる効果であること。

### 非発動条件
- 自分で下げる効果（からをやぶる等）。
- 低下が無効化される状況（みがわり、しろいきり、既に最低ランク等）。

### 無効化・上書き
- 特性無効化中は通常どおり能力が下がる。
- かたやぶり系で無視される場合は反射しない。

### 交代・入場時
- 常時効果。場にいる間だけ有効。

### その他の特性との相互作用
- まけんき/かちき等の「相手に下げられた」判定と整合するようイベント属性を保持する。

## 4. 処理順・優先度
- 発動タイミング: 能力ランク低下適用直前。
- 処理: 低下対象を相手へ置換して適用、元対象の低下を取り消す。
- すばやさは参照しない。

## 5. テスト項目（実装検証用）
### 発動ケース
- いかく等で下がるはずの攻撃を反射し、相手の攻撃が下がる。
- 低下付き攻撃の追加効果を反射する。

### 非発動ケース
- 自己デメリット技の能力低下は反射しない。
- かたやぶり系には反射しない。

### 境界ケース
- 相手能力が最低ランク時、メッセージと実ランクが矛盾しない。
