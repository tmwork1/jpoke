# 仕様書: かんそうはだ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%93%E3%81%9D%E3%81%86%E3%81%AF%E3%81%A0

かんそうはだ は、ポケモンのとくせいの一種。
## 効果
- みず タイプのわざを受けても無効化し、代わりに最大 HP の1/4を回復する (小数点以下切り捨て)。
- ほのお タイプの 攻撃技 を受けたときの 威力 が5/4倍になる。
- 天気 が あめ ・ おおあめ 状態のとき、毎 ターン 終了時に最大HPの1/8を回復する (小数点以下切り捨て)。
- 天気が にほんばれ ・ おおひでり 状態のとき、毎ターン終了時に最大HPの1/8のダメージを受ける (小数点以下切り捨て)。
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代: あついと HPが へる。 みずで HPを かいふく。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは みずタイプの わざを うけたときや てんきが あめの ばあい たいりょくが かいふくするんだけど…… ほのおタイプの わざを うけたときや ひざしがつよい てんきの ばあいは たいりょくが へって しまうから ちゅうい すること
- 第七世代 ・ 第八世代 ・ 第九世代: てんきが あめのときや みずタイプの わざで HPが かいふくし はれのときや ほのおタイプの わざで へってしまう。

## こんなときに使おう
みず技を無効化するため 雨パ 対策になるが、雨パに加えても特性のデメリット効果をカバーできるため相性がよい。
## 特性の仕様
### 技を受けたときの効果
- 以下のように、 みず 技を無効化する効果は ちょすい と共通している。詳細な仕様は ちょすい#特性の仕様 を参照とする。
  - みずタイプなら 変化技 を受けたときも発動して無効化する。
  - 使用者自身を対象にする技や、場を対象とするみず技を無効化して回復することはない。
  - まもる 状態などでみず技を防いだときや、 おおひでり でみず技が不発したときは回復できない。
- 技 や 特性 の効果でタイプが変わった技を受けた場合、変更後のタイプがみずタイプであれば無効化し、 ほのお タイプであれば威力が上がる。
- みがわり 状態でみず技を受けても発動しHPを回復する。受けるほのお技の威力は上がる。
- ノーてんき / エアロック / ばんのうがさ により天気による効果が無効化されているときでも、みず技やほのお技を受けたときの効果は発動する。
### 天気に関する効果
- あめ 状態が終了するターンでは回復しない。 にほんばれ 状態が終了するターンにダメージは受けない。
- 第五世代 以降では、 かいふくふうじ 状態のときは回復できない。
  - かいふくふうじ状態が終了するターンでもあめ状態での回復ができない。
- 自身が 姿を隠している ときは、以下のように扱う。
  - あなをほる 状態/ ダイビング 状態で隠れている間は、 第五世代 以降は天気による回復とダメージのどちらも発生しない。
    - 第四世代 のみ、にほんばれ状態によるダメージは受けない一方で、あめ状態によるHPの回復はできる。
  - そらをとぶ 状態/ シャドーダイブ 状態で隠れている間は、天気による回復・ダメージのどちらも発生する。第四世代でも同様となる。
- 特性 ノーてんき / エアロック のポケモンが場にいるときは天気による回復・ダメージが発生しない。
- ばんのうがさ を持っているときは天気による回復・ダメージが発生しない。
- みがわり 状態でも天気による回復・ダメージが発生する。
### その他
- かたやぶり 効果があるみず技・ほのお技を受けてもかんそうはだは発動しない。
  - ただし、かたやぶりのポケモンがあめ・にほんばれ状態を発動させたとしても、 場の状態 にかたやぶりの効果は無いので、天気による回復・ダメージどちらも発生する。
- 第四世代まではHPが満タンの状態でみず技を受けると「<ポケモン名>の かんそうはだで <技名>は こうかが なかった!」、 HPが減っているときにみず技を受けると「<ポケモン名>は かんそうはだで かいふくした!」、HPが減っているときにあめで回復すると「<ポケモン名>は かんそうはだで かいふくした!」、にほんばれでダメージを受けると「<ポケモン名>は かんそうはだで ダメージを うけた!」とメッセージが出る。第五世代以降でHPが満タンの状態でみず技を受けるとかんそうはだの特性バーと「<ポケモン>には 効果が ないようだ...」、HPが減っているときにみず技を受けるとかんそうはだの特性バーと「<ポケモン>の 体力が 回復した!」というメッセージが出て、HPが減っているときにあめで回復するときやにほんばれでダメージをうけるときは特性バーが現れるだけメッセージは出ない。いずれの世代でもほのお技を受けたときと、HPが満タンであめ状態のときにかんそうはだに関するメッセージは現れない。
## 備考
- 同じくみず技を無効化する特性として ちょすい 、 第五世代 以降の よびみず がある。
- 同じく天気でHPを回復する特性として あめうけざら 、 アイスボディ がある。天気でダメージを受ける特性として サンパワー がある。
