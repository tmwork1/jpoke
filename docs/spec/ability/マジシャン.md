# 仕様書: マジシャン

参照URL: https://wiki.pokemonwiki.com/wiki/マジシャン

## 基本効果
| 項目 | 内容 |
|---|---|
| 効果 | 攻撃技でダメージを与えたとき、相手の持ち物を奪う |
| 発動制限 | 自身が道具を所持している場合は発動しない |
| 発動対象 | 相手をひんしにした場合でも発動 |
| 攻撃種別 | 物理・特殊・接触・非接触を問わない |

---

## 詳細仕様
### 発動条件
| 条件 | 詳細 |
|---|---|
| 技命中 | 攻撃技が命中しダメージを与えた場合 |
| 所持条件 | 自身が道具未所持 |
| 相手状態 | 相手が道具を所持している |

### 無効条件
| 分類 | 内容 |
|---|---|
| 技分類 | 変化技では発動しない |
| ダメージなし | 無効化・まもる・タイプ相性等でダメージが0の場合 |
| みがわり | みがわりに防がれた場合（貫通時は発動） |
| 回復技 | プレゼント・かふんだんごで回復効果の場合 |
| 特殊ダメージ技 | なげつける、しぜんのめぐみ、みらいよち、はめつのねがい |

---

## 処理順・優先度
| 順序 | 内容 |
|---|---|
| 1 | 技命中・ダメージ計算 |
| 2 | ダメージ確定 |
| 3 | マジシャン発動（道具奪取） |

### 詳細処理
| ケース | 処理 |
|---|---|
| 相手ひんし | ひんし後でも発動 |
| 一撃必殺技 | 第9世代では発動対象 |
| 複数対象技 | 命中した対象ごとに判定 |

---

## 競合・上書き・例外
### 技との競合
| 技 | 挙動 |
|---|---|
| どろぼう / ほしがる | 技の奪取処理と競合（通常は技側が優先） |
| はたきおとす | 道具消失が先に適用される |

### 特性との競合
| 特性 | 挙動 |
|---|---|
| ねんちゃく | 発動はするが奪えない |

## テスト項目
- [ ] 攻撃技でダメージを与えたとき自分が道具未所持なら奪取する
- [ ] 変化技やダメージ0では発動しない
- [ ] みがわりに防がれた場合は発動しない
- [ ] 相手ひんし時でも条件を満たせば発動する
