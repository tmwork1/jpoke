# 仕様書: きゅうばん

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%8D%E3%82%85%E3%81%86%E3%81%B0%E3%82%93
調査日: 2026-04-30

## 効果

きゅうばんは、ポケモンのとくせいの一種。

相手を交代させる技やレッドカードによる交代を防ぐ。

## 説明文

- **第七世代・第八世代・第九世代** 吸盤で 地面に 張り付き ポケモンを 入れ替えさせる 技や 道具が 効かなくなる。

## 特性の仕様

- 自分からの交代は制限されない。特性きゅうばんのポケモンであっても使うと交代するわざ、だっしゅつボタン・だっしゅつパックは使用できる。
- 相手のレッドカードに対しても発動し、レッドカードの効果で交代させられない。この場合もレッドカードは消費される。
- ともえなげ・ドラゴンテールを受けた場合、ダメージを受けた後に「\[ポケモン\] は きゅうばんで はりついている！」というメッセージが出て交代を防ぐ。
- かたやぶりの効果がある技を受けたとききゅうばんは無視され、交代させられる。
- ダイマックスしているときは、ダイマックスの天覇の力による無効が優先され、きゅうばんに関するメッセージは出ない。

## 備考

- フィールド上で同様の効果のあるとくせいとして、ねんちゃくがある。

## 関連項目

- ねをはる（状態変化）
- ばんけん
- ねんちゃく
- 発動ケース: 「相手の交代させる技の効果を受けない」が成立する条件で期待どおりに発動すること。
- 非発動ケース: 条件を満たさない場合や無効化条件下で発動しないこと。
- 境界ケース: 交代直後、特性無効化、同時処理競合時に破綻しないこと。
