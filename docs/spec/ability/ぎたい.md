# 仕様書: ぎたい

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8E%E3%81%9F%E3%81%84

ぎたい は、ポケモンの とくせい の一種。
## 効果
フィールド が変化したとき、自身が以下の単 タイプ に変化する。フィールドが無いときは本来のタイプに変化する。
- フィールド: タイプ
- エレキフィールド: でんき
- グラスフィールド: くさ
- ミストフィールド: フェアリー
- サイコフィールド: エスパー
## 説明文
- 第八世代 ・ 第九世代: フィールドの じょうたいに あわせて ポケモンの タイプが かわる。

## こんなときに使おう
## 特性の仕様
- 交代 するか、 ひんし になるか、戦闘が終了するとタイプは元に戻る。
- 自身が 地面にいない ときでも発動する。
- 特性が発動するタイミングは、 フィールド が発生したとき、あるいは解除されたとき、フィールドが発生している場に繰り出されたとき、特性がぎたいに変わったときのいずれかになる。
  - 自身が場にいるときにフィールドが変化した場合は、即座にぎたいが発動する。
  - フィールドが発生している場に繰り出されたときは、 設置技 の効果や、多くの 場に出たときに発動する特性 が発動した後にぎたいが発動する ( ターン#1.ポケモンを繰り出す も参照)。
    - ステルスロック / キョダイコウジン 状態の場にマッギョを繰り出したときは、 じめん ・ はがね タイプの相性に応じたダメージを受けた後にぎたいが発動する。
    - どくびし 状態の場にマッギョを繰り出したときは、どくびしの効果がはがねタイプで無効化された後にぎたいが発動する。
    - ぎょぐん のポケモンと同時に場に出たときは、現在の すばやさ が高い順に特性が発動する。
    - すばやさ 関係によらず、 シード系アイテム の発動よりぎたいの方が優先して発動する。
    - 特性 の効果でフィールドが発生した場合は、他の処理より優先して即座にぎたいが発動する。
  - フィールドが発生しているときに他の特性からぎたいに変わると、即座にぎたいが発動してフィールドに合わせたタイプに変わる。フィールドが消えるとそのポケモン本来のタイプに戻る。
  - 現在のタイプとぎたいによる変化先のタイプが同じである場合、ぎたいは発動しない。
  - フィールドが無い状態で特性ぎたいのポケモンに へんしん すると、即座にぎたいが発動してタイプだけへんしん前のものに戻る。
  - ぎたいの特性を失ったときは、フィールドに関係なくタイプはそのまま変化しない。
- フィールドが発生中に、ぎたいのポケモンが みずびたし / まほうのこな を受けてタイプを変えられた場合や、 ミラータイプ を使用してタイプを変えた場合でも、即座にフィールドに合わせたタイプに戻ることはない。
  - みずびたし/まほうのこなでタイプが変わった後にぎたいが発動した場合、みずびたし/まほうのこなの影響は消える。
  - フィールドが無い状態で、みずびたし/まほうのこななどでタイプが変わっていたポケモンの特性がぎたいになった場合、即座にぎたいが発動して本来のタイプに戻る。
- 特性 リベロ でタイプが変わっていたポケモンの特性がぎたいになった場合、ぎたいが発動するとタイプは上書きされる。
  - フィールドが無い状態で、特性がリベロからぎたいになった場合、即座にぎたいが発動して本来のタイプになる。
- もりののろい / ハロウィン の効果はぎたいが発動しても解除されない。
  - もりののろい状態のときに グラスフィールド 状態になりくさタイプに変化したとしても、くさタイプともりののろい状態が重複して存続する。
  - もりののろいかハロウィン状態のときにフィールドが解除されて本来のタイプに戻るとき、本来のタイプがくさタイプか ゴースト タイプを持つ場合でも、そのタイプが重複して存続する。
  - ぎたいの効果により、本来のタイプと追加されたタイプが重複しているポケモンであっても、受ける ダメージ の 相性 補正はタイプを1つ持つポケモンと同様にして行われる。使用する技の タイプ一致ボーナス は1.5倍のままであり、2.25倍にはならない。
  - くさタイプともりののろい状態や、ゴーストタイプとハロウィン状態を重複して持つときでも、 ミラータイプ の効果はこれらをタイプ単体とはみなさない。
  - フィールドが無い状態で、もりののろい/ハロウィンでタイプが追加されていたポケモンの特性がぎたいになった場合では、即座にぎたいが発動することはない。
- かがくへんかガス の効果によりぎたいは発動できなくなる。ぎたいでタイプが変わっているときにかがくへんかガスの効果が発動してもタイプはそのままで元に戻ることは無く、フィールドが解除されても元のタイプに戻らない。かがくへんかガスの効果が切れると即座にフィールドに合わせたタイプに変化する。
## 備考
- マッギョ (ガラルのすがた)の 専用特性 。
- 分類 が「ぎたいポケモン」の シャリタツ はこの特性は持たない。
