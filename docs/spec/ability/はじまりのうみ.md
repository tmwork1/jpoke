# 仕様書: はじまりのうみ

参照URL: https://wiki.pokemonwiki.com/wiki/はじまりのうみ

## 基本効果
場に出たとき、天気を おおあめ にする。おおあめ中は ほのおタイプの攻撃技 が失敗する。

## 詳細仕様

### 天気の発生
- 発動条件は 場に出たとき。
- 天気は通常の あめ ではなく おおあめ として扱う。

### 天気の解除
- 場から はじまりのうみ のポケモンがいなくなると、おおあめは解除される。
- 交代・ひんし・とくせいなし化（いえき など）で解除される。

### 他の天気との相互作用
- おわりのだいち / デルタストリーム などの強天候と競合する。
- 同時発動時は、その時点の発動順で最後に発動した天気が場に残る。

### ゲンシカイキ関連
- ゲンシカイオーガがこの特性になる場合、あめふらしは挟まずに はじまりのうみ を発動する。
- 同時ゲンシカイキ時は、実数値のすばやさ順で発動順が決まる。

### 特性変更技との相互作用
- スキルスワップで強天候特性同士を入れ替えると、一瞬天気が途切れた後に再発動する。

## 処理順・優先度
- 発動タイミング: 場に出た直後（入場時特性の処理順に従う）。
- 同時発動時: 入場時特性の一般的な順序規則（すばやさ順など）を適用。
