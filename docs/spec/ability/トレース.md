# 仕様書: トレース

## 1. 基本情報
- 参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%88%E3%83%AC%E3%83%BC%E3%82%B9
- 種別: 入場時特性コピー

## 2. 基本効果（1段落）
場に出たとき、相手の有効な特性を1つ選んで自分の特性をその特性へ変更する。交代して手持ちに戻ると元のトレースへ戻る。

## 3. 詳細仕様
### 発動条件
- 入場時にコピー可能な相手特性が存在すること。
- 対象が複数いる場合はランダム選択する。

### 非発動条件
- コピー不可特性しか存在しない場合。
- 特性無効化状態等でトレース自体が発動できない場合。

### 無効化・上書き
- かがくへんかガス等で無効化中は発動しない（解除後の再判定は実装方針で定義）。
- スキルスワップ等で再度トレースを得た場合の再発動有無は世代差分に注意する。

### 交代・入場時
- 交代時にトレースへ戻る。
- 入場直後イベントで発動する。

### その他の特性との相互作用
- 入場時発動特性をコピーした際の即時発動有無は実装方針で統一する。

## 4. 処理順・優先度
- 発動タイミング: 入場時イベント（ON_SWITCH_IN系）。
- 同時発動時は既存の入場イベント優先順に従う。
- すばやさ参照は、既存入場順ロジックに従う。

## 5. テスト項目（実装検証用）
### 発動ケース
- 相手特性を1つコピーし、自身の特性が変わる。
- 相手が複数いる場合に候補内から選ばれる。

### 非発動ケース
- コピー不可特性のみの場合は不発。
- 特性無効化中は不発。

### 境界ケース
- 交代後にトレースへ戻る。
- 再入場で再度コピー判定が走る。
