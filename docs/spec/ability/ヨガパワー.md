## 2. 基本効果（1段落）
物理技のダメージ計算時に、使用者のこうげきを2倍として扱う。ちからもちと同一効果。

## 3. 詳細仕様
- 発動条件: 物理分類の技使用時に適用。
- 補正位置: 攻撃側能力値決定に直接作用。
- 対象外: 特殊技には無効。
- 技別挙動: イカサマ、ボディプレス、分類変更技などはちからもちと同じ判定系に従う。

## 4. 処理順・優先度
- 適用タイミング: ダメージ計算の攻撃能力値算出。
- 同時補正: ランク補正・道具補正・天候補正と合成して最終ダメージへ反映。

## 5. テスト項目（実装検証用）
- 発動ケース: 非所持個体比で物理ダメージが2倍相当。
- 非発動ケース: 特殊技で不変。
- 境界ケース: ちからもち共通の例外技で同値になる。
