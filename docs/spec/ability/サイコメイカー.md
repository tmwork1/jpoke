# 仕様書: サイコメイカー

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%82%B5%E3%82%A4%E3%82%B3%E3%83%A1%E3%82%A4%E3%82%AB%E3%83%BC

## 2. 基本効果
場に出たとき、サイコフィールドを5ターンの間発生させる。

## 3. 詳細仕様

### 発動条件
- 戦闘に出たときに発動する

### 効果内容
- 場をサイコフィールド状態にする
- フィールドの継続ターンは5ターン

### フィールド効果
- エスパータイプの技の威力が上昇する
- 地面にいるポケモンは先制技を受けない

### 発動タイミング
- ポケモンが登場した直後に発動する

### 重複・上書き
- 他のフィールドを発生させる特性と競合する
- 同時発動時はすばやさにより処理順が決定される
- 後から発動したフィールドで上書きされる

### 無効化・上書き
- 特性が無効化されている場合は発動しない
- 特性のコピー・交換は可能

### 交代・入場時
- 交代で出た場合も発動する
- ひんしや交代によりフィールドは消失しない場合がある

## 5. テスト項目（実装検証用）

### 発動ケース
- 場に出たときサイコフィールドが展開される
- エスパー技の威力が上昇する
- 先制技が無効化される

### 非発動ケース
- 特性無効化状態では発動しない
- すでに同一フィールドが存在する場合は再発動しない

### 境界ケース
- 他フィールド特性と同時発動時に優先順位が発生する
- フィールドは交代後も継続する場合がある
