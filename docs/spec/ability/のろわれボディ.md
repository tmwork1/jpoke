# 仕様書: のろわれボディ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AE%E3%82%8D%E3%82%8F%E3%82%8C%E3%83%9C%E3%83%87%E3%82%A3

のろわれボディ は、ポケモンのとくせいの一種。
## 効果
他のポケモンから 攻撃技 による ダメージ を受けたとき、30%の確率でその技を かなしばり 状態にする。非 直接攻撃 に対しても発動する。
## 説明文
- 第五世代 ・ 第六世代: こうげきされると たまに かなしばりにする。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは あいてから こうげきを うけると こうげき してきた わざを かなしばり じょうたいに してしまう ことが あるんだ
- 第七世代 ・ 第八世代 ・ 第九世代: こうげきを うけると あいての わざを かなしばり じょうたいに することが ある。

## 特性の仕様
- この特性で発動した かなしばり 状態は、発動したターンを含めて4ターンの間続く。
- ダメージ を受けなかったときは発動しない。
  - 変化技 などダメージが無い技に対しては発動しない。 いたみわけ に対しても発動しない。
  - 相性 / まもる 状態などで攻撃を防いだときは発動しない。
  - みがわり 状態で攻撃を防いだときは発動しない。
- みらいよち / はめつのねがい に対しては発動しない。
- 他の技が出る技 で出た攻撃技を受けた場合、基本的に「出た技」ではなく「他の技が出る技」自体をかなしばりにする。
  - ただし、 溜め技 や あばれる / ころがる / がまん など 数ターン行動する技 が出て、「他の技が出る技」を使用した次のターン以降にのろわれボディが発動する場合は、「他の技が出る技」ではなく「出た技」をかなしばりにする。他の技を経由することでかなしばりにされた技でも溜めることができるが、攻撃はかなしばり状態により失敗する。例えば オウムがえし で出た ゴッドバード を次のターンに受けた場合、ゴッドバードをかなしばりにする。かなしばり状態の間でも、オウムがえしでゴッドバードを溜めることはできるが、攻撃時にかなしばりにより失敗する。
  - 溜め技や数ターン行動する技が出た場合でも、「他の技が出る技」を出したそのターンにのろわれボディが発動したときは「他の技が出る技」自体をかなしばりにする。 パワフルハーブ などの効果で溜め技を1ターンで使用したときも同様。
- Zワザ に対しても発動するが、基本的にかなしばりにする対象はZワザ自体であり、元の技をかなしばりにすることはできない。
  - ただし、 Z変化わざ で繰り出された攻撃技に対して発動した場合は、元の技をかなしばりにする。例えば、Z しぜんのちから で出た ラブリースターインパクト を受けた場合、しぜんのちからをかなしばりにする。
  - ロトムずかん の効果で1試合中に2回Zワザが使えるようになった場合、かなしばりの対象になっているZワザでも使用することができる。
- ダイマックスわざ に対しては発動しない。
  - マックスレイドバトル で通常の技を受けたときは、 ダイマックス したポケモンに対してものろわれボディが発動することがある。
- 連続攻撃技 に対しては1発ごとに発動判定があるが、発動してもその技を中断させることはない。
- その攻撃で自身が ひんし になったときでも発動する。
- 相手が みがわり 状態でも発動する。
- 相手がすでに他の技をかなしばりにされてる場合は発動しない。
- かたやぶり の効果がある技を受けたときも発動する。
- 相手が特性 アロマベール に守られているときは発動しない。
- アロマベールに守られているポケモンから、かたやぶりの効果がある攻撃を受けた場合、かたやぶりの効果で味方のアロマベールを無効にするためのろわれボディは発動する。
## 備考
- この特性を所有するポケモンは全て ゴースト タイプを持つ。
