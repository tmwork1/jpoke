## こんなときに使おう
パーティの補完ができる任意のタイプに変更できる。 マルチアタック のタイプも自身のタイプと同様に変化するため、 タイプ一致技 として使用できる。
## 特性の仕様
- この特性により シルヴァディ のタイプが変わると、それに合わせて フォルム も変わる。
  - シルヴァディに メモリ を持たせても手持ち画面に映される小アイコンは通常時のフォルムと変わらない。 通信対戦 の見せ合い時点で相手にメモリの有無と種類を知らせることは無い。
  - バトルに出てきたシルヴァディのタイプは、見た目や相性の他、 第七世代 では下画面、 第八世代 ではじょうきょう画面の表示でも判別できる。
- シルヴァディが持っているメモリを無効化したり、シルヴァディにメモリを渡すことはできない。
  - シルヴァディに対する トリック / すりかえ / ギフトパス は、自分と相手のどちらかがメモリを持っている場合失敗する。
  - シルヴァディの持つメモリを はたきおとす / どろぼう / ほしがる / ふしょくガス で無効化したり奪ったりすることはできない。はたきおとすの威力も上がらない。
  - シルヴァディの持つメモリは マジシャン / わるいてぐせ の効果で奪うことはできない。シルヴァディに対して きょうせい でメモリを渡すことはできない。
  - さしおさえ / マジックルーム 状態の効果を受けてもシルヴァディのタイプ・姿は変わらないが、 マルチアタック を使うときはメモリの効果を受けずに ノーマル タイプになる。
- ARシステムを無効化することや、他のポケモンにARシステムを付与することはできない。
  - この特性は 上書きできない 。 とくせいなし 状態にもならず、 バトンタッチ で引き継がれるときはとくせいなし状態が消える。 かがくへんかガス の効果はシルヴァディや マルチアタック のタイプに影響を与えない。
  - この特性は コピーできない 。この特性のポケモンが なかまづくり を マジックコート 状態で跳ね返して使用しても失敗する。
  - この特性は スキルスワップ / さまようたましい で交換することができない。
- シルヴァディに へんしん したポケモンは、ARシステムもコピーすることができる。
  - へんしん使用者がメモリを持っていたとしても関係なく、対象のシルヴァディと同じフォルムになる。 マルチアタック のタイプはへんしん使用者が持つメモリに対応するタイプになる。
  - へんしんでシルヴァディの姿になったポケモンに対しては、メモリを奪ったり無効化したりすることや、メモリを渡すことができる。はたきとすの威力も上がる。
  - へんしんでシルヴァディの姿になったポケモンに対しても、特性を書き換えたり無効化したりすることはできない。
- かたやぶり の効果で無効化できない。シルヴァディは特性がなければ ノーマル タイプだが、かたやぶりの効果がある技がシルヴァディをノーマルタイプとみなすことはない。かたやぶりの効果がある技でもシルヴァディのメモリを無効化したり奪ったりすることや、ARシステムの特性を書き換えたり無効化したりすることはできない。
## 備考
- シルヴァディ の 専用特性 。
- 似たような効果の特性として、持たせた プレート でタイプが変化する アルセウス の マルチタイプ がある。
  - 英語名が「RKS System」であり、「Arceus」の発音が英語では「アルケイウス」のようになるため、ARは「アルセウス」を意味するものと考えられるが、公式からのソースはない。
