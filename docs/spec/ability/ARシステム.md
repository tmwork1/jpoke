# 仕様書: ARシステム

参照URL: https://wiki.xn--rckteqa2e.com/wiki/AR%E3%82%B7%E3%82%B9%E3%83%86%E3%83%A0

## 基本効果
シルヴァディの専用特性。持っているメモリに応じて自身のタイプとフォルムが変わる。メモリなしの場合はノーマルタイプ。

## 詳細仕様

### タイプ・フォルム変更
メモリとタイプに応じた対応：
- メモリなし（タイプ:ノーマル）：ノーマルタイプ
- ファイトメモリ：かくとうタイプ、タイプ:ファイト フォルム
- フライングメモリ：ひこうタイプ、タイプ:フライング フォルム
- （他17種のメモリ/タイプペアについて spec/ability.md の一覧表参照）
- マルチアタック のタイプはシルヴァディが持つメモリのタイプに対応する

### メモリの保護
- シルヴァディが持つメモリは トリック / すりかえ / ギフトパス で奪えない
- はたきおとす / どろぼう / ほしがる / ふしょくガス で無効化・奪取できない
- マジシャン / わるいてぐせ で奪えない
- メモリを渡すことはできない

### マジックルーム・さしおさえとの相互作用
- マジックルーム / さしおさえ 状態でもシルヴァディのタイプ・フォルムは変わらない
- ただし マルチアタック は ノーマルタイプ になる

### かたやぶり との相互作用
- かたやぶり で無効化できない
- かたやぶり 効果の技でもメモリを無効化・奪取できない
- ARシステムを書き換え・無効化できない

### 無効化・上書き
- この特性は 上書きできない 。とくせいなし 状態にもならない
- バトンタッチ で とくせいなし 状態を引き継いだ場合は消える
- かがくへんかガス でも影響を受けない（マルチアタックのタイプまで無効化されない）
- この特性は コピーできない 。スキルスワップ / さまようたましい で交換できない

### へんしん との相互作用
- へんしん したポケモンはARシステムもコピーする
- メモリの効果は「へんしん 使用者が持つメモリ」で決まる
- マルチアタック のタイプはへんしん 使用者が持つメモリのタイプになる
- へんしん状態のシルヴァディに対してはメモリを奪ったり無効化できる
- 特性は書き換え・無効化できない

## 処理順・優先度
- フォルムチェンジは場に出たとき、メモリの有無・種類に応じて自動判定
- タイミング：バトルに出た時点でタイプ・フォルムが反映される

## テスト項目

- [ ] メモリなしでノーマルタイプになる
- [ ] メモリを持たせるとそのメモリのタイプに変わる
- [ ] フォルムもメモリに対応して変わる
- [ ] マルチアタック がメモリのタイプになる
- [ ] トリック / すりかえ でメモリを奪えない
- [ ] マジックルーム 中はメモリのタイプ効果が無い（マルチアタックはノーマル）
- [ ] へんしん でARシステムをコピーし、マルチアタックはへんしん使用者のメモリで判定される
