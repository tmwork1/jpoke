# 仕様書: きよめのしお

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8D%E3%82%88%E3%82%81%E3%81%AE%E3%81%97%E3%81%8A

## 2. 基本効果
状態異常にならず、さらにゴーストタイプの技で受けるダメージが半減する。

## 3. 詳細仕様

### 状態異常無効
- 対象：どく・まひ・やけど・こおり・ねむり
- 相手・味方・自分問わず、すべての状態異常付与を無効化
- 追加効果や特性による付与も無効
- すでに状態異常である場合：
  - 特性取得後も回復はしない（新規付与のみ防ぐ）

### ゴースト技ダメージ軽減
- ゴーストタイプの攻撃技によるダメージを0.5倍にする
- タイプ相性・他補正と乗算で処理される

### 交代・入場時
- 常時効果
- 入場時の即時効果なし
- 場にいる間のみ有効

### その他の特性との相互作用
- かたやぶりの影響を受ける（状態異常無効・ダメージ軽減ともに無視される）
- 他のダメージ軽減効果と重複した場合は乗算で適用される

## 5. テスト項目（実装検証用）

### 発動ケース
- すべての状態異常が付与されない
- ゴースト技のダメージが半減される

### 非発動ケース
- 特性無効化状態で状態異常を受ける
- 特性無効化状態でゴースト技軽減が無効

### 境界ケース
- 既に状態異常の場合は治らない
- かたやぶりで状態異常・ダメージ軽減が無視される
- 他ダメージ軽減と正しく乗算される
