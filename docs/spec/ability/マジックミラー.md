# 仕様書: マジックミラー

## 1. 基本情報
- 参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%B8%E3%83%83%E3%82%AF%E3%83%9F%E3%83%A9%E3%83%BC
- 種別: 変化技反射

## 2. 基本効果（1段落）
相手から受ける対象変化技を無効化し、使用者へ跳ね返す。マジックコート相当の反射効果を常時持つ特性として扱う。

## 3. 詳細仕様
### 発動条件
- 相手の技が反射対象の変化技であること。

### 非発動条件
- 攻撃技本体や、反射対象外の変化技。
- かたやぶり系により特性を無視されるケース。

### 無効化・上書き
- 特性無効化中は反射しない。
- 特性変更後は新特性の仕様に従う。

### 交代・入場時
- 常時効果。場にいる間だけ有効。

### その他の特性との相互作用
- ミラーアーマー等の反射系と連鎖したときは、既存イベント順で無限反射を起こさないよう制御する。

## 4. 処理順・優先度
- 発動タイミング: 対象技命中前の効果適用判定。
- 処理: 反射対象なら受け手への適用を中断し、元使用者へ再適用する。
- すばやさは参照しない。

## 5. テスト項目（実装検証用）
### 発動ケース
- でんじは等の反射対象変化技を跳ね返す。
- 跳ね返し後、元使用者に効果が適用される。

### 非発動ケース
- 攻撃技は反射しない。
- 反射対象外変化技は通常どおり受ける。

### 境界ケース
- かたやぶり系を持つ相手の変化技では反射しない。
