# 仕様書: マジックミラー

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%B8%E3%83%83%E3%82%AF%E3%83%9F%E3%83%A9%E3%83%BC

マジックミラー は、ポケモンのとくせいの一種。
## 効果
戦闘中、常に マジックコート状態 になる。
## 説明文
- 第五世代 ・ 第六世代: へんかわざを かえす ことが できる。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは あいての つかった へんかわざを あいてへ そのまま はねかえすんだ しかも! あいてが つかった へんかわざは むこうに なるんだよ
- 第七世代 ・ 第八世代 ・ 第九世代: あいてに だされた へんかわざを うけずに そのまま かえす ことが できる。

## こんなときに使おう
変化技 を中心に戦うポケモンに対して有利になる。相手が変化技を使うことを読んでマジックミラーのポケモンに 交代 するという使い方もできる。この特性を持つポケモンはマジックミラーが選ばれることが大半で、見せ合いでは相手が変化技の使用そのものを控えてくれることも期待できる。
## Pokémon UNITEにおけるマジックミラー
Pokémon UNITE では、 エーフィ の特性として登場する。
## 特性の仕様
- 跳ね返せる変化技、跳ね返せない変化技については マジックコート (状態変化)#対象技 を参照とする。
- かたやぶり 系の効果がある 変化技 は跳ね返せない。
- 設置技 の変化技を受けたときは跳ね返せるが、すでに設置されているこれらの場の状態の効果を防いだり跳ね返したりすることはできない。
  - がんせきアックス / ひけん・ちえなみ など 攻撃技 による 追加効果 や、 どくげしょう など特性による効果は跳ね返せない。
- 相手のマジックミラー、およびマジックコートで跳ね返ってきた効果はマジックミラーで再び跳ね返すことはできない。
- ミラーアーマー のポケモンから ランク を下げる変化技を受けた場合、マジックミラーで跳ね返した効果はミラーアーマーで跳ね返される。マジックミラーのポケモンがミラーアーマーのポケモンのランクを下げようとした場合、ミラーアーマーで跳ね返された効果はマジックミラーで跳ね返されない。
- 自分が ダイマックス している場合、 アンコール / いちゃもん / かなしばり / ふきとばし / ほえる / なかまづくり は無効化される前にマジックミラーで跳ね返す。
## 備考
- 第九世代 時点では 隠れ特性 ・ メガシンカ 限定の特性となる。
  - メガシンカポケモンを除く所有者は全て エスパー タイプを持つ。
- スカーレット・バイオレット では マジックコート が使用できない技であるため、 マジックコート状態 になれる唯一の手段となる。
