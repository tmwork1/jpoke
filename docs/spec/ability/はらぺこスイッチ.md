# 仕様書: はらぺこスイッチ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AF%E3%82%89%E3%81%BA%E3%81%93%E3%82%B9%E3%82%A4%E3%83%83%E3%83%81

## 基本効果
モルペコの専用特性。ターン終了時に「まんぷくもよう」と「はらぺこもよう」を交互に切り替わる。モルペコが「はらぺこもよう」のときは、オーラぐるまはあく状態になる。

## 詳細仕様

### フォルムチェンジの仕様
- フォルムチェンジは ターン終了時 に発動する。スリップダメージや場の状態解除より後。
- フォルムチェンジするたびに、オーラぐるまのタイプがでんき/あくに原語される。
- 万一タイプが変更されていた場合（みずびたし など）、フォルムチェンジにより本来のタイプに戻る。
- パワーシェア / ガードシェア / パワートリック / スピードスワップ により ステータス が変更されている場合、フォルムチェンジにより本来のステータスに戻る。

### 発動タイミング
- あらゆるスリップダメージと状態変化・場の状態解除の後。
- 同じターン終了時に発動する他の フォルムチェンジ特性 より遅い。
  - ダルマモード / リミットシールド / スワームチェンジ / ぎょぐん よりは後に発動。
- モルペコ同士の場合、すばやさが高い方が先に発動。トリックルーム 状態ではその効果を考慮。

### 交代・入場時の挙動
- 交代により場に出たときは、その直後に（同ターン）はらぺこスイッチが発動してはらぺこもようになる。
- ひんし になった代わりに死に出しされたときは、そのターンの発動判定は終わっているので、次のターン終了時に発動。
- 交代で引っ込むときは、基本的にはらぺこもよう → まんぷくもよう に戻る。
  - はらぺこもよう で テラスタル後に交代した場合は、はらぺこもようを維持。次に場に出るときもはらぺこもよう のまま。
  - テラスタルしてひんしになった場合は、まんぷくもように戻る。

### 無効化・上書き
- この特性は コピーできない 。スキルスワップ / さまようたましい で交換することはできない。
- いえき により とくせいなし 状態になる。
- かがくへんかガス の効果が発動している間は、はらぺこスイッチは発動しなくなる。特性が消された場合、モルペコはその時点のフォルムを維持し変化しなくなる。
- 特性を変える技 で他の特性に上書きされると、フォルムチェンジできなくなる。

### テラスタル時の挙動
- テラスタルするとその時点のフォルムを維持し、はらぺこスイッチは発動しなくなる。

### へんしん との相互作用
- モルペコに へんしん したポケモンははらぺこスイッチで模様が変わることはない。
- はらぺこもようのモルペコに変身したポケモンは、オーラぐるまをあく タイプの技として使える（画面上では表示がでんき のまま）。

## 処理順・優先度
- タイミング：ターン終了時（スリップダメージ・状態変化解除後）
- 同じターン終了時フォルムチェンジ内での優先度：
  1. ダルマモード / リミットシールド / スワームチェンジ / ぎょぐん
  2. はらぺこスイッチ
- モルペコ同士はすばやさで順序決定（トリックルーム考慮）

## テスト項目

- [ ] フォルムチェンジ発動後、オーラぐるまのタイプが正しく変更される
- [ ] 交代で場に出たターンに はらぺこもよう に変わる
- [ ] 死に出しされたときは次のターン終了時に発動する
- [ ] かがくへんかガス 発動中は フォルムチェンジしない
- [ ] テラスタル後はフォルムが固定される
- [ ] 交代時の フォルム戻し（通常は まんぷくもよう に戻る）
- [ ] タイプ変更技で変更されたタイプが フォルムチェンジで原状復帰した際に、タイプ変更技の効果も失う
