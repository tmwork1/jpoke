調査日: 2026-02-07

ミラクルスキン は、ポケモンの とくせい の一種。
## 効果
この特性のポケモンが受ける 変化技 の 命中率 は50になる。
## 説明文
- 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代 ・ 第九世代: へんかわざを うけにくい からだに なっている。
- ブラック2・ホワイト2 （ ライブキャスター の チェレン ）: この とくせいの ポケモンは へんかわざに たいして のみ かいひりつが ばいに なるんだ じぶんで じぶんに つかう へんかわざには こうかが ないから おぼえておくと いいよ

## こんなときに使おう
変化技を中心に使う相手に強くなるが、命中率を半減させるだけなので不確実ではある。この特性を持つポケモンは、他に防御面で優秀な特性を持っていることが多いので、もちものやわざに頼らずに 状態異常 や ちょうはつ を受けずに起点を作りたい場合など、ピンポイントな採用理由になる。
## 特性の仕様
- 必中技 など、 命中 判定が無い技を回避することはできない。 あくび / ほえる / どく タイプが使用した どくどく などや、 自分が対象の変化技 、 味方の場 / 相手の場 / 全体の場が対象の技 などが該当する。
- 状態/ もちもの / ランク補正 など、 命中 が変動する他の効果はミラクルスキンの後に計算する。
  - 例えば 回避率 が6段階上がったミラクルスキンのポケモンが命中75の変化技を受ける場合、ミラクルスキンの効果で命中が50に変化した後にランク補正の効果で1/3倍 (小数点以下切り捨て) になり、結果として命中率は16%になる。
  - 詳しい仕様は 命中#第五世代以降 を参照。
- ミラクルスキンによる回避率上昇はランク補正によるものではない。特性 するどいめ / てんねん のポケモンが使用した変化技もミラクルスキンで回避することができる。
- 複数のポケモンを対象とする変化技に巻き込まれた場合、この特性のポケモンに対する命中判定にだけ特性の効果がある。
- 他の技が出る技 によって出された攻撃技に対しては、ミラクルスキンは発動しない。
- かたやぶり の効果がある変化技に対してミラクルスキンは発動しない。
- ミラクルスキンであると同時に マジックコート 状態である場合、命中判定より前にマジックコートの判定があるため、変化技を跳ね返す前にミラクルスキンの効果で回避することはない。
  - 他のポケモンのマジックコートで跳ね返ってきた変化技はミラクルスキンで回避することができる。
- ミラクルスキンの効果で技を回避した場合、技の使用者が持つ からぶりほけん は発動する。
## 備考
- この特性を通常特性として持つポケモンは シンボラー のみ。
- 命中の処理において、ミラクルスキンの効果は基本的な命中率が50を超える変化技にのみ発動するように設定されている。ただし、最も命中率が低い変化技は 第七世代 以降の ダークホール の50であるので、 第九世代 現在では必中でない全ての変化技の命中率が50に変わるものと取れる。
- マジックミラー の効果対象にならず、ミラクルスキンの効果対象になる技 ( 反射の対象技 ではなく、命中判定がある技) には以下がある。
  - あくむ / うつしえ / おきみやげ / サイコシフト / さきおくり / すりかえ / たこがため / テクスチャー2 / トリック / フラフラダンス
- おうごんのからだ の効果対象にならず、ミラクルスキンの効果対象になる技は無い。ただし、おうごんのからだは いのちのしずく / てだすけ など、対象にとってメリットがある変化技も無効化するため、一概にミラクルスキンがおうごんのからだの下位互換特性とは言えない。
