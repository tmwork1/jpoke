# 仕様書: かちき

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%81%A1%E3%81%8D

## 2. 基本効果（1段落）
相手によって自身の能力が下げられたとき、自身の特攻が2段階上昇する。能力低下のたびに発動判定が行われる。

## 3. 詳細仕様

### 発動条件・基本挙動
- 相手によって能力ランクが1段階以上下げられたときに発動
- 発動時、自身の特攻が+2される
- 複数能力が同時に下がった場合でも発動は1回
- 能力が実際に下がらなかった場合（無効化・既に最低など）は発動しない

### 対象となる能力低下
- 技・特性・追加効果による能力低下が対象
  - 例：いかく、バークアウトの追加効果など
- 味方による能力低下では発動しない
- 自身の技・反動による低下では発動しない

### 無効化・上書き
- 特性が無効化されている場合（例：かがくへんかガス）は発動しない
- かたやぶり等の影響は受けない（自己強化のため）
- 特性のコピー・交換は可能

### 交代・入場時
- 常時監視型特性
- 入場時に即時発動はしない
- 場にいる間のみ有効

### その他の特性との相互作用
- かいりきバサミ等で能力低下が防がれた場合：
  - 能力低下が発生しないため発動しない
- あまのじゃくとの関係：
  - 能力低下が反転して上昇した場合、低下が発生していないため発動しない
- クリアボディ等で無効化された場合も発動しない

## 4. 処理順・優先度
- 発動タイミング：能力ランク変化適用後
- 処理順：
  1. 能力低下処理
  2. かちき発動判定
  3. 特攻+2適用
- 同時発動時の優先度：
  - 通常の能力変化処理順に従う
- すばやさは無関係

## 5. テスト項目（実装検証用）

### 発動ケース
- いかくで能力が下がった際に特攻が+2される
- 技の追加効果で能力が下がった際に発動する
- 複数能力低下でも1回だけ発動する

### 非発動ケース
- 能力低下が無効化された場合に発動しない
- 味方による能力低下で発動しない
- 自身の技による能力低下で発動しない
- 特性無効化状態で発動しない

### 境界ケース
- 能力が最低で下がらない場合は発動しない
- あまのじゃくで上昇に変換された場合は発動しない
- 複数回能力低下が発生した場合に都度発動する
