# 仕様書: すなおこし

参照URL: https://wiki.pokemonwiki.com/wiki/%E3%81%99%E3%81%AA%E3%81%8A%E3%81%93%E3%81%97
調査日: 2026-04-30

## 2. 基本効果（1段落）
場に出たとき、天気をすなあらし状態にする。第6世代以降は通常5ターン（さらさらいわで8ターン）継続する。

## 3. 詳細仕様
- 発動条件: 入場時または特性書き換えで即時発動する。
- 持続: 発動者が交代/ひんしでも天気は継続する。
- 上書き関係: 既にすなあらし状態のときは継続ターンを更新しない。
- 強天候との関係: おおひでり/おおあめ/らんきりゅう中は天気変更に失敗する。
- 無効化: かがくへんかガス中は発動しない。

## 4. 処理順・優先度
- 発動タイミング: 入場時特性処理。
- 同時発動時の優先度: 天候始動特性同士は後処理側が最終天気。
- 継続ターン: 第6世代以降はターン終了時に消費。

## 5. テスト項目（実装検証用）
- 発動ケース: 入場で砂嵐化、継続ターン反映。
- 非発動ケース: 強天候中に砂嵐へ変更できない。
- 境界ケース: 既存砂嵐中の再入場でターン更新されない。
