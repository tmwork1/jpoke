# 仕様書: クォークチャージ

## 2. 基本効果
エレキフィールド中、またはブーストエナジーを持っているとき、自身の最も高い能力が上昇する。

## 3. 詳細仕様

### 能力上昇条件
- 以下のいずれかで発動：
  - エレキフィールドが展開されている
  - ブーストエナジーを所持している（発動時に消費）
- 条件を満たしたとき一度だけ発動

### 上昇対象能力
- 自身の能力値（実数値）を比較し最も高い能力を対象とする
- 対象能力：
  - 攻撃 / 防御 / 特攻 / 特防 / 素早さ
- 同値の場合は内部優先順位に従う

### 上昇量
- 攻撃・防御・特攻・特防：1.3倍
- 素早さ：1.5倍
- 能力ランクではなく実数値補正（内部倍率）

### 発動タイミング・維持
- 入場時、または条件成立時に発動
- エレキフィールドによる発動：
  - フィールドが消えると効果も終了
- ブーストエナジーによる発動：
  - 消費後は場を離れるまで効果が持続

### 再発動条件
- フィールドで発動していた場合：
  - フィールドが再展開されると再発動
- ブーストエナジーは1回限り

### 無効化・上書き
- 特性が無効化されている場合：
  - 効果は発動しない、または解除される
- 無効化解除後：
  - 条件を満たしていれば再発動する
- 特性のコピー・交換は可能

### 交代・入場時
- 入場時に条件を満たせば発動
- 交代すると効果はリセットされる

### その他の特性との相互作用
- パラドックスポケモン専用の能力補正系特性
- 他の能力補正（こだわり系等）と乗算で適用
- ランク変化とは別枠で処理される

## 5. テスト項目（実装検証用）

### 発動ケース
- エレキフィールド中に入場して能力が上昇する
- ブーストエナジー所持で入場時に発動する
- 最も高い能力が正しく選択される

### 非発動ケース
- 条件を満たさない場合発動しない
- 特性無効化状態で発動しない

### 境界ケース
- 同値能力時に優先順位どおり選択される
- フィールド終了で効果が解除される
- ブーストエナジー発動時は場にいる限り維持される
- 再展開で再発動する
