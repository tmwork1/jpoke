# 仕様書: じんばいったい

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%98%E3%82%93%E3%81%B0%E3%81%84%E3%81%A3%E3%81%9F%E3%81%84

## 2. 基本効果
きんちょうかんの効果に加えて、はくばじょうのすがたでは「しろのいななき」、こくばじょうのすがたでは「くろのいななき」の効果も併せ持つ。

## 3. 詳細仕様

### 発動内容
- きんちょうかんの効果が発動する
- はくばじょうのすがたの場合：「しろのいななき」の効果を併せ持つ
- こくばじょうのすがたの場合：「くろのいななき」の効果を併せ持つ

### 対象ポケモン
- バドレックス（はくばじょうのすがた）
- バドレックス（こくばじょうのすがた）

### 発動タイミング
- 場に出たときに特性として常時発動する

### 無効化・上書き
- 特性はかがくへんかガス等でも無効化されない
- 特性のコピー・交換はできない
- とくせいなし状態にもならない

### 交代・入場時
- 場に出たときに自動的に発動する
- 常時効果として維持される

## 5. テスト項目（実装検証用）

### 発動ケース
- きんちょうかん＋各フォルム固有特性が同時に発動する
- 場に出た直後に特性が有効化される

### 非発動ケース
- なし状態やかがくへんかガスでも無効化されない
- 特性のコピー・交換は発生しない

### 境界ケース
- すばやさ順で複数の同時発動特性と処理順が決定される
