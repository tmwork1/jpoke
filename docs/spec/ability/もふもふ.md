# 仕様書: もふもふ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%82%E3%81%B5%E3%82%82%E3%81%B5

もふもふ は、ポケモンの とくせい の一種。
## 効果
- 直接攻撃 を受けたとき、 ダメージ を半減する。
- ほのお タイプの 攻撃技 を受けたとき、ダメージが2倍になる。
## 説明文
- 第七世代 ・ 第八世代 ・ 第九世代: あいてから うけた せっしょくする わざの ダメージを はんげんするが ほのおタイプの わざの ダメージは 2ばいになる。

## 特性の仕様
- ほのお 技を受けるとダメージが2倍になるが、 相性 判定は変わらない。
- ほのお技の 直接攻撃 を受けたときは、特性の効果が両方発動してダメージは等倍となる。
- 特性 かたやぶり から技を受けた時、特性の効果がどちらも無効化される。ほのお技の非直接攻撃、ほのお技でない直接攻撃のどちらもダメージは等倍となる。
- みがわり 状態で攻撃を防いだときも特性が発動する。
- 特性 えんかく や道具の パンチグローブ により直接攻撃でなくなった技は半減できない。
- ぼうごパット を持ったポケモンによる直接攻撃は半減できる。
- こんらん 時の自傷ダメージは半減しない。
- 技 の効果でタイプが変わった攻撃技を受けた場合、変更後のタイプがほのおタイプであるときに発動する。
## 備考
- 第七世代 では ヌイコグマ 系統の 専用特性 だった。
