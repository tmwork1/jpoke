# でんじふゆう（Magnet Rise）仕様書

参照: docs/wiki/moves/でんじふゆう.html（ポケモンWiki）
関連: docs/spec/volatiles/でんじふゆう.md（揮発性状態側の仕様）

## 基本情報

* タイプ: でんき
* 分類: 変化
* 威力: -
* 命中: なし（必ず自分に成功する）
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。Gen9本家のPP10ではない）
* 対象: 自分
* 優先度: 0
* 接触: なし

## 効果

* 5ターンの間、『ふゆう』と同じ『でんじふゆう』状態となる（揮発性状態。詳細は
  `docs/spec/volatiles/でんじふゆう.md` を参照）。
  * `じめん` タイプの技を受けなくなる（サウザンアロー等の例外あり）。
  * まきびし・どくびし・ねばねばネットの効果を受けなくなる。
  * グラスフィールド・ミストフィールド・エレキフィールド・サイコフィールドなどの
    「地面にいる」ことを条件とする効果を受けなくなる。
* すでにでんじふゆう状態になっているときに使うと失敗する
  （揮発性状態の重複付与不可という汎用ルールで処理される。個別のハンドラは不要）。
* ねをはる状態、またはうちおとす状態のポケモンが使うと失敗する。
* 場がじゅうりょく状態のときは使えない。
* くろいてっきゅうを持っていると、でんじふゆう状態でも「地面にいる」扱いになる
  （`ON_CHECK_FLOATING` をアイテム側が上書きする）。ただし、でんじふゆうを
  使用すること自体は可能（付与は成功する）。

## 判定フラグ

* 急所: なし（変化技）
* 命中判定: なし（自分対象のため必ず命中）
* まもる: 防がれない（`src/jpoke/data/ps-champ-ja/moves.json` の「守×」、対象が自分のため）
* マジックコート・マジックミラー: 反射されない（対象が自分のため）
* よこどり: 対象になる（○）。ただしよこどり自体は本実装ではスコープ外

## 実装上の注意

* `MoveData` には `target="self"` を必ず指定する
  （欠落すると `Move.is_blocked_by_protect` / `Move.is_reflectable` が
  `target == "foe"` を基準に判定するため、まもる・マジックコートに誤反応する）。
