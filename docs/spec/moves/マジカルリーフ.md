# マジカルリーフ（Magical Leaf）仕様書

参照: docs/wiki/moves/マジカルリーフ.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 60
* 命中: 必中（「必ず命中する」と明記）
* PP: 20
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 光る葉を相手に当てて攻撃する。追加効果なし。
* 命中率・回避率のランク補正に関係なく必ず命中する。

## 判定フラグ

* 急所: 通常判定（急所ランク補正なし）
* 命中判定: なし（必中。`accuracy=None`）
* 追加効果: なし
* まもる・キングシールド・たたみがえし・ブロッキング・スレッドトラップ状態の相手には命中しない
  （通常のダメージ技と同様の挙動であり、専用実装は不要）
* あなをほる・そらをとぶ・ダイビング・シャドーダイブ等の回避状態の相手には命中しない
  （同上、必中フラグは命中率判定のみをスキップするものであり、無効化状態自体は通常通り貫通しない）
* おうじゃのしるし: 発動する（第三世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 実装状況

* `src/jpoke/data/moves/move_ma.py` の `MOVES["マジカルリーフ"]` に実装済み
  （`type="くさ"`, `category="special"`, `pp=20`, `power=60`, `accuracy=None`, `handlers={}`）。
* `docs/champions/move_list.txt` には掲載がないが、これは対象外を意味しない。
  同ファイルはSVからChampionsへの数値変更点（PP統一等）のみを転記した表であり、Gen9の全技を
  網羅したものではない。PPは元々20であり、Champions基準の4段階（8/12/16/20）にも合致するため
  変更不要。
