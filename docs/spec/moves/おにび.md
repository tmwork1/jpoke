# おにび 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | ほのお |
| 分類 | 変化 |
| PP | 15 |
| 命中率 | 85 |
| 範囲 | 相手1体 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

相手を **やけど** 状態にする。

## 失敗条件（ON_BEFORE_APPLY_MOVE priority=130）

以下の場合、おにびは失敗する：

1. 対象がすでに **状態異常**（どく・もうどく・まひ・やけど・ねむり・こおり）を持っている
2. 対象が **ほのおタイプ** である（やけど無効）

## 防がれる条件

以下の場合、おにびは防がれる（`ailment_manager.apply` 内での判定）：

- しんぴのまもり状態（使用者にすりぬけがある場合は無視）
- みがわり状態（使用者にすりぬけがある場合は無視）
- みずのベール特性（priority=140）

## イベントフロー

```
ON_BEFORE_APPLY_MOVE (priority=130)
  → すでに状態異常 OR ほのおタイプなら失敗 (stop_event=True)

ON_STATUS_HIT (priority=100, MoveHandler デフォルト)
  → ailment_manager.apply(defender, "やけど")
```

## ポケモンチャンピオンズ向けの簡略化

- Z変化技付加効果は実装しない
- コンテスト効果は実装しない
