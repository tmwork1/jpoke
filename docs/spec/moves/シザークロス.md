# シザークロス（X-Scissor）仕様書

参照: docs/wiki/moves/シザークロス.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 物理
* 威力: 80
* 命中: 100
* PP: 16（champions基準。src/jpoke/data/ps-champ-ja/moves.json に準拠。
  Gen9本家のPP15はchampions側でPPが8/12/16/20の4段階に統一されたことによる差分）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`, `slash`（切る技）

## 効果

* 通常攻撃。追加効果なし。

## 判定フラグ

* 急所: 通常（急所ランク補正なし）。
  ※Pokémon LEGENDS アルセウスでは急所に当たりやすい効果があるが、
  本編シリーズ（対象範囲）ではそのような効果は無いことがwiki備考欄に明記されている。
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 発動する（第四世代・第五世代以降ともに○）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 技の仕様（きれあじとの相互作用）

* 「切る技」の1つ。特性きれあじのポケモンが使用すると威力が1.5倍になる
  （`flags` に `"slash"` を含めることで既存の `きれあじ_modify_power` ハンドラが適用される）。

## リーサル計算

対象外（追加効果なしの確定ダメージ技であり、`docs/progress/move.md` の
「リーサル実装」列は `n/a`）。
