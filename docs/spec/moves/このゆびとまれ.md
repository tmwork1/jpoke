# このゆびとまれ（Follow Me）仕様書

参照: docs/wiki/moves/このゆびとまれ.html（ポケモンWiki）
参照: docs/champions/move_list.txt L282

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: - （必中、命中判定なし）
* PP: 20
* 対象: 自分
* 優先度: +3（第五世代以前）/ +2（第六世代以降）
* 接触: なし（直接攻撃×）
* 粉・胞子のわざには分類されない（いかりのこなとの相違点。第六世代以降も本技のみ非対象）

## 効果（本来のダブル/トリプルバトルでの効果）

* 使用すると自分が「ちゅうもくのまと」状態になり、そのターン中、相手が対象を選択する技を使う際は
  すべて自分に向けられる（いかりのこなと同種の効果）。
* 全体技・場を対象とする技など、対象を選択しない技には効果がない。
* くさタイプ・特性「ぼうじん」・道具「ぼうじんゴーグル」による無効化は存在しない
  （これらはいかりのこな等の粉技専有の無効化手段であり、このゆびとまれは粉技に分類されないため
  無効化されない）。
* シングルバトルで使用しても成功するが、対象となる相手が1体しかいないためバトルに影響を与えない。
* テラレイドバトルで使用すると失敗する。
* 味方の複数のポケモンが同じターンに使った場合、先に使った方の効果が優先される。
* すでにちゅうもくのまと状態のときに使用しても成功する。
* 判定: 急所 × / 命中判定 × / 追加効果 × / まもる × / おうじゃのしるし × /
  マジックコート × / よこどり × / オウムがえし ×

## シングルバトルでの扱い（本プロジェクトのスコープ）

* 本プロジェクトの対象はポケモンチャンピオンズのシングルバトルのみであり、`Battle` は常に
  各プレイヤー1体ずつの場（`battle.actives` は常に2体以下）しか扱わない。
* 「ちゅうもくのまと」による対象リダイレクトは、相手に2体以上の選択肢がある場合にのみ意味を持つ
  ダブル/トリプルバトル専有の効果であり、本プロジェクトの対象範囲外
  （`docs/progress/move.md` で本技は全列「ダブル専用」＝対象外として扱う）。
* `docs/spec/turn.md` の `Event.ON_TRY_MOVE_1`（priority=30）に「このゆびとまれ: シングルバトルである」
  という失敗条件の記載があるが、本プロジェクトの `Battle` は常に `len(battle.actives) <= 2`
  （各プレイヤー1体ずつ）という構造的制約を持つため、この失敗条件を実装すると
  「このゆびとまれは常に必ず失敗する」という自明で分岐のないハンドラになる。
  他の全ダブル専用技（いかりのこな・おさきにどうぞ・てだすけ等）との一貫性を欠くため、
  この失敗条件も含めて未実装のまま据え置く。

## 実装方針

* 通常のダメージ計算を伴わない変化技として、追加効果なし（`handlers={}`）で実装する。
* `ちゅうもくのまと` 状態の付与はダブル専用機能のため未実装のままとする。
