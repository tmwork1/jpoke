# マッドショット（Mud Shot）仕様書

参照: docs/wiki/moves/マッドショット.html（ポケモンWiki）

## 基本情報

* タイプ: じめん
* 分類: 特殊
* 威力: 55
* 命中: 95
* PP: 16（`docs/champions/move_list.txt` / `docs/progress/move.md` 準拠。本家シリーズのPPは15だが、
  Championsでは全技PPが8/12/16/20の4段階に統一されており16になる）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、100％の確率で相手の『すばやさ』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率95）
* 追加効果: あり（100%・すばやさ-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第三世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射される

## 備考

* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 110（本プロジェクトでは対象外）。
