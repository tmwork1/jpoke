# じゅうでん（Charge）仕様書

参照: docs/wiki/moves/じゅうでん.html、docs/wiki/volatiles/じゅうでん_(状態変化).html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 変化
* 威力: -
* 命中: -（命中判定なし）
* PP: 20
* 対象: 自分
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 使用すると自分に **じゅうでん** 揮発性状態を付与する。
  * じゅうでん状態の詳細仕様（威力補正・消滅条件など）は
    `docs/spec/volatiles/じゅうでん.md` を参照する。
  * 既にじゅうでん状態のポケモンが使用しても成功し、再度じゅうでん状態になる
    （効果は重複しない。次に使うでんき技の威力は2倍のままで増えない）。
* 同時に自分の **とくぼうを1段階上げる**（第四世代以降）。
  * とくぼうランクが既に最大（+6）の場合でもじゅうでん状態の付与自体は成功する
    （ランク上昇のみ不発、「とくぼうはもうあがらない!」のメッセージ）。
  * このとくぼうランク上昇は、じゅうでん状態が解除されても消えない（独立した効果）。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果: なし
* まもる: 対象外（自分自身に対する効果のため、まもるでは防がれない）
* マジックコート: 反射不可（相手を対象としない技のため）
* よこどり: 対象外（相手の技を奪う本技自体は変化技なのでよこどりされ得るが、
  よこどりされた場合は奪った側の使用者自身にじゅうでん状態が付与される）

## 備考

* Zワザ・ダイマックスわざは本プロジェクトでは対象外。
