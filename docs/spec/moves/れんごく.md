# れんごく（Inferno）仕様書

参照: docs/wiki/moves/れんごく.html（ポケモンWiki）
参照: docs/spec/moves/_move_secondary_1.md（146行目、やけど列=1）
参照: docs/spec/ailments/こおり.md（31-32行目、ほのお技被弾によるこおり回復）

## 基本情報

* タイプ: ほのお
* 分類: 特殊
* 威力: 100
* 命中: 50
* PP: 8（ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されており、
  `docs/champions/move_list.txt:960` 記載の値を採用する）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、100%の確率で相手を『やけど』状態にする。
* ほのおタイプの攻撃技であるため、被弾した相手がこおり状態の場合はこおりを解凍する
  （`docs/spec/ailments/こおり.md` 31-32行目、`こおり_cure_by_thaw_move` が担当）。
  本プロジェクトでは `flags` に `"thaw"` を設定することでこの解凍処理が有効になる
  （`かえんほうしゃ` 等、既存の同種特殊ほのお技と同じ扱い）。

## 判定フラグ

* 急所: 発生する（通常判定）
* 命中判定: あり（命中率50）
* 追加効果判定: あり（やけど付与、100%）
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
