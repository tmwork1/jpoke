# エアカッター（Air Cutter）仕様書

参照: docs/wiki/moves/エアカッター.html（ポケモンWiki）

## 基本情報

* タイプ: ひこう
* 分類: 特殊
* 威力: 60（第六世代以降。本プロジェクトはチャンピオンズ基準のためこの値を採用）
* 命中: 95
* PP: 20（`docs/champions/move_list.txt` 準拠。本家基準は25だが、チャンピオンズでは全技PPが
  8/12/16/20の4段階に統一されているため20を採用）
* 対象: 相手全員（本プロジェクトはシングルバトルのみ対象のため実質「相手1体」と同じ挙動）
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果なし。
* 急所ランク+1（急所に当たりやすい）。
* 「切る技」の1つ。特性きれあじのポケモンが使用すると威力が1.5倍になる。
* 「風技」の1つ（かぜのり等の対象判定に該当）。

## 判定フラグ

* 急所: あり（急所ランク+1）
* 命中判定: あり（命中率95）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ/ダイマックスわざ関連仕様は本プロジェクトでは対象外。
* 第三世代〜第五世代は威力55だったが、第六世代で60に変更された。本プロジェクトは
  チャンピオンズ（最新世代）基準のため60を採用する。
