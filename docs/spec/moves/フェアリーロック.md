## 効果

使用した次のターンが終了するまで、場に出ている全てのポケモン（両陣営、ゴーストタイプも含む）は
交代・にげるができなくなる（『フェアリーロック』場の状態）。詳細な効果は
`docs/spec/fields/フェアリーロック.md` を参照。

すでに『フェアリーロック』状態が有効な場合、再度使用しても技は失敗する
（`battle.global_manager.activate` が `max_count<=1` のため `False` を返す）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（対象が `foe` ではなく全体の場のため、`is_blocked_by_protect` の
  `move.target == "foe"` 条件を満たさず防がれない） |
| おうじゃのしるし（第五世代以降） | × |
| マジックコート | ×（対象が `foe`/`foe_side` ではないため `is_reflectable` が False。
  跳ね返されない） |
| よこどり | ×（変化技だが対象が場全体のため対象外。※`よこどり` 自体は本プロジェクト未実装） |
| オウムがえし | ○（場を対象にした技だがコピー可能） |

`target="field"`（`MoveData.target` のデフォルト値 `"foe"` ではない）を明示しているため、
`is_blocked_by_protect` / `is_reflectable` の判定はいずれも自動的に False になる。

## 技の仕様（一次情報より）

- 使用した次のターンに再度フェアリーロックを使った場合、場はすでにフェアリーロック状態で
  あるため失敗する。
- 場を対象にした技だが、オウムがえしでコピーすることができる。
- クレッフィの専用技。ただし本プロジェクトの実装上は他のポケモンも使用できる `MoveData` として
  登録している（既存の専用技実装パターンに準じる）。
