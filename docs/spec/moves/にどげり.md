# にどげり（Double Kick）仕様書

参照: docs/wiki/moves/にどげり.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 30（1ヒットあたり）
* 命中: 100
* PP: 30
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: 特になし（パンチ技・切る技等の対象外）

## 効果

* 2回連続で攻撃する固定回数の連続攻撃技（2回攻撃固定技）。
  * 命中判定は1発目のみ行う（`check_hit_each_time` は使用しない）。
  * 各ヒットの威力は固定で30（`power_sequence` なし）。
* 追加効果なし。

## 判定フラグ

* 急所: 通常判定（急所ランク補正なし、各ヒットごとに独立して判定）
* 命中判定: あり（命中率100、1発目のみ判定）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（ヒットごとに追加効果判定の対象）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* 2回攻撃により実質威力は60相当。
* Zワザ威力: 100、ダイマックスわざ威力: 80（本プロジェクトでは対象外）。
