# オーバードライブ（Overdrive）仕様書

参照: docs/wiki/moves/オーバードライブ.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 80
* 命中: 100
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` に本技の記載はない。本家Gen8/9基準PPは10で、
  本プロジェクトの対象（ポケモンチャンピオンズ）は全技PPが8/12/16/20の4段階に統一されている
  （`docs/champions/changes_from_sv.md`）ため、同じ本家基準PP=10を持つハイパーボイス・
  むしのさざめき・フレアソング（いずれも `src/jpoke/data/ps-champ-ja/moves.json` でPP=12）と同様に
  12へ変換されると判断した）
* 対象: 相手全員（本プロジェクトの対象範囲はシングルバトルのみのため、実質的に相手1体）
* 優先度: 0
* 接触: なし（非直接攻撃）
* フラグ: 音（sound）技

## 効果

* 通常攻撃。追加効果はない。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。本プロジェクトの対象世代では発動する）
* マジックコート: 反射不可（変化技ではないため対象外）

## Zワザ・ダイマックスわざ

* ダイマックスわざ威力: 130
* （本プロジェクトの対象範囲外のため実装しない）

## 備考

* ストリンダーの専用技。特性パンクロックのストリンダーにとってメインウェポンとなる。
* 音技であるため、ぼうおん・パンクロック・うるさがた・そうめん等の音技関連特性の対象になる
  （各特性側のハンドラで汎用的に処理されるため、本技側に個別のガードは不要）。
