# はねる（Splash）仕様書

参照: docs/wiki/moves/はねる.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -（対象が自分のため必ず成功する）
* PP: 40（`docs/champions/move_list.txt` に記載なし。同じ「効果のないわざ」である
  おいわい・みねうち等も移行時PPの変更対象外として PP40 のまま実装されている
  既存実装との整合性を優先し、PP40 を維持する）
* 対象: 自分
* 優先度: 0
* 接触: なし

## 効果

* ピョンピョンと跳ねるだけで、戦闘上の効果は一切ない（何も起こらない）。

## 判定フラグ

* 急所: なし（変化技）
* 命中判定: なし（自分対象のため必ず成功する）
* 追加効果: なし
* まもる: 防がれない（対象が自分のため）
* おうじゃのしるし: 対象外
* マジックコート・マジックミラー: 反射されない（対象が自分のため）
* よこどり: 対象外（本実装ではよこどり自体がスコープ外）
* オウムがえし: 対象外（本実装ではオウムがえし自体がスコープ外）

## じゅうりょくとの相互作用

* 場が『じゅうりょく』状態の間は技を選択できない。
* はねるを選択したターン、先に場がじゅうりょく状態になった場合、はねるは失敗する
  （`gravity_restricted` フラグ + `ON_TRY_MOVE_1` priority=30 の既存汎用ハンドラ
  `gravity_restricted_fail` で処理する。他の gravity_restricted 技と共通）。

## ちょうはつとの相互作用

* 攻撃技ではないので、ちょうはつ状態では使用できない（変化技全般の汎用ルールで
  処理されるため、個別ハンドラは不要）。

## 実装上の注意

* `MoveData` には `target="self"` を必ず指定する
  （欠落すると `Move.is_blocked_by_protect` / `Move.is_reflectable` が
  `target == "foe"` を基準に判定するため、まもる・マジックコートに誤反応する。
  `docs/spec/moves/でんじふゆう.md` と同様の注意点）。

## ポケモンチャンピオンズ向けの簡略化

* コンテスト効果（アピール等）は実装しない。
* Zワザによる付加効果（こうげき3段階上昇）は本実装のスコープ外（Zワザ未実装）。
* バトルパレス・アピールタイプ等の判定基準は実装しない。
