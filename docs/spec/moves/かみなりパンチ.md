# かみなりパンチ（Thunder Punch）仕様書

## 基本情報

* タイプ: でんき
* 分類: 物理
* 威力: 75
* 命中: 100
* PP: 15
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: パンチ技

## 効果

* 追加効果として、10%の確率で相手を「まひ」状態にする
* パンチ技のため、特性「てつのこぶし」所持時は威力が1.2倍になる

## 判定フラグ

* 急所: あり
* 命中判定: あり
* 追加効果: あり（10%・まひ）
* まもる: 防がれる
* マジックコート: 反射不可（相手への追加効果だが直接攻撃技のため対象外）
* オウムがえし: 反射可能

## 特記事項

* まひ無効のポケモン（でんきタイプ・じゅうなん等）には追加効果が発動しない（既存の共通処理に従う）
