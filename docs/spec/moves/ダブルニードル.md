# 仕様書: ダブルニードル

## 対象範囲外（実装しない）

`docs/wiki/moves/ダブルニードル.html` のページカテゴリ（`wgCategories`）に
「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」が明記されている（本文中の技データテーブルでも
赤字警告表示で「ソード・シールドで使用不可」「BDSPで使用不可」「SVで使用不可」と重複して確認できる）。

`docs/champions/move_list.txt`（ポケモンチャンピオンズの技一覧）にも本技のエントリは存在しない。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に
該当するため、本技は実装対象外とする（`ダブルパンツァー` / `ひっさつまえば` / `ふわふわフォール` /
`まわしげり` と同様のケース）。

過去のレビュー（`docs/plan/moves/攻撃技_どく系.md` の一括実装）にて `data/moves/move_ta.py` に
本技の `MoveData` と `handlers/move_attack.py` の専用ハンドラ、
`tests/moves_attack/test_move_ta.py` にテストが存在していたことが判明したため、本レビューで削除した。
詳細は `docs/review/moves/ダブルニードル.md` を参照。

## 参考: 第1〜7世代時点のデータ（実装しないが記録として残す）

| 項目 | 値 |
|---|---|
| タイプ | むし |
| 分類 | 物理 |
| 威力 | 25 |
| 命中率 | 100 |
| PP | 20 |
| 範囲 | 相手1体選択 |
| 優先度 | 0 |
| 直接攻撃 | × |
| 連続攻撃技 | 2回固定（1発ごとに追加効果判定） |
| 追加効果 | 1発ごとに20%の確率で相手をどく状態にする（2発合計で約36%: 1-(1-0.2)^2） |
| 急所判定 | ○ |
| まもる | ○（有効） |
| マジックコート | ×（無効） |
| よこどり | ×（無効） |
| おうじゃのしるし | 第2世代: ○ / 第3〜4世代: × / 第5世代以降: ○ |

（出典: `docs/wiki/moves/ダブルニードル.html`）

第1世代では技のタイプと同じタイプの相手には追加効果が発動しないことがあるが、ダブルニードルの場合は
むしタイプの相手にもどく状態にできる例外仕様があった。第2世代のみ、はがねタイプの相手にもどく状態に
できた。第3世代以降ではダブルニードルにはがねタイプをどく状態にする効果はない
（特性ふしょくの使用者が使用した場合を除く）。いずれも実装対象外のため参考情報として記録するのみ。
