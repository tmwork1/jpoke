# ハードプレス（Hard Press）仕様書

参照: docs/wiki/moves/ハードプレス.html（ポケモンWiki）
参照: docs/spec/moves/_variable_power.md（変動威力技共通仕様、第5節「対象HP依存威力技」）

## 基本情報

* タイプ: はがね
* 分類: 物理
* 威力: 可変（1〜100）
* 命中: 100
* PP: 12（Gen9本家PP10 → Champions: PPは8/12/16/20の4段階に統一されており、
  base PP10の技は12に変換される。`docs/champions/move_list.txt` に記載あり）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 対象（相手）の残りHPが多いほど、威力が高くなる。
* 威力は以下の式で算出する（小数点以下切り捨て）。ただし最小値は1。

  ```
  威力 = max(1, floor(100 × 対象の残りHP ÷ 対象の最大HP))
  ```

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）

## リーサル計算

* 威力が対象（相手）自身の残りHPに依存するため、リーサル計算で複数のHP分布枝を扱う場合、
  枝ごとの正確な威力を反映するには専用のリーサルハンドラが必要になる可能性があるが、
  `docs/progress/move.md` の該当行は `リーサル実装 n/a` / `リーサルテスト n/a`
  （本タスクの対象外、変更なし）。
