# ワイルドボルト（Wild Charge）仕様書

参照: docs/wiki/moves/ワイルドボルト.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 物理
* 威力: 90
* 命中: 100
* PP: 16（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。Gen9本家のPP15はチャンピオンズのPP統一仕様で16に変更されている）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 相手に与えたダメージの1/4を自分も受ける（反動技）。
* 追加効果なし。

## 反動ダメージの計算

* 反動量 = `max(1, round_half_down(与えたダメージ * 1/4))`
  （端数は五捨五超入。参照: `docs/spec/moves/_move_secondary_2.md`）
* 反動は `reason="recoil"` で処理する（いしあたま特性による無効化のため）

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## Zワザ・ダイマックスわざ

* Zワザ威力: 175
* ダイマックスわざ威力: （本プロジェクトの対象範囲外のため実装しない）

## 備考

* 同系統の反動技（1/4反動）: とっしん、じごくぐるま、アフロブレイク
* みがわりに命中した場合も反動ダメージは発生する（ON_HIT はみがわり被弾時も発動）。
  この場合の反動量は、みがわりへの与ダメージを基準に算出する。
* 特性が『いしあたま』『マジックガード』の場合、反動によるダメージを受けない。
* 特性が『すてみ』の場合、威力が1.2倍になる（`すてみ`特性側で汎用的に処理される）。
