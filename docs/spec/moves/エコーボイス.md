# エコーボイス（Echoed Voice）仕様書

参照: docs/wiki/moves/エコーボイス.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 特殊
* 威力: 40（可変。効果参照。最大200）
* 命中: 100
* PP: 15（本編）。ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されており、
  `src/jpoke/data/ps-champ-ja/moves.json` には本技のエントリ自体が存在しない（同ドキュメントの
  他の項目に登場する『じごくづき』の説明文中にのみ技名が出現し、本技単独の行が抜け落ちて
  いる）。本編PP15の他の技（エアスラッシュ・エレキネットなど）は同ドキュメントで
  すべてPP16に変換されているため、同じ変換規則を適用してPP16を採用する。
* 対象: 相手1体
* 優先度: 0
* 直接攻撃: なし
* 音技（sound）: あり

## 効果

* 毎ターン、場の誰か（自分・相手を問わない）がこの技を使い続けた場合、使用するたびに
  威力が40ずつ高くなっていく。最大威力は200（5ターン目以降の使用で頭打ち）。
* 直前のターンにこの技が（誰であれ）成立した使用がない場合、威力は基本値40に戻る
  （別の技を使った場合や、状態異常・状態変化・なまけ等により技が出せなかった場合を含む）。
* 1ターン中に複数回（自分・相手の双方が選択した場合など）使用されても、そのターンの
  威力上昇は1回分のみ。同じターン内で後から命中判定に進んだ側が、先に使われたこの技の
  影響を受けてさらに威力が上がることはない。
* 技が外れた場合や、ゴーストタイプ・まもる状態などに防がれた場合でも、
  「使われたこと」自体は次のターンの威力上昇に引き継がれる
  （画面に「○○のエコーボイス!」が表示されていれば成立したとみなされる）。
* 前のターンに使用したポケモンと今回使用するポケモンが異なっていても、威力は引き継がれる。
* テラレイドバトルでは味方ごとに独立したターンがあるため、自分だけが毎ターン使用しても
  威力は上がらない（本プロジェクトは対象外）。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 関連技
