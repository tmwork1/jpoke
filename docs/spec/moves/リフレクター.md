# リフレクター 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | エスパー |
| 分類 | 変化 |
| PP | 20 |
| 命中率 | - |
| 範囲 | 味方の場 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

自陣営の場を **5ターン** の間「リフレクター」状態にする。リフレクター状態の間、自陣営に出ている
ポケモンが**物理技**で受けるダメージを **1/2** に軽減する（シングルバトル固定。ダブルバトル等での
2/3軽減は本プロジェクト対象外）。すでに自陣営がリフレクター状態のときに使用すると失敗する。

リフレクター状態そのものの詳細仕様（軽減対象外となる技・急所判定・すりぬけとの相互作用・
ひかりのねんどによる延長など）は `docs/spec/fields/リフレクター.md` を参照。

## 判定

- 急所: なし（変化技）
- 命中判定: なし（命中率 `-` の技）
- 追加効果: なし（本体の効果がフィールド設定そのもの）
- まもるで防がれる: ×（範囲が「味方の場」のため対象を取らず、まもるの影響を受けない）
- マジックコートで跳ね返される: ×
- よこどりで奪われる: ○
- オウムがえしでコピーされる: ×（相手に向けて撃つ技ではないため）

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- Zワザによる付加効果は実装しない（Zワザ非対応）
- わるわるゾーン・マックスレイドバトル中のおうえんによる発生は対象外（ダイマックス非対応）
- ダブルバトル・トリプルバトルでの2/3軽減は対象外（シングルバトル専用）
- PPは `src/jpoke/data/ps-champ-ja/moves.json` の値（20）を採用する（Gen9本家と同値）
