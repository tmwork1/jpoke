# きりさく（Slash）仕様書

参照: docs/wiki/moves/きりさく.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 70
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果なし。
* 急所ランク+1（急所に当たりやすい）。
* 「切る技」の1つ。特性きれあじのポケモンが使用すると威力が1.5倍になる。

## 判定フラグ

* 急所: あり（急所ランク+1）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ/ダイマックスわざ関連仕様は本プロジェクトでは対象外。
