# らいめいげり（Thunderous Kick）仕様書

参照: docs/wiki/moves/らいめいげり.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 90
* 命中: 100
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、100%の確率で相手の『ぼうぎょ』ランクを1段階下げる。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・ぼうぎょ-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。本プロジェクトの対象世代では発動する）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（相手のランク補正を下げる追加効果を持つため、ちからずくのポケモンが
  使用すると威力1.3倍・ぼうぎょ低下の追加効果は発動しなくなる。`resolve_secondary_chance`
  経由で処理される）

## ダイマックスわざ

* ダイマックスわざ威力: 90
* （本プロジェクトの対象範囲外のため実装しない）
