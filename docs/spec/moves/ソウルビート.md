## 効果

自分の**最大HPの1/3（小数点以下切り捨て）** を消費し、自分の『こうげき』『ぼうぎょ』『とくこう』
『とくぼう』『すばやさ』ランクをすべて1段階ずつ上げる。

- ダンス技（`dance` フラグ）: おどりこ等のコピー対象になる。
- 音系の技（`sound` フラグ）: ぼうおん等の音技無効化・じごくづきの音技封じ対象になる。

## 失敗条件

以下のいずれかに該当する場合は失敗し、HPは減少しない（このときもPPは消費する）。

1. 現在のHPが**最大HPの1/3以下**のとき（`現在HP <= 最大HP // 3` が失敗条件。ちょうど1/3のときも失敗する）。
2. こうげき・ぼうぎょ・とくこう・とくぼう・すばやさの**すべて**がすでに最大（+6）のとき。
   一部のランクのみ最大の場合は失敗せず、上げられるランクだけを1段階上げてHPを消費する
   （`battle.modify_stats` の汎用挙動に従う）。

はらだいこ・みがわりと同様、HPコストは状態変化の代償であり、いしあたま・マジックガード等の
ダメージ無効化特性の影響を受けない。

## 判定フラグ

- 急所: 対象外（ダメージを与えない）
- 命中判定: なし
- 追加効果: なし
- まもる: 無効化されない（自分対象の変化技のためまもる判定自体が発生しない。
  `docs/champions/move_list.txt` の「守×」表記と一致）
- マジックコート・マジックミラー: 反射不可（自分対象の変化技のため対象外）
- すりぬけ: 影響なし（自分対象の技のためみがわり貫通判定自体が発生しない）

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。
