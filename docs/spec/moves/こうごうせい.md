## 効果

使用者自身の HP を、天候に応じた割合で回復する。`あさのひざし` と完全に同一の効果を持つ
（一次情報の「技の仕様」節でも「基本的にあさのひざしと同様の効果を持つ」と明記されている）。

| 天候 | 回復量（最大HPに対する割合） |
|------|------------------------------|
| はれ・おおひでり | 2/3 |
| あめ・おおあめ・すなあらし・ゆき | 1/4 |
| 天候なし・らんきりゅう | 1/2 |

HP が満タンの場合は失敗する。

## ばんのうがさ・ノーてんき・エアロックとの相互作用

- 使用者が **ばんのうがさ** を持つ場合、はれ/おおひでり・あめ/おおあめによる補正を受けず、
  通常時と同じ 1/2 回復になる。すなあらし・ゆきによる 1/4 補正はそのまま適用される
  （`battle.weather_for(mon)` が使用者視点の有効天候を返す）。
- 場に **ノーてんき**・**エアロック** 持ちのポケモンがいる場合、天候効果自体が無効化されるため
  常に 1/2 回復になる（`battle.weather` / `battle.weather_for` が天候無効時は「なし」を返す）。

## かいふくふうじとの相互作用

- 使用者が **かいふくふうじ** 状態の場合、`heal` フラグを持つ技として選択できない
  （`かいふくふうじ_try_action` が汎用的に処理するため、本技専用の実装は不要）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない） |
| おうじゃのしるし | × |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |
| よこどり | ○ |
| オウムがえし | × |

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。

## ポケモンチャンピオンズ向けの簡略化

- 第二世代の時間帯（朝/昼/夜）による回復量変化は実装しない
- ダイヤモンド・パールのみの「さわぐ」「トリックルーム」「じゅうりょく」による変化は実装しない
  （プラチナ以降で廃止された仕様のため）
