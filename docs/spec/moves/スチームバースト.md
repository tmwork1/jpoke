## 基本情報

* タイプ: みず
* 分類: 特殊
* 威力: 110
* 命中: 95
* PP: 8（Gen9本家は5。`docs/champions/changes_from_sv.md` により全技PPは8/12/16/20の4段階に
  統一されているため、Gen9基準のPP5はそのままでは使用できない。`docs/champions/move_list.txt`
  にはボルケニオンの専用技であるスチームバーストの行自体が存在しない（信頼できる直接の
  出典がない）が、Gen9でPP5だった同系統の特殊技（だいもんじ・ハイドロポンプ・れんごく・
  りゅうせいぐん・リーフストーム等）は同ファイル内でいずれもPP=8に統一されていることが
  確認できる。この一貫したパターンから、スチームバーストのchampions PPも8であると
  推定するのが最も妥当と判断した。詳細な根拠は docs/plan/moves/スチームバースト.md を参照）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 追加効果として、30%の確率で相手を『やけど』状態にする。
* 使用者が『こおり』状態の場合、こおり状態を治してから攻撃する。
  - かえんぐるま・かえんボール・せいなるほのお・もえつきる・ねっさのだいち・ねっとう等と
    同じ「使用宣言時に無条件でこおりを解除してから攻撃する」効果である
    （通常のこおり状態のターン開始時解凍判定＝25%抽選とは独立した、専用の解除処理）。
    みずタイプの技だが、ねっとう・ねっさのだいちと同様にこの特殊な解凍仕様を持つ。
  - 本プロジェクトでは `flags` に `"thaw"` を設定した技が、この「無条件解凍」対象技を表す
    共通ラベルとして扱われている（`docs/spec/moves/_move_labels.md` の thaw リスト、
    533行目にスチームバースト記載済み）。
  - 使用者自身がこおり状態のときに無条件解凍する処理は技ごとに個別実装が必要
    （実装例: `かえんぐるま_thaw_attacker` / `ねっとう_thaw_attacker`,
    `Event.ON_TRY_ACTION`, priority=5）。
* 第六世代以降では、ダメージを与えた相手のこおり状態も治す。
  - `"thaw"` フラグにより `こおり_cure_by_thaw_move`（`Event.ON_DAMAGE_HIT`, defender:self）で
    実装されている。この処理は技のタイプに関わらず `"thaw"` フラグの有無のみで判定するため、
    スチームバーストにも既に適用されている。
  - この効果は、やけど付与の追加効果判定やひんし判定の後に発動するため、こおり状態を治した
    攻撃で直後にやけど状態にすることはない（こおり状態が残っている間はやけどを重ねて付与でき
    ないため、通常の状態異常排他制御により自然に成立する）。
  - 使用者の特性が『ちからずく』の場合、この効果は発動せず相手のこおりを治さない
    （`docs/spec/abilities/ちからずく.md` 13行目・39行目）。ただしちからずくでも自分の
    こおりを治すことはできる（自分のこおりを治す効果は追加効果に該当しないため）。
    - ほのおタイプの技がダメージを通じて相手のこおりを治す効果はタイプ由来の効果であり
      追加効果に該当しないため、ちからずくの影響を受けない。スチームバーストはみずタイプで
      この効果を持つ「例外技」（ねっとう・シャカシャカほう・ねっさのだいち・ハイドロスチーム
      等と同種）であるため、この効果に限りちからずくで無効化される。
  - 相手の特性が『りんぷん』である場合でもこおり状態を治せる
    （`docs/spec/abilities/りんぷん.md` 33行目）。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率95）
* 追加効果判定: あり
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降。第二〜第四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
