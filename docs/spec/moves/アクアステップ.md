# アクアステップ（Aqua Step）仕様書

参照: docs/wiki/moves/アクアステップ.html（ポケモンWiki）
参照: docs/spec/moves/_move_secondary_1.md（71行目、追加効果共通仕様）

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 80
* 命中: 100
* PP: 12（champions版。本家(SVまで)は10だが、`docs/champions/moves.md` 24行目の
  記載どおり12になる。`docs/champions/changes_from_sv.md` の技PP統一ルールに準拠）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* おどりの技の1つ（`docs/wiki/moves/アクアステップ.html` の「技の仕様」節に明記。
  おどりこの対象になる）
* ウェーニバルの専用技

## 効果

* 追加効果として、100%の確率で自分の『すばやさ』ランクを1段階上げる。
* 使用者の特性が『ちからずく』の場合、威力が1.3倍になる代わりに追加効果（すばやさ上昇）は
  発動しなくなる（`flags` に `"secondary_effect"` を設定することで、既存の ちからずく 汎用処理
  `ちからずく_boost` / `ちからずく_disable_secondary_effect` が威力上昇・追加効果無効化を担当する）。

## 判定フラグ（一次情報より）

* 急所: 通常判定（○）
* 命中判定: あり（命中率100）
* 追加効果: あり（○、100%）
* まもる: 防がれる（○）
* おうじゃのしるし: 第五世代以降対象（○）
* マジックコート: 対象外（×、変化技ではないため）

## リーサル計算

* `docs/progress/move.md` の該当行は「リーサル実装: n/a」「リーサルテスト: n/a」であり、
  自分のすばやさランクを上げるだけの追加効果は相手へのダメージ確定計算（HP分布）に
  一切影響しないため、`くさわけ` / `オーラぐるま` / `ニトロチャージ` 等の同種技と同様に
  対象外（本レビューの対象範囲外）とする。
