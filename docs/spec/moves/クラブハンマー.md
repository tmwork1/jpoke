# クラブハンマー（Crabhammer）仕様書

参照: docs/wiki/moves/クラブハンマー.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 100
* 命中: 95（Champions基準。docs/champions/move_list.txt を正とする）
* PP: 12（Champions基準。docs/champions/move_list.txt を正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果なし。
* 急所ランク+1（Championsでは「きゅうしょアップ+1で攻撃する」という説明文になっており、急所に当たりやすい効果）。

## 判定フラグ

* 急所: あり（急所ランク+1）
* 命中判定: あり（命中率95）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ/ダイマックスわざ関連仕様は本プロジェクトでは対象外。
* 世代による威力・命中率の変遷: 威力90→100（第六世代以降）、命中率85→90（第五世代～SV）→95（Champions）。本プロジェクトはChampions基準の値（威力100・命中95・PP12）を採用する。
