# エレキボール（Electro Ball）仕様書

参照: docs/wiki/moves/エレキボール.html（ポケモンWiki）
参照: docs/spec/moves/_variable_power.md（変動威力技共通仕様、第6節「速度差依存威力技」）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 可変（40〜150）
* 命中: 100
* PP: 12（本家基準10。champions は全技PP 8/12/16/20 の4段階に統一されており、
  `docs/champions/move_list.txt` の値を正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: なし
* フラグ: 弾のわざ（ぼうだん所持ポケモンに無効化される）

## 効果

* 自分（使用者）が相手より実効素早さが速いほど、威力が大きくなる。
* 比率 = floor(自分の実効素早さ ÷ 相手の実効素早さ)
* 比率から以下の表で威力を決定する。

  | 自分の実効素早さが相手の実効素早さの | 威力 |
  |---|---|
  | 1倍未満（比率0） | 40 |
  | 1倍以上2倍未満（比率1） | 60 |
  | 2倍以上3倍未満（比率2） | 80 |
  | 3倍以上4倍未満（比率3） | 120 |
  | 4倍以上（比率4以上） | 150 |

* 相手の実効素早さが0の場合はゼロ除算を防ぐ例外処理があり、
  自分の素早さに依らず威力は40固定になる。
* 実効素早さは、両者のランク補正・特性・もちもの・まひ・おいかぜ・しつげんの影響を受ける。
* トリックルーム状態でも威力は変化しない（トリックルームは行動順序のみを反転させ、
  実際の素早さ数値自体は変化しないため）。
* Zワザ・ダイマックスわざにしたときの威力は固定（Zワザ160、ダイマックスわざ130）で、
  両者の素早さとは無関係（本プロジェクトはZワザ・ダイマックスわざが対象外のため実装不要）。

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: ひるみ効果あり（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能（本プロジェクトはオウムがえし自体が未実装のため対応不要）
* ぼうだん: 無効化される（弾のわざ）

## リーサル計算

* 威力は使用者・対象双方の実効素早さに依存するが、ダメージ計算時点で確定する威力の変動要素であり、
  相手HPの分布（`StateDist`）を直接左右する要素ではないため、`lethal.py` に専用ハンドラは不要。
  （`docs/progress/move.md` のリーサル実装/リーサルテスト列は n/a）
