# フライングプレス（Flying Press）仕様書

参照: docs/wiki/moves/フライングプレス.html（ポケモンWiki）、src/jpoke/data/ps-champ-ja/moves.json

## 基本情報

* タイプ: かくとう（同時にひこうタイプでもある。相性計算のみ複合、他の効果はかくとう単タイプ扱い）
* 分類: 物理
* 威力: 100
* 命中: 95
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。Gen9本家は10だが、チャンピオンズでは全技PP8/12/16/20の
  4段階に統一されているため12を採用する）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* じゅうりょく状態では選択・使用不可（`gravity_restricted`）

## 効果

* かくとうタイプとひこうタイプの2つの相性を組み合わせてダメージを計算する（相性判定のみの複合）。
  * かくとう相性表とひこう相性表をそれぞれ防御側の各タイプに適用し、掛け合わせた値が最終的なタイプ相性になる。
  * 相性早見表（防御側タイプ別、かくとう×ひこうの合成値）:
    * 2倍: ノーマル・くさ・こおり・かくとう・あく
    * 1倍: ほのお・みず・じめん・むし・いわ・ドラゴン・はがね（むし・いわ・はがねは2倍と0.5倍が相殺されて1倍）
    * 0.5倍: でんき・どく・ひこう・エスパー・フェアリー
    * 0倍: ゴースト
* 「ちいさくなる」状態の相手に対しては必ず命中し、ダメージが2倍になる（`minimize` ラベル共通処理）。
  ただし、そらをとぶ等で姿を隠している相手には当たらない。

## わざのタイプに関する効果への扱い

フライングプレスは、タイプ相性判定以外の「わざのタイプに関する効果」については単純にかくとうタイプの技とみなす。

* タイプ一致ボーナス（STAB）はかくとうタイプに対してのみ補正がかかる。
* 相手の特性きけんよちは、この技を通常のかくとうタイプの技とみなし、ひこうタイプを複合した相性は考慮しない
  （`docs/spec/abilities/きけんよち.md`）。
* 使用者の持ち物くろおび/こぶしのプレートにより威力が増える。するどいくちばし/あおぞらプレートでは威力は
  増えない（`docs/spec/items/するどいくちばし.md`）。
* とくせいノーマルスキンの効果を受けるとノーマル・ひこうタイプの技になる（`docs/spec/abilities/ノーマルスキン.md`）。
* とくせいへんげんじざいを使用した場合、かくとう単タイプに変わる（かくとう・ひこうタイプにはならない）
  （`docs/spec/abilities/へんげんじざい.md`）。
* とくせいへんしょくでこの技を受けた場合はかくとうタイプになる（ひこうタイプにはならない）
  （`docs/spec/abilities/へんしょく.md`）。
* そうでんを受けるとでんき・ひこうタイプの技になる。

## 相性・ダメージ計算関連の特性・アイテムとの相互作用

* フライングプレスを抜群で受けるポケモンは、ヨプのみの効果でダメージを半減できる。ひこうタイプの相性により
  抜群となるポケモンであってもバコウのみの効果は発動しない
  （`docs/spec/items/_タイプ半減きのみ.md`）。
* とくせいブレインフォース・いろめがねは、フライングプレスによる相性自体が変わる効果（複合相性）を
  考慮して発動判定を行う（`docs/spec/abilities/ブレインフォース.md`、`docs/spec/abilities/いろめがね.md`）。
* とくせいハードロック・フィルター・プリズムアーマーも同様に、複合後の最終相性を基準に発動する
  （`docs/spec/abilities/ハードロック.md`）。

## 判定フラグ

* 急所: 通常判定
* 命中判定: あり（命中率95。ただし「ちいさくなる」状態の相手には必中）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
