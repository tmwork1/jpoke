# シェルアームズ（Shell Side Arm）仕様書

参照: docs/wiki/moves/シェルアームズ.html（ポケモンWiki）、
https://bulbapedia.bulbagarden.net/wiki/Shell_Side_Arm_(move)（分類判定の裏取り）

## 基本情報

* タイプ: どく
* 分類: 特殊（ただし判定次第で物理技に変化する。詳細は「分類の判定」を参照）
* 威力: 90
* 命中: 100
* PP: 12（champions/move_list.txt の値。Gen9本家では10だが、本プロジェクトは
  champions/changes_from_sv.md によりPPが8/12/16/20に統一されているためこちらを正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: 物理技として計算されたときのみ直接攻撃になる。特殊技として計算されたときは
  直接攻撃にならない（弾のわざにも分類されず、ぼうだんのポケモンにも通常通り効果がある）
* ヤドラン（ガラルのすがた）専用技

## 効果

* 追加効果として、20%の確率で相手を『どく』状態にする。

## 分類（物理・特殊）の判定

* 以下のダメージ計算式に対して、Aに使用者の攻撃・Dに対象の防御を代入した値の方が、
  Aに使用者の特攻・Dに対象の特防を代入した値より高ければ分類が物理技に変わる。
  両者が等しい場合、実機では物理か特殊かがランダム（50%）で決まる。

  ```
  ((レベル × 2/5 + 2) × 90 × A/D) / 50
  ```

  威力・レベルは両辺で共通のため、実質的には `攻撃/防御` の比と `特攻/特防` の比の
  比較（`攻撃 × 特防` と `特攻 × 防御` の比較）と同値になる。
* ランク補正／パワートリック／パワーシェア／ガードシェアの影響を考慮した実数値
  （使用者の攻撃・特攻、対象の防御・特防）を代入する。
* 物理か特殊かの判定が決まった後に、こだわりハチマキ／とつげきチョッキ等のもちもの、
  ちからもち／ファーコート／てんねん等の特性、壁／やけど／すなあらし状態の影響を
  考慮してダメージを計算する。これらの効果は分類の判定そのものには影響しない
  （`くさのけがわ`・`ふしぎなうろこ`・`ファーコート` の各仕様書を参照）。
* 相手の特性「てんねん」は分類判定には影響しない。分類が確定した後のダメージ計算で
  ランク無視が適用される（`docs/spec/abilities/てんねん.md` 参照）。
* 使用者の特性が「スロースタート」であっても、分類判定における能力比較には影響しない
  （`docs/spec/abilities/スロースタート.md` 参照）。物理技になった場合は、
  スロースタートによる攻撃ダウン補正がダメージ計算に適用される。
* ワンダールーム状態の相互作用は本プロジェクトでは対象外（ワンダールーム自体が未実装）。
* 使用者の特性が「ちからずく」のときは、どく状態にできなくなる代わりに威力が1.3倍になる
  （`flags={"secondary_effect"}` により既存の汎用ちからずく処理で対応済み）。
  分類が変わる効果はちからずくの影響を受けず、通常どおり発動する。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（20%・どく）
* まもる: 防がれる
* おうじゃのしるし: 対象（第五世代以降）。本技自体はひるみ追加効果を持たないため、
  おうじゃのしるしを持たせても発動しない（既存の汎用ひるみ処理の対象外というだけで、
  本技固有の実装は不要）
* マジックコート: 反射不可（変化技ではないため対象外）
* ちからずく: 対象技（追加効果があるため）。「分類の判定」の項を参照

## Zワザ・ダイマックスわざ

* ダイマックスわざ威力: 90（ダイマックスわざにしたときはステータスに関係なく特殊技になる）
* 本プロジェクトの対象範囲外（ダイマックス・Zワザ非対応）のため実装しない

## 備考

* ヤドラン（ガラルのすがた）の専用技。
* 分類次第で戦闘アニメーションが変わる（物理: 叩きつけ、特殊: 毒液発射）が、
  ゲームプレイに影響しないため実装対象外。
