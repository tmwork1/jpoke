# つぼをつく（Acupressure）仕様書

参照: docs/wiki/moves/つぼをつく.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: - （必中）
* PP: 20（champions 基準。src/jpoke/data/ps-champ-ja/moves.json 参照）
* 対象: 自分か味方1体（本家仕様。ただしシングルバトルでは常に自分が対象になるため、
  本プロジェクトの対象範囲では `target="self"` として実装する）
* 優先度: 0
* 直接攻撃: しない

## 効果

* 「こうげき」「ぼうぎょ」「とくこう」「とくぼう」「すばやさ」「命中率」「回避率」の
  7つの能力のうち、いずれか1つのランクがランダムで2段階上がる。

## 技の仕様（一次情報より）

* すでに最大（+6）まで上がっている能力はランダム選択の対象から除外される。
  対象の全ての能力が最大まで上がっている場合、この技は失敗する。
* 味方に使う場合でも命中判定は行われず必中するが、姿を隠している（溜め技実行中の）味方には
  当たらない（シングルバトルでは自分専用の技であるため実質的に影響しない）。
* 味方に使う場合でもまもる状態やダイウォールを無視する（シングルバトルでは自分専用の技で
  あるため実質的に影響しない）。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化がそのまま効果）
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* よこどり: 対象外（第五世代以降は奪われない。自分を対象とするため）
* オウムがえし: 反射不可（相手を対象としないため対象外）
