# からみつく（Constrict）仕様書

参照: docs/wiki/moves/からみつく.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 10
* 命中: 100
* PP: 35
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、10％の確率で相手の『すばやさ』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常判定（急所ランク0）
* 命中判定: あり（命中率100）
* 追加効果: あり（10%・すばやさ-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。第二〜四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: 反射される
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・すばやさ低下効果は
  発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）

## 本プロジェクトでの対象範囲について

* `docs/champions/move_list.txt` に からみつく 自体が収録されておらず、ポケモンチャンピオンズのシングルバトルでは
  現状いずれのポケモンも本技を習得できない（一次情報にも「ソード・シールド」「BDSP」「スカーレット・バイオレット」
  で使用不可と明記されている）。ただし本プロジェクトの技エンジンは汎用実装のため、他の同種の未使用技と同様に
  通常のダメージ技として実装する。

## 備考

* 全ての技の中でも最も威力が低い技の一つ（威力10）。
