# みがわり（Substitute）仕様書

参照: docs/wiki/moves/みがわり.html（ポケモンWiki）、docs/wiki/volatiles/みがわり_(状態変化).html（ポケモンWiki）、
docs/spec/volatiles/みがわり.md、docs/spec/moves/しっぽきり.md

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -（命中判定なし）
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 基準。本家SVでは10だが、チャンピオンズは全技PPを8/12/16/20に統一しているため12）
* 対象: 自分
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 使用者の**最大HPの1/4（小数点以下切り捨て）** を消費し、消費量と同じHPを持つ「みがわり」状態になる。
* みがわりは本体の代わりに相手の攻撃を受ける（詳細な防御・貫通仕様は `docs/spec/volatiles/みがわり.md` を参照）。
* 技が成功すると、使用者の**バインド状態を解除する**（第三世代以降の仕様）。

## 失敗条件

以下のいずれかに該当する場合は失敗し、HPは減少しない（このときもPPは消費する）。

1. 使用者がすでにみがわり状態のとき
   （「しかし 〈使用者〉の 身代わりは すでに でていた」というメッセージが出る）。
   HP不足による失敗より優先して表示される。
2. 使用者の残りHPが**最大HPの1/4以下**のとき
   （「しかし 身代わりを だすには 体力が 足りなかった!」というメッセージが出る）。
   ちょうど1/4のときも失敗する（＝消費後にHPが0になる場合は使用不可、という一般則）。

いしあたま・マジックガードなどHP消費を無効化する特性を持っていても、上記の消費は通常どおり発生する
（HPコストは技の直接ダメージではなく状態変化の代償のため）。

## 判定フラグ

* 急所: 対象外（ダメージを与えない）
* 命中判定: なし
* 追加効果: なし
* まもる: **無効化されない**（自分対象の変化技のためまもる判定自体が発生しない。
  `src/jpoke/data/ps-champ-ja/moves.json` の「守×」表記と一致）
* マジックコート・マジックミラー: 反射不可（自分対象の変化技のため対象外）
* すりぬけ: 影響なし（自分対象の技のためみがわり貫通判定自体が発生しない）

## みがわり状態が出ている間の制約

* すでにみがわり状態のポケモンが「みがわり」または「しっぽきり」を使おうとした場合は失敗する
  （PPは消費する）。
