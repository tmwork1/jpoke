# 仕様書: ニードルアーム

## 対象範囲外（実装しない）

本技は第3世代から存在する技だが、**ポケットモンスター ソード・シールド以降（ソード・シールド／BDSP／スカーレット・バイオレット）で使用不可**
（`docs/wiki/moves/ニードルアーム.html` のカテゴリ「ソード・シールドで使用不可能なわざ」
「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」「スカーレット・バイオレットで使用不可能なわざ」、
および本文中の赤字表記「ソード・シールドで使用不可」「BDSPで使用不可」「SVで使用不可」より確認）。
第八世代の説明文にも「このわざは つかえません」と明記されている。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に該当するため、
本技は実装対象外とする。`docs/champions/move_list.txt`（ポケモンチャンピオンズの技一覧）にも項目が存在しないことも確認した。

過去のレビューにて `data/moves/move_na.py` に本技の `MoveData`（PP15の誤った値。champions PP規定は
8/12/16/20の4段階のみで15は存在しない）と `handlers/move_attack.py` の専用ハンドラ
（`ニードルアーム_apply_flinch`）、`types/move.py` の `MoveName` Literal、
`handlers/item.py` の `_INNATE_FLINCH_MOVES`、
`tests/moves_attack/test_move_na.py` / `tests/moves_status/test_move_na.py`（重複登録されていた）
にテストが存在していたことが判明したため、本レビューで削除した。詳細は `docs/review/moves/ニードルアーム.md` を参照。
