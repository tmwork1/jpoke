# じゃれつく（Play Rough）仕様書

参照: docs/wiki/moves/じゃれつく.html（ポケモンWiki）

## 基本情報

* タイプ: フェアリー
* 分類: 物理
* 威力: 90
* 命中: 90
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、10%の確率で相手の『こうげき』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率90）
* 追加効果: あり（10%・こうげき-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。本プロジェクトの対象世代では発動する）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・こうげき低下の
  追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）

* ダイマックスわざ威力: 130
