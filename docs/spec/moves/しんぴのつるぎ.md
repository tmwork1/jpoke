# しんぴのつるぎ（Secret Sword）仕様書

参照: docs/wiki/moves/しんぴのつるぎ.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 特殊
* 威力: 85
* 命中: 100
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 相手の『とくぼう』ではなく、『ぼうぎょ』でダメージ計算をする（分類は特殊のまま）。
  * サイコショック・サイコブレイクと同種の「物理判定の特殊技」（`docs/spec/moves/カテゴリ.md` の
    「物理として判定される / physical」区分に該当）。
  * 追加効果はない。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 技の分類（切り技）

* 「切り / slash」フラグを持つ（`docs/spec/moves/カテゴリ.md` 参照）。

## Zワザ・ダイマックスわざ

* Zワザ威力: 160
* ダイマックスわざ威力: 90
* （本プロジェクトの対象範囲外のため実装しない）

## 専用技について（本プロジェクトでは対象外）

* ケルディオの専用技。フォルムチェンジ（かくごのすがた）に関する仕様はシングルバトルの
  ダメージ計算とは無関係のため対象外。
