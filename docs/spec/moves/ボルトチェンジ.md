# ボルトチェンジ（Volt Switch）仕様書

参照: docs/wiki/moves/ボルトチェンジ.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 70
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: なし（非直接攻撃）

基本的には[とんぼがえり](とんぼがえり.md)のでんきタイプ版であり、判定・交代条件などの仕様は
とんぼがえりと共通（`docs/spec/moves/とんぼがえり.md` を参照）。以下はボルトチェンジ独自の点のみ記述する。

## 効果

* 攻撃した後、使用者は手持ちのポケモンと交代する。
* 追加効果とは見なされない（ちからずくで威力は上がらないが、交代効果自体は発動する。
  りんぷん・おんみつマントの影響も受けない）。

## 判定フラグ

* 急所: 通常判定
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる（防がれた場合、交代も発動しない）
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## でんきタイプ固有の注意点

* でんきタイプの技であるため、じめんタイプには無効。また特性ひらいしん・ちくでん・
  でんきエンジンによって技が無効化される（吸収・威力反射等）。これらの場合、攻撃自体が
  不発となるため、交代（ボルトチェンジ自身の効果）も発動しない。
* 相手の特性とびだすなかみ/うのミサイル、持ち物レンブのみ、みちづれ状態の効果を受けて
  使用者がひんしになった場合、技の効果（交代）は発動せず、次のポケモンはターン終了時の
  通常の瀕死交代として繰り出される（とんぼがえりの「攻撃側がゴツゴツメット等の反動で
  瀕死になった場合」と同種のケース。詳細は `docs/plan/moves/とんぼがえり.md` の
  エッジケース5を参照）。

## 交代の発動条件

とんぼがえりと同一（`docs/spec/moves/とんぼがえり.md` の「交代の発動条件」を参照）。

## 備考

* バトンタッチとは異なり、能力ランク変化や補助効果を交代後のポケモンに引き継がせる効果はない。
* Zワザ威力: 140、ダイマックスわざ威力: 120（本プロジェクトでは対象外）。
