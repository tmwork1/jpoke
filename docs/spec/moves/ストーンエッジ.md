# ストーンエッジ（Stone Edge）仕様書

参照: docs/wiki/moves/ストーンエッジ.html（ポケモンWiki）

## 基本情報

* タイプ: いわ
* 分類: 物理
* 威力: 100
* 命中: 80
* PP: 5（本家基準。チャンピオンズでは全技PPが8/12/16/20の4段階に統一されるため、
  実装上は 8 とする。docs/champions/move_list.txt の該当行を参照）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: なし（`contact` を持たない）

## 効果

* 追加効果なし。急所に当たりやすい（急所ランク+1）。

## 判定フラグ

* 急所: 急所ランク+1（通常より高い確率で急所に当たる）
* 命中判定: あり（命中率80）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 対象外（ひるみ追加効果を持たないため無関係）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* 追加効果を持たないため、専用ハンドラは不要。`critical_rank=1` を `MoveData` に設定するのみで、
  既存の急所判定ロジック（`core/move_executor.py` の `CRIT_RATES` 参照処理）が自動的に適用される。

## リーサル計算

対象外（追加効果を持たない通常の物理攻撃技のため、`docs/progress/move.md` の
「リーサル実装」列は `n/a`）。
