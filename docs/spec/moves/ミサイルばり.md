# ミサイルばり（Pin Missile）仕様書

参照: docs/wiki/moves/ミサイルばり.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 物理
* 威力: 25（1ヒットあたり、第六世代以降。本プロジェクトはこの値を採用）
* 命中: 95（第六世代以降。本プロジェクトはこの値を採用）
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 2～5回連続で攻撃する可変回数の連続攻撃技（2-5回攻撃技）。
  * 命中判定は1発目のみ行う（`check_hit_each_time` は使用しない）。
  * 各ヒットの威力は固定で25（`power_sequence` なし）。
  * ヒット回数の分布は `docs/spec/moves/_multi_hit.md` の「2.2 可変回数技（2～5回）」に従う
    （第9世代: 2回37.5%、3回37.5%、4回12.5%、5回12.5%）。スキルリンクで常に5回ヒットになる。
* 急所判定は各ヒットごとに独立して行われる。
* 追加効果なし。

## 判定フラグ

* 急所: 各ヒットごとに独立判定（急所ランク補正なし）
* 命中判定: あり（命中率95、1発目のみ判定）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第二世代以降、各ヒットごとに判定）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（攻撃技のため）
* オウムがえし: 発動する

## 専用技

* なし。

## 備考

* 第五世代までは威力14・命中85だったが、第六世代以降は威力25・命中95に変更された。
* Zワザ威力: 140、ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
