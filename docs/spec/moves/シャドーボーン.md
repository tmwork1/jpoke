# 仕様書: シャドーボーン

## 対象範囲外（実装しない）

本技（英名 Shadow Bone）は第七世代で登場し、第八世代（ソード・シールド）までは使用可能だったが、
**ブリリアントダイヤモンド・シャイニングパール（BDSP）およびポケットモンスター スカーレット・バイオレット
（SV）では使用不可**（`docs/wiki/moves/シャドーボーン.html` のインフォボックスに赤字で
「BDSPで使用不可」「SVで使用不可」と明記、カテゴリにも「ブリリアントダイヤモンド・シャイニングパールで
使用不可能なわざ」「スカーレット・バイオレットで使用不可能なわざ」とある）。
`docs/champions/move_list.txt`（ポケモンチャンピオンズの技一覧）にも本技の項目は存在せず、
ポケモンチャンピオンズ側でも採用されていないことが確認できる。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に
該当するため、本技は実装対象外とする。

過去のレビュー（一括実装計画 `docs/plan/moves/攻撃技_防御側ランク変化系.md`）にて、対象範囲の確認が
行われないまま `data/moves/move_sa.py` に本技の `MoveData`（威力85・命中100・PP10、`Event.ON_DAMAGE_HIT`
での防御ダウン追加効果）と `handlers/move_attack.py` の専用ハンドラ `シャドーボーン_lower_defender_def`、
`types/move.py` の `MoveName` Literal、および `tests/moves_attack/test_move_sa.py` のテストが誤って
登録されていたことが判明したため、本レビューで削除した。詳細は `docs/review/moves/シャドーボーン.md` を参照。
