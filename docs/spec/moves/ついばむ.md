# ついばむ（Peck）仕様書

参照: docs/wiki/moves/ついばむ.html（ポケモンWiki）

## 基本情報

* タイプ: ひこう
* 分類: 物理
* 威力: 60
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: なし（追加効果に確率発動はない）

## 効果

* 相手が「バトルに効果のある」きのみを持っていれば、命中後にそのきのみを奪い、
  自分（攻撃者）がそのきのみの効果を受ける（消費する）。
* むしくい（虫タイプ・同威力・同命中・同PP）と全く同じ追加効果を持つ、姉妹技。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし（確率発動要素はない。きのみを持っていれば必ず発動）
* まもる: 防がれる
* おうじゃのしるし: 対象外（追加効果がないため無関係）
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: コピー可能

## きのみ奪取の詳細

* 発動タイミングはダメージ命中後（ON_HIT）。みがわりに阻まれた場合でも、みがわりを貫通して
  相手本体からきのみを奪う（ON_HIT はみがわり被弾時にも発動する仕様のため）。
* 相手がきのみを持っていない場合、または「バトルに効果のある」きのみでない場合は不発。
* 相手がねんちゃく（ねんちゃく特性等）を持つ場合、奪取自体が失敗し追加効果は発生しない。
* 攻撃者が既にアイテムを持っている場合、奪取は失敗する（アイテム交換の一般規則）。
* タンガのみ等、ダメージ計算前に自動発動して消費されるきのみは、命中後の奪取処理が発火する前に
  既に消費されているため、ついばむによる奪取は不発になる。
* 奪取に成功したきのみは、攻撃者が即座に消費し効果を受ける（`force_trigger_berry` 相当の処理）。
