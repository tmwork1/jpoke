# パワフルエッジ（Mighty Cleave）仕様書

参照: docs/wiki/moves/パワフルエッジ.html（ポケモンWiki）

## 基本情報

* タイプ: いわ
* 分類: 物理
* 威力: 95
* 命中: 100
* PP: 5（`src/jpoke/data/ps-champ-ja/moves.json` に本技のエントリは存在しないため、
  Champions側のPP再定義（8/12/16/20統一）の対象外。Gen9本家の値をそのまま採用する）
* 対象: 相手1体
* 優先度: 0
* 直接攻撃: あり
* テツノイワオの専用技

## 効果

* 相手の「まもる」状態を無視して攻撃する。
* 「切る技」の1つ（特性「きれあじ」を持つポケモンが使用すると威力が1.5倍になる）。
* 威力・命中率に対する追加効果はない（追加効果: ×）。

## 急所ランクについて

* Wiki判定表では「急所: ○」（通常の急所判定が発生しうる）とのみ記載されており、
  「急所ランク+1」等の急所に当たりやすい効果を示す注記はない。
* `ストーンエッジ` の備考には「第七世代までは いわタイプの技の中で唯一急所に当たりやすい
  効果を持つ技だった」「スカーレット・バイオレットでは（`ツタこんぼう` 以外の）
  いわタイプの技に急所に当たりやすい効果はない」旨の記載があり、
  パワフルエッジには急所ランク補正が無いことを裏付ける。
* よって本技の急所ランク補正は **なし**（`critical_rank` は 0 のまま）。

## 「まもる」無視の仕様

* まもる状態を無視してダメージを与えるが、まもる系の状態自体を解除する効果はない
  （`フェイント` の仕様とは異なり、まもる状態はそのまま残る）。
  相手は本技を受けたターンも他のポケモンからの技を防ぐことができる。
  （※本シミュレータはシングルバトルのみ対象のため「他のポケモンからの技を防ぐ」効果は
  実質的に観測されないが、まもる状態自体を解除しない点は次のターンの
  「連続成功率ペナルティ」の扱いに影響するため明記する）
* 次のターンに使うまもる系統の技の成功率は、通常通り連続使用による低下の対象となる
  （まもる状態が解除されていないため）。
* `かえんのまもり` / `スレッドトラップ` / `トーチカ` / `ニードルガード` などの
  「まもる」系統の変化技による防御も無視するため、直接攻撃時にこれらの技が発動する
  反撃効果（やけど付与・拘束・とげとげ返し等）は受けない。
* `ゆびをふる` で本技が選ばれることがある（`ゆびをふる` から出現しうる技として仕様書上
  明記されている）。

## 判定フラグ

* 急所: ○（急所ランク補正なし）
* 命中判定: ○
* 追加効果: ×
* まもる: ×（`まもる` 状態を無視して命中する）
* おうじゃのしるし・するどいキバ: ○（第五世代以降）
* マジックコート: ×（反射不可、変化技ではないため対象外）
