# はるのあらし（Springtide Storm）仕様書

## 基本情報

* タイプ: フェアリー
* 分類: 特殊
* 威力: 100
* 命中: 80
* PP: 5
* 対象: 相手1体選択（原作では相手全員だが、シングルバトルのため相手1体と等価）
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* その他分類: 風の技

## 効果

* 追加効果として、30%の確率で相手の「こうげき」ランクを1段階下げる

## 判定フラグ

* 急所: あり
* 命中判定: あり
* 追加効果: あり（30%・相手のこうげき-1）
* まもる: 防がれる
* マジックコート: 反射不可（攻撃技のため対象外）
* おうじゃのしるし（追加効果発動率アップ系）: 対象（第五世代以降）
* 風の技: かぜのり（無効化＋こうげき+1）、ふうりょくでんき（じゅうでん付与）等の対象

## 特記事項

* ラブトロス専用技
* ゆびをふるで選ばれない
