## 効果

相手を **ちょうはつ** 状態（揮発性状態）にする。
ちょうはつ状態のポケモンは、変化技を選択・使用できなくなり、攻撃技しか出せなくなる。

- すでにちょうはつ状態の相手には失敗する（重複付与不可、持続ターンも上書きされない）。
- みがわり状態のポケモンにも効果がある（ちょうはつはみがわりを貫通する）。
  `MoveData` に `flags={"bypass_substitute"}` を設定する必要がある。
- 特性どんかん・アロマベール・おうごんのからだを持つ相手には無効。
  メンタルハーブを持つ相手には付与直後に即時解除される。
  （いずれも `docs/spec/volatiles/ちょうはつ.md` を参照）

揮発性状態「ちょうはつ」自体の詳細仕様（行動制限・解除条件など）は
`docs/spec/volatiles/ちょうはつ.md` を参照。

## 判定

- 急所: なし（変化技）
- 命中判定: あり（100%、回避率等の補正を受ける）
- 追加効果: なし（本体効果が100%発動するため「追加効果」扱いではない）
- まもる: 防がれる（`Move.is_blocked_by_protect` は対象が `foe` の変化技を自動的に対象とするため、専用フラグは不要）
- マジックコート: 跳ね返される（`マジックコート_reflect` が「対象が相手・変化技」であるすべての技を汎用的に跳ね返すため、専用対応は不要）

## イベントフロー

```
ON_STATUS_HIT (priority=100, MoveHandler デフォルト)
  → ちょうはつ_apply
    → volatile_manager.apply(defender, "ちょうはつ", count=3)
```

持続ターン数（3ターン）は Pokémon Champions 仕様として固定。
原作の「行動後に受けた場合は4ターン、行動前/交代直後に受けた場合は3ターン」という
分岐は実装を単純化して省略している（詳細は `docs/spec/volatiles/ちょうはつ.md` 参照）。
