# ドラゴンハンマー（Dragon Hammer）仕様書

参照: docs/wiki/moves/ドラゴンハンマー.html（ポケモンWiki）

## 基本情報

* タイプ: ドラゴン
* 分類: 物理
* 威力: 90
* 命中: 100
* PP: 16（Champions基準。SV本家PP15はdocs/champions/changes_from_sv.mdの4段階統一ルールにより16に変換される。
  docs/champions/move_list.txtに本技のエントリはないが、同一のSV本家PP15を持つ技
  （かえんほうしゃ・10まんボルト・ドラゴンクロー）がいずれも16に変換されていることから、
  同じ変換規則を適用する）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果なし（通常攻撃）。

## 判定フラグ

* 急所: あり（通常の急所判定。急所ランク補正なし）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ/ダイマックスわざ関連仕様は本プロジェクトでは対象外。
* 「切る技」フラグは付与しない（wikiのカテゴリに「切る技」の記載なし。ハンマーで叩く攻撃）。
