# メテオドライブ（Sunsteel Strike）仕様書

参照: docs/wiki/moves/メテオドライブ.html（ポケモンWiki）

## 基本情報

* タイプ: はがね
* 分類: 物理
* 威力: 100
* 命中: 100
* PP: 5
  （docs/champions/move_list.txt にメテオドライブの記載がないため、champions 値による
  上書きは行わない。シャドーレイ・フォトンゲイザーなど他の同系専用技も同様に一覧に
  記載がなく、Gen9 本家準拠の値のまま扱う）
* 対象: 相手1体
* 優先度: 0
* 直接攻撃: あり（接触技）

## 効果

* 相手の特性を無視して攻撃する。
* 追加効果なし。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* マジックコート: 対象外（変化技ではないため）

## 技の仕様

* 無視できる特性は「かたやぶり」と同じ対象（本プロジェクトでは特性データの
  `mold_breaker_ignorable` フラグを持つ特性が対象）。
* ファントムガード／プリズムアーマーは無視できない
  （両特性とも `mold_breaker_ignorable` フラグを持たないため、実装上も対象外）。
* 第七世代のみ、使用者の特性自身も「かたやぶり」で無視できる特性の場合はメテオドライブの
  効果で無視される仕様があったが、第八世代以降は使用者自身の特性は無視しない。
  本プロジェクトは SV（チャンピオンズ）基準のため、使用者自身の特性は無視しない
  （defender のみを対象とする現行実装が正しい）。
* 他の技が出す技（ゆびをふる等）経由でメテオドライブが発動した場合、特性を無視する効果は
  発生しない。ただし本プロジェクトでは技呼び出し系の技（ゆびをふる等）自体が未実装
  （`src/jpoke/data/moves/move_ya.py` の「ゆびをふる」は `handlers={}` の TODO 状態）
  のため、この分岐は現時点では対象外。

## 学習ポケモン

* ソルガレオの専用技として登場し、ウルトラサン・ウルトラムーン以降は日食ネクロズマも
  使用できる。本プロジェクトの実装上はポケモンの技構成として設定すれば誰でも使用可能で、
  専用技制限は考慮しない。
