# かいでんぱ 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | でんき |
| 分類 | 変化 |
| PP | 16 |
| 命中率 | 100 |
| 範囲 | 相手1体 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

相手の **特攻** を **2段階** 下げる。

## 失敗条件

明示的な失敗チェックなし（既存の能力低下技に準じる）。
特攻がすでに最低ランク（-6）の場合はランク変化が発生せず、ログも記録されない。

## 防がれる条件

以下の場合、かいでんぱは防がれる（`modify_stats` 内での判定）：

- クリアボディ特性（優先度140）
- みがわり状態（能力低下は防がれる）

## イベントフロー

```
ON_STATUS_HIT (priority=100, MoveHandler デフォルト)
  → battle.modify_stats(defender, {"C": -2}, source=attacker)
```

## ポケモンチャンピオンズ向けの簡略化

- マジックコート反射は実装しない
- コンテスト効果は実装しない
