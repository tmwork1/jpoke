# ニトロチャージ（Flame Charge）仕様書

参照: docs/wiki/moves/ニトロチャージ.html（ポケモンWiki）
参照: docs/spec/moves/_move_secondary_1.md（76行目、追加効果共通仕様）

## 基本情報

* タイプ: ほのお
* 分類: 物理
* 威力: 50
* 命中: 100
* PP: 20（champions版。参照: src/jpoke/data/ps-champ-ja/moves.json 630行目）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、100%の確率で自分の『すばやさ』ランクを1段階上げる。
* 使用者の特性が『ちからずく』の場合、威力が1.3倍になる代わりに追加効果（すばやさ上昇）は
  発動しなくなる（`flags` に `"secondary_effect"` を設定することで、既存の ちからずく 汎用処理
  `ちからずく_boost` / `ちからずく_disable_secondary_effect` が威力上昇・追加効果無効化を担当する）。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率100）
* 追加効果判定: あり（100%）
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
