# にぎりつぶす（Crush Grip）仕様書

参照: docs/wiki/moves/にぎりつぶす.html（ポケモンWiki）
参照: docs/spec/moves/威力変動技.md（変動威力技共通仕様、第5節「対象HP依存威力技」）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 可変（1〜120）
* 命中: 100
* PP: 5
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 対象（相手）の残りHPが多いほど、威力が高くなる。
* 威力は以下の式で算出する。第五世代以降（本プロジェクトが対象とする第9世代を含む）は
  端数を**五捨五超入**（`round_half_down`。ちょうど0.5は切り捨て、0.5を超えたら切り上げ）で
  丸める。ただし最小値は1。

  ```
  威力 = max(1, round_half_down(120 × 対象の残りHP ÷ 対象の最大HP))
  ```

* 第四世代のみ `威力 = floor(1 + 120 × 対象の残りHP ÷ 対象の最大HP)`（範囲1〜121）だが、
  本プロジェクトの対象世代（第9世代）では非該当。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代・第五世代以降とも）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 180（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 140（本プロジェクトでは対象外）。

## リーサル計算

* 威力が対象（相手）自身の残りHPに依存するため、リーサル計算で複数のHP分布枝を扱う場合、
  枝ごとの正確な威力を反映するには専用のリーサルハンドラが必要になる可能性があるが、
  `docs/progress/move.md` の該当行は `リーサル実装 n/a` / `リーサルテスト n/a`
  （本タスクの対象外、変更なし）。
