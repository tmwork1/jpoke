# おさきにどうぞ（After You）仕様書

参照: docs/wiki/moves/おさきにどうぞ.html（ポケモンWiki）
参照: src/jpoke/data/ps-champ-ja/moves.json L146

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -（必中、命中判定なし）
* PP: 16
* 対象: 自分以外の1体
* 優先度: 0（てだすけとは異なり優先度は上がらないため、使用者自身が素早くなければ意味がない）
* 接触: なし（直接攻撃×） / まもる: 効果なし（守備側の技ではない） / ゆびをふるでは選ばれない

## 効果（本来のダブル/トリプルバトルでの効果）

* 対象となったポケモンは、自身の「すばやさ」・技の優先度・特性「あとだし」/「きんしのちから」・
  持ち物「こうこうのしっぽ」/「まんぷくおこう」・「さきおくり」の効果を無視して、
  使用者の直後に行動できるようになる。
* 行動順が変わるだけで、対象が選択した技の優先度の情報自体は失われない。
  （例: 特性「いたずらごころ」で優先度+1にしたおさきにどうぞで対象を繰り上げても、
  対象が出す優先度0以下の技は「ファストガード」などに防がれない。）
* 第八世代以降、元々使用者の次に行動する予定だったポケモンに使用しても成功する
  （第七世代までは技が失敗した）。
* 失敗条件:
  * すでにそのターン行動したポケモンを対象にした場合は失敗する。
  * 「さいはい」「おどりこ」の効果で行動したポケモンに対しても、同様に失敗する。

## シングルバトルでの扱い（本プロジェクトのスコープ）

* 本プロジェクトの対象はポケモンチャンピオンズのシングルバトルのみであり、`Battle` は常に
  各プレイヤー1体ずつの場（`battle.actives` は常に2体以下）しか扱わない。
* シングルバトルでは、行動側とその対象（相手1体）しか存在しないため、「対象を使用者の直後に
  割り込ませる」という本技の核心効果は意味を持たない。対象がすでに行動済みなら本技は失敗し、
  対象が未行動ならそもそも次に行動するのは対象自身であり、行動順は本技を使っても使わなくても
  変化しない。
* この「常に空振りの分岐にしかならない」という性質は、`docs/spec/turn.md`（`Event.ON_APPLY_MOVE`
  priority 100「おさきにどうぞ: 失敗条件」）に記載された失敗条件チェックを実装したとしても、
  シングルバトル上で観測可能な効果を生まない。類似技「このゆびとまれ」「いかりのこな」
  「てだすけ」等、同様に対象リダイレクト・行動順操作を主目的とするダブル/トリプル専有技と
  同じ扱いとし、`docs/progress/move.md` で本技を全列「ダブル専用」（対象外）として扱う。

## 実装方針

* 通常のダメージ計算を伴わない変化技として、追加効果なし（`handlers={}`）で実装する。
* 行動順操作（対象を使用者の直後に割り込ませる処理）はダブル/トリプル専有機能のため未実装のままとする。
