# ふるいたてる（Work Up）仕様書

参照: docs/wiki/moves/ふるいたてる.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -
* PP: 20（champions 基準。docs/champions/move_list.txt にエントリなし。本家 PP30 の
  自己ランク上昇技〔きあいだめ・つるぎのまい・こうそくいどう等〕がすべて champions PP20 に
  統一されているため、同一パターンとして PP20 とする）
* 対象: 自分
* 優先度: 0
* 直接攻撃: いいえ

## 効果

* 自分の『こうげき』『とくこう』ランクを1段階ずつ上げる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化がそのまま効果）
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* よこどり: 対象になる（○）
* オウムがえし: 反射不可（相手を対象としないため対象外）
