# ねんりき（Confusion）仕様書

参照: docs/wiki/moves/ねんりき.html（ポケモンWiki）

※ `docs/champions/move_list.txt` には本技の記載がない（ポケモンチャンピオンズの技リストに未収録）。
本仕様書は本家シリーズ（ポケモンWiki）の値をそのまま採用する。

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: 50
* 命中: 100
* PP: 25
* 対象: 相手1体
* 優先度: 0
* 接触: なし
* フラグ: `secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、10％の確率で相手をこんらん状態にする。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（10%・こんらん付与）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象外（本追加効果はこんらん付与効果のため、第四世代までは発動率上昇の対象だったが第五世代以降は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。

## リーサル計算

* HP依存の追加効果・威力変動を持たない通常のダメージ技であり、リーサル計算上の特別な考慮は不要（対象外）。
