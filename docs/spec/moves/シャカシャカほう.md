## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 80
* 命中: 90（相手の実数値等による変動はない。固定値）
* PP: 16（champions基準。Gen9本家では15だが、champions側でPPは8/12/16/20の4段階に統一されている）
* 対象: 相手全員（本プロジェクトはシングルバトルのみが対象のため、対象は相手1体として扱う）
* 優先度: 0
* 接触: なし
* 弾（bullet）: 該当しない（「ほう」と名が付くが弾のわざには分類されず、特性ぼうだんの影響を受けない）
* ヤバソチャの専用技

## 効果

* 相手に与えたダメージの半分だけ、自分のHPを回復する（`_drain_hp` の `heal_ratio=0.5`。四捨五入・最低1回復）。
* 追加効果として、20%の確率で相手を『やけど』状態にする。
* 使用者が『こおり』状態の場合、こおり状態を治してから攻撃する。
  - かえんぐるま・かえんボール・せいなるほのお・もえつきる・ねっさのだいち・ねっとう等と同じ
    「使用宣言時に無条件でこおりを解除してから攻撃する」効果である
    （通常のこおり状態のターン開始時解凍判定＝25%抽選とは独立した、専用の解除処理）。
    くさタイプの技だが、ねっとう・ねっさのだいちと同様にこの特殊な解凍仕様を持つ。
  - 本プロジェクトでは `flags` に `"thaw"` を設定した技が、この「無条件解凍」対象技を表す
    共通ラベルとして扱われている（`docs/spec/moves/_move_labels.md` の thaw リスト参照）。
  - 使用者自身がこおり状態のときに無条件解凍する処理は技ごとに個別実装が必要
    （実装例: `ねっとう_thaw_attacker` / `ねっさのだいち_thaw_attacker`,
    `Event.ON_TRY_ACTION`, priority=5）。
* ダメージを与えた相手のこおり状態も治す。
  - `"thaw"` フラグにより `こおり_cure_by_thaw_move`（`Event.ON_DAMAGE_HIT`, defender:self）で
    実装されている。
  - この効果は、やけど付与の追加効果判定やひんし判定の後に発動するため、こおり状態を治した
    攻撃で直後にやけど状態にすることはない（こおり状態が残っている間はやけどを重ねて付与でき
    ないため、通常の状態異常排他制御により自然に成立する）。
  - 使用者の特性が『ちからずく』の場合、この効果は発動せず相手のこおりを治さない
    （`docs/spec/abilities/ちからずく.md` 13行目・39行目）。ただしちからずくでも自分の
    こおりを治すことや、HPを吸収することはできる
    （どちらも追加効果に該当しないため）。
    - くさタイプの技でありながら相手のこおりを治す効果を持つ「例外技」（ねっとう・スチーム
      バースト・ねっさのだいち・ハイドロスチーム等と同種）であるため、この効果に限り
      ちからずくで無効化される。
  - 相手の特性が『りんぷん』である場合でもこおり状態を治せる
    （`docs/spec/abilities/りんぷん.md` 33行目）。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率90）
* 追加効果判定: あり
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 技の仕様（本プロジェクトのスコープ外）

* 複数の相手に当てた場合の吸収順序（左のポケモンから順に吸収）や、みちづれ発動時の
  こおり回復可否といった仕様は、ダブルバトル以上を前提とした仕様のため、
  シングルバトルのみを対象とする本プロジェクトでは対象外とする
  （CLAUDE.md「対象はポケモンチャンピオンズシングルバトルのみ」）。
