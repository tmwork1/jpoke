# ブレイブチャージ（Take Heart）仕様書

参照: docs/wiki/moves/ブレイブチャージ.html（ポケモンWiki）

## 基本情報

* タイプ: エスパー
* 分類: 変化
* 威力: -
* 命中: -
* PP: 15（第九世代基準。`docs/champions/move_list.txt` に本技のエントリは存在しない。
  フィオネ・マナフィの専用技であり、両者はチャンピオンズのシングルバトルで実戦投入されないため
  championsのPP再編表に含まれていないと考えられる。championsでの上書き値が存在しないため、
  第九世代の基準値をそのまま採用する）
* 対象: 自分
* 優先度: 0
* 直接攻撃: いいえ
* 専用技: フィオネ・マナフィ

## 効果

* 自分の『とくこう』『とくぼう』ランクをそれぞれ1段階ずつ上げる。
* 自分の状態異常（どく・もうどく・やけど・まひ・こおり・ねむり）を回復する。
* 上記2つの効果は互いに独立している。
  * 状態異常に罹っていないときでも、とくこう・とくぼうランクは上がる。
  * とくこう・とくぼうランクがどちらも最大（+6）であるときでも、状態異常は治る。
  * ランクが最大かつ状態異常もない場合でも技自体は失敗しない（「とくこうは もう 上がらない!」等の
    メッセージのみが表示される）。
* 状態異常（ねむり・まひ等）で技が出せなかった場合、本技の効果自体は発動しない（技を出せなかったため）。
  * `ねごと` で本技が選ばれて実行された場合は、自身の『ねむり』状態を解除できる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化・状態異常回復がそのまま効果）
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* よこどり: 対象になる（○、自分を対象とする変化技）
* オウムがえし: 反射不可（変化技のため対象外）
* おうじゃのしるし等の追加効果依存アイテム: 対象外
