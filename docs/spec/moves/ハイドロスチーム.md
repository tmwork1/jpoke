## 基本情報

* タイプ: みず
* 分類: 特殊
* 威力: 80
* 命中: 100
* PP: 16（Gen9本家は15。`docs/champions/changes_from_sv.md` により全技PPは8/12/16/20の
  4段階に統一されているため、Gen9基準のPP15はそのままでは使用できない。
  `src/jpoke/data/ps-champ-ja/moves.json` にはウネルミナモの専用技であるハイドロスチームの行自体が
  存在しない（信頼できる直接の出典がない）が、Gen9でPP15だった同系統の特殊技
  （あくのはどう・シャドーボール・ねっとう等）は同ファイル内でいずれもPP=16に統一されて
  いることが確認できる。この一貫したパターンから、ハイドロスチームのchampions PPも16で
  あると推定するのが最も妥当と判断した。詳細な根拠は docs/plan/moves/ハイドロスチーム.md
  を参照）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* にほんばれ（おおひでり含む）状態によるダメージ補正率が、通常のみずタイプ技の0.5倍ではなく
  1.5倍になる（みずタイプ技として唯一、晴れで弱体化せずむしろ強化される）。
  - 使用者がにほんばれ状態の効果を受けている（＝使用者自身がばんのうがさを持たず、
    ノーてんき等で天候が無効化されていない）ときにこの効果が発動する。
  - 使用者のみがばんのうがさを持っている場合、この特殊補正は発動せず、通常のみずタイプ技
    と同じ晴れ下0.5倍の弱体化を受ける（対象がばんのうがさを持たない前提）。
  - 使用者と対象がどちらもばんのうがさを持っている場合、天気補正は1.0倍になる（使用者の
    ばんのうがさにより1.5倍化が発動せず、かつ対象のばんのうがさにより通常の0.5倍弱体化も
    発動しないため）。
  - 使用者がにほんばれ状態の効果を受けていれば、対象がばんのうがさを持っていても
    1.5倍になる効果は発動する（対象側のばんのうがさによる天候無効化は、この特殊な
    威力上昇効果には影響しない）。
  - 特性ノーてんきのポケモンが場にいるときはにほんばれ状態の効果自体が無くなるため、
    天気補正は1.0倍になる。
  - あめ（おおあめ含む）状態下では、通常のみずタイプ技と同様にダメージが1.5倍になる
    （対象がばんのうがさを持っていると天気補正は1.0倍になる、通常のあめ補正仕様のまま）。
* 使用者が『こおり』状態の場合、こおり状態を治してから攻撃する。
  - かえんぐるま・かえんボール・せいなるほのお・もえつきる・ねっさのだいち・ねっとう・
    スチームバースト等と同じ「使用宣言時に無条件でこおりを解除してから攻撃する」効果である
    （通常のこおり状態のターン開始時解凍判定＝25%抽選とは独立した、専用の解除処理）。
    みずタイプの技だが、ねっとう・スチームバースト・ねっさのだいちと同様にこの特殊な
    解凍仕様を持つ。
  - 本プロジェクトでは `flags` に `"thaw"` を設定した技が、この「無条件解凍」対象技を表す
    共通ラベルとして扱われている（`docs/spec/moves/_move_labels.md` の thaw リスト、
    538行目にハイドロスチーム記載済み）。
  - 使用者自身がこおり状態のときに無条件解凍する処理は技ごとに個別実装が必要
    （実装例: `かえんぐるま_thaw_attacker` / `ねっとう_thaw_attacker` /
    `スチームバースト_thaw_attacker`, `Event.ON_TRY_ACTION`, priority=5）。
* ダメージを与えた相手のこおり状態も治す。
  - `"thaw"` フラグにより `こおり_cure_by_thaw_move`（`Event.ON_DAMAGE_HIT`, defender:self）で
    実装されている。
  - この効果は、ひんし判定の後に発動する。
  - **ハイドロスチームはちからずくの対象技ではない**（`docs/spec/abilities/ちからずく.md`
    13行目・39行目のちからずく無効化リストに、シャカシャカほう/スチームバースト/
    ねっさのだいち/ねっとうは含まれるが、ハイドロスチームは含まれない）。
    そのため使用者の特性が『ちからずく』であっても、威力は上がらず（そもそもハイドロスチーム
    に「追加効果」＝ちからずく対象となる確率発動効果が無いため）、にほんばれ状態による
    威力上昇効果、および自分・相手のこおり状態を治す効果はいずれも通常通り発動する
    （ねっとう・スチームバースト等とは異なり、ちからずくによる無効化を一切受けない）。
  - 相手の特性が『りんぷん』である場合でもこおり状態を治せる
    （`docs/spec/abilities/りんぷん.md` 33行目）。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率100）
* 追加効果判定: なし（にほんばれ威力上昇・こおり回復効果はいずれもタイプ・技固有の
  効果であり、確率発動の「追加効果」には該当しない）
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降。第二〜第四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
