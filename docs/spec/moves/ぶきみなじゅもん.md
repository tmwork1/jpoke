## 効果

ダメージを与えた後、相手が最後にPPを消費した技
（`ctx.defender.last_pp_consumed_move`）のPPを3減らす。

一次情報（`docs/wiki/moves/ぶきみなじゅもん.html`「技の仕様」節）:

> 場に出てから1度もPPを消費する行動を取っていない相手のPPは減らせない。
> 1度PPを消費したポケモンでも、それが交代で出てきた直後に当てた場合などでは
> PPは減らせない。

- PP減少の対象は「最後に**PPを消費した**技」。うらみ・かなしばりと同じ定義であり、
  ねごとで実行されたサブ技（PP消費0）は対象にならず、ねごと自身のPPが減る。
- 相手が場に出てからPPを消費する行動をしていない場合（`last_pp_consumed_move` が
  `None`）は、**ダメージは通常どおり与えるがPP減少のみ発生しない**（技自体は
  失敗しない）。交代直後の相手に対しても同様（`last_pp_consumed_move` は
  交代時にリセットされる）。
- 減少量は3。対象技の残りPPが3未満の場合は0まで減る（`Move.modify_pp` の
  下限クリップをそのまま利用する）。

## 判定

- 急所: あり（攻撃技）
- 命中判定: あり（100%）
- 追加効果: PP減少は追加効果扱い（一次情報。ただし後述のとおり本実装では
  確定効果として簡略化している）
- まもる: 防がれる
- マジックコート: 跳ね返されない（攻撃技）
- みがわり: 貫通する（音のわざ。`sound` フラグによりコア側の汎用ルールで処理）

## イベントフロー

```
ON_DAMAGE_HIT (priority=100, MoveHandler デフォルト)
  → ぶきみなじゅもん_reduce_defender_pp
    → move = ctx.defender.last_pp_consumed_move
    → move が None ならPP減少をスキップ（ダメージのみ）
    → move.modify_pp(-3)
```

## 本実装での対象外・既知の差分（スコープ外）

一次情報には以下の仕様があるが、本実装では対応していない（将来対応の候補）:

- **ひんし判定後の発動**: 「この技で相手をひんしにさせたときはPPを減らさない」。
  本実装の `ON_DAMAGE_HIT` はひんしの有無に関わらず発火するため、
  相手を倒したヒットでもPPが減る。
- **追加効果扱いの無効化**: りんぷん・おんみつマントでPP減少が防がれ、
  ちからずくで威力1.3倍と引き換えにPP減少が消える（一次情報）。
  本実装ではPP減少を確定効果として実装しており、これらの無効化に対応していない。
- **へんしん・ものまね・レベルアップで最後の技を忘れたポケモン**: PPを減らせない
  （へんしん・ものまねは本エンジンで実装保留のため到達不能）。
- **ダイマックス関連**: ポケモンチャンピオンズはダイマックス機構自体が対象外。
