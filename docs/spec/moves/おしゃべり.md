# おしゃべり（Chatter）仕様書

参照: docs/wiki/moves/おしゃべり.html（ポケモンWiki）

## 基本情報

* タイプ: ひこう
* 分類: 特殊
* 威力: 60（第五世代まで）→65（第六世代以降。本プロジェクトはこちらを採用）
* 命中: 100
* PP: 20
* 対象: 相手1体（遠隔）
* 優先度: 0
* 接触: なし（直接攻撃×）
* フラグ: 音のわざ

## 効果

* 追加効果として、100%の確率で相手をこんらん状態にする（第六世代以降）。
  * 第四・第五世代では、ニンテンドーDSのマイクに録音した音声の音量によって追加効果の発生率が変動した
    （未録音/小音量: 低確率、大音量: 高確率）。この仕様は現行のハードウェア機能に依存するため、
    本プロジェクト（ポケモンチャンピオンズ、現行世代準拠）では対象外とし、100%固定とする。

## 判定フラグ

* 急所: あり（通常の急所判定に従う。確定急所ではない）
* 命中判定: あり（命中率100）
* 追加効果: あり（こんらん、確率100%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。ひるみ効果はダメージ技に対して汎用的に処理されるため本技固有の対応は不要）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 第四・第五世代はコピー不可、第六世代以降はコピー可能
  （本プロジェクトはオウムがえし自体が未実装のため対応不要。実装時は本技の除外リストからおしゃべりを外すこと）
* さきどり: コピー不可（さきどり自体が未実装のため対応不要）
* ちからずく: 威力が1.3倍になるが、こんらんの追加効果は付与されない
  （`flags={"secondary_effect"}` により汎用のちからずくハンドラで処理済み。本技固有の対応は不要）

## 技の仕様（その他）

* 音のわざの一つ（ぼうおん・パンクロックの対象、じごくづきの音系封じ対象等）。
* ものまね・スケッチで習得できない。
* まねっこ・さきどりでコピーできない（さきどり自体が未実装のため対応不要）。
* ゆびをふる・ねこのてで選ばれない（いずれも未実装のため対応不要）。
* ねごとでは選ばれない技のリストに含まれない（`ねごと.html` の「選ばれない技」節に記載なし）。
  → 本技に `non_negoto` フラグを付与する必要はない。

## 専用技

* ペラップの専用技（バトル上の挙動には影響しない。習得可否のみに関わる）。

## 備考

* Zワザ威力: 120（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 120（本プロジェクトでは対象外）。
* 第四・第五世代ではソード・シールド、スカーレット・バイオレットで使用不可という記載があるが、
  これは当該作品の技マシン/技教え未収録によるものであり、ポケモンチャンピオンズでの扱いとは無関係。
  本プロジェクトは実装済み・テスト済みとして扱う。
