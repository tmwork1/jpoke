# かたきうち（Retaliate）仕様書

参照: docs/wiki/moves/かたきうち.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 70（可変。効果参照）
* 命中: 100
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃 ○）

## 効果

* 味方がひんしになった**次のターン**に使用すると、威力が2倍（70→140）になる。
* 「次のターン」とは、味方がひんしになったターンの直後の1ターンのみを指す。
  ひんしになったそのターン中に使用しても威力は上がらない
  （例: ダブルバトルで味方がひんしになったターン中に本技を使用しても2倍にならない）。
  ひんしになってから2ターン以上経過した場合も威力は上がらない。
* シングルバトル（本プロジェクトの対象）では、味方がひんしになったことでポケモンを交代した
  「次のターン」に本技を使用した場合のみ威力が上がる。

## 本プロジェクトでの実装方針

* 「味方がひんしになったターン」をプレイヤー単位で記録し、現在のターンがその翌ターンであれば
  威力を2倍にする。
* 味方（自分の手持ちの誰か）がひんしになったタイミング（`Battle.modify_hp` でHPが0になった瞬間）
  でそのポケモンが所属するプレイヤーの状態に「ひんしになったターン」を記録する。
* 本技の威力計算時、記録された「ひんしになったターン」が `現在のターン - 1` と一致するかどうかで
  判定する。

## 判定フラグ

* 急所: あり（通常判定）

* 追加効果: なし

* マジックコート: 反射不可（変化技ではないため対象外）
