# ぼうふう（Hurricane）仕様書

参照: docs/wiki/moves/ぼうふう.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: ひこう
* 分類: 特殊
* 威力: 110（第六世代以降。第五世代までは120）
* 命中: 70
* PP: 12
  * 一次情報（ポケモンWiki）ではPP10だが、`docs/champions/move_list.txt` の `ぼうふう` 行がPP12であり、
    ポケモンチャンピオンズ側の値を正とする
    （`docs/spec/moves/なみのり.md` と同様、Champions仕様のPP4段階統一〈8/12/16/20〉に基づく）。
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 追加効果として、30%の確率で相手を『こんらん』状態にする。
* 天気が『あめ』のときは必ず命中する（命中率補正・命中判定を無視して必中になる）。
* 天気が『にほんばれ』のときは命中率が50になる。
* 天気が『あめ』でも『にほんばれ』でもない場合は通常通り命中率70で判定する。
* `そらをとぶ` の溜め状態（回避状態）の相手にも命中する。この仕様は `handlers/volatile.py` の
  `HIDDEN_MOVE_ALLOWED_MOVES["そらをとぶ"]` に `ぼうふう` が登録済みであることで実現している。

## 天候による命中率補正の優先順位

* あめ（防御側から見た天候）: 必中。ばんのうがさ等の天候無効特性を持つ側は通常の命中判定に戻る。
* にほんばれ（攻撃側から見た天候）: 命中率50。
* 上記のいずれでもない場合: 通常の命中率70で判定する。
* 天候の有無判定は `battle.weather_for(pokemon)` を使う
  （攻撃側・防御側それぞれの特性による天候無効化を考慮するため）。

## 風の技（wind）フラグ

* 「風の技」に分類され、特性 かぜのり／ふうりょくでんき の効果対象になる
  （`flags={"wind"}` として実装、`かぜのり_absorb_wind` / `ふうりょくでんき_on_damage` が対象判定に使用）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（天候により必中・命中率変化。上記参照）
* 追加効果: あり（30%・こんらん）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## リーサル計算

* 固定威力・追加効果（追加効果を除けばダメージ量に影響しない状態異常付与のみ）の通常攻撃技のため、
  `lethal.py` に専用ハンドラは不要
  （`docs/progress/move.md` のリーサル実装／リーサルテスト列は n/a のまま対象外）。
