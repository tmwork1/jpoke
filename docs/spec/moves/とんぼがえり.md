# とんぼがえり（U-turn）仕様書

参照: docs/wiki/moves/とんぼがえり.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 物理
* 威力: 70
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 攻撃した後、使用者は手持ちのポケモンと交代する。
* 追加効果とは見なされない（ちからずくで威力は上がらないが、交代効果自体は発動する。
  りんぷん・おんみつマントの影響も受けない）。

## 判定フラグ

* 急所: 通常判定
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる（防がれた場合、交代も発動しない）
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 交代の発動条件

以下の場合、攻撃は成功してもとんぼがえりによる交代は発動しない。

* 技がまもる状態などで防がれた場合、姿を隠していた場合、命中しなかった場合。
* 控えに戦えるポケモンが1匹も残っていない場合（攻撃自体は成功しダメージを与える）。
* 対象のレッドカードが発動した場合（レッドカードが先に発動し、攻撃側の交代先はランダムで選ばれる。
  攻撃側自身による交代選択は発生しない）。
* 対象のだっしゅつボタンが発動した場合（相手はアイテム効果により交代するが、攻撃側は交代できない）。
* 対象の特性にげごし・ききかいひが発動した場合（相手は特性により交代するが、攻撃側は交代できない）。

以下の場合でも、とんぼがえりによる交代は発動する。

* 技がみがわり状態に防がれた場合や、特性ばけのかわ・アイスフェイスにダメージを肩代わりされた場合。
* 使用者がバインド状態・にげられない状態・ねをはる状態であっても交代できる
  （とらわれ状態を無視して発動する）。
* 場がフェアリーロック状態であっても交代できる。
* 使用者が特性かげふみ・ありじごく・じりょくの効果を受けている場合も交代できる。
* 対象を倒した場合でも交代する。このとき、とんぼがえりの交代先が出てきた後に、
  相手が次のポケモンを選択して繰りだす（攻撃側の交代が先）。

## 備考

* バトンタッチとは異なり、能力ランク変化や補助効果を交代後のポケモンに引き継がせる効果はない。
* Zワザ威力: 140、ダイマックスわざ威力: 120（本プロジェクトでは対象外）。
