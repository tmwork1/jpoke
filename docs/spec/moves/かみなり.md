# かみなり（Thunder）仕様書

参照: docs/wiki/moves/かみなり.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 110（第六世代以降。第五世代までは120）
* 命中: 70
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 追加効果として、30%の確率で相手を『まひ』状態にする
  （第一世代のみ10%。本プロジェクトの対象世代では30%）。
* 天気が『あめ』のときは必ず命中する（命中率補正・命中判定を無視して必中になる）。
* 天気が『にほんばれ』のときは命中率が50になる。
* 天気が『あめ』でも『にほんばれ』でもない場合は通常通り命中率70で判定する。
* `そらをとぶ`・`とびはねる` の溜め状態（回避状態）の相手にも命中する
  （第二世代以降）。この仕様は `handlers/volatile.py` の
  `HIDDEN_MOVE_ALLOWED_MOVES["そらをとぶ"]` に `かみなり` が登録済みであることで実現している。
  なお `とびはねる` は本プロジェクトでは1ターン完結の即時攻撃技として簡略実装されており
  （溜めターン・回避状態を持たない）、この命中仕様との相互作用は現状発生しない。

## 天候による命中率補正の優先順位

* あめ（防御側から見た天候）: 必中。ばんのうがさ等の天候無効特性を持つ側は通常の命中判定に戻る。
* にほんばれ（攻撃側から見た天候）: 命中率50。
* 上記のいずれでもない場合: 通常の命中率70で判定する。
* 天候の有無判定は `battle.weather_for(pokemon)` を使う
  （攻撃側・防御側それぞれの特性による天候無効化を考慮するため）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（天候により必中・命中率変化。上記参照）
* 追加効果: あり（30%・まひ）
* まもる: 防がれる
* おうじゃのしるし: 第二世代は対象外、第三〜四世代は対象外、第五世代以降は対象
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
