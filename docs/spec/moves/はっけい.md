# はっけい（Force Palm）仕様書

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 60
* 命中: 100
* PP: 10
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、30%の確率で相手を「まひ」状態にする

## 判定フラグ

* 急所: あり
* 命中判定: あり
* 追加効果: あり（30%・まひ）
* まもる: 防がれる
* マジックコート: 反射不可（直接攻撃技のため対象外）
* オウムがえし: 反射可能
* おうじゃのしるし（追加効果発動率アップ系）: 対象

## 特記事項

* まひ無効のポケモン（でんきタイプ・じゅうなん等）には追加効果が発動しない（既存の共通処理に従う）
