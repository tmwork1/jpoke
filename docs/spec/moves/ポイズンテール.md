# ポイズンテール（Poison Tail）仕様書

参照: docs/wiki/moves/ポイズンテール.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 物理
* 威力: 50
* 命中: 100
* PP: 25（`docs/champions/move_list.txt` に本技のエントリが存在しないため、本家(Gen9)基準のPPをそのまま採用）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`, `secondary_effect`

## 効果

* 急所に当たりやすい（急所ランク+1）。
* 追加効果として、10%の確率で相手を『どく』状態にする。

## 判定フラグ

* 急所: 急所ランク+1（通常より高い確率で急所に当たる）
* 命中判定: あり（命中率100）
* 追加効果: あり（10%でどく状態）
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 発動する（追加効果を持つ攻撃技のため）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* 追加効果（どく付与）は `ON_DAMAGE_HIT` で処理する。ダメージ計算内部のイベントで
  `docs/spec/turn.md` には未掲載のため、既存の同種ハンドラ（クロスポイズン・どくづき 等）と
  同じ `ON_DAMAGE_HIT` / priority 100（デフォルト）に合わせる。
* `critical_rank=1` を `MoveData` に設定するのみで、既存の急所判定ロジック
  （`core/move_executor.py` の `CRIT_RATES` 参照処理）が自動的に適用される。

## リーサル計算

対象外（どく状態異常の付与は確定ダメージではなくターン経過依存の追加効果であり、
本プロジェクトのリーサル計算対象に含めない方針。`docs/progress/move.md` の
「リーサル実装」列は `n/a`）。
