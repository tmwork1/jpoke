# むしのていこう（Struggle Bug）仕様書

参照: docs/wiki/moves/むしのていこう.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 特殊
* 威力: 50（第六世代以降。第五世代は30）
* 命中: 100
* PP: 20
* 対象: 相手全員（シングルバトルでは相手1体）
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、100%の確率で相手の『とくこう』ランクを1段階下げる。

## 判定フラグ

* 急所: あり
* 命中判定: あり
* 追加効果: あり（100%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 奪われない（変化技ではないため対象外）
* オウムがえし: コピーされる
