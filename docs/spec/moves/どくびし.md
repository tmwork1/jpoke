# どくびし（Toxic Spikes）仕様書

参照: docs/wiki/moves/どくびし.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 変化
* 威力: -
* 命中: -（命中判定なし。相手の場を対象とする設置技のため必ず命中する）
* PP: 20（`docs/champions/move_list.txt` 準拠）
* 対象: 相手の場（target="foe_side"）
* 優先度: 0
* 接触: なし

## 効果

* 相手の場を『どくびし』状態にする（最大2層）。
* 『どくびし』状態の詳細（毒付与のタイミング・無効条件・消滅条件）は
  `docs/spec/side_fields/どくびし.md` を参照。技側の責務は「相手の場に1層設置する」ことのみ。
* すでに『どくびし』が最大2層設置されている場合、この技は失敗する。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（相手の場を対象とするため命中率の概念がない）
* 追加効果: なし（設置自体が主効果であり、追加効果扱いではない）
* まもる: 防がれない（`まもる`は使用者自身を対象とする技を防ぐものであり、
  相手の場を対象とする本技には無効。`Move.is_blocked_by_protect`は`target == "foe"`のみを
  対象とするため、`target="foe_side"`の本技は自動的にまもるの対象外になる）
* みがわり: 防がれない（相手の場を対象とするため、みがわりの影響を受けない）
* おうじゃのしるし: 対象外（変化技のため追加効果によるひるみ判定は発生しない）
* マジックコート: 反射可能（第五世代以降。`Move.is_reflectable`は
  `category == "status" and target in ("foe", "foe_side")`で判定するため、
  target="foe_side"の本技も反射対象に含まれる）
* よこどり: 対象外（本技は場を対象とするためよこどりで奪えない。ただし本プロジェクトでは
  よこどり自体が未実装のため実装上の考慮は不要）
* オウムがえし: 対象外（使用者自身を直接対象とする技ではないため、オウムがえしでは
  コピーできない。本プロジェクトではオウムがえし自体が未実装のため実装上の考慮は不要）
