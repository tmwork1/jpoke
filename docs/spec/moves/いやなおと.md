# いやなおと（Screech）仕様書

参照: docs/wiki/moves/いやなおと.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中率: 85（`src/jpoke/data/ps-champ-ja/moves.json` に準拠）
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` に準拠。Wiki記載のPP40はGen9本家基準であり、
  `docs/champions/changes_from_sv.md` の全技PP統一（8/12/16/20）により本プロジェクトでは20を採用する）
* 範囲: 相手1体選択
* 優先度: 0
* 直接攻撃: ×
* フラグ: 音のわざ（みがわりを貫通する）

## 効果

相手の **ぼうぎょ** ランクを **2段階** 下げる。

## 判定フラグ

* 急所: なし（変化技）
* 命中判定: あり（命中率85）
* 追加効果: なし（本体の効果がランクダウンそのもの）
* まもる: 防がれる
* みがわり: 貫通する（音のわざのため）
* おうじゃのしるし: 対象外（Wiki記載どおり全世代で無関係）
* マジックコート: 反射される
* よこどり: 奪えない（Wiki記載どおり）
* オウムがえし: コピーされる

## ポケモンチャンピオンズ向けの簡略化

* コンテスト効果（アピール等）は実装しない
* バトルパレスの判定基準（「ずるい」技）は実装しない
* ポケモン不思議のダンジョン・ポケモンマスターズ・ポケモンカードゲーム・アニメでの挙動は対象外
