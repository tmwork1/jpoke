# ロッククライム（Rock Climb）仕様書

参照: docs/wiki/moves/ロッククライム.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 90
* 命中: 85
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`（直接攻撃）、`secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、20%の確率で相手をこんらん状態にする。
* フィールド上で使うとゴツゴツした岩肌を登れる（ひでんわざとしての効果。本プロジェクトはバトルのみ
  対象のため対象外）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率85）
* 追加効果: あり（20%・こんらん付与）
* まもる: 防がれる
* おうじゃのしるし: 発動する（本技はひるみの追加効果を持たないため、通常通り10%の追加ひるみ効果が
  乗る。第四世代・第五世代以降ともに有効）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT`
  を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が
    行われなかった場合には発動しない仕様を担保する。

## リーサル計算

* HP依存の追加効果・威力変動を持たない通常のダメージ技であり、リーサル計算上の特別な考慮は不要
  （対象外）。

## 備考

* Zワザ威力: 175（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
* 第四世代のひでんわざであり、`docs/champions/move_list.txt` には掲載がない
  （ポケモンチャンピオンズ／スカーレット・バイオレットで使用不可の技）。PP・威力・命中は
  ポケモンWiki記載の値（第四世代以降変更なし）を採用する。
