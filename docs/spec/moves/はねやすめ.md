# はねやすめ 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | ひこう |
| 分類 | 変化 |
| PP | 8（championsルールにより8/12/16/20の4段階へ統一。docs/champions/move_list.txt準拠） |
| 命中率 | ー（命中判定なし） |
| 対象 | 自分 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

使用者自身の最大HPの1/2（切り捨て）を回復する。

さらに、使用者が **ひこうタイプを持つ場合**、そのターンが終わるまでひこうタイプを失う
（1タイプの場合は無タイプ、複合タイプの場合は残りのタイプのみになる）。

HP が満タンの場合は失敗する（回復・タイプ変化のいずれも発生しない）。

## ひこうタイプ除去の詳細

- タイプ除去は使用直後（`ON_STATUS_HIT`）に発生し、ターン終了時（`docs/spec/turn.md`
  priority 120「はねやすめのひこうタイプ復帰」）に自動で復帰する。
- **テラスタル中**は、テラスタイプがひこうであっても除去されない（実際のタイプはテラスタイプの
  みになっているため、除去対象がそもそも存在しない）。
- **ひこうタイプを持たないポケモン**（テラスタルも含めて現在ひこうタイプでない場合）が使用しても
  タイプ変化は発生せず、HP回復のみが発生する。
- タイプが失われている間は「ひこう」を要求・除外する各種判定（じめんタイプの技の命中、
  浮遊判定、フィールド・特性によるひこう限定効果など）に影響する。
- 交代した場合、はねやすめによるタイプ変化は即座に解除される（ボルティクル等と同様に
  控えのポケモンには影響しない）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない） |
| おうじゃのしるし | × |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |
| よこどり | ○ |
| オウムがえし | × |

`まもる`・`マジックコート` はいずれも `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。

## かいふくふうじとの相互作用

- 使用者が **かいふくふうじ** 状態の場合、`heal` フラグを持つ技として選択できない
  （`かいふくふうじ_try_action` が汎用的に処理するため、本技専用の実装は不要）。

## ポケモンチャンピオンズ向けの簡略化

- 特になし（第四世代以降、効果に世代差はない）
