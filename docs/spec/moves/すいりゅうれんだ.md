# すいりゅうれんだ（Surging Strikes）仕様書

参照: docs/wiki/moves/すいりゅうれんだ.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 25（1ヒットあたり）
* 命中: 100
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: パンチ技（てつのこぶし・パンチグローブの対象）

## 効果

* 3回連続で攻撃する固定回数の連続攻撃技（3回攻撃固定技）。
  * 命中判定は1発目のみ行う（`check_hit_each_time` は使用しない）。
  * 各ヒットの威力は固定で25（`power_sequence` なし）。
* 必ず急所に当たる（急所ランク+3 = 確率100%）。急所判定は各ヒットごとに独立して行われるため、
  全3ヒットがそれぞれ確定急所となる。
* 追加効果なし。

## 判定フラグ

* 急所: 確定（急所ランク3、各ヒットごと）
* 命中判定: あり（命中率100、1発目のみ判定）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）

## 急所無効化との関係

* 確定急所ではあるが、防御側の特性カブトアーマー・シェルアーマーを持つポケモンに対しては急所に当たらない
  （急所無効化特性が確定急所効果より優先される）。
  この無効化はカブトアーマー・シェルアーマー側のハンドラで処理されるため、本技側の実装には影響しない。

## 専用技

* ウーラオス（れんげきのかた）の専用技。

## 備考

* 確定急所を含めた実質威力は37相当（1ヒットあたり）。
* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
