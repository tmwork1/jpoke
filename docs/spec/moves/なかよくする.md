# なかよくする 仕様書

参照: docs/wiki/moves/なかよくする.html（ポケモンWiki）

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | ノーマル |
| 分類 | 変化 |
| 威力 | - |
| 命中率 | - （必中） |
| PP | 20 |
| 範囲 | 相手1体選択 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

相手の **こうげき** ランクを **1段階** 下げる。

自分の命中率、相手の回避率に関係なく必ず命中する（溜め技で姿を隠している相手を除く）。
相手の **まもる** 状態・**みがわり** 状態を無視して発動する。

## 判定

- 急所: なし（変化技）
- 追加効果: なし
- まもるで防がれる: ×（無視して発動する）
- みがわりを無視する: ○（貫通する）
- マジックコートで跳ね返される: ○
- オウムがえしでコピーされる: ○
- よこどりで奪われる: ×（対象を持たない変化技のため無効）
- トリックガードで防がれる: ○（トリックガードは本ライブラリ未実装のため対象外）

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- ポケモン不思議のダンジョンの仕様は対象外
