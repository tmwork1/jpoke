# うたう（Sing）仕様書

参照: docs/wiki/moves/うたう.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: なし
* 命中: 55（本プロジェクトはポケモンチャンピオンズ基準。champions/move_list.txt の値を正とする）
* PP: 16（champions/move_list.txt の値。Gen9本家は15だが、changes_from_sv.md によりPPは
  8/12/16/20の4段階に統一されているため、champions側では16になっている）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし
* フラグ: 音のわざ（ぼうおん・パンクロック・ヴィヴィヨン(音)・ドラゴンエールの対象、みがわりを貫通）

## 効果

* 相手を『ねむり』状態にする。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: あり（命中率55）
* 追加効果: なし（技自体の主効果としてねむりを付与する）
* まもる: 防がれる
* マジックコート: 反射される
* オウムがえし: 複製される
* みがわり: 貫通する（音のわざのため）
* おうじゃのしるし: 対象外（元々追加効果を持たない技のため無関係）
* エレキフィールド: 効果を受ける側が地面にいる場合、ねむり付与不可（フレームワーク側で処理）
* さわぐ状態: 相手が『さわぐ』状態の間はねむり状態にできない（フレームワーク側で処理）

## 備考

* 第一世代から存在する、ねむり状態を付与する代表的な変化技。命中率55は歴代のねむり付与技の中でも低い部類。
