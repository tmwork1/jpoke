# こわいかお（Scary Face）仕様書

参照: docs/wiki/moves/こわいかお.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: 100（第五世代以降。第四世代までは90だが本プロジェクトの対象範囲外）
* PP: 12（`docs/champions/move_list.txt` 準拠。本家基準は10だが、
  `docs/champions/changes_from_sv.md` により全技PPが8/12/16/20の4段階に統一されている）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 相手の『すばやさ』ランクを2段階下げる。

## 判定フラグ

* 急所: なし（対象外）
* 命中判定: あり（命中率100）
* 追加効果: なし（本効果自体が確定効果）
* まもる: 防がれる
* マジックコート: 反射される
* よこどり: 対象外（相手に向けた効果のため奪えない）
* オウムがえし（ミラームーブ）: コピーされる
* おうじゃのしるし: 対象外（ひるみ関連の効果ではないため無関係）

## ランク低下無効化との相互作用（フレームワーク側で処理、実装済みの汎用ロジックに委譲）

* 特性「クリアボディ」「メタルプロテクト」、道具「しろいけむり」、
  特性「フラワーベール」やフィールド効果「しろいきり」を持つ相手には
  すばやさのランク低下が無効化される。
* 特性「まけんき」「かちき」を持つ相手に使うと、ランク低下発動時にそれぞれの特性が
  発動する。

## 備考

* 同じ効果（相手のすばやさを2段階下げる変化技）を持つ技に「わたほうし」「いとをはく」がある。

* わたほうし・いとをはく（同系統のすばやさ低下技）
