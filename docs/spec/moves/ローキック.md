# ローキック（Low Sweep）仕様書

参照: docs/wiki/moves/ローキック.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 60（第五世代）→ 65（第六世代以降）
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、100％の確率で相手の『すばやさ』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・すばやさ-1、`flags={"secondary_effect"}` によりちからずく等の対象）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射される

## 備考

* Zワザ威力: 120（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 80（本プロジェクトでは対象外）。
