# 仕様書: オクタンほう

## 対象範囲外（実装しない）

本技は第2世代から存在する技だが、**ポケットモンスター ソード・シールド以降（ソード・シールド／BDSP／スカーレット・バイオレット）で使用不可**
（`docs/wiki/moves/オクタンほう.html` のカテゴリ「スカーレット・バイオレットで使用不可能なわざ」および
本文中の赤字表記「SVで使用不可」より確認）。`src/jpoke/data/ps-champ-ja/moves.json`（ポケモンチャンピオンズの技一覧）
にも本技の項目は存在せず、`docs/spec/abilities/ぼうだん.md` の対応表でも第9世代の列が「-」（技が登場していない）
となっており、ポケモンチャンピオンズ側でも採用されていないことが二重に確認できる。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に該当するため、
本技は実装対象外とする。

過去のレビューにて `data/moves/move_a.py` に本技の `MoveData`（威力110・PP10、`Event.ON_DAMAGE_HIT` での
命中率ダウン追加効果）と `handlers/move_attack.py` の専用ハンドラ `オクタンほう_lower_acc`、
`types/move.py` の `MoveName` Literal、および `tests/moves_attack/test_move_a.py` にテストが
誤って登録されていたことが判明したため、本レビューで削除した。詳細は `docs/review/moves/オクタンほう.md` を参照。
