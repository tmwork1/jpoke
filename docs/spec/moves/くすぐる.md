# くすぐる（Tickle）仕様書

参照: docs/wiki/moves/くすぐる.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: 100
* PP: 20（`docs/champions/move_list.txt` 準拠。本家と同じ値）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 音技（sound）: なし。相手の『みがわり』状態には防がれる（第三世代のみ例外的にみがわりを貫通するが、本プロジェクトの対象範囲外）

## 効果

* 相手の『こうげき』『ぼうぎょ』ランクをそれぞれ1段階下げる。

## 判定フラグ

* 急所: なし（対象外）
* 命中判定: あり（命中率100）
* 追加効果: なし（本効果自体が確定効果）
* まもる: 防がれる
* マジックコート: 反射される
* よこどり: 対象外（相手に向けた効果のため奪えない）
* オウムがえし（ミラームーブ）: コピーされる
* おうじゃのしるし: 対象外（ひるみ関連の効果ではないため無関係）

## ランク低下無効化との相互作用（フレームワーク側で処理、実装済みの汎用ロジックに委譲）

* 特性「クリアボディ」「メタルプロテクト」、道具「しろいけむり」、
  特性「フラワーベール」やフィールド効果「しろいきり」を持つ相手には
  こうげき・ぼうぎょのランク低下がいずれも無効化される。
* 特性「かいりきバサミ」を持つ相手には、こうげきのランク低下のみ無効化され、
  ぼうぎょのみ下がる。
* 特性「まけんき」「かちき」を持つ相手に使うと、ランク低下発動時にそれぞれの特性が
  2回（こうげき低下・ぼうぎょ低下のそれぞれで）発動し、まけんきの場合はこうげきが
  最終的に3段階上昇しぼうぎょが1段階下降、かちきの場合はこうげき・ぼうぎょが
  1段階ずつ下降しとくこうが4段階上昇した状態になる。
* こうげきランクがすでに-6の状態でも、ぼうぎょのランク低下は独立して適用される
  （片方が下限で下がらなくても、もう片方は下がる）。

## 備考

* 同じ効果を持つ技に「おたけび」（こうげき・とくこう低下、音技）がある。
* 2種類の相手のステータスを1段階ずつ下げる技は他に、おたけび・すてゼリフ・
  たこがため・なみだめなどがある。
