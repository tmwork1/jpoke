# アームハンマー（Hammer Arm）仕様書

参照: docs/wiki/moves/アームハンマー.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 100
* 命中: 90
* PP: 12（`docs/champions/move_list.txt` 準拠。本家シリーズ(SVまで)のPPは10だが、
  Championsでは全ての技のPPが8・12・16・20の4段階に統一されており、
  `docs/champions/changes_from_sv.md` の記載どおり本技は12になる）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）、パンチ技（`punch`。てつのこぶしの対象）
* フラグ: `contact`、`punch`

## 効果

* 技が成功すると、自分の『すばやさ』ランクが1段階下がる（確定効果）。

## 判定フラグ（一次情報より）

* 急所: 通常判定（○）
* 命中判定: あり（命中率90）
* 追加効果: ×（一次情報の判定表でも「追加効果: ×」と明記されている。
  自分自身への確定ランク低下であり、相手への追加効果ではない）
* まもる: 防がれる（○）
* おうじゃのしるし: 第四世代・第五世代以降ともに対象（一次情報の判定表に記載。
  本技の追加効果はひるみ効果ではないため実際には無関係）
* マジックコート: 対象外（変化技ではないため）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 対象になる（○）
* ちからずく: 対象外。`docs/wiki/abilities/ちからずく.html` の一次情報によれば、
  ちからずくの効果対象となる技は「相手のランク補正を下げる技」「自分のランク補正を
  上げる技」等に限られ、「自分のランク補正を下げる技」は対象に含まれない。
  したがって `secondary_effect` フラグを付与しない
  （`docs/spec/moves/いじげんラッシュ.md` の同種の判断と同じ）。

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_HIT`（ダメージ発生後の処理）の
  「10 自分のランクが下がる反動」に該当する処理だが、本プロジェクトの既存実装では
  自分自身への確定ランク低下を伴う同種の技（りゅうせいぐん・リーフストーム・
  いじげんラッシュ等）はいずれも `Event.ON_HIT` の優先度100（デフォルト）で
  実装されており、本技もそのパターンに合わせる。
* `subject_spec` はデフォルトの `attacker:self`（`MoveHandler` 既定値）を使用する。

## リーサル計算

* `docs/progress/move.md` の該当行は「リーサル実装: n/a」「リーサルテスト: n/a」であり、
  本レビューの対象範囲外（対象外）とする。
