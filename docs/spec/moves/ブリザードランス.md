# ブリザードランス（Glacial Lance）仕様書

参照: docs/wiki/moves/ブリザードランス.html（ポケモンWiki）

## 基本情報

* タイプ: こおり
* 分類: 物理
* 威力: 130（第八世代）→ 120（第九世代）
* 命中: 100
* PP: 5（Gen9本家）→ Champions: 8（PPは8/12/16/20の4段階に統一。
  `src/jpoke/data/ps-champ-ja/moves.json` には本技の記載がないが、base PP5の技（アストラルビット・
  でんじほう・はかいこうせん・じばく・だいばくはつ等）が軒並み8へ変換されていることから
  同様に8へ変換されるものとして扱う）
* 対象: 相手全員（ダブルバトル用の範囲だが、本プロジェクトはシングルバトルのみ対象のため
  実質的に相手1体を対象とする通常の攻撃技として扱う）
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 追加効果: なし（通常攻撃）
* 急所ランク補正: なし（急所には当たりうるが、ランク補正を持つ技ではない。
  ストーンエッジ等とは異なり「急所に当たりやすい」効果は記載されていない）

## 効果

* 通常のダメージ技。追加効果は持たない。
* バドレックス（はくばじょうのすがた）専用技。

## 判定フラグ

* 急所: あり（通常の急所判定のみ。ランク補正なし）
* 命中判定: あり（通常通り命中率100で判定）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）

## 備考

* バドレックス（はくばじょうのすがた）の専用技。キズナのタヅナではくばじょうのすがたに
  なると覚え、通常のすがたに戻ると忘れる。
* デメリットのないこおりタイプの技で最も威力が高い。
* こおりタイプ物理技の中で唯一の「相手全員」を対象とする技（本プロジェクトではシングル
  バトルのみを扱うため、この特性による追加のゲームルール上の違いはない）。
