# アーマーキャノン（Armor Cannon）仕様書

参照: docs/wiki/moves/アーマーキャノン.html（ポケモンWiki）、src/jpoke/data/ps-champ-ja/moves.json

## 基本情報

* タイプ: ほのお
* 分類: 特殊
* 威力: 120
* 命中: 100
* PP: 8（champions 版で 8/12/16/20 の4段階に統一されているため、src/jpoke/data/ps-champ-ja/moves.json の
  値を正とする。Gen9 本家Wiki記載の PP5 は移行元の値であり本プロジェクトでは採用しない）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 通常攻撃。技が成功した（命中し、ダメージを与えた）後、自分の『ぼうぎょ』『とくぼう』ランクが
  1段階ずつ下がる（確率100%・確定効果）。
* この能力ランクダウンは「追加効果」には分類されない（下記判定フラグ参照）。そのため、
  シールドダスト等の追加効果を無効化する特性・効果や、ちからずくによる威力上昇・追加効果無効化の
  対象にはならない。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし（自分の能力ランクダウンは追加効果として扱われない）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は発動する（追加効果を持たないため実質影響なし）
* マジックコート: 反射不可（変化技ではないため対象外）

## 備考

* グレンアルマの専用技。特殊版インファイトと言える性能で、だいもんじより威力・命中が高い一方、
  自分の防御・特防を下げるデメリットを持つ。
