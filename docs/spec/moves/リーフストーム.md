# リーフストーム（Leaf Storm）仕様書

参照: docs/wiki/moves/リーフストーム.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 140（第五世代まで）→ 130（第六世代以降。本プロジェクトの対象世代ではこちらを採用）
* 命中: 90
* PP: 8（ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されており、
  Gen9本家基準の5ではなくこの値を採用する。`docs/champions/move_list.txt:934` 記載の値）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 命中して技が成功した後、反動として自分の『とくこう』ランクが2段階下がる（確定効果）。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率90）
* 追加効果: なし（自分自身への確定効果であり、相手への「追加効果」には該当しない）
* まもる: 防がれる
* おうじゃのしるし: 対象外（「追加効果なし」技のため発動しない）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能（本プロジェクトでは対象外の古い技のため実装しない）
* ちからずく: 非対象技（自分への確定ランク低下は「追加効果」扱いではないため、
  ちからずくによる威力上昇・追加効果無効化の対象にならない）

## 備考

* 反動でとくこうが下がる代表的な高火力特殊技。同系統の技（オーバーヒート・
  りゅうせいぐん・サイコブースト・フルールカノン等）と同様の「反動技#能力が低下する」
  仕様に従う。
* クリアチャームなど「相手の技によるランク低下を防ぐ」効果は、自分自身の技による
  ランク低下には無効（`tests/items/test_item_ka.py` の
  `test_クリアチャーム_自分の技の低下は防げない` で確認済み）。
* しろいハーブを持たせた場合、命中後に下がったランクをその場で元に戻す
  （`tests/items/test_item_sa.py` で確認済み）。
