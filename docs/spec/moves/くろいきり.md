# くろいきり（Haze）仕様書

参照: docs/wiki/moves/くろいきり.html（ポケモンWiki）

## 基本情報

* タイプ: こおり
* 分類: 変化
* 威力: -
* 命中: - （必中）
* PP: 20（`docs/champions/move_list.txt` 準拠）
* 対象: 全体の場（`target="field"`）
* 優先度: 0
* 接触: なし

## 効果

* 黒い霧を出して、場に出ているポケモン全員の能力ランク変化を元に戻す（±0にリセットする）。
* 自分・相手を問わず、場にいるすべてのポケモンが対象。

## 判定フラグ

* 命中判定: 必中（命中判定自体が行われない）
* まもる: 効果を受けない（`守×`。まもるで防がれない）
* マジックコート: 反射不可（`_move_labels.md` の反射対象外の変化技）
* よこどり: 対象外
* オウムがえし: コピー可能（本実装ではオウムがえし自体は対象外の仕様のため実質適用外）

## ランクリセット効果の詳細

* 場にいる全ポケモン（シングルバトルでは使用者・相手の2体）の能力ランク変化
  （こうげき・ぼうぎょ・とくこう・とくぼう・すばやさ・命中率・回避率）を全て±0にリセットする。
* みがわり状態のポケモンや、姿を隠している（溜め技の実行中など）ポケモンのランク補正も
  リセット対象になる（全体の場を対象とした技であるため、通常の「対象選択」を経由しない）。
* きゅうしょランク（きゅうしょアップ状態など）はリセット対象外
  （`Pokemon.rank` の管理する `Stat` に急所ランクは含まれないため、実装上自然に満たされる）。

### リセット効果がその他の要素から影響を受けない点

* 特性クリアボディ／しろいけむり／メタルプロテクト／かいりきバサミ／はとむね／するどいめ／
  フラワーベール／ミラーアーマー、しろいきり状態（場の状態）を無視してランク補正を
  リセットする（`battle.modify_stats` を経由せず `Pokemon.rank` を直接書き換えるため、
  これらのランク低下防止・反射系ハンドラが発動する `ON_BEFORE_MODIFY_STAT` を経由せず、
  自然に無視できる）。
* この技によるランク補正の変化は特性あまのじゃく／たんじゅんの影響を受けない
  （同上の理由で自然に満たされる）。
* 結果的にランクが下がったことで特性まけんき／かちきが発動することや、上がったことで
  特性びんじょう／持ち物ものまねハーブの効果が発動することはない（`battle.modify_stats` を
  経由しないため、これらの「ランク変化検知」系ハンドラが発火しない）。
* この技によりランク補正がリセットされたことで、しっとのほのお／うっぷんばらしの効果が
  発動することはない（同上）。

## 発動タイミング

| イベント | Priority | 根拠 |
|---------|----------|------|
| Event.ON_STATUS_HIT | 100（デフォルト） | `docs/spec/turn.md` に未掲載のイベント。`move_executor.py` が変化技の命中処理として発火する内部イベントであり、既存の変化技ハンドラ（くろいまなざし等）と同様にデフォルトpriorityを使用する |

## 備考

* 似た名前の技にしろいきりがあるが、こちらは他のポケモンからランク補正を下げられることを
  防ぐ「場の状態」であり、くろいきりの「使用したポケモン全員のランクを±0にリセットする」
  効果とは全く異なる。くろいきりはしろいきりの効果を無視してランクをリセットできる。
* クリアスモッグとの違い: クリアスモッグは攻撃技（ダメージ+ダメージを与えた相手のみの
  ランクリセット）で `ON_DAMAGE_HIT` 後に発動するが、くろいきりは純粋な変化技
  （ダメージなし・場にいる全ポケモンが対象）。
* 第一世代のみ、状態変化・状態異常の一部を回復する効果があったが、第二世代以降は
  ランク変動のみに影響する仕様に変更されている。本プロジェクトは対象外の世代仕様のため
  実装しない。
