# クロスチョップ（Cross Chop）仕様書

参照: docs/wiki/moves/クロスチョップ.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 100
* 命中: 80
* PP: 8（ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されており、
  Gen9本家基準の5ではなくこの値を採用する）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* 急所ランク: +1（急所に当たりやすい）

## 効果

* 通常攻撃。追加効果なし。
* 急所ランク+1（急所に当たりやすい）。

## 判定フラグ

* 急所: あり（急所ランク+1）
* 命中判定: あり（命中率80）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 対象外（追加効果を持たないため発動率上昇の余地がない）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* 追加効果を持たないため、専用ハンドラは不要（`handlers={}`）。
* `critical_rank=1` により、`core/move_executor.py` が急所判定に反映する（追加実装不要）。

## リーサル計算

* HP依存の追加効果・威力変動を持たない通常のダメージ技であり、リーサル計算上の特別な考慮は不要（対象外）。
