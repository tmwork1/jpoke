# スパーク（Spark）仕様書

参照: docs/wiki/moves/スパーク.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 物理
* 威力: 65
* 命中: 100
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` に本技のエントリなし。第九世代基準のPPが既に統一4段階
  （8/12/16/20）のひとつであるため、champions側の変更は無いものと判断し変更しない）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、30%の確率で相手をまひ状態にする。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（30%でまひ）
* まもる: 防がれる
* おうじゃのしるし: 第二世代: ×、第三世代〜第四世代: ×、第五世代以降: ○
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 120（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 120（本プロジェクトでは対象外）。
* でんきタイプの相手はまひ無効（`ailment_manager` 側で共通判定されるため本技固有の実装は不要）。
