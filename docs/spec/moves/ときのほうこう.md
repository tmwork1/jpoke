# ときのほうこう（Roar of Time）仕様書

参照: docs/wiki/moves/ときのほうこう.html（ポケモンWiki）

## 基本情報

* タイプ: ドラゴン
* 分類: 特殊
* 威力: 150
* 命中: 90
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 直接攻撃: なし
* 分類フラグ: なし

## 効果

* 技を命中させて成功した次のターン、使用者は反動で行動できなくなる（いわゆる「リチャージ」状態）。
  * がんせきほう・ギガインパクト・はかいこうせん・ブラストバーン・ハードプラント・ハイドロカノンと同じ効果。
* まもるなどで技が防がれた場合や、命中しなかった場合はリチャージ状態にならない。
* リチャージ状態のポケモンが交代すると、リチャージ状態は解除される。
* 音の技のような技名だが、音技には分類されず、ぼうおん等で無効化されない（本プロジェクトでは音技フラグを付与しない）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率90）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代・第五世代以降ともに対象）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射される

## 備考

* ディアルガの専用技。
* Zワザ威力: 200（本プロジェクトでは対象外）。
