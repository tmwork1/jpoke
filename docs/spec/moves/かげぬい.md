# かげぬい（Spirit Shackle）仕様書

参照: docs/wiki/moves/かげぬい.html（ポケモンWiki）

## 基本情報

* タイプ: ゴースト
* 分類: 物理
* 威力: 90（`src/jpoke/data/ps-champ-ja/moves.json` / `docs/progress/move.md` 準拠。本家シリーズ(SVまで)の威力は80だが、Championsでは90）
* 命中: 100
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` / `docs/progress/move.md` 準拠。本家シリーズのPPは10だが、Championsでは12）
* 対象: 相手1体（本家の範囲は「自分以外」だが、本プロジェクトはシングルバトルのみ対象のため相手1体で問題ない）
* 優先度: 0
* 接触: なし
* フラグ: なし（ちからずくの対象技となる `secondary_effect` フラグのみ付与）

## 効果

* 追加効果として、相手を『にげられない』状態にする（100%）。
* ゴーストタイプのポケモンは『にげられない』状態の効果を無視できる（`にげられない` 揮発性状態自体の仕様。docs/spec/volatiles/にげられない.md 参照）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・にげられない状態の付与）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象になる（本追加効果はひるみ効果ではないため無関係。ただし一次情報の判定表に記載されている）
* よこどり: 対象外
* オウムがえし: 対象になる
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、にげられない状態の付与効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）
* りんぷん・おんみつマント: 対象の特性がりんぷん、または対象がおんみつマントを持つ場合、にげられない状態の付与効果が発動しない。
  * `resolve_secondary_chance` が Event.ON_MODIFY_SECONDARY_CHANCE 経由でりんぷん・おんみつマントの無効化ハンドラを呼び出すため、本技側の実装で個別対応する必要はない。
* ノーマルタイプなどに技自体が無効化された場合、追加効果（にげられない状態の付与）は発生しない（ダメージが発生しなかった場合はイベントが発火しないため、本技側の実装で個別対応する必要はない）。

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当。`はたきおとす` / `うたかたのアリア` 等の実装コメント参照）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。

## 専用技

* ジュナイパーの専用技。
* 同じタイプの組み合わせを持つダダリンは同じ効果を持つ専用技「アンカーショット」を覚える（SVまでは威力が同じだが、タイプ・直接攻撃判定・PPが異なる）。
