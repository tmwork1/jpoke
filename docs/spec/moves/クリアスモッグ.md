# クリアスモッグ（Clear Smog）仕様書

参照: docs/wiki/moves/クリアスモッグ.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 特殊
* 威力: 50
* 命中: - （必中）
* PP: 16（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。Gen9本家は15だが、チャンピオンズ全技は
  `docs/champions/changes_from_sv.md` により8/12/16/20の4段階に統一されているため16が正）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 特殊な泥の塊を相手に投げつけて攻撃する。ダメージを与えた相手の能力ランク変化を全て±0に戻す。
* 自分の命中率、相手の回避率に関係なく必ず命中する。

## 判定フラグ

* 急所: あり
* 命中判定: 必中（命中判定自体が行われない）
* 追加効果: なし（ランクリセットは「追加効果」に分類されない）
* まもる: 防がれる
* おうじゃのしるし: 発動する（通常の攻撃技と同様の一般ルール）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能（本実装ではオウムがえし自体は対象外の仕様のため実質適用外）

## ランクリセット効果の詳細

* ダメージを与えた対象のみ、能力ランク変化（こうげき・ぼうぎょ・とくこう・とくぼう・すばやさ・
  命中率・回避率）を全て±0にリセットする。くろいきりと異なり、リセット対象は攻撃を当てた
  相手のみで、場にいる全ポケモンには影響しない。
* きゅうしょランク（急所ランクアップ状態・キョダイシンゲキ等）はリセット対象外
  （`Pokemon.rank` の管理する `Stat` に急所ランクは含まれないため、実装上自然に満たされる）。

### リセットできない場合

* 相性・特性などにより攻撃が無効化された場合（技が命中判定より前に失敗するため、
  ダメージが発生せずリセットも発動しない）。
* まもる状態に防がれた場合。
* みがわり状態に防がれた場合。ただし、使用者がすりぬけ特性を持ち、みがわりを無視して
  攻撃した場合はリセットが発動する。

### 0ダメージでもリセットされる場合（既知の未実装ニュアンス）

一次情報（Wiki）によれば、特性ばけのかわでダメージを肩代わりされた場合や、対象がHP1で
こらえるを使用していた場合など、実ダメージが0になった場合でもランクリセットは発動する。

本実装では `docs/spec/turn.md` の Event.ON_DAMAGE（実装上の `Event.ON_DAMAGE_HIT`）
priority=10 に「クリアスモッグによるランクリセット」が明記されており、`ON_DAMAGE_HIT` は
実ダメージが0の場合（みがわり肩代わり・ばけのかわ・こらえる等）は発火しない設計になっている。
そのため、通常のダメージが発生するケースとみがわりに阻まれるケースは正しく再現できるが、
ばけのかわ・こらえるによる0ダメージのケースでのリセットは本実装では発動しない
（`docs/spec/turn.md` に記載済みの設計判断を優先し、本レビューでは変更しない）。

### リセット効果がその他の要素から影響を受けない点

* ダメージを与えられた場合、特性クリアボディ／しろいけむり／メタルプロテクト／
  かいりきバサミ／はとむね／するどいめ／フラワーベール／ミラーアーマー、しろいきり状態を
  無視してランク補正をリセットする（`battle.modify_stats` を経由せず `Pokemon.rank` を直接
  書き換えるため、これらのランク低下防止・反射系ハンドラが発動する `ON_BEFORE_MODIFY_STAT`
  を経由せず、自然に無視できる）。
* この技によるランク補正の変化は特性あまのじゃく／たんじゅんの影響を受けない
  （同上の理由で自然に満たされる）。
* 結果的にランクが下がったことで特性まけんき／かちきが発動することや、上がったことで
  特性びんじょう／持ち物ものまねハーブの効果が発動することはない（`battle.modify_stats` を
  経由しないため、これらの「ランク変化検知」系ハンドラが発火しない）。
* この技によりランク補正がリセットされたことで、しっとのほのお／みわくのボイス／
  うっぷんばらしの効果が発動することはない（同上）。
* いかりによる攻撃上昇の後にリセット効果が発動した場合、いかりによるランク変動もリセットされる
  （同一時点の `Pokemon.rank` を直接書き換えるため、発動順序どおりに自然に満たされる）。
* リセット効果発動後に特性じきゅうりょく／いかりのつぼ／いかりのこうら／ぎゃくじょう、
  持ち物じゃくてんほけん／チイラのみ等のピンチきのみ／タラプのみの効果が発動する場合、
  これらによるランク変動はリセットされない（`docs/spec/turn.md` の Event.ON_DAMAGE
  priority=10 より後（20以降）にこれらの効果が配置されており、本実装のハンドラを
  priority=10 で登録しているため、発動順序が自然に守られる）。
* 使用者の特性がちからずくである場合でも威力は上がらず、技の効果は発動する（この技には
  `secondary_effect` フラグを付与していないため、ちからずくの威力上昇・追加効果無効化の
  対象外となる）。
* 対象が特性りんぷんや持ち物おんみつマントを持つ場合でも技の効果は発動する
  （`ON_MODIFY_SECONDARY_CHANCE` を経由しないため、これらの「追加効果無効化」系ハンドラの
  影響を受けない）。

## 備考

* くろいきりの効果とは異なり、ランクをリセットするのは攻撃を当てた対象のみ。
* マックスレイドバトルのふしぎなバリアに関する仕様は、本プロジェクトの対象外
  （ポケモンチャンピオンズシングルバトルのみ）のため実装しない。
