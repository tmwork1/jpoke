# パワーウィップ（Power Whip）仕様書

参照: docs/wiki/moves/パワーウィップ.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 120
* 命中: 85
* PP: 12（Champions基準。src/jpoke/data/ps-champ-ja/moves.json に記載のとおり）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果なし（通常攻撃）。

## 判定フラグ

* 急所: あり（通常の急所判定。急所ランク補正なし）
* 命中判定: あり（命中率85）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ/ダイマックスわざ関連仕様は本プロジェクトでは対象外。
* 「切る技」フラグは付与しない（wikiのカテゴリに「切る技」の記載なし。つるや尻尾を叩きつける攻撃）。
