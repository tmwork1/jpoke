# はなふぶき（Petal Blizzard）仕様書

参照: docs/wiki/moves/はなふぶき.html（ポケモンWiki）
参照: docs/champions/move_list.txt, docs/champions/changes_from_sv.md（ポケモンチャンピオンズ仕様）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 90
* 命中: 100
* PP: 16
  * 一次情報（ポケモンWiki、第9世代基準）ではPP15だが、`docs/champions/changes_from_sv.md` に
    「すべての技のPPが8・12・16・20の4段階に統一」とある通り、本プロジェクトの対象である
    ポケモンチャンピオンズでは基礎PP15が最も近い段階の16に丸められる。
    `docs/champions/move_list.txt` の記載値（16）を正とする。
* 対象: 自分以外の全員（本プロジェクトはシングルバトルのみ対象のため、相手1体への通常攻撃と
  同義に扱う）
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 追加効果: なし（通常攻撃のみ）

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: ひるみ効果あり（第五世代以降。直接攻撃フラグはないが判定上は対象）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可（本プロジェクトはオウムがえし自体が未実装のため対応不要）

## 風の技（wind）フラグ

* 「風の技」に分類され、特性 かぜのり／ふうりょくでんき の効果対象になる
  （`flags={"wind"}` として実装、`かぜのり_absorb_wind` / `ふうりょくでんき_on_damage` が対象判定に使用）。

## リーサル計算

* 固定威力・追加効果なしの通常攻撃技のため、`lethal.py` に専用ハンドラは不要
  （`docs/progress/move.md` のリーサル実装／リーサルテスト列は n/a のまま対象外）。
