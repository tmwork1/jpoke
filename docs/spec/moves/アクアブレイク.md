# アクアブレイク（Aqua Break）仕様書

参照: docs/wiki/moves/アクアブレイク.html（ポケモンWiki）
※ 本セッションでは wiki.pokemonwiki.com へのネットワークアクセスがタイムアウトし一次情報を
取得できなかったため、`src/jpoke/data/ps-champ-ja/moves.json`・`docs/spec/moves/_move_secondary_1.md`・
既存実装（`src/jpoke/data/moves/move_a.py` / `src/jpoke/handlers/move_attack.py`）・
既存テスト（`tests/moves_attack/test_move_a.py`, `tests/abilities/test_ability_ma.py`）・
`docs/spec/abilities/みずがため.md`（アクアブレイク/シェルブレードとの発動順序について既に
一次情報照合済み）を突き合わせて内容を記載した。

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 85
* 命中: 100
* PP: 12（champions基準。src/jpoke/data/ps-champ-ja/moves.json 準拠。本家基準は10）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、20%の確率で相手の『ぼうぎょ』ランクを1段階下げる
  （`docs/spec/moves/_move_secondary_1.md` 10行目: 対象=defender、確率=0.2、B=-1）。
* 「切る技」には該当しない（`docs/spec/abilities/きれあじ.md` の切る技一覧に記載なし）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（20%・ぼうぎょ-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・ぼうぎょ低下の
  追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）

## 特性との相互作用

* みずがため: アクアブレイクを受けた場合、追加効果（ぼうぎょ-1）が先に適用されてから
  みずがための『ぼうぎょ』2段階上昇が発動する（`docs/spec/abilities/みずがため.md` 参照）。
  この順序はランク上限のクランプ挙動に影響するため、実装上は
  `Event.ON_DAMAGE_HIT priority=10`（みずがための`priority=20`より先）で登録する必要がある。
