# フリーズドライ（Freeze-Dry）仕様書

参照: docs/wiki/moves/フリーズドライ.html（ポケモンWiki）

## 基本情報

* タイプ: こおり
* 分類: 特殊
* 威力: 70
* 命中: 100
* PP: 20（`docs/champions/move_list.txt` 準拠）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* みずタイプのポケモンに対しても効果抜群（2倍）になる特殊なタイプ相性を持つ。
  * これは追加効果ではなく、タイプ相性計算そのものの特例。通常こおりタイプはみずタイプに1/2倍だが、
    本技に限りみずタイプへの相性が2倍として扱われる（他のタイプとの相性は通常のこおりタイプと同じ）。
  * さかさバトル等でタイプ相性が反転する場合でも、みずタイプへの効果抜群は維持される。
  * ノーマルスキンでノーマルタイプに変化しても、みずタイプへの効果抜群は残る。
  * テクスチャー2を使用しても通常のこおりタイプの技として扱われ、みずタイプに変化することがある
    （本プロジェクトでは対象外の効果）。
  * 特性りんぷん・おんみつマントを持つみずタイプに対してもバツグンになる。
  * 対象の特性がテラスシェルの場合は、特性の効果が優先されて効果はいまひとつ（半減）に変化する。
* **Champions（本プロジェクト対象）における変更**: SVまでは追加効果として10%の確率で相手を『こおり』状態にしたが、
  Championsではこの追加効果が削除されている（`docs/champions/changes_from_sv.md` 参照）。
  * 追加効果が存在しないため、特性ちからずくの効果（威力1.3倍・追加効果無効化）も発動しない。

## 判定フラグ

* 急所: 攻撃時のみ判定
* 命中判定: あり（命中率100）
* 追加効果: なし（Championsでは削除済み）
* まもる: 防がれる（○）
* おうじゃのしるし: 発動する（第五世代以降、○）
* マジックコート: 反射不可（×、変化技ではないため対象外）
* よこどり: 対象外（×、変化技ではないため）
* オウムがえし: 反射可能（○）

## 備考

* Zワザ威力: 140（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 120（本プロジェクトでは対象外）。
* みずタイプへの効果抜群により、使用者の持ち物たつじんのおびが発動し、相手の持ち物ヤチェのみが発動してダメージが半減する。
  相手の特性フィルター/ハードロック/プリズムアーマーの効果でダメージは3/4倍になる。使用者の特性ブレインフォースにより
  ダメージが増える。使用者の特性いろめがねの効果は発動しない（みずタイプに対しては「いまひとつ」ではなくなるため）。
* みずタイプに抜群になる効果は特性きけんよちの判定（相手の技が効果抜群かどうかの警告）には影響しない。
