# アシッドボム（Acid Spray）仕様書

参照: docs/wiki/moves/アシッドボム.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 特殊
* 威力: 40
* 命中: 100
* PP: 20（champions基準。Gen9本家は20のため変更なし）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない、弾のわざ）

## 効果

* 命中時、100%の確率で相手の『とくぼう』ランクを2段階下げる。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: あり（100%）
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する（第五世代以降。第三・四世代では
  既に追加効果を持つ技のため発動しなかったが、第五世代以降は既存の追加効果と
  併せてひるみが発動しうる仕様に変更された。この挙動はアイテム側の汎用実装で
  処理され、技側の実装に特別な対応は不要）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 相互作用・エッジケース

* ちからずく: 追加効果（とくぼうダウン）を無効化する代わりに威力が1.3倍になる。
* てんのめぐみ: 追加効果の発動率に影響しない（すでに100%のため）。
* かたやぶり/ハンター等の特性無効化はランク変化を防ぐ特性
  （クリアボディ・しろのつばさ等）を無視する。
* とくぼうが既に-6のときは、下降自体が発生しないため
  ミラーアーマー等の反射系特性は反応しない。
