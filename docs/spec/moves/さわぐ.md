# さわぐ 仕様書

対象: Pokémon Champions シングルバトル（SVシリーズ準拠）

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | ノーマル |
| 分類 | 特殊 |
| 威力 | 90（第五世代以降。第四世代までは50） |
| 命中率 | 100 |
| PP | 10 |
| 範囲 | 相手1体（ダブル以上ではランダム1体だが、シングルのみ対象のため常に唯一の相手） |
| 優先度 | 0 |
| 直接攻撃 | × |
| フラグ | `sound`（音technique）、`non_negoto`（ねごとで選ばれない） |

## 効果

- 命中したとき、使用者に **さわぐ** 揮発性状態（3ターン固定）を付与する。
  - さわぐ状態の詳細（強制行動・ねむり無効化・さわがしい補助状態など）は
    `docs/spec/volatiles/さわぐ.md` を参照。
- 最初の使用が命中しなかった場合（外れ・無効化・まもる等で防がれた）は、
  さわぐ状態にならない。次のターンは通常どおり技を選択できる。
- すでにさわぐ状態（2・3ターン目の強制行動）の場合、この効果は何もしない
  （二重付与を防ぐ）。

## 技としての特徴

- **音の技**: `sound` フラグにより、命中判定時にみがわり状態を貫通してダメージを与える
  （第六世代以降の仕様。実装は `move_executor.check_hit_substitute` の
  `has_flag("sound")` チェックで共通対応済み）。
- **ねごとで選ばれない**: `non_negoto` フラグにより「ねごと」の候補技から除外される。
  また `non_negoto` フラグの技は `battle.last_used_move_name` に記録されないため、
  「さいはい」でこの技をコピーすることはできない（さいはいは失敗する）。
- **追加効果に分類されない**: さわぐ状態の付与は「追加効果」ではないため、
  ちからずく（威力上昇・追加効果無効化）の対象にならない。
  `secondary_effect` フラグは付与しない。
- **おうじゃのしるし等**: ダメージを与える技であるため、おうじゃのしるし・
  するどいキバ等によるひるみ付与は通常どおり対象になる（`flinch_on_hit_10pct` の
  対象からさわぐを除外する特別な処理はない）。
- PP消費は最初の使用時の1回のみ（さわぐ状態継続中の強制行動ではPPを消費しない）。
