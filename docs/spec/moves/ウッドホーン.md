# ウッドホーン（Horn Leech）仕様書

参照: docs/wiki/moves/ウッドホーン.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 75
* 命中: 100
* PP: 12（チャンピオンズ準拠。`docs/champions/move_list.txt` の値。第九世代本家では10）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 相手に与えたダメージの1/2（50%）だけ、自分のHPを回復する（ドレイン技）。
* 追加効果なし。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。第三〜第四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 技の仕様（ドレイン技共通）

* 回復量は「相手に与えたダメージ × 1/2」で算出する。
* 端数処理: 第四世代までは切り捨て、**第五世代以降は四捨五入**する
  （公式Wiki「技の仕様」節: 「回復量を算出する際、小数点以下は第四世代までは切り捨て、
  第五世代以降は四捨五入する。」）。
  * heal_ratio=0.5 の場合、ダメージが整数であれば端数は必ず0か0.5となる。
    四捨五入では0.5はちょうど切り上げられるため、ダメージが奇数のとき
    単純な切り捨て（`int()`）より回復量が1多くなる（例: 与ダメ101 → 回復51）。
  * これは `むねんのつるぎ`（専用に「切り上げ」と明記されている技）と同じ結果になる
    （heal_ratio=0.5 では四捨五入と切り上げが常に一致するため）。
* 道具『おおきなねっこ』を持たせたポケモンが使う場合、回復量は通常の1.3倍になる。
* 特性『ヘドロえき』のポケモンを攻撃した場合、回復効果が同量の反動ダメージに変わる。
* みがわりに命中した場合は、みがわりへのダメージ分を吸収する。
* 『かいふくふうじ』状態のポケモンは回復できない。
