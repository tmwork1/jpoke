# バーンアクセル（Blazing Torque）仕様書

参照: docs/wiki/moves/バーンアクセル.html（ポケモンWiki）

## 基本情報

* タイプ: ほのお
* 分類: 物理
* 威力: 80
* 命中: 100
* PP: 10（本家）→ 12（Champions、docs/champions/changes_from_sv.md によりPP8/12/16/20の4段階に統一。
  本技は `src/jpoke/data/ps-champ-ja/moves.json` に未収録のため、本家PP10の技（10まんばりき・
  マジカルシャイン・オーラウイング等）が12に変換されている前例と同様の変換規則を適用する）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない。物理技だが接触判定は発生しない特殊な技）

## 効果

* 追加効果として、30%の確率で相手を「やけど」状態にする。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: あり（30%・相手をやけど状態にする）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降、追加効果の確率にひるみ判定を上乗せ可能）
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: 反射可能（技の仕様欄に制限記載なし）

## 技の仕様（一次情報より）

* この技に対する **アンコール・ものまね・まねっこ・さいはい は失敗する**
  （スターモービル専用の「アクセル技」共通仕様。他に該当する技: ダークアクセル・
  ポイズンアクセル・ファイトアクセル・マジカルアクセル）。
  * 本プロジェクトでは `MoveFlag` に `non_encore`（アンコール失敗）・`non_copycat`
    （まねっこ失敗）が既存実装されているため、この2つを付与する。
  * `ものまね`（Mimic）・`さいはい`（Instruct）は本プロジェクトで未実装
    （`handlers={}` のまま実装保留）のため、対応する制限フラグは存在しない。
    将来これらの技を実装する際は、本技を含む「アクセル技」ファミリーがコピー・
    再使用対象から除外されることを踏まえる必要がある。
* ゆびをふる（Metronome）では選ばれない。ただし `ゆびをふる` 自体が本プロジェクトで
  未実装（`handlers={}`、TODO）のため、現時点では対応不要。

## 備考

* メロコが使用するブロロローム（シェダル・スターモービル）専用技。
  本プロジェクトはポケモン種族・専用技制限を対象としないため、習得制限は対象外。
* 同じ「アクセル技」ファミリー（ダークアクセル・ポイズンアクセル・ファイトアクセル・
  マジカルアクセル）は既存実装済みの `non_copycat` フラグ（「スターモービル専用アクセル技」
  という趣旨のコメント付き）を共有しているが、本レビューの対象はバーンアクセルのみのため
  他技の仕様の当否は別レビューの範囲とする。
