# コールドフレア（Ice Burn）仕様書

参照: docs/wiki/moves/コールドフレア.html（ポケモンWiki）

## 基本情報

* タイプ: こおり
* 分類: 特殊
* 威力: 140
* 命中: 90
* PP: 5
  * `docs/champions/move_list.txt` に本技の記載がないため、本家シリーズ（Wiki）の値をそのまま採用する
    （`ねんりき` 等の前例に準拠。champions側のPP正規化(8/12/16/20)は対象外の技として扱う）。
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 1ターン目に溜め、2ターン目で攻撃する（溜め技）。
  * 溜めている間は姿を隠さない（あなをほる・そらをとぶ等の「隠れる技」とは異なり、回避判定は発生しない）。
  * `ソーラービーム` ・ `メテオビーム` ・ `エレクトロビーム` と同系統の「その場で溜める」技グループに属する。
* 追加効果として、30%の確率で相手をやけど状態にする。

## 判定フラグ

* 急所: 攻撃時のみ判定
* 命中判定: 攻撃時のみ
* 追加効果: 攻撃時のみ
* まもる: 防がれる（○）
* おうじゃのしるし: 第五世代以降 発動する（○）
* マジックコート: 反射不可（×、変化技ではないため対象外）
* よこどり: 対象外（×、変化技ではないため）
* オウムがえし: 反射可能（○）

## 備考

* Zワザ威力: 200（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 140（本プロジェクトでは対象外）。
* 溜めている間（1ターン目）はパワフルハーブで溜めを省略できる（他の溜め技と共通の汎用処理）。
* キュレム（ホワイトキュレム）の専用技。
