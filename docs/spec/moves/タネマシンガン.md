# タネマシンガン（Bullet Seed）仕様書

参照: docs/wiki/moves/タネマシンガン.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 25（第四世代までは10。第五世代以降25。`src/jpoke/data/ps-champ-ja/moves.json` も25で一致）
* 命中: 100
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。本家シリーズのPPは30だが、
  Championsでは全ての技のPPが8・12・16・20の4段階に統一されており、
  `docs/champions/changes_from_sv.md` の記載どおり本技は20になる）
* 対象: 相手1体
* 直接攻撃: なし（弾のわざ）
* フラグ: `bullet`（特性「ぼうだん」を持つポケモンに対して無効化される）

## 効果

* 1ターンに2～5回連続で攻撃する連続攻撃技。
* 追加効果なし。

## 連続攻撃技としての仕様

* 汎用の「2-5回攻撃技」区分に該当し、命中判定は初回のみ、急所判定は各ヒットごとに行う。
* 詳細な共通仕様（ヒット回数分布、テクニシャン/スキルリンクとの相互作用、
  みがわり・ひんし判定など）は `docs/spec/moves/_multi_hit.md` の
  「9.3 2-5回攻撃技」区分を参照（本技もその一覧に掲載済み）。

## 判定フラグ

* 急所: 各ヒットごとに独立判定
* 命中判定: 初回のみ（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 対象外（ひるみ効果を持たないため。一次情報の判定表にも記載なし）
* 弾のわざ: 特性「ぼうだん」により無効化される

## イベント / 優先度

* 追加効果を持たない純粋なダメージ技のため、専用ハンドラは不要。
* `src/jpoke/data/moves/move_ta.py` の `multi_hit` 設定（`min=2, max=5,
  check_hit_each_time=False, power_sequence=()`）により連続攻撃の共通ロジック
  （`core/move_executor.py`）で処理される。

## リーサル計算

* `docs/progress/move.md` の該当行は「リーサル実装: n/a」「リーサルテスト: n/a」であり、
  本レビューの対象範囲外（対象外）とする。
