# いかりのこな（Rage Powder）仕様書

参照: docs/wiki/moves/いかりのこな.html（ポケモンWiki）
参照: src/jpoke/data/ps-champ-ja/moves.json L66-67

## 基本情報

* タイプ: むし
* 分類: 変化
* 威力: -
* 命中: - （必中、命中判定なし）
* PP: 20
* 対象: 自分
* 優先度: +2（第六世代以降。第五世代のみ+3）
* 接触: なし（直接攻撃×）
* 粉・胞子のわざ（第六世代以降からラベル付与）

## 効果（本来のダブル/トリプルバトルでの効果）

* 使用すると自分が「ちゅうもくのまと」状態になり、そのターン中、相手が対象を選択する技を使う際は
  すべて自分に向けられる（このゆびとまれと同種の効果）。
* 全体技・場を対象とする技など、対象を選択しない技には効果がない。
* 第六世代以降、以下のいずれかを満たすポケモンの行動は、いかりのこなによる「ちゅうもくのまと」状態を無視する
  （粉技として扱われるため）。
  * くさタイプ
  * 特性「ぼうじん」
  * 道具「ぼうじんゴーグル」
  * いかりのこな自体は自分を対象とする技のため、使用者自身がくさタイプ・ぼうじん・ぼうじんゴーグルを
    持っていても使用は成功し、「ちゅうもくのまと」状態にはなれる（無効化されるのはあくまで
    「相手の行動がこの状態を無視できるかどうか」の判定）。
  * 上記の無効化判定は、いかりのこな使用時ではなく、無効化する側のポケモンの行動開始時に行われる
    （使用後に条件を得失した場合は都度再判定される）。
* テラレイドバトルで使用すると失敗する。
* 判定: 命中判定 × / 追加効果 × / まもる × / おうじゃのしるし（第五世代以降）× /
  マジックコート × / よこどり × / オウムがえし ×
* 第五世代ではこのゆびとまれと全く同じ効果だったが、第六世代から粉技に分類され、
  くさタイプ等による無効化手段が追加された。

## シングルバトルでの扱い（本プロジェクトのスコープ）

* 本プロジェクトの対象はポケモンチャンピオンズのシングルバトルのみであり、`Battle` は常に
  各プレイヤー1体ずつの場（`battle.actives` は常に2体以下）しか扱わない。
* 「ちゅうもくのまと」による対象リダイレクトは、相手に2体以上の選択肢がある場合にのみ意味を持つ
  ダブル/トリプルバトル専有の効果であり、本プロジェクトの対象範囲外
  （`docs/progress/move.md` で本技は全列「ダブル専用」＝対象外として扱う）。
* くさタイプ・特性ぼうじん・道具ぼうじんゴーグルによる無効化判定も、上記のリダイレクト機構自体が
  未実装のダブル専用機能であるため、同様に実装対象外。
  * 注意: 本技自身に `powder` フラグ（粉技の汎用無効化ラベル）を付与してはならない。
    このコードベースの `powder` 判定（`ぼうじん`・`ぼうじんゴーグル`・くさタイプ免疫）は
    技の `ctx.defender`（＝技を受ける側）に対して働く汎用ルールだが、いかりのこなの
    `ctx.defender` は本プロジェクトの実行モデル上「使用者の相手」に固定される。
    いかりのこなは自分を対象とする技であり、無効化対象は「いかりのこなを使った側」ではなく
    「いかりのこなで発生したちゅうもくのまと状態を無視して行動する相手側」であるため、
    `powder` フラグを付けると意味の異なる（相手がくさタイプなら本技自体が無効化されるという誤った）
    判定になってしまう。

## 実装方針

* 通常のダメージ計算を伴わない変化技として、追加効果なし（`handlers={}`）で実装する。
* `ちゅうもくのまと` 状態の付与・くさタイプ等による無効化はダブル専用機能のため未実装のままとする。
