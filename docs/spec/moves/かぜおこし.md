# かぜおこし（Gust）仕様書

参照: docs/wiki/moves/かぜおこし.html（ポケモンWiki）

## 基本情報

* タイプ: ひこう
* 分類: 特殊
* 威力: 40
* 命中: 100
* PP: 35
* 対象: 相手1体（遠隔）
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: 風技（wind）

## 効果

* 通常攻撃。追加効果なし。
* 第二世代以降、`そらをとぶ` で滞空中の相手にも命中し、その場合威力が2倍になる
  （`そらをとぶ` volatile 側の共通処理 `そらをとぶ_boost_power` / `そらをとぶ_can_hit_hidden_target`
  で実装済み。本技固有のハンドラは不要）。
* 風技（wind フラグ）であるため、`ふうりょくでんき`（じゅうでん付与）などの
  風技依存の特性・効果の対象になる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
