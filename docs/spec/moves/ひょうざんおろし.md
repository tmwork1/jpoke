# ひょうざんおろし（Mountain Gale）仕様書

## 基本情報

* タイプ: こおり
* 分類: 物理
* 威力: 100（SVまで）→ 120（Pokémon Champions）
* 命中: 85
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 直接攻撃（接触）: なし
* 専用技: クレベース（ヒスイのすがた）の専用技（本プロジェクトの対象範囲では専用技かどうかは
  習得可否のみに影響し、技自体の効果には影響しない）

一次情報: `docs/wiki/moves/ひょうざんおろし.html`。威力の世代差は同ページ冒頭のインフォボックスに
「100 (SV) → 120 (Champions)」と明記されている。本プロジェクトは Champions を優先するため、
実装上の威力は 120 とする（README.md の対象範囲定義に基づく）。

## 効果

* 追加効果として、30%の確率で相手をひるみ状態にする。

## 判定フラグ

* 急所: あり
* 命中判定: あり
* 追加効果: あり
* まもる: 防がれる
* おうじゃのしるし: 発動しない（第五世代以降、本来ひるみを追加効果に持つ技のため対象外）
* マジックコート: 反射されない

## 備考

* `docs/spec/moves/_move_secondary_1.md` の一覧表（183行目）にも「ひるみ 30%」の追加効果技として
  記載されている。
* 直接攻撃を伴わない物理技であるため、`かたきうち`・`ゴツゴツメット` 等の接触依存効果は発動しない。
