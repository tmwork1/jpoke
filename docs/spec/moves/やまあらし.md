# やまあらし（Storm Throw）仕様書

参照: docs/wiki/moves/やまあらし.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 40（第五世代）→ 60（第六世代以降。本プロジェクトは第六世代以降準拠）
* 命中: 100
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 必ず急所に当たる（急所ランク3扱い）。
* 追加効果なし。

## 判定フラグ

* 急所: 必ず急所（急所ランク3）
  * シェルアーマー/カブトアーマーの相手には急所に当てられない（一般ハンドラで処理、かたやぶりで無視）
  * おまじない状態の相手には急所に当てられない（一般ハンドラで処理。本プロジェクトでは未実装のため対象外）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* SV（スカーレット・バイオレット）では技マシン未収録のため使用不可と一次情報に記載があるが、
  `src/jpoke/data/ps-champ-ja/moves.json` には登録されており、本プロジェクト（ポケモンチャンピオンズ
  シングルバトル）の対象範囲では使用可能な技として扱う。
* 第六世代で急所倍率が1.5倍に下がった代わりに本技の威力が40→60に上がっており、
  急所込みでの実質威力（90相当）はほぼ変わらない。
* Zワザ威力: 120（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 80（本プロジェクトでは対象外）。
* 急所ランク3による確定急所判定・シェルアーマー等との相互作用は、既存の急所判定共通ロジック
  （`critical_rank` を参照する汎用処理）で扱われるため、本技固有のハンドラは不要。
