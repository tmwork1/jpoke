# じしん（Earthquake）仕様書

参照: docs/wiki/moves/じしん.html（ポケモンWiki）

## 基本情報

* タイプ: じめん
* 分類: 物理
* 威力: 100
* 命中: 100
* PP: 10
* 対象: 自分以外の全員（本プロジェクトはシングルバトル専用のため、実質「相手1体」として扱う）
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果なし。
* 『あなをほる』状態の相手にも命中し、ダメージが2倍になる（`あなをほる_boost_power` で実装済み。
  `handlers/volatile.py` の `HIDDEN_MOVE_ALLOWED_MOVES["あなをほる"]` にも `じしん` が登録されている）。
* 『グラスフィールド』の影響を受けている（接地している）相手に対しては、威力が半減する
  （`グラスフィールド_power_modifier` で実装済み）。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第三世代以降。第二世代のみ×）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ威力: 180（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
* 『あなをほる』状態の相手への威力2倍・『グラスフィールド』での威力半減は、じしん固有のハンドラでは
  なく `あなをほる` 状態側・場の状態側のハンドラで処理されるため、`data/move.py` の `じしん` エントリ
  自体には追加効果のハンドラを登録しない。
