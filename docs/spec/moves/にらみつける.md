# にらみつける（Leer）仕様書

参照: docs/wiki/moves/にらみつける.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中率: 100
* PP: 30（`docs/champions/move_list.txt` に単独項目が存在しないため、`なきごえ`・`ちょうおんぱ`
  等の前例に倣い Gen9本家準拠のPPを採用する。現行実装と一致）
* 対象: 相手全員（シングルバトルのため相手1体と等価）
* 優先度: 0
* 直接攻撃: ×

## 効果

* 相手の『ぼうぎょ』ランクを1段階下げる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: あり（`accuracy=100`）
* 追加効果: なし（本効果自体が確定効果）
* まもる: 防がれる（`unprotectable` フラグなし。一般の相手対象変化技と同様）
* みがわり: 貫通しない（音技ではないため、一般ルールに従いみがわりの相手には命中しない）
* マジックコート: 反射される
* よこどり: 対象外（相手に向けた効果のため奪えない。未実装）
* オウムがえし（ミラームーブ）: コピーされる
* おうじゃのしるし: 対象外（ひるみ関連の効果ではないため無関係）

## ランク低下無効化との相互作用（フレームワーク側で処理、実装済みの汎用ロジックに委譲）

* 特性「クリアボディ」「メタルプロテクト」、道具「しろいけむり」、
  特性「フラワーベール」やフィールド効果「しろいきり」を持つ相手にはぼうぎょのランク低下が無効化される。
* 特性「まけんき」「かちき」を持つ相手に使うと、ランク低下発動時にそれぞれの特性が発動する。

## 一次情報との差異（本プロジェクトのスコープ外）

* Zワザによる付加効果（自分のこうげきを1段階上げる）は本プロジェクトでは非対応
* ポケモン不思議のダンジョンでの通路制限は対象外
* コンテスト効果（アピール等）は実装しない

## 備考

* 同じ効果（相手のぼうぎょを1段階下げる変化技）を持つ技に「なかよくする」「わるだくみ返し的な
  性質」等はなく、`docs/wiki` 上は「にらみつけると同じ効果のわざ」として複数挙げられているが、
  本プロジェクトで実装済みの近縁技としては `Gのちから`（追加効果として同ランク低下を付与）が挙げられる。
