# おどろかす（Astonish）仕様書

参照: docs/wiki/moves/おどろかす.html（ポケモンWiki）

## 基本情報

* タイプ: ゴースト
* 分類: 物理
* 威力: 30
* 命中: 100
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、30%の確率で対象をひるませる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（30%でひるみ）
* まもる: 防がれる
* おうじゃのしるし: 第三世代〜第四世代: ×、第五世代以降: ×（本技の追加効果はひるみだが、おうじゃのしるしでは発動しない仕様）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* 第三世代のみ、ちいさくなる状態の相手に使用した場合ダメージが2倍になる（第四世代以降は適用されない、過去世代限定の仕様のため本プロジェクトでは対象外）。
* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
