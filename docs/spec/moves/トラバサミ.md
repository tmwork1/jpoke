# トラバサミ（Snap Trap）仕様書

参照: docs/wiki/moves/トラバサミ.html（ポケモンWiki）

## 基本情報

* タイプ: はがね（第八世代はくさタイプだったが、Championsではがねタイプに変更された）
* 分類: 物理
* 威力: 35
* 命中: 100
* PP: 16（`docs/champions/move_list.txt` の値を採用）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* マッギョ（ガラルのすがた）の専用技

## 効果

* 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける。
  * バインド状態の詳細仕様（継続ターン、しめつけバンドによる強化、マジックガードでの無効化、交代禁止とその例外、解除条件など）は `docs/spec/volatiles/バインド.md` を参照する。
* 命中率100%のバインド技であるため、ゴースト以外の相手を確実に拘束できる（ゴーストタイプはバインド状態でも交代可能）。
* タイプがはがねに変更されたため、そうしょく（くさタイプの技を無効化する特性）による無効化は発生しない
  （第八世代まではくさタイプだったためそうしょくで無効化されていた）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし（バインド付与自体が本体効果であり追加効果としての判定はない）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
* しめつける・まきつく・ほのおのうず・からではさむ・うずしお・すなじごく・まとわりつく・サンダープリズン・
  マグマストームと同系統の「バインド」技グループに属する。
* 直接攻撃技のため、ゆびをふれる等の接触判定が発生する。
