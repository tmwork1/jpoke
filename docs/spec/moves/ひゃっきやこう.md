# ひゃっきやこう（Infernal Parade）仕様書

参照: docs/wiki/moves/ひゃっきやこう.html（ポケモンWiki）

## 基本情報

* タイプ: ゴースト
* 分類: 特殊
* 威力: 60（SV）→ 65（Pokémon Champions）。本プロジェクトの対象はチャンピオンズのため65を採用する。
* 命中: 100
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、30%の確率で相手をやけど状態にする。
* 攻撃対象が状態異常（どく・もうどく・まひ・やけど・ねむり・こおり）のとき、威力が2倍になる。
  * 威力が2倍になる条件は「たたりめ」と同様（`docs/spec/moves/たたりめ.md` を参照）。
  * 特性『ぜったいねむり』を持つ相手（ネッコアラ）は常に状態異常『ゆめうつつ』を抱えているため、
    命中すれば常に威力2倍となる。
* 追加効果（やけど付与）の判定はダメージ計算の後に行う。この技で相手をやけど状態にした
  ターンのダメージ自体は威力2倍になっていない（やけど付与前の状態異常有無で威力を判定する）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（30%でやけど）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）

## 備考

* バクフーン（ヒスイのすがた）の専用技。
* やけど効果を持つ技のうち、初めて登場したゴースト技。
