# コットンガード（Cotton Guard）仕様書

参照: docs/wiki/moves/コットンガード.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 変化
* 威力: -
* 命中: -
* PP: 12（champions 基準。src/jpoke/data/ps-champ-ja/moves.json 参照。本家PPは10だが、
  champions は全技PPを8/12/16/20の4段階に統一しており、10は12に繰り上げられる）
* 対象: 自分
* 優先度: 0
* 直接攻撃: いいえ

## 効果

* 自分の『ぼうぎょ』ランクを3段階上げる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化がそのまま効果）
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* おうじゃのしるし: 対象外
* よこどり: 対象になる（○）
* オウムがえし: 反射不可（変化技のため対象外）
