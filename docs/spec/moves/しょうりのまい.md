# しょうりのまい（Victory Dance）仕様書

参照: docs/wiki/moves/しょうりのまい.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 変化
* 威力: -
* 命中: -
* PP: 10（src/jpoke/data/ps-champ-ja/moves.json に記載なし。同技は専用技でありチャンピオンズで
  習得可能なポケモンが存在しないため未掲載と推定される。第9世代本家基準の値10を採用）
* 対象: 自分
* 優先度: 0
* 直接攻撃: いいえ
* 踊り技（`dance` ラベル）

## 効果

* 自分の『こうげき』『ぼうぎょ』『すばやさ』ランクを1段階ずつ上げる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化がそのまま効果）
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* よこどり: 対象外（自分を対象とするため奪えない）
* オウムがえし: 反射不可（相手を対象としないため対象外）
