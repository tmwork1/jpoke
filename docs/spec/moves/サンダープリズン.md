# サンダープリズン（Thunder Cage）仕様書

参照: docs/wiki/moves/サンダープリズン.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 80
* 命中: 90
* PP: 15（`docs/champions/move_list.txt` に本技の記載はない。レジエレキ専用技であり、
  同ファイルは記載漏れと判断し、第八/第九世代の原作値である15をそのまま採用する）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* レジエレキの専用技

## 効果

* 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける。
  * バインド状態の詳細仕様（継続ターン、しめつけバンドによる強化、マジックガードでの無効化、交代禁止とその例外、解除条件など）は `docs/spec/volatiles/バインド.md` を参照する。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率90）
* 追加効果: なし（バインド付与自体が本体効果であり追加効果としての判定はない）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
* うずしお・しめつける・すなじごく・まきつく・ほのおのうず・からではさむ・まとわりつく・トラバサミ・マグマストームと同系統の「バインド」技グループに属する。
* バインドにする技の中ではマグマストーム（威力100）の次に威力が高い。
* 直接攻撃ではないため、ゆびをふれる等の接触判定は発生しない。
