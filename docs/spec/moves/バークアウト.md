# バークアウト（Snarl）仕様書

参照: docs/wiki/moves/バークアウト.html（ポケモンWiki）

## 基本情報

* タイプ: あく
* 分類: 特殊
* 威力: 55
* 命中: 95
* PP: 16（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。Gen9本家は15だが、チャンピオンズでは全技PPが
  8/12/16/20の4段階に統一されているため16が正）
* 対象: 相手1体（原作では相手全員が対象の技だが、本プロジェクトはシングルバトルのみを扱うため
  `defender` 単体として扱う）
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 音技: あり（`sound` フラグ。みがわり状態を貫通して本体にダメージを与える）

## 効果

* 追加効果として、100%の確率で相手の『とくこう』ランクを1段階下げる。

## 判定フラグ

* 急所: あり
* 命中判定: あり
* 追加効果: あり（100%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
