# ジェットパンチ（Jet Punch）仕様書

参照: docs/wiki/moves/ジェットパンチ.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 60
* 命中: 100
* PP: 16（ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されており、
  `src/jpoke/data/ps-champ-ja/moves.json:360` 記載の値を採用する。Wiki記載の15はGen9本家の値）
* 対象: 相手1体選択
* 優先度: +1
* 接触: あり（直接攻撃）
* フラグ: パンチ技

## 効果

* 先制攻撃技（優先度+1）。追加効果なし。
* パンチ技のため、特性「てつのこぶし」所持時は威力が1.2倍になる。
* 同じ優先度の技を他のポケモンも使った場合はすばやさに則って先攻が決定される
  （通常の優先度処理と同じ）。

## 判定フラグ

* 急所: あり（急所ランク0）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: 反射可能
* ゆびをふる: 選ばれない（先制攻撃技のため）

## 備考

* イルカマンの専用技。
* SV Ver.1.3.2以前の本家データ上では「相手の能力を下げる技」に分類され特性「ちからずく」の
  対象になっていたが、Ver.2.0.1以降は通常攻撃技として扱われる（本プロジェクトは対象外の
  過去挙動のため実装不要）。
