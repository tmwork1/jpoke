# とどめばり（Fell Stinger）仕様書

参照: docs/wiki/moves/とどめばり.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 物理
* 威力: 50（第七世代以降。第六世代のみ30）
* 命中: 100
* PP: 25
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* この技で相手を倒すと、自分の『こうげき』ランクが3段階上がる（第六世代のみ2段階）。
* 相手を倒せなかった場合は何も起きない。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: **なし**（攻撃ランク上昇は「追加効果」として扱われない）
* まもる: 防がれる
* おうじゃのしるし/するどいキバ: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 「追加効果」扱いではないことの影響

攻撃ランク上昇は、ゲーム内仕様上「追加効果」に分類されない。そのため、
使用者の特性ちからずくは発動しない（威力1.3倍にならない）が、効果自体は発動する。

## 相互作用・エッジケース

* 相手を倒すと自分のランクが上がる特性（じしんかじょう、ビーストブースト等）を持つ
  ポケモンがとどめばりを使用して相手を倒した場合、とどめばりの効果でこうげきが
  上がった直後にその特性が発動する（とどめばりの効果が先）。
* 特性がたんじゅん/あまのじゃくのポケモンが使用して、相手を倒すと同時に相手の
  ミイラ/さまようたましい/とれないにおいが発動して自分の特性が変化した場合、
  特性が変更された後にとどめばりの効果が発動する。そのため変更前の特性
  （たんじゅん/あまのじゃく）による補正は受けない。
  * 相手がかがくへんかガスだった場合、相手を倒してガスの効果が切れた（特性が
    元に戻った）後にとどめばりの効果が発動する。
* 第八世代以降、特性ばけのかわの相手に使用し、ばけのかわが剥がれるときの
  フォルムチェンジ消費ダメージ（最大HPの1/8）で相手を倒した場合も、
  とどめばりの効果は発動してこうげきが上がる
  （このときの技自体のダメージは0だが、効果は発動する）。
