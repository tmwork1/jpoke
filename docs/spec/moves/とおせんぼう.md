# とおせんぼう（Block）仕様書

参照: docs/wiki/moves/とおせんぼう.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: 必中（`docs/champions/move_list.txt` では命中率欄が空。命中判定自体を行わない）
* PP: 8（`docs/champions/move_list.txt` / `docs/progress/move.md` 準拠。本家シリーズのPPは5だが、Championsでは8）
* 対象: 相手1体
* 優先度: 0
* 接触: なし
* まもる: 防がれない（第六世代以降。`docs/champions/move_list.txt` の「守×」に対応）

## 効果

* 相手を『にげられない』状態にし、交代・逃げることができなくする。
* ゴーストタイプのポケモンは『にげられない』状態の効果を無視できる
  （`にげられない` 揮発性状態自体の仕様。docs/spec/volatiles/にげられない.md 参照）。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: 必中（`accuracy=None`。ランク補正・回避率に関係なく必ず命中する）
* 追加効果: なし
* まもる: 防がれない（第六世代以降。第三〜第五世代では防がれたが、本プロジェクトの対象範囲では無効）
* みがわり: 防がれる（`bypass_substitute` フラグなし。第三世代以降の一般ルールにより自動的にブロックされる。
  ただし、にげられない状態になった後にみがわりを張っても逃げられるようにはならない）
* おうじゃのしるし: 対象外（ひるみ効果を持たないため無関係）
* マジックコート: 反射可能（変化技かつ対象が相手1体のため、category=status/target=foe の一般ルールで自動的に反射対象になる）
* よこどり: 対象外（未実装のため本仕様書では対象外）
* オウムがえし: コピー可能

## 一次情報との差異（本プロジェクトのスコープ外）

* トリックガード状態により防がれる（本プロジェクト未実装のため対象外）
* ダイウォールにより防がれる（本プロジェクト未実装のため対象外。同じ効果のくろいまなざしは
  ダイウォールに防がれない点で異なる）
* 必中技だが、姿を隠している相手（穴を掘っている・宙に浮いている等の溜め技の隠れ状態）には
  当たらない（本プロジェクトの命中判定フレームワークの一般仕様に準拠）

## イベント / 優先度

* `Event.ON_STATUS_HIT`（優先度100・デフォルト）で相手に『にげられない』状態を付与する。
* ゴーストタイプへの無効化はフレームワーク側（`docs/spec/volatiles/にげられない.md` の
  `Event.ON_CHECK_TRAPPED` 経由の `check_trapped_not_ghost`）が処理するため、本技のハンドラでの
  個別対応は不要。
* すでに『にげられない』状態の場合の失敗は `battle.volatile_manager.apply()` の重複付与チェック
  （既に同じ揮発性状態があれば失敗）により自動的に処理される。

## 備考

* 「にげられない」状態の詳細な仕様（消滅条件・ゴーストタイプの扱いなど）は
  `docs/spec/volatiles/にげられない.md` を参照。
* くろいまなざしと同じ『にげられない』状態を使用する（付与対象が異なる: くろいまなざしは
  にらんで発動し、とおせんぼうは立ちはだかって発動するが、効果自体は同一）。
* クモのすとも同じ『にげられない』状態を使用する。
