# ヘドロウェーブ（Sludge Wave）仕様書

参照: docs/wiki/moves/ヘドロウェーブ.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 特殊
* 威力: 95
* 命中: 100
* PP: 12（Champions基準。docs/champions/move_list.txt により全技PPは8/12/16/20に統一されている。
  Gen9本家基準ではPP10）
* 対象: 自分以外全体（本プロジェクトはシングル戦のみのため、実質的に相手1体と同義）
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、10%の確率で相手を『どく』状態にする。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（10%・どく）
* まもる: 防がれる
* おうじゃのしるし: 対象外（本技はもともと状態異常の追加効果を持つため、おうじゃのしるし
  によるひるみ追加効果とは無関係。本技自体の追加効果発動率には影響しない）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・
  どくの追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）
* どく・はがねタイプの相手 / めんえき等の特性: どく付与は無効（`ailment_manager.apply` 内部で判定）

## Zワザ・ダイマックスわざ

* Zワザ威力: 175（本プロジェクトの対象範囲外のため実装しない）
* ダイマックスわざ威力: 90（本プロジェクトの対象範囲外のため実装しない）
