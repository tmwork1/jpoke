# ふぶき（Blizzard）仕様書

参照: docs/wiki/moves/ふぶき.html（ポケモンWiki）
参照: src/jpoke/data/ps-champ-ja/moves.json（ポケモンチャンピオンズ仕様）

## 基本情報

* タイプ: こおり
* 分類: 特殊
* 威力: 110（第六世代以降。第五世代までは120）
* 命中: 70（第二世代以降。第一世代は90）
* PP: 8
  * 一次情報（ポケモンWiki）ではPP5だが、`src/jpoke/data/ps-champ-ja/moves.json` の `ふぶき` 行がPP8であり、
    ポケモンチャンピオンズ側の値を正とする。
* 対象: 相手全員（本プロジェクトはシングルバトルのみ対象のため、相手1体への通常攻撃と同義に扱う）
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、10%の確率で相手を『こおり』状態にする
  （第一世代のみ30%。本プロジェクトの対象世代では10%）。
* 天気が『ゆき』のときは必ず命中する（命中率補正・命中判定を無視して必中になる。第四世代以降）。
* 天候の有無判定は `battle.weather_for(pokemon)` を使う
  （エアロック・ノーてんきで天候自体が無効化されている場合は必中補正の対象外になる）。
* ばんのうがさは「はれ・あめ」系天候のみを無効化する持ち物であり『ゆき』は対象外のため、
  ばんのうがさ所持者に対してもゆき天候による必中補正は通常通り適用される。

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり（天候により必中になる。上記参照）
* 追加効果: あり（10%・こおり）
* まもる: 防がれる
* おうじゃのしるし: 第二世代は対象外、第三〜四世代は対象外、第五世代以降は対象
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可（本プロジェクトはオウムがえし自体が未実装のため対応不要）

## 風の技（wind）フラグ

* 「風の技」に分類され、特性 かぜのり／ふうりょくでんき の効果対象になる
  （`flags={"wind"}` として実装、`かぜのり_absorb_wind` / `ふうりょくでんき_on_damage` が対象判定に使用）。

## リーサル計算

* 固定威力・追加効果（追加効果を除けばダメージ量に影響しない状態異常付与のみ）の通常攻撃技のため、
  `lethal.py` に専用ハンドラは不要
  （`docs/progress/move.md` のリーサル実装／リーサルテスト列は n/a のまま対象外）。
