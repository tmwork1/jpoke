# なみのり（Surf）仕様書

参照: docs/wiki/moves/なみのり.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: みず
* 分類: 特殊
* 威力: 90
* 命中: 100
* PP: 16（Champions仕様。マスターデータ準拠: すべての技のPPは8/12/16/20の4段階に統一。
  参考: 本家PPは15）
* 対象: 自分以外の全員（本プロジェクトはシングルバトル専用のため、実質「相手1体」として扱う）
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果なし。
* 『ダイビング』状態の相手にも命中し、ダメージが2倍になる（`ダイビング_boost_power` で実装済み。
  `handlers/volatile.py` の `HIDDEN_MOVE_ALLOWED_MOVES["ダイビング"]` にも `なみのり` が登録されている）。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ威力: 175（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
* 『ダイビング』状態の相手への威力2倍は、なみのり固有のハンドラではなく `ダイビング` 状態側の
  ハンドラで処理されるため、`data/move.py` の `なみのり` エントリ自体には追加効果のハンドラを
  登録しない。
