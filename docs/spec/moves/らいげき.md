# らいげき（Bolt Strike）仕様書

参照: docs/wiki/moves/らいげき.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 物理
* 威力: 130
* 命中: 85
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、20%の確率で相手を『まひ』状態にする。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率85）
* 追加効果: あり（20%・まひ）
* まもる: 防がれる
* おうじゃのしるし: 対象（第五世代以降、20%のまひ判定に加えておうじゃのしるしによる
  追加のひるみ判定が行われる。ひるみ自体はおうじゃのしるし側の汎用処理で扱われるため、
  本技側での追加実装は不要）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・
  まひの追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）

## Zワザ・ダイマックスわざ

* ダイマックスわざ威力: （本プロジェクトの対象範囲外のため実装しない）
