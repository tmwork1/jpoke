# へびにらみ 仕様書

参照: docs/wiki/moves/へびにらみ.html（ポケモンWiki）

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | ノーマル |
| 分類 | 変化 |
| PP | 20（champions基準。src/jpoke/data/ps-champ-ja/moves.json と一致） |
| 命中率 | 100（第六世代以降の値。第四世代までは75、第五世代は90） |
| 範囲 | 相手1体 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

相手を **まひ** 状態にする。

## 失敗条件

`ailment_manager.apply(defender, "まひ")` が内部で行う汎用チェックにより、以下の場合は失敗する。

- 対象がすでに **状態異常**（どく・もうどく・まひ・やけど・ねむり・こおり）を持っている
- 対象が **でんきタイプ** である（まひ無効。第六世代以降の仕様）

へびにらみ自体はノーマルタイプの変化技であり、でんじはのような「技のタイプによる
対象タイプ相性判定」は存在しない（ノーマル技はタイプ相性を持たず、変化技はそもそも
攻撃技用のタイプ相性判定 `_check_hit_by_type` の対象外）。

## 防がれる条件

以下の場合、へびにらみは防がれる（`ailment_manager.apply` 内、`ON_BEFORE_APPLY_AILMENT` 経由の
汎用判定、および `ON_BEFORE_APPLY_MOVE` 経由の特性判定）。

- じゅうなん・ぜったいねむり・きよめのしお特性（まひ無効）
- フラワーベール特性・しんぴのまもり状態・ミストフィールド状態
- みがわり状態（使用者にすりぬけがある場合は無視）
- まもる・みきり等（変化技かつ target=foe のデフォルト挙動）
- マジックコートによる跳ね返し（変化技のデフォルト挙動）

## 技の仕様（過去世代の挙動・参考情報）

- 第二世代・第三世代のみ、ゴーストタイプに対して効果がなかった（みやぶられている状態の
  ゴーストタイプには有効）。第一世代および第四世代以降ではゴーストタイプに対しても
  効果がある。ポケモンチャンピオンズは現行仕様のためゴースト無効化は実装しない。
- 第六世代以降ではでんきタイプのポケモンに対しては効果がない（上記「失敗条件」参照）。

## イベントフロー

```
ON_STATUS_HIT (priority=100, MoveHandler デフォルト)
  → ailment_manager.apply(defender, "まひ")
     - すでに状態異常 OR でんきタイプなら失敗（apply内部の汎用チェック）
     - ON_BEFORE_APPLY_AILMENT を発火し、じゅうなん等の特性・
       しんぴのまもり等の場の状態・みがわりによる無効化を判定
```

## ポケモンチャンピオンズ向けの簡略化

- Z変化技付加効果（とくぼう+1段階）は実装しない
- コンテスト効果は実装しない
- 第二世代・第三世代限定のゴーストタイプ無効化は実装しない（対象外世代）
- おうじゃのしるし等の追加効果判定は対象外（へびにらみに追加効果〈secondary_effect〉は存在しない）
