# つめとぎ 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | あく |

| PP | 15（第9世代）／16（ポケモンチャンピオンズ、下記参照） |

| 範囲 | 自分 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

自分の **こうげき** と **命中率** のランクをそれぞれ **1段階** 上げる。

## 判定

- 急所判定: なし
- 命中判定: なし（自分が対象のため必ず成功する）
- 追加効果: なし（本効果自体が確定効果）
- まもる: 効かない（自分が対象のため、相手のまもる状態の影響を受けない）
- マジックコート: 跳ね返されない（自分が対象のため）
- よこどり: 対象になる（変化技のためよこどりで奪われる）
- オウムがえし: 対象にならない（相手を対象とする技ではないため）

## PPについて

`src/jpoke/data/ps-champ-ja/moves.json` に本技のエントリが存在しない（スクレイピング漏れの可能性）。
`docs/champions/changes_from_sv.md` により全技PPは8/12/16/20の4段階に統一されている。
第9世代でPP15だった他の技（あくのはどう、シャドーボール、10まんボルト等）が
move_list.txt上でいずれもPP16に統一されていることから、同じ第9世代PP15である
つめとぎもPP16に統一されていると推測し、その値を採用する。
