# ポルターガイスト（Poltergeist）仕様書

参照: docs/wiki/moves/ポルターガイスト.html（ポケモンWiki）

## 基本情報

* タイプ: ゴースト
* 分類: 物理
* 威力: 110
* 命中: 90
* PP: 8（src/jpoke/data/ps-champ-ja/moves.json準拠。SVの原作値は5だが、チャンピオンズでは全技PPが
  8/12/16/20の4段階に統一されているため8を採用する）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 相手が「もちもの」を持っているときのみ成功する。相手が道具を持っていない場合は失敗する。
* 技を受けても相手の持ち物が無くなるわけではない（アイテムを奪う・無効化する効果はない）。
* 場が「マジックルーム」状態のときや、相手の特性が「ぶきよう」であるときでも、
  相手が持ち物を持っているなら成功する（アイテムが無効化されていても「持っている」こと自体は
  変わらないため）。
* 特性「ねんちゃく」のポケモンに対しても通常通り成功する。
* シルヴァディの「メモリ」やザシアンの「くちたけん」など、奪ったり無効化したりできない
  持ち物を持つ相手にも成功する。
* 技が当たると相手の持ち物が判明する（アイテム所持の有無が分かる）。
* カシブのみなど、効果抜群のダメージを半減するきのみを持った相手が受けた場合、
  アイテム所持チェック（技の成否判定）はダメージ計算より先に行われるため、
  技自体は成功しつつ、カシブのみによるダメージ半減が発生する（きのみは消費される）。

## 判定フラグ

* 急所: あり
* 命中判定: あり
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）

## 備考

* ゴーストタイプの攻撃技で唯一、命中率が100より小さい（90）。
* 「もちものに影響を与えるわざ」には分類されないため、マックスレイドバトル・テラレイドバトルでも
  使用できる（対象外効果のため今回の実装スコープには影響しない）。
