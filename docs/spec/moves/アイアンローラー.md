# 技仕様書: アイアンローラー（Steel Roller）

**対象世代**: 第9世代（スカーレット・バイオレット）
**バトル形式**: シングルバトル

---

# 基本データ

- タイプ: はがね
- 分類: 物理
- 威力: 130
- 命中率: 100
- PP: 8（`src/jpoke/data/ps-champ-ja/moves.json` に基づく。Gen9本家は5だが
  チャンピオンズでは全技PPが8/12/16/20に統一されているため8とする）
- 対象: 相手ポケモン1体選択
- 優先度: 0
- 直接攻撃: ○
- 専用技: スカーレット・バイオレット（Ver.2.0.1現在）ではテツノワダチの専用技
  （本プロジェクトでは移動先のポケモンを限定しないため実装上の制約はなし）

---

# 基本効果

場に何らかのフィールド（エレキフィールド・グラスフィールド・サイコフィールド・
ミストフィールド）が発生しているときのみ技が成立し、通常のダメージを与えると
同時にそのフィールドを解除する。

- 使用者・対象がフィールドの効果範囲（地面）にいなくても、フィールドを解除できる。
- 技の効果は「追加効果」とはみなされない。
  - 使用者の特性ちからずくの効果は発動せず、確率に関わらずフィールドを解除できる。
  - 対象の特性りんぷんや持ち物おんみつマントを持つ場合であっても、フィールドの
    解除は防がれない。

---

# 技成立条件

## フィールドの有無判定

- 技を実際に行動するタイミングでフィールドが発生していない場合は技が失敗する。
- 行動選択時点ではフィールドが存在していても、実際に行動するまでの間に別の効果で
  フィールドが解除されていた場合は技が失敗する。
- フィールドが発生していない状態でも、技を選択すること自体はできる（PPは消費される）。

## 命中判定

- 通常の命中率判定（100%）に従う。
- 技が外れた場合、防がれた場合（まもる等）、無効化された場合（タイプ相性・特性等）は
  フィールドを解除できない。

---

# 相互作用

## まもる

- まもる・キングシールド等で防がれる（`○`）。この場合フィールドは解除されない。

## マジックコート・マジックミラー

- 物理技であるため反射・跳ね返りの対象外（`×`）。

## みがわり

- みがわり状態の相手に技を受けられた（みがわりにダメージを肩代わりさせた）場合でも、
  フィールドは解除できる。
  - みがわりによって対象の実HPダメージが0になった場合でも、技自体は命中して
    ヒットしているため、フィールド解除効果は独立して発動する。

## 使用者がひんしになった場合

- 対象の特性てつのトゲ・ゴツゴツメット、または自分の持ち物いのちのたまの効果を
  受けて使用者のHPが0になった場合でも、ひんしになった後にフィールドを解除できる。

## 場の効果を発生させる特性・仕様との絡み

- この技で場の効果を発生させる特性（ミストメイカー等）を持つポケモンを倒し、
  味方のレシーバーで特性を引き継ぐ場合、レシーバーによる引き継ぎでフィールドが
  発生した後にこの技のフィールド解除効果が発動することで、フィールドを完全に
  解除できる。
- この技でかがくへんかガス状態のポケモンを倒したことにより、無効化されていた
  フィールド生成特性が有効になる場合も同様に、特性でフィールドが発動した後に
  技の効果が発動することで、フィールドを完全に解除できる。

## 対象の持ち物・特性による強制交代との絡み

- 対象の持ち物レッドカードが発動した場合、交代先が引きずり出された結果、
  フィールドを発生させる特性を持つポケモンが出てきたときは新たにフィールドが
  発生する。
- 対象の特性にげごし・ききかいひが発動した場合も同様に、交代先が繰り出された
  結果としてフィールドを発生させる特性を持つポケモンが出てきたときは新たに
  フィールドが発生する。
- 対象の特性カーリーヘアー・ぬめぬめ・わたげによりだっしゅつパックが発動する
  場合も同様。

## そうでん状態

- そうでん状態でタイプがでんきタイプに変化した場合、フィールドがエレキフィールドで
  あれば技の威力が上昇した状態で使用され、その後にエレキフィールドを解除できる。

# 世代別仕様差分（第9世代で有効なもののみ）

- SwSh時代のワイルドエリアで天候由来（豪雨・霧）によりフィールドが自然発生している
  場合の特例挙動（技は常に成功するがフィールドを解除できない）は、本プロジェクトの
  対象（チャンピオンズ・シングルバトル）に存在しない仕組みのため対象外。
- ダイマックスわざ威力: 140（本プロジェクトでは対象外）。
