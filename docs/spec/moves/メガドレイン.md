# メガドレイン（Mega Drain）仕様書

参照: docs/wiki/moves/メガドレイン.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 40
* 命中: 100
* PP: 15（第四世代以降。第三世代までは10）。
  `docs/champions/move_list.txt` に記載がないため、Gen9本家のPP（15）をそのまま採用する。
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 相手に与えたダメージの1/2（50%）だけ、自分のHPを回復する（ドレイン技）。
* 追加効果なし。
* 威力とPPを除けばギガドレインと同じ効果を持つ（docs/wiki/moves/メガドレイン.html「技の仕様」参照）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。第三〜第四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 技の仕様（ドレイン技共通）

* 回復量は「相手に与えたダメージ × 1/2」で算出する。
  小数点以下は第四世代までは切り捨て、第五世代以降は四捨五入する
  （heal_ratio=0.5 の場合、ダメージは整数のため端数は必ず 0 か 0.5 となり、
  切り捨てと四捨五入の結果は一致する）。
* 道具『おおきなねっこ』を持たせたポケモンが使う場合、回復量は通常の1.3倍になる。
* 特性『ヘドロえき』のポケモンを攻撃した場合、回復効果が同量の反動ダメージに変わる。
* みがわりに命中した場合は、みがわりへのダメージ分を吸収する。
