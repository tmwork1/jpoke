# ドラゴンエナジー（Dragon Energy）仕様書

参照: docs/wiki/moves/ドラゴンエナジー.html（ポケモンWiki）
参照: docs/spec/moves/_variable_power.md（変動威力技共通仕様、第3節）

## 基本情報

* タイプ: ドラゴン
* 分類: 特殊
* 威力: 可変（1〜150）
* 命中: 100
* PP: 5（Gen9本家）→ Champions: 8（PPは8/12/16/20の4段階に統一。base PP5の技は
  でんじほう・はかいこうせん・じばく・だいばくはつ・アストラルビット等と同様に8へ変換される。
  `src/jpoke/data/ps-champ-ja/moves.json` にはレジドラゴが選出対象外のため本技の記載自体がないが、
  威力計算式が全く同一の「しおふき」「ふんか」（いずれも base PP5→champions PP8として
  `move_list.txt` に記載あり）と同じ変換規則を適用する）
* 対象: 相手全員（本プロジェクトはシングルバトルのみ対象のため、対象は相手1体として扱う）
* 優先度: 0
* 接触: なし
* 専用技: レジドラゴ専用

## 効果

* 自分（使用者）の残りHPが多いほど、威力が大きくなる。
* 威力は以下の式で算出する（小数点以下切り捨て）。ただし、最小値は1。

  ```
  威力 = floor(150 × 残りHP ÷ 最大HP)
  ```

* しおふき・ふんかと全く同じ威力計算式を使用する（タイプ違いの同効果技）。

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）

## リーサル計算

* 威力が自分自身の残りHPに依存するが、ダメージ計算対象の変化ではなく攻撃力側の可変要素であるため、
  `lethal.py` の想定する「相手HPの分布」（`StateDist`）には直接影響しない。
  リーサル計算は使用時点の実HPで威力を確定させる通常のダメージ計算ロジックで扱われるため、
  専用のリーサルハンドラは不要（`docs/progress/move.md` のリーサル実装/リーサルテスト列は n/a）。
