# 仕様書: グラスミキサー

## 対象範囲外（実装しない）

本技（英名 Leaf Tornado）は第5世代で登場し、第8世代（ソード・シールド／BDSP）までは使用可能だったが、
**ポケットモンスター スカーレット・バイオレットでは使用不可**（`docs/wiki/moves/グラスミキサー.html` の
インフォボックスに赤字で「SVで使用不可」と明記、カテゴリにも「スカーレット・バイオレットで使用不可能なわざ」
とある）。`docs/champions/move_list.txt`（ポケモンチャンピオンズの技一覧）にも本技の項目は存在せず、
ポケモンチャンピオンズ側でも採用されていないことが確認できる（WebSearch でも「Leaf Tornado is not a
usable move in Pokémon Scarlet and Violet」と確認済み）。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に
該当するため、本技は実装対象外とする。

過去のレビュー（一括実装計画 `docs/plan/moves/攻撃技_防御側ランク変化系.md`）にて、対象範囲の確認が
行われないまま `data/moves/move_ka.py` に本技の `MoveData`（威力60・命中55・PP10、`Event.ON_DAMAGE_HIT`
での命中率ダウン追加効果）と `handlers/move_attack.py` の専用ハンドラ `グラスミキサー_lower_acc`、
`types/move.py` の `MoveName` Literal が誤って登録されていたことが判明したため、本レビューで削除した。
詳細は `docs/review/moves/グラスミキサー.md` を参照。

## 参考: 第5〜8世代時点のデータ（実装しないが記録として残す）

| よこどり | ×（横取りされない） |
| オウムがえし | ○（コピーされる） |
| Zワザ威力 / ダイマックスわざ威力 | 120 |

（出典: `docs/wiki/moves/グラスミキサー.html`）

なお過去に誤って登録されていた `MoveData.power=60, accuracy=55` は上記の基礎威力65・命中率90とは
一致しない値であり、出典・根拠は不明（他技との混同または入力ミスと推測される）。ハンドラ本体の効果
（`modify_defender_stats(..., stats={"accuracy": -1}, chance=0.5)`）自体は一次情報の追加効果と一致していた。
