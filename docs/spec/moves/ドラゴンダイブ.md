# ドラゴンダイブ仕様書

参照: docs/wiki/moves/ドラゴンダイブ.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: ドラゴン
* 分類: 物理
* 威力: 100
* 命中: 75
* PP: 12（`docs/champions/move_list.txt` の値。Gen9本家の値は10だが、本プロジェクトは
  champions 基準のため全技PPが8/12/16/20の4段階に統一されている
  `docs/champions/changes_from_sv.md` に基づき12を正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`（接触技）, `secondary_effect`（追加効果を持つ技。ちからずく対象技）,
  `minimize`（ちいさくなる状態の相手に対して威力2倍・必中になる技）

## 効果

* 追加効果として、20%の確率で相手をひるませる。
* ちいさくなる状態の相手に対しては必ず命中し、ダメージ（威力）が2倍になる（第六世代以降）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率75。ただしちいさくなる状態の相手には必中）
* 追加効果: あり（20%・ひるみ付与）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象外（本追加効果はひるみ効果のため、第四世代までは発動率上昇の対象だったが第五世代以降は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、ひるみの追加効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。
* ちいさくなるとの相互作用（威力2倍・必中）は `minimize` フラグを介して
  `handlers/volatile.py` の `ちいさくなる_boost_power` / `ちいさくなる_guaranteed_hit` が
  汎用的に処理するため、本技固有のハンドラは不要（他の `minimize` フラグ技と共通）。
