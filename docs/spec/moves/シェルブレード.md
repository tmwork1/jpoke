# シェルブレード（Razor Shell）仕様書

参照: docs/wiki/moves/シェルブレード.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 75
* 命中: 95
* PP: 12（champions基準。src/jpoke/data/ps-champ-ja/moves.json 準拠。一次情報の本家基準は10）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、50%の確率で相手の『ぼうぎょ』ランクを1段階下げる。
* 「切る技」の1つ。特性きれあじのポケモンが使用すると威力が1.5倍になる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率95）
* 追加効果: あり（50%・ぼうぎょ-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・ぼうぎょ低下の
  追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）
