# なりきり（Role Play）仕様書

参照: docs/wiki/moves/なりきり.html（ポケモンWiki）、
docs/spec/abilities/トレース.md、docs/spec/abilities/うのミサイル.md、
docs/spec/abilities/ふしぎなまもり.md（関連特性の仕様書）

## 基本情報

* タイプ: エスパー
* 分類: 変化
* 威力: -
* 命中: - （必中）
* PP: 12（champions/move_list.txt準拠。第三世代の本家では表示上「100」だが、
  実際には命中判定自体が行われず必ず成功する）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 自分の特性を、対象の特性と同じものに変化させる。
* 成功すると特性バーが表示され、対象から何の特性をコピーしたかが分かる。
* 場に出たときに発動する特性（いかく等）をコピーした場合、コピーした直後に発動する。

## 失敗条件

以下のいずれかに該当する場合、技は失敗する。

* 対象の特性が「コピーできない特性」（`uncopyable` フラグを持つ特性。
  マルチタイプ・イリュージョン・かわりもの・ダルマモード・バトルスイッチ・
  ARシステム・ぎょぐん・スワームチェンジ・ぜったいねむり・ばけのかわ・
  リミットシールド・レシーバー・アイスフェイス・じんばいったい・
  はらぺこスイッチ・かがくへんかガス・おもかげやどし・クォークチャージ・
  こだいかっせい・しれいとう・ゼロフォーミング・テラスシェル・
  テラスチェンジ・どくくぐつ・マイティチェンジ・てんきや・トレース・
  フラワーギフト等）である場合。
  * うのミサイルは第八世代とSV(Ver.2.0.1-2.0.2)ではコピーできなかったが、
    SV(Ver.3.0.0以降、本プロジェクトが対象とする最新版)ではコピー可能になった
    （`docs/spec/abilities/うのミサイル.md` 参照）。
* 対象の特性がふしぎなまもりである場合。
  * ふしぎなまもりは上記の一般的な「コピーできない特性」のリストには
    含まれない特殊な例外で、特性トレース・なかまづくりでは正常にコピー
    できるが、なりきりに対してのみコピーできない
    （`docs/spec/abilities/トレース.md` 備考、`docs/spec/abilities/ふしぎなまもり.md`
    「特性の変更」節を参照）。
* 使用者自身の特性が「上書きできない特性」（`protected` フラグを持つ特性。
  マルチタイプ・ARシステム・アイスフェイス・うのミサイル・おもかげやどし・
  ぎょぐん・じんばいったい・スワームチェンジ・ぜったいねむり・ダルマモード・
  テラスチェンジ・ばけのかわ・バトルスイッチ・マイティチェンジ・
  リミットシールド等）である場合。
  * ただし使用者自身がふしぎなまもりを持っている場合は例外的に成功し、
    他の特性へ上書きできる（`docs/spec/abilities/ふしぎなまもり.md` 参照）。
* 自分の特性と対象の特性が既に同じ場合（第五世代以降の仕様。第四世代までは
  同じ特性同士でも成功していた）。
* 使用者の持ち物がとくせいガードである場合。
* 使用者の特性変更がその他の理由（とくせいガード等）でブロックされる場合。

## 特性コピーに関する仕様

* コピー先の特性は交代すると元の特性に戻る（`base_ability_name` は変化しない）。
* 相手の特性がいえき等により無効化されている（とくせいなし状態）場合でも失敗せず、
  無効化された特性の名前（空文字列）をそのままコピーする。
* コピーした特性がいかく等の「場に出たとき発動する特性」の場合、コピー直後に
  ON_ABILITY_ENABLED が発火し即座に効果が発動する。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中技）
* 追加効果: なし
* まもる: 防がれない（`unprotectable`）
* みがわり: 防がれない（`bypass_substitute`）
* おうじゃのしるし: 対象外（変化技）
* マジックコート・マジックミラー: 反射不可（`unreflectable`）
* よこどり: 対象外（未実装）
* オウムがえし: コピー不可（対象外・変化技のため未実装のオウムがえしとは無関係）
* 特性おうごんのからだ: 防がれる（変化技共通の無効化処理。使用者の特性が
  かたやぶりの場合は防がれずにコピーできる）
* トリックガード: 防がれる（未実装のため現状は影響なし）
* 姿を隠している（そらをとぶ等の溜め技中）対象には当たらない
  （必中技だが semi-invulnerable 判定は通常どおり適用される）

## 備考

* 第三世代のルビー・サファイア・エメラルドでは「あいてに なりきって あいての
  とくせいを コピーする」（自分の特性を変えず対象の特性を模倣するのみ）
  という異なる効果だったが、第三世代ファイアレッド・リーフグリーン以降は
  自分の特性そのものを対象と同じ特性に変化させる現行仕様になった。
* 第四世代までは自他の特性が同じでも成功したが、第五世代以降は失敗するように
  変更された。
