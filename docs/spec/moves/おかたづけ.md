# 基本効果

1. 自分含む場の全ポケモンの「みがわり」状態を解除する。
2. 敵味方双方の場に設置された以下のトラップ（設置技）をすべて解除する。
   - まきびし
   - どくびし
   - ステルスロック
   - ねばねばネット
3. 自分の「こうげき」・「すばやさ」を1段階ずつ上げる。

---

# 技の仕様（wikiより）

- みがわりかトラップを解除する効果が発動したときのみ、専用メッセージ（「かたづけ おわり!」）が
  表示される。解除対象が何もなかった場合はこのメッセージは出ない。
- みがわり・トラップを解除しなかった場合でも、能力ランク上昇は独立して発生する。
- 自分の「こうげき」・「すばやさ」のランクがすでに最大（+6）だったときも、みがわり・トラップを
  解除する効果自体は問題なく発動する。
- 姿を隠している（そらをとぶ・あなをほる等の溜め技中の）ポケモンのみがわり状態も解除できる。
- 特性おうごんのからだのポケモンのみがわり状態も解除できる。
- 使用者の特性がすりぬけであっても、みがわり状態を解除できる（通常通り無条件に解除される）。
- 複数の状態を解除する場合の解除順序（ダブルバトル・トリプルバトル向けの規定であり、
  本プロジェクトはシングルバトルのみが対象のため実質的な影響はない）:
  1. みがわり状態を、ホストトレーナー側→もう片方の順（各陣営内は左から）で解除する。
  2. ホストトレーナー側のトラップを、まきびし→どくびし→ステルスロック→ねばねばネットの順で解除する。
  3. もう片方のトレーナー側のトラップを、同じ順序で解除する。
- ゆびをふる（Metronome）で選ばれない（`ゆびをふる` 自体が本プロジェクトで未実装のため対象外）。
