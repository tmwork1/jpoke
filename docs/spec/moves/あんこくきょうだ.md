# あんこくきょうだ（Wicked Blow）仕様書

参照: docs/wiki/moves/あんこくきょうだ.html（ポケモンWiki）

## 基本情報

* タイプ: あく
* 分類: 物理
* 威力: 75（第九世代。第八世代では80）
* 命中: 100
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: パンチ技（てつのこぶし・パンチグローブの対象）

## 効果

* 必ず急所に当たる（急所ランク+3 = 確率100%）。
* 追加効果なし。

## 判定フラグ

* 急所: 確定（急所ランク3）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）

## 急所無効化との関係

* 確定急所ではあるが、防御側の特性カブトアーマー・シェルアーマーを持つポケモンに対しては急所に当たらない
  （急所無効化特性が確定急所効果より優先される）。
  この無効化はカブトアーマー・シェルアーマー側のハンドラで処理されるため、本技側の実装には影響しない。

## 専用技

* ウーラオス（いちげきのかた）の専用技。

## 備考

* 確定急所効果を持つ技の中では威力が最も高い。
* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
