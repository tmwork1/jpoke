# ドリルライナー（Drill Run）仕様書

参照: docs/wiki/moves/ドリルライナー.html（ポケモンWiki）

## 基本情報

* タイプ: じめん
* 分類: 物理
* 威力: 80
* 命中: 95
* PP: 12（Champions基準。docs/champions/move_list.txt を正とする。Wiki記載の10はGen9本家基準のため不採用）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`

## 効果

* 追加効果なし。急所に当たりやすい（急所ランク+1）。

## 判定フラグ

* 急所: 急所ランク+1（通常より高い確率で急所に当たる）
* 命中判定: あり（命中率95）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 対象外（ひるみ追加効果を持たないため無関係）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* 追加効果を持たないため、専用ハンドラは不要。`critical_rank=1` を `MoveData` に設定するのみで、
  既存の急所判定ロジック（`core/move_executor.py` の `CRIT_RATES` 参照処理）が自動的に適用される。

## リーサル計算

対象外（追加効果を持たない通常の物理攻撃技のため、`docs/progress/move.md` の
「リーサル実装」列は `n/a`）。

## 備考

* Zワザ/ダイマックスわざ関連仕様は本プロジェクトでは対象外。
