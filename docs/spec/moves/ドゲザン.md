# ドゲザン（Kowtow Cleave）仕様書

参照: docs/wiki/moves/ドゲザン.html（ポケモンWiki）

## 基本情報

* タイプ: あく
* 分類: 物理
* 威力: 85
* 命中: - （必中）
* PP: 12（`docs/champions/move_list.txt` 準拠。本家では10だが、本プロジェクトの
  対象範囲であるチャンピオンズでは全技PPが8/12/16/20の4段階に統一されているため12）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`, `slash`（切る技。特性きれあじで威力1.5倍）

## 効果

* 自分の攻撃は必ず命中する（自分の命中率、相手の回避率に関係なく必ず当たる）。
* 急所に当たりやすい（急所ランク+1）。
* 追加効果なし。

## 判定フラグ

* 急所: 急所ランク+1（通常より高い確率で急所に当たる）
* 命中判定: 必中（命中判定自体が行われない）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）

## イベント / 優先度

* 追加効果を持たないため専用ハンドラは不要。`MoveData` に `accuracy=None`（必中）と
  `critical_rank=1`（急所ランク+1）を設定するのみで、既存の命中判定・急所判定ロジック
  （`core/move_executor.py`）が自動的に適用される。

## リーサル計算

対象外（追加効果を持たない通常の物理攻撃技のため、`docs/progress/move.md` の
「リーサル実装」列は `n/a`）。

## 備考

* ドドゲザンの専用技。
* あくタイプの必中技は他にだましうち、いじげんラッシュ、どげざつきがある。
* 技の分類上「切る技」に属し、特性きれあじの対象。
