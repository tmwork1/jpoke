# きしかいせい（Reversal）仕様書

参照: docs/wiki/moves/きしかいせい.html（ポケモンWiki）
参照: docs/spec/moves/_variable_power.md（変動威力技共通仕様、第4節）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 可変（20〜200）
* 命中: 100
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 自分（使用者）の残りHPが少ないほど、威力が大きくなる。
* 威力算出用の値 X は、使用者の残りHP・最大HPから以下の式で算出する（小数点以下切り捨て）。

  ```
  X = floor(残りHP × 48 / 最大HP)
  ```

* X の値から以下の表で威力を決定する（第9世代基準。第2〜3世代・第5世代以降と同じ式）。

  | X の範囲 | 威力 | 残りHP割合（概算） |
  |---|---|---|
  | X ≧ 33 | 20 | 68.7%以上 |
  | 17 ≦ X ≦ 32 | 40 | 35.4〜68.7% |
  | 10 ≦ X ≦ 16 | 80 | 20.8〜35.4% |
  | 5 ≦ X ≦ 9 | 100 | 10.4〜20.8% |
  | 2 ≦ X ≦ 4 | 150 | 4.2〜10.4% |
  | 0 ≦ X ≦ 1 | 200 | 4.2%以下 |

* じたばたと全く同じ威力計算式を使用する。

## 判定フラグ

* 急所: 通常通り発生する（第三世代以降。第二世代のみ急所に当たらない仕様だが、本プロジェクトはSV基準のため非対象）。
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: ひるみ効果あり（通常の直接攻撃技として扱う）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能（本プロジェクトはオウムがえし自体が未実装のため対応不要）

## リーサル計算

* 威力が自分自身の残りHPに依存するが、ダメージ計算対象の変化ではなく攻撃力側の可変要素であるため、
  `lethal.py` の想定する「相手HPの分布」（`StateDist`）には直接影響しない。
  リーサル計算は使用時点の実HPで威力を確定させる通常のダメージ計算ロジックで扱われるため、
  専用のリーサルハンドラは不要（`docs/progress/move.md` のリーサル実装/リーサルテスト列は n/a）。
