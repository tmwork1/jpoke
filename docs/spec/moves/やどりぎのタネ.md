# やどりぎのタネ（Leech Seed）仕様書

参照: docs/wiki/moves/やどりぎのタネ.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 変化
* 威力: -
* 命中: 90
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 準拠）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 相手を『やどりぎのタネ』状態にする。
* 『やどりぎのタネ』状態のポケモンはターン終了時に最大HPの1/8のダメージを受け、
  その分だけ技の使用者がいた場所のポケモンのHPが回復する。
* 自分が交代しても効果は引き継がれる（回復先は「使用者がいた場所」であり、
  使用者自身ではない）。
* 『くさ』タイプのポケモンには技が失敗する（付与できない）。
  * 特性ノーマルスキンやそうでん状態などにより技のタイプがくさタイプ以外に
    変わっていたとしても、やどりぎのタネはくさタイプのポケモンに対して効果が無い
    （技固有の失敗条件のため）。
  * 対象がねらいのまとを持っているときでも効果は無い
    （ねらいのまとが無効化するのは「タイプ相性による無効」のみであり、
    本技のくさタイプ無効はそれとは別の失敗条件のため）。
* くさタイプの技なので、特性そうしょくのポケモンに対しては無効となる。
  * 技のタイプがくさタイプ以外に変化していれば有効となる。
  * 特性かたやぶりのポケモンが使用したときなど、特性を無視して使用できれば
    有効となる（そうしょくは `mold_breaker_ignorable`）。
* すでに対象がやどりぎのタネ状態になっている場合は失敗する
  （揮発性状態の重複付与禁止の一般規則により処理される）。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: あり（命中率90）
* 追加効果: なし（本技の状態変化付与自体が主効果であり、追加効果扱いではない）
* まもる: 防がれる
* みがわり: 防がれる（`move_executor.check_hit_substitute` が一律に判定するため技側の実装は不要）
* おうじゃのしるし: 対象外
* マジックコート: 反射可能
  * マジックコート状態で技を跳ね返された場合、マジックコート状態のポケモンから
    本来の使用者に対して技を使用したことになる。回復先はマジックコート状態の
    ポケモンがいる位置になる。
* よこどり: 対象外（本技はよこどりで奪えない）
* オウムがえし: 反射可能
