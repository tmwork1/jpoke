# ムーンフォース（Moonblast）仕様書

参照: docs/wiki/moves/ムーンフォース.html（ポケモンWiki）

## 基本情報

* タイプ: フェアリー
* 分類: 特殊
* 威力: 95
* 命中: 100
* PP: 15（本家）→ 16（Champions。`src/jpoke/data/ps-champ-ja/moves.json` に本技のPPが16と明記されている）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、相手の『とくこう』ランクを1段階下げる。
  * スカーレット・バイオレットまで: 発動確率30%
  * **Champions: 発動確率10%**（一次情報の説明文に「10%の確率で相手の特攻を1段階下げる。」と
    明記。`docs/champions/changes_from_sv.md` にも「ムーンフォース 特攻ダウン30%→10%」と
    記載されている）

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（10%・とくこう-1、`flags={"secondary_effect"}` によりちからずく等の対象）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## しぜんのちからとの関係

* ミストフィールド状態で「しぜんのちから」を使用すると、ムーンフォースに変化する
  （本プロジェクトでは `しぜんのちから` 側の実装で対応する範囲であり、本技側の実装には影響しない）。

## 備考

* Zワザ威力: 175（本プロジェクトでは対象外）。
* ピッピ・ナゾノクサの専用技だったが、第八世代以降は進化後も習得可能。
