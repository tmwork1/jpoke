# クロスフレイム（Fusion Flare）仕様書

参照: docs/wiki/moves/クロスフレイム.html（ポケモンWiki）
参照: docs/wiki/moves/クロスサンダー.html（コンビネーション技の相方。デュアルクロスの記述を相互参照）
参照: docs/spec/moves/かえんぐるま.md（自分のこおり状態を無条件解凍する処理の実装パターン）
参照: docs/spec/moves/_move_labels.md（82行目 自分のこおり状態を解除カテゴリ、531行目 thaw ラベル）

## 基本情報

* タイプ: ほのお
* 分類: 特殊
* 威力: 100
* 命中: 100
* PP: 8（`docs/champions/move_list.txt` に記載なし。ただし対になるクロスサンダーを含め、
  オーバーヒート・ハイドロカノン・はかいこうせん・りゅうせいぐん・てっていこうせん・
  ガリョウテンセイ等、第9世代PP5の技はchampionsで一様にPP8へ変換されている前例が多数あり、
  この一般則に従う）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃×）
* ホワイトキュレムの専用技

## 効果

* 使用者が『こおり』状態の場合、こおり状態を治してから攻撃する（無条件解凍。通常のこおり状態の
  ターン開始時解凍抽選＝25%とは独立した専用の解除処理）。
  - `docs/spec/moves/_move_labels.md` の「自分のこおり状態を解除 / unfreeze」カテゴリに掲載されており、
    かえんぐるま・かえんボール・もえつきる等と同じ実装パターンが必要。
  - `flags` の `"thaw"` は現状、被弾側（相手がこの技のダメージを受けた際にこおりを解凍する:
    `こおり_cure_by_thaw_move`, `Event.ON_DAMAGE_HIT`）にのみ使われており、使用者自身が
    こおり状態のときに無条件解凍する処理は技ごとに個別実装が必要
    （実装例: `もえつきる_thaw_attacker`, `かえんぐるま_thaw_attacker`, いずれも
    `Event.ON_TRY_ACTION`, priority=5）。クロスフレイムにはこの使用者側の解凍処理が
    未実装だったため、本レビューで追加する。
* 追加効果なし（一次情報「追加効果: ×」）。
* 直前に成功した技がクロスサンダーだった場合、クロスフレイムの威力が2倍になる
  （通称「デュアルクロス」）。
  - **判定条件に「味方」「自分以外」の制限はない。** 一次情報の文言は「直前に成功した技が
    クロスサンダーだったとき」であり、そのターン中に直前に成功した技を主体を問わず参照する。
    シングルバトルでも、相手がそのターンにクロスサンダーを先に命中させていれば
    （速度でクロスフレイム使用者より先に行動した場合）成立する。1体のポケモンが両方の技を
    同時に覚える手段がない（ホワイトキュレム/ブラックキュレムは排他フォルム）ことは事実だが、
    それは「発動条件を満たす」ことと無関係であり、対象外理由にはならない。
  - `docs/wiki/moves/クロスサンダー.html` の「こんなときに使おう」節にある
    「ダブルバトルでクロスフレイムとのデュアルクロスを狙う場合、味方2体のすばやさが
    近くなければ発動しにくい」という記述は、ダブルバトルで味方2体を使って狙い撃つ場合の
    運用上の注意であり、シングルバトルでの発動を否定するものではない。
  - よって本プロジェクトでも実装対象とする。実装は `battle.fusion_bolt_used_turn` /
    `battle.fusion_flare_used_turn`（クロスサンダー側と対になる）を用いて、`Event.ON_HIT`
    で命中したターンを記録し、`Event.ON_CALC_POWER_MODIFIER` で「同じターン内でクロスサンダーが
    既に命中しているか」を判定する（`docs/spec/moves/クロスサンダー.md` 参照）。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率100）
* 追加効果判定: なし（追加効果なし）
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降。第二〜第四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## Zワザ・ダイマックスわざ

* Zワザ威力180・ダイマックスわざ威力130は本プロジェクトの対象範囲外のため実装しない。

## 実装方針

* ダメージ計算を行う通常の特殊技として実装する。
* `flags` に `"thaw"` を維持する（被弾側の解凍は `こおり_cure_by_thaw_move` により既に有効）。
* 使用者自身の無条件解凍は `クロスフレイム_thaw_attacker`（`Event.ON_TRY_ACTION`, priority=5）を
  新規実装し登録する。
* デュアルクロス（威力2倍）は `クロスフレイム_calc_power`（`Event.ON_CALC_POWER_MODIFIER`）と
  `クロスフレイム_record_hit`（`Event.ON_HIT`）で実装する（クロスサンダー側と対になる実装。
  `docs/spec/moves/クロスサンダー.md` 参照）。
