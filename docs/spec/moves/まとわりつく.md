# まとわりつく（Infestation）仕様書

参照: docs/wiki/moves/まとわりつく.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 特殊
* 威力: 20
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける。
  * バインド状態の詳細仕様（継続ターン、しめつけバンドによる強化、マジックガードでの無効化、交代禁止とその例外、解除条件など）は `docs/spec/volatiles/バインド.md` を参照する。
* むしタイプの技であり、まきつく・しめつけるのようなノーマルタイプ技とは異なりゴーストタイプに対する無効化は発生しない（むし対ゴーストは効果いまひとつの通常の相性判定に従う）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし（バインド付与自体が本体効果であり追加効果としての判定はない）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
* うずしお・しめつける・まきつく・ほのおのうず・からではさむ・すなじごく・トラバサミ・サンダープリズン・マグマストームと同系統の「バインド」技グループに属する。
* 第六世代で新規追加された技。
