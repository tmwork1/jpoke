# ラスターパージ（Luster Purge）仕様書

参照: docs/wiki/moves/ラスターパージ.html（ポケモンWiki）

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: 95（第八世代までは70。第九世代で95に強化された）
* 命中: 100
* PP: 8（本家基準は5。src/jpoke/data/ps-champ-ja/moves.json にはラスターパージの記載がないが、
  changes_from_sv.md の「全技PPが8/12/16/20の4段階に統一」により、本家PP5の技
  （りゅうせいぐん・じばく・だいばくはつ等）は move_list.txt 上で一律PP8に変換されている
  ため、同じ本家PP5のラスターパージもPP8として扱う）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（間接攻撃）

## 効果

* 追加効果として、50％の確率で相手の『とくぼう』ランクを1段階下げる。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率100）
* 追加効果: あり（50%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: コピー可能（特殊技のため）

## 備考

* Zワザ威力: 140（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 対象外。
