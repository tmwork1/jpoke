# どろかけ（Mud-Slap）仕様書

参照: docs/wiki/moves/どろかけ.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: じめん
* 分類: 特殊
* 威力: 20
* 命中: 100
* PP: 12（`docs/champions/move_list.txt` の値を採用。ポケモンWiki記載の本家シリーズ値は10だが、
  本プロジェクトはポケモンチャンピオンズを対象とするため `docs/champions/move_list.txt` の値を優先する）
* 対象: 相手1体
* 優先度: 0
* 接触: なし
* フラグ: `secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、100%の確率で相手の命中率を1段階下げる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・命中率ランク-1）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象（第二世代〜第四世代は対象外）。本追加効果は能力ランクを下げる効果であるため、
  発動率上昇の対象となる（実装上は `battle.resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、
  命中率を下げる追加効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。
