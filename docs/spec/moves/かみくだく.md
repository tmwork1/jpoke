# かみくだく（Crunch）仕様書

参照: docs/wiki/moves/かみくだく.html（ポケモンWiki）

## 基本情報

* タイプ: あく
* 分類: 物理
* 威力: 80
* 命中: 100
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、20%の確率で相手の『ぼうぎょ』ランクを1段階下げる
  （第四世代以降。第二・第三世代は『とくぼう』-1だったが本プロジェクトでは対象外）。
* かみつき技の1つ。特性『がんじょうあご』のポケモンが使うと威力が1.5倍になる
  （`flags` に `"bite"` を付与するのみでよく、特性側で汎用的に処理される）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（20%・ぼうぎょ-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・ぼうぎょ低下の
  追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）
