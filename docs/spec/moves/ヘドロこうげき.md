# ヘドロこうげき（Sludge）仕様書

参照: docs/wiki/moves/ヘドロこうげき.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 特殊
* 威力: 65
* 命中: 100
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、30%の確率で相手を『どく』状態にする
  （第一世代のみ40%だったが、本プロジェクトは現行仕様のみ対象のため30%）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（30%・どく）
* まもる: 防がれる
* おうじゃのしるし: 対象外（本技はもともと状態異常の追加効果を持つため、おうじゃのしるし
  によるひるみ追加効果とは無関係。本技自体の追加効果発動率には影響しない）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## Zワザ・ダイマックスわざ

* Zワザ威力: 120
* ダイマックスわざ威力: （本プロジェクトの対象範囲外のため実装しない）

## Champions固有仕様

* PP: src/jpoke/data/ps-champ-ja/moves.json には本技の記載がないが、本家Gen9基準のPP(20)が
  Championsで採用される4段階(8/12/16/20)の最大値と一致するため変換不要。実装値(20)も
  この結論と一致している。
