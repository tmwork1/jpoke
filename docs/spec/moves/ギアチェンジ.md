# ギアチェンジ（Shift Gear）仕様書

参照: docs/wiki/moves/ギアチェンジ.html（ポケモンWiki）

## 基本情報

* タイプ: はがね
* 分類: 変化
* 威力: -
* 命中: -
* PP: 10（docs/champions/move_list.txt に該当行なし。docs/wiki/moves/ギアチェンジ.html の値10と現行実装が一致するためそのまま採用）
* 対象: 自分
* 優先度: 0
* 直接攻撃: いいえ

## 効果

* 自分の『こうげき』ランクを1段階、『すばやさ』ランクを2段階上げる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化がそのまま効果）
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* よこどり: 対象になる（○）
* オウムがえし: 反射不可（相手を対象としないため対象外）
