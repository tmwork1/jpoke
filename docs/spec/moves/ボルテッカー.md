# ボルテッカー（Volt Tackle）仕様書

参照: docs/wiki/moves/ボルテッカー.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 物理
* 威力: 120
* 命中: 100
* PP: 16（`docs/champions/move_list.txt` 準拠。Gen9本家のPP15はチャンピオンズのPP統一仕様で16に変更されている）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 相手に与えたダメージの33%（第五世代以降の仕様。第四世代までは1/3切り捨て）を自分も受ける（反動技）。
* 追加効果として、10%の確率で相手を『まひ』状態にする。

## 反動ダメージの計算

* 反動量 = `max(1, round_half_down(与えたダメージ * 1/3))`
  （端数は五捨五超入。参照: `docs/spec/moves/_move_secondary_2.md`）
* 反動は `reason="recoil"` で処理する（いしあたま特性による無効化のため）
* みがわりに命中した場合も反動ダメージは発生する。この場合の反動量は、
  みがわりへの与ダメージを基準に算出する。

## 追加効果（まひ）の判定

* 発動確率: 10%
* `secondary_effect` フラグを持つため、ちからずく・てんのめぐみ等の確率補正特性の対象。
* みがわりに阻まれた場合（被ダメージが0の場合）、まひの追加効果は発動しない
  （`ON_DAMAGE_HIT` はみがわりに完全に阻まれた場合は発火しないため）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（10%・まひ）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## Zワザ・ダイマックスわざ

* Zワザ威力: 190（スパーキングギガボルト） / 210（専用Z技『ひっさつのピカチュート』）
* ダイマックスわざ威力: （本プロジェクトの対象範囲外のため実装しない）

## 備考

* 反動比率は1/3（とっしん・じごくぐるま・アフロブレイク・ワイルドボルト等は1/4反動であり、
  本技はそれらより反動比率が大きい）。
* 特性が『いしあたま』『マジックガード』の場合、反動によるダメージを受けない。
* 特性が『すてみ』の場合、威力が1.2倍になる（`すてみ`特性側で汎用的に処理される）。
* ピチューが『でんきだま』を持たせた状態で生まれることで習得できる専用の遺伝技
  （詳細は `docs/spec/items/でんきだま.md` を参照）。
