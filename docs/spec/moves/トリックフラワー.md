# トリックフラワー（Flower Trick）仕様書

参照: docs/wiki/moves/トリックフラワー.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 70
* 命中: -（必中）
* PP: 12（champions/move_list.txt準拠。Gen9本家は10）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* マスカーニャ専用技（進化時に覚える）

## 効果

* 必ず命中する（自分の命中率、相手の回避率に関係なく必中）。
* 必ず急所に当たる（急所ランク3扱い）。
* 追加効果なし。

## 判定フラグ

* 命中判定: 必中（`accuracy=None`）
* 急所: 必ず急所（急所ランク3）
  * シェルアーマー/カブトアーマーの相手には急所に当てられない（一般ハンドラで処理、かたやぶりで無視）
  * おまじない状態の相手には急所に当てられない（一般ハンドラで処理）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* 必ず急所に当たる技でありながら必中技でもあるため、相手の防御ランク・回避率ランクの
  両方を無視して攻撃できる（`docs/spec/moves/こおりのいぶき.md`・`やまあらし.md` と同系統の技）。
* 急所ランク3による確定急所判定・シェルアーマー等との相互作用は、既存の急所判定共通ロジック
  （`critical_rank` を参照する汎用処理）で扱われるため、本技固有のハンドラは不要。
