# ブレイズキック（Blaze Kick）仕様書

参照: docs/wiki/moves/ブレイズキック.html（ポケモンWiki）

## 基本情報

* タイプ: ほのお
* 分類: 物理（第三世代は特殊。第四世代以降は物理。本プロジェクトの対象範囲では物理固定）
* 威力: 85
* 命中: 90
* PP: 10（一次情報の値。ただし本プロジェクトは Champions 仕様のため
  `src/jpoke/data/ps-champ-ja/moves.json` の値（PP12）を正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、10%の確率で相手を『やけど』状態にする。
* 急所に当たりやすい（急所ランク+1）。
* ほのおタイプの攻撃技であるため、被弾した相手が『こおり』状態の場合は解凍する
  （`docs/spec/ailments/こおり.md` 31-32行目、一般的なほのお技の解凍効果）。
  一次情報には自技解説として明記されていないが、`かえんほうしゃ`・`ひのこ`等の他のほのお攻撃技と
  同様の共通効果として扱う。
* 一次情報には「使用者がこおり状態のとき無条件で解凍してから攻撃する」旨の記載はなく、
  `かえんぐるま`・`かえんボール`等が持つ使用者側の無条件解凍効果（`_thaw_attacker` 系ハンドラ）は
  対象外（未記載のため実装しない）。

## 判定フラグ

* 急所: 発生する。急所ランク+1（`docs/spec/critical.md` 89行目に記載）。
* 命中判定: あり（命中率90）
* 追加効果判定: あり（10%でやけど）
* まもる: 防がれる
* おうじゃのしるし: 第三世代〜第四世代は対象外、第五世代以降は効果あり
  （ただし本追加効果は「ひるみ」ではなく「やけど」であり、おうじゃのしるしはひるみ限定の効果のため
  実質関係しない。一次情報の「判定」欄の記載をそのまま記録するのみ）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* `Event.ON_DAMAGE_HIT`（デフォルト priority=100, `attacker:self`）でやけど付与を判定する。
  他のやけど付与技（かえんほうしゃ・だいもんじ等）と同じ標準パターン。
* `critical_rank=1` を `MoveData` に設定するのみで、既存の急所判定ロジック
  （`core/move_executor.py` の `CRIT_RATES` 参照処理）が自動的に適用される。専用ハンドラは不要。
* こおり解凍は `flags` に `"thaw"` を設定することで、`data/ailment.py` に登録済みの
  `こおり_cure_by_thaw_move`（`Event.ON_DAMAGE_HIT`, `defender:self`）が汎用的に処理する。

## リーサル計算

対象外。`docs/progress/move.md` の「リーサル実装」列は `n/a`
（確率発動の追加効果を持つ通常の物理攻撃技のため）。
