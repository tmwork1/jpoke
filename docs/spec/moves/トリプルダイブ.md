# トリプルダイブ（Triple Dive）仕様書

参照: docs/wiki/moves/トリプルダイブ.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 30（1ヒットあたり、固定）
* 命中: 95
* PP: 10（`docs/champions/move_list.txt` に該当行なし。TMの存在しない専用技のため掲載対象外と
  推測される。README.md の対象範囲定義に従い、champions側の情報が存在しない場合は
  第9世代本家/Wiki準拠の値をそのまま採用する。既存実装の `pp=10` は変更不要）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 3回連続で攻撃する固定回数の連続攻撃技（3回攻撃固定技）。
  * 命中判定は1発目のみ行う（`check_hit_each_time` は使用しない）。1発目が当たれば、
    攻撃は必ず3回続く。
  * 各ヒットの威力は固定で30（`power_sequence` なし）。
  * どちらかがひんしになった場合、そこで中断する（連続攻撃技全般の仕様。
    `docs/spec/moves/_multi_hit.md` 参照）。
* 追加効果なし。

## 判定フラグ

* 急所: 通常判定（各ヒットごと）
* 命中判定: あり（命中率95、1発目のみ判定）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）

## 専用技

* ウミトリオの専用技。
