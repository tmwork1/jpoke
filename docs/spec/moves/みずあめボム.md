# みずあめボム（Syrup Bomb）仕様書

参照: docs/wiki/moves/みずあめボム.html（ポケモンWiki）
参照: docs/spec/volatiles/あめまみれ.md（追加効果で付与する揮発性状態の詳細仕様）

## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 60
* 命中: 90（Champions。SVでは85だが、本プロジェクトは対象範囲の定義によりChampions値を優先する）
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 直接攻撃: なし
* 弾の技（`bullet` フラグ）: あり。特性ぼうだんを持つ相手には無効化される。

## 効果

* 追加効果として、100%の確率で相手を「あめまみれ」状態にする。
  * あめまみれの詳細仕様（3ターンの間ターン終了時にすばやさランクを1段階下げ続ける効果、
    各特性・アイテムとの相互作用）は `docs/spec/volatiles/あめまみれ.md` を参照。

## 追加効果としての制約

* 技の効果は追加効果（`secondary_effect`）と見なされる。以下の場合はあめまみれ状態にできない。
  - 対象が特性りんぷんを持つ場合
  - 対象が持ち物おんみつマントを持つ場合
  - 使用者の特性がちからずくの場合
* 技をみがわり状態に防がれたときはあめまみれ状態にできない（追加効果はみがわりに直接当たらない
  ため、通常の追加効果処理と同様に防がれる）。

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり（命中率90）
* 追加効果: あり（100%、上記の制約あり）
* まもる: 防がれる
* おうじゃのしるし: 対象外（ひるみを発生させる技ではない）
* マジックコート: 反射不可（変化技ではないため対象外）

## リーサル計算

* `docs/progress/move.md` の「リーサル実装」「リーサルテスト」列は `n/a`。
  あめまみれによるすばやさ低下はダメージ計算・HP分布に直接影響しないため、
  専用のリーサルハンドラは不要（対象外）。
