# ずつき（Headbutt）仕様書

参照: docs/wiki/moves/ずつき.html（ポケモンWiki）、src/jpoke/data/ps-champ-ja/moves.json

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 70
* 命中: 100
* PP: 15（`src/jpoke/data/ps-champ-ja/moves.json` に本技のエントリなし。本家シリーズの値をそのまま使用）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`（接触技）, `secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、30%の確率で相手をひるませる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（30%・ひるみ付与）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象外（本追加効果はひるみ効果のため、第四世代までは発動率上昇の対象だったが第五世代以降は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、ひるみの追加効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。
