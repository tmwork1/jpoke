# てっていこうせん（Steel Beam）仕様書

参照: docs/wiki/moves/てっていこうせん.html（ポケモンWiki）
参照: docs/wiki/moves/クロロブラスト.html（同系統技。ただし特性との相互作用が異なる点に注意）
参照: docs/spec/moves/いのちがけ.md, じばく.md（HP消費系の既存実装パターン）

## 基本情報

* タイプ: はがね
* 分類: 特殊
* 威力: 140
* 命中: 95
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 使用者は、攻撃前に自分の最大HPの1/2（**小数点以下切り上げ**、最低1）を消費してから、
  通常のダメージ計算（攻撃・防御・威力・急所・タイプ相性など）で相手にダメージを与える。
* 追加効果なし。

## HP消費の順序

* `Event.ON_PAY_HP` で、命中判定・タイプ相性判定より **前** に無条件でHPを消費する。
  - まもる状態に防がれたときや、技が無効化されたとき（タイプ相性0倍等）でも使用者のHPは減る。
  - 技が外れた場合（命中判定失敗）や、姿を隠している相手（あなをほる等）に当たらなかった場合でも
    使用者のHPは減る。
  - HP消費によりHPが0になった状態でも、そのまま命中判定・ダメージ計算が実行され、相手に攻撃が届く
    （`ON_PAY_HP` はヒット処理より前に発火するため）。
* 攻撃対象がすでにひんしで相手の場に誰もいない場合は、技が外れたときとは異なり使用者のHPは減らない
  （一次情報）。本プロジェクトはシングルバトル専用で、行動時点で相手の場にひんしのポケモンが
  居続けることは想定されない（ひんしになった直後に交代フェーズで入れ替わる）ため、この分岐は
  通常のゲーム進行では発生しない。

## 特性との相互作用（重要・クロロブラストとの違い）

* 使用者の特性が **マジックガード** のときは、反動ダメージを受けない。
* 使用者の特性が **いしあたま** であっても、反動ダメージは無効化されない（通常の反動技とは異なる）。
* この2点の組み合わせ（マジックガードのみ防げる／いしあたまでは防げない）は、
  `HPChangeReason = "recoil"`（いしあたま・マジックガード双方で防げる）や
  `HPChangeReason = "self_cost"`（いしあたま・マジックガードどちらでも防げない）のどちらにも
  該当しない。そのため専用の理由 `HPChangeReason = "fixed_recoil"` を新設して区別する。
  - `マジックガード_ignore_damage` は `fixed_recoil` を除外リストに含めない（＝防がれる）。
  - `いしあたま_ignore_recoil` は `reason == "recoil"` のみを見るため `fixed_recoil` には反応しない
    （＝防がれない）。
* 参考: `クロロブラスト` は一次情報上、いしあたま・マジックガードの双方で反動を防げる
  （`HPChangeReason = "recoil"` 相当）。本技とは異なる仕様のため混同しないこと
  （クロロブラスト自体は本レビュー対象外）。
* 使用者の特性が **すてみ** であっても、技の威力は上がらない
  （すてみは `move.has_flag("recoil")` を見て判定するため、本技にその flag を付与しないことで
  対応済み。反動ダメージの理由と、すてみが参照する `flags={"recoil"}` は別概念であることに注意）。

## HP0でのひんし・全滅判定

* 反動によりHPが0になるときでも技は発動できる。この技で両者のHPが0になった場合、使用者が先に
  倒れてから相手が倒れる。この反動と攻撃の両方でお互いのポケモンが全滅した場合、
  技の使用者側の負けとなる（`ON_PAY_HP` を相手へのダメージ付与より前に処理することで、
  この裁定と同じ結果を再現できる）。
* きあいのタスキ／きあいのハチマキ、特性がんじょうを持っていても、それらは反動ダメージには
  介入しない（通常の被ダメージ処理＝相手からの攻撃時のみに介入するため、既存実装のままで
  対応済み）。

## ダメおしとの相互作用（未実装・既知の制限）

* 一次情報によれば、この技の反動ダメージを受けたポケモンが、そのターン中に別の相手から
  `ダメおし` を受けた場合、`ダメおし` の威力は2倍になる（＝反動ダメージも「そのターンに
  ダメージを受けた」扱いになる）。
* ただし、これは `てっていこうせん` に限らず「反動技」全般に共通する`ダメおし`側の仕様であり
  （一次情報: `docs/wiki/moves/ダメおし.html` 技の仕様節）、本プロジェクトの `hits_taken`
  （`ダメおし` が参照する「そのターンに攻撃を受けたか」のフラグ）は現状、通常の対戦相手からの
  攻撃ヒット時のみ加算され、反動・自己HP消費では加算されない。
* この相互作用は `ダメおし` 自体の仕様実装（`docs/progress/move.md` の `ダメおし` 行、
  現状レビュー未了）で反動技全般の扱いとして横断的に対応すべき内容であり、本技単独では
  対応しない（スコープ外）。

## 判定フラグ

* 急所: 通常判定
* 命中判定: あり（命中率95）
* 追加効果: なし
* まもる: 防がれる（防がれても反動は受ける）
* おうじゃのしるし: 効果あり（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
