# はっぱカッター（Razor Leaf）仕様書

参照: docs/wiki/moves/はっぱカッター.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 55
* 命中: 95
* PP: 25
* 対象: 相手（本プロジェクトはシングルバトルのみ対象のため相手1体として扱う。
  原作ではダブルバトル等で相手全員が対象の拡散技だが、シングル戦では影響なし）
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: `slash`（切る技）

## 効果

* 追加効果なし。急所に当たりやすい（急所ランク+1）。

## 判定フラグ

* 急所: 急所ランク+1（通常より高い確率で急所に当たる）
* 命中判定: あり（命中率95）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 対象外（ひるみ追加効果を持たないため無関係）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* 追加効果を持たないため専用ハンドラは不要。`critical_rank=1` を `MoveData` に設定するのみで、
  既存の急所判定ロジック（`core/move_executor.py` の `CRIT_RATES` 参照処理）が自動的に適用される。
* `slash` フラグにより、きれあじ特性のポケモンが使用すると威力が1.5倍になる（`きれあじ_modify_power` が
  `has_flag("slash")` を判定して処理する既存ロジックで対応済み）。

## リーサル計算

対象外（追加効果を持たない通常の物理攻撃技のため、`docs/progress/move.md` の
「リーサル実装」列は `n/a`）。
