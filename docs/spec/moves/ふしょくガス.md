# ふしょくガス（Corrosive Gas）仕様書

参照: docs/wiki/moves/ふしょくガス.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 変化
* 威力: -
* 命中: 100
* PP: 20（Wikiのゲーム内表記は40だが、src/jpoke/data/ps-champ-ja/moves.json によりポケモン
  チャンピオンズでは全技PPが8/12/16/20の4段階に統一されているため20を採用する）
* 対象: 自分以外全員（本プロジェクトはシングルバトルのみ対象のため、実質相手1体）
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 自分以外のポケモンの持ち物を消失させる（溶かす）。ダメージは与えない。
* はたきおとすと異なり、持ち物を奪って自分のものにするわけではなく、単純に消滅させる。

## 判定フラグ

* 急所: なし（変化技のため対象外）
* 命中判定: あり（命中率100）
* 追加効果: なし（持ち物消失がこの技の唯一かつ主効果であり、追加効果としては扱われない）
* まもる: 防がれる
* みがわり: 防がれる（変化技のため。使用者の特性がすりぬけならば無視できる）
* おうじゃのしるし: 発動しない（第五世代以降）
* マジックコート: 反射可能
* よこどり: 対象外（相手に効果を及ぼす変化技のため、よこどりで奪えない）
* オウムがえし: 反射可能

## 奪えない・発動しないケース

* 対象が持ち物を持っていない場合、または溶かせない持ち物を持つ場合は
  「〈対象者〉には うまく きまらなかった!!」というメッセージで失敗する。
* 対象の特性がねんちゃくの場合は溶かせない（使用者がかたやぶり、または
  かがくへんかガスが発動している場合は溶かせる）。
  * このアイテム変更可否判定は既存の `ON_CHECK_ITEM_CHANGE` イベント（はたきおとす等と共通の
    仕組み）で処理されるため、本技専用の重複ガードは不要。
* 対象がみがわり状態の場合は技自体が無効化される（変化技はみがわりに防がれるため）。
  使用者の特性がすりぬけならばみがわりを無視して溶かせる。
* 特定のポケモンと専用道具の組み合わせ（ギラティナとはっきんだま・だいはっきんだま／
  ゲノセクトとカセット／シルヴァディとメモリ／ザシアンとくちたけん／ザマゼンタとくちたたて等）は
  `removable=False` のアイテムとして既存の仕組みで奪取・除去不可判定がされるため溶かせない。
* 姿を隠している（溜め技の1ターン目等）ポケモンに対しては当たらない。
  どちらかのポケモンがノーガードならば当たる（フレームワーク共通の命中判定処理）。

## 持ち物との発動順序・他の効果との関係

* ダメージを与えないため、ゴツゴツメット・じゃくてんほけん等の被ダメージ時発動アイテムは発動しない。
* この技で消滅した持ち物は、技リサイクル・特性しゅうかく・ものひろいの復元/拾得対象にならない
  （remove_item は消費フラグを立てないため、ほおぶくろ・ゲップの発動条件も満たさない）。
* 特性かるわざのポケモンの持ち物を溶かした場合、かるわざが発動しすばやさが2倍になる。
* 特性ぶきようのポケモンの持ち物も溶かせる。マジックルーム状態でも持ち物を溶かせる。
* マジックコート状態に跳ね返された場合、使用者自身の持ち物が溶かされる
  （本プロジェクトはシングルバトルのみのため、跳ね返り時の対象は使用者1体のみ）。

## 備考

* 自分以外の全員が対象の変化技としては、フラフラダンスに次ぐ2つ目の技（Wikiより）。
  本プロジェクトはシングルバトルのみを対象とするため、事実上「相手1体」を対象とする技として扱う。
* 第八世代で登場。第九世代（スカーレット・バイオレット）時点では習得者がいないが、
  ゆびをふる等で選択されうるため実装対象とする。
