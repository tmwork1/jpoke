# ほのおのムチ（Fire Lash）仕様書

参照: docs/wiki/moves/ほのおのムチ.html（ポケモンWiki）

## 基本情報

* タイプ: ほのお
* 分類: 物理
* 威力: 90（`docs/champions/move_list.txt` / `docs/progress/move.md` 準拠。本家シリーズ(SVまで)の威力は80だが、Championsでは90）
* 命中: 100
* PP: 16（`docs/champions/move_list.txt` / `docs/progress/move.md` 準拠。本家シリーズのPPは15だが、Championsでは16）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`、ちからずくの対象技となる `secondary_effect`

## 効果

* 追加効果として、100%の確率で相手の『ぼうぎょ』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・ぼうぎょ1段階ダウン）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象になる（本追加効果はひるみ効果ではないため無関係。ただし一次情報の判定表に記載されている）
* マジックコート: 対象外
* よこどり: 対象外
* オウムがえし: 対象になる
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、ぼうぎょダウン効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）
* ノーマルタイプなどに技自体が無効化された場合、追加効果（ぼうぎょダウン）は発生しない（ダメージが発生しなかった場合はイベントが発火しないため、本技側の実装で個別対応する必要はない）。

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当。`Gのちから` 等の同種の能力ランクダウン効果の実装コメント参照）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。

## 備考

* 第七世代ではクイタランの専用技だったが、SVではエンニュートの専用技になっている。
* この技を覚えるポケモンはいずれも威力が勝るフレアドライブを覚えるが、ほのおのムチは追加効果により2発目以降ではフレアドライブと同等以上のダメージを与えられるようになる。
