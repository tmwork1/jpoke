# 仕様書: ホネこんぼう

## 対象範囲外（実装しない）

本技は第1世代から存在する技だが、**ポケットモンスター ソード・シールド以降（ソード・シールド／BDSP／スカーレット・バイオレット）で使用不可**
（`docs/wiki/moves/ホネこんぼう.html` のカテゴリ「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」および、本文中の技データ表の「ソード・シールドで使用不可」「BDSPで使用不可」「SVで使用不可」の記載、
第八世代の説明文「このわざは つかえません」より確認）。`src/jpoke/data/ps-champ-ja/moves.json`（ポケモンチャンピオンズの技一覧）にも本技のエントリは存在しない。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に該当するため、
本技は実装対象外とする。

過去のレビューにて `data/moves/move_ha.py` に本技の `MoveData`（威力100・PP10の誤った値）と
`handlers/move_attack.py` の専用ハンドラ、`tests/moves_attack/test_move_ha.py` /
`tests/moves_status/test_move_ha.py`（重複登録されていた）にテストが存在していたことが判明したため、
本レビューで削除した。詳細は `docs/review/moves/ホネこんぼう.md` を参照。
