# きょけんとつげき（Glaive Rush）仕様書

参照: docs/wiki/moves/きょけんとつげき.html（ポケモンWiki）

## 基本情報

* タイプ: ドラゴン
* 分類: 物理
* 威力: 120
* 命中: 100
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: あり

## 効果

* 技が成功した場合（相手のタイプ相性による無効化・まもる等での無効化を除く）、
  使用者に自身の次の行動が始まるまで持続する専用状態を付与する。
* この状態の間、使用者が相手から受ける技は次のようになる。
  * 必ず命中する（自分の回避率・相手の命中率補正に関係なく必中。一撃必殺技を受けた場合も必中になる）。
  * ダメージが2倍になる。
* ダメージ固定技（いのちがけ等）による被ダメージは、威力計算を経由しないため2倍にならない。

## 効果の発動条件・非発動条件

* フェアリータイプに技が無効化された場合、状態は付与されない。
* まもる・キングシールド等で技が無効化された場合、状態は付与されない。
* 技が外れた場合、状態は付与されない。
* みがわりに命中した場合は技が成功したものとして状態が付与される
  （`Event.ON_HIT` はみがわりに被弾した場合も発動するため）。

## 状態の解除タイミング

* 使用直後から、次に自身の行動が始まるまで持続する。
* 次の自分の行動が始まった時点（技の使用可否判定が始まるタイミング）で、実際に行動できるかどうかに
  関わらず状態は解除される。
  * 状態異常（まひ・ねむり・こおり等）や状態変化（ひるみ等）で動けなかった場合でも解除される。
  * こんらんの自傷などで自身を攻撃するより前に解除される。
  * 次のターンも続けてきょけんとつげきを使用した場合は、成功すれば再度状態が（新規に）付与される。

## 判定フラグ

* 急所: ○
* 命中判定: ○
* 追加効果: ×
* まもる: ○（防がれた場合、デメリットは発動しない）
* おうじゃのしるし: ○（第五世代以降）
* マジックコート: ×（攻撃技のため対象外）
