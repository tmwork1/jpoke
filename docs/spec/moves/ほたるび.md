# ほたるび（Tail Glow）仕様書

参照: docs/wiki/moves/ほたるび.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 変化
* 威力: -
* 命中: -
* PP: 20（champions 未収録。src/jpoke/data/ps-champ-ja/moves.json に本技のエントリはないが、
  既存実装の PP=20 は champions の PP 正規化4段階（8/12/16/20）の範囲内であるため変更しない）
* 対象: 自分
* 優先度: 0
* 直接攻撃: しない

## 効果

* 自分の『とくこう』ランクを3段階上げる（第四世代までは2段階。本プロジェクトは現行仕様の3段階を実装）。

## 判定フラグ（一次情報より）

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし（本体の能力ランク変化がそのまま効果）
* まもる: 防がれない（自分を対象とする変化技）
* おうじゃのしるし: 対象外
* マジックコート: 反射不可（自分を対象とするため対象外）
* よこどり: 奪える（自分を対象とする変化技のため。本プロジェクトはよこどり自体が未実装のため対応不要）
* オウムがえし: 対象外

## エッジケース

* すでに『とくこう』ランクが+6の場合、ランクが変化せず技は失敗する
  （`battle.modify_stats` が空の変化辞書を返し、フレームワーク側で失敗扱いになる）。
