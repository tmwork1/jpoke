## 効果

使用者と相手の **こうげき（A）・とくこう（C）のランク補正** を入れ替える。
実数値（種族値・努力値・個体値・性格補正等）は一切変化せず、ランク変化量のみが
両者間で交換される。

- ぼうぎょ（B）・とくぼう（D）・すばやさ（S）・命中率・回避率のランクは対象外。
- 交換後もランクは常に ±6 の範囲に収まる（既存のランク値同士を入れ替えるだけのため、
  範囲外になることはない）。
- 双方のランクが同値（0 同士等）の場合は見かけ上変化しないが、技自体は失敗しない。

## 判定

- 急所: なし（変化技）
- 命中判定: なし（必中技。ただし「姿を隠している」相手（そらをとぶ中の相手等）には当たらない）
- 追加効果: なし（本体効果がそのまま発動するため「追加効果」扱いではない）
- まもる: 防がれる（`Move.is_blocked_by_protect` は対象が `foe` の変化技を自動的に対象とするため、専用フラグは不要）
- マジックコート: 跳ね返されない。`MoveData` に `flags={"unreflectable"}` を設定する必要がある
  （デフォルトでは対象が `foe` の変化技はすべて跳ね返り対象になるため、明示的な除外が必要）
- みがわり: 貫通する。`MoveData` に `flags={"bypass_substitute"}` を設定する必要がある
- 必中技: `accuracy=None` を設定する必要がある（`accuracy=100` のままだと ON_MODIFY_ACCURACY
  経由でランク補正（回避率上昇等）の影響を受けてしまい、必中にならない）

## 技の仕様（本家一次情報）

- この技によるランク変動は、特性かいりきバサミ/クリアボディ/しろいけむり/フラワーベール/
  メタルプロテクト/ミラーアーマー、しろいきり状態、持ち物クリアチャームの影響を受けずに
  ランクを下げる（下げ無効特性等をすり抜ける）。
- この技によるランク変動はあまのじゃくの効果で逆転することがない。第五世代以降は特性
  たんじゅんの影響も受けない。
- この技によりランクを下げられた場合も特性まけんき/かちき、持ち物だっしゅつパックは
  発動しない。
- この技によりランクが上がった場合も、相手の場の特性びんじょう、持ち物ものまねハーブは
  発動しない。
- この技でランクが上がったことで、そのターンに受けるしっとのほのお/みわくのボイスの効果が
  発動することはない。
- この技でランクが下がったことで、そのターンに使ううっぷんばらしの威力が上がることはない。

これらはいずれも「ランクの直接入れ替え」を経由するため、`ON_MODIFY_STAT` を発火させない
実装（後述）で自然に満たされる。

## イベントフロー

```
ON_STATUS_HIT (priority=100, MoveHandler デフォルト)
  → パワースワップ_swap_ranks
    → attacker.rank["atk"] ⇔ defender.rank["atk"]
    → attacker.rank["spa"] ⇔ defender.rank["spa"]
```

`battle.modify_stats()` はランク変化量（デルタ）を加算するAPIのため、
「現在値そのものを入れ替える」目的には使用できない。`Pokemon.rank` dict の
該当キーを直接入れ替える方式を採る（`ガードスワップ`・`ハートスワップ` と同一パターン）。

直接代入のため `ON_MODIFY_STAT` イベントは発火しない。これにより、クリアボディ・
しろいけむり・フラワーベール・メタルプロテクト・ミラーアーマー等のランク低下無効特性や、
しろいきり・クリアチャームの影響を受けずにランクを入れ替える（ゲーム仕様上正しい挙動）。
同様にあまのじゃく・たんじゅんによる反転も発生しない。
まけんき・かちき・だっしゅつパック（ランク低下トリガー）や、びんじょう・ものまねハーブ
（ランク上昇トリガー）も発動しない。

## 備考

- ガードスワップ（ぼうぎょ・とくぼうの**ランク**入れ替え）、パワースワップ（こうげき・とくこうの
  **ランク**入れ替え）、ハートスワップ（全ステータスの**ランク**入れ替え）という整合関係がある。
- スピードスワップ（すばやさの**実数値**入れ替え）とは系統が異なる点に注意。
- 本プロジェクトの対象範囲はポケモンチャンピオンズシングルバトルのみのため、対象は
  常に相手1体で固定となる。
