# でんこうそうげき（Double Shock）仕様書

参照: docs/wiki/moves/でんこうそうげき.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 物理
* 威力: 120
* 命中: 100
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* パーモットの専用技

## 効果

* 自分が「でんき」タイプでないと失敗する（技を選択した時点ではなく、命中判定等の前段階で失敗判定される）。
* 技が成功すると、自分の「でんき」タイプを無くす。
  * でんき単タイプのときにこの技が成功すると、タイプが完全になくなる
    （相性・タイプ一致ボーナスを一切受けなくなる）。
  * 交代すると、失ったタイプは元に戻る。
* 相性・特性・まもる状態等により技が無効化されたときや、命中しなかったときはでんきタイプを失わない
  （ダメージを与えた場合のみタイプが消える）。
* テラスタルとの相互作用:
  * でんきタイプにテラスタルしたポケモンが使用した場合、成功メッセージは出るが、でんきタイプは消えない
    （テラスタル中はタイプ消失処理を無視する）。
  * 元々でんきタイプを持たないポケモンが、ものまね・まねっこ等ででんこうそうげきを使った場合でも、
    でんきタイプにテラスタルしているなら成功する。
  * でんきタイプ以外にテラスタルした場合、元々でんきタイプを持っていたとしてもこの技は失敗する。
  * ステラタイプにテラスタルしたでんきタイプのポケモンが使用した場合、ステラタイプは維持され、
    2回目以降も使用できる（ただしステラ補正は初回のみ）。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* ゆびをふるで選ばれない
