# ブラストバーン（Blast Burn）仕様書

参照: docs/wiki/moves/ブラストバーン.html（ポケモンWiki）

## 基本情報

* タイプ: ほのお
* 分類: 特殊
* 威力: 150
* 命中: 90
* PP: 5（本家基準。本プロジェクトでは docs/champions/move_list.txt により PP8 に統一）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 分類フラグ: なし

## 効果

* 技を命中させて成功した次のターン、使用者は反動で行動できなくなる（いわゆる「リチャージ」状態）。
  * がんせきほう・ギガインパクト・はかいこうせん・ハードプラント・ハイドロカノンと同じ効果。
* まもるなどで技が防がれた場合や、命中しなかった場合、相性・特性により無効化された場合はリチャージ状態にならない。
* 技が命中すれば、相手のHPが減らなかった場合（みがわり状態など）でもリチャージ状態になる。
* 相手をひんしにしたときもリチャージ状態になる。
* リチャージ状態のポケモンが交代すると、リチャージ状態は解除される。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率90）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第三世代・第四世代、第五世代以降ともに対象）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射される

## 備考

* 第八世代までのゲーム中では「究極技」として教えてもらえる技の1つ。
  第四世代以降は基本的にほのおタイプの御三家最終進化系ポケモンの専用技だが、
  わざマシンになった第九世代からはミュウも覚える。
* Zワザ威力: 200（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 150（本プロジェクトでは対象外）。
