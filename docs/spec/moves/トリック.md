# トリック（Trick）仕様書

参照: docs/wiki/moves/トリック.html（ポケモンWiki）
       docs/spec/moves/すりかえ.md
       ※ すりかえはタイプを除きトリックと全く同じ効果を持つ（トリックWiki備考節に明記）。

## 基本情報

* タイプ: エスパー
* 分類: 変化
* 威力: -
* 命中: 100
* PP: 12（第五世代までは10。本プロジェクトはチャンピオンズ仕様に基づき
  src/jpoke/data/ps-champ-ja/moves.json の値を採用）
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 自分と相手の持ち物を交換する。

## 失敗条件

* 使用者と対象がどちらも道具を持っていない場合は失敗する。
  * どちらか片方でも道具を持っているなら成功する。
* 対象のみがわり状態に防がれる（第六世代以降では、特性すりぬけのポケモンが使う場合はみがわりを無視して成功する）。
* 対象の特性がねんちゃくのときは失敗する。ねんちゃくのポケモンが道具を持っていない場合でも失敗する。
  * 特性がねんちゃくのポケモンが、他の特性のポケモンにトリックを使う場合は成功する。
* 使用者か対象が、以下の特定のポケモンと持ち物の組み合わせであるときや、特定のポケモンにその持ち物が
  渡ろうとしたときは失敗する。ただし、これらのポケモンに変身した他のポケモンならこの制約を受けない。
  * ディアルガとだいこんごうだま／パルキアとだいしらたま／ギラティナとだいはっきんだま／
    アルセウスとプレート／ゲノセクトとカセット／シルヴァディとメモリ／ザシアンとくちたけん／
    ザマゼンタとくちたたて／オーガポンと仮面。
  * 特性こだいかっせい・クォークチャージを持つパラドックスポケモン（ウガツホムラ・タケルライコ・
    テツノイワオ・テツノカシラを除く）が持つブーストエナジー。これらの特性が無効化されたときや、
    他の特性に上書きされたときでも関係無く、ポケモンの種族自体が効果判定に影響する。
  * ポケモンに対応したメガストーン（本プロジェクトはメガシンカ非対応のため対象外）。

## 他の効果との関係

* 使用者や対象の特性がぶきようであっても成功する（道具の効果が発動しないだけで、交換自体は可能）。
* 使用者や対象がさしおさえ状態であるときや、マジックルーム状態で使ったときでも成功する
  （道具の効果が使えないだけで、交換自体は可能）。
* 使用者がこだわり系アイテムを手にした場合、次のターンは別の技を選ぶことができる
  （このトリックの使用自体ではロックされない）。すでにこだわりでロックされていた場合も、
  このロックは解除される。
* ゆびをふる・ねこのてで選ばれない。
* まねっこでコピーできない。
* よこどりの対象にならない（変化技だが、よこどりでは奪えない）。
* 経験値の入らないトレーナー戦・対戦施設で交換された道具は、バトル終了後に元の持ち主に返却される
  （本プロジェクトのバトル内シミュレーションでは対象外）。
* 相手に渡った道具は場に存在し続けるため、渡した側のリサイクル・しゅうかく・ものひろいの
  復元・拾得対象にはならない。

## 判定フラグ

* 急所: 対象外（変化技のため）
* 命中判定: あり（命中率100）
* 追加効果: なし（本効果自体が追加効果を持たない単一効果技）
* まもる: 防がれる
* マジックコート: 反射不可（トリック・すりかえは持ち物授受系の変化技として明示的に反射対象外とされている）
* よこどり: 対象外
* オウムがえし: 反射可能（対象にできる）

## 備考

* Zワザにすることで自分のすばやさランクが2段階上がる付加効果を得られるが、そのためには自身の道具は
  必ずZクリスタルでなければならず、トリックを成功させることができない（本プロジェクトはZワザ非対応のため対象外）。
* わざのタイプを除き、すりかえと全く同じ効果を持つ（威力・命中・PP・追加効果・判定はすべて共通）。
