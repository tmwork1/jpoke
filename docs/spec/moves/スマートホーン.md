# スマートホーン仕様書

参照: docs/wiki/moves/スマートホーン.html（ポケモンWiki）

## 基本情報

* タイプ: はがね
* 分類: 物理
* 威力: 70
* 命中: - （必中）
* PP: 12（チャンピオンズ基準。docs/champions/move_list.txt に準拠。本家第七〜第九世代の値は10）
* 対象: 相手1体
* 優先度: 0
* 接触: あり

## 効果

* 自分の攻撃は必ず命中する（自分の命中率、相手の回避率に関係なく必ず当たる）。
* 追加効果なし。

## 判定フラグ

* 急所: あり
* 命中判定: 必中（命中判定自体が行われない）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし（第五世代以降）: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* 必中効果を持つが威力が低いため、採用は限定的（アイアンヘッドを覚えられないポケモンでの採用例がある）。
