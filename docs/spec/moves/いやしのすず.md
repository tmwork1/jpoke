# いやしのすず（Heal Bell）仕様書

参照: docs/wiki/moves/いやしのすず.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: なし
* 命中: なし（必中）
* PP: 8（`docs/champions/move_list.txt` 準拠。Gen9本家のPP=5ではない）
* 対象: 味方全体（自分と選出チーム全員）
* 優先度: 0
* 音の技（ラベル `sound`）

## 効果

* 自分と味方全体（シングルバトルでは選出した控えポケモンを含む）の状態異常を回復する。
* チーム内に状態異常のポケモンが1体もいない場合、技は失敗する。
* 音の技のため、みがわり状態の味方に対しても効果が発生する（第六世代以降の「音技はみがわり貫通」仕様に加え、
  そもそもみがわりは相手の技を防ぐものであり自分側を対象とする本技には無関係）。
* 『ダイウォール』の効果を受けない（本プロジェクトはダイマックスを対象外とするため実装上の対応は不要）。

## 判定フラグ

* 命中判定: なし（必中）
* まもる: 防がれない（`守×`。対象が「味方全体」であり相手を対象としないため、`is_blocked_by_protect`
  の判定上も `target != "foe"` により自動的に防がれない）
* マジックコート/マジックバウンス: 反射されない（相手を対象としないため `target in {"foe", "foe_side"}` を満たさない）
* かいふくふうじ: 対象外（`heal` ラベルを持たないため、かいふくふうじ状態でも使用できる）
* よこどり: 奪われた場合、よこどりした側のポケモンの状態異常が回復する
  （本プロジェクトのよこどり実装は `docs/spec/moves/よこどり.md` を参照）

## ぼうおんとの相互作用

* 使用者自身がぼうおんの場合: 第五世代・第八世代以降では自身の特性の影響を受けずに状態異常を回復できる
  （第四世代以前・第六世代・第七世代のみ無効化されるが、本プロジェクトは最新世代基準のため対象外）。
* 場にいる味方がぼうおんの場合: 第六世代以降では無効化される。ただし本プロジェクトはシングルバトル限定のため、
  「場にいる味方」は使用者自身以外に存在せず、この効果は実質的に発生しない。
* 控えにいる味方がぼうおんの場合: 第五世代以降では無効化されず、通常通り回復できる。
* 相手（対戦相手側）のぼうおんは本技の対象外であり、一切関係しない
  （相手を対象とする音の技のみをぼうおんは無効化する）。

## 備考

* Zワザ・ダイマックスわざは本プロジェクトでは対象外。
* 同様の効果を持つ技にアロマセラピー（くさタイプ、音の技ではない）がある
  （`docs/spec/moves/アロマセラピー.md` を参照）。
