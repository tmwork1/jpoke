# ぜったいれいど（Sheer Cold）仕様書

参照: docs/wiki/moves/ぜったいれいど.html（ポケモンWiki）
参照: docs/spec/moves/_ohko.md（一撃必殺技 共通仕様）

## 基本情報

* タイプ: こおり
* 分類: 特殊
* 威力: -（一撃必殺技のため通常のダメージ計算式を使用しない）
* 命中: 30（固定。使用者Lvと対象Lvの差による変動は対人戦では無視して実装しない）
  * ただし、使用者が「こおり」タイプでない場合、基礎命中率が20に下がる
    （`一撃必殺技`共通の命中率仕様とは異なる、本技固有の仕様）。
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 命中した場合、対象のHPを0にする（`一撃必殺技`の共通仕様。docs/spec/moves/_ohko.md 参照）。
* 反動・追加効果・急所判定なし。

## 無効化条件

* 対象が「こおり」タイプの場合、本技は無効となる（本技専用の仕様。通常のタイプ相性表では
  こおり技はこおりタイプに対して0.5倍であり無効ではないため、専用の判定処理が必要）。
  * `docs/spec/turn.md` の `Event.ON_TRY_MOVE_2`（優先度120）「こおりタイプ: ぜったいれいど無効」
    に該当する。通常のタイプ相性判定（ON_BEFORE_APPLY_MOVE 相当）より後に判定される。
* その他、一撃必殺技共通の無効化条件（がんじょう・きあいのタスキ等での生存、まもる等での防御）は
  docs/spec/moves/_ohko.md を参照。

## 判定フラグ

* 急所: ×
* 命中判定: ○
* 追加効果: ×
* まもる: ○
* おうじゃのしるし・するどいキバ: ×（一撃必殺技には追加効果が発生しない）
* オウムがえし: ○
