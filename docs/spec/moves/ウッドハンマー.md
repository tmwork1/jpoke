# ウッドハンマー（Wood Hammer）仕様書

参照: docs/wiki/moves/ウッドハンマー.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 120
* 命中: 100
* PP: 16（`docs/champions/move_list.txt` の値。本家Gen9の15から
  `docs/champions/changes_from_sv.md` の「全技PPを8/12/16/20に統一」により変更されている）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 相手に与えたダメージの1/3（33%）を自分も受ける（反動技）。
* 追加効果なし。

## 反動ダメージの計算

* 反動量 = `max(1, round_half_down(与えたダメージ * 1/3))`
  （端数は五捨五超入。参照: `docs/spec/moves/_move_secondary_2.md`。
  一次情報にも「反動ダメージは第四世代では与ダメージ×1/3(切り捨て)、第五世代以降では
  与ダメージ×0.33(四捨五入)」と明記されている）
* 反動は `reason="recoil"` で処理する（いしあたま特性・マジックガード特性による
  無効化のため）
* みがわりに命中した場合も反動ダメージは発生する（ON_HIT はみがわり被弾時も発動）。
  この場合の反動量は、みがわりへの与ダメージを基準に算出する。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代・第五世代以降とも○）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 特性との関係

* いしあたま・マジックガード: 反動ダメージを無効化する。
* すてみ: 威力が1.2倍になる（`flags={"recoil"}` により自動的に適用対象となる）。
* がんじょう: 本技の反動ダメージによる瀕死は防がない
  （がんじょうが防ぐのはメインの攻撃ダメージによる瀕死のみ）。

## 備考

* 同系統の反動技（1/3反動）: ウェーブタックル、すてみタックル、フレアドライブ、
  ブレイブバード、ボルテッカー
* Zワザ・ダイマックスわざの威力は本プロジェクトの対象範囲外のため実装しない。
