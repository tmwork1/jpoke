# どくガス（Poison Gas）仕様書

参照: docs/wiki/moves/どくガス.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 変化
* 威力: -
* 命中: 90（第六世代以降。第四世代までは55、第五世代は80。第九世代でも90のまま変更なし）
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` に単独項目が存在しないが、championsのPP圧縮則
  （8/12/16/20の4段階。SV基準PPが20以上の技は一律20に圧縮される）から導出。
  Gen9本家（SVでも歴代通り40のまま変更なし）は40だが、本プロジェクトはchampions基準のためPP20。
  同じ「SV基準PP≧20は一律20に圧縮」パターンは`どくのこな`（SV35→champions20）、
  `いとをはく`（SV40→champions20）、`でんこうせっか`（SV30→champions20）で確認済み）
* 対象: 相手1体（第四世代までは1体選択、第五世代以降は相手全員が対象になったが、本プロジェクトは
  シングルバトルのみが対象のため実質的に相手1体で変わらない）
* 優先度: 0
* 接触: なし

## 効果

* 相手を『どく』状態にする。
* 『どく』タイプ・『はがね』タイプには通常の毒付与と同様に無効（`ailment_manager` が
  一律に判定するため技側の実装は不要）。
* 粉・胞子技ではない（`powder` フラグは不要）。どくのこな等と異なり、くさタイプ・特性
  『ぼうじん』・道具『ぼうじんゴーグル』による無効化は受けない。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: あり（命中率90）
* 追加効果: なし（本技の状態異常付与自体が主効果であり、追加効果扱いではない）
* まもる: 防がれる
* みがわり: 防がれる（`move_executor.check_hit_substitute` が音技等の例外を除き一律に判定するため技側の実装は不要）
* おうじゃのしるし: 対象外
* マジックコート: 反射可能
* よこどり: 対象外（本技はよこどりで奪えない）
* オウムがえし: 反射可能
