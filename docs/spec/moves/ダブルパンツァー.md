# 仕様書: ダブルパンツァー

## 対象範囲外（実装しない）

本技はメルメタルの専用技であり、`docs/wiki/moves/ダブルパンツァー.html` の
ページカテゴリに「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」が明記されている
（本文中の赤字警告表示でも「BDSPで使用不可」「SVで使用不可」と重複して確認できる）。
使用可能な世代は「Let's Go! ピカチュウ・Let's Go! イーブイ・第八世代」（ソード・シールド）までであり、
第9世代（スカーレット・バイオレット）では使用できない。

`src/jpoke/data/ps-champ-ja/moves.json`（ポケモンチャンピオンズの技一覧）にも本技のエントリは存在しない。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に
該当するため、本技は実装対象外とする（`ひっさつまえば` / `ふわふわフォール` / `まわしげり` と同様のケース）。

過去のレビュー（`docs/plan/moves/攻撃技_ひるみ系.md` の一括実装）にて `data/moves/move_ta.py` に
本技の `MoveData`（威力50・PP10という誤った値）と `handlers/move_attack.py` の専用ハンドラ、
`handlers/item.py` の `_INNATE_FLINCH_MOVES` への登録、および
`tests/moves_attack/test_move_ta.py` / `tests/moves_status/test_move_ta.py`（重複登録）に
テストが存在していたことが判明したため、本レビューで削除した。詳細は `docs/review/moves/ダブルパンツァー.md` を参照。

## 参考: 第7・8世代時点のデータ（実装しないが記録として残す）

（出典: `docs/wiki/moves/ダブルパンツァー.html`）

`MoveData.power=50, pp=10` という過去の誤った登録値は、実際のゲームデータ（威力60・PP5）と一致しない上、
`multi_hit`（2回攻撃固定）の設定自体が丸ごと欠落しており、1回しか攻撃しない実装になっていた
（`docs/spec/moves/_multi_hit.md` / `docs/spec/moves/連続技.md` / `docs/spec/abilities/てつのこぶし.md` /
`docs/spec/items/パンチグローブ.md` にある威力60・PP5の記載と照合して発覚）。

## 参考文献との差異について

`docs/spec/moves/_multi_hit.md` と `docs/spec/moves/連続技.md`（第二次情報源: wiki.xn--rckteqa2e.com）では
本技の対象世代が「7-」（第7世代から継続して存在）と記載されているが、これは技データテーブル上の存在を示すに
とどまり、実際の教え方（レベルアップ／技マシン／わざおしえ等）が第9世代・BDSPで提供されていないことと矛盾しない
（メルメタルが過去作／Pokémon GOから連れてきた場合のみ保持できる、教えることのできない専用技のため）。
本仕様書はダブルパンツァー固有のwikiページ（一次情報）の記載を優先し、対象範囲外と判断した。
