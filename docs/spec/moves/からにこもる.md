# からにこもる（Withdraw）仕様書

参照: docs/wiki/moves/からにこもる.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 変化
* 威力: -
* 命中: -
* PP: 40（`src/jpoke/data/ps-champ-ja/moves.json` に本技のエントリなし。第9世代の基準値をそのまま採用）
* 対象: 自分
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 自分の『ぼうぎょ』ランクを1段階上げる。

## 判定フラグ

* 急所: 発生しない（変化技のため）
* 命中判定: なし（対象が自分自身のため必中）
* 追加効果判定: なし
* まもる: 防がれない（自分自身を対象とする技のため）
* おうじゃのしるし: 効果なし（対象外）
* マジックコート: 反射不可（自分自身を対象とする技のため）
* よこどり: 対象になる（相手が『よこどり』を使っていた場合、効果を奪われる。
  ※本プロジェクトでは『よこどり』自体が未実装のため現時点では影響なし）
* オウムがえし: コピー不可（自分自身を対象とする技のため）
