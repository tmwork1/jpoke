# アイアンテール（Iron Tail）仕様書

参照: docs/wiki/moves/アイアンテール.html（ポケモンWiki）

## 基本情報

* タイプ: はがね
* 分類: 物理
* 威力: 100
* 命中: 75
* PP: 16（`src/jpoke/data/ps-champ-ja/moves.json` 準拠。本家シリーズ(SVまで)のPPは15だが、
  Championsでは全ての技のPPが8・12・16・20の4段階に統一されており、
  `docs/champions/changes_from_sv.md` の記載どおり本技は16になる）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`、ちからずくの対象技となる `secondary_effect`

## 効果

* 追加効果として、30%の確率で相手の『ぼうぎょ』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率75）
* 追加効果: あり（30%・ぼうぎょ-1）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象になる（本追加効果はひるみ効果ではないため無関係。
  ただし一次情報の判定表に記載されている）
* マジックコート: 対象外（変化技ではないため）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 対象になる
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる
  代わりに、ぼうぎょダウン効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、
  本技側の実装への影響はない）

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当。
  `Gのちから`・`ほのおのムチ` 等の同種の能力ランクダウン効果の実装コメント参照）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、
  イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合や
    ダメージ計算自体が行われなかった場合には発動しない仕様を担保する。

## リーサル計算

  本レビューの対象範囲外（対象外）とする。
