# ひっかく（Scratch）仕様書

参照: docs/wiki/moves/ひっかく.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 40
* 命中: 100
* PP: 35
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 通常攻撃。追加効果なし。

## 判定フラグ

* 急所: 通常判定（急所ランク0）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* Zワザ威力: 100、ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
