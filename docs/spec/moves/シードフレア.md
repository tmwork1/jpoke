# シードフレア（Seed Flare）仕様書

参照: docs/wiki/moves/シードフレア.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 120
* 命中: 85
* PP: 5（`docs/champions/move_list.txt` に本技のエントリは存在しない。本技はシェイミの専用技であり、
  シェイミは幻のポケモンのためポケモンチャンピオンズのレギュレーションでは使用不可。move_list.txt は
  レギュレーション内で実際に使用可能な技から生成されているため、専用技を持つ幻のポケモンが使用不可の
  場合はエントリが存在しない（例: Vジェネレート＝ビクティニ専用、プラズマフィスト＝ゼラオラ専用、
  ダイヤストーム＝ディアンシー専用も同様に move_list.txt に存在しない）。よってPPはGen9本家基準の
  `5` のまま据え置く（8/12/16/20への変換対象外）。
* 対象: 相手1体
* 優先度: 0
* 接触: なし（非直接攻撃）

## 効果

* 追加効果として、40%の確率で相手の『とくぼう』ランクを2段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率85）
* 追加効果: あり（40%・とくぼう-2）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代・第五世代以降とも対象）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・とくぼう低下の
  追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）

## Zワザ

## 備考

* シェイミ（ランドフォルム／スカイフォルム）専用技。レベルアップで覚える（SVでは Lv.100）。
* Pokémon LEGENDS アルセウスでは技皆伝（早業／通常／力業）で仕様が異なるが、本プロジェクトの
  対象世代（ポケモンチャンピオンズ = SV基準）には無関係のため実装対象外。
