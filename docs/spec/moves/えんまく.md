# えんまく 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | ノーマル |
| 分類 | 変化 |
| PP | 20 |
| 命中率 | 100 |
| 範囲 | 相手1体選択 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

相手の **命中率** ランクを **1段階** 下げる。

## 判定

- 急所: なし（変化技）
- 追加効果: なし（本体の効果がランクダウンそのもの）
- まもるで防がれる: ○
- マジックコートで跳ね返される: ○
- オウムがえしでコピーされる: ○
- よこどりで奪われる: ×（Wiki記載どおり、対象を持たない変化技のため無効）

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- バトルパレスの判定基準（「ずるい」技）は実装しない
- Zワザによる付加効果（自分の回避率を1段階上げる）は実装しない（Zワザ非対応）
- `docs/champions/move_list.txt` にえんまくの記載はない（本移動先データセットには未収録の技）。
  PPは一次情報（Wiki）記載の基礎値20を採用する（8/12/16/20の4段階にすでに合致するため変更不要）
