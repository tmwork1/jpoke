# もろはのずつき（Head Smash）仕様書

参照: docs/wiki/moves/もろはのずつき.html（ポケモンWiki）

## 基本情報

* タイプ: いわ
* 分類: 物理
* 威力: 150
* 命中: 80
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 相手に与えたダメージの1/2を自分も受ける（反動技）。
* 追加効果なし。

## 反動ダメージの計算

* 反動量 = `max(1, round_half_down(与えたダメージ * 1/2))`
  （端数は五捨五超入。参照: `docs/spec/moves/_move_secondary_2.md`。比率1/2の場合、
  端数は必ず0か0.5となるため `int()` 切り捨てと `round_half_down` の結果は一致する）
* 反動は `reason="recoil"` で処理する（いしあたま特性・マジックガード特性による
  無効化のため）
* みがわりに命中した場合も反動ダメージは発生する（ON_HIT はみがわり被弾時も発動）。
  この場合の反動量は、みがわりへの与ダメージを基準に算出する。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率80）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代・第五世代以降とも○）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 特性との関係

* いしあたま・マジックガード: 反動ダメージを無効化する。
* すてみ: 威力が1.2倍になる（`flags={"recoil"}` により自動的に適用対象となる）。
* がんじょう: 本技の反動ダメージによる瀕死は防がない
  （がんじょうが防ぐのはメインの攻撃ダメージによる瀕死のみ。
  参照: `docs/spec/abilities/がんじょう.md`）。

## 備考

* 同系統の反動技（1/2反動）: はめつのひかり
* Zワザ・ダイマックスわざの威力は本プロジェクトの対象範囲外のため実装しない。
