# こごえるかぜ（Icy Wind）仕様書

参照: docs/wiki/moves/こごえるかぜ.html（ポケモンWiki）

## 基本情報

* タイプ: こおり
* 分類: 特殊
* 威力: 55
* 命中: 95
* PP: 16（`docs/champions/move_list.txt` 準拠。本家シリーズのPPは15だが、Championsでは16）
* 対象: 相手1体（本家の範囲は「相手全員」だが、本プロジェクトはシングルバトルのみ対象のため相手1体で問題ない）
* 優先度: 0
* 接触: なし（非直接攻撃）
* フラグ: `wind`（風技）, `secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、相手の『すばやさ』ランクを1段階下げる（100%）。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率95）
* 追加効果: あり（100%・すばやさ1段階ダウン）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象になる（本追加効果はひるみ効果ではないため無関係。ただし一次情報の判定表に記載されている）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、すばやさダウンの追加効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）
* かぜのり・つむじかぜ等の「風」関連特性: `wind` フラグにより判定対象になる。

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。
