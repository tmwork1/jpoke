# 仕様書: ふわふわフォール

## 対象範囲外（実装しない）

本技は `docs/wiki/moves/ふわふわフォール.html` によれば **第七世代（`ポケットモンスター Let's Go! ピカチュウ・Let's Go! イーブイ`）専用の「相棒わざ」** であり、
相棒ピカチュウのみが使用できる専用技として追加されたものである。ソード・シールド以降（BDSP・SVを含む）には
一度も実装されておらず、README.md の「対象範囲」の定義

> 第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない

に該当するため、本技は実装対象外とする。

また、`docs/champions/move_list.txt`（ポケモンチャンピオンズの技一覧）にも「ふわふわフォール」の項目は
存在しない。ポケモンチャンピオンズ側でも採用されていないことが確認できるため、二重の根拠から対象外と判断した。

過去のレビューにて `docs/plan/moves/攻撃技_ひるみ系.md`（ひるみ系追加効果の一括実装計画）で
「スペック要確認」のまま実装され、`data/moves/move_ha.py` に誤った `MoveData`（タイプ:フェアリー・威力70・PP15、
本来は第七世代時点のひこうタイプ・威力90・PP15をベースに何らかの変換ミスが生じたものと推測される）と
`handlers/move_attack.py` の専用ハンドラ、`handlers/item.py` の `_INNATE_FLINCH_MOVES` への登録、
`tests/moves_attack/test_move_ha.py` のテストが存在していたことが判明したため、本レビューで削除した。
詳細は `docs/review/moves/ふわふわフォール.md` を参照。
