# オーロラビーム（Aurora Beam）仕様書

参照: docs/wiki/moves/オーロラビーム.html（ポケモンWiki）

## 基本情報

* タイプ: こおり
* 分類: 特殊
* 威力: 65
* 命中: 100
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` に本技の記載はないが、本家Gen9基準のPP20は
  ポケモンチャンピオンズのPP統一表（8/12/16/20の4段階）にすでに含まれる値であり、変換不要）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（非直接攻撃）

## 効果

* 追加効果として、10%の確率で相手の『こうげき』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（10%・こうげき-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。本プロジェクトの対象世代では発動する）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・こうげき低下の
  追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）

## Zワザ・ダイマックスわざ

* Zワザ威力: 120
* ダイマックスわざ威力: 120
* （本プロジェクトの対象範囲外のため実装しない）
