# かいふくふうじ 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | エスパー |
| 分類 | 変化 |
| PP | 15 |
| 命中率 | 100 |
| 範囲 | 相手全員（本家）／本プロジェクトはシングルバトルのみのため相手1体として扱う |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

相手を **5ターンの間「かいふくふうじ」状態** にする。かいふくふうじ状態の詳細な効果
（HP回復無効化の範囲・例外・持続ターン・交代での解除など）は
`docs/spec/volatiles/かいふくふうじ.md` を参照。

## 失敗条件

- 相手がすでにかいふくふうじ状態の場合は失敗する（`volatile_manager.apply` が
  既に同じ揮発性状態がある場合に `False` を返すため、専用の失敗チェックは不要）
- 相手の場にアロマベール特性のポケモンがいる場合は失敗する
  （`ON_BEFORE_APPLY_VOLATILE` でアロマベール側が阻止するため、本技側の実装は不要）

## 防がれる条件

- **まもる**: 防がれる（フレームワーク側の共通処理）
- **みがわり**: 本技自体は音技・接触技のいずれでもないため、みがわり状態の相手には
  無効化される（フレームワーク側の共通処理。サイコノイズは音技のため貫通する点と対照的）
- **マジックコート**: 第五世代以降は跳ね返される。本プロジェクトの `MoveData` は
  `unreflectable` フラグを持たないため、共通のマジックコート反射処理がそのまま適用される

## イベントフロー

```
ON_STATUS_HIT (priority=100, MoveHandler デフォルト)
  → apply_volatile_to_defender(volatile="かいふくふうじ", count=5)
```

## 対象範囲についての注記

本家では「相手全員」が対象の技だが、本プロジェクトの対象はポケモンチャンピオンズ
シングルバトルのみであり、相手は常に1体のため機能的な差異はない。`MoveData.target`
はデフォルトの `"foe"`（相手1体）のままでよい。

## ポケモンチャンピオンズ向けの注記

- 本技は第八世代（ソード・シールド）以降 SV を含めて「使用不可能なわざ」に分類される
  （出典: `docs/wiki/moves/かいふくふうじ.html` インフォボックス「SVで使用不可」）。
  ただし本プロジェクトでは、既存実装の `かいりき`・`つばめがえし`・`みねうち` 等と同様に
  使用不可能技も実装対象としている既存方針に倣い、第四〜七世代時点の仕様
  （PP15・命中100・5ターン）に基づいて実装する
- コンテスト効果・Zワザ追加効果は実装しない
