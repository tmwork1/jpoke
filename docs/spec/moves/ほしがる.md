# ほしがる（Covet）仕様書

参照: docs/wiki/moves/ほしがる.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 60（第四世代までは40。本プロジェクトは第五世代以降の値を採用）
* 命中: 100
* PP: 25（第五世代までは40。本プロジェクトは第六世代以降の値を採用）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃。第四世代から直接攻撃に変更）

## 効果

* 自分が道具を持っていない場合、ダメージを与えた相手の持ち物を奪って自分の持ち物にする。
* 自分がすでに道具を持っている場合、または相手が道具を持っていない場合は追加効果が発動せず、ダメージのみを与える。
* 追加効果は確率ではなく、条件を満たせば必ず発動する。
* 技どろぼう（あくタイプ）と全く同じ追加効果を持つ。威力・命中・PP・タイプ以外は共通。

## 発動タイミング・奪えない/発動しないケース・持ち物との発動順序・他の効果との関係

* どろぼう（docs/spec/moves/どろぼう.md）を参照。タイプが異なる点を除き、詳細な仕様（発動タイミング、
  奪えない・発動しないケース、持ち物との発動順序、他の効果との関係）はすべて共通。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（アイテム奪取。上記条件を満たせば確定発動）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* 第三世代までは特殊技だったが、第四世代で物理技に変更された。
* 技どろぼうは本技と全く同じ追加効果を持つ（威力・命中・PP・タイプ以外は共通）。
* Zワザ威力: 120、ダイマックスわざ威力: 110（本プロジェクトでは対象外）。
