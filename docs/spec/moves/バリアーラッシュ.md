# バリアーラッシュ（Psyshield Bash）仕様書

参照: docs/wiki/moves/バリアーラッシュ.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: エスパー
* 分類: 物理
* 威力: 70（SV基準）→ 90（Champions基準）
* 命中: 90
* PP: 10（SV基準）→ 12（Champions基準。`docs/champions/move_list.txt` 714行目に記載）
* 対象: 相手1体選択
* 優先度: 0
* 接触: あり（直接攻撃）
* オドシシ・アヤシシの専用技

## 効果

* 追加効果として、100%の確率で自分（攻撃側）の『ぼうぎょ』ランクを1段階上げる。
  相手のランクではなく**自分**のランクが上がることに注意。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率90）
* 追加効果: あり（100%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
