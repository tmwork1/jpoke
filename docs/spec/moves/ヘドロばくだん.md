# ヘドロばくだん（Sludge Bomb）仕様書

参照: docs/wiki/moves/ヘドロばくだん.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 特殊
* 威力: 90
* 命中: 100
* PP: 12（`docs/champions/move_list.txt` の値を正とする。Gen9本家基準のPP10は
  championsのPP再編（4段階: 8/12/16/20）により12へ変更されている）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: `bullet`（弾の技。ぼうだんで無効化される）、`secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、30%の確率で相手を『どく』状態にする。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（30%でどく）
* まもる: 防がれる
* おうじゃのしるし: 第二世代: ×、第三世代〜第四世代: ×、第五世代以降: ○
  （`resolve_secondary_chance` 経由で `Event.ON_MODIFY_SECONDARY_CHANCE` により共通処理されるため、
  本技固有の実装は不要）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、
  どくの追加効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）
* ぼうだん: 弾の技のため、ぼうだん特性を持つ相手には無効化される
  （`docs/spec/abilities/ぼうだん.md` 参照。本技固有の実装は不要）

## どく無効の扱い

* どく・はがねタイプへの無効化、めんえき等の特性による無効化は `ailment_manager.apply` 内部で
  共通判定されるため、本技固有の実装は不要。

## Zワザ・ダイマックスわざ

* Zワザ威力: 175（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
