# ないしょばなし（Confide）仕様書

参照: docs/wiki/moves/ないしょばなし.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: 必中（`src/jpoke/data/ps-champ-ja/moves.json` には単独の項目が存在しない。ちょうおんぱ・なきごえ等
  他の一般的な音技も同様に単独項目を欠くため、本プロジェクトはこれを一覧のスクレイピング漏れと判断し、
  第九世代の値（PP20・必中）をそのまま採用する）
* PP: 20（本家シリーズと同じ。Championsの4段階PP規格（8/12/16/20）に元々合致するため変更なし）
* 対象: 相手1体
* 優先度: 0
* 接触: なし
* 音技（`sound` フラグ）

## 効果

* 相手の『とくこう』ランクを1段階下げる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: 必中（`accuracy=None`。ランク補正・回避率に関係なく必ず命中する）
* 追加効果: なし
* まもる: 防がれない（`まもる状態やダイウォールを無視する。`と明記されている。`unprotectable` フラグが必要）
* みがわり: 貫通する（`sound` フラグによる第六世代以降の一般ルール。個別のフラグ付与は不要、
  `move_executor.check_hit_substitute` が sound フラグから自動的に判定する）
* おうじゃのしるし: 対象外（第五世代以降。ひるみ効果を持たないため無関係）
* マジックコート: 反射可能（変化技かつ対象が相手1体のため、category=status/target=foe の
  一般ルールで自動的に反射対象になる）
* よこどり: 対象外（相手を対象とする技のため、よこどりの対象外という一般ルールに従う。未実装）
* オウムがえし: コピー可能

## 一次情報との差異（本プロジェクトのスコープ外）

* トリックガードにより防がれる（本プロジェクト未実装のため対象外）
* ダイウォールを無視する（本プロジェクトはダイマックスを実装していないため対象外）
* 必中技だが、姿を隠している相手（穴を掘っている・宙に浮いている等の溜め技の隠れ状態）には
  当たらない（本プロジェクトの命中判定フレームワークの一般仕様に準拠）
* 特性『かちき』の相手に使用すると差し引きでとくこうが1段階上昇する（`かちき` 特性側の
  一般仕様であり、本技固有のハンドラは不要）

## イベント / 優先度

* `Event.ON_STATUS_HIT`（優先度100・デフォルト）で相手の『とくこう』ランクを1段階下げる。
