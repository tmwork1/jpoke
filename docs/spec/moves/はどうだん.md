# はどうだん（Aura Sphere）仕様書

参照: docs/wiki/moves/はどうだん.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 特殊
* 威力: 80（第六世代以降。第五世代までは90）
* 命中: - （必中）
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 直接攻撃: なし
* フラグ: 弾（bullet）、波動（pulse）

## 効果

* 自分の攻撃は必ず命中する（自分の命中率、相手の回避率に関係なく必ず当たる）。
* 追加効果なし。
* 特性がぼうだんのポケモンには無効化される（弾のわざ）。
* 特性がメガランチャーのポケモンが使うと、威力が1.5倍になる（波動のわざ）。

## 判定フラグ

* 急所: あり
* 命中判定: 必中（命中判定自体が行われない。あなをほる・そらをとぶ・ダイビング・
  シャドーダイブ等の回避状態には、必中であっても命中しない）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* 必中技に分類される技の中では効果抜群を取れるタイプが5種類と最多である
  （他にはあてみなげがある）。
