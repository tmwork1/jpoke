# バレットパンチ（Bullet Punch）仕様書

参照: docs/wiki/moves/バレットパンチ.html（ポケモンWiki）

## 基本情報

* タイプ: はがね
* 分類: 物理
* 威力: 40
* 命中: 100
* PP: 20（本家PPは30。champions では PP が 8/12/16/20 の4段階に統一されているため
  docs/champions/move_list.txt の値を正とする）
* 対象: 相手1体
* 優先度: +1
* 接触: あり（直接攻撃）
* パンチ技: あり（てつのこぶし・パンチグローブ等の対象）

## 効果

* 優先度が高い（先制攻撃技）。追加効果なし。

## 判定フラグ

* 急所: あり（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* パンチ技の1つ。てつのこぶしやパンチグローブの効果対象となる（威力1.2倍）。
* はがねタイプの技の中では最も威力が低い。
* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
