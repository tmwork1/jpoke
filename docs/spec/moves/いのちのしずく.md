一次情報: `docs/wiki/moves/いのちのしずく.html`

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | みず |
| 分類 | 変化 |
| PP | 12（championsルールにより8/12/16/20の4段階へ統一。docs/champions/move_list.txt準拠。Gen9本家は10） |

## 効果

自分と場にいる味方ポケモンの HP を、それぞれ最大 HP の 1/4 ずつ回復する。
**端数は四捨五入で丸める**（一次情報「技の仕様」節）。

- シングルバトルでは「味方全員」＝使用者自身のみが対象になるため、実質「自分の HP を最大HPの
  1/4（四捨五入）回復する」技として振る舞う
- 使用者自身の HP が満タンの場合、技は失敗する
- 同じ 1/4 回復技である `ジャングルヒール` `みかづきのいのり` は端数を **切り捨て** るのに対し、
  本技のみ四捨五入である点に注意（一次情報「技の仕様」節に明記されている既知の差異）

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない。トリックガード・ダイウォールも同様に防がれない） |
| おうじゃのしるし | × |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |

`まもる`・`マジックコート` は `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。

## みずタイプ吸収特性との相互作用

- みずタイプの技であるため、味方（このプロジェクトのシングルバトルでは使用者自身）が
  `よびみず` / `ちょすい` / `かんそうはだ` を持つ場合、通常はこれらの特性が技を無効化して
  発動する対象になり得るが、**使用者自身の特性はよこどりする側の技には反応しない
  （自分自身の技に対して自分自身の吸収特性は発動しない）**ため、シングルバトルでは
  常に技の回復効果がそのまま適用される
- 一次情報によると、使用者の敵が `よびみず` を持っていても、特性に技を引き寄せられることなく
  味方（使用者自身）を回復できる。これは `よびみず` 等の吸収判定が `move.target == "foe"` を
  前提とするため、`target="self"` の本技では敵側の特性が発動しないことで実現される
- `かんそうはだ` の味方に対して本技を使った場合は特性の効果で最大HPの1/4（切り捨て）を回復するが、
  `よびみず` の味方に対して使った場合はHPが回復しない（本プロジェクトのシングルバトルでは
  使用者自身が対象のため、この相互作用は発生しない）

## 変化技に対する防御特性との相互作用

- 変化技であるため、味方が `おうごんのからだ` を持つ場合は技を防がれるが、使用者自身が
  `おうごんのからだ` を持つ場合は自身の特性は発動せず、技の効果で回復できる
  （`target="self"` により、`おうごんのからだ` の判定条件 `move.target == "foe"` を満たさないため）

## 特性 ヒーリングシフト との相互作用

`heal` フラグを持つ回復技のため、使用者が `ヒーリングシフト` を持つ場合は優先度が+3される
（`docs/spec/abilities/ヒーリングシフト.md` 参照。特性側は `heal` フラグを汎用的に参照するため、
本技専用の実装は不要）。

## かいふくふうじとの相互作用

- 使用者が `かいふくふうじ` 状態の場合、`heal` フラグを持つ技として選択できない
  （`かいふくふうじ_try_action` が汎用的に処理するため、本技専用の実装は不要）

## 備考

- 自分1体しか対象がいない（シングルバトル）場合でも、回復量は最大HPの1/4のまま変わらない
- あめ状態などの天候による回復量の変化はない（`あさのひざし` 等とは異なり天候の影響を受けない）

## ポケモンチャンピオンズ向けの簡略化

- ダブルバトル・トリプルバトルでの複数対象への回復処理は対象外（シングルバトル専用実装のため）
