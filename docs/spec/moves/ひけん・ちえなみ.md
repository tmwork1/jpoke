# ひけん・ちえなみ（Ceaseless Edge）仕様書

参照: docs/wiki/moves/ひけん・ちえなみ.html（ポケモンWiki）

## 基本情報

* タイプ: あく
* 分類: 物理
* 威力: 65
* 命中: 90
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* 切る技の1つ（特性「きれあじ」で威力1.5倍）

## 効果

* 追加効果として、相手の場を『まきびし』状態にする（1層追加。最大3層）。
* すでに相手の場にまきびしが3層設置されている場合、追加効果は発動しない（攻撃自体はダメージを与える）。
* 味方を対象に攻撃したときも、相手の場をまきびし状態にする（がんせきアックスと同様）。
* 変化技ではないため、相手の場に特性「マジックミラー」のポケモンがいてもまきびしを設置できる（がんせきアックスと同様）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし。Pokémon LEGENDS アルセウスの技仕様では急所ランク+1だが、本プロジェクトの対象範囲外）
* 命中判定: あり（命中率90）
* 追加効果: あり（100%・まきびし1層追加。ちからずく対象技）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射される

## 追加効果が発動しない場合

* 特性「ちからずく」の使用者が使ったとき（威力は1.3倍になるが、まきびしの追加効果は発生しない）。がんせきアックスと同様の仕様。
* 技がまもる状態で無効化されたとき、または命中しなかったとき。
* すでに相手の場がまきびし3層設置済みのとき。

## 追加効果が発動する場合（りんぷん・おんみつマントの影響を受けない）

* 対象が特性「りんぷん」や、持ち物「おんみつマント」を持っているとき（相手個体ではなく場を変化させる効果のため、これらの特性・持ち物の影響を受けない）。
* 対象を「ひんし」にしたとき。

## 備考

* ダイケンキ（ヒスイのすがた）の専用技。
* 発生する状態（ステルスロックの代わりにまきびし）を除けば、がんせきアックスと同様の効果を持つ。
* まきびしの詳細は `docs/spec/fields/まきびし.md` を参照。
* Zワザ・ダイマックスわざの威力は本プロジェクトの対象範囲外。
