# ドレインパンチ（Drain Punch）仕様書

参照: docs/wiki/moves/ドレインパンチ.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 物理
* 威力: 75（第五世代以降。第四世代までは60）
* 命中: 100
* PP: 12（`docs/champions/move_list.txt` 準拠。本家基準は10（第五世代以降。第四世代までは5））
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* パンチ技（`punch` フラグ）に該当する。特性『てつのこぶし』・道具『パンチグローブ』の効果対象。

## 効果

* 相手に与えたダメージの1/2（50%）だけ、自分のHPを回復する（ドレイン技）。
* 追加効果なし。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代・第五世代以降とも対象）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 技の仕様（ドレイン技共通）

* 回復量は「相手に与えたダメージ × 1/2」で算出する。
  小数点以下は第四世代までは切り捨て、第五世代以降は四捨五入する
  （heal_ratio=0.5 の場合、ダメージは整数のため端数は必ず 0 か 0.5 となるが、
  切り捨てと四捨五入で結果が変わる境界（与ダメが奇数のとき）が存在する。
  例: 与ダメ101 → round_half_up(101 * 0.5) = 51（int() 切り捨てなら50になる）。
  最低1回復を保証する）。
* 道具『おおきなねっこ』を持たせたポケモンが使う場合、回復量は通常の1.3倍になる。
* 特性『ヘドロえき』のポケモンを攻撃した場合、回復効果が同量の反動ダメージに変わる。
* みがわりに命中した場合は、みがわりへのダメージ分を吸収する。
* 特性『てつのこぶし』の時、パンチ技として威力が1.2倍になる。
  道具『パンチグローブ』を持たせている場合も、パンチ技として威力が1.1倍になる
  （代わりに接触判定がなくなる）。
* かくとうタイプで唯一のHP回復技。
* 『かいふくふうじ』状態のポケモンはこの技を選択できず、行動前にかいふくふうじ状態に
  されると失敗する（第六世代以降。第五世代までは技の使用自体はできるがHP回復効果が
  無くなる。本プロジェクトはチャンピオンズ基準のため選択不可・失敗の挙動を採用）。
