# うたかたのアリア（Sparkling Aria）仕様書

参照: docs/wiki/moves/うたかたのアリア.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 特殊
* 威力: 90
* 命中: 100
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` / `docs/progress/move.md` 準拠。本家シリーズのPPは10だが、Championsでは12）
* 対象: 相手1体（本家の範囲は「自分以外」だが、本プロジェクトはシングルバトルのみ対象のため相手1体で問題ない）
* 優先度: 0
* 接触: なし
* フラグ: 音のわざ（ぼうおん・パンクロック・ヴィヴィヨン(音)・ドラゴンエールの対象、みがわりを貫通）

## 効果

* この技でダメージを受けたポケモンの『やけど』状態を治す（追加効果）。やけど以外の状態異常は治さない。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・やけど回復）
* まもる: 防がれる（まもる状態などで技を無効化された相手のやけどは治せない）
* みがわり: 貫通する（音のわざのため。みがわりに当たっても本体のやけどを治す）
* おうじゃのしるし: 対象外（無関係）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外
* オウムがえし: 対象になる
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・やけど回復効果は発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）
* りんぷん・おんみつマント: 対象の特性がりんぷん、または対象がおんみつマントを持つ場合、やけど回復効果が発動しない。
  * 一次情報では「ダブルバトルなどで複数体にダメージを与えた場合は治る」とあるが、本プロジェクトはシングルバトルのみ対象のため、りんぷん・おんみつマント所持時は常に発動しない扱いでよい。
  * `resolve_secondary_chance` が Event.ON_MODIFY_SECONDARY_CHANCE 経由でりんぷん・おんみつマントの無効化ハンドラを呼び出すため、本技側の実装で個別対応する必要はない。
* だっしゅつボタン・レッドカードの効果はやけどを治した後に発動する（優先度で担保。本技の効果はデフォルト優先度100、これらの発動はターン終了に近い後方の処理のため問題なし）。
* とびだすなかみ/うのミサイル・レンブのみ・みちづれの効果で使用者がひんしになった場合、本家シリーズ（スカーレット・バイオレットまで）ではやけど回復効果が発動しないが、Championsでは発動する。
  * 本プロジェクトはChampions基準のため、攻撃側の生死に関わらずやけど回復効果を発動させてよい（使用者の生死をチェックする実装は不要）。

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当。`はたきおとす` 等の実装コメント参照）に
  「100 | うたかたのアリアでやけど回復」と明記されている。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。

## 専用技

* 第七世代ではアシレーヌの専用技。第八世代以降はラプラスなどにタマゴわざとして追加。

## 備考

* うたかた（泡沫）は泡のこと。アリアはオペラなどで歌われる独唱曲のこと。
* きつけ・めざましビンタと違い、やけど状態の相手に威力が上がる効果は無い。
