# 仕様書: まわしげり

## 対象範囲外（実装しない）

本技は第1世代から存在する技だが、**ポケットモンスター ソード・シールド以降（ソード・シールド／BDSP／スカーレット・バイオレット）で使用不可**
（`docs/wiki/moves/まわしげり.html` のカテゴリ「ソード・シールドで使用不可能なわざ」
「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」「スカーレット・バイオレットで使用不可能なわざ」より確認。
本文中の第八世代の説明文も「このわざは つかえません」となっている）。

`docs/champions/move_list.txt`（ポケモンチャンピオンズの技一覧）にも本技のエントリは存在しない。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に該当するため、
本技は実装対象外とする。

過去のレビューにて `data/moves/move_ma.py` に本技の `MoveData`（威力85・命中90・PP10の誤った値）と
`handlers/move_attack.py` の専用ハンドラ、`handlers/item.py` の `_INNATE_FLINCH_MOVES` への登録、
および `tests/moves_attack/test_move_ma.py` / `tests/moves_status/test_move_ma.py`（重複登録されていた）に
テストが存在していたことが判明したため、本レビューで削除した。詳細は `docs/review/moves/まわしげり.md` を参照。
