# りんごさん（Apple Acid）仕様書

参照: docs/wiki/moves/りんごさん.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 特殊
* 威力: 90（`docs/champions/move_list.txt` / `docs/progress/move.md` 準拠。本家シリーズ(SVまで)の威力は80だが、Championsでは90）
* 命中: 100
* PP: 12（`docs/champions/move_list.txt` / `docs/progress/move.md` 準拠。本家シリーズのPPは10だが、Championsでは12）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: ちからずくの対象技となる `secondary_effect`

## 効果

* 追加効果として、100%の確率で相手の『とくぼう』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・とくぼう1段階ダウン）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象になる（本追加効果はひるみ効果ではないため無関係。ただし一次情報の判定表に記載されている）
* マジックコート: 対象外（変化技ではないため）
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、とくぼうダウン効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）
* ノーマルタイプなどに技自体が無効化された場合、追加効果（とくぼうダウン）は発生しない（ダメージが発生しなかった場合はイベントが発火しないため、本技側の実装で個別対応する必要はない）。

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当。`Gのちから`（`handlers/move_attack.py` の `Gのちから_lower_defender_def`）等の同種の能力ランクダウン効果の実装パターン参照）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が行われなかった場合には発動しない仕様を担保する。

## 専用技

* タルップルの専用技。

## 備考

* 威力はそこそこあり、追加効果も優秀。とくこうが高めなタルップルとは相性が良い。
