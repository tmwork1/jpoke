# サイコキネシス（Psychic）仕様書

参照: docs/wiki/moves/サイコキネシス.html（ポケモンWiki）

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: 90
* 命中: 100
* PP: 12（本家基準は10。championsでは全技PPが8/12/16/20の4段階に統一されており、
  docs/champions/move_list.txt の値（12）を正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（間接攻撃）

## 効果

* 追加効果として、10%の確率で相手の『とくぼう』ランクを1段階下げる。（第二世代以降の仕様。
  第一世代では33.2%の確率で『とくしゅ』を下げる仕様だったが、本プロジェクトの対象外）

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率100）
* 追加効果: あり（10%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: コピー可能（特殊技のため）

## 備考

* Zワザ威力: 175（マキシマムサイブレイカー） / 185（オリジンズスーパーノヴァ）。本プロジェクトでは対象外。
* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
