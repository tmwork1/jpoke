# スモッグ（Smog）仕様書

参照: docs/wiki/moves/スモッグ.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 特殊
* 威力: 30（第五世代までは20、第六世代以降30。本プロジェクトは現行仕様のみ対象のため30）
* 命中: 70
* PP: 20（`src/jpoke/data/ps-champ-ja/moves.json` に本技のエントリなし。第九世代基準のPPが既に統一4段階
  （8/12/16/20）のひとつであるため、champions側の変更は無いものと判断し変更しない）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: `secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、40%の確率で相手を『どく』状態にする。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率70）
* 追加効果: あり（40%でどく）
* まもる: 防がれる
* おうじゃのしるし: 第二世代: ×、第三世代〜第四世代: ×、第五世代以降: ○
  （`resolve_secondary_chance` 経由で `Event.ON_MODIFY_SECONDARY_CHANCE` により共通処理されるため、
  本技固有の実装は不要）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、
  どくの追加効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）における
  「追加効果（特殊なもの除く）」（priority=10）に該当する処理だが、10まんボルト・スパーク・ずつき等、
  既存の同種「追加効果付与」ハンドラがすべて `Event.ON_DAMAGE_HIT` + デフォルト priority=100 で
  登録されているため、同一パターンを踏襲し明示的な priority 指定はしない。

## どく無効の扱い

* どく・はがねタイプへの無効化は `ailment_manager.apply` 内部で共通判定されるため、本技固有の実装は不要。

## Zワザ・ダイマックスわざ

* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 70（本プロジェクトでは対象外）。
