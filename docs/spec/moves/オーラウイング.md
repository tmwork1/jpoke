# オーラウイング（Esper Wing）仕様書

参照: docs/wiki/moves/オーラウイング.html（ポケモンWiki）

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: 80
* 命中: 100
* PP: 10（本家）→ 12（Champions、docs/champions/changes_from_sv.md によりPP8/12/16/20の4段階に統一。
  本家PP10の技（10まんばりき・ラスターカノン等）は12に変換されているため、同様に12とする）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 急所ランク: +1（当たりやすい）

## 効果

* 追加効果として、100%の確率で自分の「すばやさ」ランクを1段階上げる
* 急所に当たりやすい（急所ランク+1）

## 判定フラグ

* 急所: あり（急所ランク+1）
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・自分のすばやさ+1）
* まもる: 防がれる
* マジックコート: 反射不可（自分への追加効果のため対象外）
* オウムがえし: 反射可能

## 特記事項

* 追加効果は使用者自身に発動するため、みがわりに命中した場合でも発動する
* ウォーグル（ヒスイのすがた）の専用技（Pokémon LEGENDS アルセウス）。SV/Championsでは専用技制限なし。
* Pokémon LEGENDS アルセウスの「早業・力業」に伴う行動順補正・威力/命中率の変化は本プロジェクトの対象外。
