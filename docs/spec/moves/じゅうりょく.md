## 効果

場を『じゅうりょく』状態にする。詳細な効果（命中率補正・接地化・使用禁止技・
Gのちから威力上昇など）は `docs/spec/fields/じゅうりょく.md` を参照。

継続ターン数は5ターン（発動ターンを含む）。すでに『じゅうりょく』状態が有効な場合、
再度使用しても延長・解除されず技は失敗する（`battle.global_manager.activate` が
`max_count<=1` のため `False` を返す）。

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 命中判定 | × |
| まもる | ×（`守×`。対象が `foe` ではなく全体の場のため、`is_blocked_by_protect` の
  `move.target == "foe"` 条件を満たさず防がれない） |
| マジックコート | ×（対象が `foe`/`foe_side` ではないため `is_reflectable` が False。
  跳ね返されない） |
| よこどり | ×（変化技だが対象が場全体のため対象外） |

`target="field"`（`MoveData.target` のデフォルト値 `"foe"` ではない）を明示しているため、
`is_blocked_by_protect` / `is_reflectable` の判定はいずれも自動的に False になる。
