# オーラぐるま（Aura Wheel）仕様書

参照: docs/wiki/moves/オーラぐるま.html（ポケモンWiki）
参照: docs/spec/moves/_move_secondary_1.md（73行目、追加効果共通仕様）

## 基本情報

* タイプ: でんき（モルペコが『はらぺこもよう』のときは あく に変わる）
* 分類: 物理
* 威力: 110
* 命中: 100
* PP: 12（champions版。本家の10から変換済み。参照: docs/champions/move_list.txt 136行目）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* モルペコの専用技

## 効果

* 追加効果として、100%の確率で自分の『すばやさ』ランクを1段階上げる。
* 使用者（モルペコ）のフォルムによって技のタイプが変わる。
  - まんぷくもよう: でんきタイプ
  - はらぺこもよう: あくタイプ
* 使用者の特性が『ちからずく』の場合、威力が上がる代わりに追加効果（すばやさ上昇）は発動しなくなる。
  タイプがフォルムにより変わる効果はちからずくの影響を受けず、通常通り発動する
  （`flags` に `"secondary_effect"` を設定することで、既存の ちからずく 汎用処理
  `ちからずく_boost` / `ちからずく_disable_secondary_effect` が威力上昇・追加効果無効化を担当する。
  タイプ変更はこの追加効果とは独立した `ON_MODIFY_MOVE_TYPE` ハンドラで処理されるため、
  ちからずくの影響を受けない仕様を自然に満たす）。
* 特性『はらぺこスイッチ』が消えている場合や、テラスタルによりモルペコのフォルムが
  変化しなくなった場合であっても、オーラぐるまのタイプは現在のモルペコのフォルム
  （`is_hangry` フラグ）に応じて変わる。これは特性の発動有無に依存しない、技自体に
  組み込まれた仕様である。
* 使用者が『そうでん』状態のときは、はらぺこもようであっても でんきタイプ になる
  （`そうでん_move_type` が `Event.ON_MODIFY_MOVE_TYPE` priority=110 で本技のタイプ判定より
  後に評価され、でんきタイプへの変換を優先する。実装済み・共通処理のため本技での追加対応は不要）。
* 特性『きけんよち』は、はらぺこもようのモルペコに対しても、オーラぐるまを でんきタイプ の
  技とみなす（相手から見た警戒表示の仕様であり、本プロジェクトの対象範囲外）。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率100）
* 追加効果判定: あり（100%）
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
