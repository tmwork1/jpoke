# 仕様: ルミナコリジョン

一次情報: `docs/wiki/moves/ルミナコリジョン.html`

## 基本情報

- タイプ: エスパー
- 分類: 特殊
- 威力: 80
- 命中率: 100
- PP: 12（`docs/champions/move_list.txt` 準拠。Gen9本家値は10だが、チャンピオンズは
  全技PPを8/12/16/20の4段階に統一しているため12が正）
- 対象: 1体選択（本プロジェクトはシングルバトルのみのため `defender` 単体固定）
- 優先度: 0
- 直接攻撃: ×（接触技ではない）

## 効果

追加効果として、100%の確率で相手の『とくぼう』ランクを2段階下げる。

## 判定

- 急所: ○
- 命中判定: ○
- 追加効果: ○
- まもる: ○（まもるで防がれる）
- おうじゃのしるし: ○（第五世代以降、ひるみ追加効果を持つ技の対象になる特性だが本技はひるみ
  効果を持たないため実質影響なし）
- マジックコート: ×（攻撃技の追加効果はマジックコートで跳ね返らない。攻撃技共通の一般仕様であり
  ルミナコリジョン固有の特記事項ではない）

## その他

- クエスパトラの専用技。
- 100%発動の追加効果を持つため、`secondary_effect` フラグを付与する必要がある
  （`ちからずく` の威力1.3倍・追加効果無効化の対象になる）。
- 追加効果の対象・確率は `docs/spec/moves/_move_secondary_1.md` にも掲載済み
  （対象=defender、確率=1、D列=-2）。
