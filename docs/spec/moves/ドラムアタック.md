# ドラムアタック（Drum Beating）仕様書

参照: docs/wiki/moves/ドラムアタック.html（ポケモンWiki）

## 基本情報

* タイプ: くさ
* 分類: 物理
* 威力: 80
* 命中: 100
* PP: 10（第八世代・第九世代共通。第九世代の技一覧に掲載されており使用可能。BDSPのみ使用不可）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない。ドラムの根っこによる攻撃のため音技にも分類されない）
* ゴリランダーの専用技（進化時に習得）

## 効果

* 追加効果として、100%の確率で相手の『すばやさ』ランクを1段階下げる。
  （自分ではなく相手のランクが下がることに注意。「ドラムの根っこをドラミングでコントロールして
  攻撃することで相手のすばやさを下げる」という説明文の通り。）

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率100）
* 追加効果: あり（100%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
