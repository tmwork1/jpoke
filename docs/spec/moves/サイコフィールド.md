# サイコフィールド 仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | エスパー |
| 分類 | 変化 |
| PP | 12 |
| 命中率 | - |
| 範囲 | 全体の場 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

**5ターン** の間、地形を「サイコフィールド」にする。サイコフィールドの間、「ひこう」タイプや特性「ふゆう」など
地面にいないポケモンを除くすべてのポケモンは、相手（味方は含まない）の優先度+1以上の技を受けなくなり、また
「エスパー」タイプの技の威力が1.3倍になる。すでにサイコフィールドが発動している場合に使用すると失敗する
（エレキフィールド・グラスフィールド・ミストフィールドが発動している場合は上書きする）。

サイコフィールドそのものの詳細仕様（威力補正・先制技無効化・カウントダウン・グランドコートによる
延長など）は `docs/spec/fields/サイコフィールド.md` を参照。

## 判定

- 急所: なし（変化技）
- 命中判定: なし（命中率 `-` の技）
- 追加効果: なし（本体の効果がフィールド設定そのもの）
- まもるで防がれる: ×（範囲が「全体の場」のため対象を取らず、まもるの影響を受けない）
- マジックコートで跳ね返される: ×
- よこどりで奪われる: ×（フィールド系の変化技は例外的によこどり対象外）
- オウムがえしでコピーされる: ×（相手に向けて撃つ技ではないため）

## ポケモンチャンピオンズ向けの簡略化

- コンテスト効果（アピール等）は実装しない
- Zワザによる付加効果（自分のとくこうが1段階上がる）は実装しない（Zワザ非対応）
- ポケモン不思議のダンジョン・ポケモンクエスト・ポケモンマスターズ固有の仕様は対象外
- PPは `src/jpoke/data/ps-champ-ja/moves.json` の値（12）を採用する
  （Gen9本家の10から8/12/16/20の4段階統一により変更されている）
