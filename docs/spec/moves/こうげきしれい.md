# こうげきしれい（Attack Order）仕様書

参照: docs/wiki/moves/こうげきしれい.html（ポケモンWiki）

## 基本情報

* タイプ: むし
* 分類: 物理
* 威力: 90
* 命中: 100
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 急所ランク: +1（急所に当たりやすい）

## 効果

* 急所に当たりやすい。追加効果なし。

## 判定フラグ

* 急所: 高確率（急所ランク+1）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第四世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 175（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
