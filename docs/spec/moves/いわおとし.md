# いわおとし（Rock Throw）仕様書

参照: docs/wiki/moves/いわおとし.html（ポケモンWiki）

## 基本情報

* タイプ: いわ
* 分類: 物理
* 威力: 50
* 命中: 90（第一世代のみ65。第二世代以降90）
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 通常攻撃。追加効果なし。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率90）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第二世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 100（本プロジェクトでは対象外）。
