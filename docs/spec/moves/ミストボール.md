# ミストボール（Mist Ball）仕様書

参照: docs/wiki/moves/ミストボール.html（ポケモンWiki）

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: 70（第八世代まで。第九世代以降95）
* 命中: 100
* PP: 5（`src/jpoke/data/ps-champ-ja/moves.json` に記載なし。Championsの技リストに存在しない場合は
  本家（Gen9）のPPをそのまま採用する方針のため5のまま）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない。弾のわざに分類される）

## 効果

* 追加効果として、50％の確率で相手の『とくこう』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（50%・とくこう-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。第三〜四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射される
* 弾のわざ: ○（特性『ぼうだん』所持者には無効化される）

## 備考

* ラティアスの専用技。
* Zワザ威力: 140（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 120（本プロジェクトでは対象外）。
