# ジャイロボール（Gyro Ball）仕様書

参照: docs/wiki/moves/ジャイロボール.html（ポケモンWiki）
参照: docs/spec/moves/_variable_power.md（変動威力技共通仕様、第6節「速度差依存威力技」）

## 基本情報

* タイプ: はがね
* 分類: 物理
* 威力: 可変（1〜150）
* 命中: 100
* PP: 8（本家基準5。champions は全技PP 8/12/16/20 の4段階に統一されており、
  `src/jpoke/data/ps-champ-ja/moves.json` の値を正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: あり
* フラグ: 弾のわざ（ぼうだん所持ポケモンに無効化される）

## 効果

* 自分（使用者）が相手より実効素早さが遅いほど、威力が大きくなる。
* 威力 = floor(25 × 相手の実効素早さ ÷ 自分の実効素早さ) + 1
* ただし最大威力は150（自分の実効素早さが相手の約5.96倍以下＝比率が約1/5.96以下のとき150に到達）。
* 自分の実効素早さが0の場合はゼロ除算を防ぐ例外処理があり、第六世代以降は威力を1として扱う
  （第五世代までは使用者の素早さを1として計算する旧仕様だが、本プロジェクトは第六世代以降のみ対象のため
  「威力1固定」を採用する）。
* 実効素早さは、両者のランク補正・特性・もちもの・まひ・おいかぜ・しつげんの影響を受ける。
* トリックルーム状態でも威力は変化しない（トリックルームは行動順序のみを反転させ、
  実際の素早さ数値自体は変化しないため）。
* Zワザ・ダイマックスわざにしたときの威力は固定（Zワザ160、ダイマックスわざ130）で、
  両者の素早さとは無関係（本プロジェクトはZワザ・ダイマックスわざが対象外のため実装不要）。

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: ひるみ効果あり（第四世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能（本プロジェクトはオウムがえし自体が未実装のため対応不要）
* ぼうだん: 無効化される（弾のわざ）

## リーサル計算

* 威力は使用者・対象双方の実効素早さに依存するが、ダメージ計算時点で確定する威力の変動要素であり、
  相手HPの分布（`StateDist`）を直接左右する要素ではないため、`lethal.py` に専用ハンドラは不要。
  （`docs/progress/move.md` のリーサル実装/リーサルテスト列は n/a）
