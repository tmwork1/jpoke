# フェイタルクロー（Dire Claw）仕様書

参照: docs/wiki/moves/フェイタルクロー.html（ポケモンWiki）

## 基本情報

* タイプ: どく
* 分類: 物理
* 威力: 80
* 命中: 100
* PP: 16（`docs/champions/move_list.txt` 準拠。スカーレット・バイオレット基準は15だが、
  Champions では全技PPが8/12/16/20の4段階に統一されているため16が正）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: 切る技（きれあじの対象。Champions で追加された特徴）

## 効果

* 追加効果として、一定確率で相手を「どく」「まひ」「ねむり」のいずれかの状態にする
  （どの状態になるかは均等に各1/3の確率で決まる）。
  * スカーレット・バイオレット: 発動確率50%
  * **Champions: 発動確率30%**（`docs/plan/archives/champions_migration.md` に記載の変更点）
* 全体を通すと、どく・まひ・ねむりのいずれか1つが選ばれる確率は Champions では約10%
  （30% × 1/3）。

## 追加効果の判定仕様

* 発動確率判定とどの状態異常になるかの選択は、まとめて単一の乱数で処理される
  （`r < base/3` → どく、`base/3 <= r < base*2/3` → まひ、`base*2/3 <= r < base` → ねむり、
  `r >= base` → 不発）。`base` は `battle.resolve_secondary_chance` で
  てんのめぐみ・にじ状態（2倍）やちからずく（無効化）などの補正を受けた後の実効確率。
* 選ばれた状態異常に対象が耐性を持つ場合（どく・はがねタイプへのどく、でんきタイプへのまひ、
  ねむり無効特性等）、その状態異常のみが無効化される（付与判定自体はまとめて1回であり、
  無効化されても再抽選はしない）。これらの無効化は `ailment_manager.apply` 内部の
  タイプ・特性チェックで処理されるため、本技のハンドラ側に重複ガードは不要。
* 使用者特性りんぷん・持ち物おんみつマントを対象が持つ場合、追加効果自体が発動しなくなる
  （`resolve_secondary_chance` 内の `ON_MODIFY_SECONDARY_CHANCE` ハンドラで処理）。

## 急所ランク補正について（注意）

* 「急所に当たりやすい（急所ランク+1、力業では+2）」という追加効果は
  **Pokémon LEGENDS アルセウス限定**の仕様であり、スカーレット・バイオレットおよび
  Champions には存在しない。
* スカーレット・バイオレットの Ver.1.3.2 以前の説明文には「急所を狙って攻撃」という記載が
  あったが、これは説明文の誤りであることが公式に告知され、Ver.2.0.1 以降で
  「急所を狙って」の記述が削除された（実際の急所ランク補正は元から存在しない）。
* 本プロジェクト（Champions シングルバトル対象）では急所ランク補正なし
  （`critical_rank` は設定しない = デフォルト0）が正しい。

## 判定フラグ

* 急所: 通常（補正なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（判定仕様は上記参照）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）

## 特性きれあじとの関係

* Champions では「切る技」の1つに分類され、特性きれあじを持つポケモンが使用すると
  威力が1.5倍になる（`きれあじ_modify_power` ハンドラが `ctx.move.has_flag("slash")` で
  判定するため、MoveData の `flags` に `"slash"` を含める必要がある）。
* スカーレット・バイオレットでは切る技に含まれない（Champions で新規に追加された特徴）。

## 専用技

* オオニューラの専用技（進化時習得）。

## 備考

* 特性どくしゅを持つオオニューラがこの技を使用した場合、フェイタルクローとどくしゅの
  発動判定はフェイタルクローが優先される。そのためフェイタルクローの判定で状態異常となり
  ラムのみ等で一度回復しても、どくしゅの判定で改めてどく状態にすることができる
  （どくしゅ側の仕様であり、本技側の実装には影響しない）。
