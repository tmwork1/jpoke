# ぶんまわす（Brutal Swing）仕様書

参照: docs/wiki/moves/ぶんまわす.html（ポケモンWiki）

## 基本情報

* タイプ: あく
* 分類: 物理
* 威力: 60
* 命中: 100
* PP: 20
* 対象: 自分以外の全員（本プロジェクトはシングルバトルのみを対象とするため、実質的に相手1体）
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 通常攻撃。追加効果なし。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 120（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 110（本プロジェクトでは対象外）。
* ダブルバトル・バトルロイヤルでは自分以外の全ポケモンを攻撃するが、
  本プロジェクトの対象範囲（シングルバトルのみ）では相手1体が対象となるため、
  通常の単体攻撃技と挙動は変わらない。
