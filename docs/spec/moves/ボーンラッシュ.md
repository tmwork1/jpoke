# ボーンラッシュ（Bone Rush）仕様書

参照: docs/wiki/moves/ボーンラッシュ.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: じめん
* 分類: 物理
* 威力: 25（SVまで）→ **30**（Pokémon Champions。本プロジェクトの対象範囲）
* 命中: 80（第四世代まで）→ **90**（第五世代以降。本プロジェクトの対象範囲）
* PP: 10（Gen9本家）→ **12**（Pokémon Champions。`docs/champions/move_list.txt` 準拠）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃×。手に持ったホネで攻撃するが、接触技には分類されない）
* フラグ: 特になし（弾・パンチ・切る技等の対象外）

## 効果

* 2～5回連続で攻撃する連続攻撃技（連続攻撃技の可変回数タイプ）。
  * 命中判定は1発目のみ行う（`check_hit_each_time` は使用しない）。
  * 各ヒットの威力は固定で30（Champions基準。`power_sequence` なし）。
* 追加効果なし。

## 判定フラグ

* 急所: 各ヒットごとに独立して判定（急所ランク補正なし）
* 命中判定: あり（命中率90、1発目のみ判定）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（ヒットごとに追加効果判定の対象、第二世代〜全世代で有効）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 備考

* 最大5回ヒットした場合の実質威力は150（Champions基準）相当。
* Zワザ威力: 140、ダイマックスわざ威力: 130（本プロジェクトでは対象外）。
* `docs/spec/moves/連続技.md` の「9.3 2-5回攻撃技」に本技が掲載されているが、そちらはGen9本家基準の
  数値（威力25・命中90・PP10）であり、Champions基準の本仕様書の数値（威力30・PP12）とは異なる。
  実装・進捗表はChampions基準（本仕様書）を正とする。
