# 仕様書: ハードローラー

## 対象範囲外（実装しない）

本技は第五世代で登場した技だが、**ポケットモンスター ソード・シールド以降（ソード・シールド／BDSP／スカーレット・バイオレット）で使用不可**
（`docs/wiki/moves/ハードローラー.html` の `wgCategories` に「ソード・シールドで使用不可能なわざ」
「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」「スカーレット・バイオレットで使用不可能なわざ」が
含まれることより確認。本文中の第八世代の説明文も「このわざは つかえません」となっている）。

`src/jpoke/data/ps-champ-ja/moves.json`（ポケモンチャンピオンズの技一覧）にも本技のエントリは存在しない
（`ハードプレス` と `ばかぢから` の間にエントリなし）。

README.md の「対象範囲」の定義「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」に該当するため、
本技は実装対象外とする。

過去のレビューにて `data/moves/move_ha.py` に本技の `MoveData`
（タイプ:ノーマル・威力100・命中95・PP10という誤った値）と `handlers/move_attack.py` の専用ハンドラ、
`handlers/item.py` の `_INNATE_FLINCH_MOVES` への登録、
`types/move.py` の `MoveName` Literal への登録、
`tests/moves_attack/test_move_ha.py` / `tests/moves_status/test_move_ha.py`（重複登録されていた）に
テストが存在していたことが判明したため、本レビューで削除した。詳細は `docs/review/moves/ハードローラー.md` を参照。

## 参考: 第5〜7世代時点のデータ（実装しないが記録として残す）

| 項目 | 値 |
|---|---|
| タイプ | むし |
| 分類 | 物理 |

| 命中率 | 100 |
| PP | 20 |

| 追加効果 | 30%の確率で相手をひるませる |

（出典: `docs/wiki/moves/ハードローラー.html`）

`MoveData(type="ノーマル", power=100, accuracy=95, pp=10)` という過去の誤った登録値は、実際のゲームデータ
（むしタイプ・威力65・命中100・PP20）と一致しない。急所ランクを上げる効果は一次情報上どこにも記載がなく、
本技は高急所率技ではない（過去の実装にも急所ランク補正は含まれていなかった）。
