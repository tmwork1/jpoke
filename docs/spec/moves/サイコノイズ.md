# サイコノイズ（Psychic Noise）仕様書

参照: docs/wiki/moves/サイコノイズ.html（ポケモンWiki）、src/jpoke/data/ps-champ-ja/moves.json

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: 75
* 命中: 100
* PP: 12（Champions基準。Gen9本家は10）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 音の技（ラベル `sound`）

## 効果

* 追加効果（発動確率100%）として、命中した相手を『かいふくふうじ』揮発性状態にし、
  2ターンの間、技・特性・持ち物による HP 回復を一切できなくする。
  * かいふくふうじ状態の詳細仕様（対象外の回復・解除条件など）は
    `docs/spec/volatiles/かいふくふうじ.md` を参照する。
* 音の技であるため、`ちょうはつ` 中でも使用でき、`みがわり` 状態を貫通して
  本体にダメージ・追加効果を与える（`sound` フラグによる汎用処理、
  `MoveExecutor.check_hit_substitute` を参照）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（100%）
  * 特性りんぷん・持ち物おんみつマントを持つ相手には発動しない（`resolve_secondary_chance` 経由）。
  * 特性ちからずくのポケモンが使用すると威力が1.3倍になるが、かいふくふうじ状態にできない
    （`secondary_effect` フラグによる汎用処理）。
  * 特性アロマベールを持つ相手（および味方）にはかいふくふうじ状態を付与できない
    （`アロマベール_prevent_volatile` の対象に「かいふくふうじ」が含まれる）。
  * 対象がみがわり状態でも、音の技のため貫通してかいふくふうじ状態にできる。
  * まもる・みがわり以外の技・効果で無効化された場合はかいふくふうじ状態にできない。
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。ただしこの技はもともと追加効果を持つため実質影響なし）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）

## 特殊な相互作用（備考）

* この技のダメージでオボンのみ・混乱きのみ・きのみジュースの発動圏に入ったとしても、
  かいふくふうじ状態が発動すればこれらの持ち物は発動しない
  （かいふくふうじが解除されるまで発動しない）。
* この技で効果バツグンのダメージを与えたときでも、かいふくふうじ状態が発動すれば
  相手のナゾのみは発動しない。かいふくふうじ解除後も、再び弱点のダメージを受けるまで
  ナゾのみは発動しない。
* 特性ほおぶくろのポケモンが持つウタンのみを発動させた場合、かいふくふうじ状態の効果が
  発動した後にきのみが発動したことを表すメッセージが流れるが、ほおぶくろによる回復は
  かいふくふうじによって防がれる（`docs/spec/abilities/ほおぶくろ.md` 参照）。
