# いやしのねがい 仕様書

参照: docs/wiki/moves/いやしのねがい.html（ポケモンWiki）

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | エスパー |
| 分類 | 変化 |
| PP | 12（チャンピオンズ仕様。本家 SV 基準は 10） |
| 命中率 | なし（必中・命中判定自体が発生しない） |
| 範囲 | 自分 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

- 自分が **ひんし** になる代わりに、次に出てくる自分のポケモンの **HP** と **状態異常** を回復する。
- `みかづきのまい` と異なり **PP は回復しない**。
- 控えのポケモンが（生存した状態で）いないときに使用した場合は **失敗** し、使用者はひんしにならない。

## 発動タイミング（第五世代以降。チャンピオンズもこの仕様に準拠）

- 技が成功した場合、使用者は**そのターンの終了時**にひんしになり、死に出し交代が発生する（`ON_STATUS_HIT` で即座に `battle.faint` を呼び、既存の瀕死処理フローに乗せる）。
- 回復効果は自陣営に「いやしのねがい」状態（保留された回復効果）として設置され、**次にその位置に出てきたポケモンが場に出た瞬間**（`ON_SWITCH_IN`）に発動する。
  - 出てきたポケモンの HP が満タンかつ状態異常もない場合、効果は発動せず状態は保留され続ける（第八世代以降の仕様）。
  - HP が満タンでも状態異常があれば発動する（状態異常のみ回復）。
  - こんらんなどの「状態変化」（`VolatileName`）は回復対象に含まれない。状態異常（`AilmentName`）のみが対象。
- 場に設置技（ステルスロックなど）がある場合、設置技の効果を受けるより前に回復効果が発動する。
- 1 陣営に「いやしのねがい」状態は重複しない。

## 判定

- 命中判定: なし（対象を持たない自分専用技）
- まもる: ×（自分を対象とする技のため、相手の `まもる` では防がれない）
- ちょうはつ: 変化技のため使用不可
- よこどり: 第五世代以降は奪われる対象になる（本プロジェクトでは未実装の場合は対象外）
- オウムがえし: ×
- マジックコート: ×
- おうじゃのしるし等の追加効果関連特性: 該当なし（追加効果を持たない技）

## ポケモンチャンピオンズ向けの簡略化

- ダブルバトル・トリプルバトル固有の仕様（トリプルバトルの MOVE 移動時の扱い、サイドチェンジとの相互作用等）は対象外（本プロジェクトはシングルバトルのみ）。
- コンテスト・アピール関連の効果は実装しない。
- Z ワザ関連効果は実装しない（付加効果なし）。
