# ナイトバースト（Night Daze）仕様書

参照: docs/wiki/moves/ナイトバースト.html（ポケモンWiki）、src/jpoke/data/ps-champ-ja/moves.json

## 基本情報

* タイプ: あく
* 分類: 特殊
* 威力: 90（`src/jpoke/data/ps-champ-ja/moves.json` の値を採用。ポケモンWiki記載の本家SVまでの値は85だが、
  Pokémon Champions で90に変更されているため、本プロジェクトでは90を採用する）
* 命中: 95
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` の値を採用。ポケモンWiki記載の本家シリーズ値は10だが、
  本プロジェクトはポケモンチャンピオンズを対象とするため `src/jpoke/data/ps-champ-ja/moves.json` の値を優先する）
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: `secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、40%の確率で相手の命中率を1段階下げる。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率95）
* 追加効果: あり（40%・命中率ランク-1）
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 第五世代以降は対象（本技は元々ひるみの追加効果を持つ技や
  一撃必殺技ではないため、通常のダメージ技として10%のひるみ追加抽選の対象になる）。
  この10%ひるみ抽選は命中率を下げる本技本来の追加効果（40%）とは独立した別処理であり、
  実装上は `handlers/item.py` の `flinch_on_hit_10pct` が担当するため、本技側の実装への影響はない
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍になる代わりに、
  命中率を下げる追加効果が発動しなくなる。`resolve_secondary_chance` 経由で処理され、本技側の実装への影響はない）

## イベント / 優先度

* `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上の `Event.ON_DAMAGE_HIT` に相当）に準じ、
  ダメージが実際に適用された後に発動させる。優先度100（デフォルト）、イベントは `Event.ON_DAMAGE_HIT` を使用する。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ計算自体が
    行われなかった場合には発動しない仕様を担保する。

## リーサル計算

対象外（命中率ランク低下のみでHPに影響しないため、`docs/progress/move.md` の「リーサル実装」列は `n/a`）。
