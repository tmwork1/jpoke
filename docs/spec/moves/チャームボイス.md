# チャームボイス（Disarming Voice）仕様書

参照: docs/wiki/moves/チャームボイス.html（ポケモンWiki）

## 基本情報

* タイプ: フェアリー
* 分類: 特殊
* 威力: 40
* 命中: - （必中）
* PP: 16（チャンピオンズ換算。Gen9本家PP15を8/12/16/20の4段階に変換した値。
  同じGen9PP15の技（うたう・アイアンヘッド・ムーンフォース等）がPP16に統一されている実例に準拠。
  ※ src/jpoke/data/ps-champ-ja/moves.json には本技の行が欠落しているため、他の同PP帯の技から類推した値）
* 対象: 相手1体（シングル）
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* フラグ: 音技（音のわざ）

## 効果

* 自分の攻撃は必ず命中する（自分の命中率、相手の回避率に関係なく必ず当たる）。
* 追加効果なし。

## 判定フラグ

* 急所: あり
* 命中判定: 必中（命中判定自体が行われない）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能

## 技の仕様

* 音のわざの1つ。
* 必中技だが、まもる・あなをほる・そらをとぶ・ダイビング・シャドーダイブ状態などの
  姿を隠している相手には命中しない。
* 音技のため、みがわり状態を貫通してダメージを与える（第六世代以降の仕様）。
