# てだすけ（Helping Hand）仕様書

参照: docs/wiki/moves/てだすけ.html（ポケモンWiki）
参照: docs/champions/move_list.txt L508-509

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -（必中、命中判定なし）
* PP: 20
* 対象: 味方1体（第三世代では自分。第四世代以降は味方1体選択）
* 優先度: +5（全ての技の中で最も優先度が高い）
* 接触: なし（直接攻撃×） / まもる: 効果なし（守備側の技ではない） / おうじゃのしるし: 効果なし
  （第三〜四世代・第五世代以降ともに×） / マジックコート: 反射不可 / よこどり: 不可 /
  オウムがえし: コピー不可 / 追加効果判定: なし（急所判定なし）

## 効果（本来のダブル/トリプルバトルでの効果）

* 1ターンの間、対象の味方を「てだすけ」状態（volatile）にし、対象がそのターンに繰り出す技の
  威力を1.5倍にする。
* 対象が行動する前に、てだすけを使用したポケモンがひんしになった場合や場から去った場合でも、
  てだすけ状態の効果は残る。
* 対象がすでにてだすけ状態になっているときも成功する。1ターンで同じポケモンに複数回てだすけを
  使用できれば、その分威力補正は累積する（トリプルバトルで両端のポケモンが中央のポケモンに
  使用した場合や、さいはいで2回使用した場合に起こり得る）。ただしテラレイドバトルでは、
  同じポケモンに複数回使用しても威力は1.5倍のままで累積しない。
* 失敗条件:
  * 対象の味方が、そのターンにすでに行動していた場合は失敗する（さいはい／おどりこの効果で
    行動していた場合も同様）。
  * 対象が交代したターンに使用した場合、第四世代では失敗するが、第三世代・第五世代以降では成功する。
* 無効化:
  * 特性「おうごんのからだ」を持つ対象には無効。
  * トリックガードの後に行動した場合、てだすけはトリックガードに防がれる。
* 対象がみがわり状態でも成功する。対象があなをほる／そらをとぶ／ダイビング／シャドーダイブ状態で
  姿を隠していても基本的に成功する（ダイヤモンド・パールのみ、姿を隠している味方に使うと失敗する）。
* ゆびをふる／ねこのてでは選ばれない。オウムがえし／まねっこではコピーできない。
* Zワザによる付加効果: 自分の下がった能力ランクを元に戻す（本プロジェクトの対象外）。
* 単体で完結する優先度+5の変化技であり、味方の攻撃力を底上げするダブル/トリプルバトル専用の
  サポート技。公式解説でも「ダブルバトル以上において、自分より味方の技の方が有効になる状況で
  使える」と説明されており、シングルバトルでは味方が存在しないため意味を持たない。

## シングルバトルでの扱い（本プロジェクトのスコープ）

* 本プロジェクトの対象はポケモンチャンピオンズのシングルバトルのみであり、`Battle` は常に
  各プレイヤー1体ずつの場（`battle.actives` は常に2体以下）しか扱わない。
* シングルバトルには「対象にできる味方」がそもそも存在しないため、てだすけの核心効果
  （味方の技の威力を1.5倍にする）を発動させることができない。ポケモンWikiにも
  「シングルバトルでもてだすけを選択することはできるが、失敗する」と明記されている。
* この「常に失敗する」という性質は、`docs/spec/turn.md` に失敗条件チェックを実装したとしても
  シングルバトル上で観測可能な効果を生まない。類似技「さきおくり」「おさきにどうぞ」
  「このゆびとまれ」「いかりのこな」等、同様に対象リダイレクト・行動順操作・味方支援を
  主目的とするダブル/トリプル専有技と同じ扱いとし、`docs/progress/move.md` で本技を全列
  「ダブル専用」（対象外）として扱う。

## 実装方針

* 通常のダメージ計算を伴わない変化技として、追加効果なし（`handlers={}`）で実装する。
* 味方への威力補正（てだすけ状態の付与）はダブル/トリプル専有機能のため未実装のままとする。
* `MoveTarget`（`src/jpoke/types/literals.py`）には「味方」を表す選択肢が存在しない
  （`foe` / `foe_side` / `own_side` / `self` / `field` のみ）ため、`target` は指定せず
  既定値の `foe` のまま据え置く（`docs/progress/move.md` の対象列で「defender」と表記される
  他のダブル専用・味方対象技と同じ扱い）。
