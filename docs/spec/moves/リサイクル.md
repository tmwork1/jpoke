# リサイクル（Recycle）仕様書

参照: docs/wiki/moves/リサイクル.html（ポケモンWiki）
       docs/spec/items/ふうせん.md（割れたふうせんの復元対象外規定）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -（命中判定なし）
* PP: 12（第九世代本家は10。本プロジェクトはチャンピオンズ仕様に基づき
  docs/champions/move_list.txt の値を採用）
* 対象: 自分
* 優先度: 0
* 接触: なし

## 効果

* そのバトルで、自分が最後に消費した（失った）持ち物を復活させる。

## 失敗条件

* 使用者がすでに持ち物を持っている場合は失敗する。
* 使用者がそのバトルで一度も持ち物を失ったことがない場合は失敗する。

## 持ち物を消費したかどうかの判定（復元対象になるかどうか）

* 復元対象になるもの:
  * 通常のきのみ消費（HP割合トリガー・状態異常治療等）。
  * なげつけるを使用して自分の持ち物を消費した場合。
  * しぜんのめぐみを使用して消費したきのみ。
  * おちゃかい／ほおばるの効果で消費したきのみ。
* 復元対象にならないもの:
  * 使用者以外のポケモンが消費した持ち物（第五世代以降。第三・四世代は
    「使用者の元の位置で最後に消費された持ち物」が対象になる仕様だったため、
    ダブルバトルで交代した場合などに挙動が異なったが、本プロジェクトは
    第五世代以降の仕様を採用する）。
  * どろぼう・ほしがる・トリック・すりかえ・ギフトパス、特性マジシャン・
    わるいてぐせ・ものひろい・きょうせいの効果で他のポケモンの持ち物になった場合
    （場に存在したまま持ち主が変わるだけで、消費されたことにならない）。
  * はたきおとす・やきつくす・ふしょくガスにより失った持ち物
    （場に存在したまま消滅する扱いのため）。ただし、その後トリック等で
    新しい持ち物を入手し、それを消費した場合は新しい方が復元対象になる。
  * 相手からなげつけるを受けて効果を得たきのみ等（自分の持ち物として
    消費した扱いにならない）。
  * 相手からついばむ／むしくいを受けて奪われたきのみ（第五世代以降。
    第四世代までは復元対象になったが、本プロジェクトは第五世代以降の仕様を採用する）。
  * ついばむ／むしくいを使用して奪って消費したきのみ（奪って得た持ち物を
    消費しても、自分がもともと持っていた持ち物を失った扱いにはならない）。
  * 割れたふうせん（攻撃を受けて割れた場合。なげつけるでふうせんを消費した場合は
    復元対象になる）。
  * 他のポケモンの持ち物になったくっつきバリ。

## 他の効果との関係

* 持ち物の利用を制限する状態・特性を受けていても、持ち物の復活自体は制限されない。
  * さしおさえ・かいふくふうじ状態でも失敗しない。
  * 特性ぶきようでも失敗しない。
  * 場がマジックルーム状態でも失敗しない（復活した持ち物の効果自体は
    マジックルームで無効化される）。
  * 特性きんちょうかんの効果を受けていても、きのみを復活させることはできる
    （きんちょうかんはきのみの発動条件を無視させない特性であり、リサイクルによる
    復活自体には無関係）。
* 持ち物を消費してから経過したターン数に関係なく成功する。消費後に交代や
  ひんし・復帰が挟まれていても成功する。
* 取り戻した持ち物が即時発動条件を満たす場合はその場で発動する。

## 判定フラグ

* 急所: 対象外（変化技のため）
* 命中判定: なし（必ず命中判定を介さず成否が決まる）
* 追加効果: なし
* まもる: 防がれる
* マジックコート: 反射不可
* よこどり: 奪われる（第四世代までは奪われなかったが、第五世代以降は奪われるようになった。
  本プロジェクトは第五世代以降の仕様を採用する。なお本プロジェクトはよこどり自体を
  未実装のため実装上の対象外）
* オウムがえし: 反射不可

## 備考

* Zワザにすることで自分のすばやさランクが2段階上がる付加効果を得られるが、
  そのためには自身の持ち物が消費されないZクリスタルである必要があるため、
  リサイクル自体は必ず失敗する（本プロジェクトはZワザ非対応のため対象外）。
