# 仕様書: リフレッシュ

調査日: 2026-07-10
参照URL: docs/wiki/moves/リフレッシュ.html

## 対象範囲外の判断（重要）

一次情報（`docs/wiki/moves/リフレッシュ.html`）を確認したところ、本技は以下のように
「ソード・シールド」「BDSP（ブリリアントダイヤモンド・シャイニングパール）」
「スカーレット・バイオレット」のいずれでも**使用不可能なわざ**に分類されている
（wgCategories に「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・
シャイニングパールで使用不可能なわざ」「スカーレット・バイオレットで使用不可能なわざ」
が含まれる。またページ冒頭の技情報ボックスにも「ソード・シールドで使用不可」
「BDSPで使用不可」「SVで使用不可」と明記されている）。

第八世代（ソード・シールド）以降、本技を習得できるポケモンが存在しないため実質的に
使用不可能であり、第9世代（スカーレット・バイオレット）でも復活していない。
`docs/champions/move_list.txt`（ポケモンチャンピオンズの全技一覧）にも
「リフレッシュ」の項目が存在しないことを確認し、ポケモンチャンピオンズ側でも
採用されていないという二重の根拠を得た。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは
実装しない」）に該当するため、**本技は実装対象外と判断した**
（前例: `docs/review/moves/オクタンほう.md`, `docs/review/moves/グラスミキサー.md`,
`docs/review/moves/シャドーボーン.md`, `docs/review/moves/シグナルビーム.md`,
`docs/review/moves/ふわふわフォール.md` 等と同様の判断基準）。

対象外と判断したが、参考情報として過去世代の仕様を記録する。

- 効果は「どく・もうどく・まひ・やけど」の3種のみを回復し、「こおり」「ねむり」は
  回復対象に含まれない（第三世代ルビー・サファイア〜第七世代まで説明文の表現に揺れは
  あるが、いずれも「どく まひ やけどの 状態異常」とのみ記載されており、範囲の変更はない）。
  状態異常がない場合、または回復対象外の状態異常（こおり・ねむり）の場合は技が失敗する。
- 第八世代（ソード・シールド）で「このわざは つかえません」に変更され、技マシン・技レコード等の
  習得手段も一切存在しなくなった。

## 実装方針

- 実装しない。データエントリ・専用ハンドラ・専用テストはすべて削除済み
  （旧実装は `battle.ailment_manager.remove()` を無条件呼び出ししており、
  こおり・ねむりまで回復してしまう誤り（範囲の誤り）があったことも判明したが、
  対象範囲外のため修正ではなく削除で対応した）。
