# たくわえる（Stockpile）仕様書

参照: docs/wiki/moves/たくわえる.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: -
* PP: 20（champions 基準。docs/champions/move_list.txt 参照。第三世代は10、第四世代以降20）
* 対象: 自分
* 優先度: 0
* 直接攻撃: いいえ

## 効果

* 自分の「たくわえた回数」を1増やす（volatile「たくわえる」の count で管理）。
* 最大3回までたくわえられる。
* 自分の『ぼうぎょ』『とくぼう』ランクをそれぞれ1段階ずつ上げる
  （ランクがすでに上限の場合はそのステータスのみ上がらないが、カウントの増加自体は成功する）。
* たくわえた回数が多いほど、技『のみこむ』の回復量・技『はきだす』の威力が上がる
  （連動する技セット。詳細は各技の仕様書を参照）。

## 使用条件

* すでに3回たくわえている状態でたくわえるを使用した場合は失敗する
  （このとき『ぼうぎょ』『とくぼう』も上がらない）。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: なし（必中）
* 追加効果判定: なし
* まもる: 防がれない（自分に対する変化技）
* マジックコート: 反射不可（相手を対象としないため対象外）
* よこどり: 対象になる（○）
* オウムがえし: 反射不可（変化技のため対象外）
