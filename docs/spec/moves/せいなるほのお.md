## 基本情報

* タイプ: ほのお
* 分類: 物理
* 威力: 100
* 命中: 95
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: なし

## 効果

* 追加効果として、50%の確率で相手を『やけど』状態にする。
* 使用者が『こおり』状態の場合、こおり状態を治してから攻撃する。
  - かえんぐるま等と同じ「使用宣言時に無条件でこおりを解除してから攻撃する」効果である
    （通常のこおり状態のターン開始時解凍判定＝25%抽選とは独立した、専用の解除処理）。
  - 本プロジェクトでは `flags` に `"thaw"` を設定した技が、この「無条件解凍」対象技を表す
    共通ラベルとして扱われている（`docs/spec/moves/_move_labels.md` の thaw リスト参照）。
  - ただし `"thaw"` フラグは現状、被弾側（相手がこの技のダメージを受けた際にこおりを解凍する:
    `こおり_cure_by_thaw_move`, `Event.ON_DAMAGE_HIT`）にのみ使われており、
    使用者自身がこおり状態のときに無条件解凍する処理は技ごとに個別実装が必要
    （実装例: `かえんぐるま_thaw_attacker` / `もえつきる_thaw_attacker`, `Event.ON_TRY_ACTION`, priority=5）。

## 判定フラグ

* 急所: 発生する
* 命中判定: あり（命中率95）
* 追加効果判定: あり
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降。第二〜第四世代は対象外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可能
