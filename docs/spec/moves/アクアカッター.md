# アクアカッター（Aqua Cutter）仕様書

参照: docs/wiki/moves/アクアカッター.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 70
* 命中: 100
* PP: 20
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃 ×）
* フラグ: `slash`（切る技。`contact` は付与しない）

## 効果

* 追加効果なし。急所に当たりやすい（急所ランク+1）。

## 判定フラグ

* 急所: 急所ランク+1（通常より高い確率で急所に当たる）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 対象外（ひるみ追加効果を持たないため無関係）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## イベント / 優先度

* 追加効果を持たないため、専用ハンドラは不要。`critical_rank=1` を `MoveData` に設定するのみで、
  既存の急所判定ロジック（`core/move_executor.py` の `CRIT_RATES` 参照処理）が自動的に適用される。
* `slash` フラグにより、特性きれあじ（`handlers/ability.py` の `ctx.move.has_flag("slash")` 判定）
  を持つポケモンが使用すると威力が1.5倍になる（既存の共通ロジックで自動適用）。

## リーサル計算

対象外（追加効果を持たない通常の物理攻撃技のため、`docs/progress/move.md` の
「リーサル実装」列は `n/a`）。
