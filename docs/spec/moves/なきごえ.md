# なきごえ（Growl）仕様書

参照: docs/wiki/moves/なきごえ.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中率: 100
* PP: 40（Gen9本家基準。`src/jpoke/data/ps-champ-ja/moves.json` に単独の項目が存在しない。
  ちょうおんぱ・ないしょばなし等、他の一般的な音技も同様に単独項目を欠くため、
  本プロジェクトはこれをChampionsの固定ロースターに本技を採用するポケモンがいない
  ことによる欠落と判断し、Gen9本家のPP値をそのまま採用する（`docs/spec/moves/ちょうおんぱ.md`
  ・`docs/spec/moves/ないしょばなし.md` と同じ扱い）
* 対象: 相手全員（シングルバトルのため相手1体と等価）
* 優先度: 0
* 直接攻撃: ×
* 音技（`sound` フラグ）

## 効果

* 相手の『こうげき』ランクを1段階下げる。

## 判定フラグ

* 急所: 対象外（変化技）
* 命中判定: ○（`accuracy=100`）
* 追加効果: なし
* まもる: 防がれる（`unprotectable` フラグなし。一般の相手対象変化技と同様）
* みがわり: 貫通する（`sound` フラグによる第六世代以降の一般ルール。個別のフラグ付与は不要、
  `move_executor` が sound フラグから自動的に判定する）
* マジックコート: 反射可能（変化技かつ対象が相手のため、category=status/target=foe の
  一般ルールで自動的に反射対象になる）
* よこどり: 対象外（相手を対象とする技のため、よこどりの対象外という一般ルールに従う。未実装）
* オウムがえし: コピー可能

## 一次情報との差異（本プロジェクトのスコープ外）

* ポケモン不思議のダンジョンでの通路制限（2マス先まで）は対象外
* コンテスト効果（ボルテージ維持等）は実装しない

## イベント / 優先度

* `Event.ON_STATUS_HIT`（優先度100・デフォルト）で相手の『こうげき』ランクを1段階下げる。
