# ワイドブレイカー（Breaking Swipe）仕様書

参照: docs/wiki/moves/ワイドブレイカー.html（ポケモンWiki）

## 基本情報

* タイプ: ドラゴン
* 分類: 物理
* 威力: 60
* 命中: 100
* PP: 16（`docs/champions/move_list.txt` 準拠。Gen9本家は15だが、チャンピオンズでは
  全技PPが8/12/16/20の4段階に統一されているため16が正）
* 対象: 相手全員（本プロジェクトはシングルバトルのみ対象のため、実質相手1体への
  通常攻撃として扱う。ダブルバトル向けの範囲攻撃威力補正は実装しない）
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 追加効果として、100%の確率で相手の『こうげき』ランクを1段階下げる。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率100）
* 追加効果: あり（100%・こうげき-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。本プロジェクトの対象世代では発動する）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（相手のランク補正を下げる追加効果を持つため、ちからずくのポケモンが
  使用すると威力1.3倍・こうげき低下の追加効果は発動しなくなる。`resolve_secondary_chance`
  経由で処理される）
