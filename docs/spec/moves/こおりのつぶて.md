# こおりのつぶて（Ice Shard）仕様書

参照: docs/wiki/moves/こおりのつぶて.html（ポケモンWiki）

## 基本情報

* タイプ: こおり
* 分類: 物理
* 威力: 40
* 命中: 100
* PP: 30
* 対象: 相手1体
* 優先度: +1
* 接触: なし（非直接攻撃）
* フラグ: なし

## 効果

* 追加効果なし。優先度+1により必ず先制攻撃できる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（追加効果を持たない技だが、おうじゃのしるしのひるみ判定は
  ダメージを与えた技全般に対して独立して発動するため対象になる）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象外（追加効果を持たないため無関係）
