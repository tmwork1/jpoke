# おきみやげ 仕様書

参照: docs/wiki/moves/おきみやげ.html（ポケモンWiki）

## 基本情報

| 項目 | 値 |
|------|-----|
| タイプ | あく |
| 分類 | 変化 |
| PP | 12（チャンピオンズ仕様。本家 SV 基準は 10） |
| 命中率 | 100 |
| 範囲 | 相手1体選択 |
| 優先度 | 0 |
| 直接攻撃 | × |

## 効果

- 使用者が自分の **HP を 0 にしてひんし** になる代わりに、相手の **こうげき** と **とくこう** ランクを
  それぞれ **2 段階** ずつ下げる。
- 追加効果ではなく確定効果（発動率 100%）。急所の概念はない。

## 発動タイミング（第五世代以降。チャンピオンズもこの仕様に準拠）

- 命中判定・`まもる`/`みがわり` による失敗判定を通過した場合のみ、使用者はひんしになる。
  - 命中しなかった場合: 使用者はひんしにならない。
  - `まもる`/`みがわり` で防がれた場合: 使用者はひんしにならない。
  - トリックガード状態やおうごんのからだ等、優先度の高い技を防ぐ効果に阻まれた場合: 使用者はひんしにならない。
- 命中判定・まもる判定を通過した後は、以下の場合でも使用者は必ずひんしになる。
  - 相手のこうげき/とくこうランクが共に -6 で、ランク低下の効果が実際には無かった場合。
  - 相手の特性 クリアボディ/しろいけむり/メタルプロテクト/フラワーベール や、持ち物 クリアチャーム、
    場の状態 しろいきり などによってランク低下がすべて防がれた場合。
  - 相手のミラーアーマーによってランク低下effectが跳ね返された場合（跳ね返された側が使用者自身のみがわり等で
    防いだとしても、使用者はひんしになる）。
- 自分の他に交代できるポケモンが残っていない場合でも技は成功し、使用者はひんしになる
  （バトルに敗北することもある）。

## 判定

- 命中判定: あり（100%）
- 急所: ×
- 追加効果: ×（メインの効果自体が確定効果のため区分なし）
- まもる: ○（防がれる。防がれた場合は使用者はひんしにならない）
- みがわり: ○（防がれる。防がれた場合は使用者はひんしにならない）
- おうじゃのしるし等の追加効果関連特性: 該当なし
- マジックコート: ×（反射されない）
- よこどり: ×（奪えない）
- オウムがえし: ○（コピー可能。ただし対象が場にいないと成立しないため、実質おきみやげが失敗した
  ときにしか成立しない）

## Zワザによる付加効果（対象外）

- Zワザ版おきみやげは、控えから出てくるポケモンの HP を全回復させる（`ON_SWITCH_IN` 相当）。
- 本プロジェクトは Zワザ非対応のため実装しない。

## ポケモンチャンピオンズ向けの簡略化

- ダブルバトル・トリプルバトル固有の仕様は対象外（本プロジェクトはシングルバトルのみ）。
- コンテスト・アピール関連の効果は実装しない。
- バトルパレス（AI 選択傾向）関連の記述は実装しない。
