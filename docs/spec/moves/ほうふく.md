# ほうふく（Comeuppance）仕様書

参照: docs/wiki/moves/ほうふく.html（ポケモンWiki）
参照: docs/spec/moves/固定ダメージ.md（4.3 メタルバースト、ほうふく）

## 基本情報

* タイプ: あく
* 分類: 物理
* 威力: -（固定ダメージ技のため威力欄なし）
* 命中: 100
* PP: 10
* 対象: 相手1体（不定表記だが、シングルバトルでは常に唯一の相手）
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 技を出す前にそのターン中に最後に相手から受けた攻撃技のダメージを **1.5倍**（小数点以下切り捨て）にして、
  その相手にそのまま与える固定ダメージ技。
* メタルバーストとタイプ・直接攻撃判定を除いて同じ効果を持つ
  （`docs/spec/moves/固定ダメージ.md` 4.3 を参照）。
  * ほうふくは直接攻撃（接触）技であり、メタルバーストは非接触技という点が異なる。
  * ほうふくはあくタイプ、メタルバーストははがねタイプという点が異なる。
* そのターンに攻撃技のダメージを受けていない場合は失敗する。
  * 状態異常（どく・やけど等）や天候ダメージ、反動ダメージなど、相手の攻撃技以外に起因する
    HP減少はここでいう「受けたダメージ」に含まれない。
* 物理・特殊を問わず、直前に受けた攻撃技のダメージが対象になる。
* 連続攻撃技を受けた場合は、最後のヒットのダメージのみが記録される（合計ではない）。
* 通常のダメージ計算式（攻撃・防御・威力・急所・乱数補正）は使用しない固定ダメージ技のため、
  タイプ相性による半減・高倍率は適用されない
  （例: フェアリータイプ相手でも、あくの半減は適用されず1.5倍のダメージがそのまま通る）。
  ただし無効化（0倍）は仕様上適用される。なお現行の第九世代のタイプ相性表では、
  あくタイプの攻撃技に対して0倍（無効）となる防御タイプは存在しないため、
  実際には無効化が発生するケースはない。

## 判定フラグ

* 急所: 発生しない（固定ダメージ技のため）
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 効果あり（第五世代以降は直接攻撃技であるため通常のダメージ技として扱われる）
* マジックコート: 反射不可（変化技ではないため対象外）
* せいぎのこころ・びびり: あくタイプのダメージ固定技を受けた際にも通常通り発動する
  （`docs/spec/abilities/せいぎのこころ.md`、`docs/spec/abilities/びびり.md` を参照）。
* さきどり: 対象外（さきどりで選べない技として個別に指定されている）
* ゆびをふる: 対象外（ゆびをふるで選ばれない技として個別に指定されている）

## 特性・道具との相互作用

* プレッシャー: ほうふくで反撃した際、プレッシャーのポケモンに対してPPが多く消費される
  （`docs/spec/abilities/プレッシャー.md` を参照）。
* よちむ: ほうふくの技威力区分は120として扱われる
  （`docs/spec/abilities/よちむ.md` を参照）。

## リーサル計算

* ほうふくのダメージ量は「そのターンに攻撃側自身が相手から受けたダメージ」に依存するため、
  防御側（相手）のHP分布を直接左右する専用の事前計算要素はない。
* `docs/progress/move.md` のリーサル実装・リーサルテスト列は `n/a`（対象外）。
