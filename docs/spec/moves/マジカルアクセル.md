# マジカルアクセル（Magical Torque）仕様書

参照: docs/wiki/moves/マジカルアクセル.html（ポケモンWiki）

## 基本情報

* タイプ: フェアリー
* 分類: 物理
* 威力: 100
* 命中: 100
* PP: 10（本家）→ 12（Champions、docs/champions/changes_from_sv.md によりPP8/12/16/20の4段階に統一。
  本技は `docs/champions/move_list.txt` に未収録のため、本家PP10の技（10まんばりき・
  バーンアクセル等）が12に変換されている前例と同様の変換規則を適用する）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない。物理技だが接触判定は発生しない特殊な技）

## 効果

* 追加効果として、30%の確率で相手を「こんらん」状態にする。
* こんらんの付与ターン数は本プロジェクト（ポケモンチャンピオンズ）の共通仕様に従い 2〜5 ターンの
  ランダム（`docs/spec/volatiles/こんらん.md` 参照）。

## 判定フラグ

* 急所: あり（通常判定）
* 命中判定: あり（命中率100）
* 追加効果: あり（30%・相手をこんらん状態にする）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降、追加効果の確率にひるみ判定を上乗せ可能）
* マジックコート: 反射不可（変化技ではないため対象外）
* オウムがえし: コピー不可（一次情報の「技の仕様」欄に明記。ダイマックスわざや他の
  「アクセル技」共通の失敗リストに含まれる）

## 技の仕様（一次情報より）

* この技に対する **アンコール・ものまね・まねっこ・さいはい は失敗する**
  （スターモービル専用の「アクセル技」共通仕様。他に該当する技: バーンアクセル・
  ダークアクセル・ポイズンアクセル・ファイトアクセル）。
  * 本プロジェクトでは `MoveFlag` に `non_encore`（アンコール失敗）・`non_copycat`
    （まねっこ失敗）が既存実装されているため、この2つを付与する。
  * `ものまね`（Mimic）・`さいはい`（Instruct）は本プロジェクトで未実装
    （`handlers={}` のまま実装保留）のため、対応する制限フラグは存在しない。
* ゆびをふる（Metronome）では選ばれない。ただし `ゆびをふる` 自体が本プロジェクトで
  未実装（`handlers={}`、TODO）のため、現時点では対応不要。

## 備考

* オルティガが使用するブロロローム（ルクバー・スターモービル）専用技。
  本プロジェクトはポケモン種族・専用技制限を対象としないため、習得制限は対象外。
* 同じ「アクセル技」ファミリー（バーンアクセル・ダークアクセル・ポイズンアクセル・
  ファイトアクセル）は既存実装済みの `non_copycat` フラグ（「スターモービル専用アクセル技」
  という趣旨のコメント付き）を共有している。
