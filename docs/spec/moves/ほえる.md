# ほえる（Roar）仕様書

参照:
* docs/wiki/moves/ほえる.html（ポケモンWiki）
* docs/champions/move_list.txt（ほえる行）
* docs/spec/moves/強制交代技.md（ほえる・ふきとばし・ドラゴンテール・ともえなげの共通仕様）
* docs/spec/moves/_blow.md（同内容）
* docs/spec/moves/ふきとばし.md（同系統の技。効果はほぼ同一）

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: -
* 命中: - （第六世代以降は必中。命中ランク補正・回避率補正の影響を受けない）
* PP: 20
* 対象: 相手1体
* 優先度: -6
* 直接攻撃: なし
* フラグ: 音技（sound）

## 効果

* トレーナー戦で使用すると、相手の場のポケモンを強制的に控えのポケモンとランダムに交代させる。
  ダメージは与えない。
* 野生戦での戦闘終了効果は本プロジェクトの対象外（シングルバトルのトレーナー戦のみを扱うため）。

## 判定フラグ

* 急所: 対象外（変化技のため）
* 命中判定: なし（必中）。ただし、そらをとぶ・あなをほる・ダイビング等で姿を隠している相手には
  当たらない（`HIDDEN_MOVE_ALLOWED_MOVES` に「ほえる」は含まれないため、既存の潜伏判定機構で
  自動的に失敗する）。
* 追加効果: なし
* まもる・みきり・トーチカ: 防がれない（貫通する。第六世代以降の仕様。
  `docs/champions/move_list.txt` にも「守×」および「相手の『まもる』『みきり』『トーチカ』
  『ニードルガード』『ブロッキング』『スレッドトラップ』『かえんのまもり』の効果を受けない」と明記）
* マジックコート: 反射される（第五世代以降の仕様。変化技かつ対象が相手のため反射可能）
* よこどり: 未実装の技のため対象外
* オウムがえし: コピー可能（ポケモンWiki記載。本プロジェクトでの対応状況は
  `docs/spec/moves/オウムがえし.md` を参照）
* みがわり: 貫通する（みがわり状態の相手にも効果がある。音技のため第六世代以降の一般則でも
  貫通するが、ほえる自体は第一世代から一貫してみがわりを貫通する仕様）
* 音技: はい。特性ぼうおんを持つ相手には無効化される
  （`ぼうおん` は既存実装済み。ON_TRY_MOVE 等のサウンド無効化ハンドラで判定）
* 特性きゅうばん・ばんけん: 効果を無効化する
* ねをはる状態: 効果を無効化する
* 特性かげふみ・ありじごく・じりょく、にげられない状態・バインド状態、
  場のフェアリーロック状態: いずれも無視して強制交代を発動する
  （`get_forced_switch_commands` が通常の交代可否判定を経由しないため）
* テラスタル: 影響なし（テラスタル状態でも通常どおり交代する）
* ゴーストタイプ: ノーマル技だが通常のタイプ相性無効化を受けず、ゴーストタイプの相手にも効果がある
  （本プロジェクトでは変化技はダメージ計算を経由しないため、タイプ相性による無効化判定自体が
  発生せず自動的にこの仕様を満たす）
* ダイマックス: 本プロジェクトの対象外（ダイマックス機構自体が非対応）

## 失敗条件

* 相手に交代できる控えポケモン（生存している控え）がいない場合、技自体が失敗する。
* 特性きゅうばん・ばんけん、またはねをはる状態により無効化された場合、技が失敗する。

## 交代先の選択

* 交代先は相手の控え（生存しているベンチポケモン）からランダムに選ばれる
  （ふきとばし・ドラゴンテール・ともえなげと共通のロジック `get_forced_switch_commands`）。

## 備考

* Zワザによる付加効果（自分のぼうぎょが1段階上がる）は本プロジェクトの対象外
  （Zワザ機構自体が非対応）。
