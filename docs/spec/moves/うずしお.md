# うずしお（Whirlpool）仕様書

参照: docs/wiki/moves/うずしお.html（ポケモンWiki）

## 基本情報

* タイプ: みず
* 分類: 特殊
* 威力: 35（第五世代以降。第四世代までは15）
* 命中: 85（第五世代以降。第四世代までは70）
* PP: 15
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける。
  * バインド状態の詳細仕様（継続ターン、しめつけバンドによる強化、マジックガードでの無効化、交代禁止とその例外、解除条件など）は `docs/spec/volatiles/バインド.md` を参照する。
* ダイビング状態（『ダイビング』を参照）の相手にも命中し、ダメージが2倍になる。
  * この処理は `うずしお` 固有の実装ではなく、`ダイビング` 揮発性状態側の `ダイビング_boost_power` ハンドラで `なみのり` と共通に処理される（`src/jpoke/handlers/volatile.py`）。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率85）
* 追加効果: なし（バインド付与自体が本体効果であり追加効果としての判定はない）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第三世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
* しめつける・まきつく・ほのおのうず・からではさむ・すなじごく・まとわりつく・トラバサミ・サンダープリズン・マグマストームと同系統の「バインド」技グループに属する（`docs/spec/moves/カテゴリ.md` を参照）。
