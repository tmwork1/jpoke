# フレアソング（Torch Song）仕様書

参照: docs/wiki/moves/フレアソング.html（ポケモンWiki）

## 基本情報

* タイプ: ほのお
* 分類: 特殊
* 威力: 80
* 命中: 100
* PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 基準。一次情報のGen9基準PPは10だが、
  Championsでは全技のPPが8/12/16/20の4段階に統一されているため12を採用する）
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）
* 音技（`sound`フラグ）: ぼうおんで無効化される・みがわりを貫通する・のどスプレー等と
  相互作用する

## 効果

* 追加効果として、100%の確率で自分の『とくこう』ランクを1段階上げる。

## 判定フラグ

* 急所: 通常通り発生する。
* 命中判定: あり（命中率100）
* 追加効果: あり（確定効果。ちからずくの対象＝威力1.3倍・追加効果無効化の対象）
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降ひるみ効果の対象（直接攻撃技ではないが一次情報上は対象）
* マジックコート: 反射不可（変化技ではないため対象外）

## リーサル計算

* 命中後、追加効果（確定発動）によって攻撃側の『とくこう』ランクが1段階上がるため、
  以降の攻撃のダメージ分布に影響する。しんぴのちから・ほのおのまい等の同種の
  「自分のとくこうが上がる追加効果」を持つ技と同様、専用のリーサルハンドラ
  （`LethalEvent.ON_HIT`, subject="attacker"）で対応する。
