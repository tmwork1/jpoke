## 効果

自分と場にいる味方ポケモンの HP を、それぞれ最大 HP の 1/4 ずつ回復し、状態異常を治す。
**端数は切り捨て**（一次情報「効果」節に明記）。

タイプと PP を除けば、基本的に `ジャングルヒール` と同様の効果を持つ。ただし、以下の点が異なる
（一次情報「技の仕様」節）。

- みがわり状態の味方は回復できない。ただし使用者自身のみがわり状態は無視して回復できる
  （シングルバトルでは「味方全員」＝使用者自身のみが対象になるため、この差異は実質発生しない）
- `ゆびをふる`（メトロノーム技相当）で選ばれる対象になる

- シングルバトルでは「味方全員」＝使用者自身のみが対象になるため、実質「自分の HP を最大HPの
  1/4（切り捨て）回復し、状態異常を治す」技として振る舞う
- 使用者自身の HP が満タンで、かつ状態異常でもない場合、技は失敗する
  （`docs/spec/turn.md` Event.ON_APPLY_MOVE 節「ジャングルヒール: HP満タン・状態異常無」と同様の
  判定。一次情報でも「タイプとPPを除けば基本的にジャングルヒールと同様の効果」と明記されており、
  失敗条件はこの一覧の差異に含まれていないため、ジャングルヒールと同一の判定を継承する）
- HP が満タンでも状態異常があれば技は成功し、状態異常のみを治す
- 同じ 1/4 回復技でも `いのちのしずく` は端数を四捨五入するのに対し、本技は `ジャングルヒール` と
  同様に切り捨てである点に注意

## 判定（一次情報より）

| 項目 | 値 |
|------|-----|
| 急所 | × |
| 命中判定 | × |
| 追加効果 | × |
| まもる | ×（自分を対象とする技のため防がれない。トリックガード・ダイウォールも同様に防がれない） |
| おうじゃのしるし | ×（第五世代以降） |
| マジックコート | ×（自分を対象とする技のため跳ね返されない） |

`まもる`・`マジックコート` は `move.target == "foe"` を前提に判定されるため、
実装上は `MoveData.target="self"` を明示する必要がある（未設定だとデフォルト値 `"foe"` になり、
誤って防がれる/跳ね返される）。

## 特性 ヒーリングシフト との相互作用

`heal` フラグを持つ回復技のため、使用者が `ヒーリングシフト` を持つ場合は優先度が+3される
（`docs/spec/abilities/ヒーリングシフト.md` 参照。特性側は `heal` フラグを汎用的に参照するため、
本技専用の実装は不要）。

## かいふくふうじとの相互作用

- 使用者が `かいふくふうじ` 状態の場合、`heal` フラグを持つ技として選択できない
  （`かいふくふうじ_try_action` が汎用的に処理するため、本技専用の実装は不要）

## 備考

- 自分1体しか対象がいない（シングルバトル）場合でも、回復量は最大HPの1/4のまま変わらない
- あめ状態などの天候による回復量の変化はない（`あさのひざし` 等とは異なり天候の影響を受けない）
- クレセリアの専用技

## ポケモンチャンピオンズ向けの簡略化

- ダブルバトル・トリプルバトルでの複数対象への回復処理は対象外（シングルバトル専用実装のため）
- みがわり状態の味方に関する挙動差はシングルバトルでは発生しないため対象外
