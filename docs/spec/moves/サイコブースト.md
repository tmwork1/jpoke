# サイコブースト（Psycho Boost）仕様書

参照: docs/wiki/moves/サイコブースト.html（ポケモンWiki）

## 基本情報

* タイプ: エスパー
* 分類: 特殊
* 威力: 140（第三世代から一貫して変更なし。ソード・シールドでは使用不可だが、
  第九世代（スカーレット・バイオレット）では引き続き使用可能）
* 命中: 90
* PP: 8（ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一されている。
  Gen9本家基準の5ではなくこの値を採用する。`src/jpoke/data/ps-champ-ja/moves.json` には
  サイコブースト自体のエントリが存在しないが、同系統の反動技（りゅうせいぐん・
  オーバーヒート・リーフストーム）はいずれも本家PP5→ChampionsPP8で統一されており
  〔`src/jpoke/data/ps-champ-ja/moves.json:942`(りゅうせいぐん) / `:134`(オーバーヒート) /
  `:934`(リーフストーム)、いずれもPP8〕、じばく・だいばくはつ・はかいこうせん・
  ぜったいれいど等の他の本家PP5技も同様にPP8へ統一されていることから、
  サイコブーストも同じ規則でPP8になると判断する)
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 命中して技が成功した後、反動として自分の『とくこう』ランクが2段階下がる（確定効果）。

## 判定フラグ

* 急所: あり
* 命中判定: あり（命中率90）
* 追加効果: なし（自分自身への確定効果であり、相手への「追加効果」には該当しない）
* まもる: 防がれる
* おうじゃのしるし: 対象外（「追加効果なし」技のため発動しない）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能（本プロジェクトでは対象外の古い技のため実装しない）
* ちからずく: 非対象技（自分への確定ランク低下は「追加効果」扱いではないため、
  ちからずくによる威力上昇・追加効果無効化の対象にならない）

## 備考

* 基本的にはデオキシスの専用技（XDでリライブ後のルギアも習得可能だが第八世代以降は
  習得手段がない）。反動でとくこうが下がる代表的な高火力特殊技であり、りゅうせいぐん・
  オーバーヒート・リーフストーム・フルールカノン等と同様の「反動技#能力が低下する」
  仕様に従う。
* `docs/spec/moves/_move_secondary_1.md` にも `サイコブースト	attacker	1	0	0	-2	...`
  として、りゅうせいぐん等と同一の自分C-2パターンで掲載済み。
