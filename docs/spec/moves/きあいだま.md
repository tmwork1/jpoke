# きあいだま（Focus Blast）仕様書

参照: docs/wiki/moves/きあいだま.html（ポケモンWiki）

## 基本情報

* タイプ: かくとう
* 分類: 特殊
* 威力: 120
* 命中: 70
* PP: 5
* 対象: 相手1体
* 優先度: 0
* 接触: なし（非直接攻撃）
* フラグ: 弾（bullet）技

## 効果

* 追加効果として、10%の確率で相手の『とくぼう』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率70）
* 追加効果: あり（10%・とくぼう-1）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降。本プロジェクトの対象世代では発動する）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・とくぼう低下の
  追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）

## Zワザ・ダイマックスわざ

* Zワザ威力: 190
* ダイマックスわざ威力: 95
* （本プロジェクトの対象範囲外のため実装しない）
