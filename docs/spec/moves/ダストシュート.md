# ダストシュート（Gunk Shot）仕様書

参照: docs/wiki/moves/ダストシュート.html（ポケモンWiki）
参照: src/jpoke/data/ps-champ-ja/moves.json（ポケモンチャンピオンズ仕様）

## 基本情報

* タイプ: どく
* 分類: 物理
* 威力: 120
* 命中: 80（第六世代以降。第五世代までは70）
* PP: 8
  * 一次情報（ポケモンWiki）ではPP5だが、`src/jpoke/data/ps-champ-ja/moves.json` の `ダストシュート` 行がPP8であり、
    ポケモンチャンピオンズ側の値を正とする。
* 対象: 相手1体選択
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、30%の確率で相手を『どく』状態にする。

## 判定フラグ

* 急所: 通常通り発生する
* 命中判定: あり
* 追加効果: あり（30%・どく）
* まもる: 防がれる
* おうじゃのしるし: 対象（追加効果の発動判定に上乗せされる）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: コピー可（本プロジェクトはオウムがえし自体が未実装のため対応不要）

## リーサル計算

* 固定威力・追加効果（追加効果を除けばダメージ量に影響しない状態異常付与のみ）の通常攻撃技のため、
  `lethal.py` に専用ハンドラは不要
  （`docs/progress/move.md` のリーサル実装／リーサルテスト列は n/a のまま対象外）。
