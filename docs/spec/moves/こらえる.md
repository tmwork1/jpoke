# こらえる（Endure）仕様書

参照: docs/wiki/moves/こらえる.html（ポケモンWiki）、docs/champions/move_list.txt

## 基本情報

* タイプ: ノーマル
* 分類: 変化
* 威力: なし
* 命中: なし（必中）
* PP: 12（champions基準。docs/champions/move_list.txt 参照。Gen9本家は10）
* 対象: 自分
* 優先度: +4
* 接触: なし

## 効果

* 使用したターンに、攻撃技でひんしになるようなダメージを受けても必ずHPが1残る。
* 具体的には、最初は成功するが、連続で使う度に成功率が1/3になっていく（失敗するとリセットされる）。
  * 本プロジェクトでは、まもる系技共通の連続使用失敗判定（`まもる系_連続使用失敗チェック`）を採用し、
    「直前ターンに守る系技を成功させていた場合、今ターンは必ず失敗する」という簡略化モデルで実装する
    （確率的に成功率が1/3ずつ低下していく詳細モデルは docs/spec/volatiles/まもる.md の方針に合わせ
    未実装・スコープ外とする）。
* 耐えることができるのは攻撃技によるダメージのみ。どく・やけど・バインド・のろい等の状態変化ダメージや、
  天候等の場の状態によるダメージは耐えられない。
* すでに残りHPが1のときでもこらえるの効果で耐えることができる。これにより受けたダメージが0になったとしても、
  攻撃を無効化したことにはならない。具体的には、このとき「受けた技の追加効果が発生する」
  「ダメージを受けたときに発動する特性が発動する」「持っていたふうせんが割れる」などの効果が発生する
  （みねうちと同様の扱い）。
* 一撃必殺技を含むダメージ固定技も耐えることができる。
* こらえるの効果は、成功したターンが終了するときまで継続する。1ターンに複数の相手から攻撃を受けても、
  連続攻撃技を受けても、全てのダメージを持ちこたえる。

## 判定フラグ

* 急所: なし（急所に当たっても効果は変わらない）
* 命中判定: なし
* 追加効果: なし
* まもる: 効果を受けない（まもるで防がれる対象外＝自分を対象とする変化技のため通常は問題にならない）
* おうじゃのしるし: 発動しない
* マジックコート: 反射不可
* よこどり: 対象外
* オウムがえし: コピー不可

## 優先度（docs/spec/turn.md 参照）

* `Event.ON_STATUS_HIT`（`attacker:self`）: 自分に「こらえる」揮発状態を付与する。turn.md に個別 priority
  行の記載がないため、既存の変化技ハンドラ（まもる_apply 等）と同様にデフォルト（100）を採用する。
* `Event.ON_TRY_MOVE_2`（`attacker:self`）: まもる系連続使用失敗チェック。turn.md では
  `Event.ON_TRY_MOVE_1` の30番「まもる系: 連続使用により失敗」に該当する記載があるが、既存実装
  （まもる・みきり・キングシールド等）はすべて `Event.ON_TRY_MOVE_2` のデフォルト priority（100）で
  実装されているため、同一パターンに合わせる。
* `Event.ON_MODIFY_MOVE_DAMAGE`（`defender:self`）: turn.md 表記 `Event.ON_MODIFY_DAMAGE` には
  60番「こらえる」（みねうち・てかげんと同番）と記載されているが、実装では **95** を採用する。
  turn.md の表には一撃必殺技の確定ダメージ算出（`ohko_damage`, priority=90, turn.md未掲載）が
  含まれておらず、こらえるは「一撃必殺技を含むダメージ固定技も耐えることができる」仕様のため
  `ohko_damage`（90）より後に実行する必要がある。がんじょう（100）・きあいのタスキ（100）・
  きあいのハチマキ（100）より先に発動させる点はturn.mdの意図と変わらない
  （詳細は docs/plan/moves/こらえる.md「Priority 根拠」参照）。
* `Event.ON_TURN_END`（`source:self`）: 「こらえる」揮発状態を解除する。turn.md に個別記載がないため、
  まもる・かえんのまもり等と同様にデフォルト priority を採用する。

## 関連インタラクション

* きあいのタスキ・がんじょうと同時に発動条件を満たした場合、こらえるの効果が優先されてタスキ・がんじょうは
  消費・発動しない（priority 60 が 100 より先に処理されるため実現される）。
* みねうち・てかげんの攻撃をHP1で耐えた場合も、こらえるの効果と結果は変わらない（いずれもHP1残し）。
  メッセージの優先順位（第五世代以降はこらえる優先）は本プロジェクトのログ表示仕様に依存するためスコープ外。

## 既知の未実装範囲（備考）

* 連続使用時の確率的な成功率低下（1/3ずつ低下、下限1/729）は未実装。本プロジェクトはまもる系共通の
  簡略化モデル（連続使用で必ず失敗）を採用する（docs/spec/volatiles/まもる.md 参照）。
* さいはい・おどりこ等ダブルバトル専用技との相互作用は、本プロジェクトの対象範囲（シングルバトル）外のため未実装。
* フリーフォール（連れ去り技）との相互作用は本プロジェクト非対応技のため未実装。
