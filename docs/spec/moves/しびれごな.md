## 効果の詳細

- 相手をまひ状態にする（直接付与）。
- 粉・胞子技（`powder` ラベル）: でんきタイプ・くさタイプの相手、特性『ぼうじん』所持ポケモン、
  『ぼうじんゴーグル』所持ポケモンには無効。詳細は `docs/spec/moves/_move_labels.md` の
  `powder` を参照。
- 対象がすでに状態異常を持っている場合は失敗する（`ailment_manager` が保証）。
- 対象が特性『じゅうなん』『そうしょく』『ぜったいねむり』『きよめのしお』の場合、まひを
  付与できない（各特性側の実装が担保する）。
- でんきタイプへの免疫は、まひ状態異常自体の仕様（`docs/spec/ailments/まひ.md`）として
  `ailment_manager` が保証するため、しびれごな固有の実装は不要。

## Pokémon Champions での補足

- Zワザによる付加効果（とくぼう+1）は実装しない（Zワザ自体が本プロジェクトの対象外）。
- コンテスト効果は実装しない。

## 実装方針

- イベント: `ON_STATUS_HIT`
- ハンドラ: `apply_ailment_to_defender(battle, ctx, value, ailment="まひ")`
- パターン: `でんじは`・`へびにらみ`と同様
- 粉技の無効化（でんき/くさタイプ・ぼうじん・ぼうじんゴーグル）はフレームワーク側
  （`move_executor.py` の `_check_grass_type_powder_immunity` や `ぼうじん`・
  `ぼうじんゴーグル` の各ハンドラ）が `powder` ラベルから汎用的に処理するため、
  しびれごな側での個別対応は不要。
