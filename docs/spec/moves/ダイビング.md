# ダイビング（Dive）仕様書

参照: docs/wiki/moves/ダイビング.html（ポケモンWiki）
関連: docs/spec/volatiles/ダイビング.md

## 基本情報

* タイプ: みず
* 分類: 物理
* 威力: 80（第四世代以降。第三世代のみ60）
* 命中: 100
* PP: 10（vanilla。本実装ではチャンピオンズ仕様の PP=12 を採用、`src/jpoke/data/ps-champ-ja/moves.json` 参照）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 2ターン技（溜め技）。
  * 1ターン目: 水中に深く潜る。攻撃は行わない。揮発状態「ダイビング」が付与される。
  * 2ターン目: 相手を攻撃し、命中後に揮発状態が解除される。
* 水中に潜っている間はほとんどの技を受けないが、以下は命中する（詳細は `docs/spec/volatiles/ダイビング.md` を参照）:
  * なみのり・うずしお（命中時ダメージ2倍）
  * 一次情報ではロックオン状態・ノーガード特性・どくどく（毒タイプ、第六世代以降）でも命中するとされるが、
    本実装では未実装（`docs/spec/volatiles/ダイビング.md` の「本実装での対象外」を参照）
* 追加効果はなし。
* 通常の姿のウッウが使用した場合、1ターン目の溜める行動が成功すれば、2ターン目の行動が失敗しても
  特性「うのミサイル」が発動する（本実装のウッウ側の対応状況は `docs/progress/ability.md` の
  うのミサイル行を参照。本仕様書はダイビング技側の一次情報として記載するのみで、
  うのミサイル自体の実装可否はうのミサイルのレビュー対象）。
* スカイバトルでは使用できず、ゆびをふる等でこの技が選ばれた場合は失敗する
  （本実装はスカイバトル非対応のため対象外）。

## 判定フラグ

* 急所: あり（攻撃時のみ、通常判定）
* 命中判定: あり（攻撃時のみ、命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 発動する（第三世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## ポケモンチャンピオンズ向けの簡略化

* フィールド上の効果（水の底への移動、マインドバッジ等）は実装しない（バトル外機能のため）
* Zワザ・ダイマックスわざとしての追加威力（Zワザ160・ダイマックスわざ130）は実装しない
* コンテスト・アピール関連の効果は実装しない
* スカイバトル関連仕様は実装しない（本実装は非対応ルールのため対象外）
