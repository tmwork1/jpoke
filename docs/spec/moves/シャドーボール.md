# シャドーボール（Shadow Ball）仕様書

参照: docs/wiki/moves/シャドーボール.html（ポケモンWiki）

## 基本情報

* タイプ: ゴースト
* 分類: 特殊
* 威力: 80
* 命中: 100
* PP: 16（champions/move_list.txt の値。Gen9本家では15だが、本プロジェクトは
  champions/changes_from_sv.md によりPPが8/12/16/20に統一されているためこちらを正とする）
* 対象: 相手1体
* 優先度: 0
* 接触: 直接攻撃ではない
* 弾のわざ: 該当する（特性「ぼうだん」で無効化される）

## 効果

* 追加効果として、20%の確率で相手の『とくぼう』ランクを1段階下げる。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: あり（20%・とくぼうランク-1）
* まもる: 防がれる
* おうじゃのしるし: 現行世代（第五世代以降）では通常のダメージ技として汎用の
  おうじゃのしるし処理の対象になる（本技固有の実装は不要。世代間の仕様変遷は
  本プロジェクトの対象範囲外）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能（攻撃技のため）
* ちからずく: 対象技（相手のランク補正を下げる技のため）。
  `flags={"secondary_effect"}` により、既存の汎用ちからずく処理
  （`ちからずく_boost` / `ちからずく_disable_secondary_effect`）で対応する。

## Zワザ・ダイマックスわざ

* Zワザ威力: 160 / ダイマックスわざ威力: 130
* 本プロジェクトの対象範囲外（ダイマックス・Zワザ非対応）のため実装しない

## 備考

* 弾のわざ（ボールわざ）の1つ。特性「ぼうだん」のポケモンに対しては無効化される。
