## 効果の詳細

- 相手をねむり状態にする（直接付与、あくびのように揮発効果を経由しない）。
- ねむりの継続ターン数（カウント）は Pokémon Champions 共通仕様に従う。詳細は
  `docs/spec/ailments/ねむり.md` を参照（通常技によるねむり付与は 1/3 の確率でカウント2、
  2/3 の確率でカウント3）。
- 対象がすでに状態異常を持っている場合は失敗する（ailment_manager が保証）。
- 粉・胞子技（`powder` フラグ）のため、第六世代以降は以下の場合に無効化される。
  - 相手が『くさ』タイプを持つ
  - 相手の特性が『ぼうじん』
  - 相手の持ち物が『ぼうじんゴーグル』
  - 上記はいずれも `powder` フラグから導出される汎用ルールとして
    `core/move_executor.py` の `_check_grass_type_powder_immunity`、
    `handlers/ability.py` の `ぼうじん_block_powder`、`handlers/item.py` の
    `ぼうじんゴーグル_block_powder_move` が共通処理する。技側の実装は不要。

## Pokémon Champions での補足

- Zワザによる付加効果（すばやさ+1）は実装しない（Zワザ自体が本プロジェクトの対象外）。
- コンテスト効果は実装しない。
- PP は Pokémon Champions 基準（`docs/champions/move_list.txt`）で 16（Gen9本家は15）。

## 実装方針

- イベント: `ON_STATUS_HIT`
- ハンドラ: `apply_ailment_to_defender(battle, ctx, value, ailment="ねむり")`
- パターン: `さいみんじゅつ`（ねむり付与、粉技フラグなし）・`どくのこな`（粉技フラグあり）と同様
