# イナズマドライブ（Electro Drift）仕様書

参照: docs/wiki/moves/イナズマドライブ.html（ポケモンWiki）

## 基本情報

* タイプ: でんき
* 分類: 特殊
* 威力: 100
* 命中: 100
* PP: 5（ミライドンの専用技。`docs/champions/move_list.txt` に本技は掲載されておらず、
  Championsで一律適用される「全技PPを8・12・16・20の4段階に統一」ルールの対象になった
  形跡がない。したがって本家シリーズ（SVまで）の値である5をそのまま採用する。
  同様にmove_list.txtに掲載のない専用技である`アクセルブレイク`・`ソウルビート`も
  Gen9基準のPPのまま実装されている）
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）
* フラグ: `contact`
* 専用技: ミライドンのみが習得可能

## 効果

* 効果がバツグン（タイプ相性が1倍を超える）の場合、ダメージが4/3倍になる。
  正確な補正率は5461/4096倍。

## 判定フラグ

* 急所: 通常
* 命中判定: あり（命中率100）
* 追加効果: なし
* まもる: 防がれる
* おうじゃのしるし: 第五世代以降は対象になる（本技はひるみ効果を持たないため実質無関係。
  一次情報の判定表に記載されているため参考として記載）
* マジックコート: 対象外（変化技ではないため）
* ゆびをふる: 選ばれない（一次情報の「技の仕様」に明記）

## イベント / 優先度

* 威力補正はダメージ計算内部の `Event.ON_CALC_POWER_MODIFIER` で行う。
  `docs/spec/turn.md` に本イベントは掲載されていないため、同種の効果抜群時威力ブースト技
  （`アクセルブレイク`）で採用済みの実装パターン・優先度（デフォルト）に合わせる。
* タイプ相性は `damage_calculator.calc_def_type_modifier(ctx)` の戻り値（4096が等倍）を
  参照し、4096を超える（=効果抜群）場合に `apply_fixed_modifier` で5461/4096倍を適用する。

## リーサル計算

* `docs/progress/move.md` の該当行は「リーサル実装: n/a」「リーサルテスト: n/a」であり、
  本レビューの対象範囲外（対象外）とする。
