# シグナルビーム（Signal Beam）仕様書

参照: docs/wiki/moves/シグナルビーム.html（ポケモンWiki）

## 対象範囲外の判定（重要）

`docs/wiki/moves/シグナルビーム.html` は本技をカテゴリ「ソード・シールドで使用不可能なわざ」
「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」に分類しており、第9世代で新規に習得可能な
ポケモンが存在しない。`src/jpoke/data/ps-champ-ja/moves.json` にも該当項目は存在せず、二重の根拠が
得られた。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは
実装しない」）に該当するため、**本技は実装対象外と判断した**（`docs/review/moves/オクタンほう.md`
`docs/review/moves/グラスミキサー.md` `docs/review/moves/シャドーボーン.md` と同様の前例）。

以降は参考として第2〜8世代時点の技データを記録する。
