# シグナルビーム（Signal Beam）仕様書

参照: docs/wiki/moves/シグナルビーム.html（ポケモンWiki）

※ docs/champions/move_list.txt に本技の記載なし（未収録。移行対象733技のうち一部は
  move_list.txt に未掲載であり、したでなめる等の前例と同様、Gen9本家基準の値を採用する）。
  なお本技はポケモンWiki上で「ソード・シールドで使用不可能」「BDSPで使用不可能」
  「スカーレット・バイオレットで使用不可能」なわざに分類されている（第9世代で新規に
  習得可能なポケモンが存在しない）が、技自体のデータ・効果はGen9のシステム上でも
  定義されており、既存実装（PP・威力・命中・効果）もGen9本家基準の値と一致している。
  対象範囲の判断（実装継続の可否）は本レビューの範囲外のため、既存実装を維持しつつ
  この事実を明記するに留める。

## 基本情報

* タイプ: むし
* 分類: 特殊
* 威力: 75
* 命中: 100
* PP: 15
* 対象: 相手1体（遠隔）
* 優先度: 0
* 接触: なし
* フラグ: `secondary_effect`（追加効果を持つ技。ちからずく対象技）

## 効果

* 追加効果として、10%の確率で相手をこんらん状態にする。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（10%・こんらん付与）
* まもる: 防がれる
* おうじゃのしるし・するどいキバ: 対象（本技はこんらん付与技であり、ひるみの追加効果は
  持たないため、全世代を通じておうじゃのしるし等による追加のひるみ効果と重複しうる）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能
* ちからずく: 対象技（追加効果があるため、ちからずくのポケモンが使用すると威力1.3倍・
  こんらんの追加効果は発動しなくなる。`resolve_secondary_chance` 経由で処理される）

## Zワザ

* Zワザ威力: 140
* ダイマックスわざ: （本プロジェクトの対象範囲外のため実装しない）

## イベント / 優先度

* `docs/spec/turn.md` に `Event.ON_DAMAGE_HIT` 自体の記載はないため、既存の同種ハンドラ
  （みずのはどう・したでなめる・あくのはどう 等）と同じ `Event.ON_DAMAGE_HIT` / 優先度100
  （デフォルト）のパターンに従う。
  * ダメージが実際に適用された後に発動することで、まもる等で無効化された場合やダメージ
    計算自体が行われなかった場合には発動しない仕様を担保する。

## リーサル計算

* HP依存の追加効果・威力変動を持たない通常のダメージ技であり、リーサル計算上の特別な
  考慮は不要（対象外）。
