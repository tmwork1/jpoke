# まきつく（Wrap）仕様書

参照: docs/wiki/moves/まきつく.html（ポケモンWiki）

## 基本情報

* タイプ: ノーマル
* 分類: 物理
* 威力: 15
* 命中: 90
* PP: 20
* 対象: 相手1体
* 優先度: 0
* 接触: あり（直接攻撃）

## 効果

* 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける。
  * バインド状態の詳細仕様（継続ターン、しめつけバンドによる強化、マジックガードでの無効化、交代禁止とその例外、解除条件など）は `docs/spec/volatiles/バインド.md` を参照する。
* 相手がゴーストタイプである場合、ノーマルタイプの技はゴーストタイプに無効（0倍）であるため、まきつくはそもそも命中せず、バインド状態にすることもできない（第二世代以降）。
  * これは `まきつく` 固有の実装ではなく、通常のタイプ相性判定（`_check_hit_by_type`、`src/jpoke/core/move_executor.py`）により技自体が失敗するため自然に成立する。
  * 第一世代のみ「1-4ターンの間バインド状態にする（相手は技を出せなくなるが交代は可能）。相手がゴーストタイプでもバインド状態にできる」という異なる仕様だったが、本プロジェクトの対象世代（Champions/第九世代）には適用されない。

## 判定フラグ

* 急所: 通常判定（急所ランク変化なし）
* 命中判定: あり（命中率90）
* 追加効果: なし（バインド付与自体が本体効果であり追加効果としての判定はない）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第三世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）
* よこどり: 対象外（変化技ではないため）
* オウムがえし: 反射可能

## 備考

* Zワザ威力: 100（本プロジェクトでは対象外）。
* ダイマックスわざ威力: 90（本プロジェクトでは対象外）。
* うずしお・しめつける・ほのおのうず・からではさむ・すなじごく・まとわりつく・トラバサミ・サンダープリズン・マグマストームと同系統の「バインド」技グループに属する。
* しめつけると同タイプ・同威力・同効果だが、命中率が高い（まきつく90 / しめつける85）。
