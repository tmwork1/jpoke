# うらみつらみ（Bitter Malice）仕様書

参照: docs/wiki/moves/うらみつらみ.html（ポケモンWiki）

## 基本情報

* タイプ: ゴースト
* 分類: 特殊
* 威力: 75
* 命中: 100
* PP: 10
* 対象: 相手1体
* 優先度: 0
* 接触: なし（直接攻撃ではない）

## 効果

* 追加効果として、100%の確率で相手の『こうげき』ランクを1段階下げる。

## 判定フラグ

* 急所: あり（命中率通常判定。急所ランク補正なし）
* 命中判定: あり（命中率100）
* 追加効果: あり（100%）
* まもる: 防がれる
* おうじゃのしるし: 発動する（第五世代以降）
* マジックコート: 反射不可（変化技ではないため対象外）

## 世代差異（本プロジェクトでは対象外）

* Pokémon LEGENDS アルセウスでは「相手が状態異常だと威力2倍」「相手をしもやけ状態にする追加効果」を持つが、
  第九世代（本プロジェクト対象）ではこれらの効果は存在しない。

## 専用技

* ゾロア・ゾロアーク（ヒスイのすがた）の専用技。
