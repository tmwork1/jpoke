# 仕様書: はっきんだま

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AF%E3%81%A3%E3%81%8D%E3%82%93%E3%81%A0%E3%81%BE

## 効果
- 効果: ギラティナに持たせると、ドラゴンタイプとゴーストタイプの技の威力が1.2倍になる。
 - 第八世代までは、やぶれたせかいでなくてもギラティナがオリジンフォルムにフォルムチェンジする。
- なげつける: ギラティナの場合は失敗する。それ以外は威力60

## 詳細な仕様
- タイプが変わる技を使用したときは、変更後のタイプがドラゴンかゴーストであればアイテムの効果で威力が上がる。

#### 第九世代
- ギラティナに持たせてもドラゴンとゴーストタイプの技の威力が1.2倍になるだけで、フォルムチェンジは起きない。第八世代以前のはっきんだまと同じ効果はだいはっきんだまに引き継がれた。
- 戦闘中、技や特性の効果でギラティナからはっきんだまを奪うことや失わせることはできる。はたきおとすを受けて失ったとき、技の威力は1.5倍になる。
 - ギラティナにはっきんだまを渡すこともできる。
