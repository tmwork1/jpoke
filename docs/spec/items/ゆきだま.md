## 効果

## 詳細な仕様
- ダメージが無かった場合は発動しない。以下の状況においてゆきだまは発動しない。
 - 技が外れたとき
 - 技が無効化されたとき
 - まもる状態で防いだとき
 - みがわり状態で防いだとき
- ばけのかわ・アイスフェイスでダメージを肩代わりしたときは発動する。
- こおりタイプの連続攻撃技を受けた場合、1発目が当たった時点で発動する。
- 所持者のこうげきランクがすでに最大まで上がっている場合、ゆきだまは発動しない。
 - 所持者の特性があまのじゃくである場合、こうげきランクが最小まで下がっているなら発動しない。
 - こうげきランク+5のときいかり状態とゆきだまが同時に発動条件を満たした場合、いかり状態が優先されるためゆきだまは消費しない。
- 所持者がたんじゅんである場合、こうげきランクが2段階上がる。
- 所持者があまのじゃくである場合、こうげきランクが1段階下がる。
- かたやぶりの効果があるこおり技に対してはたんじゅん・あまのじゃくの効果は発動せず、いずれもゆきだまでこうげきランクが1段階上がる。
- 特性マジシャンのこおり技を受けた場合、ゆきだまの効果が先に発動するため奪われない。
- ぜったいれいどのダメージを受けたがこらえるなどにより耐えた場合、ゆきだまは発動する。
