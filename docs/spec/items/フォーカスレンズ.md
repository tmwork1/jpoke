# 仕様書: フォーカスレンズ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%95%E3%82%A9%E3%83%BC%E3%82%AB%E3%82%B9%E3%83%AC%E3%83%B3%E3%82%BA

## 効果
- 効果:そのターンに、すでに行動している相手に対して使う技の命中率が1.2倍になる。
- なげつける:威力10

## 詳細な仕様
- 交代で出てきた相手に攻撃する場合、第四世代のみフォーカスレンズの効果がある。第五世代以降では効果がない。
- 一撃必殺技の命中率は上がらない。
- 複数のポケモンを対象とする技を使用した場合、道具の効果が発動するかは1体1体別々に判定される。
- 第五世代以降では、正確な補正率は4915/4096倍となる。

## 備考
- 似たような道具として、無条件に命中率が1.1倍になるこうかくレンズがある。発動条件がある分、フォーカスレンズの方が上昇率が高い。
