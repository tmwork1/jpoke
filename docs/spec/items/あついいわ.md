# 仕様書: あついいわ

調査日: 2026-07-04
参照URL: https://wiki.pokemonwiki.com/wiki/あついいわ

## 効果
- べにタマ×40-49DPPt/べにタマL×30BDSP
- 効果:持たせたポケモンがにほんばれ状態にするわざを使ったとき、持続ターンが8ターンに伸びる。
- 効果:持たせたポケモンが特性ひでり・ひひいろのこどうを発動させると、にほんばれ状態の持続ターンが8ターンに伸びる。(第六世代以降)
- なげつける:威力60
- ウッウロボ:素材タイプほのお、素材ポイント6

## 詳細な仕様
- ダイマックスわざ（たいようのあいさつ等）で天気を変えたときも効果がある。
- 第七世代以降のじょうきょう画面での天気の持続ターン表示において、あついいわの効果を発動させたプレイヤー側は最初からにほんばれが8ターン続くと表示されるが、対戦相手側では通常通りの5ターンと表示されるため、あついいわの存在を相手に知らせることはない。5ターンを過ぎても天気が消えず、あついいわが相手視点からも確定すると持続ターンの表記が8ターンに書き換わる。

## 備考
- 発動させる天気の効果を延長する道具の一種。
