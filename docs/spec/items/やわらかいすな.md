# 仕様書: やわらかいすな

調査日: 2026-07-04
参照URL: https://wiki.pokemonwiki.com/wiki/やわらかいすな

## 効果
- 持たせると、じめんタイプの技の威力が1.2倍になる（第四世代以降）。
- なげつける:威力10

## 詳細な仕様
- タイプが変わる技（わざレンジャー等）は、変更後のタイプがじめんタイプであればやわらかいすなの効果がある
  （`ctx.move.type`は判定時点のタイプを参照するため、既存の共通実装`_modify_power_by_type`で自然に対応済み）。
