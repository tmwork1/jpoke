# 仕様書: くろいてっきゅう

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8F%E3%82%8D%E3%81%84%E3%81%A6%E3%81%A3%E3%81%8D%E3%82%85%E3%81%86

## 効果
- こんごうダマ×40-49DPPt/こんごうダマL×15BDSP
- 効果:持たせると、すばやさが0.5倍になる。地面にいないポケモンは地面にいるようになる。
- なげつける:威力130

## 詳細な仕様
- 持たせるとでんじふゆう状態でも地面にいるようになるが、でんじふゆうを使うことはできる。
- くろいてっきゅうの効果で地面にいるひこうタイプポケモンへのじめんタイプ攻撃は、第四世代ではひこうタイプ以外の相性通りとなる。第五世代以降ではひこうタイプ以外の相性に関わらず等倍となる。
 - 第五世代以降でも、くろいてっきゅうを持ちつつ、うちおとす状態・じゅうりょく状態・ねをはる状態のいずれかの効果を受けているポケモンへのじめんタイプ攻撃は、ひこうタイプを無視したタイプ相性となる。
 - さかさバトルではひこうタイプに対するじめん技の相性は2倍になり、ひこうタイプ以外の相性の影響も受ける。
- 特性ぶきようのポケモンやさしおさえ状態のポケモンは、第四世代では地面にいるようになる効果は無視するがすばやさが0.5倍になる効果は残る。第五世代以降はすばやさが0.5倍になる効果も含めて無視する。
- すばやさがランクで変動している場合、変動後のステータスを0.5倍にする。まひの効果はくろいてっきゅうやその他の補正の効果を考慮した後に計算する(すばやさ#すばやさ補正も参照)。
- この道具が譲渡されたり消滅したりした場合、第七世代まではそのターンの行動順には影響しない。第八世代ではそのターンから行動順に影響が出る。

## 備考
- 類似の効果を持つアイテムにきょうせいギプス・パワー系アイテムがある。若干効果やメリットが違うので意識して使い分けたい。
- 持たせたポケモンのおもさは変化しない。また、フリーフォールに対しても影響しない。
