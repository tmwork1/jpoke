# 仕様書: ばんのうがさ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B0%E3%82%93%E3%81%AE%E3%81%86%E3%81%8C%E3%81%95

## 効果
- 効果:持たせたポケモンはにほんばれ状態とあめ状態の影響を受けなくなる。
- なげつける:威力60

## 詳細な仕様
- 晴れと雨状態に関する効果のみ防ぐことができる。ダメージを始めとしたすなあらし・あられ・ゆき関連の効果は一切防ぐことができない。
- この道具を持ったポケモンでも、特性や技の効果で天候を変えることはできる。
 - 晴れ状態のときのにほんばれ・ひでり、雨状態のときのあまごい・あめふらしがそれぞれ不発する効果は防げず、天候の継続ターンは更新されない。
- 晴れ状態でほのお技のダメージが1.5倍になり、みず技のダメージが半減する効果は防御側の効果とみなされる。攻撃側がばんのうがさを持っていても天気の効果を受け、逆に防御側がばんのうがさを持っていれば天気の影響は無視される。雨状態のみず技とほのお技も同様にして、防御側がばんのうがさを持っているときのみ天気の影響が無視される。
- ソーラービーム・ソーラーブレードは攻撃側がばんのうがさを持っていると晴れ状態でも溜めるようになるが、雨でも威力が下がらない。防御側が持っていても影響は無い。
 - 晴れ状態でソーラービーム・ソーラーブレードを溜め始めた後にばんのうがさを失っても、溜めは無くならない。
- エレクトロビームは攻撃側がばんのうがさを持っていると雨状態でも溜めるようになる。
- ウェザーボールは、攻撃側がばんのうがさを持っていると晴れや雨状態でもノーマルタイプとなり威力も上がらない。防御側が持っている場合は天気の効果で技のタイプと威力が変わる。
- せいちょうは晴れ状態でもこうげき・とくこうが1段階しか上がらない。
- こうごうせい・あさのひざし・つきのひかりは晴れや雨状態でも回復量が最大HPの1/2になる。
- 晴れ状態でのかみなり・ぼうふうの命中率は、攻撃側がばんのうがさを持っていると低下しない。
 - 防御側のみばんのうがさを持っている場合は、晴れ状態の効果で命中率が50%になる。
- 雨状態でのかみなり・ぼうふう・こがらしあらし・かみなりあらし・ねっさのあらしは、防御側がばんのうがさを持っていると必中にならない。
 - 攻撃側のみばんのうがさを持っている場合は、雨状態の効果で必中になる。
- ハイドロスチームは、攻撃側がばんのうがさを持っていると晴れ状態でダメージが1.5倍になる効果が発動しない。
 - 攻撃側のみばんのうがさを持っている場合、通常のみず技と同様に晴れの効果でダメージが0.5倍になる。防御側もばんのうがさを持っている場合、晴れの効果は完全に無視されてダメージは等倍となる。
 - 防御側のみばんのうがさを持っている場合、晴れ状態で技の効果が発動してダメージは1.5倍になる。
- 特性サンパワー・リーフガード・ようりょくそ、あめうけざら・うるおいボディ・すいすいは発動しなくなる。
- 特性かんそうはだによる晴れ状態でのダメージと、雨状態での回復は発動しなくなる。みず技を無効にする効果や受けるほのお技の威力が上がる効果は発動する。
- チェリムがばんのうがさを持つと、晴れ状態でもポジフォルムになれず、フラワーギフトによる能力上昇効果も発動しない。すでにポジフォルムのチェリムがばんのうがさを手に入れるとネガフォルムに戻る。
 - ポジフォルムのチェリムが味方にいるポケモンがばんのうがさを持っている場合は、晴れ状態でフラワーギフトによる能力上昇効果を受ける。
- 特性こだいかっせいは発動し、能力の上昇効果も受ける。
- 特性ひひいろのこどうで場をにほんばれに変えることはできるが、攻撃の上昇効果は受けられなくなる。
- 持たせたポケモンは晴れ状態でもこおり状態になる。逆にばんのうがさを持ったポケモンが持っていないポケモンを晴れ状態のときにこおり状態にすることはできない。
- ヌメイルのヌメルゴンへの進化には影響が無く、ばんのうがさを持っていても条件が揃えば進化できる。
