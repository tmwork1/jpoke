# 仕様書: ルームサービス

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%AB%E3%83%BC%E3%83%A0%E3%82%B5%E3%83%BC%E3%83%93%E3%82%B9

## 効果
- 効果:場がトリックルーム状態であるとき発動して、すばやさが1段階下がる。1度発動すると無くなる。
- なげつける:威力100

## 詳細な仕様
- 発動しても「<所持者>の 素早さが 下がった!」というメッセージが流れるのみで、ルームサービスを使ったことは直接明示されない。
 - 経験値が入手できるバトルでは、他の消費アイテムと同様、使用すると「ルームサービスは 役目を 終え なくなって しまった......」とメッセージが流れる。
- 所持者が場にいるときにトリックルームが発動した場合と、トリックルーム状態の場に所持者が繰り出されたときのどちらでも発動する。
 - 所持者が場に出たときは、場に出たときに発動する特性より後にルームサービスが発動する。
 - トリックルーム状態になったことで、場にいる複数の所持者が同時にルームサービスを発動させるとき、手持ち画面で確認できるすばやさ実数値が高い順に発動させる。このときトリックルームやランク補正などの効果や、へんしんやスピードスワップなどの実数値変動を含めたあらゆるすばやさ補正を考慮しない。
 - トリックルーム状態の場に、複数の所持者が同時に死に出しされるとき、場に出た瞬間のすばやさが高い順に発動させる。このとき特性やトリックルームなどのすばやさ補正を考慮する。場に出た後にねばねばネット・かわりものや、天候を変える特性などによりルームサービス発動時のすばやさが変動しても、発動順には影響しない。
 - トリックルーム状態で、こだいかっせい・クォークチャージの発動条件も満たしている場に出たときは、ルームサービスの後に特性が発動する。
- 設置技の効果でダメージや状態異常を受けた場合は、他の処理より優先してルームサービスの発動が直後に割り込む。
 - 複数の設置技の効果が仕掛けられている場合、最初の効果を受けた直後にルームサービスが発動する。
 - 場に設置技が仕掛けられていてもその効果を防いだときや、ねばねばネット状態の効果を受けただけのときはルームサービスの発動は割り込まない。
 - 同時にこだいかっせい・クォークチャージの発動条件を満たしているときは、特性が発動した後にルームサービスが発動するようになる。
- 発動すると、その瞬間から行動順に影響が出る。ただし、トリックルームの優先度の関係上、そのターンの行動順が変わるケースは限られる。
- トリックルーム中にルームサービスを入手したときは即座に発動させる。特性ぶきようのポケモンからトリックなどで受け取ったときや、リサイクルで道具を復活させたときなどに見られる。
- すでに所持者のすばやさが最低まで下がっているときは発動しない。すばやさが最低でルームサービスを発動できなかった場合、トリックルーム中に使用した変化技などですばやさランクが上がったとしてもルームサービスは発動しない。
 - このとき攻撃技のダメージや、天候のスリップダメージなどを受けたとき即座にルームサービスが発動する。ただし、こんらんの自傷ダメージを受けたときは発動しない。HPが回復するときは発動しない。
 - どろぼう・ほしがる・特性マジシャンの攻撃を受けたときは、道具を奪う効果が優先される。はたきおとすを受けたときは道具を失う効果が優先される。
 - 新たに状態異常になったり、こんらんやアンコールなどの状態変化になったときは発動する。
 - トリックルームが解除されてから再度トリックルーム状態になったときや、ポケモンを一度手持ちに戻して再び場に繰り出したときは通常どおりに発動する。
- 他のポケモンによるトリックルームでルームサービスを発動させる場合でも、所持者自身が自分の能力を下げたことになる。
 - クリアボディ・ミラーアーマーなどの特性や、しろいきり状態を無視してすばやさが下がる。
 - 敵が使ったトリックルームでルームサービスを発動しても、特性まけんき・かちきは発動しない。
- 所持者の特性があまのじゃくである場合、すばやさが1段階上がり、すでにすばやさが最大まで上がっているときは発動しない。所持者の特性がたんじゅんである場合、すばやさが2段階下がる。
 - 特性かたやぶりのポケモンがトリックルームを発動させたことで場にいる特性あまのじゃく・たんじゅんのポケモンのルームサービスが発動した場合、特性の効果は無視されていずれもすばやさが1段階下がる。
 - かたやぶりのポケモンがトリックルームを発動させた後に、あまのじゃく・たんじゅんのポケモンが場に出てきてルームサービスを発動させる場合はかたやぶりの効果は無く、それぞれの特性の効果でランク変動値が変わる。
- マジックルーム状態の間は、トリックルーム状態であってもルームサービスは発動しない。
 - ターン経過により、トリックルームとマジックルームが同じターンに解除される場合、使用された順に関わらずトリックルームが解除された後にマジックルームが解除されるため、ルームサービスは発動しない。
 - トリックルームが発動したときに場にいたが、マジックルームによりルームサービスを使えなかったポケモンは、トリックルーム中にマジックルームが解除されれば即座にルームサービスを使用できる。
 - マジックルーム状態のときにトリックなどの効果でルームサービスを手に入れたポケモンは、マジックルームが解除されればルームサービスを発動できる。このとき、トリックルーム発動時に場にいたかは関係しない。
 - トリックルームが発動した後に場に出たが、マジックルームによりルームサービスを発動できなかったポケモンは、マジックルーム状態が解除された後もルームサービスを発動できなくなる。
 - このとき、攻撃技などでダメージを受けたときや、トリックルームを解除してから再発動させたとき、マジックルームを再発動させてから解除したときなどでもルームサービスは発動しない。
 - このときも、一旦手持ちに戻ってからマジックルームでない場に繰り出したときや、他のポケモンからトリックなどでルームサービスを手にしたときは発動できる。
 - ぶきようの効果でルームサービスを発動できなかったポケモンは、特性の効果が消えると同時にルームサービスを発動できる。

## 備考
- 名前にルームが入っているが、場がワンダールーム状態になるだけでは発動しない。
