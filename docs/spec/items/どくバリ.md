# 仕様書: どくバリ

調査日: 2026-07-04
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A9%E3%81%8F%E3%83%90%E3%83%AA

## 効果
- 効果:持たせると、どくタイプの技の威力が1.2倍になる。
- なげつける:威力70、相手をどく状態にする。

## 詳細な仕様
- 威力補正は技のタイプがどくタイプであるかどうかで判定する（他の道具の
  タイプ強化アイテムと同様、`ON_CALC_POWER_MODIFIER` で `move.type` を参照して判定する）。

## 備考
- 類似の効果を持つアイテムに他のタイプ強化アイテム（もくたん・じしゃく・りゅうのキバ 等）がある。

## 関連項目
- もくたん - 同系統のタイプ強化アイテム
- なげつける - どくバリを投げた場合の追加効果
