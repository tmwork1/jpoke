# 仕様書: メタルコート

調査日: 2026-07-04
参照URL: https://wiki.pokemonwiki.com/wiki/メタルコート

## 効果
- 持たせると、はがねタイプの技の威力が1.2倍になる（第四世代以降）。
- なげつける:威力30

## 詳細な仕様
- 所有者が使用する技のタイプが（わざレンジャー等でタイプが変わった場合を含め）はがねタイプであれば威力補正がかかる。威力補正は攻撃側の持ち物にのみ依存する。
