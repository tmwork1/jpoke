# 仕様書: でんきだま

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A7%E3%82%93%E3%81%8D%E3%81%A0%E3%81%BE

## 効果
- 効果:
 - ピカチュウに持たせると、
 - 第二世代・第三世代:とくこうが2倍になる。
 - 第四世代:使用する技の威力が2倍になる。
 - 第五世代以降:攻撃技を使うときにこうげき・とくこうが2倍になる。
 - ピチューのタマゴが見つかる組み合わせで、両親のどちらかにでんきだまを持たせておくと、生まれるピチューがボルテッカーを覚える (エメラルド以降)。
- なげつける:威力30、相手をまひ状態にする

## 詳細な仕様
- おきがえピカチュウ・サトシの帽子をかぶったピカチュウ・キョダイマックスするピカチュウに持たせたときも効果がある。
 - ピチュー・ライチュウに持たせても効果は無い。
- こんらんの自傷ダメージを受けるときは、第四世代ではでんきだまの影響を受ける。第五世代以降はでんきだまの影響が無い。
- へんしん状態のポケモンがでんきだまを持っている場合、第三世代・第四世代では変身後の姿がピカチュウであれば効果がある。第二世代と第五世代以降は変身前がピカチュウでなければ効果が無い。
- でんきだまの効果を得ているポケモンがイカサマを受けても、でんきだまの影響は無い。
 - でんきだまの効果を得ているポケモンがイカサマを使用したときは、でんきだまの影響がある。ボディプレスを使用したときはぼうぎょを2倍にする。
- なげつけるで使うと、追加効果として100%の確率で相手をまひ状態にする。
- ピチューにボルテッカーを覚えさせるときの仕様についてはボルテッカー#習得における仕様を参照とする。
