# 仕様書: ひかりのねんど

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B2%E3%81%8B%E3%82%8A%E3%81%AE%E3%81%AD%E3%82%93%E3%81%A9

## 効果
- しらタマ×40-49DPPt/しらタマL×15BDSP
- 効果:持たせたポケモンが使用した技でリフレクター、ひかりのかべ、オーロラベール状態が発生したとき、持続ターンが8ターンに伸びる。
- なげつける:威力30

## 詳細な仕様
- この道具を所持しながら技を使用したときしかアイテムの効果は無い。ひかりのねんどを持たずに壁を張り、効果が残っているうちにひかりのねんどを手に入れたとしても持続ターンは伸びない。逆に、ひかりのねんどを持って壁を張った後にひかりのねんどを失ったとしても延長した効果は消えない。
- 壁を張る変化技をよこどりで奪われた場合、よこどりしたポケモンがひかりのねんどを持っていれば持続ターンが長くなる。奪われたポケモンがひかりのねんどを所持していても効果は無い。
- ひかりのねんどにより効果が延長された壁がコートチェンジで入れ替わる場合、ひかりのねんどの効果ごと入れ替わる。コートチェンジの使用者がひかりのねんどを持っていても、ひかりのねんどを持たずに発動された壁の効果は延長されない。
- 第七世代以降のバトルではじょうきょう画面で壁の残りターン/持続ターンが確認できる。ひかりのねんどの効果を発動させたプレイヤー側は最初から8ターン続くと表示されるが、対戦相手側では通常どおり5ターンと表示されるため、ひかりのねんどの存在を相手に知らせることはない。5ターンを過ぎても壁が消えずひかりのねんどの存在が確定したとき、対戦相手視点から見た継続時間が8ターンに書き換わる。
 - コートチェンジにより壁が対戦相手側に移った場合でも、相手視点では5ターン表記のままであり、ひかりのねんどの存在は知らされない。
