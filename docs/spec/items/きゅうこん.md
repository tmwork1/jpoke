# 仕様書: きゅうこん

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8D%E3%82%85%E3%81%86%E3%81%93%E3%82%93

## 効果
持たせると、みずタイプの攻撃技を受けたときにとくこうが1段階上がる。1回使うとなくなる（なげつける威力30）。

## 詳細な仕様
- ダメージが無かった場合は発動しない。以下の状況においてきゅうこんは発動しない。
 - みずびたしなど変化技を受けたとき。
 - 技が外れたとき。
 - 技が無効化されたとき。
 - まもる状態で防いだとき。
 - みがわり状態で防いだとき。
- ばけのかわ・アイスフェイスでダメージを肩代わりしたときは発動する。
- みずタイプの連続攻撃技を受けた場合、1発目が当たった時点で発動する。
- 所持者のとくこうランクがすでに最大まで上がっている場合、きゅうこんは発動しない。
 - 所持者の特性があまのじゃくである場合、とくこうランクが最小まで下がっているなら発動しない。
- 所持者がたんじゅんである場合、とくこうランクが2段階上がる。
- 所持者があまのじゃくである場合、とくこうランクが1段階下がる。
- かたやぶりの効果があるみず技に対してはたんじゅん・あまのじゃくの効果は発動せず、いずれもきゅうこんでとくこうランクが1段階上がる。
- 特性マジシャンのみず技を受けた場合、きゅうこんの効果が先に発動するため奪われない。
- タイプが変わる技は、変化後のタイプがみずタイプであるときに発動する。
- うるおいボイスの効果で、みずタイプに変わった技を受けたときも発動する。

## 備考
- みず技のダメージを無効化する効果はない。
