# 仕様書: おんみつマント

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8A%E3%82%93%E3%81%BF%E3%81%A4%E3%83%9E%E3%83%B3%E3%83%88

## 効果
- 効果:相手の攻撃技による追加効果を受けなくなる。
- なげつける:威力30

## 詳細な仕様
- 特性りんぷんのアイテム版といった効果を持つ。防げる効果に関しては追加効果/りんぷん#特性の効果の項目も参照とする。
 - 自分のランクを上げる追加効果は、所持者が使用したときも、所持者に対して使われたときも発動する。
 - クリアスモッグ/はたきおとすなど、追加効果ではない技の効果は防げない。
 - なげつけるにより追加効果がある道具を投げられたとき、防御側にとってメリットとなる効果も含めて全てが防がれる。きのみをなげつけられてもゲップは使えるようにはならず、無効化されたきのみが特性ほおぶくろ/しゅうかく/はんすうの効果対象になることもない。
- おんみつマントで追加効果を防いだときも、それを明示するメッセージが流れることはない。
- 特性どくしゅ/どくのくさりによるどく/もうどく状態も防げる。
- 特性あくしゅう・道具おうじゃのしるし/するどいキバによるひるみ効果も防ぐ。
- ひるみ状態を無効化する効果の判定は攻撃技を受けた時点で行われる。おんみつマントを持つポケモンが行動するときではない。
- 特性シンクロ/どくくぐつにより発生する状態異常/状態変化はおんみつマントで防ぐことができない。

## 備考
- りんぷんと異なり特性ではないので、かたやぶりでは無視されない。
