## するどいくちばし

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%99%E3%82%8B%E3%81%A9%E3%81%84%E3%81%8F%E3%81%A1%E3%81%B0%E3%81%97

### 効果
- 効果:持たせると、
 - 第二世代:ひこうタイプのわざによるダメージが1.1倍される。
 - 第三世代:ひこうタイプのわざを使った時のこうげきが1.1倍になる。
 - 第四世代以降:ひこうタイプのわざの威力が1.2倍になる。
- なげつける:威力50

### 詳細な仕様
- タイプが変わる技・ノーマルスキン・スカイスキンの効果でタイプが変わった技は、変更後のタイプがひこうタイプであればアイテムの効果がある。
- フライングプレスはかくとうタイプの技とみなされ、するどいくちばしの効果は無い。
