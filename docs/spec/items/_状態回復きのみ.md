| きのみ | 回復する状態異常 | なげつける |

| カゴのみ | ねむり | 威力10 |
| クラボのみ | まひ | 威力10 |
| チーゴのみ | やけど | 威力10 |

| ラムのみ | 全状態異常 | 威力10 |

- 共通: 対応する状態異常になった瞬間に自動消費して回復。ラムのみはねむり以外の揮発性状態は回復しない。
