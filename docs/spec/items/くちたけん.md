# 仕様書: くちたけん

調査日: 2026-06-06

## 効果
- 効果: ザシアンに持たせると戦闘中けんのおうの姿になる。
- なげつける: 失敗する

## 詳細な仕様
- この道具を持ったポケモンがなげつけるを使うと失敗する。
- この道具を持ったザシアンに対して、トリック・すりかえは失敗する。ふしょくガスの効果は受けない。ほしがる・どろぼう・特性マジシャン・わるいてぐせの効果で道具を奪えない。はたきおとすの威力は上がらず道具も失われない。
 - ザシアン以外のポケモンならばこれらの効果でくちたけんを失う。はたきおとすの威力も1.5倍になる。
- れきせんのゆうしゃのザシアンに対してトリック・すりかえ・特性きょうせいでくちたけんを渡すことはできない。
