# 仕様書: だっしゅつボタン

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A0%E3%81%A3%E3%81%97%E3%82%85%E3%81%A4%E3%83%9C%E3%82%BF%E3%83%B3

## 効果
- 効果:持たせたポケモンが攻撃技を受けると手持ちと入れ替わる。1回発動するとなくなる。
- なげつける:威力30

## 詳細な仕様
- 交代先のポケモンは自由に選ぶことができる。
- 所有者が野生ポケモンであるときや、交代できるポケモンが残っていないときは発動しない。
- にげられない・バインド・ねをはる・フェアリーロック状態、特性きゅうばん・ありじごく・じりょく・かげふみの影響を受けずに交代することができる。
- だっしゅつボタンが発動するとダイマックスポケモンも交代する。
- フリーフォールを使用したり受けたりして空中にいる間は、攻撃を受けてもだっしゅつボタンは発動しない。
- ヘイラッシャに持たせたときは、味方の特性しれいとうで合体している間に攻撃を受けてもだっしゅつボタンは発動しない。
- ダメージが無い技を受けたときは発動しない。
 - 変化技を受けたときは発動しない。
 - 相性・特性・まもる状態などで攻撃技を無効にしたときは発動しない。
 - みがわりで攻撃を防いだときは発動しない。
- 特性ばけのかわ/アイスフェイスでダメージを肩代わりしたときや、、HP1でこらえるを使ったときなど、攻撃技で受けたダメージが0だったときはだっしゅつボタンは発動する。
- 所有者が相手の攻撃でひんしになったときは発動しない。
- だっしゅつボタンを持つ複数のポケモンが全体攻撃技で同時に攻撃を受けた場合、すばやさ実数値が一番高いポケモンのだっしゅつボタンのみが発動する。このとき比較するすばやさはステータス画面で確認できるすばやさ実数値であり、ランク補正やトリックルームなど、あらゆるすばやさ補正は除外する。他のポケモンのだっしゅつボタンは発動しない。
- 連続攻撃技は最後の攻撃まで受けてから発動する。
- 相手がとんぼがえりなど、こちらに攻撃しつつ交代する技を使った場合、だっしゅつボタンによる交代のみ発動し、技を使った側は交代できない。
- 特性ちからずくの効果が発動した技を受けたとき、だっしゅつボタンは発動しない。
 - 特性の効果が発動していなければ、ちからずくのポケモンから受けた技であってもだっしゅつボタンは発動する。
- はたきおとす・ほしがる・どろぼう・マジシャンの効果はだっしゅつボタンより早く発動する。このため攻撃を受けた側はもちものを失うか奪われ交代することができない。
- ドラゴンテール・ともえなげによる強制交代効果はだっしゅつボタンより早く発動する。このためだっしゅつボタンは消費されず、交代先を自由に選ぶことができない。
- 所有者の攻撃により、相手のわるいてぐせが発動してだっしゅつボタンを奪われた場合、その際の攻撃ではだっしゅつボタンは発動しない。
- だっしゅつボタンとレッドカードの発動条件を同時に満たした場合、補正を除外したすばやさ実数値が高い順にアイテムが発動する。ただし、いずれの場合でもレッドカードによる交代先が出てきた後にだっしゅつボタンによる交代先が出てくる。
 - レッドカードが先に発動した場合、レッドカードによる交代先が出てきた後にだっしゅつボタンが発動する。
 - だっしゅつボタンが先に発動した場合、だっしゅつボタン所有者が引っ込んだ後、レッドカードが発動しその交代先が出てきて、その後だっしゅつボタンによる交代先を選択することになる。
- だっしゅつボタンとききかいひ/にげごしの発動条件を同時に満たした場合、だっしゅつボタンが先に発動して控えに引っ込む。
 - 複数のポケモンのだっしゅつボタンとききかいひ/にげごしが同時に発動した場合、すばやさに関わらずだっしゅつボタンが先に、ききかいひ/にげごしが後に発動して控えに引っ込む。交代先は両者同時に繰り出す。
- だっしゅつボタンとだっしゅつパックの発動条件を同時に満たした場合、すばやさに関わらずだっしゅつボタンのみが発動し、だっしゅつパックは発動しない。
- 発動しただっしゅつボタンは特性ものひろいで拾う対象にならない。
 - なげつけるでだっしゅつボタンを消費したときはものひろいで拾える。
 - リサイクルの場合は、発動しただっしゅつボタンでも復活させることができる。
- 特性きょうせいのポケモンの味方がだっしゅつボタンを発動した場合、第七世代以降ではきょうせいは発動しない。
 - 第六世代では交代前にきょうせいが発動し、だっしゅつボタンを消費したポケモンにもちものを渡す。次に場に出てきたとき、だっしゅつボタンを持っていたポケモンは受け取った道具を2つ持っていると見なされる。このポケモンが再び交代したり、アイテムを失うか渡したりすることでこの現象は解消される。詳細はきょうせい#バグを参照。
