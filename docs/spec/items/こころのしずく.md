# 仕様書: こころのしずく

調査日: 2026-06-06

## 効果
- 効果:
 - 第六世代まで: ラティオスかラティアスに持たせると、とくこうととくぼうを1.5倍にする。
 - 第七世代以降: ラティオスかラティアスに持たせると、エスパータイプとドラゴンタイプの技の威力が1.2倍になる。
- なげつける:威力30

## 詳細な仕様
- こころのしずくを持つ他のポケモンがラティオス・ラティアスにへんしんしている場合、第四世代までは効果がある。第五世代以降は効果が無い。

### 第七世代以降
- 対戦施設・レーティングバトル・公式大会などでも使用できる。
- 正確な倍率は4915/4096となる。
- タイプが変わる技を使用したときは、変更後のタイプがエスパーかドラゴンタイプであれば効果を受ける。
- そうでん状態により、エスパー・ドラゴンタイプの技がでんきタイプに変わったときは、こころのしずくの効果を受けられない。
