# 仕様書: しんぴのしずく

調査日: 2026-07-04
参照URL: https://wiki.pokemonwiki.com/wiki/しんぴのしずく

## 効果
- 持たせると、みずタイプの技の威力が1.2倍になる（第四世代以降）。
- なげつける:威力30

## 詳細な仕様
- 所有者が使用する技のタイプが（タイプが変わる技やノーマルスキン・うるおいボイス等でタイプが変わった場合を含め）みずタイプであれば威力補正がかかる。威力補正は攻撃側の持ち物にのみ依存する。
