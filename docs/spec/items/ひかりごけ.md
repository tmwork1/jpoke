## 効果

## 詳細な仕様
- ダメージが無かった場合は発動しない。以下の状況においてひかりごけは発動しない。
 - みずびたしなど変化技を受けたとき。
 - 技が外れたとき。
 - 技が無効化されたとき。
 - まもる状態などで防いだとき。
 - みがわり状態で防いだとき。
- ばけのかわ・アイスフェイスでダメージを肩代わりしたときは発動する。
- みずタイプの連続攻撃技を受けた場合、1発目が当たった時点で発動する。
- 所持者のとくぼうランクがすでに最大まで上がっている場合、ひかりごけは発動しない。
 - 所持者の特性があまのじゃくである場合、とくぼうランクが最小まで下がっているなら発動しない。
- 所持者がたんじゅんである場合、とくぼうランクが2段階上がる。
- 所持者があまのじゃくである場合、とくぼうランクが1段階下がる。
- かたやぶりの効果があるみず技に対してはたんじゅん・あまのじゃくの効果は発動せず、いずれもひかりごけでとくぼうランクが1段階上がる。
- 特性マジシャンのみず技を受けた場合、ひかりごけの効果が先に発動するため奪われない。
- タイプが変わる技は、変化後のタイプがみずタイプであるときに発動する。
- うるおいボイスの効果で、みずタイプに変わった技を受けたときも発動する。
