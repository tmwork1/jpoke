# 仕様書: からぶりほけん

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%89%E3%81%B6%E3%82%8A%E3%81%BB%E3%81%91%E3%82%93

## 効果
- 効果:持たせたポケモンが使用した技が命中判定に失敗したとき、自分のすばやさが2段階上昇する。1度発動すると無くなる。
- なげつける:威力80

## 詳細な仕様
- すでにすばやさランクが最大のときは発動しない。
- 一撃必殺技を外したときは発動しない。
- 説明文にあるとおり、命中判定に失敗して技が外れた場合のみ発動する。
 - 以下の効果による技の失敗は命中判定とは関係が無いので、からぶりほけんは発動しない。
 - 相性・特性に技を無効化されたときや、まもる状態などで防がれたとき。
 - 対象が姿を隠しているために技が当たらなかったとき。
 - 対象がすでにひんしになっていたときなど、対象の場に誰もいなかったため技が失敗したとき。
 - 以下の効果は命中判定に関わるので、からぶりほけんが発動する。
 - 所持者の特性はりきり、または相手の特性すながくれ/ゆきがくれ/ちどりあし/ミラクルスキン、もちものひかりのこな/のんきのおこうの効果で技が当たらなかったとき。
 - なかよし度の効果で技が当たらなかったとき。
 - 無効化判定の前に命中判定があるとき（例：みがわり状態に対するさいみんじゅつ）、命中判定に失敗して技が外れたなら発動する（参考：ターン#技の成功判定）。
- 連続攻撃技は最初の1発が外れると発動する。最大回数当たらなかっただけでは発動しない。
 - トリプルキック/トリプルアクセル/ネズミざんも他の連続攻撃技と同様に、最初の1発が外れたときのみ発動する。1発でも当たれば、最大回数当たらなかったとしても発動しない。
- ドラゴンアローは1体の命中判定に失敗した場合、もう片方を2回攻撃したとしても発動する。
 - 片方にタイプやまもる状態で無効化されたため、もう片方を2回攻撃したときは発動しない。
- マジックコート状態により跳ね返された技が外れた場合、マジックコートのポケモンが持つからぶりほけんが発動し、元の使用者のものは発動しない。
- 全体攻撃技を使用したときは、対象の内いずれか1体にでも回避されたなら発動する。
 - 全体攻撃の変化技を使用し、外れたポケモンとマジックコート状態のポケモンがいた場合、からぶりほけんが発動した後にマジックコートで跳ね返される。
 - 全体攻撃の攻撃技を使用し、外れたポケモンと当たったポケモンがいた場合、またはドラゴンアローが命中判定により片方を2回攻撃した場合、以下のようになる。
 - 相手のわたげやミラーアーマーなどの特性の後にからぶりほけんが発動する。
 - 相手のレッドカードが発動した場合、からぶりほけんは発動する前に交代させられる。
 - 相手のわるいてぐせが発動した場合、からぶりほけんは発動する前に奪われる。
 - マジシャンのポケモンがからぶりほけんを発動させるときは、特性が不発した後に消費する。
 - アイテムが無いマジシャンのポケモンがからぶりほけんを奪ったときは、他に技を外した相手がいたとしてもその攻撃でからぶりほけんは発動しない。
