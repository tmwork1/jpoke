# 仕様書: いかさまダイス

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%84%E3%81%8B%E3%81%95%E3%81%BE%E3%83%80%E3%82%A4%E3%82%B9

## 効果
- 効果:
 - 2~5回当たる連続攻撃技を使用したとき、攻撃回数が必ず4回以上になる。
 - トリプルアクセル・トリプルキック・ネズミざんを使用したとき、最初の1回しか命中判定を行わない。
- なげつける:威力30

## 詳細な仕様
- 2~5回当たる連続攻撃技を使ったとき、4~5回が選ばれたときは影響しないが、2~3回が選ばれたときは0~1のランダムな数を5から引いた数が攻撃回数になる[1]。つまり各50%の確率で4回か5回攻撃することになる。
 - 攻撃回数の期待値は4.5回で、通常の3.1回と比較して約1.45倍になる。
- トリプルアクセル・トリプルキックを使ったときは最初の1回しか命中判定を行わない。命中補正が変化していない場合、10%の確率で0回、90%の確率で3回になる。
 - 攻撃の期待値は2.7回で、通常の約2.439回と比較して約1.107倍になる。合計威力の期待値は "1発目の威力×5.4" で、通常の "1発目の威力×約4.71" と比較し、約1.147倍になる。
- ネズミざんを使ったときは、最初の1回しか命中判定を行わず、0~6のランダムな数を10から引いた数が攻撃回数になる。命中補正が変化してない場合、10%の確率で0回、各12.9%の確率で4~10回になる。
 - 攻撃回数の期待値は6.3回で、通常の約5.862回と比較して約1.075倍になる。
- 攻撃回数の最大値が通常より増えることはない。必ず2回攻撃になる技や、3回攻撃になる技を使用しても効果は無い。
- 所持者の特性がスキルリンクである場合、特性の効果が優先されて必ず攻撃は最大回数になる。
- 攻撃側か防御側どちらかのHPが0になったときか、防御側の特性ほうしの効果により攻撃側がねむり状態になったときはそこで攻撃が中断される (連続攻撃技#命中・回数の決定と最大威力・期待値も参照) 。

## 備考
- 性能だけ見れば「スキルリンクのダウングレード版」とも取れるが、こちらは道具であり、他の特性と組み合わせて使えるというメリットもある。
