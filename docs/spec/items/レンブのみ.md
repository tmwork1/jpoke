# 仕様書: レンブのみ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%AC%E3%83%B3%E3%83%96%E3%81%AE%E3%81%BF

## 効果
- 効果:ポケモンに持たせると、特殊技を受けたときに消費し、攻撃してきた相手ポケモンは自身の最大HPの1/8だけダメージを受ける。

## 詳細な仕様
- 特殊技の連続攻撃技を受けた場合、最初の1発が当たった直後に発動する。
- 所持者のみがわり状態で攻撃を防いだときは発動しない。
- レンブのみのダメージは対象のみがわり状態を貫通してダメージを与える。
- 所持者がやきつくすを受けた場合、発動せずに普通に消滅する(ジャポのみがはたきおとすなどを受けたときの裁定とは異なる)。
- 特性マジシャンの特殊技を受けた場合、レンブのみが先に発動するため奪われない。
- 特性マジックガードのポケモンに対しては発動せず、きのみも消費されない。
- みらいよち/はめつのねがいの攻撃を受けた場合は、使用者が場に残っていたとしても発動しない。
- 所持者がひんしになったときでも発動する。
 - 所持者がひんしになった際のレンブのみのダメージでお互いのポケモンが全滅した場合、第六世代までレンブ所持者側が勝ちとなる。第七世代以降はレンブ所持者側が負けとなる。
 - いのちがけの攻撃と、それに対して発動したレンブのダメージでお互い全滅した場合は、第七世代以降でも特別にレンブ側の勝ちとなる。
- 技を使用したポケモンがすでにひんしになっていた場合、レンブのみは発動しない。
 - てっていこうせん/ビックリヘッド/ミストバーストを受けたときや、相手が特性ヘドロえき/ゆうばく/とびだすなかみ/さめはだ/てつのトゲ/うのミサイルによるダメージでひんしになっていたときなどが該当する。
 - 相手がいのちのたまのダメージでひんしになる場合、レンブのみが発動してからひんしとなる。
 - 相手がいのちがけでひんしになる場合、第五世代までレンブのみは発動しない。第六世代以降はレンブのみが発動してからひんしとなる。
