# 仕様書: しろいハーブ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%97%E3%82%8D%E3%81%84%E3%83%8F%E3%83%BC%E3%83%96

## 効果
- 効果:持たせると、能力がマイナスになったとき0に戻す。1度発動すると無くなる。
- なげつける:威力10、相手の下がったランク補正を0に戻す

## 詳細な仕様
- ランク補正が合計で-1以下になったときにのみ発動する。2段階上がっている能力が2段階下げられたときなど、下がった結果のランクが+0以上だった場合は発動しない (だっしゅつパックの発動条件とは異なる)。
- すでにランクが下がっているポケモンがこの道具を手に入れた場合、即座に発動して能力が戻る。
- 下がった能力をバトンタッチで引き継いだポケモンがしろいハーブを持っていた場合、即座に発動して能力をリセットする。
 - このポケモンが同時にねばねばネットに引っかかった場合、すばやさが下がった後にしろいハーブを発動させて全ての下がった能力をリセットする。
- へんしん状態やじこあんじで-1以下のランク補正をコピーした場合も発動する。
- くすぐるなどで複数の能力を同時に下げられた場合、技が終わった後に発動して下がった能力全てを戻す。
- 相手の特性いかく・ぬめぬめ・カーリーヘアー・ミラーアーマー・わたげや、自身の特性くだけるよろいにより能力が下がったときも発動する。
 - 所持者のばかぢからなど自分の能力を下げる攻撃技で対象のぬめぬめ・カーリーヘアー・わたげが発動した場合、「わざの反動→相手の特性→しろいハーブ」という順で処理されて一連の流れで下がったすべての能力がリセットされる。
- なげつけるでしろいハーブをなげつけた場合、対象の下がった能力をリセットする。
 - 特性くだけるよろいのポケモンがなげつけるでしろいハーブを投げられた場合、「投げられたしろいハーブ→くだけるよろい」という順で処理されるため、この際のくだけるよろいによるぼうぎょランク低下は回復できない。
- どろぼう・ほしがる・はたきおとす、特性マジシャン・わるいてぐせの効果はしろいハーブの発動より優先される。
 - 所持者がくだけるよろいでこれらの攻撃を受けた場合、「くだけるよろい→相手の技・特性の効果」の順番に表示されるが、しろいハーブは奪われ、ランク低下は回復できない。
 - 所持者のばかぢからなど自分の能力が下がる直接攻撃で対象のわるいてぐせが発動した場合、「わざの反動→わるいてぐせ」の順番に表示されるが、しろいハーブは奪われ、わざで下がった能力は回復できない。
- 所持者が特性まけんき・かちきでありそれぞれこうげき・とくこうを下げられた場合、まけんき・かちきの発動が優先される。まけんき・かちき発動後も依然能力がマイナスでない限りしろいハーブは発動しない。
- 所持者があまのじゃくであり、Oパワーの効果を逆に受けて能力が下がった場合、しろいハーブは発動しない。
 - 何かしら技を使用すれば、それが引き金となりしろいハーブで能力をリセットできる。
 - ロトポンのおうえんポンの効果で能力が下がった場合、しろいハーブが発動し能力がリセットされる。
