# 仕様書: せいれいプレート

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%97%E3%83%AC%E3%83%BC%E3%83%88

## 効果
- 持たせたプレートのタイプに応じてそのタイプのわざの威力が1.2倍になる。
- なげつける: 威力90（こわもてプレートをなげつけた場合のみプレート自体の補正が乗り威力108になる）。
- さばきのつぶてを覚えたポケモンに持たせると、持たせたプレートのタイプに応じてさばきのつぶてのタイプが変わる。
- アルセウスに持たせると、持たせたプレートのタイプに応じてタイプシフトする（フォルムとタイプが変わる）。
 - 第四世代で他のポケモンがアルセウスにへんしんする場合、一旦相手のアルセウスと同じ色の姿になったあと、へんしん使用者が持っている道具（プレート）の種類・有無に応じたフォルムに変化する。
 - 第五世代以降ではへんしん使用者が持っている道具（プレート）の種類・有無にかかわらず、相手のアルセウスと同じフォルムになる。
 - 『LEGENDS アルセウス』では道具を持たせられないため、ポーチから直接使用してフォルムチェンジする。レジェンドプレートは素ではノーマルだが、さばきのつぶて使用時に相手の弱点を突けるタイプに変化する。選ばれるタイプは相手の弱点を突けて、相手の一致技を半減または無効化するタイプが優先的に選ばれる。相手が複数持つタイプに対して弱点を突け、なおかつ半減または無効化するタイプが複数ある場合、相手のタイプ1が優先される。
- プレートを持ったアルセウスに対するトリック・すりかえは失敗する。どろぼう・ほしがる、特性マジシャン・わるいてぐせの効果で道具を奪えない。はたきおとすの威力は上がらず道具も失わない。プレートを持ったアルセウスが使用するトリックも失敗する。
 - 第四世代ではプレートの有無にかかわらずアルセウスに対するトリック・すりかえは失敗する。アルセウスが使用するトリックも失敗する。
 - 第五世代以降はプレートを持っていないアルセウスに対するトリック・すりかえは成功し、プレートを持っていないアルセウスが使用するトリックも成功する。アルセウスに対してトリック・すりかえ・ギフトパス、特性きょうせいでプレートを渡すことはできない。

## 詳細な仕様
- タイプが変わる技（ノーマルスキン等）は、変更後のタイプがフェアリータイプであればせいれいプレートの効果がある。
