# 仕様書: かいがらのすず

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%81%84%E3%81%8C%E3%82%89%E3%81%AE%E3%81%99%E3%81%9A

## 効果
- 効果:攻撃技を使用した後、相手に与えたダメージの1/8分だけ自分のHPが回復する（小数点以下切捨て）。
- なげつける:威力30

## 詳細な仕様
- 回復量を算出するとき小数点は切り捨てるが、1以上のダメージを与えた際には1回復できる。
- 攻撃が無効化されたときや、HP1のポケモンをみねうちで攻撃したとき、相手のばけのかわ/アイスフェイスで攻撃を肩代わりされたときなど、与えたダメージが0だった場合は回復しない。
- 連続攻撃技を使用した場合、回復は最後の攻撃の後に発動する。回復量は、第三世代のみ最後のダメージのみから、第四世代以降は全ての攻撃のダメージの合計から算出する。
- 全体攻撃技で複数のポケモンに同時にダメージを与えた場合、全員に与えたダメージを合計してから回復量を算出する。
- 特性ヘドロえきの影響を受けない。かいがらのすずによる回復効果は吸収技による回復効果とは異なる。
- 発動したとき、「<所持者>は かいがらのすずで すこし かいふく」というメッセージが出る。
- こんらんで自傷ダメージを受けたときは発動しない。
- みらいよち/はめつのねがいによるダメージでは発動しない。
- 特性ちからずくのポケモンが特性の対象技を使うときは、かいがらのすずの回復効果も無くなる。
- 吸収技による効果、さめはだなどの特性やゴツゴツメット/ジャポのみ/レンブのみによるダメージ、反動技によるダメージの後に回復効果が発動する。
- 反動技や、ひんしになる攻撃技などの効果で攻撃側のHPが0になったときは、回復効果が発動しないままひんしになる。
- 相手のレッドカードが発動して攻撃側が交代させられた場合、回復はできない。
- とんぼがえり/ボルトチェンジ/クイックターンを使用した場合、回復の後に交代する。
- どろぼう/ほしがる/マジシャンで相手のかいがらのすずを奪った場合、即座にその際のダメージに応じた回復効果が発動する。
- わるいてぐせのポケモンを直接攻撃してかいがらのすずを奪われる場合、攻撃者がかいがらのすずで回復した後にわるいてぐせが発動する。
- おどりこで踊りの攻撃技をコピーして使ったときも発動する。

### 第五世代以降
- 交代前のポケモンへのおいうちでも発動する。
- かいふくふうじ状態では回復できない。
- 攻撃がみがわり状態に防がれた場合も回復できる。みがわりに与えたダメージから回復量を求める。
- 一撃必殺技が成功したとき、第五世代・第六世代では発動しない。第七世代以降では発動する。
