# 仕様書: ヒメリのみ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%92%E3%83%A1%E3%83%AA%E3%81%AE%E3%81%BF

## 効果
- 効果: ポケモン1匹の1つの技のPPを10回復する。ポケモンに持たせると戦闘中にPPが0になったわざのPPを10回復する。

## 詳細な仕様
- ヒメリのみを持つポケモンが戦闘に出たときにPPが0の技があった場合、戦闘に出た直後に使う。
- へんしん状態のポケモンが持っている場合、へんしん前の技のPPが0でも発動しない。へんしん後の技のPPが0になったら発動する。
- うらみ/ぶきみなじゅもんの効果でPPが0になったときは発動する。
- キョダイゲンスイの効果でPPが0になったときは発動しない。技を受けたポケモンも、その味方のポケモンも同様。
 - キョダイゲンスイを受けたターンでPPが0になった技を使おうとして、使用できなかったときはその直後に使用する。
 - キョダイゲンスイを受けたターンでPPが0になった技以外を使った場合、行動を完了してもヒメリのみは使用されない。
- なげつけるでヒメリのみを投げた場合、相手の技のPPを10回復する。PPが減っていないと効果は発動しない。
- むしくい・ついばむで相手のヒメリのみを食べた場合、自分の技のPPを10回復する。その際技のPPを消費した後に効果が発動する。
- PPが減っている技が複数あるポケモンが、なげつける・むしくい・ついばむ・ほおばる・おちゃかいによるヒメリのみの効果を受ける場合、回復対象の優先順位は世代により変わる。
 - 第四世代では、最も多く消費されている技のPPを回復する。最も多く消費されている技が複数ある場合、その中で一番上のスロットにある技が対象になる。
 - 第五世代以降では、減っている技のうち一番上のスロットにある技のPPを回復する。ただし、PP0の技がある場合はその技が優先される。PP0の技が複数ある場合、最後に使われた技かつPP0の技があればその技が対象になり、そのような技がなければPP0の技のうち一番上のスロットにある技が対象になる。
 - 例えば、なげつけるを受けたポケモンが覚えている技の残りPP/最大PPが、上から (10/10, 9/10, 13/15, 3/5) の順番であるとする。第四世代では一番多く2消費されている3番目・4番目の技の内、上にある3番目の技を回復する。第五世代以降では1番目の技が消費されていないので2番目の技を回復する。

## 備考
- 持たせた場合に自分で使ってくれることを除けば、ピーピーエイドと同じ効果。
