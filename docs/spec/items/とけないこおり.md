# 仕様書: とけないこおり

調査日: 2026-07-04
参照URL: https://wiki.pokemonwiki.com/wiki/とけないこおり

## 効果
- 持たせると、こおりタイプの技の威力が1.2倍になる（第四世代以降）。
- なげつける:威力30

## 詳細な仕様
- タイプが変わる技（わざレンジャー等）や、フリーズスキン特性の効果でタイプが変わった技は、変更後のタイプがこおりタイプであればとけないこおりの効果がある。
