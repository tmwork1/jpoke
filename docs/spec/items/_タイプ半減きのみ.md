## ヤチェのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%A4%E3%83%81%E3%82%A7%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、効果バツグンのこおりわざを受けたときに消費して、1度だけダメージを1/2にする。

### 詳細な仕様
- みずタイプのポケモンがフリーズドライを抜群で受けたときも発動する。
- 特性あついしぼうによる効果は相性を変えるものではないため、ヤチェのみの発動に影響を与えない。

## ソクノのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%BD%E3%82%AF%E3%83%8E%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、効果バツグンのでんきわざを受けたときに消費して、1度だけダメージを1/2にする。

### 詳細な仕様
- そうでん・プラズマシャワー・エレキスキンの効果ででんきタイプになった攻撃技は、元のタイプに関係なくソクノのみの効果対象になる。
 - でんきタイプになったどろぼう・ほしがる・ついばむ・むしくい・やきつくすの攻撃を抜群で受けた場合、ソクノのみの効果が優先して発動してダメージを半減し、技の効果は受けない。
 - でんきタイプになったはたきおとすを抜群で受けた場合、はたきとおすの威力は1.5倍になるが、ソクノのみの効果でダメージは半減できる。
 - でんきタイプになったポルターガイストの攻撃を受けた場合、ソクノのみでダメージを半減して攻撃を受ける。

## ホズのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9B%E3%82%BA%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、ノーマルタイプの攻撃技を受けたときに消費して、1度だけダメージを1/2にする。

### 詳細な仕様
- ホズのみの効果が発動するため、通常はほしがるでは奪えない。技のタイプが変わる特性や技の効果でほしがるがノーマルタイプでなくなったときや、あくタイプのどろぼうを受けたときは奪われる。
- ノーマルスキンの効果でノーマルタイプになったついばむ・むしくい・はたきおとすを受けた場合、技の効果よりホズのみの効果が優先されてダメージを半減できる。第六世代以降では、このときもはたきおとすの威力は1.5倍になる。
- わるあがきを受けたときは、第四世代では発動する。第五世代以降では発動しない。

### 備考
- 全きのみで唯一しぜんのめぐみ使用時にノーマルタイプで攻撃する。
- ノーマル技が効果抜群になるタイプはさかさバトルルールであるときを除き存在しないため、他の半減系きのみと違いノーマル技を受けたら相性に関係なく発動する。

## ヨプのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%A8%E3%83%97%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、効果バツグンのかくとうわざを受けたときに消費して、1度だけダメージを1/2にする。

### 詳細な仕様
- フライングプレスが効果抜群になるとき発動するきのみはヨプのみであり、バコウのみは影響を与えない。
 - ひこうタイプを複合した相性が抜群になるときのみヨプのみが発動する。例えばはがね単タイプのポケモンにヨプのみを持たせてフライングプレスを受けても等倍ダメージで受け、半減にはならない。
 - ひこうタイプによる相性でフライングプレスが抜群になるときでも、ヨプのみの効果でダメージが半減する。例えばくさ単タイプのポケモンがフライングプレスを受けたときなど。
