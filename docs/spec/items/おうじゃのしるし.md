# 仕様書: おうじゃのしるし

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8A%E3%81%86%E3%81%98%E3%82%83%E3%81%AE%E3%81%97%E3%82%8B%E3%81%97

## 効果
- 効果:
 - ポケモンに持たせると、攻撃したとき10%の確率で相手をひるませる。
 - ニョロゾ・ヤドン（通常のすがた）に持たせて通信交換すると、それぞれニョロトノ・ヤドキング（通常のすがた）に進化する。このときおうじゃのしるしは消費される。
- なげつける:威力30、相手をひるませる

## 詳細な仕様
- 技のダメージが無効化されたときはおうじゃのしるしの効果は無い。
 - みがわり状態に攻撃を防がれた場合も効果は無い。
- 持たせたポケモンが連続攻撃技を使用した場合、第二世代では最後の1回だけにひるみ効果がある。第三世代以降では1回のヒットそれぞれに10%のひるみ効果が発動する。
 - 連続攻撃のヒット数をnとすると、ひるむ確率は (1-(1-0.1)n)×100 = (1-0.9n)×100 (%)となる。5回当たった場合は41%になる。
- 持たせたポケモンが、元々ひるみの追加効果がある技を使用した場合の仕様は、世代により異なる。
 - 第四世代までは効果が累積する。例えばエアスラッシュを使用した場合、ひるむ確率は (1-(1-0.3)×(1-0.1))×100 = 37 (%) になる。
 - 第五世代以降ではおうじゃのしるしの効果が発動しない。例えばエアスラッシュを使用した場合も、ひるむ確率は30%のまま変わらない。
- 一撃必殺技をきあいのタスキなどにより耐えられたときはにおうじゃのしるしの効果が無い。
- おうじゃのしるしの効果は特性りんぷんにより無効化される。
- 所持者の特性てんのめぐみは第四世代までは影響を受けなかったが、第五世代以降は影響を受け、ひるむ確率は20%になる。
 - 例えば特性てんのめぐみのポケモンがおうじゃのしるしを持ってエアスラッシュを使用した場合、ひるむ確率は第四世代までは (1-(1-0.3×2)×(1-0.1))×100 = 64 (%) になるが、第五世代以降はおうじゃのしるしの効果が無いので60%になる。
- にじ状態の効果を受けると、ひるむ確率は20%となる。
 - てんのめぐみ状態とにじ状態の効果を同時に受けてもひるむ確率は20%のまま変わらない。
- 所持者が特性あくしゅうであってもひるむ確率は10％のまま変わらない。
- 所持者が特性ちからずくであっても、おうじゃのしるしの効果で技の威力が増えることはなく、おうじゃのしるしによるひるみ効果は発動する。元々ひるみ効果を持つためおうじゃのしるしの効果が無い技は、ちからずくのポケモンが使用した場合でもひるみ効果は無い。
- なげつけるを使っておうじゃのしるしを投げた場合は、相手を100%の確率でひるませる追加効果が発生する。

## 備考
- 全く同じ効果の道具にするどいキバがある。なげつけるで使用したときの威力・効果も同じ。進化するポケモン及び進化方法が異なるのみ。
