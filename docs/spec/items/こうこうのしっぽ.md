# 仕様書: こうこうのしっぽ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%93%E3%81%86%E3%81%93%E3%81%86%E3%81%AE%E3%81%97%E3%81%A3%E3%81%BD

## 効果
- 効果:持たせたポケモンは、同じ優先度の行動の中で最後に行動する。
- なげつける:威力10

## 詳細な仕様
- 優先度が同じ技を使ったポケモン同士の中で一番行動順が遅くなる。相手が自分より優先度が低い技を使っていた場合、自分が先制することになる。
- トリックルーム状態でも関係なく後攻になる。
- こうこうのしっぽが譲渡されたり消滅したりしてもそのターンの行動順には影響せず、次のターンから影響が出る。これは、すばやさや優先度の変化がそのターンの行動順に即座に反映される第八世代においても変わらない。
- こうこうのしっぽを持ったポケモンの特性クイックドロウが発動した場合、道具の効果は無視されて行動順が早くなる。同じターン他にクイックドロウ・せんせいのツメ・イバンのみを発動させたポケモンがいても、すばやさが高いポケモンから行動する。
- 行動順の判定については、ターン#4.ポケモンの行動も参照とする。

### 第五世代以降
- ターン終了時の状態異常の発動順や、ポケモンチェンジの順番にこうこうのしっぽは影響しない。(まんぷくおこうも同様)
- こうこうのしっぽ (まんぷくおこう)を持ったポケモン同士では、すばやさの高い方が先に行動する。
- 特性あとだし・きんしのちからの効果で行動が遅くなったポケモンとは、すばやさの高い方が先に行動する。
- トリプルバトルではMOVEの使用順も遅くなる。
- おさきにどうぞ・りんしょう・コンビネーションわざ・トラップシェルによる行動順を引き上げる効果は、こうこうのしっぽの効果よりも優先される。(まんぷくおこうも同様)
 - こうこうのしっぽを持つポケモンを含めた3体以上のポケモンがりんしょうを選択していた場合、行動を引き上げられた2体目以降のポケモンの行動順にはこうこうのしっぽの効果があり、こうこうのしっぽを持つポケモンの行動順が最も遅くなる。
 - こうこうのしっぽを持つポケモンを含めた3体以上の味方が誓い技を選択していた場合、行動が引き上げられる2体目のポケモンには、こうこうのしっぽを持っていないポケモンが優先される。

## 備考
- まんぷくおこうとはバトルにおいてこうこうのしっぽと同じ効果を持つ。なげつけるの威力もどちらも10で差は無い。
