# 仕様書: きあいのタスキ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8D%E3%81%82%E3%81%84%E3%81%AE%E3%82%BF%E3%82%B9%E3%82%AD

## 効果
- 効果:HPが満タンのとき一撃でひんしになる攻撃技のダメージを受けても、HPを1残して持ちこたえる。1度発動すると無くなる。
- なげつける:威力10

## 詳細な仕様
- 第八世代までは、野生ポケモン戦、トレーナー戦、マックスレイドバトルで発動すると戦闘終了後も戻ってこない。
 - 通信対戦や経験値が入らない対戦施設では、発動しても戦闘が終わった後に手元に戻る。
 - 第九世代では、野生ポケモン戦や、経験値が入るトレーナー戦、テラレイドバトルで発動したときも、戦闘が終われば手元に戻る。
- 僅かでもHPを減らされると効果がなくなるが、その後HPを回復して再び満タンになれば発動可能になる。
- どろぼう・ほしがる・はたきおとす・特性マジシャンの攻撃を受けた場合、技や特性の効果より先にきあいのタスキが発動する。きあいのタスキが消費されると奪うことができない。
- 特性わるいてぐせと同時に発動条件を満たした場合、きあいのタスキを消費した後にわるいてぐせで相手の道具を奪う。
- ダメージ固定技や一撃必殺技のダメージも耐える。
- こんらんによる自分への攻撃も耐える。
- 状態異常・こんらん以外の状態変化によるダメージは耐えない。
- 自滅技を使用したときの効果、みちづれ・ほろびのうたでひんしにさせる効果に対してきあいのタスキは発動しない。
- 反動技の反動ダメージ、はじけるほのおの火花によるダメージは耐えない。
- 連続攻撃技を受けた場合、第四世代では全ての攻撃を耐える。第五世代以降は最初の1発目に対してのみ発動し、2発目以降は耐えない。
- みらいよち・はめつのねがいの攻撃を受けた場合も発動する。
- 交代際においうちの攻撃を受けた場合、プラチナまでは発動しない。ハートゴールド・ソウルシルバーからは発動する。
- こらえる状態だった場合、技の効果が先に発動してきあいのタスキは消費されない。
- 第五世代以降で、所有者が特性がんじょうのポケモンである場合、がんじょうが優先して発動してきあいのタスキは消費されない。
- なかよし度レベルが3以上である場合でも、きあいのタスキが発動する場合はきあいのタスキが先に発動して消費される。
