# 仕様書: しめつけバンド

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%97%E3%82%81%E3%81%A4%E3%81%91%E3%83%90%E3%83%B3%E3%83%89

## 効果
- 効果:持たせたポケモンがバインド状態で与えるダメージが最大HPの1/6になる（第五世代以前は最大HPの1/8）。
- なげつける:威力30

## 詳細な仕様
- 道具の効果は相手をバインド状態にするときに持っていなければ発動しない。
 - しめつけバンドを持っていないポケモンが、相手をバインド状態にしてからしめつけバンドを入手してもダメージは増えない。
 - しめつけバンドを持っているポケモンが、相手をバインド状態にしてからしめつけバンドを失ったとしてもダメージは減らない。
 - しめつけバンドを持ってキョダイサジン・キョダイヒャッカを使用したポケモンが場からいなくなってもダメージは減らない。
