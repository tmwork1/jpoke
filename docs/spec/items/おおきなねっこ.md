# 仕様書: おおきなねっこ

調査日: 2026-06-06

## 効果
- 効果:持たせたポケモンは、以下の効果によりHPが回復する量が1.3倍になる。
 - HPを吸収する攻撃技、ちからをすいとるを使用したときの回復量。
 - アクアリング状態、ねをはる状態での回復量。
 - 相手がやどりぎのタネ状態により自分が回復する量。
- なげつける:威力10

## 詳細な仕様
- 通常時の回復量を算出し、端数処理も行った後におおきなねっこの効果を計算する。
- 小数点以下は第四世代では切り捨て、第五世代以降は五捨五超入する。
- 第五世代以降では、正確な倍率は5324/4096倍になる。
- 相手の特性ヘドロえきにより回復効果がダメージ効果に変わったときも、受けるダメージが1.3倍になる。
- やどりぎのタネ状態による回復量は、回復するポケモン自身がおおきなねっこを持っているかによって決まる。やどりぎのタネ使用者だけが持っていても影響しない。
- HPを吸収する攻撃技ややどりぎのタネ状態で与えるダメージに関しては変化しない。
- グラスフィールド状態での回復量は増えない。
