# 仕様書: ナゾのみ

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%8A%E3%82%BE%E3%81%AE%E3%81%BF

## 効果
- 効果:
 - 第三世代:どろぼう・ほしがる・トリックの効果を受けない。
 - 第四世代以降:ポケモンに持たせると、効果バツグンの攻撃技でダメージを受けた直後に消費して、最大HPの1/4だけ回復する (小数点以下切り捨て)。ひんしになったときは発動しない。

## 詳細な仕様
### 第四世代以降
- ほしがる/どろぼう/ついばむ/むしくい/はたきおとす/やきつくす、特性マジシャンの攻撃が弱点であった場合、以下のようになる。
 - 第四世代では、技の効果が優先されてナゾのみの効果は発動しない。
 - 第五世代以降では、技の効果より先にナゾのみの効果が発動しHPを回復する(ほしがるはノーマルタイプであるため、技のタイプが変わったときやさかさバトル限定)。弱点でなければナゾのみは発動しないので、これらの効果で失われる。
 - 第六世代以降では、このときもはたきおとすの威力は1.5倍になる。
- ついばむ/むしくいで食べたとき、第七世代までは使用者のHPは回復しない。第八世代では使用者の最大HPの1/4が回復する。
- ほおばる/おちゃかい/はんすうで食べたときも最大HPの1/4が回復する。
- なげつけるで使用しても、対象のHPは回復しない (第八世代/第九世代でも同様)。
- 弱点となる連続攻撃技を受けた場合、1発目を受けた直後に発動する。
- 弱点のタイプの一撃必殺技をこらえるなどにより耐えた場合、ナゾのみは発動する。一撃必殺技以外のダメージ固定技を受けたときは、弱点のタイプであっても発動しない。
- みがわり状態で攻撃を防いだときは発動しない。
- ばけのかわ/アイスフェイスでダメージを肩代わりしたときは発動しない。
- らんきりゅうによりひこうタイプの相性が変化した場合は発動に影響がある。弱点技を等倍にして受けた場合は発動せず、らんきりゅう状態でなお抜群であった場合は発動する。
- タールショット状態によりほのお技に対する相性が変化した場合は発動に影響がある。
