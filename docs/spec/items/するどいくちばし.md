# 仕様書: するどいくちばし

調査日: 2026-07-04
参照URL: https://wiki.pokemonwiki.com/wiki/するどいくちばし

## 効果
- 持たせると、ひこうタイプの技の威力が1.2倍になる（第四世代以降）。
- なげつける:威力50

## 詳細な仕様
- タイプが変わる技（ノーマルスキン・スカイスキン等）は、変更後のタイプがひこうタイプであればするどいくちばしの効果がある。
- フライングプレスはかくとうタイプの技とみなされ（相性計算のみひこうタイプが追加される）、するどいくちばしの効果は無い。
