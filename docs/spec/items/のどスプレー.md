# 仕様書: のどスプレー

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AE%E3%81%A9%E3%82%B9%E3%83%97%E3%83%AC%E3%83%BC

## 効果
- 効果:音のわざを使用した後、自分のとくこうを1段階上げる。1回発動すると無くなる。
- なげつける:威力30

## 詳細な仕様
- 消費するのは技の使用後なので、特殊攻撃の音技を使用して発動させてもその際の攻撃のダメージは上がらない。
 - 味方の特性きょうせいがのどスプレーの消費に対して発動するのも技の使用後になる。
- すでにとくこうが最大まで上がっている場合は消費しない。
- 相性・特性・まもるなどで技を無効化されたときや、うまく決まらなかったとき、命中しなかったときは消費しない。
- マジックコート状態で音の変化技が跳ね返された場合、マジックコート側ののどスプレーが発動する。他に技が有効なポケモンがいなかった場合、跳ね返された側ののどスプレーは発動しない。有効なポケモンがいた場合は、そのポケモンに技を使用しのどスプレーが発動してからマジックコートの効果が発動する。
 - ミラーアーマーでいやなおとなどの効果を跳ね返した場合は、技の使用者側ののどスプレーが発動する。ミラーアーマー側ののどスプレーは発動しない。
- すてゼリフが成功して交代した場合、のどスプレーは消費されずに技の効果で交代する。すてゼリフが成功したが交代先がいないため使用者が場に残った場合はのどスプレーを消費する。
- いやしのすずを使用した場合、「鈴の音が 響きわたった！」のメッセージが出ただけではなく、手持ちのポケモンの状態異常を治す効果が発動しなければのどスプレーは発動しない。
- 音の攻撃技に対して相手のレッドカードが発動して交代させられた場合、のどスプレーは消費されずにレッドカードの効果で交代する。
- 特性マジシャンの発動判定は、のどスプレーの消費判定前に行われる。
 - マジシャンのポケモンがのどスプレーを持ちながら音の攻撃技を使った場合、マジシャンが不発した後にのどスプレーを消費する。
 - マジシャンのポケモンがのどスプレーを持つ相手に音の攻撃技を使った場合、マジシャンで奪ったのどスプレーを即座に消費する。
- 野生ポケモンに対してほえるを使い戦闘を終わらせると、のどスプレーが消費されてから戦闘が終わる。
- 野生ポケモン戦やトレーナー戦で、音の攻撃技で相手を倒して戦闘が終わった場合は、消費されずに戦闘が終わる。
