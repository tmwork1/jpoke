# 仕様書: ぎんのこな

調査日: 2026-07-04
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8E%E3%82%93%E3%81%AE%E3%81%93%E3%81%AA

## 効果
- 効果:持たせると、むしタイプの技の威力が1.2倍になる。
- なげつける:威力10。

## 詳細な仕様
- 威力補正は技のタイプがむしタイプであるかどうかで判定する（他の道具の
  タイプ強化アイテムと同様、`ON_CALC_POWER_MODIFIER` で `move.type` を参照して判定する）。

## 備考
- 類似の効果を持つアイテムに他のタイプ強化アイテム（もくたん・じしゃく・りゅうのキバ 等）がある。

## 関連項目
- どくバリ - 同系統のタイプ強化アイテム
- なげつける - ぎんのこなを投げた場合の威力
