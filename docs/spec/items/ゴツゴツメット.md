# 仕様書: ゴツゴツメット

調査日: 2026-06-06
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%B4%E3%83%84%E3%82%B4%E3%83%84%E3%83%A1%E3%83%83%E3%83%88

## 効果
- 効果:直接攻撃でダメージを受けたとき、攻撃してきた相手に対して相手の最大HPの1/6分のダメージを与える。(小数点以下切り捨て。ただし最低でも1以上)。
- なげつける:威力60

## 詳細な仕様
- 効果が発動する際、「<攻撃者>は ゴツゴツメットで ダメージを受けた!」というメッセージが流れる。
- 特性マジックガードのポケモンはゴツゴツメットのダメージを受けない。このとき、ゴツゴツメットやマジックガードの所有を明示するメッセージは流れない。
- アイテムぼうごパットを持つポケモンはゴツゴツメットのダメージを受けない。このとき、お互いの所有するアイテムを明示するメッセージは流れない。
- 特性えんかくやアイテムパンチグローブの効果で直接攻撃ではなくなった技に対しては発動しない。
- 直接攻撃を受けても、相性/特性/まもる状態などで攻撃を無効化したときは発動しない。
 - ばけのかわ/アイスフェイスでダメージを肩代わりしたときは発動する。
- みがわり状態で攻撃を防いだときは発動しない。
- みがわり状態のポケモンから直接攻撃を受けた場合、みがわり状態を貫通して本体にダメージを与える。
- 特性がさめはだ/てつのトゲである場合、特性によるダメージの後にゴツゴツメットによるダメージが発生し、合計で最大HPの7/24のダメージを与える。途中で相手がきのみの発動圏に入ったときは回復を挟むことができる。
- ゴツゴツメットを持つポケモンがひんしになったときも発動する。
 - このときのダメージで相手のポケモンもひんしになり、お互いのポケモンが全滅した場合は、サン・ムーンのVer.1.0までは攻撃側が先にひんしになり、ゴツゴツメット側の勝利となる。サン・ムーンのVer.1.1以降はゴツゴツメット側が先にひんしになり、ゴツゴツメット側の敗北となる。
- 連続攻撃技に対しては、1発ダメージを受けるたびに発動する。
 - 途中で使用者がひんしになると攻撃は中断する。相手の特性がスキルリンク/おやこあいだったとしても同様に中断する。
- HPを吸収する効果がある直接攻撃を受けた場合、回復の後にゴツゴツメットのダメージが入る。
- とんぼがえり/クイックターンを受けたが、使用者がゴツゴツメットのダメージでひんしになったときは、技の効果は発動しない。
- 以下の技を受けた場合、ゴツゴツメットのダメージの後に技の効果が発動する。ゴツゴツメットで使用者がひんしになったときは技の効果が発動しない。
 - こうそくスピン/キラースピン (設置技を解除する効果)
 - どろぼう/ほしがる/はたきおとす
 - きつけ/めざましビンタ
 - ドラゴンテール/ともえなげ
 - プラズマフィスト
 - アイススピナー
 - 特性マジシャンによるアイテムを奪う効果
- どろぼう/ほしがる/はたきおとす/特性マジシャンの直接攻撃技を受けた場合、相手にダメージを与えた後でゴツゴツメットは失われる。

## 備考
- 実装上の既知の制限: ばけのかわ/アイスフェイスがダメージを0に変換して肩代わりした場合、
  現在の実装では`Event.ON_DAMAGE_HIT`自体が発火しないため、ゴツゴツメットは発動しない
  （本来の仕様では発動するべき）。この修正には技実行フロー（`MoveExecutor._execute_hit`）
  および該当特性ハンドラの横断的な変更が必要なため、本アイテム単体のレビューでは未対応。
  詳細は`docs/plan/items/ゴツゴツメット.md`を参照。
