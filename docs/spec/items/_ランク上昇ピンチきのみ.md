## チイラのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%81%E3%82%A4%E3%83%A9%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、残りHPが1/4以下となったときに消費して、こうげきが1段階上がる。
 - ロメのみ+ヤチェのみ→チイラのみ
 - チイラのみ+リュガのみ→アッキのみ

### 詳細な仕様
- 所持者がひんしになったときは発動しない。
- すでにこうげきランクが最大であるときは、残りHPが1/4以下でも発動しない。
- 特性くいしんぼうのポケモンはHP1/2以下で発動できる。
- なげつける・むしくい・ついばむ・ほおばる・おちゃかいで消費するときは、残りHPに関わらず効果が発動する。
- 第四世代以降では、すでにHPが発動圏内の所持者が場に出たときは即座にきのみが発動する。
 - 設置技や、いやしのねがい・みかづきのまいの効果が発動した後にきのみの発動判定がある。

#### 第五世代以降
- 以下のタイミングで発動判定がある。
 - 所持者が場に出たとき。
 - 所持者のHPが減少したとき (ただし、こんらんの自傷ダメージを除く)。
 - アイテムを手に入れたときや、アイテムの効果が有効になったとき。
- 攻撃技のダメージを受けてHPが発動圏内になったときは、基本的に技や特性の効果が発動した後にきのみの発動判定がある。(ターン#技の効果も参照)
 - 追加効果、ダイマックスわざの効果、コアパニッシャー・やきつくす・クリアスモッグの効果、特性せいぎのこころ・いかりのつぼ・さまようたましいの効果の後にきのみの発動判定がある。
 - 連続攻撃技を受けたときは、1発ごとにきのみの発動判定が行われる。特性おやこあいの攻撃を受けたときも同様となる。
 - はたきおとす・むしくい・ついばむ・どろぼう・ほしがる・特性マジシャンの効果はきのみより優先して発動する。ただし、おやこあいで2回攻撃になったときは除く。
 - ドラゴンテール・ともえなげで交代させられたときは発動しない。
 - 特性ききかいひ・にげごし・わるいてぐせ・おどりこに関しては、きのみの消費判定より後に特性が発動する。
 - 特性きょうせいは、第六世代のみ味方が道具を消費した直後に割り込んで発動する。第七世代以降ではきのみの消費判定より後に特性が発動する。
- 攻撃技を使用したポケモンが、さめはだやゴツゴツメットなどのダメージを受けてHPが発動圏内になったときは、即座にきのみが発動する。
- みがわり・のろい(呪い)を使用してHPが発動圏内に入ったときも発動する。いたみわけを使用したときや、受けたときも発動判定がある。
 - はらだいこ・ソウルビートを使用したときは、先にランクが上がった後にHPが減少する。技の効果でランクが最大になれば、きのみは発動しない。
- こんらんの自傷ダメージを受けたときは、HPが発動圏内になっても発動しない。
 - すでにHPが発動圏内になっているときにこんらんの自傷ダメージを受けたときも発動しない。
 - ポケモンがお願いをきいてくれずに自分を攻撃して発動圏内になったときも、同様にして発動しない。
- ランクが最大できのみを発動できなかった場合、HPが発動圏内のときにランクが変化技や特性の効果で下がったとしても発動しない。ただし、攻撃技の効果で最大だったランクが下がった場合は、即座にきのみが発動する。
- こんらんやランクが最大だったことによりきのみが発動できなかった場合、上記の発動判定がされるまでの間きのみが使えない。
 - 変化技を受けたときや、状態異常になったときも発動しない。発動圏内に収まるHP回復が起きても発動しない。
 - こんらんのダメージを除けば、攻撃技のダメージ、天候や状態異常、いたみわけなど何かしらの要因でHPが減少すれば、そのときに発動する。
- 場に出たときに発動条件を満たしていた場合、いかくなど多くの場に出たときに発動する特性とはすばやさ順に発動判定がある (ターン#1.ポケモンを繰り出すも参照)。
 - 自身がこれらの特性を持つ場合、特性の後にきのみが発動する。
- 特性たんじゅん・じゅくせいのポケモンはランクが2段階上がる。
- 特性あまのじゃくのポケモンはランクが1段階下がる。ランクが最大のときは発動するようになるが、ランクが最小のときは発動しなくなる。

## アッキのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%A2%E3%83%83%E3%82%AD%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、物理技を受けた直後に使用して、ぼうぎょを1段階上げる。

### 詳細な仕様
- 物理技を受けたが、相性・特性・まもる状態などの効果でダメージが無かったときは発動しない。
- みがわり状態で物理技を防いだときは発動しない。
- すでに防御ランクが最大まで上がっているときは発動しない。
- ちからずくの効果が発動した技を受けたときは発動しない。
- ばけのかわ/アイスフェイスでダメージを肩代わりしたときは、第八世代までは発動する。第九世代では発動しない。
- HP1のときにこらえるやみねうちなどの効果で物理技を耐えたため、受けたダメージが0になったときは発動する。これは第九世代でも同様となる。
- アッキのみの効果は特性あまのじゃく/たんじゅんの影響を受ける。
 - あまのじゃくがアッキのみを使用すると防御が下がる。自身で下げたとみなされ、しろいきりや特性フラワーベールの効果でも防げない。
 - かたやぶりの効果がある技でアッキのみが発動した場合、あまのじゃく/たんじゅんの効果はない。
- 物理技の連続攻撃技を受けた場合、全ての攻撃がヒットした後に効果が発動する。
- どろぼう/ほしがる、特性マジシャンの物理技を受けた場合、アッキのみの効果は発動せずに奪われる。
- はたきおとすを受けた場合、アッキのみの効果は発動せず失われる。
- ついばむ/むしくいを受けた場合、食べられて相手の防御が1段階上がる。
- ポルターガイストを受けた場合、技が成功した後にアッキのみが発動する。
- ほおばる/おちゃかい、特性はんすうの効果で食べたときも防御が1段階上がる。

## スターのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%B9%E3%82%BF%E3%83%BC%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、残りHPが1/4以下となったときに消費して、こうげき・ぼうぎょ、とくこう、とくぼう、すばやさの内いずれか1つが2段階上がる。

### 詳細な仕様
- 上がる能力や上昇量を除くと、チイラのみ系統のきのみと共通する仕様が多い。チイラのみ#詳細な仕様も参照とする。
- 上がる能力はランダムに選ばれる。
- すでにランクが最大になっている能力は、上がる能力に選ばれない。
 - ランクが+5段階の能力は選ばれるため、結果1段階しか上がらないことはある。
- 命中率・回避率のランクは選ばれない。
- 5箇所の能力全てがすでに最大になっているときは発動しない。

#### 第五世代以降
- 5箇所の能力全てが最大できのみを発動できなかった場合、HPが発動圏内のときにランクが変化技や特性の効果で下がったとしても発動しない。
- こんらんの自傷ダメージを受けたときは、HPが発動圏内になっても発動しない。
- 特性たんじゅん・じゅくせいのポケモンはいずれかのランクが4段階上がる。
- 特性あまのじゃくのポケモンはいずれかのランクが2段階下がる。ランクが最大の能力は選ばれるようになるが、ランクが最小の能力は選ばれない。5箇所の能力全てが最小のときは発動しない。

### 備考
- ランクを2段階上げる効果がある唯一のきのみとなる。

## イバンのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%A4%E3%83%90%E3%83%B3%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、HPが1/4以下のときに技を選択した直後に消費し、行動順が早くなる。ただし、わざの優先度は変わらない。

### 詳細な仕様
- きのみを消費するタイミングは、HPが1/4以下の状態で技を選択した直後となる。このときターンの最初にきのみを食べるアニメが発動し、「<所持者>は イバンのみで 行動が はやくなった！」とアナウンスが流れる。発動のタイミングは他のポケモンが交代するより前となる。
 - 技選択時はきのみを食べていないため、ゲップを選択することはできない。
 - 技選択時はきのみを持っているため、第七世代まではそのターンの行動順にかるわざは考慮されない。第八世代からは発動させたターンから行動順に影響が出る。
 - 特性ほおぶくろのポケモンは、アナウンスが流れた後、交代より前に発動させてHPを回復する。
 - 相手の場にいるきんちょうかん・じんばいったいのポケモンが交代で引っ込んだターンでは発動しない。技選択時は相手の場にいなかったが、交代でそのポケモンが出てきたターンでは発動する。
- 特性くいしんぼうのポケモンの場合、HPが1/2以下になったときに発動する。
- 所持者が交代を選んだときは発動しない。
 - バッグからもちものを使ったときも発動しない。
 - 第四世代のみ、相手も自分も交代を選んだときはすばやさや特性あとだしの効果に関わらず先制で交代でき、イバンのみを消費しない。
- 所持者がにげる/降参を選んだときは発動しない。
 - 相手がにげる/降参を選んだが、所持者が技を選択していたなら発動する。
- 第四世代のみ、相手が交代し自分は技を選んだときや、自分と相手の選んだ技の優先度が異なり先攻が確定しているときは発動しない。第五世代以降はこのような場合でも発動する。
- トリプルバトルでMOVEを選択したときは発動する。
- すばやさが異なる複数のポケモンのイバンのみ・せんせいのツメ・クイックドロウが同時に発動した場合、アナウンスは行動順に流れる。同じ優先度ならばすばやさが高い方が先に、優先度が異なれば優先度が高い順にアナウンスされる。
 - すばやさが同じ複数のポケモンのイバンのみ・せんせいのツメ・クイックドロウが同時に発動した場合、アナウンスと行動順は必ずしも一致しない。
 - イバンのみを持つポケモンが特性クイックドロウを発動させた場合、HPが1/4以下のときでもイバンのみは消費されない。HP1/4以下でクイックドロウが発動しなかったときにイバンのみが発動する。
- トリックルーム状態でも行動順が早くなる。
- 使用する技の優先度が0以下であるなら、イバンのみの効果で先制した行動はじょおうのいげん・ビビッドボディ・サイコフィールド・ファストガードの効果では防がれない。
- 行動順の判定について、詳しくはターン#4.ポケモンの行動を参照とする。

## ミクルのみ

参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9F%E3%82%AF%E3%83%AB%E3%81%AE%E3%81%BF

### 効果
- 効果:ポケモンに持たせると、HPが1/4以下になったときに消費し、次に使う技の命中率が1.2倍になる。

### 詳細な仕様
- 詳細な倍率は4915/4096倍となる。
 - 補正される位置は、他のアイテムによる効果とは異なる(命中#第五世代以降参照)。
- 一撃必殺技の命中率は上がらない。
- ミクルのみを消費した後、次の行動を完了すると命中上昇の効果は消える。
 - 状態異常/状態変化などにより行動できなかった場合は効果は持続する。
 - 技が外れた場合や、無効化された場合、命中判定が無い変化技などを使用した場合でも効果は消える。
 - 交代すると効果は消える。
- さいはい/おどりこによる行動もミクルのみの効果対象であり、これらの行動を完了すると効果は消える。
- 効果が発動してから次に行動するまでの間に再度ミクルのみを入手した場合、HPが1/4以下でもすぐには使わず、次の行動を終えた後に使う。
 - ミクルのみの発動が保留されているときの行動で、相手のわるいてぐせが発動した場合、わるいてぐせが先に発動してミクルのみを奪われる。

### 備考
- 命中率のランク補正を上げる効果ではない。ミクルのみによる命中倍率は、命中ランクが0段階から1段階まで上がったときの倍率(4/3≒1.33倍)より低い。
