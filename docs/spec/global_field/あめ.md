# 天候: あめ（Rain）仕様書（第9世代・シングルバトル）

**対象世代**: 第9世代（SV）  
**カテゴリ**: 場の状態（天候）  

---

# 概要

「あめ」は場に影響する天候状態の1つであり、技威力・命中率・回復量・特性発動条件など複数の戦闘仕様に影響する。

---

# 天候継続仕様

## 継続ターン
- 通常: **5ターン**
- しめったいわ所持時: **8ターン**

---

# 基本効果

## 技威力補正
- みずタイプ技の威力: **1.5倍**
- ほのおタイプ技の威力: **0.5倍**

---

## 命中率補正
- 以下の技は命中判定を無視し必中化:
  - かみなり
  - ぼうふう
- 晴れ状態ではこれらの技の命中率が低下する（参考仕様）

---

## 技挙動変更

### ソーラービーム / ソーラーブレード
- 威力が **0.5倍**

### ウェザーボール
- タイプ: みずタイプに変化  
- 威力: **2倍（基礎威力50→100）**

---

## 回復技補正

以下の回復技の回復量が変化する。

| 技 | 回復量 |
|----|--------|
| こうごうせい | 最大HPの1/4 |
| あさのひざし | 最大HPの1/4 |
| つきのひかり | 最大HPの1/4 |

---

# 特性への影響

## 雨状態で効果が発動する特性

| 特性 | 効果 |
|------|------|
| すいすい | すばやさが2倍 |
| うるおいボディ | ターン終了時に状態異常回復 |
| あめのうけざら | ターン終了時に最大HPの1/16回復 |
| かんそうはだ | ターン終了時に最大HPの1/8回復 |
| てんきや | ポワルンがみずタイプ形態に変化 |

---

# 技との相互作用

## 天候依存技
- かみなり / ぼうふう: 命中必中化
- ウェザーボール: タイプ変化・威力倍化
- ソーラービーム / ソーラーブレード: 威力低下
- 回復技（こうごうせい等）: 回復量低下

---

# ダメージ・定期処理

## 定期ダメージ
- **なし**（砂嵐・雪と異なり定期ダメージは存在しない）

---

# 天候発生手段

## 技
- あまごい

## 特性
- あめふらし（場に出た時に天候を雨にする）

---

# 天候上書きルール

- 他の天候（晴れ・砂嵐・雪）と排他
- 新しい天候効果が優先される
- ノーてんき・エアロックなどにより天候効果は無効化される

---

# 仕様カテゴリ

- 場の状態（フィールドとは独立）
- 非永続状態（ターン制限あり）
- 天候排他カテゴリ
