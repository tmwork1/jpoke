# 天候: おおあめ（Heavy Rain）仕様書（第9世代・シングルバトル）

**対象世代**: 第6世代以降（ゲンシカイキ要素含む）  
**カテゴリ**: 場の状態（天候・特殊天候）  

---

# 概要

「おおあめ」は通常の「あめ」とは異なる特殊天候であり、主に特性「はじまりのうみ」によって発生する。  
ほのおタイプ技を無効化するなど、通常の雨を上回る強力な効果を持つ。

---

# 天候継続仕様

## 継続ターン
- **無限（ターン制限なし）**
- 以下の場合にのみ終了:
  - 他の特殊天候（おおひでり・らんきりゅう）による上書き
  - 特性無効化（ノーてんき・エアロック等）
  - 発動ポケモンの状態変化（フォルム解除など）

---

# 基本効果

## 技威力補正（雨相当）
- みずタイプ技の威力: **1.5倍**
- ほのおタイプ技の威力: **通常は0.5倍だが後述の通り無効化**

---

## 特殊効果

### ほのおタイプ技の無効化
- すべての**ほのおタイプ攻撃技が無効化される**
- ダメージ・追加効果ともに発生しない

---

# 技挙動変更

## 雨依存挙動をすべて適用

### かみなり / ぼうふう
- 命中判定を無視し**必中**

### ウェザーボール
- タイプ: みず
- 威力: 2倍（50→100）

### ソーラービーム / ソーラーブレード
- 威力: **0.5倍**

---

## 回復技補正

| 技 | 回復量 |
|----|--------|
| こうごうせい | 最大HPの1/4 |
| あさのひざし | 最大HPの1/4 |
| つきのひかり | 最大HPの1/4 |

---

# 特性への影響

## 発動特性

| 特性 | 効果 |
|------|------|
| はじまりのうみ | 場をおおあめ状態にする |
| すいすい | すばやさ2倍 |
| うるおいボディ | 状態異常回復 |
| あめのうけざら | 毎ターン回復 |
| かんそうはだ | 毎ターン回復 |
| てんきや | みずタイプに変化 |

---

# 技との相互作用

## 天候依存技
- かみなり / ぼうふう: 必中
- ウェザーボール: 水化＋威力倍
- ソーラービーム / ブレード: 威力低下
- 回復技: 回復量低下

---

# ダメージ・定期処理

## 定期ダメージ
- **なし**

---

# 天候発生手段

## 特性
- はじまりのうみ（ゲンシカイオーガ専用）

---

# 天候上書きルール

- 通常天候（雨・晴れ・砂嵐・雪）では**上書き不可**
- 以下の特殊天候のみ上書き可能:
  - おおひでり（おわりのだいち）
  - らんきりゅう（デルタストリーム）
- ノーてんき・エアロックで効果無効

---

# 仕様カテゴリ

- 場の状態（天候）
- 永続状態（条件付き解除）
- 特殊天候カテゴリ（通常天候より優先度が高い）

---

# 補足仕様

- 通常特性「あめふらし」はこの天候を上書きできない
- ゲンシカイキ時に発動し、通常の天候発生特性とは競合処理が異なる
