# 天候: おおひでり（Extremely Harsh Sunlight）仕様書（第9世代・シングルバトル）

**対象世代**: 第6世代以降（ゲンシカイキ要素含む）  
**カテゴリ**: 場の状態（天候・特殊天候）  

---

# 概要

「おおひでり」は通常の「はれ」とは異なる特殊天候であり、主に特性「おわりのだいち」によって発生する。  
みずタイプ技を無効化するなど、通常の晴れを上回る強力な効果を持つ。

---

# 天候継続仕様

## 継続ターン
- **無限（ターン制限なし）**
- 以下の場合にのみ終了:
  - 他の特殊天候（おおあめ・らんきりゅう）による上書き
  - 特性無効化（ノーてんき・エアロック等）
  - 発動ポケモンの状態変化（フォルム解除など）

---

# 基本効果

## 技威力補正（晴れ相当）
- ほのおタイプ技の威力: **1.5倍**
- みずタイプ技の威力: **通常は0.5倍だが後述の通り無効化**

---

## 特殊効果

### みずタイプ技の無効化
- すべての**みずタイプ攻撃技が無効化される**
- ダメージ・追加効果ともに発生しない :contentReference[oaicite:0]{index=0}

---

## 状態異常制限
- 通常の晴れと同様:
  - **こおり状態にならない**

---

# 技挙動変更

## 晴れ依存挙動をすべて適用

### ソーラービーム / ソーラーブレード
- 溜めなしで発動

### ウェザーボール
- タイプ: ほのお
- 威力: 2倍（50→100）

### かみなり / ぼうふう
- 命中率: 50%

---

# 回復技補正

| 技 | 回復量 |
|----|--------|
| こうごうせい | 最大HPの2/3 |
| あさのひざし | 最大HPの2/3 |
| つきのひかり | 最大HPの2/3 |

---

# 特性への影響

## 発動特性

| 特性 | 効果 |
|------|------|
| おわりのだいち | 場をおおひでり状態にする |
| ようりょくそ | すばやさ2倍 |
| サンパワー | 特攻1.5倍＋毎ターンダメージ |
| リーフガード | 状態異常無効 |
| てんきや | ほのおタイプに変化 |

---

# 技との相互作用

## 天候依存技
- ソーラービーム / ブレード: 溜め不要
- ウェザーボール: 炎化＋威力倍
- かみなり / ぼうふう: 命中低下
- 回復技: 回復量増加

---

# ダメージ・定期処理

## 定期ダメージ
- **なし**

---

# 天候発生手段

## 特性
- おわりのだいち（ゲンシグラードン専用）

---

# 天候上書きルール

- 通常天候（雨・晴れ・砂嵐・雪）では**上書き不可**
- 以下の特殊天候のみ上書き可能:
  - おおあめ（はじまりのうみ）
  - らんきりゅう（デルタストリーム）
- ノーてんき・エアロックで効果無効

---

# 仕様カテゴリ

- 場の状態（天候）
- 永続状態（条件付き解除）
- 特殊天候カテゴリ（通常天候より優先度が高い）

---

# 補足仕様

- 通常特性「ひでり」はこの天候を上書きできない :contentReference[oaicite:1]{index=1}
- ゲンシカイキ時に発動し、通常の天候発生特性とは競合処理が異なる :contentReference[oaicite:2]{index=2}
