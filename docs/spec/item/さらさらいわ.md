# 仕様書: さらさらいわ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%95%E3%82%89%E3%81%95%E3%82%89%E3%81%84%E3%82%8F

さらさらいわ は、アイテムの一種。
## 効果
- 買値:非売品 4-8 ,8000円 9
- 売値:100円 4-6, BD SP ( 第五世代 いしマニア は1000円), 2000円 7, Sw Sh ,9
  - こんごうダマ ×35-39 D P Pt / こんごうダマL ×10 BD SP
- 効果
  - 持たせたポケモンが 天気 を すなあらし 状態にするわざを使ったとき、持続ターンが8ターンに伸びる。
  - 持たせたポケモンが特性 すなおこし ・ すなはき を発動させると、すなあらし状態の持続ターンが8ターンに伸びる。( 第六世代 以降)
- なげつける :威力10
- ウッウロボ :素材タイプ いわ 、素材ポイント6
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代: もたせた ポケモンが すなあらしを つかうと いつもより すなあらしの じかんが ながくなる。
- 第九世代: 持たせたポケモンが すなあらしを 起こすと いつもより すなあらしの時間が 長くなる。
## 入手方法
- ダイヤモンド・パール ・ プラチナ: ちかつうろ の カセキほり で手に入れる。
- ハートゴールド・ソウルシルバー: ポケウォーカー のコース「 さむいやまみち 」でダウジングする。
- 第五世代: 8ばんどうろ の東にいる パラソルおねえさん に 夕方 に話しかけると貰える。
- X・Y: 13ばんどうろ ・ チャンピオンロード で拾う。
- オメガルビー・アルファサファイア: てんきけんきゅうじょ でイベント後、受付のお姉さんから貰う。
- サン・ムーン ・ ウルトラサン・ウルトラムーン: マリエシティ の地域センター内の天候教室にいる女性から貰う。
- ソード・シールド: ナックルシティ の天気の話をする ジェントルマン の話を4回聞くと貰える。その後も4日話を聞くごとに1個貰える。
- ブリリアントダイヤモンド・シャイニングパール: 地下大洞窟 の カセキほり で手に入れる。
- スカーレット・バイオレット: ジムバッジ 4個入手後、 カラフシティ の デリバードポーチ で買う。
## こんなときに使おう
第六世代 以降では特性 すなおこし のポケモンに持たせると効果を発揮する。
## 詳細な仕様
- ダイマックスわざ で天気を変えたときも効果がある。
- 第七世代 以降のじょうきょう画面での天気の持続ターン表示において、さらさらいわの効果を発動させたプレイヤー側は最初からすなあらしが8ターン続くと表示されるが、対戦相手側では通常通りの5ターンと表示されるため、さらさらいわの存在を相手に知らせることはない。5ターンを過ぎても天気が消えず、さらさらいわが相手視点からも確定すると持続ターンの表記が8ターンに書き換わる。
## 備考
- 発動させる 天気 の効果を延長する道具の一種。
