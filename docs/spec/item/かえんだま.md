# 仕様書: かえんだま

調査日: 2026-02-07

かえんだま は、アイテムの一種。
## 効果
- 買値:非売品,15000円 9
- 売値:100円 4-6, BD SP ,2000円 7, Sw Sh ,3750円 9
- 効果:持たせると、 ターン 終了時に やけど 状態になる。
- なげつける :威力30、相手を やけど 状態にする
- ウッウロボ :素材タイプ ほのお 、素材ポイント7
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代: さわると ねつをだす ふしぎなたま。もたせると せんとうちゅうに やけどの じょうたいに なる。
- 第九世代: 触ると 熱をだす 不思議な玉。 持たせると 戦闘中に やけどの状態に なる。
## 入手方法
- ダイヤモンド・パール ・ プラチナ: バトルパーク ・ バトルフロンティア で16 BP と交換する。
- ハートゴールド・ソウルシルバー: バトルフロンティアで16BPと交換する。
- ブラック・ホワイト: バトルサブウェイ で16BPと交換する。
- ブラック2・ホワイト2: リバースマウンテン で拾う W2 。
- 第六世代: バトルハウス で16BPと交換する。
- サン・ムーン ・ ウルトラサン・ウルトラムーン: ウラウラのはなぞの で ゴルファー のツカサとの勝負に勝つと貰える。
- ソード・シールド: 殿堂入り 後、 シュートスタジアム でトーナメントに1日2回以上優勝すると ボールガイ から貰えることがある。
- ブリリアントダイヤモンド・シャイニングパール: バトルパーク で16 BP と交換する。
- スカーレット・バイオレット: ハッコウシティ の デリバードポーチ で購入する。
## こんなときに使おう
持たせたポケモンは能動的に 特性 こんじょう ・ ふしぎなうろこ ・ ねつぼうそう を発動できる。 はやあし も発動するが、物理型のポケモンに持たせても相性が悪い。
トリック や すりかえ で相手に押し付けてやけど状態にすれば 物理技 のダメージを落とせる。相手が 状態異常 の回復手段を持っていても、治した矢先に再びやけどになるため効果が出やすい。やけど状態にならないか、やけどのダメージを受けないポケモンに持たせればデメリットが少ない。 サイコシフト で自分のやけど状態を相手に移す戦法もある。

## 詳細な仕様
- 発動しても消費しない。
- 以下のポケモンに対してかえんだまは発動しない。
  - ほのお タイプ
  - 特性 みずのベール 、 すいほう 、 ぜったいねむり 、 きよめのしお 、 ねつこうかん
  - 特性 リーフガード 、 リミットシールド に守られている
  - ミストフィールド の影響下にいる
  - すでに 状態異常 になっている
- 以下のポケモンに対してもかえんだまは発動する。
  - みがわり 状態
  - しんぴのまもり 状態
  - フラワーベール に守られている
- 仲良し度 や かいふくアイテム 等の手段で回復しても、そのターン終了直前に再発動する。
- 交代 でかえんだまを持ったポケモンを出した場合、そのターンからかえんだまは発動する。
  - 死に出し で出した場合は発動しない。
- 発動のタイミングは、 やけど ダメージ、 ねむけ 、場の状態、フィールド、 はねやすめ の効果の処理が終わってからとなる。
  - 発動するターンにやけどダメージは受けない。
  - ねむけ から ねむり 状態になるターンとかえんだまの発動ターンが重なった場合、ねむり状態が優先される。
  - ミストフィールド が解除されたターンからかえんだまが発動する。
  - さしおさえ ・ マジックルーム 状態が解除されたターンからかえんだまが発動する。
  - そのターン はねやすめ を使用した ひこう タイプも 地面から浮いた ことになるため、ミストフィールドでかえんだまの発動を防げない。
- だっぴ ・ うるおいボディ ・ いやしのこころ の発動タイミングの後にかえんだまは発動する。かえんだまが発動したターンにだっぴ・うるおいボディ・いやしのこころでやけどが治ることは無い。これらの特性で状態異常が治ったターンからかえんだまは発動する。
- かえんだまは ものひろい で拾ったターンでは発動しない。
## 備考
- 第六世代 まで、場に居座るのが3ターン以内ならダメージ量は どくどくだま ≦かえんだまなので、4ターン以上場に居続ける場合に持たせると有効。
- 第七世代 以降はやけどダメージが1/16に変わったので、1ターン目で両方のダメージ量が同じ、2ターン目でどくどくだまの方がダメージ量が多くなる。
