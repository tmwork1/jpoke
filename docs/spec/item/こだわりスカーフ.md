# 仕様書: こだわりスカーフ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%93%E3%81%A0%E3%82%8F%E3%82%8A%E3%82%B9%E3%82%AB%E3%83%BC%E3%83%95

こだわりスカーフ は、アイテムの一種。
## 効果
- 買値:非売品 4-8 ,100000円 9
- 売値:100円 4-6, BD SP ,2000円 7, Sw Sh ,25000円 9
- 効果:ポケモンに持たせると、最初に選択したわざしか使用できなくなるが、 すばやさ が1.5倍になる。
- なげつける :威力10
- ウッウロボ :素材タイプ ノーマル 、素材ポイント10
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代: ちょっと こだわった スカーフ。もたせると すばやさは あがるが おなじ わざしか だせなくなる。
- 第九世代: ちょっと こだわった スカーフ。 持たせると 素早さは あがるが 同じ技しか だせなくなる。
## 入手方法
- ダイヤモンド・パール ・ プラチナ: バトルパーク ・ バトルフロンティア で48 BP と交換する。
- ハートゴールド・ソウルシルバー: バトルフロンティアで48BPと交換する。
- ブラック・ホワイト: 自分が持っているポケモンの ID のうち、自分以外の ID が30個以上のとき、 ヒウンシティ の ポケモンセンター にいる黒服の男性から貰える。
- ブラック2・ホワイト2: バトルサブウェイや PWT で24BPと交換する。
- 第六世代: バトルハウスで48BPと交換する。
- サン・ムーン ・ ウルトラサン・ウルトラムーン: バトルツリー で48BPと交換する。
- ソード・シールド: ふるびたてがみ を アラベスクタウン の民家にいるフランクに届けると、お礼にもらえる。
- ブリリアントダイヤモンド・シャイニングパール: バトルパーク で25 BP と交換する。
- スカーレット・バイオレット: ジムバッジ 4個入手後、 テーブルシティ にある デリバードポーチ で買う。
## こんなときに使おう
攻撃力はあるがすばやさが足りないポケモンの すばやさ を上げるために持たせることが多い。他にすばやさを上げる効果を持つアイテムには カムラのみ ・ スピードパウダー ・ ビビリだま ・ からぶりほけん など発動条件が限られているものが多いため、こだわりスカーフの需要は大きい。
すりかえ や トリック で相手に押し付け行動を制限するコンボもある(通称「スカーフトリック」)。この相手に かなしばり や いちゃもん を使えば わるあがき を強要させることもできる。先制して押し付けやすく、他の こだわり系アイテム では相手の火力を上げてしまう可能性もあるので、この用途ではこだわりスカーフが一番使用われやすい。
## 詳細な仕様
- こだわり系アイテム の1つ。こだわり系アイテム全体で共通の仕様については こだわり系アイテム#仕様 を参照。
- すばやさが ランク で変動している場合、変動後のステータスを1.5倍にする。 まひ の効果はこだわりスカーフなどの効果を考慮した後に計算する( すばやさ#詳細な仕様 も参照)。
  - 特性 ・ おいかぜ ・ しつげん の効果を同時に受けている場合は、全ての効果をまとめて出した補正率をすばやさにかけて五捨五超入する。
- この道具が譲渡されたり消滅したりした場合、 第七世代 ではそのターンの行動順には影響しない。 第八世代 ではそのターンから行動順に影響が出る。
## 備考
- 対戦系のサイト等では「スカーフ」と略されることが多い。スカーフがつくアイテムは他に シルクのスカーフ が存在するが、基本的にはこちらのアイテムを指す。
- 第二世代 と 第三世代 ではこのアイテムは存在していなかったため、 せんせいのツメ 、 カムラのみ が代用として使われていた。
