# 仕様書: しめったいわ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%97%E3%82%81%E3%81%A3%E3%81%9F%E3%81%84%E3%82%8F

しめったいわ は、アイテムの一種。
## 効果
- 買値:非売品 4-8 ,8000円 9
- 売値:100円 4-6, BD SP ( 第五世代 いしマニア は1000円), 2000円 7, Sw Sh ,9
  - あおタマ ×40-49 D P Pt / あおタマL ×30 BD SP
- 効果:
  - 持たせたポケモンが 天気 を あめ 状態にするわざを使ったとき、持続ターンが8ターンに伸びる。
  - 持たせたポケモンが特性 あめふらし を発動させると、あめ状態の持続ターンが8ターンに伸びる。( 第六世代 以降)
- なげつける :威力60
- ウッウロボ :素材タイプ みず 、素材ポイント6
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代: もたせた ポケモンが あまごいを つかうと いつもより あめの じかんが ながくなる。
- 第九世代: 持たせた ポケモンが 天気を あめにすると いつもより あめの時間が 長くなる。
## 入手方法
- ダイヤモンド・パール ・ プラチナ: ちかつうろ の カセキほり で手に入れる。
- ハートゴールド・ソウルシルバー: ポケウォーカー のコース「 さむいやまみち 」でダウジングする。
- 第五世代: 8ばんどうろ の東にいる パラソルおねえさん に 朝 に話しかけると貰える。
- X・Y: 14ばんどうろ 、 19ばんどうろ で拾う。
- オメガルビー・アルファサファイア: てんきけんきゅうじょ でイベント後、受付のお姉さんから貰う。
- サン・ムーン ・ ウルトラサン・ウルトラムーン: マリエシティ の地域センター内の天候教室にいる女性から貰う。
- ソード・シールド: ナックルシティ の天気の話をする ジェントルマン の話を2回聞くと貰える。その後も4日話を聞くごとに1個貰える。
- ブリリアントダイヤモンド・シャイニングパール: 地下大洞窟 の カセキほり で手に入れる。
- スカーレット・バイオレット: ジムバッジ 4個入手後、 カラフシティ の デリバードポーチ で買う。
## こんなときに使おう
雨パ でなら採用の価値がある。
## 詳細な仕様
- ダイマックスわざ で天気を変えたときも効果がある。
- 第七世代 以降のじょうきょう画面での天気の持続ターン表示において、しめったいわの効果を発動させたプレイヤー側は最初からあめが8ターン続くと表示されるが、対戦相手側では通常通りの5ターンと表示されるため、しめったいわの存在を相手に知らせることはない。5ターンを過ぎても天気が消えず、しめったいわが相手視点からも確定すると持続ターンの表記が8ターンに書き換わる。
## 備考
- 発動させる 天気 の効果を延長する道具の一種。
