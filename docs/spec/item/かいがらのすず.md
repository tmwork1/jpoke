かいがらのすず は、アイテムの一種。
## 効果
- 買値:20000円 S V
- 売値:100円 3-6, BD SP ,2000円 7, Sw Sh , 5000円 S V
- 効果: 攻撃技 を使用した後、相手に与えた ダメージ の1/8分だけ自分の HP が回復する（小数点以下切捨て）。
- なげつける :威力30
- ウッウロボ :素材タイプ みず 、素材ポイント7
## 説明文
- ルビー・サファイア ・ エメラルド: もたせると てきに ダメージを あたえたとき たいりょくが かいふく
- ファイアレッド・リーフグリーン: もたせた ポケモンが あいてに ダメージを あたえたとき HPが すこし かいふくする。
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代 ・ 第九世代: もたせた ポケモンが こうげきして あいてに ダメージを あたえたとき HPを すこし かいふく できる。

## こんなときに使おう
与えられる ダメージ が少ないポケモンに持たせても回復量には期待できないが、相手のHPを満タンから0にしてもその1/8しか回復できない。 持ち物 のラインナップが少ない 第三世代 では回復手段の候補となる。 トリプルバトル では真ん中に配置するポケモンに持たせて しおふき や ふんか を使えば威力の維持にもつながる。 第五世代 以降では、 レベル1 の特性 がんじょう のポケモンが がむしゃら でHPを全回復するために持つことがある。
たべのこし とは機能するタイミングが違うため、戦闘終了時にも回復する。シナリオ攻略や 経験値 / 努力値 稼ぎの際に、交代時のダメージがかさみやすいポケモンに持たせる道具の候補になる。戦闘後に回復できない 対戦施設 である バトルチューブ や バトルピラミッド 、 しばり組手 でも有用な道具になる。
高難易度の テラレイドバトル では相手のHP量が数十倍にもなるため、通常の戦闘より高い回復量が見込める。 吸収技 を覚えないポケモンでも攻撃と回復を両立して行えるようになる。 マックスレイドバトル でも似たことが言えるが、HP倍率が最大でも3倍で、バリアによるダメージ切り捨ても大きいことからテラレイドバトルほどの効果は見込めない。
## 詳細な仕様
- 回復量を算出するとき小数点は切り捨てるが、1以上のダメージを与えた際には1回復できる。
- 攻撃が無効化されたときや、HP1のポケモンを みねうち で攻撃したとき、相手の ばけのかわ / アイスフェイス で攻撃を肩代わりされたときなど、与えたダメージが0だった場合は回復しない。
- 連続攻撃技 を使用した場合、回復は最後の攻撃の後に発動する。回復量は、 第三世代 のみ最後のダメージのみから、 第四世代 以降は全ての攻撃のダメージの合計から算出する。
- 全体攻撃技で複数のポケモンに同時にダメージを与えた場合、全員に与えたダメージを合計してから回復量を算出する。
- 特性 ヘドロえき の影響を受けない。かいがらのすずによる回復効果は 吸収技 による回復効果とは異なる。
- 発動したとき、「<所持者>は かいがらのすずで すこし かいふく」というメッセージが出る。
- こんらん で自傷ダメージを受けたときは発動しない。
- みらいよち / はめつのねがい によるダメージでは発動しない。
- 特性 ちからずく のポケモンが特性の対象技を使うときは、かいがらのすずの回復効果も無くなる。
- 吸収技 による効果、 さめはだ などの特性や ゴツゴツメット / ジャポのみ / レンブのみ によるダメージ、 反動技 によるダメージの後に回復効果が発動する。
- 反動技や、 ひんしになる攻撃技 などの効果で攻撃側のHPが0になったときは、回復効果が発動しないまま ひんし になる。
- 相手の レッドカード が発動して攻撃側が交代させられた場合、回復はできない。
- とんぼがえり / ボルトチェンジ / クイックターン を使用した場合、回復の後に交代する。
- どろぼう / ほしがる / マジシャン で相手のかいがらのすずを奪った場合、即座にその際のダメージに応じた回復効果が発動する。
- わるいてぐせ のポケモンを直接攻撃してかいがらのすずを奪われる場合、攻撃者がかいがらのすずで回復した後にわるいてぐせが発動する。
- おどりこ で 踊りの攻撃技 をコピーして使ったときも発動する。
### 第三世代・第四世代
- 交代前のポケモンへの おいうち では発動しない。
- かいふくふうじ 状態でも回復できる。
- 攻撃が みがわり 状態に防がれた場合、 第三世代 では回復できる。 第四世代 では、基本的に攻撃でみがわりを壊したとしても回復できない。
  - 第四世代の 連続攻撃技 でみがわり状態を途中で壊し、本体にもダメージを与えた場合は、みがわりに与えたダメージと本体に与えたダメージの合計から回復量を算出する。
  - 全体攻撃技でみがわり状態のポケモンとそうでないポケモンを攻撃した場合、第四世代でもみがわりに与えたダメージも含めた合計から回復量を算出する。
- 一撃必殺技 が成功したときも発動する。
### 第五世代以降
- 交代前のポケモンへの おいうち でも発動する。
- かいふくふうじ 状態では回復できない。
- 攻撃が みがわり 状態に防がれた場合も回復できる。みがわりに与えたダメージから回復量を求める。
- 一撃必殺技 が成功したとき、 第五世代 ・ 第六世代 では発動しない。 第七世代 以降では発動する。
## 備考
- ルビー・サファイア ・ エメラルド の時計用内蔵電池が切れてしまっている場合、 あさせのほらあな が満潮か干潮のどちらかで固定化されてしまい作れなくなる。
  - ポケモンXD ではポケクーポンの景品として入手可能。
