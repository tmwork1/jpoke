# 仕様書: おんみつマント

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8A%E3%82%93%E3%81%BF%E3%81%A4%E3%83%9E%E3%83%B3%E3%83%88

おんみつマント は、アイテムの一種。
## 効果
- 買値:20000円
- 売値:5000円
- 効果:相手の 攻撃技 による 追加効果 を受けなくなる。
- なげつける :威力30
## 説明文
- スカーレット・バイオレット: 敵の目を あざむく フード付き マント。 持たせると 体を 包み隠し 技の 追加効果を 受けなくなる。
## 入手方法
- スカーレット・バイオレット: ハッコウシティ にある デリバードポーチ で買う。
## 詳細な仕様
- 特性 りんぷん の持ち物版といった効果を持つ。防げる効果に関しては 追加効果 / りんぷん#特性の効果 の項目も参照とする。
  - 自分のランクを上げる 追加効果 は、所持者が使用したときも、所持者に対して使われたときも発動する。
  - クリアスモッグ / はたきおとす など、追加効果ではない技の効果は防げない。
  - なげつける により追加効果がある道具を投げられたとき、防御側にとってメリットとなる効果も含めて全てが防がれる。 きのみ をなげつけられても ゲップ は使えるようにはならず、無効化されたきのみが特性 ほおぶくろ / しゅうかく / はんすう の効果対象になることもない。
- おんみつマントで追加効果を防いだときも、それを明示するメッセージが流れることはない。
- 特性 どくしゅ / どくのくさり による どく / もうどく 状態も防げる。
- 特性 あくしゅう ・道具 おうじゃのしるし / するどいキバ による ひるみ 効果も防ぐ。
- ひるみ状態を無効化する効果の判定は攻撃技を受けた時点で行われる。おんみつマントを持つポケモンが行動するときではない。
- 特性 シンクロ / どくくぐつ により発生する状態異常/状態変化はおんみつマントで防ぐことができない。
## 備考
- りんぷんと異なり特性ではないので、 かたやぶり では無視されない。
