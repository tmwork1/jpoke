## こんなときに使おう
ねむりごな ・ キノコのほうし を無力化することで ビビヨン や キノガッサ の無限戦術に対して強くなる。 ラムのみ と比較して、技を防いでも無くならない点がメリットだが、 さいみんじゅつ や あくび など粉ではない技は無効化できない。
ダブルバトル はで いかりのこな で攻撃を吸われなくなるので、とりわけメジャーな モロバレル に対する対策になる。粉技ではない このゆびとまれ の影響は受けるので、 ちゅうもくのまと を完全に無視できるようになるわけではない。
天候ダメージのみを防ぐためだけの目的ならば たべのこし 、どくタイプならば くろいヘドロ でも十分であるため、粉技を使う相手に役割があるポケモンに持たせたい。特性 がんじょう が天候ダメ―ジで潰れないように粉系の技対策のついでに持たせられることもある。
天候ダメージを受ける 野生ポケモン を捕獲するとき、 トリック ・ すりかえ でぼうじんゴーグルを渡せば自滅されることはない。ただし、渡した後は状態異常にする粉技が効かなくなる点と、 第八世代 以前だと捕獲に失敗した場合はぼうじんゴーグルを失う点に注意がいる。特性 マジックガード を スキルスワップ で渡す方法なら注意点が減るので、ぼうじんゴーグルを使う方法はマジックガードを渡す方法が使えないときの代用となる。
## 詳細な仕様
- 粉系の技を防いだ際には、「<ポケモン>は ぼうじんゴーグルで <粉系の技>を 受けない!」というメッセージが表示される。
  - 特性 ぼうじん のポケモンがぼうじんゴーグルを持って粉のわざを防いだ場合、特性による無効化が優先される。
  - すなあらし ・ あられ 、 いかりのこな 、特性 ほうし を無視したときにメッセージは流れない。
- いかりのこな による ちゅうもくのまと を無視するかどうかはポケモンが行動するときに決定する。
  - ぼうじんゴーグルを持っていない状態でいかりのこなが使用されたが、行動する前にぼうじんゴーグルを手に入れたポケモンの行動はいかりのこなを無視することができる。逆にいかりのこなが使用された時点ではぼうじんゴーグルを持っていたが、行動する前にぼうじんゴーグルを失ったポケモンの行動はいかりのこなに引き寄せられる。
  - いかりのこなは自分を対象とする粉の技だが、ぼうじんゴーグルを持ったポケモンがいかりのこなを使った場合でも技は成功し、ちゅうもくのまと状態になる。
- 技 ふんじん を受けたときは無効化できるが、 ふんじん 状態の効果自体を防ぐことはできない。
  - ふんじん状態になった後にぼうじんゴーグルを得た場合、そのターンにほのお技を使おうとするとダメージを受ける。
- さしおさえ ・ マジックルーム 状態がターン経過で解除されるときは、天候ダメージを受けてから状態が解除されて道具が有効になる。
- すなあらしによる いわ タイプの特防上昇や、あられによる ふぶき の必中化など、スリップダメージ以外の天候の効果は防がない。
  - ゆき 状態に対しては効果が無い。
  - 特性 アイスボディ のポケモンに持たせたときも、あられ状態による回復ができる。
## 備考
- 持たせたポケモンに特性 ぼうじん の効果を追加させるアイテムと言える。このアイテムが登場した 第六世代 から、ぼうじんは粉系の技も無効になるように強化された。
  - 特性と道具という違いがあるため、 かたやぶり や かがくへんかガス はぼうじんのみを無効化し、ぼうじんゴーグルに効果は無い。 さしおさえ ・ マジックルーム 状態はぼうじんゴーグルを無効化するが、ぼうじんには効果が無い。
- アニメでは SM編第41話 に登場し、デンヂムシレースの砂漠地帯に入る際に サトシ と デンヂムシ の両方が着用した。
- ゴーゴーゴーグル というものも存在するが、こちらはトレーナーが さばく 地帯に入るための たいせつなもの である。
