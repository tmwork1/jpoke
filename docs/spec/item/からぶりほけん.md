# 仕様書: からぶりほけん

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%89%E3%81%B6%E3%82%8A%E3%81%BB%E3%81%91%E3%82%93

からぶりほけん は、アイテムの一種。
## 効果
- 買値:非売品 8 , 30000円 9
- 売値:2000円 8 , 7500円 9
- 効果:持たせたポケモンが使用した技が 命中 判定に失敗したとき、自分の すばやさ が 2段階 上昇する。1度発動すると無くなる。
- なげつける :威力80
- ウッウロボ :素材タイプ あく 、素材ポイント12
## 説明文
- 第八世代 ・ 第九世代: めいちゅうりつに よって わざが はずれたとき すばやさが ぐーんと あがる。
## 入手方法
- ソード・シールド: バトルタワー で20BPと交換する。
- スカーレット・バイオレット: エンディング後、 テーブルシティ の デリバードポーチ で買う。
## こんな時に使おう
さいみんじゅつ など、命中率が低い ねむらせるわざ と組み合わせれば当たっても外れても利益になる。上がったすばやさは バトンタッチ で引き継ぐこともできる。
## 詳細な仕様
- すでに すばやさ ランク が最大のときは発動しない。
- 一撃必殺技 を外したときは発動しない。
- 説明文にあるとおり、 命中 判定に失敗して技が外れた場合のみ発動する。
  - 以下の効果による技の失敗は命中判定とは関係が無いので、からぶりほけんは発動しない。
    - 相性 ・ 特性 に技を無効化されたときや、 まもる 状態などで防がれたとき。
    - 対象が 姿を隠している ために技が当たらなかったとき。
    - 対象がすでに ひんし になっていたときなど、対象の場に誰もいなかったため技が失敗したとき。
  - 以下の効果は命中判定に関わるので、からぶりほけんが発動する。
    - 所持者の特性 はりきり 、または相手の特性 すながくれ / ゆきがくれ / ちどりあし / ミラクルスキン 、もちもの ひかりのこな / のんきのおこう の効果で技が当たらなかったとき。
    - なかよし度 の効果で技が当たらなかったとき。
  - 無効化判定の前に命中判定があるとき（例： みがわり 状態に対する さいみんじゅつ ）、命中判定に失敗して技が外れたなら発動する（参考： ターン#技の成功判定 ）。
- 連続攻撃技 は最初の1発が外れると発動する。最大回数当たらなかっただけでは発動しない。
  - トリプルキック / トリプルアクセル / ネズミざん も他の連続攻撃技と同様に、最初の1発が外れたときのみ発動する。1発でも当たれば、最大回数当たらなかったとしても発動しない。
- ドラゴンアロー は1体の命中判定に失敗した場合、もう片方を2回攻撃したとしても発動する。
  - 片方にタイプやまもる状態で無効化されたため、もう片方を2回攻撃したときは発動しない。
- マジックコート 状態により跳ね返された技が外れた場合、マジックコートのポケモンが持つからぶりほけんが発動し、元の使用者のものは発動しない。
- 全体攻撃技を使用したときは、対象の内いずれか1体にでも回避されたなら発動する。
  - 全体攻撃の変化技を使用し、外れたポケモンとマジックコート状態のポケモンがいた場合、からぶりほけんが発動した後にマジックコートで跳ね返される。
  - 全体攻撃の攻撃技を使用し、外れたポケモンと当たったポケモンがいた場合、またはドラゴンアローが命中判定により片方を2回攻撃した場合、以下のようになる。
    - 相手の わたげ や ミラーアーマー などの特性の後にからぶりほけんが発動する。
    - 相手の レッドカード が発動した場合、からぶりほけんは発動する前に交代させられる。
    - 相手の わるいてぐせ が発動した場合、からぶりほけんは発動する前に奪われる。
    - マジシャン のポケモンがからぶりほけんを発動させるときは、特性が不発した後に消費する。
    - 持ち物が無いマジシャンのポケモンがからぶりほけんを奪ったときは、他に技を外した相手がいたとしてもその攻撃でからぶりほけんは発動しない。
