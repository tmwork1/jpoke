# 仕様書: こだわりメガネ

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%93%E3%81%A0%E3%82%8F%E3%82%8A%E3%83%A1%E3%82%AC%E3%83%8D

こだわりメガネ は、アイテムの一種。
## 効果
- 買値:非売品 4-8 ,100000円 9
- 売値:100円 4-6, BD SP ,2000円 7, Sw Sh ,25000円 9
- 効果:ポケモンに持たせると、最初に選択したわざしか使用できなくなるが、 とくこう が1.5倍になる。
- なげつける :威力10
- ウッウロボ :素材タイプ エスパー 、素材ポイント10
## 説明文
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代: ちょっと こだわった メガネ。もたせると とくこうは あがるが おなじ わざしか だせなくなる。
- 第九世代: ちょっと こだわった メガネ。 持たせると 特攻は あがるが 同じ技しか だせなくなる。
## 入手方法
- ダイヤモンド・パール ・ プラチナ: カンナギタウン の左上の民家で 午前4時～午前9時59分 の間に貰う。
- ハートゴールド・ソウルシルバー: いかりのみずうみ で拾う。
- ポケウォーカー: チャンプのみち でダウジングする。
- ポケモンバトルレボリューション: 7200 ポケクーポン で交換する。
- ブラック・ホワイト: バトルサブウェイ で48 BP と交換する。
- ブラック2・ホワイト2: バトルサブウェイや PWT で24BPと交換する。
- X・Y: バトルハウス で48BPと交換する。
- オメガルビー・アルファサファイア: バトルハウスで48BPと交換する。
- サン・ムーン: バトルツリー で48BPと交換する。
- ウルトラサン・ウルトラムーン: カプのむら の ベテラントレーナー に勝利すると貰える。
- ソード・シールド: スパイクタウン で拾う。
- ブリリアントダイヤモンド・シャイニングパール: カンナギタウン の左上の民家で 朝 に貰う。
- スカーレット・バイオレット: エンディング後に テーブルシティ にある デリバードポーチ で買う。
## こんなときに使おう
基本的に 特殊技 のみ覚えたポケモンの持ち物として対戦でもよく使われる。 りゅうせいぐん などの高威力技で負担をかけ、受けられたらすぐに 交代 する「撃ち逃げ」の戦法が取られることが多い。

## 備考
- こだわり系アイテム の1つ。対となる こだわりハチマキ より遅い 第四世代 で初登場した。
- 使用率の高さから、対戦スラングでは「眼鏡」と略されて呼ばれることもある。この呼び方が ものしりメガネ や くろいメガネ を指すことは少ない。
