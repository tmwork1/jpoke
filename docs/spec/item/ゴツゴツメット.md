# 仕様書: ゴツゴツメット

調査日: 2026-02-07
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%B4%E3%83%84%E3%82%B4%E3%83%84%E3%83%A1%E3%83%83%E3%83%88

ゴツゴツメット は、アイテムの一種。
## 効果
- 買値:非売品 5 - 8 、50000円 9 -
- 売値:100円 5 - 6 、2000円 7 - 8 、12500円 9 -
- 効果: 直接攻撃 で ダメージ を受けたとき、攻撃してきた相手に対して相手の最大 HP の1/6分のダメージを与える。(小数点以下切り捨て。ただし最低でも1以上)。
- なげつける :威力60
- ウッウロボ :素材タイプ いわ 、素材ポイント13
## 説明文
- 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代: ポケモンに もたせると だげきわざを うけたとき あいてにも ダメージを あたえる。
- 第九世代: ポケモンに 持たせると 自分に さわった相手に ダメージを 与える。
## 入手方法
- ブラック・ホワイト: れいとうコンテナ で貰う。
- ブラック2・ホワイト2: こだいのぬけみち で拾う。
- X・Y: 9ばんどうろ 西ゲートで貰う。
- オメガルビー・アルファサファイア: てんきけんきゅうじょ でイベント後1Fにいる助手から貰う。
- サン・ムーン ・ ウルトラサン・ウルトラムーン: バトルツリー で48BPと交換する。
- ソード・シールド: ラテラルタウン で拾う。
- スカーレット・バイオレット: ジムバッジ 4個入手後、 テーブルシティ の デリバードポーチ で買う。
## こんなときに使おう
接触されるだけで ダメージ を稼げるため、物理耐久に優れてなおかつ じこさいせい や はねやすめ などの高速再生技を持つポケモンに持たせるとよい。 がんじょう 、 きあいのタスキ を潰す手段としても有用。
## 詳細な仕様
- 効果が発動する際、「<攻撃者>は ゴツゴツメットで ダメージを受けた!」というメッセージが流れる。
- 特性 マジックガード のポケモンはゴツゴツメットのダメージを受けない。このとき、ゴツゴツメットやマジックガードの所有を明示するメッセージは流れない。
- 持ち物 ぼうごパット を持つポケモンはゴツゴツメットのダメージを受けない。このとき、お互いの所有する持ち物を明示するメッセージは流れない。
- 特性 えんかく や持ち物 パンチグローブ の効果で直接攻撃ではなくなった技に対しては発動しない。
- 直接攻撃 を受けても、 相性 /特性/ まもる 状態などで攻撃を無効化したときは発動しない。
  - ばけのかわ / アイスフェイス でダメージを肩代わりしたときは発動する。
- みがわり 状態で攻撃を防いだときは発動しない。
- みがわり状態のポケモンから直接攻撃を受けた場合、みがわり状態を貫通して本体にダメージを与える。
- 特性が さめはだ / てつのトゲ である場合、特性によるダメージの後にゴツゴツメットによるダメージが発生し、合計で最大HPの7/24のダメージを与える。途中で相手が きのみ の発動圏に入ったときは回復を挟むことができる。
- ゴツゴツメットを持つポケモンが ひんし になったときも発動する。
  - このときのダメージで相手のポケモンもひんしになり、お互いのポケモンが全滅した場合は、 サン・ムーン のVer.1.0までは攻撃側が先にひんしになり、ゴツゴツメット側の勝利となる。サン・ムーンのVer.1.1以降はゴツゴツメット側が先にひんしになり、ゴツゴツメット側の敗北となる。
- 連続攻撃技 に対しては、1発ダメージを受けるたびに発動する。
  - 途中で使用者がひんしになると攻撃は中断する。相手の特性が スキルリンク / おやこあい だったとしても同様に中断する。
- HPを吸収する 効果がある直接攻撃を受けた場合、回復の後にゴツゴツメットのダメージが入る。
- とんぼがえり / クイックターン を受けたが、使用者がゴツゴツメットのダメージでひんしになったときは、技の効果は発動しない。
- 以下の技を受けた場合、ゴツゴツメットのダメージの後に技の効果が発動する。ゴツゴツメットで使用者がひんしになったときは技の効果が発動しない。
  - こうそくスピン / キラースピン ( 設置技 を解除する効果)
  - どろぼう / ほしがる / はたきおとす
  - きつけ / めざましビンタ
  - ドラゴンテール / ともえなげ
  - プラズマフィスト
  - アイススピナー
  - 特性 マジシャン による持ち物を奪う効果
- どろぼう/ほしがる/はたきおとす/特性マジシャンの直接攻撃技を受けた場合、相手にダメージを与えた後でゴツゴツメットは失われる。
## Pokémon Trading Card Game Pocketにおけるゴツゴツメット
トレーナーズ として登場する。詳しくは ゴツゴツメット (ポケポケ) を参照。
## 備考
- ブラック・ホワイト と 第六世代 の作品では、1回のプレイにつき1つしか入手できない。
- アニメでは、 XY編第59話 にてサトシが ヒヨクシティ の誓いの木の祭りでポケモンにプレゼントする物を考えていたところ、通りすがりの店の店員に きょうせいギプス ・ みがわり人形 ・ ハスボーじょうろ と共に勧められていた。
