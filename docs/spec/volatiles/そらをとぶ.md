# 仕様書: そらをとぶ（揮発状態）

調査日: 2026-06-20
参照URL: https://wiki.pokemonwiki.com/wiki/そらをとぶ_(状態変化)

## 概要

そらをとぶを使用した1ターン目に付与される揮発状態。ポケモンが空中に飛び上がっている状態を表す。

## 発生条件

- 技「そらをとぶ」の1ターン目（溜めターン）に発生する。
- 技「とびはねる」の1ターン目（溜めターン）にも発生する。
- 技「フリーフォール」の1ターン目にも発生する（本実装スコープ外）。

## 消滅条件

- そらをとぶ・とびはねるの2ターン目（攻撃ターン）が完了した時点で解除される。
- まひ・こんらん・なまけなどで2ターン目に攻撃できなかったときも解除される（状態変化が解除され、攻撃もできない）。
- うちおとす・サウザンアローを受けた場合も解除される（本実装スコープ外）。
- じゅうりょく状態になった場合も解除される（本実装スコープ外）。

## 効果

### 回避

- そらをとぶ状態のポケモンに対して、ほとんどの技は命中しない（ミス）。
- 以下の技は命中する（必中ではなく通常の命中判定が行われる）：
  - かぜおこし（第二世代以降）
  - かみなり（第二世代以降）
  - たつまき（第二世代以降）
  - ぼうふう（第五世代以降）
  - うちおとす（第五世代以降。命中時に揮発状態を解除する）
  - スカイアッパー（第三世代〜第七世代）
- 以下の条件を満たす技も命中する：
  - ロックオン状態のポケモンが使用した技
  - 自身か相手が特性ノーガードの場合
  - 毒タイプのポケモンがどくどくを使用した場合（第六世代以降）

### 威力2倍

- そらをとぶ状態のポケモンに対して「かぜおこし」「たつまき」を使用した場合、威力が2倍になる（第三世代以降の仕様）。
  - 本実装（チャンピオンズ）: `ON_CALC_POWER_MODIFIER` イベントで威力を2倍補正する。

### 行動固定

- そらをとぶ状態のポケモンのコマンドは強制（FORCED）に固定される。
- PPは消費されない（溜めターン）。

### 交代禁止

- そらをとぶ状態中は交代できない（ゴーストタイプを除く）。

### フィールド効果

- そらをとぶ状態の間、フィールドの影響を受けない（本実装スコープ外）。

### 天候ダメージ

- すなあらし・あられのダメージは通常通り受ける。

## 本実装での対象外（スコープ外）

- うちおとす命中時の揮発状態解除（うちおとす揮発状態側で浮遊無効化のみ実装済み）。
- じゅうりょく状態による解除は未実装。
- スカイアッパーの命中（技が未実装のため）。
- フィールド効果回避は未実装。
- フリーフォールによる付与は未実装。
- ロックオン状態による命中は `ロックオン` 揮発状態側で実装済み。
- ノーガード特性による命中は特性側で実装済み。
