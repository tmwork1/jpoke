# いちゃもん (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

いちゃもんとは揮発性状態の一つ。
直前に使用した技を選択できなくなる。

## 効果

- いちゃもん状態のポケモンは、直前に選択した技（`move_name` に記録された技）をコマンド選択時に選択できない。
- 「技を選択できなくなる」効果であり、「技を使用できなくなる」効果ではない。
- 他のすべての技が選択可能な状態のとき、その技を除いた技から選択できる。

## 付与条件

- 技「いちゃもん」を相手に使用したとき、相手がいちゃもん状態になる。

## 消滅条件

- **交代**によって解除される（バトンタッチは除く）。
- ターン経過では解除されない（技「いちゃもん」による場合）。
- メンタルハーブを持っているポケモンが付与されたとき即座に解除される（第五世代以降）。

## 詳細仕様

- 選択できる技がなくなった場合（直前の技しか持っていない場合、またはこだわり系などとの重複でほかに選べる技がない場合）は、わるあがきを使用する。
- わるあがきは連続で使用できる（いちゃもんの制限を受けない）。
- バトンタッチで引き継がれない。
- いちゃもん状態になったターンは、直前に使用した技を選択していても失敗しない（技の使用後にいちゃもん状態になるため、その後のターンで制限される）。
- 技の選択制限であるため、アンコール・溜め技・連続攻撃などでコマンド選択を経ずに技が出される場合は制限されない。

## 本実装での対象外（スコープ外）

- キョダイユウゲキによるいちゃもん（3ターン時間制限）: Champions ではキョダイユウゲキは対象外。
- ダイマックスによる回復・予防: Champions ではダイマックスは対象外。
- アロマベール特性による予防: 特性の実装は別途行う。
- メンタルハーブによる回復: アイテム実装時に対応。

## ターン処理の優先度

`ON_MODIFY_COMMAND_OPTIONS` でコマンド選択肢から禁止技を除外する（priority=100, デフォルト）。
