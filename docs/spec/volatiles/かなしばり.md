# かなしばり (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

かなしばりとは揮発性状態の一つ。
相手が最後に使用した技を封じ、一定ターンの間その技を使えなくする。

## 効果

- かなしばり状態のポケモンは、揮発状態に記録された `move_name` の技をコマンド選択時に選択できない。
- 行動実行時にも封じた技を使おうとすると行動がブロックされる（`action_success = False`）。

## 付与条件

- 技「かなしばり」を相手に使用したとき（命中判定あり）。
- 発動条件:
  - 相手がそのバトルで一度でもPPを消費する行動をしていること
    （`pp_consumed_move` が None でないこと）
  - 相手が最後にPPを消費した技がわるあがきでないこと
  - 相手がすでにかなしばり状態でないこと

## ターン数

- 第五世代以降（本実装）: 4ターン固定

## 消滅条件

- **ターン経過**: 4ターン経過後（`ON_TURN_END` priority=110 でカウントダウン）
- **交代**によって解除される（交代時に全揮発状態が自動解除）
- メンタルハーブ（アイテム実装時に対応）

## ターン処理の優先度

`docs/spec/turn.md` より:

| イベント | Priority | 処理 |
|---------|----------|------|
| `ON_MODIFY_COMMAND_OPTIONS` | 100（デフォルト） | コマンド選択肢から封じた技を除外 |
| `ON_TRY_ACTION` | 100 | 封じた技の実行をブロック |
| `ON_TURN_END` | 110 | かなしばりのターンカウント消費・終了 |
| `ON_STATUS_HIT` | 100 | かなしばりの失敗条件チェック |

## コマンド制限

- `ON_MODIFY_COMMAND_OPTIONS` で封じた技のコマンドのみを除外する。
- 他の技は引き続き選択可能。
- 封じた技以外に選択可能な技がない場合（こだわり系アイテム・アンコールとの重複等）は、わるあがきを使用する。

## 詳細仕様（Champions版）

- バトンタッチで引き継がれない（交代時に全揮発状態が解除される）。
- アンコール状態の技と重複した場合、どちらか解除されるまでわるあがきを使用する
  （コマンド選択レベル: アンコール制限で固定技のみ残り、かなしばり制限で固定技が除外 → コマンドなし → わるあがき）。
- こだわり系アイテムを持っているポケモンが固定技にかなしばりを受けた場合、わるあがきを使用する。

## 関連する特性・アイテムとの相互作用

- のろわれボディ特性: 直接攻撃を受けたとき30%の確率で相手の攻撃技をかなしばりにする
  （`handlers/ability.py` の `のろわれボディ_maybe_disable_move` で実装済み）。
- メンタルハーブ: 装備者にかなしばりが付与された瞬間に即解除する
  （`handlers/item.py` の `メンタルハーブ_cure_mental_volatile` で実装済み）。
- アロマベール特性: 味方（シングルバトルでは自分自身）へのかなしばり付与を無効化する
  （`handlers/ability.py` の `アロマベール_prevent_volatile` で実装済み）。

## 本実装での対象外（スコープ外）

- ダイマックスわざへの変換による回避: Champions ではダイマックスは対象外。
