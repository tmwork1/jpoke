# タールショット (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

タールショットとは揮発性状態の一つ。
技「タールショット」の命中時に付与される。

## 効果

ほのおタイプが弱点に追加される。
具体的には、ダメージ計算のタイプ相性（ON_CALC_DEF_TYPE_MODIFIER）において
ほのおタイプの技に対する相性補正値を2倍にする。

| 元の相性 | タールショット後 |
|----------|-----------------|
| 無効(0倍) | 変化なし（無効のまま） |
| 1/4倍 | 1/2倍 |
| 1/2倍 | 1倍（等倍） |
| 1倍（等倍） | 2倍（弱点） |
| 2倍（弱点） | 4倍 |
| 4倍 | 8倍 |

## 付与条件

- 技「タールショット」の命中時に付与される（常時・ON_STATUS_HIT）。
- 同時にすばやさを1段階下げる。

## 消滅条件

- **交代**によって解除される（バトンタッチは引き継がない）。
- ターン経過では解除されない（count=None、永続）。

## テラスタルとの関係

- テラスタル状態のポケモンはタールショット状態にならない（予防）。
  - **未実装**: 本プロジェクトでは揮発性状態全般に対するテラスタル予防の共通機構が存在せず、
    タールショットに限らず他の状態変化技でも同様の予防は実装されていない
    （`docs/plan/volatiles/タールショット.md` 参照）。挙動としては本項目のとおりだが、
    実装との差分として明記する。
- すでにタールショット状態のポケモンがテラスタルしても、タールショット状態は解除されない。

## 詳細仕様

- 変化する相性は2倍であり、ほのおタイプに対する相性自体が変化する。
  - 元々ほのおが弱点のポケモンも影響を受ける（2倍→4倍、4倍→8倍）。
  - 元々ほのおが半減のポケモンは等倍になる（1/2倍→等倍）。
  - 元々ほのおが無効のポケモンには変化なし（0のまま）。
- じゃくてんほけん・オッカのみ・たつじんのおびなどは、タールショットによって
  ほのおが抜群になった場合に発動する。
- もりののろい・ハロウィン等のタイプ追加状態と重複する。
- フォルムチェンジで解除されない。
- バトンタッチで引き継がれない。
- タールショット状態のポケモンのタイプが変更されても解除されない。

## イベント

| イベント | subject_spec | priority | 内容 |
|----------|--------------|----------|------|
| ON_CALC_DEF_TYPE_MODIFIER | defender:self | 100 | ほのおタイプ技の相性補正を2倍にする |

## 技「タールショット」の仕様

- タイプ: いわ
- 分類: 変化
- PP: 20
- 命中率: 100
- 効果: 命中時に相手をタールショット状態にし、すばやさを1段階下げる（ON_STATUS_HIT）
- 技本体の詳細仕様は `docs/spec/moves/タールショット.md` を参照。
