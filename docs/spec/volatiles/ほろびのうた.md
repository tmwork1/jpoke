# ほろびのうた (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

ほろびのうたとは揮発性状態の一つ。
発生から3ターン後に対象ポケモンがひんし状態になる。

## 効果

| 効果 | 値 |
|------|-----|
| ターン終了時カウント減少 | Priority 120 で毎ターン終了時にカウントが1減る |
| カウント0になった時の処理 | ひんし状態になる（HP を0にする） |

- ほろびのうた状態のポケモンは、3ターン後（カウント3→2→1→0時に発動）にひんしになる。
- ターン終了時（ON_TURN_END, Priority 120）にカウントを1減らし、カウントが0になると ON_VOLATILE_END が発火してひんしにする。

## 付与条件

- 技「ほろびのうた」を使用したとき（相手に付与）
- 特性「ほろびのボディ」で直接攻撃を受けた時（攻撃者に付与）

## 消滅条件

- **交代**によって解除される
- **バトンタッチ**によって引き継がれる（バトンタッチ先にも状態が移動する）

## 詳細仕様

- きあいのタスキ・きあいのハチマキ・特性がんじょうでひんしを耐えることはできない。
- 特性**マジックガード**のポケモンもひんしになる。
- バトンタッチで引き継がれる。特性ぼうおんのポケモンにも引き継がれる。
- 複数体がほろびのうた状態の場合、第三世代以降はすばやさが高い順にひんしになる（トリックルーム状態ではすばやさが遅い順）。

## ターン処理の優先度

`docs/spec/turn.md` の `Event.ON_TURN_END` より:

| Priority | 処理 |
|----------|------|
| 120 | ほろびのうた（カウント減少 / ON_TURN_END） |
| 100 | ON_VOLATILE_END（カウント0時のひんし処理） |

## 関連インタラクション

- **マジックガード**: 通常の間接ダメージは無効化するが、ほろびのうたによるひんしは防げない（`reason="perish"` で発動）
- **きあいのタスキ / きあいのハチマキ / がんじょう**: これらで耐えることはできない（`ON_MODIFY_NON_MOVE_DAMAGE` を通さず、HP を直接 0 に設定するため）

## 実装上の注意

- ダメージ適用: `battle.modify_hp(mon, v=-mon.hp, reason="perish")`
  - `reason="perish"` を指定することでマジックガードの無効化を回避する
  - `HPChangeReason` に `"perish"` が定義されている
- `subject_spec`: 非攻撃フェーズのため `"source:self"`
- ON_TURN_END priority: 120（`docs/spec/turn.md` のほろびのうた行より）
- ON_VOLATILE_END priority: 100（デフォルト値）
