# 仕様書: きゅうしょアップ（揮発状態）

調査日: 2026-06-20
参照URL: https://wiki.pokemonwiki.com/wiki/きゅうしょアップ

## 概要

きゅうしょアップ状態とは、技「きあいだめ」などを使用したポケモンに付与される揮発状態。
自分の技が急所に当たりやすくなる。第6世代まで「きあいだめ状態」と呼ばれていた。

## 効果

### 急所ランク上昇

- 第3世代以降: 急所ランク+2
- ポケモンチャンピオンズ（第9世代）: 急所ランク+2

急所ランクに基づく急所確率は以下の通り（第6世代以降）:
- ランク0: 1/24 (約4.2%)
- ランク1: 1/8 (12.5%)
- ランク2: 1/2 (50%)
- ランク3以上: 確定急所 (100%)

## 発生条件

- 技「きあいだめ」の命中時（使用者自身に付与）
- アイテム「サンのみ」がHP1/4以下で発動した時（count=2, 急所ランク+2）

## 消滅条件

- 交代（ポケモンチェンジ）で解除される

## 詳細な仕様

- バトンタッチで引き継がれる。
- 第2世代以降では、くろいきりやクリアスモッグ等で消すことはできない。
- すでにきゅうしょアップ状態であるとき、きゅうしょアップ状態になる技や道具の効果は発動しない（重ねがけ不可）。

## 内部実装上の仕様

- `count` フィールドに急所ランク加算値（+2）を格納する。
- きあいだめ使用時: `count=2` で `volatile_manager.apply` を呼ぶ。
- ハンドラ `ON_CALC_CRITICAL_RANK` で `value += volatile.count` として急所ランクを加算する。

## 本実装での対象外（スコープ外）

- ドラゴンエールによる+1（シングルバトル向けのドラゴンエール自体が未実装）
- じこあんじ・へんしんによるコピー（第6世代以降の挙動だが優先度低）
- きょうえん技（未実装）
