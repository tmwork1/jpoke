# みがわり（揮発性状態）

## 概要

自分のHPを1/4消費してみがわりを生成する。みがわりは本体の代わりに攻撃を受ける。

## 発生条件

- 技「みがわり」の使用（HP1/4以上ある場合のみ）
- 技「しっぽきり」の使用（HP1/2消費、みがわりHPは1/4）

## 消滅条件

- みがわりのHPが0になったとき
- 本体がひんしになったとき
- 本体が交代したとき

## みがわりのHP

- 最大HPの 1/4（小数点以下切り捨て）
- みがわりのHPを回復する手段はない

## 効果

### みがわりが防ぐもの

- 相手の攻撃技のダメージ（みがわりのHPが削られる）
- 攻撃技の追加効果（状態異常・ランク低下・ひるみなど）
- 相手の変化技（対象側へのもの）：状態異常付与・ランク変化・状態変化など

### みがわりを貫通する技・効果

- **音技**（soundラベルを持つ技）：みがわりを無視して本体にダメージを与える
- **bypass_substituteラベルを持つ技**（スキルスワップ等）
- **すりぬけ特性**のポケモンが使用した技（ON_CHECK_HIT_SUBSTITUTEハンドラで制御）
- 自分が対象の技（target != "foe" の技）
- 場の状態によるダメージ・効果（設置技・フィールド・天気など）
- 既にかかっている状態異常・状態変化の効果

## 詳細仕様

- 攻撃技のダメージはみがわりのHPから差し引かれ、本体のHPは変化しない
- みがわりのHPが攻撃ダメージを上回る場合、差分のHPがみがわりに残る
- みがわりのHPが攻撃ダメージ以下の場合、みがわりが消滅する（残ダメージは本体に入らない）
- バインド状態はみがわり成功時に解除される（第三世代以降）
- マジックコート状態での変化技跳ね返しは、みがわりよりも優先される
