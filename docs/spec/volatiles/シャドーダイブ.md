# 仕様書: シャドーダイブ（揮発状態）

調査日: 2026-06-20
参照URL: https://wiki.pokemonwiki.com/wiki/シャドーダイブ_(状態変化)

## 概要

シャドーダイブ・ゴーストダイブを使用した1ターン目に付与される揮発状態。ポケモンが影の中に潜んでいる状態を表す。

## 発生条件

- 技「シャドーダイブ」または「ゴーストダイブ」の1ターン目（溜めターン）に発生する。

## 消滅条件

- シャドーダイブ・ゴーストダイブの2ターン目（攻撃ターン）が完了した時点で解除される。
- 状態異常・状態変化などにより技が失敗した場合も解除される。

## 効果

### 回避

- シャドーダイブ状態のポケモンに対して、ほとんどの技は命中しない（ミス）。
- 以下の条件を満たす技は命中する：
  - ロックオン状態のポケモンが使用した技
  - 自身か相手が特性ノーガードの場合
  - （本実装スコープ外：てだすけを使用した場合、毒タイプのポケモンがどくどくを使用した場合）
- フリーフォールは上記条件を満たしていても必ず失敗する（本実装スコープ外）。
- 必中わざも当たらなくなる。
- へんしん・ものまね・さきどりなどの「対象に影響を与えないわざ」も失敗する。

### フィールド効果回避

- シャドーダイブ状態中はフィールドの効果を受けない（本実装スコープ外）。

### まもる貫通

- シャドーダイブ・ゴーストダイブはまもる系の状態を取り除いてから攻撃する。
- 本実装では `unprotectable` ラベルにより、まもる中の相手にも攻撃が通る。
- ワイドガード・ファストガード・トリックガードなどもまとめて貫通する。
- ニードルガード・キングシールド・トーチカ・ブロッキング・スレッドトラップの効果は発動しない（本実装スコープ外：まもる系を「取り除く」という挙動まで再現しない）。

### 行動固定

- シャドーダイブ状態のポケモンのコマンドは強制（FORCED）に固定される。
- PPは消費されない（溜めターン）。

### 交代禁止

- シャドーダイブ状態中は交代できない（ゴーストタイプを除く）。

### その他（スコープ外）

- 天候ダメージ（すなあらし・あられ）は通常通り受ける。
- いかく・きんちょうかんの効果は受け続ける。
- わたげによるすばやさ低下は受けない（本実装スコープ外）。
- ボールによる捕獲はできなくなる（本実装スコープ外）。
- おどりこが発動しなくなる（本実装スコープ外）。

## 本実装での対象外（スコープ外）

- フィールド効果回避（フィールドの影響を受けない）は未実装。
- てだすけによる命中（第四世代では失敗）は未実装。
- フリーフォール強制失敗は未実装。
- まもる系揮発状態の解除メッセージ（「守りを打ち破った！」）は未実装。
- ニードルガード・キングシールド・トーチカ・ブロッキング・スレッドトラップの効果不発動は未実装。
- わたげによるすばやさ低下回避は未実装。
- ボール捕獲不可は未実装。
- おどりこ不発動は未実装。
