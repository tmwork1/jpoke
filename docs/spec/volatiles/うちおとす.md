# 仕様書: うちおとす（揮発状態）

調査日: 2026-06-19
参照URL: https://wiki.pokemonwiki.com/wiki/うちおとす_(状態変化)

## 概要

うちおとす状態とは、技「うちおとす」または「サウザンアロー」を受けたポケモンに付与される揮発状態。
対象を地面にいる状態として扱う。

## 発生条件

- 技「うちおとす」が命中した際に付与される。
- 技「サウザンアロー」が命中した際に付与される（サウザンアローは未実装）。
- 元々地面にいるポケモンには付与できない。

## 消滅条件

- 交代（ポケモンチェンジ）で解除される。

## 効果

### 浮遊判定の無効化

以下の要因によって浮いているポケモンを地面にいる扱いにする：

- ひこうタイプのポケモン
- 特性「ふゆう」のポケモン
- アイテム「ふうせん」を持つポケモン
- でんじふゆう状態のポケモン
- テレキネシス状態のポケモン（テレキネシスは現時点で未実装）

### でんじふゆう / テレキネシス状態の解除

技「うちおとす」が命中した際、対象がでんじふゆう状態またはテレキネシス状態にある場合、それらを解除した上でうちおとす状態を付与する。

（この解除処理は揮発状態ハンドラではなく、技「うちおとす」のハンドラ側で行う。揮発状態ハンドラは `ON_CHECK_FLOATING` に反応するのみ。）

## 詳細な仕様

- うちおとす状態であっても、そらをとぶ・とびはねる・フリーフォールは使用できる（じゅうりょくとは異なる）。
- ひこうタイプがこの状態になっても、ひこうタイプが消えるわけではない（はねやすめとは異なる）。
- バトンタッチで引き継がれない。
- 最初から地面にいないポケモンがうちおとす状態になっても、すでに設置されたまきびし・どくびしの効果を受けない。
- うちおとす状態のポケモンが使用するでんじふゆうは失敗する。
- うちおとす状態のポケモンに対するテレキネシスは失敗する。

## 本実装での対象外（スコープ外）

- テレキネシス状態の解除処理（テレキネシス揮発状態が未実装のため）。
- サウザンアロー技からの付与（技が未実装のため）。
- まきびし・どくびしの免除（それらの効果がうちおとすと相互作用する仕様はエントリーハザードの実装次第）。
