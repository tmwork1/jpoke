# まもる (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

まもるとは揮発性状態の一つ。
技「まもる」「みきり」を使用することで付与される（自分対象）。
そのターンに受ける攻撃技・変化技（一部を除く）を無効化する。

## 技の基本情報

| 項目 | 値 |
|------|-----|
| タイプ | ノーマル |
| 分類 | 変化 |
| PP | 10 |
| 優先度 | +4 |
| 範囲 | 自分 |
| 直接攻撃 | なし |
| 急所 | なし |
| 命中判定 | なし |
| 追加効果 | なし |
| まもる貫通 | × |

## 効果

- 使用したターンに受ける**攻撃技および相手を対象とした変化技**を無効化する
- **接触時の追加効果はなし**（キングシールド・ニードルガード・トーチカ等と異なる）

## 付与条件

- 技「まもる」または「みきり」を使用したとき（自分に付与、count=1）
- `volatile_manager.apply()` が False を返した場合（すでに付与済み）は失敗

## 消滅条件

- ターン終了時に解除される（count=1 のため使用したターンのみ有効）
- **交代**によって解除される
- フェイント / ゴーストダイブ / シャドーダイブ / いじげんホール / いじげんラッシュ のダメージを受けた場合も取り除かれる（第九世代では次ターンのまもるは成功する）

## 詳細仕様

### まもるの発生

- まもる / みきり / ニードルガード / トーチカ を使用したとき

### 防御対象

- 相手の技の効果を受けない（一部を除く）
- 必中技（スピードスター・雨のかみなり等）も防ぐことができる
- 特性ノーガードであっても防ぐことができる
- **防げないもの**:
  - 特性いかく・わたげによるランク低下
  - あられ・すなあらし等の天気ダメージ
  - 状態異常ダメージ
  - 全体の場が対象の技 / 味方の場が対象の技
  - 相手の場が対象の技（まきびし・どくびし・ステルスロック・ねばねばネット）
  - 特性ふかしのこぶしのポケモンが使用した直接攻撃技

### 変化技の無効化

- `protect_non_attack=True`（デフォルト）のため、変化技も防ぐ
- キングシールド・かえんのまもり・スレッドトラップは変化技を防げない点で異なる

### フェイントによる解除

- フェイント・ゴーストダイブ・シャドーダイブ・いじげんホール・いじげんラッシュ を受けた場合、まもる状態が取り除かれてダメージを受ける
- 攻撃がタイプ相性で無効の場合は取り除かれない
- 第六世代以降は次ターンのまもる系技は連続使用扱いにならず成功する

### 連続使用による失敗率上昇

- まもる / みきり / こらえる / ニードルガード / トーチカ / スレッドトラップ / かえんのまもり / キングシールド を連続使用すると成功率が低下する
  - 成功するごとに次ターンの成功率が 1/3 になる（下限 1/729）
  - 連続使用しなかった場合・失敗した場合・交代を挟んだ場合はリセットされる
  - **注意**: 本実装では連続使用失敗チェックは `ON_TRY_MOVE_1` の priority=30 で行われる（未実装 → スコープ外として保留）

## ターン処理の優先度

`docs/spec/turn.md` の `Event.ON_TRY_MOVE_1` より:

| Priority | 処理 |
|----------|------|
| 30 | まもる系: 連続使用により失敗（未実装・保留） |
| 100 | まもる状態による技の無効化 |

`Event.ON_TURN_END`:
- priority 未指定（デフォルト）で source:self に対して解除する

## 関連インタラクション

- **ふかしのこぶし**: `ON_CHECK_PROTECT` でFalseを返し、守りを貫通する
- **フェイント / ゴーストダイブ / シャドーダイブ / いじげんホール / いじげんラッシュ**: まもる状態を取り除く
- **みきり**: まもる volatile と同じ揮発状態を付与する（みきりを使用→まもる volatileが付与される）
- **まもる / ニードルガード / トーチカ / スレッドトラップ / かえんのまもり / キングシールド**: 連続使用成功率共有
