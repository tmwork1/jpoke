# 仕様書: ちいさくなる（揮発状態）

調査日: 2026-06-20
参照URL: https://wiki.pokemonwiki.com/wiki/ちいさくなる_(状態変化)

## 概要

技「ちいさくなる」を使用したときに付与される揮発状態。ポケモンが小さくなっている状態を表す。

## 発生条件

- 技「ちいさくなる」の使用時に発生する。

## 消滅条件

- 交代（控えに引っ込む）。

## 効果

### 特定技への必中化（第六世代以降）

Champions（第九世代相当）では、ちいさくなる状態のポケモンに対して、以下の特定技は必ず命中する：

| 技名 | タイプ |
|------|--------|
| ふみつけ | ノーマル |
| のしかかり | ノーマル |
| ドラゴンダイブ | ドラゴン |
| ヒートスタンプ | ほのお |
| ヘビーボンバー | はがね |
| フライングプレス | かくとう |
| サンダーダイブ | でんき |

これらの技は `minimize` ラベルを持つ。

### 特定技へのダメージ2倍

上記の `minimize` ラベルを持つ技を、ちいさくなる状態のポケモンに対して使用した場合、ダメージが2倍になる。
威力補正は `ON_CALC_POWER_MODIFIER` イベントで実装する（4096基準で2倍 = 8192）。

### 必中と威力2倍の相互作用

- 必中化と威力2倍は同じ技に対して同時に発動する。
- ダメージが2倍になる効果は、タイプ相性など他のダメージ補正と重ねがけされる。

## 本実装での対象外（スコープ外）

- 交代以外の消滅条件（ダイマックス）は Champions に存在しないため対象外。
- バトンタッチによる引き継ぎは Champions 未実装のため対象外。
- 第六世代以降でも、溜め技で姿を隠しているちいさくなる状態のポケモンへは当たらない仕様は、溜め技側（あなをほる等）の揮発状態で管理される。
