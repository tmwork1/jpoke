# スレッドトラップ (状態変化) 仕様書

対象: Pokémon Champions シングルバトル

## 概要

スレッドトラップとは揮発性状態の一つ。
技「スレッドトラップ」を使用することで付与される（自分対象）。
そのターンに受ける攻撃技を無効化し、直接攻撃をしてきた相手の素早さランクを1段階下げる。
変化技は防ぐことができない点でまもると異なる。
第九世代（スカーレット・バイオレット）専用技（ワナイダーの専用技）。

## 技の基本情報

| 項目 | 値 |
|------|-----|
| タイプ | むし |
| 分類 | 変化 |
| PP | 10 |
| 優先度 | +4 |
| 範囲 | 自分 |
| 直接攻撃 | なし |
| 急所 | なし |
| 命中判定 | なし |
| 追加効果 | なし |
| まもる貫通 | × |

## 効果

- 使用したターンに受ける**攻撃技**を無効化する（**変化技は防げない**）
- 直接攻撃技を防いだ場合、攻撃者の素早さ（S）ランクを1段階下げる

## 付与条件

- 技「スレッドトラップ」を使用したとき（自分に付与、count=1）

## 消滅条件

- ターン終了時に解除される（count=1 のため、使用したターンのみ有効）
- **交代**によって解除される

## 詳細仕様

### 攻撃技の無効化

- そのターンに受ける攻撃技（`is_attack=True` の技）を無効化する
- **変化技は防げない**（`protect_non_attack=False`）
  - まもる / みきり / ニードルガード / トーチカ が変化技を防ぐのと異なる
  - かえんのまもり / キングシールド / ブロッキング と同様に攻撃技のみを防ぐ
- まもる系と同じ `ON_TRY_MOVE_1` イベントで処理される（priority=100）

### 素早さランクの低下

- 直接攻撃（`is_contact=True`）を防いだ場合のみ攻撃者の素早さ（S）ランクを1段階下げる
- 非接触の攻撃技（れいとうビームなど）を防いでも素早さランクは下がらない
- `ぼうごパット` を持つ攻撃者は接触判定が False になるため素早さランクが下がらない
- タイプ相性で無効な直接攻撃を防いだ場合も素早さランクが下がる

### 防御が無視されるケース（今世代スコープ外）

- `ふかしのこぶし` 特性の直接攻撃では守りを貫通し、素早さランクも下がらない
  - `ON_CHECK_PROTECT` ハンドラで `ふかしのこぶし` が False を返す

### ランクダウンの相互作用

- `クリアボディ` / `しろいけむり` / `メタルプロテクト` / `かいりきバサミ` や `しろいきり` 状態のポケモンは素早さランクが下がらない
- `まけんき` / `かちき` / `あまのじゃく` / `たんじゅん` の効果対象となる
- `ミラーアーマー` のポケモンの直接攻撃を防ぐと、効果が跳ね返されてスレッドトラップ使用者の素早さランクが下がる

### 連続使用による失敗率上昇

- まもる/みきり/こらえる/ニードルガード/トーチカ/スレッドトラップ/かえんのまもり/キングシールド を連続使用すると成功率が低下する
  - 成功するごとに次ターンの成功率が 1/3 になる（下限 1/729）
  - 連続使用しなかった場合・失敗した場合はリセットされる
  - **注意**: 本実装では連続使用失敗チェックは `ON_TRY_MOVE_1` の priority=30 で行われる

## ターン処理の優先度

`docs/spec/turn.md` の `Event.ON_TRY_MOVE_1` より:

| Priority | 処理 |
|----------|------|
| 30 | まもる系: 連続使用により失敗 |
| 100 | スレッドトラップ状態による攻撃技の無効化 |

`Event.ON_TURN_END`:
- priority 未指定（デフォルト）で source:self に対して解除する

## 関連インタラクション

- **ぼうごパット**: 攻撃者のis_contactをFalseにするため、素早さランクが下がらない
- **ふかしのこぶし**: `ON_CHECK_PROTECT` でFalseを返し、守りを貫通する（素早さランクも下がらない）
- **まもる / ニードルガード / トーチカ / キングシールド / かえんのまもり**: 連続使用成功率共有
- **ワイドガード / ファストガード**: 連続使用成功率共有に含まれるが、スレッドトラップの失敗率に影響しない
