# まきびし（場の状態）仕様書

## 概要

- **種別**: サイドフィールド（相手の場）
- **最大層数**: 3 層
- **発生源**: 技「まきびし」

## まきびしの効果

交代により **地面に着地した** ポケモンが場に出てきたとき、層数に応じた割合の最大 HP ダメージを受ける。

| 層数 | ダメージ割合 |
|------|--------------|
| 1 層 | 最大HPの 1/8 |
| 2 層 | 最大HPの 1/6 |
| 3 層 | 最大HPの 1/4 |

- ダメージは小数点以下を切り捨てる。ただし、切り捨てた結果が0になる場合は1ダメージを与える。
- 特性「マジックガード」のポケモンや、「あつぞこブーツ」を持つポケモンはダメージを受けない。
- このダメージでHPが0になったポケモンは、かがくへんかガス・きんちょうかん以外の「場に出たときに発動する特性」を発動できずにひんしとなる。

## 無効条件（地面にいない扱い）

以下のポケモンには **ダメージが発生しない**（まきびしは消滅しない）。

| 条件 | 詳細 |
|------|------|
| ひこうタイプ | タイプによる無効 |
| 特性ふゆう | 特性による無効 |
| でんじふゆう・テレキネシス状態をバトンタッチで引き継いだポケモン | 状態変化による無効 |
| ふうせんを持っているポケモン | アイテムによる無効 |

以下の条件に当てはまる場合は、上記のポケモンも地面にいる扱いになりダメージを受ける。

| 条件 | 詳細 |
|------|------|
| くろいてっきゅうを持っている場合 | 浮遊無効を上書き |
| 場がじゅうりょく状態の場合 | 浮遊無効を上書き |
| ねをはる状態もバトンタッチで引き継いだ場合 | 浮遊無効を上書き |

- 特性ぶきよう・バトンタッチでさしおさえ状態を引き継がれたポケモン・マジックルーム状態の場に出てきたポケモンは、持ち物を無視して判定を受ける（あつぞこブーツ・ふうせんを無視する）。
- バトンタッチでいえき状態を引き継がれたポケモン・かがくへんかガスが発動中の場に出てきたポケモンは、特性を無視して判定を受ける（ふゆうを無視する）。

## まきびしの消滅

| 消滅条件 | 対象の場 |
|----------|----------|
| こうそくスピン・キラースピン を使用 | 使用者の場のみ |
| きりばらい を使用 | 両陣営 |
| キョダイフウゲキ・おかたづけ | 両陣営 |

## 関連

- 技「まきびし」は変化技（じめんタイプ、対象は相手の場）で、使用すると相手陣営にこのフィールドを1層設置する（最大3層）。
- 状態異常・変化技を跳ね返す効果（マジックコート・マジックミラー）の対象になる。
