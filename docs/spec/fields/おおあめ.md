# おおあめ 仕様書

## 概要

- カテゴリ: 強天候
- 継続ターン: カウントダウンなし（永続。特性「はじまりのうみ」を持つポケモンが場にいる間）
- 上書き制限: 通常天候（はれ・あめ・ゆき・すなあらし）および強天候（おおひでり）から上書き不可
  - らんきりゅう（優先度 2）のみ上書き可能

## 天候優先度

| 天候 | 優先度 |
|------|--------|
| 通常天候（はれ・あめ等） | 0 |
| おおひでり・おおあめ | 1 |
| らんきりゅう | 2 |

## 効果

### 威力補正（ON_CALC_POWER_MODIFIER）

「あめ」と共通のハンドラ `あめ_power_modifier` を使用する。

- みずタイプ技の威力が 1.5 倍（固定倍率 6144 / 4096）
- ほのおタイプ技の威力が 0.5 倍（固定倍率 2048 / 4096）
- **防御側の効果**とみなされる
  - 防御側がばんのうがさを持つ場合、天候補正は無効になる

### ほのおタイプ技の失敗（ON_TRY_MOVE_1, priority=10）

- ほのおタイプの技（攻撃技・変化技を問わない）を使用すると失敗する
- ログ: `LogCode.MOVE_FAILED`、payload `{"reason": "おおあめ"}`
- `HandlerReturn(value=False, stop_event=True)` を返す
- subject_spec: `"attacker:self"`（AttackContext）

### こおり状態

- おおあめ中にこおり状態が付与されることはある（こおり防止ハンドラは存在しない）
- はれ・おおひでりとは異なり、こおりを防ぐ効果はない

## ターン終了処理

ON_TURN_END ハンドラなし（カウントダウン不要のため）。

## ばんのうがさ相互作用

- 防御側がばんのうがさを持つ場合、みず/ほのお技威力補正は無効（あめと同仕様）
- ほのお技の失敗ブロックはばんのうがさの影響を受けない
