# オーロラベール 仕様書

## 概要

- カテゴリ: 自陣サイドフィールド
- 継続ターン: 通常5ターン（ひかりのねんど持ちポケモンが発動した場合は8ターン）
- 対象: 自陣のポケモン

## 効果一覧

| 効果 | イベント |
|------|----------|
| 攻撃技による物理・特殊ダメージを1/2に軽減 | ON_CALC_DAMAGE_MODIFIER |
| ターン終了時にカウントダウンし、0になると終了 | ON_TURN_END |

## 詳細な仕様

- シングルバトルではダメージを1/2に軽減（×2048/4096）
- ダブルバトルでは2/3に軽減（×2732/4096）※本プロジェクト対象外
- 急所に当たったときはダメージを軽減できない
- こんらん状態で自分を攻撃したときのダメージは軽減できない（こんらんによるダメージは攻撃ではない）
- 特性「すりぬけ」の敵ポケモンから受けるダメージは軽減できない（`ctx.can_bypass_screen` で判定）
- かわらわり・サイコファング・レイジングブルのダメージは軽減できない（これらの技は使用時に壁を解除する）
- リフレクター（物理技）またはひかりのかべ（特殊技）が同時に有効でも効果は重複せず、1/2のまま変わらない

## 発動条件（技としての使用）

- 天気が「ゆき」でない場合は失敗する（ON_TRY_MOVE_1, priority=30）
- すでにオーロラベールが発動中の場合は失敗する

## 解除条件

- ターン経過（カウント0になったとき）
- きりばらい
- かわらわり・サイコファング・レイジングブル（これらを受けた場合、ダメージ後に解除）
- キョダイフウゲキ（ダメージ軽減後に解除）
- バリアフリー

| イベント | Priority | 根拠 |
