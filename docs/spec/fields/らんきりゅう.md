# らんきりゅう 仕様書

## 概要

- カテゴリ: 強天候
- 継続ターン: カウントダウンなし（永続。特性「デルタストリーム」を持つポケモンが場にいる間）
- 上書き制限: 通常天候・おおひでり・おおあめで上書き不可
  - はじまりのうみ・おわりのだいちのみ上書き可能

## 天候優先度

| 天候 | 優先度 |
|------|--------|
| 通常天候（はれ・あめ・ゆき・すなあらし） | 0 |
| おおひでり・おおあめ | 1 |
| らんきりゅう | 2 |

らんきりゅうは最高優先度 2 を持つため、通常天候・おおひでり・おおあめいずれも上書きできない。
逆に、らんきりゅうはすべての天候（通常・強天候）を上書きできる。

## 発生方法

特性「デルタストリーム」を持つポケモンが場に出た際に発動する。
ポケモンが引っこめられるか特性が書き換えられると解除される。

## 効果

### タイプ相性補正（ON_CALC_DEF_TYPE_MODIFIER）

- ひこうタイプのポケモンに対して、ひこうの弱点タイプ（でんき・いわ・こおり）の技のダメージが 0.5 倍に軽減される
- 軽減の対象: 防御側がひこうタイプを持ち、かつ技タイプがでんき・いわ・こおりのいずれか
- 本来 2 倍弱点の技が 0.5 倍で軽減されると実質等倍（4096）になる
- subject_spec: `"defender:self"`（AttackContext）

### 補足仕様（実装対象外）

- ひこう以外のタイプで弱点を相殺できる場合でも、ひこう弱点タイプであれば必ず半減される
  - 例: メガレックウザへのでんき技、ランドロスへのいわ技
- ひこうタイプを持たないポケモンには適用されない（通常のタイプ相性計算を行う）
- エアロック・ノーてんきで無効化される
- ウェザーボールの威力・タイプは変化しない（いつも通り通常威力50・ノーマルタイプ）
- こうごうせい等の回復量は 1/2 のまま変化しない
- ステルスロックのダメージは変化しない

## ターン終了処理

ON_TURN_END ハンドラなし（カウントダウン不要のため）。
