# ワンダールーム 仕様書

## 概要

- カテゴリ: 全体の場の状態（グローバルフィールド）
- 継続ターン: 5ターン（発動ターンを含む）

## 効果一覧

| 効果 | 対象 | イベント |
|------|------|----------|
| 物理技受時に防御実数値→特防実数値へ差し替え（比率換算） | 防御側ポケモン | ON_CALC_DEF_MODIFIER |
| 物理技受時のランク補正参照を防御→特防へ差し替え | 防御側ポケモン | ON_CALC_DEF_RANK_MODIFIER |
| 特殊技受時に特防実数値→防御実数値へ差し替え（比率換算） | 防御側ポケモン | ON_CALC_DEF_MODIFIER |
| 特殊技受時のランク補正参照を特防→防御へ差し替え | 防御側ポケモン | ON_CALC_DEF_RANK_MODIFIER |
| ターン終了時にカウントダウンし、0になると終了 | 両プレイヤー共通（1P側のみ実行） | ON_TURN_END（priority=140） |

## ダメージ計算の詳細

ワンダールーム中、技を受けるポケモンは実数値とランク補正の参照先が入れ替わる:

- **物理技**: 特防実数値 × 防御ランク補正（通常は防御実数値 × 防御ランク補正）
- **特殊技**: 防御実数値 × 特防ランク補正（通常は特防実数値 × 特防ランク補正）

技の分類自体は変化しない（リフレクター→物理技軽減、ひかりのかべ→特殊技軽減 は通常通り）。

## 発動方法

- 技: ワンダールーム（エスパータイプ・変化技・優先度 -7）

## 消滅条件

- 5ターン経過（発動ターン含む）
- 発動中にワンダールームを再使用（即時解除）

## 再使用時の挙動

ワンダールームが発動中に再度ワンダールームを使用すると、フィールドが即時解除される。
move_status ハンドラ `ワンダールーム_activate_global_field` がアクティブ状態を確認して分岐する。

## 実装の注意

`ON_CALC_DEF_MODIFIER` イベントの `value` は 4096 ベースの補正値。
ワンダールームでは `(swap_value / base_value)` の比率を 4096 倍値で渡すことで、
ダメージ計算内の「実数値 × ランク → DEF_MODIFIER」の順で正しい値を出す。

## 範囲外（現行実装非対象）

以下の仕様は複雑な相互作用を含むため、現行実装では対象外とする:

- サイコショック・サイコブレイク・しんぴのつるぎ との相互作用
  （通常時: 特防実数値 × 防御ランク補正 → ワンダールーム時: 防御実数値 × 特防ランク補正）
- せいなるつるぎ・なしくずし・DDラリアット・てんねん特性との相互作用
- ビーストブースト・こだいかっせい・クォークチャージ との相互作用
- パワートリック との相互作用
- ルームサービス との相互作用（ワンダールームでは発動しない）
