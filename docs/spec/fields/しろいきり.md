# しろいきり 仕様書

## 概要

- カテゴリ: 自陣サイドフィールド
- 継続ターン: 5ターン
- 対象: 自陣のポケモン

## 効果一覧

| 効果 | イベント |
|------|----------|
| 相手（または第三者）による能力ランク低下を防ぐ | ON_BEFORE_MODIFY_STAT |
| ターン終了時にカウントダウンし、0になると終了 | ON_TURN_END |

## 詳細な仕様

- 自分以外のポケモンによる技や特性の効果では能力ランク（こうげき・ぼうぎょ等）が下がらない。
- すでに下がったランクを回復する効果はない。
- **防げる例**:
  - 相手の技による能力低下（おたけび、ガリョウテンセイの副効果等）
  - いかくによる攻撃ランク低下
  - ねばねばネット・ねばねばネット状態（たこがため）によるランク低下
- **防げない例**:
  - 自分自身の技の効果による自己ランク低下（からをやぶる・のろい等の反動）
  - 特性くだけるよろい・ムラっけによる自己ランク低下
  - ルームサービスによるランク低下
  - くろいきり・クリアスモッグ・シャドースチールによるランクリセット
  - パワースワップ・ガードスワップ・ハートスワップ・ひっくりかえすによるランク変動
  - バトンタッチで引き継いだ下降ランク
- 特性すりぬけを持つ相手が使う技・特性の効果は防げない（`can_bypass_status_guard` で判定）。

## 発動条件（技としての使用）

- 技「しろいきり」または特性「エフェクトガード」によって自陣に設定される。
- 本プロジェクトでは技「しろいきり」のハンドラとして実装する（技実装が完了すれば自動的に発動する）。

## 解除条件

- ターン経過（カウント0になったとき）
- きりばらい（解除直前のきりばらいによるランク低下は防ぐ）
