# おいかぜ 仕様書

## 概要

- カテゴリ: 自陣サイドフィールド
- 継続ターン: 4ターン（発動ターンを含む）
- 対象: 自陣のポケモン

## 効果一覧

| 効果 | イベント |
|------|----------|
| 場に出ているポケモンのすばやさが2倍になる | DomainEvent.ON_CALC_SPEED |
| ターン終了時にカウントダウンし、0になると終了 | ON_TURN_END |

## 詳細な仕様

- すばやさの実数値を2倍にする（ランク補正の後、まひ状態の前に計算）
- 第8世代以降では、発動したターンから行動順においかぜの影響がある
- ジャイロボール/エレキボールの威力算出にも影響する（発動ターンから影響）
- 重ねがけはできない（すでに発動中の場合は失敗）

## 発動条件（技としての使用）

- すでにおいかぜが発動中の場合は失敗する（ON_STATUS_HIT で stop_event=True）
- 自陣営に「おいかぜ」を4ターン設定する

## 解除条件

- ターン経過（カウント0になったとき、Priority 130 ON_TURN_END）
- きりばらいでは解除されない

## 特性との相互作用

- **かぜのり**: おいかぜ発生時、またはおいかぜ状態の場に場出したとき、こうげきが1段階上がる
  - すでにおいかぜ状態の場に出たとき → かぜのりは発動する
  - コートチェンジでおいかぜが移動してきたとき → かぜのりは発動しない
- **ふうりょくでんき**: おいかぜが発生したときにじゅうでん状態になる
  - すでにおいかぜ状態の場に出たとき → ふうりょくでんきは発動しない
  - コートチェンジでおいかぜが移動してきたとき → ふうりょくでんきは発動しない

## 発動優先度

| イベント | Priority | 根拠 |
|----------|----------|------|
| ON_TURN_END（カウントダウン） | 130 | docs/spec/turn.md: 「おいかぜの継続・終了」Priority 130 |
| DomainEvent.ON_CALC_SPEED（素早さ2倍） | 100（デフォルト） | 既存サイドフィールドと同じデフォルト値 |
| ON_STATUS_HIT（発動） | 100（デフォルト） | 既存サイドフィールドと同じデフォルト値 |
