# みかづきのまい 仕様書

## 概要

- カテゴリ: 自陣サイドフィールド（場の状態として管理）
- 継続ターン: ターン経過では解除されない（消費されるまで永続）
- 対象: 使用者の自陣に入場したポケモン

技としての基本情報（タイプ・PP・命中率など）は `docs/spec/moves/みかづきのまい.md` を参照。

## 効果一覧

| 効果 | イベント |
|------|----------|
| 控えのポケモンがいない場合は失敗する | ON_BEFORE_APPLY_MOVE |
| 使用者をひんしにして自陣に「みかづきのまい」フィールドを設置 | ON_STATUS_HIT |
| 場に出たポケモンの HP・全ての技の PP を全回復し、状態異常を解除（第八世代以降：
  HP満タン・状態異常なし・PP満タンなら保留） | ON_SWITCH_IN（priority=100） |

## 詳細な仕様

### 技の使用時

- 控えのポケモンがいない場合は失敗し、使用者はひんしにならない（`ON_BEFORE_APPLY_MOVE` で判定）
- 成功した場合、使用者は「みかづきのまい」フィールドを自陣に設置した後、ひんしになる

### フィールドの発動（第五世代以降、本プロジェクト対象世代）

- 自陣にポケモンが場に出たとき（`ON_SWITCH_IN`）に発動判定
- HP を最大値まで回復する
- 状態異常（状態変化ではなく `ailment`）を治す。こんらんなどの揮発性状態（`volatile`）は治さない
- 全ての持っている技の PP を最大値まで回復する（`いやしのねがい` との違い）
- いずれかを回復したら（＝発動条件を満たしたら）フィールドは解除される

### 第八世代以降（ポケモンチャンピオンズ対象）の特殊ルール

- 場に出たポケモンの HP が満タン **かつ** 状態異常がない **かつ** 全ての技の PP が満タンの場合は
  発動しない（フィールドは保留される）
- いずれか1つでも回復できる要素があれば効果が発動する

### かいふくふうじとの相互作用

- かいふくふうじ状態のポケモンにも、みかづきのまいの効果自体は発動する
- HP は回復できない（`ON_MODIFY_HEAL` 経由で `かいふくふうじ_block_heal` にブロックされる）が、
  状態異常および PP は回復する（`ailment_manager.remove` と PP リセットは `ON_MODIFY_HEAL` を
  経由しないため、かいふくふうじの影響を受けない）
- 本プロジェクト（シングルバトル）では、場に出た直後のポケモンは switch-out 時に
  volatile が全解除されるため、実質的にかいふくふうじ状態で ON_SWITCH_IN を迎えることはない。
  ただし将来的な拡張や不正な状態遷移に備え、実装は原作仕様に準拠しておく

### 設置技との順序

- 第五世代以降：みかづきのまいによる回復が設置技（まきびし・ステルスロックなど）の効果より
  **前**に発動する（`docs/spec/turn.md` の `ON_SWITCH_IN` priority 順）

### いやしのねがい状態との重複

- 1箇所にみかづきのまい状態は重複しない（`FieldManager.activate` は既に有効な場合は再設置しない）
- 1箇所にみかづきのまい状態といやしのねがい状態を重複させることはできる。この場合、
  先に使われた技の方が優先して発動する（`ON_SWITCH_IN` の同一 priority 内では、フィールドの
  `register_handlers` が呼ばれた順、すなわち先に設置された技のハンドラが先に実行されるため、
  実装上の追加対応は不要）

### フィールドの持続

- ターン経過では解除されない（`ON_TURN_END` ハンドラなし）
- 消費されるか、きりばらい等の解除手段が使用されるまで有効

## 発動優先度

| イベント | Priority | 根拠 |
|----------|----------|------|
| ON_BEFORE_APPLY_MOVE（失敗チェック） | 100（デフォルト） | `docs/spec/fields/ねがいごと.md` の同種チェックに合わせたデフォルト |
| ON_STATUS_HIT（ひんし＋フィールド設置） | 100（デフォルト） | 既存の変化技ハンドラに合わせたデフォルト |
| ON_SWITCH_IN（HP・状態異常・PP回復） | 100 | `docs/spec/turn.md`: 「みかづきのまいによる回復」Priority 100 |
