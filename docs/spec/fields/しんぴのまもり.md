# しんぴのまもり（場の状態）仕様書

## 概要

「しんぴのまもり」は味方の場に発生するサイドフィールド。
5ターンの間、場のポケモンが**他のポケモンからのわざの効果**で状態異常・こんらん・ねむけ状態にならない。

## 効果

- 他のポケモンのわざによる**状態異常**（どく・もうどく・まひ・やけど・ねむり・こおり）を防ぐ
- 他のポケモンのわざによる**こんらん**を防ぐ
- 他のポケモンのわざによる**ねむけ**を防ぐ
- 自分が使う**ねむる**は対象外（ねむり状態になれる）
- 特性（第五世代以降）：どくのトゲ・せいでんき・ほのおのからだ・ほうし・シンクロによる状態異常は防ぐ
- 特性：どくしゅ・うのミサイルによる状態異常は防ぐ
- すでに発生している状態異常・こんらん・ねむけは治らない
- メロメロ・ふういん・ほろびのうたなどの状態変化は防げない

## 防がないもの

- いばる（こんらんは防ぐが、こうげき上昇は防げない）
- おだてる（こんらんは防ぐが、とくこう上昇は防げない）
- あばれる解除によるこんらん（第三世代以降）
- 自分のわざ（ねむる）による状態異常

## 発生

- 技「しんぴのまもり」を使用する

## 消滅

- 5ターン経過（priority=130 で毎ターン終了時にカウントダウン）
- きりばらい
- ダークリムーブ（未実装）
- キョダイフウゲキ（未実装）

## 特性との相互作用

- **すりぬけ**：敵側のしんぴのまもりを無視する（自分側は無視しない）
- **バリアフリー**：しんぴのまもりは解除されない（壁とは異なる）

## その他

- ひかりのねんどの効果で持続時間は変更されない
- かわらわり・サイコファングでは解除されない
- コートチェンジで入れ替わる対象になる

## イベント優先度

| イベント | priority | 備考 |
|---|---|---|
| ON_BEFORE_APPLY_AILMENT | 100 (default) | target:self |
| ON_BEFORE_APPLY_VOLATILE | 100 (default) | target:self（こんらん・ねむけのみ） |
| ON_TURN_END | 130 | source:self（ターンカウントダウン） |
