# 仕様書: マジックガード

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%B8%E3%83%83%E3%82%AF%E3%82%AC%E3%83%BC%E3%83%89

## 効果
相手の攻撃技以外によるダメージを無効化する。具体的には以下のダメージを受けない。
- わざの補助効果によるダメージ
  - 反動技による反動ダメージ : ビックリヘッド/てっていこうせん/クロロブラストの反動、とびげり/とびひざげり/かかとおとし/サンダーダイブの失敗時の反動も受けない。
    - 例外として、わるあがきの反動ダメージは受ける。
  - ニードルガード
  - はじけるほのおの火花
- 状態異常によるダメージ
  - どく/もうどく
  - やけど (物理技のダメージは通常どおり半減する)
- 状態変化によるダメージ : ただし、こんらん/ほろびのうた/みちづれ状態によるダメージは防げない。
  - あくむ/のろい/しおづけ
  - やどりぎのタネ : ダメージを無効化するため、使用者の位置にいるポケモンを回復させることもない。
  - バインド : ダメージのみを防ぐ。にげる/交代はできなくなる。
  - ふんじん : ほのおタイプの技を使おうとすると失敗するが、ダメージは受けない。
- 場の状態によるダメージ
  - 自分の場のステルスロック/まきびし/キョダイコウジン
  - ひのうみ/キョダイベンタツ/キョダイゴクエン/キョダイホウゲキ/キョダイフンセキ : 戦闘アニメが再生される環境で、無効化できるタイプを持たない場合は、ターン終了時にこれらの状態のエフェクトだけが発生する。
  - てんきのすなあらし/あられ
- もちものによるダメージ : 道具の効果が発動するときのメッセージも流れなくなる。
  - 自分のいのちのたま/くっつきバリ/くろいヘドロ
  - 相手のゴツゴツメット
  - 相手のジャポのみ/レンブのみ : これらのきのみは消費もされない。
- 相手が持つダメージを与える特性によるダメージ
  - うのミサイル/さめはだ/てつのトゲ/とびだすなかみ/ナイトメア/ヘドロえき/ゆうばく : 第五世代以降では、相手の特性バーだけは表示される。特性発動時のメッセージは流れない。

## 特性の仕様
- 攻撃技のダメージでなくても、以下のダメージは受ける。
  - ひんしになるわざ/はらだいこ/みがわり/のろい (呪い) /ソウルビート/しっぽきり/みをけずる使用時のHP消費
  - こんらん状態/命令無視による自傷ダメージ
  - わるあがきの反動ダメージ
  - いたみわけの使用時、または受けた時
  - ほろびのうた/みちづれ状態が発動したことによるひんし
- みらいよち/はめつのねがいによるダメージは受ける。
- マジックガードのポケモンが特性ミイラ/とれないにおい/さまようたましいのポケモンに直接攻撃を使用した場合、マジックガードが上書き/交換された後に、相手のゴツゴツメット/ジャポのみ/レンブのみのダメージや、自分の反動技のダメージといのちのたまの反動ダメージを受ける。
- 特性かがくへんかガスのポケモンを倒したことでかがくへんかガスが解除される場合、相手のゴツゴツメット/ジャポのみ/レンブのみは発動してダメージは受けるが、反動技のダメージといのちのたまの反動ダメージは受けない。
  - 攻撃技に対して、かがくへんかガスがだっしゅつボタンの効果で交代したことでガスが解除される場合、反動技のダメージは受けるが、いのちのたまのダメージは受けない。
- かたやぶりの効果がある技を受けてもマジックガードは無視されない。
- マジックガードの効果が発動しても、それを明示するメッセージは流れない。第五世代以降でも、発動時にマジックガードの特性バーは表示されない。
- マジックガードによりもうどくのダメージを防いでいる間も、もうどく状態でいるターンは記録されている。例えばもうどく状態になってから5ターン目に特性を失った場合、そのターン終了時に最大HPの5/16分のダメージを受ける。

## 備考
- 初登場の第四世代ではピィ系統の専用特性だった。
- いしあたまの上位互換と言える効果を持つ。
  - 第五世代でのみ、ぼうじんの上位互換とも言える効果だった。第六世代以降ではぼうじんの仕様が変更されたため、上位互換ではなくなっている。
- 第四世代のみ、フィールド上でのどくダメージは防げなかった。また、ダメージを防ぐ以外にも
  「まひ状態でも行動不能にならない（すばやさは通常どおり1/4）」「どくびしでどく/もうどくに
  ならない」という効果があった（対象範囲外の過去仕様のため実装対象外）。
