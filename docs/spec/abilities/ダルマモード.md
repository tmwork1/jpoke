# 仕様書: ダルマモード

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%80%E3%83%AB%E3%83%9E%E3%83%A2%E3%83%BC%E3%83%89

## 効果
自分のHPが最大HPの1/2以下になると、そのターン終了時にダルマモードへ変化する。回復してHPが1/2を超過すると、ターン終了時に元のすがたに戻る。

## 特性の仕様
- フォルムチェンジを発動させる特性の1つ。
  - フォルムチェンジ前でタイプやステータスの実数値が変更されていた場合、フォルムチェンジと同時にタイプとステータスがフォルム本来のものに再設定される。
- フォルムチェンジするタイミングはターン終了時、場の状態/状態異常/状態変化/もちものの効果が発動した後になる。(ターン#5.ターン終了時の処理を参照)
  - みらいよち/はめつのねがいはフォルムチェンジ前の姿で受ける。
  - すばやさに関係なく、特性の発動はかそく/ムラっけ/しゅうかく/ものひろい/ナイトメアより遅い。はらぺこスイッチよりは早い。
  - スワームチェンジ/リミットシールド/ぎょぐんとは同じ優先順位で発動する。すばやさ補正は考慮せず、ボールから出るときのフォルムのすばやさ実数値が発動順に影響する。
- ダルマモードの状態で交代するとダルマモードは解除され通常の姿に戻る。
  - HP1/2以下の状態で場に出ても直後にダルマモードにはならない。発動にはターンの終了まで待つ必要がある。
- ダルマモードに対してトレースは発動しない。
- へんしん/かわりものにより、他のポケモンが特性ダルマモードのヒヒダルマに変身することは可能だがフォルムチェンジはできず、変身した際のフォルムを維持する。
- ダイマックスしているときも、ダイマックス後のHPの割合によって特性が発動する。

### 第七世代以降
- この特性は上書きできない。とくせいなし状態になることや、かがくへんかガスで無効化されることもない。
  - とくせいなし状態をバトンタッチにより引き継いだ場合、とくせいなし状態が消える。
- この特性はコピーできない。
- この特性をスキルスワップ/さまようたましいで交換することはできない。

## 備考
- ヒヒダルマの専用特性。
- 隠れ特性せいしんりょくを持つダルマッカが特性ダルマモードを持つヒヒダルマに進化する。通常特性のヒヒダルマからせいしんりょくを持つダルマッカは生まれない。
  - 入手方法についてはダルマッカ#隠れ特性の入手方法、ヒヒダルマ#隠れ特性の入手方法を参照。
