# 仕様書: かんろなミツ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%93%E3%82%8D%E3%81%AA%E3%83%9F%E3%83%84

## 効果
その戦闘で初めて場に出たとき、相手全員の回避率を1段階下げる。

## 特性の仕様
- 発動時、「<特性所有者>の ミツから あまいかおりが ただよっている!」というメッセージが出て、その後相手の回避率を下げる。
  - いかくなど、多くの場に出たときに発動する特性とは素早さが高い順に発動する。
- 下げる能力を除けばいかくと同様の効果を持つ。詳細な効果はいかく#特性の仕様も参照とする。
  - 特性を変える技/特性によってかんろのミツを得たときも特性が発動する。
  - 発動タイミングで相手の場にポケモンがいなかった場合、特性は発動しない。次のポケモンが出された後でも発動しない。
  - みがわり状態の相手には効果が無い。
  - 姿を隠している相手の回避率も下げる。
- いかくと異なり、かんろなミツは戦闘中に1回しか発動できない。この特性を発動させたことがあるポケモンは、交代やひんしを経て再度場に出たとしても特性を発動できない。
  - 相手の場にポケモンがいなかったため特性が発動しなかった場合、次に場に出たときに発動させることができる。
  - 特性は発動したなら、クリアボディなどにより効果が無かった場合でも次に場に出たときに発動することはない。
  - その他、戦闘中1回しか発動しない仕様としては、ふとうのけんと同様の仕様を持つ。ふとうのけん#第九世代以降を参照とする。
  - 1回の戦闘中に、かんろなミツ/ふとうのけん/ふくつのたてをそれぞれ1回ずつ発動させることはできる。

## 備考
- カミッチュ・カミツオロチの専用特性。特性がじゅくせいのカジッチュをカミッチュに進化させるとこの特性になる。
