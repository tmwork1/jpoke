# 仕様書: サイコメイカー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%B5%E3%82%A4%E3%82%B3%E3%83%A1%E3%82%A4%E3%82%AB%E3%83%BC

## 効果
場に出たとき、フィールドを5ターンの間サイコフィールド状態にする。

## 特性の仕様
- フィールド#フィールドを発生させる特性も参照とする。
  - 特性ミストメイカー/エレキメイカー/グラスメイカー/ハドロンエンジンのポケモンと同時に場に出てきた場合、補正込みのすばやさが高い順に特性が発動する。先に発動したフィールドは後に発動したフィールドにより上書きされる。
  - 所有者が交代したときや、ひんしになったときもフィールドは消えない。
  - ポケモンの特性がサイコメイカーに書き換わった場合、即座に特性が発動する。
- 設置技の効果は特性より前に発動する。場に出た直後に設置技のダメージでひんしになった場合、サイコメイカーは発動しない。
- グランドコートを持つポケモンが発動させた場合、継続ターンは8ターンになる。
- サイコメイカーでフィールドが書き換わった直後、場に居るポケモンはサイコシードを消費することができる。
  - 同時に他のフィールドを張る特性でフィールドが書き換えられる場合も、すばやさに関係なくサイコシードを消費できる。
- すでに場がサイコフィールド状態になっているときは発動せず、サイコフィールドの継続ターンが書き換わることも無い。
  - 同時に他のフィールドを張る特性でフィールドが書き換えられる場合、すばやさに関係なくサイコシードを消費できない。

## 備考
- 第七世代ではカプ・テテフの専用特性だった。第八世代では隠れ特性として持つイエッサンが追加された。
