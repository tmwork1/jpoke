# 仕様書: ミラーアーマー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9F%E3%83%A9%E3%83%BC%E3%82%A2%E3%83%BC%E3%83%9E%E3%83%BC

## 効果
自身に対するランク補正を下げる効果を相手に跳ね返す。

## 特性の仕様
- 変化技/攻撃技による追加効果、ダイマックスわざによるランク低下、いかく/わたげなどのとくせいなどの効果で、他のポケモンによりミラーアーマーのポケモンの能力が下げられようとしたとき、効果を跳ね返して下げようとしたポケモンの能力を低下させる。ランク低下に関する効果以外は受けることになる。
  - ちからをすいとるを受けた場合、技の使用者の攻撃ランクを下げるが、回復量はミラーアーマーのポケモンの攻撃実数値と攻撃ランクにより決まる。ちからをすいとるの使用者がみがわり状態などで跳ね返した効果が無かったときでも回復は発動する。
  - すてゼリフを受けた場合、技の使用者の攻撃/特攻ランクを下げた後に使用者は交代する。すてゼリフ使用者のみがわり状態などで跳ね返した効果が無かったときも相手は交代する。
  - おきみやげを受けた場合、技の使用者の攻撃/特攻ランクを下げた後に使用者はひんしになる。
  - きりばらいを受けた場合、技の使用者の回避率ランクを下げるが、ミラーアーマーのポケモンを対象にして場の状態が解除される。
  - ハバネロエキスを受けた場合、自身の攻撃が上がった後に使用者の防御を下げる。
  - うのみのすがたのうのミサイルを受けた場合、ダメージは受けるがぼうぎょを下げる効果は跳ね返す。
  - いかくの効果を跳ね返した時点では、ミラーアーマーのポケモンが持つビビリだまは発動しない。場に残っている内に自身のランクが変動するとその直後にビビリだまが発動する。
    - いかくのポケモンがビビリだまを持っている場合、ミラーアーマーでいかくの効果を跳ね返されてもビビリだまは発動しない。
  - かんろなミツの効果は、1度の戦闘中に2回以上跳ね返すことができる。
  - 技の使用者と対象の特性がどちらもミラーアーマーである場合、ミラーアーマーで跳ね返したランク低下を再度ミラーアーマーで跳ね返すことはできない。
  - たこがため状態によるぼうぎょ/とくぼうランク低下は跳ね返せない。
  - あめまみれ状態による素早さの低下は防ぎ、使用者に跳ね返すことができる。
  - ねばねばネットにひっかかった場合、「<特性所持者>は ねばねばネット にひっかかった!」というメッセージが出るがすばやさは下がらない (ただし、基本的にひこうタイプのアーマーガアは地面にいないため、ねばねばネットにはひっかからない) 。
    - 第八世代のみ、このとき場にねばねばネットの使用者がいればそのすばやさが下がる。使用者が場にいなければどのポケモンのランクも下がらない。
      - コートチェンジで場の状態を入れ替えただけのポケモンは使用者とは見なされない。
      - ねばねばネットがマジックコート状態で跳ね返されていた場合、マジックコートのポケモンが使用者とみなされる。
    - 第九世代では、場に使用者がいたとしてもその素早さは下がらない。
- 自分の能力を下げる攻撃技や、のろい/からをやぶる/じこあんじなどの変化技を使って自分の能力を下げた場合や、持っていたルームサービスが発動した場合はミラーアーマーのポケモンの能力がそのまま下がる。
- 自身のみがわり/しろいきり状態、クリアチャーム/おんみつマントによりランク低下を無効にしたときや、すでに能力が最低まで下がっているため技が失敗したときはミラーアーマーは発動しない。
- 相手のみがわり/しろいきり状態やクリアボディ等の特性、クリアチャーム、ふしぎなバリアによりミラーアーマーで跳ね返した効果は防がれる。
  - 音のわざによる効果を跳ね返した場合、跳ね返す効果に音のわざの性質は無くなる。たとえばこの効果はみがわりを無視することができず、特性ぼうおんで無効化されない。
  - 特性おうごんのからだのポケモンが使用した変化技の効果を跳ね返した場合、跳ね返す効果は変化技ではなく特性の効果とみなされるため、おうごんのからだに防がれることなくランクを下げることができる。
  - テラレイドバトルでは、テラスタルシールドに対してランク低下を跳ね返すことができる。
- 命中判定より後にミラーアーマーの判定があるため、ランクを下げる変化技を回避して相手に効果を跳ね返せないことがある。
- 相手のランクがすでに最低で、跳ね返してもこれ以上下げられない場合、ミラーアーマーの特性バーは現れずに「相手の <能力>は もう 下がらない!」というメッセージだけが出る。
- 複数のポケモンの能力を下げる効果を使われた場合、この特性を持っているポケモンのみが跳ね返し、その他のポケモンが守られることはない。ミラーアーマーでランク低下を跳ね返す対象は、技や特性を発動させたポケモン単体になる。
  - 複数のミラーアーマーのポケモンが同時に能力を下げる効果を受けた場合、攻撃技による効果は使用者の味方→左側の敵→右側の敵の優先順で、いかく/わたげ/かんろなミツによる効果は左側の敵→右側の敵→使用者の味方の優先順にミラーアーマーが発動し、その分使用者は複数回ランクを低下させられる。
- 同時に複数の能力を下げる技を受けた場合、それぞれのランク低下に対して逐一ミラーアーマーが発動する。
- くろいきり/クリアスモッグ/シャドースチール/きみょうなくすりによるランクリセットは跳ね返せない(ただし相性でクリアスモッグやシャドースチールを無効にできれば能力はリセットされない)。
- パワースワップ/ガードスワップ/ハートスワップ/ひっくりかえすによるランク低下は跳ね返せない。
- バトンタッチで下がった能力が引き継がれることはある。
- ふしぎなバリアが破られたときのランク低下は受ける。
- 相手のマジックコート状態によりランクを低下させる技を跳ね返された場合、ミラーアーマーでランク低下効果を跳ね返しマジックコート状態のポケモンの能力を下げることができる。
- かたやぶりの効果が発動した技によりミラーアーマーは無視される。
- かがくへんかガスによりミラーアーマーは無効化される。
  - かがくへんかガスのポケモンからおきみやげを受けたときも、特性を無視してランクを下げられた後にガスの効果が切れる。
- 特性まけんき/かちきは、敵のミラーアーマーによりランクを下げられると発動する。味方のミラーアーマーにより下げられた場合は発動しない。
- 特性あまのじゃくのポケモンから受けたランク低下を跳ね返す場合、下がる効果は上がる効果に逆転する。
- 特性たんじゅんのポケモンから受けたランク低下を跳ね返す場合、下がる大きさは2倍になる。

## 備考
