# 仕様書: メガソーラー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%A1%E3%82%AC%E3%82%BD%E3%83%BC%E3%83%A9%E3%83%BC

## 効果
この特性を持つポケモンが行動する際、天気がにほんばれ状態であるものとして技を使用できる。

## 特性の仕様
詳細についてはにほんばれ (天気)を参照。
- 行動時ににほんばれ状態と同じ効果を受けるだけで、実際に天候が変わるわけではない。
- 場にノーてんきのポケモンがいるときでも特性の効果を受けることができる。
- ソーラービーム/ソーラーブレードを使用する場合、「光を 吸収した!」のメッセージの直後にメガソーラーの特性バーが現れ、即座に攻撃に移ることができる。
  - 場がにほんばれ以外の天候であっても威力が下がらない。
  - メガソーラーの特性バーは、特性の効果で溜めを短縮できた際にのみ表示され、他の技を使ったときには表示されない。実際の場がにほんばれ状態のため溜めターン無しで使える場合も特性バーは出ない。
- ウェザーボールは天候に関係無く威力100のほのお技になる。この際、技の選択画面からほのおタイプの技として相性予測ができる。
- ダメージ計算においてもにほんばれ状態のときと同じ効果を受けることができる。
  - ほのおタイプの技を使う際はダメージが1.5倍になる。実際の場があめ状態であるときでもダメージが半減することは無い。
  - みずタイプの技を使う際はダメージが半減する。
  - 実際の場がすなあらし状態であるときでもいわタイプによる特防補正を無視できる。ゆき状態のこおりタイプの防御補正についても同様となる。
- こうごうせい/つきのひかり/あさのひざしを使えば、天候に関係無く最大HPの2/3を回復できる。
- この特性であっても技のにほんばれは使用でき、場をにほんばれ状態に変えることができる。
- この特性により相手の特性リーフガードの効果が発動することはなく、場がにほんばれ状態でなければ状態異常にできる。
