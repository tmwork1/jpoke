# 仕様書: わるいてぐせ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%8F%E3%82%8B%E3%81%84%E3%81%A6%E3%81%90%E3%81%9B

## 効果
直接攻撃でダメージを受けたとき、相手からアイテムを奪う。
自分がすでにアイテムを持っているときや、ひんしにされた場合は奪えない。反動ダメージなどにより相手がひんしになった場合は発動する。

## 特性の仕様
### 発動条件
- 相手と相手のアイテム、もしくは自分と相手のアイテムが特定のポケモンとアイテムの組み合わせであるとき、道具を奪う効果は発動しない。このとき、わるいてぐせの特性バーも表示されない。これらのポケモンに変身した他のポケモンならこの制約を受けない。
  - ディアルガのだいこんごうだま/パルキアのだいしらたま/ギラティナのはっきんだま5-8/ギラティナのだいはっきんだま9-/アルセウスのプレート/ゲノセクトのカセット/シルヴァディのメモリ/ザシアンのくちたけん/ザマゼンタのくちたたて/オーガポンと仮面
  - メガシンカできるポケモンとメガストーン
  - ゲンシカイキできるポケモンとべにいろのたま/あいいろのたま
  - 特性こだいかっせい/クォークチャージを持つパラドックスポケモンとブーストエナジー (ウガツホムラ/タケルライコ/テツノイワオ/テツノカシラを除く）: これらの特性が無効化されたときや、他の特性に上書きされたときでも関係無く、ポケモンの種族自体が効果判定に影響する。
- 以下の道具は奪うことができない。
  - Zクリスタル
  - 敵が持つメール : 味方が持つメールは奪うことができる。
- 特性ちからずくのポケモンから追加効果がある技を受けた場合、第五世代からSVではわるいてぐせは発動しない。Championsではこのときもわるいてぐせで道具を奪える。
- 特性ねんちゃくに対してわるいてぐせは発動するが、ねんちゃくにより道具を奪えない。
  - ねんちゃくのポケモンが技の反動などでひんしになったときは、ねんちゃくは発動せずに道具を奪える。
  - 自身のレッドカードが発動し、相手を交代させたときはねんちゃくを発動させずに道具を奪える。
- 野生ポケモンが持つわるいてぐせは発動しない。
- トレーナーのポケモンが持つわるいてぐせの場合、野生ポケモンから道具を奪うことができる。
  - 第八世代まで、奪った道具はわるいてぐせのポケモンのアイテムになっている。
  - 第九世代ではその相手を倒すと戦闘終了後に道具は自動的にバッグに送られる。逃げたときも道具はバッグに送られるが、再エンカウントすると相手は道具を持った状態に戻っている。捕獲したときはその相手が道具を持っており、バッグの道具は増えない。
- トレーナー戦で奪った道具は、戦闘終了後には元の持ち主に返される。
- マックスレイドバトル/テラレイドバトルでは、敵味方関係なくわるいてぐせは発動しない。
- 自身がみがわり状態で技を防いだ場合、わるいてぐせは発動しない。
- 相手がみがわり状態である場合では、わるいてぐせの効果で道具を奪える。
- ぼうごパットを所持した相手の直接攻撃に対しても反応し、ぼうごパットを奪うことができる。
- 特性えんかくや、アイテムパンチグローブにより直接攻撃ではなくなった技に対してわるいてぐせは発動しない。

### 効果の処理順
- 直接攻撃の連続攻撃技を受けた場合、最後の攻撃を受けてから発動する。
- わるいてぐせの発動タイミングはダメージ計算の終了後となる。
  - いのちのたまは相手が反動ダメージを受けてから奪う。奪う際も技のダメージは増えている。
  - かいがらのすずは相手が回復効果を受けてから奪う。
  - おうじゃのしるし/するどいキバを奪ってもこれらの効果でひるむことがある。
  - 自身のきあいのタスキ/じゃくてんほけん/じゅうでんち/ゆきだま/ひかりごけ/きゅうこん/ふうせん/各種きのみ/きのみジュースを消費した場合、即座にわるいてぐせが発動し相手のアイテムを奪う。
  - 自身のだっしゅつボタンが発動して交代する場合、引っ込んだことでわるいてぐせは発動しなくなる。
  - 相手のとんぼがえりまたは自身のレッドカードの効果により相手の交代と同時にわるいてぐせが発動した場合、引っ込んだ相手のアイテムを奪う。
  - どろぼう/ほしがる/特性マジシャンで自身の道具を奪われた場合、即座に道具を奪い返す。
  - ついばむ/むしくい/はたきおとすで自身の道具を失った場合、即座に相手の道具を奪う。
  - 自身のくっつきバリが相手にくっついた場合、即座にわるいてぐせでくっつきバリを奪い返す。
- しろいハーブ/だっしゅつパック/ものまねハーブの消費判定はわるいてぐせの判定の後に行われる。この内、だっしゅつパック/ものまねハーブは所持した状態で起きた能力変化に対してのみ発動する仕様がある。
  - しろいハーブを/だっしゅつパックを持つ相手から、使用者の能力が下がる直接攻撃 (ばかぢからなど) を受けた場合、相手が道具を発動させるより前にわるいてぐせが発動してしろいハーブ/だっしゅつパックを奪う。
  - しろいハーブ/だっしゅつパックを自身が持っているときに、対象の能力を下げる直接攻撃 (ローキックなど) を受けた場合、わるいてぐせは発動せずにしろいハーブで自分の能力を戻すか、だっしゅつパックで交代する。
  - だっしゅつパックを持つ相手から、対象の能力を下げる直接攻撃を受けた場合、能力が下がった後にわるいてぐせが発動する。自身の能力が変化した時点では所持していなかったため、このときだっしゅつパックは発動しない。
  - ものまねハーブを持つ相手から直接攻撃を受けて、防御側の能力が上昇した場合 (アッキのみなどの効果による) 、攻撃側が道具を発動させるより前にわるいてぐせが発動する。
  - ものまねハーブを自身が持っているときに、使用者の能力が上がる直接攻撃 (ニトロチャージなど) を受けた場合、わるいてぐせは発動せずにものまねハーブのみが発動する。
  - ものまねハーブを持つ相手から使用者の能力が上がる直接攻撃を受けた場合、能力が上がった後にわるいてぐせが発動する。相手の能力が変化した時点では所持していなかったため、このときものまねハーブは発動しない。
- 相手のパワフルハーブ/ジュエルが発動した場合、わるいてぐせより先に消費されるため奪えない。
- 自身が動く前にこだわり系アイテムを奪うと、そのターン使用した行動で技が固定される。
- 自身が動く前にとつげきチョッキを奪っても、そのターンに限り変化技も使用できる。
- わるいてぐせのポケモンがさまようたましいのポケモンに直接攻撃を使用した場合、特性を交換した直後に相手に渡ったわるいてぐせが発動し、道具を奪われる。
- 特性かがくへんかガスのポケモンがわるいてぐせのポケモンに直接攻撃を使用すると同時に反動ダメージでひんしになった場合、ガスの解除後にわるいてぐせは発動する。

## 備考
- 第八世代までは隠れ特性限定の特性だった。
- 似た効果の特性として、ダメージを与えたときに相手の道具を奪う特性マジシャンがある。マジシャンは直接攻撃でなくても発動する。
