# 仕様書: テラスチェンジ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%86%E3%83%A9%E3%82%B9%E3%83%81%E3%82%A7%E3%83%B3%E3%82%B8

## 効果
場に出たとき、テラスタルフォルムにフォルムチェンジする。

## 特性の仕様
- すばやさに関係無く、かがくへんかガスを含めた他の場に出たときに発動する特性より先に発動する。
  - 複数のテラスチェンジが同時に場に出た場合、補正抜きの素早さ実数値が高い順に発動する。発動順にこだわりスカーフなどのアイテムや、トリックルーム状態などの場の状態の効果は考慮しない。
- この特性は設置技の効果より早く発動する。設置技のダメージはフォルムチェンジ後のテラスタルフォルムのHP実数値から算出される。
- この特性は上書きできない。
  - すでにかがくへんかガスが発動している場に繰り出したときもテラスチェンジは発動する。ただし、フォルムチェンジ後のテラスシェル/ゼロフォーミングは上書き可能な特性であるため、かがくへんかガスにより効果を打ち消される。
  - とくせいなし状態をバトンタッチで引き継いだ場合、とくせいなし状態は消える。場に出た瞬間にとくせいなし状態が消えるため、再度とくせいなし状態にされない限りフォルムチェンジ後のテラスシェル/ゼロフォーミングは発動できる。
- 交代によってテラスチェンジのポケモンを出したターンで、その後味方が使用するサイドチェンジが成功する場合、テラパゴスの外見だけが味方と同じ姿になる。変わるのは場に出ている外見のみで、タイプやステータスはテラパゴス (テラスタルフォルム) と同様になる。詳細はサイドチェンジ#第九世代を参照とする。
  - テラパゴスをテラスタルさせたときは、テラスタルジュエルを含めて味方がテラスタルしたときと同じ外見になる。変わるのは外見のみで、テラパゴスのタイプはステラとして扱われる。
  - テラスタル後のテラクラスターは、相手から見て左側の1体だけを対象にしたように演出されるが、隣のポケモンにもダメージを与えられている。

## 備考
- テラパゴス (ノーマルフォルム) の専用特性。
- 場に出た直後に発動し、効果を打ち消すこともできないため、基本的にテラパゴスをノーマルフォルムのまま戦闘させることはできない。
  - レッツゴーのおまかせバトルでも、テラパゴスは戦闘のたびにテラスタルフォルムに変化する。
  - イリュージョンによりノーマルフォルムに化けることはできるので、ノーマルフォルムの戦闘アニメーションを見ることはできる。
