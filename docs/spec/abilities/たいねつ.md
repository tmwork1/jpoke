# 仕様書: たいねつ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%9F%E3%81%84%E3%81%AD%E3%81%A4

## 効果
- 相手からほのおタイプの攻撃技を受けたとき、相手のこうげき・とくこうを半減してダメージ計算を行う (第九世代)。
  - 第八世代までは、ほのお技の威力を半減する。
- やけど状態で受けるダメージが半減する。

## 特性の仕様
- 相性判定は変化しない。この特性は自身のナゾのみ/オッカのみ/じゃくてんほけんの発動や、相手のたつじんのおびの発動などに影響を与えない。
- 第八世代以前であっても、特性テクニシャンの判定には影響しない。
- やけどのダメージは、第六世代までは最大HPの1/16になる。第七世代以降は最大HPの1/32になる。ただし、いずれも最低でも1のダメージを受ける。

## 備考
- 第七世代まではドータクン系統の専用特性だった。
- 第九世代では威力補正ではなくなったため、例えばほのおタイプにテラスタルしたポケモンから威力60以上120未満のほのお技を受けたとしても、ダメージは約半分になり、威力60相当に底上げされることはない。
