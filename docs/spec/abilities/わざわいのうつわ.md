# 仕様書: わざわいのうつわ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%8F%E3%81%96%E3%82%8F%E3%81%84%E3%81%AE%E3%81%86%E3%81%A4%E3%82%8F

## 効果
場にいる間、この特性を持つポケモンを除いた全員の特攻を3/4倍にしてダメージを計算する。

## 特性の仕様
- この特性のポケモンが場に出たとき、「<所有者>の わざわいのうつわで まわりの 特攻が 弱まった!」とメッセージが出る。
  - 他の多くの場に出たときに発動する特性とは補正込みの素早さが高い順に発動する。
- 特性の効果はメッセージが出た瞬間ではなく、この特性のポケモンが場にいる限り継続する。
  - メッセージの後に場に出てきたポケモンの特攻も下がる。
  - この特性のポケモンが場から去るか、他の特性に書き変えられると効果は消える。
- 場に複数体この特性を持つポケモンがいても、それ以外のポケモンに効果は累積せず、特攻は3/4倍のままとなる。
  - わざわいのうつわを持つポケモンは全員がこの特性の効果を受けず、特攻が下がらない。
  - この特性をコピーや交換によって手に入れたポケモンの特攻も下がらなくなる。
- かたやぶりの効果で無視できない。
- 他の特性のポケモンが使う分類が変わる技による分類の判定に、この特性の効果は考慮されない。分類が特殊に決まったときは、特性の効果でダメージが下がる。
- 特殊技であれば、他の特性のポケモンが使うサイコショック/サイコブレイク/しんぴのつるぎも特性の効果対象であり、ダメージが下がる。[1]
- みらいよち/はめつのねがいのダメージは、場にわざわいのうつわのポケモンがいるなら技の使用者が場にいなくても特性の効果でダメージが下がる。ただし、使用者が場にいてその特性もわざわいのうつわであるときは特攻が下がらない。
- この特性のポケモンがあなをほる状態になっても、特性の効果は切れない。
