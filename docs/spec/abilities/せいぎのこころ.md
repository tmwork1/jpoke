# 仕様書: せいぎのこころ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%9B%E3%81%84%E3%81%8E%E3%81%AE%E3%81%93%E3%81%93%E3%82%8D

## 効果
あくタイプの攻撃技でダメージを受けたとき、自分のこうげきが1段階上がる。

## 特性の仕様
- あく技のダメージ自体を無効化したり、軽減したりする効果は無い。
- 特性の発動にはあく技のダメージを受ける必要がある。
  - あくタイプの変化技を受けても発動しない。
  - まもる状態などで技を無効化したときは発動しない。
  - みがわり状態で技を防いだときは発動しない。
- あく技であるなら、カタストロフィ/ほうふくなどのダメージ固定技を受けたときでも発動する。
- すでに攻撃が最大まで上がっているときは発動しない。
- ふくろだたきなど、あくタイプの連続攻撃技を受けると、ダメージを受けるたびに特性が発動する。
- タイプが変わる技を受けた場合は、実際に出されたタイプがあくタイプであれば発動する。
  - ノーマルスキンやそうでんの効果であくタイプでなくなった技を受けても発動しない。
- イカサマを受けた場合、ダメージ計算の後に特性が発動してこうげきが上がるので、その際の上昇分はダメージに影響していない。

## 備考
- 自身があくタイプであるアブソルもこの特性を持っている。
- この特性を通常特性で持つポケモンは聖剣士とそのトリオマスターのケルディオのみで、それ以外のポケモンは全て隠れ特性で所有する。
