# 仕様書: ミストメイカー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9F%E3%82%B9%E3%83%88%E3%83%A1%E3%82%A4%E3%82%AB%E3%83%BC

## 効果
場に出たとき、フィールドを5ターンの間ミストフィールド状態にする。

## 特性の仕様
- フィールド#フィールドを発生させる特性も参照とする。
  - 特性エレキメイカー/グラスメイカー/サイコメイカー/ハドロンエンジンのポケモンと同時に場に出てきた場合、補正込みのすばやさが高い順に特性が発動する。先に発動したフィールドは後に発動したフィールドにより上書きされる。
  - 所有者が交代したときや、ひんしになったときもフィールドは消えない。
  - ポケモンの特性がミストメイカーに書き換わった場合、即座に特性が発動する。
- 設置技の効果は特性より前に発動する。
  - どくびし状態の場に出た場合、ミストメイカーが発動する前にどくびしの効果を受ける。
  - 設置技のダメージでひんしになった場合、ミストメイカーは発動しない。
- グランドコートを持つポケモンが発動させた場合、継続ターンは8ターンになる。
- ミストメイカーでフィールドが書き換わった直後、場に居るポケモンはミストシードを消費することができる。
  - 同時に他のフィールドを張る特性でフィールドが書き換えられる場合も、すばやさに関係なくミストシードを消費できる。
- すでに場がミストフィールド状態になっているときは発動せず、ミストフィールドの継続ターンが書き換わることも無い。
  - 同時に他のフィールドを張る特性でフィールドが書き換えられる場合、すばやさに関係なくミストシードを消費できない。

## 備考
- 第七世代ではカプ・レヒレの専用特性だった。
- スカーレット・バイオレットのシナリオ上ではオルティガが使用するブロロローム (ルクバー・スターモービル) もこの特性を持つ。
  - ミストフィールドが無くてもスターモービル固有の仕様として状態異常/こんらん状態は無効で、ドラゴン技も効かないが、使用するアイアンローラーが成功するようになる。
