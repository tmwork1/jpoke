# 仕様書: レシーバー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%AC%E3%82%B7%E3%83%BC%E3%83%90%E3%83%BC

## 効果
味方のポケモンがひんしになったとき、そのポケモンの特性と同じ特性になる。
コピーできない特性を持つ味方がひんしになったときは発動しない。
- スカーレット・バイオレット (Ver.1.3.2以前) では、ハドロンエンジン/ひひいろのこどうはコピーできなかったが、Ver.2.0.1以降はコピーできるようになった。
- 第八世代ではうのミサイルはコピーできなかったが、スカーレット・バイオレット (Ver.3.0.0) ではコピーできるようになった。

## 特性の仕様
- かがくのちからと全く同じ効果を持つ。かがくのちから#特性の仕様を参照とする。

## 備考
- ナゲツケサルの専用特性。
- 全く同じ効果を持つ特性としてかがくのちからがある。
