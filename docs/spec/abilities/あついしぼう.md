# 仕様書: あついしぼう

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%82%E3%81%A4%E3%81%84%E3%81%97%E3%81%BC%E3%81%86

## 効果
- 第三世代: ほのおタイプ・こおりタイプの攻撃技を受けるとき、相手のとくこうを半減してダメージを計算する。
- 第四世代: ほのおタイプ・こおりタイプの攻撃技を受けるとき、威力が半減する。
- 第五世代以降: ほのおタイプ・こおりタイプの攻撃技を受けるとき、相手のこうげき・とくこうを半減してダメージを計算する。

## 特性の仕様
- タイプ相性の判定は変わらない。
- ダメージ固定技に対しては効果が無い。
- かたやぶりの効果がある技に対しては効果が無い。
- 第三世代・第五世代以降でもランク補正を変えるわけではないので、クリアボディのポケモンなどから受ける技に対しても効果がある。
- 第四世代でもテクニシャンの判定に影響しない。
