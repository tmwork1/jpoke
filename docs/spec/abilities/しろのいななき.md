# 仕様書: しろのいななき

## 効果
攻撃技で他のポケモンをひんしにさせたとき、自分のこうげきが1段階上がる。

## 特性の仕様
- じしんかじょうと全く同じ効果を持つ。じしんかじょう#特性の仕様を参照とする。

## 備考
- ブリザポスの専用特性。レイスポスのくろのいななきとは対となる効果を持つ。
- バドレックス(はくばじょうのすがた)はきんちょうかんとしろのいななきの効果を併せ持つ特性じんばいったいを持つ。
