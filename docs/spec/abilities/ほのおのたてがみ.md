# 仕様書: ほのおのたてがみ

調査日: 2026-07-12
参照URL: https://wiki.pokemonwiki.com/wiki/ほのおのたてがみ

## 効果
ほのおタイプの攻撃技を使うとき、自分のこうげき・とくこうを1.5倍にしてダメージを計算する。

## 特性の仕様
- タイプ一致ボーナスとは補正箇所が異なる。この特性のポケモンがほのおタイプを持っていれば、特性の補正とタイプ一致ボーナスは両立する。
- タイプが変わる技、技のタイプを変える技により使用する技のタイプが変わった場合、変更後のタイプがほのおタイプであれば特性の効果がある。

## 備考
- メガカエンジシの専用特性。ポケモンチャンピオンズで新たに追加された特性。
- 同型の特性に、いわはこび（いわ）・はがねつかい（はがね）・りゅうのあぎと（ドラゴン）・トランジスタ（でんき、倍率1.3倍）・すいほう（みず、倍率2.0倍）がある。これらとは対象タイプ・倍率が異なるのみで、補正箇所・発動条件は共通。
