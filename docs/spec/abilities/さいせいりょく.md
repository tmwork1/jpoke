# 仕様書: さいせいりょく

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%95%E3%81%84%E3%81%9B%E3%81%84%E3%82%8A%E3%82%87%E3%81%8F

## 効果
他のポケモンに交代すると、自身の最大HPの1/3が回復する (小数点以下は切り捨て)。

## 特性の仕様
- 使うと交代する技/相手を交代させる技や、だっしゅつボタン/だっしゅつパック/レッドカードの効果で交代したときも発動する。
  - しっぽきりで交代する場合、技の効果でHPが減少した後にさいせいりょくの発動判定がある。途中できのみの発動圏内に入った場合、きのみの発動が挟まれる。
- さいせいりょくによる回復は、交代先のポケモンを選択するタイミングですでに完了している。
- 本来他の特性を持つポケモンが、特性を変える技/特性の効果で一時的にさいせいりょくを得ているときに交代した場合、さいせいりょくの効果は発動する。
- 本来さいせいりょくを持つポケモンが、この特性を失った状態で交代した場合、さいせいりょくは発動しない。とくせいなし状態のときに交代した場合でも同様に発動しない。
- かがくへんかガスが発動しているときは交代しても回復しない。かがくへんかガスのポケモンと同時に交代した場合、後に交代した方のポケモンの特性が優先される。
- かいふくふうじ状態でもこの特性の効果で回復することはできる。バトンタッチでかいふくふうじ状態を引き継いだときでも同様となる。
- かたやぶりの効果がある技を受けてだっしゅつボタンで交代したときや、かたやぶりの効果がある相手を交代させる技を受けた場合であってもさいせいりょくは発動する。
- しぜんかいふくとは異なり、第七世代以前でも戦闘が終了したときに発動することは無い。

## 備考
- 第六世代と第七世代では、いのちのたま/かいがらのすずを持つ相手から攻撃技を受けて特性所持者のだっしゅつボタンが発動したとき、いのちのたま/かいがらのすずによるHP変動と同時にさいせいりょくのポケモンのHPが回復していく様が映し出される。
  - 全体攻撃技を受けてさいせいりょくのだっしゅつボタンが発動し、その際の攻撃で他のポケモンがきのみによる回復を発動させたときも、同様の現象が起きる。
  - 第五世代と第八世代以降ではだっしゅつボタンの発動させたポケモンのHPバーは表示されず、回復する様子は映し出されない。
