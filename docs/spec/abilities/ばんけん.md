# 仕様書: ばんけん

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B0%E3%82%93%E3%81%91%E3%82%93

## 効果
- いかくを受けてもこうげきが下がらず、逆に1段階上がる。
- 交代させる技やレッドカードの効果による交代を防ぐ。

## 特性の仕様
- いかくを受けたとき、特性バーが表れて攻撃が上がる。
  - ビビリだまを持っているときは、攻撃が上がった後にビビリだまの効果が発動する。
  - クリアチャームを持っているときでも、ばんけんの発動が優先されて攻撃が上がる。
  - みがわり/しろいきり状態でいかくを防いだとき、ばんけんは発動しない。
  - すでに攻撃が最低まで下がっていていかくの効果が無かったとき、ばんけんは発動しない。
  - ばんけんのポケモンがくさタイプで味方にフラワーベールのポケモンがいる場合、補正抜きの素早さ実数値が高い方の特性が優先して発動する。フラワーベールでいかくの効果が防がれるとばんけんは発動しない。
- 相手を交代させる技を受けたとき、特性バーは現れずに交代が無効化される。
  - ほえる/ふきとばしを受けたときは「しかし うまく 決まらなかった!!」とメッセージが出る。
  - ともえなげ/ドラゴンテールはダメージだけ受ける。
  - かたやぶりの効果がある技の場合、ばんけんは無効化されて交代させられる。
- このポケモンの攻撃技に対してもレッドカードは消費され、「レッドカードを <攻撃者>に たたきつけた!」のメッセージも出るが、ばんけんの特性バーやメッセージは流れないまま交代は無効化される。
- 持っているだっしゅつボタン/だっしゅつパックが発動したときは交代する。
