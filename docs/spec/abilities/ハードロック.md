# 仕様書: ハードロック

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%8F%E3%83%BC%E3%83%89%E3%83%AD%E3%83%83%E3%82%AF

## 効果
相手から受ける効果バツグンの技のダメージが3/4倍になる。

## 特性の仕様
- かたやぶりの効果がある技に対しては効果が無い。
- もりののろい/ハロウィンで追加されたタイプや、タールショット状態で追加された弱点もハードロックの発動に影響する。
- らんきりゅうの効果で抜群の技が等倍になるとハードロックは発動しなくなる。
- フリーズドライ/フライングプレスによる相性自体が変わる効果は、ハードロックの発動に影響する。
- 弱点半減のきのみなど、相性自体は変わらない効果はハードロックの発動に影響しない。ハードロックのポケモンがきのみで受けるダメージを半減したとき、2倍弱点のダメージは3/4倍になる。

## 備考
- 同様の効果を持つ特性として、フィルターやプリズムアーマーがある。ただし、プリズムアーマーはかたやぶりに対しても有効という違いがある。
- 単純計算では2倍弱点のダメージは1.5倍、4倍弱点のダメージは3倍までに軽減する効果だが、ハードロックや上記類似特性は相性のダメージが変化する効果ではない。ダメージ計算において相性補正の位置とハードロックの補正位置は異なる。
- この特性のポケモンは全て4倍弱点を持つ。
