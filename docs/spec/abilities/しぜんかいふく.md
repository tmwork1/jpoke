# 仕様書: しぜんかいふく

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%97%E3%81%9C%E3%82%93%E3%81%8B%E3%81%84%E3%81%B5%E3%81%8F

## 効果
他のポケモンに交代すると状態異常が回復する。
第四世代から第七世代までは、戦闘が終了したときも状態異常が回復する。

## 特性の仕様
- 使うと交代する技/相手を交代させる技や、だっしゅつボタン/だっしゅつパック/レッドカードの効果で交代したときも発動する。
- 本来他の特性を持つポケモンが、特性を変える技/特性の効果で一時的にしぜんかいふくを得ているときに交代した場合、状態異常が回復する。
- 本来しぜんかいふくを持つポケモンが、スキルスワップなどによって別の特性に書き変えられた状態で交代した場合、状態異常は回復しない。
  - とくせいなし状態のときに交代した場合、第四世代では状態異常が回復したが、第五世代以降では状態異常は回復しない。
- かがくへんかガスが発動しているときは引っ込んでも発動しない。かがくへんかガスのポケモンと同時に交代した場合、後に交代した方のポケモンの特性が優先される。
- かたやぶりの効果がある技を受けてだっしゅつボタン/だっしゅつパックで交代したときや、かたやぶりの効果がある相手を交代させるわざを受けた場合であっても発動する。

### 戦闘終了時の効果について
- 世代や状況によっては、経験値を入手できる戦闘が終了したときにも特性の発動判定があり、状態異常を回復できる。
- 第三世代では、戦闘を終了したときは回復しない。
- 第四世代から第七世代では、場に出ている状態で戦闘を終了させたなら状態異常が回復する。
  - にげるや捕獲による戦闘終了でも発動する。
  - ただし、場に出てない状態で戦闘が終了した場合や、状態異常になったしぜんかいふくの野生ポケモンを捕獲して手持ちポケモンにした場合は発動せず、状態異常は回復しない。
  - 本来しぜんかいふくを持つポケモンが、特性を変更されたり、とくせいなし状態のまま戦闘が終了したときも発動する。逆に、本来しぜんかいふくを持たないポケモンが特性を得た状態で戦闘が終了したときは発動しない。
- 第八世代以降では、場に出ている状態で戦闘を終了したとしても状態異常は回復しない。
