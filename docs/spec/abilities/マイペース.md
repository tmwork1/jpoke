# 仕様書: マイペース

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%A4%E3%83%9A%E3%83%BC%E3%82%B9

## 効果
- こんらん状態にならない。こんらん状態になっても回復する。
- 第八世代から、特性いかくを無効化する。

## 特性の仕様
- 混乱状態のポケモンの特性がマイペースに書き換わった場合、直後に混乱を回復する。
- かたやぶりの効果がある技によってマイペースのポケモンが混乱状態になった場合、直後に混乱を回復する。
  - このとき同時にラムのみ/キーのみを持っていた場合、第四世代まではマイペースが優先して発動するためきのみは消費されない。第五世代以降はきのみが優先して発動するため消費される。
- かがくへんかガスで特性が無効化されて混乱状態になった場合、かがくへんかガスのポケモンが場から去ると同時に混乱が回復する。
- マイペースのポケモンがバトンタッチで混乱状態を引き継いだ場合、第四世代までは場に出た直後に混乱が治る。第五世代以降は、マイペースのポケモンが場に出てからポケモンやトレーナーが何かしらの行動を完了した後にマイペースが発動して混乱が治る。第五世代以降では以下のような仕様がある。
  - 場に出てから最初に行動するポケモンがマイペース自身である場合、技を使うか自傷した後にマイペースが発動する。
  - 場に出てから最初に行動するポケモンが他のポケモンである場合、その行動が完了した後にマイペースが発動する。状態異常などで動けなかったときや、技が無効化されたときでも同様にマイペースが発動する。
  - 他のポケモンがトレーナーに交代させられた場合や、相手を交代させる技の効果で交代した場合は、次のポケモンが繰り出された後にマイペースが発動する。
    - 他のポケモンが使うと交代する技の効果で交代したときは、ポケモンが引っ込んだ直後、そのトレーナーが次に繰り出すポケモンを選ぶ前にマイペースが発動する。
  - 自身を含めた場にいるポケモンがメガシンカ/ダイマックス/テラスタルを発動させたときも、それを引き金にしてマイペースが発動する。
  - きあいパンチの準備などは行動とはみなされず、マイペースは発動しない。
- 攻撃技の追加効果の効果による混乱を無効化した際は、マイペースが発動したことを明示するメッセージは流れない。
- 変化技の効果による混乱を無効化した際は、第四世代では「<マイペースのポケモン>は マイペースで こんらんしない!」というメッセージが、第五世代以降ではマイペースの特性バーと共に「<ポケモン名>は 混乱しない!」というメッセージが流れる。
  - いばる/おだてるを受けたときは、能力が上昇した後にマイペースの表示が出て混乱を防ぐ。
- あばれる状態や混乱きのみの効果で自分に発生する混乱も防ぐ。この際、特性が発動したことを明示するメッセージは流れない。
- 特性どくくぐつの効果による混乱を防いだ際は、どちらの特性バーも表示されない。受けた技が変化技と攻撃技のどちらであっても同様となる。
- 味方の場がしんぴのまもり状態である場合、第四世代ではマイペースによる混乱の無効化が優先される。第五世代以降ではしんぴのまもり状態による混乱の無効化が優先される。
- 自身がミストフィールド状態7-の効果を受けている場合、ミストフィールド状態による混乱の無効化が優先される。
- はなびらのまいなどでミイラ/さまようたましい/とれないにおいのポケモンを直接攻撃し、同時にあばれる状態も終了した場合、ミイラ/さまようたましい/とれないにおいへの変更が優先されて混乱状態になる。

## 備考
- 第四世代のみ、状態異常になっているポケモンがマイペースで混乱状態を治した場合、状態異常の表示が消える。表示が消えるだけで状態異常が治るわけではない。
