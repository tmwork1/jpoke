# 仕様書: リベロ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%AA%E3%83%99%E3%83%AD

## 効果
技を出す直前に、使用者のタイプがその技と同じタイプに変化する。交代して手持ちに戻ると元のタイプに戻る。
第九世代では、1回場に出るごとにつき1回しか発動しない。

## 特性の仕様
- へんげんじざい#特性の仕様を参照。

## 備考
- ヒバニー系統の専用特性。かつかくれとくせいのみの特性。
  - リベロとは、サッカー用語の一つであるが、この特性を持つエースバーンのモチーフはストライカーである。
- へんげんじざいと同じ効果を持つ。
