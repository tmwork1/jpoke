# 仕様書: カブトアーマー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%AB%E3%83%96%E3%83%88%E3%82%A2%E3%83%BC%E3%83%9E%E3%83%BC

## 効果
他のポケモンから受ける攻撃技が急所に当たらなくなる。

## 特性の仕様
- こおりのいぶき/やまあらし/あんこくきょうだ/すいりゅうれんだ/トリックフラワーを受けても急所に当たらない。
- 自身がどく/もうどく状態のとき特性ひとでなしの攻撃を受けても急所に当たらない。
- 第六世代以降で急所ランクが+3以上の攻撃を受けても急所に当たらない。
- かたやぶりの効果が発動した技に対してはこの特性は無視され、急所に当たることがある。

## 備考
- 同様の効果を持つとくせいとしてシェルアーマーがある。
