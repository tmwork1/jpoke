# 仕様書: あまのじゃく

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%82%E3%81%BE%E3%81%AE%E3%81%98%E3%82%83%E3%81%8F

## 効果
自分のランク補正が変動するとき、変動量のプラスマイナスが反転する。

## 特性の仕様
- この特性が発動したときも特にメッセージなどは流れず、能力変化を表すメッセージの文が変わるのみとなる。たとえばいかくを受けたときは「<ポケモン>の 攻撃が 上がった!」とだけ表示される。
- しろいきり状態・フラワーベールでランク低下が防がれるかは、あまのじゃくの効果を適用した後に判定する。本来ランクを上げる効果は防がれるようになり、本来ランクを下げる効果は反転してランクを上げることができる。
- 自身のアイテムによる能力変化や、戦闘用の道具による能力変化も反転する。
  - アイテムや戦闘用の道具で能力が下がる場合、自分で能力を下げたことになり、しろいきり・フラワーベールで防ぐことができない。他のポケモンの技でアイテムが発動したときも同様に防げない。
- かたやぶりの効果がある技を受けた場合、あまのじゃくは無効化され、通常の効果通り能力が上下する。
  - かたやぶりの効果がある技を受けて発動したわたげは、あまのじゃくの効果を無視してすばやさを下げる。
  - かたやぶりの効果がある技を受けた瞬間に発動したアイテムの効果は反転しない。同様に、特性を無視する技を受けた瞬間に発動したアイテムの効果も反転しない。
- かがくへんかガスが発動している間はあまのじゃくの効果は無い。
  - かがくへんかガスのポケモンを倒したときの追加効果や反動を受けるときや、かがくへんかガスのポケモンからおきみやげを受けたときでもあまのじゃくの効果は無い。
- バトンタッチで引き継いだ能力変化は反転しない。
- くろいきり・クリアスモッグ・じこあんじ・パワースワップ・ガードスワップ・ハートスワップ・ひっくりかえすによる能力変化は反転しない。
- シャドースチールを受けたときのランクリセット効果は反転しない。あまのじゃくのポケモンがシャドースチールを使用したときは吸い取った分だけ能力が下がる効果に反転する。
- Z変化ワザの付加効果によるランク変動は反転しない。
  - 例えばせいちょうをZワザに変えた場合、Zワザの効果はあまのじゃくの影響を受けずにとくこうが1段階上がるが、直後の技の効果はあまのじゃくの影響を受けてこうげきととくこうが1段階下がる。
  - ナインエボルブーストを使えば全能力が2段階下降する。
- ダイマックスわざの効果によるランク変動は反転する。
- マックスレイドバトルでふしぎなバリアが破壊されたときの能力変化にも適用され、ぼうぎょ・とくぼうが2段階ずつ上昇する。
- ランクが限界だったことによる技の失敗条件や道具の発動条件が、他の特性のときと反転するものと、変わらないものがある。
  - めいそう・ニトロチャージなど、自分の能力を変化させる技は、効果対象の能力が全て+6のときも発動してランクが下がる。
  - いばる・おだてる・アロマミスト・たがやす・フラワーガード・デコレーション・コーチングは、効果対象の能力が全て+6のときも発動してランクが下がる。
  - ほおばるは防御ランクが+6のときも成功してきのみを消費する。-6のときに使うと失敗してきのみを消費しない。
  - ソウルビートは効果対象の能力が全て+6のときも発動してランクが下がり、HPも減少する。全て-6のときは失敗してHPが減らない。
  - はいすいのじんは効果対象の能力が全て+6のときも発動してランクが下がり、にげられない状態になる。全て-6のときは失敗してにげられない状態にならない。
  - ピンチ系きのみや、じゃくてんほけん・シード系アイテム・からぶりほけんは対応する能力がすでに+6であるときは発動して能力が下がり、-6であるときは発動しない。
  - ルームサービスは素早さランクが-6であるときは発動し、+6であるときは発動しない。
  - ビビリだまは素早さランクが+6であるときは発動し、-6であるときは発動しない。すでに攻撃ランクが+6でいかくの効果が無かったときは発動しない。
  - はらだいこを使う場合、攻撃のランク補正が-6まで下がる。攻撃ランクが+6のときは失敗し、-6のときは成功してHPが減少するだけとなる。このときも「体力を削って パワー全開！」のメッセージが表示される。
  - ちからをすいとるを受けた場合、攻撃が上昇した後に使用者のHPが回復する。攻撃ランクが+6のときに受けたときは成功して回復されるが、-6のときは失敗する。
  - アッキのみ・タラプのみは対応する能力がすでに+6であるときも、-6であるときのどちらでも発動しない。
    - 能力が+6のときについばむ・むしくいで食べた場合や、なげつけるを受けた場合でも発動しない。おちゃかいで食べた場合も能力が+6から下がらないが、きのみは消費される。
    - ほおばるで特防+6のときにタラプのみを食べても特防は下がらないが、きのみは消費される。このときもゲップは使用可能になる。防御+6のときにアッキのみを食べると、ほおばるの効果で防御が2段階下がった後、アッキのみの効果で1段階下がる。
  - じゅうでんち・きゅうこん・ひかりごけ・ゆきだま・のどスプレーは対応する能力がすでに+6であるときも、-6であるときのどちらでも発動しない。
  - プラスパワー・おうえんポン・ダイキノコなど、ランクを上げる戦闘用の道具を使った場合、対応する能力全てが+6であるときは効果が無く、使おうとした道具は戻ってくる。-6のときは「<ポケモン>の <能力>は もう 下がらない!」と表示されるものの道具は消費される。
- しろいハーブ・だっしゅつパックを持っているときに戦闘用の道具で能力を下げた場合、使用した道具の効果が終わった直後にアイテムの消費判定がある。ダブルバトルで他の味方が道具を使う場合でも、アイテムの発動が優先される。
  - 味方の行動で使われた道具により、あまのじゃくのポケモンがだっしゅつパックで交代した場合、そのターンあまのじゃくのポケモンが選んでいた行動はキャンセルされる。道具を使おうとしていたときも同様にキャンセルされる。
- Oパワーの効果で能力が下がった場合、その瞬間にしろいハーブは発動しない。何かしら技を使用すれば、それが引き金となりしろいハーブで能力をリセットできる。

## 備考
- 第五世代では隠れ特性限定であり、通常特性として持つのはカラマネロ系統のみ。
