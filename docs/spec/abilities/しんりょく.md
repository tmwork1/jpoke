# 仕様書: しんりょく

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%97%E3%82%93%E3%82%8A%E3%82%87%E3%81%8F

## 効果
- 第四世代まで: 自分のHPが1/3以下になると、くさタイプの攻撃技の威力が1.5倍になる。
- 第五世代以降: 自分のHPが1/3以下になると、くさタイプの技で攻撃するときのこうげき・とくこうが1.5倍になる。

## 特性の仕様
- タイプが変わる技を使用した場合、変更後のタイプがくさタイプであればしんりょくの効果がある。
- そうでん状態でくさタイプの技がでんきタイプに変えられた場合、しんりょくの効果は無くなる。
- みがわり状態のときは本体のHP残量により発動するかが決まる。みがわりのHPは関係しない。
- ダメージ計算では、しんりょくが発動するか判定がされた後に、技の効果や相手の特性の効果などが発動する。
  - HP1/3以下のときにギガドレインなどのHPを吸収する技を使用した場合、回復してHPが1/3を超えるときでも、その際のダメージにしんりょくの効果はある。
  - HP1/3以上のときにくさ技を使用したが相手のさめはだのダメージなどを受けた場合、ダメージを受けてHPが1/3以下になったときでも、その際のダメージにしんりょくの効果は無い。
- HPが1/3以上のときにタネマシンガンなどの連続攻撃技を使用したが、途中でジャポのみ/うのミサイルのダメージを受けてHPが1/3以下になった場合、即座にしんりょくの効果が発動する。
- ダイマックスしているときでも、ダイマックス後のHPが1/3以下になったときに発動する。

## 備考
- 御三家のくさタイプは全てこの特性を持つ。
- 御三家以外でこの特性を所有するポケモンはヤナップ系統しかいない。
- 分類が「しんりょくポケモン」のリーフィアはこの特性を持たない。
