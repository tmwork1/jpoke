# 仕様書: ゆきかき

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%86%E3%81%8D%E3%81%8B%E3%81%8D

## 効果
天気があられ/ゆき状態のとき、すばやさが2倍になる。

## 特性の仕様
- ゆきかきによるすばやさ補正はランク補正の後に計算する。まひの効果はゆきかきやその他の補正の効果を考慮した後に計算する(すばやさ#すばやさ補正も参照)。
- この特性を受け取ったり失ったりした場合、第七世代まではそのターンの行動順には影響ない。第八世代からはそのターンから行動順に影響が出る。
- 天気があられ/ゆき状態でも特性ノーてんき/エアロックのポケモンが場にいる場合、ゆきかきは発動しない。

## 備考
- すなかきのあられ/ゆき版といえる効果がある。
- 第九世代現在では、特性ゆきかきを所有するポケモンは全てこおりタイプを持つ。
- 第八世代以前ではゆきがくれ/すながくれ/すなかきなどと異なり、タイプによらず天候ダメージを受けなくなるような効果は無い。特性ゆきかきを持つポケモンのタイプがこおりタイプ以外に変更された場合や、こおりタイプを持たないポケモンがゆきかきを手に入れた場合、あられのダメージを受ける。
