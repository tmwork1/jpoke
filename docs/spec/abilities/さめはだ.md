# 仕様書: さめはだ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%95%E3%82%81%E3%81%AF%E3%81%A0

## 効果
直接攻撃でダメージを受けたとき、攻撃してきた相手に対して相手の最大HPの1/8分のダメージを与える (小数点以下切り捨て。ただし最低でも1以上)。
- 第三世代のみ、最大HPの1/16のダメージを与える。

## 特性の仕様
- この特性が発動し、攻撃側がダメージを受ける際のメッセージは以下のようになる。
  - 第四世代までは「<防御側名>の さめはだで <攻撃側名>は きずついた!」というメッセージが流れる。
  - 第五世代からスカーレット・バイオレットではさめはだの特性バーが現れ「<攻撃側名>は きずついた!/傷ついた!」というメッセージが出る。
  - Championsでは特性バーが出てダメージが入るのみで、メッセージは流れない。
- 特性マジックガードのポケモンはさめはだのダメージを受けない。第五世代以降ではこのとき、さめはだの特性表示はされるが、マジックガードに関するメッセージは流れない。
- アイテムぼうごパットを持つポケモンはさめはだのダメージを受けない。このとき、さめはだの特性表示の直後に「<所持者>は ぼうごパットで 防いだ！」とメッセージが流れる。
- 特性えんかくやアイテムパンチグローブの効果で直接攻撃ではなくなった技に対しては発動しない。
- 直接攻撃を受けても、相性/まもる状態などで技を無効化したときは発動しない。
- みがわり状態で技を防いだときは発動しない。
- まもる/みがわり状態のときでも、それらを無視する直接攻撃で本体がダメージを受けたときは発動する。
- みがわり状態のポケモンから直接攻撃を受けた場合、本体のHPにダメージを与える。
- さめはだのポケモンがひんしになったときも発動する。
  - このときのダメージで相手のポケモンもひんしになり、お互いのポケモンが全滅した場合、第六世代まではさめはだ側の勝ちとなる。第七世代以降ではさめはだ側の負けとなる。
- 連続攻撃技に対しては、1発ダメージを受けるたびに発動する。
  - 途中で使用者がひんしになると攻撃は中断する。相手の特性がスキルリンク/おやこあいだったとしても同様に中断する。
- アイテムがゴツゴツメットである場合、さめはだの後にゴツゴツメットによるダメージが発生し、合計で最大HPの7/24のダメージを与える。途中で相手がきのみの発動圏に入ったときは回復を挟むことができる。さめはだのダメージでひんしになったときはゴツゴツメットのメッセージは出ない。
- アイテムであるジャポのみ/レンブのみが発動する場合、さめはだの後にきのみのダメージが発生し、合計で最大HPの1/4のダメージを与える。途中で相手がきのみの発動圏に入ったときは回復を挟むことができる。さめはだのダメージでひんしになったときはジャポのみ/レンブのみは消費されない。
- HPを吸収する効果がある直接攻撃を受けた場合、相手は技の効果で回復した後にさめはだのダメージを受ける。
- とんぼがえり/クイックターンを受けたが、使用者がさめはだのダメージでひんしになったときは、技の効果は発動しない。代わりのポケモンはターンの終わりに繰り出すことになる。Championsでも同様。
- 以下の技を受けた場合、第四世代までは技の効果が発動した後にさめはだのダメージが発生する。第五世代以降ではさめはだのダメージの後に技の効果が発動する。さめはだのダメージで技の使用者がひんしになった場合、第四世代、第五世代からスカーレット・バイオレット、Championsのそれぞれで技の発動判定が異なる。

## 備考
- 同じ効果を持つ特性として、てつのトゲがある。
- 似た効果を持つ特性としてうのミサイルがある。
- 第四世代まではキバニア系統の専用特性だった。
- 特性いしあたまのポケモンもさめはだのダメージを受ける。
- さめはだのダメージでフリーフォールを使ったポケモンをひんしにした場合、さめはだのポケモンのグラフィックが消えることがある。フリーフォール#さめはだバグ(レッドカードバグ)を参照とする。
