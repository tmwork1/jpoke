# 仕様書: フェアリーオーラ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%95%E3%82%A7%E3%82%A2%E3%83%AA%E3%83%BC%E3%82%AA%E3%83%BC%E3%83%A9

## 効果
場にいるポケモン全員が使用するフェアリータイプの技の威力が1.33倍になる。

## 特性の仕様
- 正確な補正率は5448/4096倍となる。
- 特性オーラブレイクを持つポケモンが場に出ていると、威力が3/4倍になる効果に変化する。
- 場に複数フェアリーオーラを持つポケモンがいても、効果は累積せず、1.33倍のまま変わらない。
- 効果対象は場にいるポケモン全員なので、自身や味方の他に、敵が使用するフェアリー技の威力も上げる。
- だいちのはどうなどのタイプが変わる技や、フェアリースキンなどのタイプを変える特性の効果を受けた技は、変動後のタイプがフェアリータイプであれば特性の効果を受ける。
- かたやぶりの効果がある技は、第七世代まではフェアリーオーラの影響を受けない。第八世代ではフェアリーオーラの影響を受ける。
- この特性のポケモンが場に出ると、特性バーと同時に「<ポケモン>は フェアリーオーラを 放っている!」というメッセージが出る。他の多くの場に出たときに発動する特性とは補正込みのすばやさ順にメッセージが出る (ターン#1.ポケモンを繰り出すも参照)。

## 備考
- 第八世代まではゼルネアスの専用特性だった。
- 類似効果を持つ特性にダークオーラがある。
