# 仕様書: あめうけざら

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%82%E3%82%81%E3%81%86%E3%81%91%E3%81%96%E3%82%89

## 効果
天気があめ・おおあめ状態のとき、ターン終了時に最大HPの1/16を回復する (小数点以下切り捨て)。

## 特性の仕様
- あめが終了したターンでは回復しない。
- 状態異常や状態変化のダメージやたべのこし/くろいヘドロ等の効果は、あめうけざらによる回復の後に発動する。
- 第五世代以降では、かいふくふうじ状態では回復できない。
  - かいふくふうじ状態が終了するターンでも回復できない。
- あなをほる/ダイビング状態により隠れている間は回復しない。
  - そらをとぶ/シャドーダイブ状態で隠れているならば回復する。
- 特性ノーてんき/エアロックのポケモンが場にいる間は発動しない。
- ばんのうがさを持っているときは発動しない。

## 備考
- 第四世代まではハスボー系統の専用特性だった。それ以外のポケモンは隠れ特性限定である。
