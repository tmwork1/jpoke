# 仕様書: てんきや

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A6%E3%82%93%E3%81%8D%E3%82%84

## 効果
天気に対応した姿にフォルムチェンジし、タイプが変わる。
| フォルム名 | タイプ | 天気 |
| --- | --- | --- |
| ポワルンのすがた | ノーマル | 通常下記以外の天気 |
| たいようのすがた | ほのお | にほんばれおおひでり |
| あまみずのすがた | みず | あめおおあめ |
| ゆきぐものすがた | こおり | あられゆき |

## 特性の仕様
- すなあらし/くらやみ/きり/らんきりゅう状態に対応したフォルムは無い。
- フォルムチェンジ特性であり、特性が発動して姿が変わるたびに以下の効果がある。
  - みずびたし/もりののろい/ハロウィン/ミラータイプなどによりタイプが変化していた場合、これらの効果を無くす形で天気に対応した姿/タイプに変わる。
  - パワーシェア/ガードシェア/パワートリック/スピードスワップによりステータスが変化していた場合、元のステータスに戻る。
- 他の多くのフォルムチェンジ特性とは異なり、この特性を上書きすることや交換することはできる。技なかまづくり/スキルスワップ/なやみのタネ/シンプルビーム・特性ミイラの効果でてんきやは失われる。いえき/コアパニッシャーを受けるととくせいなし状態となる。
  - フォルムチェンジ後にこの特性を失った場合の挙動は世代により異なる。コピーが可能かは世代により異なる (下記参照)。
  - この特性のポケモンがなりきりを使用したときは成功し、てんきやは失われる。
  - かがくへんかガスが発動している間は、ポワルンはフォルムチェンジできなくなる。天候が変動中にかがくへんかガスの効果が切れたときは、即座にフォルムチェンジを発動できる。
- 天候が変わった瞬間に特性が発動する。回数制限はないため、特性や技で1ターンの間に天候を何度も変えた場合、1ターンの間に何度も特性が発動することになる。
- ステルスロックのダメージはてんきやの発動前に受ける。たいよう/ゆきぐものすがたにフォルムチェンジするときも、最大HPの1/8のダメージで済む。
- 場に出たとき発動する特性の効果で天気を発動させるポケモンが同時に複数場に出てきた場合、最終的な天気が決まるまでてんきやは発動しない。
- 場にいる複数のポケモンのてんきや/フラワーギフトが同時に発動する場合、補正込みのすばやさが高い順に特性が発動する。
- フォルムチェンジ中、場にノーてんき/エアロックのポケモンが出てきて天候が無効化された場合、即座にポワルンは通常の姿に戻る。ノーてんき/エアロックの効果が続く限り、天気が変わってもフォルムチェンジできない。
- かたやぶり効果を持つ技でも、この特性を無視することはできない。

### 第五世代以降
- へんしん/かわりもので他のポケモンがポワルンにへんしんした場合、天気が変わってもコピーした姿のままでフォルムチェンジできない。
- ポワルンがてんきやを失った場合、ポワルンは通常の姿に戻り、フォルムチェンジできなくなる。
  - 再び特性がてんきやに戻った場合は、またフォルムチェンジ可能になる。
- この特性はコピーできない。
  - この特性に対するなりきりは失敗し、トレースも発動しない。
  - この特性が使用するなかまづくりは失敗する。この特性がなかまづくりを受けたときはてんきやは失われる。
  - この特性に対してかがくのちから/レシーバーは発動しない。
- 第六世代以降では、フィールドに自然発生した永続天候を技や特性で一時的に書き換えていた場合、一時的な天候のターンが切れた時点で一旦天候なし状態となり、その後改めて永続天候が再発生する。このため、天候が切れた時点で一旦てんきやが発動して通常フォルムに戻り、永続天候が再発生した時点で再度てんきやが発動してフォルムが変わる。
  - 例えばたいようのすがたのとき、おわりのだいちの特性を持つポケモンが他のポケモンとスキルスワップで特性を入れ替えた場合、特性が入れ替わり一瞬天候が戻った際にポワルンは通常フォルムに戻る。もう片方のポケモンが特性を再発動することで天気がおおひでりに戻り、ポワルンのフォルムもまたたいようのすがたに戻る。
