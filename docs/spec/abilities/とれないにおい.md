# 仕様書: とれないにおい

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A8%E3%82%8C%E3%81%AA%E3%81%84%E3%81%AB%E3%81%8A%E3%81%84

## 効果
直接攻撃でダメージを受けたとき、相手の特性をとれないにおいにする。
上書きできない特性に対しては発動しない。

## 特性の仕様
- ミイラと同じ効果を持つ。ミイラ#特性の仕様を参照とする。
  - 発動時のメッセージは異なり、とれないにおいの場合は「<相手ポケモン名>は においが うつって とれなくなっちゃった!」と流れる。

## 備考
- ♂のパフュートンの専用特性。♀のパフュートンと進化前のグルトンは代わりにアロマベールを持つ。
