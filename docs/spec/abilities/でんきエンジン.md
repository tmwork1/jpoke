# 仕様書: でんきエンジン

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A7%E3%82%93%E3%81%8D%E3%82%A8%E3%83%B3%E3%82%B8%E3%83%B3

## 効果
でんきタイプの技を受けても無効化し、自分のすばやさを1段階上げる。

## 特性の仕様
- 発動条件自体はちくでんと共通しているため、その詳細はちくでん#特性の仕様を参照とする。ちくでんは発動するとHPが回復するのに対し、でんきエンジンが発動したときは自身のすばやさが1段階上がる。
  - でんきタイプなら変化技を受けたときも発動して無効化する。
  - 使用者自身を対象にする技や、場を対象とする技に対しては発動しない。
  - みがわり状態ででんき技を受けたときでも発動し、すばやさが上がる。
  - まもる状態などででんき技を防いだときは発動しない。
  - でんきタイプの連続攻撃技も無効化するが、すばやさは1段階だけ上昇する。
  - かたやぶりの効果がある技に対してでんきエンジンは発動しない。
- でんき技を無効化した際は特性の発動を示すメッセージが流れる。
  - 第四世代では「<ポケモン名>は でんきエンジンで すばやさが あがった!」、すばやさが最大なら「<ポケモン名>の でんきエンジンで <技名>は こうかが なかった!」とメッセージが出る。
  - 第五世代以降ではでんきエンジンの特性バーが出現し「<ポケモン>の 素早さが 上がった!」、すばやさが最大なら「<ポケモン>には 効果が ないようだ...」というメッセージが出る。

## 備考
- 第四世代ではエレキブルの専用特性だった。
- 同じくでんき技を無効化する特性としてちくでん/ひらいしんがある。
