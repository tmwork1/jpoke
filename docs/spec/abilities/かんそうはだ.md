# 仕様書: かんそうはだ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%93%E3%81%9D%E3%81%86%E3%81%AF%E3%81%A0

## 効果
- みずタイプのわざを受けても無効化し、代わりに最大HPの1/4を回復する (小数点以下切り捨て)。
- ほのおタイプの攻撃技を受けたときの威力が5/4倍になる。
- 天気があめ・おおあめ状態のとき、毎ターン終了時に最大HPの1/8を回復する (小数点以下切り捨て)。
- 天気がにほんばれ・おおひでり状態のとき、毎ターン終了時に最大HPの1/8のダメージを受ける (小数点以下切り捨て)。

## 特性の仕様
### 技を受けたときの効果
- 以下のように、みず技を無効化する効果はちょすいと共通している。詳細な仕様はちょすい#特性の仕様を参照とする。
  - みずタイプなら変化技を受けたときも発動して無効化する。
  - 使用者自身を対象にする技や、場を対象とするみず技を無効化して回復することはない。
  - まもる状態などでみず技を防いだときや、おおひでりでみず技が不発したときは回復できない。
- 技や特性の効果でタイプが変わった技を受けた場合、変更後のタイプがみずタイプであれば無効化し、ほのおタイプであれば威力が上がる。
- みがわり状態でみず技を受けても発動しHPを回復する。受けるほのお技の威力は上がる。
- ノーてんき/エアロック/ばんのうがさにより天気による効果が無効化されているときでも、みず技やほのお技を受けたときの効果は発動する。

### 天気に関する効果
- あめ状態が終了するターンでは回復しない。にほんばれ状態が終了するターンにダメージは受けない。
- 第五世代以降では、かいふくふうじ状態のときは回復できない。
  - かいふくふうじ状態が終了するターンでもあめ状態での回復ができない。
- 自身が姿を隠しているときは、以下のように扱う。
  - あなをほる状態/ダイビング状態で隠れている間は、第五世代以降は天気による回復とダメージのどちらも発生しない。
    - 第四世代のみ、にほんばれ状態によるダメージは受けない一方で、あめ状態によるHPの回復はできる。
  - そらをとぶ状態/シャドーダイブ状態で隠れている間は、天気による回復・ダメージのどちらも発生する。第四世代でも同様となる。
- 特性ノーてんき/エアロックのポケモンが場にいるときは天気による回復・ダメージが発生しない。
- ばんのうがさを持っているときは天気による回復・ダメージが発生しない。
- みがわり状態でも天気による回復・ダメージが発生する。

### その他
- かたやぶり効果があるみず技・ほのお技を受けてもかんそうはだは発動しない。
  - ただし、かたやぶりのポケモンがあめ・にほんばれ状態を発動させたとしても、場の状態にかたやぶりの効果は無いので、天気による回復・ダメージどちらも発生する。
- 第四世代まではHPが満タンの状態でみず技を受けると「<ポケモン名>の かんそうはだで <技名>は こうかが なかった!」、 HPが減っているときにみず技を受けると「<ポケモン名>は かんそうはだで かいふくした!」、HPが減っているときにあめで回復すると「<ポケモン名>は かんそうはだで かいふくした!」、にほんばれでダメージを受けると「<ポケモン名>は かんそうはだで ダメージを うけた!」とメッセージが出る。第五世代以降でHPが満タンの状態でみず技を受けるとかんそうはだの特性バーと「<ポケモン>には 効果が ないようだ...」、HPが減っているときにみず技を受けるとかんそうはだの特性バーと「<ポケモン>の 体力が 回復した!」というメッセージが出て、HPが減っているときにあめで回復するときやにほんばれでダメージをうけるときは特性バーが現れるだけメッセージは出ない。いずれの世代でもほのお技を受けたときと、HPが満タンであめ状態のときにかんそうはだに関するメッセージは現れない。

## 備考
- 同じくみず技を無効化する特性としてちょすい、第五世代以降のよびみずがある。
- 同じく天気でHPを回復する特性としてあめうけざら、アイスボディがある。天気でダメージを受ける特性としてサンパワーがある。
