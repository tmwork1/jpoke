# 仕様書: ドラゴンスキン

調査日: 2026-06-03

## 効果
使用するノーマルタイプの技がドラゴンタイプに変わり、威力が1.2倍になる。

## 特性の仕様
- 変更後の技のタイプを除けばフェアリースキンと同様の効果を持つ。細かな仕様はフェアリースキン#特性の仕様を参照とする。
