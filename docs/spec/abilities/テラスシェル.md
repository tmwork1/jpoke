# 仕様書: テラスシェル

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%86%E3%83%A9%E3%82%B9%E3%82%B7%E3%82%A7%E3%83%AB

## 効果
自分のHPが満タンのとき、等倍以上でダメージを受ける攻撃技の相性が全て今ひとつ (1/2) になる。

## 特性の仕様
- 発動すると特性バーが出現し、「<所有者名>は 甲羅を かがやかせ タイプ相性を 歪める!!」とメッセージが出る。
- この特性はコピーできない。この特性に対するスキルスワップは失敗する。
  - この特性のポケモンに変身したポケモンの場合は、特性の効果を得ることができる。
- この特性を他の特性に上書きすることはできる。この特性はかがくへんかガスによって無効化される。
- かたやぶりの効果がある技や、特性を無視する技に対してこの特性は発動しない。
- みがわり状態である場合は、みがわりではなく本体のHPが満タンのときに特性が発動する。
- 一度HPが減っても、回復してHP満タンになれば特性は何度でも発動する。
- 相性で無効にできる技を受けたときは発動せず、そのまま無効にする。
  - みずびたし/まほうのこな/もりののろいの効果でタイプが変わった結果、相性で半減以下にできるようになった攻撃技を受けたときも特性は発動せず、そのままの相性でダメージを受ける。
- 特性の効果が発動すると、相性判定そのものが変化する。
  - 本来抜群となる攻撃を受けた場合でも、特性の効果が優先されて相性が書き変わるため、攻撃側のたつじんのおびや、防御側のじゃくてんほけん/ナゾのみ/弱点半減のきのみの効果、技アクセルブレイク/イナズマドライブの効果は発動しない。
  - 攻撃側の特性いろめがねの効果は発動し、受けるダメージが2倍になる。結果的に、テラスシェルの効果と相殺してほぼ等倍のダメージになる。
  - ねらいのまとにより等倍となった技に対しても発動し、今ひとつの相性にする、
  - ホズのみなど、相性に影響しないアイテムの効果は特性と同時に発動し得る。
- 連続攻撃技に対しては、全ての攻撃を今ひとつで受けることができる。
- みずびたしでみずタイプになったときに受けるフリーズドライも今ひとつで受けることができる。
- タールショット状態により弱点になったほのおタイプの技も今ひとつで受けることができる。
- ステラタイプの技のダメージも半減することができる。
- タイプなしのめざめるダンスを受けるときも発動し、ダメージを半減する。
- ダメージ固定技を受けたときは、特性バーとメッセージが現れるものの、実際にはダメージは半減されない。
  - 一撃必殺技が当たったときは、ダメージを受けた後に特性と「一撃必殺!!」のメッセージが現れる。それ以外のダメージ固定技の場合は、特性バーが現れた後にダメージを受ける。
- わるあがきを受けたときは、基本的に特性バーとメッセージが現れず、ダメージは半減されない。ただし、特性バーとメッセージが現れる（ダメージは半減されない）こともある（ダブルバトルで観測）。
- こんらんの自滅ダメージを受けるときは発動しない。
- ステルスロック状態により受けるダメージは減少しない。

## 備考
- テラパゴス (テラスタルフォルム) の専用特性。
- 似た特性として、HP満タンのときに受けるダメージを半減するマルチスケイル/ファントムガードがある。これらはダメージを1/2にする効果であるのに対し、テラスシェルは相性補正が直接1/2に書き変わるという違いがある。
  - 第五世代以降の連続攻撃技は、1発目が当たったときしか相性判定を行わない仕様があるため、マルチスケイル/ファントムガードと異なり、2発目以降のダメージにもテラスシェルは発動する。
