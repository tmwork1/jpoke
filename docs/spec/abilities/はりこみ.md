# 仕様書: はりこみ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AF%E3%82%8A%E3%81%93%E3%81%BF

## 効果
ポケモンチェンジで出てきたポケモンへ攻撃するときのこうげき・とくこうが2倍になる。

## 特性の仕様
- 使うと交代するわざで交代して出てきた相手に攻撃したときも発動する。
- アイテムレッドカード・だっしゅつボタン・だっしゅつパック、特性にげごし・ききかいひの効果で交代して出てきた相手に攻撃したときも発動する。
- 初手で繰り出された相手や、死に出しで繰り出された相手に攻撃しても発動しない。
- かわりものが発動したポケモンに対しては発動しない。

## 備考
- 第七世代ではヤングース系統の専用特性だった。
