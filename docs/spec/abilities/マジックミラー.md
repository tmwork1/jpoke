# 仕様書: マジックミラー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%82%B8%E3%83%83%E3%82%AF%E3%83%9F%E3%83%A9%E3%83%BC

## 効果
戦闘中、常にマジックコート状態になる。

## 特性の仕様
詳細についてはマジックコート (状態変化)#詳細な仕様を参照。
- 跳ね返せる変化技、跳ね返せない変化技についてはマジックコート (状態変化)#対象技を参照とする。
- かたやぶり系の効果がある変化技は跳ね返せない。
- 設置技の変化技を受けたときは跳ね返せるが、すでに設置されているこれらの場の状態の効果を防いだり跳ね返したりすることはできない。
  - がんせきアックス/ひけん・ちえなみなど攻撃技による追加効果や、どくげしょうなど特性による効果は跳ね返せない。
- 相手のマジックミラー、およびマジックコートで跳ね返ってきた効果はマジックミラーで再び跳ね返すことはできない。
- ミラーアーマーのポケモンからランクを下げる変化技を受けた場合、マジックミラーで跳ね返した効果はミラーアーマーで跳ね返される。マジックミラーのポケモンがミラーアーマーのポケモンのランクを下げようとした場合、ミラーアーマーで跳ね返された効果はマジックミラーで跳ね返されない。
- 自分がダイマックスしている場合、アンコール/いちゃもん/かなしばり/ふきとばし/ほえる/なかまづくりは無効化される前にマジックミラーで跳ね返す。

## 備考
- 第九世代時点では隠れ特性・メガシンカ限定の特性となる。
  - メガシンカポケモンを除く所有者は全てエスパータイプを持つ。
- スカーレット・バイオレットではマジックコートが使用できない技であるため、マジックコート状態になれる唯一の手段となる。
