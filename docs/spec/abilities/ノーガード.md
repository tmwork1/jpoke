# 仕様書: ノーガード

調査日: 2026-06-03

## 効果
### バトルでの効果
ノーガードのポケモンが使用する技と、ノーガードのポケモンが受ける技は必ず命中するようになる。

### フィールドでの効果
手持ちの先頭にこのとくせいを持つポケモンがいると、野生ポケモンの出現率が2倍になる (第八世代まで)。ひんし状態でも効果がある。

## 特性の仕様
### バトルでの効果
- 相性/特性/まもる状態などの効果で技が無効化されることはある。
  - すながくれ/はりきり/ミラクルスキンなどの特性による、命中率が下がる効果はノーガードで無視できる。
  - 特性しれいとうによりヘイラッシャに飲み込まれているシャリタツへの攻撃は、ノーガードの技であっても当たらない。
- あなをほる/そらをとぶ/ダイビング/シャドーダイブ状態により姿を隠している状態に対する攻撃も必中になる。
- 特性かたやぶりの効果がある技であっても、ノーガードの効果は無視されず必中になる。
- 一撃必殺技にも効果が適用される。
  - こおりタイプ以外のポケモンが使うぜったいれいども必中になる。
- 全体攻撃技を受けたときは、ノーガードのポケモンが受ける分のみ特性の効果がある。

### フィールドでの効果
- フィールドで同様の効果をもつ特性としてありじごく/はっこうがある。はっこう#特性の仕様を参照とする。

## 備考
- 第四世代ではワンリキー系統の専用特性だった。
- 以下の方法により、ノーガードのポケモン自身が一撃必殺技を使うことができる。
  - 一撃必殺技を持つポケモンの特性を特性を変える技でノーガードに変える。
  - カイリキーの場合、第三世代のおしえわざによるものまねで一撃必殺技を覚えることができる。
    - 第一世代ではわざマシンでじわれを覚えられるが、VC版から送ったカイリキーは隠れ特性（ふくつのこころ）で固定される。第八世代ではとくせいパッチによって隠れ特性を通常特性に変えることができず、第九世代ではとくせいパッチによって隠れ特性を通常特性に変えることができるように変更されたが、カイリキーはスカーレット・バイオレットに送ることができないため、スカーレット・バイオレット時点では実現できない。
      - ブリリアントダイヤモンド・シャイニングパール以降は覚えている技がソフトごとに保存されるものの、変更した特性は世代・ソフト間で共有される。そのため、仮に今後隠れ特性を通常特性に変えることができるソフトにカイリキーを送ることができるようになった場合、VC産のじわれを習得した個体をとくせいパッチととくせいカプセルで特性をノーガードに変更し、ソード・シールドに送ることでノーガードとじわれの両立が可能になる。
  - メガピジョットの場合、相手の使った一撃必殺技をまもるやみがわりなどで防ぐことができれば、オウムがえしで返すことができる。
- ダクマにはスカーレット・バイオレット (Ver.1.0.0 - Ver.1.1.0) までデータ上の隠れ特性としてノーガードが設定されていたが、Ver.1.2.0からは通常特性と同じせいしんりょくに戻された。
