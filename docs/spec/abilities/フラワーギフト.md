# 仕様書: フラワーギフト

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%95%E3%83%A9%E3%83%AF%E3%83%BC%E3%82%AE%E3%83%95%E3%83%88

## 効果
- 天気がにほんばれ・おおひでり状態のとき、自分と味方のこうげきととくぼうが1.5倍になる。
- 天気がにほんばれ・おおひでり状態のとき、チェリムがネガフォルムからポジフォルムへとフォルムチェンジする (第五世代以降)。

## 特性の仕様
- にほんばれ状態でも、場にノーてんき/エアロックのポケモンがいる場合はフラワーギフトは発動しない。ポジフォルムのチェリムがいる場にこれらの特性のポケモンが出てきた場合は、ネガフォルムに戻される。
- にほんばれ状態でも、この特性のポケモンがばんのうがさを持っている場合はフラワーギフトは発動しない。ポジフォルムのチェリムがばんのうがさを持つとネガフォルムに戻る。ポジフォルムのチェリムが味方にいるポケモンがばんのうがさを持っている場合は、フラワーギフトの効果を受けることができる。
- 場がワンダールーム状態のときは、攻撃・防御が1.5倍になる。
- 特性かたやぶりの効果がある技により特防上昇の効果は無効化される。
- この特性のポケモンがなかまづくりを使うと失敗する。
- 他の多くのフォルムチェンジ特性とは異なり、この特性を上書きすることや交換することはできる。

### 第五世代以降
- この特性を失った場合や、とくせいなし状態になった場合、かがくへんかガスが発動している場合は、チェリムはにほんばれ状態でもポジフォルムになれず、ポジフォルムであっても強制的にネガフォルムに戻されフォルムチェンジできなくなる。
- この特性はコピーできない。
  - スキルスワップ/さまようたましいで交換することは依然できる。交換かへんしん/かわりものによってチェリム以外のポケモンがこの特性を得ても攻撃・特防の上昇効果は発動しない。
- チェリムにへんしんしたポケモンは変身時点のフォルムを維持し、天候や特性が変わってもフォルムチェンジしない。
- 味方のかたやぶりの効果がある技は、攻撃上昇の効果を受けない。
- 特性を無視する技は、フラワーギフトによる能力上昇効果を受けない。
- 味方の場にポジフォルムのチェリムが複数いる場合、その分フラワーギフトによる能力上昇効果は累積する。
- フォルムチェンジが発動すると、タイプとステータスはチェリム本来のものに戻る。
  - みずびたし/ハロウィンなどでタイプが変更されていてもくさタイプに戻る。
  - パワーシェア/ガードシェア/スピードスワップでステータスが変更されていても元に戻る。パワートリック状態は解除される。

## 備考
- チェリムの専用特性。
- Pokémon LEGENDS アルセウスでは特性が存在せず、フラワーギフトの効果はポジフォルム時のこうげき・とくぼう種族値が1.5倍に上がることで再現されている。同作ではチェリムのフォルムチェンジの条件も異なる。
