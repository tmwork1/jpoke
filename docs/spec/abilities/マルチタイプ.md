# 仕様書: マルチタイプ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%9E%E3%83%AB%E3%83%81%E3%82%BF%E3%82%A4%E3%83%97

## 効果
- 持っているプレートに応じて自身のフォルムとタイプが変わる（プレート#一覧を参照）。
- 第七世代のみ、持っているZクリスタルに応じて自身のフォルムとタイプが変わる。

## 特性の仕様
- この特性によりアルセウスのタイプが変わると、それに合わせてフォルムも変わる。
  - 手持ち画面に映される小アイコンの姿に持たせたプレートが反映されるかは、世代によって異なる。
    - 第八世代以前では持たせたプレートに関係無く通常時のフォルムが表示される。通信対戦の対戦相手は、見せ合い時点ではプレートの有無と種類を判別できない。
    - スカーレット・バイオレットでは持たせたプレートがアイコンに反映される。通信対戦の対戦相手は、見せ合い時点でフォルムからプレートの有無と種類を判別できる。
  - 第六世代以前・BDSPでは、相手の場に出てきたアルセウスのタイプは見た目や相性などから見分けるしかない。第七世代では下画面、第九世代では「様子を見る」画面に表示されるタイプから判別できる。
- マルチタイプを無効化することや、他のポケモンにマルチタイプを付与することはできない。
  - この特性は上書きできない。とくせいなし状態にもならず、バトンタッチで引き継がれるときはとくせいなし状態が消える。かがくへんかガスの効果はアルセウスやさばきのつぶてのタイプ・威力に影響を与えない。
  - この特性はコピーできない。この特性のポケモンがなかまづくりをマジックコート状態で跳ね返して使用しても失敗する。
  - この特性はスキルスワップで交換することができない。
  - アルセウスにへんしんしたポケモンに対しても、マルチタイプを書き換えたり無効化したりすることはできない。
- かたやぶりの効果で無効化できない。アルセウスは特性がなければノーマルタイプだが、かたやぶりの効果がある技がアルセウスをノーマルタイプとみなすことはない。かたやぶりの効果がある技でもマルチタイプの特性を書き換えたり無効化したりすることはできない。
- アルセウスのタイプはみずびたし/まほうのこなで変更できないが、これは特性の効果ではなくアルセウスという種族固有の効果となる。
  - アルセウスにへんしんしたポケモンに対しては、みずびたし/まほうのこなでタイプを変更できる。
  - アルセウスがテラスタルしたときは、持たせたプレートとは別のタイプに変化できる。

### 第五世代以降
- アルセウスがプレートを持っている場合か、アルセウスにプレートを渡そうとした場合、アルセウスのアイテムに影響を与える技の効果は無効になる。
  - アルセウスに対するトリック/すりかえ/ギフトパス、またはアルセウスが使うトリックは自分か相手のどちらかがプレートを持っているときのみ失敗する。どちらもプレートを持っていなければ成功する。
  - アルセウスの持つプレートをはたきおとす/どろぼう/ほしがるで無効化したり奪ったりすることはできない。プレートを持っていなければ技の効果は発動する。
    - 第六世代以降では、プレートを持つアルセウスに対するはたきおとすの威力は上がらない。 プレート以外の道具を持っていれば威力が上がり、道具を落とすこともできる。
  - プレートを持っているアルセウスに対してマジシャン/わるいてぐせの効果は発動しない。アルセウスに対してきょうせいでプレートを渡すことはできない。
- さしおさえ/マジックルーム状態の効果を受けてもアルセウスのタイプ・姿は変わらないが、さばきのつぶてを使うときはプレートの効果を受けずにノーマルタイプになり、威力も上がらない。
- アルセウスにへんしんしたポケモンは、マルチタイプをコピーすることができる。
  - 変身使用者が持っているプレート/Zクリスタルに関わらず、対象のアルセウスと同じフォルムになる。ただしさばきのつぶては変身したポケモンが持っているプレートの種類・有無によりタイプが決まる。
  - アルセウスにへんしんしたポケモンに対しては、プレートを無効化させたり奪ったりすることや、プレートを渡すことができる。第六世代以降でははたきおとすの威力も上がる。

## 備考
- アルセウスの専用特性。
- 似たような効果の特性として、持たせたメモリでタイプが変化するシルヴァディのARシステムがある。
- ソード・シールドでは所持者がいないが、ボックスの検索機能により名前と説明を見ることができる。
