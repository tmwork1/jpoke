# 仕様書: フェアリースキン

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%95%E3%82%A7%E3%82%A2%E3%83%AA%E3%83%BC%E3%82%B9%E3%82%AD%E3%83%B3

## 効果
使用するノーマルタイプの技がフェアリータイプに変わり、威力が1.2倍になる (第七世代以降)。
- 第六世代のみ、威力補正率は1.3倍だった。

## 特性の仕様
- 正確な威力補正率は第六世代では5325/4096倍、第七世代以降では4915/4096倍となる。
- もともとフェアリータイプである技の威力は上昇しない。
- タイプが変わる技の内、ウェザーボール/さばきのつぶて/しぜんのめぐみ/だいちのはどう/マルチアタック/めざめるダンスを使用したときにこの特性は発動しない。
  - めざめるパワーは、ノーマルタイプになることが無いので特性は発動しない。
  - レイジングブルは、ケンタロス以外のポケモンが使えば特性が発動し、威力も上がる。通常の姿のケンタロスが使う場合、この特性は発動しない。
  - テラバースト/テラクラスターは、テラスタルしていないときなら特性が発動し、威力が1.2倍のフェアリー技になる。テラスタル中なら特性は発動せず、テラスタイプと同じタイプの技になる。テラスタイプがノーマル/ステラであるときも同様。
- わるあがきを使用したときにこの特性は発動しない。
- プラズマシャワー/そうでん状態の効果を受けた場合、特性の効果の後に状態の効果が考慮される。
  - プラズマシャワー状態では、フェアリー技に変化することでプラズマシャワー状態の効果は無くなり、ノーマル技はフェアリー技として使える。
  - そうでん状態では、フェアリー技に変化してからそうでん状態の効果を受けるため、使用するノーマル技はでんき技に変わる。特性による威力補正も無くなる。
- この特性でフェアリータイプに変わったノーマル技には以下の効果がある。
  - 第七世代以降の技選択画面では、特性の効果を考慮した相性が表示される。
  - 自身がフェアリータイプのときのみタイプ一致ボーナスが付く。
  - 自身のせいれいプレート/ようせいのハネの効果対象になり、ノーマルジュエル/シルクのスカーフの効果は受けない。
  - 相手のロゼルのみの効果対象になり、ホズのみの効果は受けない。
  - テクスチャー2はフェアリー技に耐性のあるタイプに変わる。テクスチャー2使用時に対象の特性がフェアリースキンでなくても、最後に使用した技がフェアリースキンの効果を受けていればフェアリー技とみなされる。
- 変化技を使用したときも発動する。
  - フェアリータイプの変化技に対して発動する効果が限られているため、バトルに影響するのはプラズマシャワー状態やテクスチャー2の対象になったときなどに限られる。
- Z攻撃技を使用しても特性は発動しない。
  - Z変化ワザを使用したときは発動する。
- ダイマックスわざにするときもノーマルタイプの技はフェアリータイプになるが、特性による威力補正は無い。
  - ウェザーボールなどタイプが変わる技でも、ダイマックスわざにしたときは特性の効果を受ける。
  - 技の選択画面ではダイアタックのままだが、実際に技を繰り出すときにダイフェアリーに変わる。相性は特性の効果を考慮する。
- テラスタル時は特性の効果が発動し、ノーマル技は威力が1.2倍のフェアリー技になる (タイプが変わる技を除く) 。
  - このとき、テラスタルによるタイプ一致補正もフェアリータイプにかかる。元々フェアリータイプのポケモンの場合、フェアリータイプにテラスタルすればタイプ一致ボーナスは2.0になるが、他のタイプにテラスタルすればタイプ一致ボーナスは1.5となる。
  - ステラタイプにテラスタルした場合、特性の効果を受けた技はフェアリータイプのステラ補正を消費する。すでにフェアリータイプのステラ補正を消費している場合、特性の効果を受けた技にステラ補正はかからない。
- きけんよちはこの特性の効果を考慮しない。
- 覚えている一番上の技がノーマルタイプであるポケモンがテクスチャーを使用しても、フェアリータイプではなくノーマルタイプになる。
- 直接攻撃の連続攻撃技を使用し、相手のミイラ/さまようたましい/とれないにおいに特性を書き変えられた場合、その行動が完了するまで相性やタイプ一致ボーナスは1発目の影響が残る。ただし、スキン系特性による威力補正は特性を失うと同時に消える。

## 備考
- メガシンカができない第八世代・第九世代では事実上ニンフィアの専用特性になっている。
