# 仕様書: にげごし

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AB%E3%81%92%E3%81%94%E3%81%97

## 効果
自分のHPが最大HPの1/2超から1/2以下に減ったとき、トレーナー戦では交代する。野生ポケモン戦ではにげる。

## 特性の仕様
詳細な仕様はききかいひ#特性の仕様を参照とする。以下に、捕獲の際にも影響が出やすい効果のみ抜粋する。
- 状態異常/状態変化/場の状態などのスリップダメージにより半分以下になったときも発動する。
- こんらんによる自傷ダメージや、いたみわけを受けてHPが半分以下になったときは発動しない。
- 特性かげふみ/ありじごくや、にげられない/バインド/フェアリーロック状態の効果を無視して逃げる。

## 備考
- コソクムシの専用特性。グソクムシャに進化すると特性がききかいひに変わるが、名前が変わるだけで効果は変わらない。
