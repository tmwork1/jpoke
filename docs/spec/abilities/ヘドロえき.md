## 特性の仕様
- ヘドロえきのポケモンがひんしになったときも発動する。
  - このときのダメージでお互いのポケモンが全滅した場合、第六世代まで攻撃側のポケモンが先に倒れるためヘドロえき側の勝ちになる。第七世代以降はヘドロえきのポケモンが先に倒れるためヘドロえき側の負けになる。
- パラボラチャージ/シャカシャカほうなど、複数のポケモンを対象とする吸収技を受けた場合、ヘドロえきのポケモンからの吸収分のみがダメージに変わる。ヘドロえきにより攻撃側のHPが0になった場合、まだ吸収できていない対象がいたとしてもその時点で攻撃側はひんしになる。
- 攻撃側がおおきなねっこを持つ場合、ヘドロえきによるダメージは1.3倍になる。
  - ヘドロえき側だけがおおきなねっこを持っていても効果は無い。
- 相手がかいふくふうじ状態であってもダメージ効果は発動する。
- 特性がマジックガードのポケモンに対しては、ヘドロえきでダメージを与えることはできない (回復効果は無くなる)。
- 相手のみがわり状態を無視してダメージを与える。
- かいがらのすずによる回復効果はダメージに変えない。
- キョダイダンエンによる回復効果はダメージに変えない。
- かたやぶりの効果がある技に対しても発動し、相手にダメージを与える。
- ヘドロえき側のアイテムがゴツゴツメット/ジャポのみ/レンブのみである場合、ヘドロえきの効果の後にアイテムの効果によるダメージが発生する。途中で相手がきのみの発動圏に入ったときは回復を挟むことができる。
- 味方からやどりぎのタネを受けた後サイドチェンジで位置を入れ替えた場合、自分のやどりぎのタネ状態にヘドロえきが発動して自分がダメージを受ける。

## 備考
- この特性を所有するポケモンは全てどくタイプを持つ。
- どくタイプのポケモンがこの特性のポケモンに吸収技を使ってもダメージを受ける。
