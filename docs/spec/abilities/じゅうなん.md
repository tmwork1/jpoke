# 仕様書: じゅうなん

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%98%E3%82%85%E3%81%86%E3%81%AA%E3%82%93

## 効果
まひ状態にならない。まひ状態になっても回復する。

## 特性の仕様
- すでにまひ状態になっているポケモンの特性がじゅうなんに書き換わった場合、即座にまひを治す。
- すでにまひ状態になっているじゅうなんのポケモンを戦闘に出すと、即座にまひを治す。
  - 同時に場がどくびしの場合、どくびしのどくはまひ状態により不発してからじゅうなんでまひを治すため、どくびしの効果は防がれる。
  - まひ状態のポケモンが戦闘中以外でじゅうなんを得ても即座にまひは治らず、次に戦闘の場に出たときにまひを治す。ニャース/バルキーがそれぞれペルシアン/サワムラーに進化してじゅうなんを得たときや、とくせいカプセル/とくせいパッチで特性をじゅうなんに変えたときが該当する。
  - 第三世代のみ、先発で出したとき、ひんしのポケモンの代わりに出したとき、トレースでじゅうなんをコピーしたときは、ポケモンが何かしら行動するまでまひは治らない(後出しで繰り出したときは即座に治る)。
- かたやぶりの効果がある技によってじゅうなんのポケモンがまひ状態になった場合、直後に回復する。
- 特性じゅうなんでまひ状態を治せるポケモンが、同時にクラボのみかラムのみ等のまひを治すきのみも持っていた場合、第四世代まで特性が先に発動してきのみは消費されない。第五世代以降は特性よりきのみが先に発動して消費される。
- 本来じゅうなんを持つポケモンが、特性を変更されてまひ状態にされた場合、第四世代のみ交代で引っ込めるとまひが治っている(このときメッセージは無い)。第三世代と第五世代以降は交代してもまひは治らない。
- 第六世代以降で特性じゅうなんのでんきタイプのポケモンがまひ状態になる変化技を受けた場合、タイプによる無効化が優先され、特性に関するメッセージは出ない。

## 備考
- ルビー・サファイアから存在していた特性だが、発売当時はこの特性を持つポケモンはいなかったため、ファイアレッド・リーフグリーンからの登場となった。
- 第六世代以降ではでんきタイプでまひを防げるため、マッギョはほぼじゅうなんを活用できなくなった。一応ほごしょくなどでタイプを変えた後もまひを防げるという差別点はある。
