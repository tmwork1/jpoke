# 仕様書: サーフテール

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%B5%E3%83%BC%E3%83%95%E3%83%86%E3%83%BC%E3%83%AB

## 効果
フィールドがエレキフィールド状態のとき、すばやさが2倍になる。

## 特性の仕様
- 所有者が地面におらず、エレキフィールド状態の効果を受けられない場合でも、この特性の効果は発動する。
- すばやさがランクで変動している場合、変動後のステータスを2倍にする。まひの効果はサーフテールやその他の補正の効果を考慮した後に計算する(すばやさ#すばやさ補正も参照)。
- この特性を受け取ったり失ったりした場合、第七世代ではそのターンの行動順には影響しない。第八世代以降ではそのターンから行動順に影響が出る。

## 備考
- ライチュウ(アローラのすがた)の専用特性。
- 大乱闘スマッシュブラザーズ SPECIALでは、モンスターボールから出現するライチュウ(アローラのすがた)が使用する技の名前になっている。
