# 仕様書: ビーストブースト

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%93%E3%83%BC%E3%82%B9%E3%83%88%E3%83%96%E3%83%BC%E3%82%B9%E3%83%88

## 効果
攻撃技で他のポケモンをひんしにさせたとき、自分の最もステータスが高い能力のランクが1段階上がる。
最も高い能力が複数ある場合は、こうげき・ぼうぎょ・とくこう・とくぼう・すばやさの順で最初にあるものが選ばれる。

## 特性の仕様
- 上がる能力にHPは考慮しない。
- 上がる能力が選ばれる際、参照にする数値は実数値である。ランク補正/もちもの/味方のとくせい/すなあらしの効果は考慮しない。
  - パワーシェア/ガードシェア/パワートリック/スピードスワップ/ワンダールームにより実数値が変動した場合、それに応じ上がる能力も変化しうる。実数値の変動はステータスの比較より先に考慮する。例えば、攻撃と防御が同じポケモンがパワートリックを使用しても上がる能力は攻撃のままであり、特攻と特防が同じで通常なら特攻が上昇するポケモンはワンダールーム状態では防御が上がる。
- すでに上げる能力が最大ランクまで上がっている場合、この特性は発動しない。上がる能力が2番目に高い能力になることは無い。
- 上がる能力が自分のステータスに依存する点を除けば、じしんかじょうと同様の効果を持つ。特性の発動条件に関してはじしんかじょう#特性の仕様を参照とする。

## 備考
- ウルトラビーストの専用特性と捉えることもできる。
- 設定上のビーストブーストは、ウルトラホールを通過する際に浴びて身にまとうオーラを指す。ぬしポケモンが身にまとうオーラも、過去ウルトラホールが開いた際地上に残ったビーストブーストであると捉えることもできる。
  - ただし、ウルトラビーストとの捕獲イベントにおいては、オーラによる能力上昇を無視してこの特性を発動させる。
  - オーラとしてのビーストブーストは色違いエフェクトより先に処理される。
