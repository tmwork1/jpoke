# 仕様書: こおりのりんぷん

調査日: 2026-06-03

## 効果
特殊技を受けたときのダメージが半減する。

## 特性の仕様
- ダメージ固定技のダメージは半減できない。
- サイコショック/サイコブレイク/しんぴのつるぎを受けるときはぼうぎょの値を参照をするが、特殊技なのでダメージを半減できる。
- 分類が変わる技を受けた場合、特殊技のままであればダメージを半減できるが、物理技に変わったときはダメージを半減できない。
- 場がワンダールーム状態のときでも特殊技のダメージを半減する。

## 備考
- ユキハミ系統の専用特性。
- こおりのりんぷんの物理版のような特性としてファーコートがあるが、物理技のダメージを半減する効果ではなくぼうぎょが2倍になる効果となる。
