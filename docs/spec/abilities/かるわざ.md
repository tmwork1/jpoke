# 仕様書: かるわざ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8B%E3%82%8B%E3%82%8F%E3%81%96

## 効果
場に出た後に自分のアイテムが無くなるとすばやさが2倍になる。再びアイテムを手にするとすばやさは元に戻る。場に出たときからアイテムがない場合は発動しない。

## 特性の仕様
- 一度アイテムを消費して、かるわざを発動させてから交代で引っ込んだ場合、再び場に出てきても「初めから何も持っていなかった」扱いになり、かるわざは発動しない。
- 場に出たときと同時にアイテムを消費したときはかるわざは発動する。
- 道具を消費する手段は問わない。以下の状況でかるわざは発動する。
  - 普通に道具を消費した場合 (なげつける/しぜんのめぐみ/ほおばるを使用したときも有効)。
  - どろぼう/ほしがる/トリック/すりかえ、特性マジシャン/わるいてぐせにより道具を失った場合。
  - ついばむ/むしくい/はたきおとす/やきつくす/おちゃかい/ふしょくガスを受けて道具が失われた場合。
  - 第四世代では、はたきおとすの効果ははたきおとす状態にするだけでアイテム欄から道具は消えないが、かるわざは発動する。
- 発動中に次のように道具を手に入れた場合、素早さは元に戻る。再び消費できれば発動する。
  - どろぼう/ほしがる/トリック/すりかえ/リサイクル/キョダイサイセイで道具を持った場合。
  - 他者からギフトパス、特性きょうせいで道具を受け取った場合。
- 道具が無い状態で、相手からなげつけるを受けてきのみや道具を消費してもかるわざは発動しない。
- 道具が無い状態で、ついばむ/むしくいを使用して相手のきのみを消費してもかるわざは発動しない。
- さしおさえ/マジックルーム状態では道具を失ったとは見なされず、かるわざは発動しない。
- アイテムが発動しても、消費されない場合はかるわざは発動しない。
- かるわざが他の特性に書き換わった場合、素早さは元に戻る。
  - 他の特性を持つポケモンが道具を消費した後に、特性がかるわざになった場合、かるわざは発動しない。
  - かるわざが発動中に、かるわざが他の特性に書き換わり、再度かるわざに書き換わった場合、かるわざは発動しない。
  - かるわざが発動中に、かるわざ同士でスキルスワップをした場合、かるわざは発動しなくなる。
- かるわざが発動後にかがくへんかガスが発動した場合、その間かるわざの効果は無くなる。かがくへんかガスが解除されればかるわざは再度発動する。
  - かるわざが発動していないときにかがくへんかガスが発動し、その間にアイテムを消費してからかがくへんかガスが解除された場合、かるわざは発動しない。
- かるわざによって上昇した素早さはランク補正とも状態変化とも異なる。この素早さ上昇はバトンタッチでは引き継げない。
  - 素早さがランクで変動している場合、変動後のステータスを2倍にする。まひの効果はかるわざやその他の補正の効果を考慮した後に計算する(すばやさ#すばやさ補正も参照)。
- 他のポケモンの行動によって道具が失われかるわざが発動する場合や、道具を持たされかるわざの効果が消える場合、第七世代まではそのターンの行動順は変化しない。第八世代からはそのターンから行動順に影響が出る。

## 備考
- 第四世代ではフワンテ系統の専用特性だった。
