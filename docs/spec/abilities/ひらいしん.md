# 仕様書: ひらいしん

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B2%E3%82%89%E3%81%84%E3%81%97%E3%82%93

## 効果
### バトルでの効果
- 第三世代：ダブルバトルで相手がでんきタイプの技を使ったとき、その攻撃対象を自分にする。
- 第四世代：ダブルバトルで自分以外がでんきタイプの技を使ったとき、その攻撃対象を自分にする。
- 第五世代以降：自分以外のポケモンが自分以外のポケモンに対してでんきタイプの技を使ったとき、その攻撃対象を自分にする。さらにでんき技を無効化し、とくこうを1段階上げる。

### フィールドでの効果
- エメラルドのみ、手持ちの先頭にこのとくせいを持つポケモンがいると、ポケナビの呼び出し確率が2倍になる。
- 第八世代でのみ、手持ちの先頭にこのとくせいを持つポケモンがいると、出現可能なでんきタイプの野生ポケモンが50%の確率で出現する。ひんし状態のときも効果がある。

## 特性の仕様
- 第四世代までは、でんき技を引き寄せるだけで無効化する効果は無い。以下、無効化に関する効果は第五世代以降の仕様である。
- でんじは/かいでんぱなどの変化技も引き寄せる対象になる。第五世代以降はさらに無効化して特攻が上がる。
  - 同時に自身がマジックコート状態である場合、引き寄せた変化技は跳ね返し特攻は上がらない。
  - 他のポケモンがマジックコート状態である場合も、自身に引き寄せる効果の方が優先される。
  - マジックコート状態で跳ね返されたでんきタイプの変化技もひらいしんで引き寄せる対象になる。
- じゅうでんなど、使用者自身を対象にする技は引き寄せず、自身が使用したときも無効化しない。
- エレキフィールドなど、場を対象とする技に対しては発動しない。
- エレキネット/ほうでんなど、複数のポケモンが対象の技は引き寄せない。自分が受ける分は無効化して特攻が上がる。
- めざめるパワーなど、タイプが変わる技がでんきタイプになった場合、第四世代以前は引き寄せることができないが、第五世代以降なら引き寄せることができる。
- そうでん/プラズマシャワー状態や、特性エレキスキンなどの効果ででんきタイプになった技も、基本的には引き寄せて無効化できる。例外として、以下の技はでんき技にしたとしても特性が発動しないことがある。
  - くさのちかい/ほのおのちかい/みずのちかい/ねらいうちは引き寄せない。直接ひらいしんのポケモンが狙われた場合は無効化する。
  - コーチング/たがやす/とおぼえ8-に対して特性は発動しない。
  - アロマミストは引き寄せることができず、特性で無効化することもできない。トリプルバトルにおいても同様。
  - デコレーションは引き寄せることができるが、特性で無効化することはできない。
- でんきタイプの連続攻撃技も引き寄せて無効化するが、特攻は1回のみ上昇する。
- 自身がみがわり状態でも技を引き寄せる効果や、無効化して特攻が上がる効果は発動する。
  - 他のポケモンのみがわり状態に対してでんき技が使われた場合でも、ひらいしんで引き寄せる効果が優先される。
- じめんタイプのポケモンがこの特性を持っている場合も引き寄せる効果は働く。第七世代以降では相性よりも特性による無効化の方が優先されため、特攻は上がる。
  - 第五世代と第六世代ではタイプ相性による無効化の方が優先されるため、特攻は上がらない。
- 自身がまもる状態のときでもでんき技を引き寄せる効果は発動する。まもる状態で防げる技の場合特攻は上がらない。
  - 特性じょおうのいげん/ビビッドボディ/テイルアーマー、サイコフィールド/ファストガード/ワイドガード/トリックガード/たたみがえし状態で守られているときも同様にして、引き寄せる効果は発動するが特攻は上がらない。
  - 第七世代以降で、いたずらごころで優先度が上がったでんき技がひらいしんのあくタイプに無効化される場合、ひらいしんの効果が優先されて特攻が上がる。
  - 他のポケモンに対してこれらの効果で防御可能なでんき技が使われた場合でも、ひらいしんで引き寄せる効果が優先される。
- このポケモンが姿を隠している間でもひらいしんの効果は残り、でんきタイプの技を引き寄せる。
  - ただし、引き寄せた攻撃がひらいしんのポケモンに当たらないと、特攻上昇は起こらない。ひらいしんのポケモンがそらをとぶ状態ならばかみなりは当たるため特攻上昇する。
  - 他の姿を隠したポケモンに対してでんき技が使われた場合でも、ひらいしんで引き寄せる効果が優先される。
- 第五世代以降では命中判定より先に特性の発動判定があるので、姿を隠しているときを除けばでんき技を回避することなく特攻が上昇する。
  - 第四世代では引き寄せたでんき技が外れる可能性もある。
- エレクトロビームなど、でんきタイプの溜め技が使われたときは、各ターンごとに攻撃が引き寄せられるかが判定される。
  - 溜め始めたときにひらいしんが場にいなくても、攻撃時にはひらいしんがいた場合は、ひらいしんへ攻撃が引き寄せられる。
  - 溜め始めたときにひらいしんが場にいても、攻撃までに場から去っていた場合は、ひらいしんがいた位置ではなく技の対象に選んだ位置へ攻撃が行われる。
- 他のポケモンがじんらいを選んだ場合、技の対象はひらいしんの効果で引き寄せられ、ひらいしんのポケモンが選んだ技とその行動順によって技の成否が決まる。失敗した場合は特攻が上がらず、対象が変更されたときでもひらいしんの特性バーとメッセージは現れない。
- ひらいしん以外を対象としたエレクトロビームが溜められたか、あるいはじんらいが失敗した際は、ひらいしんの特性バーや「攻撃を 引き寄せた!」のメッセージは現れないものの、内部上はひらいしんに攻撃が引き寄せられる処理が行われている。これらの技の直後に行動するポケモンがひらいしんのポケモンを対象に技を使用する場合、それがでんき技でなくてもこのタイミングでひらいしんの特性バーとメッセージが流れる。メッセージが後にずれ込んで表示されるだけであり、でんきタイプ以外の技が無効化されるわけではない。元々ひらいしんのポケモンが対象だった技でしか発生しないため、対象が変更されたわけでもない。
  - 直後に使われた技が変化技や、全体攻撃技だったときや、ひらいしんのポケモン自身を対象にした変化技だったときもこの現象が起きる。
- じばそうさを受けたときは、特性不一致による技の失敗が優先され、ひらいしんは発動しない。
  - そうでん状態のアシストギアを受けたときも、同様にして特性は発動しない。
- この特性ででんき技を無効化したときに、アイテムのじゅうでんちは発動しない。

### 技を引き寄せる効果について
- ちゅうもくのまと状態のポケモンが他に場にいる場合、そちらが優先されるためでんき技を引き寄せられない。
- 場に複数のひらいしんのポケモンがいる場合、すばやさが一番高いポケモンのひらいしんが優先される。ランク補正やトリックルーム状態などの効果を除外したすばやさ実数値を比較する。
- ひらいしんのポケモン自身が使うでんき技は、自身のひらいしんには引き寄せられない。
- 第四世代以降では味方が使用するでんき技も引き寄せる。
- でんき技を引き寄せた際、特性の発動を示すメッセージが流れる。元々ひらいしんのポケモンが対象とされていればこのメッセージは流れない。
  - 第四世代では「<ポケモン名>は ひらいしんで こうげきを うけた!」と流れる。
  - 第五世代以降では特性バーが表示されて「<ポケモン名>は こうげきを ひきよせた! / <ポケモン名>は 攻撃を 引き寄せた!」と流れる。その後、技の無効化を示すメッセージも流れる。
- かたやぶり効果があるでんき技を引き寄せたり無効化することはできない。
- 特性スクリューおびれ/すじがねいりのポケモンが使用したでんき技は引き寄せない。直接使用された場合は無効化する。
- トリプルバトルでは、自分とでんき技の使用者が対角の位置にいるときなど、自分に届かないでんき技は引き寄せない。
  - そうでん状態などで、でんき技になった離れたポケモンも対象になる技の場合は、お互いの位置に関係無く引き寄せることができる。
- ダイサンダーなどの、でんきタイプのダイマックスわざは味方を対象に選べないが、味方のひらいしんに引き寄せられる。
- テラレイドバトルでは引き寄せる効果が発動しない。直接でんき技を受けたときは特性が発動して無効化する。

## 備考
- でんきタイプの代わりにみずタイプの技を引き寄せる特性としてよびみずがある。よびみずに比べ、ひらいしんは所持するポケモンが多い。
- 第五世代以降のひらいしんのようにでんき技を無効化する効果がある特性としてちくでん/でんきエンジンがある。
- ウルトラサン・ウルトラムーンでは、マーマネの試練での漏電をトゲデマルがひらいしんで吸収した際、マーレインが「トゲデマルの電気袋に収まる量ではない」と発言している。そのため、ゲームシステムとして反映されることはないものの、世界観設定上では吸収して無効化できる限界容量が存在すると思われる。
