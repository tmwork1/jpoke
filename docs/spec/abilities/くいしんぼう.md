# 仕様書: くいしんぼう

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8F%E3%81%84%E3%81%97%E3%82%93%E3%81%BC%E3%81%86

## 効果
本来は残りHPが最大HPの1/4以下になったときしか発動しないきのみを、最大HPの1/2以下になったときに発動できる。
もとから最大HPの1/2以下で発動するきのみ（オボンのみなど）に対しては効果は無い。発動条件が残りHPでないきのみ（ラムのみなど）も同様に効果は無い。
対象となるきのみは以下。
- フィラのみ (第七世代以降)
- ウイのみ (第七世代以降)
- マゴのみ (第七世代以降)
- バンジのみ (第七世代以降)
- イアのみ (第七世代以降)
- チイラのみ
- リュガのみ
- カムラのみ
- ヤタピのみ
- ズアのみ
- サンのみ
- スターのみ
- ミクルのみ
- イバンのみ

## 特性の仕様
- HPが最大HPの1/4超-1/2以下になった瞬間に対応するきのみを発動できる（通常のきのみ発動タイミングと同じ、HP変化直後）。
- 他の特性を持つポケモンのHPが1/4超-1/2以下のときに、特性がくいしんぼうに書き換わると、その瞬間に対応するきのみを食べることができる。
- この特性のポケモンが場に出た時点ですでにHPが1/4超-1/2以下である場合、その瞬間に対応するきのみを食べることができる。
- かたやぶりの効果がある技を受けてHPが1/4超-1/2以下になったときもきのみを発動できる。
- HPが1/4超-1/2以下になったがきんちょうかんの効果できのみを食べられなかった場合、きんちょうかんの効果が切れた瞬間にきのみを食べることができる。
- HPが1/4超-1/2以下になったがかがくへんかガスの効果できのみを食べられなかった場合、かがくへんかガスの効果が切れてもイバンのみ以外のきのみを食べることはできない。この状態になっても以下の要因できのみを発動できる。
  - ミクルのみ以外は回復したり変化技を受けたりしても発動しないが、定数ダメージを含めた何かしらのダメージを受けると発動する（HPが1/4以下にならなくてもよい）。
  - ミクルのみは自身が行動を終えた後に発動する。状態異常・状態変化などにより行動ができなかったときでも発動する。
- HPが1/4超-1/2以下のとき、かがくへんかガスのポケモンが交代して場から去ったターンにイバンのみは発動しない。逆に、交代してかがくへんかガスのポケモンが出てくるターンにイバンのみは発動できる。
- 特性ミイラ/さまようたましい/とれないにおいのポケモンを直接攻撃したとき、相手のゴツゴツメット/ジャポのみ/レンブのみや、反動技のダメージでHPが1/4超-1/2以下になった場合、特性の変更が優先されてきのみは発動しない。

## 備考
- 現行実装は「HP変化時に閾値以下だと発動」という既存のきのみ発動ロジック（他のHP閾値系きのみ全般に共通）をそのまま踏襲し、その閾値をcuいしんぼう所持時のみ1/4→1/2に緩和する形で実現している。
- 特性が後から書き換わった瞬間の再判定・場に出た瞬間に既にHP1/4超-1/2以下である場合の判定・かがくへんかガス解除後のイバンのみ限定発動などの詳細仕様は、既存のHP閾値系きのみ全般が同様に未対応（本特性固有の欠落ではない）であるため、本実装では対象外とする。
