# 仕様書: きもったま

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8D%E3%82%82%E3%81%A3%E3%81%9F%E3%81%BE

## 効果
- ゴーストタイプに対して使うノーマル・かくとうタイプのわざの相性が、無効ではなく等倍になる。
- 第八世代から、特性いかくを無効化する。

## 特性の仕様
- この特性で当たるようになったノーマル・かくとうタイプの技はゴーストタイプには等倍（ダメージ1倍）として扱われ、他のタイプの相性は通常通り処理される。例えばはがね・ゴーストのギルガルドに対してノーマル技を使えばいまひとつに、かくとう技を使えばばつぐんになる。
- きもったまのポケモンが、ノーマル・かくとうタイプで直接攻撃の連続攻撃技をミイラ/さまようたましい/とれないにおいのゴーストタイプに対して使用した場合、1発目できもったまはミイラ/さまようたましい/とれないにおいに変更されるが、技が終わるまではきもったまの効果は残り、ダメージを与え続ける。

## 備考
- 第四世代からこの特性を持つガルーラ・ミルタンクは共に性別が♀のみしか存在しない。
- 第四世代のこの特性の説明文には具体的にどの技がゴーストタイプに当たるかは書かれておらず、第五世代・第六世代では、かくとうタイプの技もゴーストタイプに当たる効果については書かれていない。ライブキャスターのチェレンからはかくとう技が当たる効果についても説明される。第七世代以降の記述ではどちらの点も明記されている。
- いかくを無効にする効果がついたのは第八世代からだが、説明文に明記されるようになったのは第九世代からとなる。
- 似たような効果を持つ特性としてしんがんがある。しんがんには命中ランクを下げられなくなる等の効果も付いているが、きもったまと異なりいかくを防ぐ効果は無い。
