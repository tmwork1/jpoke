# 仕様書: たんじゅん

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%9F%E3%82%93%E3%81%98%E3%82%85%E3%82%93

## 効果
- 第四世代: 能力のランク数が通常の2倍の扱いになる。ランクの扱いが+6より大きくなったり、-6より小さくなることはない。
例:+1の状態なら+2のときと同じ効果を得られ、逆に-1なら-2と同じ扱いになる。+4以上でも全て+6の扱いになり、-4以下でも全て-6の扱いになる。
- 第五世代以降: 能力のランク数の変化量が通常の2倍になる。

## 特性の仕様
- 命中率/回避率のランクに関しても効果がある。
- 急所ランクには効果が無い。

### 第五世代以降
- ランクが低下する量も2倍になる。
- パワーシェア/ガードシェア/スピードスワップ/パワートリックなど、実数値が変化する効果は特性の影響で変化しない。
- 以下の状況によるランク変動に対しては発動しない。(ランク補正#概要の項も参照)
  - バトンタッチでランク補正を受け継いだとき。
  - じこあんじを使用したとき。
  - パワースワップ/ガードスワップ/ハートスワップ/ひっくりかえすを受けたとき、または使用したとき。
  - くろいきり/クリアスモッグ、きみょうなくすりでランク補正がリセットされたとき。
  - Z変化ワザの付加効果で能力が変化するとき。
    - 例えばまるくなるをZワザに変えて使用した場合、付加効果の命中率上昇には特性の効果が無く1段階しか上がらないが、技の効果には特性の効果がありぼうぎょが2段階上昇する。
    - Z変化ワザではないナインエボルブーストやブレイジングソウルビートによる上昇量は2倍になる。
- たんじゅんのポケモンがシャドースチールを受けたときのランクリセット効果に対しては発動しない。たんじゅんのポケモンがシャドースチールを使用した場合は吸収した量の2倍分能力が上がる。
- ダイマックスわざの効果によるランク変動はたんじゅんの効果対象となる。
- ちょうのまい/からをやぶるなど、複数のランクが変動する技に対しても、全ての能力が対象になる。
- シード系アイテム/プラスパワー系アイテム/ものまねハーブなど、アイテムの効果によるランク変動にも適用される。
- ねばねばネット/たこがため/あめまみれ状態によるランク変動にも適用される。
- ちからをすいとるを受けるとこうげきが2段階下がるが、相手の回復量は変わらない。
- かたやぶりの効果が発動した技によりランクが変動する場合、2倍にはならない。
  - この攻撃技によりじゃくてんほけん/アッキのみ/タラプのみ/きゅうこん/じゅうでんち/ひかりごけ/ゆきだまが発動しても上昇量は2倍にはならない。
  - たんじゅんのポケモンが場にいるときにかたやぶりの技の効果でエレキシード/サイコシード/グラスシード/ミストシード/ルームサービスが発動する場合、変化量は2倍にはならない。かたやぶりの技が発動した後の場にたんじゅんのポケモンを繰り出してこれらのアイテムが発動する場合、変化量は2倍になる。
  - かたやぶり効果がある技で発動したわたげを受けた場合、たんじゅんの効果を無視してすばやさが1段階下がる。
- マックスレイドバトルでバリアが破壊された後のランク変動にも適用され、ぼうぎょ・とくぼうが4段階ずつ低下する。
