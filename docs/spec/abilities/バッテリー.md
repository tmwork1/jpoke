# 仕様書: バッテリー

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%90%E3%83%83%E3%83%86%E3%83%AA%E3%83%BC

## 効果
自分以外の味方が使う特殊技の威力が1.3倍になる。

## 特性の仕様
- 正確な補正率は5325/4096倍。
- 自身が使う技の特殊技の威力は上がらない。
- パッテリーのポケモンが場にいるとき、味方が使用したみらいよち/はめつのねがいの威力も上げる。シングルバトルにおいても特性の効果がある。
- 味方の場に複数のバッテリーのポケモンがいる場合、効果はその分累積する。2匹のバッテリーの効果を得たポケモンの特殊技の威力は6923/4096(≒1.69)倍、3体なら9000/4096(≒2.12)倍になる。

## 備考
- デンヂムシの専用特性。
- 似た効果を持つ特性にパワースポットがあり、これは物理特殊問わず味方の技の威力が1.3倍になる。
