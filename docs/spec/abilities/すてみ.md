## 特性の仕様
- いのちのたまを持つポケモンが上記以外の技を使ってもすてみの効果は発動しない。
- 以下の技はすてみの効果対象ではなく、威力は増えない。
  - わるあがき
  - ビックリヘッド/てっていこうせん/クロロブラスト
  - じばく/だいばくはつ/ミストバースト
  - 反動で動けなくなる技
  - 反動で能力が下がる技
- 第五世代以降では正確な補正率は4915/4096倍となる。

## 備考
- 第四世代ではサワムラーの専用特性だった。
- 第九世代現在では、正規手段では覚えるポケモンがいないはめつのひかりを除く対象技はすべて物理技で直接攻撃である。実質的に、より補正率が高く対象技も多いかたいツメの下位互換に相当するが、すてみとかたいツメを両方所有するポケモンはいない。
