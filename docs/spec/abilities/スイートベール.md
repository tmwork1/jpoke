# 仕様書: スイートベール

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%B9%E3%82%A4%E3%83%BC%E3%83%88%E3%83%99%E3%83%BC%E3%83%AB

## 効果
自分を含めた味方のポケモンはねむり・ねむけ状態にならなくなる。

## 特性の仕様
- すでにねむり状態になったポケモンを治す効果は無い。
- すでにねむけ状態になったポケモンを治す効果は無いが、ねむけによるねむりに落ちることはスイートベールで防がれる。
- トリプルバトルにおいては隣接していない味方にも効果がある。
- スイートベールに守られているポケモンは、ねむるを使おうとしてもスイートベールに防がれて失敗する。
- 特性ほうしによるねむり状態も防ぐことができる。
- かたやぶりの効果がある技はスイートベールの効果を無視する。
  - このときスイートベールの影響下にいるポケモンもねむり・ねむけ状態にできる。ただし、ねむけからねむりに落ちることはスイートベールにより防がれる。
  - かたやぶりのポケモンは味方のスイートベールも無視するため、使用するねむるは成功し、相手の特性ほうしの効果でねむり状態になることがある。
  - 特性を無視する技を使用した際も同様に味方のスイートベールを無視する。第七世代では使用者自身のスイートベールも無視する。第八世代以降では使用者自身のスイートベールは無視しない。

## 備考
- 似た効果を持つ特性としてやるき/ふみんがある。ねむり状態を防ぐ点は同じだが、スイートベールはねむり状態を回復することはできない代わり、自分以外の味方にも効果がある違いがある。
