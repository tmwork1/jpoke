# 仕様書: みつあつめ

調査日: 2026-06-03

## 効果
### バトルでの効果
なし。

### フィールドでの効果
バトル終了後、あまいミツを拾ってくることがある。
- バトルに参加したかは問わない。
- レベルが高いほど拾ってくる確率が高くなる。
| レベル | 確率 |
| --- | --- |
| 1～10 | 5% |
| 11～20 | 10% |
| 21～30 | 15% |
| 31～40 | 20% |
| 41～50 | 25% |
| 51～60 | 30% |
| 61～70 | 35% |
| 71～80 | 40% |
| 81～90 | 45% |
| 91～100 | 50% |

## 特性の仕様
- 野生ポケモンから逃げた場合、とくせいは発動しない。捕まえた場合は拾うことがある。
- バトルタワー (ガラル地方)・しばり組手でバトルチームのポケモンを使用した場合も、戦闘終了後に手持ちにいる「みつあつめ」のポケモンがあまいミツを拾ってくることがある。

## 備考
- 第四世代ではミツハニーの専用とくせいだった。
  - 第六世代以前では進化前のポケモン専用のとくせいだった。
- このとくせいを持つポケモンは全てあまいかおりを覚えることができる。
- フィールド上で似たような効果のあるとくせいとして「ものひろい」がある。「みつあつめ」は「あまいミツのみ拾えるものひろい」といえる効果を持つ。
  - 「ものひろい」であまいミツは拾えないため下位互換ではない。
  - 「ものひろい」はレベルが高いと拾うどうぐが変わる（ルビー・サファイア/ファイアレッド・リーフグリーン除く）のに対し、「みつあつめ」はレベルが高いと拾う確率が高くなる。
