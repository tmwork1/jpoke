# 仕様書: きんしのちから

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8D%E3%82%93%E3%81%97%E3%81%AE%E3%81%A1%E3%81%8B%E3%82%89

## 効果
変化技を使用するとき、同じ優先度の行動の中で最後に行動するようになるが、他のポケモンの特性を無視して技を出せる。
無視できる特性の詳細についてはかたやぶり#効果を参照。

## 特性の仕様
- 変化技を選択したときに限り、あとだしとかたやぶりが複合したような効果が発動する。あとだし#特性の仕様/かたやぶり#特性の仕様も参照。
  - 攻撃技を選んだときは特性の効果は発動しない。
  - ねごとなど他の技が出る技を選択し、それにより攻撃技が選ばれた場合、かたやぶり効果が発動して特性を無視できる。
- あとだしと同じく、使用する技の優先度自体は変化しない。
  - 同じターンで複数の特性きんしのちから/あとだし・アイテムこうこうのしっぽを持ったポケモンが同じ優先度の技を使用したときは、補正込みの素早さが高い順に行動する。きんしのちからのポケモンがこうこうのしっぽを持っているときも同様となる。
  - 場がトリックルーム状態のときでも行動順は遅くなる。
- 行動順が遅くなる効果に関しては技の選択時に、特性を無視する効果に関しては技の使用直前において特性の発動判定が行われる。
  - きんしのちからのポケモンが、変化技で行動する前に特性を消されたとしても行動順は早くならないが、相手の特性を無視することはできなくなる。
  - 他の特性のポケモンが、変化技で行動する前にきんしのちからの効果を得たとしても行動順は遅くならないが、相手の特性を無視できるようになる。きんしのちからがかがくへんかガスで無効化されていたが、行動前にガスが解除されたときも同様になる。
- さいはいにより1ターンに複数回技を使う機会を得た場合、そのターン選択した技の分類に関係無く、繰り出す技が変化技であるときに特性を無視できる。
  - 攻撃技を選択したターンでも、行動前に受けたさいはいで出した変化技なら特性を無視できる。選択していた攻撃技は特性を無視できない。
  - 変化技を選択したターンでも、行動前に受けたさいはいで出した攻撃技では特性を無視できない。選択していた変化技は特性を無視できる。
- 変化技を選択し、おさきにどうぞの効果で行動順を繰り上げた場合、相手の特性を無視できる。
- 変化技を選択したターンでは、持たせたせんせいのツメ/イバンのみは発動することがない (あとだしの仕様とは異なる)。
  - 攻撃技を選択したターンならこれらのアイテムの効果も発動する。
  - かがくへんかガス等できんしのちからが無効化されている場合は発動し得る。発動したターン、行動する前にかがくへんかガス等が解除されるなどして特性の効果が戻った場合でも行動順に特性の効果は無く、同じ優先度の中で最初に行動することができる。
- かたやぶりと同じく、かたやぶりで無視できない特性 (ぜったいねむりなど) は無視できない。
  - ふみん/やるき/めんえき/じゅうなん/マイペース/どんかんなどの特性には、状態異常/状態変化を防ぐだけでなく、回復する効果もあるので、きんしのちからの効果で特性を無視して与えた状態異常/状態変化も即座に回復される。
  - おうごんのからだ/きよめのしおには状態異常を回復する効果は無いので、きんしのちからの効果で状態異常にすれば回復しない。
  - とくせいガードを持つポケモンの特性は無視できない。
