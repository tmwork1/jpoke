# 仕様書: いかりのつぼ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%84%E3%81%8B%E3%82%8A%E3%81%AE%E3%81%A4%E3%81%BC

## 効果
相手の攻撃が急所に当たると、こうげきが最大まで上がる。

## 特性の仕様
- いかりのつぼが発動すると、こうげきランクが-6の状態からでも+6まで上がる。
- みがわり状態で防いだ技が急所に当たった場合、第四世代のみいかりのつぼは発動する。第五世代以降は発動しない。
  - 第五世代以降でも、音のわざやすりぬけなどの効果でみがわりを無視した技や、連続攻撃技により途中でみがわりが壊されて本体が急所のダメージを受けたときは発動する。
- 急所に当たった攻撃の追加効果でこうげきのランク補正が低下した場合も、技の効果の後にいかりのつぼが発動してこうげきは最大まで上がる。
- クリアスモッグ/ダイドラグーンが急所に当たった場合、技の効果が発動した後にいかりのつぼが発動してこうげきは最大まで上がる。
- シャドースチールが急所に当たる場合でも、ランクを吸収された後に攻撃が行われるため、いかりのつぼでこうげきは最大まで上がる。
- 後攻のコアパニッシャーが急所に当たって特性が消される場合、いかりのつぼは発動しない。
- 特性かがくへんかガスのポケモンが使用したひんしになる攻撃技が急所に当たった場合、かがくへんかガスの解除後にいかりのつぼの発動判定があり、特性は発動する。
