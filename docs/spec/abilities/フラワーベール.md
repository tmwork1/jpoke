# 仕様書: フラワーベール

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%95%E3%83%A9%E3%83%AF%E3%83%BC%E3%83%99%E3%83%BC%E3%83%AB

## 効果
味方の場にいるすべてのくさタイプのポケモンは、他のポケモンからランク補正を下げられず、状態異常・ねむけにもされない。

## 特性の仕様
- 自身や隣接していない味方もくさタイプならば効果対象となる。
- もりののろいによりくさタイプが追加されたポケモンや、技/特性/テラスタルの効果でくさタイプに変わっているポケモンも対象となる。逆に、本来持っていたくさタイプを失っているポケモンには効果が無くなる。
- すでに下がったランクを元に戻す効果や、すでに状態異常になったポケモンを治す効果は無い。
- こんらん状態など、ねむけ以外の状態変化は防げない。
- ねむけ状態を防ぐことができるが、すでにねむけ状態になったポケモンがねむり状態になることは防げない。
- 味方からのランク低下や状態異常も防ぐ。
- 自発的にランクを下げたり状態異常になったりする技/特性/もちものに対しては発動しない。以下の効果はフラワーベールで防げない。
  - リーフストームなど自分のランク補正を下げる技、からをやぶるなど自身の能力を下げる変化技
  - くだけるよろい/ムラっけなど自身の特性によるランク低下
  - ねむるによるねむり状態
  - かえんだま/どくどくだまの効果
- 相手の特性によるランク低下や状態異常も防ぐ。ほのおのからだ/せいでんき/どくのトゲ/シンクロ/いかく/ぬめぬめ/カーリーヘアー/ミラーアーマー/わたげの効果を防ぐ。うのミサイルのダメージは受けるがランク低下や状態異常は防ぐ。
  - いかくをフラワーベールで防いだ場合、ビビリだまは発動する。ただし第八世代以降でもびびりは発動しない。
  - くさタイプの味方の特性がばんけんである場合、フラワーベールのポケモンと比べ補正抜きのすばやさが高い方の特性が発動する。
- くさタイプの味方の特性がミラーアーマーである場合、フラワーベールによる無効化が先に発動する。
- クリアチャームの効果より先にフラワーベールによる無効化が発動する。
- トーチカ/キングシールド/ブロッキング/スレッドトラップ/かえんのまもり状態のポケモンを直接攻撃したときに発動する効果も防ぐ。
- どくびし/ねばねばネット状態の効果も防ぐ。
  - くさ/どくタイプを複合して持つポケモンは、地面にいるならどくびしを除去することができる。
- くろいきり/クリアスモッグ/シャドースチール、特性きみょうなくすりの効果を受けた際のランクリセットは防げない。
- パワースワップ/ガードスワップ/ハートスワップ/ひっくりかえすを受けたときによるランク低下は防げない。
- バトンタッチにより下がったランクを引き継ぐことはある。
- あまのじゃくのくさポケモンの場合、あまのじゃくの効果込みでランクを下げられることを防ぐ。
- おきみやげ/きりばらい/ちからをすいとる/ハバネロエキスを受けた場合、ランク低下効果は防いげるが、それ以外の効果は発動する。
- すてゼリフの効果を防いだ場合、相手は第六世代では交代できるが、第七世代では交代できない。
- かたやぶりの効果がある技はフラワーベールを無視する。
  - かたやぶりのポケモンは味方のフラワーベールも無視するため、ランク低下や状態異常を起こす相手の特性が発動してもフラワーベールで防げない。
  - 特性を無視する技を使用した際も同様に味方のフラワーベールを無視する。第七世代では使用者自身がフラワーベールを持つ場合も無視する。第八世代以降では使用者自身のフラワーベールは無視しない。

## 備考
- くさタイプのポケモンにしか効果が無い特性だが、第九世代現在この特性を所有するくさタイプのポケモンはいない。
  - 第七世代まで、フラベベ系統はグラスフィールド状態でほごしょくを使うことでくさタイプになることはできる。
  - スカーレット・バイオレットではくさタイプにテラスタルすればフラワーベール所有者自身も効果を得られる。東1番エリアにはくさタイプにテラスタルする野生のフラエッテ (あかいはな) が出現する。
