# 仕様書: どくくぐつ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A9%E3%81%8F%E3%81%8F%E3%81%90%E3%81%A4

## 効果
この特性のポケモンが使用した技でどく・もうどく状態になったポケモンは、こんらん状態にもなる。

## 特性の仕様
- この特性はコピーできない。
- この特性に対するスキルスワップは失敗する。
- この特性を他の特性に上書きすることはできる。とくせいなし状態やかがくへんかガスにより無効される。
- モモワロウにへんしんしたポケモンが使用した技でもどくくぐつは発動する。
- 攻撃技と変化技のどちらを使用したときも特性は発動する。
- 場にどくくぐつのポケモンがいるときでも、どくびし状態やどくどくだまで自分からどく/もうどく状態になった相手や、この特性以外のポケモンの技でどく/もうどく状態になった相手は混乱状態にならない。
- この特性による混乱状態が発動したときはどくくぐつの特性バーが表示される。
  - 相手がすでに混乱状態である場合、どくくぐつの特性バーは表示されない。
  - 相手が特性マイペースで混乱状態を無効化する場合、どくくぐつとマイペースのどちらも特性バーも表示されない。
- 相手がラムのみ/モモンのみを持つ場合、どくくぐつの発動後にきのみが発動してどく/もうどく状態を治す。ラムのみならどくくぐつで発生した混乱状態も治せる。
- 相手の特性がシンクロである場合、お互いの特性の発動順は補正抜きのすばやさが高い順になる。
  - モモワロウがどく状態を無効化する場合、シンクロの特性バーの代わりに「モモワロウには 効果が ないようだ…」というメッセージが流れる。
  - モモワロウのタイプが変わっていたためシンクロのどく状態が有効になったとしても、それによりモモワロウが混乱状態になることはない。
  - シンクロ側の方がすばやさが低く、さらにラムのみを持っている場合、どくくぐつで混乱になった直後にラムのみが発動してシンクロ側のどくと混乱状態が治るが、このときもシンクロは発動する。
- どくガス/ヘドロウェーブなどを使用して1度に複数の相手をどく状態にした場合、1体ごとにどく状態と混乱状態の付与を行う。
- 変化技でどく/もうどくにした相手がりんぷん/おんみつマントを持つ場合でも混乱状態は発動する。

## 備考
- モモワロウの専用特性。
  - スカーレット・バイオレットでは、この特性はボックスの検索リストには表示されない。Pokémon HOMEでは検索できる。
