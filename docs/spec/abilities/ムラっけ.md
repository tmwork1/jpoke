# 仕様書: ムラっけ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%A0%E3%83%A9%E3%81%A3%E3%81%91

## 効果
ターン終了時、自分のこうげき・ぼうぎょ・とくこう・とくぼう・すばやさの内、いずれか1つのステータスが2段階上がり、別のいずれか1つのステータスが1段階下がる。
第七世代以前では、命中率・回避率も上下するステータスとして選ばれることがある。

## 特性の仕様
- すでに+6の能力を上げたり、-6の能力を下げたりする変化は選ばれない。
  - 上げる能力に+5の能力が選ばれ、結果1段階しか上がらないことは起こりうる。
- 下がる能力はそのターン上がらなかった能力から選ばれる。1ターンに同じ能力が上下することはない。
- 対象となる全能力が+6まで上がっている場合、能力が下がる効果だけ発動する。ムラっけだけで、対象となる能力を全て最大まで上げることはできない。
- 対象となる全能力が-6まで下がっている場合、能力が上がる効果だけ発動する。
- しろいきり/フラワーベール/クリアチャームの効果では、能力の下降を防ぐことはできない。
- 発動はターンの終了時となる (ターン#5.ターン終了時の処理も参照)。
  - 他のムラっけや、特性かそく/ナイトメア/スロースタートの解除、ものひろい/しゅうかく/たまひろい/はんすう、アイテムかえんだま/どくどくだま/くっつきバリとは、補正を考慮したすばやさが高い順に発動する。すばやさが変動するときでも、変動前のすばやさを比較する。
  - トリックルーム状態の効果も考慮する。トリックルーム状態が解除されるターンでは、トリックルームが解除された後にムラっけの発動判定があるため、トリックルームの影響は無い。
- 場に出てから一度も行動していないときでもムラっけは発動する (かそくの仕様とは異なる)。
  - 交代によりムラっけのポケモンを繰り出した場合や、使うと交代する技、相手を交代させる技、にげごし/ききかいひ、だっしゅつボタン/レッドカード/だっしゅつパックの効果でムラっけのポケモンが出てきた場合、そのターンからムラっけは発動する。
  - ひんしになったポケモンの代わりのポケモンを繰り出すタイミングは、ムラっけの発動タイミングの後になる。

## 備考
- 隠れ特性限定の特性となる。
