# 仕様書: ノーてんき

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%8E%E3%83%BC%E3%81%A6%E3%82%93%E3%81%8D

## 効果
このポケモンが場にいる間は、あらゆる天気の効果が無くなる。

## 特性の仕様
- この特性を持つポケモンが場に出たとき、第四世代まではメッセージは出ない。第五世代以降では「天気の影響が なくなった!」というメッセージが出る。
  - 他の場に出たときに発動する特性と異なり、ノーてんきとエアロックはスキルスワップで手に入れたときにメッセージは出ないが、このときも特性の効果は継続している。
  - かがくへんかガスが発動している間はノーてんきの効果は無くなる。かがくへんかガスが解除されるとノーてんきの効果は復活するが、ノーてんきの特性バーとメッセージは現れない。
- この特性を持つポケモンだけではなく、場にいるポケモン全員が天気の効果を受けなくなる。
- ノーてんきのポケモンが交代して場から去ったときや、特性を失ったときは、天気の効果は復活する。
- 効果を無くすだけであり、天気自体を止めることはできない。ノーてんきのポケモンが場にいても、技や他の特性の効果で天気自体を変更することはできる。
  - ノーてんきが発動している間でも、すでに発動している天気を発動させようとして失敗することはある。例えばあめ状態であまごいを使うと失敗する。
  - ノーてんきが発動している間でも、天気の経過ターン数はカウントされている。
- おおひでり/おおあめ/らんきりゅうの効果も無くなる。
  - ただし、ノーてんきの発動中でもこれらの天候をにほんばれ/あめ/あられ/すなあらし状態に変えることはできない。
- ノーてんきが発動している間は以下の特性も発動しなくなる。特記しない限り、ノーてんきのポケモンが場から去ればこれらの特性の効果は即座に復活する。
  - アイスボディ/あめうけざら/うるおいボディ/サンパワー/すいすい/すなかき/すながくれ/すなのちから/ゆきかき/ゆきがくれ/ようりょくそ/リーフガードは発動しなくなる。
  - かんそうはだはにほんばれ状態によるダメージや、あめ状態による回復が発動しなくなる。
  - しゅうかくはにほんばれ状態でもきのみを復活させる確率が50%になる。
  - てんきやは発動しなくなる。ノーてんきのポケモンが場に出たとき、フォルムチェンジしていたポワルンは通常の姿に戻る。
  - フラワーギフトは発動しなくなる。ノーてんきのポケモンが場に出たとき、ポジフォルムのチェリムはネガフォルムに戻る。
  - アイスフェイスは、ナイスフェイスからアイスフェイスに戻れなくなる。ノーてんきのポケモンが場からいなくなってもフォルムチェンジできない。
  - こだいかっせいは、場がにほんばれに変わっても能力を上げることができなくなる。
    - こだいかっせいが未発動の状態でノーてんきが解除された場合、場がにほんばれでも即座にこだいかっせいが発動することはない。次にポケモンが繰り出された後でこだいかっせいの発動判定がある。
    - こだいかっせいで能力が上がった後にノーてんきのポケモンが場に出ても、こだいかっせいによる能力の補正が消えることはない。
  - ひひいろのこどうは、場がにほんばれ状態の間も攻撃の補正を得られなくなる。
- あられが降っている状態でもオーロラベールが失敗するようになる。
- かたやぶりの効果がある技も天気の影響を受けなくなる。

## 備考
- 同様の効果を持つ特性としてエアロックがある。
- 似た効果を持つ特性としてゼロフォーミングがあるが、これはノーてんきと異なり天気自体を解除することができるが、代わりに特性を得た瞬間しか発動判定がない。
- 第四世代まではコダック系統の専用特性だった。通常特性として持っているのは第九世代現在もこの系統のみである。
  - なお、ソード・シールド発売時点では隠れ特性のジジーロン限定だった。他のこの特性を持つポケモンはエキスパンションパス配信後に解禁された。
- この特性の名前は「能天気」と「no」「天気」を掛けた洒落になっており、効果は「no」「天気」を意識したものになっている。
  - 「のうてんき」という性格もある。
  - 分類が「のうてんきポケモン」のルンパッパは、この特性を持たない。
