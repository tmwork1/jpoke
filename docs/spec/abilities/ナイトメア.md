# 仕様書: ナイトメア

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%8A%E3%82%A4%E3%83%88%E3%83%A1%E3%82%A2

## 効果
毎ターン終了時、相手の場にいるねむり状態のポケモンに対して、相手の最大HPの1/8分のダメージを与える (小数点以下切り捨て。ただし、切り捨てた結果0になる場合は1ダメージ)。

## 特性の仕様
- 特性マジックガードのポケモンに対してダメージは無い。
- 特性ぜったいねむりのポケモンに対してもダメージを与える。
- ナイトメア所有者が複数いる場合、ねむり状態の相手はその分複数回ダメージを受ける。
- トリプルバトルでは、所有者の対角にいるポケモンに対してダメージは無い。
- ダメージが発動するタイミングはターンの終わりとなる (ターン#5.ターン終了時の処理も参照)。
  - 交代でナイトメア所有者が出てきたターンからダメージは発動する。
  - ターンの途中でナイトメア所有者が場からいなくなった場合、そのターンにナイトメアのダメージは発動しない。
  - 天候・ねがいごと・たべのこし等の効果より後にナイトメアのダメージが発動する。
  - あくむ状態の相手は、あくむのダメージの後にナイトメアのダメージを受ける。
  - 特性だっぴ/うるおいボディ/いやしのこころの発動よりも後なので、これらでねむり状態を治したターンにダメージは無い。

### 第五世代以降
- ナイトメアのダメージは、ナイトメアの特性所有者の補正込みのすばやさが高い順に発動する。
- 相手の場にねむり状態のポケモンが複数いる場合、すばやさに関係なく相手から見て左側のポケモンからダメージを受ける。
- さしおさえ/マジックルーム状態の解除はナイトメアの前なので、これによりねむりを治すアイテムを発動させたターンにダメージは無い。
- ねむけ状態によるねむり状態の発動はナイトメアの前なので、ねむけでねむり状態になったターンもダメージを受ける。
- 相手の場のねむり状態のポケモンが特性ものひろい/しゅうかく/はんすうでねむりを治す道具の効果を発動させる場合、特性所有者のすばやさにより発動順が決まる。
