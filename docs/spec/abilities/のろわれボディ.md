# 仕様書: のろわれボディ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AE%E3%82%8D%E3%82%8F%E3%82%8C%E3%83%9C%E3%83%87%E3%82%A3

## 効果
他のポケモンから攻撃技によるダメージを受けたとき、30%の確率でその技をかなしばり状態にする。非直接攻撃に対しても発動する。

## 特性の仕様
- この特性で発動したかなしばり状態は、発動したターンを含めて4ターンの間続く。
- ダメージを受けなかったときは発動しない。
  - 変化技などダメージが無い技に対しては発動しない。いたみわけに対しても発動しない。
  - 相性/まもる状態などで攻撃を防いだときは発動しない。
  - みがわり状態で攻撃を防いだときは発動しない。
- みらいよち/はめつのねがいに対しては発動しない。
- 他の技が出る技で出た攻撃技を受けた場合、基本的に「出た技」ではなく「他の技が出る技」自体をかなしばりにする。
  - ただし、溜め技やあばれる/ころがる/がまんなど数ターン行動する技が出て、「他の技が出る技」を使用した次のターン以降にのろわれボディが発動する場合は、「他の技が出る技」ではなく「出た技」をかなしばりにする。他の技を経由することでかなしばりにされた技でも溜めることができるが、攻撃はかなしばり状態により失敗する。例えばオウムがえしで出たゴッドバードを次のターンに受けた場合、ゴッドバードをかなしばりにする。かなしばり状態の間でも、オウムがえしでゴッドバードを溜めることはできるが、攻撃時にかなしばりにより失敗する。
  - 溜め技や数ターン行動する技が出た場合でも、「他の技が出る技」を出したそのターンにのろわれボディが発動したときは「他の技が出る技」自体をかなしばりにする。パワフルハーブなどの効果で溜め技を1ターンで使用したときも同様。
- Zワザに対しても発動するが、基本的にかなしばりにする対象はZワザ自体であり、元の技をかなしばりにすることはできない。
  - ただし、Z変化わざで繰り出された攻撃技に対して発動した場合は、元の技をかなしばりにする。例えば、Zしぜんのちからで出たラブリースターインパクトを受けた場合、しぜんのちからをかなしばりにする。
  - ロトムずかんの効果で1試合中に2回Zワザが使えるようになった場合、かなしばりの対象になっているZワザでも使用することができる。
- ダイマックスわざに対しては発動しない。
  - マックスレイドバトルで通常の技を受けたときは、ダイマックスしたポケモンに対してものろわれボディが発動することがある。
- 連続攻撃技に対しては1発ごとに発動判定があるが、発動してもその技を中断させることはない。
- その攻撃で自身がひんしになったときでも発動する。
- 相手がみがわり状態でも発動する。
- 相手がすでに他の技をかなしばりにされてる場合は発動しない。
- かたやぶりの効果がある技を受けたときも発動する。
- 相手が特性アロマベールに守られているときは発動しない。
- アロマベールに守られているポケモンから、かたやぶりの効果がある攻撃を受けた場合、かたやぶりの効果で味方のアロマベールを無効にするためのろわれボディは発動する。

## 備考
- この特性を所有するポケモンは全てゴーストタイプを持つ。
