## 備考
- 第五世代においてのみ、マジックガードの下位互換の特性なのでランクルスはぼうじんを選ぶ理由が無い。
- アイテムぼうじんゴーグルと同様の効果を発揮する。
  - 特性と道具という違いがあるため、かたやぶりやかがくへんかガスはぼうじんのみを無効化し、ぼうじんゴーグルに効果は無い。さしおさえ・マジックルーム状態はぼうじんゴーグルを無効化するが、ぼうじんには効果が無い。
