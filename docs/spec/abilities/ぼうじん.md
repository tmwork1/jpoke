## 特性の仕様
- 天候ダメージを防ぐときに特性に関するメッセージは出ない。
- すなあらしによるいわタイプの特防上昇など、スリップダメージ以外の天候効果は防がない。
- ゆき状態には元々ダメージが無いため、実質的に効果が無い（第九世代仕様）。
- かたやぶりの効果がある粉技は防ぐことができない。
- 自身がくさタイプも持っている場合、粉技はくさタイプによる無効化とぼうじんによる無効化が重複するが、
  第七世代以降は特性による無効化が優先されるため特性発動のメッセージが表示される。

## 備考
- 第五世代においてのみ、マジックガードの下位互換の特性なのでランクルスはぼうじんを選ぶ理由が無い。
- アイテムぼうじんゴーグルと同様の効果を発揮する。
  - 特性と道具という違いがあるため、かたやぶりやかがくへんかガスはぼうじんのみを無効化し、ぼうじんゴーグルに効果は無い。さしおさえ・マジックルーム状態はぼうじんゴーグルを無効化するが、ぼうじんには効果が無い。
