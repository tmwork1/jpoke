# 仕様書: すいほう

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%99%E3%81%84%E3%81%BB%E3%81%86

## 効果
- ほのおタイプの攻撃技を受けるとき、相手のこうげき・とくこうが半減してダメージを計算する。
- みずタイプの攻撃技を使うとき、自分のこうげき・とくこうを2倍にしてダメージを計算する。
- やけど状態にならない。やけど状態になっても回復する。

## 特性の仕様
- ほのお技を受けるときの相性判定自体は変わらない。
- かたやぶりの効果が発動した技に無視され、ほのお技を半減することができない。
  - このときすいほうのポケモンもやけど状態にすることができるが、すいほうの効果により即座に回復する。同時にラムのみ/チーゴのみを持っていた場合、道具が優先して消費される。
  - すいほうのポケモンが、かたやぶりのポケモンに対してみず技で攻撃する場合は、すいほうの効果が無視されることはない。
- すでにやけど状態になっているポケモンの特性がすいほうに書き換わった場合もすいほうの効果によりやけどは即座に回復する。
- かえんだまを持っていても発動しない。
- みず技を使用したときの効果については、タイプや補正率が異なるがはがねつかいと似通った使用がある。発動条件の詳細ははがねつかい#特性の仕様を参照とする。
