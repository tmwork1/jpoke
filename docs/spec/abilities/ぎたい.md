# 仕様書: ぎたい

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%8E%E3%81%9F%E3%81%84

## 効果
フィールドが変化したとき、自身が以下の単タイプに変化する。フィールドが無いときは本来のタイプに変化する。
| フィールド | タイプ |
| --- | --- |
| エレキフィールド | でんき |
| グラスフィールド | くさ |
| ミストフィールド | フェアリー |
| サイコフィールド | エスパー |

## 特性の仕様
- 交代するか、ひんしになるか、戦闘が終了するとタイプは元に戻る。
- 自身が地面にいないときでも発動する。
- 特性が発動するタイミングは、フィールドが発生したとき、あるいは解除されたとき、フィールドが発生している場に繰り出されたとき、特性がぎたいに変わったときのいずれかになる。
  - 自身が場にいるときにフィールドが変化した場合は、即座にぎたいが発動する。
  - フィールドが発生している場に繰り出されたときは、設置技の効果や、多くの場に出たときに発動する特性が発動した後にぎたいが発動する (ターン#1.ポケモンを繰り出すも参照)。
    - ステルスロック/キョダイコウジン状態の場にマッギョを繰り出したときは、じめん・はがねタイプの相性に応じたダメージを受けた後にぎたいが発動する。
    - どくびし状態の場にマッギョを繰り出したときは、どくびしの効果がはがねタイプで無効化された後にぎたいが発動する。
    - ぎょぐんのポケモンと同時に場に出たときは、現在のすばやさが高い順に特性が発動する。
    - すばやさ関係によらず、シード系アイテムの発動よりぎたいの方が優先して発動する。
    - 特性の効果でフィールドが発生した場合は、他の処理より優先して即座にぎたいが発動する。
  - フィールドが発生しているときに他の特性からぎたいに変わると、即座にぎたいが発動してフィールドに合わせたタイプに変わる。フィールドが消えるとそのポケモン本来のタイプに戻る。
  - 現在のタイプとぎたいによる変化先のタイプが同じである場合、ぎたいは発動しない。
  - へんしん状態のポケモンの場合、フィールドが無い状態で変化するタイプは以下になる。
    - 第八世代では変身前の自身のタイプとなる。フィールドが無い状態で変身すると即座にタイプだけ変身前のものに戻ることになる。
    - Championsでは変身対象のポケモンのタイプとなる。
  - ぎたいの特性を失ったときは、フィールドに関係なくタイプはそのまま変化しない。
- フィールドが発生中に、ぎたいのポケモンがみずびたし/まほうのこなを受けてタイプを変えられた場合や、ミラータイプを使用してタイプを変えた場合でも、即座にフィールドに合わせたタイプに戻ることはない。
  - みずびたし/まほうのこなでタイプが変わった後にぎたいが発動した場合、みずびたし/まほうのこなの影響は消える。
  - フィールドが無い状態で、みずびたし/まほうのこななどでタイプが変わっていたポケモンの特性がぎたいになった場合、即座にぎたいが発動して本来のタイプに戻る。
- 特性へんげんじざい/リベロでタイプが変わっていたポケモンの特性がぎたいになった場合、ぎたいが発動するとタイプは上書きされる。
  - フィールドが無い状態で、特性がへんげんじざい/リベロからぎたいになった場合、即座にぎたいが発動して、場に出てきたときと同じそのポケモン本来のタイプになる。
- もりののろい/ハロウィンの効果はぎたいが発動しても解除されない。Championsでも同様。
  - もりののろい状態のときにグラスフィールド状態になりくさタイプに変化したとしても、くさタイプともりののろい状態が重複して存続する。
  - もりののろいかハロウィン状態のときにフィールドが解除されて本来のタイプに戻るとき、本来のタイプがくさタイプかゴーストタイプを持つ場合でも、そのタイプが重複して存続する。
  - ぎたいの効果により、本来のタイプと追加されたタイプが重複しているポケモンであっても、受けるダメージの相性補正はタイプを1つ持つポケモンと同様にして行われる。使用する技のタイプ一致ボーナスは1.5倍のままであり、2.25倍にはならない。
  - くさタイプともりののろい状態や、ゴーストタイプとハロウィン状態を重複して持つときでも、ミラータイプの効果はこれらをタイプ単体とはみなさない。
  - フィールドが無い状態で、もりののろい/ハロウィンでタイプが追加されていたポケモンの特性がぎたいになった場合では、即座にぎたいが発動することはない。
- かがくへんかガスの効果によりぎたいは発動できなくなる。ぎたいでタイプが変わっているときにかがくへんかガスの効果が発動してもタイプはそのままで元に戻ることは無く、フィールドが解除されても元のタイプに戻らない。かがくへんかガスの効果が切れると即座にフィールドに合わせたタイプに変化する。

## 備考
- マッギョ(ガラルのすがた)の専用特性。
- 分類が「ぎたいポケモン」のシャリタツはこの特性は持たない。
