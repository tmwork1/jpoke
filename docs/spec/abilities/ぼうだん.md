## 特性の仕様
- くちばしキャノンを溜めたポケモンに直接攻撃を行ったことによるやけどの効果は防げない。
- 味方のポケモンが使う、回復効果のかふんだんごも無効化する。
- じばくとだいばくはつは無効化したり威力を下げて受けることはできない。

## 備考
- 第六世代ではブリガロン系統の専用特性で、なおかつ隠れ特性限定だった。
- 技名に「ボール」「ばくだん」「ボム」の単語が含まれている技は全てが弾の技となる。それ以外の単語はあまり弾の技の分類と紐づいていない。説明文やエフェクトから、ビーム系の技であることが分かる技は弾の技には含まれないことが多い。
