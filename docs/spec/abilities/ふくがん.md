# 仕様書: ふくがん

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B5%E3%81%8F%E3%81%8C%E3%82%93

## 効果
### バトルでの効果
使う技の命中率が1.3倍になる。

### フィールドでの効果
手持ちの先頭にこのとくせいを持つポケモンがいると、もちものを持った野生ポケモンが出やすくなる。ふくがんのポケモンがひんしであっても効果がある (エメラルドから第八世代まで)。
- 50%→60%、5%→20%、1%→5%

## 特性の仕様
### バトルでの効果
- 詳細な倍率は5325/4096倍となる。
- 一撃必殺技には適用されない。

### フィールドでの効果
- 乱入バトルでは、乱入時点で場に出ていないと効果が発揮されない。
- スカーレット・バイオレットでは道具を持ったポケモンが出やすくなる効果は無い。

## 備考
- 第八世代のみ、フィールドで同じ効果を持つ特性にきょううんがある。
- この特性を所有するポケモンは全てむしタイプを持つ。
