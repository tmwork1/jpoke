# 仕様書: はりきり

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AF%E3%82%8A%E3%81%8D%E3%82%8A

## 効果
### バトルでの効果
物理技を使用するとき、自分のこうげきを1.5倍にしてダメージを計算するが、命中率が0.8倍に下がる。

### フィールドでの効果
手持ちの先頭にこの特性を持つポケモンがいると、野生ポケモンのレベルが出現可能な中で最も高い個体になる確率が50%になる（エメラルドから第七世代・ブリリアントダイヤモンド・シャイニングパール）。ひんし状態でも効果がある。

## 特性の仕様
### バトルでの効果
- 第五世代以降での命中の詳細な補正率は3277/4096倍となる。
- 特殊技/変化技を使用したときは発動しない。
- 一撃必殺技の命中率は下がらない。
- 一撃必殺技を除く物理攻撃のダメージ固定技 (カウンターなど) は、特性の効果でダメージが増えることが無いももの、命中率は下がる。
- 必中技の命中率は下がらない。
- 相手の特性ノーガード、ロックオン/テレキネシス/ちいさくなる状態の効果で必中になった物理技の命中率は下がらない。
- はりきりの効果で技が命中しなかったときも、アイテムからぶりほけんは発動する。
- こんらんで自傷ダメージを受けるときは、第四世代までは特性の効果で攻撃が1.5倍になるが回避はできない。第五世代以降では特性の効果は無い。
- 自分がイカサマを使用するときは、相手の攻撃を1.5倍にしてダメージを計算する。
- 相手からイカサマを受けるときは、特性の効果は無い。
- 自分がボディプレスを使用するときは、自分の防御を1.5倍にしてダメージを計算する。
- パワートリック状態で物理技を使用するときは、攻撃の代わりに自分の防御を1.5倍にしてダメージを計算する。
- 分類が変わる技を使用するときは、分類の判定には特性の効果は影響しない。技が物理技に決まった場合は特性の効果が発動し、攻撃は1.5倍で命中率が0.8倍になる。

### フィールドでの効果
- 第四世代以前・ブリリアントダイヤモンド・シャイニングパールでは釣りによるエンカウントにも効果がある。
- 第五世代から第七世代では釣りによるエンカウントには効果が無い。
- ソード・シールド・スカーレット・バイオレットでは、レベルの高い野生ポケモンと出会いやすくなる効果は無くなった。

## 備考
- バトルの効果ではちからもち/ヨガパワーの下位互換といえる。
- フィールド上で同様の効果のあるとくせいとして、プレッシャー/やるきがある。
