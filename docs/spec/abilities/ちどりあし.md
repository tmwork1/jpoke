# 仕様書: ちどりあし

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A1%E3%81%A9%E3%82%8A%E3%81%82%E3%81%97

## 効果
このポケモンがこんらん状態の間、自分が受ける技の命中率が半減する。

## 特性の仕様
- こんらん状態、およびそれによる自傷は防げない。
- この特性による回避率上昇はランク補正によるものではない。
  - こんらんが治ると回避率は元に戻る。
  - 攻撃側の特性がてんねん/するどいめ6-/はっこう9-/しんがんである場合でも技を避けることができる。
- かたやぶり系の効果がある技に対しては発動しない。

## 備考
- ポケモンの本編ゲームにおいて、命中率が半減する効果は、回避率が2倍になる効果と同義となる。命中も参照。
