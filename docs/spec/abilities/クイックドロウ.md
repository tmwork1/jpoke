# 仕様書: クイックドロウ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%AF%E3%82%A4%E3%83%83%E3%82%AF%E3%83%89%E3%83%AD%E3%82%A6

## 効果
攻撃技を選択したとき、30%の確率で行動順が早くなる。優先度は変化しない。

## 特性の仕様
- 発動すると「<ポケモン>は クイックドロウで 行動が はやくなった!」というアナウンスが流れる。アナウンスが流れるタイミングは交代/ダイマックス/テラスタルより前となる。
  - 複数のすばやさが異なるポケモンが、同時にクイックドロウやせんせいのツメ/イバンのみを発動させた場合、アナウンスは行動順に流れる。同じ優先度の技を選択していればすばやさの高い方から、異なる優先度の技を選択していれば優先度が高い方から行動する。
  - 複数のすばやさが同じポケモンが、同時にクイックドロウ/せんせいのツメ/イバンのみを発動させた場合、アナウンスの順番と行動順は必ずしも一致しない。
- この特性のポケモンがせんせいのツメを所持している場合、クイックドロウとせんせいのツメの効果は累積し、44%の確率で行動順が早くなる。クイックドロウとせんせいのツメが同時に発動するときは、クイックドロウのアナウンスのみが流れる。よってこのとき、せんせいのツメのアナウンスが流れる確率は14%になる。
- クイックドロウが発動したポケモンは、HPが1/4以下のときでもイバンのみを発動させない。HPが1/4以下で、かつクイックドロウが発動しなかったときにイバンのみが発動する。
- クイックドロウが発動したポケモンが持つこうこうのしっぽ/まんぷくおこうの効果は無視される。このとき他にクイックドロウ/せんせいのツメを発動させたポケモンがいた場合、素早さが高いポケモンから動く。
- かがくへんかガスが発動している間はクイックドロウは発動しない。
  - かがくへんかガスのポケモンが交代で引っ込むターンにクイックドロウは発動しない。
  - クイックドロウが発動したターンに、他のポケモンが交代してかがくへんかガスのポケモンが出てきた場合、クイックドロウは無効化されずに、早く行動することができる。
- わるあがきを使うときも、攻撃技を選択したときと同様の扱いとなり、クイックドロウは発動することがある。
- 優先度は変わらないため、クイックドロウの効果で先制した行動はじょおうのいげん系の特性や、ファストガード/サイコフィールドの効果では防がれない。
- トリックルーム状態でも行動順が早くなる。
- 行動順の判定についてはターンも参照。

## 備考
- ヤドラン(ガラルのすがた)の専用特性。特性くいしんぼうのヤドン(ガラルのすがた)が進化するとこの特性になる。
- クイックドロウ(quick draw)とは早撃ちの意味。
- 似た効果を持つせんせいのツメの英語名Quick Clawとは語感が似ている。
