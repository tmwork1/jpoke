# 仕様書: ふしぎなまもり

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%B5%E3%81%97%E3%81%8E%E3%81%AA%E3%81%BE%E3%82%82%E3%82%8A

## 効果
効果バツグンではないタイプの攻撃技によるダメージを無効化する。

## 特性の仕様
### 防げる技について
- 変化技は無効化できない。その技のタイプとの相性は関係しない。
  - 相性の影響を受ける変化技のでんじはであっても、ふしぎなまもりで無効化することはできない。
- 攻撃技であれば、ダメージ固定技/一撃必殺技であっても効果抜群ではないタイプの技なら無効化できる。効果抜群になるタイプの技は無効化できない。
- 状態異常/状態変化/場の状態によるダメージや、特性によるダメージなど、攻撃技以外によるダメージは受ける。
  - 自分のアイテムいのちのたま/くろいヘドロ/くっつきバリや、相手のアイテムゴツゴツメット/ジャポのみ/レンブのみによるダメージも受ける。
  - 反動技や、いのちがけなどひんしになる技を使用したことによるダメージも受ける。
  - ニードルガードによるダメージも受ける。
  - はじけるほのおの火花よるダメージも受ける。
- こんらんによる自傷ダメージは無効化できない。
- わるあがきは無効化できない。
- かたやぶりの効果が発動した技、特性を無視する攻撃技によりふしぎなまもりは無視され、抜群でなくてもダメージを受ける。
- 特性所有者のタイプが変わった場合や、さかさバトルなどで相性判定そのものが変わった場合は、それに応じてふしぎなまもりで防げるタイプも変化する。
  - もえつきるなどによりタイプなしになった場合、わるあがきとかたやぶり効果がある技以外の全ての攻撃技を無効化する。
  - ひこうタイプである場合、らんきりゅうの効果で等倍になったひこうタイプの弱点技は無効化する。
  - みずタイプである場合、抜群になったフリーズドライのダメージは受ける。
  - タールショット状態によりほのおタイプが弱点になった場合、ほのお技を無効化できなくなる。
  - じめんタイプを弱点とするタイプになった場合、じめん技を無効化できなくなるが、ふうせんやでんじふゆうにより地面にいない状態になると、まるでふしぎなまもりの効果で防いだかのように特性バーを表示しながらじめん技を無効化する。
  - 抜群技に対してオッカのみなどの半減系きのみが発動するときも相性自体は変わらないため、特性で防ぐことはできない。
- ポケモンコロシアムやポケモンXDに登場するダークわざはコロシアムではタイプがなく、XDではダークポケモン以外に効果は抜群であるため無効化できない。
- 第四世代のみ、効果抜群にならなくともほのおのキバのダメージは受ける。
- 第四世代までのふくろだたき/みらいよち/はめつのねがいは無効化できない。第五世代以降は効果抜群であれば無効化できないが、効果抜群でなければ無効化する。
- タイプなしのポケモンが使用するめざめるダンスは無効化できる。

### 特性の変更
- この特性を上書きすることはできる。
  - なやみのタネ/シンプルビーム/なかまづくり/ミイラにより特性は上書きされる。かがくへんかガスにより特性は無効化される。
  - かがくへんかガスのポケモンがひんしになる技を使用したときも、その攻撃に対してふしぎなまもりは発動しない。
  - いえきを受けるととくせいなし状態になる。
  - コアパニッシャーを抜群で受けて技の効果が発動すると特性を消される。しかし特性の効果で攻撃を無効にしたときはとくせいなしにもならない。
  - 特性所有者が使うなりきりは成功し、ふしぎなまもりを他の特性に上書きできる。
- この特性はスキルスワップ/さまようたましいで交換することができない。
  - 特性所有者がとくせいなし状態のときや、かがくへんかガスが発動している間でも入れ替えはできない。
  - かたやぶりの効果があるスキルスワップでも特性を入れ替えることはできない。
  - 特性所有者が使うスキルスワップも失敗する。
- この特性をコピーする効果は、手段によって成否が異なる。
  - 以下の方法でコピーすることはできる。
    - この特性と対面した相手は、特性トレースでコピーすることができる。
    - 特性所有者が使うなかまづくりで相手に特性をコピーすることも可能となる。
  - 以下の方法でコピーすることはできない。
    - この特性に対するなりきりは失敗する。
    - この特性に対してレシーバー/かがくのちからの効果は発動しない。

## 備考
- ヌケニンの専用特性。
- 以下の条件でヌケニン以外のポケモンにこの特性を持たせることはできる。
1. 特性トレースを持つポケモンなら、相手の場にいるヌケニンのふしぎなまもりをコピーできる。
2. トレースをスキルスワップで他のポケモンに移し、移した相手がトレースでふしぎなまもりをコピーすれば、スキルスワップが有効な任意のポケモンの特性をふしぎなまもりにできる。以下第五世代まで弱点が1つも無かったミカルゲやヤミラミに持たせることを考える。第五世代でのみトレースをスキルスワップで移しても発動しないため、第四世代か第六世代以降である必要がある。
  - 「いえきが使えるポケモン」「バトンタッチが使えるポケモン」「特性トレースを持ち、スキルスワップが使えるポケモン(以下サーナイトとする)」の3体がミカルゲ(ヤミラミ)自身の他に自分の手持ちにいること、ダブルバトル以上であり、相手がヌケニンを場に出していること。
  - まずいえき持ちとバトンタッチ持ちを出し、いえきをバトンタッチ持ちに使い、とくせいなし状態にする。
  - バトンタッチ持ちはとくせいなし状態のままサーナイトにバトンタッチする(こうする事で相手の特性をトレースしないまま場に出られる)。同時にいえき持ちをミカルゲ(ヤミラミ)と交代させる。
  - サーナイトはミカルゲ(ヤミラミ)を対象にしてスキルスワップをする。そうするとミカルゲ(ヤミラミ)が特性トレースを得る。
  - 自動的に、とくせいなし状態でないミカルゲ(ヤミラミ)は特性トレースの効果を発揮し、相手のヌケニンのふしぎなまもりをトレースする。相手の場にヌケニン以外のポケモンがいても、そのポケモンがトレースできない特性であれば確実にふしぎなまもりをトレースできる。
3. 第三世代の教え技でものまねを覚えたヌケニンを第五世代以降に連れて来て、他のポケモンが使ったなかまづくりを真似すれば、なかまづくりが有効な任意のポケモンの特性をふしぎなまもりにできる。
- ポケウッドのゴーストイレイザー4にはゴースト・あくタイプで特性がふしぎなまもりのセットが登場するが、こちらはデスカーンのミイラかオクタンのみずびたしで特性かタイプを変えて対処するよう台本にも書かれている。
