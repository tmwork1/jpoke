# 仕様書: フレンドガード

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%95%E3%83%AC%E3%83%B3%E3%83%89%E3%82%AC%E3%83%BC%E3%83%89

## 効果
自分以外の味方ポケモンが受ける攻撃技のダメージが3/4倍になる。

## 特性の仕様
- 味方から受けた攻撃技によるダメージも減らせる。
- ダメージ固定技によるダメージは減らない。
- こんらんによる自傷ダメージは減らない。
- トリプルバトルで自分に隣接していない味方へのダメージも減る。
- トリプルバトル/マックスレイドバトル/テラレイドバトルでフレンドガードのポケモンが2体以上いる場合、他の味方にはフレンドガードの効果が累積してかかる。2体のフレンドガードの効果を受けるとダメージは9/16倍(=56.25%)に、3体のフレンドガードの効果を受けるとダメージは27/64倍(=42.1875%)になる。

## 備考
- 第八世代までは隠れ特性限定の特性だった。
- 自身へのダメージは減らせないので、シングルバトルで発動することはない。
