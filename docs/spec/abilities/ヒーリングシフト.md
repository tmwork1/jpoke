## 特性の仕様
- 以下のわざには適用されない。いずれも、出してすぐに回復させるわざでは無く、それぞれの状態変化にさせるわざであるため。
  - アクアリング
  - グラスフィールド
  - ねをはる
  - やどりぎのタネ
- いたみわけは回復技とは見なされずに適用されない。
- プレゼント/かふんだんごで回復効果が出る場合も、回復技とは見なされず適用されない。
- ヒーリングシフト対象外の攻撃技は、かいがらのすずを持って使用しても特性は適用されない。
- Z変化わざによる回復効果を持つ技にも適用されない、
- キョダイダンエンには適用されない。
- この特性でHPを吸収する攻撃技の優先度を上げて使用するターンでは、行動前に受けるはやてがえしが成功する。
  - 特性の効果があっても、変化技に対してはやてがえしは成功しない。
