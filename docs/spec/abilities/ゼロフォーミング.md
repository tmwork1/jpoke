# 仕様書: ゼロフォーミング

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%BC%E3%83%AD%E3%83%95%E3%82%A9%E3%83%BC%E3%83%9F%E3%83%B3%E3%82%B0

## 効果
テラパゴスがテラスタルしてステラフォルムになったとき、場の天候とフィールドを消滅させる。

## 特性の仕様
- マップの効果で自然に発生している天候は解除することができない。
- この特性はコピーできない。この特性に対するスキルスワップは失敗する。
- この特性を他の特性に上書きすることはできる。この特性はかがくへんかガスによって無効化される。
- この特性は、テラパゴスをテラスタルさせた瞬間にのみ発動判定がある。
  - テラスタルの発動後なら、この特性のポケモンが場にいる間でも天候とフィールドを発生させることはできる。
  - テラスタル時に天候とフィールドが無かったため特性が発動しなかった場合でも、一度引っ込めて再度場に繰り出したときに発動することはない。
  - かがくへんかガスの発動中にテラパゴスがテラスタルした場合、特性の発動機会は失われる。ガスが解除された後もゼロフォーミングを発動させることはできない。
- ノーてんき/エアロックで天候の効果が無効化されているときでも天候を解除できる。
- ふうせんなどで地面にいないときでもフィールドを解除できる。

## 備考
- テラパゴス (ステラフォルム) の専用特性。
- この特性と同様に天候メタの特性としてノーてんき/エアロックがあるが、これらは特性所有者が場にいる限り効果のみを打ち消して場の状態自体は消滅させないのに対し、ゼロフォーミングは1度きり場の状態自体を消滅させる効果となる。
- 天候をターン経過や上書き以外で解除できる初めての手段となる。
- フィールドを解除する技はいくつかあるが、フィールドを解除できる特性としては初となる。
