# 仕様書: じしんかじょう

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%98%E3%81%97%E3%82%93%E3%81%8B%E3%81%98%E3%82%87%E3%81%86

## 効果
攻撃技で他のポケモンをひんしにさせたとき、自分のこうげきが1段階上がる。

## 特性の仕様
- すでに攻撃が最大ランクまで上がっているときは発動しない。
- 味方を倒した場合でも発動する。
- 複数匹をまとめて倒した場合、倒した数だけ攻撃が上がる。
  - 第五世代では複数回特性が発動し、第六世代以降では1回にまとめて発動する。最終的に上がる量自体は世代間で変化していない。
- 相手が状態異常/天気/反動ダメージ/ゴツゴツメットなど、特性所有者の攻撃技によるダメージ以外で倒れたときは発動しない。
  - ただし、特性ばけのかわをもつポケモンを攻撃し、ばけのかわが剥がれた時のダメージで倒れた場合、じしんかじょうは発動する。
- ハサミギロチンなどの一撃必殺技で倒した場合は第五世代・第六世代では発動せず、第七世代以降では発動する。
- ちきゅうなげなど、その他のダメージ固定技で倒したときは第五世代から一貫して発動する。
- 第五世代のみ、倒せばバトルが終了する段階でも1段階上がる。第六世代以降はそのようなときにじしんかじょうは発動しない。
- 特性ミイラ/さまようたましい/とれないにおいのポケモンを直接攻撃で倒した場合、ミイラ/さまようたましい/とれないにおいへの変化が優先されこうげきは上がらない。
- 特性かがくのちから/レシーバーを持つ味方の攻撃技によりじしんかじょうのポケモンが倒された場合、特性は引き継がれ即座に能力が上がる。
- 特性かがくへんかガスのポケモンを倒したことでガスの効果が切れた場合、じしんかじょうの効果は発動でき、攻撃が上がる。

## 備考
- 同様の効果を持つ特性にしろのいななきがある。同じ発動条件でとくこうが上がる特性にくろのいななきがある。同じ発動条件だが自分のステータスにより上がる能力が変わる特性にビーストブーストがある。
