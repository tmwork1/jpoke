# 仕様書: ちくでん

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%A1%E3%81%8F%E3%81%A7%E3%82%93

## 効果
でんきタイプの技を受けても無効化し、最大HPの1/4だけ回復する (小数点以下切り捨て)。

## 特性の仕様
- でんきタイプの変化技は、第三世代のみ無効化できない。第四世代以降は無効化できる。
  - ちくでんのポケモンが同時にマジックコート状態だった場合、でんきタイプの変化技は跳ね返し回復はしない。
  - マジックコート状態で跳ね返されたでんきタイプの変化技をちくでんのポケモンが受けた場合は回復する。
  - 第四世代以降ででんじはなどの状態異常にするでんきタイプの変化技を受けた場合、しんぴのまもり/ミストフィールド状態・特性フラワーベールで無効化するより先にちくでんが発動してHPを回復する。第六世代以降では、でんきタイプの性質でまひを無効化するより先にちくでんが発動してHPを回復する。
- じゅうでんなど、使用者自身を対象にする技を使用しても発動しない。
- エレキフィールドなど、場を対象にする技に対しては発動しない。
- 技や特性、そうでん/プラズマシャワー状態の効果でタイプが変わった技を受けた場合、基本的に変化後のタイプがでんきタイプであれば発動する。
  - 例外的に、そうでん等ででんき技になったアロマミスト/コーチング/たがやす/デコレーション/とおぼえ8-を受けても特性は発動しない。
- でんきタイプの連続攻撃技も無効化するが、回復は1回のみ発動する。
- みがわり状態ででんき技を受けたときでも発動してHPを回復する。
- じめんタイプのポケモンがこの特性を持っている場合、第六世代までじめんタイプによる無効化が優先されて、でんき技を受けてもHPは回復しない。第七世代以降は特性が優先され、でんき技で回復できる。
- まもる状態などででんき技を防いだときは発動しない。
  - 特性じょおうのいげん/ビビッドボディ/テイルアーマー、サイコフィールド/ファストガード/ワイドガード/トリックガード/たたみがえし状態などで防いだときも同様にして発動しない。
  - 第七世代以降で、いたずらごころで優先度が上がったでんき技があくタイプに無効化される場合、ちくでんが先に発動して回復する。
- じんらいの対象にされた場合でも、技の仕様で失敗した場合ちくでんは発動しない。
- じばそうさを受けたときは、特性不一致による技の失敗が優先され、ちくでんは発動しない。
  - そうでん状態のアシストギアを受けたときも、同様にして特性は発動しない。
- かたやぶり効果があるでんき技に対してちくでんは発動しない。
- ちくでんででんき技を無効化したときに、アイテムのじゅうでんちは発動しない。
- 第四世代までは命中判定の後にちくでんの判定があるため、でんき技を回避しちくでんが発動しないことがある。第五世代以降では命中判定より前にちくでんが発動するため、技を回避してちくでんが発動しなくなることはない。
  - ただし第五世代以降でも姿を隠しているため技が当たらない場合、ちくでんは発動しない。
- でんき技を無効化した際は特性の発動を示すメッセージが流れる。
  - 第四世代では、HPが減っている状態なら「<ポケモン名>は ちくでんで かいふくした!」、HPが満タンの状態で発動すると「<ポケモン名>の ちくでんで <技名>は こうかが なかった!」とメッセージが出る。
  - 第五世代以降ではちくでんの特性バーが出現し、さらにHPが減っている状態なら「<ポケモン>の 体力が 回復した!」、HPが満タンの状態なら「<ポケモン>には 効果が ないようだ...」というメッセージが出る。
- かいふくふうじ状態でもでんき技の無効化はできる。第四世代のみ回復はできるが、第五世代以降では回復もできない。

## 備考
- 第九世代現在、この特性を持つポケモンは全てでんきタイプを持つ。
