# 仕様書: ねつこうかん

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%AD%E3%81%A4%E3%81%93%E3%81%86%E3%81%8B%E3%82%93

## 効果
- ほのおタイプの攻撃技でダメージを受けたとき、自分のこうげきが1段階上がる。
- やけど状態にならない。やけど状態になっても回復する。

## 特性の仕様
- ほのお技を無効化したり、ダメージを軽減する効果は無い。
- ほのおタイプの攻撃技を受けても、ダメージが無かったときは発動しない。
  - まもる状態などで技を無効化したときは発動しない。
  - みがわり状態で攻撃を防いだときは発動しない。
- おにびを受けても無効化するが、その際攻撃は上昇しない。
- すでに攻撃が最大まで上がっているときは、ほのおタイプの攻撃技を受けても発動しない。
- 攻撃技でひんしになったときは発動しない。
- タイプが変わる攻撃技を受けた場合は、実際に出されたタイプがほのおタイプであれば発動する。
- かたやぶりの効果がある技を受けたときは、特性の効果は2つとも発動しない。
  - かたやぶりの効果があるほのおタイプの攻撃技を受けても攻撃は上がらない。
  - かたやぶりの効果がある技によりやけど状態にされることはあるが、即座に回復する。

## 備考
- セビエ系統の専用特性。
