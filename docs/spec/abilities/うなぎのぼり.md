# 仕様書: うなぎのぼり

調査日: 2026-07-10
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%81%86%E3%81%AA%E3%81%8E%E3%81%AE%E3%81%BC%E3%82%8A

## 効果
1. この特性のポケモンは地面にいない状態となる。
   - じめんタイプの攻撃技を無効化する。
   - まきびし・どくびし・ねばねばネット状態による効果を受けない。
   - 特性ありじごくの影響を受けない。
   - フィールドによる効果をほとんど受けない (一部の効果は地面にいなくても受ける)。
2. 攻撃技で他のポケモンをひんしにさせたとき、自分の最もステータスが高い能力のランクが1段階上がる。
   最も高い能力が複数ある場合は、こうげき・ぼうぎょ・とくこう・とくぼう・すばやさの順で最初にあるものが選ばれる。

## Champions説明文
地面にいないことになり じめんタイプのわざ まきびし どくびし ねばねばネット状態が効かない。攻撃で相手を倒すと 自分の最も高い能力が1段階上がる。

## 特性の仕様
効果1・効果2とも、それぞれ特性ふゆう・特性ビーストブーストと同一の仕様を持つ（詳細は各仕様書を参照）。
- 効果1（地面にいない扱い）
  - じめんタイプの技であっても、サウザンアローなど一部の技は防げない。
  - うちおとす/ねをはる状態、じゅうりょく状態、くろいてっきゅう所持、とくせいなし状態のときはこの特性は無効化され、地面にいる扱いとなる。
  - かたやぶりの効果がある技を受けるときは、地面にいるポケモンとして扱われる。
- 効果2（撃破時のランク上昇）
  - 上がる能力にHPは考慮しない。
  - 参照する数値は実数値であり、ランク補正/もちもの/味方のとくせい/すなあらしの効果は考慮しない。
  - 上げる能力がすでに最大ランクの場合、この特性は発動しない。2番目に高い能力が代わりに上がることはない。
  - 発動条件（相手の状態異常/天気/反動ダメージ等では発動しない等）はじしんかじょうと同様。

## 備考
- メガシビルドンの専用特性。Championsで新たに追加された特性である。
- ふゆうとビーストブーストを複合したような効果を持つ。
- Championsの仕様では個体値による能力差をつけられないため、上げられる能力に制限がある
  （メガシビルドンの実数値は攻撃・特攻が同点で最も高くなりやすく、素早さ努力値を攻撃/特攻に回すことで
  片方を確定で高くする調整が可能）。
