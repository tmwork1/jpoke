# 仕様書: クリアボディ

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%82%AF%E3%83%AA%E3%82%A2%E3%83%9C%E3%83%87%E3%82%A3

## 効果
他のポケモンからランク補正を下げられなくなる。

## 特性の仕様
- 自分の能力を下げる攻撃技や、のろい/からをやぶる/じこあんじなどを使い、自分の技の効果でランクが下がるときは発動しない。
- 自身のアイテムルームサービスが発動したことによる素早さ低下は防げない。他のポケモンが使用したトリックルームで発動したとしても同様に防げない。
- 味方からランクを下げられようとしたときは防げる。
- 交代したときやひんしになったときは、上がっていたランクもリセットされる。
- くろいきり/クリアスモッグ/シャドースチール、特性きみょうなくすりの効果を受けた際のランクリセットは防げない。
- パワースワップ/ガードスワップ/ハートスワップ/ひっくりかえすによりランクが下がることはある。
- バトンタッチにより下がったランクが引き継がれることはある。
- 上がった防御/特防ランクは、急所の効果や、特性てんねんの効果によって無視される。
- かたやぶりの効果がある技には無視されてランクを下げられる。
- キングシールド/ブロッキング/スレッドトラップ状態のポケモンを直接攻撃したときのランク低下も防ぐ。
- ダイマックスわざの効果によるランク低下も防ぐ。
- おきみやげ/きりばらい/ちからをすいとる/ハバネロエキスを受けたときは、ランクを下げられる効果だけ防ぎ、それ以外の技の効果は発動する。
  - かがくへんかガスのポケモンが使うおきみやげを受けた場合、能力を下げられた後に相手がひんしになってクリアボディの効果が戻るため、ランク低下は防げない。
- すてゼリフを受けてもランクは下げられない。このとき相手は第六世代のみ交代はできるが、第七世代以降は交代もできない。
- 特性いかく/カーリーヘアー/かんろなミツ/ぬめぬめ/ミラーアーマー/わたげによるランク低下も防ぐ。
  - うのミサイルの効果はうのみのすがたによるぼうぎょ低下のみ防げる。
  - いかくをクリアボディで防いだときもビビリだまは発動する。
- ねばねばネット/たこがため/あめまみれ状態によるランク低下も防ぐ。
- マックスレイドバトルのレイドボスの場合、ふしぎなバリアが破られたときのランク低下も防ぐ。
- 低下したランクを元に戻す効果は無い。他の特性のポケモンが、ランクを下げられた後に特性がクリアボディに変わったときもランクは戻らない。
- ランク補正によらない能力の低下効果は防げない。
  - まひ/しつげん状態によるすばやさ低下は防げない。
  - くろいてっきゅう/パワー系アイテムによるすばやさ低下は防げない。
  - パワーシェア/ガードシェア/スピードスワップによる能力値の変更は防げない。
  - わざわいのうつわ/わざわいのおふだ/わざわいのたま/わざわいのつるぎによる能力値の低下は防げない。
- 能力を下げる変化技や特性の効果をクリアボディで防いだとき、第四世代までは「<ポケモン>は クリアボディで のうりょくが さがらない!」、第五世代からは特性バーと「<ポケモン>の 能力は 下がらない!」というメッセージが流れる。
  - 場がしろいきり状態である場合、しろいきり状態による無効化が優先され、クリアボディに関するメッセージは流れない。
  - クリアチャームを持っている場合、クリアボディによる無効化が優先され、クリアチャームに関するメッセージは流れない。
  - 攻撃技の追加効果やダイマックスわざによるランク低下を防いだときはクリアボディに関するメッセージは流れない。
  - いかく/カーリーヘアー/かんろなミツ/ぬめぬめ/ミラーアーマー/わたげの効果を防ぐときはメッセージが流れる。
  - うのミサイルによるランク低下を防いだときにメッセージは流れない。
  - キングシールド/ブロッキング/スレッドトラップ状態や、あめまみれ状態の効果を防ぐときはメッセージが流れる。
  - ねばねばネット/たこがため状態や、ふしぎなバリアが破られたときのランク低下を防ぐときにメッセージは流れない。

## 備考
- 同様の効果がある特性としてしろいけむり/メタルプロテクトがある。
  - しろいけむりのみフィールド上での効果がある。
  - メタルプロテクトのみかたやぶりの効果で無視することができない。
- 同様の効果があるアイテムとしてクリアチャームがある。
