# 仕様書: ファーコート

調査日: 2026-06-03
参照URL: https://wiki.xn--rckteqa2e.com/wiki/%E3%83%95%E3%82%A1%E3%83%BC%E3%82%B3%E3%83%BC%E3%83%88

## 効果
攻撃技を受けるとき、自分のぼうぎょを2倍にしてダメージを計算する。

## 特性の仕様
- こんらんで自傷ダメージを受けるときは効果が無い。
- サイコショック/サイコブレイク/しんぴのつるぎを受けるときも防御を2倍にしてダメージ計算をする。
- シェルアームズを受けるとき、分類の判定にファーコートの効果は影響しない。技が物理技に決まった場合は防御を2倍にしてダメージ計算をする。
- ボディプレスを使用するときは特性の効果は発動せず、自分の防御を2倍にすることは無い。
- 場がワンダールーム状態のときは、特防が2倍になる効果に変わる。

## 備考
- 第六世代ではトリミアンの専用特性だった。
- ぼうぎょではなくこうげきを2倍にする特性としてちからもち/ヨガパワーがある。
- ファーコートの特殊版のような特性としてこおりのりんぷんがあるが、とくぼうが2倍になる特性ではなく特殊技のダメージを半減する特性になる。
