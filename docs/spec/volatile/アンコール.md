# アンコール（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | アンコール（状態変化） |
| 分類 | 状態変化（わざ効果） |
| 発生方法 | 技「アンコール」を対象に命中させる |
| 効果内容 | 対象ポケモンは最後に使用した技しか選べなくなる |
| 持続目安 | 3ターン |
| 重ね掛け | 再付与・延長不可 |
| 継続停止 | PP切れ、交代、戦闘不能、メンタルハーブなどで解除 |

## 発動条件 / 失敗条件

### 発動条件

- 対象がすでに技を使用済みであることが必要  
- 最後に使用した技が対象となる

### 失敗条件

- 対象が技をまだ使用していない  
- 最後に使用した技が以下の場合：  
  - Transform、Mimic、Sketch、Mirror Move、Encore自身  
  - Struggle  
  - PPが0の技  
  - Sleep TalkやMetronomeなど一部の補助技  
- ダイマックス技・Zワザなどの特殊技も失敗する場合あり

## 効果詳細

### 制限内容

| 内容 | 仕様 |
|---|---|
| 選択可能技 | 最後に使用した技のみ |
| 他技選択制限 | 他の技は選択不可・実行不可 |
| PP関連 | 対象技のPPが0になると状態解除 |
| みがわり・防御効果 | みがわり越しでも効果は適用 |
| 優先度・対象選択 | 反射効果（Magic Coat）で反射可能 |

### 例外・処理仕様

- 対象が行動前にアンコールを受けた場合、そのターンの行動は対象技に固定  
- ちょうはつなどで通常選択不可技を選ぶ場合、Struggle等に置き換わることがある

## 持続と解除

アンコール状態は以下で解除される：

- 対象技のPPが0になる  
- 対象ポケモンの交代・戦闘不能  
- 道具「メンタルハーブ」などの状態解除効果  
- 3ターン経過

## 競合・相互作用

| 条件・効果 | 仕様 |
|---|---|
| 特性「アロマベール」 | 味方へのアンコール効果を防ぐ |
| ダイマックス中の技 | 世代によってアンコール対象外になる場合あり |
| ちょうはつ等制限効果 | 対象技使用不可と競合するとStruggleに置き換わる場合あり |

## ターン処理順（概要）

1. 技「アンコール」が成功 → 状態変化付与  
2. 対象が場にいる間は最後に使用した技のみ選択可能  
3. PP減少・動作制限など状態条件を維持  
4. 解除条件到達で状態消滅
