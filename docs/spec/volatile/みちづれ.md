# みちづれ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | みちづれ（状態変化） |
| 分類 | 状態変化（Destiny Bond / みちづれ） |
| 効果概要 | 対象ポケモンが **次に行動するまでの間、自身が相手の攻撃技でひんしになった場合、その攻撃をしたポケモンもひんし状態にする** |
| 発生源 | 技「みちづれ」 |
| 対象 | 技を使用したポケモン自身 |
| 重ね掛け | 非対応（既に状態中は重ね・延長なし） |
| 状態解除 | 対象ポケモンの **次の行動時**／交代／戦闘不能 |

## 発動条件 / 無効条件

### 発動条件

- 技「みちづれ」を **使用した後** に、「みちづれ状態」が発生する。  
- 技自体には命中率・回避補正はなく、成功すれば状態発生となる。  
- 対象がすでにみちづれ状態の場合は **重ね掛けされない**。 :contentReference[oaicite:0]{index=0}

### 無効・付与不能条件

| 条件 | 結果 |
|---|---|
| 技「みちづれ」が失敗 | 状態発生なし |
| 対象が交代・戦闘不能 | 状態発生・維持なし |
| 技が選択不可（行動不能等） | 状態発生なし |

## 効果詳細

### 条件発動

- 対象が **みちづれ状態の間に相手の攻撃技によって自身がひんし状態になる** と、  
  → **その攻撃技を使用した相手ポケモンもひんし状態になる**。  
- 発動判定は **攻撃技によるひんし** のみで、やどりぎ等の継続ダメージや状態異常でのひんしでは発動しない。 :contentReference[oaicite:1]{index=1}

### 解除条件

| 状況 | 結果 |
|---|---|
| 対象の **次の行動時** | みちづれ状態が解除される |
| 対象が **交代** | 状態解除 |
| 対象が **戦闘不能** | 状態解除（発動判定前に解除） |

## 維持とタイミング

- みちづれ状態は **次の行動まで持続**（ターン経過では解除されない）。  
- ターンが変わっても対象が行動するまでは みちづれ状態は有効。  
- 状態中に対象が攻撃しなかった場合、その後のターンに攻撃をしても **発動判定は残る**。 :contentReference[oaicite:2]{index=2}

## 相互作用・例外

| 条件・効果 | 挙動 |
|---|---|
| 対象が「がんじょう」「きあいのタスキ」等で耐える | 発動対象条件は満たされない |
| 相手が「みがわり」状態 | 発動条件を満たせば **みがわりを貫通**してひんしにする |
| 技「みちづれ」の連続使用 | 連続で使用した場合、2回目以降は **失敗する**可能性がある |
| 自身が状態異常で行動不能 | 次の行動ができない場合でも状態は解除される場合がある |

## ターン処理順（概要）

1. 技「みちづれ」を **使用成功 → みちづれ状態付与**  
2. 以降、対象の **次の行動判定まで状態維持**  
3. その間に対象が **攻撃技でひんしになる** と判定  
4. 状態効果が **発動 → 攻撃者をひんしにする**  
5. 対象の **次の行動時に状態解除**

## 補足

- みちづれ状態は “条件付きのひんし効果保証” として機能し、発動条件を満たすと相手を同時に倒すことができる。  
- 発動条件は **攻撃技で受けたダメージでひんしになること** が必須であり、やどりぎ等の継続ダメージでは発動しないという世代仕様がある。  
- 相手が **みちづれ状態のポケモンを倒す前に交代した場合**、状態判定は発生しない。 :contentReference[oaicite:3]{index=3}
