# おんねん（状態変化）仕様書

※「おんねん」は 技「おんねん」により発生する状態変化として扱われるが、**第9世代の対戦では技自体の選択が不可**（使用できない）となっているため、仕様としては“過去世代の挙動”を基準にまとめる。:contentReference[oaicite:0]{index=0}

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | おんねん（状態変化） |
| 分類 | 状態変化（Grudge） |
| 効果概要 | 状態保持者が **ひんし状態で倒された場合**、その時倒した **相手ポケモンが最後に使った技の PP を 0 にする** |
| 発生源 | 技「おんねん」の成功による付与 |
| 対象 | 技使用者自身 |
| 持続 | **次の行動まで**（状態は持続し、技を受けるまで維持） |
| 重ね | 再付与・延長なし |
| 状態解除 | 対象が交代 / 戦闘不能 / 次の行動後 |

## 発動条件 / 無効条件

### 発動条件

- 技「おんねん」を **自身に使用して成功** すると状態が発生する。  
- 命中率・回避補正とは独立した判定であり、対象が状態付与可能な状況で通常通り発生する。:contentReference[oaicite:1]{index=1}

### 無効・付与不能条件

| 条件 | 結果 |
|---|---|
| 技「おんねん」の選択自体が不可 | 状態は発生しない（第9世代以降の対戦ルールでは不可）:contentReference[oaicite:2]{index=2} |
| 状態付与前に使用者が **ひんし / 戦闘離脱** | 状態付与できない |
| 付与後、状態が消失している | 効果は発動しない |

## 効果詳細

### PP 無効化条件

- 使用者が **おんねん状態でひんしになる** と、直前に倒した相手が **最後に使用した技の PP を 0 にする**。  
- 無効化されるのは **倒した技そのものの PP** である。  
- **Struggle（わるあがき）** や **継続ダメージ（やどりぎ 等）による戦闘不能** では効果は発動しない。:contentReference[oaicite:3]{index=3}

### 継続と解除

- おんねん状態は **次の行動まで持続** し、対象が倒されるか交代するまで維持する。  
- 対象が戦闘不能・交代・状態解除効果を受けた場合、状態は消失する。

## 競合・相互作用

| 条件・特性 | 挙動 |
|---|---|
| PP管理に関わる効果 | 主に **PP を減らす効果と独立して発動** （他の状態異常とは異なる処理） |
| 相手の技が PP 無制限の場合 | 対象の PP は減少せず（例：無限 PP 装備） |
| 特性や特殊効果 | 状態そのものの処理は **標準の状態変化扱い** |

## 注意・世代差異

- 第9世代以降では **技「おんねん」が対戦ルール上選択不可** のため、通常の対戦では状態発生効果そのものを利用できない。:contentReference[oaicite:4]{index=4}  
- 過去世代での仕様として扱う場合、状態発生中に倒されるタイミングが **特殊な判定となる点** が挙動の要注意ポイント。  
- 技効果は通常の状態異常とは異なり、**特定行動結果に対するトリガー条件**で発動する仕組み。:contentReference[oaicite:5]{index=5}
