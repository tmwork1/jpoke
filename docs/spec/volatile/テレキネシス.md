# テレキネシス（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | テレキネシス（状態変化） |
| 分類 | 状態変化（わざ「テレキネシス」による付与） |
| 効果概要 | 対象が **地面にいない状態（浮いている）になる** と同時に、一部の攻撃が **必中** になる |
| 効果対象 | 状態を付与されたポケモン |
| 持続 | **3ターン** |
| 重ね掛け | 再付与・延長なし |
| 状態解除 | ターン経過・交代・戦闘不能・解除効果 |

## 発動条件

- 技「テレキネシス」を **対象に命中させたとき** 状態が付与される  
- 技自体の命中が成功しない場合は付与されない  
- 対象がすでに状態中の場合は **再付与・延長されない**

## 効果詳細（浮遊・必中）

### 浮遊効果

- 対象は **地面にいない扱いとなる**（地面技・フィールドなどの影響を受けない）  
  - 例：じめんタイプ技・エントリー ハザード（まきびし/どくびし/ねばねばネット）等の影響を受けない  
- この状態は **重力場（じゅうりょく）・うちおとす・サウザンアロー等で解除** されうる  
- 「ねをはる」などで地面に固定されたポケモンには付与/効果が無効になる

### 命中補正

- 状態中の対象に対して **一撃必殺技以外の技は「必中」扱い** になる  
  - すばやさ変動や命中・回避ランクの影響を受けず命中判定が通る  
  - 半減など耐性は影響するが **命中率判定が存在しない**  
- 半減対象の判定は、対象がフリーフォール/浮遊扱いのままにならない条件下では異なる場合あり

## 無効化・解除

テレキネシス状態は以下の条件で解除／発生しなくなる：

| 条件 | 結果 |
|---|---|
| ターン経過（3ターン） | 状態解除 |
| 対象交代・戦闘不能 | 状態解除 |
| 「じゅうりょく」状態が発生 | 状態解除／付与不可 |
| 「うちおとす」「サウザンアロー」など地面化効果 | 状態解除 |
| 「ねをはる」状態の対象 | 付与失敗（既存状態持続扱いになる場合あり） |
| 対象が本来地面状態に一体化しているポケモン（ディグダ系等） | 付与不可（状態は付与されない） |
| Gravity 効果中 | 技の選択ができない／付与不可 |

## 相互作用・例外

| 効果・条件 | 挙動 |
|---|---|
| みがわり等で防御 | 技命中・状態付与の判定は対象の状況に準ずる |
| 特性・技効果で浮遊化している対象 | 状態付与の条件判定に影響する場合あり |
| バトンタッチで引き継ぎ | 状態は **引き継がれる**（条件に応じて影響継続） |
| 状態中に Gravity 発生 | 全てのテレキネシス状態が解除 |

## ターン処理順（概要）

1. 技「テレキネシス」が成功 → 対象に状態付与  
2. 対象は **地面にいない扱い** とする  
3. 対象に対する攻撃技の命中判定は **必中扱い**（一撃必殺除く）  
4. ターン終了でカウント進行  
5. 3ターン経過で **状態解除**

## 補足・注意点

- 対象がすでに **地面にいない状態**（例: ひこう/でんじふゆうなど）でも追加的に必中効果が付与される  
- 状態は **技効果の付与管理** として扱われ、フリーフォール等の他の浮遊継続効果と交差する場合は優先処理順が存在  
- 一撃必殺技はこの必中補正の対象外であり、通常通り命中判定を行う
