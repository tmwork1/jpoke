# あなをほる（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | あなをほる（状態変化） |
| 分類 | 状態変化（Semi‑invulnerable 状態） |
| 関連技 | わざ「あなをほる」 |
| 効果概要 | 使用者が **地中に潜った状態** となり、 **多くの技を受けない** 状態に入る |
| 対象 | 使用したポケモン自身 |
| 持続 | **1ターンのみ**（次ターン攻撃発動で解除） |
| 重ね掛け | 重ね不可・延長不可 |
| 状態解除 | 攻撃実行時 / 交代・戦闘不能 / 何らかの中断 |

## 発動条件 / 無効条件

### 発動条件

- 技「あなをほる」を **使用したターンに行動が成功** すること。  
- 技が **選択・進行判定を通過** している必要あり。  
- 潜伏状態中は **半透明状態** として扱われる。  
- 自身が行動不能の場合、潜伏は進まず攻撃へ移行（中断扱い）。 :contentReference[oaicite:0]{index=0}

### 無効・付与不能

| 条件 | 結果 |
|---|---|
| 技が **無効化された場合**（まもる等） | 状態は発生しない |
| 使用者が **交代 / 戦闘不能** | 状態は発生しない |
| 同一ターン内に行動不能 | 潜伏は中断扱い |

## 効果詳細

### 潜伏状態（Semi‑invulnerable）

- 潜伏中は **多くの技・効果が命中しない / 避ける** 判定となる。  
  - 地中にいるため、通常の直接命中攻撃は **ほとんど当たらない**。  
  - **じしん / マグニチュード / フォール系（Fissure）など一部技のみ命中する**。  
  - 潜伏中は天候ダメージ（すなあらし・あられ）や一部フィールド影響を回避扱いとなる。  
  - ロックオン等の必中誘導効果や特性ノーガードの場合は命中するケースあり。  
  - 状態中は **場の影響（天候など）を受けない処理** になる場合がある。 :contentReference[oaicite:1]{index=1}

### 攻撃実行

- 潜伏した次のターンに「あなをほる」の **ダメージ判定・攻撃が実行される**。  
- この攻撃時に状態は **解除される**。  
- 攻撃処理では通常の命中・ダメージ計算が行われる。 :contentReference[oaicite:2]{index=2}

## 維持と解除条件

| 条件 | 結果 |
|---|---|
| 攻撃発動時 | 潜伏状態は解除 |
| 対象が **交代 / 戦闘不能** | 潜伏状態は解除（中断） |
| 行動不能で処理中断 | 潜伏は解除として扱う場合あり |

## 競合・相互作用

| 条件/効果 | 挙動 |
|---|---|
| ロックオン効果 | 潜伏中でも特定の必中判定で命中する可能性あり |
| 特性「ノーガード」 | 潜伏中でも命中判定が成立する可能性あり |
| 保護系技（まもる等） | 潜伏自体は別扱いのため、潜伏判定と混同しない |
| 天候・フィールド | 潜伏中は多くの継続ダメージ・効果を **受けない / 無効扱い** になる |

## ターン処理順（概要）

1. 使用者が技「あなをほる」を選択し行動成功  
2. **潜伏状態が付与（1ターン目）**  
3. 潜伏中の攻撃判定（大半の技を避ける）  
4. 次ターン：攻撃実行 → **状態解除**

## 補足・注意点

- あなをほるの潜伏状態は **「地中に潜っている」判定** として内部処理され、多くの攻撃が命中しない扱いとなる。  
- じしん・マグニチュード系は例外的に命中しやすく、ダメージも **通常より大きく扱われる** 場合あり。  
- 技効果中に状態が解除された場合、潜伏は中断される。  
- 天候などの **継続効果を受けない** 判定が適用されるが、タイプ相性や一部効果は影響を受けることもある。 :contentReference[oaicite:3]{index=3}
