# ダイビング（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ダイビング（状態変化） |
| 分類 | 状態変化（わざによる付与） |
| 関連わざ | 技「ダイビング（Dive）」 |
| 効果概要 | 自身が **水中に潜っている状態** となり、多くの攻撃が命中しなくなる |
| 持続 | **2ターン構成（1ターン目：潜る、2ターン目：攻撃実行）** |
| 重ね掛け | 同一状態は **重ね不可 / 延長不可** |
| 状態解除 | 2ターン目攻撃時に解除、または行動阻害で中断 |

## 発動条件 / 無効条件

### 発動条件

- 技「ダイビング」を **使用したターンの行動選択が成功** した場合に発生  
- 技そのものが **成功（選択通りに進行）** する必要あり  
- **みがわりや反射技そのものでは状態付与自体が消費されないが、行動阻害で中断する可能性あり**

### 無効 / 付与不能

| 条件 | 結果 |
|---|---|
| 技「ダイビング」選択不可（例：ねむり等） | 状態発生せず |
| 技が **防がれ/無効化された場合** | 状態は発生しない |
| 状態継続中に **交代/ひんし** | 状態解除 |
| 同世代ルールで使用不可 | 状態発生しない |

## 効果詳細（潜伏・命中無効化）

### 潜伏状態

- **1ターン目で潜る動作（semi‑invulnerable）** が発生  
- 潜伏中、自身は多くの攻撃技を **命中させない（回避・無効化）** と扱われる  
- ただし以下の攻撃は **潜伏中でも命中判定が可能**：
  - 「なみのり（Surf）」および「うずしお（Whirlpool）」：潜伏中でも命中し、**威力が2倍** となる世代仕様がある（第4世代以降を除く一部世代） :contentReference[oaicite:0]{index=0}  
  - 「ロックオン」「マインドリーダー」等の必中誘導効果がある場合
  - 特性「ノーガード／No Guard」による命中無効化解除時
- 潜伏中は **天候ダメージ（すなあらし／あられ）等の紐付くダメージ処理** も回避対象になることがある :contentReference[oaicite:1]{index=1}

### 攻撃実行（2ターン目）

- **2ターン目** に潜っていた状態が解除され、通常の **ダメージ攻撃処理** が行われる  
- 攻撃成功時は **与ダメージ・命中判定等は通常の攻撃処理** に戻る  
- 潜伏中に特性や技などで行動不能になった場合は **処理中断** となり、ダイビングは失敗扱いになることがある :contentReference[oaicite:2]{index=2}

## 無効化・中断条件

| 条件 | 挙動 |
|---|---|
| 行動不能効果（まひ・ねむり等） | 潜伏行動がキャンセルされる |
| 特性により命中無効化解除（例：ノーガード） | 潜伏中でも攻撃を受けることがある |
| 技が選択後に阻害（混乱自傷など） | ダイビングの処理が **進行しない / 中断扱い** |

## 競合・相互作用

| 相互作用 | 挙動 |
|---|---|
| みがわり | 潜伏中でも基本的に **状態付与は成立** するが、実際の命中判定は別処理 |
| バトンタッチ等 | 潜伏状態は解除（行動が中断） |
| 連続保護系・干渉系 | 他二ターン技同様の扱い（例：ふいうち等は潜伏貫通が発生する場合あり） |
| アイテム「パワーハーブ」 | 潜伏を**カットして1ターン攻撃**にできる :contentReference[oaicite:3]{index=3} |

## ターン処理順（概要）

1. 使用者が技「ダイビング」を選択  
2. **1ターン目**：潜伏状態（semi‑invulnerable）へ移行  
3. 命中可能攻撃の判定  
4. **2ターン目**：潜伏解除 → ダメージ攻撃処理  
5. 潜伏状態解除

## 補足

- ダイビング中でも天候などの **継続ダメージを回避できる場合がある**（例：すなあらし・あられ） :contentReference[oaicite:4]{index=4}  
- 技「ダイビング」は **二回行動技** として扱われ、潜伏ターンと攻撃ターンで別々の判定が走る :contentReference[oaicite:5]{index=5}
