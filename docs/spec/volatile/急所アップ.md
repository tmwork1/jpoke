# きゅうしょアップ（状態／急所率上昇）仕様書

## 基本情報

| 項目 | 内容 |
|---|---|
| 表示名 | きゅうしょアップ |
| 分類 | 状態（急所ランク上昇） |
| 効果概要 | 急所（クリティカルヒット）に当たりやすくなるランク補正。 |
| 発生源 | 状態変化・技「きあいだめ」・特性・アイテム等 |
| 対象 | 発動したポケモン自身 |
| 持続 | **自身が場を離れるまで**（交代・ひんしで解除） |
| 重ね掛け | 同種状態の **重複不可**（再付与は無効） |

## 発動条件 / 無効条件

### 発動条件

- 技 **「きあいだめ」** 等によって発生する。  
  - きあいだめ使用で **急所ランクが上昇** し、急所率が上がる（世代依存で上昇量が変動）。 :contentReference[oaicite:0]{index=0}

### 無効・付与不能条件

- すでに同状態が付与されている場合 **再付与不可**。 :contentReference[oaicite:1]{index=1}  
- 行動が妨害され状態付与が **成立しない場合** 発動しない。

## 効果詳細

### 急所ランクと急所率

- 急所は**特定乱数判定で発生するクリティカル判定**であり、発生時はダメージが通常より増加（第6世代以降は1.5倍）。 :contentReference[oaicite:2]{index=2}  
- **きゅうしょアップ状態**はこの急所に当たる確率を上昇させるランク補正の一種である。 :contentReference[oaicite:3]{index=3}  
- 状態の具体的な急所率への変換は世代ごとに異なるが、一般的に急所ランクが上昇することで **急所への当たりやすさが高まる**。 :contentReference[oaicite:4]{index=4}

### 技「きあいだめ」による付与

- 技「きあいだめ」は対象の急所ランクを上昇させる **状態変化として機能**する。  
- 状態付与後は交代・ひんしまで持続し、急所率補正が適用される。 :contentReference[oaicite:5]{index=5}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 対象が **交代** | 状態解除 |
| 対象が **ひんし** | 状態解除 |
| 対象が **戦闘不能・場離脱** | 状態解除 |

## 相互作用・注意点

- 同じ状態（急所率上昇）は重複せず **付与済み状態への再付与は無効**。 :contentReference[oaicite:6]{index=6}  
- 特性・アイテム等による補正は別枠で適用・累積する場合がある。  
- 急所ランク補正は他の補正と**独立した内部処理**として扱われる。 :contentReference[oaicite:7]{index=7}

## ターン処理順（概要）

1. 技「きあいだめ」を **使用し成功**  
2. 急所ランク補正の状態（きゅうしょアップ）を **付与**  
3. 急所判定（攻撃技命中時の急所発生確率に反映）

## 補足説明

- きゅうしょアップを受けている状態では、急所発生確率が通常より高くなる。  
- 具体的な確率変化は世代・状況・他補正要素に依存する。 :contentReference[oaicite:8]{index=8}
