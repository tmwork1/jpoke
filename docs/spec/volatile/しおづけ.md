# しおづけ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | しおづけ（状態変化） |
| 分類 | 状態変化（技「しおづけ」などによる付与） |
| 効果概要 | 毎ターン終了時に持続ダメージを受ける |
| 種別 | ダメージ継続系（揮発性に近いが交代では解除） |
| 持続条件 | 対象が **場にいる間** |

## ダメージ量

| 条件 | ダメージ |
|---|---|
| 通常 | **最大HPの1/8** |
| 対象が みずタイプ または はがねタイプ | **最大HPの1/4** |

※ ダメージ計算後は **小数点以下切り捨て**。  
※ みず/はがね両方を持つ場合も **1/4**（2倍ではない）。  
※ 特性「マジックガード」などでダメージ無効化可能。 :contentReference[oaicite:0]{index=0}

## 発動条件 / 無効条件

### 発動

- 技「しおづけ」（Salt Cure）等の効果で付与される。  
- 技命中後、対象に状態変化として付与される。 :contentReference[oaicite:1]{index=1}

### 無効／解除条件

| 条件 | 結果 |
|---|---|
| 対象が **交代** | 状態解除（交代判定で消滅） |
| 状態防止特性（例：りんぷん、おんみつマント） | 状態付与を **予防** 可能 |
| ダメージ無効特性（例：マジックガード） | ダメージを **受けない** が状態自体は残る |

## 競合・相互作用

| 効果 / 状態 | 備考 |
|---|---|
| みず・はがねタイプの判定変化 | タイプ変化系（へんげんじざい等）でタイプが変わるとダメージ量が変動 |
| 持続ダメージ優先順 | 他の持続系（やけど/どく等）とは **別途処理** |
| バトンタッチ | 交代扱いとなるため **解除対象** |

## ターン処理順（概要）

1. 対象に技「しおづけ」が命中 → 状態変化付与  
2. 行動処理進行  
3. 各ターン終了時に **持続ダメージ判定**  
4. 次ターン以降も継続（交代で解除）

## 補足

- 対象が **場に居続ける限り**毎ターンダメージが発生。  
- タイプ耐性によるダメージ量の変化は **対象の現在タイプ判定**で処理。 :contentReference[oaicite:2]{index=2}
