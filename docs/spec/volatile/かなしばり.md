# かなしばり（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | かなしばり（状態変化） |
| 分類 | 状態変化（わざ/特性による付与） |
| 効果概要 | 対象の **最後に使った技を使えなくする** 制限状態 |
| 効果持続 | 基本 **4ターン（SV 以降）** |
| 重ね掛け | 再付与・延長は発生しない |
| 状態解除 | 交代・戦闘不能・状態解除系で解除 |

## 発動条件 / 失敗条件

### 発動条件

- 技「かなしばり」などで対象に状態変化を付与した場合に発生  
- 「のろわれボディ」等による付与も含む

### 失敗条件

以下の場合、かなしばりは付与されないまたは失敗扱いとなる：

- 対象が **まだ技を使用していない**  
- 最後に使った技が **わるあがき（Struggle）**  
- 対象がすでに同じ状態異常を持つ（重複不可）  
- 技が無効化（まもる等）された場合

## 効果内容（技制限）

- 対象は **最後に使用した技を「使用不可」** とする  
- 使用不可技は技選択画面で選べない  
- PPが0になっても解除されず、制限はターン経過で消える  
- 技を封じられたターン中は該当技を選択不可として扱う  
- 制限技を対象が選択しようとした場合、行動自体が不可となる

## 効果持続と解除

かなしばり状態は以下の条件で解除される：

- 付与から **4ターン経過**  
- 対象ポケモンの **交代・戦闘不能**  
- 道具（メンタルハーブなど）による状態解除

## ターン処理・持続

1. 技「かなしばり」が命中 → 対象へ状態変化付与  
2. 状態有効ターン数が **4** に設定  
3. ターン終了時にカウント進行  
4. 4ターン経過で状態解除

## 競合・相互作用

| 他効果 / 条件 | 挙動 |
|---|---|
| 「アンコール」状態 | 他技固定系と重複した場合、挙動が競合（次行動以降に判定） |
| 「こだわり系アイテム」 | 封じられた技を選択できなくなるため、わるあがき状態になる場合あり |
| 技の無効化・反射 | 魔法こーと等で付与前に跳ね返される/防がれる可能性あり |

## 補足・注意点

- かなしばり状態は **技の使用不可制限のみ** を付与し、他の行動制御（移動不能など）を伴わない  
- 第5世代以降はターン持続が標準化されている  
- 制限技は攻撃以外の補助技も対象となる
