# さしおさえ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | さしおさえ（状態変化） |
| 分類 | 状態変化（わざ / 技効果） |
| 発生方法 | 技「さしおさえ」を対象に命中させる |
| 効果内容 | 対象ポケモンの **持ち物の効果を無効化し、アイテム使用も不可にする** |
| 効果持続 | **5ターン** |
| 重ね掛け | 同じ状態は **重ね不可 / 延長不可** |
| 状態解除 | 交代（引っ込める）で解除 |

## 発動条件 / 無効条件

### 発動条件

- 技「さしおさえ」を **対象に命中** させることで状態が付与される。  
- 対象が技を無効化している状態（まもる / みがわり）では付与されない。  
- 状態を付与されたポケモンを **交代させると解除** される。  
- 状態は使用者が引っ込んでも **残る**（引き継がれる）。 :contentReference[oaicite:0]{index=0}

### 無効条件

| 条件 | 結果 |
|---|---|
| 技による防御（まもる・みがわり） | 状態付与が発生しない |
| 特性や効果による状態防止 | 状態付与が発生しない場合あり |
| 状態付与済 | 重ねがけ・延長はされない |

## 効果詳細（持ち物無効化）

### 無効になるもの

さしおさえ状態の対象は、次のような **持ち物・効果を受けなくなる / 使用不可**：

- アイテムの **使用（きのみ含む）** ができない  
- 持たせたアイテムの **効果が発動しない**  
- 「HPが増減するもの全般」のアイテム効果  
- 「状態異常付与 / 回復」などの持ち物効果（状態異常系など）  
- 「こだわり○○」系による能力上昇・制限効果  
- 「くっつきバリ」の設置・効果  
- 「しぜんのめぐみ」の発動制限（失敗する）  
- 「くろいてっきゅう」「ふゆう」での地面技理由付与効果を無効化  
- 上記状態中は対象の持ち物による追加効果が適用されない :contentReference[oaicite:1]{index=1}

### 無効にならないもの

- スピード低下系アイテム（アイアンボール / ギプス系 / パワー系等）の低下効果  
- 「トリック」「すりかえ」「リサイクル」「なげつける」などのアイテム関連行動  
- 状態中に付与されたままの効果の **直接影響を受けないもの**  
- 特性や状態固有の効果（持ち物とは別扱い） :contentReference[oaicite:2]{index=2}

## 競合・相互作用

| 効果・状態 | 結果 |
|---|---|
| バトンタッチ | さしおさえ状態は引き継がれる |
| アイテム使用反射系 | さしおさえ状態中はアイテム使用自体が不可 |
| 特性・状態防止 | 状態付与前の条件によって防がれる可能性がある |
| 交代（引っ込める） | 状態は解除される |
| 使用者引っ込み後 | 状態は残る | :contentReference[oaicite:3]{index=3}

## ターン処理順（概要）

1. 対象に技「さしおさえ」が成功 → 状態が付与  
2. 対象は **5ターン**状態となる  
3. 対象の持ち物効果・使用が **無効化・不可**  
4. 5ターン経過または交代処理で状態を解除

## 注意点

- 状態中は **きのみを食べる・使う・発動する効果全般が不可**。  
- 「くろいてっきゅう」「ふゆう」の影響による地面技追加効果は無効化される。  
- アイテム効果が切れるタイミングは **状態解除時まで保留** として扱われる場合がある。 :contentReference[oaicite:4]{index=4}
