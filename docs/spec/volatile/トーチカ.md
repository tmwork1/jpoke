# トーチカ（技）仕様書

## 基本情報

| 項目 | 内容 |
|---|---|
| 表示名 | トーチカ |
| 英語名 | **Wall‑Shield (or similar protective move name)** |
| 分類 | 変化技 |
| タイプ | どく |
| 対象 | 自分 |
| PP | 10（最大16） |
| 優先度 | +4（**先制で発動**） |

## 効果概要

- **このターン中、自分への相手の攻撃技を防ぐ**。  
- 防いだ場合、その攻撃で**直接攻撃を使用した相手にどく状態を付与する**。  
- 同じターンに複数回防いでも、どく付与は対象技ごとに処理される可能性がある。  
- 同一技を連続使用すると**失敗しやすくなる**（保護系成功率低下システム）。 :contentReference[oaicite:0]{index=0}

## 発動条件 / 失敗条件

### 発動条件

- 使用者が技「トーチカ」を選択し、**行動が成立**した場合に効果が発生する。  
- 発動は必中（命中率判定なし）で、優先度 +4 により先制扱い。 :contentReference[oaicite:1]{index=1}

### 失敗条件

- そのターンで **最後に行動する場合**（行動順の関係上）や  
- 同じターン内に他の保護系技使用による **成功率低下** が発生している場合、効果が **失敗** する可能性あり。  
（まもる系の保護成功率低下と同一計算） :contentReference[oaicite:2]{index=2}

## 効果詳細

### 防護効果

- **対象自身への相手の攻撃技ダメージ・効果を無効化**する（そのターン中のみ）。  
- 保護の適用は **物理・特殊問わず** 防ぐが、  
  保護系妨害（ふいうち等）・必中効果・特性によっては防げない場合あり。  
- 防護は保護系技（まもる／みきり等）と共通の処理ルール。 :contentReference[oaicite:3]{index=3}

### どく付与効果

- トーチカ防護中に **相手が直接攻撃（接触判定ではなくヒット技）を使用してきた場合**、  
  その対象ポケモンに **どく状態を付与** する。  
- 防護中に接触以外の攻撃を受けてもどく付与は発生しない。  
- どく状態は通常の継続ダメージなどの仕様に従う。 :contentReference[oaicite:4]{index=4}

## 維持と解除

- トーチカの効果は **そのターン中のみ** 維持される。  
- ターン終了時に **効果は消滅**。  
- 技としては保護系の一種であり、次ターン以降は効果なし。 :contentReference[oaicite:5]{index=5}

## 競合・相互作用

| 条件・効果 | 挙動 |
|---|---|
| 保護系技の成功率低下 | まもる等と共通して **成功率低下判定** の対象 |
| 直接攻撃によるどく付与 | 防護判断後、条件を満たす攻撃を受けた場合に発生 |
| 特性・必中効果 | ふいうち等で防護が破られるケースあり |
| トーチカの後続防護 | 効果はターン内のみのため、継続しない | :contentReference[oaicite:6]{index=6}

## ターン処理順（概要）

1. トーチカを **使用 → 防護状態付与**  
2. 相手の攻撃技処理  
3. 防護判定（成立すれば無効化）  
4. 防護成立時に **どく付与判定**  
5. ターン終了で効果解除

## 補足

- トーチカは保護系技に **どく付与要素** を付けた形の技であり、  
  保護効果は **そのターン内の攻撃を無効化**、かつ条件付きで敵にどくを与える。  
- 技の成功率低下や保護系競合は **一般的な保護技と同一ルール** となる。 :contentReference[oaicite:7]{index=7}
