# ほろびのうた（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ほろびのうた（状態変化） |
| 分類 | 状態変化（わざ「ほろびのうた」による付与） |
| 効果概要 | 対象ポケモンが **付与から3ターン後にひんし状態になる** |
| 発生源 | 技「ほろびのうた」・特性「ほろびのボディ」などの効果 |
| 対象 | 付与当時バトルに出ていたポケモン全員 |

## 発動条件 / 無効条件

### 発動条件

- 技「ほろびのうた」が **成功（命中判定）** すると、フィールド上の Pokémon に状態が付与される。  
- 一部特性（例：ぼうおん）の Pokémon は **音系技判定で効果を受けない**場合あり。  
- 「みがわり」越しでも付与される。  
- すでに同状態のカウントを持つポケモンには付与されない。 :contentReference[oaicite:0]{index=0}

### 無効/付与不能ケース

| 条件 | 結果 |
|---|---|
| 技が無効化された場合（まもる等） | 状態付与は発生しない |
| 対象が既に同状態カウントを持っている場合 | 新たな付与は発生しない |
| 交代・離脱済のポケモン | 付与対象にはならない |

## 効果内容

### ほろびカウント

- ほろびのうた状態は **カウント方式** で管理される。  
- 付与された時点ですぐにカウントが **3ターン分セット** される。  
- 各ターン終了時にカウントが **1減少**（そのターンも含む）。  
- カウントが **0 になったターン終了時点で対象がひんしになる**。  
- 対象が対象状態の間に **交代した場合、付与は解除される**。 バトンタッチ等で後続に引き継ぐことは可能。 :contentReference[oaicite:1]{index=1}

### ひんし判定

- カウントが0になった時点で **HP削りではなく即座にひんし扱いになる**仕様。  
- HPベースの割合ダメージとは扱いが異なるため **Endure等では防げない**。 :contentReference[oaicite:2]{index=2}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 対象交代（通常交代） | ほろび状態が **解除** |
| 戦闘離脱（ひんし含む） | 状態解除 |
| バトンタッチ等で引き継いで交代 | ほろびカウントは **後続に引き継がれる** |
| 3ターン経過でカウント0 | ひんしになる（解除相当） | :contentReference[oaicite:3]{index=3}

## 優先処理・相互作用

- 「Soundproof」特性は第8世代以降でも **自身以外の Pokémon に対する付与を無効化**する場合がある。 :contentReference[oaicite:4]{index=4}  
- カウントはフィールド上の各 Pokémon 個別で管理される。  
- 一度カウントが付与された後に **別のほろびのうたを受けても上書きされない**。  
- 他の行動不能・状態異常とは独立して処理される。  
- 攻撃技ではないため **Focus Sash / Magic Guard / Sturdy** 等の防護は影響しない。 :contentReference[oaicite:5]{index=5}

## ターン処理順（概要）

1. 使用者が「ほろびのうた」を成功させる  
2. 対象 Pokémon に **ほろびカウント（3）付与**  
3. 各ターン終了時に **カウントを1減少**  
4. カウントが **0 になった場合、ひんし状態になる**

## 注意点

- 「みがわり」防御は無効化しないため、**みがわり越しでも状態付与**が発生する。 :contentReference[oaicite:6]{index=6}  
- 交代で状態は消えるため、戦略的に交代で回避可能。  
- ひんし判定は HP によるダメージではないため、Endure系の防止効果は無効。 :contentReference[oaicite:7]{index=7}
