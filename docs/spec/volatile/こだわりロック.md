# 共通仕様

## 効果（定量値）

| 項目 | 内容 |
|---|---|
| ステータス補正 | 対応する能力値 ×1.5 |
| 技選択制限 | 最初に使用した技以外を選択不可（交代で解除） |

---

## 発動条件 / 無効条件

### 発動条件

| 条件 | こだわり発生判定 |
|---|---|
| 技が実際に使用されPP消費・技メッセージ表示 | こだわる |
| 技を「さいはい」で使用 | こだわる |
| 溜め技・連続技 | 各ターンごとに判定 |
| 技の効果でこだわり系アイテムを取得し行動成功 | こだわる |

### 発動しない条件（こだわらない）

| 条件 | 備考 |
|---|---|
| 技が状態異常・状態変化・なまけ等で不発 | 技選択固定されない |
| じゅうりょく・ちょうはつ等で使用不能メッセージ | 同上 |
| PP0の技選択 | 同上 |
| ねこのて等で技が出せず失敗 | その技でこだわる（例外あり） |
| マジックコート反射技 | 反射技ではこだわらない |
| マジックミラー反射技 | こだわらない |
| トリック等でアイテム入手 | こだわらない（第5世代以降） |
| マジシャンで奪取 | こだわらない |

---

## ターン処理順（概念フロー）

1. 技選択  
2. 行動可否判定  
3. 技使用成功時にこだわり判定  
4. 技確定後、技固定状態付与  
5. 次ターン以降技選択制限適用  

---

## 競合・上書き・例外

### 技が上書き・変化した場合

| 条件 | こだわり状態 |
|---|---|
| 変身・ものまね・スケッチ・レベルアップで技上書き | 原則解除 |
| 特定条件で再習得 | 継続 |

### アイテム無効化・解除関連

| 条件 | 技固定状態 |
|---|---|
| ぶきよう / さしおさえ / マジックルーム | 一時解除 |
| 無効化解除後 | 元の技に再固定 |
| 単にアイテム喪失 | 完全解除 |

### ダイマックス（第8世代仕様だが概念参考）

| 状態 | 効果 |
|---|---|
| ダイマックス中 | ステータス補正・技固定ともに無効 |
| ダイマックス終了後 | 最初の技で固定 |

---

## 相互作用

### 技・特性との相互作用

| 種別 | 技・特性 | 相互作用 |
|---|---|---|
| 特性 | おどりこ | 条件により出た技で固定 |
| 特性 | マジックミラー | 反射技で固定しない |
| 特性 | ぶきよう | 効果無効・復帰で固定挙動変化 |
| 技 | トリック / すりかえ | 交換時固定解除 |
| 技 | どろぼう / ほしがる | 入手時固定しない |
| 技 | マジックルーム | 持ち物無効化により固定一時解除 |
| 技 | さしおさえ | 同上 |
| 技 | アンコール / さいはい | 固定技以外指定時失敗条件あり |
| 技 | 溜め技・数ターン技 | 無効復帰直後失敗処理あり |

---

# 第9世代固有注意点（第7世代以降共通仕様継承）

| 条件 | 挙動 |
|---|---|
| 固定後に固定技以外を出そうとする | 失敗（PP消費なし） |
| こだわり解除可能状態で選択した別技 | 条件復帰時失敗 |
| アンコール等で固定技以外指定 | わるあがき発生条件あり |
