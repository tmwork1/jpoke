# タールショット（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | タールショット（状態変化） |
| 分類 | 状態変化（技「タールショット」命中による付与） |
| 発生方法 | 技「タールショット」を **対象に命中** させる |
| 効果内容 | 対象ポケモンの **すばやさランクを1段階低下（–1）** |
| 効果持続 | **永続（交代するまで）** |
| 重ね掛け | 同一ポケモンに **再付与・延長なし** |
| 状態解除 | 交代、戦闘不能、状態解除効果で解除 |

## 発動条件 / 無効条件

### 発動条件

- 技「タールショット」が **対象に命中** すること  
- まもる・みがわり等で命中を無効化された場合は付与されない

### 無効 / 発生しないケース

| 条件 | 結果 |
|---|---|
| 対象がすでにタールショット状態 | 追加効果なし（重ね不可） |
| 技が命中しなかった | 状態は付与されない |
| 状態防止効果・特性（例: クリアボディ等） | ランク低下が無効になる |

## 効果詳細（すばやさランク低下）

- 対象は **即時にすばやさランクが–1** される  
- その後、状態は **維持され続ける**  
- ランク低下は **最低値（–6）まで可能**  
- すばやさランクに対する影響は、ゲーム内部のランク補正ルールに準拠

## 持続と解除

タールショットによる状態は以下で解除される：

- **交代したとき**  
- **戦闘不能になったとき**  
- 道具・技・効果（例：メンタルハーブ等）による **状態変化解除**

## 競合・相互作用

| 効果・条件 | 挙動 |
|---|---|
| 特性「クリアボディ」「しろいけむり」「メタルプロテクト」 | ランク低下が **無効化** |
| 特性「あまのじゃく」 | 低下時に **逆にランク上昇** として扱われる |
| 特性「かちき」「まけんき」 | ランク低下が条件となり **特性発動** |
| ランク低下耐性の持ち物（例: クリアチャーム） | ランク低下を **無効化** |

## ターン処理順（概要）

1. 技「タールショット」が命中 → 判定成功  
2. 対象の **すばやさランクを–1**  
3. 状態変化として **保持**（永続）  
4. 交代等で解除されるまで継続

## 補足

- タールショットは **単独で継続ペナルティを付与** する技であり、毎ターンごとのランク低下ではなく1回のみのランク低下と状態保持が特徴  
- 交代後に再びタールショットを命中させない限り、ランク低下は再発しない
