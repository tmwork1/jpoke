# でんじふゆう（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | でんじふゆう（状態変化） |
| 分類 | 状態変化（技「でんじふゆう / Magnet Rise」による付与） |
| 効果概要 | 対象ポケモンが **地面にいない扱い（浮遊系扱い）になる** |
| 効果主な影響 | ・じめんタイプの技・効果を **無効化**<br>・まきびし/どくびし/ねばねばネット 等の **地面関連効果を無効** |
| 効果持続 | **5ターン** |
| 重ね掛け | 再付与・延長 **なし** |
| 継続停止 | 交代・戦闘不能・状態解除系で解除 |

## 発動条件 / 無効条件

### 発動条件

- 技「でんじふゆう」を **対象（多くは自身）に命中** させる  
- 技効果が成功する必要あり  
- 効果付与後、対象ポケモンは状態変化として扱われる  

### 無効／付与失敗条件

| 条件 | 結果 |
|---|---|
| 技選択不可（重力 / じゅうりょく状態） | 技自体が **選択不可 / 失敗** になる |
| すでに地面に特定の固定状態になる（例: ねをはる） | 付与が **失敗** する |
| 命中前に防御効果（まもる等） | 状態付与が **されない** |
| 「うちおとす」「サウザンアロー」等で地面化後 | でんじふゆう状態は **解除** される |
| バトンタッチ等 | 状態は **引き継ぎ可能**（交代先でも有効） | :contentReference[oaicite:0]{index=0}

## 効果詳細（地面関連処理）

### 地面無効扱い

- 対象は **地面にいない扱い** となり、以下が無効化される：  
  - 地面タイプ技の **ダメージ・追加効果**  
  - Entry hazard の地面関連（まきびし / どくびし / ねばねばネット）効果  
  - フィールド地面関連処理（地面判定フィールド等影響）  
- 「でんじふゆう」状態でも **特性「ふゆう」やアイテム「ふうせん」** と同様の地面無効扱いになるが、重力・うちおとす等で解除される扱いになる。 :contentReference[oaicite:1]{index=1}

### 地面無効以外の影響

- 例外として **接地中の障害（まきびし等）を無効化** するが、戦闘の地面判定が別途処理されるケースでは個別判定が必要になる  
- でんじふゆう状態中に **技は成功する** ものの、地面無効処理が優先される  

## 維持と解除

| 条件 | 結果 |
|---|---|
| 5ターン経過 | 状態が解除 |
| 対象の交代/戦闘不能 | 状態が解除 |
| 地面に戻る効果発生（うちおとす等） | 即時解除 |
| 重力 / じゅうりょく発生 | 技選択不可・状態付与不可 |
| バトンタッチ | 状態は **引き継ぎ可能** が、継続時間は引き継いだ先でも残存 | :contentReference[oaicite:2]{index=2}

## 競合・相互作用

| 効果・条件 | 挙動 |
|---|---|
| 重力（じゅうりょく）状態 | 技の選択・付与が **不可** |
| うちおとす / サウザンアロー | 対象を地面化し **でんじふゆう状態を解除** |
| バトンタッチ | 状態を **交代先にも引き継ぐ** |
| 地面技以外のタイプ攻撃 | 通常通り処理される |

## ターン処理順（概要）

1. 技「でんじふゆう」を使用 → 命中判定  
2. 対象に **でんじふゆう状態** 付与  
3. 被判定段階で **地面技・地面関連効果を無効化**  
4. ターン経過ごとにカウント進行  
5. 5ターン後または解除条件で **状態解除**

## 補足

- でんじふゆうは **地面無効効果の一時付与状態** として扱われる  
- 重力等との相性で技選択・付与条件が制限されることがある  
- バトンタッチなどで別ポケモンに状態を引き継ぐことが可能だが、地面無効への処理は別途ゲーム処理に準じること
