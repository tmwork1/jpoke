# いちゃもん（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | いちゃもん（状態変化） |
| 分類 | 状態変化（わざによる付与） |
| 発生源 | 技「いちゃもん」命中 |
| 効果概要 | 直前に使用した技を次ターン以降に選択できなくする |
| 影響期間 | 交代するまで継続（ターン経過では消えない） |
| 重ね掛け | 再付与・延長は発生しない |
| 引き継ぎ | バトンタッチなど交代で消滅 |

## 発動条件 / 無効条件

### 発動条件

- 技「いちゃもん」で対象に状態変化を付与した場合  
- 対象が場に出てから技選択を1回以上している必要はない  
- いちゃもんの効果は技の選択制限であり、該当技の実行そのものを阻害するものではない  
- いちゃもん状態になったそのターン内の技実行には影響を与えない

### 無効・解除条件

| 条件 | 結果 |
|---|---|
| 対象が交代 | いちゃもん状態が解除 |  
| 道具（メンタルハーブなど） | 状態変化解除で解除 |  
| 特性「アロマベール」 | 対象への付与を防ぐ可能性あり |  
| ダイマックス状態 | 世代によって効果が無効化・解除される場合あり |

## 効果詳細

### 技制限

- いちゃもん状態のポケモンは直前に使用した技を次ターン以降に選択できない  
- 直前技が混乱・まひ・なまけ等でPPが減らない場合でも、次ターンで同じ技は選択不可扱い  
- 技選択画面へ入る前に状態になる場合でも、その後の選択で同じ技は出せない  
- 溜め技や複数ターン技（ころがる・げきりん等）は連続行動継続中は中断されない  
- 「さいはい」や「おどりこ」など別技として選出される場合は連続で使用可能  
- 全ての技のPPが0になった場合、わるあがきはいちゃもん状態でも使用可能

## 競合・相互作用

| 他効果 / 条件 | 挙動 |
|---|---|
| アンコール状態 | 次ターンの技固定と競合し、わるあがき等と交互になる場合あり |
| こだわり系アイテム | 選択可能技が1つの場合、いちゃもん効果との併用で制限が強化される場合あり |
| 状態変化解除 | 状態消去系効果（交代・メンタルハーブなど）で解除 |

## ターン処理順（概要）

1. 技「いちゃもん」の命中判定が成功  
2. 対象ポケモンへ状態変化付与  
3. 対象は直前に使用した技を次ターン以降に選択不可とする  
4. 交代・解除効果発動まで状態が維持される

## 補足・注意点

- いちゃもん状態は技選択制限のみであり、技使用不能化とは異なる  
- 制限判定は技選択フェーズで行われる
