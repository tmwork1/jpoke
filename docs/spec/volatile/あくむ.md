# あくむ（状態変化 / わざ効果）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | あくむ |
| 分類 | わざの状態変化 / 付与効果 |
| 技タイプ | ゴースト（状態変化技） |
| 発生方法 | 技「あくむ」を **相手に命中** させた場合 |
| 発動条件 | 対象が **ねむり状態** であること |
| 効果内容 | 毎ターン終了時に 対象ポケモンの **HPを最大値の1/4 減少** させる |
| 効果持続 | 対象が **ねむり状態の間継続** |
| 重ね掛け | **重複しない**（同時に複数付与できない） |
| 状態解除 | 対象が **ねむり状態を解除** した瞬間に同時に解除される |

## 発動条件 / 無効条件

### 発動

- 対象ポケモンが **ねむり状態** であることが必須。  
- ねむり状態でないポケモンに対しては **効果が発揮されず失敗** する。  
- みがわり等の状態で防がれる場合は **効果なし** と判定される。 :contentReference[oaicite:0]{index=0}

### 無効／解除

| 条件 | 結果 |
|---|---|
| 対象が **ねむり状態を解除** | あくむの状態変化も同時に解除される |
| 対象が場を離れる（交代等） | 状態変化は対象と共に **消失** |
| 他付与系妨害・防止効果 | ねむりが防がれる場合、あくむは **付与されない** |

## 効果詳細（ダメージ処理）

| 発生タイミング | 効果 |
|---|---|
| 各ターン終了時 | 対象の **HPを最大HPの1/4 減少** |
| HP減少の最小値 | 計算後 **小数点以下切り捨て** |
| 対象が瀕死になる場合 | HPが0になるまで減少 |

## 競合・相互作用

| 条件・特性 | 仕様 |
|---|---|
| 特性「マジックガード」 | **状態付与自体は受けるが** ダメージが発生しない場合がある（HP減少を防ぐ） |
| みがわり | あくむを付与できない / 防がれる |
| バトンタッチ等 | 状態は **引き継がれない**（対象変更で解除） |

## ターン処理順（概要）

1. 技「あくむ」命中判定  
2. 対象がねむり状態であるか確認  
3. 条件成立 → **あくむ状態付与**  
4. ターン毎の終了処理で **HP減少**  
5. ねむり解除または交代等で **ステータス解除**
