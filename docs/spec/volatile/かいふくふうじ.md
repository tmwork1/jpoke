# かいふくふうじ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | かいふくふうじ（状態変化） |
| 分類 | 状態変化（わざ「かいふくふうじ」の付与効果） |
| 発生方法 | 技「かいふくふうじ」を対象に命中させる |
| 効果対象 | 対象ポケモン |
| 効果内容 | 一定ターンの間、HP回復に関する効果を **受けなくなる** |
| 効果持続 | **5ターン** |
| 重ね掛け | 再付与・延長は不可（既存状態は更新されない） |

## 発動条件 / 無効条件

### 発動条件

- 技「かいふくふうじ」で対象ポケモンに状態変化を付与した場合に発生。  
- 技は命中する必要がある。

### 無効・解除条件

| 条件 | 結果 |
|---|---|
| 対象が交代 | 状態変化は解除 |
| 対象が戦闘不能 | 状態変化は解除 |
| 状態変化解除系の効果（道具「メンタルハーブ」等） | 状態変化は解除 |
| 対象が既に同状態の場合 | 重ね掛け不可（効果時間は延長されない） |

## 効果詳細（HP回復阻止）

かいふくふうじ状態の対象は、以下のような **HP回復関連の効果を受けなくなる**：

### 回復行為の制限

- 回復技（じこさいせい、ねがいごと、いやしのねがい等）の使用効果を受けない  
- みずあめボム等の回復付与効果を受けない  
- 特性による回復（しぜんかいふく、リジェネレータ等）を受けない  
- 環境・フィールドによる回復（とくせい／きのみ／場の効果）を受けない  
- ドレイン技（ギガドレイン等）による回復は **回復分が発生しない**（ダメージのみ発生）  
- どく／もうどく等でダメージを受けても回復しない  

ただし、以下のような効果は **制限対象外** または例外となる場合がある：

- 自己回復以外の効果（ステータス補正や非HP回復処理）には影響しない  
- ストリガーによるHP吸収や他の解除によって回復が発動する場合は個別に処理判定が必要

## 競合・相互作用

| 相互作用 | 仕様 |
|---|---|
| みがわり | 状態変化付与を防ぐことはないが、回復阻止効果は別処理 |
| 反射系（マジックコート／マジックバウンス） | 技自体が反射される場合あり |
| 状態解除アイテム（メンタルハーブ等） | 状態変化は即解除される |
| 技の無効化（まもる等） | 技命中が成立しない場合、状態変化は付与されない |

## ターン処理順（概要）

1. 技「かいふくふうじ」が成功 → 対象へ状態変化付与  
2. 対象が場にいる間はHP回復に関する効果を阻止  
3. ターン経過ごとにカウント進行（最大5ターン）  
4. 5ターン経過または解除条件発生で状態解除

## 補足

- かいふくふうじ状態は **HP回復関連処理の阻止** を管理するため、回復効果の開始・終了タイミングで個別に処理判定を挟む必要あり。  
- すでに回復済のHPに対して遡及して変更するものではない。
