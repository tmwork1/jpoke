# キングシールド（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | キングシールド（状態変化／技「キングシールド」由来） |
| 分類 | 状態変化（保護系の一種） |
| 効果概要 | 使用したターン中、自身への **攻撃技を防ぐ** とともに、**接触した敵の攻撃ランクを下げる** |
| 発生源 | 技「キングシールド」 |
| 対象 | 技使用者自身 |
| 発生タイミング | 技発動後、そのターン戦闘中 |
| 消滅タイミング | **そのターン終了** |
| 重ね掛け | なし（同一ターンの連続発生なし） |
| 関連形式 | 第6世代〜第9世代で存在（ただし第9世代で選択不可） :contentReference[oaicite:0]{index=0}

## 発動条件 / 無効条件

### 発動条件

- 技「キングシールド」を **選択して行動が成立** すると状態が付与される  
- 優先度 +4 の保護系技として扱われる :contentReference[oaicite:1]{index=1}

### 無効・失敗条件

- **同ターン内にすでに保護系成功率減少** が発生している場合、成功率低下により失敗する可能性あり（まもる等と成功率を共有） :contentReference[oaicite:2]{index=2}  
- 技が **無効化（例：行動不能など）** された場合、状態は発生しない  
- 対象が **交代・戦闘不能** などで場を離れた場合、状態は発生しない

## 効果内容

### 保護効果

- 使用した **そのターン中に対象ポケモンが受ける攻撃技（物理・特殊）のダメージおよび副次効果を無効化**  
  - 保護できるのは「攻撃技」で、変化技（状態変化のみを伴う技）は防げない :contentReference[oaicite:3]{index=3}  
  - 保護効果は「まもる／みきり」等と同じ成功率低下処理の対象  
  - 攻撃を防いだ場合、**その技によるダメージは発生しない** :contentReference[oaicite:4]{index=4}

### ランク変化

- 防いだ攻撃が **接触（直接攻撃）技だった場合**、その攻撃をした相手の **こうげきランクが −2 段階** になる（世代標準） :contentReference[oaicite:5]{index=5}  
  - 多段ヒット技でも、攻撃者1体につき **1回分** のランク低下判定  
  - 攻撃がタイプ無効で接触扱いでも低下判定が発生する場合あり :contentReference[oaicite:6]{index=6}  
  - 特性「みえざるこぶし（Unseen Fist）」など一部の特性を持つ攻撃は、保護無効およびランク低下効果を無効化する場合あり :contentReference[oaicite:7]{index=7}

## 維持と解除

| 条件 | 結果 |
|---|---|
| その**ターン終了** | 状態は **解除** |
| 技の効果が失敗 | 状態は発生しない |
| 対象が交代・戦闘不能 | 状態は **発生せず／解除** |

## 相互作用・例外

| 相互作用 | 挙動 |
|---|---|
| 保護系成功率低下 | 「まもる」「みきり」「ワイドガード」などと共有して成功率が低下する :contentReference[oaicite:8]{index=8} |
| ふいうち等の先制攻撃 | 先制技でも保護判定の対象だが、成功率低下や不発判定により防げない場合がある |
| みがわり | 分身状態には保護効果は「先に分身ヒット処理」→その後保護判定 |
| 攻撃無効化条件（タイプ無効等） | 保護成功時でもランク低下効果の有無が世代・判定による差異あり :contentReference[oaicite:9]{index=9} |
| 特性「バトルスイッチ」 | Aegislash がキングシールドを使用した場合、使用時点で **Shield Forme に変化**（対応能力） :contentReference[oaicite:10]{index=10} |

## ターン処理順（概要）

1. 使用者が **キングシールドを選択**  
2. 成功判定 → **保護状態付与**  
3. 相手の攻撃技処理時に **保護判定**  
4. 保護成立時は **ダメージ・効果無効化**  
5. 接触攻撃時は **相手のこうげき−2**  
6. ターン終了で **状態解除**

## 補足

- キングシールドは **保護系技の一種** であり、まもる・みきり・クイックガード等と同じ成功率低下体系を共有することがある :contentReference[oaicite:11]{index=11}  
- 世代によっては **Z‑技化（Z‑King’s Shield）等の追加効果** があるが、基礎仕様は保護＋ランク低下判定 :contentReference[oaicite:12]{index=12}  
- 第9世代では対戦ルール上 **使用不可** となっている場合がある（技リストから削除） :contentReference[oaicite:13]{index=13}
