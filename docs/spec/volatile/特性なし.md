# とくせいなし（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | とくせいなし（状態変化） |
| 分類 | 状態変化（Ability suppression / 特性無効化） |
| 効果概要 | 対象ポケモンの **本来持つ特性の効果を一時的に無効化** する |
| 対象 | 1体（効果付与対象） |
| 持続条件 | 付与後 **継続**（解除条件を満たすまで） |
| 重ね掛け | 付与済みの場合 **上書き・延長なし** |
| 状態解除 | 対象交代 / 戦闘不能 / 所定ターン経過 / 状態解除効果による解除 |

## 発動条件 / 無効条件

### 発動条件

- 対象に **特定の技・効果によって付与** される状態変化  
  （例：技「いえき / Gastro Acid」「コアパニッシャー / Core Enforcer」「特性 ニュートラライジングガス」の効果）  
- 状態付与には **命中・効果判定の成功** が必要  
- 付与対象がすでに「とくせいなし」状態である場合、再付与や延長は発生しない  

### 無効・付与失敗条件

| 条件 | 結果 |
|---|---|
| 対象に付与前に技・効果が **無効化された場合** | 状態付与は発生しない |
| 対象が交代済・戦闘不能 | 状態付与は不可能 |
| 対象がすでに「とくせいなし」状態 | 追加付与なし |

## 効果詳細／仕様

### 特性無効化

- 対象が「とくせいなし」状態の間、**本来持っている特性の効果はすべて無効化**される  
  - 特性の効果（戦闘中のステータス変化・能力補正・反応効果・場効果等）は発動せず、機能しない  
  - 対象の **特性名自体は変化しない** が、効果が消失した状態として扱われる  
  - 特性による **自動発動判定・影響効果すべてが抑止**される  
- 状態中の特性効果は、状態が解除されるまで **一切効果を発揮しない**

### 特性の扱い

- 状態「とくせいなし」は **特性を持たない状態になる** と内部で扱われる  
- 特性を持つポケモンが状態となっても、**特性名自体は残る** が効果は発動しない  
- 状態中の特性が他の手段で変更（例：スキルスワップ・トレースなど）された場合、新たに得た特性も **発動せず無効のまま** 扱われる  
- バトンタッチなどで別ポケモンに状態を引き継いだ場合、状態効果もそのまま引き継がれる  
- 対象が持つ **特性によって「とくせいなし」状態にならないもの**（例：一部例外特性）は無効対象とならない場合がある（仕様依存）

## 維持と解除

| 条件 | 結果 |
|---|---|
| 対象が **交代 / 戦闘不能** | 「とくせいなし」状態が解除される |
| 所定ターンの経過（技/効果により付与ターン指定がある場合） | 状態が解除される |
| 状態解除効果（特性/技/道具） | 状態が解除される |
| 状態予防効果（例：とくせいガード） | 状態付与が発生しない |

## 競合・相互作用

| 相互作用 | 挙動 |
|---|---|
| 交代技（バトンタッチ等） | 状態が **交代先でも継続** |
| 状態付与後に別効果で特性変更 | 新たに得た特性も **無効化中は発動しない** |
| 「とくせいなし」対象に特性効果が発生する効果 | 状態中は **効果を発揮しない** |
| 特性自体が「とくせいなし状態にならない」例外特性 | 対象外（特性効果が持続するか無効化解除扱いになる場合あり） |

## ターン処理順（概要）

1. 技/効果が対象に **命中・判定成功** → 状態付与  
2. 対象は **特性効果を失った状態** となる  
3. 各ターン継続中は特性効果を **無効化**  
4. 交代・戦闘不能・所定ターン経過・解除効果で **状態解除**

## 補足

- 「とくせいなし」は **特性効果そのものの発動を完全に無効化する状態変化**であり、持っている特性が意味を持たない状態となる。  
- 状態中に **特性名が表示されない扱い**になる場合あり（ゲームUI上の表示は変化）。  
- 一部例外や世代ごとの差異が存在するが、基本挙動は特性効果の無効化となる。 :contentReference[oaicite:0]{index=0}
