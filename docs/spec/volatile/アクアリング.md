# アクアリング（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | アクアリング（状態変化） |
| 分類 | 状態変化（恒常的） |
| 発生方法 | 技「アクアリング」の命中成功 |
| 効果対象 | 使用したポケモン自身 |
| 効果内容 | 毎ターン終了時に最大HPの1/16回復 |
| 回復量詳細 | 小数点以下切り捨て（切り捨て0でも1回復） |
| 持続条件 | 対象ポケモンが場に居続ける限り継続 |
| 消滅条件 | 対象ポケモンが交代・戦闘不能・状態解除で消滅 |

## 発動条件

- 技「アクアリング」を使用して成功し、対象ポケモンが状態変化を得る。  
- 技は自身にしか影響せず、他者には付与されない。

## 回復挙動

| タイミング | 効果 |
|---|---|
| ターン終了時 | 対象ポケモンのHPを最大HPの1/16回復 |
| 交代・戦闘不能 | 回復効果は消滅 |
| バトンタッチ | 状態は引き継がれ、次のポケモンにも継続適用 |

## 相互作用・例外

### 技・状態・効果との関係

| 効果・条件 | 挙動 |
|---|---|
| かいふくふうじ状態 | 回復効果は発動しない |
| リーキシード等 他回復効果 | 他回復効果と併用可能（別個に加算） |
| 状態変化解除技 | 状態消去効果によりアクアリングも解除 |

### アイテムとの影響

| アイテム | 効果 |
|---|---|
| おおきなねっこ | 回復量が1.3倍（約1/12）になる |
| 回復アイテム | 個別回復処理（直接HP回復）は別効果 |

## 競合・注意点

| 条件 | 仕様 |
|---|---|
| 複数回の再使用 | すでにアクアリング状態がある場合、効果は上書きせず継続時間は延びない |
| バトンタッチ時の継続 | 交代先に状態が継続し、回復は次ターン以降も発生 |
| 天候・フィールド等 | 天候や場の効果による回復補正は直接影響しない |

## ターン処理順（概要）

1. 技「アクアリング」を使用 → 状態変化を付与  
2. 通常の行動処理進行  
3. ターン終了時フェーズでHP回復処理  
4. 次ターン以降も同様の回復判定（条件満たす限り）
