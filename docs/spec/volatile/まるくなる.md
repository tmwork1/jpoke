# まるくなる（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | まるくなる（状態変化） |
| 分類 | 状態変化（硬球／硬化） |
| 効果概要 | 使用者が **硬球（Defense Curl）状態** になり、特定技の威力が増加する |
| 発生源 | 技「まるくなる」 |
| 対象 | 使用したポケモン自身 |
| 重ね掛け | **不可**（既に状態中は再付与・延長なし） |
| 状態解除 | 交代／戦闘不能／一部特殊処理（世代仕様） |

## 発動条件 / 無効条件

### 発動条件

- 技「まるくなる」を **使用して成功** すること  
- 技自体の効果は **防御ランクを上昇させる** 仕様  
- 状態変化「硬球」は **技成功時点で付与** される  
- 命中率判定はなく、技の使用成立で状態発生  

### 無効・付与不可

| 条件 | 結果 |
|---|---|
| 既に「硬球」状態中 | 再付与・延長なし |
| 技使用が **失敗／無効化（まもる等）** | 状態付与なし |
| 対象が **交代／戦闘不能** | 状態付与なし |

## 効果詳細

### 防御強化

- 状態付与と同時に **ぼうぎょランクが1段階上昇** する。  
  - これは単純な能力ランク変化であり、以後のダメージ計算に反映される。 ([turn0search1][turn0search14])

### 硬球状態の効果

- 状態変化「硬球」は以下の技に対して追加効果をもたらす（世代依存仕様あり）：  
  - 技「ころがる」「アイスボール」などの **威力が2倍になる**（内部倍率）  
  - 硬球状態の効果自体は **技の性能補正として扱われる**  
  - この強化は **状態が付与されている限り持続**  
- 「硬球」状態は **まるくなるの付与によってのみ発生する**。  
- 状態自体は **内部的な強化フラグ扱い** であり、他の状態とは別途管理されることがある。 ([turn0search2][turn0search10])

## 維持と解除

| 条件 | 結果 |
|---|---|
| 対象が **交代／戦闘不能** | 「硬球」状態解除 |
| 世代仕様でのバトンタッチ | 第3世代以降では状態は **引き継がれない**（ただし上昇したぼうぎょランクは残る場合あり） |
| 通常のターン経過 | 状態は **持続** |

## 競合・相互作用

| 条件・効果 | 挙動 |
|---|---|
| 同一技「まるくなる」連発 | 状態は **上書き・重ね不可** |
| 技「ころがる」「アイスボール」 | 「硬球」状態で **威力2倍** の補正が入る |
| 能力ランク変化との共存 | ぼうぎょランク上昇は **他の補正と重複** して加算処理される |
| 状態変化解除技（例：リフレッシュ等） | 「硬球」状態の解除とは通常関連しない |

## ターン処理順（概要）

1. 技「まるくなる」を **使用・成功**  
2. 対象の **ぼうぎょランクが上昇**  
3. 状態変化「硬球」が **付与**  
4. 次以降の攻撃処理で「硬球」効果が **反映**

## 補足

- 「まるくなる」は **状態変化を主目的とする技** であり、直接ダメージは与えない。  
- 付与される「硬球」状態は **攻撃性能強化に絡む内部フラグ扱い** で、第9世代でも基本同様の仕様が継続している。  
- 状態は **視認表示されない場合がある** が、効果自体は内部で扱われる。 ([turn0search0][turn0search12])
