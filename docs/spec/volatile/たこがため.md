# たこがため（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | たこがため（状態変化） |
| 分類 | 状態変化（技「たこがため」による付与） |
| 効果概要 | 対象ポケモンは毎ターン終了時に **防御・特防ランクが各1段階低下** |
| タイプ | 技「たこがため」は かくとうタイプ技（状態変化付与） |
| 付与方法 | 技「たこがため」命中後に状態として付与 |
| 持続条件 | 対象が **場に居続ける間継続** |
| 重ね掛け | 同一状態は **重ね不可・延長不可** |
| 状態解除 | 交代 / 戦闘不能 / 効果解除（使用者交代） |

## 発動条件 / 無効条件

### 発動条件

- 技「たこがため」が **対象に命中** した時に付与される。  
- 対象がゴーストタイプの場合、技効果・状態付与は発生しない（命中しても効果なし）。  
- みがわり中の場合、**状態付与自体は付くが効果は妨害される**（みがわりが消えるまで防御低下は発生しない）。  

### 無効条件

- 技「たこがため」が **命中前に無効化**（まもる等）された場合、状態付与は発生しない。  
- 対象が **すでにたこがため状態** の場合、重ね掛け・効果の延長は発生しない。  
- みがわり中であり、使用者の特性が **すりぬけでない・効果防止特性あり** の場合、ランク低下が阻止される。 ([turn1search0])

## 効果詳細（持続ダメージ処理）

| 処理タイミング | 効果 |
|---|---|
| 各ターン終了時 | 対象の **ぼうぎょランク・とくぼうランクをそれぞれ–1段階** 低下 |
| ランク最低時 | ランクがすでに最低（–6）でも状態は残り、メッセージ表示のみ控える |

### 補足処理

- みがわり状態の場合、みがわりが残っている間はランク低下は発生しないが状態は保持される。  
- 状態効果によりランク変化が起こるため、特性「まけんき／かちき」等は **防御低下に応じて発動可能**。  
- 特性「あまのじゃく」の場合、ランク低下が逆に **ランク上昇として適用される** ことがある。  
- 防御低下耐性特性（クリアボディ／しろいけむり／メタルプロテクト等）や場効果によって **ランク低下そのものが無効化**される場合がある。 ([turn1search0])

## 維持と解除

たこがため状態は以下の条件で解除される：

- 対象ポケモンが **交代・戦闘不能になる**  
- 技使用者（付与源）が **交代・戦闘不能になる**  
- 状態解除処理（例: みがわり以外の状態リセット）  
- 戦闘終了時

## 相互作用・例外

| 条件／効果 | 仕様 |
|---|---|
| 技「たこがため」使用者の **すりぬけ特性** | みがわり越しでもランク低下が適用される |
| みがわり状態（対象） | 状態は付与されるがランク低下はみがわりが壊れるまで行われない |
| 特性「クリアボディ」「しろいけむり」「メタルプロテクト」など | ランク低下処理が **発生しない** |
| 特性「あまのじゃく」 | 防御・特防ランク低下が **逆に上昇** として扱われる |
| 特性「まけんき／かちき」 | ランク低下ごとに **攻撃／特攻が上昇** 可能 |
| ゴーストタイプ（対象） | 技の効果として **状態付与が無効** |

## ターン処理順（概要）

1. 技「たこがため」命中 → 対象へ状態変化付与  
2. 対象ポケモン行動処理  
3. ターン終了時 → ランク低下処理（–1 各）  
4. 次ターン以降も同様に処理（交代等で解除するまで）
