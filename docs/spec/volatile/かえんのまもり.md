# かえんのまもり（Burning Bulwark／技）仕様書

## 基本情報

| 項目 | 内容 |
|---|---|
| 表示名 | かえんのまもり |
| 英語名 | Burning Bulwark |
| 分類 | 変化技 |
| タイプ | ほのお |
| 対象 | 自分 |
| PP | 10（最大16） |
| 優先度 | +4 |

## 効果概要

**かえんのまもり** は自身に炎の防護をまとわせ、そのターンに受ける攻撃を防ぎつつ、物理接触してきた相手を **やけど状態にする** 技です。  
- 技使用中に受ける物理・特殊攻撃を一定条件内で防ぐ。  
- 接触攻撃を防いだ場合、相手を**やけど**状態にする。  
- 連続使用すると成功率が低下する。  
- みがわり越しでもやけど付与は発生する。  
- 技による防護効果中に他の保護系技と同様の成功率低下要因が共有される。 :contentReference[oaicite:0]{index=0}

## 発動条件 / 失敗条件

### 発動条件

- 使用したターンに技を選択・実行する。  
- 自身が行動可能な状態である。  
- 直前に **保護系技使用で失敗していない**。 :contentReference[oaicite:1]{index=1}

### 失敗条件

- **そのターンの末行動で使用した場合（行動順が最後）** は失敗する。  
- 同一ターン内に **過去の保護系技やかえんのまもりを連続使用** している場合、成功率が低くなり失敗する可能性がある。  
- 命中判定・回避ランクの影響はなく、行動順で失敗判定のみ発生。 :contentReference[oaicite:2]{index=2}

## 効果詳細

### 防護効果

- 使用したターン中に、**自分に向けられた攻撃技の無効化** を行う。  
  - 物理・特殊の判定に限らず、攻撃ダメージと付随効果を受けない。  
  - ただし **フェイント / ゴーストダイブ / シャドーダイブ / いじげん系** の攻撃を受けた場合、かえんのまもりの防護は解除され、通常通りダメージを受ける。  
  - 保護効果は「物理・特殊攻撃のみ保護」と扱われる一部例外あり。 :contentReference[oaicite:3]{index=3}

### やけど付与

- 防護中に **接触する攻撃技を受けた場合**、相手ポケモンを **やけど状態** にする。  
  - 対象がやけど耐性を持つ場合やけど付与は発生しない。  
  - 防護が解除されたケースでも **やけど付与は発生しない**。  
  - みがわり越しの接触でもやけどは発生する。 :contentReference[oaicite:4]{index=4}

## 持続と解除

- 防護・やけど付与の効果は **そのターンのみ** 発生し、次ターンには消滅する。  
- 技の防護効果は継続ターンや持続フラグを持たない。 :contentReference[oaicite:5]{index=5}

## 競合・例外

| 相互作用 / 条件 | 効果 |
|---|---|
| 特性「ふかしのこぶし」 | かえんのまもりの防護効果を **無視して攻撃できる技もある** |
| 保護系技の成功率低下 | まもる系技、ワイドガード等と同様に **成功率が低下** する（共有） |
| みがわり状態（自身/相手） | みがわり越しでもやけど付与は発生 |
| いじげん系技 | 防護を解除し攻撃を成立させる |

## ターン処理順（概要）

1. 技「かえんのまもり」を **使用 → 防護状態付与**  
2. 同ターン内に敵の攻撃が来た場合、防護処理優先  
3. 接触攻撃なら **やけど状態を付与**  
4. ターン末時まで保護効果維持（防護判定）  
5. 技効果消滅

## 補足

- かえんのまもりは「保護系技」扱いであり、保護系成功率低下システムに組み込まれている。  
- やけど付与と防護効果は独立した扱いであり、防護成功後の接触判定でのみ発生する。 :contentReference[oaicite:6]{index=6}
