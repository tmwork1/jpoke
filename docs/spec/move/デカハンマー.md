デカハンマー は、ポケモンの技の一種。
## 説明文
- 第九世代: 大きな ハンマーを 体ごと ぶんまわして 攻撃する。 この技は 2回連続で だせない。

## こんなときに使おう
高威力の技にしては反動ダメージ・行動不能になるなどのデメリットはないが連続で選べない。当然、 こだわり系アイテム との相性も悪い。次のターン、相手はデカハンマーを使われないことを分かって行動できるので、交代や起点化を妨害できる変化技を覚えておくとよい。

## 技の仕様
- 使用した次のターンにデカハンマーを選ぼうとすると、「デカハンマーは 2回 続けて 出すことが できない!」というメッセージが出て選択できない。
  - デカハンマー以外の技なら選択して行動できる。
  - 使用直後に 交代 した場合は、次に場に出た直後に選択できる。
  - デカハンマーしか選べる技が無い場合、使用した次のターンに「たたかう」を押すと わるあがき を使用することになる。
- デカハンマーの PP が消費されなかったときは、次のターンも選ぶことができる。
  - PPを消費したが まもる 状態で防がれたときや、 外れた ときは次のターン選択できなくなる。
  - デカハンマーを選択したが 状態異常 ・ 状態変化 で動けなかったときは、次のターン続けて選択できる。
- デカハンマーを使用した後は、他の技のPPを消費しなければ再び選べるようにならない。
  - 他の技を使ったがまもる状態で防がれたときや、外れたときは次のターン選択できるようになる。
  - 他の技を使おうとしたが状態異常・状態変化で動けなかったときは、次のターンはデカハンマーを選べない。ただし、わるあがきを出そうとしたが動けなかった場合は、次のターンからデカハンマーを選べる。
  - バッグ から道具を使用しても、デカハンマーを選べるようにはならない。
- デカハンマーの効果は次のターン技の選択ができなくなるだけで、使用自体は制限されない。選択画面を経ずにデカハンマーが出された場合は2回連続で使用できる。
  - 使用直後に、 他の技が出る技 や さいはい でデカハンマーを出したときは成功する。
  - 他の技が出る技でデカハンマーを出した直後に、覚えているデカハンマーを選ぶことはできない。
  - 使用した次のターン、行動する前に アンコール を受けてデカハンマーの使用を強制されたときはデカハンマーを出せる。次のターンからアンコールが解除されるまでの間は、わるあがきとデカハンマーを交互に使用する。
  - 先制 さいはい によってデカハンマーを出した後に通常行動で別の技を出していれば、次のターンはデカハンマーを選択できる。
- デカハンマーの使用直後にアンコールを受けたときは、次のターンからわるあがきとデカハンマーを交互に使用する。
