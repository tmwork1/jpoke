## こんなときに使おう
他にじめん技を覚えないポケモンであれば候補になるが、 溜め技 に共通した欠点として潜っている間に 交代 ・ まもる ・ みがわり ・能力上昇などの猶予を与えることになる。攻撃技として使うより、 どくどく などと組み合わせたターン稼ぎを目的として使われることもある。潜っている間でもダメージ2倍で当たってしまう技として、採用率が高い技である じしん もあるため、他の溜め技より1ターン稼ぐ目的も果たしにくい点もあなをほるの弱点と言える。 パワフルハーブ と併用するか、 Zワザ / ダイマックス に変えて使えば溜め無しで攻撃できる。
シナリオでは隙になることは少なく、威力80とそこそこの威力があるじめん技として使える。また第六世代以前と BDSP では、フィールドで あなぬけのヒモ と同じ効果として使えるため、移動要員の技スペースが余ったとき覚えさせる技の候補となる。

## ポケモン不思議のダンジョンにおけるあなをほる

## 技の仕様
### バトルにおける効果
- 1ターン姿を隠す 溜め技 の1つ。溜め技に共通する仕様は 溜め技 の記事を参照。
- スカイバトル では使用できず、 ゆびをふる などでこのわざが選ばれた場合失敗する。
### フィールド上における効果
- 第六世代 までと、 BDSP では洞窟・森・塔などのフィールドで使うと入り口に戻る( 第一世代 では最後に使った ポケモンセンター に戻る)。
  - 第一世代ではテレポートブロック（シルフカンパニーなどにある移動装置）の上で使用すると、移動元ではテレポートブロックの演出が優先され、主人公が回転せずにいきなり画面上部に飛び去り、移動先では通常時同様に回転しながら着地する。
    - あなぬけのヒモ や テレポート を使った際と同様に、ワープする際に主人公の姿が「アイウエ」(文字が反転している)となる現象がある。この現象は スーパーゲームボーイ や ポケモンスタジアム のGBビルでプレイした場合は起こらない。
  - 第一世代で洞窟などのダンジョン以外で使おうとすると、本来道具を不適切な場所で使ったときのメッセージと同じ「オーキドのことば…（以下略）」が表示される。 そらをとぶ や いあいぎり の場合は専用のメッセージが表示される。
- 第七世代 からはフィールドで使うことができなくなった ( BDSP を除く)。
## 備考
- 第一世代 の攻略本ではほとんどの書籍で威力60と書かれていた。 第二世代 では実際に威力60に変更されている。
- SM / USUM ではわざマシンに収録されていないので習得できるポケモンが限られる。
- BDSP ではフィールドわざとして復活したが、本作の他のわざマシンと異なり、あなをほるが収録されている わざマシン28 は1つのセーブデータにつき1つしか手に入らない。
- 本技を覚える ホルビー 、 ホルード は、 分類 も「あなほりポケモン」である。
- ポケモンバトリオ では、 サイホーン のわざとして登場する。
