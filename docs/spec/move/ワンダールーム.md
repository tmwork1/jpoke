## こんなときに使おう
防御と特防の能力値が離れているポケモンであっても疑似的な両受けができるようになる。例えば イシヘンジン などの「防御は高いが特防は低いポケモン」は特殊技に、 ウツロイド などの「特防は高いが防御は低いポケモン」は物理技に強くなる。ワンダールーム発動中は本来強かったはずの分類の技に弱くなるが、再度ワンダールームを使うことで能力を戻せる。

## 技の仕様
- すでにワンダールーム状態のときに使用すると、ワンダールーム状態を解除する。
## 備考
- ポケットモンスターシリーズの全てのわざを五十音順に並べると、このわざが最後になる。
- ムウマ の タマゴわざ だが、 ブリリアントダイヤモンド・シャイニングパール 単体では 輝きの空洞 ・ 星影の大空洞 でランダムで出現する、この技を覚えているムウマからしか遺伝経路が無い。他のソフトから遺伝経路として連れて来れるポケモンも、ワンダールームを覚えたムウマ系統のみとなる。
