## こんなときに使おう
主に ゴースト タイプのポケモンが覚える、2ターンかけて攻撃するゴーストタイプ 物理技 。他の 溜め技 と同様に、ただ使っても1ターンの猶予を与えてしまい、能力アップや交代の隙を作って逆に不利になることも考えられるが、他の溜め技と異なり まもる を無効化できる特長がある。1ターンは相手の攻撃から逃れられる所を生かし、 どく や やけど のターン稼ぎとして使うという方法もある。シングルバトルなら普通に ノーマル タイプに交代されたら無効化されてしまうので、 くろいまなざし などで交換を縛る、または ステルスロック を撒くなど交換を躊躇させる準備があると決まりやすい。
ダブルバトル なら、ただ居座れば巻き込まれる味方の全体攻撃をかわすために使いつつ、シングルバトルよりも重要な技となる まもる や ワイドガード などを牽制できることもできる。

## 技の仕様
- 1ターン姿を隠す 溜め技 の1つ。溜め技に共通した仕様は当該項目を参照。
- 威力とPPを除き シャドーダイブ と同じ性能を持つ。詳細な仕様は シャドーダイブ#技の仕様 、および シャドーダイブ を参照とする。
## 備考
- ギラティナ の 専用技 である シャドーダイブ の威力を落としてPPを増やしたような性能を持つ。こちらは多くのポケモンが覚えられる。
- 専用技や使用条件がある技を除けば、ゴーストタイプの物理技で最も威力が高い。
- ゴーストタイプ以外でこの技を覚えるポケモンは ミュウ / アルセウス / イベルタル / フーパ （ときはなたれしすがた）のみ（ アルセウス は プレート を持たせることでゴーストタイプになれる）。
