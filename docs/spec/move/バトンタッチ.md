## 説明文
### たたかうわざ
- 第二世代 ・ ルビー・サファイア ・ エメラルド: ついかこうかを のこしたまま ほかの ポケモンと いれかわる
- ポケモンスタジアム金銀: じぶんに かかっている ほじょこうかを こうたいする ポケモンに ひきつがせる。
- ファイアレッド・リーフグリーン: じぶんの のうりょくの へんかを のこしたまま ひかえの ポケモンと いれかわる。
- 第四世代 ・ 第五世代 ・ 第七世代 ・ 第八世代 ・ 第九世代: ひかえの ポケモンと いれかわる。のうりょく へんかは かわった ポケモンが そのまま うけつぐ。
### コンテストわざ
- 第三世代: このあと アピールする ポケモン みんなを きんちょう させる
- 第四世代: ボルテージが ひくければひくいほど たかいひょうかになる。
- 第六世代: ちょうしが いいときに だすと アピールが とても うまくいく
- 第八世代: ついかこうか なし

## こんなときに使おう
バトンタッチと積み技を覚えたポケモン（バトン役）とエースポケモン（バトン先）を用意し、バトン役をバトン先のサポート係とする使い方がある。 かそく などの特性や じゃくてんほけん などの持ち物を併用しても使われる。引き継ぐと有利になる みがわり などの状態変化だけでなく、 こんらん などのデメリットとなる状態変化も引き継ぐ欠点はある。
能力変化を引き継ぐためでなく、相手から技を受けずに交代先のポケモンを場に出したいときにも使われる。単に 交代 するときは交代先のポケモンが攻撃されることになるが、相手より後にバトンタッチで行動できればバトン先を攻撃されずにすむ。
## ポケモン不思議のダンジョンにおけるバトンタッチ
- 冒険団シリーズ まで: 同じ部屋にいる全てのポケモンの場所を変える。 ばしょがえだま の広範囲版。
- マグナゲートと∞迷宮: 味方と場所を入れ替える。このとき入れ替わる味方の強さが上がる。

## 技の仕様
### 交代効果
- 交代先のポケモンは自由に選ぶことができる。
- 交代できる控えのポケモンがいないときに使うと失敗する。
- 特性 かげふみ / じりょく / ありじごく や、 ねをはる / にげられない / バインド 状態、場の フェアリーロック 状態の影響を受けずに交代できる。
- バトンタッチによる交代は おいうち の対象にならない。バトンタッチを使用する前においうちを受けた場合も威力は2倍にならない。
- バトンタッチによる交代でも、バトン先は 設置技 の効果を受ける。
  - みがわり 状態を引き継いでも設置技の効果は防げない。
  - でんじふゆう / テレキネシス 状態を引き継いだときを含め、バトン先が 地面にいない 場合は まきびし ・ どくびし / ねばねばネット 状態の効果を受けない。
  - ねをはる 状態を引き継いだ場合、バトン先は地面にいるものとして設置技の効果を受ける。
    - ただし、 第四世代 でバトン先として ひこう タイプのポケモンを出した場合や、でんじふゆう状態も同時に引き継いだときはまきびし/どくびし状態の効果を受けない。
  - とくせいなし 状態を引き継いだ場合、バトン先が特性 ふゆう のポケモンでも地面にいるものとして設置技の効果を受ける。バトン先が 場に出たときに発動する特性 を持つ場合も発動しない。
- 第三世代 ・ 第四世代 では、バトン先のポケモンが出てきた直後に受ける スケッチ / ものまね は成功し、バトンタッチを覚えられる。 第二世代 と 第五世代 以降ではスケッチ/ものまねは失敗する。
- 第四世代 ではバトン先のポケモンもバトンタッチを覚えている場合、場に出た直後に受ける アンコール ・ うらみ ・ かなしばり は成功して、バトン先が覚えている技に対してこれらの効果がかかる。 第五世代 以降ではバトン先に対するこれらの技は失敗する。
### 能力変化の引き継ぎ
- 能力変化 はマイナスのランクも引き継ぐ。
  - マイナスのランク補正を引き継いだバトン先が しろいハーブ を持つ場合、場に出た直後に発動させる。
  - マイナスのランク補正を引き継いでも、バトン先の だっしゅつパック は発動しない。
  - 場が しろいきり である場合でもマイナスのランクは引き継がれる。
- バトン先のポケモンが特性 クリアボディ / たんじゅん 5 - / あまのじゃく 、持ち物 クリアチャーム などを持つ場合でも、能力変化は特性・持ち物の効果で変化すること無くそのまま引き継がれる。
- 能力上昇を引き継いだだけでは、そのターンに しっとのほのお を受けて やけど 状態になることや、 みわくのボイス を受けて こんらん 状態になることはない。
- 能力上昇の引き継ぎ時に相手の場の びんじょう / ものまねハーブ は発動しない。
- きゅうしょパワー / キョダイシンゲキ による 急所 ランクの上昇は引き継がれない。
  - きゅうしょアップ 状態は引き継がれる。 ドラゴンエール で発生した効果の場合は上昇段階ごと引き継がれる。
### 状態変化の引き継ぎ
- 世代ごとの、引き継がれる 状態変化 と引き継がれない状態変化の一覧は以下の通りとなる。
- 持続ターンがある状態変化の場合、そのターン数ごと引き継がれる。
- みがわり 状態の残りHPは引き継がれる。
- 最大HP割合で決まる回復量やダメージ量は、交代先のポケモンの最大HPで決まるようになる。
- 状態異常 は引き継がれない。
- バトン先が状態変化に耐性を持つ場合、引き継いだ状態変化が回復することがある。ただし、基本的にはバトンタッチで引き継がれると本来耐性を持つ状態変化にかかってしまうケースが多い。
  - 特性 マイペース のポケモンに こんらん 状態を引き継ぐと、 第三世代 ・ 第四世代 では場に出た直後にこんらんが治る。 第五世代 以降では直後ではなく、ポケモンやトレーナーが何かしらの行動を完了した後にマイペースが発動しこんらんが治る。
  - とくせいなし 状態を無効化する特性のポケモンにとくせいなし状態を引き継ぐ場合、とくせいなし状態は治る。
    - とくせいガード を持ったポケモンにとくせいなし状態を引き継いだ場合は治らない。
  - テレキネシス にならない性質を持つポケモンの内、 ディグダ / ダグトリオ / スナバァ / シロデスナ に引き継ぐ場合はテレキネシス状態は治らない。 メガゲンガー に引き継ぐときはテレキネシス状態が治る。
  - 第六世代 ・ 第七世代 では、 ゴースト タイプに引き継がれた にげられない 状態は治る。
  - くさ タイプ自体に やどりぎのタネ 状態を回復する効果は無いので、やどりぎのタネ状態が引き継がれても状態は治らない。
- パワートリック 状態をバトンタッチする場合、第五世代以降ではパワートリックを使ったことがあるという情報が引き継がれる。
  - パワートリック を偶数回使って攻撃と防御が元に戻ったポケモンがバトンタッチを使用した場合、 第四世代 ではバトン先のポケモンの攻撃と防御は入れ替わらない。 第五世代 以降では、奇数回のパワートリックを引き継いだときと同様に、バトン先のポケモンの攻撃と防御は入れ替わる。
  - 第五世代以降で、パワートリック状態で フォルムチェンジ / メガシンカ して攻撃と防御が元に戻ったポケモンがバトンタッチを使用する場合、バトン先のポケモンの攻撃と防御は入れ替わる。
### 第三世代以前のみ
- 第二世代 のみ、 もうどく 状態のポケモンがバトンタッチで どく 状態・ やけど 状態のポケモンに交代した場合、もうどくの効果が引き継がれ交代後のポケモンはどく・やけどのダメージが増加する。バトン先が状態異常がないポケモンであっても、どく状態・やけど状態になればもうどくの継続カウントが再開する。
- バトルアリーナ で使うと失敗する。
- 日本語版 エメラルド では以下2つの条件を満たしているとき、プレイヤー側のポケモンは通常の交代を行うだけでバトンタッチを使用したときのようにランク補正や状態変化を引き継いでしまうバグがある。
  1. 相手がCPUのトレーナー戦であり、技の使用傾向 (参照 ) において「チェック2」が入っている。
  2. CPU側のポケモンが覚えている技の4番目がバトンタッチである。
