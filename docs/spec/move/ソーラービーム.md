## 説明文
### たたかうわざ
- 第二世代 ・ ルビー・サファイア ・ エメラルド: 1ターンめで ひかりを きゅうしゅう つぎのターンで てきを こうげき
- ポケモンスタジアム金銀: くさタイプの さいきょうのわざ。さいしょに ひかりを きゅうしゅうし つぎに ビームを はっしゃする。
- ファイアレッド・リーフグリーン: 1ターンめで ひかりを きゅうしゅうして 2ターンめで あいてに ほうしゅつ して こうげきする。
- ファイアレッド・リーフグリーン （ わざマシン22 ）: 1ターンめで ひかりを きゅうしゅうして つぎのターンに はっしゃして あいてを こうげきする。
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代 ・ 第九世代: 1ターンめに ひかりを いっぱいに あつめ 2ターンめに ひかりの たばを はっしゃして こうげきする。
### コンテストわざ
- 第三世代: たくさん アピール できる
- 第四世代: ほかの ポケモンと しんさいんが かさならないと +3。
- 第六世代: 1つまえの ポケモンの アピールの うまさに えいきょう される
- 第八世代: 10びょうの あいだ ハートゲージが とても たまりやすくなる

## こんなときに使おう
多くの 溜め技 は まもる や 交代 で受けられやすいというデメリットがあるが、ソーラービームの場合は 晴れ 状態だと1ターンで攻撃できる特長がある。これを生かして晴れパの主力技として使用できる。
肝心の くさ タイプは晴れパで使うと ほのお タイプまで強化されてしまうので採用率が芳しくなく、同じ威力で溜めなしで攻撃できる はなびらのまい や安定技の エナジーボール が主に使われる。
ほのおタイプで習得できるポケモンも多く、弱点を突かれる みず / いわ / じめん タイプへの対策になる。 パワフルハーブ / Zワザ / ダイマックス などで溜めずに使用する手段があるなら、晴れパに限らずとも採用され得る。
第一世代 の 対戦 では、 モンジャラ が かげぶんしん と ねむる で粘りながら せいちょう を最大まで積んでからこの技を撃つことにより、相手のかげぶんしん+ねむる戦術を対策するという戦術が考えられる。

## 技の仕様
- その場で溜める 溜め技 の1つ。溜め技に共通する仕様は当該項目を参照とする。
- 天気 が にほんばれ 状態のときに使用すると、溜めずに攻撃できる。
  - 技を選んでから溜め始めるまでの間に天気が変わった場合、溜め始めたときの天気がにほんばれ状態であることが溜めが無くなる条件となる。
  - 溜め始めた後から天気がにほんばれ状態に変わっても溜めが無くなることはない。
  - おおひでり 状態のときも、にほんばれ状態と同じ扱いを受け、溜めずに攻撃できる。
- パワフルハーブ を持っているときに使用すると、道具を消費して溜めずに攻撃できる。
  - 天気がにほんばれ状態でもある場合は、天気の効果が優先されてパワフルハーブは消費しない。
- 溜め始めた際、「<使用者>は ひかりを きゅうしゅうした!/光を 吸収した!」というメッセージが流れる。
  - にほんばれ状態の効果で溜めずに攻撃するときは、 第四世代 までは溜めのメッセージは流れず、即座に「<使用者>の ソーラービーム!」とメッセージが出る。 第五世代 以降では、溜めのメッセージが流れた直後に「<使用者>の ソーラービーム!」のメッセージが出る。
  - パワフルハーブの効果で溜めずに攻撃するときは、第四世代から一貫して溜めのメッセージが流れた後、パワフルハーブの発動と攻撃のメッセージが流れる。
- 天気が あめ / すなあらし 3 - / あられ / ゆき 状態のときに ダメージ を与えた場合、 第四世代 まではダメージが半分に、 第五世代 以降では威力が半分になる。
  - 溜め始めてから攻撃するまでの間に天気が変わった場合、攻撃時に発生している天気がダメージに影響する。
  - 第二世代 のすなあらし状態のみ、ダメージに影響は無い。
  - おおあめ 状態のときも威力が半分になる。
  - らんきりゅう 状態では威力に影響は無い。
  - 天気の影響で威力が半分になった場合でも、特性 テクニシャン の補正は受けられない。
- 特性 ノーてんき / エアロック のポケモンが場にいるときは、天気の影響は完全に無視される。
- 使用者が ばんのうがさ を持っている場合、にほんばれ状態でも溜めるようになるが、あめ状態でも威力が下がらなくなる。すなあらし/あられ/ゆき状態では威力が半減する。
  - 技の対象がばんのうがさを持っていても影響は無い。
- ノーてんき/エアロック/ばんのうがさの効果でにほんばれ状態でも技を溜め始めた場合、攻撃までにノーてんき/エアロック/ばんのうがさの効果が失われたとしても溜めは無くならない。
### 第四世代
- 天気が きり 状態のときもダメージが半分になる。
- ダイヤモンド・パール のみ、にほんばれ状態で相手がいないときにソーラービームを使うと、ソフトが応答しなくなる。
  - プラチナ ・ ハートゴールド・ソウルシルバー ではこのバグは解消されている。
## 備考
- 第六世代 では多くの特殊技の威力が下方修正された中で、ソーラービームの威力に関しては変更されていない。
- ポケットモンスター Let's Go! ピカチュウ・Let's Go! イーブイ では、晴れ状態や持ち物のシステムが無いため溜めずに使うことができない代わりに、威力200になっている。
