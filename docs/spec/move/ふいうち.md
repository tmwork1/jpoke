# 仕様調査書: ふいうち（Sucker Punch）

**対象世代**: 第9世代（スカーレット・バイオレット）  
**対象ルール**: シングルバトル  
**仕様範囲**: 技仕様（対戦影響のみ）

---

# 1. 技概要

| 項目 | 値 |
|------|------|
| 技名 | ふいうち |
| 英語名 | Sucker Punch |
| タイプ | あく |
| 分類 | 物理 |
| 威力 | 70 |
| 命中率 | 100 |
| 優先度 | +1 |
| 基本PP | 5 |

---

# 2. 発動条件

## 2.1 成功条件
- 対象が **ダメージを与える攻撃技** を選択している場合のみ成功する。

---

## 2.2 失敗条件（条件不成立）

以下のいずれかに該当すると **技は必ず失敗** する。

### 対象の行動条件
- 対象が **攻撃技以外の技（変化技）を選択** している場合  
- 対象が **交代・テラスタル・どうぐ使用など技以外の行動** を選択している場合  

### 行動順条件
- 対象が **ふいうちより先に行動した場合**
  - 優先度が高い技を使用
  - 同じ優先度で素早さが高く先行
- 対象が **行動不能状態** で攻撃技が実行されない場合  

---

# 3. 優先度処理

- 優先度は **+1**  
- 行動順は以下の順で決定される。

優先度 → 素早さ → その他行動順補正

---

# 4. ダメージ計算

## 4.1 ダメージ式
- 通常の物理技ダメージ計算式を使用する。

---

## 4.2 補正適用
- タイプ一致補正（STAB）
- タイプ相性
- 急所補正
- 天候・フィールド・特性・どうぐ補正
- テラスタル補正

---

# 5. 相互作用仕様

## 5.1 優先度技との関係
- 対象が **より高い優先度技** を使用した場合、ふいうちは必ず失敗。
- 対象が **同一優先度技** を使用し、対象が先行した場合も失敗。

---

## 5.2 行動不能時
- 対象が攻撃技を選択していても、  
  行動不能（まひ・こおり・ひるみ等）で攻撃が実行されなければ失敗。

---

# 6. 対象選択仕様

- 通常の単体攻撃技と同様に対象指定される。
- まもる等で無効化可能。

---

# 7. 状態異常・バリア相互作用

## 防御系効果
- まもる系統技により無効化される。
- みがわりに吸収される。

---

# 8. テラスタル相互作用

- テラスタルによるタイプ変更・威力補正は通常通り適用される。
- テラスタル選択自体は攻撃技ではないため、  
  対象がテラスタルのみを行動として選択した場合、ふいうちは失敗条件に該当する。

---

# 9. その他仕様

## 技カテゴリ
- 物理技カテゴリに属するため、やけどによる攻撃低下の影響を受ける。
