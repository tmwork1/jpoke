## 技の仕様
- その場で溜める 溜め技 の1つ。溜め技に共通する仕様は当該項目を参照。
- 切る技 の1つ。特性 きれあじ のポケモンが使うと威力が1.5倍になる。
- 溜め行動や 天気 に関する効果は ソーラービーム と同様の仕様を持つ。詳細な効果は ソーラービーム#技の仕様 も参照とする。
  - 天気が にほんばれ 状態のときに使用すると、溜めずに攻撃できる。
  - パワフルハーブ を持っているときに使用すると、道具を消費して溜めずに攻撃できる。にほんばれ状態でもある場合は、天気の効果が優先されてパワフルハーブは消費しない。
  - 溜め始めた際、「<使用者>は ひかりを きゅうしゅうした!/光を 吸収した!」というメッセージが流れる。にほんばれ状態/パワフルハーブの効果が発動したときでも、溜めのメッセージが流れた後に「<使用者>の ソーラーブレード!」のメッセージが出る。
  - 天気が あめ / すなあらし / あられ / ゆき 状態のときに ダメージ を与えた場合、威力が半分になる。 らんきりゅう 状態では威力に影響は無い。
  - 場に特性 ノーてんき / エアロック のポケモンがいる場合、天気の影響は完全に無視される。
  - 使用者が ばんのうがさ を持っている場合、にほんばれ状態でも溜めるようになり、あめ状態でも威力が下がらなくなる。すなあらし/あられ/ゆき状態では威力が半減する。
  - 一度溜め行動を始めると、攻撃までの間ににほんばれ状態の効果を得られたとしても溜め行動は無くならない。
## 備考
- 似た効果の 特殊技 である ソーラービーム と比べて、威力が5高い。
- 第七世代 では ラランテス の 専用技 だった。 第八世代 以降では剣をモチーフにしたポケモンの習得者が増えた。
- ソーラービームと異なり、ほのおタイプの習得者は限られる。第八世代では ギャロップ 系統、 第九世代 では ソウブレイズ / オーガポン (かまどのめん) が覚える。
