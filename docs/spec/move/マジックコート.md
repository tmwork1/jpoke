## 説明文
### たたかうわざ
- ルビー・サファイア ・ エメラルド: とくしゅな わざの こうかを はねかえして あいてに かけてしまう
- ファイアレッド・リーフグリーン: どく まひ ねむり こんらんの わざや やどりぎのタネを あいてに はねかえす。
- 第四世代: じょうたい いじょうや のうりょくを さげる わざ やどりぎのタネ などを だされたとき あいてに はねかえす。
- 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代: じょうたい いじょうに なる ワザや やどりぎのタネ などを だされたとき あいてに はねかえす。
- 第九世代: この技は 使えません 思い出すことが できなくなりますが 技を 忘れることを おすすめします
### コンテストわざ
- 第三世代: ほかの ポケモンに おどかされても がまんできる
- 第四世代: さいごの アピールで ひょうかが 2ばいに なる。
- 第六世代: 1ばん さいごに アピールすると アピールが すごく うまくいく
- 第八世代: 15びょうの あいだ ノーツはんていが しっぱいに ならない

## こんなときに使おう
使用したターンしか効果がないため、相手がマジックコートの効果対象になる技を使うタイミングを読み切る必要がある。変化技の対策は みがわり や ちょうはつ でもできるが、これらは相手より先に行動できなければ効果的でない。優先度が高いマジックコートならば、すばやさが低いポケモンでもちょうはつなどを対策できる。

## 技の仕様
- 使用したターンが終了するとマジックコートの効果は消える。
- その ターン の最後に使用したときは失敗する。
- よこどり の対象にならない。
- こだわり系アイテム を持つポケモンが使用した場合、跳ね返した技ではなくマジックコートでこだわる。
### 第四世代まで
- マジックコートの効果は一度技を跳ね返すことでも消滅する。
- 特性 プレッシャー のポケモンに対して技を跳ね返した場合、マジックコートの PP は2減る。
### 第五世代以降
- マジックコートの効果はそのターンの終了時まで継続する。
  - 技を跳ね返してもマジックコート状態は解除されない。
  - さいはい でマジックコートを使用した後、そのターン中に別の行動をしたときもマジックコート状態は解除されない。
- すでに自身がマジックコート状態であるときは失敗する。さいはいでマジックコートを続けて使用したときに起こり得る。このときもマジックコート状態は解除されない。
  - 特性 マジックミラー のポケモンがマジックコートを使用した場合は成功する。このときマジックコートの効果対象になる技を受けると、マジックコートが先に発動して特性の表示は出ない。
- 特性 プレッシャー のポケモンに対して技を跳ね返した場合でも、マジックコートの PP は1だけ減る。
## 備考
- 第三世代 では 命中 の表記が100になっているが、自分を対象に選択する変化技なので命中に関係無く マジックコート 状態になることができる。反射される技が当たるかは、跳ね返す技の命中に依存する点も 第四世代 以降と変わらない。
- 第五世代 で登場した 特性 マジックミラー のポケモンは常にマジックコート状態になる。
  - 第九世代 ではマジックコートは使用できなくなったので、特性マジックミラーでしかマジックコート状態になることができない。
- ウルトラサン・ウルトラムーン では ズガドーン のレベル技だが、教え技では覚えることができない。
