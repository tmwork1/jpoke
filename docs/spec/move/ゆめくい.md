## 説明文
### たたかうわざ
- 第二世代 ・ ルビー・サファイア ・ エメラルド: ねているてきに あたえた ダメージの はんぶん たいりょくを かいふくする
- ポケモンスタジアム金銀: ねている あいてにしか つかえない。あいての HPをすいとり はんぶんを じぶんのものにするわざ。
- ファイアレッド・リーフグリーン: ねむり じょうたいの あいての ゆめを たべて あたえた ダメージの はんぶん HPが かいふくする。
- 第四世代: ねている あいての ゆめを たべて こうげきする。ダメージの はんぶん HPを かいふくする。
- 第五世代 ・ 第六世代: ねている あいての ゆめを たべて こうげきする。 ダメージの はんぶんの HPを かいふく する。
- 第七世代 ・ 第八世代 ・ 第九世代: ねている あいての ゆめを たべて こうげきする。 ダメージの はんぶんの HPを かいふく する。
### コンテストわざ
- 第三世代: アピールが おわっている ポケモン みんなを おどろかす
- 第四世代: 2ひき れんぞくして ボルテージが あがったら +3。
- 第六世代: 1つまえの ポケモンの アピールと タイプが おなじなら きにいられる
- 第八世代: ポケモンの テンションが マイナスの ばあい もらえる ショーポイントが 2ふえる

## こんなときに使おう
眠らせる技 と組み合わて初めて有効になるHP吸収技。相手の場のポケモンが ねむり 状態でも、使用する前に 交代 されることや、先に行動した相手が起きて無効にされることもあるため、使い所は難しい。エスパー技には使用条件が無い サイコキネシス があるので、対人戦で使われることは少ない。

## 技の仕様
- ねむり 状態ではない相手に使うと失敗するが、技を選択すること自体はできる。選択してから使用するまでに相手がねむり状態になった場合は成功する。
  - 逆に、相手がねむり状態のときに選択しても、使用時までにねむり状態が回復していた場合は失敗する。
- 特性 ぜったいねむり のポケモンに対しても効果がある。
- 回復量を算出する際、小数点以下は 第四世代 までは切り捨て、 第五世代 以降は四捨五入する。
- おおきなねっこ を持たせたポケモンが使う場合は回復量が通常の1.3倍になる。
- 特性 ヘドロえき のポケモンを攻撃した場合は、 第四世代 までは特性の影響を受けずに回復できるが、 第五世代 以降は回復効果がダメージに変わる。
- かいふくふうじ 状態のポケモンは回復できない。
  - 第五世代 まで、技を使用することはできるが、HPを回復する効果が無くなる。
  - 第六世代 以降では選択することもできない。攻撃する前にかいふくふうじ状態にされた場合は失敗する。
- 第四世代 まで みがわり 状態のポケモンには無効だった。
## 備考
- 第一世代 で覚える エスパー ポケモンは スリープ ・ スリーパー ・ ミュウ のみだった。
- 第四世代 までは使用後に「てきの（又はあいての、やせいの）<相手ポケモン名>の ゆめを くった！」という特殊なメッセージが表示される。
- HP吸収技 の中では最も威力が高い。
- 本技を覚える ムンナ は、 分類 も「ゆめくいポケモン」である。
