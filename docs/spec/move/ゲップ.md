## 技の仕様
- 使用条件を満たしていないときにゲップを選ぼうとした場合、「<ポケモン名>は きのみを 食べていないので 技が だせない!」というメッセージが出て選択はキャンセルされる。このときゲップ以外に選べる技が無いポケモンは、「たたかう」を押した時点で わるあがき を出すことになる。
  - ゲップを選べずわるあがきを使用し始めた後にきのみを消費した場合、次のターンからゲップを使えるようになる。わるあがきで行動する前にきのみを消費したとしても、そのターンはわるあがきを使用する。
  - HP ・ PP ・ 状態異常 が全快した状態で開始されるバトルの1ターン目でゲップを使用する方法は無い。
- オウムがえし / さきどり でコピーできず、 ねごと / ねこのて / まねっこ / ゆびをふる / さいはい により選ばれない。
  - これらの技を出すポケモンがゲップの使用条件を満たしていても選ばれない。
- 第八世代 までは ものまね でコピーできる。 第九世代 ではできない。
### 使用条件
- 使用条件は きのみ を 食べる こと。きのみを食べるアニメーションが発動する以下の状況においてゲップは使用できるようになる。
  - きのみを持たせて普通に食べて消費したとき。
  - なげつける で相手からきのみを投げつけられたとき。
    - 追加効果 がないきのみを投げられても食べるアニメは出ず、ゲップは使えない( ブリーのみ などコンテスト用きのみ・半減の実など)。
    - 追加効果はあるが、それが発動しなかった場合は食べるアニメは出ないが、ゲップは使用できるようになる (状態異常でないときの ラムのみ など)。
  - むしくい / ついばむ を使用して相手のきのみを食べたとき。
    - 効果があるきのみを食べた場合、発動/非発動を問わずゲップを使用できるようになる。
    - 半減の実/ ナゾのみ 6 - 7 / イバンのみ / ジャポのみ / レンブのみ の効果は無く、きのみを食べるアニメは起きないがゲップを使用することができる( なげつける とは異なる)。
    - コンテスト用のきのみは食べてもゲップを使用することはできない。
  - ほおばる / おちゃかい の効果で自分のきのみを食べたとき。
    - 半減の実/イバンのみなどバトルにおける効果があるきのみを食べたなら、発動/非発動を問わずゲップを使用できるようになる。
    - 基礎ポイント を下げるきのみはバトルにおける効果が無いので、食べるアニメも発動せず、ゲップも使用できない。
  - なげつける を使用して消費したきのみを次のターン はんすう で食べたとき。
    - 半減の実/イバンのみなど、バトルにおける効果があるきのみを食べたなら、発動/非発動を問わず使用できるようになる。
    - 基礎ポイントを下げるきのみを食べてもゲップは使用できない。
- きのみを食べた後に リサイクル / しゅうかく / ものひろい などできのみが復活したとしても使用できる。
- 持ち物にきのみを持っていたとしても、それを食べる以外の方法で消費してしまうと使用できない。したがって、以下の状況においてゲップは使用できない。
  - なげつける / ギフトパス を使用して自分のきのみを手放したとき。
  - しぜんのめぐみ を使用してきのみを消費したとき。
  - むしくい / ついばむ を受けて自分のきのみを食べられたとき。
  - どろぼう / ほしがる / トリック / すりかえ 、特性 マジシャン / わるいてぐせ / きょうせい などの効果で、持っているきのみが他のポケモンに渡ったとき。
  - やきつくす / はたきおとす / ふしょくガス の効果できのみを失ったとき。
  - ストーリーにおいて、ポケモンの もちもの ではなく バッグ からきのみを食べさせたとき。
- きのみを食べた後に 交代 しても使用できる。
  - きのみを食べてから ひんし になった後、戦闘中に げんきのかけら / さいきのいのり などで復活したときでも使用できる。
  - 使用できるのはきのみを食べた戦闘中のみ。戦闘終了後に再び戦闘になれば、またきのみを食べて消費しなければ使用できない。
- きのみを食べたポケモンに へんしん したとしても、自身がきのみを食べていないかぎり使用できない。
- きのみを食べた状態は バトンタッチ で引き継がれない。
- きのみ以外の食べ物 ( きのみジュース / たべのこし など) や、HPを回復する手段 ( くろいヘドロ / かいがらのすず など) では使用条件を満たせない。
- マックスレイドバトル でのレイドボスはきのみを食べていなくてもゲップを使用できる。
  - テラレイドバトル でのレイドボスはきのみを食べてなければゲップを使えない。
