ブラッドムーン は、ポケモンの技の一種。
## 説明文
- 第九世代: 血のように 赤い満月から ありったけの 気迫を 撃ちだす。 この技は 2回連続で だせない。

## こんなときに使おう
デカハンマー#こんなときに使おう も参照。
同威力の ばくおんぱ と比べて ぼうおん に無効化されず、 ダブルバトル では味方を巻き込まないメリットがあるが、 みがわり を貫通できず、2回連続で使えないデメリットがある。
## 技の仕様
- デカハンマー と同様の仕様を持つ。 デカハンマー#技の仕様 を参照。
  - デカハンマーの次にブラッドムーンを使うことや、あるいはその逆もできる。
## 備考
- ガチグマ (アカツキ) の 専用技 。
- ガチグマ（アカツキ）の姿以外のポケモンが使用する場合、赤いエネルギー球を生み出してそこから発射する。
