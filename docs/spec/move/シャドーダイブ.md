シャドーダイブ は、ポケモンの技の一種。
## 説明文
### たたかうわざ
- 第四世代: 1ターンめで きえて 2ターンめに あいてを こうげきする。 まもっていても こうげきは あたる。
- 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代 ・ 第九世代: 1ターンめで すがたを けして 2ターンめに あいてを こうげきする。 まもっていても こうげきは あたる。
- Pokémon LEGENDS アルセウス: 闇に姿を隠しつつ攻撃する。 相手の技を回避しやすくなる。
### コンテストわざ
- 第四世代: まえの ポケモンの ボルテージマックスの あとだと +3。
- 第六世代: ほかの ポケモンに おどろかされても 1かい くらいは がまんできる
- 第八世代: 10びょうの あいだ ハートゲージが とても たまりやすくなる

## こんなときに使おう
まもる 系の状態を解除できる効果は特に ダブルバトル で生かしやすい。姿を隠している間は他のポケモンの攻撃を受けないので、 おにび などによるスリップダメージを稼げる。
他の多くの 溜め技 と異なりまもる系を無視できる利点はあるが、それでも 交代 による受け出しを許す点や、積み技を使う猶予を与える可能性があるという欠点は共通している。

## 技の仕様
- 1ターン姿を隠す 溜め技 の1つ。溜め技に共通した仕様は当該項目を参照。
- 姿を消している間の特徴については シャドーダイブ (状態変化) も参照。
- 対象が まもる ・ たたみがえし 状態の場合、それを取り除いてから攻撃する。この際、「<対象>の 守りを 打ち破った!」とメッセージが出る。
  - ワイドガード ・ ファストガード ・ トリックガード などこの技に対し直接抵抗を持たない状態でも、それらを取り除いて攻撃できる。
  - ニードルガード ・ キングシールド ・ トーチカ ・ ブロッキング ・ スレッドトラップ の効果は発動しない。
  - ノーマル タイプなどによりダメージを無効化された場合や、技が 命中 しなかった場合はまもる系の状態を取り除けない。
  - この技でまもる状態を解除された相手が次のターンに まもる 系統の技を使う場合、 第五世代 までは連続使用扱いになり成功率が下がる。 第六世代 以降は連続使用扱いにはならずに成功する。
- ダイウォール によりダメージは防がれ、守りを解除することはできない。
- 第六世代 のみ、 ちいさくなる 状態の相手に対して必中になりダメージが2倍になる。第七世代以降はこの効果は無い。
## 備考
- ギラティナ の 専用技 。
  - 配信限定技として、2009年の劇場版で配布された アルセウス も習得している。
  - 第六世代 では同様の効果で威力が低い代わりに、習得できるポケモンが多い ゴーストダイブ が登場した。
- 伝説のドラゴン の 専用技 の中で唯一の ドラゴン タイプでなく、物理技である ( ディアルガ と パルキア はそれぞれドラゴンタイプ特殊の専用技)。
- ダメージ・威力が変動する技 ・ Zワザ を除き、 ゴースト タイプのわざの中で最も威力が高い。
- Pokémon LEGENDS アルセウス でボスとして登場する アルセウス は姿を消して突進するシャドーダイブのような技を使う。
- LEGENDS アルセウス では 色違い の ギラティナ （オリジンフォルム）がシャドーダイブを使用すると、ゲームが フリーズ するバグがある 。
- ブリリアントダイヤモンド・シャイニングパール において、 228ばんどうろ にいる サイキッカー のシュウが通常は覚えられないこの技を覚えた ジュペッタ を使用する。同じ効果の ゴーストダイブ と取り違えた可能性があるが、最新バージョンであるVer. 1.3.0時点でも修正されていない。
