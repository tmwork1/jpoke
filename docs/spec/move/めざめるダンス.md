## 説明文
### たたかうわざ
- 第七世代 ・ ソード・シールド ・ 第九世代: ぜんりょくで おどって こうげきする。 この わざの タイプは じぶんの タイプと おなじになる。
- ブリリアントダイヤモンド・シャイニングパール: このわざは つかえません おもいだすことが できなくなりますが わざを わすれることを おすすめします

## 技の仕様
- 踊りの技 の1つ。使用者以外の特性 おどりこ によりコピーされる。
- 技のタイプはタイプ1と同じになるため、 タイプ無し になっていない限り タイプ一致 になる。
- 特性おどりこ/ 他の技が出る技 / スケッチ / ものまね などにより、オドリドリ以外のポケモンがこの技を使用した場合も使用者のタイプ1と同じタイプの技となる。
- 複合タイプのポケモンが もえつきる や はねやすめ を使用しタイプ1がなくなった後にこの技を使用した場合、めざめるダンスのタイプはタイプ2を参照する。
  - はねやすめしたターンにめざめるダンスを使うには、はねやすめ中に特性 おどりこ で使用するか、 さいはい を受けてはねやすめを出した後めざめるダンスを使う方法がある。
- ほのお 単タイプのポケモンがもえつきるを使用し、かつ ハロウィン / もりののろい 状態のときにこの技を使用した場合、めざめるダンスのタイプは ゴースト または くさ タイプになる。ほのお・ひこうタイプかつハロウィン/もりののろい状態のポケモンがもえつきるとはねやすめを使用したときも同様。
- ほのお単タイプのポケモンがもえつきるを使用しタイプ無しとなった後にこの技を使用した場合、めざめるダンスのタイプも無くなる。このタイプの無いめざめるダンスは以下の効果を持つ。ほのお・ひこうタイプのポケモンがもえつきるとはねやすめを使用しタイプ無しとなっている場合も同様。
  - 全タイプに等倍となる。
  - タイプ一致ボーナス は無い。
  - 対象の特性 へんしょく は発動しない。
  - 対象の特性 ふしぎなまもり により防がれる。
  - この技に対する テクスチャー2 は失敗する。
- ひこう 単タイプのポケモンがはねやすめを使用しタイプ無しとなった後にこの技を使用した場合、めざめるダンスのタイプはノーマルになる。ハロウィン/もりののろい状態の有無は問わない。
- この技を使用した際、使用者の特性 ノーマルスキン / フェアリースキン / フリーズスキン / スカイスキン / エレキスキン の効果は発動しない。
- そうでん 状態の効果を受けると、使用者のタイプに関係無く でんき タイプになる。タイプ1がノーマルのポケモンが プラズマシャワー 状態のときに使用したときもでんきタイプに変わる。
- Zワザ に変えるときは、自身のタイプに関係無く ノーマルZ で ウルトラダッシュアタック に変えることしかできない。
- テラスタル している場合もその時のタイプ1を参照するため、基本的にはテラスタイプと同じタイプの技になる。
  - ステラ タイプにテラスタルしている場合は、元のタイプのタイプ1を参照するため、ステラタイプの技にはならない。
