レイジングブル は、ポケモンの技の一種。
## 説明文
- 第九世代: 怒り狂う あばれうしの 猛烈な タックル。 フォルムで 技のタイプが 変わり ひかりのかべや リフレクターなども 破壊できる。

## 技の仕様
- 技のタイプはケンタロスの フォルム によって決まる。
  - みずびたし や テラスタル などでケンタロス自身のタイプが変更されても、この技のタイプは変更されない。
  - 他のポケモンがケンタロスに へんしん したときは、変身対象のフォルムに従って技のタイプが変わる。
  - ケンタロスの姿ではないポケモンが ものまね / まねっこ などによって使用したときは、元の使用者や自分のタイプに関係なくノーマルタイプになる。 技のタイプを変える特性 の効果を受けて別のタイプに変わる。
  - ケンタロスが使用したときは 技のタイプを変える特性 の効果を受けない。
- 相性 / 特性 / まもる 状態などに技が無効化されたときは 壁 を解除できない。
- 技のダメージは リフレクター / オーロラベール 状態の影響で減衰しない。
- 壁を解除する効果は 追加効果 ではないので、 りんぷん / おんみつマント では防げず、使用者の特性 ちからずく の効果も発動しない。
- 味方に使用したときは味方の場の壁を解除する。
- ゆびをふる で選ばれない。
## 備考
- ケンタロス とその リージョンフォーム の 専用技 。
- 壁 を解除する技において最も威力が高い。
