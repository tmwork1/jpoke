## 説明文
### たたかうわざ
- ルビー・サファイア ・ エメラルド: てんきに よって タイプと いりょくが へんかする
- ファイアレッド・リーフグリーン: つかった ときの てんきに よって あいてに あたえる ダメージの りょうと タイプが へんかする。
- 第四世代 ・ 第五世代 ・ 第六世代 ・ 第七世代 ・ 第八世代 ・ 第九世代: つかった ときの てんきに よって わざの タイプと いりょくが かわる。
### コンテストわざ
- 第三世代: たくさん アピール できる
- 第四世代: しんさいんの ボルテージが あがったら +2。
- 第六世代: つづけて だしても かんきゃくに あきられずに アピール できる
- 第八世代: ポケモンの テンションが マイナスの ばあい もらえる ショーポイントが 1ふえる

### 特別なわざ
- ライコウ (クラウン) ：配布
- フシギダネ ： ポケモン集めてもらおうキャンペーン でプレゼントされたタマゴから生まれたフシギダネが覚えている。
## こんなときに使おう
威力は50と低いが、天気を変えれば実質威力100を安定した命中率で繰り出せる高性能のわざになるので、天気を変える状況下で戦うポケモンに採用されうる。
すなあらし のときは いわ タイプ での貴重な 特殊わざ になる。 にほんばれ と あめ のときは天気の効果で威力150まで強化されるが特性 ノーてんき のポケモンには威力50と素の威力しか出せないので注意。
## ポケモン不思議のダンジョンにおけるウェザーボール
- 救助隊: PP15。効果範囲は直線上。天気によってタイプ・ダメージが変化する。
- 探検隊 ・ 冒険団: わざ の 分類 が タイプ ごとだったのが各わざごとに変わり、ノーマル・いわタイプでも 特殊わざ となった。それ以外のわざとしての性質は救助隊と同様。
- 超不思議のダンジョン: PP14と少しだけ減少したが、わざを鍛錬することでPPを上げることが可能。

## Pokémon GOにおけるウェザーボール
Pokémon GO ではスペシャルアタックとして登場する。 2019年3月30日 の ハスボー限定リサーチ イベントで実装された。本編とは異なり、わざのタイプはポケモンごとに決まっており てんき には依存しない。
実装当初のウェザーボールは ノーマル ・ ほのお ・ みず ・ こおり の4種類があったが、 2021年6月1日 の ディスカバリーシーズン ( GOバトルリーグ シーズン8 )開始時にノーマルタイプのウェザーボールが無くなり いわ タイプに変更された。
ノーマルタイプのウェザーボールは 2021年8月31日 の GOバトルリーグ シーズン9 開始時に新しいわざとして改めて実装された。

## マンガにおけるウェザーボール
### ポケットモンスターSPECIALにおけるウェザーボール
- ポケットモンスターSPECIAL では、 ルビー の ポワルン などが使用する。
## 技の仕様
- 弾のわざ の1つ。 特性 が ぼうだん のポケモンに対しては無効化される。
- 天気 が らんきりゅう 状態のときはタイプ・威力いずれも対応した変化は無い。
- 技を選んだターンにおいて、使用する前に天気が変化した場合は使用時点の天気を参照してタイプと威力が決まる。
- 特性 ノーてんき 、 エアロック を持つポケモンが場にいるときは天候が変化していてもノーマルタイプになり、威力は増えない。
- ばんのうがさ を持つポケモンが使用したときは、 晴れ ・ 雨 状態でもノーマルタイプになり、威力は増えない。 すなあらし ・ あられ ・ ゆき ではタイプが変わり、威力も増える。
- 第四世代 のみ ノーマルスキン の効果を受ける。 第五世代 以降はノーマルスキン・ フェアリースキン ・ フリーズスキン ・ スカイスキン ・ エレキスキン の効果は発動しない。
- わざがノーマルタイプのまま変わらなかった場合のみ、 プラズマシャワー の効果を受ける。
- そうでん の効果を受けると、どの天候でもでんきタイプになる。
- どの天候であっても ノーマルZ を持っているときのみ Zワザ に変えて使うことができる。通常の天候下では ウルトラダッシュアタック であるが、天候により対応したタイプのZワザに変わる。例えば 晴れ 状態ではウルトラダッシュアタックは ダイナミックフルフレイム に変わる。威力はいずれも160。
  - Z ねごと など、他のZワザの効果でウェザーボールをZワザにして繰り出した場合、天候でタイプが変わる効果はない。いずれも威力160のウルトラダッシュアタックとなる。
- ダイマックスわざ にした場合、天候に対応したタイプのダイマックスわざになる。威力は130。
  - 技選択画面ではどの天候でも ダイアタック であり、相性予測もノーマルタイプのものになるが、実際に技を繰り出すときは ダイバーン 、 ダイストリーム 、 ダイロック 、 ダイアイス に変わる。使用者がキョダイマックスポケモンで対応したタイプのキョダイマックスわざがあれば、特定天候下で専用のキョダイマックスわざになる。
## 備考
- 第三世代 では ポワルン の 専用技 だった。
- 第三世代以前は タイプ ごとに 分類 が決められていたため、 ノーマル / いわ タイプで繰り出すと 物理わざ になるが、 第四世代 でどのタイプで繰り出しても 特殊わざ になる。
- ほのお タイプになったときは晴れの効果でさらにダメージが1.5倍に、 みず タイプになったときは雨の効果でさらにダメージが1.5倍になる。
