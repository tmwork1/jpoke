## 技の仕様
- フェアリーロック (場の状態) も参照。
- 使用した次のターンに再度フェアリーロックを使った場合、場はすでに フェアリーロック 状態であるため失敗する。
- 場を対象にした技だが、 オウムがえし でコピーすることができる。
## 備考
- クレッフィの 専用技 。
- ソード・シールド では、Ver.1.2.0でクレッフィが登場する以前においても使用不可能な技ではなく、 ゆびをふる で出現することがあった。
  - Ver.1.1.0では戦闘アニメが再生される環境でフェアリーロックが使用されるとソフトがクラッシュ(強制終了)する不具合があった。戦闘アニメが再生されなければ問題なく使用することができる。Ver.1.0ではこの現象は起こらない。Ver.1.1.1の配信で修正された。
