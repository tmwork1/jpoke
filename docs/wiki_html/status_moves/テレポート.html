バトルにおける効果

USUMまでの対人戦では効果が無い。バトルピラミッドでは普通の逃走成功率が低いため、先制さえすれば確実に逃げられる技として使える。第七世代でZワザに変えれば自分のHPを全回復できる。

ピカブイ以降では、通常の交代や他の交代を行う技と異なり、相手の行動を見てから安全に味方に交代できる技として使える。

フィールドにおける効果

第六世代以前およびBDSPでは、そらをとぶが使えないストーリー序盤から即座に街に戻る手段として使える。またそらをとぶでは選べない場所にあるポケモンセンターに行くこともできる。第二世代・HGSSでは、そらをとぶで直接移動不可能な別地方への移動も可能。特に第二世代のポケモンリーグに連続で挑戦する場合は便利である。「最後に休んだ～」と表示されるが、回復したかどうかではなく建物自体に入ったかどうかだけで判定される。

野生ポケモンのテレポート

野生ポケモンにテレポートを使われて成功すると、その相手を捕まえることができない。特にこの技しか覚えていないケーシィは、1度でも行動を許してしまうと捕まえられない。「即座にクイックボールなど被捕獲度の高いボールを投げる」「にげられない・バインド状態、特性かげふみ・ありじごくでテレポートを失敗させる」「ねむり状態やちょうはつ状態にして捕獲の試行回数を増やす」などの工夫をすることで捕まえやすくなる。

バトルにおける効果

第七世代 (USUM) まで

  • トレーナー戦・通信対戦では効果が無く、使うと必ず失敗する。
  • 野生ポケモン戦では、トレーナー側か野生ポケモン側が使うとバトルが終了する。
  • ダブルバトル・群れバトル・乱入バトル・ 大試練中は味方側も野生ポケモン側もテレポートが失敗する。
  • 第二世代では、ロケットだんアジトのマルマイン・いかりのみずうみのあかいギャラドス・スズのとうのスイクン・ウバメのもりのセレビィとの戦闘で使用すると失敗する。
  • 使用したポケモンがバインド状態・にげられない状態のときや場がフェアリーロック状態のとき、使用したポケモンにかげふみ・ありじごく・じりょくの効果が働いているときは失敗する。
    • ただし、特性にげあしやもちものけむりだまを持つ場合は、これらの状態や特性を無視して確実に逃げることができる。
    • 第六世代以降はゴーストタイプのポケモンの場合は、これらの状況を無視して確実に逃げることができる。
  • 使用したポケモンがねをはる状態の場合、以下のようになる。
    • 第四世代以前は失敗する。ただし、特性にげあしやもちものけむりだまを持つ場合は成功する。
    • 第五世代以降は、ねをはる状態を無視してテレポートを成功させることができる。
  • 自分よりレベルの高いポケモンに使う場合、以下のようになる。
    • 第一世代・第二世代では相手のレベルが自分のレベルと比較して高いほど失敗しやすい。テレポートの成功可否は以下の式Rにより求まり、R≦T/4なら失敗する。
R=floor{(T+U)*X/256}+1
T: 対象のレベル, U: 使用者のレベル, X: 0-255までの乱数, floor関数では小数点以下を切り捨てる。
  • 失敗率は(T/4+1)/(T+U)と概算できる。
  • 第二世代では、野生ポケモンが使用するテレポートは相手よりレベルが低くても失敗しない。
  • 第三世代以降の野生ポケモン戦では、レベルが相手よりも低くてもテレポートが失敗することはない。
  • テレポートの効果で戦闘が終了した場合、ネコにこばんのお金を拾うことはできない。
    • 第一世代のみ、テレポートの効果で戦闘が終了してもネコにこばんを使ったポケモンがひんしになっていなければ拾える。

ピカブイ・第八世代以降

  • トレーナーのポケモンが使用すると優先度-6で控えのポケモンと交代する。
  • 交代できる控えのポケモンがいない場合は失敗する。
  • 第八世代以降では、野生ポケモンがテレポートを使用した場合は戦闘が終了する。野生ポケモン戦でもトレーナーのポケモンが使用したときは交代するだけで、戦闘は終わらない。
    • 第九世代では、テレポートで逃げた野生ポケモンのシンボルは消滅する。
  • バインド・にげられない・フェアリーロック状態や、特性かげふみ・ありじごく・じりょくの効果を受けているポケモンがテレポートを使用した場合、以下のようになる。
    • トレーナーのポケモンがテレポートを使用するときは、これらの状態や特性の効果を無視してテレポートで交代することができる。
    • 野生ポケモンがテレポートを使用するときは、これらの状態や特性の効果でテレポートが失敗する。ただし、特性にげあしやもちものけむりだまを持つ場合や、ゴーストを持つ場合は成功して戦闘が終了する。相手の特性がかげふみである場合は、味方のかげふみを無視して戦闘が終了する。
  • マックスレイドバトルでトレーナーが使用した場合、イベントバトルで手持ちが複数いるならば成功する。敵ダイマックスポケモンが使用した場合は失敗する。
  • テラレイドバトルでトレーナーが使用した場合は失敗する。野生ポケモンが使用した場合、マルチプレイ時だと失敗するが、ソロプレイ時は成功して戦闘が終わる。テレポートで戦闘が終わっても結晶は残り続ける。

フィールドにおける効果

  • 第一世代のみ、最後に立ち寄ったポケモンセンターではなく、最後にポケモンの回復をしたポケモンセンターに飛ぶ。
  • 3ばんどうろなど、そらをとぶに対応していない場所のポケモンセンターでも飛べる。
  • そらをとぶと同様に、屋外でしか使うことができない。
  • 第四世代では町の中で使うことができない。
  • 第四世代以降では、主人公の自宅も移動先の候補になる。
  • 第七世代・SwShではフィールドで使うことができない。
  • シナリオの進行でも移動先のポケモンセンターの位置が再設定されることがある。

備考

  • ポケモンセンターや野生ポケモンが存在しないポケモンスタジアムシリーズ・ポケモンバトルレボリューション等の作品では全く使い道のない技であり、ポケモンバトルレボリューションでは使用時のモーションも存在しない。
  • 第一世代、第二世代ではゲーム中のNPCトレーナーが使用するケーシィやユンゲラーなどは無意味にこの技を使用することがある(特に第一世代では顕著)。
  • 第二世代では殿堂入り前のみセキエイこうげんのおじいさんと会話するとケーシィが強力なテレポートで主人公の家に送ってくれる。
    • ハートゴールド・ソウルシルバーでは、過去の話として上記イベントの話をするだけに変更されている。
  • ポケモンレンジャー バトナージでは、エーフィ・キルリア・バリヤード・フーディン・リーシャン・ルージュラのマップわざとして登場。使うとダンジョンの入口まで戻ることができる。

大乱闘スマッシュブラザーズにおけるテレポート

  • 大乱闘スマッシュブラザーズシリーズでは、ミュウツーの上必殺ワザとして登場。攻撃力を持たない、移動・復帰専用の必殺ワザとなっている。
  • 大乱闘スマッシュブラザーズ SPECIALでは、モンスターボールから登場するケーシィが使用する。ケーシィ自身だけでなく、近くにいるファイターも一緒にテレポートさせる。

ポッ拳シリーズにおけるテレポート

  • ポッ拳ではミュウツー・ダークミュウツーのわざとして登場する。
  • ポッ拳DXではサーナイト・ミュウツー・ダークミュウツーのわざとして登場する。
