ポケットモンスターSPECIALにおけるでんじは

  • ポケットモンスターSPECIALでは、1巻の9話でレッドのピカチュウなどが使用する。

技の仕様

詳細についてはまひを参照。
  • 特性じゅうなん/ぜったいねむり/きよめのしおのポケモン、特性フラワーベールやしんぴのまもり/ミストフィールド状態に守られているポケモンはまひしない。第六世代以降ではでんきタイプのポケモンもまひしない。
  • 特性ちくでんのポケモンには第三世代のみ有効だが、第四世代以降は無効となる。
  • 特性でんきエンジンのポケモンには無効となる。
  • 特性ひらいしんの影響を受ける。
  • 変化技だが、例外的に「効果がない」場合のみタイプ相性の影響を受けて無効になる。
    • じめんタイプのポケモンには無効となる。相手がねらいのまとを持っている場合は有効になる。
    • 特性がノーマルスキンのポケモンが使用した場合は、ゴーストタイプのポケモンに無効、じめんタイプ/ちくでん/でんきエンジン/ひらいしんのポケモンに有効となる。
    • さかさバトルではじめんタイプのポケモンにも有効となる。
    • ふしぎなまもりの影響は受けない。

備考

  • 他にタイプ相性の影響を受ける変化技としては、第二世代・第三世代のへびにらみがある。へびにらみは第四世代から相性の影響を受けなくなった。
