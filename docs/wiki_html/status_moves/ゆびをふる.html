こんなときに使おう

最も不安定といってよい技で、ストーリー攻略でも対戦でも使われることは少ない。一方「ゆびをふる以外の技を全て忘れさせたポケモンのみで戦う」などの縛りは有名な遊び方の一つであり、第七世代のスペシャルレートでも採用されたことがある。古くはポケスタ1の攻略本にゆびをふるだけで対戦する企画が掲載されていた。

第二世代の対戦では、ねむる+ねごとと併用して試行回数を稼げば上振れを狙うことが可能である。

ポケモン不思議のダンジョンにおけるゆびをふる

PPは22。効果範囲や威力は選ばれるわざに依存。青・赤では繰り出される可能性のあるわざは105種類と少なかったが、時・闇以降はすべてのわざ（さばきのつぶてを除く。シードフレアは空の探検隊では繰り出す）の中からランダムで繰り出すようになった。

時・闇の探検隊ではしんぴのもりに出現するゴンベなどが使ってくることがある。わざの予想が全くつかないため下手したらピンチになりかねない。

大乱闘スマッシュブラザーズシリーズにおけるゆびをふる

技の仕様

ゆびをふるで出る技

  • 自分が覚えている技が出るかは、世代により仕様が異なる。
    • 第一世代・第三世代・第五世代以降では出る。
    • 第二世代・第四世代では出ない。
  • 第一世代では、ゆびをふるとわるあがき以外の技が（自分が覚えている技も含めて）ランダムで出る。
  • 第二世代以降では、以下の表で×になっている技は選ばれない。他の技が出る技、まもる系統の技、行動順に関する技、第五世代以降の伝説・幻ポケモンの専用技の一部 (全てではない) などが含まれる。第八世代以降では、一般ポケモンの専用技の一部 (全てではない) や、特に特徴が無い技でも選ばれないことがある。

凡例
○ : ゆびをふるで選ばれる
× : ゆびをふるで選ばれない
- : 技自体が登場していない
  • ピカブイではオウムがえし/カウンター/へんしん/ものまね/わるあがきを除く第一世代の技のみ選ばれる。
  • 第八世代・第九世代では、そのソフトにおいて使用できない技は選ばれない (ソード・シールドで使用不可能なわざ/ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ/スカーレット・バイオレットで使用不可能なわざ)。

ゆびをふる自体の仕様

  • ゆびをふるで出た技は、アンコールすることができない。
  • ねごとで選ばれない(第三世代以降)。
  • ねこのてで選ばれない。
  • この技に対するさいはいは失敗する。
  • 選ばれた技の優先度は関係なく、ゆびをふるの優先度で繰り出される。
  • ノーマルZを持たせることで、出た技のタイプと威力に対応したZワザが出せる。
    • 変化技を出した場合、Z変化ワザによる付加効果は受けられない。

第二世代のみ

  • ゆびをふるで出た技でダメージを受けた場合は、カウンター/ミラーコートで反撃できない。
  • 相手が直前に使った技がゆびをふるの場合、テクスチャー2は失敗する。
  • ねごとで選ばれる。

第四世代まで

  • ものまねでコピーできない。
  • メトロノームの効果によりゆびをふるで出た技で与えられるダメージが増える。
  • じゅうりょく状態でそらをとぶ/とびはねるが出た場合、じゅうりょくの影響を受けずに飛び上がるが、じゅうりょくの効果が切れるまで攻撃できず飛び上がったままとなる。
  • 回復技が出た場合、かいふくふうじ状態の影響を受けずに回復する。

第五世代以降

  • ゆびをふるに対するものまねは、第八世代まではゆびをふる自体をコピーする。第九世代ではものまねは失敗する。
  • ゆびをふるで同じ技が続けて出ない限り、メトロノームの効果は無い。数ターン行動する技が出た場合は効果を受ける。
  • じゅうりょく状態で使用できない技が出た場合、じゅうりょくの影響で失敗する。
  • 回復技が出た場合、かいふくふうじ状態であると失敗する。
  • 第七世代以降では、こだわり系アイテムを持っている状態でゆびをふるを使い数ターン続く技や溜め技を出した場合、最初のターンは成功するが、2ターン目の行動は必ず失敗する。

備考

  • ゆびをふるとメトロノームの英名はいずれも"Metronome"である。
  • 赤・緑・青ではスプーンまげを覚えるポケモンがいないため、ゆびをふるを使うことでしか見られない技だった。
    • これ以外にもアルセウス専用のさばきのつぶてなど、未解禁だったポケモンの技が出ることがあった。第五世代以降では専用技は出ないことが多くなっているが、じゃどくのくさりは習得者が解禁される前からゆびをふるで出現した。
  • ポケモンバトルレボリューションでは、ピッピ・ピクシーがこの技を使用すると専用のモーションが見られる。
  • ポケウッドでは使用禁止技に指定されている。
  • 様々な技がランダムに出る仕様は、どれにしようかな(子供の数え歌の一種)が元ネタになっていると思われる。
  • 2022年9月3日～11日の期間にポケモンセンター (店舗)でスタッフに「ゆびをふる」とピッピ (おつきみ22)を受け取ることができた。
