こんなときに使おう

相手に依存している分シングルバトルでの安定性は低い。基本的にはダブルバトル・トリプルバトル用の技と言える。ダブル・トリプルバトルでの主なコンボは以下の通り。

  • ヨガパワーなど強力な特性を持つ味方のポケモンの特性をコピーする。
  • あまのじゃくをコピーして使うとに能力が下がる技の効果を反転させる。
  • 特性で発動させた天気を相手に上書きされたとき、なりきりで天候特性をコピーし天気を奪い返す。
  • ノーガードをコピーして必中の一撃必殺技を使う。

ポケモン不思議のダンジョンにおけるなりきり

探検隊
PP17。正面の敵の特性を自分にコピーする。次のフロアに行くと元の特性に戻る。

ポケモンカードゲームにおけるなりきり

この節はポケモンカードゲームに関連した書きかけです。加筆、訂正して下さる協力者を求めています。(プロジェクト)

マンガにおけるなりきり

ポケットモンスターSPECIALにおけるなりきり

  • ポケットモンスターSPECIALでは、グリーンのフーディンなどが使用する。

技の仕様

  • 成功すると「<使用者>は <相手ポケモン>の <特性>を コピーした!」とメッセージが表示され、何の特性を手にしたかが分かる。
    • 第五世代以降ではこれに加えて、使用者が元々何の特性を持っていたかも特性バーが表示されることで分かる。
  • お互いの特性が同じである場合、第四世代までは成功する。第五世代からは失敗する。
  • 場に出たときに発動する特性は、基本的にはコピーした直後に発動する。
    • 第四世代まで、使用者が場に出てから発動させたことがある特性をコピーした場合は発動しない。第五世代以降はこれらに関係なく発動する。
    • 第五世代以降でノーてんき/エアロックをコピーしても特性バーは現れないが、特性の効果はコピーできている。
    • 第五世代のみ、きんちょうかんをコピーしてもその瞬間に特性の効果は発動せず、その次にポケモンの入れ替えが起きた後に発動する。第六世代以降はコピーしたその瞬間からきんちょうかんが発動する。
  • 対象の特性が以下の特性である場合は失敗する。多くはコピーできない特性に該当する。
    • スカーレット・バイオレット (SV) では、Ver.1.3.2まではハドロンエンジン/ひひいろのこどうに対しては無効だったが、Ver.2.0.1からは有効になった。
    • 第八世代とSV (Ver.2.0.1 - 2.0.2) ではうのミサイルはコピーできない。SV (Ver.3.0.0 - 3.0.1) ではコピーできる。

凡例
○: なりきりでコピーできる。
×: なりきりでコピーできない。
－: その世代では特性が存在しない。
  • 変化技なので、対象の特性がおうごんのからだであるときは防がれる。使用者の特性がかたやぶりなら特性に防がれることなくコピーできる。
  • 使用者の特性がきずなへんげなどの上書きできない特性であるときは失敗する。
  • 使用者の持ち物がとくせいガードであるときは失敗する。
  • まもる/みがわり状態のポケモンに使っても失敗しない。ダイウォールに防がれることもない。
  • トリックガードには防がれる。
  • 必中技だが、姿を隠している対象には当たらない。
  • とくせいなし状態のポケモンに使っても失敗しない。
  • オウムがえしでコピーできない。

備考

  • 第三世代では命中の欄に「100」と表記されているが、実際には命中判定が行われることなく成功する。
