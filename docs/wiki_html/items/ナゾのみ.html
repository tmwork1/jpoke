値段・効果

  • 買値:非売品
  • 売値:10円3-7, BDSP (第五世代でアイテムマニアに売ると30000円), 40円SwSh, 20円SV
  • 効果:
    • 第三世代:どろぼう・ほしがる・トリックの効果を受けない。
    • 第四世代以降:ポケモンに持たせると、効果バツグンの攻撃技でダメージを受けた直後に消費して、最大HPの1/4だけ回復する (小数点以下切り捨て)。ひんしになったときは発動しない。
  • きのみクラッシュ: 基礎こな量200、難易度150
  • アイスクリーム作り (すくいやすさ/くっつきやすさ): 1個目3/3、2個目4/4、3個目5/5
  • しるや: 紫色、グループ3。ロゼルのみとブレンドするとワンダフルソーダになる。
  • カロス地方のきのみばたけ: 紫色
  • カレーライス: 20ポイント
  • ウッウロボ: 素材タイプむし、素材ポイント5

こんなときに使おう

第三世代においては敵の使うトリックを無効化できる唯一の手段だが、それ以外の効果は無い。さらにポケモンコロシアムやポケモンXDでは使用不可能なために、一般的な対戦ルールでは禁止扱いされていることが多い。一応バトルフロンティア等では使用可能である。

第四世代以降においてはきのみとしての効果が設定された。回復量が同じオボンのみとは発動タイミングが異なるが、弱点技を受けてHPが半分以上残る耐久力が高いポケモンでなければオボンのみでよいことになる。ダブルバトルでは、オボンのみの発動を1発目で回避し、2発目にてだすけ込みで倒そうとする相手の作戦に対しても、ナゾのみなら発動機会を逃さずに済むことがある。オボンのみははたきおとす・ついばむ・むしくいを受けると回復できないが、第五世代以降でこれらが弱点のポケモンならナゾのみを失う前に回復できる。

カードe+について

  • ルビー・サファイアでカードeリーダー+に「ナゾのみカード」を読み込ませた後で、センリと会話すると受け取ることができる。
    • バッグとパソコンの中にNo.43のきのみが一つもないことが条件。ポケモンに持たせたりふかふかの土に植えてある分はノーカウント。
  • 土に植えてある状態で別のきのみに切り替えた場合、成長段階や結実数は次の成長段階から反映される。
    • 成長は早いが個数の少ないホズのみと、逆に成長は遅いが個数の多いエドマのみをタイミングよく切り替えることで短時間での大量収穫が可能になる。
  • ルビー・サファイア以外の第三世代のソフトに対応した「ナゾのみカード」は発売されていない。
  • 日本版と北米版では、カードeリーダー+本体・カードともに共用不可能で、各リージョンごとに専用のものとしか通信できない。
    • 日本版と北米版で通信してポロックを作る場合、同じきのみであっても別種として扱われる。例えばラッカのみとNutpea Berryは設定上は同種だが、ブレンドで併用しても「くろいポロック」にはならない。
  • 第2弾は日本でしか発売されていない。海外においては、日本語名のローマ字表記が便宜上用いられている。

詳細な仕様

第四世代以降

  • ほしがる/どろぼう/ついばむ/むしくい/はたきおとす/やきつくす、特性マジシャンの攻撃が弱点であった場合、以下のようになる。
    • 第四世代では、技の効果が優先されてナゾのみの効果は発動しない。
    • 第五世代以降では、技の効果より先にナゾのみの効果が発動しHPを回復する(ほしがるはノーマルタイプであるため、技のタイプが変わったときやさかさバトル限定)。弱点でなければナゾのみは発動しないので、これらの効果で失われる。
    • 第六世代以降では、このときもはたきおとすの威力は1.5倍になる。
  • ついばむ/むしくいで食べたとき、第七世代までは使用者のHPは回復しない。第八世代では使用者の最大HPの1/4が回復する。
  • ほおばる/おちゃかい/はんすうで食べたときも最大HPの1/4が回復する。
  • なげつけるで使用しても、対象のHPは回復しない (第八世代/第九世代でも同様)。
  • 弱点となる連続攻撃技を受けた場合、1発目を受けた直後に発動する。
  • 弱点のタイプの一撃必殺技をこらえるなどにより耐えた場合、ナゾのみは発動する。一撃必殺技以外のダメージ固定技を受けたときは、弱点のタイプであっても発動しない。
  • みがわり状態で攻撃を防いだときは発動しない。
  • ばけのかわ/アイスフェイスでダメージを肩代わりしたときは発動しない。
  • らんきりゅうによりひこうタイプの相性が変化した場合は発動に影響がある。弱点技を等倍にして受けた場合は発動せず、らんきりゅう状態でなお抜群であった場合は発動する。
  • タールショット状態によりほのお技に対する相性が変化した場合は発動に影響がある。

備考

  • ルビー・サファイアでは、カチャのみ～トポのみのデフォルトの姿。これらのきのみはデータがソフト内には元々入っておらず、カードを読み込むことで画像や味のデータをインストールする。
    • 上記のきのみの総称としても用いられる。例えばボックスに預けたポケモンの持ち物としては、上記のきのみは全て「ナゾのみ」表記になる。
    • 第三世代でナゾのみを持たせたポケモンは通信交換不可。
  • 第四世代以降は独立したきのみとして効果等が設定された。
