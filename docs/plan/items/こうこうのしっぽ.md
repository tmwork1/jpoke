# こうこうのしっぽ 実装計画

## 効果概要
技の行動順を最後にする（あとだしと同様の効果）。素早さに関わらず最も遅く行動する。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `DomainEvent.ON_CALC_BACK_TIER` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def こうこうのしっぽ_calc_back_tier(battle, ctx, value):
    # -1 を返すことで後攻ティアを付与（あとだしと同パターン）
    return HandlerReturn(value=-1)
```

## テスト観点
- こうこうのしっぽ所持時、素早さに関わらず最後に行動すること
- あとだし（特性）と同じ後攻ティアになること
- こうこうのしっぽがない場合は行動順が通常通りであること
- 特性クイックドロウが発動した場合、道具の効果（後攻ティア-1）が無視され先攻になること
  （`handlers/ability.py` の `クイックドロウ_maybe_fast_attack` で、攻撃者がこうこうのしっぽを
  持つ場合は +1 ではなく +2 を加算し、道具側の -1 を打ち消して正味 +1 にする）
