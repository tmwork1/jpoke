# とつげきチョッキ 実装計画

## 効果概要
変化技の使用を禁止する + とくぼうを 1.5 倍にする（非消費）。

## 実装仕様

### イベント 1: 変化技禁止

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TRY_MOVE_1` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |

### イベント 2: とくぼう 1.5 倍

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CALC_DEF_MODIFIER` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False |

### ハンドラロジック（疑似コード）
```python
def とつげきチョッキ_block_status_move(battle, ctx, value):
    if ctx.move.category == "変化":
        # 変化技を使用禁止にしてイベント停止
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)


def とつげきチョッキ_boost_spdef(battle, ctx, value):
    if ctx.move.category == "特殊":
        value = apply_fixed_modifier(value, 6144)  # 1.5倍
    return HandlerReturn(value=value)
```

## テスト観点
- 変化技を選択したとき失敗すること
- 物理技・特殊技は通常通り使えること
- 特殊技に対してとくぼうが 1.5 倍になること
- 物理技に対してとくぼう倍率が変化しないこと
