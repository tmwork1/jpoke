# とつげきチョッキ 実装計画

## 効果概要
変化技をコマンド選択時に除外する（実行自体を禁止するわけではない） + とくぼうを 1.5 倍にする（非消費）。

## 実装仕様

### イベント 1: 変化技をコマンド選択肢から除外

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_COMMAND_OPTIONS` |
| subject_spec | `"source:self"` |
| priority | 100（デフォルト。turn.md 未掲載イベントのため既存の同種ハンドラ（こだわり・アンコール等の command_options 系ハンドラ）に合わせる） |
| once | False |

選択可能なコマンドの一覧から、変化技（`category == "status"`）に対応する技コマンド（通常技・テラスタル・メガシンカ等、`cmd.is_type("move")` が True になるもの全て）を除外する。
変化技しか持たないポケモンの場合、`CommandManager.get_available_action_commands` 側の「技コマンドが無ければわるあがきを追加する」フォールバックにより自動的にわるあがきのみが選択可能になる（本アイテム側で追加の分岐は不要）。

### イベント 2: とくぼう 1.5 倍

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CALC_DEF_MODIFIER` |
| subject_spec | `"defender:self"` |
| priority | 100（デフォルト。ダメージ計算内部イベントのため、他アイテムの防御補正ハンドラ（とくせいガード等）に合わせる） |
| once | False |

### ハンドラロジック（疑似コード）
```python
def とつげきチョッキ_modify_command_options(battle, ctx, value):
    mon = ctx.source
    return HandlerReturn(value=[
        cmd for cmd in value
        if not (cmd.is_type("move") and mon.moves[cmd.index].category == "status")
    ])


def とつげきチョッキ_boost_spdef(battle, ctx, value):
    if ctx.move.category == "special":
        value = apply_fixed_modifier(value, 6144)  # 1.5倍
    return HandlerReturn(value=value)
```

## 実行時に変化技を禁止する `Event.ON_TRY_MOVE_1` ハンドラを設けない理由

とつげきチョッキは「コマンド選択画面で変化技を選べなくする」効果であり、選択済み・強制された変化技の実行自体を妨げるものではない
（`docs/spec/items/とつげきチョッキ.md` 詳細な仕様を参照）。
`Event.ON_TRY_MOVE_1` で `ctx.move.category == "status"` を一律ブロックすると、以下の仕様上の例外を満たせなくなる:

- 変化技を選択してから行動するまでに他のポケモンからとつげきチョッキを受け取った場合、そのターンのみは変化技でも使用できる
- とつげきチョッキを持っている状態で変化技をアンコールされた場合、そのターンのみは変化技も使用できる
  （アンコールは `Event.ON_MODIFY_MOVE` で `ctx.move` を強制上書きするため、選択時点のコマンドフィルタを経由せず変化技が実行され得る）
- さいはい・おどりこの効果によって変化技を使用することはできる
- 特性ぶきよう・さしおさえ／マジックルーム状態では変化技も選択できる

これらはいずれも「選択時点のコマンド一覧からの除外」のみで解決すべき事象であり、選択後の実行を追加でブロックする仕組みは
こだわり系アイテム・ごりむちゅう（`こだわり` 揮発性状態、`Event.ON_MODIFY_COMMAND_OPTIONS` のみで技を固定し、
`Event.ON_TRY_MOVE_1` 側のブロックは持たない）にも存在しない。この既存実装と整合させるため、とつげきチョッキも
コマンド選択の除外のみを実装し、`Event.ON_TRY_MOVE_1` での実行時ブロックは設けない。

## テスト観点
- 物理技は通常通り使えて、とくぼう補正が掛からないこと
- 特殊技に対してとくぼうが 1.5 倍になること
- 変化技はコマンド選択肢から除外されること（物理/変化混在の技構成で確認）
- 変化技しか持たない場合、選択可能コマンドがわるあがきのみになること
- アンコールで変化技を強制されたターンは、とつげきチョッキを持っていても変化技が正常に実行できること（実行時ブロックが無いことの確認）
