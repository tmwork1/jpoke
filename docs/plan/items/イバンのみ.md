# イバンのみ 実装計画

## 効果概要
HP が最大値の 25% 以下になった瞬間にフラグを立て（消費）、次に使う技で先制する。タラプのみと同効果の可能性あり（要確認）。

## 実装仕様

### イベント 1: HP 閾値トリガー（フラグ付与）

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HP_CHANGED` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### イベント 2: 行動順の先制付与

| 項目 | 内容 |
|------|------|
| イベント | `DomainEvent.ON_CALC_BACK_TIER` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
