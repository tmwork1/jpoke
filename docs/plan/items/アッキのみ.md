# アッキのみ 実装計画

## 効果概要
物理の弱点でダメージを受けたとき、ぼうぎょ（B）を +2 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def アッキのみ_boost_defense_on_physical_super_effective(battle, ctx, value):
    mon = ctx.defender
    if (
        ctx.move.category == "物理"
        and battle.damage_calculator.calc_def_type_modifier(ctx) > 1
    ):
        battle.modify_stats(mon, {"B": +2})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- 物理の弱点ダメージを受けたとき B が +2 されること
- 特殊技の弱点では発動しないこと（タラプのみの守備側特殊版と対称）
- 物理技でも等倍以下では発動しないこと
- 発動後にアイテムが消費されること
