# アッキのみ 実装計画

## 効果概要
物理技でダメージを受けたとき、ぼうぎょ（B）を +1 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def アッキのみ_boost_defense_on_physical_hit(battle, ctx, value):
    mon = ctx.defender
    if ctx.move.category == "物理":
        battle.modify_stats(mon, {"B": +1})
        _announce_item_triggered(battle, mon)
        battle.consume_item(mon)
    return HandlerReturn(value=value)
```

## 注意
- じゅくせい持ちでも効果量は+1のまま（スター以外のピンチきのみと違い2倍にならない）
- ダメージが0のとき（相性無効・みがわり等）は ON_DAMAGE_HIT が発火しないため別途チェック不要

## テスト観点
- 物理技でダメージを受けたとき B が +1 されること
- 特殊技では発動しないこと
- 発動後にアイテムが消費されること
