# するどいくちばし 実装計画

## 効果概要
ひこうタイプ技の威力を1.2倍にする（タイプ強化アイテムと同パターン）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `docs/spec/items/するどいくちばし.md` |
| 実装状態 | **実装済み** |
| イベント | `Event.ON_CALC_POWER_MODIFIER` |
| subject_spec | `"attacker:self"` |
| priority | 100（デフォルト、他のタイプ強化アイテムと同一） |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック
```python
def するどいくちばし_modify_power_by_type(battle, ctx, value):
    return _modify_power_by_type(ctx.move, value, type_="ひこう", modifier=4915)
```

- `ctx.move.type` は攻撃時点で確定済みの技タイプ（ノーマルスキン・スカイスキン等でひこうタイプに変化した場合も反映される）を参照するため、追加の分岐は不要。
- フライングプレスは相性計算のみひこうタイプが追加される特殊技だが、`move.type` 自体は「かくとう」のまま変わらないため、既存の `_modify_power_by_type` ロジックだけで自動的に効果対象外となる（追加実装不要）。

## テスト観点
- ひこうタイプ技の威力が1.2倍になること
- 他タイプの技には効果がないこと
- ノーマルスキン等でひこうタイプに変化した技には効果があること
- フライングプレス（かくとうタイプ扱い）には効果がないこと
