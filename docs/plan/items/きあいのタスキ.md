# きあいのタスキ 実装計画

## 効果概要
フルHP状態から OHKO（一撃ひんし）ダメージを受けたとき、HP1 で耐える（消費）。HP が満タンでない場合は発動しない。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_MOVE_DAMAGE` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def きあいのタスキ_survive_ohko(battle, ctx, value):
    mon = ctx.defender
    # HP が満タン かつ ダメージがHP以上の場合のみ発動
    if mon.hp == mon.max_hp and value >= mon.hp:
        value = mon.hp - 1
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- フルHP状態から OHKO ダメージを受けたとき HP1 で耐えること
- HP が満タンでない場合（半減状態等）は発動しないこと
- 発動後にアイテムが消費されること
- 消費後に再度 OHKO を受けたとき耐えないこと
- すなあらし・まきびし等の継続ダメージには発動しないこと（ON_MODIFY_MOVE_DAMAGE は技ダメージのみ）
