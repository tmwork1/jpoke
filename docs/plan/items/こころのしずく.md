## 実装ゴール（完了定義）

- ラティオス/ラティアスがエスパーまたはドラゴン技を使うと威力が 4915/4096 倍になる
- ラティオス/ラティアス以外が持っても効果がない
- タイプ変換後のタイプで判定する（既存の `ON_CALC_POWER_MODIFIER` 発火順が保証）

## 実装対象イベント

- `Event.ON_CALC_POWER_MODIFIER` — 技の威力倍率を計算する。`value` は累積倍率（int）

## テスト観点

1. ラティオスがエスパー技を使うと威力が約 1.2 倍になる。
2. ラティアスがドラゴン技を使うと威力が約 1.2 倍になる。
3. ラティオス/ラティアスがノーマル技等を使っても効果がない。
4. 他のポケモンが持っても効果がない。
