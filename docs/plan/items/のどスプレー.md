# のどスプレー 実装計画

## 効果概要
音技（sound ラベル）を使用したとき、とくこう（C）を +1 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MOVE_END` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def のどスプレー_boost_spatk_on_sound(battle, ctx, value):
    mon = ctx.attacker
    if ctx.move.has_label("sound"):
        battle.modify_stats(mon, {"C": +1})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## 注意
`ON_MOVE_END` の subject_spec: `attacker:self` (AttackContext) か確認。
event.py のコメントによると `ON_MOVE_END` は `AttackContext` で発火するため `attacker:self` が正しい。

## テスト観点
- 音技を使用したとき、とくこうが +1 されること
- 音技以外では発動しないこと
- 発動後にアイテムが消費されること
- 消費後に再度音技を使っても発動しないこと
