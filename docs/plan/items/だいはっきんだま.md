# だいはっきんだま 実装計画

更新日: 2026-06-12

## スコープ

- 仕様書: `docs/spec/items/だいはっきんだま.md`
- 実装状態: 未実装
- 実装方針: `ON_SWITCH_IN` / `ON_ITEM_GAINED` でギラティナ(アナザー)をオリジンフォルムにフォルムチェンジし、`ON_CALC_POWER_MODIFIER` でドラゴン・ゴースト技を 1.2 倍にする

## 実装ゴール（完了定義）

- ギラティナ(アナザー)がだいはっきんだまを持って登場するとギラティナ(オリジン)になる
- ギラティナ(オリジン)がドラゴンまたはゴースト技を使うと威力が 4915/4096 倍になる
- `removable=False` はすでに ItemData に設定済み（オリジンフォルムは奪われない）
- ギラティナ以外が持っても効果なし
- マジックルーム中は威力補正が無効になるがフォルムは維持（ItemHandler が item_enabled チェックを通じて自動的に無効化される）

## 実装対象イベント

- `Event.ON_SWITCH_IN` — 登場時にフォルムチェンジを発動する
- `Event.ON_ITEM_GAINED` — バトル中にアイテムを入手したとき
- `Event.ON_CALC_POWER_MODIFIER` — ドラゴン/ゴースト技威力 1.2 倍

## 変更対象（ファイル/関数単位）
