# パンチグローブ 実装計画

## 効果概要
パンチ技（punch ラベル）の威力を 1.1 倍にする + パンチ技が接触技でなくなる（非消費）。

## 実装仕様

### イベント 1: パンチ技威力 1.1 倍

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CALC_POWER_MODIFIER` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |

### イベント 2: パンチ技の接触判定を無効化

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_CONTACT` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |

### ハンドラロジック（疑似コード）
```python
def パンチグローブ_boost_punch_power(battle, ctx, value):
    if ctx.move.has_label("punch"):
        value = apply_fixed_modifier(value, 4506)  # 1.1倍
    return HandlerReturn(value=value)


def パンチグローブ_negate_punch_contact(battle, ctx, value):
    if ctx.move.has_label("punch"):
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)
```

## 注意
`ctx.move.has_label("punch")` の実装が `move.labels` リストによるチェックかどうか確認する。

## テスト観点
- パンチ技の威力が 1.1 倍になること
- パンチ技で接触判定が発生しないこと（ゴツゴツメット・さめはだ等が発動しない）
- パンチ技以外は威力・接触判定が変化しないこと
