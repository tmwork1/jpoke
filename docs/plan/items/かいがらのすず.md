# かいがらのすず 実装計画

## 効果概要
ダメージを与えたとき、与えたダメージの 1/8 を HP として回復する（吸収技とは別枠）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HIT` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def かいがらのすず_drain_on_hit(battle, ctx, value):
    # value は ON_HIT イベントで渡される実ダメージ量
    heal_amount = value // 8
    if heal_amount > 0 and battle.modify_hp(ctx.attacker, v=heal_amount):
        _announce_item_triggered(battle, ctx.attacker)
    return HandlerReturn(value=value)
```

## 注意
`ON_HIT` の `value` がダメージ量（正の整数）として渡されるか、イベント定義を確認する。

## テスト観点
- ダメージ技命中時、与ダメージの 1/8 が回復すること
- 変化技では回復しないこと（ON_HIT は通常ダメージ技にのみ発火する）
- HP が満タンの場合は回復ログが出ないこと
- みがわり人形へのダメージでは回復しないこと（ON_HIT の発火条件による）
