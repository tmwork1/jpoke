# ラムのみ 実装計画

## 効果概要
ターン終了時、何らかの状態異常（やけど・まひ・どく・もうどく・こおり・ねむり）があれば回復する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 50 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ラムのみ_cure_ailment(battle, ctx, value):
    mon = ctx.source
    if mon.has_ailment():
        battle.ailment_manager.remove(mon)
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- やけど・まひ・どく等いずれかの状態異常時にターン終了で回復すること
- 状態異常がない場合は発動しないこと
- 発動後にアイテムが消費されること
- 消費後の次のターンに状態異常が付与されても回復しないこと
