# ひかりごけ 実装計画

## 効果概要
みず技でダメージを受けたとき、とくぼう（D）を +1 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ひかりごけ_boost_spdef_on_water(battle, ctx, value):
    mon = ctx.defender
    if ctx.move.type == "みず":
        battle.modify_stats(mon, {"D": +1})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- みず技でダメージを受けたとき D が +1 されること
- みず以外の技では発動しないこと
- 発動後にアイテムが消費されること
