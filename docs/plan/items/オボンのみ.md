# オボンのみ 実装計画

## 効果概要
HP が最大値の 50% 以下になった瞬間に最大 HP の 25% を回復する（消費）。
ダメージを受けて閾値を跨いだ瞬間にのみ発動する。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HP_CHANGED` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def オボンのみ_heal_on_half_hp(battle, ctx, value):
    mon = ctx.target
    hp_after = mon.hp
    hp_before = hp_after + value  # value は正のダメージ量
    # 閾値を跨いだとき（50% 超 → 50% 以下）のみ発動
    if hp_before * 2 > mon.max_hp and hp_after * 2 <= mon.max_hp:
        battle.modify_hp(mon, r=1/4)
        battle.consume_item(mon)
    return HandlerReturn(value=value)
```

## テスト観点
- HP が 50% 超 → 50% 以下になった瞬間に 25% 回復すること
- すでに 50% 以下の状態でさらにダメージを受けた場合は発動しないこと
- 発動後にアイテムが消費されること
- 消費後に再度ダメージを受けても発動しないこと
