# ナナシのみ 実装計画

## 効果概要
ターン終了時、こんらん（揮発性状態）があれば解除する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 50 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ナナシのみ_cure_confusion(battle, ctx, value):
    mon = ctx.source
    if mon.has_volatile("こんらん"):
        battle.volatile_manager.remove(mon, "こんらん")
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- こんらん状態のとき、ターン終了で解除されること
- こんらん以外の揮発性状態（ひるみ等）では発動しないこと
- 発動後にアイテムが消費されること
