# くろいメガネ 実装計画

## 効果概要
あくタイプ技の威力を1.2倍にする（タイプ強化アイテムと同パターン）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | なし（タイプ強化アイテム共通パターン） |
| 実装状態 | **実装済み** |
| イベント | `Event.ON_CALC_POWER_MODIFIER` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック
```python
def くろいメガネ_modify_power_by_type(battle, ctx, value):
    return _modify_power_by_type(ctx.move, value, type_="あく", modifier=4915/4096)
```

## テスト観点
- あくタイプ技の威力が1.2倍になること
- 他タイプの技には効果がないこと
