# エレキシード 実装計画

## 効果概要
エレキフィールドが発動中に登場したとき、または発動したとき、ぼうぎょ（B）を +1 する（消費）。

## 実装仕様

### イベント 1: 登場時

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_SWITCH_IN` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### イベント 2: フィールド変化時

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_FIELD_CHANGE` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | True（消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def _エレキシード_try_boost(battle, mon):
    if battle.terrain.name == "エレキフィールド":
        battle.modify_stats(mon, {"B": +1})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)


def エレキシード_on_switch_in(battle, ctx, value):
    _エレキシード_try_boost(battle, ctx.source)
    return HandlerReturn(value=value)


def エレキシード_on_field_change(battle, ctx, value):
    _エレキシード_try_boost(battle, ctx.source)
    return HandlerReturn(value=value)
```

## テスト観点
- エレキフィールド中に登場したとき B が +1 されること
- エレキフィールドが発動したとき B が +1 されること
- エレキフィールドがない場合は発動しないこと
- 発動後にアイテムが消費されること
