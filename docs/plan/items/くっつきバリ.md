# くっつきバリ 実装計画

## 効果概要
直接攻撃（接触技）を受けたとき、攻撃者にアイテムを転送する（自分はアイテムを失う）。
攻撃者がすでにアイテムを持っている場合は転送しない。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_CONTACT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False |
| 消費条件 | 転送時に自分のアイテムを失う |

## テスト観点
- 接触技を受けたとき、攻撃者にくっつきバリが転送されること

- 転送後、自分のアイテム欄が空になること
- 非接触技を受けたときは転送されないこと
