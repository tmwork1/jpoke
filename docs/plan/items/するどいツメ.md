# するどいツメ 実装計画

## 効果概要
急所ランクを +1 する。攻撃技の急所ランク計算時に常時発動する。ピントレンズと同効果。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CALC_CRITICAL_RANK` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def するどいツメ_boost_critical_rank(battle, ctx, value):
    return HandlerReturn(value=value + 1)
```

## テスト観点
- するどいツメ所持時、急所ランクが +1 されること
- ピントレンズと効果が同一であること（実装を共通化してもよい）
- するどいツメがない場合は急所ランクが変化しないこと
