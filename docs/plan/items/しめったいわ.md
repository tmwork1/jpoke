# しめったいわ 実装計画

## 効果概要
あめ の継続ターンを +3 する（5 ターン → 8 ターン）。さらさらいわと同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION` |
| subject_spec | `"source:self"` |
| priority | 100（デフォルト値。`ON_MODIFY_DURATION` は `docs/spec/turn.md` に未掲載のため、既存の同種ハンドラ（さらさらいわ・あついいわ・つめたいいわ・ひかりのねんど・グランドコート）がいずれも priority を明示指定していない事実を根拠に採用） |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def しめったいわ_resolve_field_count(battle, ctx, value):
    return _resolve_field_count(value, field="あめ", additonal_count=3)
```

## テスト観点
- しめったいわ所持時、あめが 8 ターン継続すること
- しめったいわがない場合は 5 ターンで終了すること
- あめ以外の天候は延長されないこと

## 補足
あまごい（技）・あめふらし（特性）のどちらの発生源でも `WeatherManager.apply()` が唯一の発動経路であるため、
本ハンドラは発生源の区別なく延長を適用する。
