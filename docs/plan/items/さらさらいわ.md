# さらさらいわ 実装計画

## 効果概要
すなあらし の継続ターンを +3 する（5 ターン → 8 ターン）。しめったいわ・あついいわ・つめたいいわと同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def さらさらいわ_resolve_field_count(battle, ctx, value):
    return _resolve_field_count(value, "すなあらし", additonal_count=3)
```

`ON_MODIFY_DURATION` は `field_manager.py` の天候発動処理（技・特性すなおこし/すなはき問わず）で共通して
発火されるため、技によるすなあらしと特性によるすなあらしの両方で自動的に延長される。

## 実装状態
- 実装済み: `src/jpoke/data/item.py` の `さらさらいわ` に `ON_MODIFY_DURATION` ハンドラを登録済み。
- `src/jpoke/handlers/item.py` の `さらさらいわ_resolve_field_count` で実装済み。

## テスト観点
- さらさらいわ所持時、すなあらしが 8 ターン継続すること（技発動時）
- さらさらいわ所持時、特性すなおこしで発動したすなあらしも 8 ターン継続すること
- さらさらいわがない場合は 5 ターンで終了すること
- すなあらし以外の天候は延長されないこと
