# しろいハーブ 実装計画

## 効果概要
ランク低下が発生したとき即座に低下分を打ち消す（消費・1回のみ）。上昇は打ち消さない。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_STAT` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 低下を打ち消したとき `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def しろいハーブ_cancel_stat_drop(battle, ctx, value):
    # value は {stat: change} の辞書
    if any(v < 0 for v in value.values()):
        # 負の変化を 0 に戻す（正の変化は維持）
        value = {s: max(v, 0) for s, v in value.items()}
        mon = ctx.target
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## 注意
`ON_MODIFY_STAT` の `value` が `{stat: change}` 辞書形式かどうか確認する。
だっしゅつパックの `だっしゅつパック_reserve_switch` が同イベントを使っているため参照する。

## テスト観点
- ランク低下が発生したとき、低下を打ち消して発動すること
- ランク上昇のみの場合は発動しないこと
- 上昇と低下が混在する場合、低下分のみ打ち消すこと
- 発動後にアイテムが消費されること
- 消費後はランク低下を打ち消せないこと
