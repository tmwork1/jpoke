# ジャポのみ 実装計画

## 効果概要
物理技でダメージを受けたとき、攻撃者に最大 HP の 1/8 のダメージを与える（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ジャポのみ_retaliate_physical(battle, ctx, value):
    return _retaliate_on_category(battle, ctx, value, "physical")

def _retaliate_on_category(battle, ctx, value, category):
    mon = ctx.defender
    if ctx.move.category == category:
        # マジックガードなどで実際にダメージが通らなかった場合
        # （攻撃者がすでにひんしの場合を含む）は発動・消費しない
        if battle.modify_hp(ctx.attacker, r=-1/8):
            _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)
```

レンブのみ（特殊技版）と実装を共通化し、`_retaliate_on_category` に category を渡して分岐する。

## テスト観点
- 物理技でダメージを受けたとき相手に 1/8 ダメージが入ること
- 特殊技・変化技では発動しないこと
- 発動後にアイテムが消費されること
- 相手へのダメージでひんしになるケース
- 攻撃者がマジックガード持ちの場合は発動せず、アイテムも消費されないこと
