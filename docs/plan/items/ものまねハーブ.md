# ものまねハーブ 実装計画

## 効果概要
相手のポケモンのランクが上昇したとき、同じ能力を +1 コピーする（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_STAT` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | コピー時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ものまねハーブ_copy_stat_boost(battle, ctx, value):
    mon = ctx.target
    # 自分以外のランク上昇が対象
    if ctx.source == mon:
        return HandlerReturn(value=value)
    # 正の変化（上昇）のみコピー
    boosts = {s: 1 for s, v in value.items() if v > 0}
    if boosts:
        battle.modify_stats(mon, boosts)
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## 注意
`ON_MODIFY_STAT` の `ctx.source` が誰のランク変化かを示すかどうか確認する。
複雑な相互作用（マジックミラー等）があるため、詳細な仕様確認が必要。TODO として残すケースあり。

## テスト観点
- 相手のランクが上昇したとき同じ能力が自分にも +1 されること
- 自分自身のランク上昇ではコピーしないこと
- 相手のランク低下ではコピーしないこと
- 発動後にアイテムが消費されること
