# たべのこし 実装計画

## 効果概要
ターン終了時、最大 HP の 1/16 を回復する（切り捨て、最低1）。すでに HP が満タンのときは発動しない。非消費アイテム。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 60 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def たべのこし_heal(battle, ctx, value):
    mon = ctx.source
    if battle.modify_hp(mon, r=1/16):
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

- `battle.modify_hp(mon, r=1/16)` が内部で最大HPの1/16（切り捨て、最低1）を計算し、`ON_MODIFY_HEAL` イベント経由でかいふくふうじ状態による回復無効化（`かいふくふうじ_block_heal`）が自動的に適用される。専用のガードは不要。
- HP が満タンのときは `modify_hp` が 0 を返すため `_announce_item_triggered` は呼ばれず、アイテムも露見しない。

## テスト観点
- HP が満タンのときはターン終了時に回復しないこと（アイテムも露見しない）
- HP が減っているときはターン終了時に最大HPの1/16回復すること
- かいふくふうじ状態のときは回復しないこと
