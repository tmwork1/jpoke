# ぼうごパット 実装計画

## 効果概要
自分が使う技の接触判定を無くす（攻撃側として技を使うとき、接触扱いにならない）（非消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_CONTACT` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def ぼうごパット_negate_contact(battle, ctx, value):
    # 接触判定を強制的に無効にする
    return HandlerReturn(value=False, stop_event=True)
```

## テスト観点
- ぼうごパット持ちが接触技を使ったとき、相手のゴツゴツメット・さめはだ等が発動しないこと
- ぼうごパット持ちが接触技を使ったとき、ほのおのからだ・せいでんき等が発動しないこと
- パンチグローブと組み合わせ: パンチグローブは威力上昇・ぼうごパットは接触無効（同効果の接触無効）
