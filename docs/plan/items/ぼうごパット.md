# ぼうごパット 実装計画

## 効果概要
所持者の技は接触技のままである（パンチグローブ/特性えんかくとは異なり、接触判定自体は消えない）。
そのため、所持者自身が接触技であることに由来する効果（かたいツメ/どくしゅ/ふかしのこぶし）はそのまま発動する。
一方で、相手が「自分の接触攻撃を受けたこと」に反応して発動する効果（さめはだ・ゴツゴツメット・ほのおのからだ等）は防ぐ（非消費）。
ただし、もふもふ（被ダメージ半減）とわるいてぐせ（アイテム強奪）は防げない。

## 実装仕様

新規イベント `Event.ON_CHECK_CONTACT_REACTION` を追加する。
`docs/spec/turn.md` に記載のないイベント（ダメージ計算内部の補助イベント）のため、
既存の `Event.ON_CHECK_CONTACT`（同じくturn.md未掲載）と同様に扱い、Handler優先度はデフォルト(100)とする。

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_CONTACT_REACTION` |
| subject_spec | `"attacker:self"` |
| priority | 100（デフォルト、根拠: 同種の`ON_CHECK_CONTACT`ハンドラに準拠） |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### query.py 側の実装
`PokemonQuery.is_contact_reaction(ctx)` を新設し、
`is_contact(ctx)` の結果を初期値として `Event.ON_CHECK_CONTACT_REACTION` を発火する。

```python
def is_contact_reaction(self, ctx: AttackContext) -> bool:
    return self._events.emit(
        Event.ON_CHECK_CONTACT_REACTION,
        ctx,
        self.is_contact(ctx),
    )
```

- 「相手が接触を受けたことに反応する効果」（さめはだ・てつのトゲ・ぬめぬめ・カーリーヘアー・のろわれボディ・
  ほうし・ほのおのからだ・どくのトゲ・せいでんき・ほろびのボディ・メロメロボディ・ミイラ・とれないにおい・
  さまようたましい・ゆうばく・ゴツゴツメット・キングシールド等の接触カウンター・くちばしキャノンの反撃やけど）
  は `is_contact(ctx)` ではなく `is_contact_reaction(ctx)` を呼ぶように変更する。
- 「自分の技が接触技であることに由来する効果」（かたいツメ/どくしゅ/ふかしのこぶし）と、
  ぼうごパットの対象外と明記されている効果（もふもふ/わるいてぐせ）は、従来どおり `is_contact(ctx)` を呼ぶ
  （ぼうごパットの影響を受けない）。

### ハンドラロジック（疑似コード）
```python
def ぼうごパット_block_contact_reaction(battle, ctx, value):
    # 相手の反応効果の発動を強制的に防ぐ
    return HandlerReturn(value=False, stop_event=True)
```

## テスト観点
- ぼうごパット持ちが接触技を使ったとき、相手のゴツゴツメット・さめはだ・てつのトゲが発動しないこと
- ぼうごパット持ちが接触技を使ったとき、ほのおのからだ・せいでんき・どくのトゲ・ほうし等の状態異常付与が発動しないこと
- ぼうごパット持ちが接触技を使ったとき、ミイラ/とれないにおい/さまようたましいによる特性書き換えが発動しないこと
- ぼうごパット持ちが接触技を使ったとき、かたいツメによる威力上昇はそのまま適用されること（ぼうごパットの影響を受けない）
- ぼうごパット持ちが接触技を使ったとき、どくしゅによる追加効果（相手をどくにする）はそのまま発動すること
- ぼうごパット持ちが接触技を使ったとき、相手のもふもふによるダメージ半減はそのまま適用されること
- ぼうごパット持ちが接触技で相手を攻撃したとき、相手のわるいてぐせによってぼうごパットを奪われること
- パンチグローブと組み合わせ: パンチグローブは威力上昇に加え接触判定自体を無効化する点でぼうごパットと異なる
