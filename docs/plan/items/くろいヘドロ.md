# くろいヘドロ 実装計画

## 効果概要
ターン終了時、ポイズンタイプなら最大 HP の 1/16 を回復する。それ以外のタイプなら最大 HP の 1/8 のダメージを受ける。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 60 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def くろいヘドロ_turn_end(battle, ctx, value):
    mon = ctx.source
    if "どく" in mon.types:
        if battle.modify_hp(mon, r=1/16):
            _announce_item_triggered(battle, mon)
    else:
        battle.modify_hp(mon, r=-1/8)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- どくタイプ持ちの場合、ターン終了時に最大 HP の 1/16 回復すること
- 非どくタイプの場合、ターン終了時に最大 HP の 1/8 ダメージを受けること
- HP が満タンの場合、どくタイプは回復が発生しないこと（modify_hp の空チェック）
- ダメージで HP が 0 になった場合の処理
