# グラスシード 実装計画

## 効果概要
グラスフィールドが発動中に登場したとき、または発動したとき、ぼうぎょ（B）を +1 する（消費）。エレキシードと同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント 1 | `Event.ON_SWITCH_IN` / `"source:self"` |
| イベント 2 | `Event.ON_FIELD_CHANGE` / `"source:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def グラスシード_on_switch_in(battle, ctx, value):
    if battle.terrain.name == "グラスフィールド":
        battle.modify_stats(ctx.source, {"B": +1})
        battle.consume_item(ctx.source)
    return HandlerReturn(value=value)
```

## テスト観点
- グラスフィールド中に登場したとき B が +1 されること
- グラスフィールドが発動したとき B が +1 されること
- グラスフィールドがない場合は発動しないこと
- 発動後にアイテムが消費されること
