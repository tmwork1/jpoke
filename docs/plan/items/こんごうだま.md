# こんごうだま 実装計画

更新日: 2026-06-12

## スコープ

- 仕様書: `docs/spec/items/こんごうだま.md`
- 実装状態: 未実装
- 実装方針: `ON_CALC_POWER_MODIFIER` でディアルガのドラゴン・はがね技を 1.2 倍（4915/4096）にする

## 実装ゴール（完了定義）

- ディアルガがドラゴンまたははがね技を使うと威力が 4915/4096 倍になる
- ディアルガ以外が持っても効果がない（第五世代以降）
- ディアルガ(オリジン)が持つ場合も同様に有効（だいこんごうだまとは別アイテム）

## 実装対象イベント

- `Event.ON_CALC_POWER_MODIFIER` — 技の威力倍率を計算する

## 変更対象（ファイル/関数単位）

- `src/jpoke/handlers/item.py` — `こんごうだま_modify_power` を実装する
- `tests/test_item.py` — テストを追加する

## テスト観点

1. ディアルガがドラゴン技を使うと威力が約 1.2 倍になる。
2. ディアルガがはがね技を使うと威力が約 1.2 倍になる。
3. ディアルガが他のタイプの技を使っても効果がない。
4. 他のポケモンが持っても効果がない。

## 疑似コード

```python
BOOSTED_TYPES = {"ドラゴン", "はがね"}

def こんごうだま_modify_power(battle, ctx, value):
    if "ディアルガ" not in ctx.attacker.name:
        return HandlerReturn(value=value)
    if ctx.move.type not in BOOSTED_TYPES:
        return HandlerReturn(value=value)
    return HandlerReturn(value=apply_fixed_modifier(value, 4915))
```

登録:
```python
"こんごうだま": ItemData(
    fling_power=60,
    handlers={
        Event.ON_CALC_POWER_MODIFIER: h.ItemHandler(
            h.こんごうだま_modify_power,
            subject_spec="attacker:self",
        )
    }
),
```

Note: こんごうだま自体はフォルムチェンジを伴わない通常アイテムであり、はたきおとす・トリック等で
奪う/奪われることができる（`removable` は既定値の `True` のまま）。なげつけるの威力は一次情報の通り60。
フォルムチェンジを伴い奪われない `だいこんごうだま`（`removable=False`, `fling_power=0`）とは区別すること。
