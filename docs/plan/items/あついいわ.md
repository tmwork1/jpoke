# あついいわ 実装計画

## 効果概要
はれ の継続ターンを +3 する（5 ターン → 8 ターン）。さらさらいわ（すなあらし延長）と同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def あついいわ_resolve_field_count(battle, ctx, value):
    return _resolve_field_count(value, field="はれ", additonal_count=3)
```

## テスト観点
- あついいわ所持時、はれが 8 ターン継続すること
- あついいわがない場合は 5 ターンで終了すること
- はれ以外の天候（あめ・ゆき・すなあらし）は延長されないこと
