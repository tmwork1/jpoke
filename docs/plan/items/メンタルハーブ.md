# メンタルハーブ 実装計画

## 効果概要
いちゃもん・アンコール・なりきり・かなしばり・ちょうはつ・さしおさえのいずれかの揮発性状態が付与されたとき、即座に解除する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_VOLATILE_START` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 解除時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
_MENTAL_HERB_TARGETS = frozenset({
    "いちゃもん", "アンコール", "なりきり", "かなしばり", "ちょうはつ", "さしおさえ"
})

def メンタルハーブ_cure_mental_volatile(battle, ctx, value):
    mon = ctx.target
    volatile_name = ctx.volatile_name  # 付与された揮発性状態名（要確認）
    if volatile_name in _MENTAL_HERB_TARGETS:
        battle.volatile_manager.remove(mon, volatile_name)
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## 注意
`ON_VOLATILE_START` の `ctx` に付与された揮発性状態名がどのようにアクセスできるか確認する。
`EventContext` に `volatile_name` フィールドがあるか調査が必要。

## テスト観点
- いちゃもん・アンコール・なりきり・かなしばり・ちょうはつ・さしおさえが付与されたとき即解除されること
- 対象外の揮発性状態（こんらん等）では発動しないこと
- 発動後にアイテムが消費されること
- 消費後は対象揮発性状態が付与されたままになること
