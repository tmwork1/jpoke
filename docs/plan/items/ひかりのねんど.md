# ひかりのねんど 実装計画

## 効果概要
リフレクター・ひかりのかべ・オーロラベールの持続ターンを +3 する（5 ターン → 8 ターン）。
かるいしと同効果・同実装パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def ひかりのねんど_resolve_field_count(battle, ctx, value):
    for field_name in ("リフレクター", "ひかりのかべ", "オーロラベール"):
        result = _resolve_field_count(value, field=field_name, additional_count=3)
        if result.value != value:
            return result
    return HandlerReturn(value=value)
```

## テスト観点
- ひかりのねんど所持時、リフレクターが 8 ターン継続すること
- ひかりのねんど所持時、ひかりのかべが 8 ターン継続すること
- ひかりのねんど所持時、オーロラベールが 8 ターン継続すること
- ひかりのねんどがない場合は 5 ターンで終了すること
- 対象外の場状態は延長されないこと
