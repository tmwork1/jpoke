# タラプのみ 実装計画

## 効果概要
特殊技でダメージを受けた直後、とくぼう（D）を +1 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def タラプのみ_boost_spdef(battle, ctx, value):
    mon = ctx.defender
    if ctx.move.category == "特殊":
        battle.modify_stats(mon, {"D": +1})
        _announce_item_triggered(battle, mon)
        battle.consume_item(mon)
    return HandlerReturn(value=value)
```

## 注意
- アッキのみの特殊版（アッキ=物理被弾でB+1、タラプ=特殊被弾でD+1）
- じゅくせい持ちでも効果量は+1のまま（スター以外のピンチきのみと違い2倍にならない）
- HP1/4閾値とは無関係。特殊技でダメージを受けたら即発動

## テスト観点
- 特殊技でダメージを受けたとき D が +1 されること
- 物理技では発動しないこと
- 発動後にアイテムが消費されること
