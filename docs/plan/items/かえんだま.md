# かえんだま 実装計画

## 効果概要
ターン終了時に持ち主にやけどを付与する。すでにやけど状態の場合は付与しない（ailment_manager が重複チェックする）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 90 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def かえんだま_apply_burn(battle, ctx, value):
    mon = ctx.source
    if battle.ailment_manager.apply(mon, "やけど", source=mon):
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- ターン終了時、かえんだま持ちにやけどが付与されること
- すでにやけど状態の場合は再付与されないこと
- みずのベール等でやけどが無効化されている場合は付与されないこと
- どくタイプ・はがねタイプにも付与されること（タイプによる免疫はない）
