# ねばりのかぎづめ 実装計画

## 効果概要
バインド技（まきつく・ほのおのうず等）の継続ターンを常に 7 ターンに固定する。

## 実装仕様

バインド系の継続ターン設定がどのフローで行われるかを確認する必要がある。
`ON_MODIFY_DURATION` または専用のバインドターン設定フックが必要。

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION`（要確認） |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def ねばりのかぎづめ_fix_bind_duration(battle, ctx, value):
    # value が [volatile_name, count] 形式の場合
    name, count = value
    if name in {"まきつく", "ほのおのうず", "まとわりつく", "なみのり", "うずしお", "すなじごく"}:
        count = 7
    return HandlerReturn(value=[name, count])
```

## 注意
バインド技の継続ターン設定フローの確認が必要。`ON_MODIFY_DURATION` で対応できるか、
または専用のイベントフックが必要かを調査すること（TODO）。

## テスト観点
- ねばりのかぎづめ持ちがバインド技を使ったとき継続ターンが 7 ターンになること
- ねばりのかぎづめがない場合は 4〜5 ターン（乱数）になること
