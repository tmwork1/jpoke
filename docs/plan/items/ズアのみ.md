# ズアのみ 実装計画

## 効果概要
HP が最大値の 25% 以下になった瞬間に とくぼう（D）を +1 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HP_CHANGED` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ズアのみ_boost_spdef(battle, ctx, value):
    mon = ctx.target
    hp_after = mon.hp
    hp_before = hp_after + value
    if hp_before * 4 > mon.max_hp and hp_after * 4 <= mon.max_hp:
        battle.modify_stats(mon, {"D": +1})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- HP が 25% 超 → 25% 以下になった瞬間にとくぼうが +1 されること
- チイラのみと同パターンで実装できること
- 発動後にアイテムが消費されること
