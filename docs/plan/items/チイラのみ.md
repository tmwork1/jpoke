# チイラのみ 実装計画

## 効果概要
HP が最大値の 25% 以下になった瞬間に こうげき（A）を +1 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HP_CHANGED` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def チイラのみ_boost_attack(battle, ctx, value):
    mon = ctx.target
    hp_after = mon.hp
    hp_before = hp_after + value
    # 閾値を跨いだとき（25% 超 → 25% 以下）のみ発動
    if hp_before * 4 > mon.max_hp and hp_after * 4 <= mon.max_hp:
        battle.modify_stats(mon, {"A": +1})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- HP が 25% 超 → 25% 以下になった瞬間にこうげきが +1 されること
- すでに 25% 以下でさらにダメージを受けた場合は発動しないこと
- 発動後にアイテムが消費されること
- こうげきランクが +6 のとき発動してもランク上限で止まること
