更新日: 2026-06-12

## スコープ

- 仕様書: `docs/spec/items/しらたま.md`
- 実装状態: 未実装
- 実装方針: `ON_CALC_POWER_MODIFIER` でパルキアのドラゴン・みず技を 1.2 倍（4915/4096）にする

## 実装ゴール（完了定義）

- パルキアがドラゴンまたはみず技を使うと威力が 4915/4096 倍になる
- パルキア以外が持っても効果がない（第五世代以降）
- パルキア(オリジン)が持つ場合も有効（だいしらたまとは別アイテム）

## 実装対象イベント

- `Event.ON_CALC_POWER_MODIFIER` — 技の威力倍率を計算する

- `src/jpoke/handlers/item.py` — `しらたま_modify_power` を実装する

## テスト観点

1. パルキアがドラゴン技を使うと威力が約 1.2 倍になる。
2. パルキアがみず技を使うと威力が約 1.2 倍になる。
3. パルキアが他のタイプの技を使っても効果がない。
4. 他のポケモンが持っても効果がない。

```python
BOOSTED_TYPES = {"ドラゴン", "みず"}

def しらたま_modify_power(battle, ctx, value):
    if "パルキア" not in ctx.attacker.name:
        return HandlerReturn(value=value)
    if ctx.move.type not in BOOSTED_TYPES:
        return HandlerReturn(value=value)
    return HandlerReturn(value=apply_fixed_modifier(value, 4915))
```
