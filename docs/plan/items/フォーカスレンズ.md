# フォーカスレンズ 実装計画

## 効果概要
後攻（自分より速い相手に対して行動する）ときに命中率を 1.2 倍にする。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_ACCURACY` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def フォーカスレンズ_modify_accuracy(battle, ctx, value):
    # アナライズ特性と同じ後攻判定ロジック
    attacker_player = battle.get_player(ctx.attacker)
    is_second = battle.query.is_second_actor(attacker_player)
    if is_second is None:
        defender_player = battle.get_player(ctx.defender)
        defender_state = battle.player_states[defender_player]
        is_second = (
            ctx.defender.executed_move is not None
            or defender_state.has_switched
        )
    if is_second:
        value = apply_fixed_modifier(value, 4915)  # 1.2倍
    return HandlerReturn(value=value)
```

## テスト観点
- 後攻時（自分が遅い）に命中率が 1.2 倍になること
- 先攻時（自分が速い）は命中率が変化しないこと
