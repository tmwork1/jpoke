# ゴツゴツメット 実装計画

## 効果概要
直接攻撃（接触技）を受けた際に、攻撃した相手に最大 HP の 1/6 のダメージを与える。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_CONTACT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def ゴツゴツメット_contact_damage(battle, ctx, value):
    # value は接触判定の結果（True=接触）
    # さめはだ特性と同パターン
    common.apply_contact_counter_chip(battle, ctx, ratio=1/6)
    return HandlerReturn(value=value)
```

## 注意
- `handlers/ability.py` の `さめはだ` や `ほのおのからだ` と同パターン
- `common.py` に `apply_contact_counter_chip` ヘルパーがあるか確認する。なければ `handlers/item.py` 内にプライベート実装する

## テスト観点
- 接触技を受けたとき相手に最大 HP の 1/6 ダメージが入ること
- 非接触技を受けたときはダメージが入らないこと
- ゴツゴツメットがない場合は接触ダメージが発生しないこと
- 相手が接触ダメージでひんしになるケース
