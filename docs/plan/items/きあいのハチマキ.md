# きあいのハチマキ 実装計画

## 効果概要
ひんしになるダメージを受けたとき 11.7% の確率で HP1 で耐える（非消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_MOVE_DAMAGE` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False（非消費アイテム） |
| 消費条件 | なし |

### ハンドラロジック（疑似コード）
```python
def きあいのハチマキ_survive_by_chance(battle, ctx, value):
    mon = ctx.defender
    # ひんしになるダメージ かつ 確率発動
    if value >= mon.hp and battle.random.random() < 0.117:
        value = mon.hp - 1
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- ひんしダメージを受けたとき 11.7% で HP1 で耐えること
- 耐えないケース（確率外）では通常通りひんしになること
- HP が 1 以上残る（非ひんし）ダメージには発動しないこと
- 乱数固定テストで発動・非発動の両ケースを確認すること
