# きゅうこん 実装計画

## 効果概要
みずタイプの技でダメージを受けたとき、とくこう（C）を +1 する（消費）。
仕様書: `docs/spec/items/きゅうこん.md`

ひかりごけ（みず技被弾でとくぼう+1）・じゅうでんち（でんき技被弾でこうげき+1）と同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `_announce_and_consume_item(battle, mon)` |

### ハンドラロジック（疑似コード）
```python
def きゅうこん_boost_spatk_on_water_hit(battle, ctx, value):
    """きゅうこん: みず技でダメージを受けたときとくこう+1。"""
    return _boost_stat_on_type_hit(battle, ctx, value, type_="みず", stats={"C": +1})
```

`_boost_stat_on_type_hit` は既存ヘルパー（ひかりごけ・じゅうでんち等で利用）。

## 実装ステップ

1. `src/jpoke/handlers/item.py` に `きゅうこん_boost_spatk_on_water_hit` を追加。
2. `src/jpoke/data/item.py` の `きゅうこん` に `Event.ON_DAMAGE_HIT` ハンドラを登録（`handlers={}` を書き換える）。
3. `tests/test_item.py` にテストを追加し、全テストが通ることを確認。

## テスト観点
- みず技でダメージを受けたとき、とくこうが +1 されること
- みず以外のタイプ（でんき・ほのお等）では発動しないこと
- 発動後にアイテムが消費されること
- ランク上限（+6）での処理が正しいこと（発動しない）
- たんじゅん持ちはとくこうが +2 されること
- あまのじゃく持ちはとくこうが -1 されること
