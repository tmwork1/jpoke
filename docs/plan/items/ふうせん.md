# ふうせん 実装計画

## 効果概要
浮遊状態になり地面技が当たらない。攻撃を受けると消滅する（非消費だがダメージで除去される）。

## 実装仕様

### イベント 1: 浮遊判定を返す

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_FLOATING` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |

### イベント 2: ダメージを受けたとき消滅

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False |

### ハンドラロジック（疑似コード）
```python
def ふうせん_check_floating(battle, ctx, value):
    return HandlerReturn(value=True)


def ふうせん_pop_on_hit(battle, ctx, value):
    mon = ctx.defender
    # ダメージを受けたとき消滅（use_item で除去）
    battle.consume_item(mon)
    _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- ふうせん持ちに地面技が当たらないこと
- ダメージ技でふうせんが消滅し、その後地面技が当たること
- 変化技ではふうせんが消滅しないこと（ON_DAMAGE_HIT はダメージ後に発火）
- じゅうりょく状態下でふうせんの浮遊効果が打ち消されること（別ハンドラで対応）
