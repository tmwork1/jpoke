# くろいてっきゅう 実装計画

## 効果概要
素早さを 0.5 倍にする。また、浮遊状態を無効化し、地面技を受けるようになる。

## 実装仕様

### イベント 1: 素早さ半減

| 項目 | 内容 |
|------|------|
| イベント | `DomainEvent.ON_CALC_SPEED` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### イベント 2: 浮遊無効化

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CHECK_FLOATING` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def くろいてっきゅう_halve_speed(battle, ctx, value):
    return HandlerReturn(value=value // 2)


def くろいてっきゅう_negate_floating(battle, ctx, value):
    # 浮遊状態を強制的に無効にする
    return HandlerReturn(value=False, stop_event=True)
```

## テスト観点
- くろいてっきゅう所持時、素早さ実数値が半減すること
- ふゆう特性・マグネットライズ等の浮遊状態が無効化されること（地面技が当たること）
- くろいてっきゅうがない場合は素早さ・浮遊判定が通常通りであること
- 消費条件なしのため交代後も継続して効果が発動すること
