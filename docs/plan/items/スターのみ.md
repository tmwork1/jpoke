# スターのみ 実装計画

## 効果概要
HP が最大値の 25% 以下になった瞬間に、こうげき・ぼうぎょ・とくこう・とくぼう・すばやさのうちランダムな1つが +2 される（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HP_CHANGED` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def スターのみ_random_boost(battle, ctx, value):
    mon = ctx.target
    hp_after = mon.hp
    hp_before = hp_after + value
    if hp_before * 4 > mon.max_hp and hp_after * 4 <= mon.max_hp:
        stat = battle.random.choice(["A", "B", "C", "D", "S"])
        battle.modify_stats(mon, {stat: +2})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- HP が 25% 超 → 25% 以下になった瞬間にランダムな能力が +2 されること
- 5種（A/B/C/D/S）のいずれかが上昇すること
- 発動後にアイテムが消費されること
- 乱数固定テストで各能力の上昇を確認すること
