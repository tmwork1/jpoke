# じゃくてんほけん 実装計画

## 効果概要
効果抜群のダメージを受けたとき、こうげき（A）と とくこう（C）を +2 ずつ上昇させる（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def じゃくてんほけん_boost_on_super_effective(battle, ctx, value):
    mon = ctx.defender
    if battle.damage_calculator.calc_def_type_modifier(ctx) > 1:
        battle.modify_stats(mon, {"A": +2, "C": +2})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- 効果抜群のダメージを受けたとき A・C が +2 ずつ上昇すること
- 効果抜群でない（等倍・いまひとつ）の攻撃では発動しないこと
- 発動後にアイテムが消費されること
- ランク上限（+6）での処理が正しいこと
