# はっきんだま 実装計画

更新日: 2026-06-12

## スコープ

- 仕様書: `docs/spec/items/はっきんだま.md`
- 実装状態: 未実装
- 実装方針: `ON_CALC_POWER_MODIFIER` でギラティナのドラゴン・ゴースト技を 1.2 倍（4915/4096）にする

## 実装ゴール（完了定義）

- ギラティナ（アナザー/オリジン）がドラゴンまたはゴースト技を使うと威力が 4915/4096 倍になる
- ギラティナ以外が持っても効果がない
- SV (第九世代) ではフォルムチェンジ効果なし（だいはっきんだまに移行済み）
- SV では持ち主がギラティナでもアイテムを奪われる可能性がある（`removable` は現状 True）

## 実装対象イベント

- `Event.ON_CALC_POWER_MODIFIER` — 技の威力倍率を計算する

## 変更対象（ファイル/関数単位）

- `src/jpoke/handlers/item.py` — `はっきんだま_modify_power` を実装する
- `tests/test_item.py` — テストを追加する

## テスト観点

1. ギラティナ(アナザー)がドラゴン技を使うと威力が約 1.2 倍になる。
2. ギラティナ(オリジン)がゴースト技を使うと威力が約 1.2 倍になる。
3. ギラティナが他のタイプの技を使っても効果がない。
4. 他のポケモンが持っても効果がない。
5. アイテムを奪われた後は効果が消える。

BOOSTED_TYPES = {"ドラゴン", "ゴースト"}

def はっきんだま_modify_power(battle, ctx, value):
    if "ギラティナ" not in ctx.attacker.name:
        return HandlerReturn(value=value)
    if ctx.move.type not in BOOSTED_TYPES:
        return HandlerReturn(value=value)
    return HandlerReturn(value=apply_fixed_modifier(value, 4915))
