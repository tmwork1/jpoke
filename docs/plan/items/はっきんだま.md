# はっきんだま 実装計画

更新日: 2026-07-04

## スコープ

- 仕様書: `docs/spec/items/はっきんだま.md`
- 実装状態: 実装済み
- 実装方針:
  - `ON_CALC_POWER_MODIFIER` でギラティナ（アナザー/オリジン問わず）のドラゴン・ゴースト技を 1.2 倍（4915/4096）にする
  - SV では通常の持ち物（`removable` 既定値 `True`）として扱い、なげつける・はたきおとす等の対象にする
  - なげつけるはギラティナが使用した場合のみ失敗し、それ以外のポケモンが使用した場合は威力60で成功する

## 実装ゴール（完了定義）

- ギラティナ（アナザー/オリジン問わず）がドラゴンまたはゴースト技を使うと威力が 4915/4096 倍になる
- ギラティナ以外が持っても威力上昇効果はない
- SV (第九世代) ではフォルムチェンジ効果なし（だいはっきんだまに移行済み）
- SV では通常の持ち物と同様にトリック・はたきおとす等で奪われる／失われる（`removable` は既定値 `True` のまま）
- ギラティナ以外が持っている場合の `なげつける` は威力60で成功する
- ギラティナ（アナザー/オリジン問わず）が持っている状態で `なげつける` を使用すると失敗する（一次情報に明記された固有の仕様）

## 実装対象イベント

- `Event.ON_CALC_POWER_MODIFIER` — 技の威力倍率を計算する
- `なげつける_check_item`（`handlers/move_attack.py` の共通ハンドラ）— ギラティナ + はっきんだまの組み合わせのみ失敗させる特殊分岐を追加

## 変更対象（ファイル/関数単位）

- `src/jpoke/data/item.py` — `はっきんだま` の `ItemData`（`fling_power=60`、`removable` は既定値のまま）と `handlers` 登録
- `src/jpoke/handlers/item.py` — `はっきんだま_modify_power`（ギラティナ両フォルムを対象にする）
- `src/jpoke/handlers/move_attack.py` — `なげつける_check_item` にギラティナ+はっきんだまの失敗分岐を追加（他アイテムには影響しない）
- `tests/test_item.py` / `tests/moves_attack/test_move_na.py` — テストを追加

## 実装ステップ

1. `src/jpoke/handlers/item.py` の `はっきんだま_modify_power` で `_dedicated_item_modify_power` に渡す許可名を
   `frozenset({"ギラティナ(アナザー)", "ギラティナ(オリジン)"})` にする（こんごうだま・しらたまと同じパターン）。
2. `src/jpoke/data/item.py` の `はっきんだま` に `fling_power=60` を設定し、`removable=False` は設定しない
   （だいはっきんだま等のフォルムロック専用アイテムと異なり、SV では通常通り奪われる）。
3. `src/jpoke/handlers/move_attack.py` の `なげつける_check_item` に、
   `attacker.item.base_name == "はっきんだま"` かつ `attacker.name.startswith("ギラティナ")` の場合のみ失敗する分岐を追加する。
4. テストを追加し、全テストが通ることを確認する。

## テスト観点

1. ギラティナ(アナザー)がドラゴン技を使うと威力が約 1.2 倍になる。
2. ギラティナ(オリジン)がゴースト技を使うと威力が約 1.2 倍になる。
3. ギラティナが他のタイプの技を使っても効果がない。
4. 他のポケモンが持っても効果がない。
5. アイテムを奪われた後は効果が消える（トリック・はたきおとす等で通常通り奪われる）。
6. ギラティナ(アナザー)/ (オリジン) がはっきんだまを持って `なげつける` を使うと失敗する。
7. ギラティナ以外がはっきんだまを持って `なげつける` を使うと威力60で成功する。

## リスクと対策

- リスク: `removable` を誤って `False`（フォルムロック専用アイテム扱い）にしてしまうミス。
  だいはっきんだまとの実装コピーの際に誤って `removable=False, fling_power=0` を流用したバグが過去に存在したため、
  レビュー時に必ず「SV のはっきんだまは通常の持ち物として奪われる／なげつけられる」ことを確認する。
- リスク: `はっきんだま_modify_power` の対象名にギラティナの両フォルムを含めないと、
  オリジンフォルムのギラティナが持っている場合に威力補正が効かなくなる（こんごうだま・しらたまの実装との非対称に注意）。
- リスク: なげつけるの失敗分岐はギラティナ+はっきんだまの組み合わせのみに限定し、
  他の道具（こんごうだま・しらたま等）や他のポケモンがはっきんだまを持つ場合には影響しないようにする。
