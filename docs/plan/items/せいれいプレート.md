# せいれいプレート 実装計画

## 効果概要
フェアリータイプ技の威力を1.2倍にする（タイプ強化アイテムと同パターン）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `docs/spec/items/せいれいプレート.md` |
| 実装状態 | **実装済み** |
| イベント | `Event.ON_CALC_POWER_MODIFIER` |
| subject_spec | `"attacker:self"` |
| priority | 100（デフォルト、他のタイプ強化アイテムと同一） |
| once | False |
| 消費条件 | なし（非消費アイテム） |
| fling_power | 90（プレート共通。こわもてプレートをなげつけた場合のみ自身の補正がかかり108になる） |

### ハンドラロジック
```python
def せいれいプレート_modify_power_by_type(battle, ctx, value):
    return _modify_power_by_type(ctx.move, value, type_="フェアリー", modifier=4915)
```

- `ctx.move.type` は攻撃時点で確定済みの技タイプを参照するため、追加の分岐は不要。
- アルセウスのフォルム・タイプシフトおよびプレートに対するトリック・すりかえ・はたきおとす等の道具変更阻害は、アルセウス専用特性マルチタイプ側（`ability.py` の `マルチタイプ_apply_type` / `マルチタイプ_block_item_change`）で実装済み。本アイテム自体には追加実装しない（アイテム単体の効果はタイプ別威力上昇のみ）。
- さばきのつぶてが持たせたプレートに応じてタイプ変化する挙動は技側（`さばきのつぶて`）の実装範囲であり、本アイテムの計画には含めない（`docs/progress/move.md` で別途管理）。

## テスト観点
- フェアリータイプ技の威力が1.2倍になること
- 他タイプの技には効果がないこと
