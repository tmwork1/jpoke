# せんせいのツメ 実装計画

## 効果概要
約 23.4% の確率で先制する（行動順が1上がる）。クイックドロウ特性と同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `DomainEvent.ON_CALC_BACK_TIER` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def せんせいのツメ_calc_back_tier(battle, ctx, value):
    # 約 23.4% の確率で先制（+1）
    if battle.random.random() < 0.234:
        return HandlerReturn(value=value + 1)
    return HandlerReturn(value=value)
```

## テスト観点
- せんせいのツメ発動時（確率）、行動順ティアが +1 されること
- 発動しない場合は行動順が変化しないこと
- クイックドロウ特性と同パターンで実装できること
- 乱数を固定したテストで発動・非発動の両ケースを確認すること
