# こうかくレンズ 実装計画

## 効果概要
攻撃側の命中率を 1.1 倍にする。技の命中率計算時に常時発動する。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_ACCURACY` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def こうかくレンズ_modify_accuracy(battle, ctx, value):
    # value は命中率整数値（4096 基準）
    return HandlerReturn(value=apply_fixed_modifier(value, 4506))
```

## テスト観点
- こうかくレンズ所持時、命中率 90 の技が約 99 になること
- こうかくレンズがない場合は命中率が変化しないこと
- ひかりのこな（防御側）との重複適用が正しく計算されること
