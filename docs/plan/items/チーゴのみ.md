# チーゴのみ 実装計画

## 効果概要
ターン終了時、こおり状態であれば回復する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 50 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def チーゴのみ_cure_freeze(battle, ctx, value):
    mon = ctx.source
    if mon.ailment.name == "こおり":
        battle.ailment_manager.remove(mon)
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- こおり状態のとき、ターン終了で回復すること
- こおり以外の状態異常では発動しないこと
- 発動後にアイテムが消費されること
