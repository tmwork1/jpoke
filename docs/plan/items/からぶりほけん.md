# からぶりほけん 実装計画

## 効果概要
自分の技が外れたとき、すばやさ（S）を +2 する（消費）。

## 実装仕様

技が外れたことを検知するイベントが必要。現時点では `ON_MOVE_END` 内で「技が外れたフラグ」の確認が想定されるが、専用イベントの有無を確認する。

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MOVE_END`（技外れフラグ確認）または専用イベント |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 技外れ時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def からぶりほけん_boost_speed_on_miss(battle, ctx, value):
    mon = ctx.attacker
    if ctx.move_missed:  # 仮のフラグ名
        battle.modify_stats(mon, {"S": +2})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## 注意
技が外れたことを検知するイベントまたはフラグが未整備の可能性がある。
`AttackContext` に `move_missed` フィールドが存在するか確認し、なければ TODO として残す。

## テスト観点
- 技が外れたとき、すばやさが +2 されること
- 技が命中したときは発動しないこと
- 発動後にアイテムが消費されること
