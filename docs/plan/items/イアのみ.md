# イアのみ 実装計画

## 効果概要
HP が最大値の 25% 以下になった瞬間に最大 HP の 33% を回復する（消費）。SV では混乱しない。フィラのみと同効果。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HP_CHANGED` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def イアのみ_heal_on_quarter_hp(battle, ctx, value):
    mon = ctx.target
    hp_after = mon.hp
    hp_before = hp_after + value
    if hp_before * 4 > mon.max_hp and hp_after * 4 <= mon.max_hp:
        battle.modify_hp(mon, r=1/3)
        battle.consume_item(mon)
    return HandlerReturn(value=value)
```

## テスト観点
- HP が 25% 超 → 25% 以下になった瞬間に 33% 回復すること
- フィラのみ系5種（フィラ・ウイ・マゴ・バンジ・イア）は同パターンで実装できること
- 発動後にアイテムが消費されること
- SV では混乱しないこと
