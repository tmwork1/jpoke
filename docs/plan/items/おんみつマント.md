# おんみつマント 実装計画

## 効果概要
技の追加効果（secondary effect）を受けない（非消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_SECONDARY_CHANCE` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def おんみつマント_negate_secondary_effect(battle, ctx, value):
    # 追加効果の確率を 0 にする
    return HandlerReturn(value=0)
```

## 注意
`ON_MODIFY_SECONDARY_CHANCE` の `subject_spec` が `"target:self"` か `"defender:self"` かを確認する（EventContext なので `"target:self"` が正しい）。
`ctx.target` が追加効果を受ける側（防御側）を指すことを確認する。

## テスト観点
- おんみつマント持ちへの追加効果（まひ・やけど等）が発動しないこと
- 本体ダメージ自体は通常通り受けること
- 追加効果の確率 0% 化により完全無効になること
