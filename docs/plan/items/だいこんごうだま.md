# だいこんごうだま 実装計画

更新日: 2026-06-12

## スコープ

- 仕様書: `docs/spec/items/だいこんごうだま.md`
- 実装状態: 未実装
- 実装方針: `ON_SWITCH_IN` / `ON_ITEM_GAINED` でディアルガをオリジンフォルムにフォルムチェンジし、`ON_CALC_POWER_MODIFIER` でドラゴン・はがね技を 1.2 倍にする

## 実装ゴール（完了定義）

- ディアルガがだいこんごうだまを持って登場するとディアルガ(オリジン)にフォルムチェンジする
- ディアルガ(オリジン)がドラゴンまたははがね技を使うと威力が 4915/4096 倍になる
- `removable=False` はすでに ItemData に設定済み（オリジンフォルムは奪われない）
- ディアルガ以外が持っても効果なし

## 実装対象イベント

- `Event.ON_SWITCH_IN` — 登場時にフォルムチェンジを発動する
- `Event.ON_ITEM_GAINED` — バトル中にアイテムを入手したとき
- `Event.ON_CALC_POWER_MODIFIER` — ドラゴン/はがね技威力 1.2 倍

## 変更対象（ファイル/関数単位）

## テスト観点

1. ディアルガがだいこんごうだまを持って登場するとディアルガ(オリジン)になる。
2. ディアルガ(オリジン)がドラゴン技を使うと威力が約 1.2 倍になる。
3. ディアルガ(オリジン)がはがね技を使うと威力が約 1.2 倍になる。
4. ディアルガ以外が持っても効果がない。

BOOSTED_TYPES = {"ドラゴン", "はがね"}

def だいこんごうだま_form_change(battle, ctx, value):
    mon = ctx.source
    if mon.name != "ディアルガ":
        return HandlerReturn(value=value)
    mon.set_form("ディアルガ(オリジン)")
    return HandlerReturn(value=value)

def だいこんごうだま_modify_power(battle, ctx, value):
    if "ディアルガ" not in ctx.attacker.name:
        return HandlerReturn(value=value)
    if ctx.move.type not in BOOSTED_TYPES:
        return HandlerReturn(value=value)
    return HandlerReturn(value=apply_fixed_modifier(value, 4915))
