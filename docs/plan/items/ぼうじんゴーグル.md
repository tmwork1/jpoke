# ぼうじんゴーグル 実装計画

## 効果概要
粉技（powder ラベル）を無効化する + 砂嵐・あられのダメージを受けない（非消費）。

## 実装仕様

### イベント 1: 粉技無効化

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_BEFORE_APPLY_MOVE` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False |

### イベント 2: 天候ダメージ無効化

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_NON_MOVE_DAMAGE` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |

### ハンドラロジック（疑似コード）
```python
def ぼうじんゴーグル_block_powder_move(battle, ctx, value):
    if ctx.move.has_label("powder"):
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)


def ぼうじんゴーグル_block_weather_damage(battle, ctx, value):
    if ctx.hp_change_reason == "weather":
        return HandlerReturn(value=0, stop_event=True)
    return HandlerReturn(value=value)
```

## 注意
`ctx.hp_change_reason` の値が `"weather"` で砂嵐・あられの両方を判別できるか確認する。

## テスト観点
- こなゆき・どくのこな等の粉技を受けたとき無効化されること
- 砂嵐・あられのターン終了ダメージを受けないこと
- 砂嵐中でも岩/はがね/こおりタイプ以外のポケモンがダメージを受けないこと
- 粉技以外の技は通常通り発動すること
