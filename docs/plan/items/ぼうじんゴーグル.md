# ぼうじんゴーグル 実装計画

## 効果概要
粉・胞子技（powder フラグ）を無効化する + 特性ほうし（Effect Spore）による状態異常付与を無効化する
+ 天候（すなあらし）のターン終了ダメージを受けない（非消費）。

現行仕様（ポケモンチャンピオンズ／第九世代）には「あられ」天候が存在せず、天候ダメージの対象は
すなあらしのみ（ゆきはダメージを与えない）。

## 実装仕様

### イベント 1: 粉技無効化

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_BEFORE_APPLY_MOVE` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False |

ぼうじん特性（同じイベント・同じ優先度で登録）は switch_manager でポケモン登場時に
特性→アイテムの順にハンドラ登録されるため、同速の場合は特性側が先に評価され、
特性ぼうじんとぼうじんゴーグルを同時に持つ場合は特性側の無効化が優先される
（`docs/spec/items/ぼうじんゴーグル.md` の「特性による無効化が優先される」に対応）。

粉技を無効化した際は `_announce_item_triggered` でアイテムを公開し、
`LogCode.MOVE_IMMUNED` 相当のメッセージ（「ぼうじんゴーグルが発動した」）を表示する。

### イベント 2: 天候ダメージ無効化

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_NON_MOVE_DAMAGE` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | False |

`ぼうじん_block_sandstorm_damage`（特性ぼうじん）と同じイベント・ロールを使用する。

### イベント 3: 特性ほうし（Effect Spore）の無効化

特性ほうしの発動箇所（`handlers/ability.py` の `ほうし_maybe_inflict_ailment_on_contact`）で、
接触してきた攻撃側（`ctx.attacker`）がぼうじんゴーグルを持っている場合は状態異常を付与しない。
アイテム側に専用のイベントハンドラを追加するのではなく、既存の特性ハンドラに直接ガード条件を
追加する（`ctx.attacker.has_item("ぼうじんゴーグル")`）。理由: `ON_BEFORE_APPLY_AILMENT` は
あらゆる状態異常付与で発火する汎用イベントであり、そこで「特性ほうし由来かどうか」を
安全に判別する手段がないため。

### ハンドラロジック（疑似コード）
```python
def ぼうじんゴーグル_block_powder_move(battle, ctx, value):
    if ctx.move.has_flag("powder"):
        _announce_item_triggered(battle, ctx.defender)
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)


def ぼうじんゴーグル_block_weather_damage(battle, ctx, value):
    if ctx.hp_change_reason == "sandstorm":
        return HandlerReturn(value=0, stop_event=True)
    return HandlerReturn(value=value)


# handlers/ability.py 側に追記
def ほうし_maybe_inflict_ailment_on_contact(battle, ctx, value):
    if (
        not battle.query.is_contact_reaction(ctx)
        or ctx.attacker.has_item("ぼうじんゴーグル")
    ):
        return HandlerReturn(value=value)
    ...
```

## 注意
- 現行仕様に「あられ」天候は存在しないため、天候ダメージ無効化の対象は実質すなあらしのみ。
- いかりのこなによるちゅうもくのまと（攻撃対象の強制）はダブルバトル専用の仕様であり、
  本プロジェクトはシングルバトルのみを対象とするため実質的に影響しない。
  `いかりのこな` の `MoveData` には `powder` フラグを付与していないが、シングルバトルでは
  攻撃対象強制効果自体が意味を持たないため実害はない。
- ふんじん状態そのものの発動は防げない（受けた瞬間の技のみ無効化）。

## テスト観点
- こなゆき・ねむりごな等の粉技を受けたとき無効化されること
- 粉技を無効化した際にアイテムが公開される（`item.revealed`）こと
- すなあらしのターン終了ダメージを受けないこと
- 特性ぼうじんと同時に持つ場合、特性側の無効化が優先されること（メッセージ・処理の重複がないこと）
- 特性ほうし（Effect Spore）を持つ相手に接触しても状態異常にならないこと
- 粉技以外の技は通常通り発動すること
