# ねらいのまと 実装計画

## 効果概要
タイプ相性による無効（0 倍）を無効化する。例: ゴーストタイプへのノーマル技が当たるようになる（非消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_CALC_DEF_TYPE_MODIFIER` |
| subject_spec | `"defender:self"` |
| priority | -100（他の倍率計算後に適用） |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def ねらいのまと_negate_immunity(battle, ctx, value):
    # value が 0（無効）の場合、4096（1倍）に戻す
    if value == 0:
        value = 4096
    return HandlerReturn(value=value)
```

## 注意
複雑な相互作用あり。以下を要確認：
- ゴーストタイプが「ノーマル・かくとう 無効」を持つ場合
- 地面タイプへのでんき技（浮いていない場合）との競合
- ふゆう特性による浮遊（ON_CHECK_FLOATING）との関係（こちらは別イベント）
- priority を低くすることで他の倍率補正後に適用する

## テスト観点
- ねらいのまと持ちに対して通常無効になる技が命中すること
- 倍率 0 の技が 1 倍として計算されること
- ねらいのまとがない場合は通常通り無効になること
- 複数の無効要因が重なるケースの確認
