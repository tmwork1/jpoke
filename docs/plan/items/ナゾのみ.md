# ナゾのみ 実装計画

## 効果概要
効果抜群のダメージを受けたとき、HP を最大 HP の 25% 回復する（消費）。
ただしダメージ固定技（一撃必殺技を除く）はタイプ相性上抜群であっても発動しない。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT`（`AttackContext`、`docs/spec/turn.md`未掲載イベント。実際にダメージを与えた場合のみ`MoveExecutor._execute_hit`から発火されるため、まもる・特性無効化・みがわり・ばけのかわ/アイスフェイスで防いだ場合は自動的に発火しない） |
| subject_spec | `"defender:self"` |
| priority | 100（他のON_DAMAGE_HITハンドラ同様の既定値。既存の同種アイテムハンドラ（じゃくてんほけん等）に合わせる） |
| once | 不要（発動可否は`battle.query.is_super_effective`で判定し、都度消費処理を行う） |
| 消費条件 | 発動時に `_announce_and_consume_item(battle, mon)` を呼ぶ（アナウンス＋消費） |

### 発動条件の詳細
- `ctx.move.has_flag("fixed_damage")`が真の場合は発動しない（いかりのまえば・カタストロフィ・
  ちきゅうなげ・ナイトヘッドなど、タイプ相性を無視した固定ダメージ技）。
  - 一撃必殺技（`ohko`フラグ）は`fixed_damage`フラグを持たせず、通常通りタイプ相性判定を行う
    （耐えた場合に弱点なら発動する仕様のため）。
- `battle.query.is_super_effective(ctx)`（内部で`calc_def_type_modifier`を使用）で判定する。
  - らんきりゅう・タールショットはタイプ相性自体（`ON_CALC_DEF_TYPE_MODIFIER`）を変更するため、
    自然に反映される。
- ほしがる/どろぼう/ついばむ/むしくい/はたきおとす/やきつくす、特性マジシャンで技のタイプ・
  対象が変わる場合の相互作用は`docs/spec/items/ナゾのみ.md`の「詳細な仕様」を参照。
  これらの技は`Event.ON_DAMAGE_HIT`で処理されるが、本ハンドラの方がpriorityが高い
  （先に評価される）ため、アイテムを失う前にナゾのみが発動する。

### ハンドラロジック（疑似コード）
```python
def ナゾのみ_heal_on_super_effective(battle, ctx, value):
    mon = ctx.defender
    if ctx.move.has_flag("fixed_damage"):
        return HandlerReturn(value=value)
    if battle.query.is_super_effective(ctx):
        battle.modify_hp(mon, r=1/4)
        _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)
```

## 既知の制限
- HPが1の状態でこらえるにより攻撃技のダメージが0に補正された場合、`ON_DAMAGE_HIT`自体が
  発火しないため発動しない。詳細は`docs/spec/items/じゃくてんほけん.md`の既知の制限と同様。

## テスト観点
- 効果抜群のダメージを受けたとき HP が 25% 回復すること
- 効果抜群でない（等倍・いまひとつ）の攻撃では発動しないこと
- 発動後にアイテムが消費されること
- ダメージ固定技（例: ちきゅうなげ）で弱点タイプでも発動しないこと
- 一撃必殺技（例: じわれ）を弱点タイプで受けて倒れなかった場合は発動すること
