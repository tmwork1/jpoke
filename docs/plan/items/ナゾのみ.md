# ナゾのみ 実装計画

## 効果概要
効果抜群のダメージを受けたとき、HP を最大 HP の 25% 回復する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ナゾのみ_heal_on_super_effective(battle, ctx, value):
    mon = ctx.defender
    if battle.damage_calculator.calc_def_type_modifier(ctx) > 1:
        battle.modify_hp(mon, r=1/4)
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## 注意
効果抜群判定には `battle.damage_calculator.calc_def_type_modifier(ctx) > 1` を使う（じゃくてんほけんと同パターン）。

## テスト観点
- 効果抜群のダメージを受けたとき HP が 25% 回復すること
- 効果抜群でない（等倍・いまひとつ）の攻撃では発動しないこと
- 発動後にアイテムが消費されること
- HP が満タンに近い場合でも 25% 回復処理が走ること
