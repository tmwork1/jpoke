# かるいし 実装計画

## 効果概要
持ち主の体重を半分にする（くさむすび・はかりにかけるのダメージ算出に影響）。
小数点第二位を切り捨て、切り捨てた結果0.1kgより軽くなる場合は0.1kgとする。
なげつける威力は30。

## 実装状態

体重半減効果は `src/jpoke/model/pokemon.py` の `weight` プロパティで直接実装済み。
ハンドラ登録は不要（`data/item.py` には `fling_power=30` のみ登録）。

`weight` プロパティは末尾で `max(w, 0.1)` を返し、ライトメタル・ヘヴィメタル・
かるいしのいずれの組み合わせでも最低値0.1kgを下回らないようにする
（例: 体重0.1kgのポケモンがかるいしを持つと `int(0.1*0.5*10)/10` が0.0になってしまう
浮動小数点の丸め込みバグがあったため、下限クランプを追加した）。

## テスト観点
- かるいし所持時、`mon.weight` が `mon.data.weight * 0.5` であること
- かるいしなしでは通常体重であること
- 体重0.1kgのポケモン（例: ゴース）がかるいしを持っても `mon.weight` が0.1kgのままであること（下限クランプの確認）
