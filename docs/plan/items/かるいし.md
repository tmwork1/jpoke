# かるいし 実装計画

## 効果概要
リフレクター・ひかりのかべ・オーロラベールの持続ターンを +3 する（5 ターン → 8 ターン）。
さらさらいわと同じ `_resolve_field_count` パターンを使う。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def かるいし_resolve_field_count(battle, ctx, value):
    # リフレクター・ひかりのかべ・オーロラベールの3種に対してそれぞれ延長
    for field_name in ("リフレクター", "ひかりのかべ", "オーロラベール"):
        result = _resolve_field_count(value, field=field_name, additional_count=3)
        if result.value != value:
            return result
    return HandlerReturn(value=value)
```

## 注意
`_resolve_field_count` は `handlers/item.py` 内のプライベートヘルパー関数（さらさらいわで使用済み）。
同関数の引数名スペルが `additonal_count`（typo）か `additional_count` か実装時に確認が必要。

## テスト観点
- かるいし所持時、リフレクターが 8 ターン継続すること
- かるいし所持時、ひかりのかべが 8 ターン継続すること
- かるいし所持時、オーロラベールが 8 ターン継続すること
- かるいしがない場合は 5 ターンで終了すること
- 対象外の場状態（すなあらし等）は延長されないこと
