# どくどくだま 実装計画

## 効果概要
ターン終了時に持ち主にもうどくを付与する。すでに状態異常がある場合は付与しない。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_TURN_END` |
| subject_spec | `"source:self"` |
| priority | 90 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def どくどくだま_apply_poison(battle, ctx, value):
    mon = ctx.source
    if battle.ailment_manager.apply(mon, "もうどく", source=mon):
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- ターン終了時、どくどくだま持ちにもうどくが付与されること
- すでに状態異常がある場合は付与されないこと
- どくタイプ・はがねタイプには付与されないこと（ailment_manager の通常チェックで防ぐ）
- かえんだまとの比較: もうどく（蓄積ダメージ増加）vs やけど（攻撃半減）の違い
