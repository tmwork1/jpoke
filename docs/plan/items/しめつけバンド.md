# しめつけバンド 実装計画

## 効果概要
バインドダメージを最大 HP の 1/6 に増加する（通常は最大 HP の 1/8）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_BIND_DAMAGE` |
| subject_spec | `"source:foe"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

`ON_MODIFY_BIND_DAMAGE` は `バインド_damage` が `EventContext(source=バインドされた側)` で発火する。
バインドした側（しめつけバンド持ち）は `battle.foe(ctx.source)` に当たるため `subject_spec="source:foe"`。

## 発火側の改修

`handlers/volatile.py` の `バインド_damage` に `ON_MODIFY_BIND_DAMAGE` 発火を追加する。

### `handlers/volatile.py` の `バインド_damage` を改修
```python
def バインド_damage(battle, ctx, value):
    mon = ctx.source
    battle.volatile_manager.tick(mon, "バインド")
    if not mon.has_volatile("バインド"):
        return HandlerReturn(value=value)

    base_ratio = mon.volatiles["バインド"].bind_damage_ratio
    r = battle.events.emit(
        Event.ON_MODIFY_BIND_DAMAGE,
        ctx,  # source=バインドされた側、foe=バインドした側
        base_ratio
    )
    battle.modify_hp(mon, r=-r)
    return HandlerReturn(value=value)
```

### ハンドラロジック（アイテム側）
```python
def しめつけバンド_boost_bind_damage(battle, ctx, value):
    return HandlerReturn(value=1/6)
```

## テスト観点
- しめつけバンド持ちがバインド技を使ったとき毎ターンのバインドダメージが最大 HP の 1/6 になること
- しめつけバンドがない場合は最大 HP の 1/8 のバインドダメージになること
