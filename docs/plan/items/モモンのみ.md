# モモンのみ 実装計画

## 効果概要
どく・もうどく状態を回復する（消費）。
- どく・もうどく状態が付与された瞬間に自動消費して回復する。
- かがくへんかガス解除・ぶきよう解除などでアイテム効果が有効化された瞬間、既にどく・もうどく状態であれば回復する。
- ほおばる・おちゃかい等による強制発動でも回復する。
- 上記のいずれでも回復しなかった場合の保険として、ターン終了時点でどく・もうどく状態が残っていれば回復する。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_APPLY_AILMENT`（どく・もうどく付与直後・即時回復） |
| subject_spec | `"target:self"` |
| priority | 50 |
| イベント | `Event.ON_ITEM_ENABLED`（アイテム効果の有効化直後） |
| subject_spec | `"source:self"` |
| priority | 50 |
| イベント | `Event.ON_FORCE_BERRY_TRIGGER`（ほおばる等による強制発動） |
| subject_spec | `"source:self"` |
| priority | 50 |
| イベント | `Event.ON_TURN_END`（保険としてのターン終了時チェック） |
| subject_spec | `"source:self"` |
| priority | 50 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)`（`_announce_and_consume_item` 経由） |

### ハンドラロジック（疑似コード）

`_cure_ailment_berry` / `_cure_ailment_berry_on_apply` という状態異常回復きのみ共通のヘルパーを利用する。

```python
def モモンのみ_cure_poison(battle, ctx, value):
    # ON_TURN_END / ON_ITEM_ENABLED / ON_FORCE_BERRY_TRIGGER 用
    mon = ctx.source
    if mon.ailment.name in ("どく", "もうどく") and battle.ailment_manager.remove(mon):
        _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)


def モモンのみ_cure_poison_on_apply(battle, ctx, value):
    # ON_APPLY_AILMENT 用: どく・もうどく付与直後に治療して消費する
    mon = ctx.target
    if value in ("どく", "もうどく") and battle.ailment_manager.remove(mon):
        _announce_and_consume_item(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- どく・もうどく付与直後（`ON_APPLY_AILMENT`）にターン終了を待たず即座に回復し消費されること
- どく・もうどく状態のとき、ターン終了で回復すること
- どく・もうどく以外の状態異常では発動しないこと
- 発動後にアイテムが消費されること
