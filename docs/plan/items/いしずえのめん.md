# いしずえのめん 実装計画

## 効果概要
オーガポン専用。いしずえのめんオーガポンにフォルムチェンジし、物理技の攻撃補正を1.2倍にする。
battle中はremovable=Falseで取得・移動不可。

## 実装仕様

| 項目 | 内容 |
|------|------|
| 仕様書 | `docs/spec/items/_おめん.md` |
| 実装状態 | **実装済み**（物理技のみ。特殊技は未対応） |
| イベント | `Event.ON_CALC_ATK_MODIFIER` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（removable=False） |

### ハンドラロジック（共通: `オーガポンのめん_boost_atk`）
```python
def オーガポンのめん_boost_atk(battle, ctx, value):
    if ctx.move.category == "物理":
        value = apply_fixed_modifier(value, 4915)  # ×1.2 (4915/4096)
    return HandlerReturn(value=value)
```

## テスト観点
- いしずえのめんを持たせたオーガポンの物理技ダメージが1.2倍になること
- 持たせていない場合は補正なしであること
- battle中にアイテムを取られないこと（removable=False）
