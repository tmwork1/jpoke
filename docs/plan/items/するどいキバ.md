# するどいキバ 実装計画

## 効果概要
ダメージ技命中時、10% の確率でひるみを付与する（非消費）。おうじゃのしるしと同効果。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_HIT` |
| subject_spec | `"attacker:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def するどいキバ_flinch_on_hit(battle, ctx, value):
    if ctx.move.category != "変化":
        common.apply_secondary(battle, ctx, volatile="ひるみ", chance=0.1)
    return HandlerReturn(value=value)
```

## テスト観点
- ダメージ技命中時、10% の確率で相手がひるむこと
- 変化技では発動しないこと
- おうじゃのしるしと同パターンで実装できること（関数名のみ異なる）
- おんみつマント持ちの相手にはひるみが発生しないこと
