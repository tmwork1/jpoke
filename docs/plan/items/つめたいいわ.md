# つめたいいわ 実装計画

## 効果概要
ゆき の継続ターンを +3 する（5 ターン → 8 ターン）。さらさらいわと同パターン。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_DURATION` |
| subject_spec | `"source:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def つめたいいわ_resolve_field_count(battle, ctx, value):
    return _resolve_field_count(value, field="ゆき", additonal_count=3)
```

## テスト観点
- つめたいいわ所持時、ゆきが 8 ターン継続すること
- つめたいいわがない場合は 5 ターンで終了すること
- ゆき以外の天候は延長されないこと
