# ゆきだま 実装計画

## 効果概要
こおり技でダメージを受けたとき、こうげき（A）を +1 する（消費）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ゆきだま_boost_attack_on_ice(battle, ctx, value):
    mon = ctx.defender
    if ctx.move.type == "こおり":
        battle.modify_stats(mon, {"A": +1})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- こおり技でダメージを受けたとき A が +1 されること
- こおり以外の技では発動しないこと
- 発動後にアイテムが消費されること
