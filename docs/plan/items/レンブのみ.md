# レンブのみ 実装計画

## 効果概要
特殊技でダメージを受けたとき、攻撃者に最大 HP の 1/8 のダメージを与える（消費）。ジャポのみの特殊版。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_DAMAGE_HIT` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def レンブのみ_retaliate_special(battle, ctx, value):
    mon = ctx.defender
    if ctx.move.category == "特殊":
        battle.modify_hp(ctx.attacker, r=-1/8)
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## テスト観点
- 特殊技でダメージを受けたとき相手に 1/8 ダメージが入ること
- 物理技・変化技では発動しないこと
- 発動後にアイテムが消費されること
- ジャポのみ（物理版）と対称的な実装であること
