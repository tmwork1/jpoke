# カシブのみ 実装計画

## スコープ
- 優先度: -
- 仕様書: `docs/spec/items/_タイプ半減きのみ.md`（カシブのみ節）
- 実装状態: **実装済み**

## 効果概要
- 効果バツグンのゴーストわざを受けたときに消費して、1度だけダメージを1/2にする（17種の通常タイプ半減きのみ共通処理）。
- なげつける威力: 10。
- ポルターガイストを効果抜群で受けた場合、カシブのみの効果は発動しダメージを半減するが、
  ポルターガイスト自体はアイテム所持チェック（`ON_BEFORE_APPLY_MOVE`, priority=130）が
  ダメージ計算（`ON_CALC_DAMAGE_MODIFIER`）より先に行われるため成功する。

## 実装方針
- `Event.ON_CALC_DAMAGE_MODIFIER` に `subject_spec="defender:self"` でハンドラを登録し、
  共通処理 `handlers/item.py::_modify_super_effective_damage(type_="ゴースト", modifier=2048/4096)` を呼ぶ
  （他の17種タイプ半減きのみと同一パターン）。
- リーサル計算用に `lethal_handlers` の `LethalEvent.ON_BEFORE_HIT` にも
  `handlers/lethal.py::カシブのみ_resist_ghost`（内部で共通処理 `_type_resist_berry(type_="ゴースト")` を呼ぶ）を登録する。
  この共通処理は `calc_damages` 側で既にダメージ半減が反映されているため、
  アイテム消費と `item_enabled` の更新のみを行う。

## 実装済み内容
