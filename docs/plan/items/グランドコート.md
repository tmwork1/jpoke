# グランドコート 実装計画

## 効果概要
持たせたポケモンがフィールドを発動させたとき、継続ターンを +3 する（5ターン→8ターン）。
あついいわ（天候延長）と同パターン。4種フィールドすべてが対象。

### ハンドラロジック（疑似コード）
```python
def グランドコート_resolve_field_count(battle, ctx, value):
    for field_name in ("エレキフィールド", "グラスフィールド", "ミストフィールド", "サイコフィールド"):
        result = _resolve_field_count(value, field=field_name, additonal_count=3)
        if result.value != value:
            return result
    return HandlerReturn(value=value)
```

ひかりのねんど（スクリーン延長）と同じ構造。`_resolve_field_count` を流用。

3. `tests/test_item.py` にテストを追加し、全テストが通ることを確認。

## テスト観点
- グランドコート所持時、フィールドが8ターン継続すること（4種すべて）
- 所持していない場合は5ターンで終了すること
- フィールド以外の持続（天候・スクリーン等）は延長されないこと
