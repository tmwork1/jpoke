# ビビリだま 実装計画

## 効果概要
いかく でこうげきが下がったとき、すばやさ（S）を +1 する（消費しない）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_STAT` |
| subject_spec | `"target:self"` |
| priority | 100 |
| once | True（消費アイテム） |
| 消費条件 | 発動時に `battle.consume_item(mon)` |

### ハンドラロジック（疑似コード）
```python
def ビビリだま_boost_speed_on_intimidate(battle, ctx, value):
    mon = ctx.target
    # いかく による A の低下かどうか判定
    # ctx.source の ability が "いかく" かどうか（または専用フラグ）
    if (
        "A" in value
        and value["A"] < 0
        and hasattr(ctx, "ability_name")
        and ctx.ability_name == "いかく"
    ):
        battle.modify_stats(mon, {"S": +1})
        battle.consume_item(mon)
        _announce_item_triggered(battle, mon)
    return HandlerReturn(value=value)
```

## 注意
いかく判定方法（`ctx.source.ability.name == "いかく"` または専用フィールド）を確認する。
だっしゅつパック（`ON_MODIFY_STAT` でランク低下を検知）のパターンを参照する。

## テスト観点
- いかく によってこうげきが下がったとき S が +1 されること
- いかく以外のランク低下では発動しないこと（あまのじゃく等）
- 発動後にアイテムが消費されること
