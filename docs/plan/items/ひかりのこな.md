# ひかりのこな 実装計画

## 効果概要
防御側が持つアイテム。相手の命中率を 0.9 倍にする（回避率を上げる効果と同義）。

## 実装仕様

| 項目 | 内容 |
|------|------|
| イベント | `Event.ON_MODIFY_ACCURACY` |
| subject_spec | `"defender:self"` |
| priority | 100 |
| once | False |
| 消費条件 | なし（非消費アイテム） |

### ハンドラロジック（疑似コード）
```python
def ひかりのこな_reduce_accuracy(battle, ctx, value):
    # value は命中率整数値（4096 基準）
    # 防御側が持つことで相手の命中率を下げる
    # 一撃必殺技（ohkoフラグ）は第三世代以降ひかりのこなの影響を受けないため除外する
    if not ctx.move.has_flag("ohko"):
        return HandlerReturn(value=apply_fixed_modifier(value, 3686))
    return HandlerReturn(value=value)
```

## テスト観点
- ひかりのこな所持時、相手の命中率が約 0.9 倍になること
- 攻撃側視点では命中率が下がること
- こうかくレンズ（攻撃側）との重複適用が正しく計算されること
- ひかりのこながない場合は命中率が変化しないこと
- 一撃必殺技（つのドリル等）の命中率には影響しないこと
- 必中技（アイテムの効果があっても回避できない技）はそもそも `ON_MODIFY_ACCURACY` が発火せず必中のままであること（`move_executor.py` 側の共通処理）
