# だいしらたま 実装計画

更新日: 2026-06-12

## スコープ

- 仕様書: `docs/spec/items/だいしらたま.md`
- 実装状態: 未実装
- 実装方針: `ON_SWITCH_IN` / `ON_ITEM_GAINED` でパルキアをオリジンフォルムにフォルムチェンジし、`ON_CALC_POWER_MODIFIER` でドラゴン・みず技を 1.2 倍にする

## 実装ゴール（完了定義）

- パルキアがだいしらたまを持って登場するとパルキア(オリジン)にフォルムチェンジする
- パルキア(オリジン)がドラゴンまたはみず技を使うと威力が 4915/4096 倍になる
- `removable=False` はすでに ItemData に設定済み（オリジンフォルムは奪われない）
- パルキア以外が持っても効果なし

## 実装対象イベント

- `Event.ON_SWITCH_IN` — 登場時にフォルムチェンジを発動する
- `Event.ON_ITEM_GAINED` — バトル中にアイテムを入手したとき
- `Event.ON_CALC_POWER_MODIFIER` — ドラゴン/みず技威力 1.2 倍
