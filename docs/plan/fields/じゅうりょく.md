# じゅうりょく 実装計画書

## 概要

全体の場の状態の一つ。命中率の上昇・全ポケモンの接地化・一部技の使用禁止・Gのちから強化の効果を持つ。

## 実装対象イベント

### 1. ON_FIELD_ACTIVATE（揮発性状態解除）

**優先度**: priority=100（デフォルト）
**subject_spec**: `"source:self"`
**実装関数**: `じゅうりょく_activate_release_volatiles`
**場所**: `handlers/field.py`

- 各プレイヤーのアクティブポケモンからそらをとぶ・でんじふゆう揮発性状態を解除する
- `battle.volatile_manager.remove(mon, volatile_name)` を使用
- 揮発性状態が存在しない場合は何もしない（idempotent）

### 2. ON_CALC_POWER_MODIFIER（Gのちから威力補正）

**優先度**: priority=100（デフォルト）
**subject_spec**: `"attacker:self"`
**実装関数**: `じゅうりょく_g_power_modifier`
**場所**: `handlers/field.py`

- 技名が "Gのちから" の場合、威力を1.5倍（固定倍率 6144/4096）にする
- 条件: `ctx.move.name == "Gのちから"`

### 3. ON_TRY_MOVE_1（使用禁止技ブロック）

**優先度**: priority=100（デフォルト）
**subject_spec**: `"attacker:self"`
**実装関数**: `じゅうりょく_block_gravity_move`
**場所**: `handlers/field.py`

- `gravity_restricted` フラグを持つ技を失敗させる
- 条件: `"gravity_restricted" in ctx.move.flags`
- ログ: `LogCode.MOVE_FAILED` reason="じゅうりょく"

### 4. ON_MODIFY_ACCURACY（命中率補正）

**優先度**: priority=100（デフォルト）
**subject_spec**: `"attacker:self"`
**実装関数**: `じゅうりょく_modify_accuracy`
**場所**: `handlers/field.py`

- 命中率を約1.67倍（固定倍率 6840/4096）にする
- `apply_fixed_modifier(value, 6840)` を使用

### 5. ON_CHECK_FLOATING（接地化）

**優先度**: priority=100（デフォルト）
**subject_spec**: `"source:self"`
**実装関数**: `じゅうりょく_grounded`
**場所**: `handlers/field.py`

- 全ポケモンを接地状態（浮いていない）にする
- `HandlerReturn(value=False, stop_event=True)` を返す

### 6. ON_TURN_END（カウントダウン・終了）

**優先度**: priority=140（docs/spec/turn.md 準拠）
**subject_spec**: `"source:self"`
**実装関数**: `じゅうりょく_tick_global_field`（共通 tick_global_field のラッパー）
**場所**: `handlers/field.py`

## MoveFlag 追加

`MoveFlag = Literal[...]` に `"gravity_restricted"` を追加する（`types/literals.py`）。

以下の技に `gravity_restricted` フラグを追加する:

| 技名 | 場所 |
|------|------|
| はねる | data/move.py |
| とびげり | data/move.py |
| とびひざげり | data/move.py |
| でんじふゆう | data/move.py |
| そらをとぶ | data/move.py |
| とびはねる | data/move.py |
| フライングプレス | data/move.py |
