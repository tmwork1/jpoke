## ハンドラ一覧

### 1. いやしのねがい_faint_and_set_side_field（発動・ひんし）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_STATUS_HIT |
| subject_spec | （MoveHandler デフォルト: `attacker:self`） |
| priority | 100（デフォルト） |

**ロジック:**
1. 自陣 side を取得
2. `side.activate("いやしのねがい", 1)` でフィールドを設置（count=1: ターン経過では解除されない）
3. `battle.faint(attacker)` で使用者をひんしにする

### 2. いやしのねがい_heal_on_switch_in（交代先のHP・状態異常回復）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_SWITCH_IN |
| subject_spec | `source:self`（EventContext: 場に出たポケモンが自陣のとき） |
| priority | 100 |
| 根拠 | docs/spec/turn.md: 「いやしのねがいによる回復」Priority 100 |

**ロジック:**
1. `ctx.source`（場に出たポケモン）と自陣 side を取得
2. `mon.hp == mon.max_hp and not mon.ailment.is_active` なら **return**（フィールド保留・第八世代以降仕様）
3. `battle.modify_hp(mon, v=mon.max_hp - mon.hp)` で HP を全回復
4. `battle.ailment_manager.remove(mon)` で状態異常を解除（揮発性状態は対象外）
5. `side.deactivate("いやしのねがい")` でフィールドを解除
