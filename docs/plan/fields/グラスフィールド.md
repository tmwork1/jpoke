# グラスフィールド 実装計画書

## 概要

地形（テレイン）の一つ。くさ技の威力強化・地面技の威力弱化・ターン終了時HP回復の3つの効果を持つ。

## 実装対象イベント

### 1. ON_CALC_POWER_MODIFIER（威力補正）

**優先度**: priority=100（デフォルト）  
**subject_spec**: `"attacker:self"`  
**実装関数**: `グラスフィールド_power_modifier`  
**場所**: `handlers/field.py`

- くさタイプ技の威力を 1.3 倍（固定倍率 5325 / 4096）にする
  - 条件: `ctx.move.type == "くさ"` かつ `not battle.query.is_floating(ctx.attacker)`
- じしん・じならし・マグニチュードの威力を 0.5 倍（固定倍率 2048 / 4096）にする
  - 条件: `ctx.move.name in ["じしん", "じならし", "マグニチュード"]` かつ `not battle.query.is_floating(ctx.defender)`
- 両効果は同一ハンドラ内で処理（1ターンに1回発火、attacker:self で登録）

### 2. ON_TURN_END（HP回復）

**優先度**: priority=60（docs/spec/turn.md 準拠）  
**subject_spec**: `"source:self"`  
**実装関数**: `グラスフィールド_heal`  
**場所**: `handlers/field.py`

- 接地ポケモンのHPを `max_hp // 16` 回復する
- `battle.modify_hp(ctx.source, r=1/16)` を使用
- かいふくふうじは ON_MODIFY_HEAL 経由で自動的にブロックされる

### 3. ON_TURN_END（カウントダウン・終了）

**優先度**: priority=140（docs/spec/turn.md 準拠）  
**subject_spec**: `"source:self"`  
**実装関数**: `tick_terrain`（共通関数）  
**場所**: `handlers/field.py`

## データ登録（data/field/terrain.py）

```python
"グラスフィールド": FieldData(
    handlers={
        Event.ON_CALC_POWER_MODIFIER: h.FieldHandler(
            h.グラスフィールド_power_modifier,
            subject_spec="attacker:self",
        ),
        Event.ON_TURN_END: [
            h.FieldHandler(
                h.グラスフィールド_heal,
                subject_spec="source:self",
                priority=60,
            ),
            h.FieldHandler(
                h.tick_terrain,
                subject_spec="source:self",
                priority=140,
            ),
        ]
    },
),
```
