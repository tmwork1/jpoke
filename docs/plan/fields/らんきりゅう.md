### らんきりゅう_type_modifier（ON_CALC_DEF_TYPE_MODIFIER）

```python
# src/jpoke/handlers/field.py
def らんきりゅう_type_modifier(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """らんきりゅう中にひこうタイプの弱点（でんき/いわ/こおり）を0.5倍に軽減する"""
    if (
        ctx.defender.has_type("ひこう")
        and ctx.move.type in {"でんき", "いわ", "こおり"}
    ):
        value = apply_fixed_modifier(value, 2048)  # ×0.5
    return HandlerReturn(value=value)
```

- subject_spec: `"defender:self"`（AttackContext のため）
- priority: デフォルト 100
- ON_CALC_DEF_TYPE_MODIFIER は turn.md に記載なし。同イベントの既存ハンドラ（きもったま等）に倣い priority=100（デフォルト）を使用。
