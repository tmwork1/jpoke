# おいかぜ 実装計画書

## 仕様参照

`docs/spec/fields/おいかぜ.md` を参照。

## 実装ファイル

- ハンドラ実装（サイドフィールド）: `src/jpoke/handlers/field.py`（おいかぜ_speed_boost, おいかぜ_tick_side_field）
- ハンドラ登録（サイドフィールド）: `src/jpoke/data/field/side_field.py`
- ハンドラ実装（技）: `src/jpoke/handlers/move_status.py`（おいかぜ_set_side_field）
- ハンドラ登録（技）: `src/jpoke/data/move.py`

## ハンドラ一覧

### 1. おいかぜ_speed_boost（素早さ2倍）

| 項目 | 値 |
|------|----|
| イベント | DomainEvent.ON_CALC_SPEED |
| subject_spec | `source:self`（EventContext: 非攻撃フェーズ） |
| priority | 100（デフォルト） |

**ロジック:** `value * 2`

### 2. おいかぜ_tick_side_field（カウントダウン）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_TURN_END |
| subject_spec | `source:self`（EventContext: 非攻撃フェーズ） |
| priority | 130 |
| 根拠 | docs/spec/turn.md: 「おいかぜの継続・終了」Priority 130 |

**ロジック:** `tick_side_field(battle, ctx, value, name="おいかぜ")`

### 3. おいかぜ_set_side_field（発動）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_STATUS_HIT |
| subject_spec | `attacker:self`（AttackContext: 攻撃フェーズ） |
| priority | 100（デフォルト） |

**ロジック:** 自陣営に「おいかぜ」を4ターン設定する。すでに発動中の場合は `stop_event=True` で失敗。
