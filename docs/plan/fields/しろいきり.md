## ハンドラ一覧

### 1. しろいきり_prevent_stat_drop（ランク低下無効）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_BEFORE_MODIFY_STAT |
| subject_spec | `target:self`（EventContext: 非攻撃フェーズ） |
| priority | 130 |
| 根拠 | docs/spec/turn.md: 「しろいきりによるランク補正下げ無効」Priority 130 |

**ロジック:**
1. `ctx.is_foe_target()` が True（発動元が対象と異なる）かつ `ctx.can_bypass_status_guard(battle)` が False（すりぬけ等でない）の場合
2. `value` dict からマイナス値（ランク低下）のエントリを取り除き、プラス値（ランク上昇）のみ残す

### 2. しろいきり_tick_side_field（カウントダウン）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_TURN_END |
| subject_spec | `source:self`（EventContext: 非攻撃フェーズ） |
| priority | 130 |
| 根拠 | docs/spec/turn.md: 「しろいきりの継続・終了」Priority 130 |

**ロジック:** `tick_side_field(battle, ctx, value, name="しろいきり")`
