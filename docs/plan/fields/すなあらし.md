## ハンドラ一覧

### 1. すなあらし_D_boost（いわタイプ特防1.5倍）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_CALC_DEF_MODIFIER |
| subject_spec | `defender:self`（AttackContext: 攻撃フェーズ） |
| priority | 100（デフォルト） |

**ロジック:**
1. `ctx.defender.has_type("いわ")` かつ `ctx.move.category == "特殊"` → `apply_fixed_modifier(value, 6144)`（×1.5）

**注意:** `ON_CALC_DEF_MODIFIER` は攻撃フェーズイベントなので subject_spec は `defender:self`。
ばんのうがさとの相互作用はなし（はれ/あめの威力補正とは異なる）。

### 2. tick_weather（カウントダウン）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_TURN_END |
| subject_spec | `source:self`（EventContext: 非攻撃フェーズ） |
| priority | 10 |
| 根拠 | docs/spec/turn.md: 「にほんばれ等の天気終了」Priority 10 |

**ロジック:** 1P側のみ `battle.weather_manager.tick_down()` を呼ぶ（共通関数 `tick_weather` を使用）。

### 3. すなあらし_turn_end（ターン終了ダメージ）

| 項目 | 値 |
|------|----|
| イベント | Event.ON_TURN_END |
| subject_spec | `source:self`（EventContext: 非攻撃フェーズ） |
| priority | 20 |
| 根拠 | docs/spec/turn.md: 「すなあらし・あられのダメージ」Priority 20 |

**ロジック:**
1. `battle.weather.name != "すなあらし"` → 早期リターン（カウントが0になった直後のターンはダメージなし）
2. `ctx.source.has_type("いわ") or ctx.source.has_type("じめん") or ctx.source.has_type("はがね")` → 早期リターン（タイプ免疫）
3. `battle.modify_hp(ctx.source, r=-1/16, reason="sandstorm")`

**注意:** `reason="sandstorm"` を渡すことで、特性（すなかき/すながくれ/すなのちから/ぼうじん）の
`ON_MODIFY_NON_MOVE_DAMAGE` ハンドラが `ctx.hp_change_reason == "sandstorm"` を検出して無効化できる。

## 特性免疫の実装（ability.py 側）

以下の特性が `ON_MODIFY_NON_MOVE_DAMAGE` で `reason="sandstorm"` を検出して damage を 0 に変更する:
- `すなかき`: `すなかき_ignore_sandstorm_damage`
- `すながくれ`: `すながくれ_ignore_sandstorm_damage`
- `すなのちから`: `すなのちから_ignore_sandstorm_damage`
- `ぼうじん`: `ぼうじん_block_sandstorm_damage`

共通実装: `_ignore_sandstorm_damage` ヘルパー関数を各ハンドラから呼び出す。

## data/field.py 登録

```python
"すなあらし": FieldData(
    handlers={
        Event.ON_CALC_DEF_MODIFIER: h.FieldHandler(
            h.すなあらし_D_boost,
            subject_spec="defender:self",
        ),
        Event.ON_TURN_END: [
            h.FieldHandler(
                h.tick_weather,
                subject_spec="source:self",
                priority=10,
            ),
            h.FieldHandler(
                h.すなあらし_turn_end,
                subject_spec="source:self",
                priority=20,
            ),
        ],
    },
),
```
