# 特性実装計画: せいぎのこころ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/せいぎのこころ.md`

## 実装ゴール

- あくタイプの攻撃技を受けると、こうげきが1段階上がる。
- あく以外のタイプでは発動しない。
- A が最大のときは発動しない。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=100 で実装。
- `ctx.move.type == "あく"` チェック後、`battle.modify_stats(mon, {"A": +1}, ...)` を呼ぶ。
- **ひらいしん**（でんき吸収でC↑1）と同パターン。

def せいぎのこころ_on_damage(battle, ctx, value):
    if ctx.move.type != "あく":
        return HandlerReturn(value=value)
    mon = ctx.defender
    if battle.modify_stats(mon, {"A": +1}, source=ctx.attacker):
        announce_ability_triggered(battle, ctx, value, mon=mon)
