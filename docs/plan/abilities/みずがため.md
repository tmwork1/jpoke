# 特性実装計画: みずがため

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/みずがため.md`

## 実装ゴール

- みずタイプの攻撃技でダメージを受けたとき、ぼうぎょが2段階上がる。
- 変化技のみずタイプでは発動しない（ON_DAMAGE_HIT は damage > 0 のときのみ発火）。
- B が最大のときは発動しない。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=20 で実装。
- priority は `turn.md` ON_DAMAGE 「みずがため等: 特定タイプを受けた」の priority 20 に従う。
- `ctx.move.type == "みず"` でタイプチェック後、`battle.modify_stats(mon, {"B": +2}, ...)` を呼ぶ。
- **こんがりボディ**（ほのお吸収でB↑2）と同パターン。

## 変更候補

- `src/jpoke/data/ability.py` — `みずがため` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `みずがため_on_damage` を実装する

## テスト観点

1. みず技受けると B が 2 段階上がる。
2. ほのお等他タイプでは発動しない。
3. B が最大のときは発動しない。
