# 特性実装計画: うるおいボイス

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/うるおいボイス.md`

## 実装ゴール

- 音技をみずタイプに変換し、威力を 1.2 倍にする。
- 非音技には適用されない。
- タイプ変換はノーマル → みずのみ（元々みずタイプの音技は対象外）。

## 実装方針

- `Event.ON_MODIFY_MOVE_TYPE`（タイプ変換）と `Event.ON_CALC_POWER_MODIFIER`（威力1.2倍）の 2 ハンドラ構成。
- `ctx.move.has_label("sound")` で音技判定。
- **ノーマルスキン・フェアリースキン**と同一パターン（対象ラベル・変換先タイプの違いのみ）。

## 変更候補

- `src/jpoke/data/ability.py` — `うるおいボイス` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `うるおいボイス_modify_move_type`、`うるおいボイス_modify_power` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 音技（ラスターカノン等）がみずタイプになる。
2. みずタイプ変換後に威力が 1.2 倍になる。
3. 非音技には適用されない。

def うるおいボイス_modify_move_type(battle, ctx, value):
    if ctx.move.has_label("sound") and value == "ノーマル":
        return HandlerReturn(value="みず")
    return HandlerReturn(value=value)

def うるおいボイス_modify_power(battle, ctx, value):
    if ctx.move.has_label("sound") and ctx.move.type == "みず":
        value = apply_fixed_modifier(value, 4915)  # 1.2倍
    return HandlerReturn(value=value)
