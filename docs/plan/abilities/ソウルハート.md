# 特性実装計画: ソウルハート

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ソウルハート.md`

## 実装ゴール

- 場のポケモンがひんしになるたびに、とくこうが1段階上がる。
- 技によるひんし（ON_MOVE_KO）で発動する。
- C が最大のときは発動しない。

## 実装方針

- `Event.ON_MOVE_KO`、`subject_spec="source:self"` で実装（自分が攻撃者の場合）。
- **くろのいななき**（倒したらC↑1）と同じ `_boost_on_move_ko` パターンを参照。
- 仕様では「誰がひんしになっても発動」だが、`ON_MOVE_KO` は攻撃者の subject に対して発火するため、まず自分が倒した場合を実装する。倒された側への発動は別途 `ON_HP_CHANGED` 等で拡張を検討。

## 変更候補

- `src/jpoke/data/ability.py` — `ソウルハート` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `ソウルハート_on_ko` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 自分が技で相手を倒すと C が 1 段階上がる。
2. C が最大のときは発動しない。

def ソウルハート_on_ko(battle, ctx, value):
    mon = ctx.source
    if not battle.is_active(mon):
        return HandlerReturn(value=value)
    if battle.modify_stats(mon, {"C": +1}, source=mon):
        announce_ability_triggered(battle, ctx, value, mon=mon)
