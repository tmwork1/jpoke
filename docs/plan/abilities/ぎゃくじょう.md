# 特性実装計画: ぎゃくじょう

更新日: 2026-07-11

## 仕様参照

- `docs/spec/abilities/ぎゃくじょう.md`

## 実装ゴール

- 相手の攻撃技を受けて、自分のHPが最大HPの1/2超から1/2以下に減らされたとき、とくこうが1段階上がる。
- HP が 50% 超から 50% 以下へ跨いだ瞬間に発動する（一度跨いだら再発動しない。既にとくこうが最大なら不発）。
- ひんしになった場合は発動しない（`ON_DAMAGE_HIT` は KO 後にも発火するため、`mon.alive` を明示的に確認する）。
- 第七世代からSVまでのちからずく無効化は、Championsでは撤廃されている（いかりのこうらと同様）。本実装は
  ちからずくの効果発動有無を判定していないため、対応は不要（Champions仕様に合致）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=100（デフォルト値、明示不要） で実装。
- ハンドラ先頭で `mon.alive` を確認し、瀕死のときは発動しない。
- `_crossed_half_hp(hp_before, hp_after, mon.max_hp)` で閾値跨ぎを確認（**いかりのこうら**と同ヘルパー）。
- `value` は `ON_DAMAGE_HIT` 時の damage 量（正値）。
- 連続攻撃技対応: `ctx.hit_index == 1` のときの HP（`mon.hp + value`）を `ctx._berserk_hp_before`
  に保持し、最終ヒット（`ctx.hit_index == ctx.hit_count` または `ctx.attacker.fainted`）でのみ、
  その保持した「1発目を受ける前のHP」と現在HPで `_crossed_half_hp` を判定する
  （`いかりのこうら`と同じ idiom。`handlers/ability.py` の
  `いかりのこうら_boost_on_half_hp` を参照）。

## テスト観点

1. HP 50% 超から 50% 以下へ跨いだとき、とくこうが1段階上がる。
2. もともと HP が半分以下の状態で攻撃を受けても再発動しない。
3. すでに とくこう ランクが最大のときは発動しない。
4. ひんしになった場合は発動しない。
5. 連続攻撃技で途中のヒットでHPが半分を下回っても、最終ヒット終了後にまとめて判定される
   （1発目を受ける前のHPを基準に判定する）。

## 疑似コード

```python
def ぎゃくじょう_boost_spa_on_half_hp(battle, ctx, value):
    mon = ctx.defender
    if not mon.alive:
        return HandlerReturn(value=value)

    if ctx.hit_index == 1:
        ctx._berserk_hp_before = mon.hp + value

    is_last_hit = ctx.hit_index == ctx.hit_count or ctx.attacker.fainted
    if not is_last_hit:
        return HandlerReturn(value=value)

    hp_before = ctx._berserk_hp_before
    hp_after = mon.hp
    if (
        _crossed_half_hp(hp_before, hp_after, mon.max_hp)
        and battle.modify_stats(mon, {"spa": +1}, source=ctx.attacker, reason="ぎゃくじょう")
    ):
        announce_ability_triggered(battle, mon)
    return HandlerReturn(value=value)
```
