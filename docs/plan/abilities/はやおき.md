## 実装ゴール（完了定義）

- ねむり状態のはやおき所持者は、ターンごとにねむりカウントを 2 ずつ消費する。
- バッグ使用・フリーフォールなどカウント消費が行われないターンでは効果が発動しない。
- 既存テストを壊さない。

## 実装対象イベント

独立イベントは使用しない。`handlers/ailment.py` の `ねむり_check_action` に直接追記する。

## 実装ステップ

1. 仕様書 `docs/spec/abilities/はやおき.md` でカウント消費ルールを確認する。
2. `src/jpoke/handlers/ailment.py` の `ねむり_check_action` を修正する。
   - `battle.ailment_manager.tick(mon)` の直後に下記を追加:
     ```python
     if mon.ability.name == "はやおき" and mon.ability.enabled:
         battle.ailment_manager.tick(mon)
         if not mon.has_ailment("ねむり"):
             battle.ailment_manager.remove(mon)
             return HandlerReturn(value=True)
     ```
3. テストを追加し、全テストが通ることを確認する。

## テスト観点

1. ねむりカウントが 3 のとき、1 ターンで 2 消費して残り 1 になる（通常は残り 2）。
2. ねむりカウントが 2 のとき、1 ターンで起きる（通常は 1 ターン眠ったまま）。
