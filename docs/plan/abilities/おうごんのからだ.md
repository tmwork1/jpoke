# 特性実装計画: おうごんのからだ

更新日: 2026-07-11

## スコープ

- 優先度: H
- 仕様書: `docs/spec/abilities/おうごんのからだ.md`
- 実装状態: 実装済み（既存実装のレビュー・不具合修正）
- 実装方針: `Event.ON_BEFORE_APPLY_MOVE` で、相手から受ける「対象が自分（`target == "foe"`）の
  変化技」を無効化する。ただし「自分含む全員が対象の技」（おちゃかい・ほろびのうた）は
  使用者自身にも効果が及ぶため技全体を無効化できない。これらは通常の全体無効化の対象から
  除外し、各技のハンドラ側（`handlers/move_status.py`）で対象ポケモンごとに免疫判定を行う。

## 実装ゴール（完了定義）

- 相手から受ける対象が自分の変化技（例: なきごえ）を無効化する。
- 自分が対象の変化技（例: つるぎのまい）・場が対象の変化技（例: にほんばれ）は無効化しない。
- 攻撃技・攻撃技の追加効果は無効化しない。
- かたやぶり・きんしのちからの効果が発動した変化技は防げない（`mold_breaker_ignorable`
  フラグにより中央のイベント基盤が処理するため、ハンドラ側で個別チェックは行わない）。
- 自分含む全員が対象の技（おちゃかい・ほろびのうた）を相手から受けたとき、自分への効果
  （きのみの強制消費・ほろびのうた状態の付与）のみを防ぐ。使用者自身への効果は防がない。
- おうごんのからだ所持ポケモン自身がこれらの技を使用した場合、自身の特性で自身の効果が
  防がれることはない。
- 既存テストを壊さない。

## 実装対象イベント

- `Event.ON_BEFORE_APPLY_MOVE`（priority=100, `docs/spec/turn.md` 表のデフォルト値） —
  対象が自分の変化技を技全体として無効化する。

## 変更対象（ファイル/関数単位）

- `src/jpoke/data/ability.py` — `おうごんのからだ` に `mold_breaker_ignorable` フラグと
  `Event.ON_BEFORE_APPLY_MOVE` ハンドラを登録済み（変更なし）。
- `src/jpoke/handlers/ability.py`
  - `おうごんのからだ_block_status_move` — 対象が自分（`target == "foe"`）の変化技を無効化する。
    「自分含む全員が対象の技」（`_OUGON_NO_KARADA_EXCLUDED_MOVES = {"ほろびのうた"}`）は
    無効化対象から除外する。
    - `おちゃかい` は `target="self"` のためもとから対象外。
- `src/jpoke/handlers/move_status.py`
  - `_blocked_by_ougon_no_karada(battle, mon)` — 全員対象の技の対象ポケモンごとに
    おうごんのからだの免疫を判定する共通ヘルパー。
  - `おちゃかい_force_consume_berries` — 使用者以外できのみが強制消費される直前に
    このヘルパーで免疫判定を行う。
  - `ほろびのうた_apply` — 使用者以外へのほろびのうた状態付与前にこのヘルパーで免疫判定を行う。
- `tests/abilities/test_ability_a.py` — テストを追加・修正する。

## 実装ステップ

1. 仕様書 `docs/spec/abilities/おうごんのからだ.md` を確認する。
2. `おうごんのからだ_block_status_move` の除外リストに `ほろびのうた` を追加する
   （すでに `おちゃかい` は `target="self"` のため対象外）。
3. `handlers/move_status.py` に共通ヘルパー `_blocked_by_ougon_no_karada` を実装する。
4. `おちゃかい_force_consume_berries` と `ほろびのうた_apply` にヘルパーを組み込み、
   使用者以外のおうごんのからだ所持ポケモンへの効果を防ぐ。
5. `tests/abilities/test_ability_a.py` にテストを追加し、全テストが通ることを確認する。

## テスト観点

1. 相手の変化技（対象が自分）を無効化する（既存）。
2. 自分が対象の変化技は無効化しない（既存）。
3. 場が対象の変化技は無効化しない（既存）。
4. 攻撃技は無効化しない（既存）。
5. かたやぶりで無効化される（既存）。
6. ほろびのうたを相手から受けたとき、自分への状態付与のみ防ぎ、使用者自身には付与される。
7. おちゃかいを相手から受けたとき、自分のきのみの強制消費のみ防ぎ、使用者自身のきのみは
   通常通り消費される。
8. おうごんのからだ所持ポケモン自身がほろびのうた・おちゃかいを使用した場合、自身の効果は
   防がれない。

## 対象外・既知の制約

- `ぼうおん` の音技（ほろびのうた等）無効化は `Event.ON_BEFORE_APPLY_MOVE` で技全体を
  無効化する既存実装のままであり、使用者自身への効果まで防いでしまう既知の制約が残る
  （本計画のスコープ外。おうごんのからだ側は使用者自身への効果を防がないよう対応済み）。
- 「味方全員が対象の技」（いのちのしずく・とおぼえ）「味方1体が対象の技」（アロマミスト・
  てだすけ）は、本プロジェクトの対象範囲（シングルバトル）では相手側に「味方」が存在しない
  ため実質的に到達しない組み合わせであり、個別対応は行わない。
