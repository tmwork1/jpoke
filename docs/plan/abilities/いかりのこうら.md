# 特性実装計画: いかりのこうら

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/いかりのこうら.md`

## 実装ゴール

- HPが半分以下になったとき、こうげき・とくこう・すばやさが1段階上がり、ぼうぎょ・とくぼうが1段階下がる。
- HP が 50% 超から 50% 以下へ跨いだ瞬間に発動する（一度跨いだら再発動しない）。
- ひんしになっても発動する（ON_DAMAGE_HIT は KO 後にも発火）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=100 で実装。
- `_crossed_half_hp(hp_before, hp_after, mon.max_hp)` で閾値跨ぎを確認（**ぎゃくじょう**と同ヘルパー）。
- B↓・D↓を先に適用し、A・C・S↑ が成立したときにアナウンスを出す。
- `value` は `ON_DAMAGE_HIT` 時の damage 量（正値）。`hp_before = mon.hp + value` で算出。

## 変更候補

- `src/jpoke/data/ability.py` — `いかりのこうら` に handlers を登録する

## テスト観点

1. HP 50% 超から 50% 以下へ跨いだとき、A・C・S↑1、B・D↓1 が発動する。
2. もともと HP が半分以下の状態で攻撃を受けても再発動しない。
3. ひんしになっても発動する。

def いかりのこうら_on_damage(battle, ctx, value):
    mon = ctx.defender
    hp_after = mon.hp
    hp_before = hp_after + value
    if not _crossed_half_hp(hp_before, hp_after, mon.max_hp):
        return HandlerReturn(value=value)
    battle.modify_stats(mon, {"B": -1, "D": -1}, source=ctx.attacker)
    if battle.modify_stats(mon, {"A": +1, "C": +1, "S": +1}, source=ctx.attacker):
        announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
