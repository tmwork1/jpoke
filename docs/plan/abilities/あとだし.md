# 特性実装計画: あとだし

更新日: 2026-05-03

## 仕様参照

- `spec/ability/あとだし.md`

## 実装ゴール

- 同一優先度の行動の中で、あとだし所持ポケモンが最後に行動する。
- 他のポケモンより優先度が高い技を使用した場合は通常どおり先攻。
- せんせいのツメ/イバンのみ発動時は後攻効果が消える。
- トリックルーム状態でも後攻になる。

## 実装方針

- 技の優先度そのものは変更せず、行動速度（action speed）の計算で後攻補正を適用する。
- `DomainEvent.ON_CALC_SPEED` で「同優先度帯では最後に行動する」ための後攻ティア補正を付与する。
- せんせいのツメ/イバンのみの割り込みが発生した場合は後攻補正を無効化する。

## テスト観点

1. 通常技使用時、あとだし所持者が同優先度の相手より後に行動する。
2. 相手より優先度が高い技を使用した場合は先攻になる。
3. トリックルーム状態でも後攻になる（素早さ逆転の影響を受けない）。
4. イバンのみ発動時は後攻効果が無効化される。
5. あとだし同士では素早さが高い方が先攻になる（第五世代以降）。

## 保留・論点

- 行動順ティアの実装方式（行動速度への後攻ティア補正 vs 専用ティアフラグ）を設計時に決定する。
