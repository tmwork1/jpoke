# 特性実装計画: フラワーベール

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/フラワーベール.md`

## 実装ゴール

- 自分がくさタイプのとき、状態異常にならない。
- 自分がくさタイプのとき、敵から能力ランクを下げられない。
- くさタイプでないポケモンには効果なし（シングルでは自分のみ）。
- かたやぶり系には無効。

## 実装方針

- `Event.ON_BEFORE_APPLY_AILMENT`（状態異常ブロック）と `Event.ON_BEFORE_MODIFY_STAT`（能力低下ブロック）の 2 ハンドラ構成。
- `"くさ" in mon.types` でくさタイプを確認。
- `flags=["mold_breaker_ignorable"]` は data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `フラワーベール` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `フラワーベール_prevent_ailment`、`フラワーベール_prevent_stat_drop` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. くさタイプへの状態異常 → 無効。
2. くさタイプへの能力低下 → 無効。
3. くさタイプでないポケモンには保護なし。
4. かたやぶり系では無効。
