# 特性実装計画: フェアリースキン

更新日: 2026-05-03

## 仕様参照

- `spec/ability/フェアリースキン.md`

## 実装ゴール

- ノーマルタイプの技をフェアリータイプに変換し、威力を4915/4096倍にする。
- タイプ変換後はフェアリータイプ一致補正が適用される。

## 実装方針

- `ON_MODIFY_MOVE_TYPE` でノーマルタイプの技をフェアリータイプに変換し、威力倍率に4915/4096を適用する。
- スカイスキン・フリーズスキン・ドラゴンスキンと共通の `_skin_modify_move_type` / `_skin_boost_power` を使用する（変換先タイプが異なるだけ）。
- わるあがきはタイプなし判定のため、わるあがき実装側で除外し、特性側で個別分岐は持たない。
- ウェザーボール/さばきのつぶて/だいちのはどう/マルチアタック/めざめるダンスは、使用時に結果としてノーマルタイプになっていてもスキン系特性の対象外とする（技名で判定する。type値の一致判定だけでは、天候・地形・所持アイテムなしなどでたまたまノーマルタイプのままだった場合に誤って変換してしまうため不十分）。
- テラバースト/テラクラスターはテラスタル中のみ対象外（テラスタルしていなければ通常のノーマル技として対象になる）。これも技名＋`ctx.attacker.is_terastallized`で判定する。

## テスト観点

1. ノーマル技をフェアリータイプに変換してダメージ計算する。
2. 威力が4915/4096倍になる。
3. フェアリータイプ一致補正が適用される。
4. わるあがきはわるあがき実装側の判定で変換対象外になる。
5. ウェザーボール/さばきのつぶて/だいちのはどうは、結果がノーマルタイプのままでも変換対象外になる。
6. めざめるダンスは使用者がノーマルタイプで技タイプがノーマルのままでも変換対象外になる。
7. テラバーストはテラスタルしていなければ対象になり、テラスタル中は対象外になる。
