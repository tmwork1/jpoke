# 特性実装計画: すなはき

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/すなはき.md`

## 実装ゴール

- 攻撃を受けたとき、天候をすなあらし（5ターン）にする。
- すでにすなあらしの場合は発動しない。
- おおひでり等の強天候中は上書きされない。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"` で実装。
- `battle.weather.apply("すなあらし", 5, source=mon)` の成否でアナウンスを制御。
- **すなおこし**（ON_SWITCH_IN）と同様の天候適用だが、被弾トリガー版。

## 変更候補

- `src/jpoke/data/ability.py` — `すなはき` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `すなはき_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 攻撃を受けるとすなあらしが発動する（5ターン）。
2. すでにすなあらし中は変化なし。
3. おおひでり等の強天候中は上書きされない。
