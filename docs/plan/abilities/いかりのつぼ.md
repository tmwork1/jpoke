# 特性実装計画: いかりのつぼ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/いかりのつぼ.md`

## 実装ゴール

- 攻撃を急所に受けたとき、こうげきが最大ランク（+6）まで上がる。
- 急所に当たっていない場合は発動しない。
- A がすでに最大のときは発動しない（ただしアナウンスなし）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=20 で実装。
- priority は `turn.md` ON_DAMAGE priority 20「いかりのつぼ: 急所に当たった」に従う。
- 急所確認は `battle.move_executor.critical` を参照する。
  `AttackContext.critical` フィールドはダメージ計算専用に `damage.py` 内で生成される別インスタンスにのみ
  設定され、`move_executor._execute_hit` が保持し `ON_DAMAGE_HIT` に渡すメインの `ctx` には反映されない
  （常に `False` のまま）ため、`ctx.critical` では急所判定を取得できない。
- A を +6 にするには `6 - mon.rank["atk"]` の差分を計算して渡す。

## テスト観点

1. 急所被弾で A が最大ランク（+6）になる。
2. 急所でない被弾では発動しない。
3. A がすでに +6 のときは急所被弾しても発動しない（変化なし）。

def いかりのつぼ_max_atk_on_crit(battle, ctx, value):
    if not battle.move_executor.critical:
        return HandlerReturn(value=value)
    mon = ctx.defender
    diff = 6 - mon.rank["atk"]
    if diff > 0 and battle.modify_stats(mon, {"atk": diff}, source=ctx.attacker):
