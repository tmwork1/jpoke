# 特性実装計画: いかりのつぼ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/いかりのつぼ.md`

## 実装ゴール

- 攻撃を急所に受けたとき、こうげきが最大ランク（+6）まで上がる。
- 急所に当たっていない場合は発動しない。
- A がすでに最大のときは発動しない（ただしアナウンスなし）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=20 で実装。
- priority は `turn.md` ON_DAMAGE priority 20「いかりのつぼ: 急所に当たった」に従う。
- `ctx.critical` で急所確認。A を +6 にするには `6 - mon.rank["A"]` の差分を計算して渡す。

## 変更候補

- `src/jpoke/data/ability.py` — `いかりのつぼ` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `いかりのつぼ_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 急所被弾で A が最大ランク（+6）になる。
2. 急所でない被弾では発動しない。
3. A がすでに +6 のときは発動しない。

## 疑似コード

```python
def いかりのつぼ_on_damage(battle, ctx, value):
    if not ctx.critical:
        return HandlerReturn(value=value)
    mon = ctx.defender
    delta = 6 - mon.rank["A"]
    if delta > 0 and battle.modify_stats(mon, {"A": delta}, source=ctx.attacker):
        announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
```
