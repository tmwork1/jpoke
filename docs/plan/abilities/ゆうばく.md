# 特性実装計画: ゆうばく

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ゆうばく.md`

## 実装ゴール

- 直接攻撃でひんしになったとき、攻撃者に相手の最大HPの1/4ダメージを与える（最低1）。
- 直接攻撃でない技でのひんしでは発動しない。
- 攻撃者がじばく等ですでにひんしのときは発動しない。
- マジックガード持ちの攻撃者にはダメージを与えられない（modify_hp 側で処理）。

## 実装方針

- `Event.ON_MOVE_KO`、`subject_spec="defender:self"` で実装。
- **とびだすなかみ**（被弾直前HPをダメージとして返す）と同じ `ON_MOVE_KO` イベントを使用。
- 接触判定後、`max(1, ctx.attacker.max_hp // 4)` を計算し `battle.modify_hp(attacker, -damage, ...)` を呼ぶ。

## テスト観点

1. 直接攻撃でKOされると攻撃者に最大HP1/4ダメージが入る。
2. 非接触技でのKOでは発動しない。
3. 攻撃者がじばくでひんしなら発動しない。
4. マジックガード持ちの攻撃者にはダメージが入らない。

## 疑似コード

```python
def ゆうばく_on_ko(battle, ctx, value):
    if not battle.query.is_contact(ctx):
        return HandlerReturn(value=value)
    attacker = ctx.attacker
    if attacker is None or attacker.fainted:
        return HandlerReturn(value=value)
    damage = max(1, attacker.max_hp // 4)
    battle.modify_hp(attacker, -damage, reason="ability")
    announce_ability_triggered(battle, ctx, value, mon=ctx.defender)
    return HandlerReturn(value=value)
```
