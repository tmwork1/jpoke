# 特性実装計画: とうそうしん

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/とうそうしん.md`

## 実装ゴール

- 相手と同性のとき、こうげき・とくこうが 1.25 倍（5120/4096）になる。
- 相手と異性のとき、こうげき・とくこうが 0.75 倍（3072/4096）になる。
- 性別不明（双方どちらか）のときは補正なし。

## 実装方針

- `Event.ON_CALC_ATK_MODIFIER`、`subject_spec="attacker:self"` で実装。
- `mon.gender` の値（`"♂"`, `"♀"`, `""` 等）を確認して実装。性別不明は `""` を想定。
- **はりきり**（物理技ATK1.5倍）と同一の `ON_CALC_ATK_MODIFIER` イベントを使用。

## 変更候補

- `src/jpoke/data/ability.py` — `とうそうしん` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `とうそうしん_modify_atk` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 同性対決 → こうげき/とくこうが 1.25 倍。
2. 異性対決 → こうげき/とくこうが 0.75 倍。
3. 性別不明（片方または双方）→ 補正なし。

def とうそうしん_modify_atk(battle, ctx, value):
    a_gender = ctx.attacker.gender
    d_gender = ctx.defender.gender if ctx.defender else ""
    if not a_gender or not d_gender:
        return HandlerReturn(value=value)
    if a_gender == d_gender:
        value = apply_fixed_modifier(value, 5120)
    else:
