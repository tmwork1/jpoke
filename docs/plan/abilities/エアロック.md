## 実装ゴール

- 場の天候自体は維持しつつ、天候効果だけを無効化する。
- 通常天候と強天候の両方に対して同じ抑制方針を適用する。

## 実装方針

- 共通ヘルパー `is_weather_effect_active` 相当を導入し、全天候依存処理をそこへ寄せる。
- 素早さ補正、威力補正、命中補正、回復補正、定期ダメージ補正を統一して抑制する。
- ノーてんきと同一実装を許容し、差分はデータ定義側に閉じる。

## テスト観点

1. 天候状態は残る。
2. すいすい / ようりょくそ 等の天候依存特性が止まる。
3. かみなり / ぼうふう / ふぶき など命中補正が止まる。
4. 強天候と同居時の挙動が仕様どおりになる。

## 保留・論点

- 強天候の「対応タイプ攻撃技失敗」を抑制対象に含めるかを先に明文化する。
