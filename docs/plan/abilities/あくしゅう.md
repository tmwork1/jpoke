# 特性実装計画: あくしゅう

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/あくしゅう.md`

## 実装ゴール

- 攻撃技を命中させたとき、10%の確率で相手をひるませる。
- みがわりに当たった場合は発動しない（ON_DAMAGE_HIT は damage > 0 のときのみ発火）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="attacker:self"` で実装。
- 10%の乱数判定後、`battle.volatile_manager.apply(ctx.defender, "ひるみ", ...)` で付与。
- 既存の **ほのおのからだ・せいでんき** と同一パターン。

## 変更候補

- `src/jpoke/data/ability.py` — `あくしゅう` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `あくしゅう_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 攻撃技が命中したとき 10% でひるみが付与される。
2. ひるみ耐性（せいしんりょく・どんかん等）を持つ相手には付与されない。

## 疑似コード

```python
def あくしゅう_on_damage(battle, ctx, value):
    if battle.random.random() < 0.1:
        battle.volatile_manager.apply(ctx.defender, "ひるみ", source=ctx.attacker, ctx=ctx)
    return HandlerReturn(value=value)
```
