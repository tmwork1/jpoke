# 特性実装計画: ヘヴィメタル

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ヘヴィメタル.md`

## 実装ゴール

- 自分の重さが2倍になる（けたぐり・くさむすびのダメージ増加等）。

## 実装方針

**モデルレベルで実装済み。**

`src/jpoke/model/pokemon.py` の `Pokemon.weight` プロパティがすでに対応している：

```python
@property
def weight(self) -> float:
    w = self.data.weight
    match self.ability.name:
        case 'ヘヴィメタル':
            w = w * 2
    ...
```

`data/ability.py` の `flags=["mold_breaker_ignorable"]` も設定済み。

**追加の実装作業は不要。テストの追加のみ行う。**

## 変更候補

- `tests/test_ability.py` — テストを追加する

## テスト観点

1. `pokemon.weight` が基本値の 2 倍になる。
2. かたやぶり系技受け時は ヘヴィメタル が無視され基本値の重さになる。

## 検証コマンド

```powershell
python -m pytest tests/test_ability.py -k "ヘヴィメタル" -v
python -m pytest tests/ -v
```
