# 特性実装計画: くだけるよろい

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/くだけるよろい.md`

## 実装ゴール

- 物理技を受けると、ぼうぎょが1段階下がりすばやさが2段階上がる。
- 特殊技では発動しない。
- B↓は必ず発生し、S↑2 が成立したときに特性発動アナウンスが出る。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=20 で実装。
- priority は `turn.md` ON_DAMAGE 「くだけるよろい等: 物理技を受けた」の priority 20 に従う。
- `battle.resolve_move_category(ctx.attacker, ctx.move) == "物理"` でフィルタ。
- B↓ → S↑2 の順に `battle.modify_stats` を呼ぶ。

## 変更候補

- `src/jpoke/data/ability.py` — `くだけるよろい` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `くだけるよろい_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 物理技を受けると B↓1、S↑2 が発生する。
2. 特殊技では発動しない。
3. S が最大でも B↓1 は発生する（アナウンスなし）。

## 疑似コード

```python
def くだけるよろい_on_damage(battle, ctx, value):
    if battle.resolve_move_category(ctx.attacker, ctx.move) != "物理":
        return HandlerReturn(value=value)
    mon = ctx.defender
    battle.modify_stats(mon, {"B": -1}, source=ctx.attacker)
    if battle.modify_stats(mon, {"S": +2}, source=ctx.attacker):
        announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
```
