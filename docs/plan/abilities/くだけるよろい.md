# 特性実装計画: くだけるよろい

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/くだけるよろい.md`

## 実装ゴール

- 物理技を受けると、ぼうぎょが1段階下がりすばやさが2段階上がる。
- 特殊技では発動しない。
- B↓は必ず発生し、S↑2 が成立したときに特性発動アナウンスが出る。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=20 で実装。
- priority は `turn.md` ON_DAMAGE 「くだけるよろい等: 物理技を受けた」の priority 20 に従う。
- `battle.resolve_move_category(ctx.attacker, ctx.move) == "物理"` でフィルタ。
- B↓ → S↑2 の順に `battle.modify_stats` を呼ぶ。

## テスト観点

1. 物理技を受けると B↓1、S↑2 が発生する。
2. 特殊技では発動しない。
3. S がすでに最大でも B↓1 は発生する（アナウンスは B↓ ではなく S↑ の成立時のみ。
   `いかりのこうら` と同じ idiom）。
4. B がすでに最低でも S↑2 は発生する。
5. 物理技の連続攻撃技を受けた場合、1発ごとに B↓1・S↑2 が発動する（`ON_DAMAGE_HIT` が
   ヒットごとに発火するため、いかりのこうらのような最終ヒットまとめ判定は不要）。

def くだけるよろい_drop_B_boost_S(battle, ctx, value):
    if ctx.move.category != "physical":
        return HandlerReturn(value=value)
    mon = ctx.defender
    battle.modify_stats(mon, {"def": -1}, source=ctx.attacker)
    if battle.modify_stats(mon, {"spe": +2}, source=ctx.attacker):
        _announce_ability_triggered(battle, mon)
