# 特性実装計画: はとむね

更新日: 2026-07-12

## 仕様参照

- `docs/spec/abilities/はとむね.md`

## 実装ゴール

- 相手からぼうぎょランクを下げられなくなる。
- 自己効果（からをやぶる・ばかぢから・じこあんじ等）によるぼうぎょランク低下は防げない。
- かたやぶり系の効果を持つ技・特性には無視されてぼうぎょランクを下げられる。

## 実装方針

- クリアボディ・かいりきバサミ・するどいめと同一パターン。
- `ON_BEFORE_MODIFY_STAT`（`subject_spec="target:self"`）でランク変化辞書を受け取り、
  共通ヘルパー `_block_stat_drop_by_foe(value, ctx, stat="def")` を呼び出して、
  `source != target`（相手由来）かつ低下方向（`v < 0`）の「ぼうぎょ」項目だけを取り除く。
- かたやぶり判定は `AbilityData.flags` に `mold_breaker_ignorable` を付与し、
  共通処理に委ねる（個別実装しない）。

## 変更候補

- `src/jpoke/data/ability.py` — `はとむね` の `AbilityData` に `ON_BEFORE_MODIFY_STAT` ハンドラを登録
- `src/jpoke/handlers/ability.py` — `はとむね_block_B_drop`
- `tests/abilities/test_ability__group.py` — `test_能力低下防止特性_param` にパラメータ追加

## テスト観点

1. 相手の技・特性によるぼうぎょランク低下（くすぐる等）を防ぐ。
2. 自己効果（からをやぶる等）によるぼうぎょランク低下は防げない。
3. くろいきり/クリアスモッグによるリセットは防げない。
