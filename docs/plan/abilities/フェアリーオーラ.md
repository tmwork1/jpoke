# 特性実装計画: フェアリーオーラ

更新日: 2026-07-12

## スコープ

- 優先度: H（ゼルネアス・メガフラエッテ専用特性、ダークオーラと対の実装）
- 仕様書: `docs/spec/abilities/フェアリーオーラ.md`
- 実装状態: 部分実装済み（`ON_CALC_POWER_MODIFIER` のみ登録済み。本レビューで
  `mold_breaker_ignorable` フラグの誤り修正・登場時アナウンス追加を行う）
- 実装方針: `ON_CALC_POWER_MODIFIER` で場にいる自分・相手のフェアリー技威力を
  5448/4096倍にする。オーラブレイクによる 0.75 倍への反転は
  `フェアリーオーラ_boost_power` 側で対応済み（オーラブレイク所持者の有無を見て
  補正率を切り替える）。`ON_SWITCH_IN`・`ON_ABILITY_ENABLED` で登場時アナウンス
  を出す。かたやぶりでは無視されない（`mold_breaker_ignorable` フラグを付けない）。

## 実装ゴール（完了定義）

- 場にいるポケモン（攻撃側・防御側どちらか）が使うフェアリー技の威力が5448/4096倍になる。
- 複数のフェアリーオーラ所持者が場にいても補正は累積しない（シングル対象のため実質1体分のみ）。
- フェアリーオーラを持つポケモンが場に出ると「<ポケモン>は フェアリーオーラを 放っている!」の
  ログが記録され、特性が開示される。
- かたやぶりを持つ相手の技であっても、フェアリーオーラの威力補正は無視されない（現行世代仕様）。
- オーラブレイクが場にいる場合は補正率が0.75倍（3072/4096倍）に反転する。

## 実装対象イベント

- `Event.ON_SWITCH_IN` — 登場時にアナウンスを出す（priority 100, `docs/spec/turn.md` に記載あり）
- `Event.ON_ABILITY_ENABLED` — 特性が有効化されたときにアナウンスを出す
  （`turn.md` 未掲載のイベント。オーラブレイク・ダークオーラ等の既存実装に合わせ
  デフォルト priority=100 とする）
- `Event.ON_CALC_POWER_MODIFIER` — フェアリー技威力を5448/4096倍にする（`turn.md` 未掲載。
  ダメージ計算内部処理のため、ダークオーラ等の既存の同種ハンドラに合わせ
  デフォルト priority=100 とする）

## 変更対象（ファイル/関数単位）

- `src/jpoke/handlers/ability.py` — `フェアリーオーラ_boost_power`（既存、変更なし）。
  アナウンスは既存の汎用関数 `announce_ability_triggered` を再利用する（新規関数追加なし）
- `src/jpoke/data/ability.py` — `フェアリーオーラ` の `flags` から `mold_breaker_ignorable`
  を削除し、`ON_SWITCH_IN` / `ON_ABILITY_ENABLED` に `announce_ability_triggered` を
  `subject_spec="source:self"` で登録する
- `tests/abilities/test_ability_ha.py` — テストを追加する

## 実装ステップ

1. 仕様書で補正率（5448/4096倍）・オーラブレイクによる反転・かたやぶり非対象であることを確認する。
2. `フェアリーオーラ_boost_power`（既存実装）が `ctx.move.type == "フェアリー"` のときのみ補正し、
   オーラブレイク所持者の有無で補正率を切り替えていることを確認する。
3. `data/ability.py` の `フェアリーオーラ` エントリから `mold_breaker_ignorable` フラグを削除する
   （オーラブレイクとは異なり、フェアリーオーラ自体はかたやぶりに無視されないため。
   `docs/spec/abilities/オーラブレイク.md` の「第八世代ではフェアリーオーラ/ダークオーラは
   かたやぶりで無視されない」の記述が根拠）。
4. `data/ability.py` の `フェアリーオーラ` エントリに `ON_SWITCH_IN` / `ON_ABILITY_ENABLED` の
   `announce_ability_triggered` ハンドラを追加する（`subject_spec="source:self"`）。
5. テストを追加し、全テストが通ることを確認する。

## テスト観点

1. フェアリーオーラ所持ポケモンの自分のフェアリー技威力が5448/4096倍になる。
2. フェアリーオーラ所持ポケモンがいると、相手のフェアリー技威力も5448/4096倍になる。
3. フェアリー技以外の技には効果がない。
4. 場に出たとき特性が開示される（アナウンス）。
5. かたやぶりを持つ相手のフェアリー技でも、フェアリーオーラの補正は無視されない。
6. オーラブレイクが場にいる場合、フェアリー技の威力補正が0.75倍（3072/4096倍）に反転する。
