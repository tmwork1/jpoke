# 特性実装計画: こんじょう

更新日: 2026-07-11

## 仕様参照

- `docs/spec/abilities/こんじょう.md`

## 実装ゴール

- 状態異常（どく・もうどく・やけど・まひ・ねむり・こおり）のとき、こうげきが1.5倍になる。
  - 物理技（ボディプレスの防御流用・イカサマの相手こうげき流用を含む）にのみ適用する。
  - こんらん・メロメロなどの状態変化のみでは発動しない（`ailment`のみを見る）。
- やけど状態になっても、使用する物理技のダメージが半減しない。
- こんらんの自傷ダメージ（内部技 `"_こんらん"`）には攻撃補正が乗らない
  （Champions仕様＝第五世代以降の仕様に準拠）。

## 実装方針

- こうげき1.5倍: `ON_CALC_ATK_MODIFIER` で `subject_spec="attacker:self"` を使用し、
  `ctx.attacker.ailment.is_active` かつ `ctx.move.category == "physical"` かつ
  `ctx.move.name != "_こんらん"` のとき6144/4096倍を適用する。
  - `ON_CALC_ATK_MODIFIER` はステータス（攻撃対象の`atk`/`def`/相手`atk`の置換後）に対して発火するため、
    ボディプレス・イカサマにも自然に対応できる（`core/damage.py`の`_calc_final_attack`参照）。
  - priorityは`docs/spec/turn.md`に`ON_CALC_ATK_MODIFIER`の記載が無いため、
    既存の同種ハンドラ（`はりきり_modify_atk`, `とうそうしん_modify_atk`, `ちからもち_boost_physical`等）
    に合わせてデフォルト（100）とする。
  - `"_こんらん"`除外は同様のパターンを持つ`こだわりハチマキ_boost_physical`（item.py）を参照する。
- やけど半減無効化: `ON_CALC_BURN_MODIFIER` で `subject_spec="attacker:self"`、
  `priority=200`（`やけど_modifier`のデフォルト優先度100より後に実行し上書きする）で、
  常に`value=4096`を返す。やけどでない・特殊技のときは元々4096なので副作用は無い。
  - priorityの根拠は同一パターンの `からげんき_ignore_burn_modifier`（move_ka.py, priority=200）を参照する。

## 変更候補

- `src/jpoke/handlers/ability.py` — `こんじょう_modify_atk`, `こんじょう_ignore_burn_penalty` を実装する
- `tests/abilities/test_ability_ka.py` — テストを追加する

## テスト観点

1. 行動可能な状態異常（どく・もうどく・やけど）で物理技のこうげきが1.5倍になる。
2. やけど状態でも物理技ダメージが半減しない（`burn_modifier == 4096`）。
3. こんらんの自傷ダメージ（`"_こんらん"`）には攻撃補正が乗らない。
