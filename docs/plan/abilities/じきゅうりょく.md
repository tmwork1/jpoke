# 特性実装計画: じきゅうりょく

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/じきゅうりょく.md`

## 実装ゴール

- 攻撃技のダメージを受けたとき（物理・特殊問わず）、ぼうぎょが1段階上がる。
- 天候・状態異常等のスリップダメージでは発動しない。
- B が最大ランクのときは発動しない。
- 連続攻撃技は1発ヒットするたびに発動する。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=100 で実装。
- `battle.modify_stats(mon, {"B": +1}, source=ctx.attacker)` の成否でアナウンスを制御。
- 既存の **ぎゃくじょう** と同パターン（閾値チェックなし版）。

## 変更候補

- `src/jpoke/data/ability.py` — `じきゅうりょく` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `じきゅうりょく_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 物理技を受けると B が 1 段階上がる。
2. 特殊技を受けても B が 1 段階上がる。
3. B が最大（+6）のときは発動しない。
4. 連続攻撃技: 1発ごとに B が上がる。
