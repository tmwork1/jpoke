# わるいてぐせ 実装計画

## スコープ
- 仕様書: `docs/spec/abilities/わるいてぐせ.md`（作成済み）
- 効果: 直接攻撃でダメージを受けたとき、攻撃側からアイテムを奪う。
  自分がすでにアイテムを持っている場合や、自分がひんしになった場合は発動しない。

## 実装方針

### イベント・priority
- `Event.ON_DAMAGE_HIT`（`docs/spec/turn.md` の `Event.ON_DAMAGE` に対応）
  priority=180（turn.md に明記の「わるいてぐせ」の行と一致）。
  いのちのたまの反動(160)・かいがらのすずの回復(160)・オボンのみ等回復きのみ(170)より後、
  しろいハーブ/だっしゅつパック/ものまねハーブ(190)より先に発動する。
- `subject_spec="defender:self"`（AttackContext。わるいてぐせ保持側が防御側になる）。

### 処理内容
1. `battle.query.is_contact(ctx)` で接触判定する（`is_contact_reaction` ではなく `is_contact`
   を使う。ぼうごパットは接触判定そのものを変えないため、ぼうごパット所持の相手からも発動する。
   `is_contact_reaction` はぼうごパット等で「反応効果」だけを防ぐための判定であり、
   `src/jpoke/core/query.py` の `is_contact_reaction` docstring にも
   「もふもふ/わるいてぐせのようにぼうごパットの対象外となる効果を判定する場合は
   `is_contact()` を使う」と明記されている）。
2. 自分（`ctx.defender`）がひんしでなければ `battle.item_manager.take_item(ctx.attacker, ...)`
   でアイテムを奪う。
   - `take_item` 内部で「対象(攻撃側)がアイテムを持たない」「自分(防御側)が既にアイテムを
     持っている」場合は自動的に失敗する（アイテムマネージャ側の共通処理）。
   - `ignore_sticky_hold=ctx.attacker.fainted` を指定する。攻撃側が反動ダメージ等で
     ひんしになった場合、その攻撃側の特性ねんちゃくは発動せず奪取を阻止できない
     （仕様書「ねんちゃくのポケモンが技の反動などでひんしになったときは、ねんちゃくは
     発動せずに道具を奪える」）。この分岐は `マジシャン_steal_item` と同じパターン。
3. Zクリスタル・メール・ディアルガのだいこんごうだま等の特定ポケモン専用アイテムの保護、
   メガストーン・ゲンシカイキの秘石、パラドックスポケモンのブーストエナジー等は
   `Event.ON_CHECK_ITEM_CHANGE` 側（各アイテムの `data/item.py` ハンドラ）で汎用的に
   ブロックされるため、わるいてぐせ側で個別に判定する必要はない。
4. ちからずくとの相互作用: 本プロジェクトの対象範囲（チャンピオンズ）では、ちからずくは
   `Event.ON_MODIFY_SECONDARY_CHANCE` のみを無効化し `Event.ON_DAMAGE_HIT` には影響しない
   構造のため、わるいてぐせは特性ちからずくの技に対しても常に発動する（仕様書の
   「Championsではこのときもわるいてぐせで道具を奪える」と自然に一致する）。追加実装不要。
5. さまようたましいとの相互作用（仕様書「効果の処理順」に明記の例外）: てつのトゲ等の
   「ダメージを受けたときに発動する特性」は、さまようたましいで手に入れてもその場のダメージ
   では発動しないのが基本だが、わるいてぐせだけは「特性を交換した直後に発動判定がある」
   という例外がある。イベントハンドラの発火リストは `EventManager.emit()` 実行開始時点で
   スナップショットされるため、さまようたましいで交換の途中に新規登録した
   わるいてぐせハンドラは同一 `ON_DAMAGE_HIT` の通常ループには乗らない。そのため
   `さまようたましい_swap_ability_on_contact` 内で `swap_ability` 直後に防御側の新しい特性が
   わるいてぐせであれば `わるいてぐせ_steal_item` を直接呼び出す。

## テスト観点
- 接触技を受けたら相手のアイテムを奪う。
- 非接触技には反応しない（ぼうごパット所持相手にも発動する／えんかく・パンチグローブで
  非接触化された技には発動しない）。
- 自分がすでにアイテムを持っている場合は奪わない。
- 自分がひんしになった場合は発動しない。反動ダメージ等で相手がひんしになった場合は発動する。
- 相手のねんちゃくで通常は阻止されるが、相手がその攻撃の反動等でひんしになった場合は
  ねんちゃくを無視して奪える。
- さまようたましいで自分に渡ってきた直後に発動し、相手のアイテムを奪える。
