# 特性実装計画: はっこう

更新日: 2026-07-12

## 仕様参照

- `docs/spec/abilities/はっこう.md`
- `docs/spec/abilities/するどいめ.md`（第九世代以降ははっこうと同一効果）

## 実装ゴール

- 他のポケモンから命中率（accuracy）のランク低下を受けない。
  - 自分自身の効果による低下（ランク変動を伴う自傷等）までは防がない＝相手由来のみ無効化する。
- ACC↑ は通常通り適用される。
- A↓ 等 ACC 以外の能力低下は通常通り適用される。
- 攻撃時、相手の回避率（evasion）ランクを上昇・低下とも無視する。
- かたやぶり系には無効（`flags=["mold_breaker_ignorable"]`）。

## 実装方針

- **するどいめ**と第九世代以降は完全に同一効果のため、`するどいめ_block_ACC_drop` /
  `するどいめ_ignore_evasion` と対になる実装を行う。
- 命中率低下無効: `Event.ON_BEFORE_MODIFY_STAT`、`subject_spec="target:self"`。
  - 共通ヘルパー `_block_stat_drop_by_foe(value, ctx, "accuracy")` を使い、
    `ctx.source != ctx.target` の場合のみ ACC 低下エントリを除去する（相手由来のみ無効化）。
- 回避率無視: `Event.ON_GET_STAT_RANK`、`subject_spec="attacker:self"`。
  - `value["evasion"] = 0` として相手の回避率ランクを無視する。
- `flags=["mold_breaker_ignorable"]` は data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `はっこう` に ON_BEFORE_MODIFY_STAT / ON_GET_STAT_RANK の
  両ハンドラを登録する
- `src/jpoke/handlers/ability.py` — `はっこう_block_acc_drop`（`_block_stat_drop_by_foe` 使用に修正）
  と `はっこう_ignore_evasion`（新規）を実装する
- `tests/abilities/test_ability_ha.py` — テストを追加する

## テスト観点

1. にらみつける等、相手からの ACC↓ を試みる → 無効になる。
2. ACC↑ は通常通り適用される。
3. A↓ 等他の能力低下は通常通り通る。
4. かたやぶり系では ACC↓ 無効化が効かない（相手が命中率を下げられる）。
5. 攻撃時、相手の回避率ランクが上昇していても無視して命中判定する。
6. 攻撃時、相手の回避率ランクが低下していても命中率を上げない（無視する）。
