# 特性実装計画: はっこう

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/はっこう.md`

## 実装ゴール

- 命中率（ACC）を下げられない。
- ACC↑ は通常通り適用される。
- A↓ 等 ACC 以外の能力低下は通常通り適用される。
- かたやぶり系には無効。

## 実装方針

- `Event.ON_BEFORE_MODIFY_STAT`、`subject_spec="target:self"` で実装。
- **するどいめ**（ACC↓無効 + EVA無視）の ACC↓無効部分のみを抽出して実装。
- `value` dict から `ACC < 0` のエントリを除いて返す。
- `flags=["mold_breaker_ignorable"]` は data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `はっこう` に ON_BEFORE_MODIFY_STAT handler を登録する
- `src/jpoke/handlers/ability.py` — `はっこう_block_acc_drop` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. にらみつける等で ACC↓ を試みる → 無効になる。
2. ACC↑ は通常通り適用される。
3. A↓ 等他の能力低下は通常通り通る。
4. かたやぶり系では無効。
