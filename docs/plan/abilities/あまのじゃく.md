# 実装計画: あまのじゃく

作成日: 2026-07-10

## 効果
自分のランク補正が変動するとき、変動量のプラスマイナスが反転する（`docs/spec/abilities/あまのじゃく.md` 参照）。

## イベント／ハンドラ

| イベント | priority | subject_spec | 関数名 |
|---------|---------|-------------|--------|
| `ON_BEFORE_MODIFY_STAT` | 100（`turn.md` 未掲載。イベント計算内部のため、同種の `シンプル`（ランク変化量を2倍にするハンドラ）に合わせてデフォルト値を採用） | `target:self` | `あまのじゃく_reverse_stat` |

`ON_BEFORE_MODIFY_STAT` は `StatusManager.modify_stats()` が能力ランク変更のたびに発火するイベントで、
`value` は `{stat: 変化量}` の辞書。`subject_spec="target:self"` により、
**あまのじゃくを持つポケモン自身が変化の対象（target）になるときのみ** 発動する
（あまのじゃく所持者が相手の能力を下げる技を使った場合は `target` が相手になるため発動しない）。

## 実装

```python
def あまのじゃく_reverse_stat(battle: Battle, ctx: EventContext, value: dict[str, int]) -> HandlerReturn:
    """あまのじゃく特性: 能力変化量の符号を反転する。"""
    return HandlerReturn(value={stat: -delta for stat, delta in value.items()})
```

## data/ability.py 登録

```python
"あまのじゃく": AbilityData(
    flags={
        "mold_breaker_ignorable"
    },
    handlers={
        Event.ON_BEFORE_MODIFY_STAT: h.AbilityHandler(
            h.あまのじゃく_reverse_stat,
            subject_spec="target:self",
        )
    }
),
```

`mold_breaker_ignorable` フラグにより、かたやぶり等でとくせいを無視する技を受けた場合は
あまのじゃくが無効化され、通常通りランクが変化する（仕様書「特性の仕様」節）。

## 実装済みの補足対応（他ファイル）

以下は本特性に依存する別特性・アイテム・技側の実装で、あまのじゃく単体のハンドラとは別に
それぞれの `subject_spec` / 実装内で「自分の能力変化かどうか」を判定して整合させている。
本計画では変更しないが、参照用に記載する。

- `くろいきり` / `クリアスモッグ` / `じこあんじ` / `パワースワップ` / `ガードスワップ` /
  `ハートスワップ` / `ひっくりかえす` / `バトンタッチ`: `modify_stats()` を経由しない、
  または対象外ロールで発動するため、あまのじゃくの影響を受けない。
- `シャドースチール`: ランクリセット・ランク吸収は別経路（直接リセット）のため反転しない。
- 各種アイテム（アッキのみ・タラプのみ・じゅうでんち・きゅうこん・ひかりごけ・ゆきだま・
  のどスプレー・ビビリだま等）・技（はらだいこ・ほおばる・ちからをすいとる等）は、
  あまのじゃく所持者の `ON_BEFORE_MODIFY_STAT` 経由で自然に反転する。それぞれのテストは
  `tests/items/`, `tests/moves_status/` に存在する。

## 既知の未対応（対象外・低優先度）

- `わたげ`（`handlers/ability.py` の `わたげ_lower_spd_on_hit`）は `battle.modify_stats(attacker, ...)`
  を通常経路で呼ぶため、攻撃者があまのじゃくを持つ場合は反転してしまう。仕様書には
  「かたやぶりの効果がある技を受けて発動したわたげは、あまのじゃくの効果を無視してすばやさを下げる」
  とあるが、これは「かたやぶり技で わたげ 自体の発動可否判定を無視した場合」の非常に限定的な
  edge case であり、既存実装は一般ケース（かたやぶりでない技でわたげが発動する場合）を
  正しく処理している。優先度が低いため本レビューでは対応を見送る。

## テストケース
- あまのじゃく所持者が対象（`target`）となるランク変化は、上昇下降が反転する
  （`tests/abilities/test_ability_a.py::test_あまのじゃく_能力変化の符号反転`）。
- かたやぶりの効果がある技を受けた場合はあまのじゃくが無効化され、通常通り下降する
  （`tests/abilities/test_ability_a.py::test_あまのじゃく_かたやぶりで無効`）。
- あまのじゃく所持者が相手の能力を下げる技（なきごえ等）を使っても、`source` 側であって
  `target` ではないため反転しない
  （`tests/abilities/test_ability_a.py::test_あまのじゃく_相手の能力を下げる技は反転しない`）。
- アイテム・戦闘道具・追加効果技との相互作用は `tests/items/`, `tests/moves_status/`,
  `tests/moves_attack/` に多数のテストが存在する。
