# 特性実装計画: そうだいしょう

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/そうだいしょう.md`

## 実装ゴール

- 場に出たとき、その戦闘でひんしになった味方の数×10%の威力補正を得る（最大 +50%）。
- 威力補正は入場時点で確定し、入場後の変化は反映されない。
- ひんし味方が0体のときはアナウンスなし・補正なし。

## 実装方針

- `Event.ON_SWITCH_IN`（入場時に補正率を `ability.state` に保存）と `Event.ON_CALC_POWER_MODIFIER`（保存値を威力に乗算）の2ハンドラ構成。
- ひんし味方数は `battle.get_player(mon).party` から `p.fainted and p is not mon` でカウント。
- 補正率: `min(count * 0.1, 0.5)` → `apply_fixed_modifier(value, int(4096 * (1 + multiplier)))` で適用。
- `ability.state` に float（0.0〜0.5）を格納。

## 変更候補

- `src/jpoke/data/ability.py` — `そうだいしょう` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `そうだいしょう_on_switch_in`、`そうだいしょう_modify_power` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 味方1体ひんし状態で入場 → 攻撃技の威力が 1.1 倍。
2. 味方5体ひんし状態で入場 → 威力が 1.5 倍（上限 50%）。
3. 入場後に味方がひんしになっても補正は変わらない。
4. ひんし味方なし → アナウンスなし・補正なし。
