# 特性実装計画: びびり

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/びびり.md`

## 実装ゴール

- あく・ゴースト・むしタイプの技を受けると、すばやさが1段階上がる。
- 上記3タイプ以外では発動しない。
- S が最大のときは発動しない。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=100 で実装。
- `ctx.move.type in ("あく", "ゴースト", "むし")` チェック後、`battle.modify_stats(mon, {"S": +1}, ...)` を呼ぶ。
- **せいぎのこころ** と同パターン（対象タイプが複数の点が異なる）。

## 変更候補

- `src/jpoke/data/ability.py` — `びびり` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `びびり_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. あく技受けると S が 1 段階上がる。
2. ゴースト技受けると S が 1 段階上がる。
3. むし技受けると S が 1 段階上がる。
4. ノーマル等他タイプでは発動しない。

## 疑似コード

```python
def びびり_on_damage(battle, ctx, value):
    if ctx.move.type not in ("あく", "ゴースト", "むし"):
        return HandlerReturn(value=value)
    mon = ctx.defender
    if battle.modify_stats(mon, {"S": +1}, source=ctx.attacker):
        announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
```
