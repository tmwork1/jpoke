# 特性実装計画: しんがん

更新日: 2026-06-03

## スコープ

- 優先度: H
- 仕様書: `docs/spec/abilities/しんがん.md`
- 実装状態: 部分実装（flags=[mold_breaker_ignorable] のみ設定済み）
- 実装方針: きもったまのゴースト無効バイパスと、するどいめの命中ランク低下阻止・回避ランク無視を組み合わせる。既存の `きもったま_ghost_immune_bypass`・`するどいめ_block_ACC_drop`・`するどいめ_ignore_evasion` のロジックを再利用する。

## 実装ゴール（完了定義）

- ノーマル・かくとう技がゴーストタイプに等倍で当たる（きもったまと同じ効果）。
- 相手から命中ランクを下げられない（するどいめと同じ効果）。
- 相手の回避ランクを無視して攻撃できる（するどいめと同じ効果）。
- いかく無効の効果はない（きもったまとの差異）。
- 既存テストを壊さない。

## 実装対象イベント

- `Event.ON_CALC_DEF_TYPE_MODIFIER`（または同等のイベント）— ゴースト無効バイパス
- `Event.ON_BEFORE_MODIFY_STAT` — 命中ランク低下を阻止
- `Event.ON_GET_STAT_RANK` — 回避ランクを無視

## 変更対象（ファイル/関数単位）

- `src/jpoke/data/ability.py` — `しんがん` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `しんがん_ghost_immune_bypass` を実装する（するどいめの実装は共有関数として再利用）
- `tests/test_ability.py` — テストを追加する

## 実装ステップ

1. きもったま実装後、`きもったま_ghost_immune_bypass` を共有関数に切り出せるか検討する。
2. するどいめの `するどいめ_block_ACC_drop` と `するどいめ_ignore_evasion` を `しんがん` にも登録できるか確認する（同一関数を再利用）。
3. `src/jpoke/data/ability.py` の `しんがん` に上記 handlers を登録する。
4. テストを追加し、全テストが通ることを確認する。

## テスト観点

1. ゴーストタイプにノーマル技を使うと等倍ダメージが出る。
2. 相手からの命中ランク低下技を受けても命中ランクが下がらない。
3. 相手の回避ランク上昇に関係なく命中する。
4. いかくではこうげきが下がる（きもったまとの差異）。
