# 特性実装計画: かんろなミツ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/かんろなミツ.md`

## 実装ゴール

- バトル中一度だけ、場に出たとき相手の回避率を1段階下げる。
- 2回目以降の入場では発動しない。
- クリアボディ等で EVA↓ 無効の相手には発動しない。

## 実装方針

- `Event.ON_SWITCH_IN`、`subject_spec="source:self"` で実装。
- **ふくつのたて** と同一パターン。相手（`battle.foe(mon)`）の EVA を↓1 する。
- `per_battle_once` フラグは data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `かんろなミツ` に ON_SWITCH_IN handler を登録する
- `src/jpoke/handlers/ability.py` — `かんろなミツ_on_switch_in` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 初回入場で相手の EVA が 1 段階下がる。
2. 2回目以降の入場では発動しない。
3. クリアボディ等の相手では EVA↓ が失敗する。

## 疑似コード

```python
def かんろなミツ_on_switch_in(battle, ctx, value):
    mon = ctx.source
    if mon.ability.state == "used":
        return HandlerReturn(value=value)
    mon.ability.state = "used"
    foe = battle.foe(mon)
    if battle.modify_stats(foe, {"EVA": -1}, source=mon):
        announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
```
