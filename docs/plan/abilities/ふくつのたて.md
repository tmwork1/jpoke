# 特性実装計画: ふくつのたて

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ふくつのたて.md`

## 実装ゴール

- バトル中一度だけ、場に出たときにぼうぎょが1段階上がる。
- 2回目以降の入場では発動しない。

## 実装方針

- `Event.ON_SWITCH_IN`、`subject_spec="source:self"` で実装。
- `ability.state == "used"` で使用済みチェック。初回発動時に `"used"` にセット。
- `per_battle_once` フラグは data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `ふくつのたて` に ON_SWITCH_IN handler を登録する
- `src/jpoke/handlers/ability.py` — `ふくつのたて_on_switch_in` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 初回入場で B が 1 段階上がる。
2. 2回目以降の入場では発動しない。

def ふくつのたて_on_switch_in(battle, ctx, value):
    mon = ctx.source
    if mon.ability.state == "used":
        return HandlerReturn(value=value)
    mon.ability.state = "used"
    if battle.modify_stats(mon, {"B": +1}, source=mon):
        announce_ability_triggered(battle, ctx, value, mon=mon)
