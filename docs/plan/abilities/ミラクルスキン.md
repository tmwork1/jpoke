# 特性実装計画: ミラクルスキン

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ミラクルスキン.md`

## 実装ゴール

- 相手の変化技の命中率を 50% に固定する（命中率がすでに 50% 以下なら変化なし）。
- 攻撃技には適用されない。
- 必中の変化技には適用されない。
- かたやぶり系には無効。

## 実装方針

- `Event.ON_MODIFY_ACCURACY`、`subject_spec="defender:self"` で実装。
- `ctx.move.category == "変化"` と `ctx.move.accuracy is not None`（必中除外）を確認。
- 命中率の内部スケールを確認し、50% に相当する値（`2048` if 4096 スケール）を上限とする。
- `flags=["mold_breaker_ignorable"]` は data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `ミラクルスキン` に ON_MODIFY_ACCURACY handler を登録する
- `src/jpoke/handlers/ability.py` — `ミラクルスキン_reduce_accuracy` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 変化技（どくどく等）の命中率が 50% になる。
2. 攻撃技には適用されない。
3. 必中の変化技には適用されない。
4. かたやぶり系では無効。

## 疑似コード

```python
def ミラクルスキン_reduce_accuracy(battle, ctx, value):
    if ctx.move.category != "変化":
        return HandlerReturn(value=value)
    if ctx.move.accuracy is None:  # 必中技
        return HandlerReturn(value=value)
    return HandlerReturn(value=min(value, 2048))  # 4096スケールで50%
```
