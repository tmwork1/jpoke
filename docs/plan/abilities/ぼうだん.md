# 特性実装計画: ぼうだん

更新日: 2026-05-03

## 仕様参照

- `docs/spec/abilities/ぼうだん.md`

## 実装ゴール

- 弾の技（ボール・ばくだん・ボム系の技）を無効化する。
- じばく・だいばくはつは防げない。
- かふんだんごも無効化する（味方からの回復効果含む）。

## 実装方針

- `Event.ON_BEFORE_APPLY_MOVE`（`docs/spec/turn.md` の同種特性 ぼうじん と同じ priority=100 の
  `h.AbilityHandler` デフォルト値を使用）で `ctx.move.has_flag("bullet")` の技を無効化する
  （`subject_spec="defender:self"`）。
- 弾技フラグ `"bullet"` は `data/moves/*.py`（五十音別モジュール）に技ごとに個別付与済み
  （アシッドボム・ウェザーボール・エナジーボール・エレキボール・かえんだん・かえんボール・
  かふんだんご・がんせきほう・きあいだま・くちばしキャノン・ジャイロボール・シャドーボール・
  タネばくだん・タネマシンガン・でんじほう・どろばくだん・はどうだん・ヘドロばくだん・
  みずあめボム・ミストボール・ロックブラスト）。
- じばく・だいばくはつには意図的に `"bullet"` を付与しない（`"explosion"` フラグのみ）ため、
  ぼうだんでは防げない。
- かたやぶり無効化は `"mold_breaker_ignorable"` フラグ（`AbilityData.flags`）で対応する。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `src/jpoke/data/moves/*.py`（弾技フラグの整備）
- `tests/abilities/test_ability__group.py`

## テスト観点

1. かえんボール等の弾の技を受けて無効化する。
2. かたやぶり所持の攻撃側には無効化されない（`mold_breaker_ignorable`）。
3. じばく・だいばくはつは通常通りダメージを受ける（弾の技でないため）。

## 保留・論点

- なし（弾技フラグ `"bullet"` の付与対象は spec 一覧と突合済み）。
- かふんだんごの「味方への回復効果も無効化する」仕様は、本プロジェクトの対象範囲が
  シングルバトルのみ（味方が存在しない）のため実装上は考慮不要
  （`data/moves/move_ka.py` のコメント参照）。
