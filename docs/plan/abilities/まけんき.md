# 特性実装計画: まけんき

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/まけんき.md`

## 実装ゴール

- 敵から能力ランクを下げられたとき、こうげきが2段階上昇する。
- 自分の技の追加効果や味方からの能力低下では発動しない。
- A↑ が成立しなくてもメッセージは出る（すでに最大のとき）。

## 実装方針

- `Event.ON_MODIFY_STAT`、`subject_spec="target:self"` で実装。
- **かちき** と同一パターン。違いは上昇させる能力が C → A の点のみ。
- `ctx.source != ctx.target` で自傷チェック。

## 変更候補

- `src/jpoke/data/ability.py` — `まけんき` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `まけんき_on_stat_down` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 相手からいかくを受けると A が 2 段階上がる（差し引き A+1）。
2. 自分の技の追加効果で能力が下がっても発動しない。
3. 複数能力が同時に下がる技を受けると下がった数だけ発動する。
4. A が最大でもメッセージ表示（バー非表示）。

def まけんき_on_stat_down(battle, ctx, value):
    has_negative = any(v < 0 for v in value.values())
    if (
        has_negative
        and ctx.source != ctx.target
        and battle.modify_stats(ctx.target, {"A": +2}, source=ctx.source)
    ):
        announce_ability_triggered(battle, ctx, value, mon=ctx.target)
    return HandlerReturn(value=value)
