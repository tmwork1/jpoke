# 特性実装計画: カーリーヘアー

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/カーリーヘアー.md`

## 実装ゴール

- 直接攻撃を受けたとき、攻撃者のすばやさを1段階下げる。
- 直接攻撃でない技（念力等）では発動しない。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"` で実装。
- `battle.query.is_contact(ctx)` で接触判定後、`battle.modify_stats(ctx.attacker, {"S": -1}, ...)` を呼ぶ。
- 既存の **ぬめぬめ** と同一効果。ハンドラ関数を共有するか専用関数を作成する。

## 変更候補

- `src/jpoke/data/ability.py` — `カーリーヘアー` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `カーリーヘアー_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 直接攻撃を受けると攻撃者の S が 1 段階下がる。
2. 非接触技（念力、波動弾等）では発動しない。
3. えんかく持ち・パンチグローブ所持者の攻撃では発動しない。
