# 特性実装計画: ヘドロえき

更新日: 2026-06-03

## スコープ

- 優先度: H
- 仕様書: `docs/spec/abilities/ヘドロえき.md`
- 実装状態: 未実装
- 実装方針: `ON_MODIFY_DRAIN_HEAL`（または吸収処理のイベント）で、ヘドロえき所持者が被吸収対象のとき、回復量を攻撃者へのダメージに反転させる。

## 実装ゴール（完了定義）

- 吸収技（ギガドレイン・ドレインパンチ等）を受けたとき、攻撃者は回復せずにダメージを受ける。
- やどりぎのタネを植えられている場合も毎ターンダメージに変換する。
- マジックガード持ちの攻撃者にはダメージを与えられない（回復効果も消える）。
- みがわりを貫通してダメージを与える。
- 既存テストを壊さない。

## 実装対象イベント

- `Event.ON_MODIFY_DRAIN_HEAL`（または同等のイベント）— 吸収ダメージの回復量を攻撃者へのダメージに変換する

## 変更対象（ファイル/関数単位）

- `src/jpoke/data/ability.py` — `ヘドロえき` に handler を登録する
- `src/jpoke/handlers/ability.py` — `ヘドロえき_reverse_drain` を実装する
- `tests/test_ability.py` — テストを追加する

## 実装ステップ

1. 吸収技の回復処理がどのイベントで行われているか `src/jpoke/core/` で特定する。
2. `ヘドロえき_reverse_drain` を実装する（吸収回復量を攻撃者への `battle.modify_hp` ダメージとして付与し、回復は 0 を返す）。
3. `src/jpoke/data/ability.py` の `ヘドロえき` に handler を登録する。
4. テストを追加し、全テストが通ることを確認する。

## テスト観点

1. ギガドレイン等の吸収技を受けると攻撃者が回復せずにダメージを受ける。
2. やどりぎのタネを植えられたとき、毎ターン攻撃者がダメージを受ける。
3. マジックガード持ちの攻撃者には吸収ダメージを与えられない。
