# 特性実装計画: ヘドロえき

更新日: 2026-06-03

## スコープ

- 優先度: H
- 仕様書: `docs/spec/abilities/ヘドロえき.md`
- 実装状態: ハンドラ実装済み（`handlers/ability.py`）、data への登録のみ残
- 実装方針: `Event.ON_MODIFY_HEAL` に subject_spec="target:self" でフックし、`ctx.hp_change_reason == "drain"` のとき回復をキャンセルして攻撃者にダメージを与える。吸収技の回復処理で hp_change_reason="drain" を設定済みであることが前提。

## 実装ゴール（完了定義）

- 吸収技（ギガドレイン・ドレインパンチ等）を受けたとき、攻撃者は回復せずにダメージを受ける。
- やどりぎのタネを植えられている場合も毎ターンダメージに変換する。
- マジックガード持ちの攻撃者にはダメージを与えられない（回復効果も消える）。
- みがわりを貫通してダメージを与える。
- 既存テストを壊さない。

## 実装対象イベント

- `Event.ON_MODIFY_HEAL` — subject_spec="target:self"（ヘドロえき所持者が target）。`ctx.hp_change_reason == "drain"` でフィルタ。

`ON_MODIFY_HEAL` は `core/battle.py` の HP 回復処理で発火済み。ctx.source=吸収技使用者（回復予定）、ctx.target=ヘドロえき所持者。
