# 特性実装計画: しょうりのほし

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/しょうりのほし.md`

## 実装ゴール

- 場に出ている間、使用する技の命中率が 4506/4096 倍（約1.1倍）になる。
- 一撃必殺技の命中率は上昇しない。

## 実装方針

- `Event.ON_MODIFY_ACCURACY`、`subject_spec="attacker:self"` で実装。
- `ctx.move.has_flag("ohko")` で一撃必殺技を除外。
- `apply_fixed_modifier(value, 4506)` で補正。
- **ふくがん**（命中率1.3倍）と同パターン（倍率のみ異なる）。
- ダブルバトル以上では味方の技の命中率も補正するが、本プロジェクトはシングルバトルのみが対象のため `subject_spec="attacker:self"` で自分の技のみを対象とする。

## 変更候補

- `src/jpoke/data/ability.py` — `しょうりのほし` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `しょうりのほし_modify_accuracy` を実装する

## テスト観点

1. 技の命中率が 4506/4096 倍になる（例: 85% → 約 93%）。
2. 一撃必殺技は命中率変化なし。
