# 特性実装計画: すてみ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/すてみ.md`

## 実装ゴール

- 反動を受けるわざの威力が 1.2 倍になる（4915/4096）。
- 反動のない技には適用されない。

## 実装方針

- `Event.ON_CALC_POWER_MODIFIER`、`subject_spec="attacker:self"` で実装。
- `ctx.move.has_label("recoil")` で反動技を識別。
- `apply_fixed_modifier(value, 4915)` で補正（1.2 倍）。
- **いしあたま**（反動なし）と対になる特性。既存の技ラベル "recoil" が付与されている技を確認する。

## 変更候補

- `src/jpoke/data/ability.py` — `すてみ` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `すてみ_modify_power` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 反動技（とびひざげり等）の威力が 1.2 倍になる。
2. 反動なし技（たいあたり等）には適用されない。
