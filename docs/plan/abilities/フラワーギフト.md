# 特性実装計画: フラワーギフト

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/フラワーギフト.md`

## 実装ゴール

- 天気が晴れのとき、こうげき・とくぼうが 1.5 倍になる（シングルでは自分のみ）。
- 晴れ以外では発動しない。

## 実装方針

- `Event.ON_CALC_ATK_MODIFIER`（A と D 両方に適用）で実装。
- `battle.weather.sunny` で晴れ確認後、`apply_fixed_modifier(value, 6144)` を返す。
- **サンパワー**（晴れ中C↑1.5倍）と同イベント。こうげき/とくぼうへの適用可否は `ON_CALC_ATK_MODIFIER` の subject_spec で調整。
- フォルムチェンジ（はなやかなすがた）は `turn.md` ON_SWITCH_IN priority 140 に記載あり。本計画では能力補正のみ対象とし、フォルムチェンジは別途対応。
- `flags=["uncopyable", "mold_breaker_ignorable"]` は data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `フラワーギフト` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `フラワーギフト_modify_atk` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 晴れ中 → こうげきが 1.5 倍になる。
2. 晴れ中 → とくぼうが 1.5 倍になる。
3. 雨等他天候では変化なし。
