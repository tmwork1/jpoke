# 特性実装計画: ぜったいねむり

更新日: 2026-06-04

## スコープ

- 優先度: H
- 仕様書: `docs/spec/abilities/ぜったいねむり.md`
- 実装状態: 部分実装（flags=[uncopyable, protected, gas_proof] のみ設定済み）
- 実装方針: **「ゆめうつつ」を `AilmentName` の一種として追加する。** ailment が active な間は他の ailment 付与が自動ブロックされる既存ロジックを利用して状態異常の無効化を実現する。`AilmentData` に `is_sleep: bool`・`uncurable: bool` フラグを追加し、「ねむり扱い」の判定を ailment 経由で一元化する。`AilmentManager` の `remove` / `apply(overwrite=True)` に `uncurable` ガードを追加する。いびき・ねごとのねむり判定は `ailment.is_sleep` を参照するよう修正する。

## 実装ゴール（完了定義）

- あらゆる状態異常（ねむり含む）にならない。
- いびき・ねごとが使える（`ailment.is_sleep == True` で判定）。
- めざましビンタを受けると威力が 2 倍になる。
- たたりめ・ひゃっきやこうを受けると威力が 2 倍になる（タイプ相性無効の場合除く）。
- かがくへんかガスで無効化されない（gas_proof は実装済み）。
- ラムのみ・アロマセラピー等で「ゆめうつつ」ailment が除去されない。
- 既存テストを壊さない。
