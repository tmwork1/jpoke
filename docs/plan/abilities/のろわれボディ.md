# 特性実装計画: のろわれボディ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/のろわれボディ.md`

## 実装ゴール

- 直接攻撃を受けたとき、30%の確率で攻撃技をかなしばり状態にする。
- 直接攻撃でない技では発動しない。
- みがわりに当たった場合は発動しない。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"`、priority=20 で実装。
- priority は `turn.md` ON_DAMAGE 「のろわれボディ等: 攻撃技を受けた」の priority 20 に従う。
- `battle.query.is_contact(ctx)` 後に 30% 乱数判定、成立で `battle.volatile_manager.apply(ctx.attacker, "かなしばり", ...)` を呼ぶ。

## 変更候補

- `src/jpoke/data/ability.py` — `のろわれボディ` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `のろわれボディ_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 直接攻撃を受けて 30% でかなしばりが発動する。
2. 非接触技では発動しない。
3. かなしばり状態になると攻撃者は同じ技を使えなくなる。

def のろわれボディ_on_damage(battle, ctx, value):
    if not battle.query.is_contact(ctx):
        return HandlerReturn(value=value)
    if battle.random.random() < 0.3:
        battle.volatile_manager.apply(
            ctx.attacker, "かなしばり",
            source=ctx.defender, ctx=ctx, move=ctx.move,
        )
        announce_ability_triggered(battle, ctx, value, mon=ctx.defender)
    return HandlerReturn(value=value)
