- `flags=["mold_breaker_ignorable"]` は data/ability.py に既定済み。

- `src/jpoke/data/ability.py` — `テイルアーマー` に ON_TRY_MOVE_1 handler を登録する（じょおうのいげんの関数を共有）
- `tests/test_ability.py` — テストを追加する

## テスト観点

2. かたやぶり系 → 通る。
