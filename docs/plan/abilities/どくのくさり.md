# 特性実装計画: どくのくさり

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/どくのくさり.md`

## 実装ゴール

- 攻撃を当てると、30%の確率で相手をもうどく状態にする。
- ちからずく使用時は追加効果が発動しない。
- どく耐性を持つ相手（どく・はがねタイプ等）には無効。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="attacker:self"` で実装。
- `battle.resolve_secondary_chance(ctx, 0.3)` で確率を取得してランダム判定後、`battle.ailment_manager.apply(ctx.defender, "もうどく", ...)` を呼ぶ。
- **どくしゅ**（どく付与）と同パターン。違いはもうどくを付与する点のみ。

## 変更候補

- `src/jpoke/data/ability.py` — `どくのくさり` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `どくのくさり_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 攻撃命中時に 30% でもうどく付与される。
2. ちからずく使用者には発動しない。
3. どく・はがねタイプへは ailment_manager が拒否する。

def どくのくさり_on_damage(battle, ctx, value):
    if battle.random.random() < battle.resolve_secondary_chance(ctx, 0.3):
        battle.ailment_manager.apply(
            ctx.defender, "もうどく", source=ctx.attacker, ctx=ctx,
        )
    return HandlerReturn(value=value)
