# 特性実装計画: すいすい

更新日: 2026-07-11

## 仕様参照

- `docs/spec/abilities/すいすい.md`

## 実装ゴール

- 天気があめ・おおあめ状態のとき、すばやさが2倍になる。
- あめ状態でも特性ノーてんき/エアロックのポケモンが場にいる場合は発動しない。
- あめ状態でもばんのうがさを持っている場合は発動しない。

## 実装方針

- `DomainEvent.ON_CALC_SPEED`、`subject_spec="source:self"` で実装。
- `battle.weather_for(ctx.source).rainy` で判定する。
  - `weather_for` は `ON_CHECK_WEATHER_IMMUNE`（ばんのうがさ等）が発動している場合や、
    ノーてんき/エアロックで天候効果が無効化されている場合に「なし」天候を返すため、
    追加の重複ガードは不要。
  - `Field.rainy` はあめ・おおあめの両方を True とするため、おおあめも自動的にカバーされる。
- **サーフテール**（エレキフィールド中 S×2）・**すなかき**（すなあらし中 S×2）と同一パターン
  （天候/フィールドの違いのみ）。ばんのうがさの影響を受けるのはあめ・はれ系天候のみのため、
  すなかきは `battle.weather.name` を直接参照している一方、すいすいは `weather_for` を使う。

## テスト観点

1. あめ中 → S 実数値が2倍になる。
2. おおあめ中 → S 実数値が2倍になる。
3. あめ以外（天候なし等）では変化なし。
4. ばんのうがさ所持時はあめ中でも発動しない。
