# 特性実装計画: ぬめぬめ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ぬめぬめ.md`

## 実装ゴール

- 直接攻撃を受けたとき、攻撃者のすばやさを1段階下げる。
- 直接攻撃でない技では発動しない。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"` で実装。
- **カーリーヘアー** と同一効果。ハンドラ関数を共有するか専用関数を用意する。
- `battle.query.is_contact(ctx)` → `battle.modify_stats(ctx.attacker, {"S": -1}, ...)` の流れ。

## 変更候補

- `src/jpoke/data/ability.py` — `ぬめぬめ` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `ぬめぬめ_on_damage` を実装する（またはカーリーヘアーと共有）
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 直接攻撃を受けると攻撃者の S が 1 段階下がる。
2. 非接触技では発動しない。
3. S が最低ランクの攻撃者には発動しない（modify_stats が False）。

## 検証コマンド

```powershell

```
