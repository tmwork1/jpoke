# 特性実装計画: こぼれダネ

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/こぼれダネ.md`

## 実装ゴール

- 攻撃技でダメージを受けたとき、グラスフィールドを展開する（5ターン）。
- すでにグラスフィールドが展開されている場合は発動しない。
- HP が 0 になったときも、特性を発動させてからひんしになる。
- グランドコートを持っているとき 8 ターンに延長される（terrain.apply の延長機能で対応）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"` で実装。
- `battle.terrain.apply("グラスフィールド", 5, source=mon)` の成否でアナウンスを制御。
- **グラスメイカー**（ON_SWITCH_IN でグラスフィールド展開）と同一の terrain.apply パターン。

## 変更候補

- `src/jpoke/data/ability.py` — `こぼれダネ` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `こぼれダネ_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 攻撃受けてグラスフィールドが展開される（5ターン）。
2. すでにグラスフィールド中は発動しない。
3. 連続攻撃の1発目でフィールドが発動し、2発目以降はフィールド効果を受ける。
