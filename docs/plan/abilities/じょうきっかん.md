# 特性実装計画: じょうきっかん

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/じょうきっかん.md`

## 実装ゴール

- みずタイプかほのおタイプの攻撃技でダメージを受けたとき、すばやさが6段階上がる。
- S がすでに最大のときは発動しない。
- 変化技（みずびたし等）では発動しない（ON_DAMAGE_HIT は damage > 0 のときのみ発火）。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"` で実装。
- `ctx.move.type in ("みず", "ほのお")` でタイプチェック後、`battle.modify_stats(mon, {"S": +6}, ...)` を呼ぶ。
- **びびり**（タイプ被弾→S↑1）と同パターン（対象タイプ・上昇量が異なる）。

## 変更候補

- `src/jpoke/data/ability.py` — `じょうきっかん` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `じょうきっかん_on_damage` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. みず技受けると S が 6 段階上がる。
2. ほのお技受けると S が 6 段階上がる。
3. でんき等他タイプでは発動しない。
4. S がすでに最大のときは発動しない。

```python
def じょうきっかん_on_damage(battle, ctx, value):
    if ctx.move.type not in ("みず", "ほのお"):
        return HandlerReturn(value=value)
    mon = ctx.defender
    if battle.modify_stats(mon, {"S": +6}, source=ctx.attacker):
        announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
```
