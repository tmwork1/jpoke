# 特性実装計画: するどいめ

更新日: 2026-05-02

## 仕様参照

- `spec/ability/するどいめ.md`

## 実装ゴール

- 相手から命中率のランク補正を下げられなくなる。
- 第六世代以降: 相手の回避率ランクを無視して攻撃できる。

## 実装方針

- 命中率ランク低下防止: `ON_BEFORE_MODIFY_STAT`（subject_spec="target:self"）で、相手（`ctx.source != ctx.target`）由来の命中率ランク低下のみを除去する（かいりきバサミ・はとむね・クリアボディと同じ `_block_stat_drop_by_foe` を再利用）。
- 回避率無視: `ON_GET_STAT_RANK`（subject_spec="attacker:self"）で、命中判定直前に構築される `{"accuracy": 攻撃側の命中ランク, "evasion": 防御側の回避ランク}` の `evasion` を常に0にする。上昇・低下いずれのランクも無視する（第六世代以降の仕様）。

## テスト観点

1. 命中率ランクを下げる技・特性を受けても命中率ランクが低下しない（かたやぶり等の無効化不可効果では下がる）。
2. 相手の回避率ランクが上がっていても命中率の計算に影響しない。
3. 相手の回避率ランクが下がっていても命中率の計算に影響しない（低下分を無視して有利にならない）。
