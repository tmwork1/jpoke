# 特性実装計画: ナイトメア

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ナイトメア.md`

## 実装ゴール

- ねむり状態の相手のHPを毎ターン最大HPの1/8削る。
- 相手が目覚めると発動しない。
- 相手がひんしなら発動しない。

## 実装方針

- `Event.ON_TURN_END`、`subject_spec="source:self"`、priority=150 で実装。
- priority は `turn.md` ON_TURN_END_4 priority 150「ナイトメア」に従う。
- `battle.foe(mon).ailment.name == "ねむり"` で条件確認後、`battle.modify_hp(foe, r=-1/8, reason="ability")` を呼ぶ。

## 変更候補

- `src/jpoke/data/ability.py` — `ナイトメア` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `ナイトメア_on_turn_end` を実装する
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. 相手がねむり中 → ターン終了時に最大 HP の 1/8 ダメージ。
2. 相手が目覚めると発動しない。
3. 相手がひんしなら発動しない。

def ナイトメア_on_turn_end(battle, ctx, value):
    mon = ctx.source
    foe = battle.foe(mon)
    if foe is None or not foe.alive:
        return HandlerReturn(value=value)
    if foe.ailment.name != "ねむり":
        return HandlerReturn(value=value)
    if battle.modify_hp(foe, r=-1/8, reason="ability"):
        announce_ability_triggered(battle, ctx, value, mon=mon)
    return HandlerReturn(value=value)
