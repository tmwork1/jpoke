# 特性実装計画: ナイトメア

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ナイトメア.md`

## 実装ゴール

- ねむり状態（特性ぜったいねむりによる「ゆめうつつ」を含む）の相手のHPを毎ターン最大HPの1/8削る
  （小数点以下切り捨て、切り捨てて0になる場合は1ダメージ）。
- 相手が目覚めると発動しない。
- 特性マジックガードの相手には無効。
- 相手がひんしなら発動しない（`modify_hp` が hp=0 のポケモンに対しては差分0を返すため、
  追加のガード不要）。

## 実装方針

- `Event.ON_TURN_END`、`subject_spec="source:self"`、priority=150 で実装。
- priority は `turn.md` ON_TURN_END_4 priority 150「ナイトメア」に従う。
- 判定は `foe.ailment.is_sleep`（「ねむり」と、ぜったいねむり由来の「ゆめうつつ」の両方を含む）で行う。
  `ailment.name == "ねむり"` のような名前比較では「ゆめうつつ」を取りこぼすため使わない。
- `battle.modify_hp(foe, r=-1/8)`（reason省略＝""＝「その他のダメージ」）を呼ぶ。
  reason を "" にすることでマジックガードの対象（`hp_change_reason not in {...}` に該当）となり無効化される。

## 変更候補

- `src/jpoke/data/ability.py` — `ナイトメア` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `ナイトメア_damage_sleeping_foe` を実装する
- `tests/abilities/test_ability_na.py` — テストを追加する

## テスト観点

1. 相手がねむり中でない → 発動しない。
2. 相手がねむり中 → ターン終了時に最大 HP の 1/8 ダメージ（切り捨て）。
3. 特性マジックガードの相手 → ダメージ無効。
4. 特性ぜったいねむり相当の「ゆめうつつ」状態の相手 → ダメージを受ける。

## 疑似コード

```python
def ナイトメア_damage_sleeping_foe(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """ナイトメア特性: 相手がねむり状態のとき毎ターン最大HPの1/8を削る。"""
    mon = ctx.source
    foe = battle.foe(mon)
    if (
        foe.ailment.is_sleep
        and battle.modify_hp(foe, r=-1/8)
    ):
        _announce_ability_triggered(battle, mon)
    return HandlerReturn(value=value)
```
