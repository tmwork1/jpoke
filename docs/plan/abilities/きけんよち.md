# 特性実装計画: きけんよち

更新日: 2026-06-03

## スコープ

- 優先度: H
- 仕様書: `docs/spec/abilities/きけんよち.md`
- 実装状態: 未実装
- 実装方針: `ON_SWITCH_IN` で相手の技一覧を走査し、こちらのタイプに対してバツグンになる攻撃技か一撃必殺技があれば「みぶるいした」ログを出す。バトル結果への影響はなし（情報系特性）。

## 実装ゴール（完了定義）

- 登場時に相手がバツグン技または一撃必殺技を持つ場合にメッセージが表示される。
- 相手がそのような技を持たない場合はメッセージが表示されない。
- 既存テストを壊さない。

## 実装対象イベント

- `Event.ON_SWITCH_IN` — 登場時に相手の技リストを確認してメッセージを出す

## 変更対象（ファイル/関数単位）

- `src/jpoke/data/ability.py` — `きけんよち` に handler を登録する
- `src/jpoke/handlers/ability.py` — `きけんよち_on_switch_in` を実装する
- `tests/test_ability.py` — テストを追加する

## 実装ステップ

1. 仕様書 `docs/spec/abilities/きけんよち.md` でタイプ判定の細かい仕様（一撃必殺・変化技除外）を確認する。
2. `src/jpoke/handlers/ability.py` に `きけんよち_on_switch_in` を実装する（相手の技を走査し条件を満たせばログを出す）。
3. `src/jpoke/data/ability.py` の `きけんよち` に handler を登録する。
4. `tests/test_ability.py` にテストを追加し、全テストが通ることを確認する。

## テスト観点

1. 相手がバツグンになる攻撃技を持つとき、登場時にメッセージが出る。
2. 相手が一撃必殺技を持つとき、メッセージが出る。
3. 相手が等倍・いまひとつ・無効の技しか持たないとき、メッセージが出ない。
4. 相手が変化技のみを持つとき、メッセージが出ない。
