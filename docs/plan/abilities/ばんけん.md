# 特性実装計画: ばんけん

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ばんけん.md`

## 実装ゴール

- いかくを受けると A が 1 段階上がる（いかくによる A↓ は通常通り適用され、その後 A↑1 が加算される）。
- 交代させる技（ほえる・ふきとばし等）の効果を受けない。
- かたやぶり系には無効。

## 実装方針

- `Event.ON_BEFORE_MODIFY_STAT`（いかく反応）と `Event.ON_TRY_BLOW`（強制交代防止）の 2 ハンドラ構成。
- いかく検出は `ctx.stat_change_reason == "いかく"` で判定し、`battle.modify_stats(mon, {"A": +1}, ...)` を追加。
- ON_TRY_BLOW は **きゅうばん** と同一ハンドラ関数を共有する。
- `flags=["mold_breaker_ignorable"]` は data/ability.py に既定済み。

## 変更候補

- `src/jpoke/data/ability.py` — `ばんけん` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `ばんけん_on_intimidate` を実装する（ON_TRY_BLOW はきゅうばんと共有）
- `tests/test_ability.py` — テストを追加する

## テスト観点

1. いかく発動時 → A↓1 の後に A↑1 が加算される（差し引き変化なし）。
2. ほえる等の強制交代 → 無効になる。
3. いかく以外の A↓ 技（あまえる等）には A↑ は発動しない。
4. かたやぶり系では無効。
