# 特性実装計画: ほうし

更新日: 2026-06-05

## 仕様参照

- `docs/spec/abilities/ほうし.md`

## 実装ゴール

- 直接攻撃を受けたとき、30%の確率でどく・まひ・ねむりのいずれかにする。
  確率内訳: どく 9%、まひ 10%、ねむり 11%（合計 30%）。
- 直接攻撃でない技では発動しない。
- くさタイプ・ぼうじん・ぼうじんゴーグル所持者には無効（ailment_manager 側で制御）。
- ちからずくでは追加効果とは扱われないため、ちからずくの無効化対象外。

## 実装方針

- `Event.ON_DAMAGE_HIT`、`subject_spec="defender:self"` で実装。
- `battle.query.is_contact(ctx)` の後、乱数で状態異常を決定して `battle.ailment_manager.apply` を呼ぶ。
- 状態異常の無効化（まひ耐性等）は `ailment_manager.apply` 側で処理される。

## 変更候補

- `src/jpoke/data/ability.py` — `ほうし` に handlers を登録する
- `src/jpoke/handlers/ability.py` — `ほうし_on_damage` を実装する

## テスト観点

1. 直接攻撃受けて 30% 以内で状態異常が発動する。
2. 非接触技では発動しない。
3. どく耐性・まひ耐性を持つ相手には該当異常が付与されない。

## 疑似コード

```python
def ほうし_on_damage(battle, ctx, value):
    if not battle.query.is_contact(ctx):
        return HandlerReturn(value=value)
    r = battle.random.random()
    if r < 0.09:
        ailment = "どく"
    elif r < 0.19:
        ailment = "まひ"
    elif r < 0.30:
        ailment = "ねむり"
    else:
        return HandlerReturn(value=value)
    battle.ailment_manager.apply(ctx.attacker, ailment, source=ctx.defender, ctx=ctx)
    return HandlerReturn(value=value)
```
