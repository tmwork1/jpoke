# 特性実装計画: サンパワー

更新日: 2026-07-11

## 仕様参照

- `docs/spec/abilities/サンパワー.md`

## 実装ゴール

- にほんばれ/おおひでり中、特殊技の特攻が1.5倍になる。
- 毎ターン終了時に最大HPの1/8のダメージを受ける（小数点以下切り捨て、最低1）。
- ノーてんき/エアロック・ばんのうがさ所持中は特攻補正と天候ダメージ両方が発動しない
  （`battle.weather_for()` が自動的に判定する）。

## 実装方針

- 特攻補正: `Event.ON_CALC_ATK_MODIFIER`（`subject_spec="attacker:self"`）で
  `battle.weather_for(ctx.attacker).sunny` かつ特殊技のときに `apply_fixed_modifier(value, 6144)` で1.5倍を適用する。
- ダメージ: `Event.ON_TURN_END`（`subject_spec="source:self"`, priority=30 —
  `docs/spec/turn.md` の ON_TURN_END 表で「サンパワー」行が priority=30 と明記されている）で
  `battle.weather_for(mon).sunny` のとき `battle.modify_hp(mon, r=-1/8)` を適用する。
- `battle.weather_for()` がエアロック・ノーてんき・ばんのうがさによる天候無効化を吸収するため、
  ハンドラ側で個別に判定する必要はない。
- かんそうはだのにほんばれ中HP1/8ダメージと同パターン。

## リーサル計算

- `LethalEvent.ON_TURN_END` / `subject="defender"` で `handlers/lethal.py` の
  `サンパワー_take_sun_damage()` を登録する。
- `hp_dist` は defender（計算対象ポケモン）のHP分布のみを追跡するため、サンパワーの効果は
  「defender 自身がサンパワーを持つ場合」にのみ hp_dist に反映できる
  （attacker がサンパワーを持つ場合の自傷は hp_dist に影響を与えないため追跡不能）。
- `battle.weather_for(ctx.defender).sunny` のとき `_damage(hp_dist, max(1, ctx.defender.max_hp // 8))` を適用する
  （`かんそうはだ_weather_hp` と同パターン）。

## テスト観点

1. にほんばれ/おおひでり中の特殊技の特攻が1.5倍になる。
2. 物理技には特攻補正が適用されない。
3. 毎ターン終了時に最大HPの1/8のダメージを受ける。
4. ノーてんき/エアロック下では特攻補正もダメージも発動しない。
5. ばんのうがさ所持中は特攻補正もダメージも発動しない。
6. リーサル計算で、defender がサンパワーを持ち晴れ状態のとき、ターン終了時のダメージが
   hp_dist に反映される。

## 保留・論点

- あなをほる/ダイビング中はダメージが無いが、この判定はコードベース全体で未実装
  （かんそうはだ・すなあらし等の天候由来ターン終了ダメージも同様に未対応）。
  ゆえに本特性単体でのみ対応するのはスコープ外とし、既存実装のパターンに合わせて対応しない。
