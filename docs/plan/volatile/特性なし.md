# 特性なし — 実装プラン

## 実装状況

✅ **実装済み** (volatile キー: とくせいなし)

## 効果概要

かがくへんかガス等で相手の特性を無効化する。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_CHECK_DEF_ABILITY_ENABLED | defender:self | False で防御側特性無効 |
| ON_CHECK_ABILITY_ENABLED | source:self | False で自特性無効 |
| ON_APPLY_VOLATILE | source:self | 付与時に同期処理 |
| ON_VOLATILE_END | source:self | 解除時に復元処理 |

## テスト状況

## 実装メモ

- `ON_CHECK_ABILITY_ENABLED` / `ON_CHECK_DEF_ABILITY_ENABLED` の両方で `False` を返し、自他の特性判定を止める。
- 付与時と解除時に `refresh_effect_enabled_states()` を呼んで有効状態を同期する。

## テスト要件

- [x] 防御側の特性判定が無効化される
- [x] 付与直後に `ability.enabled` が `False` になる
- [x] 解除時に特性有効状態が復元される
