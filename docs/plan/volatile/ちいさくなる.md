# ちいさくなる — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

特定技（ふみつけ・ボディスラム等）で命中保証・威力2倍。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| ON_MODIFY_ACCURACY | defender:self | 必中フラグを返す |
| ON_CALC_POWER_MODIFIER | defender:self | 対象技に威力2倍 |

## テスト状況

## 実装メモ

- 特定技に対して `ON_MODIFY_ACCURACY` で必中化し、`ON_CALC_POWER_MODIFIER` で威力2倍を返す。
- どちらも防御側の volatile として処理する。

## テスト要件

- [x] 対象技が必中になる
- [x] 対象技の威力が2倍になる
- [x] 対象外の技ではどちらも起きない
