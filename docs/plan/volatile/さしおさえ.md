# さしおさえ — 実装プラン

## 実装状況

❌ **未実装**

## 概要

- 効果: 対象の持ち物効果を5ターン無効化（使用・効果発動ともに不可）
- 持続: 5ターン、交代で解除（使用者退場では解除されない）

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| `ON_CHECK_ITEM_ENABLED` | `source:self` | `False` を返して持ち物効果を無効化 |
| `ON_TURN_END_3` | `source:self` | 5ターン tick → 解除 |

> **確認済み**: `ON_CHECK_ITEM_ENABLED` イベントは `src/jpoke/enums/event.py` に存在する。
> `ぶきよう` 特性が同イベントを使って道具効果を抑制している（`data/ability.py` 参照）。

## 実装概要

### data/volatile.py

```python
"さしおさえ": VolatileData(
    handlers={
        Event.ON_CHECK_ITEM_ENABLED: h.VolatileHandler(
            lambda *args: HandlerReturn(value=False),
            subject_spec="source:self",
        ),
        Event.ON_TURN_END_3: h.VolatileHandler(
            partial(h.tick_volatile, name="さしおさえ", duration=5),
            subject_spec="source:self",
        ),
    }
),
```

> **NOTE**: `tick_volatile` の duration パラメータを確認すること。
> 既存実装では `tick_volatile` にデフォルト duration がある場合がある。
