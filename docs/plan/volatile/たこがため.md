❌ **未実装**

## 概要

- 効果: 毎ターン対象の防御・特防ランクを各-1
- 持続: 対象が場にいる間。**使用者が退場すると解除**（バインド同様）
- 重ね掛け不可

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| `ON_TURN_END_3` | `source:self` | 防御・特防を各-1段階 |
| `ON_SWITCH_OUT` | `source:foe` | 使用者（foe）退場時に状態解除 |

```python
def たこがため_tick(ctx: BattleContext) -> HandlerReturn:
    mon = ctx.source
    ctx.battle.modify_stat(mon, "B", -1)
    ctx.battle.modify_stat(mon, "D", -1)
    return HandlerReturn()

def たこがため_switch_out(ctx: BattleContext) -> HandlerReturn:
    # 使用者（foe）が退場したら対象の状態を解除
    target = ctx.source  # source:foe なので foe が source
    # 対象は source の相手 = self
    # NOTE: foe の退場イベントなので ctx.battle の自ポケモンを参照する必要あり
    # 既存のバインド処理を参考にする
    foe = ctx.foe  # または ctx.battle.get_opponent(ctx.source)
    ctx.battle.remove_volatile(foe, "たこがため")
    return HandlerReturn()
```

> **NOTE**: `ON_SWITCH_OUT` の subject_spec の設計はバインド（`source:foe`）と同様。
