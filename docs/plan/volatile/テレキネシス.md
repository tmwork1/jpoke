# テレキネシス — 実装プラン

## 実装状況

❌ **未実装**

## 概要

- 効果①: 対象を浮遊状態にする（地面技・ハザード等対象外）
- 効果②: 対象への技が必中になる（一撃必殺技を除く）
- 持続: 3ターン

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| `ON_CHECK_FLOATING` | `source:self` | `True` を返して浮遊扱いにする |
| `ON_MODIFY_ACCURACY` | `defender:self` | 命中率を保証（`None` または必中フラグ返す） |
| `ON_TURN_END_3` | `source:self` | 3ターン tick → 解除 |

> **参考**: `でんじふゆう` が `ON_CHECK_FLOATING` を使って浮遊化している。
> 命中保証は `ちいさくなる` の `ON_MODIFY_ACCURACY` を参考にする。

## 実装概要

### data/volatile.py

```python
"テレキネシス": VolatileData(
    handlers={
        Event.ON_CHECK_FLOATING: h.VolatileHandler(
            lambda *args: HandlerReturn(value=True),
            subject_spec="source:self",
        ),
        Event.ON_MODIFY_ACCURACY: h.VolatileHandler(
            h.テレキネシス_guaranteed_hit,
            subject_spec="defender:self",
        ),
        Event.ON_TURN_END_3: h.VolatileHandler(
            partial(h.tick_volatile, name="テレキネシス"),
            subject_spec="source:self",
        ),
    }
),
```

### handlers/volatile.py

```python
def テレキネシス_guaranteed_hit(ctx: BattleContext) -> HandlerReturn:
    # 命中率を必中（None または特定の定数）にする
    # ちいさくなる_guaranteed_hit の実装を参照すること
    return HandlerReturn(value=None)  # None = 命中保証
```
