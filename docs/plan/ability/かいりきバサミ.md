# 特性実装計画: かいりきバサミ

更新日: 2026-05-02

## 仕様参照

- `spec/ability/かいりきバサミ.md`

## 実装ゴール

- 相手から「こうげき」ランクを下げられなくなる。
- 自分の技・効果によるこうげきランク低下（ばかぢから等）は防げない。
- かたやぶりの効果がある技には無視される。
- クリアボディ系ハンドラと実装パターンを統一する。

## 実装方針

- `ON_BEFORE_MODIFY_STAT` で対象ランクが「こうげき」かつ変化量が負の場合に無効化する。
- 既存のクリアボディ実装パターンを流用し、対象能力を「こうげき」に限定した版として実装する。
- かたやぶり判定は `ON_CHECK_DEF_ABILITY_ENABLED` 経由で既存の共通処理に委ねる。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. 相手の技によるこうげきランク低下（いかく・おたけび等）を防ぐ。
2. ばかぢから等の自己効果によるこうげきランク低下は防げない。
3. かたやぶり効果の技には無視される。
4. くろいきり/クリアスモッグによるリセットは防げない。
5. ちからをすいとるを受けたときランクは下がらないが回復はされる。
