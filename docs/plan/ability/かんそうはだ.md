## 実装ゴール

✅ みずタイプ攻撃技を無効化し、HPが減っている場合は最大HPの1/4を回復する。
✅ ほのおタイプの攻撃技の威力を5/4倍にする。

## 実装方針

✅ **みず技無効化・回復**
- `Event.ON_CHECK_IMMUNE` で `_handle_type_absorb()` を使用
- HPが減っていれば最大HPの1/4回復

✅ **ほのお技威力増加**
- `Event.ON_CALC_DAMAGE_MODIFIER` で防御側として5/4倍補正を適用
- `ctx.check_def_ability_enabled(battle)` で特性有効化判定

✅ **ターン終了時天候効果**
- `Event.ON_TURN_END_3` で天候判定
- `battle.weather_manager.active` で有効な天候を確認
- あめ/おおあめ中：最大HPの1/8回復
- にほんばれ/おおひでり中：最大HPの1/8ダメージ
- ばんのうがさ判定で天候効果を抑制

```python
"かんそうはだ": AbilityData(
    flags=["mold_breaker_ignorable"],
    handlers={
        Event.ON_CHECK_IMMUNE: h.AbilityHandler(
            h.かんそうはだ_check_water_immune,
            subject_spec="target:self",
        ),
        Event.ON_CALC_DAMAGE_MODIFIER: h.AbilityHandler(
            h.かんそうはだ_modify_fire_damage,
            subject_spec="defender:self",

            h.かんそうはだ_on_turn_end,
            subject_spec="source:self",
        ),
    }
)
```
