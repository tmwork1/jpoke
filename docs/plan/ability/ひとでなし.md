# 特性実装計画: ひとでなし

更新日: 2026-05-02

## 仕様参照

- `spec/ability/ひとでなし.md`

## 実装ゴール

- どく/もうどく状態の相手への攻撃が必ず急所に当たる。
- カブトアーマー/シェルアーマー所持の相手・おまじない状態では急所に当たらない。

## 実装方針

- `ON_CALC_CRITICAL_RANK` で相手がどく/もうどく状態の場合に急所ランクを非常に大きな値（+∞相当、例えば +10）に設定し、必ず急所に当たるようにする。
- カブトアーマー/シェルアーマー等の急所無効化は既存の急所計算処理が担う。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. どく状態の相手への攻撃が必ず急所に当たる。
2. もうどく状態の相手への攻撃も必ず急所に当たる。
3. どく/もうどく状態でない相手には通常の急所率になる。
4. カブトアーマー所持の相手には急所に当たらない。
5. かがくへんかガス発動中は発動しない。
