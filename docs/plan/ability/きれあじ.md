# 特性実装計画: きれあじ

更新日: 2026-05-03

## 仕様参照

- `spec/ability/きれあじ.md`

## 実装ゴール

- 「切る技」の威力を1.5倍にする。
- 対象技: アクアカッター/エアスラッシュ/きりさく/クロスポイズン/サイコカッター/ソーラーブレード/つじぎり/ネズミざん/はっぱカッター/リーフブレード 等（仕様書掲載の完全リスト準拠）。

## 実装方針

- `ON_CALC_POWER_MODIFIER` で攻撃側として `"slash" in move.data.flags` の技に6144/4096倍を適用する。
- `MoveData.flags` の `"slash"` が未整備な技は `data/move.py` で補完する。
- がんじょうあご（かみつき技1.5倍）・メガランチャー（波動技1.5倍）と同一パターン。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `src/jpoke/data/move.py`（切る技フラグの整備）
- `tests/test_ability.py`

## テスト観点

1. アクアカッター等の切る技に威力1.5倍が適用される。
2. 切る技でない技には補正がかからない。
3. かがくへんかガス発動中は発動しない。

## 保留・論点

- `"slash"` フラグの付与対象（特に専用技）の境界を spec 一覧と突合する。
