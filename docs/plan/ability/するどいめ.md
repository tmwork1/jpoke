# 特性実装計画: するどいめ

更新日: 2026-05-02

## 仕様参照

- `spec/ability/するどいめ.md`

## 実装ゴール

- 相手から命中率のランク補正を下げられなくなる。
- 第六世代以降: 相手の回避率ランクを無視して攻撃できる。

## 実装方針

- 命中率ランク低下防止: `ON_BEFORE_MODIFY_STAT` で対象が命中率かつ変化量が負の場合に無効化する（かいりきバサミ・はとむねと同パターン）。
- 回避率無視: `ON_MODIFY_ACCURACY` で攻撃側として相手の回避率ランクを0扱いにする（しんがん系の処理を参考）。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. 命中率ランクを下げる技・特性を受けても命中率ランクが低下しない。
2. 相手の回避率ランクが上がっていても命中率の計算に影響しない。
3. ゆきがくれ/すながくれによる回避率上昇効果も無視される。
4. 必中技・一撃必殺技の命中率は変化しない。
5. かたやぶりの技には命中率ランク低下が無視される。
