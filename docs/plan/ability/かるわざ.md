# 特性実装計画: かるわざ

更新日: 2026-05-03
実装状況: **✅ 完了**

## 仕様参照

- `spec/ability/かるわざ.md`

## 実装ゴール

✅ 場に出た後に自分の持ち物が消費・奪取・除去されたとき、すばやさが2倍になる。
✅ 持ち物を再取得するとすばやさが元に戻る。
✅ 場に出た時点から持ち物がない場合は発動しない。
✅ 引っ込んで再び場に出た場合はリセットされ、未発動扱いになる。

## 実装方針

✅ `ability.state` で状態管理を実装
  - `"idle"`: 未発動状態（入場時に持ち物あり）
  - `"active"`: 発動状態（持ち物を消失）
  - `"inactive"`: 発動不可状態（入場時に持ち物なし）

✅ `DomainEvent.ON_CALC_SPEED` ですばやさ実数値を2倍にする（ランク補正ではない）。
✅ `ON_SWITCH_IN` で状態をリセットする。

## 実装完了内容

### src/jpoke/data/ability.py
```python
"かるわざ": AbilityData(
    handlers={
        Event.ON_SWITCH_IN: h.AbilityHandler(
            h.かるわざ_on_switch_in,
            subject_spec="source:self",
        ),
        DomainEvent.ON_CALC_SPEED: h.AbilityHandler(
            h.かるわざ_modify_speed,
            subject_spec="source:self",
        ),
    }
),
```

### src/jpoke/handlers/ability.py
- `かるわざ_on_switch_in()`: 入場時に持ち物有無を確認し、状態を設定
- `かるわざ_modify_speed()`: 状態遷移管理と速度補正を実装
