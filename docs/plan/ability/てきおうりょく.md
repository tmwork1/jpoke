# 特性実装計画: てきおうりょく

更新日: 2026-05-02

## 仕様参照

- `spec/ability/てきおうりょく.md`

## 実装ゴール

- タイプ一致ボーナスを1.5倍ではなく2.0倍（8192/4096）にする。
- テラスタルとの兼ね合い: テラスタイプが元タイプと一致する場合は2.25倍になる。

## 実装方針

- `ON_CALC_ATK_TYPE_MODIFIER` でタイプ一致ボーナスの倍率を6144/4096から8192/4096に置き換える。
- テラスタル中の計算は既存のテラスタル処理フローと連動させる。
- ステラタイプへのテラスタル時は発動しない（既存ステラ判定で制御）。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. タイプ一致技のSTABが2.0倍になる（通常は1.5倍）。
2. テラスタルで元タイプと一致するタイプにテラスタルした場合、STABが2.25倍になる。
3. テラスタルで元タイプと異なるタイプにテラスタルした場合、テラスタイプの技のSTABは2.0倍になる。
4. 不一致タイプの技には補正がかからない。
5. かがくへんかガス発動中は発動しない。
