# 特性実装計画: ダウンロード

更新日: 2026-05-02

## 仕様参照

- `spec/ability/ダウンロード.md`

## 実装ゴール

- 入場時に相手の防御と特防を比較し、防御 < 特防なら攻撃+1、防御 ≧ 特防なら特攻+1を得る。
- 比較に使う防御・特防はランク補正を考慮した実数値を使用する。
- かがくへんかガス中は発動しない。

## 実装方針

- `ON_SWITCH_IN` で相手の防御・特防実数値（ランク補正適用後）を取得して比較し、対応するランク+1を適用する。
- ダブルバトルでは相手2体の防御・特防の合計を比較する（実装スコープ外の場合は保留）。
- ランク変化はON_SWITCH_IN内で既存の `apply_stat_change` を呼ぶ。

## テスト観点

1. 防御 < 特防の相手に対して攻撃+1を得る。
2. 防御 ≧ 特防の相手に対して特攻+1を得る。
3. ランク補正を考慮した実数値で比較する。
4. かがくへんかガス中は発動しない。
