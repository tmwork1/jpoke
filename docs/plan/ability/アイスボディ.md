# 特性実装計画: アイスボディ

更新日: 2026-05-03

## 仕様参照

- `spec/ability/アイスボディ.md`

## 実装ゴール

- 天気がゆきのとき、毎ターン最大HPの1/16を回復する。
- 第九世代ではゆきにダメージがないため、天候ダメージ無効効果は実質影響しない。
- かいふくふうじ状態では回復しない。
- ノーてんき/エアロック存在下では発動しない。

## 実装方針

- `ON_TURN_END_3` で `battle.weather` を参照し、ゆき中に最大HPの1/16を回復する。
- 第九世代ルールでは天候ダメージ処理（あられ）を持たない前提で実装する。
- あめうけざら（あめ天候でHP1/16回復）と同一パターン（天候タイプが異なるだけ）。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. ゆき天候で毎ターン最大HPの1/16が回復する。
2. ゆき以外の天候では回復しない。
3. かいふくふうじ中は回復しない。
4. ノーてんき/エアロック存在下では回復しない。
5. HPが全快の場合でも発動しない（既存の回復上限処理に依存）。
