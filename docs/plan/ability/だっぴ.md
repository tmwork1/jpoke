# 特性実装計画: だっぴ

更新日: 2026-05-02

## 仕様参照

- `spec/ability/だっぴ.md`

## 実装ゴール

- ターン終了時、30%の確率で状態異常を回復する。
- こんらん等の状態変化は対象外。
- どく/もうどく/やけどのターン終了ダメージより先に発動する（発動ターンはそれらのダメージを受けない）。

## 実装方針

- `ON_TURN_END_2` または `ON_TURN_END_3` の前のフェーズで30%確率の状態異常解除を処理する。
- 発動タイミングはどくダメージ（`ON_TURN_END_3`）より先になる。  
  → `ON_TURN_END_2` で実装し、発動したターンのどく等ダメージをスキップさせる方針を検討する。
- 状態異常の除去は `pokemon_state.cure_ailment()` 相当の処理で実施する。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. ターン終了時に30%の確率で状態異常が回復する。
2. 発動したターンはどく/もうどく/やけどのダメージを受けない。
3. こんらん・メロメロ等の状態変化は回復しない。
4. どくどくだま/かえんだまは発動ターン後から再付与される。
5. かがくへんかガス発動中は発動しない。
