# 特性実装計画: せいでんき

更新日: 2026-05-02

## 仕様参照

- `spec/ability/せいでんき.md`

## 実装ゴール

- 直接攻撃を受けたとき、30%の確率で攻撃者をまひ状態にする。
- みがわり状態で防がれた場合は発動しない。
- まひ耐性を持つ相手（でんきタイプ等）には発動しない。

## 実装方針

- `ON_CHECK_CONTACT` で接触技受け時に30%確率のまひ付与を処理する。
- 既存の ほのおのからだ・どくのトゲ と同一パターン（発動率・状態異常の種類が異なるだけ）。
- まひ付与は `pokemon_state.apply_ailment()` 経由で実施し、耐性チェックは共通処理に委ねる。

- `src/jpoke/data/ability.py`

## テスト観点

1. 直接攻撃を受けたとき30%の確率で相手がまひ状態になる。
2. みがわり状態で防がれた場合は発動しない。
3. でんきタイプに対して（第六世代以降）は発動しない。
4. 連続攻撃技は1発ごとに発動判定がある。
5. かがくへんかガス発動中は発動しない。
