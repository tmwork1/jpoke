# すいすい 実装計画

## スコープ
- 仕様書: `spec/ability/すいすい.md`
- 優先度: H（天候連動速度補正）
- 対象: 雨時の素早さ2倍補正

## 実装方針
- `DomainEvent.ON_CALC_SPEED` にハンドラを登録し、`battle.weather.name == "あめ"` のときに2倍化する。
- 補正は `SpeedCalculator.calc_effective_speed` のイベントチェーンに乗せる。
- 他の速度補正（おいかぜ・持ち物等）と合成できる形を維持する。

## 実装ステップ
1. `src/jpoke/handlers/ability.py` に `すいすい` ハンドラを追加。
2. `src/jpoke/data/ability.py` の `すいすい` に `ON_CALC_SPEED` を登録。
3. `tests/test_ability.py` に発動/非発動テストを追加。

## テスト観点
- 雨で `calc_effective_speed` が2倍になる。
- 雨以外で速度が変化しない。
