# 特性実装計画: ノーマルスキン

更新日: 2026-05-03

## 仕様参照

- `spec/ability/ノーマルスキン.md`

## 実装ゴール

- 全ての技をノーマルタイプに変換する（わるあがきは実装側のタイプなし判定で対象外）。
- 元々ノーマルタイプ以外の技には威力を4915/4096倍にする。
- タイプ変換後はノーマルタイプ一致補正が適用される（ノーマルタイプのポケモンのみ）。

## 実装方針

- `ON_MODIFY_MOVE_TYPE` で技タイプをノーマルに変換し、元がノーマル以外の場合に威力倍率に4915/4096を適用する。
- スカイスキン・フェアリースキン・フリーズスキンと同一パターン（変換先=ノーマル、全技が対象）。
- わるあがきはタイプなし判定のため、わるあがき実装側で除外し、特性側で個別分岐は持たない。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. ノーマル以外の技がノーマルタイプに変換される。
2. 変換された技の威力が4915/4096倍になる。
3. 元々ノーマルタイプの技には威力補正がかからない。
4. わるあがきはわるあがき実装側の判定で変換対象外になる。
5. かがくへんかガス発動中は発動しない。
