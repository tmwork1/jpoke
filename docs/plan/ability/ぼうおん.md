# 特性実装計画: ぼうおん

更新日: 2026-05-03

## 仕様参照

- `spec/ability/ぼうおん.md`

## 実装ゴール

- 音の技を受けても無効化する。
- すてゼリフの攻撃・特攻低下効果と交代効果の両方を無効化する。
- かたやぶりの効果がある音技は防げない。

## 実装方針

- `ON_CHECK_IMMUNE` で `"sound" in move.data.flags` の技を無効化する。
- 音技の無効化は `MoveData.flags` の `"sound"` を基準に実装する。
- りんしょうの行動順繰り上げ・威力2倍効果は防げないことを注記する。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. おたけび等の音技を受けて無効化する。
2. すてゼリフの攻撃・特攻低下と交代効果が無効化される。
3. かたやぶりの音技には無視されてダメージを受ける。
4. ほろびのうたカウントが開始されない（バトンタッチで引き継いだカウントは継続）。
5. かがくへんかガス発動中は発動しない。
