# 特性実装計画: いしあたま

更新日: 2026-05-03

## 仕様参照

- `spec/ability/いしあたま.md`

## 実装ゴール

- 反動ダメージを伴う技（すてみタックル・フレアドライブ等）を使用しても反動ダメージを受けない。
- わるあがき・とびひざげり失敗・みがわりHP消費・いのちのたま等の反動には適用しない。

## 実装方針

- `modify_hp(reason="recoil")` の経路で `on_recoil` イベントを発火し、いしあたまで反動を無効化する。
- 反動判定は `on_recoil` に渡されるコンテキスト（反動の種別・発生源）で行い、技起因の反動だけを対象にする。
- 仕様書で対象外とされた反動（わるあがき・HP消費系等）はイベントコンテキストの種別で除外する。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. すてみタックル・フレアドライブ等の反動技を使用しても反動ダメージを受けない。
2. わるあがきの反動は通常通り受ける。
3. とびひざげり失敗時のダメージは通常通り受ける。
4. いのちのたまによるダメージは通常通り受ける。
5. かがくへんかガス発動中は反動が通常通り発生する。

## 保留・論点

- `modify_hp` の `reason` 設計を反動以外（天候・やけど等）と一貫化する。
