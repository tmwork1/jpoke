# 特性実装計画: てんのめぐみ

更新日: 2026-05-01

## 仕様参照

- `spec/ability/てんのめぐみ.md`

## 実装ゴール

- 追加効果の発生率を 2 倍（上限 100%）にする。
- 固定発動効果や本体効果には影響しない。

## 実装方針

- `Event.ON_MODIFY_SECONDARY_CHANCE` で追加効果確率のみ補正する。
- ちからずく と同じフックで処理し、競合順序を固定する。
- 多段技では各ヒットごとに確率判定を行う。

## テスト観点

1. 追加効果確率 2 倍。
2. 上限 100%。
3. 固定発動効果に影響しない。
4. 多段技で各ヒット判定。
5. ちからずく同時時の優先順が仕様どおり。

## 保留・論点

- 追加効果メタ情報を MoveData 側で統一管理する必要がある。
