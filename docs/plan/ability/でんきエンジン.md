# 特性実装計画: でんきエンジン

更新日: 2026-05-03

## 仕様参照

- `spec/ability/でんきエンジン.md`

## 実装ゴール

- でんきタイプの技を無効化し、自分のすばやさを1段階上げる。
- みがわり状態ででんき技を受けてもすばやさが上がる。
- かたやぶり効果の技には無視される。

## 実装方針

- `ON_CHECK_IMMUNE` 内ででんきタイプ技の無効化とすばやさ+1を連続処理する。
- ちくでん（HP回復）・ひらいしん（特攻+1）と同一パターンで、能力変化の種類だけ異なる実装。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. でんき技を受けて無効化し、すばやさが1段階上がる。
2. みがわり状態ででんき技を受けてもすばやさが上がる。
3. でんき連続技を受けてもすばやさ上昇は1段階のみ。
4. かたやぶりのでんき技には無視される。
5. かがくへんかガス発動中は発動しない。
