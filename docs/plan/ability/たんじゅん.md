# 特性実装計画: たんじゅん

更新日: 2026-05-02

## 仕様参照

- `spec/ability/たんじゅん.md`

## 実装ゴール

- 能力ランクの変化量（上昇・低下とも）が通常の2倍になる。
- 命中率・回避率のランクにも適用する。
- 急所ランクには適用しない。
- バトンタッチ・じこあんじ・パワースワップ等によるランク変動には適用しない。

## 実装方針

- `ON_BEFORE_MODIFY_STAT` または `ON_MODIFY_STAT` で変化量を2倍にする。
- 変化量が「技自身のランク変化」起因の場合のみ適用し、バトンタッチ引き継ぎ等は除外するフラグを参照する。
- かたやぶりの効果がある技では2倍化しない（`ON_CHECK_DEF_ABILITY_ENABLED` 経由で無効化）。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `src/jpoke/core/battle.py`（ランク変化のコンテキストにフラグが必要な場合）
- `tests/test_ability.py`

## テスト観点

1. りゅうのまい使用時にすばやさ・こうげきがそれぞれ2段階上がる。
2. ランク低下技を受けたとき変化量が2倍になる。
3. バトンタッチで引き継いだランクは2倍にならない。
4. かたやぶりの技に対しては2倍にならない。
5. 急所ランクには適用されない。

## 保留・論点

- ランク変化のコンテキスト（起因フラグ）が未整備な場合は `BattleContext` 拡張が必要。
