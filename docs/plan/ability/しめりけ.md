# 特性実装計画: しめりけ

更新日: 2026-05-03

## 仕様参照

- `spec/ability/しめりけ.md`

## 実装ゴール

- しめりけ所持ポケモンが場にいる間、全ポケモンの爆発技（じばく・だいばくはつ・ビックリヘッド・ミストバースト）を失敗させる。
- 相手の特性ゆうばくを無効化する。
- かたやぶり効果の技には無視される。

## 実装方針

- `ON_CHECK_MOVE` で爆発技（`"explosion" in move.data.flags`）を確認し、しめりけ所持ポケモンが場にいる場合に技を失敗させる。
- ゆうばく無効化は `ON_CHECK_ABILITY_ENABLED` でゆうばく特性を条件付き無効にする形で実装する。
- 場の管理は `battle.field` または `battle.side_field` を通じて確認する。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `src/jpoke/data/move.py`（`MoveData.flags` の `"explosion"` 整備）
- `tests/test_ability.py`

## テスト観点

1. じばく・だいばくはつ・ビックリヘッド・ミストバーストが失敗する。
2. 自分自身の爆発技も失敗する。
3. 爆発技のPPは消費される（失敗扱い）。
4. 相手のゆうばくが無効化される。
5. かたやぶりの技には無視されて爆発技が成功する。
