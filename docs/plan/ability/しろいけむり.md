# 特性実装計画: しろいけむり

更新日: 2026-05-02

## 仕様参照

- `spec/ability/しろいけむり.md`

## 実装ゴール

- 相手から能力ランクを下げられなくなる（全能力対象）。
- クリアボディと完全に同一のバトル効果を持つ。
- かたやぶりの効果がある技には無視される。

## 実装方針

- クリアボディと同一のハンドラ関数を登録するか、共通ヘルパー関数を呼び出す形で実装する。
- `ON_BEFORE_MODIFY_STAT` で変化量が負かつ原因が相手の場合に無効化する。
- かたやぶり判定は共通処理に委ねる。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`（クリアボディとのコード共有確認）
- `tests/test_ability.py`

## テスト観点

1. 相手の技・特性によるランク低下（いかく・おたけび等）を全能力で防ぐ。
2. 自己効果（からをやぶる等）によるランク低下は防げない。
3. かたやぶりの技には無視される。
4. クリアボディと同一の挙動になること。
