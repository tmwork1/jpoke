# 特性実装計画: ちくでん

更新日: 2026-05-03

## 仕様参照

- `spec/ability/ちくでん.md`

## 実装ゴール

- でんきタイプの技を無効化する（変化技含む）。
- HPが減っている場合は最大HPの1/4を回復する。
- かたやぶり効果の技には無視される。

## 実装方針

- `ON_CHECK_IMMUNE` 内ででんきタイプ技の無効化とHP1/4回復を連続処理する。
- 既存の もらいび・ちょすい・ひらいしん と同一パターン（対象タイプが異なるだけ）。
- でんきエンジン・ひらいしんとのコード共通化を検討する。

## 変更候補

- `src/jpoke/data/ability.py`
- `src/jpoke/handlers/ability.py`
- `tests/test_ability.py`

## テスト観点

1. でんき技を受けて無効化（HPが減っていれば1/4回復）する。
2. でんき変化技を受けても無効化する。
3. かたやぶりのでんき技には無視されてダメージを受ける。
4. HP満タン時は無効化のみでメッセージ（こうかがない旨）が出る。
5. かがくへんかガス発動中は発動しない。
