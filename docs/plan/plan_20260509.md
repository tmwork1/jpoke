'''
assert 5325 == t.calc_damage_modifier(battle_contact, Event.ON_CALC_POWER_MODIFIER)
'''
ではなく、
'''
battle.run_move(attacker, attacker.moves[0])
assert 5325 == battle.damage_calculator.damage_modifier
'''
のようにrun_move()したあとのデバッグ用変数を確認する。

すなわち、t.calc_damage_modifier()メソッドは使わない (定義削除済み)。
