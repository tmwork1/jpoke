# 計画書: アクアカッター

## 仕様要約（docs/spec/moves/アクアカッター.md 参照）

- みずタイプ / 物理 / 威力70 / 命中100 / PP20 / 対象: 相手1体 / 優先度0 / 接触なし
- 追加効果なし。急所ランク+1（急所に当たりやすい）。

## 既存実装との差分

既存実装（`data/moves/move_a.py` の `"アクアカッター"` エントリ）は仕様と一致しており、修正は不要。

1. **基本情報は仕様通り**
   `type="みず", category="physical", pp=20, power=70, accuracy=100` で一致。
   PPは `src/jpoke/data/ps-champ-ja/moves.json` の値（20）とも一致。

2. **急所ランクは `critical_rank=1` で仕様通り**
   追加効果を持たない技のため専用ハンドラは不要。`critical_rank` フィールドのみで
   `core/move_executor.py` の急所判定ロジック（`CRIT_RATES[critical_rank]`）が
   自動的に適用される。

3. **`flags` は `{"slash"}` で仕様通り**
   直接攻撃技ではない（`contact` フラグなし）が、「切る」技に分類される
   （きれあじ特性の威力1.5倍判定に影響するフラグ）。

対象外（`docs/progress/move.md` の「リーサル実装」列が `n/a`。追加効果を持たない通常の

## 注意点・エッジケース

- 急所ランクは通常のダメージ計算の急所判定に組み込まれるため、まもる等で防がれた場合は
  ダメージ自体が発生せず急所判定も行われない（既存の共通ロジックに準拠）。
- 追加効果を持たないため、ちからずくの影響を受けない（威力・効果とも変化しない）。
- `contact` フラグを持たないため、ゴツゴツメット等の接触時反動効果は発生しない。
