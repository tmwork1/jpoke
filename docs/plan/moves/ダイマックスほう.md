# 計画書: ダイマックスほう

## 仕様要約

- ドラゴンタイプ / 特殊 / 威力100 / 命中100 / PP5 / 対象: 1体選択 / 優先度0 / 接触なし
- 対象範囲判定: `src/jpoke/data/ps-champ-ja/moves.json` に記載が無いが、それのみでは対象範囲外と断定しない
  （docs/review/moves/スピードスター.md 前例）。Wikiに「SVで使用不可」相当タグが無く、
  第九世代向けの固有説明文とSVでの習得（レベルアップ56）が確認できるため対象範囲内と判断する。
  詳細根拠は `docs/spec/moves/ダイマックスほう.md` 参照。
- 「相手がダイマックスしていればダメージ2倍」の効果はダイマックス非対応の本プロジェクトでは対象外
  （前例: きょじゅうざん・きょじゅうだん）
- 追加効果なし（通常攻撃のみ）
- ものまねでコピー不可・まねっこ失敗・アンコール不可・ねごとで選ばれない
  （`non_copycat` / `non_encore` / `non_negoto` フラグで表現）
- さいはいで対象にできない（さいはい自体が本プロジェクトでは「ダブル専用」のため今回は対応不要）
- ゆびをふるで選ばれない（ゆびをふる自体が本プロジェクト未実装のため今回は対応不要）

## ハンドラ構成

ダメージ計算に関わる追加効果を持たないため、専用ハンドラは不要。
`handlers={}` のみで基本ダメージ計算に委ねる。
コピー不可・アンコール不可・ねごと対象外の挙動は
`flags={"non_copycat", "non_encore", "non_negoto"}` で表現する。
- `non_copycat` は `handlers/move_status.py` の `まねっこ_can_use` 相当処理が
  `Move(...).has_flag("non_copycat")` を見て失敗判定するため、専用ハンドラの追加は不要。
- `non_encore` は `handlers/move_status.py`（アンコール成立可否判定）が参照するため専用ハンドラ不要。
- `non_negoto` は `handlers/move_status.py`（ねごと候補選出）と
  `core/move_executor.py`（最終使用技記録）が参照するため専用ハンドラ不要。

## Priority 根拠

追加効果がなく専用ハンドラを実装しないため、priority の検討対象イベントは存在しない。

## 実装箇所

### data/moves/move_ta.py への更新

既存の `"ダイマックスほう"` エントリは以下の通り。

```python
    "ダイマックスほう": MoveData(
        type="ドラゴン",
        category="special",
        pp=5,
        power=100,
        accuracy=100,
        flags={"non_encore", "non_negoto"},
        handlers={},  # 追加効果なし
    ),
```

- type/category/pp/power/accuracy は仕様と一致（変更不要）。
- `flags` に `non_copycat` が欠落している。一次情報
  「この技に対する まねっこ・さいはい・アンコール は失敗する」の記述のうち、
  まねっこ分が未反映のため追加する。
- `target` 省略時のデフォルトは `"foe"`（相手1体）→ 仕様と一致。
- `priority` 省略時のデフォルトは `0` → 仕様と一致。
- `critical_rank` 省略時のデフォルトは `0`（通常の急所判定）→ 仕様と一致。
- 接触フラグ（`contact`）・弾技フラグ（`bullet`）はいずれも付与しない
  （一次情報で直接攻撃×、かつ名称に反し弾技でもないと明記されているため）。

## 注意点・エッジケース

- 追加効果がないため、まもる・おうじゃのしるし等の一般的な技判定はフレームワーク側の共通処理
  （`handlers/move_attack.py` の基本ダメージ計算経路）で自動的に処理され、本技固有のハンドラ実装は不要。
- ダイマックス関連の効果（相手ダイマックス時にダメージ2倍）は本プロジェクトが非対応のため実装しない。
- さいはい・ゆびをふる・ものまねは本プロジェクトで未実装/対象外のため、
  それらに対する本技の除外仕様は現時点で反映不要（将来実装時に本仕様書を参照すること）。
