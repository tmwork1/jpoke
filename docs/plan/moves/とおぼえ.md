`docs/spec/moves/とおぼえ.md` 参照。

## 実装箇所

### `src/jpoke/handlers/move_status.py`

五十音順の挿入位置: 「と」行 — `とおせんぼう_apply` の後、`とぐろをまく_modify_attacker_stats` の前。

```python
def とおぼえ_modify_attacker_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """とおぼえの効果: 自分のこうげきを1段階上げる（シングルバトルでは自分のみ）。"""
    return modify_attacker_stats(battle, ctx, value, stats={"atk": 1})
```

### `src/jpoke/data/moves/move_ta.py`

```python
"とおぼえ": MoveData(
    type="ノーマル",
    category="status",
    pp=20,
    target="self",
    flags={"sound"},
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.とおぼえ_modify_attacker_stats,
        ),
    }
),
```

## 注意点・エッジケース

- `target="self"` を必ず指定する（欠落すると `Move.is_blocked_by_protect` / `Move.is_reflectable` が `target == "foe"` を基準に判定するため、まもる・マジックコートに誤って反応してしまう）
- 対象が「味方全員」（第8世代以降）だがシングルバトルのみ対象のため、自分（attacker）に対する A+1 のみ実装する
- 音系の技ラベル（sound）はすでに `data/moves/move_ta.py` に設定済み。ぼうおんは自分に対しては無効化されない仕様のため、ハンドラ側での特別対応は不要
- `modify_attacker_stats` で A+1 を渡すと、A ランクが +6 の場合は `modify_stats` 内部の `clamp_stats` がガードするため、ハンドラ側での最大チェックは不要
