* タイプ: でんき / 分類: 特殊 / 威力: 100 / 命中率: 100 / PP: 5（ミライドン専用技。

## 既存実装の確認結果と問題点

`src/jpoke/handlers/move_attack.py` の `効果抜群時威力ブースト`（`Event.ON_CALC_POWER_MODIFIER`）が
`アクセルブレイク`・`イナズマドライブ`の両方に共有ハンドラとして登録されており、
`damage_calculator.calc_def_type_modifier(ctx)` が4096（等倍）を超える場合に
`apply_fixed_modifier(value, 5461)` で5461/4096倍を適用している。一次情報の
「正確な補正率は5461/4096倍」と一致しており問題なし。

`src/jpoke/data/moves/move_a.py` の `イナズマドライブ` エントリ
（`type="でんき"`, `category="special"`, `pp=5`, `power=100`, `accuracy=100`,
`flags={"contact"}`）も一次情報（直接攻撃○、追加効果×）と一致しており、
今回のレビューで修正すべき誤りは見つからなかった。
