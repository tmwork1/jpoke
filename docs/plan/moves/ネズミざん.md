# ネズミざん 実装計画書

## 仕様要約（docs/spec/moves/ネズミざん.md 参照）

* タイプ: ノーマル / 分類: 物理 / 威力: 20（全ヒット固定） / 命中: 90 / PP: 12
  （チャンピオンズはPP8/12/16/20の4段階に統一。第9世代本家基準は10）
* 対象: 相手1体 / 優先度: 0 / 接触あり / 切る技（きれあじ対象）
* 効果:
  * 1ターンに最大10回連続で攻撃する連続攻撃技（連続攻撃技の中で最多回数）。
    威力はヒットごとに変化せず常に20固定（トリプルアクセルとの違い）。
  * 命中判定は各ヒットごとに行う（`check_hit_each_time`）。外れた時点で終了。
  * スキルリンク／いかさまダイス所持時は初回ヒットのみ命中判定を行う。
  * 追加効果なし。

## 現状の実装状況

`src/jpoke/data/moves/move_na.py` に既に登録済み。

```python
"ネズミざん": MoveData(
    type="ノーマル",
    category="physical",
    pp=12,
    power=20,
    accuracy=90,
    flags={"contact", "slash", "check_hit_each_time"},
    multi_hit={
        "min": 10,
        "max": 10,
        "check_hit_each_time": True,
        "power_sequence": (),
    },
    handlers={},  # 追加効果なし
),
```

* `multi_hit.min == multi_hit.max == 10` のため、`_resolve_hit_count`
  （`core/move_executor.py`）は常に10を返す。
* `power_sequence == ()`（空）のため、`_resolve_hit_power` は常に基礎威力20を返す
  （トリプルアクセルのような威力上昇はない）。
* `flags` に `check_hit_each_time` を含み、`multi_hit.check_hit_each_time` も
  `True` のため、`ON_MODIFY_HIT_CHECK_EACH_TIME` が上書きされない限り
  ヒットごとに命中判定が行われる。
* スキルリンク（`data/ability.py`）・いかさまダイス（`data/item.py`）は
  `Event.ON_MODIFY_HIT_CHECK_EACH_TIME` に `False` を返すハンドラを登録しており、
  これらを持つ攻撃者は初回ヒットのみ命中判定される
  （`handlers/ability.py` の `スキルリンク_modify_hit_check_each_time`、
  `handlers/item.py` の `いかさまダイス_modify_hit_check_each_time`）。
* `flags` に `slash` を含むため、きれあじ（`handlers/ability.py` の
  `きれあじ_modify_power`）が威力1.5倍補正の対象として汎用判定する。

## ハンドラ構成

固有ハンドラなし（`handlers={}`）。連続攻撃・毎ヒット命中判定の挙動は
`core/move_executor.py` 側の汎用機構（`multi_hit`）で処理されるため、
`handlers/move_attack.py` への追加実装は不要
（`トリプルキック`・`トリプルアクセル` と同一パターン。ただし `power_sequence` が
空のため威力は変化しない）。

## Priority 根拠

該当なし（固有ハンドラを持たないため priority 検討対象外）。
`ON_MODIFY_HIT_COUNT` / `ON_MODIFY_HIT_CHECK_EACH_TIME` は `docs/spec/turn.md` 未掲載の
イベントだが、スキルリンク・いかさまダイスの既存ハンドラ（同イベント、priority未指定
=既定値）にすでに準拠済みであり、本技側で新規ハンドラを追加する必要はない。

## 注意点・エッジケース（review-test 向け）

1. **威力はヒットごとに変化しない**
   - `power_sequence == ()` のため、トリプルアクセルと異なり常に威力20で
     計算される。
2. **各ヒットごとに命中判定される**
   - 命中率0にすると1発目から外れ0回ヒットする。命中率100にすると必ず10回
     ヒットする（`test_ネズミざん_命中率0で全て外れる` / `test_ネズミざん_最大10回ヒットする`）。
3. **スキルリンク／いかさまダイス所持時は初回ヒットのみ判定**
   - 2発目以降で本来外れる状況を作っても、命中判定自体が行われず全10発命中する
     （`test_スキルリンク_毎ヒット命中判定技は初回のみ判定`、
     `test_いかさまダイス_毎ヒット命中判定技は初回のみ判定`、いずれも
     `parametrize` で本技を含む）。
4. **PPはチャンピオンズ基準の12**
   - 第9世代本家基準の10ではない点に注意（`src/jpoke/data/ps-champ-ja/moves.json` 参照）。
