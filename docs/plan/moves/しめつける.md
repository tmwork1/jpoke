# 実装計画: しめつける

## 仕様サマリ

- タイプ: ノーマル / 分類: 物理 / 威力: 15 / 命中率: 85 / PP: 20 / 接触: あり
- 効果: 命中すると相手を『バインド』状態にし、4〜5ターンの間、毎ターン終了後に最大HPの1/8のダメージを与え続ける
- バインド状態の詳細仕様は `docs/spec/volatiles/バインド.md` を参照
- まもる: ○ / マジックコート: ×（変化技ではない） / 追加効果: なし

## 対象イベント

`ON_DAMAGE_HIT` — ダメージ命中時に発火。既存のバインド系技（うずしお・まきつく・ほのおのうず・
からではさむ・すなじごく・まとわりつく・トラバサミ・サンダープリズン・マグマストーム）と共通の
`apply_bind_to_defender` ハンドラを流用する。

## Priority

`docs/spec/turn.md` に `ON_DAMAGE_HIT` の記載なし。
既存のバインド系技がすべて `MoveHandler(apply_bind_to_defender)` のみ（priority=100デフォルト）
で実装されているため、同様にデフォルト priority を採用する。

## 実装状況（実装済み）

### handlers/move_attack.py

`apply_bind_to_defender` を流用（新規実装不要）。

### data/move.py

```python
"しめつける": MoveData(
    type="ノーマル",
    category="physical",
    pp=20,
    power=15,
    accuracy=85,
    flags={"contact"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(ha.apply_bind_to_defender)
    },
    lethal_handlers={
        LethalEvent.ON_HIT: LethalHandler(l._apply_bind)
    }
),
```

## リーサル計算

`_apply_bind`（`handlers/lethal.py`）を流用（新規実装不要）。バインドの毎ターンダメージは
バインド揮発性状態の `ON_TURN_END` リーサルハンドラ側で処理される。

## テスト方針

- 命中時に相手がバインド揮発状態になる確認
- バインド中は相手が毎ターン最大HPの1/8ダメージを受ける確認
- バインド中は相手が交代できない確認
- 攻撃者が交代するとバインドが解除される確認
- （`うずしお`・`まきつく` の既存テストと同パターン）

## 注意事項

- ゴーストタイプの相手にはノーマルタイプの技が無効（0倍）のため、しめつけるはそもそも命中せず
  バインドも付与されない。`_check_hit_by_type`（`src/jpoke/core/move_executor.py`）による
  共通のタイプ相性判定で自然に成立するため、実装上の追加対応は不要。
- かがくへんかガス/かたやぶり無効化チェックは書かない
