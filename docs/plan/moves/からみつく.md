# からみつく 実装計画書

## 仕様要約（docs/spec/moves/からみつく.md 参照）

* タイプ: ノーマル / 分類: 物理 / 威力: 10 / 命中: 100 / PP: 35
* 対象: 相手1体 / 優先度: 0 / 接触あり
* 効果: 追加効果として10%の確率で相手の『すばやさ』ランクを1段階下げる。
* `src/jpoke/data/ps-champ-ja/moves.json` に本技は未収録（現行ポケモンチャンピオンズでは習得不可）だが、
  他の未使用技と同様、汎用のダメージ技として実装する。

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 10%の確率で相手の『すばやさ』ランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・修正不要）:

```python
def からみつく_lower_defender_spd(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"spe": -1}, chance=0.1)
```

`modify_defender_stats`（`handlers/move.py`）が `battle.resolve_secondary_chance` を経由するため、
ちからずくによる追加効果無効化（確率0化）は汎用的に処理される。本技側に個別のガードは不要。
同種の実装として `あわ_lower_defender_spd`（`handlers/move_attack.py`）が全く同じパターンを使用している。

## Priority 根拠

ダメージを与えた技の追加効果（能力ランク低下）は既存の同種ハンドラ（あわ・がんせきふうじ 等）と同じ
`ON_DAMAGE_HIT` / priority 100（デフォルト）のパターンに従う。`docs/spec/turn.md` にも
`ON_DAMAGE_HIT` は他の追加効果技と共通のタイミングとして扱われており、個別の priority 検討は不要。

## 注意点・エッジケース（review-test 向け）

1. **ちからずくとの相互作用**
   - `flags={"secondary_effect"}` があるため、ちからずく所持時は `resolve_secondary_chance` が確率を0にし、
     すばやさ低下が発動しなくなる（威力1.3倍化は `ちからずく_boost` 側で処理）。
2. **テスト方針**
   - `secondary_chance` を0/1に固定し、すばやさランク変化の有無を確認するテストを
     `tests/moves_attack/test_move_ka.py` に追加する（`あわ` の既存テストと同じパターン）。
