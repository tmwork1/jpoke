# 計画書: ハバネロエキス

## 仕様要約

- くさタイプ / 変化 / 命中100 / PP15 / 対象: 1体選択
- 相手の『こうげき』ランクを2段階上げる
- 相手の『ぼうぎょ』ランクを2段階下げる
- ランク上げとランク下げを同時に行う点で `いばる`（こうげき+2 + こんらん）の変形

## Priority 根拠

`docs/spec/turn.md` の `ON_STATUS_HIT` セクションにハバネロエキスの明示的な記載はない。ただし同種の変化技（いばる・おだてる・くすぐる・フェザーダンス等）はすべて `ON_STATUS_HIT` priority 100 のデフォルト値を使用しており、本技も同パターンを採用する。

ランク補正が上限または下限に達している場合の失敗判定は `ON_APPLY_MOVE` priority 100（`ランク補正上げ: すでに最大` / `ランク補正下げ: すでに最低`）でフレームワーク側が処理する。具体的には、こうげき+6かつぼうぎょ-6の場合のみ技が失敗する。

## 実装箇所

### handlers/move_status.py への追加

五十音順の挿入位置: `はらだいこ` 関連の後、`ハロウィン` の前（は行内、五十音順で「ハバネロ」<「ハロウィン」）。

```python
def ハバネロエキス_apply(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """ハバネロエキスの効果: 相手のこうげきを2段階上げ、ぼうぎょを2段階下げる。"""
    battle.modify_stats(ctx.defender, {"A": 2}, source=ctx.attacker)
    battle.modify_stats(ctx.defender, {"B": -2}, source=ctx.attacker)
    return HandlerReturn(value=value)
```

## 注意点・エッジケース

- こうげきとぼうぎょのランク変化は独立して処理される。こうげきが+6でも、ぼうぎょが-6でなければ技は成功し、適用可能な方のみ変化する。こうげき+6かつぼうぎょ-6の場合にのみフレームワーク側の `ON_APPLY_MOVE` で失敗になる
- `modify_stats` は個別にランク境界チェックを行うため、両方に対して個別に呼ぶパターン（いばると同様のアプローチ）で実装する
- みがわり状態の相手に対するランク変化はフレームワーク側で無効化されるため、ハンドラ内の追加ガードは不要
- しろいきり状態はランク低下（ぼうぎょ-2）を防ぐが、ランク上昇（こうげき+2）はみがわり・しろいきりに影響されない。この処理はランク変化ハンドラ（`modify_stats` 内）で行われるため、本ハンドラでの追加対応は不要
- クリアボディ・しろいけむり等によるランク低下無効化も `modify_stats` 内で処理されるため、ハンドラ内ガード不要
