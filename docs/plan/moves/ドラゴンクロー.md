# 計画書: ドラゴンクロー

## 仕様要約（docs/spec/moves/ドラゴンクロー.md 参照）

- ドラゴンタイプ / 物理 / 威力80 / 命中100 / PP16（チャンピオンズ、`move_list.txt` で確認済み） /
  対象: 相手1体 / 優先度0 / 直接攻撃あり（contact）
- 急所ランク+1（追加効果なし）
- 切る技（slashフラグ）。特性きれあじで威力1.5倍。
- リーサル計算: 対象外（`docs/progress/move.md` の「リーサル実装」列が `n/a`）

## 既存実装との差分

既存実装（`data/moves/move_ta.py` の `"ドラゴンクロー"` エントリ）には以下の不備がある。

1. **`critical_rank=1` が欠落**
   一次情報（Wiki「技の仕様」節・LEGENDS アルセウス節）に「急所に当たりやすい」と明記されており、
   `docs/progress/move.md` の他の急所技（あくうせつだん等）と同様に `critical_rank=1` を
   `MoveData` に設定する必要がある。既存実装は `flags={"contact"}` のみで急所ランク補正が
   欠落していたバグ。`critical_rank=1` を追加する。

2. **`flags` に `slash` が欠落**
   一次情報（Wiki「技の仕様」節）に「Championsでは切る技の1つ。特性きれあじのポケモンが使うと
   威力が1.5倍になる」と明記されている（本家SVでは切る技非対象だが、Championsでは対象という
   世代差がWikiに明記されている）。`docs/champions/changes_from_sv.md` にも
   「ドラゴンクロー　切る技(切り技)に追加」と明記されている。`flags={"contact"}` のみでは
   `きれあじ_modify_power`（`handlers/ability.py`）の `ctx.move.has_flag("slash")` 判定が
   通らず、きれあじの威力補正が発動しない。`flags={"contact", "slash"}` に修正する。

3. **PPは16で既に正しい**
   `docs/champions/move_list.txt` に `ドラゴンクロー	ドラゴン	物理	80	100	16	接○	守○` と
   明記されており、実装の `pp=16` は既にチャンピオンズ基準で正しい。

## ハンドラ構成

追加効果を持たない技のため、新規ハンドラの実装は不要。`critical_rank` はデータ属性のみで
完結する。

### data/moves/move_ta.py（修正）

```python
"ドラゴンクロー": MoveData(
    type="ドラゴン",
    category="physical",
    pp=16,
    power=80,
    accuracy=100,
    critical_rank=1,
    flags={"contact", "slash"},
    handlers={},  # 追加効果なし
),
```

## リーサル計算

対象外（`docs/progress/move.md` の「リーサル実装」列が `n/a`。追加効果を持たない通常の
物理攻撃技のためリーサル計算ハンドラは不要）。

## 注意点・エッジケース

- `slash` フラグ追加により、きれあじ特性のポケモンが使用した場合のみ別枠で威力1.5倍が乗る
  （`apply_fixed_modifier` による固定小数点乗算、`power_modifier` に加算的に反映）。
- `critical_rank=1` によって急所発生率が通常より1段階高くなる（第6世代以降の段階表: 0→1/24,
  1→1/8）。ダメージ倍率自体（急所時1.5倍）には影響しない。
