# 実装計画書: シャドーパンチ

参照仕様書: docs/spec/moves/シャドーパンチ.md

## 実装方針

シャドーパンチは追加効果を持たない攻撃技であり、以下のデータのみで挙動が完結する。
専用ハンドラは不要。

* `type="ゴースト"`
* `category="physical"`
* `pp=20`
* `power=60`
* `accuracy=None`（必中。既存の `あてみなげ` `スピードスター` `でんげきは` などと同様、
  `accuracy=None` を core（`move_executor.py`）が必中として処理するため追加実装は不要）
* `flags={"contact", "punch"}`（直接攻撃・パンチ技。パンチ技フラグは `てつのこぶし` や
  `パンチグローブ` の対象判定に既存の共通ロジックが利用するため、フラグ付与のみで良い）

## 登録

`src/jpoke/data/move.py`（実体は `src/jpoke/data/moves/move_sa.py`）の `MOVES["シャドーパンチ"]` に
上記の値を設定する。`handlers={}` とする（追加効果なし）。

## テスト方針

`tests/moves_attack/` に以下を追加する。

* 命中率・回避率を無視して必ず命中することを確認するテスト
  （相手の回避率を上げても命中する 等）
* タイプ・威力・分類（ゴースト・物理・60）が正しく反映されることを確認するテスト
