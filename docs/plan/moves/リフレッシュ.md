# リフレッシュ 実装計画書

## 結論: 実装しない（対象範囲外）

`docs/spec/moves/リフレッシュ.md` の通り、本技はソード・シールド以降（BDSP・SVを含む）で
使用不可能（習得可能なポケモンが存在しない）であり、`docs/champions/move_list.txt` にも
項目が存在しない。README.md の対象範囲定義（「第9世代で実装されていない技・アイテム・特性・
ポケモンなどは実装しない」）に該当するため実装しない。

## 削除した実装

過去のレビューで対象範囲確認なしに登録されていた実装を削除した。加えて、旧実装には
回復対象の誤り（こおり・ねむりまで回復してしまう）もあったが、対象範囲外のため修正ではなく
削除で対応した。

- `src/jpoke/data/moves/move_ra.py`: `"リフレッシュ": MoveData(...)` エントリを削除
- `src/jpoke/handlers/move_status.py`: `リフレッシュ_cure_ailment` ハンドラを削除
- `src/jpoke/types/move.py`: `MoveName` Literal から `"リフレッシュ"` を削除
  （`scripts/generate_literals/generate_move_literal.py` で再生成）
- `tests/moves_status/test_move_ra.py`: `test_リフレッシュ_まひを治す` /
  `test_リフレッシュ_やけどを治す` / `test_リフレッシュ_状態異常がなければ失敗する` を削除
- `docs/progress/move.md`: リフレッシュ行を削除（`MOVES` 辞書に存在しない技のため）

## リーサル計算

対象外（実装しないため。元々 `handlers/lethal.py` への登録もなかった）。
