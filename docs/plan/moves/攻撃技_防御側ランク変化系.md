## 仕様要約

`docs/spec/moves/_move_secondary_1.md` の `対象=defender` 行のうち、変化列（A/B/C/D/S/ACC）に値を持つものが対象。
命中時に防御側の能力ランクを下げる追加効果を実装する。

主な効果一覧（本バッチ対象のみ）：

- A-1: オーロラビーム(10%), じゃれつく(10%), はるのあらし(30%), うらみつらみ(100%)
- B-1: かみくだく(20%), シャドーボーン(20%), Gのちから(100%), ほのおのムチ(100%)
- B-1 バグ修正: はいよるいちげき（B-1 → C-1 に修正）
- C-1: ムーンフォース(30%)
- D-1: きあいだま(10%), だいちのちから(10%), ようかいえき(10%), りんごさん(100%)
- D-2: シードフレア(40%)
- S-1: あわ(10%), からみつく(10%), こがらしあらし(30%), バブルこうせん(10%)
- ACC-1: オクタンほう(50%), グラスミキサー(50%), だくりゅう(30%), どろかけ(100%), どろばくだん(30%), ナイトバースト(40%), ミラーショット(30%)
- でんじほう バグ修正: chance=0.3 → 1.0（仕様は確率=1、100%）

## Priority 根拠

`docs/spec/turn.md` の `Event.ON_HIT` セクション（474行目付近）：

```
| 10 | 自分のランクを上げる追加効果 |
| 10 | 自分のランクが下がる反動 |
| 20 | HP吸収技による回復 |
...
| 100 | ダイマックスわざの効果 |
```

`ON_HIT` は同セクションに「追加効果（特殊なもの除く）」が Priority 10 で記載されているが、この記載は攻撃側自身のランク変化（自分のランクを上げる追加効果）を指す。
防御側ランク変化は `Event.ON_DAMAGE` セクションの Priority 10「追加効果（特殊なもの除く）」に含まれるが、
既存の実装（アクアブレイク、シャドーボール等）はすべて `Event.ON_HIT priority=100` で登録されている。タスク指示も `Event.ON_HIT priority=100` と明示しているため、これに従う。

MoveHandler のデフォルト priority は 100 であるため、明示的な priority 指定は不要。

## 注意点・エッジケース（review-test 向け）

### 確率発動

- `battle.resolve_secondary_chance(ctx, chance)` は `ちからずく` や `てんのめぐみ` などの特性補正を自動適用する
- 10%, 20%, 30%, 40%, 50% 確率の技はランダム判定が入るためテストでは `accuracy=100` 等と同様の固定シードまたはパッチが必要

### はいよるいちげき バグ修正

- 仕様書では `C-1`（特攻-1）だが、現実装は `B-1`（防御-1）になっている
- `handlers/move_attack.py` の `はいよるいちげき_modify_defender_stats` を `stats={"C": -1}` に変更する
- `data/move.py` のハンドラ登録は変更不要（ハンドラ名は同一）

### でんじほう バグ修正

- 仕様書では確率=1（100%）だが、現実装は `chance=0.3`
- `handlers/move_attack.py` の `でんじほう_apply_ailment_to_defender` から `chance=0.3` 引数を削除する
- 既存テストが 30% 前提で書かれている場合は修正が必要

### ACC ランク変化

- `stats={"ACC": -1}` のように命中ランクを操作する場合、`battle.modify_stats` が ACC をサポートしている前提
- 既存の `するどいめ` 特性（ACC 低下無効）などは `modify_stats` 内部で処理されるため、ハンドラ側に追加ガード不要

### みがわり貫通

- `ON_HIT` はみがわりへの被弾時も発火される（turn.md「ダメージ発生後の処理（みがわりに被弾しても発動）」）
- ただし `battle.modify_stats` 内部でみがわりによる能力低下無効を処理するため、ハンドラ側ガード不要

### シャドーボーン・あわ・からみつく・オクタンほう・グラスミキサー・どろばくだん・ミラーショット

- move.py に MoveData が存在しない → 新規 MoveData を追加する
- 基本データ（type/category/pp/power/accuracy）は `_move_secondary_1.md` の仕様書外の情報が必要なため、
  以下の参考値を使用している（ポケモンwiki準拠）：
  - あわ: みず/特殊/PP30/威力40/命中100
  - からみつく: むし/特殊/PP20/威力55/命中100
  - グラスミキサー: くさ/特殊/PP10/威力60/命中55
  - シャドーボーン: ゴースト/物理/PP10/威力85/命中100、contact
  - どろばくだん: じめん/特殊/PP10/威力65/命中85
  - オクタンほう: みず/特殊/PP10/威力110/命中85
  - ミラーショット: はがね/特殊/PP10/威力65/命中85
