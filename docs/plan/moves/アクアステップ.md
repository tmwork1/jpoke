# アクアステップ 実装計画

参照: docs/spec/moves/アクアステップ.md

## 現状（レビュー時点）

`src/jpoke/data/moves/move_a.py` に `MoveData` は登録済みで、仕様書の内容を満たしている。

* `pp=12`（champions版の正しい値。`src/jpoke/data/ps-champ-ja/moves.json` 24行目と一致）。
* `flags={"contact", "dance", "secondary_effect"}`（直接攻撃・おどりの技・ちからずく対象の
  追加効果、いずれも一次情報と一致）。
* `Event.ON_HIT: h.MoveHandler(ha.アクアステップ_boost_attacker_spe)` が登録されており、
  追加効果（自分のすばやさ+1、100%）を処理する。

`src/jpoke/handlers/move_attack.py` の `アクアステップ_boost_attacker_spe` も
`オーラぐるま_boost_attacker_spe` / `くさわけ_boost_attacker_spe` /
`ニトロチャージ_boost_attacker_spe` と全く同じパターンで実装済み。

```python
def アクアステップ_boost_attacker_spe(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_attacker_stats(battle, ctx, value, stats={"spe": 1})
```

修正すべき誤り・欠落は見つからなかった（対応内容なし）。

## ハンドラ構成

| 技名 | イベント | priority | subject_spec | 役割 |
|---|---|---|---|---|
| アクアステップ | ON_HIT | 100（デフォルト） | attacker:self | S+1（100%、ちからずく対象） |

## Priority 根拠

* `Event.ON_HIT` は `docs/spec/turn.md` の該当セクションに「自分のランクを上げる追加効果」が
  priority=10 と記載されている。ただし既存の同種実装（`オーラウイング_boost_attacker_spe`,
  `オーラぐるま_boost_attacker_spe`, `くさわけ_boost_attacker_spe`,
  `こうそくスピン_boost_attacker_spe`, `ニトロチャージ_boost_attacker_spe` 等）はいずれも
  `MoveHandler` のデフォルト priority（未指定＝100）で登録されており、turn.md の priority=10 は
  ON_HIT イベント内での処理グループの優先度を示すものであるため、既存パターンに合わせて
  デフォルト priority のまま登録する（変更不要）。

## ちからずくとの相互作用

* `flags={"secondary_effect"}` が設定されているため、既存の汎用処理
  `ちからずく_boost`（ON_CALC_POWER_MODIFIER, 威力1.3倍）と
  `ちからずく_disable_secondary_effect`（追加効果確率を0にする）が自動的に適用される
  （`tests/abilities/test_ability_ta.py` の `test_ちからずく_追加効果が発動せず威力が1_3倍` で
  検証済み）。

## おやこあいとの相互作用

* `おやこあい`（連続攻撃化）を持つポケモンがアクアステップを使用すると、`Event.ON_HIT` が
  ヒットごとに発火するため、すばやさ+1の追加効果も2回発動する
  （`tests/abilities/test_ability_a.py` の `test_おやこあい_単発攻撃が2ヒットする` で
  すばやさランクが+2になることを検証済み）。既存の一般的な `おやこあい` 実装に依存しており、
  本技側での追加対応は不要。

## リーサル計算

`docs/progress/move.md` の該当行「リーサル実装」列は `n/a`。
自分のすばやさランクを上げる効果であり、相手へのダメージ確定計算（HP分布）には
一切影響しないため対象外。

## テスト

`tests/moves_attack/test_move_a.py` に以下が実装済み。

* 命中時、相手にダメージを与えること。
* 命中時、自分の『すばやさ』ランクが1段階上がること（100%）。

`tests/abilities/test_ability_ta.py` / `tests/abilities/test_ability_a.py` に、
ちからずく・おやこあいとの相互作用テストが既に存在する（本技を題材にした特性側のテスト）。

不足は見つからなかった（追加なし）。
