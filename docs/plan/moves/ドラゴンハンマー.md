# 実装計画書: ドラゴンハンマー

参照仕様書: docs/spec/moves/ドラゴンハンマー.md

## 実装方針

ドラゴンハンマーは追加効果を持たない攻撃技であり、以下のデータのみで挙動が完結する。
専用ハンドラは不要。

* `type="ドラゴン"`
* `category="physical"`
* `pp=16`（Champions基準。docs/spec/moves/ドラゴンハンマー.md の PP 変換根拠を参照）
* `power=90`
* `accuracy=100`
* `flags={"contact"}`（直接攻撃。「切る技」ではないため `"slash"` フラグは付与しない）

## 登録

`src/jpoke/data/moves/move_ta.py` の `MOVES_TA["ドラゴンハンマー"]` に上記の値を設定する。
`handlers={}` とする（追加効果なし）。

## リーサル計算

追加効果を持たない通常攻撃技のため、`handlers/lethal.py` への専用ハンドラ追加は不要
（docs/progress/move.md のリーサル実装列は n/a）。

## テスト方針

`tests/moves_attack/test_move_ta.py` に以下を追加する。

  （`test_ドラゴンクロー_相手にダメージを与える` と同様のパターン）
