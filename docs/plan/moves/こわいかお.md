## 実装箇所

### handlers/move_status.py への追加（実装済み）

挿入位置: 五十音順で `こらえる_apply` の後、`サイコフィールド_activate_terrain` の前（こ の並び）

```python
def こわいかお_modify_defender_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """こわいかおの効果: 相手のすばやさを 2 段階下げる。"""
    return modify_defender_stats(battle, ctx, value, stats={"spe": -2})
```

### data/moves/move_ka.py の登録（実装済み・PPのみ修正）

`こわいかお` の MoveData の `pp` を 10 → 12 に修正する（他の項目は変更なし）。

## リーサル計算

対象外（ダメージを与えない変化技のため `docs/progress/move.md` のリーサル実装は n/a）。

## 注意点・エッジケース

- すばやさ単体のランクを2段階下げるだけの単純な効果。`modify_defender_stats` の `stats` 引数に
  `{"spe": -2}` を渡せばよい
- ランク低下の無効化（しろいきり、クリアボディ等）はフレームワーク内で処理されるため、ハンドラ側のガードは不要
- `みがわり` 状態の相手には命中しない（フレームワーク側で処理済み）
