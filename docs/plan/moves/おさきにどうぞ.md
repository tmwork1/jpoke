# 実装計画書: おさきにどうぞ

## 仕様要約（docs/spec/moves/おさきにどうぞ.md 参照）

* タイプ: ノーマル / カテゴリ: 変化 / PP: 16 / 命中: -（必中） / 対象: 相手1体
* 優先度: 0 / 直接攻撃: ×
* 対象を、すばやさ・優先度・特性「あとだし」/「きんしのちから」・持ち物「こうこうのしっぽ」/
  「まんぷくおこう」・「さきおくり」の効果を無視して使用者の直後に行動させる（ダブル/トリプル向けの主効果）
* 対象がそのターンにすでに行動済み（さいはい・おどりこによる行動を含む）の場合は失敗する
* **シングルバトルでは「使用者の直後に割り込ませる」効果は実質無意味**
  （残りの行動者は相手1体のみで、対象が未行動なら次に行動するのは元々対象自身であり、
  行動済みなら本技は失敗するだけで、いずれにせよ観測可能な行動順の変化が発生しない）

## 実装しない判断

* `docs/spec/turn.md` の `Event.ON_APPLY_MOVE` priority 100 に「おさきにどうぞ: 失敗条件」が
  記載されているが、この失敗条件（対象がすでに行動済みか）をシングルバトル限定で実装しても、
  成功時に行う「行動順を割り込ませる」処理自体が対象外（未実装）のため、観測可能な効果は一切生まれない。
* 類似のダブル/トリプル専有技（このゆびとまれ・いかりのこな・てだすけ・さきおくり等）も同様の理由で
  `docs/progress/move.md` 上「ダブル専用」＝対象外として扱われており、本技のみ失敗条件だけを
  実装すると一貫性を欠く。
* 以上より、本技は `handlers={}`（追加効果なし）のまま据え置き、行動順操作・失敗条件チェックの
  いずれも実装しない。

## 注意点・エッジケース

### PP

* `src/jpoke/data/ps-champ-ja/moves.json`（L146）の値は16。Gen9本家基準（15）ではなく
  チャンピオンズ基準（16）を正とする。

### かがくへんかガス・かたやぶり

* おさきにどうぞはランク変化・状態異常を付与しないため、かがくへんかガス・かたやぶりの
  影響を受けない。ハンドラ側でのチェックも不要。

### さきおくりとの関係

* さきおくり（`docs/plan/moves/さきおくり.md`）は「行動順を最後にする」、おさきにどうぞは
  「行動順を最初（使用者の直後）にする」という逆の効果。どちらもシングルバトルでは
  行動順操作自体が無意味であり、本プロジェクトでは両技とも同様に扱う。

## テスト

* シングルバトルで観測可能な効果がないため、専用テストは追加しない
  （このゆびとまれ・いかりのこな・さきおくり等、他の全ダブル専用技にも専用テストは存在しない）。

## リーサル計算

* 対象外（ダメージを与えず、HP分布にも影響しない）。
