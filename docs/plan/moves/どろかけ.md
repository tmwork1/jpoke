# 計画書: どろかけ

## 仕様要約（docs/spec/moves/どろかけ.md 参照）

- じめんタイプ / 特殊 / 威力20 / 命中100 / PP12 / 対象: 1体選択 / 優先度0 / 接触なし
- 追加効果として、100%の確率で相手の命中率を1段階下げる。

## 既存実装との整合確認

### handlers/move_attack.py: `どろかけ_lower_acc`（既存）

```python
def どろかけ_lower_acc(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"accuracy": -1})
```

- `modify_defender_stats` ヘルパーは `chance` 省略時デフォルト1（100%）で、内部で
  `battle.resolve_secondary_chance` を経由するため、「追加効果」としてちからずく・
  おうじゃのしるし等の影響を正しく受ける（仕様書の「追加効果: あり」と一致）。
- 同系統技のどろばくだん（30%）・ナイトバースト（40%）と同一ヘルパーを使用しており、
  実装パターンとして問題なし。
- イベント: `Event.ON_DAMAGE_HIT`。同種の能力ランク低下追加効果技（どろばくだん・
  ナイトバースト等）がすべて `Event.ON_DAMAGE_HIT` を使用しており、`docs/spec/turn.md`
  に本イベントの記載はないが（ダメージ計算内部処理のため未掲載）、既存の同種ハンドラとの
  整合を優先し、priority省略時のデフォルトのままとする。
- subject_spec: `MoveHandler` デフォルト（defender側への効果、既存パターンと同一）。

### data/move.py（要修正）

修正前:

```python
    "どろかけ": MoveData(
        type="じめん",
        category="special",
        pp=10,
        power=20,
        accuracy=100,
        flags={"secondary_effect"},
        handlers={
            Event.ON_DAMAGE_HIT: h.MoveHandler(
                ha.どろかけ_lower_acc,
            )
        }
    ),
```

- `pp=10` はポケモンWikiに記載の本家シリーズ値であり、`docs/champions/move_list.txt`
  （PP=12）と `docs/progress/move.md` のどろかけ行（PP=12）に一致していない。
  本プロジェクトはポケモンチャンピオンズを対象とするため、`pp=12` に修正する。
- `flags={"secondary_effect"}` は「追加効果あり」の仕様と一致（直接攻撃ではないため
  `contact` は不要で正しい）。
- `target` 省略時のデフォルトは `"foe"`（相手1体）→ 仕様と一致。
- `priority` 省略時のデフォルトは `0` → 仕様と一致。
- `critical_rank` 省略時のデフォルトは `0`（通常の急所判定）→ 仕様と一致。

修正後は `pp=10` を `pp=12` に変更するのみで、handlers 側の変更は不要。

## リーサル計算

対象外（`docs/progress/move.md` の「どろかけ」行はリーサル実装/リーサルテストともに n/a）。
命中率を下げる効果は相手の命中判定にのみ影響し、確定数計算（HP分布）には影響しないため。

## 注意点・エッジケース

- 命中率ランク低下は「相手の命中率」であり、自分の回避率ではない点に注意
  （`modify_defender_stats` で `stats={"accuracy": -1}` を defender に適用しており、正しい）。
