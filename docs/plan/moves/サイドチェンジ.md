# 実装計画書: サイドチェンジ

## 仕様要約（docs/spec/moves/サイドチェンジ.md 参照）

* タイプ: エスパー / カテゴリ: 変化 / PP: 16 / 命中: -（必中） / 対象: 自分
* 優先度: +2 / 直接攻撃: ×
* 自分と味方1体の位置を入れ替える（ダブル/トリプル向けの主効果）。
* 第九世代では連続使用すると成功率が約1/3ずつ低下していく（まもる系とは別カウンタ）。
* **シングルバトルでは常に失敗する**（ポケモンWiki明記の通り、入れ替え先の味方が存在しないため）。

## 実装しない判断

* 位置交換という本技の効果は、シングルバトルでは対象となる「入れ替え先の味方」が存在しないため、
  常に不発となる。この失敗自体は明確な仕様だが、`handlers={}`（何もしない）と実装上の挙動が
  完全に一致するため、失敗処理をわざわざ実装する意味がない。
* 類似のダブル/トリプル専有技（てだすけ・コーチング・さきおくり・このゆびとまれ・いかりのこな・
  アロマミスト等）も同様の理由で `docs/progress/move.md` 上「ダブル専用」＝対象外として扱われて
  おり、本技のみ位置交換処理や連続使用成功率低下カウンタを実装すると一貫性を欠く。
* 以上より、本技は `handlers={}`（追加効果なし）のまま据え置き、位置交換処理・連続使用成功率
  低下カウンタ・Zワザ付加効果（自分のすばやさを2段階上げる）のいずれも実装しない。

## 現状の実装状況（本レビューで修正）

| 項目 | 状態 |
|------|------|
| `data/moves/move_sa.py` エントリ (`"サイドチェンジ"`) | 登録済み（`type="エスパー"`, `category="status"`, `pp=16`, `priority=2`, `target="self"`, `handlers={}`） |
| `handlers/move_status.py` 関数 | なし（追加効果なしのため不要） |

### 本レビューで発見・修正したバグ

1. **PP誤り**: 旧実装は `pp=15`（ポケモンWiki記載の第五世代基準値）だったが、
   `src/jpoke/data/ps-champ-ja/moves.json`（L306）は `16` である。PPはchampionsのmove_list.txtを
   正とするため、`pp=16` に修正した。
2. **`target` 未指定によるバグ**: 旧実装は `target` を指定しておらず、`MoveData` のデフォルト値
   `"foe"` が適用されていた。しかしサイドチェンジは「自分が対象の技」であり、本来 `target="self"`
   であるべき。`target` 未指定のままだと、以下の判定が誤作動する可能性があった:
   * `Move.is_blocked_by_protect`（`self.target == "foe"` で判定）: 誤って `True` になり得た
     （実際にはこの技はまもるの影響を受けない自分対象技のため、常に `False` であるべき）。
   * `Move.is_reflectable`（`self.target in ("foe", "foe_side")` で判定）: 誤って `True` になり得た
     （実際にはマジックコート/マジックミラーの対象外であるべき）。
   * `いたずらごころ_blocked_by_dark`（`ctx.move.category == "status" and ctx.move.target == "foe"
     and ctx.defender.has_type("あく")` で判定）: いたずらごころ持ちが本技を使用した際、
     相手があくタイプであれば誤ってブロックされてしまう可能性があった
     （本技は相手を対象にしないため、あくタイプ相手への無効化は本来発生しないはず）。
   * 本技は `handlers={}` でシングルバトルでは常に無効果のため上記いずれも観測可能な副作用は
     出ないが、`target="self"` を明示することで論理的な正しさを保証し、`アロマミスト`
     （`docs/spec/moves/アロマミスト.md`）と同じ扱いに統一した。

`target="self"` はゲーム内の本来のターゲット（自分。ただし効果自体は自分と味方の位置交換という
副次的なダブル専有効果）と一致する。実際に `handlers={}` のため、使用者自身に何らかの効果が
及ぶことはない。

## 注意点・エッジケース

### かがくへんかガス・かたやぶり

* サイドチェンジはダメージを与えず、位置交換のみを行う技のため、かがくへんかガス・かたやぶりの
  影響を受けない。ハンドラ側でのチェックも不要。

### 進捗表の修正

* レビュー前の `docs/progress/move.md` は「効果」列が `？` のままだった。
  `src/jpoke/data/ps-champ-ja/moves.json`（L307）の説明文に基づき、正しい効果説明に更新する。

## テスト

* シングルバトルで観測可能な効果がないため、専用テストは追加しない
  （てだすけ・いかりのこな・さきおくり・アロマミスト等、他の全ダブル専用技にも専用テストは
  存在しない）。

## リーサル計算

* 対象外（ダメージを与えず、HP分布にも影響しない）。
