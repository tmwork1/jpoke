# 実装計画書: シザークロス

参照仕様書: docs/spec/moves/シザークロス.md

## 仕様要約

* タイプ: むし / 分類: 物理 / 威力: 80 / 命中: 100 / PP: 16（champions基準）
* 対象: 相手1体選択 / 優先度: 0 / 接触あり（直接攻撃）
* 急所ランク補正なし（本編シリーズでは急所に当たりやすい効果は無い。
  LEGENDS アルセウス限定の効果であり本プロジェクトの対象範囲外）
* 追加効果なし
* 「切る技」（`flags` に `"slash"`）。特性きれあじの対象。

## 現状の実装状況

`src/jpoke/data/moves/move_sa.py` に登録済み。追加効果を持たないため専用ハンドラは不要。

```python
"シザークロス": MoveData(
    type="むし",
    category="physical",
    pp=16,
    power=80,
    accuracy=100,
    flags={"contact", "slash"},
    handlers={},  # 追加効果なし
),
```

`pp` は当初 Gen9本家基準の `15` になっていたが、champions は全技PPが8/12/16/20の
4段階に統一されているため（`docs/champions/changes_from_sv.md`）、
`src/jpoke/data/ps-champ-ja/moves.json` の値 `16` に修正した（移行漏れバグの修正）。

`critical_rank` は指定なし（デフォルト0）。きりさく・クロスポイズン等とは異なり、
シザークロスは本編シリーズで急所ランク補正を持たないため、これで仕様と一致する。

## 登録

`src/jpoke/data/moves/move_sa.py` の `MOVES_SA["シザークロス"]`（登録済み・修正不要、PPのみ修正済み）。
`handlers={}`（追加効果なし）。

## リーサル計算

追加効果を持たない通常攻撃技のため、`handlers/lethal.py` への専用ハンドラ追加は不要（n/a）。
`docs/progress/move.md` の「リーサル実装」列は `n/a`。
