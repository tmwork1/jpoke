# シャドーボール 実装計画書

## 仕様要約（docs/spec/moves/シャドーボール.md 参照）

* タイプ: ゴースト / 分類: 特殊 / 威力: 80 / 命中: 100 / PP: 16
  （champions/move_list.txt 準拠。Gen9本家の15ではない）
* 対象: 相手1体 / 優先度: 0 / 非接触 / 弾のわざ（`flags={"bullet"}`）
* 効果: 追加効果として、20%の確率で相手の『とくぼう』ランクを1段階下げる。
* ちからずく対象技（ランク低下の追加効果を持つため `flags={"secondary_effect"}` が必要）。

## 現状の実装状況

`src/jpoke/data/moves/move_sa.py` に登録済み、`src/jpoke/handlers/move_attack.py` に
`シャドーボール_lower_defender_spd` として実装済み。レビュー時点で以下の不備を発見し修正した。

1. **PP誤り**: `pp=15`（Gen9本家値）になっていた。champions/move_list.txt の値は16。
   → `pp=16` に修正。
2. **`secondary_effect` フラグの欠落**: シャドーボールは相手のとくぼうランクを下げる
   追加効果を持つため、`docs/spec/abilities/ちからずく.md` の「相手のランク補正を下げる技」に
   該当し、ちからずく対象技である。しかし `flags={"bullet"}` のみが設定されており
   `secondary_effect` が欠落していたため、ちからずくを持つポケモンが使用しても
   威力1.3倍化・追加効果無効化が発動しないバグになっていた。
   → `flags={"bullet", "secondary_effect"}` に修正。

修正後の登録内容:

```python
"シャドーボール": MoveData(
    type="ゴースト",
    category="special",
    pp=16,
    power=80,
    accuracy=100,
    flags={"bullet", "secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.シャドーボール_lower_defender_spd,
        )
    }
),
```

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_DAMAGE_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 20%の確率で相手のとくぼうランクを1段階下げる |

`src/jpoke/handlers/move_attack.py` の実装（変更なし。ロジック自体は正しい）:

```python
def シャドーボール_lower_defender_spd(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_defender_stats(battle, ctx, value, stats={"spd": -1}, chance=0.2)
```

`modify_defender_stats`（`handlers/move.py`）が `battle.resolve_secondary_chance` を
経由するため、てんのめぐみによる確率2倍化・ちからずくによる追加効果無効化と威力1.3倍化は
汎用ハンドラ側で処理される。本技側に個別のガードは不要（`secondary_effect` フラグさえ
正しく付与されていればよい）。

## Priority根拠・実装方針

* `ON_DAMAGE_HIT` は `docs/spec/turn.md` に priority 記載のない「ダメージ計算内部」の
  イベントである。CLAUDE.md の指示に従い、同種の既存ハンドラ（`シャドーボーン_lower_defender_def`、
  `どくばり_apply_poison_to_defender` 等）を参照し、いずれもデフォルト priority=100・
  `attacker:self` で競合なく動作していることを確認したうえで同じ構成を採用した（変更なし）。

## テスト方針

* シャドーボール自体の効果（20%でとくぼうランク-1）を確認する専用テストが存在しなかったため
  新規追加する（`secondary_chance=1.0` で確率100%にして確認）。
* 既存テスト `test_てんのめぐみ_追加効果確率が2倍になる`（`tests/abilities/test_ability_ta.py`）は
  シャドーボールを間借りして使用しているため、PP・flags 修正後も壊れないことを確認する。
