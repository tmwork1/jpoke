# 実装計画書: アロマミスト

## 仕様

- タイプ: フェアリー / カテゴリ: 変化 / PP: 20
- 命中: なし（必中・自分対象）
- 効果: 使用者（シングルでは自分）のとくぼうを1段階上げる
- ダイウォールを無視する

## イベント・Priority

| イベント | Priority | subject_spec | 役割 |
|----------|----------|-------------|------|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（MoveHandler デフォルト） | とくぼう+1 |

**根拠**: `ON_STATUS_HIT` は `turn.md` に独立エントリなし。`かたくなる`・`ドわすれ` 等の自己能力上昇技がすべて priority=100（デフォルト）のため同パターン採用。

## 実装

### handlers/move_status.py

配置位置: `アクアリング_apply_volatile_to_attacker` の後、`アンコール_can_apply` の前

```python
def アロマミスト_modify_attacker_stats(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    return modify_attacker_stats(battle, ctx, value, stats={"D": 1})
```

### data/move.py

既存 `"アロマミスト"` エントリの修正点:
- `accuracy=100` を削除（命中判定なし・必中）
- `target="self"` を追加
- `handlers` を追加

```python
"アロマミスト": MoveData(
    type="フェアリー",
    category="変化",
    pp=20,
    target="self",
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.アロマミスト_modify_attacker_stats,
        ),
    }
),
```

## 注意事項

- ランク最大時の失敗は `ON_TRY_MOVE_2 priority=130` の既存インフラが処理するためハンドラ側ガード不要
- `target="self"` でダイウォール無視が担保される
- かがくへんかガス/かたやぶり無効化チェックは書かない
