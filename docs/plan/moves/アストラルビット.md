# 実装計画書: アストラルビット

参照仕様書: docs/spec/moves/アストラルビット.md

## 実装方針

アストラルビットは追加効果を持たない攻撃技であり、以下のデータのみで挙動が完結する。
専用ハンドラは不要。

* `type="ゴースト"`
* `category="special"`
* `pp=8`（Gen9本家は5だが、Championsでは全技PPが8/12/16/20の4段階に統一されているため8とする。
  `docs/champions/move_list.txt` には本技の記載がないが、`でんじほう` `はかいこうせん` `じばく`
  `だいばくはつ` など base PP5 の他技が軒並み8に変換されていることから同様に扱う）
* `power=120`
* `accuracy=100`
* `flags={}`（直接攻撃ではない）
* `target`はデフォルトの`"foe"`のままでよい（本プロジェクトはシングルバトルのみを対象とし、
  「相手全員」用の別ターゲット種別は存在しないため）

## 登録

`src/jpoke/data/moves/move_a.py` の `MOVES["アストラルビット"]` に上記の値を設定する。
`handlers={}` とする（追加効果なし）。

## テスト方針

`tests/moves_attack/test_move_a.py` に以下を追加する。

* タイプ・分類・威力・命中率が正しく反映され、ダメージのみを与えることを確認するテスト
  （追加効果によるランク変化・状態異常などが発生しないこと）
* 直接攻撃ではないこと（かたやぶり等の接触判定に影響する仕様に触れる場合）を確認するテスト
  （必要に応じて、まもるや命中率等の一般的な攻撃技共通ロジックのみで完結するため省略可）

## リーサル計算

追加効果を持たない通常攻撃のため、専用のリーサルハンドラは不要（n/a）。
