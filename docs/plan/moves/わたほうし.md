## 実装箇所

### `src/jpoke/handlers/move_status.py`

五十音順の挿入位置: 「わ」行 — `ロックカット_modify_attacker_stats` の後、`わるだくみ_modify_attacker_stats` の前に挿入する。

```python
def わたほうし_modify_defender_stats(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """わたほうしの効果: 相手のすばやさを 1 段階下げる。粉技のためくさタイプ・ぼうじん無効。"""
    return modify_defender_stats(battle, ctx, value, stats={"S": -1})
```

### `src/jpoke/data/move.py`

既存の `"わたほうし"` エントリに handlers を追加する。

```python
"わたほうし": MoveData(
    type="くさ",
    category="変化",
    pp=40,
    accuracy=100,
    labels=["powder"],
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.わたほうし_modify_defender_stats,
        ),
    }
),
```

## 注意点・エッジケース

- 粉技（powder）ラベルはすでに `data/move.py` に設定済み。くさタイプ・ぼうじん特性・ぼうじんゴーグルへの無効化は `ON_TRY_MOVE_2` の「ぼうじん: 粉技無効」・「くさタイプ: 粉技無効」で処理されるため、ハンドラ側では不要
- S ランク下限チェック（-6 で効果なし）は `modify_stats` 内部で処理されるため、ハンドラ側での追加チェックは不要
