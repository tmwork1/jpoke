# りゅうせいぐん 実装計画書

## 仕様要約（docs/spec/moves/りゅうせいぐん.md 参照）

* タイプ: ドラゴン / 分類: 特殊 / 威力: 130（第六世代以降） / 命中: 90 / PP: 8（チャンピオンズ基準）
* 対象: 相手1体 / 優先度: 0 / 接触なし
* 効果: 命中して技が成功した後、自分の『とくこう』ランクが2段階下がる（確定効果）。

## 現状の実装状況

`src/jpoke/data/moves/move_ra.py` に登録済み。`handlers/move_attack.py` の
`りゅうせいぐん_sharply_lower_spa_C` と `handlers/lethal.py` の `りゅうせいぐん_lower_spa` は
仕様と一致しており修正不要。ただし `pp=5`（Gen9本家基準）になっていたため、
チャンピオンズの8/12/16/20統一ルール（`docs/champions/changes_from_sv.md`）に基づき
`pp=8` に修正した。

```python
"りゅうせいぐん": MoveData(
    type="ドラゴン",
    category="special",
    pp=8,
    power=130,
    accuracy=90,
    handlers={
        Event.ON_HIT: h.MoveHandler(
            ha.りゅうせいぐん_sharply_lower_spa_C,
        )
    },
    lethal_handlers={
        LethalEvent.ON_HIT: LethalHandler(l.りゅうせいぐん_lower_spa)
    }
),
```

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_HIT` | 100（デフォルト） | attacker:self（`MoveHandler` 既定値） | 命中後、自分の『とくこう』ランクを2段階下げる（確定効果） |

`src/jpoke/handlers/move_attack.py` の実装（登録済み・修正不要）:

```python
def りゅうせいぐん_sharply_lower_spa_C(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return modify_attacker_stats(battle, ctx, value, stats={"spa": -2})
```

`modify_attacker_stats`（`handlers/move.py`）の `chance` 既定値は1のため常に発動する。
自分自身への確定効果であり、`resolve_secondary_chance` 経由の「追加効果」判定
（ちからずく等）の対象にはならない（`flags` に `secondary_effect` を付与しない）。

## Priority 根拠

自分自身への確定ランク低下は既存の同種ハンドラ（いじげんラッシュ・りんごさん等）と同じ
`ON_HIT` / priority 100（デフォルト）のパターンに従う。`ON_HIT` は
`docs/spec/turn.md` に個別掲載されていないが、既存の同種ハンドラ（`いじげんラッシュ_lower_attacker_def`
等、命中直後に自分のランクを下げる実装）を参照して決定した。

## 注意点・エッジケース（review-test 向け）

1. **ちからずく等「追加効果」系特性・アイテムとの非該当**
   - 自分自身への確定ランク低下は「追加効果」ではないため、ちからずく・てんのめぐみ
     （`secondary_effect` フラグを参照する特性）の対象にならない。`flags` に
     `secondary_effect` を付与しない現状の実装が正しい。
2. **まもる貫通なし**
   - 通常の攻撃技であり、`まもる` 等の対象になる。個別のガードは不要
     （`unprotectable` フラグ等は付与しない）。
3. **テスト方針**
   - 100%固定効果のため `secondary_chance` の調整は不要。命中後に自分の『とくこう』
     ランクが2段階下がることを確認する通常テストと、`t.calc_lethal` を用いた
     リーサルテスト（既存の `test_りゅうせいぐん_とくこうダウン`）の両方で担保する。
