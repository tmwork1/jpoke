# 実装計画書: どくのこな

## 仕様要約

- タイプ: どく / カテゴリ: 変化 / PP: 20（champions基準） / 命中: 75 / 対象: 1 体選択
- 相手を「どく」状態にする
- どく・はがねタイプには無効（`ailment_manager` が一律判定するため技側実装は不要）
- 粉・胞子技（`powder` フラグ）: くさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』には無効
  （`move_executor` / `ability.py` / `item.py` が汎用的に処理するため技側実装は不要）

## 注意点・エッジケース

- PPはchampions基準の`docs/champions/move_list.txt`の値（20）を正とする。Gen9本家値（35）に
  戻さないこと
- `powder`フラグにより、くさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』への無効化は
  技側実装不要（フラグを付与するだけで汎用処理が効く）
- どく・はがねタイプへの無効化は`ailment_manager`が一律判定するため技側実装は不要
- 対象は相手（`target`のデフォルト値`"foe"`のまま）。自分対象ではないため`target="self"`の
  指定は不要
