# 計画書: ひっさつまえば

## 結論: 実装しない

`docs/spec/moves/ひっさつまえば.md` の通り、本技はソード・シールド以降（BDSP・SVを含む）で使用不可であり、
README.md の対象範囲定義（第9世代で実装されていない技は実装しない）に反するため実装計画は策定しない。

## 経緯

過去の一括実装計画 `docs/plan/moves/攻撃技_ひるみ系.md` で「スペック要確認」のまま実装され、
以下の誤りを含んだ状態で登録されていた:

- `src/jpoke/data/moves/move_ha.py` の `MoveData(power=120, pp=5, ...)`
  - 正しい基礎威力は80（がんじょうあご適用後の120が誤って登録されていた）、PPは15
- `src/jpoke/handlers/move_attack.py` の `ひっさつまえば_apply_flinch`
- `tests/moves_attack/test_move_ha.py` と `tests/moves_status/test_move_ha.py` の双方に
  同名テストが重複登録（後者は物理技であるにもかかわらず status 側のテストファイルに混入）

本レビュー（`docs/review/moves/ひっさつまえば.md`）にて対象範囲外と判断し、上記実装・テストを削除した。
今後、本技を再実装する必要が生じた場合は、まず README.md の対象範囲定義とポケモンチャンピオンズでの
使用可否を再確認すること。
