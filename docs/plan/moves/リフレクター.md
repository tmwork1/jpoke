# 実装計画書: リフレクター

## 仕様要約

- タイプ: エスパー / カテゴリ: 変化 / PP: 20 / 対象: 味方の場（target="own_side"）
- 5ターンの間、相手の物理攻撃のダメージを半分にする
- 味方が複数の場合は半分ではなく 2/3（シングルバトルでは半減固定）
- 急所に当たった場合は軽減されない
- 交代しても効果は続く

## 現状の実装状況

| 項目 | 状態 |
|------|------|
| サイドフィールド `リフレクター`（`data/field/side_field.py`） | 実装済み（ダメージ軽減・ターン経過は `handlers/field.py`） |
| `data/moves/move_ra.py` エントリ | 実装済み（`target="own_side"`、`Event.ON_STATUS_HIT` で `hs.リフレクター_set_side_field` を登録） |
| `handlers/move_status.py` 関数 | 実装済み（`リフレクター_set_side_field`） |

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|----------|----------|--------------|------|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（デフォルト） | 自陣営サイドフィールドに「リフレクター」を設定（5ターン） |

## Priority 根拠

`Event.ON_STATUS_HIT` は `docs/spec/turn.md` に個別記載のないイベント（`move_executor.py` が変化技の
命中処理として発火する内部イベント）。既存の同種ハンドラ（`ひかりのかべ_set_side_field` /
`オーロラベール_set_side_field` / `おいかぜ_set_side_field` 等の自陣営フィールド設定技）が
すべてデフォルト優先度（100）を使用しているため、同一パターンを採用する。

## 実装箇所

### `src/jpoke/handlers/move_status.py`

五十音順の挿入位置: `リサイクル_restore_item` の後、`リフレッシュ_cure_ailment` の前。

```python
def リフレクター_set_side_field(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """リフレクター: 自陣営に「リフレクター」を5ターン設定する。"""
    side = battle.get_side(ctx.attacker)
    if not side.apply("リフレクター", 5, source=ctx.attacker):
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)
```

### `src/jpoke/data/moves/move_ra.py`

```python
"リフレクター": MoveData(
    type="エスパー",
    category="status",
    pp=20,
    target="own_side",
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.リフレクター_set_side_field,
        ),
    }
),
```

## 注意点・エッジケース

- `target` は `"own_side"`（味方の場）
- `side.apply(...)` が False を返す（すでにリフレクターが有効）場合は失敗として `HandlerReturn(value=False, stop_event=True)` を返す
  （`SideFieldManager.apply` は `ON_MODIFY_DURATION` を発火してから `activate` する薄いラッパーで、
  ひかりのねんど所持時の8ターン延長はここで解決される）
- 物理ダメージ半減の本体は `data/field/side_field.py` の `リフレクター_reduce_damage` /
  `リフレクター_tick`（`handlers/field.py`）側で処理されており、技側はフィールド設定のみを行う
- ひかりのかべ・オーロラベール・おいかぜと同一パターン（5ターン・自陣営・`ON_STATUS_HIT` デフォルト優先度）
