## 注意点・エッジケース

- ぼうぎょ単体のランクを1段階下げるだけの単純な効果。`modify_defender_stats` の `stats` 引数に
  `{"def": -1}` を渡せばよい
- ランク低下の無効化（しろいきり、クリアボディ等）はフレームワーク内で処理されるため、ハンドラ側のガードは不要
- 音技ではないため、`みがわり` 状態の相手には命中しない（フレームワーク側で処理済み）
- まもる状態の相手には防がれる（`unprotectable` フラグを付与していないため一般ルールに従う）
