# 実装計画書: ドラゴンエール

## 仕様要約（docs/spec/moves/ドラゴンエール.md 参照）

* タイプ: ドラゴン / カテゴリ: 変化 / PP: 16（champions基準） / 命中: -（必中） / 対象: 自分以外の味方全員
* 優先度: 0 / 直接攻撃: × / まもる: 効果なし（守×） / 音の技（champions限定）
* 自分以外の味方全員を「きゅうしょアップ」状態にする。ドラゴンタイプは急所ランク+2、
  それ以外は+1（ダブル/トリプル向けの主効果）。
* 自分は対象に含まれない。**シングルバトルでは「自分以外の味方が存在しない」ため常に失敗する**
  （ポケモンWiki明記の通り）。

## 実装しない判断

* 味方をきゅうしょアップ状態にするという本技の効果は、シングルバトルでは対象となる
  「自分以外の味方」が存在しないため、常に不発となる。この失敗自体は明確な仕様だが、
  `handlers={}`（何もしない）と実装上の挙動が完全に一致するため、失敗処理をわざわざ実装する
  意味がない。
* 類似のダブル/トリプル専有技（てだすけ・コーチング・アロマミスト・いかりのこな等）も
  同様の理由で `docs/progress/move.md` 上「ダブル専用」＝対象外として扱われており、本技のみ
  きゅうしょアップ付与処理や失敗判定を実装すると一貫性を欠く。
* 以上より、本技は `handlers={}`（追加効果なし）のまま、味方対象のきゅうしょアップ付与処理は
  実装しない。

## 現状の実装状況（レビュー前）

| 項目 | レビュー前の状態 | 問題点 |
|------|------|------|
| `data/moves/move_ta.py` エントリ (`"ドラゴンエール"`) | `type="ドラゴン"`, `category="status"`, `pp=15`, `accuracy=100`, `handlers={}` | PP・accuracy・target・flagsに誤りあり |

### 発見した不具合（本レビューで修正）

1. **PP誤り**: `pp=15` はGen9本家基準の値であり、`docs/champions/move_list.txt`（L588）の
   champions基準値`16`への移行漏れ。`docs/champions/changes_from_sv.md`により全技PPは
   8/12/16/20の4段階に統一されているため、15はそもそも存在しないはずの値。`pp=16`に修正する。
2. **accuracy誤り**: `accuracy=100`が設定されていたが、本技はポケモンWiki・champions
   move_list.txt（L588、命中欄「-」）ともに必中技であり、命中判定自体が存在しない。
   本コードベースでは必中技は`accuracy`フィールドを省略する規約（`Move.accuracy`が`None`のとき
   `MoveExecutor`が命中判定をスキップする、`core/move_executor.py`参照）のため、
   `accuracy=100`の指定を削除する。
3. **target未設定によるまもる誤判定**: `target`が未設定のためデフォルト値`"foe"`が使われていたが、
   `Move.is_blocked_by_protect`は`target == "foe"`かつ`"unprotectable"`フラグなしの場合に`True`を
   返す実装になっている。本技はポケモンWiki・move_list.txt（守×）ともに「まもるに防がれない」技
   であり、デフォルトの`target="foe"`のままでは（シングルバトルでは到達しないコードパスとはいえ）
   仕様と矛盾した`is_blocked_by_protect`判定になる。`target="self"`を設定し、実際の対象
   （自分以外の味方）に近い側（使用者側の場）を表す近似値とする（`docs/spec/moves/アロマミスト.md`の
   `target="self"`近似と同じ扱い）。
4. **soundフラグ欠落**: champions版move_list.txt（L589）に「音系の技。」と明記されているが、
   `flags`に`"sound"`が含まれていなかった。`flags={"sound"}`を追加する。

## 実装方針（修正後）

```python
"ドラゴンエール": MoveData(

    category="status",
    pp=16,
```

## 注意点・エッジケース

### かがくへんかガス・かたやぶり

* ドラゴンエールはダメージを与えず、味方対象のランク変化のみを行う技のため、
  かがくへんかガス・かたやぶりの影響を受けない。ハンドラ側でのチェックも不要。

### ぼうおん（soundproof）との関係

* champions版では音の技に分類され、特性ぼうおんの味方に対しては無効化される仕様だが、
  本技はそもそも「使用者自身の相手（`ctx.defender`）」ではなく「使用者以外の味方」を対象と
  する技であり、本コードベースが対象とするシングルバトルでは対象となる味方が存在しないため
  常に失敗する。ぼうおんによる無効化判定を実装しても観測可能な効果を生まないため未実装とする。

## テスト

* シングルバトルで観測可能な効果がないため、専用テストは追加しない
  （てだすけ・いかりのこな・アロマミスト等、他の全ダブル専用技にも専用テストは存在しない）。

## リーサル計算

* 対象外（ダメージを与えず、HP分布にも影響しない）。
