# 実装計画: きんぞくおん

## 仕様サマリ

- タイプ: はがね / 分類: 変化 / PP: 20 / 命中率: 85
- 効果: 相手の **とくぼう** ランクを 2段階下げる
- まもる: ○ / マジックコート: ○ / 直接攻撃: × / 音技: ○
- 対象: 相手1体選択

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（デフォルト） | 相手のとくぼうを2段階下げる |

## Priority 根拠

`docs/spec/turn.md` に `ON_STATUS_HIT` の記載なし。
既存の同種ハンドラ（`うそなき_modify_defender_stats`、`あまえる_modify_defender_stats` など）が
すべて `MoveHandler(func)` のみ（priority=100 デフォルト）で実装されているため、同様に priority=100 を採用する。

## 実装箇所

### handlers/move_status.py への追加（既存実装済み）

```python
def きんぞくおん_modify_defender_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """きんぞくおんの効果: 相手のとくぼうを2段階下げる。"""
    return modify_defender_stats(battle, ctx, value, stats={"spd": -2})
```

### data/moves/move_ka.py の更新

```python
"きんぞくおん": MoveData(
    type="はがね",
    category="status",
    pp=20,
    accuracy=85,
    flags={"sound"},
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(

```

- PPはポケモンチャンピオンズ仕様（`src/jpoke/data/ps-champ-ja/moves.json`）に合わせて20とする。
  既存実装は`pp=40`（Gen9本家値）になっていたためレビューで修正。

## 注意点・エッジケース

- とくぼうが最速で-6段階のときは失敗する（ランク変化共通のガードで処理済み、ハンドラ側の重複ガード不要）
- 音系技のため、みがわり状態でも命中する（技フレームワーク側の`sound`フラグ判定で処理済み）
- リーサル計算: 対象外（`n/a`）。とくぼうダウンは相手の次ターン以降の被ダメージに影響するのみで、
  本ターンの命中側の生存判定・与ダメージ判定には関与しないため
