# ふしょくガス 実装計画書

## 仕様要約（docs/spec/moves/ふしょくガス.md 参照）

* タイプ: どく / 分類: 変化 / 威力: - / 命中: 100 / PP: 20
  （Wikiのゲーム内表記は40だが、`src/jpoke/data/ps-champ-ja/moves.json` によりポケモンチャンピオンズでは
  全技PPが8/12/16/20の4段階に統一されているため20を採用する）
* 対象: 自分以外全員（本プロジェクトはシングルバトルのみ対象のため実質相手1体） / 優先度: 0 / 接触なし
* 効果: 命中時に相手の持ち物を消失させる（溶かす）。ダメージは与えない。
  奪って自分のものにするはたきおとすと異なり、単純に消滅させる。

```python
def ふしょくガス_remove_item(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """ふしょくガスのアイテム除去効果。"""
    battle.item_manager.remove_item(target=ctx.defender, source=ctx.attacker)
    return HandlerReturn(value=value)
```

```python
"ふしょくガス": MoveData(

```

## 道具除去の共通仕組み（実装不要・既存機構をそのまま利用）

`ItemManager.remove_item()` は内部で `can_change_item()` を呼び出し、`Event.ON_CHECK_ITEM_CHANGE`
共通イベントで除去可否を判定する。これは `はたきおとす` と全く同じ経路であり、以下は
既存ハンドラ（`ねんちゃく_prevent_item_change`、各専用道具の `ON_CHECK_ITEM_CHANGE`、
`removable=False` メタデータ等）がそのまま処理するため、本技専用の追加実装は不要。

* ねんちゃく持ちの相手からは除去できない（かたやぶり・かがくへんかガスなら除去できる）。
* だいこんごうだま・くちたけん・くちたたて・メガストーン・オーガポンのめん等の種族+持ち物ロック
  アイテムは除去できない。

みがわり状態の相手に対しては、本技が `category="status"` であるため
`みがわり_immune`（`Event.ON_BEFORE_APPLY_MOVE`）により技自体が無効化され、
`ON_STATUS_HIT` 自体が発火しない（使用者がすりぬけなら無視できる、フレームワーク共通処理）。

マジックコートによる反射も、`MoveFlag` に `unreflectable` を設定していないため
既存の反射機構がそのまま作用する（デフォルトで反射可能）。

## Priority 根拠

`ON_STATUS_HIT` は `docs/spec/turn.md` に掲載されていないイベントのため、同種の
「自分以外全員が対象の変化技」であるフラフラダンス（`docs/plan/moves/フラフラダンス.md` 参照、
てんしのキッス・あやしいひかりと同じ扱いでデフォルト優先度100を採用）に倣い、
`MoveHandler` のデフォルト優先度（100）をそのまま使用する。

## 注意点・エッジケース（review-test 向け）

1. **通常ケース**: 相手が持ち物を持っている場合、命中すると持ち物が消失する。ダメージは発生しない。
2. **持ち物なしの相手**: `remove_item` が `has_item()` チェックで失敗を返すのみで、技自体は
   通常成功扱い（`ON_STATUS_HIT` の戻り値は `True` のまま）。Wiki記載の「うまく きまらなかった」
   演出はログ表示上の話であり、技の成否判定には影響しない（`はたきおとす` 等と同様の扱い）。
3. **ねんちゃく持ちの相手**: 持ち物は消失しない。
4. **みがわり状態の相手**: 技自体が失敗する（`ON_BEFORE_APPLY_MOVE` で無効化）。
5. **専用道具（くちたけん等）を持つ相手**: 持ち物は消失しない。
6. **かるわざ持ちの相手**: 持ち物を溶かすとかるわざが発動しすばやさが2倍になる（既存の
   `ON_ITEM_LOST` 経路で処理されるため追加実装不要）。
7. **マジックコートに跳ね返された場合**: 使用者自身の持ち物が消失する
   （本プロジェクトはシングルバトルのみのため対象は使用者1体）。
