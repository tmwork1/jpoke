# 実装計画書: マジカルリーフ

参照仕様書: docs/spec/moves/マジカルリーフ.md

## 実装方針

マジカルリーフは追加効果を持たない攻撃技であり、以下のデータのみで挙動が完結する。
専用ハンドラは不要。

* `type="くさ"`
* `category="special"`
* `pp=20`
* `power=60`
* `accuracy=None`（必中。既存の `あてみなげ` `でんげきは` `スピードスター` などと同様、
  `accuracy=None` を core（`move_executor.py`）が必中として処理するため追加実装は不要）
* `priority=0`（デフォルト値のため明示不要）
* `target="defender"`（デフォルト値のため明示不要。相手1体選択の技）
* `flags`は不要（接触なし・追加効果なし・ちからずく対象外）

## 登録

`src/jpoke/data/moves/move_ma.py` の `MOVES["マジカルリーフ"]` に上記の値を設定する。
`handlers={}` とする（追加効果なし）。
