# 実装計画書: オーロラベール

## 仕様要約

- タイプ: こおり / カテゴリ: 変化 / PP: 20 / 対象: 味方の場（target="own_side"）
- 天気が「ゆき」の時のみ使用できる
- 5ターンの間、自分と味方が受ける物理・特殊攻撃のダメージを半分にする
- 味方が複数の場合は半分ではなく 2/3（シングルバトルでは常に半分）
- 急所に当たった場合は軽減されない
- 交代しても効果は続く

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|----------|----------|--------------|------|
| `ON_TRY_MOVE_1` | 30 | `attacker:self`（デフォルト） | 天気が「ゆき」でない場合は失敗 |
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（デフォルト） | 自陣営サイドフィールドに「オーロラベール」を 5 ターン設定 |

## Priority 根拠

- `ON_TRY_MOVE_1` priority 30: `docs/spec/turn.md` の `Event.ON_TRY_MOVE_1` テーブルに「オーロラベール: あられ状態でない」が priority 30 として明記されている
  - ただし仕様はポケモンチャンピオンズ準拠（ゆき天候時）のため、チェック内容は「ゆき天候でない場合は失敗」に読み替える
- `ON_STATUS_HIT` priority 100（デフォルト）: `ひかりのかべ_set_side_field` と同じパターン

## 実装箇所

### `src/jpoke/handlers/move_status.py`

五十音順の挿入位置: 「お」は「え」の後 — `エレキフィールド_activate_terrain` の後、`おいかぜ_set_side_field` の前に挿入する。

```python
def オーロラベール_check_weather(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """オーロラベールの使用条件チェック: 天気が「ゆき」でない場合は失敗する。"""
    if not battle.weather.name == "ゆき":
        battle.add_event_log(ctx.attacker, LogCode.MOVE_FAILED,
                             payload={"reason": "オーロラベール"})
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)


def オーロラベール_set_side_field(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """オーロラベール: 自陣営に「オーロラベール」を 5 ターン設定する。"""
    side = battle.get_side(ctx.attacker)
    if not side.activate("オーロラベール", 5):
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)
```

### `src/jpoke/data/move.py`

既存の `"オーロラベール"` エントリに priority・handlers を追加する。target は `"own_side"` に修正する。

```python
"オーロラベール": MoveData(
    type="こおり",
    category="変化",
    pp=20,
    target="own_side",
    handlers={
        Event.ON_TRY_MOVE_1: h.MoveHandler(
            hs.オーロラベール_check_weather,
            priority=30,
        ),
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.オーロラベール_set_side_field,
        ),
    }
),
```

## 注意点・エッジケース

- 現在 `data/move.py` の target が `"field"` になっているが、自陣営への設置なので `"own_side"` に修正が必要
- 天候チェックは `battle.weather.name == "ゆき"` で行う（`battle.weather.snowy` のようなプロパティが存在するか事前確認を推奨）
- `side.activate("オーロラベール", 5)` が False を返すのはすでに設置済みの場合のみ（同じターンに天候が変わっても影響しない）
- ダメージ軽減効果はサイドフィールドの `ON_MODIFY_DAMAGE` ハンドラ側で処理されるため、技ハンドラ側では設置のみ行う
- 急所貫通の例外処理もフィールドハンドラ側で対応済みであることを確認すること
