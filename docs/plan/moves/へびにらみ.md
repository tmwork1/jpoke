# へびにらみ 実装計画書

## 仕様要約

- 相手を **まひ** 状態にする変化技
- ノーマルタイプ / PP 20（champions基準。src/jpoke/data/ps-champ-ja/moves.json と一致） / 命中率 100
- 失敗条件: 対象がすでに状態異常 OR でんきタイプ
  （`ailment_manager.apply` 内部の汎用チェックで処理されるため、技固有の失敗判定ハンドラは不要）
- でんじはと異なり、技自体のタイプ相性による無効化（じめんタイプ等）は存在しない
  （ノーマルタイプの変化技であり、`_check_hit_by_type` は攻撃技のみ対象のため元々影響しない）

## 実装方針

まひ付与そのものは `ailment_manager.apply(defender, "まひ")` で行う。
`apply` メソッドが既存の状態異常チェック・タイプ相性チェック（`_can_apply_by_type` の
`target.has_type("でんき")`）・`ON_BEFORE_APPLY_AILMENT` イベント（じゅうなん等の特性、
しんぴのまもり等の場の状態、みがわりによる無効化）を汎用的に処理するため、
へびにらみ固有の失敗判定ハンドラは不要（でんじはのような `ON_BEFORE_APPLY_MOVE` 追加ハンドラは
不要）。

### ハンドラ構成

| イベント | Priority | subject_spec | 役割 |
|----------|----------|---------------|------|
| `ON_STATUS_HIT` | 100（デフォルト） | attacker:self（省略） | まひ状態を相手に付与 |

### Priority 根拠

- `ON_STATUS_HIT`：MoveHandler のデフォルト priority（`docs/spec/turn.md` 未掲載。既存の
  `でんじは` `おにび` `どくどく` 等の状態異常技ハンドラと同様にデフォルト値を使用）

### subject_spec

- `attacker:self`（MoveHandlerデフォルト）。`AttackContext` の中で
  `ctx.attacker`（使用者）・`ctx.defender`（対象）を参照する。

## 実装箇所

### handlers/move_status.py（五十音順: へびにらみは「へ」行）

```python
def へびにらみ_apply_ailment_to_defender(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return apply_ailment_to_defender(battle, ctx, value, ailment="まひ")
```

### data/moves/move_ha.py（「へびにらみ」MoveData）

```python
"へびにらみ": MoveData(
    type="ノーマル",
    category="status",
    pp=20,
    accuracy=100,
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.へびにらみ_apply_ailment_to_defender,
        ),
    }
),
```

## 注意点・エッジケース（review-test 向け）

1. **でんきタイプの相手には無効（まひ免疫）**: `ailment_manager._can_apply_by_type` が
   `target.has_type("でんき")` で判定するため、追加チェック不要。
2. **命中率100%**: 第六世代以降の値。`accuracy=100` を MoveData に設定する（champions基準）。
3. **PP20**: `src/jpoke/data/ps-champ-ja/moves.json` の値と一致（Gen9本家は30だが、championsでは
   全技PPが8/12/16/20の4段階に統一されているため20が正）。
4. **まもる・マジックコート**: 変化技かつ `target=foe` のデフォルト挙動でまもる・マジックコートの
   対象になる。個別フラグ設定は不要。
5. **ゴーストタイプ無効化（第二・第三世代限定の過去仕様）**: 現行仕様（ポケモンチャンピオンズ）
   には存在しないため実装しない。
