# 実装計画書: エアロブラスト

参照仕様書: docs/spec/moves/エアロブラスト.md

## 実装方針

エアロブラストは追加効果を持たない攻撃技であり、以下のデータのみで挙動が完結する。
専用ハンドラは不要。

* `type="ひこう"`
* `category="special"`
* `pp=5`（champions/move_list.txt に本技のエントリが存在しないため現行値を維持。
  詳細は仕様書を参照）
* `power=100`
* `accuracy=95`
* `critical_rank=1`（急所ランク+1。`core/move_executor.py` が `MoveData.critical_rank` を
  参照して急所判定に反映するため追加実装は不要）
* `flags={"wind"}`（直接攻撃ではない。「風の技」フラグにより、かぜのり特性の吸収・
  ふうりょくでんき特性のじゅうでん化などの既存ハンドラが `has_flag("wind")` を
  参照して適用される）

## 登録

`src/jpoke/data/moves/move_a.py` の `MOVES["エアロブラスト"]` に上記の値を設定する。
`handlers={}` とする（追加効果なし）。

## リーサル計算

追加効果を持たない通常攻撃技のため、`handlers/lethal.py` への専用ハンドラ追加は不要（n/a）。

## テスト方針

`tests/moves_attack/test_move_a.py` に以下を追加する。

* 急所ランク+1のため乱数0で急所が発生することを確認するテスト
* 乱数が急所閾値以上のとき急所にならないことを確認するテスト
* 相手にダメージを与えることを確認するテスト
