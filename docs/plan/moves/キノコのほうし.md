# 実装計画書: キノコのほうし

## 仕様要約

- タイプ: くさ / カテゴリ: 変化 / PP: 15（Gen9本家基準。`docs/champions/move_list.txt` に
  未掲載のため champions 独自のPP変換は行わない） / 命中: 100 / 対象: 1 体選択
- 相手を「ねむり」状態にする
- 粉・胞子技（`powder` フラグ）: くさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』には無効
  （`move_executor` / `ability.py` / `item.py` が汎用的に処理するため技側実装は不要）
- 特性『そうしょく』にはくさタイプの技として無効化され、対象のこうげきが1段階上昇する
  （`そうしょく_absorb_grass` が汎用処理するため技側実装は不要）

|----------|----------|--------------|------|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（デフォルト） | 相手を「ねむり」状態にする |

## Priority 根拠

`docs/spec/turn.md` には明示記載なし。同種の状態異常付与技（`ねむりごな_apply_ailment_to_defender`、
`さいみんじゅつ_apply_ailment_to_defender`）が `ON_STATUS_HIT` にデフォルト優先度で実装されている
パターンを踏襲する。

## 注意点・エッジケース

- PPは Gen9 本家基準の 15 を正とする。`docs/champions/move_list.txt` に本技のエントリが存在しない
  （たいあたり・テレポート等の一般的な技も同様に未掲載であり、部分的なデータと判断されるため）。
  8/12/16/20 への変換は行わない。
- `powder` フラグにより、くさタイプ・特性『ぼうじん』・道具『ぼうじんゴーグル』への無効化は
  技側実装不要（フラグを付与するだけで汎用処理が効く）
- 対象は相手（`target` のデフォルト値 `"foe"` のまま）。自分対象ではないため `target="self"` の
  指定は不要
- リーサル計算は対象外（`docs/progress/move.md` の該当行で `n/a`。状態異常付与技のためダメージ
  リーサルには寄与しない）
