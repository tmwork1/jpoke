# マジカルシャイン 実装計画書

## 仕様要約（docs/spec/moves/マジカルシャイン.md 参照）

* タイプ: フェアリー / 分類: 特殊 / 威力: 80 / 命中: 100 / PP: 12（Champions）
* 対象: 相手1体（一次情報の「相手全員」はダブル専用要素のため、シングルのみ対象の本プロジェクトでは
  通常のダメージ技として扱う） / 優先度: 0 / 接触なし
* 効果: 通常攻撃。追加効果なし。

## 現状の実装状況

`src/jpoke/data/moves/move_ma.py` に登録済み。

```python
"マジカルシャイン": MoveData(
    type="フェアリー",
    category="special",
    pp=12,
    power=80,
    accuracy=100,
    handlers={},  # 追加効果なし
),
```

タイプ・分類・威力・命中・PP・`handlers={}`（追加効果なし）のいずれも仕様書と一致しており、
修正は不要。

## 注意点・エッジケース（review-test 向け）

1. **ちからずくとの非連動**
   - 本技は追加効果を持たず、`docs/spec/abilities/ちからずく.md` に記載された例外技
     （いっちょうあがり・エレクトロビーム）にも該当しないため、`flags={"secondary_effect"}`
     は付与しない。ちからずく所持ポケモンが使用しても威力補正は発生しない。

2. **相手全員（スプレッド技）の扱い**
   - 一次情報では「範囲: 相手全員」だが、本プロジェクトはシングルバトルのみ対象のため、
     ダブル特有のスプレッド技威力減衰（0.75倍）は対象外（`target="foe"` のデフォルトのまま）。
