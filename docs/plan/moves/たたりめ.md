# 計画書: たたりめ

## 仕様要約

- ゴーストタイプ / 特殊 / 威力65 / 命中100 / PP10 / 対象: 1体選択 / 優先度0 / 非接触
- 対象が状態異常（どく・もうどく・まひ・やけど・ねむり・こおり・ゆめうつつ）のとき、
  威力が2倍になる。
- 特性『ぜったいねむり』の相手は常時『ゆめうつつ』状態異常を抱えているため、命中さえ
  すれば常に威力2倍となる（タイプ相性で無効化されるかどうかは別問題）。

## ハンドラ構成

| 技名 | イベント | priority | subject_spec | 役割 |
|------|----------|----------|--------------|------|
| たたりめ | `Event.ON_CALC_POWER_MODIFIER` | デフォルト（100） | デフォルト（`attacker:self`） | 対象（`ctx.defender`）が状態異常のとき威力を2倍にする |

`Event.ON_CALC_POWER_MODIFIER` は `docs/spec/turn.md` に priority の掲載がないため、
同種の可変威力技（`からげんき_double_power_when_ailment` 等）がいずれも `MoveHandler` の
デフォルト値（priority=100, subject_spec="attacker:self"）のまま登録されていることを
根拠に、デフォルト設定のまま登録する。
ハンドラ内部では `ctx.defender.ailment.is_active` を参照するため、`subject_spec` の値自体は
ロジックに影響しない（`からげんき` 等の既存の同種ハンドラと同じ扱い）。

## 実装箇所

### data/move.py への更新

既存の `"たたりめ"` エントリは以下の通りで、仕様と一致している（変更不要）。

```python
    "たたりめ": MoveData(
        type="ゴースト",
        category="special",
        pp=10,
        power=65,
        accuracy=100,
        handlers={
            Event.ON_CALC_POWER_MODIFIER: h.MoveHandler(
                ha.たたりめ_double_power_when_ailment,
            ),
        }
    ),
```

- `flags` を指定しない → 直接攻撃ではない（仕様と一致）。
- `target` 省略時のデフォルトは `"foe"`（相手1体）→ 仕様と一致。
- `priority` 省略時のデフォルトは `0` → 仕様と一致。

### handlers/move_attack.py への実装

```python
def たたりめ_double_power_when_ailment(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """たたりめ: 対象が状態異常のとき威力が2倍になる。"""
    if ctx.defender.ailment.is_active:
        value = apply_fixed_modifier(value, 8192)
    return HandlerReturn(value=value)
```

- `ailment.is_active` は状態異常名が空でないことを判定するため、`ゆめうつつ`
  （特性『ぜったいねむり』が登場時に自身へ付与する状態異常）も対象に含まれる。
  これにより「特性『ぜったいねむり』の相手には常に威力が2倍になる」という仕様を
  追加実装なしで満たせる。
- `からげんき` のように特定の状態異常を除外する必要はない（本技は状態異常全般が対象）。

## リーサル計算

対象が状態異常かどうかでダメージが変動するが、リーサル計算（`hp_dist` の変化）には
影響しない威力補正のみのため、`handlers/lethal.py` に専用ハンドラは不要（n/a）。
実ダメージ計算自体はダメージ計算モジュール側の威力補正を通じて反映される。

## 注意点・エッジケース

- ネッコアラ（特性『ぜったいねむり』を持つポケモン）は通常ノーマルタイプのため、
  ゴーストタイプの本技はタイプ相性で無効化され命中しない。テラスタル等でタイプが
  変化している場合や、相手が『ねらいのまと』を持っている場合にのみ実際に命中し、
  威力2倍の効果を確認できる。
