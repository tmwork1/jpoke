# ほたるび 実装計画書

## 仕様要約

- 自分の『とくこう』ランクを3段階上げる自己対象の変化技（`docs/spec/moves/ほたるび.md`）
- 命中判定なし（必中）、まもるで防がれない（自分を対象とするため）

## 実装方針

`modify_attacker_stats`（`handlers/move.py`）を利用して `ON_STATUS_HIT` で
攻撃側自身の『とくこう』を3段階上げるだけの単純な自己バフ技。
既存の `わるだくみ`（とくこう2段階）・`ちょうのまい`（こうげき/とくこう/すばやさ各1段階）と
同一のパターン。

### ハンドラ構成

| イベント | Priority | 役割 |
|----------|----------|------|
| `ON_STATUS_HIT` | 100（デフォルト） | 自分の『とくこう』ランクを3段階上げる |

### Priority 根拠

`docs/spec/turn.md` に `ON_STATUS_HIT` 自体の掲載はないが、既存の同種ハンドラ
（`わるだくみ_modify_attacker_stats` 等）がすべて `MoveHandler` のデフォルト
priority=100 を使用しているため、それに合わせる。

### subject_spec

`ON_STATUS_HIT`: `attacker:self`（`MoveHandler` デフォルト）。
変化技であっても `MoveHandler` が登録するイベントは攻撃フロー（`core/move_executor.py`）上で
発火するため `AttackContext` を使う。

## 実装箇所

### handlers/move_status.py（五十音順: ほたるびは「ほ」行、ほろびのうたの前）

```python
def ほたるび_modify_attacker_stats(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """ほたるびの効果: 自分の『とくこう』ランクを3段階上げる。"""
    return modify_attacker_stats(battle, ctx, value, stats={"spa": 3})
```

### data/moves/move_ha.py（既存の「ほたるび」MoveData エントリー）

```python
"ほたるび": MoveData(
    type="むし",
    category="status",
    pp=20,
    target="self",
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.ほたるび_modify_attacker_stats,
        )
    }
),
```

`target="self"` が必須（レビューで発見した欠落）。`target` のデフォルトは `"foe"` であり、
未設定のままだと以下がすべて誤動作する：

- `Move.is_blocked_by_protect`（`self.target == "foe"` を見るため、まもるで誤って防がれてしまう）
- `MoveExecutor.check_hit_substitute`（`ctx.move.target != "foe"` でみがわり判定するため、
  みがわりに誤って防がれてしまう）
- `Move.is_reflectable`（`self.target in ("foe", "foe_side")` を見るため、
  マジックコートで誤って跳ね返ってしまう可能性がある）
- 各種「相手の技を無効化する特性」（例: `handlers/ability.py` の `_apply_type_absorb` など）が
  `ctx.move.target == "foe"` を条件にしているため、誤って発動してしまう

PP は `docs/champions/move_list.txt` に本技のエントリがないため参照不可。既存実装の
PP=20 は champions の PP 正規化4段階（8/12/16/20）の範囲内であり、Gen9本家の PP（20）とも
一致するため変更しない。

## 注意点・エッジケース（review-test 向け）

1. **ランク上限**：すでに『とくこう』が+6の場合、`battle.modify_stats` が空の変化辞書を返し、
   技は失敗扱いになる（`modify_attacker_stats` が bool の value をそのまま結果に置き換える）。
2. **target="self" の欠落**：まもる・みがわり・マジックコート関連の挙動に直結するため、
   自己対象の変化技を新規実装・レビューする際は必ず確認する。
3. **よこどり**：本プロジェクトはよこどり自体が未実装のため対応不要。
