# 計画書: ベノムショック

## 仕様要約

- どくタイプ / 特殊 / 威力65 / 命中100 / PP12（`src/jpoke/data/ps-champ-ja/moves.json` 準拠） /
  対象: 1体選択 / 優先度0 / 非接触
- 対象が『どく』または『もうどく』状態のとき、威力が2倍になる。
- 追加効果なし。まもる有効、おうじゃのしるし発動、オウムがえし反射可能。

## ハンドラ構成

| 技名 | イベント | priority | subject_spec | 役割 |
|------|----------|----------|--------------|------|
| ベノムショック | `Event.ON_CALC_POWER_MODIFIER` | デフォルト（100） | デフォルト（`attacker:self`） | 対象（`ctx.defender`）が『どく』『もうどく』状態のとき威力を2倍にする |

`Event.ON_CALC_POWER_MODIFIER` は `docs/spec/turn.md` に priority の掲載がないため、
同種の可変威力技（`たたりめ_double_power_when_ailment` / `からげんき_double_power_when_ailment`
等）がいずれも `MoveHandler` のデフォルト値（priority=100, subject_spec="attacker:self"）の
まま登録されていることを根拠に、デフォルト設定のまま登録する。
ハンドラ内部では `ctx.defender.ailment.name` を参照するため、`subject_spec` の値自体は
ロジックに影響しない（`たたりめ` 等の既存の同種ハンドラと同じ扱い）。

## 実装箇所

### data/move.py への更新

`src/jpoke/data/moves/move_ha.py` の既存の `"ベノムショック"` エントリは以下の通り
（PPを10→12へ修正）。

```python
    "ベノムショック": MoveData(
        type="どく",
        category="special",
        pp=12,
        power=65,
        accuracy=100,
        handlers={
            Event.ON_CALC_POWER_MODIFIER: h.MoveHandler(
                ha.ベノムショック_double_power_when_poisoned,
            ),
        }
    ),
```

- `flags` を指定しない → 直接攻撃ではない（仕様と一致）。
- `target` 省略時のデフォルトは `"foe"`（相手1体）→ 仕様と一致。
- `priority` 省略時のデフォルトは `0` → 仕様と一致。

### handlers/move_attack.py への実装

```python
def ベノムショック_double_power_when_poisoned(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    """ベノムショック: 対象がどく/もうどく状態のとき威力が2倍になる。"""
    if ctx.defender.ailment.name in ("どく", "もうどく"):
        value = apply_fixed_modifier(value, 8192)
    return HandlerReturn(value=value)
```

- `たたりめ` の `ailment.is_active` とは異なり、本技は「どく」「もうどく」限定のため
  `ailment.name` を直接タプルと比較する。
- `どくばりセンボン_double_power_when_poisoned` 相当のロジックと同一（本技が元祖であり、
  `どくばりセンボン` が本技のロジックを再利用する関係）。

## リーサル計算

対象がどく/もうどく状態かどうかでダメージが変動するが、リーサル計算（`hp_dist` の変化）
には影響しない威力補正のみのため、`handlers/lethal.py` に専用ハンドラは不要（n/a）。
実ダメージ計算自体はダメージ計算モジュール側の威力補正を通じて反映される。

## 注意点・エッジケース

- 状態異常の判定はダメージ計算前の値を参照するため、本技自身がどく状態を付与する
  効果を持たない（追加効果なし）ことと矛盾しない。
- 相手がまひ・やけど・ねむり・こおりなどどく/もうどく以外の状態異常の場合は威力補正
  なし（`たたりめ` と異なる点）。
