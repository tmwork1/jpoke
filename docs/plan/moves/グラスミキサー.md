# 計画書: グラスミキサー

## 結論: 実装しない

`docs/spec/moves/グラスミキサー.md` の通り、本技は第8世代までは使用可能だったがスカーレット・バイオレットでは
使用不可であり、README.md の対象範囲定義（第9世代で実装されていない技は実装しない）に反するため実装計画は
策定しない。

## 経緯

過去の一括実装計画 `docs/plan/moves/攻撃技_防御側ランク変化系.md`（防御側の能力ランクを下げる追加効果を
まとめて実装するバッチ計画）で、対象範囲の確認が行われないまま以下の内容で登録されていた:

- `src/jpoke/data/moves/move_ka.py` の `MoveData(power=60, pp=10, accuracy=55, ...)`
  - 一次情報（`docs/wiki/moves/グラスミキサー.html`）による正しい基礎威力は65・命中率90であり、
    威力60・命中55という値の出典・根拠は不明
- `src/jpoke/handlers/move_attack.py` の `グラスミキサー_lower_acc`
  （`modify_defender_stats(..., stats={"accuracy": -1}, chance=0.5)`）
- `src/jpoke/types/move.py` の `MoveName` Literal への登録

（`tests/` 側には本技を参照するテストは元々存在しなかった）

本レビュー（`docs/review/moves/グラスミキサー.md`）にて対象範囲外と判断し、上記実装を削除した。
今後、本技を再実装する必要が生じた場合は、まず README.md の対象範囲定義とポケモンチャンピオンズでの
使用可否を再確認すること。
