# 実装計画: ルミナコリジョン

## 仕様サマリ

- タイプ: エスパー / 分類: 特殊 / 威力: 80 / 命中率: 100 / PP: 12（`src/jpoke/data/ps-champ-ja/moves.json` 準拠）
- 対象: defender（原作は1体選択技であり、本プロジェクトはシングルバトルのみのためそのまま単体扱い）
- 効果: 追加効果として、100%の確率で相手の『とくぼう』ランクを2段階下げる
- flags: `secondary_effect`（100%発動の追加効果を持つため、ちからずくの威力上昇・追加効果無効化の対象になる）
- まもる: ○ / マジックコート: ×（変化技ではないため対象外） / 直接攻撃: ×

参照: `docs/spec/moves/ルミナコリジョン.md`、`docs/spec/moves/_move_secondary_1.md`
（対象=defender、D-2=100%の技として掲載済み）。
同種の「攻撃技命中時に防御側ランクを100%下げる」技（Gのちから・ほのおのムチ・りんごさん・
バークアウト等）は `docs/plan/moves/攻撃技_防御側ランク変化系.md` にまとめて計画済みで、
ルミナコリジョンも同ファイルの対象技一覧・実装状況表に含まれている（D-2・100%・`ON_HIT`系の
登録済み技として記載）。本ファイルはルミナコリジョン単体のレビュー時に見つかった残課題
（`secondary_effect` フラグ未付与・PPの移行漏れ・リーサル計算ハンドラの `move_secondary` 判定漏れ）
の修正内容を個別に記録する。

## 対象イベント

`Event.ON_DAMAGE_HIT` — ダメージ適用後に発火する「追加効果発動」イベント。
`Gのちから`・`ほのおのムチ`・`りんごさん` 等、同種の防御側ランク変化技がすべて
`Event.ON_DAMAGE_HIT` で実装済みであり、既存実装と統一する。

## Priority

`MoveHandler` のデフォルト priority=100 を使用（明示指定なし）。
既存の同種ハンドラ（`りんごさん_lower_defender_spd` 等）もすべて priority 未指定で統一されている。

## 実装状況（レビュー時点）

### 既存実装（変更なし）

- `handlers/move_attack.py` の `ルミナコリジョン_sharply_lower_defender_spd`
  （`modify_defender_stats` で `stats={"spd": -2}`, chance未指定＝デフォルト100%）
- `data/moves/move_ra.py` の `Event.ON_DAMAGE_HIT` 登録

### 今回のレビューで修正した点

1. **PP移行漏れ**: `pp=10`（Gen9本家値）になっていたが、`src/jpoke/data/ps-champ-ja/moves.json` では
   `12` であり、チャンピオンズは全技PPを8/12/16/20の4段階に統一しているため `pp=12` に修正。
2. **`secondary_effect` フラグ未付与**: `flags` が未指定だった。100%発動の追加効果
   （とくぼう-2）を持つため、`ちからずく`（威力1.3倍・追加効果無効化）の対象になるべきだが、
   フラグ欠落によりこの相互作用が機能していなかった。`flags={"secondary_effect"}` を追加。
3. **リーサル計算ハンドラの `move_secondary` 判定漏れ**: `handlers/lethal.py` の
   `ルミナコリジョン_lower_spd` が `ctx.move_secondary` を確認せず無条件にとくぼうを
   -2 していた。同種の防御側ランク変化技（`Gのちから_lower_def`・`ほのおのムチ_lower_def`・
   `りんごさん_lower_spd` 等）はすべて `if ctx.move_secondary:` で追加効果の有無を切り替えて
   いるため、これに合わせて修正した。
   - ルミナコリジョンは特殊技で防御側の『とくぼう』を下げるため、後続ヒットのダメージ計算
     （特殊技のダメージ計算式が参照する防御側ステータス）に直接影響する。したがって
     バークアウト（防御側の『とくこう』を下げるため、特殊技のダメージ計算には影響せず
     リーサル計算が `n/a` になるケース）とは異なり、リーサル計算の実装自体は妥当（`x` のまま）。

## テスト方針

- 通常ダメージ確認 + とくぼう2段階低下（`accuracy=100` 固定）
- `ちからずく` との相互作用確認（威力1.3倍・とくぼう低下は発動しない。`バークアウト` の
  既存テストと同パターン）
- リーサル計算: `secondary=True` でとくぼう低下により2発目のダメージが増加、
  `secondary=False` では変化しないことを確認（`りんごさん` の既存テストと同パターン）

## 注意事項

- かがくへんかガス/かたやぶり無効化チェックは書かない
- クエスパトラの専用技だが、実装・テストでは他の特殊アタッカー（クエスパトラ）を使用する
