# 実装計画書: パワーウィップ

参照仕様書: docs/spec/moves/パワーウィップ.md

## 実装方針

パワーウィップは追加効果を持たない攻撃技であり、以下のデータのみで挙動が完結する。
専用ハンドラは不要。

* `type="くさ"`
* `category="physical"`
* `pp=12`（Champions基準。src/jpoke/data/ps-champ-ja/moves.json を参照）
* `power=120`
* `accuracy=85`
* `flags={"contact"}`（直接攻撃。「切る技」ではないため `"slash"` フラグは付与しない）

## 登録

`src/jpoke/data/moves/move_ha.py` の `MOVES_HA["パワーウィップ"]` に上記の値を設定する。
`handlers={}` とする（追加効果なし）。

## リーサル計算

追加効果を持たない通常攻撃技のため、`handlers/lethal.py` への専用ハンドラ追加は不要
（docs/progress/move.md のリーサル実装列は n/a）。

## テスト方針

`tests/moves_attack/test_move_ha.py` に以下を追加する。

* 追加効果なしの物理くさ技で相手にダメージを与えることを確認するテスト
  （`test_ドラゴンハンマー_相手にダメージを与える` と同様のパターン）
