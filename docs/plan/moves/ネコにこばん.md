# 実装計画書: ネコにこばん

## 仕様要約

`docs/spec/moves/ネコにこばん.md` 参照。

* タイプ: ノーマル / 分類: 物理 / 威力: 40 / 命中率: 100 / PP: 20 / 対象: 1体選択
* 非接触の物理技（直接攻撃: ×）。バトル内の追加効果はなし
  （命中後のお金入手はゲームシステム上の演出でありダメージ計算・状態変化に影響しない）。

## 既存実装の確認結果

`src/jpoke/data/moves/move_na.py` に以下の通り登録済みで、仕様と一致していることを確認した。

```python
"ネコにこばん": MoveData(
    type="ノーマル",
    category="physical",
    pp=20,
    power=40,
    accuracy=100,
    handlers={},  # 追加効果なし
),
```

* `target` は指定されていないため `MoveData` のデフォルト値 `"foe"`（相手1体）が適用される。
  仕様の「対象: 相手1体」と一致する。
* `flags` に `"contact"` を含めていない点も、仕様の「直接攻撃: ×」と一致する。
* 追加効果を持たない技のため、`handlers/move_attack.py` への実装は不要。

## リーサル計算

`docs/progress/move.md` の該当行は「リーサル実装: n/a」のため対象外。
追加効果を持たない通常攻撃技であり、汎用のダメージ計算・確定/乱数リーサル計算の
枠組みでカバーされるため、専用のリーサルハンドラは不要。

## テスト

追加効果のない通常攻撃技であるため、以下の基本的な観点のみをテストする
（`tests/moves_attack/test_move_na.py` に追加）。

* 通常のダメージ計算（威力40・物理）でダメージが発生すること。
* 追加効果（状態異常・能力ランク変化等）が一切発生しないこと。
