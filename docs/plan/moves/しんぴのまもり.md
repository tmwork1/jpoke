## 実装箇所

### `src/jpoke/handlers/move_status.py`

五十音順の挿入位置: 「し」ブロック。`じこさいせい_heal_self` と `しっぽをふる_modify_defender_stats` の前後を確認し、「しん」は「じん」扱いで `じゅうりょく_activate_global_field` の後、`ステルスロック_set_field` の前に挿入する。

```python
def しんぴのまもり_set_side_field(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """しんぴのまもり: 自陣営に「しんぴのまもり」を5ターン設定する。"""
    side = battle.get_side(ctx.attacker)
    if not side.activate("しんぴのまもり", 5):
        return HandlerReturn(value=False, stop_event=True)
    return HandlerReturn(value=value)
```

### `src/jpoke/data/move.py`

既存の `"しんぴのまもり"` エントリを更新する。`target` を `"own_side"` に変更し、handlers を追加する。

```python
"しんぴのまもり": MoveData(
    type="ノーマル",
    category="変化",
    pp=25,
    target="own_side",
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.しんぴのまもり_set_side_field,
        ),
    }
),
```

## 注意点・エッジケース

- 現在 `data/move.py` の target は `"field"` になっているため `"own_side"` への変更が必要
- `side.activate` が False を返す（すでにしんぴのまもりが有効）場合は失敗として `HandlerReturn(value=False, stop_event=True)` を返す
- 状態異常無効化の本体は side_field ハンドラ側（`ON_TRY_MOVE_2` priority 130）で処理されており、技側はフィールド設定のみ
- リフレクター・ひかりのかべと同一パターン（5ターン・自陣営）
- `turn.md` の ON_TURN_END に「しんぴのまもりの継続・終了」が priority 130 として記載されている（フィールド管理は side_field 側の責任）
