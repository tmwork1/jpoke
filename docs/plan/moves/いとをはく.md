# 実装計画: いとをはく

## 仕様サマリ

- タイプ: むし / 分類: 変化 / PP: 40 / 命中率: 95
- 効果: 相手のすばやさランクを 2 段階下げる
- まもる: ○ / マジックコート: ○ / 直接攻撃: ×
- 対象: 相手全体

## 現状の実装状況

`data/move.py` にエントリは存在するが `handlers` キーが未設定（ランク低下が未実装）。

```python
"いとをはく": MoveData(
    type="むし",
    category="変化",
    pp=40,
    accuracy=95,
),
```

## ハンドラ構成

| イベント | priority | subject_spec | 役割 |
|---|---|---|---|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self`（デフォルト） | 相手のすばやさを 2 段階下げる |

## Priority 根拠

`docs/spec/turn.md` に `ON_STATUS_HIT` の記載なし。
既存の同種ハンドラ（`こわいかお_modify_defender_stats`（S-2）など）が priority=100 デフォルトで実装されているため、同様に採用する。

## 実装箇所

### handlers/move_status.py への追加

挿入位置: 五十音順で `あやしいひかり_*` と `あまいかおり_*` の間（い→う の前、あ行内での位置）

正確には: `アンコール_*` の直後、`いたみわけ_*` の直前

```python
def いとをはく_modify_defender_stats(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """いとをはくの効果: 相手のすばやさを 2 段階下げる。"""
    return modify_defender_stats(battle, ctx, value, stats={"S": -2})
```

### data/move.py の更新

`いとをはく` の MoveData に `handlers` を追加する。

```python
"いとをはく": MoveData(
    type="むし",
    category="変化",
    pp=40,
    accuracy=95,
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.いとをはく_modify_defender_stats,
        ),
    }
),
```

## 注意点・エッジケース

- ランク低下の無効化（しろいきり、クリアボディなど）は `ON_TRY_MOVE_2` priority=130/140 およびフレームワーク内で処理されるため、ハンドラ側のガードは不要
- 対象が「相手全体」だが、シングルバトルでは相手 1 体のみ。`ctx.defender` で相手を参照するため問題なし
- 命中率 95 のため、テストでは `accuracy=100` 固定で実行する
