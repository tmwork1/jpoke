# のしかかり 実装計画書

## 仕様要約（docs/spec/moves/のしかかり.md 参照）

* タイプ: ノーマル / 分類: 物理 / 威力: 85 / 命中: 100 / PP: 15
* 対象: 相手1体 / 優先度: 0 / 接触あり
* 効果: 追加効果として30%の確率で相手をまひ状態にする。
  ちいさくなる状態の相手には必ず命中し、ダメージが2倍になる（minimize フラグの共通処理）。

## 現状の実装状況

`src/jpoke/data/move.py` に既に登録済み。

```python
"のしかかり": MoveData(
    type="ノーマル",
    category="physical",
    pp=15,
    power=85,
    accuracy=100,
    flags={"minimize", "contact", "secondary_effect"},
    handlers={
        Event.ON_DAMAGE_HIT: h.MoveHandler(
            ha.のしかかり_apply_paralysis_to_defender,
        )
    }
),
```

`src/jpoke/handlers/move_attack.py` にハンドラ実装済み。

```python
def のしかかり_apply_paralysis_to_defender(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return apply_ailment_to_defender(battle, ctx, value, ailment="まひ", chance=0.3)
```

内容は仕様書と一致しており、修正不要。

## ハンドラ構成

| event | priority | subject_spec | 役割 |
|---|---|---|---|
| ON_DAMAGE_HIT | 100（デフォルト） | attacker:self（デフォルト） | 30%の確率で相手をまひ状態にする |

`apply_ailment_to_defender` は `handlers/move.py` の既存共通関数を使用する
（`battle.resolve_secondary_chance` で確率解決後 `battle.ailment_manager.apply` を呼ぶ）。

minimize フラグによる「ちいさくなる状態の相手への必中化・威力2倍」は本技固有のハンドラを持たず、
`handlers/volatile.py` の `ちいさくなる_boost_power`（ON_CALC_POWER_MODIFIER 系）・
`ちいさくなる_guaranteed_hit`（ON_MODIFY_ACCURACY 系）が `ctx.move.has_flag("minimize")` を見て
共通処理する。`flags={"minimize", ...}` を設定するだけで良い。

## Priority 根拠

まひ付与は `Event.ON_DAMAGE_HIT`（ダメージ適用後に発火、`docs/spec/turn.md` には専用イベント名では
掲載されていないが、10まんボルト・ねっぷう・ねんりき等、既存の同種「追加効果」ハンドラがすべて
`ON_DAMAGE_HIT` + デフォルト priority=100 で登録されているため、同一パターンを踏襲する。

## 注意点・エッジケース（review-test 向け）

### まひの無効条件
- でんきタイプはまひ無効。`apply_ailment_to_defender` → `battle.ailment_manager.apply` 内部で判定されるため、
  本技側ハンドラに追加ガードは不要。

### minimize フラグの共通処理
- `flags` に `"minimize"` を含めるだけで、ちいさくなる状態の相手への必中化・威力2倍が自動的に適用される。
- `tests/volatiles/test_volatile_satana.py` の `minimize_enhance_moves` パラメータ化テストに
  「のしかかり」が含まれており、既に検証済み。

### 確率発動のテスト方針
- `secondary_chance` 引数で `battle.random` を制御し、確率発動ケース（発動/非発動）を両方テストする。
- 既存テスト `test_のしかかり_まひが発動しない` / `test_のしかかり_まひが発動する` で検証済み。
