# 実装計画書: あまごい

## 仕様

- タイプ: みず / カテゴリ: 変化 / PP: 8（champions/move_list.txt準拠）
- 命中率: なし（必中） / 対象: field
- 効果: 5 ターン間、天候を「あめ」にする

強天候（おおひでり・おおあめ・らんきりゅう）発動中は `weather_manager.apply` 内で上書き拒否され失敗。
すでに「あめ」の場合も `apply` が `False` を返し失敗。

## イベント・Priority

| イベント | Priority | subject_spec | 役割 |
|----------|----------|-------------|------|
| `ON_STATUS_HIT` | 100（デフォルト） | `attacker:self` | あめ天候を5ターン設定 |

**根拠**: `ON_STATUS_HIT` は `turn.md` に独立エントリなし。`にほんばれ_activate_weather`・`すなあらし_activate_weather` が共に priority=100 デフォルトのため同パターンを採用。

## 実装

### handlers/move_status.py

```python
def あまごい_activate_weather(battle: Battle, ctx: AttackContext, value: Any) -> HandlerReturn:
    return HandlerReturn(value=battle.weather_manager.apply("あめ", 5, source=ctx.attacker))
```

### data/moves/move_a.py

```python
"あまごい": MoveData(
    type="みず",
    category="status",
    pp=8,
    target="field",
    handlers={
        Event.ON_STATUS_HIT: h.MoveHandler(
            hs.あまごい_activate_weather,
        ),
    }
),
```

## 注意事項

- `weather_manager.apply` の戻り値 bool をそのまま `HandlerReturn(value=...)` に渡す
- しめったいわによる 8 ターン延長は `ON_MODIFY_DURATION` ハンドラ（item 側）が担当
- かがくへんかガス/かたやぶり無効化チェックは書かない
