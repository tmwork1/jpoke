# とくせいなし 実装計画書

## 概要

とくせいなし揮発性状態の実装。
対象ポケモンの特性の効果が消える。
`AbilityDisabledReason("とくせいなし")` の追加/削除で制御する。

## 実装対象

### 1. とくせいなし_disable_ability（ON_VOLATILE_START）

**ファイル**: `handlers/volatile.py`
**登録**: `data/volatile.py`（とくせいなし の ON_VOLATILE_START）

- とくせいなし状態付与時に `battle.add_ability_disabled_reason(mon, "とくせいなし")` を呼ぶ。
- subject_spec: `"source:self"`（非攻撃フェーズ）
- priority: 100（デフォルト。turn.md に ON_VOLATILE_START の記載なし。さわぐ等の同種ハンドラーと同様）

### 2. とくせいなし_enable_ability（ON_VOLATILE_END）

**ファイル**: `handlers/volatile.py`
**登録**: `data/volatile.py`（とくせいなし の ON_VOLATILE_END）

- とくせいなし状態終了時に `battle.remove_ability_disabled_reason(mon, "とくせいなし")` を呼ぶ。
- subject_spec: `"source:self"`（非攻撃フェーズ）
- priority: 100（デフォルト）
