## 実装メモ

- ターン終了ごとに素早さランクを1段階下げるだけの定期ランク変化として実装されている。
- ランク変化の無効化や反転は `modify_stat` 側の共通処理に委譲している。

## テスト要件

- [x] ターン終了時に素早さが1段階下がる
- [x] ランク変化ログが出る
- [x] ランク最低時はそれ以上下がらない

※ 特性との相互作用（あまのじゃく等）は特性テスト領域で担保する。volatile 単体テストでは対象外。
