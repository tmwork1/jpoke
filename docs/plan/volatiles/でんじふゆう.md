# でんじふゆう 実装計画書

作成日: 2026-06-20
参照仕様書: docs/spec/volatiles/でんじふゆう.md

## 概要

でんじふゆう揮発性状態のハンドラを実装する。
5ターンの間、ポケモンを「地面にいない」状態にする。

## 実装済み内容

以下のハンドラは実装・登録済み:

| イベント | ハンドラ | subject_spec | priority | 説明 |
|---------|---------|-------------|----------|------|
| ON_CHECK_FLOATING | でんじふゆう_check_floating | source:self | 100 | 浮遊状態として True を返す |
| ON_TURN_END | でんじふゆう_tick_volatile | source:self | 110 | ターン終了時にカウントダウン・解除 |

- `ON_CHECK_FLOATING` は `docs/spec/turn.md` に未掲載のため、既存の同種ハンドラ（ふゆう特性、くろいてっきゅう等）の priority=100 を参照して決定。
- `ON_TURN_END` は `docs/spec/turn.md` の「でんじふゆうの終了」priority=110 に従う。
- `subject_spec="source:self"` は揮発状態を持つポケモン自身がイベント対象のパターン。
  `ON_CHECK_FLOATING` は `EventContext(source=pokemon)` を渡すため "source:self" が正しい。

## 動作確認

- でんじふゆう状態 → `is_floating()` = True → じめんタイプ技を無効化する
- でんじふゆう状態 + くろいてっきゅう → くろいてっきゅうのハンドラが stop_event=True で False を返すため、浮遊は無効
- でんじふゆう状態 + うちおとす状態 → うちおとすのハンドラが stop_event=True で False を返すため、浮遊は無効
- 5ターン経過ででんじふゆう状態が解除される
- まきびし・どくびし・ねばねばネットの効果を受けない（is_floating チェックで免疫）
- フィールド効果の恩恵を受けない（is_floating チェックで除外）

## 実装ファイル

- `src/jpoke/handlers/volatile.py`: `でんじふゆう_check_floating`・`でんじふゆう_tick_volatile` 関数（実装済み）
- `src/jpoke/data/volatile.py`: `ON_CHECK_FLOATING`・`ON_TURN_END` ハンドラ登録（実装済み）
