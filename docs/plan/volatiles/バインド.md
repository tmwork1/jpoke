# バインド 実装計画書

## 仕様概要

- 毎ターン終了時に最大HPの 1/8 ダメージ（Champions第六世代以降仕様）
- 交代・逃げることができない（ゴーストタイプは例外）
- バインドを付与した側が場からいなくなると解除（ON_SWITCH_OUT）
- 解除ターンはダメージなし（tick後にhas_volatileチェック）
- しめつけバンドで1/8→1/6に強化（ON_MODIFY_BIND_DAMAGEイベント経由）
- 継続ターン数: 4〜5ターン（Champions仕様）

## VolatileData 定義 (data/volatile.py)

```python
"バインド": VolatileData(
    handlers={
        Event.ON_TURN_END: h.VolatileHandler(
            h.バインド_damage,
            subject_spec="source:self",
            priority=100
        ),
        Event.ON_CHECK_TRAPPED: h.VolatileHandler(
            h.check_trapped_not_ghost,
            subject_spec="source:self",
        ),
        Event.ON_SWITCH_OUT: h.VolatileHandler(
            h.バインド_remove,
            subject_spec="source:foe",
        ),
    }
),
```

### Priority 根拠

- `ON_TURN_END` priority=100: `docs/spec/turn.md` の「バインドダメージ = Priority 100」に従う
- `ON_CHECK_TRAPPED`: デフォルト (100) — 既存の `にげられない` と同じ
- `ON_SWITCH_OUT`: デフォルト (100) — 既存の `ロックオン` の `source:foe` パターンと同じ

## handlers/volatile.py 実装

### バインド_damage

```python
def バインド_damage(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """バインド状態のターン終了時ダメージ"""
    mon = ctx.source

    # ターンカウント減少
    battle.volatile_manager.tick(mon, "バインド")
    if not mon.has_volatile("バインド"):
        return HandlerReturn(value=value)

    # ダメージ適用（しめつけバンドによる強化をON_MODIFY_BIND_DAMAGE経由で取得）
    r = battle.events.emit(Event.ON_MODIFY_BIND_DAMAGE, ctx, mon.volatiles["バインド"].bind_damage_ratio)
    battle.modify_hp(ctx.source, r=-r)
    return HandlerReturn(value=value)
```

### バインド_remove

```python
def バインド_remove(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """バインド状態のスイッチアウト処理（発生源が交代するとバインドを解除する）"""
    foe = battle.foe(ctx.source)
    battle.volatile_manager.remove(foe, "バインド")
    return HandlerReturn(value=value)
```

## Volatile.__init__ での bind_damage_ratio

バインドを付与する技側（data/move.py または handlers/move.py）で：

```python
battle.volatile_manager.apply(
    defender, "バインド",
    count=random.randint(4, 5),  # Champions仕様: 4〜5ターン
    bind_damage_ratio=1/8,       # しめつけバンド持ちなら 1/6
)
```
