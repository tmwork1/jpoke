# かなしばり 実装計画書

作成日: 2026-06-20
参照仕様書: docs/spec/volatiles/かなしばり.md

## 概要

かなしばり揮発性状態の実装。相手が最後に使用した技を4ターン封じる。
技「かなしばり」（data/move.py）と揮発状態ハンドラ（data/volatile.py）の両方を実装する。

## 実装対象

### 1. かなしばり_can_apply（Event.ON_STATUS_HIT）

**ファイル**: `handlers/move_status.py`
**登録**: `data/move.py`

- 相手が技を使っていない（`executed_move` が None）場合は失敗。
- 相手が直前に使った技がわるあがきの場合は失敗。
- `subject_spec`: `"attacker:self"`（AttackContext）
- `priority`: 100（`docs/spec/turn.md` の `ON_STATUS_HIT` 優先度100 に基づく）

### 2. かなしばり_apply_volatile_to_defender（Event.ON_STATUS_HIT）

**ファイル**: `handlers/move_status.py`
**登録**: `data/move.py`

- 相手に「かなしばり」揮発状態を付与する（count=4、move_name=defender.executed_move.name）。
- `apply_volatile_to_defender` ヘルパーを使用。既にかなしばり状態の場合は `volatile_manager.apply` が自動的に失敗を返す。
- `subject_spec`: `"attacker:self"`（AttackContext）

### 3. かなしばり_modify_command_options（Event.ON_MODIFY_COMMAND_OPTIONS）

**ファイル**: `handlers/volatile.py`
**登録**: `data/volatile.py`

- かなしばり状態のポケモンのコマンド選択肢から、揮発状態に記録された `move_name` と一致する技コマンドを除外する。
- 交代コマンド（`cmd.is_move_family == False`）はそのまま通す。
- 選択肢が空になった場合は `CommandManager` が自動的に `Command.STRUGGLE` を追加する。
- `subject_spec`: `"source:self"`（EventContext）
- `priority`: 100（デフォルト）

### 4. かなしばり_try_action（Event.ON_TRY_ACTION）

**ファイル**: `handlers/volatile.py`
**登録**: `data/volatile.py`

- かなしばり状態で封じた技を実行しようとした場合、行動をブロックする。
- `subject_spec`: `"attacker:self"`（AttackContext）
- `priority`: 100（`docs/spec/turn.md` の `ON_TRY_ACTION` 優先度100 に基づく）

### 5. かなしばり_tick_volatile（Event.ON_TURN_END）

**ファイル**: `handlers/volatile.py`
**登録**: `data/volatile.py`

- ターン終了時にカウントを1減らし、0になったらかなしばり状態を解除する。
- `tick_volatile` ヘルパーを使用。
- `subject_spec`: `"source:self"`（EventContext）
- `priority`: 110（`docs/spec/turn.md` の `ON_TURN_END` 優先度110 に基づく）

## 消滅条件の処理

- 交代によるかなしばり解除は `switch_manager._switch_out` の `remove_all_volatiles` で自動処理される。
- ターン経過は `ON_TURN_END` の `tick_volatile` で処理される（priority=110）。
- バトンタッチによる引き継ぎ禁止はバトンタッチ実装時に対応。

## テスト方針

- 技かなしばり: 未行動の相手には失敗する
- 技かなしばり: わるあがきを使った相手には失敗する
- 技かなしばり: 相手がかなしばり中の場合は失敗する
- 技かなしばり: 成功してかなしばり揮発状態が付与される
- 技かなしばり: かなしばり中の技は使用できない
- かなしばり揮発状態: コマンド制限（封じた技が選択肢から除外される）
- かなしばり揮発状態: ターン経過で解除される
- かなしばり揮発状態: 交代で解除される
- かなしばり揮発状態: 実行ブロック
