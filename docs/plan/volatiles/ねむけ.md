# ねむけ 実装計画書

## 仕様概要

ねむけは技「あくび」で付与される揮発状態。
count=2 のカウントダウン後にねむりへ移行する。

## 実装対象イベント

| イベント | subject_spec | priority | 処理 |
|----------|-------------|----------|------|
| ON_TURN_END | source:self | 110 | カウントを1減少、0になったら消滅→ON_VOLATILE_END 発火 |
| ON_VOLATILE_END | source:self | 100 (デフォルト) | ねむけ解除後にねむり状態を付与 |

priority の根拠:
- ON_TURN_END priority=110: `docs/spec/turn.md` の ON_TURN_END 表より「110 | ねむけ」
- ON_VOLATILE_END priority: デフォルト(100)。同種ハンドラ（ほろびのうた等）に倣う

## handlers/volatile.py に追加する関数

```python
def ねむけ_remove_and_apply_sleep(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    """ねむけを解除してねむりを付与する。

    Champions 仕様: count=2（確率1/3）または count=3（確率2/3）。
    ailment_manager.apply の ON_BEFORE_APPLY_AILMENT により、さわぐ状態中は
    ねむりへの移行が自動的に防がれる。
    """
    count = 2 if battle.random.random() < 1 / 3 else 3
    battle.ailment_manager.apply(ctx.source, "ねむり", count=count)
    return HandlerReturn(value=True)


def ねむけ_tick_volatile(battle: Battle, ctx: EventContext, value: Any) -> HandlerReturn:
    return tick_volatile(battle, ctx, value, volatile="ねむけ")
```

## data/volatile.py に追加する登録

```python
"ねむけ": VolatileData(
    handlers={
        Event.ON_TURN_END: h.VolatileHandler(
            h.ねむけ_tick_volatile,
            subject_spec="source:self",
            priority=110,
        ),
        Event.ON_VOLATILE_END: h.VolatileHandler(
            h.ねむけ_remove_and_apply_sleep,
            subject_spec="source:self",
        ),
    }
),
```

## 配置（五十音順）

- handlers/volatile.py: `ねむけ_remove_and_apply_sleep` と `ねむけ_tick_volatile`
  → `ねをはる_*` 系関数の前、`ニードルガード_*` 系関数の後
- data/volatile.py: `"ねむけ"` エントリ
  → `"にげられない"` の後、`"ねをはる"` の前
