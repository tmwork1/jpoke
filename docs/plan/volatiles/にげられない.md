# にげられない 実装計画書

## 概要

にげられない揮発性状態の実装。
対象ポケモンが交代できなくなる。ゴーストタイプは無効。

## 実装対象

### 1. check_trapped_not_ghost（ON_CHECK_TRAPPED）

**ファイル**: `handlers/volatile.py`
**登録**: `data/volatile.py`（にげられない の ON_CHECK_TRAPPED）

- `ON_CHECK_TRAPPED` イベントで、sourceがゴーストタイプでなければ交代を禁止する（`True` を返す）。
- ゴーストタイプであれば `False` を返し、交代を許可する。
- subject_spec: `"source:self"`（非攻撃フェーズ、交代しようとするポケモン自身）
- priority: 100（デフォルト）
  - 根拠: `ON_CHECK_TRAPPED` は turn.md 未掲載のイベント。
    既存の同種ハンドラ（バインド、ねをはる、あなをほる等）が priority=100（デフォルト）を使用しているため同じ値を採用。
