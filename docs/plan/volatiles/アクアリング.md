# アクアリング — 実装プラン

## 実装状況

✅ **実装済み**

## 効果概要

毎ターン終了時に使用者のHPを最大HPの1/16回復する。

## 使用イベント

| イベント | subject_spec | 効果 |
|---|---|---|
| `ON_TURN_END_3` | `source:self` | 1/16 HP回復 |

## テスト状況

## 実装メモ

- 定期回復のみを担当する単純な volatile で、解除条件は交代・戦闘不能に委譲している。
- 回復は通常の HP 回復経路を通るため、回復阻害系との相互作用は共通実装に従う。

## テスト要件

- [x] ターン終了時に最大HPの1/16だけ回復する
- [x] HP変更ログが出る
- [x] 最大HP時は過剰回復しない
- [x] かいふくふうじ中は回復しない

## 残課題

- 回復阻害との相互作用が未テスト。
