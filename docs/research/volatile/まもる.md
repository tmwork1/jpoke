# まもる（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | まもる（状態変化） |
| 分類 | 状態変化（保護系） |
| 効果概要 | この状態のポケモンは **そのターン中に来たほとんどの技・効果を無効化する** |
| 発生源 | 技「まもる」・「みきり」・「ニードルガード」など |
| 対象 | 技を成功させたポケモン自身 |
| 持続 | **発生したターンのみ** |
| 重ね掛け | なし（同一状態の重複・延長不可） |
| 状態解除 | ターン終了時、対象が交代・ひんし |

## 発動条件 / 無効条件

### 発動条件

- 技「まもる」等を **使用し行動が成立** することで状態が付与される。  
- 使用成功には **行動可能な状態で選択し実行する** 必要がある（行動不能等では状態発生なし）。  
- 状態は **技成功時点で発生** する。 ([turn0search1])

### 無効・付与不能

| 条件 | 結果 |
|---|---|
| 技が **失敗** / 無効化される | 状態発生なし |
| すでにそのターン中に状態が付与済み | 同一状態の再付与・延長なし |
| 対象が交代・ひんし | 状態付与なし・解除 |

## 効果詳細

### 攻撃・効果の無効化

- 状態中、対象は **ほとんどの技や効果の対象とならない／無効化される**。  
- ダメージ・状態異常・能力変化などのほとんどが無効化される。  
  ただし **一部の技・効果は無効化されず、ダメージを受ける／効果を受ける**。  
  例：  
  - いじげんホール／いじげんラッシュ  
  - シャドーダイブ／ゴーストダイブ等の「姿消し系」攻撃  
  - フェイント など「保護破り」技  
  - 場全体・自己対象の技・地形／状態系効果  
  - 特性「ふかしのこぶし」による接触技など  
  これらは無効化対象外として処理される。 ([turn0search1][turn0search8])

### 無効化の範囲

- 対象ポケモンへの **技ダメージ・効果・状態付与** を防ぐ。  
- 天候ダメージ・継続ダメージ・場効果の付与には **影響しない**（例：あられ・やどりぎ等は防がない）。 ([turn0search3])

## 維持と解除条件

| 条件 | 結果 |
|---|---|
| その **ターン終了** | 状態が **解除** |
| 対象が **交代・戦闘不能** | 状態は消失 |
| 行動不能や効果阻害 | 状態発生前に阻害される可能性あり |

## 競合・相互作用

| 条件 / 効果 | 挙動 |
|---|---|
| 保護系成功率低下 | 「まもる」「みきり」「まもる系技」等は **連続使用すると成功率が低下** する（共通計算） |
| 保護破り技 | 「フェイント」「いじげん系」「ゴーストダイブ系」などは **無効化対象外**（防げない） |
| 特性「ふかしのこぶし」 | 接触技について無効化を **無視して攻撃** される場合あり |
| 分身・みがわり | みがわりが優先して防護し、その後 まもる の処理へ進む |

## ターン処理順（概要）

1. 技「まもる」等を **使用・成功**  
2. 対象に **まもる状態付与**  
3. 同ターンの敵技処理時に **無効化判定**  
4. 条件に該当する攻撃/効果を **防ぐ**  
5. ターン終了で **状態解除**

## 補足

- 「まもる」状態は **そのターン限定の保護状態** として内部扱いされ、多くの攻撃や効果を受けない判定となる。  
- 無効化対象外の技・効果は例外として処理され、命中・効果が通常通り発生する。  
- 保護系の連続使用判定は同一判定プールを共有する。 ([turn0search3][turn0search8])
