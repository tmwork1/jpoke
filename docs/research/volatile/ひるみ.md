# ひるみ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ひるみ |
| 分類 | 状態変化（volatile / 一時的行動妨害） |
| 効果概要 | 対象ポケモンが **そのターンのみ技が出せなくなる** |  
| 対象 | 技効果で対象になったポケモン |
| 重ね掛け | **なし**（同一ターン内で重ねても効果は延長・累積しない） |
| 状態解除 | **そのターン終了時** 自動解除 |

## 発動条件 / 無効条件

### 発動条件

- 追加効果として発生する **攻撃技の命中時の効果**。  
  多くは攻撃技の副次効果として確率で発生する。  
- 発生条件は **対象ポケモンが攻撃を受け、かつ先に攻撃した場合のみ**。  
  （ひるみは攻撃を受けてからその同じターンで発動する）:contentReference[oaicite:0]{index=0}

### 無効・発生不能条件

- 発動対象が **先に行動して技を出している場合** はひるみ状態にならない。  
- 特性「せいしんりょく」／「りんぷん」等により **ひるみ無効** の場合、発生しない。  
- 道具「おんみつマント」等の **追加効果無効化** によって副次効果が防がれた場合、発生しない。:contentReference[oaicite:1]{index=1}

## 効果詳細

### 行動妨害

- ひるみ状態になったポケモンは **そのターン中の行動で技を出せない**。  
- 選択した技は **PP消費されない**。  
- 「ひるみ」は **“そのターン分だけの一時状態”** として扱われ、継続しない。:contentReference[oaicite:2]{index=2}

## 維持と解除

| 条件 | 結果 |
|---|---|
| **そのターン中** | 技の発動ができない |
| ターン終了 | ひるみ状態が **解除** |
| 交代・行動不能前の終了 | 状態は **影響なく解除** |

## 相互作用・例外

| 条件 / 効果 | 挙動 |
|---|---|
| **特性「せいしんりょく」** | ひるみ状態に **ならない** |  
| **特性「りんぷん」** | 追加効果自体が防がれ、ひるみ発生を **防止** |  
| **道具「おんみつマント」** | 追加効果防止でひるみが **発生しない** |  
| **サブスティチュート（みがわり）** | ひるみの発生条件が満たされても **みがわりが優先** される場合あり |  
| **優先度・先攻行動** | 先に行動した場合はひるみが **発生しない** |:contentReference[oaicite:3]{index=3}

## ターン処理順（概要）

1. 対象が **ひるみ対象技を受ける**（攻撃が先）  
2. 発動条件が成立すると **ひるみ状態付与**  
3. 対象の行動順判定  
4. ひるみ状態中の対象は **技発動不可**  
5. **そのターン終了時に解除**

## 補足

- ひるみは「状態異常」ではなく **追加効果として発生する一時的な行動妨害状態**。  
- 行動不能は他の状態変化（ねむり・まひ等）とは独立し、処理順が異なる。  
- 第9世代でも基本的に **上記処理体系が継続**。:contentReference[oaicite:4]{index=4}
