# メロメロ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | メロメロ（状態変化） |
| 分類 | 状態変化（Infatuation / 魅了） |
| 効果概要 | 対象ポケモンは **行動時に技が出せない可能性** が付与される |
| 発生源 | 技「メロメロ」・特性「メロメロボディ」・道具「あかいいと」・キョダイホーヨー等 |
| 対象 | 1体（対象ポケモン） |
| 持続条件 | **交代・戦闘不能** するまで持続 |
| 重ね掛け | 再付与 / 延長 **なし** |
| 状態解除 | 交代・戦闘不能・回復判定（道具等） |

## 発動条件 / 無効条件

### 発動条件

- 技「メロメロ」などにより **異性の対象ポケモン** に状態が付与される（性別が同じ、性別不明の場合は付与失敗）。 :contentReference[oaicite:0]{index=0}  
- 特性「メロメロボディ」や道具「あかいいと」などで付与条件に応じて発生する場合あり。 :contentReference[oaicite:1]{index=1}

### 無効・付与失敗

- 技の命中が **無効化 / 防がれた場合** は状態付与されないことがある。 :contentReference[oaicite:2]{index=2}  
- 特性「どんかん」「アロマベール」を持つポケモンは **メロメロ状態にならない**。 :contentReference[oaicite:3]{index=3}  
- 同一対象に既に状態がある場合は **重ね掛け不可**。 :contentReference[oaicite:4]{index=4}

## 効果詳細

### 行動妨害

- メロメロ状態のポケモンが **行動しようとした時点で処理** が入り、対象に対してモーション表示が出る。  
- その後、 **50% の確率でそのターンの技を発動できず行動不能となる**。 :contentReference[oaicite:5]{index=5}  
- 技が出せなかった場合でも、その技の **PP は消費されない**。 :contentReference[oaicite:6]{index=6}

### 行動判定順

- 他の状態異常（ねむり・こおり・こんらん等）がある場合、それらの **処理判定後にメロメロ判定が走る**。 :contentReference[oaicite:7]{index=7}  
  - 例：こんらん中もメロメロ判定は最後に行われる。 :contentReference[oaicite:8]{index=8}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 対象が **交代 / 戦闘不能** | メロメロ状態 **解除** |
| 道具「メンタルハーブ」等による回復 | 状態が **解除** される場合あり |
| 技で回復する場合 | 対象を回復扱いとして解除 |
| 状態保持者または付与元が交代でも継続 | 状態自体は残る | :contentReference[oaicite:9]{index=9}

## 相互作用・例外

- 性別が同じ、または性別不明の場合は **状態付与対象外** となる。 :contentReference[oaicite:10]{index=10}  
- バトンタッチ等による交代でも **状態は解除** される。 :contentReference[oaicite:11]{index=11}  
- 第5世代以降の「へんしん」状態は性別もコピーするが、メロメロ状態中の **性別変化は解除要因にならない**。 :contentReference[oaicite:12]{index=12}  
- 「しんぴのまもり」「ミストフィールド」などでは **メロメロ無効化はされない**。 :contentReference[oaicite:13]{index=13}

## ターン処理順（概要）

1. 技や特性等の効果により対象に **メロメロ状態付与**  
2. 対象の **行動開始判定**  
3. 他状態（ねむり・こおり・こんらん等）の処理  
4. **メロメロの行動妨害判定（50%）**  
5. 行動可否結果に応じた処理継続

## 補足

- 対象が **既に別状態異常を持っていても** メロメロ判定は独立して処理される。 :contentReference[oaicite:14]{index=14}  
- メロメロ状態は **ターン経過では解除されない**（継続条件を満たす限り持続）。 :contentReference[oaicite:15]{index=15}  
- 道具「メンタルハーブ」での回復は、メロメロ状態を解除する主要な手段の一つである。 :contentReference[oaicite:16]{index=16}
