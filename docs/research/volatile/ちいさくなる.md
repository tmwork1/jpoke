# ちいさくなる（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ちいさくなる（状態変化） |
| 分類 | 状態変化（Minimize／縮小） |
| 効果概要 | 対象ポケモンの **回避率ランクを上昇** させ、多くの攻撃が当たりにくくなる |
| 発生源 | 技「ちいさくなる」 |
| 対象 | 技使用者自身 |
| 重ね掛け | **不可**（同一状態の再付与・延長なし） |
| 状態解除 | 交代・戦闘不能・場離脱（バトンタッチ等で消失） |

## 発動条件 / 無効条件

### 発動条件

- 技「ちいさくなる」を **使用して成功** することで、状態が発生する。  
- 技は命中判定なし（必中扱い・自身対象）。  
- 他の行動不能状態や妨害がある場合には **技が成功しないと状態は付与されない**。 :contentReference[oaicite:0]{index=0}

### 無効・付与不能

| 条件 | 結果 |
|---|---|
| 対象がすでに状態中 | 再付与・延長なし |
| 技使用が失敗／防がれた場合 | 状態発生なし |
| 対象が交代・戦闘不能 | 状態発生なし・既存状態も解除 |

## 効果詳細

### 回避率上昇

- 状態中、対象の **回避ランクが +2 段階上昇** する（最大累積 +6 まで）。  
  - 回避率上昇は **通常のランク変化として処理される**。  
  - 過去世代では +1 段階だったが、第5世代以降は +2 段階に調整されている。 :contentReference[oaicite:1]{index=1}

### ダメージ関連影響

- 状態の間は **多くの攻撃が当たりにくくなる**（回避計算に反映）。  
- 一部の技（例：「のしかかり」「ふみつけ」「サンダープリズン」「ドラゴンダイブ」等）は、状態が付与された相手に対して **命中判定なしで命中** し、**威力が2倍** になる仕様が存在する（世代により多少差異あり）。 :contentReference[oaicite:2]{index=2}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 対象が **交代・戦闘不能** | 状態解除 |
| バトンタッチ等の交代技 | 状態解除（回避率ランクの上昇は残る） |
| 天候・フィールド効果 | 状態自体に直接影響はない（回避率ランクのみ影響） |

## 相互作用・例外

| 条件・効果 | 挙動 |
|---|---|
| 回避率補正累積上限 | 他回避率上昇と **合算可能だが最大+6まで** |
| 命中無視技（例：「ほろびのうた」「ロックオン」） | 回避率上昇を無視して命中判定が成立する場合あり |
| バトンタッチ | 回避ランクの上昇は **引き継ぐ** が、状態自体は消失 |
| 特性・場効果 | 特性や場効果での命中補正は回避率と別個に扱われる |

## ターン処理順（概要）

1. 使用者が技「ちいさくなる」を **選択・成功**  
2. 対象の **回避率ランク +2**  
3. 以降の攻撃の命中計算に回避ランクが反映  
4. 状態は **交代・戦闘不能・場離脱まで持続**

## 補足

- 「ちいさくなる」状態は **実体の「物理的縮小」ではなく、内部的に回避率ランクを上昇** させる効果であり、攻撃の命中計算に影響する。  
- 特定の攻撃技に対しては **必中＆威力2倍** になる例外的仕様も存在する。  
- 状態は **内部的フラグとして扱われる** ため、戦闘UIに表示されない場合があるが、回避率ランクには影響する。 :contentReference[oaicite:3]{index=3}
