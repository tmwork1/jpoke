# あめまみれ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | あめまみれ |
| 分類 | 状態変化 |
| 発生方法 | 技「みずあめボム」等による命中 |
| 効果対象 | 対象のポケモン |
| 効果内容 | 各ターン終了時に素早さランクが **1段階低下** |
| 効果持続 | **3ターン**（発生ターンを含む） |
| 重ね掛け | **再付与・延長不可** |

## 発動条件

- 技「みずあめボム」等が命中した際に付与される。  
- 対象が既にあめまみれ状態の場合、**再付与・ターン延長は発生しない**。  
- 付与後は毎ターン終了時に素早さランク低下を付与する状態として処理される。 :contentReference[oaicite:0]{index=0}

## 解除・消滅条件

| 条件 | 結果 |
|---|---|
| 3ターン経過 | あめまみれ状態を解除 |
| 対象ポケモンが交代 | 状態が解除 |
| 対象ポケモンが戦闘不能 | 状態が解除 |

## 効果詳細（素早さランク低下）

- 各ターン終了時に素早さランクを **–1** する。  
- ランクがすでに最低の場合でもターン経過処理は行われるが、ランクは下がらない。  
- 表示テキストはランク変動が発生した場合のみ出る。 :contentReference[oaicite:1]{index=1}

## 競合・相互作用

| 相互作用 | 仕様 |
|---|---|
| 特性「ミラーアーマー」 | ランク低下を **跳ね返す** |
| 特性「クリアボディ」「しろいけむり」「メタルプロテクト」 | ランク低下効果を **無効化** |
| 特性「クリアチャーム」 | ランク低下を **無効化** |
| 特性「あまのじゃく」 | ランク低下が **逆に上昇**として作用 |
| 特性「たんじゅん」 | 素早さランク低下幅が **–2扱いになる**場合あり |
| 特性「まけんき」「かちき」 | 素早さ低下時に **特性発動条件を満たす** | :contentReference[oaicite:2]{index=2}

## ターン処理順（概要）

1. 技「みずあめボム」成功 → あめまみれ状態付与  
2. 行動処理（攻撃・交代等）  
3. ターン終了時 → 素早さランクを低下（–1）  
4. 経過ターン数累積 → 3ターンで状態解除

## 補足

- みがわりなど一部の状態変化防止効果では **あめまみれ状態自体は付与される**場合があるが、ランク低下効果は防がれることもある。  
- 状態付与後のバトンタッチ等では状態は **引き継がれない**。 :contentReference[oaicite:3]{index=3}
