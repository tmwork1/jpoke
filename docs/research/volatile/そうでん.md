# そうでん（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | そうでん（状態変化） |
| 分類 | 技 / 状態変化 |
| 効果概要 | そのポケモンが **使う技のタイプがすべてでんきタイプになる** |
| 持続条件 | **そのターンのみ（ターン終了まで）** |
| 重ね掛け | 再付与・延長は発生しない |
| 状態解除 | ターン終了で自動解除 |

## 発動条件

- 「そうでん」状態は **効果を付与する技**（例：特定の技効果や相手の技）により付与された場合に発生  
- 対象が場に出ている間、そのターン中に使用する技すべてに効果が適用される

## 効果詳細（技タイプ変化）

- その **ポケモンが選択・使用する技すべてのタイプが「でんきタイプ」に変わる**  
- 通常の技タイプは一切無視され、**でんきタイプとして扱われる**
- わるあがきや変化技なども**でんきタイプとして処理**される（対象技すべて）

## 解除条件

- 対象が交代・戦闘不能になる  
- ターン終了時（効果はそのターン限り）

## 競合・相互作用

### タイプ一致・補正との関係

- 技がでんきタイプになるため、**タイプ一致ボーナス（STAB）はでんきタイプで判定される**  
- 元の技のタイプによるSTABや効果は **無効化される**

### 特性・場状態との相互作用

- 技のタイプ変化系の特性（スカイスキン等）と同時に発生した場合、**特性の効果 → そうでん効果の順で適用**  
  - 例：スカイスキンでノーマル技がかくとうに変わる → そうでんでさらにでんきに変わる  
- 音技・変化技もすべてでんきタイプとなるため、実質 **でんき変化技扱い** になる

## ターン処理順（概要）

1. そうでん状態付与  
2. 対象ポケモンの技選択  
3. 技のタイプ変換（すべてでんきタイプとして扱う）  
4. 技処理（ダメージ計算・効果発動等）  
5. ターン終了で状態解除

## 補足

- **持続はそのターン限り** なので、次ターン以降の技には影響しない  
- 変化技・補助技であっても **でんきタイプとして扱う** ため、耐性や特性の影響を受ける  
- ほかの技タイプ変化系効果と同時に存在する場合、効果の優先度に注意して処理する
