# ねむけ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ねむけ（状態変化） |
| 分類 | 状態変化（揮発性 / あくび等による効果） |
| 効果概要 | 次ターン終了時に対象が **ねむり状態になる可能性** を付与 |
| 対象 | 1体の相手ポケモン |
| 効果持続 | **1ターン**（付与後、次ターン終了時に判定実行） |
| 重ね掛け | 重ね不可・延長不可 |
| 状態解除 | 対象交代・戦闘不能 |

## 発動条件 / 無効条件

### 発動条件

- 技「**あくび**」などの効果を受けて状態変化として付与される。  
- 技命中時に状態として付与されるため、技が無効化された場合は発生しない。  
- 対象がすでにねむりや他の主要状態異常を持つ場合、付与が不可。またみがわり中は付与が発生せず、失敗扱いになる。 :contentReference[oaicite:0]{index=0}

### 無効 / 付与失敗

| 条件 | 結果 |
|---|---|
| 対象がすでに状態異常（ねむり・どく・まひ・やけど・こおり等） | ねむけは付与されない |
| 技が無効化（まもる・みがわり等） | 付与されない |
| 対象交代・戦闘不能 | 状態付与は解除/発生しない |

## 効果詳細（ねむり付与判定）

- ねむけ状態が付与されたポケモンは **次ターン終了時に「ねむり状態」へ移行する**。  
- **確定でねむり状態へ** なる（通常の本編ルール）条件で処理される。  
- ねむり状態になる判定前に対象が交代・戦闘不能・他状態異常付与などの条件により防がれた場合、そのねむり移行は **発生しない**。  
- 交代した場合、付与されたねむけ自体が消滅し効果は発生しない。 :contentReference[oaicite:1]{index=1}

## ターン処理順（概要）

1. 技「あくび」等が成功 → 対象に **ねむけ状態付与**  
2. 対象の次ターン行動処理進行  
3. 対象ターン終了後に **ねむり状態判定** 実行  
4. 判定成功時、対象は **ねむり状態へ移行**  
5. 対象が交代/戦闘不能の場合、それらの時点でねむけ状態が解除

## 競合・相互作用

| 効果 / 条件 | 仕様 |
|---|---|
| 対象が他の **主要状態異常** を持つ | ねむけは付与されない |
| みがわり | ねむけ付与は無効（みがわりが消えると後続判定はない） |
| 交代（引っ込める） | ねむけは解除され、次ターンのねむり移行は発生しない |
| ねむけ判定中に状態回復（例: 状態解除系技） | ねむり移行は発生せず解除扱い |

## 補足

- ねむけは「**本編ルールのねむり発動判定前段階**」として扱われ、次ターン終了時にねむり状態へ遷移させるための仮状態である。  
- 過去世代ではねむり付与の直前段階として解釈されるが、付与・判定タイミング処理が別扱いとなる。  
- バトンタッチ等で対象が交代すると **状態は解除される**。 :contentReference[oaicite:2]{index=2}
