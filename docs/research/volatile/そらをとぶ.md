# そらをとぶ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | そらをとぶ（状態変化） |
| 分類 | 状態変化（Semi‑invulnerable状態） |
| 関連技 | わざ「そらをとぶ」 |
| 効果概要 | 使用者が **空高く飛び上がり、一定ターンの間姿が消える（空中状態）**  |
| 対象 | 技使用者自身 |
| 持続 | **1ターン：空中状態 → 翌ターン攻撃実行で解除** |
| 重ね掛け | 重ね不可・延長不可 |
| 状態解除 | 攻撃発動時 / 行動妨害で中断 / 交代・戦闘不能 |

## 発動条件 / 無効条件

### 発動条件

- 技「そらをとぶ」を **使用したターンに行動選択が成功** すること  
- 技の効果が進行している必要あり  
- 戦闘中に自身が **重力（Gravity）状態** の場合は技自体が使用できない（即失敗）  
- 状態付与は **命中の独立判定ではなく技の実行判定** による（過去世代仕様だが同様に扱う） :contentReference[oaicite:0]{index=0}

### 無効 / 付与不能

| 条件 | 結果 |
|---|---|
| 技が **行動不能で選択不可（ねむり・まひ等）** | 使用不可（状態に入らない） |
| 技が **妨害・防がれて行動不能** | 状態は発生しない |
| 重力状態の場 | 技選択不可のため状態付与不能 :contentReference[oaicite:1]{index=1} |

## 効果詳細（空中／姿消し）

### 空中状態

- 使用ターン中に **空中へ飛び上がる**（Semi‑invulnerable）状態に入る  
- この状態中は **多くの技・効果が命中しない / 無効化される** と扱われる  
  - 一般的な攻撃技・変化技は当たらない判定になる  
  - 「かぜおこし」「たつまき」等の風系技は **命中し、その威力が倍増** する場合あり  
  - 「かみなり」等の一部遠距離技は **通常通り命中判定** する  
  - 「ロックオン」「ノーガード」の影響下では命中する条件あり :contentReference[oaicite:2]{index=2}

### 生存・無効復活条件

- 空中状態は **1ターンのみ継続**  
- 使用者が **状態中に攻撃を受ける/攻撃する** いずれも状態解除判定にはならず、翌ターンの攻撃処理時に解除  
- 空中状態中でも **すなあらし/あられ等の天候ダメージ** を受けることがある（世代依存の挙動） :contentReference[oaicite:3]{index=3}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 翌ターンに「そらをとぶ」攻撃実行 | 状態解除（攻撃成立） |
| 対象が **交代・戦闘不能** | 状態解除・行動終了 |
| 行動不能で中断（混乱自傷等） | 状態効果は**失敗扱い・解除** |
| 攻撃キャンセル（妨害等） | 状態解除 |

## ターン処理順（概要）

1. 技「そらをとぶ」を選択・行動成功  
2. 現ターンに **空中状態へ移行**  
3. 空中状態中の命中判定処理  
4. 翌ターン：攻撃発動 → **状態解除**

## 相互作用・例外

| 条件・効果 | 挙動 |
|---|---|
| 特性「ノーガード」等 | 空中中でも命中判定が成立する場合あり |
| 命中補正効果（ロックオン等） | 空中状態でも判定影響あり |
| 他の半透明技（ゴーストダイブ等） | 同類の状態と処理競合なし（別管理） |
| すなあらし/あられ | 空中でも継続ダメージ発生可能 :contentReference[oaicite:4]{index=4} |

## 補足

- そらをとぶの状態変化は **「空中にいる」ことで多くの攻撃を回避する**仕組みであり、命中無効判定を主目的とする  
- 状態は **技の1ターン目実行時点で発生し、翌ターンの攻撃実行で自動解除** される。 :contentReference[oaicite:5]{index=5}
