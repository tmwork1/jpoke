# じゅうでん（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | じゅうでん（状態変化） |
| 分類 | 状態変化 |
| 効果概要 | 次に使う電気タイプ技の威力を **2倍** にする |
| 対象 | じゅうでん状態のポケモン自身 |
| 効果持続 | **次に選択した電気タイプ技が発動するまで**（選択時に解除） |
| 重ね掛け | 重ねても効果は **上昇せず** 1回分のみ |

## 発動条件

- 「じゅうでん」状態が付与される技・特性の効果を受けたときに発生  
- 状態付与後、対象ポケモンは次に電気タイプ技を使用するまで効果が残る :contentReference[oaicite:0]{index=0}

## 効果詳細

- 「じゅうでん」状態中に **でんきタイプの技を選択した瞬間に状態は解除** される。  
- 選択した電気技が命中しない・無効化された場合でも、技を選んだ時点で状態解除となる。  
- 電気以外の技を選んだ場合は状態は継続する :contentReference[oaicite:1]{index=1}

### 威力補正

- 電気タイプ技の **威力は2倍（×2.0）** になる。  
- 特性「テクニシャン」など威力補正との兼ね合いは **技の最終威力に対して適用される** :contentReference[oaicite:2]{index=2}

## 解除条件

| 条件 | 結果 |
|---|---|
| 電気タイプ技を **選択したとき** | 状態は **消滅** |
| ポケモンが **交代・戦闘不能** | 状態は消滅 |
| 技命中失敗・無効化 | 選択後の解除は発生 |
| バトンタッチ等で場を離れる | 状態は消滅 |

## 競合・例外

- 技のタイプ変化（タイプ変化する技や特性）により **電気タイプとなった技**でも補正適用され、選択時に状態は解除される :contentReference[oaicite:3]{index=3}  
- 行動不能（状態異常等）で技が出せない場合も、 **でんき技を選択した時点で解除扱い** になる  
- 状態が付与された後、でんき技以外を選び続けても **状態そのものは消えない** :contentReference[oaicite:4]{index=4}

## ターン処理順（概要）

1. じゅうでん状態付与  
2. 対象ポケモン行動選択  
3. 電気タイプ技を **選択した瞬間に状態が解除**  
4. 技処理へ進行（ダメージ等計算に2倍補正を反映）
