# さわぐ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | さわぐ（状態変化） |
| 分類 | 状態変化（強制行動・異常系） |
| 効果概要 | 対象ポケモンが **技「さわぐ」を強制使用し続け、交代や他行動ができなくなる状態** |
| 発生源 | わざ「さわぐ」の命中成功 |
| 対象 | 使用者自身 |
| 継続ターン | **3ターン**（第5世代以降） |
| 重ね掛け | **不可**（同一状態は延長・重複なし） |
| 状態解除 | 交代・戦闘不能・状態中途解除条件発生 |

## 発動条件 / 無効条件

### 発動条件

- 技「さわぐ」を **対象に命中させる** と状態変化が付与される。  
- 命中前に防御系（まもる等）で無効化された場合や命中しなかった場合は、状態は付与されない。  
- 対象がすでにさわぐ状態の場合は **再付与・延長なし**。 :contentReference[oaicite:0]{index=0}

### 無効・付与不可

| 条件 | 結果 |
|---|---|
| 技の命中が無効化された場合 | さわぐ状態は発生しない |
| 対象が交代・戦闘不能 | 状態は付与されない |
| 技発動不能な状態（ねむり等） | 技自体が選択不可・状態付与不能 |

## 効果詳細

### 強制行動

- さわぐ状態のポケモンは **さわぐ以外の技・にげる・交代・道具使用等の行動が選択不可** となる。  
- 行動はスキップされず **毎ターンさわぐを自動発動** する仕様となる。  
- 行動不能・交代は **さわぐ中は基本的にできない**（例外行動は別扱い）。  
- 状態中の行動選択は固定化されるため、他の技・行動は発生しない。 :contentReference[oaicite:1]{index=1}

### ねむり関連特殊処理

- さわぐ状態のポケモンがいる間、**場のポケモンは「ねむり」状態にならない**。  
- ねむるなどの技でねむり状態にしようとしても **失敗扱い（眠れない）** となる。  
- 既にねむり状態のポケモンは、さわぐ付与時に**目を覚ます**（ただし世代仕様により詳細差異あり）。  
- さわぐ状態中に出てきたポケモンは、**ねむり状態のまま出てくる場合があるが** ねむりへの移行はその後発生しない。 :contentReference[oaicite:2]{index=2}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 3ターン経過 | さわぐ状態が解除され、通常行動可能になる |
| 対象が交代・戦闘不能 | 即座に状態解除 |
| 使用者が状態異常等で行動不能 | 状態が中断・解除されることがある（世代仕様依存） |

## 競合・相互作用

| 条件・効果 | 挙動 |
|---|---|
| みがわり等防御系 | 技「さわぐ」の攻撃判定は貫通する場面あり（技仕様依存） |
| 特性「ぼうおん」 | 音技無効化特性等による無効化は技側の影響となる |
| 技封印や強制交代技 | 状態中でも強制交代等で解除可能 |
| ねむけ・ねむり系状態 | さわぐが優先して「ねむり無効化」を行う |
| 他状態変化技との複合 | 別状態の付与は単体管理され、処理順が影響する可能性あり | :contentReference[oaicite:3]{index=3}

## ターン処理順（概要）

1. 技「さわぐ」が **命中成功** → さわぐ状態付与  
2. さわぐ状態中のターンに **強制的にさわぐを発動・行動固定**  
3. 各ターン終了時に **残存ターン数を減少**  
4. 3ターン経過で **状態解除**

## 補足

- 第3〜4世代と第5世代以降で継続ターン数やねむり無効化挙動に世代差異があるが、基本は状態中の強制行動とねむり防止が中心である。  
- さわぐ状態は **行動選択固定・交代不可** という点で他の強制系技状態と同様に扱われる。 :contentReference[oaicite:4]{index=4}
