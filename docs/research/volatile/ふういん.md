# ふういん（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ふういん（状態変化） |
| 分類 | 状態変化（技「ふういん」による付与） |
| 効果概要 | 状態保持者が覚えている技を **相手ポケモンが使用できなくする** |
| 対象 | 状態保持者／相手の場のポケモン全員に影響 |
| 持続 | 状態保持者が **交代／ひんしになるまで** |
| 重ね掛け | 既に付与されている場合、**重ね不可・延長なし** |
| 状態解除 | 状態保持者の交代・ひんし・特定解除効果 |

## 発動条件 / 無効条件

### 発動条件

- 技「ふういん」を **使用して成功すること**  
- 命中判定は存在しない（技効果で成功すれば付与）  
- すでに同状態であっても再付与は発生しない  
- みがわり等の防御判定では付与に影響しない  
- バトンタッチ等で状態を引き継ぐことはできない :contentReference[oaicite:0]{index=0}

### 無効条件

- 技が成功しなかった場合は状態付与されない  
- 対象（状態保持者）が場に存在しない場合は付与・維持されない  
- 付与対象がすでに同じ状態の場合は重ね掛け不可 :contentReference[oaicite:1]{index=1}

## 効果詳細

### 技封印（禁止）処理

- 状態中、**状態保持者が覚えている技** は相手側のポケモンが **選択・使用不可** になる  
- 使用不可になった技を選択した場合、**「〜はふういんで〜がだせない！」** と表示され、選択キャンセルとなる  
- **PP消費なし** で技は使えない状態となる  
- わるあがきは **覚えている技扱いではないため、封印の対象にならない**  
- 状態保持者が技構成を変更した場合、封印対象の技もそれに応じて変動することがある :contentReference[oaicite:2]{index=2}

## 維持と解除条件

| 条件 | 結果 |
|---|---|
| 状態保持者が **交代／戦闘不能（ひんし）** | ふういん状態は **解除** |
| 状態解除効果（例：メンタルハーブ等） | 解除される可能性あり（技依存・世代仕様） |
| バトンタッチなどの交代技 | **解除される**（引き継ぎなし） :contentReference[oaicite:3]{index=3}

## 相互作用・例外

- 状態保持者が **ものまね／スケッチ／へんしん** 等で技構成を変化させても、封印対象は構成に応じて変動する場合あり :contentReference[oaicite:4]{index=4}  
- 相手が封印対象の技しか覚えていない場合、わるあがきを使用するシーンに移行することがある（選択技なし） :contentReference[oaicite:5]{index=5}  
- ローテーションルールや複数ポケモン戦闘でも影響対象は状態保持者の覚えている技全体におよぶ :contentReference[oaicite:6]{index=6}

## ターン処理順（概要）

1. 技「ふういん」を使用 → 状態付与  
2. 対象ポケモンに **ふういん状態** が発生  
3. 以後の各ターンにおいて、相手が技選択時に封印対象の技は **選択不可**  
4. 状態保持者が交代・ひんし・解除効果で **状態解除**

## 補足

- ふういんは **攻撃技／変化技を問わず** 対象の技を封じるため、戦術的に広範囲な妨害効果となる  
- 技名称の一致だけでなく、**覚えている技そのものを封じる** 仕様であることに注意が必要 :contentReference[oaicite:7]{index=7}
