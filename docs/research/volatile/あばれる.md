# あばれる（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | あばれる（状態変化） |
| 分類 | 状態変化（強制行動系） |
| 効果概要 | ポケモンが **一定ターンの間技を自動で連続使用し、終了後にこんらん状態になる** |
| 技源 | 技「あばれる」などが原因で発生 |
| 対象 | あばれる状態になったポケモン自身 |
| 持続 | **2〜3ターン（ランダム）** |
| 重ね掛け | 再付与・延長 **なし** |
| 状態解除 | 交代・ひんし・行動不能・指定ターン終了 |

## 発動条件 / 付与

### 発動条件

- 技「あばれる」「げきりん」「はなびらのまい」などの技を **使用成功** した際に発生。  
- 技によって誘発されるため、対象が技を選択して行動成功する必要あり。  
- みがわり等の防御効果は **状態の発生には影響しない場合あり**（技依存）。 :contentReference[oaicite:0]{index=0}

### 無効・付与不可

| 条件 | 結果 |
|---|---|
| 技が **妨害・無効化された場合** | 発生しない場合あり |
| ポケモンが **行動不能** の場合 | 発生しない |
| すでに **同状態中** の場合 | 重ね掛け不可 |

## 効果詳細

### 自動行動

- あばれる状態中は **そのポケモンは強制的に技を連続使用** する（行動選択不可）。  
- 一般的な行動はキャンセルされ、**交代・バッグ・道具使用・にげる等が不可** になる。  
- 技の対象は対戦形式や状況に応じてランダムまたは仕様により決定される（シングル：相手1体）。 :contentReference[oaicite:1]{index=1}

### 持続カウント

- あばれる状態の **継続ターン数は発生時に決定** される。 :contentReference[oaicite:2]{index=2}  
- 継続中に **行動不能（ねむり・まひ・状態変化）や技失敗・命中しない状態** になると、状態は**途中解除** されることがある。 :contentReference[oaicite:3]{index=3}

### 終了時の効果

- 持続カウントが終了すると、対象は **こんらん（Confusion）状態** になる（特性「マイペース」等を除く）。  
- こんらん状態の付与は **戦闘中であればほぼ確定** で発生する仕様。 :contentReference[oaicite:4]{index=4}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 指定ターン経過 | 状態終了 → こんらん状態 |
| 交代・戦闘不能 | 状態解除 |
| 行動不能で途中終了 | 状態解除（こんらん付与なしのケースあり） |

## 競合・相互作用

| 条件・効果 | 挙動 |
|---|---|
| 特性「マイペース」 | **こんらん付与が無効** |
| みがわり状態 | あばれる状態は発生するが、みがわりの影響を別途処理 |
| 命中判定失敗・技無効 | 継続中に発動不能となった場合は強制解除される可能性あり |
| 状態変化（ねむり等） | 状態変化で動けなかった場合、**状態解除されるケースあり** | :contentReference[oaicite:5]{index=5}

## ターン処理順（概要）

1. 技誘発 → あばれる状態発生  
2. あばれる状態中の **強制行動処理**  
3. 各ターン終了時に **残り持続ターン減少**  
4. 終了時に **こんらん状態へ遷移**

## 補足

- あばれる状態は **戦闘行動を制限する特殊な状態変化** として扱われ、行動選択が完了するまで強制的に行動処理が続く。  
- 発生ターン数は技誘発時点で決定され、途中で変更されない仕様。  
- 技の性質・対戦形式によって 挙動詳細や対象判定 が微妙に変わる可能性あり（ダブルバトル等）。 :contentReference[oaicite:6]{index=6}
