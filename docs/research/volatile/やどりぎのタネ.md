# やどりぎのタネ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | やどりぎのタネ（状態変化） |
| 分類 | 状態変化（技「やどりぎのタネ」などによる付与） |
| 効果概要 | 対象に寄生のタネを植え、**毎ターン終了時にHPを奪う状態** |
| 種別 | ダメージ・回復状態変化 |
| 発生源 | 技「やどりぎのタネ」等 |
| 対象 | 単体ポケモン |
| 重ね掛け | 付与済みの場合、**再付与・延長なし** |
| 状態解除 | 交代 / 戦闘不能 / 解除効果 |

## 発動条件 / 無効

### 発動条件

- 技「やどりぎのタネ」が対象に**命中成功**した場合に付与される。  
- 対象が**くさタイプ**の場合は付与失敗（効果が発生しない）。  
- 対象がすでに同状態の場合は付与されない。 :contentReference[oaicite:0]{index=0}

### 無効条件

- 技が**無効化/防御**（まもる等）された場合、状態付与は発生しない場合がある。  
- 状態防止特性・場効果がある場合、付与されないことがある（個別判定要）。 :contentReference[oaicite:1]{index=1}

## 効果詳細

### ダメージ処理

- 各ターン終了時に、状態対象は **最大HPの1/8 を失う**。  
- ダメージ量は小数点以下切り捨て、**最低1ダメージ発生**。  
- 失った分のHPは **やどりぎのタネを付与した位置の味方ポケモンが回復**する。  
- 付与したポケモンが交代しても、**場のその位置にいるポケモン**に回復する。  
- 回復量は奪われたダメージと同じ値。  
- 状態対象が場を離れた場合、ダメージ効果は**消滅**。  
- くさタイプへの付与後にタイプ変化した場合でも、そのまま状態効果でHPを吸収される。 :contentReference[oaicite:2]{index=2}

### 回復挙動

- 回復は対象が失ったHP分だけ行われる。  
- アイテム「おおきなねっこ」を持つ回復対象では、**回復量が1.3倍** になる。  
- 回復対象が不在の場合、HP吸収・回復処理は発生しない。 :contentReference[oaicite:3]{index=3}

## 維持と解除

- 状態は対象が**場にいる限り継続**する。  
- 対象が**交代・戦闘不能**になると状態解除。  
- 技・効果で状態解除された場合も解除される。 :contentReference[oaicite:4]{index=4}

## ターン処理順（概略）

1. 技「やどりぎのタネ」が命中 → 状態付与  
2. ターン終了時フェーズで **HP奪取・回復処理**  
3. 次ターン以降も同様に処理（条件満たす限り）

## 競合・相互作用

| 効果・条件 | 挙動 |
|---|---|
| みがわり・防御系 | 状態付与に影響（付与判定要） |
| くさタイプ | 状態付与が無効化（対象にならない） |
| バトンタッチ後 | 状態は **引き継がれる** |
| リキッドオーズ特性 | 回復が**ダメージ扱いで反転される**可能性あり※（ポケットモンスターSV仕様） |

## 注意点

- やどりぎのタネは **状態変化扱いのため回復阻止系（かいふくふうじ）とは独立**に処理される。  
- 状態発生・回復優先度等は世代仕様に準拠。
