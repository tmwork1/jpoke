# のろい（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | のろい（状態変化） |
| 分類 | 状態変化（呪い状態） |
| 効果概要 | 対象ポケモンが **毎ターン終了時に HP を最大値の1/4 減少** する状態変化 |
| 効果種別 | ダメージ持続系（状態変化；揮発性扱い） |
| 持続条件 | 対象ポケモンが **その場に留まる限り** 持続 |
| 重ね掛け | **重ね不可 / 延長不可** |
| 状態解除 | 交代 / 戦闘離脱（戦闘不能）で解除 |

## 発動条件 / 無効条件

### 発動条件

- 技「のろい」を **ゴーストタイプのポケモンが使用して命中** した場合に付与される。  
  - この技は対象命中に関して **必中（命中率・回避影響を受けない）** として扱われる（みがわりも貫通）*¹*。  
- 対象がすでにのろい状態の場合、追加の状態付与・延長は発生しない。

### 無効・付与失敗

| 条件 | 結果 |
|---|---|
| 技が命中前に **無効化（まもる等）** | 状態付与は発生しない |
| 対象が既に **同状態** | 重ねて付与されない |
| 対象が場に不在（交代・戦闘不能） | 付与対象となれない |
| ゴースト以外がのろい技を使った場合 | 状態付与は発生せず、別効果となる（能力変化等） |

## 効果詳細（ダメージ処理）

- のろい状態の対象は **毎ターン終了時に最大HPの 1/4 のダメージを受ける**（小数点以下切り捨て）。  
  - たべのこし・くろいヘドロ 等の **回復効果後** にのろいダメージ処理が適用される判定順となる世代仕様あり*²*.  
  - ダメージが 0 にならない場合でも **最低1ダメージ** は発生。

- のろいによるダメージは **タイプ耐性・防御種族値等の影響を受けない割合ダメージ** である。

## 維持と解除条件

| 条件 | 結果 |
|---|---|
| 対象ポケモンの **交代 / 戦闘不能** | のろい状態は解除 |
| バトンタッチでの交代 | のろい状態は後続ポケモンへ **引き継がれる** |
| ターン経過だけでは解除されない | 理由なしでターン消滅はない |
| 状態解除効果（例：メンタルハーブ） | のろい状態の解除対象となる可能性あり（効果依存） |

## 相互作用・例外

| 条件・特性 | 挙動 |
|---|---|
| 特性「マジックガード」 | のろいによるダメージを **受けない** |
| 交代技（ボルトチェンジ等） | のろいは **解除される**（バトンタッチを除く） |
| みがわり | のろいは **みがわりを貫通** して状態付与される |
| 他ダメージ持続系 | 毎ターン処理内で並列判定される（順序はゲーム仕様に依存） |

## ターン処理順（概要）

1. ゴーストタイプの「のろい」が成功 → 対象へ状態付与  
2. 対象ポケモンが場に留まる限り状態は保持  
3. 各ターン終了時に **最大HP × 1/4 ダメージ** を発生  
4. 交代・戦闘離脱により状態解除

## 補足・注意点

- **ダメージは割合処理** であり、HP 値に対する割合で処理されるため、防御等の影響は受けない。  
- 対象が HP を 0 にするようなダメージを受けるタイミングでは、そのターンの行動の成立に影響する場合がある。  
- 元々の技「のろい」には使用者が **ゴースト以外の場合は能力変化効果** があるが、ここでは状態変化部分（target curse）にフォーカスしている。
