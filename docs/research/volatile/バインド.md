# バインド（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | バインド（状態変化） |
| 分類 | 状態変化（Trapping / 拘束） |
| 効果概要 | 対象ポケモンが **交代・逃走できなくなり、毎ターン終了時に継続ダメージを受ける** |
| 効果種別 | ダメージ付き拘束状態 |
| 対象 | 技で対象に命中したポケモン |
| 重ね掛け | **不可**（既に状態がある場合は延長・重複なし） |
| 状態解除 | 規定ターン経過 / 条件達成 / 技等で解除 |

※ 状態変化扱いは **第2世代以降**。第1世代では仕様差異あり。 ([turn0view0])  

## 発動条件 / 無効条件

### 発動条件

- 技等によって対象に **バインド状態が発生**  
- 代表的に「しめつける」「まきつく」「からではさむ」「うずしお」「ほのおのうず」「すなじごく」「まとわりつく」「トラバサミ」「サンダープリズン」などで発生。  
- 命中前に防御技（まもる等）で無効化された場合、状態は発生しないことがある。  
- 対象がすでにバインド状態の場合、**新たな付与・延長はされない**。 ([turn0view0])

### 無効・付与不能

| 条件 | 結果 |
|---|---|
| 対象がすでに **状態中** | 再付与・延長なし |
| 技が 命中無効化（まもる等） | 発生しない場合あり |
| 対象が 交代・戦闘不能 | 発生しない |

## 効果詳細

### 拘束効果

- 対象は **「いれかえる」・「にげる」コマンドが選択不可** となる（逃走・交代不可）。  
- 第9世代では **プレイヤーがテレポートを使う場合は逃走が成功可**（野生相手は失敗扱い）。  
- 技「こうそくスピン」「きらースピン」等で **バインド状態を解除可能**。  
- バインド状態でも **バトンタッチ等の技による交代は可能**（状態が解除される）。  
- 特性「にげあし」や持ち物「けむりだま」を持つ場合でも **逃走が可能になる場合あり**。  
- ゴーストタイプは **交代・逃走可能**。  
- みがわり成功でも **状態が解除** される。  
- 状態付与者が 交代・ひんし になると基本的に状態解除（※キョダイ技由来は例外）。 ([turn0view0])

### 継続ダメージ

- 各ターン終了時に **継続ダメージを受ける**：  
  - **第5世代以降**：最大HPの **1/8** の割合ダメージ  
  - **持ち物「しめつけバンド」** 装備時は **1/6** の割合ダメージ  
- **特性「マジックガード」** の場合、継続ダメージを **受けない**。  
- 継続ダメージは **ターンごとに発生**。  
- **解除ターン（状態が消えるターン）** ではダメージは発生しない。 ([turn0view0])

## 維持と解除

| 条件 | 結果 |
|---|---|
| 規定ターン経過 | 状態解除 |
| 技「こうそくスピン」「きらースピン」等 | 状態解除 |
| みがわり成功 | 状態解除 |
| 状態付与者が **交代 / ひんし** | 状態解除（例外あり） |
| バトンタッチ等で **交代** | 状態解除 |
| 持ち物 / 特性により **逃走可能** 条件満足 | 状態解除可能 |
| ゴーストタイプの対象 | 逃走・交代可能（状態は解除） |

## 競合・相互作用

| 条件・効果 | 挙動 |
|---|---|
| 特性「にげあし」等 | 状態中でも **逃走判定可能** |
| 持ち物「けむりだま」等 | **逃走判定可能** |
| 特性「マジックガード」 | **継続ダメージを受けない** |
| キョダイマックス技由来のバインド | 使用者が場を離れても継続可能 |
| みがわり | 状態を **解除** |

## ターン処理順（概要）

1. 技が命中 → 対象に **バインド状態付与**  
2. 対象は **交代・逃走不可状態** となる  
3. 各ターン終了時に **継続ダメージ処理**  
4. 規定ターン経過や解除条件で **状態消滅**

## 補足

- バインドは **交代・逃走を封じつつ継続ダメージを与える状態変化** であり、継続ダメージは割合ダメージで処理される。  
- 状態付与者が場を離れた場合でも **一部キョダイ技由来では継続する**ものがある。  
- 第1世代では挙動が大きく異なり（拘束中に攻撃できない等）別扱い。  
- バインドは **他の状態変化と独立** に処理される（追加付与・上書きなし）。 ([turn0view0][turn0search11])
