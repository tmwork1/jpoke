# ロックオン（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ロックオン（状態変化） |
| 分類 | 状態変化（わざ「ロックオン」による付与） |
| 効果概要 | 付与対象に対して、**「ロックオン」を付与した側の次の攻撃が必ず命中するようになる」** 状態 |
| 対象 | 1体の相手ポケモン |
| 効果持続 | **次に使われる1回の攻撃まで**（あるいは次ターン終了まで） |
| 重ね掛け | 再付与・延長 **不可** |
| 状態解除 | 次の攻撃発生後、対象交代、対象戦闘不能、付与者交代等で解除 |

## 発動条件 / 無効条件

### 発動条件

- 技「ロックオン」を **対象に命中させる** ことで付与される。  
- 命中が成立していない場合、状態は付与されない。  
- 対象に「みがわり」がある場合、状態付与は **失敗する**（みがわり越しでは付与されない）。  
- 同一対象にロックオン状態が既にある場合、再付与・延長は発生しない。 :contentReference[oaicite:0]{index=0}

## 効果詳細（必中判定）

- 状態付与後、その対象に対してロックオンを付与した側が行う**次の攻撃技は命中率判定を無視して必中扱い**となる。  
  - 回避ランク・命中ランクの変動に関係なく命中する。  
  - 対象が「そらをとぶ」「ダグトリオのかくれる」などの半透明状態にいる場合も、ロックオン効果が適用される攻撃は必中となる。  
- 状態は**1回の攻撃ヒット後に解除**される。  
- 対象が交代・戦闘不能になった場合も状態は解除される。  
- 「ロックオン」で付与された必中効果は**次ターン終了時まで有効**で、ターン経過で消失する場合がある。 :contentReference[oaicite:1]{index=1}

## 無効・例外条件

| 条件 | 挙動 |
|---|---|
| 対象が「みがわり」を持つ | ロックオン付与は **失敗する**（状態にならない） |
| 状態付与前に命中が **無効化（まもる等）** | 付与されない |
| 状態付与後に対象が **交代 / 戦闘不能** | 状態は **解除** |
| ロックオン付与者が交代した場合 | 状態は一般的に **解除** または消失扱いとなる |

## 競合・相互作用

- 他の命中補正効果（例：「きあいのタスキ」「必中効果の状態異常」）と**併用可能**だが、必中条件同士の処理は個別判定される。  
- ロックオンの必中効果は保護系（まもる等）の判定前後に関係なく命中率判定を無効化するが、**保護技自体を貫通するわけではない**。  
- バトンタッチによる引き継ぎは一般に行われない（付与者の交代で無効化）。 :contentReference[oaicite:2]{index=2}

## ターン処理順（概要）

1. 技「ロックオン」が命中した → 状態付与  
2. 次ターン以降に対象に攻撃技を行う  
3. 技選択時に必中判定が適用  
4. 技が実行されるとロックオン状態は解除  
5. 解除後は通常の命中/回避処理に戻る

## 補足・注意点

- ロックオンの必中効果は **対象への1回攻撃のみ** に限定され、それ以降の技には影響しない。  
- 状態は**ターン持続型ではなく「1回攻撃保証」** 型であり、次の攻撃が発生した時点で消失する。  
- 技「ロックオン」自体は命中率補正のあるわざだが、状態自体は命中補正とは独立した必中判定処理となる。 :contentReference[oaicite:3]{index=3}
