# マジックコート（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | マジックコート（状態変化） |
| 分類 | 状態変化（変化技・状態効果） |
| 効果概要 | 付与ポケモンが **そのターンに受ける変化技を無効化し、使用してきた相手にその技を跳ね返す** |
| 発生源 | 技「マジックコート」／特性「マジックミラー」等 |
| 対象 | 自分に状態が付与されたポケモン |
| 持続条件 | 状態付与ターン終了まで |
| 重ね掛け | 同一状態の重ね掛け・延長は不可 |
| 状態解除 | ターン終了・交代・ひんし・効果終了 |

## 発動条件 / 無効条件

### 発動条件

- 技「マジックコート」使用成功により状態が発生。  
- 状態は **そのターン中のみ有効**。  
- 特性「マジックミラー」持ちが場にいる場合、**常時マジックコート状態として扱われる**。  
- すでに状態付与中の場合は再付与・延長なし。  
- 命中判定不要（命中率・回避影響なし）。 :contentReference[oaicite:0]{index=0}

### 無効 / 非付与条件

- 技選択が失敗した場合、状態は発生しない。  
- 交代・ひんし等で場にいない場合、状態付与は発生しない（解除扱い）。  
- すでに状態がある場合、同一付与は発生しない。 :contentReference[oaicite:1]{index=1}

## 効果内容（跳ね返し）

### 無効化と跳ね返し

- 状態中、**他ポケモンから自分に向けて使用された変化技（状態異常付与・能力変化・トラップ等）を**：  
  1. **無効化する**  
  2. **無効化した技を使用者へ跳ね返す** という処理にする。 :contentReference[oaicite:2]{index=2}

### 跳ね返し挙動の注意点

- 跳ね返された技は **元の使用者に再対象化** される。  
- 跳ね返された技の効果は、**技自体の命中判定等は技を跳ね返した側（状態保持者）で計算**される。  
- 跳ね返された技をさらにマジックコート状態で跳ね返すことは**できない**。  
- 第五世代以降では、**設置系・場全体効果を含む変化技も跳ね返せる**（ただし対象条件あり）。 :contentReference[oaicite:3]{index=3}

## 維持と解除

- 状態は **そのターンのみ持続**。  
- ターン終了時点で自動解除。  
- 対象ポケモンが **交代・戦闘不能** となった場合も解除。 :contentReference[oaicite:4]{index=4}

## 相互作用・例外

| 条件 / 効果 | 挙動 |
|---|---|
| 特性「マジックミラー」 | **常時マジックコート状態**として変化技を跳ね返す。 |
| 複数ポケモンが同時に状態 | それぞれの状態で技跳ね返し判定が発生する場合あり。 |
| 一部技・効果 | 状態異常・能力変化・トラップ類・特殊変化技などの多くを対象化。 |
| 跳ね返された技の命中 | 元の技命中仕様とは独立し、跳ね返した側の命中計算で処理。 :contentReference[oaicite:5]{index=5} |

## ターン処理順（概要）

1. 技・特性によって **マジックコート状態発生**  
2. 対象への **変化技の使用判定時** に、状態があれば技効果を **無効化・跳ね返し**  
3. **跳ね返し技の処理** を相手側で実行  
4. **ターン終了** 時に状態は解除

## 補足

- マジックコート状態は **変化技全体の効果を一時的にミラー反射する**仕組みであり、単体技／全体技問わず対象化が可能な場合がある。  
- 跳ね返せる技は世代・個別技仕様により変動するが、一般的には状態異常・能力変化・場変化系の変化技が該当する。 :contentReference[oaicite:6]{index=6}
