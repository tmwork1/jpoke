# じごくづき（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | じごくづき（状態変化） |
| 分類 | 状態変化（わざ効果） |
| 発生方法 | 技「じごくづき」を命中させる |
| 効果内容 | 対象ポケモンは **音の技を使えなくなる** 制限状態 |
| 効果持続 | **2ターン** |
| 重ね掛け | 再付与・延長は発生しない |
| 状態解除 | **交代すると解除** |

## 発動条件 / 失敗条件

### 発動条件

- 技「じごくづき」が **対象に命中** した場合に付与される  
- 技が無効化（まもる等）された場合は付与されない

### 失敗条件

以下の場合、じごくづきは **効果が発生しない**：

- 対象が **すでにじごくづき状態** である  
- 対象が交代直後でまだ判定対象外のタイミングにある  
- 技自体が命中せずに終わった

## 効果詳細（音技制限）

じごくづき状態の対象は、**音の技を使用不可** になる。  
具体的な制限は次の通り：

- 音の技を **選択した時点で発動不可** と判定され、そのターンは他の技を使用不可ならわるあがきになる。  
- 制限中に音の技以外（非音技）は通常通り使用可能  
- 音の技であっても **Zワザ・特殊処理の一部** は制限を受けない場合がある  
- さわぐ状態の対象がじごくづき状態になると、さわぐ効果は終了（技継続が解除）  
- マジックコート等で音技が反射された場合、反射側は制限対象とはみなされず音技を発動できるケースがある  
- 音技のみを有するポケモンは制限中、行動可能な技がない場合わるあがきを行う

## 維持期間と解除

じごくづき状態は以下で解除される：

- **2ターン経過**  
- 対象ポケモンの **交代 / 戦闘離脱**  
- 状態解除効果（道具 / 技 / 特性等）

## 競合・相互作用

| 条件・効果 | 挙動 |
|---|---|
| さわぐ状態 | じごくづき状態になると **さわぐは中断・解除** される |
| 音技反射（マジックコート等） | 反射した音技は制限影響を受けない場合あり |
| 他制限系状態 | ちょうはつ等他技制限と併存可能で、挙動は状態ごとに判定 |

## ターン処理順（概要）

1. 技「じごくづき」が命中 → 状態変化付与  
2. 対象は音の技を使用不可として扱われる  
3. ターン経過により持続ターンを消費  
4. 持続ターンが 0 になるか解除条件で状態が解除

## 補足

- じごくづきは **音の技の選択制限** を管理する状態であり、ステータスダメージ等とは別処理  
- 音技の判定対象は技のカテゴリ指定（音技扱い）による
