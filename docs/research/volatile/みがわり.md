# みがわり（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | みがわり（状態変化） |
| 分類 | 状態変化（Substitute / 代わりの人形） |
| 効果概要 | 使用者が **自身の HP の 1/4 を消費して代わりの分身（人形）を設置し、攻撃・多くの効果を肩代わりさせる** |
| 発生源 | 技「みがわり」 |
| 対象 | 技使用者自身 |
| 重ね掛け | 既にみがわり状態がある場合 **失敗**（重ね不可） |
| 状態解除 | 分身の HP が 0 になる・交代・戦闘離脱・特定技等 |

## 発動条件 / 失敗条件

### 発動（付与）条件

- 技 **「みがわり」** を使用  
- 使用者の **現在 HP が最大 HP の 1/4 を超えていること**  
- 既に分身（みがわり状態）が **存在しないこと**  
- 技の使用自体が成立する（状態異常・妨害では失敗） :contentReference[oaicite:0]{index=0}

### 付与失敗条件

| 条件 | 結果 |
|---|---|
| 現在 HP が 1/4 以下 | 発動失敗 |
| すでにみがわり状態 | 発動失敗 |
| 技使用自体が妨害・無効化された | 発動失敗 |
| 交代 / 戦闘不能 | 発動不可 |

## 効果詳細

### 分身の生成

- 使用者は **最大 HP の 1/4（小数点切り捨て）** 相当の HP を消費  
- 消費した量と同じ HP を持つ **みがわり分身（doll）** を生成すると扱われる  
- 分身の HP は生成時の値固定で、その分身が **HP 減少→ 0 になるまで残る** :contentReference[oaicite:1]{index=1}

### 分身の効果（防護）

- 分身が存在する間、以下を **代わりに受ける・防ぐ**：  
  - 相手からの **直接ダメージ攻撃**  
  - 多くの **状態異常付与**（やけど・まひ・どく・ねむり等）  
  - **能力ランク変化・混乱・ひるみ効果** 等（いわゆる妨害効果多数）  
- 分身は **音技や特性インフィルトレータ（Infiltrator）等により貫通する効果あり** :contentReference[oaicite:2]{index=2}  
- 天候ダメージ・やどりぎ等 **非直接攻撃ダメージや継続効果** は分身では防げない（使用者にそのまま影響） :contentReference[oaicite:3]{index=3}

### 分身の破壊

- 分身の HP が **0 になると消滅**  
- 分身消滅と同時に元の状態のポケモンが場に立ち続ける  
- **ダメージ過多時（一撃で超過）** は残ダメージ分は元のポケモンに影響しない :contentReference[oaicite:4]{index=4}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 分身の HP が 0 | みがわり状態 **解除** |
| 分身保持のまま **交代 / 戦闘不能** | 分身は消滅 |
| 技「きれいなぬけがら」「たいどめ」等 | 分身を強制解除可能（技依存） |
| みがわりを対象とした Snatch | 分身効果を奪われる（自身に付与） :contentReference[oaicite:5]{index=5}

## 相互作用・例外

| 効果 / 条件 | 挙動 |
|---|---|
| 特性「インフィルトレータ」 | 分身防護を **貫通する** 技もある |
| 音技 | 分身で防げない場合がある |
| 天候 / 継続ダメージ | 分身では **防がない** 効果は使用者に適用 |
| バトンタッチ | 分身は **HP・状態として後続へ引き継ぐ**（HP 同値） :contentReference[oaicite:6]{index=6}

## ターン処理順（概要）

1. 技「みがわり」を **選択・成立**  
2. 使用者の HP を **最大 HP × 1/4** 消費  
3. 分身（HP 保持値）を **状態として生成**  
4. 各相手攻撃対象判定時、分身が **先にヒット処理**  
5. 分身消滅後、攻撃等は使用者に通常通り適用

## 補足

- 分身は **状態変化ではなく分身オブジェクト（doll）として内部処理** されるが、戦闘上は防護状態として扱われる  
- 分身が存在する限り、使用者の **実体への妨害・状態異常付与は無効化される**仕様が中心となる :contentReference[oaicite:7]{index=7}
