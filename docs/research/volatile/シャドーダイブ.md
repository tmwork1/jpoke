# シャドーダイブ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | シャドーダイブ（状態変化） |
| 分類 | 状態変化（技「シャドーダイブ／ゴーストダイブ」等による） |
| 効果概要 | 対象ポケモン（使用者）は **姿を消した状態になる** |
| 発生源 | 技「シャドーダイブ」「ゴーストダイブ」等の効果 |
| 対象 | シャドーダイブを使用したポケモン自身 |
| 重ね掛け | 同一状態は **重ね不可 / 延長不可** |
| 状態解除 | **攻撃発動のタイミング・解除条件達成時** |

## 発動条件 / 無効条件

### 発動条件

- 技「シャドーダイブ」や類似技を **使用したターン（行動選択成功）** に状態が発生する  
- 技が **成功・発動判定を通過** する必要あり  
- 技が **無効化された場合（まもる等）** は状態発生しない場合あり

### 無効・付与不能

| 条件 | 結果 |
|---|---|
| 技発動が無効化 | 状態は発生しない |
| 使用者が交代・戦闘離脱 | 状態は生成されない |
| 技が選択不可 | 発生しない |

## 効果詳細

### 姿消し（半透明状態）

- 状態中、対象は **姿を消した扱い** となる  
- 多くの通常技・効果が **対象へ命中しなくなる**扱いとなる（実際のヒット判定回避）  
- 対象は **フィールド効果・補助効果を受けない**判定になる場合がある  
- 技発動後、**次ターンに本来の攻撃判定が成立する**  
- 「まもる」「みきり」「キングシールド」「ニードルガード」「ブロッキング」「たたみがえし」などの保護系効果は回避扱いとなり、**効果影響を受けない**（技命中時点で消失扱いになることもある）  
- 対象は姿消し状態中も **視界上の行動は可能** だが、物理命中・多くの判定は無効となるケースが基本（世代仕様に準ずる） :contentReference[oaicite:0]{index=0}

## ターン処理と解除

- 技「シャドーダイブ」使用ターンの **1ターン目に姿を消す状態** が発生  
- **2ターン目に攻撃が実行される**と同時に状態は解除  
- 状態中でも行動選択が別途可能（世代や状況依存）  
- 状態が続く間は通常攻撃・判定に **対象不可・命中無効判定** となる端的な扱い

## 競合・相互作用

| 条件/効果 | 挙動 |
|---|---|
| 保護系技（まもる等） | 消えた状態判定により **影響を受けない**（命中効果無効含む） :contentReference[oaicite:1]{index=1} |
| 一部の必中効果 | 状態中の対象に対して処理判定が異なるため **必中効果が作用しない場合あり** |
| 特殊連続技 | 技処理・判定順で優先度が変化する可能性あり |

## 補足

- シャドーダイブは **状態変化として扱われる「姿消し」** として内部処理される  
- 攻撃処理は **次ターンに成立する**  
- 命中関連は **姿消し判定が優先されるため多くの攻撃が対象に命中しない** と扱われる  
- 対象が交代等で行動継続できない場合は状態解除扱い
