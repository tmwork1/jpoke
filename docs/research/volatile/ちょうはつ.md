# ちょうはつ（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ちょうはつ（状態変化） |
| 分類 | 状態変化（わざ「ちょうはつ」による付与） |
| 効果概要 | 対象ポケモンは **変化技（ステータス変化技）を使用できなくなる** |
| 対象 | 1体の相手ポケモン（命中対象） |
| 持続 | **3〜4ターン**（付与ターンを含む） |
| 継続停止 | 対象交代・状態解除系（例: メンタルハーブ） で解除 |
| 重ね掛け | 同じ状態は **重ね不可 / 延長不可** |
| 防御可否 | まもる等の無効化・みがわり越しにも付与可能 |

## 発動条件 / 無効条件

### 発動条件

- 技「ちょうはつ」が **対象に命中したとき** に状態変化付与。  
- 命中前に防御（まもる/みがわり等）で防がれた場合は付与されない。  
- 対象の特性「どんかん」等で即座に解除される場合あり。  
- 対象が **こうどうしない/交代時** は付与後にそのターン内カウント処理変則あり。 :contentReference[oaicite:0]{index=0}

### 無効 / 付与失敗

| 条件 | 結果 |
|---|---|
| 対象が状態防止系特性（例: アロマベール等） | 付与を防ぐ場合あり |
| 対象がすでにちょうはつ状態 | 重ね掛け・延長は発生しない |
| 技が無効化（まもる/みがわり等） | 命中処理次第で状態付与しない |

## 効果詳細（変化技制限）

- **変化技（ステータス変化・補助技）を選択不可・使用不可** とする。  
- 選択画面で変化技が選ばれた場合、失敗メッセージが出て再選択になる。  
- **ダメージ技（攻撃技）は通常通り使用可能**。  
- ステルスロック等の場設置技も変化技扱いのため制限される。  
- ちょうはつ状態中に **変化技を選択しても成功しない**。  
- 「さきどり」など一部例外で変化技を選べる場合あり。 :contentReference[oaicite:1]{index=1}

## 効果持続とターン数

- 効果は **3〜4ターン** 継続  
  - 命中ターンを含むカウントで処理  
  - 対象が行動済未済等により継続ターン判定に差異あり  
- 対象交代・状態解除系効果（例: メンタルハーブ）で **即座に解除** される。 :contentReference[oaicite:2]{index=2}

## 競合・相互作用

| 相互作用 | 仕様 |
|---|---|
| 特性「アロマベール」 | 味方へのちょうはつ等の付与を防ぐ |
| 特性「どんかん」 | ちょうはつ状態を即座に解除できる |
| みがわり | 状態付与自体は成功するが、みがわり表示と処理タイミング差異あり |
| 交代タイミング | 交代で **状態解除** が発生 |

## ターン処理順（概要）

1. 技「ちょうはつ」命中判定  
2. 対象へ **ちょうはつ状態変化付与**  
3. 対象が交代しない限り **変化技制限** を適用  
4. ターン終了時に **カウント減少**  
5. 規定ターン経過で状態解除

## 補足

- ちょうはつ状態中はステータス変化・補助・召喚・回復系などの変化技がすべて制限される。  
- 攻撃技（ダメージ技）は制限対象外でそのまま使用できる。  
- ちょうはつ効果は **交代で解除されることが多い** ため、継続戦術時の評価に注意が必要。 :contentReference[oaicite:3]{index=3}
