# こんらん（状態異常）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | こんらん（状態異常） |
| 分類 | 状態異常（揮発性） |
| 英語名 | Confusion |
| 効果 | ポケモンが混乱状態になる |
| 発生源 | 混乱付与効果を持つ技・効果（例: こんらんこうせん、いかりのこな 等） |

## 発動条件 / 無効条件

### 発動条件

- 混乱付与効果を持つ技等が対象に命中した場合に発生  
- 混乱は **揮発性状態異常** として付与される

### 無効条件・解除条件

| 条件 | 結果 |
|---|---|
| 対象が交代（戦闘から離脱） | 混乱が解除 |
| 技などで状態異常が解除された場合 | 混乱が解除 |

## 効果詳細

### 行動制御

- 混乱状態中、ポケモンが技を使用するたびに **確率判定** が行われる  
- 第7世代以降、確率は **約33%（1/3）で自分を攻撃** してしまう判定  
- 自分を攻撃した場合、そのターンは **他の行動ができない**  
- 自分への混乱ダメージは **タイプ・特性・道具の影響を受けない typeless 40威力物理攻撃に 相当** して処理される  
- 混乱したまま通常攻撃が成功した場合は通常通り相手を攻撃する  
- 捕獲や特定の行動制限技（ねむる等）のターンは混乱ターンのカウント減少に影響する  
- 混乱が続く間は交代・技発動などの行動に影響を及ぼす

### 継続ターン

- 混乱状態は **1〜4ターン** 継続（経過ターンごとにカウントが減少）  
- ターン経過の判定は **技使用を伴う行動（攻撃など）でのみ** 減少  
- 寝ている／こおり等で行動不能の場合は **カウントが減らない**

## 競合・相互作用

| 条件・特性 | 影響 |
|---|---|
| 特性「マイペース」 | 混乱状態にならない |
| フィールド「ミストフィールド」 | 地面のポケモンが混乱付与を受けない |
| 特性「すりぬけ」等 | 混乱付与の条件判定に影響を及ぼす場合あり |
| PPが0の技使用 | 混乱状態のカウント減少条件には関与しない |

## ターン処理順（概要）

1. 混乱状態が付与される  
2. 技選択時に混乱判定を表示  
3. 判定により **自分への攻撃** または **通常行動** を実行  
4. 技使用による行動が成立した場合、混乱ターンカウントを 1 減少  
5. 混乱ターンカウントが 0 になると状態解除
