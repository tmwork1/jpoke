# にげられない（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | にげられない（状態変化） |
| 分類 | 状態変化（逃走・交代制限） |
| 英語名 | Can’t Escape |
| 効果概要 | 対象ポケモンは **通常の交代・逃走ができなくなる** 状態 |
| 持続条件 | 付与されたまま交代・戦闘不能になるまで持続*¹ |
| 重ね掛け | 既に状態中の場合は **追加で延長・重複なし** |
| 状態解除 | 対象交代・戦闘不能・解除効果（例: 特定技/アイテム） |

## 発動条件 / 無効条件

### 発動条件

- 技・効果により対象に「にげられない」状態が付与される（例: **くろいまなざし / かげぬい / とおせんぼう / サウザンウェーブ / アンカーショット / はいすいのじん / たこがため / くらいつく** 等）。 :contentReference[oaicite:0]{index=0}  
- 技の命中・効果判定が成功する必要あり。  
- **みがわり** 等で効果を防ぐ/反射できる場合あり（技依存）。 :contentReference[oaicite:1]{index=1}

### 無効 / 付与失敗

| 条件 | 結果 |
|---|---|
| 対象が **ゴーストタイプ**（第6世代以降） | 技付与が **効果を受けない**（例：くろいまなざし等） :contentReference[oaicite:2]{index=2} |
| 技が **無効化 / 命中しない** | 状態付与は発生しない |
| 対象がすでに同状態 | 重ね掛け・延長は発生しない |

## 効果内容（交代・逃走制限）

- 「にげられない」状態中のポケモンは **通常の交代コマンド / 逃走行動が選択不可** となる。 :contentReference[oaicite:3]{index=3}  
- **にげる・交代** の命令が表示されても使用不可。  
- **逃走判定成功・失敗の処理は存在しない**（選択できない）。  
- 対象が戦闘中継続している限り効果が持続。 :contentReference[oaicite:4]{index=4}

## 例外・特別な交代方法

状態中でも以下は可能：

- **バトンタッチ / Uターン / ボルトチェンジ / フリップターン** 等の **置換技**（交代を伴う技）は使用可能（ただし仕様により変動あり）。 :contentReference[oaicite:5]{index=5}  
- **道具「きれいなぬけがら」** 等で交代可能になる場合あり。 :contentReference[oaicite:6]{index=6}  
- **特性「にげあし」 / 道具「けむりだま」** を持つ場合、逃走コマンド自体は成功するケースあり（逃走成功扱い）。 :contentReference[oaicite:7]{index=7}  
- 技 “テレポート” は第7世代までは状態中に失敗したが、第8〜9世代では **プレイヤー側は成功する** 仕様あり（技依存）。 :contentReference[oaicite:8]{index=8}

## 維持と解除

| 条件 | 結果 |
|---|---|
| 対象が **交代 / 戦闘不能** | 状態解除 |
| 技による解除効果 | 状態解除（技依存） |
| 対象が **バトンタッチで交代** | 通常は状態解除 (*世代依存注意*) :contentReference[oaicite:9]{index=9} |

## 競合・相互作用

| 条件 / 効果 | 挙動 |
|---|---|
| 状態付与技を使用したポケモンが戦闘離脱 | 状態解除 |
| 対象が **ゴーストタイプになる変化効果** | 状態効果を無視して交代可能（効果解除は通常扱い） :contentReference[oaicite:10]{index=10} |
| バインド系持続攻撃（やどりぎ等）との併存 | 影響なし（別効果） |

## ターン処理順（概要）

1. 対象に **逃走・交代制限技** が命中 → 「にげられない」状態付与  
2. 対象の次ターン以降 **交代/逃走選択が不可**  
3. 技影響以外で状態が解除されるまで **持続**

## 補足

- 「にげられない」は **状態変化の一種** として扱われ、他の状態変化と重複可能。 :contentReference[oaicite:11]{index=11}  
- 技の生成元や特性効果によって詳細な挙動（例外処理）は異なるため、個別の技/特性の仕様も参照が必要。 :contentReference[oaicite:12]{index=12}
