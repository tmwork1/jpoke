# うちおとす（状態変化 / 技効果）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | うちおとす |
| 分類 | 技効果（技: うちおとす / Smack Down） |
| タイプ | いわ |
| 技分類 | 物理攻撃 |
| 対象 | 1体の相手ポケモン |
| 発動方法 | 技「うちおとす」を選択し命中 |

## 効果内容

- **飛行中や浮いている対象を地面に落とす（うちおとす状態）**  
  技命中後、対象が空中状態（例: そらをとぶ／とびはねる／ひこうタイプ／でんじふゆう等）であれば、**地面に落とされる状態になる**（うちおとす状態）。  
- **ダメージ有り**  
  技ダメージが入る相手に通常の攻撃ダメージが発生する。  
- **空中技への判定**  
  Fly や Bounce 中の相手には命中時に地面に落とす。Sky Drop の場合は命中するが落下はしない。  
- **地面にいる相手**  
  元から地面にいる相手には追加の落下効果は発生しない。  
- **状態持続**  
  うちおとす状態そのものは継続するが、戦闘内での場持続時間という概念はなく、対象が場にいる限りその状態扱いになる（浮いている技キャンセル判定のため）。 :contentReference[oaicite:0]{index=0}

## 発動条件 / 無効条件

### 発動条件

- 対象に技が命中したこと。  
- 対象が **空中状態または浮いている状態** であることが効果発生の前提となる（地面既定なら落下効果なし）。 :contentReference[oaicite:1]{index=1}

### 無効 / 発生しないケース

| 条件 | 仕様 |
|---|---|
| 技「まもる」などで技が無効化 | 技効果（うちおとす）は発生しない |
| みがわりによる防御 | 技ダメージは当たる可能性ありだが、うちおとす状態付与は発生しない |
| そもそも空中状態でない場合 | 飛び状態解除効果は発生しない |
| Sky Drop 中の対象 | ダメージは入るが、落下効果は発生しない | :contentReference[oaicite:2]{index=2}

## 技処理・順序（概要）

1. 技「うちおとす」を選択・命中判定  
2. ダメージ判定・適用  
3. 対象が空中もしくは浮いている場合、**地面状態へ変更（うちおとす状態）**  
4. 以降の行動で地面状態として扱われる

## 相互作用・注意点

### 浮遊系状態との関係

- 特性「ふゆう」や状態異常「でんじふゆう」等 **浮いている扱いの対象にも命中し、地面扱いへ変更可能**（うちおとす状態付与）。  
- ただし特性・状態によっては浮遊扱いだからといって無効にならない場合もある（他場状態・特性による差異あり）。 :contentReference[oaicite:3]{index=3}

### 空中技等との関係

- 対象がそらをとぶ／とびはねる等の行動を取っている場合、**その行動をキャンセルして地面へ落とす**。  
- Sky Drop の場合、攻撃は入るが落下効果そのものは発生しない。 :contentReference[oaicite:4]{index=4}

### みがわり・防御効果

- 技ダメージ判定は対象の防御でブロックされる可能性があり、その場合うちおとす状態は付与されない。 :contentReference[oaicite:5]{index=5}
