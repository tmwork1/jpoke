# へんしん（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | へんしん（状態変化） |
| 分類 | 状態変化（変身／Transform） |
| 関連技 | わざ「へんしん」 |
| 効果概要 | 対象ポケモンが **相手ポケモンの姿・タイプ・技等をコピー** し、**変身状態になる** |
| 対象 | 技使用者自身 |
| 持続 | **場にいる間**（交代・ひんし・戦闘終了で解除） |
| 重ね掛け | 再付与・延長なし |

## 発動条件 / 失敗条件

### 発動条件

- 技「へんしん」を **対象に使用して成功** した場合に発生する  
- 命中率・回避補正の影響を受けず照準成功する  
- **みがわり状態の対象には失敗** （状態付与不可）  
- 対象がすでに **へんしん状態中の場合も失敗**  
- **Illusion（イリュージョン）状態の対象に対して失敗** する場合がある（対象が化けていると判定不能） :contentReference[oaicite:0]{index=0}

## 変身状態の効果詳細

### コピー内容

- 対象となったポケモンの以下の要素をコピーする：  
  - **見た目（Sprite）**・**種類（Species）**  
  - **タイプ**・**技構成**（各技はコピー後 **5 PP** にセット）  
  - **種族値・能力値・ランク変化**（一部世代差異あり）  
- コピーしないもの：  
  - **HP（現在・最大HP）** はコピーせず元のまま  
  - **特性**はコピー先のものになるが、条件によって無効の場合あり  
  - **状態異常**（やけど等）はコピーされない :contentReference[oaicite:1]{index=1}

### 技の扱い

- コピーされた技は **5 PP** で扱われる（元のPP量に関係なし）  
- コピー技は **元のポケモンの現在技構成を反映** する :contentReference[oaicite:2]{index=2}

## 維持と解除条件

| 条件 | 結果 |
|---|---|
| 対象が **交代する** | 変身状態が **解除** |
| 対象が **戦闘不能（ひんし）になる** | 解除 |
| 戦闘終了 | 解除 |
| バトンタッチ等（交代技） | 解除扱い |

## 無効・特例

- 対象が **みがわり・イリュージョン等の妨害判定** 下にあると **変身に失敗** する場合あり  
- 技「へんしん」は **必中だが** 対象の特定状態（影・姿隠し等）によっては動作しない  
- **へんしん状態中にさらに「へんしん」を使用しても失敗** する :contentReference[oaicite:3]{index=3}

## ターン処理順（概要）

1. 技「へんしん」を使用  
2. 命中・条件判定成功 → **変身状態に付与**  
3. 対象の姿・技・タイプ等をコピー  
4. 変身状態は **交代・戦闘終了まで持続**

## 相互作用・注意点

- 特性「Imposter」は **自動的に変身状態と同等の効果を発動**（対象ポケモンに出された瞬間にコピー）  
- 変身状態は **交代で解除される**ため、持続は場に居続けることが前提  
- 元のHP値・状態異常は **引き継がれないため別管理**  
- コピー対象に **テラスタル等の付加効果** がある場合は判定に影響するケースあり :contentReference[oaicite:4]{index=4}
