# たくわえる（状態変化 / 技効果）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | たくわえる（状態変化） |
| 分類 | 技効果 |
| 発生元 | 技「たくわえる」 |  
| 効果概要 | 使用するごとに**蓄積数（ストック）を1増やす** |
| 最大蓄積数 | **3まで**（4回目以降は失敗） |
| 関連技 | 「はきだす」「のみこむ」と効果連動 |
| ランク補正 | 使用時、**ぼうぎょ・とくぼうが各1段階ずつ上昇**（最大+6まで） |

## 発動条件 / 無効条件

### 発動条件

- 技「たくわえる」を**対象のポケモンが使用して命中、ブロックされない**こと  
- 現在の蓄積数が **0〜2** の場合に効果が発生

### 失敗 / 無効条件

| 条件 | 結果 |
|---|---|
| 蓄積数が **3** の状態で使用 | 技失敗（蓄積数増加・能力上昇なし） |
| 技が**無効化 / 防がれた場合**（例: まもる・みがわり等） | 効果なし |
| 持ち場・状態による使用制限 | 技自体が選択できない場合発動しない |  
※ ストック数は戦闘中のみ保持し、交代・戦闘離脱で消失する（後述）。

## 効果詳細

### ストック蓄積

- 使用するごとにストック数を **1増やす**（最大3）  
- 蓄積数は数値として保持し、後述の**はきだす／のみこむ**で参照される  
- 補助効果として **ぼうぎょ・とくぼう** がそれぞれ **+1段階**上昇  
  - 累積して使用すると最大 **+3段階** まで重ねられる

### 能力ランク上昇との関係

- 各使用ごとに防御・特防がそれぞれ1段階上昇  
- 「はきだす／のみこむ」を使った際、これらの上昇は **解除される**

## 関連技との連動

### はきだす（Spit Up）

- ストック数に応じて威力が変化する攻撃技  
- 使用成功時、蓄積数が **0 → 失敗**、**1〜3 → 威力増加**  
- 使用後、蓄積数は **0 にリセット**  
- 防御・特防のランク上昇は同時に解除される

### のみこむ（Swallow）

- ストック数に応じて回復量が変化する回復技  
- 使用成功時、蓄積数が **0 → 失敗**、**1〜3 → 回復発生**  
- 使用後、蓄積数は **0 にリセット**  
- 防御・特防のランク上昇は解除される

## 持続と解除

- **戦闘中のみ保持**： ポケモンが**交代／戦闘不能**になった時点で  
  - ストック数は **0にリセット**
  - ぼうぎょ・とくぼうのランク上昇はそのまま維持または保持（状況依存）  
- 状態そのものの継続時間は存在しない（技使用時のみ蓄積処理）

## 競合・相互作用

| 条件・効果 | 挙動 |
|---|---|
| 特性「ミラーアーマー」等の反射 | 技の付与効果として扱い、ストック数増加は反射対象外 |
| こうそくスピン等妨害 | 技選択そのものが不可の場合、蓄積しない |
| PP制限（PP0） | 技は選択不可、蓄積しない |
| バトンタッチ | 蓄積数は引き継がれない |

## ターン処理順（概要）

1. ポケモンが技「たくわえる」を選択  
2. 技命中判定＆防御効果の確認  
3. 成功時にストック数を +1（最大3）  
4. ぼうぎょ・とくぼうを +1段階ずつ上昇  
5. 経過ターンへ処理継続

## 補足

- 第9世代における「たくわえる」は **蓄積＋防御上昇の動作**が主仕様であり、他のストック技（はきだす／のみこむ）と組み合わせて運用される。  
- 蓄積数はバトル中の状態として管理されるが、交代・戦闘離脱でクリアされる。  
- 防御・特防のランク上昇は戦闘中持続。
