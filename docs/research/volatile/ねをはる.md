# ねをはる（状態変化）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ねをはる（状態変化） |
| 分類 | 状態変化（技「ねをはる」による付与） |
| 効果概要 | 毎ターン終了時に **最大HPの1/16回復** を行い、特定の行動制限や判定変化が発生 |
| 持続条件 | 対象ポケモンが **場に居続ける間** |
| 重ね掛け | 同一状態は **重ね不可 / 延長不可** |
| 状態解除 | 交代（通常は不可）・戦闘不能・状態解除系効果で解除 |
| 対象 | 技を使用した自分自身 |

## 発動条件 / 無効条件

### 発動条件

- 技「ねをはる」が **自分に成功** した際に状態として付与される  
- 命中判定は存在しない（自身技なので確実に付与）  
- 技使用後、状態効果が開始する :contentReference[oaicite:0]{index=0}

### 無効・解除条件

| 条件 | 結果 |
|---|---|
| 対象が **交代 / 戦闘不能** | 状態解除 |
| 状態効果解除系（例：メンタルハーブ等） | 状態解除 |
| 一部行動で交代可能な例外（きれいなぬけがら等）を利用 | 交代と同時に状態が解除または引き継ぎ判定あり :contentReference[oaicite:1]{index=1}

## 効果詳細

### HP回復処理

- 各ターン終了時に **最大HP × 1/16** を回復（小数点以下切り捨て） :contentReference[oaicite:2]{index=2}  
- 持ち物「おおきなねっこ」を装備している場合は回復量が **1.3倍** に増加 :contentReference[oaicite:3]{index=3}  
- 「かいふくふうじ」状態中は回復効果は **発動するが回復しない** :contentReference[oaicite:4]{index=4}

### 行動制限

- 状態中は **交代・にげる** コマンドが通常の方法では不可 :contentReference[oaicite:5]{index=5}  
- 「ともえなげ」「ドラゴンテール」「ふきとばし」「ほえる」などの強制交換効果は **ダメージのみで交代しない** :contentReference[oaicite:6]{index=6}  
- 道具「レッドカード」の効果は **消費されるが交代しない** :contentReference[oaicite:7]{index=7}

### 地面判定

- 第4世代以降は状態中のポケモンが **地面にいる扱い** になる :contentReference[oaicite:8]{index=8}  
  - ひこうタイプ、特性「ふゆう」、状態「でんじふゆう」「テレキネシス」、持ち物「ふうせん」などでも **じめんタイプ技のダメージを受ける**ようになる :contentReference[oaicite:9]{index=9}  
  - **まきびし / どくびし / ねばねばネット** の判定は別途処理されることがある 

## 持続と解除

- 対象が **交代 / 戦闘不能** になると状態解除  
- 通常の交代コマンドは状態中は不可だが、**きれいなぬけがら** 等の例外で交代可能な場合、状態はそのタイミングで解除または継続条件が別途判定される :contentReference[oaicite:10]{index=10}  
- 状態解除系の技・効果でも解除可能

## 競合・相互作用

| 条件・状態 | 仕様 |
|---|---|
| 特性「にげあし」 / 道具「けむりだま」等 | 状態中でも **にげる（通常逃走）可能** になる場合あり :contentReference[oaicite:11]{index=11} |
| ゴーストタイプ | 状態中でも **交代・逃走可能**（第6世代以降） :contentReference[oaicite:12]{index=12} |
| バトンタッチ | 状態を **引き継ぐことができる**（交代先でも有効） :contentReference[oaicite:13]{index=13} |
| 特性「ふゆう」等との併用 | 地面判定は変化するが状態そのものは維持 :contentReference[oaicite:14]{index=14} |
| 一部行動制限解除アイテム | 状態中でも交代可能になる例外あり :contentReference[oaicite:15]{index=15}

## ターン処理順（概要）

1. 技「ねをはる」を使用 → 状態付与  
2. 各ターン終了時に HP回復処理  
3. その後通常の行動・判定を進行  
4. 状態解除条件満了で状態消滅

## 補足

- 「ねをはる」は **HP回復効果と行動制限・判定変化を同時に付与する**状態であり、回復自体は場状態からの回復ではなく個別の状態効果として処理される :contentReference[oaicite:16]{index=16}
