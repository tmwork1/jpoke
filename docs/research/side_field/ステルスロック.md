# ステルスロック（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | ステルスロック（場の状態） |
| 発生方法 | 技「ステルスロック」などによって相手側に場状態を設置 |
| 効果対象 | 相手側に場に出てくるポケモン |
| 効果内容 | 出てきたポケモンに **ダメージを与える** |
| 効果持続 | **場に存在する限り無期限（世代による制限なし）** |
| 重ね掛け | **重ねがけ不可**（既に状態があると再設定できない） |
| 他の場条件との共存 | 他の場状態（まきびし/どくびし等）と共存可能 |
| 設置対象 | **相手側の場** |

## 発動条件 / 無効条件

### 発動

- 技「ステルスロック」使用時に成功すると相手側の場に状態が設定される。  
- 同じ場側にすでにステルスロック状態が存在している場合、技は**失敗**する。 :contentReference[oaicite:0]{index=0}

### 無効／解除条件

| 条件・技 | 結果 |
|---|---|
| 相手側にステルスロックが既に存在 | 技は失敗（再設定されない） |
| こうそくスピン / キラースピン | 使用者側の場のステルスロックを解除 |
| きりばらい / おかたづけ | 両側の場のステルスロックを解除 | :contentReference[oaicite:1]{index=1}

## 効果詳細（スイッチインダメージ）

| 判定条件 | 内容 |
|---|---|
| ダメージ発生タイミング | ポケモンが場に **出た瞬間** |
| ダメージ量 | 最大HP × (いわタイプ相性補正) ÷ 8 |
| 相性補正 | いわタイプに対するTYP相性で倍率変化 |
| 具体例（相性倍率 → ダメージ） |  
0.25× → 1/32  
0.5× → 1/16  
1× → 1/8  
2× → 1/4  
4× → 1/2 | :contentReference[oaicite:2]{index=2}

### 注意事項

- ダメージ計算時は **小数点以下切り捨て**（0になった場合最低1ダメージ）。  
- 特性「マジックガード」や持ち物「あつぞこブーツ」を持つポケモンは **ダメージを受けない**。  
- ステルスロックのダメージでHPが0になった場合、場に出たとき発動する一部の特性は **発動せずに瀕死** になる。  
- タイプ変化系効果がある場合、出てきたあとに変化したタイプで **相性判定** を行う。 :contentReference[oaicite:3]{index=3}

## 競合・相互作用

| 相互作用条件 | 効果 |
|---|---|
| 特性「マジックガード」 | ステルスロックのダメージを **無効化** |
| あつぞこブーツ | ステルスロックのダメージを **無効化** |
| 特性によるタイプ変化後の相性判定 | 出てきたポケモンの新しいタイプで判定 |

## ターン処理順（概要）

1. 技「ステルスロック」成功 → 相手側の場状態を設定  
2. その後の行動処理（技/交代）を進行  
3. ポケモンが場に出た時点で **即座にダメージ判定**  
4. 場状態は **解除されるまで継続**

## 補足

- ステルスロックは **いわタイプの設置系場状態** であり、相手の交代時にのみダメージが発生する。  
- 設置後は技の重ね掛けができず、解除効果が起きるまで永続する。 :contentReference[oaicite:4]{index=4}
