# どくびし（場の状態）仕様書

## 基本効果

| 項目 | 内容 |
|---|---|
| 表示名 | どくびし（場の状態） |
| 発生方法 | 技「どくびし」成功 / 特性「どくげしょう」で物理技を受ける |
| 効果対象 | 相手側に新たに場に出てくるポケモン |
| 効果内容 | 出てきた**地面にいるポケモン**を状態異常にする |
| 効果量 | 1層：**どく**、2層：**もうどく**（2回目の設置で強化） |
| 効果持続 | **場に存在する限り無期限** |
| 重ね掛け | **重ねられる（最大2層まで効果あり）** |
| 他場状態との共存 | 他の場状態と共存可能（例：ステルスロック等） |

## 発動条件 / 無効条件

### 発動

- 技「どくびし」を使用し命中した場合に、対象側の場に状態が設定される。  
- 特性「どくげしょう」のポケモンが物理技を受けた際にも設定される。 :contentReference[oaicite:0]{index=0}

### 無効／解除条件

| 条件・技 | 結果 |
|---|---|
| 相手側の場に既にどくびしがある | 技は成功するが**層の追加**となる（最大2） |
| こうそくスピン／キラースピン | 使用者側のどくびしを解除 |
| きりばらい・おかたづけ | 両側のどくびしを解除 |
| 地面にいるどくタイプのポケモンが場に出る | その場のどくびしを解除（味方側のみ） |
| あつぞこブーツ装備のポケモンが場に出る | どくびしの効果を**受けない**（地面免疫） |

※ 地面にいないポケモン（ひこう/特性ふゆう等）は効果対象とならない（後述）。 :contentReference[oaicite:1]{index=1}

## 効果詳細（状態付与条件）

### 条件

- ポケモンが**場に出た瞬間**に処理される。  
- 対象は**地面にいるポケモン**（地上にいるもの）で判定。 :contentReference[oaicite:2]{index=2}

### 状態付与

| 層数 | 効果 |
|---|---|
| 1層 | **どく状態**（通常どく） |
| 2層 | **もうどく状態**（重度のどく） |

## 免疫条件

以下の場合、どくびしの状態付与は**発動しない**：

| 条件 | 理由 |
|---|---|
| **ひこうタイプ** | 地面にいないため対象外 |
| **特性「ふゆう」/**磁力上昇等で地面にいない | 地面免疫 |
| **あつぞこブーツ装備** | 場に出た時の地面判定を無視 |
| **すでにどく／もうどく状態** | 追加で状態付与されない（効果なし） | :contentReference[oaicite:3]{index=3}

## 競合・特殊相互作用

| 効果・特性 | 仕様 |
|---|---|
| どくタイプが場に出る | その場のどくびし（全層）を**解除**（ただし非地面状態では解除されない） |
| ミストフィールド中 | どく状態付与を防ぐ（フィールド優先） |
| シンクロ持ち | どくびしで付与されたどく状態では**シンクロは発動しない**（効果なし） |

## ターン処理順（概要）

1. どくびしが成功 → 対象側場にどくびし状態設定  
2. ポケモンが場に出る判定  
3. 該当ポケモンが対象条件なら**どく／もうどく**付与  
4. 状態が解除されるまで維持

## 補足

- どくびしはポケモンが交代したり出てきた場面でしか効果を発揮しない。  
- すでに状態異常のポケモンが場に出た場合、どくびしの状態付与は**追加されない**。  
- 場状態として残る限り効果は持続し、ターン制限は存在しない。 :contentReference[oaicite:4]{index=4}
