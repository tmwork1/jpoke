## 📌 実装推奨順序

### 第1フェーズ: 難易度1 (50-80技)
単純な能力ランク変化のみ。`common.modify_stat` を使用して簡単に実装可能。

**例:**
```python
"ニトロチャージ": MoveData(
    type="ほのお", category="物理", pp=20, power=50, accuracy=100,
    handlers={
        Event.ON_HIT: h.MoveHandler(
            partial(common.modify_stat, stat="S", v=1, 
                    target_spec="attacker:self", source_spec="attacker:self"),
        )
    }
),
```

### 第2フェーズ: 難易度2 (80-120技)
状態異常、ひるみ、吸収、反動の実装。

### 第3フェーズ: 難易度3 (100-150技)
複合効果、条件付き効果の実装。

### 第4フェーズ: 難易度4-5 (200-300技)
複雑な計算、特殊効果の実装。
