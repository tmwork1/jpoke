# 状態変化: まもる（Protect 状態）

# 定義
- 相手の技の効果を受けない揮発性状態変化。

---

# 1. 効果仕様

# 1.1 技効果無効化
- 相手の技によるダメージおよび追加効果を無効化する。

---

# 2. 発生条件

# 2.1 技による付与
以下の技により「まもる」状態が付与される。

- まもる
- みきり
- ニードルガード
- トーチカ

# 追加効果
- ニードルガード
- 直接攻撃した相手に最大HPの 1/8 ダメージ。
- トーチカ
- 直接攻撃した相手をどく状態にする。

---

# 3. 消滅条件

# 3.1 ターン終了
- ターン終了時に解除される。

# 3.2 特定技による解除
以下の技はダメージを与え、まもる状態を解除する。

- フェイント
- シャドーダイブ
- ゴーストダイブ
- いじげんホール
- いじげんラッシュ

---

# 4. 防御仕様

# 4.1 必中技
- 必中技も無効化される。

# 4.2 ノーガード
- ノーガード状態でも無効化される。

---

# 5. 防げない効果

# 5.1 特性・能力変化
- いかく等の能力低下効果は防げない。

# 5.2 定期ダメージ
- 天候ダメージ（すなあらし、あられ）
- 状態異常ダメージ（どく、もうどく、やけど）
→ 防げない。

# 5.3 固有技効果
- 一部の技の追加効果（例：はじけるほのおの反射ダメージ等）は防げない。

---

# 6. 貫通・無視効果

# 6.1 技による貫通
- フェイント
- シャドーダイブ
- ゴーストダイブ
- いじげんホール
- いじげんラッシュ

# 6.2 特性による貫通
- ふかしのこぶしの直接攻撃は防げない。

---

# 7. 防げない技カテゴリ

# 7.1 場に作用する技
- まきびし
- どくびし
- ステルスロック
- ねばねばネット
- その他フィールド・場対象技

---

# 8. 特殊技との相互作用

# 8.1 Z技・ダイマックス技
- Z技および大半のダイマックス技はダメージが 1/4 に軽減される。
- キョダイイチゲキ・キョダイレンゲキは軽減されない。

---

# 9. 連続使用仕様

# 9.1 成功率低下
- 連続して「まもる系統技」を使用すると成功率が低下する。

# 9.2 解除時の扱い
- フェイント等で解除された場合、第6世代以降は連続使用扱いにならない。

---

# 10. 状態管理仕様

# 10.1 状態カテゴリ
- 揮発性状態変化。

# 10.2 排他仕様
- 同時に複数の「まもる」状態は存在しない。
