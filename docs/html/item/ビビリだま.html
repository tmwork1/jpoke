ビビリだまは、アイテムの一種。

概要

  • 買値:300円7, 非売品8, 5000円9
  • 売値:150円7, 2000円8, 1250円9
  • 効果:
    • ポケモンに持たせると、いかくを受けたときすばやさが1段階上がる。1度使うと無くなる。
    • 第七世代では戦闘中に使用すると、野生ポケモンが仲間を呼びやすくなる。ウルトラサン・ウルトラムーンでは野生ポケモンが何度でも仲間を呼ぶようになる。第八世代以降では戦闘中には使用できない。
  • なげつける:威力30
  • ウッウロボ:素材タイプゴースト、素材ポイント6

説明文

サン・ムーン
つかうと なかまを よびやすくなる。 もたせると いかくを うけたとき すばやさが あがる。 つかうとなくなる。
使うと 仲間を 呼びやすくなる。 持たせると いかくを 受けた時 すばやさが 上がる。 使うとなくなる。
ウルトラサン・ウルトラムーン
つかうと なんどでも なかまを よぶ。 もたせると いかくを うけたとき すばやさが あがる。 つかうとなくなる。
使うと 何度でも 仲間を 呼ぶ。 持たせると いかくを 受けた時 すばやさが 上がる。 使うとなくなる。
ソード・シールド・スカーレット・バイオレット
ポケモンに もたせると いかくを うけたとき すばやさが あがる。
ポケモンに 持たせると いかくを 受けた時 すばやさが 上がる。

入手方法

サン・ムーン・ウルトラサン・ウルトラムーン
4ばんどうろで拾う。
カキの試練を達成後、フレンドリィショップで購入する。
ハイナさばくに通じるキャプテンミニゲートでおねえさんからもらう。
エンドケイブで拾うUSUM。
ソード・シールド
バトルタワーで10BPと交換する。
(鎧の孤島) ウッウロボで合成する(ゴーストタイプの完成品ポイント11-15)。
(冠の雪原) とうちょうトンネルの隠しアイテムとして日替わりで復活することがある。
スカーレット・バイオレット
ジムバッジ4個入手後、ハッコウシティのデリバードポーチで買う。

こんなときに使おう

第七世代では野生ポケモンに仲間を呼ばせるときに使用できる。仲間を呼んだときしか出現しないポケモンも存在する (乱入バトル#仲間一覧を参照)。乱入による個体値、隠れ特性、色違い厳選をする際に必須となる道具。

詳細な仕様

ポケモンに持たせた場合

  • いかくによりこうげきランクを下げられる効果は防げない。
  • すでにすばやさランクが最大まで上がっている場合、ビビリだまは発動しない。
    • 所持者があまのじゃくならば、すばやさランクが最小まで下がっている場合、ビビリだまは発動しない。
  • すでにこうげきランクが最小まで下がっていていかくの効果が無い場合、ビビリだまは発動しない。
    • 所持者があまのじゃくならば、こうげきランクが最大まで上がっている場合、ビビリだまは発動しない。
  • 特性かいりきバサミ・クリアボディ・しろいけむり・メタルプロテクト・フラワーベール、しろいきり状態でいかくの効果を防いだ場合、ビビリだまは発動する。第八世代で、きもったま・せいしんりょく・どんかん・マイペースでいかくを無効化したときもビビリだまは発動する。
  • 特性ミラーアーマーでいかくの効果を跳ね返した場合、
    • いかくのポケモンが持つビビリだまは発動しない。
    • ミラーアーマーのポケモンが持つビビリだまはその場では発動しないが、ビビリだまを持って場に残っている間に自身のいずれかの能力が変動すると、直後にビビリだまが発動する。
      • たこがためなどで複数の能力が変化する場合、1つ目の能力の変化が発生したあとにビビリだまが発動し、その後残りの能力の変化が発生する。
      • いかくを跳ね返した後に特性たんじゅん・あまのじゃくに変化し、その後にビビリだまが発動した場合、ビビリだまによる能力変動値は特性の影響を受けて変わる。
  • みがわり状態でいかくを防いだ場合、ビビリだまは発動しない。
  • いかくに対してまけんき・かちきが発動した場合、同時にビビリだまも発動する。
  • 第八世代でいかくに対して特性びびりが発動すると、その後にビビリだまが発動する。すばやさが+5段階のときにいかくを受けたが、びびりの効果ですばやさが最大に上がったときはビビリだまは発動しない。
  • ビビリだまを持つポケモンを交代させようとしたが、先に相手が交代したことで出てきたいかくを受けた場合、ビビリだまを消費した後に交代する。
  • いかくで複数のポケモンの能力が下げられる場合、所持者の能力が下げられた直後、他のポケモンの能力が下げられるより前にビビリだまが発動する。
    • きょうせいのポケモンと、その味方のポケモン(以下Aとする)がどちらもビビリだまを持っており、同時にいかくを受けた場合、きょうせいのポケモンとの位置関係によりきょうせいが発動するかが決まる。
      • トレーナーから見て左側にきょうせいがいた場合は、きょうせいがビビリだまを発動してから、Aがいかくを受けてビビリだまを発動させる。
      • トレーナーから見て右側にきょうせいがいた場合は、Aがビビリだまを発動させた直後にきょうせいが発動し、Aにビビリだまが移されてからきょうせいがいかくを受ける。このときAに移ったビビリだまは発動しない。Aが交代する場合は、きょうせいがいかくを受けてから交代する。
    • しろいハーブ/だっしゅつパックは、いかくの対象全員が受けた後に発動する。きょうせいのポケモンがしろいハーブ/だっしゅつパックを持っており、Aがビビリだまを持っているとき、きょうせいのポケモンとの位置関係に関係なく、Aがビビリだまを消費した後にきょうせいが発動し、Aにしろいハーブ/だっしゅつパックが渡される。
      • Aの能力が下がっている場合は、しろいハーブは発動する。Aが交代する場合はしろいハーブを消費してから交代する。
      • Aがだっしゅつパックを受け取る場合も、能力を下げられた後に受け取るのでだっしゅつパックは発動しない。

戦闘中に使用した場合

  • 1度使用した場合や、トレーナー戦である場合では効果がなく、ビビリだまが減ることもなくターンが送られる。

備考

  • 第八世代から特性びびりにも同じ効果が追加された。
