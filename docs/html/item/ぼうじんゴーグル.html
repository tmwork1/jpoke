ぼうじんゴーグルは、アイテムの一種。

効果

  • 買価:20000円
  • 売価:500円6、2000円7-8、5000円9
  • 効果:持たせたポケモンは、すなあらし・あられ状態によるダメージを受けない。粉や胞子のわざ、および特性ほうしを無効化する。
  • なげつける:威力80
  • ウッウロボ:素材タイプノーマル、素材ポイント5

説明文

第六世代・第七世代・第八世代

配布ポケモンの所持

ポケモン 言語/地域 ソフト
ウインディ (Paul) 全言語/オンライン SV

こんなときに使おう

ねむりごな・キノコのほうしを無力化することでビビヨンやキノガッサの無限戦術に対して強くなる。ラムのみと比較して、技を防いでも無くならない点がメリットだが、さいみんじゅつやあくびなど粉ではない技は無効化できない。

ダブルバトルはでいかりのこなで攻撃を吸われなくなるので、とりわけメジャーなモロバレルに対する対策になる。粉技ではないこのゆびとまれの影響は受けるので、ちゅうもくのまとを完全に無視できるようになるわけではない。

天候ダメージのみを防ぐためだけの目的ならばたべのこし、どくタイプならばくろいヘドロでも十分であるため、粉技を使う相手に役割があるポケモンに持たせたい。特性がんじょうが天候ダメ―ジで潰れないように粉系の技対策のついでに持たせられることもある。

天候ダメージを受ける野生ポケモンを捕獲するとき、トリック・すりかえでぼうじんゴーグルを渡せば自滅されることはない。ただし、渡した後は状態異常にする粉技が効かなくなる点と、第八世代以前だと捕獲に失敗した場合はぼうじんゴーグルを失う点に注意がいる。特性マジックガードをスキルスワップで渡す方法なら注意点が減るので、ぼうじんゴーグルを使う方法はマジックガードを渡す方法が使えないときの代用となる。

詳細な仕様

  • 粉系の技を防いだ際には、「<ポケモン>は ぼうじんゴーグルで <粉系の技>を 受けない!」というメッセージが表示される。
    • 特性ぼうじんのポケモンがぼうじんゴーグルを持って粉のわざを防いだ場合、特性による無効化が優先される。
    • すなあらし・あられ、いかりのこな、特性ほうしを無視したときにメッセージは流れない。
  • いかりのこなによるちゅうもくのまとを無視するかどうかはポケモンが行動するときに決定する。
    • ぼうじんゴーグルを持っていない状態でいかりのこなが使用されたが、行動する前にぼうじんゴーグルを手に入れたポケモンの行動はいかりのこなを無視することができる。逆にいかりのこなが使用された時点ではぼうじんゴーグルを持っていたが、行動する前にぼうじんゴーグルを失ったポケモンの行動はいかりのこなに引き寄せられる。
    • いかりのこなは自分を対象とする粉の技だが、ぼうじんゴーグルを持ったポケモンがいかりのこなを使った場合でも技は成功し、ちゅうもくのまと状態になる。
  • 技ふんじんを受けたときは無効化できるが、ふんじん状態の効果自体を防ぐことはできない。
    • ふんじん状態になった後にぼうじんゴーグルを得た場合、そのターンにほのお技を使おうとするとダメージを受ける。
  • さしおさえ・マジックルーム状態がターン経過で解除されるときは、天候ダメージを受けてから状態が解除されて道具が有効になる。
  • すなあらしによるいわタイプの特防上昇や、あられによるふぶきの必中化など、スリップダメージ以外の天候の効果は防がない。
    • ゆき状態に対しては効果が無い。
    • 特性アイスボディのポケモンに持たせたときも、あられ状態による回復ができる。

備考

  • 持たせたポケモンに特性ぼうじんの効果を追加させるアイテムと言える。このアイテムが登場した第六世代から、ぼうじんは粉系の技も無効になるように強化された。
    • 特性と道具という違いがあるため、かたやぶりやかがくへんかガスはぼうじんのみを無効化し、ぼうじんゴーグルに効果は無い。さしおさえ・マジックルーム状態はぼうじんゴーグルを無効化するが、ぼうじんには効果が無い。
  • アニメではSM編第41話に登場し、デンヂムシレースの砂漠地帯に入る際にサトシとデンヂムシの両方が着用した。
  • ゴーゴーゴーグルというものも存在するが、こちらはトレーナーがさばく地帯に入るためのたいせつなものである。
