かいがらのすずは、アイテムの一種。

効果

  • 買値:20000円SV
  • 売値:100円3-6,BDSP,2000円7,SwSh, 5000円SV
  • 効果:攻撃技を使用した後、相手に与えたダメージの1/8分だけ自分のHPが回復する（小数点以下切捨て）。
  • なげつける:威力30
  • ウッウロボ:素材タイプみず、素材ポイント7

こんなときに使おう

与えられるダメージが少ないポケモンに持たせても回復量には期待できないが、相手のHPを満タンから0にしてもその1/8しか回復できない。持ち物のラインナップが少ない第三世代では回復手段の候補となる。トリプルバトルでは真ん中に配置するポケモンに持たせてしおふきやふんかを使えば威力の維持にもつながる。第五世代以降では、レベル1の特性がんじょうのポケモンががむしゃらでHPを全回復するために持つことがある。

たべのこしとは機能するタイミングが違うため、戦闘終了時にも回復する。シナリオ攻略や経験値/努力値稼ぎの際に、交代時のダメージがかさみやすいポケモンに持たせる道具の候補になる。戦闘後に回復できない対戦施設であるバトルチューブやバトルピラミッド、しばり組手でも有用な道具になる。

高難易度のテラレイドバトルでは相手のHP量が数十倍にもなるため、通常の戦闘より高い回復量が見込める。吸収技を覚えないポケモンでも攻撃と回復を両立して行えるようになる。マックスレイドバトルでも似たことが言えるが、HP倍率が最大でも3倍で、バリアによるダメージ切り捨ても大きいことからテラレイドバトルほどの効果は見込めない。

詳細な仕様

  • 回復量を算出するとき小数点は切り捨てるが、1以上のダメージを与えた際には1回復できる。
  • 攻撃が無効化されたときや、HP1のポケモンをみねうちで攻撃したとき、相手のばけのかわ/アイスフェイスで攻撃を肩代わりされたときなど、与えたダメージが0だった場合は回復しない。
  • 連続攻撃技を使用した場合、回復は最後の攻撃の後に発動する。回復量は、第三世代のみ最後のダメージのみから、第四世代以降は全ての攻撃のダメージの合計から算出する。
  • 全体攻撃技で複数のポケモンに同時にダメージを与えた場合、全員に与えたダメージを合計してから回復量を算出する。
  • 特性ヘドロえきの影響を受けない。かいがらのすずによる回復効果は吸収技による回復効果とは異なる。
  • 発動したとき、「<所持者>は かいがらのすずで すこし かいふく」というメッセージが出る。
  • こんらんで自傷ダメージを受けたときは発動しない。
  • みらいよち/はめつのねがいによるダメージでは発動しない。
  • 特性ちからずくのポケモンが特性の対象技を使うときは、かいがらのすずの回復効果も無くなる。
  • 吸収技による効果、さめはだなどの特性やゴツゴツメット/ジャポのみ/レンブのみによるダメージ、反動技によるダメージの後に回復効果が発動する。
  • 反動技や、ひんしになる攻撃技などの効果で攻撃側のHPが0になったときは、回復効果が発動しないままひんしになる。
  • 相手のレッドカードが発動して攻撃側が交代させられた場合、回復はできない。
  • とんぼがえり/ボルトチェンジ/クイックターンを使用した場合、回復の後に交代する。
  • どろぼう/ほしがる/マジシャンで相手のかいがらのすずを奪った場合、即座にその際のダメージに応じた回復効果が発動する。
  • わるいてぐせのポケモンを直接攻撃してかいがらのすずを奪われる場合、攻撃者がかいがらのすずで回復した後にわるいてぐせが発動する。
  • おどりこで踊りの攻撃技をコピーして使ったときも発動する。

第三世代・第四世代

  • 交代前のポケモンへのおいうちでは発動しない。
  • かいふくふうじ状態でも回復できる。
  • 攻撃がみがわり状態に防がれた場合、第三世代では回復できる。第四世代では、基本的に攻撃でみがわりを壊したとしても回復できない。
    • 第四世代の連続攻撃技でみがわり状態を途中で壊し、本体にもダメージを与えた場合は、みがわりに与えたダメージと本体に与えたダメージの合計から回復量を算出する。
    • 全体攻撃技でみがわり状態のポケモンとそうでないポケモンを攻撃した場合、第四世代でもみがわりに与えたダメージも含めた合計から回復量を算出する。
  • 一撃必殺技が成功したときも発動する。

第五世代以降

  • 交代前のポケモンへのおいうちでも発動する。
  • かいふくふうじ状態では回復できない。
  • 攻撃がみがわり状態に防がれた場合も回復できる。みがわりに与えたダメージから回復量を求める。
  • 一撃必殺技が成功したとき、第五世代・第六世代では発動しない。第七世代以降では発動する。

備考

  • ルビー・サファイア・エメラルドの時計用内蔵電池が切れてしまっている場合、あさせのほらあなが満潮か干潮のどちらかで固定化されてしまい作れなくなる。
    • ポケモンXDではポケクーポンの景品として入手可能。
