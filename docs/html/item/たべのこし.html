たべのこしは、アイテムの一種。

効果

  • 買値:非売品2-8, 20000円9
  • 売値:100円2-6,BDSP(第五世代のアイテムマニアのみ200円), 2000円7,SwSh, 5000円9
  • 効果:ポケモンに持たせておくと、毎ターン終了時にHPが最大の1/16分だけ回復する。
  • なげつける:威力10
  • ウッウロボ:素材タイプくさ、素材ポイント10

こんな時に使おう

この道具を持っている間は永続的にHPを回復できるため、シナリオでは回復どうぐの節約になる。

場に長く居座ることができるポケモンほど効果が大きいため、対戦では耐久型のポケモンに持たせるのが主流となる。しかしそれに該当しないポケモンでも、これの有無で確定数が変わることもある。実用的などうぐが現在よりも少なかった過去世代では消極的に持たせることもあった。

持たせる場合はまもるやみがわりを繰り出すと効果的。HP実数値が16の倍数に届くように調整すれば回復効率が上がる。


詳細な仕様

  • 発動しても消費されることはない。
  • 回復量の計算時、小数点以下は切り捨てる。ただし切り捨てた結果が0になるとき、回復量は1になる(ポケモンスタジアム金銀を除く)。
  • すでにHPが満タンのときは発動しない。
  • みがわり状態でも発動する。
  • 姿を隠しているときでも発動する。
  • かいふくふうじ状態のときは、第四世代までは回復できる。第五世代以降では回復できない。
  • ダイマックスしているとき、回復量はダイマックス前のHPを参照する。
  • たべのこしによる回復が発動するタイミングは毎ターンの終了時となる。
    • 複数のポケモンがたべのこしを持っている場合、すばやさ順に発動させる。
    • 天候による効果や、ひのうみやキョダイゴクエン状態などのダメージ、グラスフィールド状態による回復の後に発動する。
    • 第三世代以降では、状態異常や状態変化によるダメージの前に回復できる。
    • かいふくふうじ・さしおさえ・マジックルーム状態が解除されるターンでは回復できない。
    • ダイマックスはたべのこしが発動した後に終了する。
  • きのみではないため、バトルにおけるきのみ特有の効果は持っていない。
    • ついばむ・むしくいを受けても食べられない。
    • やきつくすで燃やされない。
    • 発動してもゲップが使えるようにはならない。
    • おちゃかいで消費する対象にならない。
    • 特性きんちょうかんの影響を受けていても回復できる。
    • 特性じゅくせいのポケモンが発動させると特性バーが現れるが、回復量は増えていない。
    • 発動した次のターンに特性はんすうが発動することはない。

Pokémon Trading Card Game Pocketにおけるたべのこし

トレーナーズとして登場する。詳しくはたべのこし (ポケポケ)を参照。

備考

  • 「残飯」という俗称で呼ばれることがある。
  • 第四世代から登場したくろいヘドロは、どくタイプのポケモンが持つ場合のみたべのこしと同じ効果を持つ。
  • ゲーム中では食べかけのリンゴのグラフィックで描かれている。
  • ウルトラサン・ウルトラムーンにおけるコックはバトルツリー以外ではすべての手持ちポケモンにたべのこしを持たせている。
