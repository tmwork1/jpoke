せんせいのツメは、アイテムの一種。

効果

  • 買値:非売品2-8, 8000円9
  • 売値:50円2-6,BDSP, 2000円7,SwSh,9
  • 効果:持たせたポケモンは、毎ターン20%の確率で、同じ優先度の行動の中で最初に行動できるようになる (第三世代以降)。
    • 第二世代のみ、発動率は60/256となる。
  • なげつける:威力80
  • ウッウロボ:素材タイプノーマル、素材ポイント13

こんな時に使おう

ストーリーでは序盤から中盤あたりで手に入ることも多い。発動しても無くならないので、ストーリーではすばやさが低いポケモンに持たせておけば多少快適にはなる。

確実性に欠けるため、わざ・道具のバリエーションが増えた第四世代以降の対人戦ではあまり使われず、別のデメリットが存在するが確実に先制しやすいこだわりスカーフの方が用いられやすい。

またすばやさが低いポケモンであっても、そのランダム性故に行動順を元にしたわざの選択[1]がしづらくなるという欠点もある。特に行動順による計画が大事なダブルバトルでは顕著になる。

バトルタワーやバトルフロンティアでは、せんせいのツメを持ったポケモンを相手にすると、このどうぐ1つで負け筋を生むことも少なくない。

相手のどうぐを制御するわざや先制技といった対策手段もなくはないが、これらの殆どは用いることで汎用的な攻撃性能が欠けるため、基本的に相手のせんせいのツメが発動しないのを願うしかない。

詳細な仕様

  • 元から自分の方が先に行動するときでも発動する。
  • 発動しても道具が消費されることはない。
  • 場の状態がトリックルームのときに発動したときも行動順が早くなる。
  • 使用する技の優先度は変わらない。せんせいのツメの発動したときも、その優先度の内でしか最初に行動することはできない。
    • 優先度が0以下であるなら、せんせいのツメの効果が発動した行動がじょおうのいげん/ビビッドボディ/テイルアーマー/サイコフィールド/ファストガードの効果で防がれることはない。
  • 行動順の判定について、詳しくはターン#4.ポケモンの行動を参照とする。
  • 複数のポケモンのせんせいのツメ/イバンのみ/クイックドロウが同じターンに発動した場合、優先度が異なれば優先度が高い順に、同じ優先度ならばすばやさが高い順に行動する。優先度もすばやさも同じである場合、行動順はランダムに決まる。
    • 第五世代以降では、ターン開始時に流れるメッセージも基本的に行動順と同じ順番で流れる。ただし、優先度もすばやさも同じだった場合はメッセージの順番とポケモンの行動順は別途ランダムに決定され、一致するとは限らなくなる。
    • 特性クイックドロウのポケモンがせんせいのツメを持っている場合、効果は累積して44%の確率で行動順が早くなる。
  • 特性あとだしのポケモンのせんせいのツメが発動した場合、あとだしの効果は無くなる。このとき他にせんせいのツメ/イバンのみが発動したポケモンがいても、あとだしに関係なくすばやさの高い方が先攻になる。
  • 所有者の特性がきんしのちからである場合、変化技を選んだターンでせんせいのツメが発動することは無い。攻撃技を選んだターンでは発動し得る。
  • 効果の発動判定は、ターンの最初で行動を決定したときに行われる。発動したターンは、行動前にせんせいのツメを失ったとしてもそのターンだけは早く行動できる。行動前にせんせいのツメを手にしたターンや、持ち物の効果が有効になったターンの行動順が早くなることは無い。
    • 第四世代では、発動は行動直前にしか分からないが、ターンの最初に行動順が決定していることは第五世代以降と変わらない。
    • 第八世代以降では、すばやさや優先度の変化がそのターンの行動順に即座に反映されるが、行動前にせんせいのツメを失ってもそのターンの行動順には影響しない。

第三世代まで

  • 発動を示すエフェクトやメッセージは出ない。

第四世代

  • せんせいのツメの効果が発動して行動が早くなったターンでは、行動開始時に所有ポケモンがもちものを使用する戦闘アニメが発生する。（メッセージでは特に何も言及されない。）
    • 設定で戦闘アニメを「みない」に設定している場合、発動を示すアニメーションは発生しない。
    • ポケモンバトルレボリューションでは発動を示すエフェクトの後に「せんせいのツメが発動しました!」という実況が流れる。
  • せんせいのツメを持っているポケモンは行動順だけでなく、場に出たときの特性の発動順や、交代の順番、ターン終了時に状態異常等のダメージを受ける順番といった、すばやさに従う処理順全般が20%の確率で早くなる。
    • 効果の発動判定は行動選択時とターン終了時の2か所で行われる。このためターン終了時の処理順は、そのターンの行動順と一致するとは限らない。
    • 交代やターン終了時の処理順が早くなっても発動を示すアニメーションは発生しない。

第五世代以降

  • 効果が発動すると、発動を示すエフェクトと共に「<所持者>は せんせいのツメで 行動が はやくなった!」というメッセージが流れる。
  • 所持者が交代を選んだときや、バッグからもちものを使ったとき、にげる/降参を選んだときは発動しない。
  • 他のポケモンが交代/バッグ/にげる/降参を選んだが、所持者が技を選択していたときは発動することがある。このとき、せんせいのツメのメッセージが流れてから交代/道具の使用/逃走処理が行われ、その後ポケモンの行動に移る。
  • トリプルバトルではMOVEを選択したときも発動することがある。
  • 特性の発動順、交代の順番、状態異常の処理順などがせんせいのツメの効果で早くなることはない。

備考

  • 効果や他言語版の名称から「先制の爪」の意であると思われるが、「先生」をかけているのかスクールのせんせいから貰うこともある。第五世代では「宣誓」とかけており、ブラック・ホワイトではからておう、ブラック2・ホワイト2ではOLから貰える。
  • 国内版のポケモンスタジアム金銀では、ジムリーダーのしろ(うら)のワタルがオーダイルとバンギラスの2匹にこのアイテムを持たせるというルール違反をしている。
    • 海外版ではオーダイルの持ち物がしんぴのしずくに変更されている。
