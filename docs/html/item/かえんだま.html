かえんだまは、アイテムの一種。

効果

  • 買値:非売品,15000円9
  • 売値:100円4-6,BDSP,2000円7,SwSh,3750円9
  • 効果:持たせると、ターン終了時にやけど状態になる。
  • なげつける:威力30、相手をやけど状態にする
  • ウッウロボ:素材タイプほのお、素材ポイント7

説明文

第四世代・第五世代・第六世代・第七世代・第八世代
さわると ねつをだす ふしぎなたま。もたせると せんとうちゅうに やけどの じょうたいに なる。
触ると 熱をだす 不思議な玉。持たせると 戦闘中に やけどの 状態に なる。
第九世代
触ると 熱をだす 不思議な玉。 持たせると 戦闘中に やけどの状態に なる。

こんなときに使おう

持たせたポケモンは能動的に特性こんじょう・ふしぎなうろこ・ねつぼうそうを発動できる。はやあしも発動するが、物理型のポケモンに持たせても相性が悪い。

トリックやすりかえで相手に押し付けてやけど状態にすれば物理技のダメージを落とせる。相手が状態異常の回復手段を持っていても、治した矢先に再びやけどになるため効果が出やすい。やけど状態にならないか、やけどのダメージを受けないポケモンに持たせればデメリットが少ない。サイコシフトで自分のやけど状態を相手に移す戦法もある。

ポケモンカードゲームにおけるかえんだま

かえんだま
説明文
(XY) 		のぞむなら、自分の[炎]エネルギーを1個選び、自分のベンチポケモンにつけ替えてよい。
説明文2
(DPt3) 		のぞむなら、自分の[炎]エネルギーを1個、自分のベンチポケモンにつけ替えてよい。
説明文3
(DPt-EP) 		自分のエネルギーを1個、自分のベンチポケモンにつけ替える。（自分のベンチポケモンがいないなら、この効果はなくなる。）
説明文4
(ADV/PCG) 		のぞむなら、自分の[炎]エネルギーを1個選び、自分のベンチポケモンにつけ替えてよい。
初登場 マグマ団のバクーダ (マグマ)

ゲーム本編でアイテムとして登場する以前より、相手にダメージを与えつつ「自分のエネルギーを1個、自分のベンチポケモンにつけ替える」効果を持つワザとして登場。炎エネルギーを指定するものとエネルギーの種類を指定しないものがある。

ポケモンカードゲームには他にエースバーンVMAX (S1a)が持つダイカエンダマというワザも存在する。

このワザを持つカード

  • マグマ団のバクーダ (マグマ)
  • サーナイトexδ-デルタ種 (PCG9-S)
  • コータス (DPt-EP)
  • ブーバーン (DPt3-B)
  • マグマ団のバクーダ (CP1)

詳細な仕様

  • 発動しても消費しない。
  • 以下のポケモンに対してかえんだまは発動しない。
    • ほのおタイプ
    • 特性みずのベール、すいほう、ぜったいねむり、きよめのしお、ねつこうかん
    • 特性リーフガード、リミットシールドに守られている
    • ミストフィールドの影響下にいる
    • すでに状態異常になっている
  • 以下のポケモンに対してもかえんだまは発動する。
    • みがわり状態
    • しんぴのまもり状態
    • フラワーベールに守られている
  • 仲良し度やかいふくアイテム等の手段で回復しても、そのターン終了直前に再発動する。
  • 交代でかえんだまを持ったポケモンを出した場合、そのターンからかえんだまは発動する。
    • 死に出しで出した場合は発動しない。
  • 発動のタイミングは、やけどダメージ、ねむけ、場の状態、フィールド、はねやすめの効果の処理が終わってからとなる。
    • 発動するターンにやけどダメージは受けない。
    • ねむけからねむり状態になるターンとかえんだまの発動ターンが重なった場合、ねむり状態が優先される。
    • ミストフィールドが解除されたターンからかえんだまが発動する。
    • さしおさえ・マジックルーム状態が解除されたターンからかえんだまが発動する。
    • そのターンはねやすめを使用したひこうタイプも地面から浮いたことになるため、ミストフィールドでかえんだまの発動を防げない。
  • だっぴ・うるおいボディ・いやしのこころの発動タイミングの後にかえんだまは発動する。かえんだまが発動したターンにだっぴ・うるおいボディ・いやしのこころでやけどが治ることは無い。これらの特性で状態異常が治ったターンからかえんだまは発動する。
  • かえんだまはものひろいで拾ったターンでは発動しない。

備考

  • 第六世代まで、場に居座るのが3ターン以内ならダメージ量はどくどくだま≦かえんだまなので、4ターン以上場に居続ける場合に持たせると有効。
  • 第七世代以降はやけどダメージが1/16に変わったので、1ターン目で両方のダメージ量が同じ、2ターン目でどくどくだまの方がダメージ量が多くなる。
