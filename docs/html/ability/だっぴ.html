だっぴは、ポケモンのとくせいの一種。

効果

ターン終了時に、自分の状態異常が30%の確率で回復する。

  • 第三世代のみ、発動率は1/3となる。

こんなときに使おう

場に居座る耐久型のポケモンであれば発動の試行回数を稼ぎやすく、もうどくの対策にもなる。ねむるを使うときも通常より早く起きれる可能性がある。

Pokémon UNITEにおけるだっぴ

Pokémon UNITEでは、サナギラスの特性として登場する。

ポケモンカードゲームにおけるだっぴ

自分のHPを回復するワザとしてゲームよりも以前に登場した。

  • ミニリュウ (闇、そして光へ...)
  • ハブネーク (DPt1-B)
  • アーボ (DP4)
  • ズルッグ (BW3)
  • ミニリュウ (BW8)
  • ハクリュー (XY6)
  • トランセル (SM3N)
  • コクーン (S3a)

特性の仕様

  • こんらんなどの状態変化は治せない。
  • 発動のタイミングはターンの終了時になる。詳細なタイミングは以下のようになる。
    • どく/もうどく/やけど/あくむ状態や、特性ナイトメアのダメージ判定より前なので、だっぴが発動するターンはこれらのダメージを受けない。
      • ルビー・サファイアにおいてのみ、ねむりをだっぴで治してもあくむ状態が回復しないバグがある (白昼夢を参照)。
    • ねむけによるねむりの発生よりも前なので、これによりねむり状態になったターンにだっぴは発動しない。ねむけ状態になった後に状態異常になったが、ねむけになった次のターンにだっぴで状態異常が治ったときはねむり状態になる。
    • どくどくだま/かえんだまの発動判定より前なので、だっぴで状態異常が治ったターンからこれらの持ち物が発動する。また、これらの持ち物が発動したターンにだっぴで状態異常を治すことはない。
    • あばれる状態のねむり状態による中断判定より前に発動する。あばれる状態の途中でねむり状態にさせられたときや、ねごとで出た技であばれる状態になったとき、だっぴが発動しなければあばれる状態は中断されて技を選べるようになるが、だっぴが発動した場合はあばれる状態は継続して技を選べない。
  • 味方の場に特性いやしのこころのポケモンがいるときは、状態異常の回復判定を2回受けられることになる。

備考

  • 似たような効果がある特性としてうるおいボディ/いやしのこころがある。
  • 本特性を持つズルッグは、分類も「だっぴポケモン」である。
