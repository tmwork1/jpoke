とびだすなかみはポケモンのとくせいの一種。

効果

相手の攻撃技でひんしになったとき、技を受ける直前に残っていたHPの分だけ、その相手にダメージを与える。

説明文

第七世代・第八世代・第九世代
あいてに たおされたとき HPの のこりの ぶんだけ あいてに ダメージを あたえる。
相手に 倒されたとき HPの 残りの ぶんだけ 相手に ダメージを あたえる。

所有ポケモン : そのポケモンの別のとくせい

特性1 特性2 隠れ特性
ナマコブシ とびだすなかみ - てんねん

こんなときに使おう

死に際で相手に反撃ができる。しかし、ナマコブシ自身はぼうぎょ・とくぼうこそ高いがHP自体は低いため、耐久戦で使用しても与えるダメージはあまり期待できない。HPに努力値を振って実数値を増やし、弱点を突いてくる相手や積み技で能力を上げた相手に出すなど、HP満タンの状態から一撃で倒されるように立ち回ると相打ちを狙いやすくなる。

特性の仕様

  • 直接攻撃でなくても、自分が攻撃技を受けてひんしになれば発動する。
  • 天候や状態異常のダメージなど、攻撃技以外のダメージでひんしになった場合は発動しない。
  • この特性の効果でお互いのポケモンのHPが0になった場合、とびだすなかみ側が先にひんしになる。このときお互いのポケモンが全滅した場合はとびだすなかみ側の負けになる。
  • 味方の攻撃に対しても発動する。
    • 味方の範囲攻撃などでこの特性が発動し、攻撃した仲間をひんしにさせて全滅した場合、同時に相手も全滅させたとしてもとびだすなかみ側の負けとなる。
  • 相手のみがわり状態を無視してダメージを与える。
  • 連続攻撃技が複数回当たってひんしになった場合、連続攻撃技が当たり始めた時点での残りHPの分だけ相手にダメージを与える。
    • 途中きのみが発動してHPが回復したときや、途中まではみがわり状態で攻撃を防いでいたときなど、最初のHP量と受けたダメージの総量が異なる場合でも、最初のHP量がとびだすなかみで与えるダメージになる。
  • ダイマックスしている間に倒されて発動したときは、残っていたHPをダイマックスレベルによる補正率で割った値が与えるダメージになる。
    • ダイマックスした相手に倒されたときは、残っていたHPそのままが相手に与えるダメージになる。
  • じばく/だいばくはつによる攻撃を受けたときなど、攻撃者がすでにひんしになって場にいない場合、とびだすなかみが発動することもない。
  • みらいよち/はめつのねがいの攻撃を受けた場合、技を出したポケモンが場にいれば発動する。場にいない場合は発動しない。
  • 後攻のコアパニッシャーを受けてひんしになった場合、コアパニッシャーの効果でとくせいなし状態になってから倒れるためとびだすなかみは発動しない。
  • HPを吸収するわざ/はじけるほのお/やきつくす/ダイマックスわざを受けてひんしになった場合、技の効果が発動してからとびだすなかみが発動する。
  • こうそくスピン/サウザンアロー/サウザンウェーブ/プラズマフィスト/とんぼがえり/ボルトチェンジ/クイックターン/オリジンズスーパーノヴァ/ラジアルエッジストーム/コンビネーションわざを受けて発動し、攻撃側もひんしになった場合は技の効果は発動しない。
  • アイアンローラーを受けて発動したとき、攻撃してきた相手をひんしにしたとしても技の効果は発動する。

備考

  • ナマコブシの専用特性。
  • 同じく自分がひんしになったときに攻撃しててきた相手にダメージを与える特性としてゆうばくがある。ゆうばくはとびだすなかみとは異なり直接攻撃に対してしか反応しない。
