おうごんのからだは、ポケモンのとくせいの一種。

効果

他のポケモンから受ける変化技を無効化する。

説明文

第九世代
酸化せず 丈夫な 黄金の体は 相手からの 変化技を 受けない。

所有ポケモン：そのポケモンの別のとくせい

ポケモン 特性1 特性2 隠れ特性
サーフゴー おうごんのからだ - -

こんなときに使おう

ほとんどの変化技を無効化するため、相手に自分の行動を妨害されにくくなる。効果対象は変化技に限るので、攻撃技の追加効果、特性、持ち物などの影響は受ける。

特性の仕様

  • 自分が対象の変化技は防がない。
  • 味方から受ける変化技も無効化する。
  • 場を対象とした変化技は防がない (ステルスロックなどの相手の場が対象の技、くろいきりなどの全体の場が対象の技、リフレクターなど味方の場が対象の技が挙げられる) 。
  • 複数のポケモンを対象とする変化技に巻き込まれた場合、この特性のポケモンのみが技を防ぎ、他のポケモンは技を受ける。
  • 相手のマジックコート状態で跳ね返ってきた変化技も無効化する。
  • かたやぶり・きんしのちからの効果が発動した変化技は防げない。
  • トリックガードやマジックコート状態とは異なる効果範囲を持つ。他のポケモンから以下の技を受けたときは無効化する。ただし、いずれの場合もこの特性のポケモン自身が使用する技が自身の特性で防がれることはない。
    • 自分含む全員が対象の技 (おちゃかい・ほろびのうた)
    • 味方全員が対象の技 (いのちのしずく・とおぼえ)
    • 味方1体が対象の技 (アロマミスト・てだすけ)
    • へんしん・なりきりなど、対象の状態をコピーする技
    • スキルスワップ・トリック・スピードスワップなど、対象と能力などを入れ替える技
    • おさきにどうぞ・さいはい・さきおくりなど、行動順に関する技
    • おきみやげ : 使用者はひんしにならない。
    • のろい (呪い) : 使用者のHPは減少しない。
    • ちからをすいとる : 使用者のHPは回復しない。
  • 変化技の対象に直接選ばれていない場合は、変化技により発動した効果でも防がない。
    • みちづれ状態の相手を倒したときは、みちづれの効果で自分もひんしにされる。
    • スレッドトラップ・トーチカ・ニードルガード状態の相手に直接攻撃したときに発動する効果は防げない。
    • 相手のふういん状態により技は出せなくなる。
    • 相手のちゅうもくのまと状態により技は引き寄せられる。
    • 他のポケモンのおかたづけにより、自分のみがわり状態は消滅する。
    • 味方のうつしえにより、自分の特性は書き換えられる。
    • 味方のねがいごと・いやしのねがい・やどりぎのタネの効果で回復することはできる。
    • 味方のさいきのいのりで復活することはできる。
    • 味方のバトンタッチによりランク補正や状態変化は引き継がれる。
    • 味方のサイドチェンジで位置を入れ替えることはできる。

備考

  • サーフゴーの専用特性。
  • ほぼ全ての変化技を無効化するものの、上書きできない特性やコピーできない特性ではない。特性の効果や、かたやぶりの効果がある技により、おうごんのからだを消されることや、他のポケモンがおうごんのからだを手にすることはできる。
