ミラクルスキンは、ポケモンのとくせいの一種。

効果

この特性のポケモンが受ける変化技の命中率は50になる。

説明文

第五世代・第六世代・第七世代・第八世代・第九世代
へんかわざを うけにくい からだに なっている。
変化技を 受けにくい 体に なっている。
ブラック2・ホワイト2（ライブキャスターのチェレン）
この とくせいの ポケモンは へんかわざに たいして のみ かいひりつが ばいに なるんだ　じぶんで じぶんに つかう へんかわざには こうかが ないから おぼえておくと いいよ

こんなときに使おう

変化技を中心に使う相手に強くなるが、命中率を半減させるだけなので不確実ではある。この特性を持つポケモンは、他に防御面で優秀な特性を持っていることが多いので、もちものやわざに頼らずに状態異常やちょうはつを受けずに起点を作りたい場合など、ピンポイントな採用理由になる。

特性の仕様

  • 必中技など、命中判定が無い技を回避することはできない。あくび/ほえる/どくタイプが使用したどくどくなどや、自分が対象の変化技、味方の場/相手の場/全体の場が対象の技などが該当する。
  • 状態/もちもの/ランク補正など、命中が変動する他の効果はミラクルスキンの後に計算する。
    • 例えば回避率が6段階上がったミラクルスキンのポケモンが命中75の変化技を受ける場合、ミラクルスキンの効果で命中が50に変化した後にランク補正の効果で1/3倍 (小数点以下切り捨て) になり、結果として命中率は16%になる。
    • 詳しい仕様は命中#第五世代以降を参照。
  • ミラクルスキンによる回避率上昇はランク補正によるものではない。特性するどいめ/てんねんのポケモンが使用した変化技もミラクルスキンで回避することができる。
  • 複数のポケモンを対象とする変化技に巻き込まれた場合、この特性のポケモンに対する命中判定にだけ特性の効果がある。
  • 他の技が出る技によって出された攻撃技に対しては、ミラクルスキンは発動しない。
  • かたやぶりの効果がある変化技に対してミラクルスキンは発動しない。
  • ミラクルスキンであると同時にマジックコート状態である場合、命中判定より前にマジックコートの判定があるため、変化技を跳ね返す前にミラクルスキンの効果で回避することはない。
    • 他のポケモンのマジックコートで跳ね返ってきた変化技はミラクルスキンで回避することができる。
  • ミラクルスキンの効果で技を回避した場合、技の使用者が持つからぶりほけんは発動する。

備考

  • この特性を通常特性として持つポケモンはシンボラーのみ。
  • 命中の処理において、ミラクルスキンの効果は基本的な命中率が50を超える変化技にのみ発動するように設定されている。ただし、最も命中率が低い変化技は第七世代以降のダークホールの50であるので、第九世代現在では必中でない全ての変化技の命中率が50に変わるものと取れる。
  • マジックミラーの効果対象にならず、ミラクルスキンの効果対象になる技 (反射の対象技ではなく、命中判定がある技) には以下がある。
    • あくむ/うつしえ/おきみやげ/サイコシフト/さきおくり/すりかえ/たこがため/テクスチャー2/トリック/フラフラダンス
  • おうごんのからだの効果対象にならず、ミラクルスキンの効果対象になる技は無い。ただし、おうごんのからだはいのちのしずく/てだすけなど、対象にとってメリットがある変化技も無効化するため、一概にミラクルスキンがおうごんのからだの下位互換特性とは言えない。
