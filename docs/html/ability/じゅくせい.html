じゅくせいは、ポケモンのとくせいの一種。

効果

このポケモンが発動させたきのみの効果が2倍になる。

  • HP・PPを回復するきのみの場合は、回復量が2倍になる。
  • ランク補正が上昇するきのみの場合は、上昇量が2倍になる。
  • 半減系きのみの場合は、受けるダメージを1/4にする。
  • ジャポのみ・レンブのみの場合は、与えるダメージが相手の最大HPの1/4になる。

説明文

第八世代・第九世代
じゅくせい させることで きのみの こうかが ばいに なる。
熟成 させることで きのみの 効果が 倍に なる。

所有ポケモン：そのポケモンの別のとくせい

特性1 特性2 隠れ特性
カジッチュ じゅくせい くいしんぼう ぼうだん
アップリュー じゅくせい くいしんぼう はりきり
タルップル じゅくせい くいしんぼう あついしぼう

こんなときに使おう

耐久型のタルップルなら消費したきのみをリサイクルで復活させることができる。

耐久が低いアップリューにはあまり採用されない。

特性の仕様

  • じゅくせいのポケモンがなげつける/おちゃかいを受けてきのみを消費させられたときも特性バーが現れ、きのみの効果が2倍になる。じゅくせいのポケモンがなげつける/おちゃかいを使用しただけでは2倍にならない。
  • じゅくせいのポケモンがむしくい/ついばむを使用して相手のきのみを食べたときも特性バーが現れ、きのみの効果が2倍になる。じゅくせいのポケモンがむしくい/ついばむを受けただけでは2倍にならない。
  • ほおばるで消費したきのみの効果は2倍になるが、特性バーは現れない。ほおばるの技自体によるランク変動量は変わらない。
  • 最大HP割合で決まるきのみの回復量は小数点以下の端数を切り捨てるが、じゅくせいの効果は切り捨てた後の値を2倍する。「1/4回復」が「1/2回復」になるわけではない。
    • 例えばHP150のポケモンがオボンのみで回復する場合、150/4=37.5なので通常は37回復するが、この特性によって2倍になると75ではなく74回復になる。
  • ジャポのみ/レンブのみで与えるダメージは「最大HPの1/4 (切り捨て) 」になる。「最大HPの1/8 (切り捨て) の2倍」ではない。
  • 半減系きのみが発動したとき、じゅくせいの特性バーは現れないが、受けるダメージは1/4になる。1/2の補正が2回掛けられるわけではない。
  • 以下のきのみを消費したときもじゅくせいの特性バーは現れるが、他の特性のポケモンが消費したときと効果は変わらない。
    • ラムのみなど状態異常を回復するきのみは、1度発動することで通常どおり無くなる。
    • サンのみによる急所ランク上昇は+2のままで、確定急所にはならない。
    • ミクルのみによる命中補正は1.2倍のまま変わらない。
    • イバンのみによる行動順が、他のイバンのみ/せんせいのツメ/クイックドロウの効果より優先されることは無い。
  • かがくへんかガスのポケモンがひんしになる攻撃技を使用してかがくへんかガスが解除されるとき、その攻撃で発動した半減系きのみにじゅくせいの効果は無い。ナゾのみ/オボンのみ/タラプのみなどが発動するときはじゅくせいの効果がある。
  • コアパニッシャーを受けてとくせいなし状態になるとき、半減系きのみにじゅくせいの効果はあるが、ナゾのみ/レンブのみ/オボンのみ/タラプのみなどにじゅくせいの効果は無くなる。
  • プレイヤーがバッグから使用したきのみにじゅくせいの効果は無い。
  • バトル外で基礎ポイントを減少させるきのみを使用したときの効果は2倍にならない。
  • 以下の道具はきのみではないため効果は変わらないが、発動するとじゅくせいの特性バーが現れる。
    • たべのこし/きのみジュース/しろいハーブ/メンタルハーブ
    • からぶりほけん/のどスプレー/じゃくてんほけん/じゅうでんち/きゅうこん/ひかりごけ/ゆきだま/ビビリだま/エレキシード/サイコシード/ミストシード/グラスシード/ルームサービス/ものまねハーブ
    • きあいのハチマキ/せんせいのツメ/パワフルハーブ/だっしゅつボタン/だっしゅつパック
    • なげつけるで受けて追加効果が発動したでんきだま/かえんだま/どくどくだま/どくバリ/おうじゃのしるし/するどいキバ/しろいハーブ/メンタルハーブ

備考

  • カジッチュ系統の専用特性。ただし、この特性のカジッチュをカミッチュに進化させるとかんろなミツに変化する。
