こんなときに使おう

威力が5上がったソーラービームの物理版。晴れ状態であれば1ターンで攻撃できる。ソーラービーム同様に雨や砂嵐だと威力が下がる。

技の仕様

  • その場で溜める溜め技の1つ。溜め技に共通する仕様は当該項目を参照。
  • 切る技の1つ。特性きれあじのポケモンが使うと威力が1.5倍になる。
  • 溜め行動や天気に関する効果はソーラービームと同様の仕様を持つ。詳細な効果はソーラービーム#技の仕様も参照とする。
    • 天気がにほんばれ状態のときに使用すると、溜めずに攻撃できる。
    • パワフルハーブを持っているときに使用すると、道具を消費して溜めずに攻撃できる。にほんばれ状態でもある場合は、天気の効果が優先されてパワフルハーブは消費しない。
    • 溜め始めた際、「<使用者>は ひかりを きゅうしゅうした!/光を 吸収した!」というメッセージが流れる。にほんばれ状態/パワフルハーブの効果が発動したときでも、溜めのメッセージが流れた後に「<使用者>の ソーラーブレード!」のメッセージが出る。
    • 天気があめ/すなあらし/あられ/ゆき状態のときにダメージを与えた場合、威力が半分になる。らんきりゅう状態では威力に影響は無い。
    • 場に特性ノーてんき/エアロックのポケモンがいる場合、天気の影響は完全に無視される。
    • 使用者がばんのうがさを持っている場合、にほんばれ状態でも溜めるようになり、あめ状態でも威力が下がらなくなる。すなあらし/あられ/ゆき状態では威力が半減する。
    • 一度溜め行動を始めると、攻撃までの間ににほんばれ状態の効果を得られたとしても溜め行動は無くならない。

備考

  • 似た効果の特殊技であるソーラービームと比べて、威力が5高い。
  • 第七世代ではラランテスの専用技だった。第八世代以降では剣をモチーフにしたポケモンの習得者が増えた。
  • ソーラービームと異なり、ほのおタイプの習得者は限られる。第八世代ではギャロップ系統、第九世代ではソウブレイズ/オーガポン (かまどのめん) が覚える。
