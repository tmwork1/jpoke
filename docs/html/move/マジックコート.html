こんなときに使おう

使用したターンしか効果がないため、相手がマジックコートの効果対象になる技を使うタイミングを読み切る必要がある。変化技の対策はみがわりやちょうはつでもできるが、これらは相手より先に行動できなければ効果的でない。優先度が高いマジックコートならば、すばやさが低いポケモンでもちょうはつなどを対策できる。

マンガにおけるマジックコート

ポケットモンスターSPECIALにおけるマジックコート

  • ポケットモンスターSPECIALでは、野生のアブソルなどが使用する。

技の仕様

詳細についてはマジックコート (状態変化)を参照。
  • 使用したターンが終了するとマジックコートの効果は消える。
  • そのターンの最後に使用したときは失敗する。
  • よこどりの対象にならない。
  • こだわり系アイテムを持つポケモンが使用した場合、跳ね返した技ではなくマジックコートでこだわる。

第四世代まで

  • マジックコートの効果は一度技を跳ね返すことでも消滅する。
  • 特性プレッシャーのポケモンに対して技を跳ね返した場合、マジックコートのPPは2減る。

第五世代以降

  • マジックコートの効果はそのターンの終了時まで継続する。
    • 技を跳ね返してもマジックコート状態は解除されない。
    • さいはいでマジックコートを使用した後、そのターン中に別の行動をしたときもマジックコート状態は解除されない。
  • すでに自身がマジックコート状態であるときは失敗する。さいはいでマジックコートを続けて使用したときに起こり得る。このときもマジックコート状態は解除されない。
    • 特性マジックミラーのポケモンがマジックコートを使用した場合は成功する。このときマジックコートの効果対象になる技を受けると、マジックコートが先に発動して特性の表示は出ない。
  • 特性プレッシャーのポケモンに対して技を跳ね返した場合でも、マジックコートのPPは1だけ減る。

備考

  • 第三世代では命中の表記が100になっているが、自分を対象に選択する変化技なので命中に関係無くマジックコート状態になることができる。反射される技が当たるかは、跳ね返す技の命中に依存する点も第四世代以降と変わらない。
  • 第五世代で登場した特性マジックミラーのポケモンは常にマジックコート状態になる。
    • 第九世代ではマジックコートは使用できなくなったので、特性マジックミラーでしかマジックコート状態になることができない。
  • ウルトラサン・ウルトラムーンではズガドーンのレベル技だが、教え技では覚えることができない。
