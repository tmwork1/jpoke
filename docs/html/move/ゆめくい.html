こんなときに使おう

眠らせる技と組み合わて初めて有効になるHP吸収技。相手の場のポケモンがねむり状態でも、使用する前に交代されることや、先に行動した相手が起きて無効にされることもあるため、使い所は難しい。エスパー技には使用条件が無いサイコキネシスがあるので、対人戦で使われることは少ない。

技の仕様

  • ねむり状態ではない相手に使うと失敗するが、技を選択すること自体はできる。選択してから使用するまでに相手がねむり状態になった場合は成功する。
    • 逆に、相手がねむり状態のときに選択しても、使用時までにねむり状態が回復していた場合は失敗する。
  • 特性ぜったいねむりのポケモンに対しても効果がある。
  • 回復量を算出する際、小数点以下は第四世代までは切り捨て、第五世代以降は四捨五入する。
  • おおきなねっこを持たせたポケモンが使う場合は回復量が通常の1.3倍になる。
  • 特性ヘドロえきのポケモンを攻撃した場合は、第四世代までは特性の影響を受けずに回復できるが、第五世代以降は回復効果がダメージに変わる。
  • かいふくふうじ状態のポケモンは回復できない。
    • 第五世代まで、技を使用することはできるが、HPを回復する効果が無くなる。
    • 第六世代以降では選択することもできない。攻撃する前にかいふくふうじ状態にされた場合は失敗する。
  • 第四世代までみがわり状態のポケモンには無効だった。

備考

  • 第一世代で覚えるエスパーポケモンはスリープ・スリーパー・ミュウのみだった。
  • 第四世代までは使用後に「てきの（又はあいての、やせいの）<相手ポケモン名>の ゆめを くった！」という特殊なメッセージが表示される。
  • HP吸収技の中では最も威力が高い。
  • 本技を覚えるムンナは、分類も「ゆめくいポケモン」である。
