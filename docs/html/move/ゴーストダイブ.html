こんなときに使おう

主にゴーストタイプのポケモンが覚える、2ターンかけて攻撃するゴーストタイプ物理技。他の溜め技と同様に、ただ使っても1ターンの猶予を与えてしまい、能力アップや交代の隙を作って逆に不利になることも考えられるが、他の溜め技と異なりまもるを無効化できる特長がある。1ターンは相手の攻撃から逃れられる所を生かし、どくややけどのターン稼ぎとして使うという方法もある。シングルバトルなら普通にノーマルタイプに交代されたら無効化されてしまうので、くろいまなざしなどで交換を縛る、またはステルスロックを撒くなど交換を躊躇させる準備があると決まりやすい。

ダブルバトルなら、ただ居座れば巻き込まれる味方の全体攻撃をかわすために使いつつ、シングルバトルよりも重要な技となるまもるやワイドガードなどを牽制できることもできる。

技の仕様

  • 1ターン姿を隠す溜め技の1つ。溜め技に共通した仕様は当該項目を参照。
  • 威力とPPを除きシャドーダイブと同じ性能を持つ。詳細な仕様はシャドーダイブ#技の仕様、およびシャドーダイブを参照とする。

備考

  • ギラティナの専用技であるシャドーダイブの威力を落としてPPを増やしたような性能を持つ。こちらは多くのポケモンが覚えられる。
  • 専用技や使用条件がある技を除けば、ゴーストタイプの物理技で最も威力が高い。
  • ゴーストタイプ以外でこの技を覚えるポケモンはミュウ/アルセウス/イベルタル/フーパ（ときはなたれしすがた）のみ（アルセウスはプレートを持たせることでゴーストタイプになれる）。
