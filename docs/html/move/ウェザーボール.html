特別なわざ

  • ライコウ (クラウン)：配布
  • フシギダネ：ポケモン集めてもらおうキャンペーンでプレゼントされたタマゴから生まれたフシギダネが覚えている。

こんなときに使おう

注意: この節には独自研究や主観的な記述、ファンによる推測が含まれています。この節の内容で適切な出典がない場合、その内容には明確な根拠が示されていません。

威力は50と低いが、天気を変えれば実質威力100を安定した命中率で繰り出せる高性能のわざになるので、天気を変える状況下で戦うポケモンに採用されうる。

すなあらしのときはいわタイプでの貴重な特殊わざになる。にほんばれとあめのときは天気の効果で威力150まで強化されるが特性ノーてんきのポケモンには威力50と素の威力しか出せないので注意。

ポケモン不思議のダンジョンにおけるウェザーボール

救助隊
PP15。効果範囲は直線上。天気によってタイプ・ダメージが変化する。
天気 タイプ ダメージ
はれ・くもり ノーマル 通常
ひざしがつよい ほのお 2倍
あめ・きり みず 2倍
ゆき・あられ こおり 2倍
すなあらし いわ 2倍
ポワルンしか覚えられないわざ。すなあらし以外ならとくせい「てんきや」でタイプが変わって自分と同じタイプのわざになる。
自分と同じタイプのわざは本編ポケットモンスター以上に強力であり、直線上の効果範囲も相まって強力。
しかし本作ポケダンでは、とくせいで変えた場合を除いて天気は数ターンで元に戻ってしまうため、目当てのタイプで繰り出すならその都度天気を変える必要がある。
戦闘アニメは、灰色の弾を発射するというもの。タイプがノーマル以外の場合はポケモンに当たった際のヒットエフェクトが加わり、タイプによって見た目と効果音が異なる。
探検隊・冒険団
わざの分類がタイプごとだったのが各わざごとに変わり、ノーマル・いわタイプでも特殊わざとなった。それ以外のわざとしての性質は救助隊と同様。
新たにロズレイドが覚えられるようになった。本作ポケダンはDPt準拠であるため、ルギアとホウオウは覚えることができない。
またわざやどうぐで天気を変えた場合も基本的に永続となり、カバルドンやユキノオー等天気を変えるとくせいのポケモンが増えたため、前回よりも活かしやすくなった。
超不思議のダンジョン
PP14と少しだけ減少したが、わざを鍛錬することでPPを上げることが可能。
本作ポケダンでははれ以外の天気ではどのタイプのポケモンもHPが自然回復せず、更にいずれの手段で天気を変えても一定ターンではれに戻る[1]ため、やや活かしづらくなった。
戦闘アニメは弾を発射してヒットエフェクトが発生するもので、ノーマル以外の場合は色が変わって効果音が少し異なるだけで、救助隊のようなタイプに応じたエフェクトはない。

Pokémon GOにおけるウェザーボール

Pokémon GOではスペシャルアタックとして登場する。2019年3月30日のハスボー限定リサーチイベントで実装された。本編とは異なり、わざのタイプはポケモンごとに決まっておりてんきには依存しない。

実装当初のウェザーボールはノーマル・ほのお・みず・こおりの4種類があったが、2021年6月1日のディスカバリーシーズン(GOバトルリーグ シーズン8)開始時にノーマルタイプのウェザーボールが無くなりいわタイプに変更された。

ノーマルタイプのウェザーボールは2021年8月31日のGOバトルリーグ シーズン9開始時に新しいわざとして改めて実装された。

マンガにおけるウェザーボール

ポケットモンスターSPECIALにおけるウェザーボール

  • ポケットモンスターSPECIALでは、ルビーのポワルンなどが使用する。

技の仕様

  • 弾のわざの1つ。特性がぼうだんのポケモンに対しては無効化される。
  • 天気がらんきりゅう状態のときはタイプ・威力いずれも対応した変化は無い。
  • 技を選んだターンにおいて、使用する前に天気が変化した場合は使用時点の天気を参照してタイプと威力が決まる。
  • 特性ノーてんき、エアロックを持つポケモンが場にいるときは天候が変化していてもノーマルタイプになり、威力は増えない。
  • ばんのうがさを持つポケモンが使用したときは、晴れ・雨状態でもノーマルタイプになり、威力は増えない。すなあらし・あられ・ゆきではタイプが変わり、威力も増える。
  • 第四世代のみノーマルスキンの効果を受ける。第五世代以降はノーマルスキン・フェアリースキン・フリーズスキン・スカイスキン・エレキスキンの効果は発動しない。
  • わざがノーマルタイプのまま変わらなかった場合のみ、プラズマシャワーの効果を受ける。
  • そうでんの効果を受けると、どの天候でもでんきタイプになる。
  • どの天候であってもノーマルZを持っているときのみZワザに変えて使うことができる。通常の天候下ではウルトラダッシュアタックであるが、天候により対応したタイプのZワザに変わる。例えば晴れ状態ではウルトラダッシュアタックはダイナミックフルフレイムに変わる。威力はいずれも160。
    • Zねごとなど、他のZワザの効果でウェザーボールをZワザにして繰り出した場合、天候でタイプが変わる効果はない。いずれも威力160のウルトラダッシュアタックとなる。
  • ダイマックスわざにした場合、天候に対応したタイプのダイマックスわざになる。威力は130。
    • 技選択画面ではどの天候でもダイアタックであり、相性予測もノーマルタイプのものになるが、実際に技を繰り出すときはダイバーン、ダイストリーム、ダイロック、ダイアイスに変わる。使用者がキョダイマックスポケモンで対応したタイプのキョダイマックスわざがあれば、特定天候下で専用のキョダイマックスわざになる。

備考

  • 第三世代ではポワルンの専用技だった。
  • 第三世代以前はタイプごとに分類が決められていたため、ノーマル/いわタイプで繰り出すと物理わざになるが、第四世代でどのタイプで繰り出しても特殊わざになる。
  • ほのおタイプになったときは晴れの効果でさらにダメージが1.5倍に、みずタイプになったときは雨の効果でさらにダメージが1.5倍になる。
