こんなときに使おう

自分と相手の行動順により使用用途が変わる。そのため、お互い優先度がある技を覚えているかどうかも重要になる。

先攻の場合、そのターン生き残ってしまうと技を晒すだけになるので、HPが残り少なく次の攻撃で倒される状況で使いたい。持ち物がきあいのタスキならHPが1で残る機会が多いため相性が良い。

後攻の場合、次のターンの相手はみちづれの効果が残った状況で行動することになるため、攻撃を躊躇わせることができる。ただしこのわざの優先度が0なので使ったターンに先制されて倒されないように。

ポケモン不思議のダンジョンにおけるみちづれ

ダメージを受けると相手に反撃するわざになっている。冒険団シリーズまではおいうちも同じ効果を持っていたが、みちづれはマグナゲートと∞迷宮でも引き続きこの効果。

ポケモンカードゲームにおけるみちづれ

「自分の[超]エネルギーを1個トラッシュ。次の相手の番、ワザのダメージで自分の残りHPがなくなったなら、そのワザを使ったポケモンをきぜつさせる。」という効果のワザとして登場する。ゲームよりも先にゴース (第1弾)のワザとして登場している。

技の仕様

詳細についてはみちづれ (状態変化)を参照。
  • ゆびをふる/ねこのて/よこどりで選ばれない。
  • まねっこで使うことができない。
  • ねごとでみちづれを出すことはできる。
  • マックスレイドバトル/テラレイドバトルで使用すると失敗する。

第二世代から第四世代まで

  • みちづれ状態の効果が発動すると、攻撃側が先にひんしになり、その後にみちづれ使用者がひんしになる。
  • 一部の公式ルールでは、最後の1匹に使わせることができない技となっている。
  • ポケモンスタジアム金銀・ポケモンバトルレボリューションでは、味方のポケモンが使用者1匹しか残っていないときに使うと失敗する。
    • 味方のポケモンが2匹以上残っていれば、相手のポケモンが残り1匹であっても成功する。

第五世代以降

  • みちづれ状態の効果が発動すると、みちづれ使用者が先にひんしになり、その後に攻撃側がひんしになる。
  • この技の効果でお互いのポケモンが全滅した場合、みちづれを使用した側が負けとなる。
  • 第六世代まで、みちづれは連続で使用しても成功する。

第七世代以降

  • みちづれが成功した直後にもう一度みちづれを使うと、必ず失敗する。
    • 連続使用してみちづれが失敗した直後に、もう一度みちづれを使った場合は成功する。
  • みちづれが成功してからPPを消費した行動を取らなかった場合も、その直後に使うみちづれは失敗する。自身のみちづれ状態は成功の可否と関係しない。2回のみちづれの間に以下の行動しか取らなかった場合、連続使用と見なされて後発のみちづれは失敗する。
    • 状態異常/状態変化/なまけにより行動できなかった。使用できなかった技がみちづれでも他の技であっても同様。
    • 集中力が途切れてきあいパンチが失敗した。
    • 特性おどりこで踊りの技を使用した。
    • バッグから道具を使った。
    • 逃げようとして失敗した。
  • ねごと/さいはいによるみちづれは、直接みちづれを使ったときと同一視される。みちづれ直後にねごと/さいはいでみちづれを出した場合、ねごと/さいはいでみちづれを出した直後にみちづれを使用した場合、ねごとでみちづれを連続で出した場合など、これら全ては連続使用と見なされる。
  • Zワザにすると自分を注目の的状態にする効果が付加される。
    • このゆびとまれで発生する状態と同じだが、技の優先度はみちづれと変わらない。また、みちづれ状態と異なり注目の的状態はターン終了で消えるため、後攻で使うみちづれをZワザに変えても、そのまま使用したときと変わらない。
    • みちづれとZみちづれを続けて成功させることもできない。みちづれ成功直後にZみちづれを使用した場合、注目の的にはなるがみちづれ自体は失敗する。

備考

  • 効果が似ている技におんねんがある。第七世代以降でもおんねんに連続使用制限はなく、おんねんとみちづれを交互に使用しても失敗しない。
  • 第二世代で相手をみちづれにした場合は倒れるエフェクトが通常の倒れ方と異なる。
  • 第三世代のポケモンコンテストではのろい（ゴーストタイプでなくても良い）からのコンボでハートを16個獲得できる。のろいの効果により4番手になるため妨害も受けにくいが、緊張はしやすい。使用可能なポケモンはドーブルとゴース系統、カゲボウズ系統など。第六世代ではコンボにならなくなっている。
