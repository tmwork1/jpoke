注意: この節には独自研究や主観的な記述、ファンによる推測が含まれています。この節の内容で適切な出典がない場合、その内容には明確な根拠が示されていません。

他にじめん技を覚えないポケモンであれば候補になるが、溜め技に共通した欠点として潜っている間に交代・まもる・みがわり・能力上昇などの猶予を与えることになる。攻撃技として使うより、どくどくなどと組み合わせたターン稼ぎを目的として使われることもある。潜っている間でもダメージ2倍で当たってしまう技として、採用率が高い技であるじしんもあるため、他の溜め技より1ターン稼ぐ目的も果たしにくい点もあなをほるの弱点と言える。パワフルハーブと併用するか、Zワザ/ダイマックスに変えて使えば溜め無しで攻撃できる。

シナリオでは隙になることは少なく、威力80とそこそこの威力があるじめん技として使える。また第六世代以前とBDSPでは、フィールドであなぬけのヒモと同じ効果として使えるため、移動要員の技スペースが余ったとき覚えさせる技の候補となる。

技の仕様

バトルにおける効果

詳細についてはあなをほる (状態変化)を参照。
  • 1ターン姿を隠す溜め技の1つ。溜め技に共通する仕様は溜め技の記事を参照。
  • スカイバトルでは使用できず、ゆびをふるなどでこのわざが選ばれた場合失敗する。

フィールド上における効果

  • 第六世代までと、BDSPでは洞窟・森・塔などのフィールドで使うと入り口に戻る(第一世代では最後に使ったポケモンセンターに戻る)。
    • 第一世代ではテレポートブロック（シルフカンパニーなどにある移動装置）の上で使用すると、移動元ではテレポートブロックの演出が優先され、主人公が回転せずにいきなり画面上部に飛び去り、移動先では通常時同様に回転しながら着地する。
      • あなぬけのヒモやテレポートを使った際と同様に、ワープする際に主人公の姿が「アイウエ」(文字が反転している)となる現象がある。この現象はスーパーゲームボーイやポケモンスタジアムのGBビルでプレイした場合は起こらない。
    • 第一世代で洞窟などのダンジョン以外で使おうとすると、本来道具を不適切な場所で使ったときのメッセージと同じ「オーキドのことば…（以下略）」が表示される。そらをとぶやいあいぎりの場合は専用のメッセージが表示される。
  • 第七世代からはフィールドで使うことができなくなった (BDSPを除く)。

備考

  • 第一世代の攻略本ではほとんどの書籍で威力60と書かれていた。第二世代では実際に威力60に変更されている。
  • SM/USUMではわざマシンに収録されていないので習得できるポケモンが限られる。
  • BDSPではフィールドわざとして復活したが、本作の他のわざマシンと異なり、あなをほるが収録されているわざマシン28は1つのセーブデータにつき1つしか手に入らない。
  • 本技を覚えるホルビー、ホルードは、分類も「あなほりポケモン」である。
  • ポケモンバトリオでは、サイホーンのわざとして登場する。
