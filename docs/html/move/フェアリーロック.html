こんなときに使おう

使用したターンに相手の攻撃でひんしになるか、だっしゅつボタンで交代して有利対面を作れば、相手はそのターン交代できない。

ポケモンカードゲームにおけるフェアリーロック

次の相手の番、このワザを受けたポケモンは逃げられなくなる効果がある。

  • クレッフィ (XY3)
  • クレッフィ (SM6)

技の仕様

  • フェアリーロック (場の状態)も参照。
  • 使用した次のターンに再度フェアリーロックを使った場合、場はすでにフェアリーロック状態であるため失敗する。
  • 場を対象にした技だが、オウムがえしでコピーすることができる。

備考

  • クレッフィの専用技。
  • ソード・シールドでは、Ver.1.2.0でクレッフィが登場する以前においても使用不可能な技ではなく、ゆびをふるで出現することがあった。
    • Ver.1.1.0では戦闘アニメが再生される環境でフェアリーロックが使用されるとソフトがクラッシュ(強制終了)する不具合があった。戦闘アニメが再生されなければ問題なく使用することができる。Ver.1.0ではこの現象は起こらない。Ver.1.1.1の配信で修正された。
