こんな時に使おう

基本的にパワフルハーブとの併用を前提で運用したほうが使いやすい。計算上、1回メテオビームを溜めて攻撃したときのダメージは、パワージェムを2回使用したときのダメージより高く、相手がいわに耐性を持つポケモンを用意していないならそのまま使用しても割に合う。

第八世代ではダイマックスわざに変えたときも1ターンで使用できる。

技の仕様

  • 1ターン溜める溜め技の1つ。溜め技に共通する仕様は当該項目を参照。
    • 技を溜め始めるとき「<ポケモン>に 宇宙の 力が あふれだす!」というメッセージが流れる。
  • パワフルハーブでこの技を溜めずに使った場合、わざが外れたり無効化されたりしてもとくこうは1段階上がる。
  • 技の効果は追加効果には分類されず、使用者の特性がちからずくでも威力は上がらず、自分のとくこうを上げることができる。

備考

  • パワージェム以来、14年振りのいわタイプの特殊技である。
  • いわタイプの特殊技の中で最も威力が高い。
  • スカーレット・バイオレットではVer.3.0.0でわざマシンに収録されてポケモンに覚えさせることができるようになった。Ver.1.0.1-2.0.2ではゆびをふるでのみ出現する技だった。
