こんなときに使おう

高威力のどく技だが、持ち物がきのみのポケモンでなければ発動機会はほぼ無い。最初のターンから使う手段が無い点もデメリットとなる。

ポケモン不思議のダンジョンにおけるゲップ

PP15。正面の敵1体を攻撃する（かどぬけ可）。本編と異なりきのみを食べなくても出せるが、おなかが減っていると失敗する。

技の仕様

  • 使用条件を満たしていないときにゲップを選ぼうとした場合、「<ポケモン名>は きのみを 食べていないので 技が だせない!」というメッセージが出て選択はキャンセルされる。このときゲップ以外に選べる技が無いポケモンは、「たたかう」を押した時点でわるあがきを出すことになる。
    • ゲップを選べずわるあがきを使用し始めた後にきのみを消費した場合、次のターンからゲップを使えるようになる。わるあがきで行動する前にきのみを消費したとしても、そのターンはわるあがきを使用する。
    • HP・PP・状態異常が全快した状態で開始されるバトルの1ターン目でゲップを使用する方法は無い。
  • オウムがえし/さきどりでコピーできず、ねごと/ねこのて/まねっこ/ゆびをふる/さいはいにより選ばれない。
    • これらの技を出すポケモンがゲップの使用条件を満たしていても選ばれない。
  • 第八世代まではものまねでコピーできる。第九世代ではできない。

使用条件

  • 使用条件はきのみを食べること。きのみを食べるアニメーションが発動する以下の状況においてゲップは使用できるようになる。
    • きのみを持たせて普通に食べて消費したとき。
    • なげつけるで相手からきのみを投げつけられたとき。
      • 追加効果がないきのみを投げられても食べるアニメは出ず、ゲップは使えない(ブリーのみなどコンテスト用きのみ・半減の実など)。
      • 追加効果はあるが、それが発動しなかった場合は食べるアニメは出ないが、ゲップは使用できるようになる (状態異常でないときのラムのみなど)。
    • むしくい/ついばむを使用して相手のきのみを食べたとき。
      • 効果があるきのみを食べた場合、発動/非発動を問わずゲップを使用できるようになる。
      • 半減の実/ナゾのみ6-7/イバンのみ/ジャポのみ/レンブのみの効果は無く、きのみを食べるアニメは起きないがゲップを使用することができる(なげつけるとは異なる)。
      • コンテスト用のきのみは食べてもゲップを使用することはできない。
    • ほおばる/おちゃかいの効果で自分のきのみを食べたとき。
      • 半減の実/イバンのみなどバトルにおける効果があるきのみを食べたなら、発動/非発動を問わずゲップを使用できるようになる。
      • 基礎ポイントを下げるきのみはバトルにおける効果が無いので、食べるアニメも発動せず、ゲップも使用できない。
    • なげつけるを使用して消費したきのみを次のターンはんすうで食べたとき。
      • 半減の実/イバンのみなど、バトルにおける効果があるきのみを食べたなら、発動/非発動を問わず使用できるようになる。
      • 基礎ポイントを下げるきのみを食べてもゲップは使用できない。
  • きのみを食べた後にリサイクル/しゅうかく/ものひろいなどできのみが復活したとしても使用できる。
  • 持ち物にきのみを持っていたとしても、それを食べる以外の方法で消費してしまうと使用できない。したがって、以下の状況においてゲップは使用できない。
    • なげつける/ギフトパスを使用して自分のきのみを手放したとき。
    • しぜんのめぐみを使用してきのみを消費したとき。
    • むしくい/ついばむを受けて自分のきのみを食べられたとき。
    • どろぼう/ほしがる/トリック/すりかえ、特性マジシャン/わるいてぐせ/きょうせいなどの効果で、持っているきのみが他のポケモンに渡ったとき。
    • やきつくす/はたきおとす/ふしょくガスの効果できのみを失ったとき。
    • ストーリーにおいて、ポケモンのもちものではなくバッグからきのみを食べさせたとき。
  • きのみを食べた後に交代しても使用できる。
    • きのみを食べてからひんしになった後、戦闘中にげんきのかけら/さいきのいのりなどで復活したときでも使用できる。
    • 使用できるのはきのみを食べた戦闘中のみ。戦闘終了後に再び戦闘になれば、またきのみを食べて消費しなければ使用できない。
  • きのみを食べたポケモンにへんしんしたとしても、自身がきのみを食べていないかぎり使用できない。
  • きのみを食べた状態はバトンタッチで引き継がれない。
  • きのみ以外の食べ物 (きのみジュース/たべのこしなど) や、HPを回復する手段 (くろいヘドロ/かいがらのすずなど) では使用条件を満たせない。
  • マックスレイドバトルでのレイドボスはきのみを食べていなくてもゲップを使用できる。
    • テラレイドバトルでのレイドボスはきのみを食べてなければゲップを使えない。
