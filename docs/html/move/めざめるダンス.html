めざめるダンスは、ポケモンのわざの一種。

説明文

たたかうわざ

第七世代・ソード・シールド・第九世代
ぜんりょくで おどって こうげきする。 この わざの タイプは じぶんの タイプと おなじになる。
(漢字) 全力で 踊って 攻撃する。 この 技の タイプは 自分の タイプと 同じになる。
ブリリアントダイヤモンド・シャイニングパール
このわざは つかえません おもいだすことが できなくなりますが わざを わすれることを おすすめします
(漢字) この技は 使えません 思い出すことが できなくなりますが 技を 忘れることを おすすめします

技の仕様

  • 踊りの技の1つ。使用者以外の特性おどりこによりコピーされる。
  • 技のタイプはタイプ1と同じになるため、タイプ無しになっていない限りタイプ一致になる。
  • 特性おどりこ/他の技が出る技/スケッチ/ものまねなどにより、オドリドリ以外のポケモンがこの技を使用した場合も使用者のタイプ1と同じタイプの技となる。
  • 複合タイプのポケモンがもえつきるやはねやすめを使用しタイプ1がなくなった後にこの技を使用した場合、めざめるダンスのタイプはタイプ2を参照する。
    • はねやすめしたターンにめざめるダンスを使うには、はねやすめ中に特性おどりこで使用するか、さいはいを受けてはねやすめを出した後めざめるダンスを使う方法がある。
  • ほのお単タイプのポケモンがもえつきるを使用し、かつハロウィン/もりののろい状態のときにこの技を使用した場合、めざめるダンスのタイプはゴーストまたはくさタイプになる。ほのお・ひこうタイプかつハロウィン/もりののろい状態のポケモンがもえつきるとはねやすめを使用したときも同様。
  • ほのお単タイプのポケモンがもえつきるを使用しタイプ無しとなった後にこの技を使用した場合、めざめるダンスのタイプも無くなる。このタイプの無いめざめるダンスは以下の効果を持つ。ほのお・ひこうタイプのポケモンがもえつきるとはねやすめを使用しタイプ無しとなっている場合も同様。
    • 全タイプに等倍となる。
    • タイプ一致ボーナスは無い。
    • 対象の特性へんしょくは発動しない。
    • 対象の特性ふしぎなまもりにより防がれる。
    • この技に対するテクスチャー2は失敗する。
  • ひこう単タイプのポケモンがはねやすめを使用しタイプ無しとなった後にこの技を使用した場合、めざめるダンスのタイプはノーマルになる。ハロウィン/もりののろい状態の有無は問わない。
  • この技を使用した際、使用者の特性ノーマルスキン/フェアリースキン/フリーズスキン/スカイスキン/エレキスキンの効果は発動しない。
  • そうでん状態の効果を受けると、使用者のタイプに関係無くでんきタイプになる。タイプ1がノーマルのポケモンがプラズマシャワー状態のときに使用したときもでんきタイプに変わる。
  • Zワザに変えるときは、自身のタイプに関係無くノーマルZでウルトラダッシュアタックに変えることしかできない。
  • テラスタルしている場合もその時のタイプ1を参照するため、基本的にはテラスタイプと同じタイプの技になる。
    • ステラタイプにテラスタルしている場合は、元のタイプのタイプ1を参照するため、ステラタイプの技にはならない。
