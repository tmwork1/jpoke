## 実装（handlers/ / data/）
- 変更なし。`すなかき_modify_speed`（`ON_CALC_SPEED`, `subject_spec="source:self"`）と

  は仕様書の内容と一致しており、`subject_spec` もイベントの発火元コンテキスト
  （`speed_calculator.py` の `EventContext(source=mon)`、`status_manager.py` の
  `EventContext(source=source, target=target, ...)`）と整合していることを確認した。
- すいすい（`weather_for(ctx.source).rainy`）との実装の違いを確認した。ばんのうがさは
  「にほんばれ・あめ」限定の効果無効化アイテムであり、すなあらしには影響しないため、すなかきが
  `battle.weather.name` を直接参照する現行実装は正しい（`weather` プロパティ自体が
  `ON_CHECK_WEATHER_ENABLED` 経由でノーてんき/エアロックの無効化を反映済み）。ゆきかきも同一パターン。
- すなあらしダメージ免除は、すながくれ・すなのちから・ぼうじんと共通の `_ignore_sandstorm_damage`
  ヘルパーを使用しており、重複実装や不要なガードもない。
