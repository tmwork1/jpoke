## 実装（handlers/ / data/）
- 修正した内容（バグ修正）:
  - `src/jpoke/data/ability.py`: 登録イベントを `Event.ON_DAMAGE_HIT` → `Event.ON_HIT` に変更。
    `Event.ON_DAMAGE_HIT` は `actual_damage<=0` のとき発火しないため、仕様の
    「HP1でこらえるを使用したときや、みねうちを受けたときなど、0のダメージを受けたときは発動する」
    を満たせていなかった（だっしゅつボタンと同様の既知の制約）。
  - `src/jpoke/handlers/ability.py` の `こぼれダネ_set_grassy_terrain`: `Event.ON_HIT` は
    みがわりに阻まれた場合（実HPダメージ0）でも発火してしまうため、`ctx.substitute_damage`
    が真の場合は何もしないガードを追加し、「みがわり状態で攻撃を防いだときは発動しない」仕様を満たした。
  - タイプ相性による無効化（`ON_BEFORE_APPLY_MOVE` が False を返すケース）は元々 `_execute_hit` 自体が
    呼ばれないため、`Event.ON_HIT` に変更した後も引き続き発動しないことを確認した。
