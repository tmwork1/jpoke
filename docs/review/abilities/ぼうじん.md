# ぼうじん レビュー結果

## 仕様書
- `docs/spec/abilities/ぼうじん.md` の空だった「## 特性の仕様」節に、一次情報（pokemonwiki）の
  「詳細な仕様」節をもとに以下を追記した。
  - 天候ダメージを防ぐ際に特性メッセージが出ない
  - すなあらしのいわタイプ特防上昇などスリップダメージ以外の天候効果は防がない
  - ゆき状態はそもそもダメージが無いため実質無効
  - かたやぶりの効果がある粉技は防げない
  - くさタイプとの重複時、第七世代以降は特性側のメッセージが優先される
- 「## 効果」「## 備考」の記述は一次情報と一致しており変更なし。

## 実装計画書
- `docs/plan/abilities/ぼうじん.md` が古い実装状態（field.py ですなあらし免除、粉技無効はON_CHECK_IMMUNE、
  ほうし無効は未実装）を記載したままだったため、現在の実装（`Event.ON_MODIFY_NON_MOVE_DAMAGE` /
  `Event.ON_BEFORE_APPLY_MOVE` によるability.py側実装、ほうし無効も実装済み）に合わせて全面的に更新した。

## 実装（handlers/ / data/）
- `handlers/ability.py` の `ほうし_maybe_inflict_ailment_on_contact` が、攻撃側の道具
  `ぼうじんゴーグル` のみをチェックしており、攻撃側の特性 `ぼうじん` を無効化条件に含めていなかった
  バグを修正した（一次情報で「特性ほうしを無効化する」と明記されているため）。
  `ctx.attacker.ability.name == "ぼうじん"` の判定を追加。
- 天候ダメージ免除（`ぼうじん_block_sandstorm_damage`）・粉技無効（`ぼうじん_block_powder`）は
  仕様書・計画書と一致しており変更なし。
- `docs/spec/turn.md` の `ON_APPLY_MOVE`（実体は `Event.ON_BEFORE_APPLY_MOVE`）節に、実装済みの
  「ぼうじん・ぼうじんゴーグル: 粉技無効」の行が抜けていたため追加した。また `ON_TRY_MOVE_2` 節に
  誤って記載されていた「ぼうじんゴーグルによる粉技無効」の行（実際は `ON_BEFORE_APPLY_MOVE` で
  処理されており該当イベントではない）を削除した。

## リーサル計算
- 対象外（`docs/progress/ability.md` の「リーサル実装」列が `n/a`）。
  同種のすなあらし免除特性（すなかき・すながくれ・すなのちから）も同様に `n/a` であり、
  `handlers/lethal.py` の `すなあらし_damage` がこれら免除特性を考慮しない実装になっている点も
  既存の設計方針（天候免除特性のリーサル計算は対象外）と整合している。

## テスト
- `tests/abilities/test_ability_ha.py` に以下を追加した。
  - `test_ぼうじん_かたやぶりの粉技は防げない`: かたやぶり持ちの粉技（キノコのほうし）は
    ぼうじんで防げないことを確認。
  - `test_ぼうじん_特性ほうしを無効化する`: 特性ほうし持ちに接触してもぼうじん側が状態異常を
    受けないことを確認（上記の実装バグ修正に対する回帰テスト）。
- 既存の `test_ぼうじん_すなあらしダメージを無効化する` / `test_ぼうじん_粉技を無効化する` は
  実装と一致しており変更なし。
