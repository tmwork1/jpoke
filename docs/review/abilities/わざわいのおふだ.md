## 実装（handlers/ / data/）
- `src/jpoke/handlers/ability.py` の `わざわいのおふだ_reduce_A` に、
  攻撃側自身がわざわいのおふだを持つ場合（コピー・交換等による取得を含む）を
  除外する条件（`ctx.attacker.ability.name != "わざわいのおふだ"`）を追加した。
  - 修正前は `ctx.attacker is not ctx.defender` のみで判定しており、
    場に複数体わざわいのおふだ所有者がいる場合（例: チオンジェン同士の対面）に、
    攻撃側自身もわざわいのおふだを持っているにもかかわらず攻撃が0.75倍に
    軽減されてしまう不具合があった（仕様書「わざわいのおふだを持つポケモンは
    全員がこの特性の効果を受けず、攻撃が下がらない」に違反）。
  - `data/ability.py` の登録内容（`ON_CALC_ATK_MODIFIER` / `subject_spec="defender:self"`）
    は仕様と整合しており変更なし。
- イカサマ・ボディプレス・こんらん自傷ダメージとの相互作用は、既存のダメージ計算経路
  （`ctx.attacker` / `ctx.defender` の実体がそれぞれの効果で入れ替わる設計）に委ねられており、
  追加の分岐なしで仕様どおりに動作することを確認した（変更不要）。
- フォトンゲイザーの分類判定（`ranked_stats` を参照し `ON_CALC_ATK_MODIFIER` の影響を受けない）
  も仕様どおりであることを確認した（変更不要）。
