## 実装（handlers/ / data/）
- `src/jpoke/data/ability.py` の `ダークオーラ` エントリを修正した：
  - `flags={"mold_breaker_ignorable"}` を削除した。`docs/spec/abilities/オーラブレイク.md` に
    「第八世代ではフェアリーオーラ/ダークオーラはかたやぶりで無視されない」と明記されており、
    現行世代を対象とする本プロジェクトではダークオーラの威力補正はかたやぶりに無視されるべきでは
    ないため、誤って付与されていたフラグを削除した（かたやぶりが無視するのはオーラブレイク側であり、
    ダークオーラ自体ではない）。
  - `Event.ON_SWITCH_IN` / `Event.ON_ABILITY_ENABLED` に `announce_ability_triggered`
    （`subject_spec="source:self"`）を追加した。仕様書に記載された登場時のアナウンス
    （「<ポケモン>は ダークオーラを 放っている!」）と特性開示が未実装だったため追加した。
    `docs/spec/turn.md` の `ON_SWITCH_IN` イベント表にも「ダークオーラ（特性）」が priority 100 で
    掲載されており、これに整合させた。オーラブレイク・かたやぶり等の既存の同種特性と同じ実装パターン。
  - `src/jpoke/handlers/ability.py` の `ダークオーラ_boost_power`（あく技威力を5448/4096倍、
    オーラブレイク所持時は3072/4096倍にする実装）は仕様と一致しており変更なし。
