## 実装計画書

  作成した。既存の類似特性 `ちからもち.md` の構成に合わせた）。

## 実装（handlers/ / data/）
- `src/jpoke/handlers/ability.py` の `よわき_modify_atk` に `"_こんらん"` 除外を追加した。
  修正前は「こんらんの自滅ダメージを受けるときは発動しない」という仕様（一次情報にも明記）が
  実装されておらず、こんらんの自傷ダメージにもよわきの攻撃半減補正が誤って適用されるバグが
  あった。コードベース内の同種ハンドラ（`こんじょう_modify_atk`、`ちからもち_boost_physical`
  など）で徹底されている `ctx.move.name != "_こんらん"` 除外パターンに合わせて修正した。
- `src/jpoke/data/ability.py` の登録内容（`Event.ON_CALC_ATK_MODIFIER`、
  `subject_spec="attacker:self"`）は仕様と整合しており変更なし。
- ボディプレス・イカサマ時の挙動、ダメージ固定技への非影響、みがわり時の判定などは
  `core/damage.py` の `_calc_final_attack` の既存実装により自動的に満たされることを確認した
  （専用分岐は不要）。
