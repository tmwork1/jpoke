# とびだすハバネロ レビュー結果

## 仕様書
- 変更なし。`docs/spec/abilities/とびだすハバネロ.md` は一次情報（`docs/wiki/abilities/とびだすハバネロ.html`）と照合し、内容の相違なし。
  - 攻撃技でダメージを受けたとき攻撃者をやけど状態にする（発動率100%、直接攻撃に限らない）
  - まもる状態/相性無効/みがわりで防いだ場合は不発動、相手がみがわり状態でも発動
  - やけど耐性を持つ相手（既に状態異常、ほのおタイプ、すいほう/きよめのしお、リーフガード/フラワーベール、しんぴのまもり、ミストフィールド）には無効
  - 連続攻撃技は1発ごとに判定、全体技はダメージ算出後に発動しダメージには影響しない
  - みらいにこうげきは使用者が場にいる場合のみ発動
  - メガスコヴィラン専用特性、ポケモンチャンピオンズで新規追加（対象範囲内）
