# じゅくせい レビュー結果

## 仕様書
- 変更なし（`docs/spec/abilities/じゅくせい.md` は既に一次情報 `docs/wiki/abilities/じゅくせい.html` と一致していた）。

## 実装計画書
- 新規作成（`docs/plan/abilities/じゅくせい.md`）。進捗表では「計画書: x」だったが実際にはファイルが
  存在しなかったため、仕様書をもとに新規作成した。
  くいしんぼう（`_gluttony_denominator`）と同様、専用ハンドラを追加せず既存のきのみ処理内で
  `mon.ability.name == "じゅくせい"` を判定するインライン方式とした。

## 実装（handlers/ / data/）
- 未実装（`AbilityData()` のみで効果ロジックが一切無い）だったため、計画書に従って新規実装した。
  - `src/jpoke/handlers/item.py`
    - `is_ripen(mon)` / `berry_heal_amount(mon, *, r=None, v=None)` を追加。
      端数切り捨て後の回復量をじゅくせい所持時に2倍にする（切り捨て前に2倍にしてから丸めるのではない）。
    - `_heal_berry` — `berry_heal_amount` 経由に変更（イア・ウイ・バンジ・フィラ・マゴ・オボン・
      オレンの各のみが対象）。
    - `_boost_on_quarter_hp` / `_boost_on_attack_category` — ランク上昇量を2倍にする
      （チイラ・リュガ・カムラ・ヤタピ・ズア・アッキ・タラプの各のみが対象）。
    - `_modify_super_effective_damage` — modifier を1回の乗算で2乗（0.5→0.25）して算出し、
      「1/2を2回掛ける」ことによる端数のズレを避けた（17種の半減系きのみ + ホズのみが対象）。
    - `_retaliate_on_category` — `-1/8` ではなく `-1/4` を直接算出する
      （ジャポのみ・レンブのみが対象。1/8を2倍にするのではなく1回の乗算で1/4を算出する）。
    - `ナゾのみ_heal_on_super_effective` — `berry_heal_amount` 経由に変更。
    - `himeri_pp_restore_cap(mon)` を追加し、`ヒメリのみ_restore_pp` /
      `ヒメリのみ_restore_pp_on_item_enabled` のPP回復上限をじゅくせい所持時に20へ変更。
  - `src/jpoke/handlers/move_attack.py`
    - `なげつける_apply_item_effect` の該当分岐（ヒメリ・オレン・オボン・フィラ/ウイ/マゴ/バンジ/
      イア・チイラ/リュガ・アッキ/カムラ/ヤタピ/ズア・タラプ・スターの各のみ）に、
      投げられた対象（`ctx.defender`）がじゅくせい所持のときの2倍化を追加した
      （`item.py` の `is_ripen` / `berry_heal_amount` / `himeri_pp_restore_cap` を import して再利用）。
      サンのみ・ミクルのみは仕様通り対象外のため変更していない。
  - `src/jpoke/data/ability.py` — `"じゅくせい": AbilityData()` は変更なし
    （くいしんぼうと同じインライン判定方式のため専用ハンドラは不要）。
  - `src/jpoke/core/move_executor.py` — 関連バグ修正: `_consume_pp` で
    `Event.ON_PP_CONSUMED` が一度も発火されておらず、`ヒメリのみ_restore_pp`
    （自分が技を使い切った直後にPPを回復する経路）が事実上デッドコードだったため、
    PP消費後に `Event.ON_PP_CONSUMED` を発火するよう追加した。
    じゅくせいのテスト作成中に発見した、じゅくせい固有ではない既存の実装漏れ。
- 対象外（変更しない）として確認したもの: 状態異常回復系（カゴ・クラボ・チーゴ・ナナシ・モモン・
  ラム・キーの各のみ）、サンのみ（急所ランク上昇は+2のまま）、ミクルのみ（命中率補正は1.2倍のまま）、
  イバンのみ（行動順の優先扱いなし）、きゅうこん・じゅうでんち・ひかりごけ・ゆきだま等
  （きのみではないアイテムのため`_boost_stat_on_type_hit`は変更していない）。
