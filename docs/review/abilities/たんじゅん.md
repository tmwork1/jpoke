## 実装（handlers/ / data/）
- 変更なし。`src/jpoke/handlers/ability.py` の `たんじゅん_double_stat` と
  `src/jpoke/data/ability.py` の `"たんじゅん"` エントリ（`mold_breaker_ignorable` フラグ +
  `ON_BEFORE_MODIFY_STAT` ハンドラ、`subject_spec="target:self"`）を仕様書・計画書と照合したが、
  誤り・欠落は見つからなかった。同型の特性 `あまのじゃく` と同一パターンで実装されている。
  - 命中率/回避率のランクにも効果が及ぶ（`value` の辞書を汎用的に2倍化するため）。
  - 急所ランクには影響しない（別経路のため `ON_BEFORE_MODIFY_STAT` を経由しない）。
  - バトンタッチ/じこあんじ/パワースワップ/ガードスワップ/ハートスワップ/ひっくりかえす/
    くろいきり/クリアスモッグは `modify_stats()` を経由しない直接代入のため影響を受けない。
  - パワーシェア/パワートリックは実数値の直接書き換えのため影響を受けない。
  - かたやぶりの効果がある技を受けた場合は `mold_breaker_ignorable` フラグにより無効化される。
  - ちからをすいとるの回復量は特性によるランク変化前の実数値を用いているため、たんじゅんで
    ランク低下量が2倍になっても回復量は変わらない（仕様どおり）。
