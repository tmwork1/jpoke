# うたかたのアリア レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/うたかたのアリア.md`）。
- 一次情報（ポケモンWiki）と `docs/champions/move_list.txt` / `docs/progress/move.md` を突き合わせ、以下を明記した。
  - Champions基準のPPは12（本家シリーズは10）。
  - ちからずくの対象技であること（威力1.3倍・やけど回復無効化）。
  - りんぷん・おんみつマント所持の相手には発動しないこと（シングルバトルのみ対象のため常に無効扱いでよい）。
  - まもる等で無効化された場合は発動しないこと、みがわりは音のわざとして貫通すること。
  - `docs/spec/turn.md` の `## Event.ON_DAMAGE`（実装上 `Event.ON_DAMAGE_HIT` に相当）優先度100で発動すること。
  - Championsでは使用者がひんしになった場合でもやけど回復が発動すること（本家SVまでとの差分）。

## 実装（handlers/ / data/）
- 以下の誤り・欠落を修正した。
  1. **PP誤り**: `pp=10` → `pp=12`（Champions基準。`docs/champions/move_list.txt` / `docs/progress/move.md` と不一致だった）。
  2. **`flags` 欠落**: `flags={"sound"}` → `flags={"sound", "secondary_effect"}`。
     うたかたのアリアはちからずく対象技（`docs/spec/abilities/ちからずく.md` に明記）だが、
     `"secondary_effect"` フラグが無く威力上昇・追加効果無効化が発動しない状態だった。
  3. **イベント誤り**: `Event.ON_HIT` → `Event.ON_DAMAGE_HIT`。
     `docs/spec/turn.md` の `## Event.ON_DAMAGE` セクションに「100 | うたかたのアリアでやけど回復」と
     明記されており、これは実装上 `Event.ON_DAMAGE_HIT` に対応する
     （`はたきおとす` のコード内コメントで対応関係を確認済み）。誤ったイベントに登録されていた。
  4. **追加効果の無効化ロジック欠落**: `src/jpoke/handlers/move_attack.py` の
     `うたかたのアリア_cure_defender_burn` が `battle.resolve_secondary_chance` を経由しておらず、
     ちからずく・りんぷん・おんみつマントによる無効化が一切反映されない状態だった。
     `handlers/move.py` の `apply_ailment_to_defender` 等と同じガードパターン
     （`resolve_secondary_chance` → `chance < 1` なら乱数判定でスキップ）を追加した。
