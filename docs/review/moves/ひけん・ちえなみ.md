## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ha.py`: `pp=16` → `pp=15` に修正（一次情報でPPは15）
- `src/jpoke/data/moves/move_ha.py`: `flags={"contact"}` → `flags={"contact", "slash", "secondary_effect"}` に修正
  - `slash`: 一次情報に「切る技」であり特性「きれあじ」で威力1.5倍になると明記されているが欠落していた
  - `secondary_effect`: 一次情報の判定表で「追加効果: ○」となっており、がんせきアックス（同系統技）の技の仕様に明記された「ちからずく使用時は威力1.3倍になるが追加効果が発動しない」の対象技であるが欠落していた
- `src/jpoke/handlers/move_attack.py`: `ひけん・ちえなみ_set_spikes` にちからずくガード（`_is_sheer_force_blocked`、既存の共通ヘルパーを再利用）を追加。ちからずく使用時にまきびし設置が発動しないようにした
  - りんぷん・おんみつマントは対象個体ではなく場を変化させる効果のため影響を受けない仕様のため、`resolve_secondary_chance` は経由させていない（一次情報のがんせきアックス技仕様の記載と整合）
- `python scripts/sort_handlers.py src/jpoke/handlers/move_attack.py` と `python scripts/sort_data/sort_moves.py` を実行済み
