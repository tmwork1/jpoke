# ハードローラー レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/ハードローラー.html` を確認したところ、本技はカテゴリ
「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」に分類されており（`wgCategories`）、第八世代の説明文も
「このわざは つかえません」となっている。`docs/champions/move_list.txt` にも本技のエントリは存在しない
（`ハードプレス` と `ばかぢから` の間にエントリなし）。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。以降の項目はこの判断に基づく対応内容を記載する。

さらに、削除前の既存実装を確認したところ `MoveData(type="ノーマル", power=100, accuracy=95, pp=10)` という
値が登録されていたが、実際のゲームでの本技の基礎値は **むしタイプ・物理・威力65・命中100・PP20** であり、
タイプ・威力・命中率のすべてが誤っていた（対象範囲外である以前に、仮に実装対象だったとしても値が誤っていた）。
なお、依頼文にあった「急所に当たりやすい技」という前提も一次情報からは確認できず、既存実装にも
急所ランク補正は含まれていなかった（本技は通常の急所判定のみで、急所ランク補正のある技ではない）。
