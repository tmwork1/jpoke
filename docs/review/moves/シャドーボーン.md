# シャドーボーン レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/シャドーボーン.html` を確認したところ、本技はインフォボックスに赤字で
「BDSPで使用不可」「SVで使用不可」と明記されており、カテゴリにも
「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」と分類されている
（第七・第八世代では使用可能だった技）。`src/jpoke/data/ps-champ-ja/moves.json`
（ポケモンチャンピオンズの技一覧）にも該当項目は存在しない。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。以降の項目はこの判断に基づく対応内容を記載する
（`docs/review/moves/オクタンほう.md` / `docs/review/moves/グラスミキサー.md` の前例と同じ扱い）。

さらに、削除前の実装を確認したところ `MoveData.flags` に `"contact"` が含まれていたが、一次情報の
「直接攻撃: ×」と矛盾する誤りだったことも判明した（対象範囲外と判断したため実装自体を削除し、
この誤りは修正ではなく削除で解消した）。

## 仕様書
- 新規作成（`docs/spec/moves/シャドーボーン.md`）。
- 対象範囲外である旨と根拠（第七・第八世代では使用可能だがBDSP/SVでは使用不可、
  ポケモンチャンピオンズにも項目なし）を明記した。
- 参考として第七・第八世代時点の技データ（タイプ:ゴースト、物理、威力85、命中100、PP10、
  追加効果20%でぼうぎょ1段階ダウン、直接攻撃なし）を記録した。
- 過去に誤って登録されていた `MoveData.flags` の `"contact"` が一次情報（直接攻撃×）と
  一致しないことも記録した。

    一次情報の追加効果と一致していた）

    `python scripts/generate_literals/generate_move_literal.py` を実行し、MOVES辞書と整合させた
  - 五十音ソート（`sort_handlers.py` / `sort_data/sort_moves.py`）はこのレビュー段階では実行しない
