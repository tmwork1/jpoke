## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ma.py`: `"ミルクのみ"` の `pp` を `5` → `8` に修正。
  `src/jpoke/data/ps-champ-ja/moves.json` にミルクのみ単独の行は存在しないが、同一効果技
  （`じこさいせい`・`タマゴうみ`・`なまける`・`はねやすめ`）のchampionsPPがいずれも8であるため、
  5（Gen9本家基準の値）は championsルールへの移行漏れバグと判断し修正した。
- `src/jpoke/data/moves/move_ma.py`: `target="self"` が未設定だったため追加。未設定のままだと
  `MoveData.target` のデフォルト値 `"foe"` になり、`is_blocked_by_protect`（まもる）・
  `マジックコート_reflect` の判定が `move.target == "foe"` を条件にしているため、自分対象の
  回復技にもかかわらず「まもるで防がれる」「マジックコートで跳ね返される」誤動作となっていた。
- `src/jpoke/handlers/move_status.py`: `ミルクのみ_self_heal` のHP満タン判定を
  `mon.hp >= mon.max_hp` → `mon.hp == mon.max_hp` に修正（HPが最大値を超えることはないため
  挙動は同じだが、`じこさいせい_heal_self` 等の他の同種ハンドラとの表記を統一した）。
