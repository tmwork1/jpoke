# りんしょう レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/りんしょう.md`）。
- 一次情報（ポケモンWiki）を基に、基本情報（ノーマル/特殊/威力60/命中100/PP16（チャンピオンズ基準）/対象1体/
  優先度0/直接攻撃なし/音技）と、同一ターン中の2体目使用で威力120になる効果、
  行動順繰り上げ効果（ダブル/トリプル・テラレイド用）を整理した。
- 本プロジェクトはシングルバトル専用のため、「本プロジェクトでの実装方針」節を設け、
  実装対象を威力上昇効果のみに絞り、行動順繰り上げ効果は通常の優先度・すばやさ順と
  常に一致するため対象外である旨を明記した。

## 実装計画書
- 新規作成（`docs/plan/moves/りんしょう.md`）。
- 既存実装が `handlers={}` のまま核心効果（同ターン2体目威力120）を欠落させていた
  バグである旨を明記。
- `Battle.round_used_turn`（りんしょうが最後に使われたターン番号）を新設し、
  `Event.ON_BEGIN_MOVE`（`attacker:self`, デフォルト優先度）で威力を直接書き換える方式を採用。
  `ON_CALC_POWER_MODIFIER` ではなく `ON_BEGIN_MOVE` を選んだ理由（無効化判定より前に
  記録する必要があるため）を明記した。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ra.py` の `MoveData` で `pp=15`（Gen9本家基準）になっていたのを
  `pp=16` に修正した。ポケモンチャンピオンズでは全技のPPが8/12/16/20の4段階に統一
  （`docs/champions/changes_from_sv.md`）されており、`src/jpoke/data/ps-champ-ja/moves.json:950`
  でも16となっている。
- `src/jpoke/core/battle.py`: `self.round_used_turn: int | None = None` を追加。
- `src/jpoke/handlers/move_attack.py`: `りんしょう_apply_chain_power` を新規実装
  （`りんごさん_lower_defender_spd` の直後、五十音順の位置に配置）。
  同じターン中に既に使われていれば `ctx.move.power = 120` にし、常に
  `battle.round_used_turn = battle.turn` を更新する。
- `src/jpoke/data/moves/move_ra.py`: `"りんしょう"` の `handlers` に
  `Event.ON_BEGIN_MOVE: h.MoveHandler(ha.りんしょう_apply_chain_power)` を追加
  （従来は `handlers={}` で核心効果が未実装だった）。
