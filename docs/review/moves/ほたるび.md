## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ha.py` の `"ほたるび"` MoveData に **`target="self"` が欠落していたバグを修正**。
  未設定だと `MoveData.target` のデフォルト `"foe"` が使われ、以下がすべて誤動作する状態だった：
  - `Move.is_blocked_by_protect`（`target == "foe"` を見るため、まもるで誤って防がれる）
  - `MoveExecutor.check_hit_substitute`（`target != "foe"` でみがわり判定するため、みがわりで誤って防がれる）
  - `Move.is_reflectable`（`target in ("foe", "foe_side")` を見るため、マジックコートで誤って跳ね返る可能性）
  - 相手を対象とする技のみに反応する特性ハンドラ（`handlers/ability.py` の `_apply_type_absorb` 等、
    `ctx.move.target == "foe"` を条件にするもの）が誤って反応する可能性
- `src/jpoke/handlers/move_status.py` の `ほたるび_modify_attacker_stats` に説明用 docstring を追加（挙動自体は変更なし）
