# ビックリヘッド レビュー結果

## 一次情報の確認結果（重要: 実装対象外と判定）

一次情報（`docs/wiki/moves/ビックリヘッド.html`）を確認した結果、ビックリヘッド（Mind Blown）は
**第七世代でズガドーン（Blacephalon）の専用技として登場したほのおタイプの特殊技**であり、
ウィキのページカテゴリに `ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ` /
`スカーレット・バイオレットで使用不可能なわざ` と明記されている通り、
**第九世代（スカーレット・バイオレット）では使用不可能**な技である。

README.md の対象範囲規定「第9世代で実装されていない技・アイテム・特性・ポケモンなどは
実装しない」に該当するため、本技は実装対象外と判定した。
`src/jpoke/data/ps-champ-ja/moves.json` にも該当行は存在しない
（作業指示にあった「ビックリヘッドは30%の確率で相手をひるませる技」という前提は、
一次情報のどの記述とも一致しない誤りであることも確認した）。

一次情報上の実データ（参考）:
- タイプ: ほのお
- 分類: 特殊
- 威力: 150
- 命中率: 100
- PP: 5
- 範囲: 自分以外の全員（スペック上は範囲技だが、本プロジェクトはシングル専用のため実質1体対象）
- 優先度: 0
- 直接攻撃: しない
- 効果: 攻撃前に自分の最大HPを1/2（切り上げ）減らしてから攻撃する。追加効果なし。
- 特性との相互作用: マジックガードで反動を防げるが、いしあたまでは防げない
  （てっていこうせんと同じ相互作用）
- 爆発する技の1つ（しめりけで不発）
- 判定: 急所○ / 命中判定○ / 追加効果× / まもる○ / おうじゃのしるし○（第五世代以降）/
  マジックコート× / よこどり× / オウムがえし○

既存実装は上記の一次情報と全く異なる（誤った）データ
（タイプ=ノーマル、分類=物理、PP=5、威力=130、命中率=100、`labels={"contact"}`）で登録されており、
「使用前に無条件で最大HPの1/2を消費してから相手1体を殴る物理技」として実装されていた。
これは一次情報のいずれの数値・分類とも一致せず、`src/jpoke/data/ps-champ-ja/moves.json` にも存在しないため、
第9世代の実在データを参照した形跡のない誤実装（存在しない技の捏造データ）と判断し、
実装・計画書・テストを削除した。

（参考: `docs/spec/moves/てっていこうせん.md`・`docs/spec/moves/クロロブラスト.md` は同じ
「自己HP半分消費」系統の技だが、いずれも一次情報上SVで使用可能であることを確認済みのため
対象外にはしていない。ビックリヘッドのみSV/BDSPで使用不可能な点が異なる。）
