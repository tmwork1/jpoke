# ダブルパンツァー レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/ダブルパンツァー.html` を確認したところ、本技はメルメタルの専用技であり、
ページカテゴリに「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」が明記されている（本文中の赤字警告表示でも
「BDSPで使用不可」「SVで使用不可」と重複して確認できる）。使用可能な世代は
「Let's Go! ピカチュウ・Let's Go! イーブイ・第八世代」（ソード・シールド）までである。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。さらに `docs/champions/move_list.txt`
（ポケモンチャンピオンズの全技一覧）にも「ダブルパンツァー」の項目が存在しないことを確認し、
ポケモンチャンピオンズ側でも採用されていないという二重の根拠を得た
（`ひっさつまえば` / `ふわふわフォール` / `まわしげり` と同様のケース）。

以降の項目はこの判断に基づく対応内容を記載する。

## 仕様書
- 新規作成（`docs/spec/moves/ダブルパンツァー.md`）。
- 対象範囲外である旨と根拠（メルメタルの専用技であり、BDSP・SVいずれにも使用できない）を明記した。
- 参考として第7・8世代時点の技データ（タイプ:はがね、物理、威力60、命中100、PP5、
  2回攻撃固定の連続攻撃技、1発ごとに30%でひるみ、パンチ技）を記録した。
- `docs/spec/moves/_multi_hit.md` / `docs/spec/moves/連続技.md`（第二次情報源）では対象世代が「7-」と
  記載されているが、これは技データテーブル上の存在を示すに留まり、実際の教え方が提供されていないことと
  矛盾しないことを注記した。

## 実装計画書

  「スペック要確認（PP・威力・命中）」のまま誤って登録されていた経緯を記録した。

  （威力: —・PP等）が誤った過去の登録内容であり削除済みであることを明記した。

## 実装（handlers/ / data/）

    2回攻撃固定の連続攻撃技であるにもかかわらず `multi_hit` 設定が丸ごと欠落しており、
    1回しか攻撃しない実装になっていた）。

    重複判定用集合）から `"ダブルパンツァー"` を削除（実体が存在しない技名の混入を解消）
  - `python scripts/generate_literals/generate_move_literal.py` を実行し、`MoveName` Literal 型を
    MOVES辞書と整合させた（726件）
