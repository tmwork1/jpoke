## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ma.py` の `"マッドショット"` エントリを修正:
  - `pp` を 15 → 16 に修正（`docs/champions/move_list.txt` 準拠。Championsは全技PPを8/12/16/20の
    4段階に統一しているため、Gen9本家基準の15のままだったのは移行漏れのバグ）。
  - `flags={"secondary_effect"}` を追加。未設定だと `ちからずく_boost`（威力1.3倍化）と
    `ちからずく_disable_secondary_effect`（すばやさ低下無効化）のいずれも発動しない状態だった
    （`がんせきふうじ` と同種の不具合）。
- `handlers/move_attack.py` の `マッドショット_lower_defender_spd` は仕様と一致しており修正不要
  （`modify_defender_stats` を使い `stats={"spe": -1}` を適用する既存実装のまま）。
