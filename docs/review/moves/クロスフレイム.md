# クロスフレイム レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/クロスフレイム.md`）。
- 一次情報（ポケモンWiki）に基づき、基本情報（ほのお/特殊/威力100/命中100/PP5/対象1体/
  優先度0/非接触）と効果（自分のこおり状態を無条件解凍してから攻撃、追加効果なし）を記載。
- クロスサンダーとの「デュアルクロス」（直前に成功した技がクロスサンダーの場合威力2倍）は、
  同ターン中に味方の別ポケモンがクロスサンダーを成功させていることが条件のダブルバトル専用効果
  であり、かつホワイトキュレム/ブラックキュレムは排他フォルムのため単体のポケモンが両技を
  同時に覚えることもなく、シングルバトル専用の本プロジェクトでは発動しえないことを明記し、
  実装対象外とした（`docs/spec/moves/いっちょうあがり.md` と同様の整理）。
- PPは `docs/champions/move_list.txt` に記載がないため、根拠のない値への書き換えを避け、
  第9世代基準の5をそのまま維持する旨を明記した。
