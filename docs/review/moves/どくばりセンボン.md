# どくばりセンボン レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/どくばりセンボン.md`）。
- 一次情報（`docs/wiki/moves/どくばりセンボン.html`）より、タイプ: どく、分類: 物理、
  威力: 60、命中: 100、PP: 10、対象: 相手1体、優先度: 0、直接攻撃ではない。
- 効果: 相手が『どく』『もうどく』状態のとき威力2倍（ベノムショックと同様の判定）。
  追加効果として50%の確率で相手をどく状態にする。追加効果判定はダメージ計算後に
  行われるため、本技自体でどく状態にした場合はそのターンの威力は2倍にならない。

## 実装計画書
- 新規作成（`docs/plan/moves/どくばりセンボン.md`）。
- 仕様書に基づき、`ON_CALC_POWER_MODIFIER`（威力2倍化）と `ON_DAMAGE_HIT`（どく付与）の
  2ハンドラ構成、priority根拠、ちからずくとの相互作用、リーサル対象外の理由を記載。

## 実装（handlers/ / data/）
- **欠落を発見・修正**: `src/jpoke/data/move.py` の `どくばりセンボン` エントリには
  どく付与ハンドラ（`ON_DAMAGE_HIT`）のみが登録されており、仕様にある
  「相手が『どく』『もうどく』状態のとき威力2倍」の効果が未実装だった。
- `src/jpoke/handlers/move_attack.py` に `どくばりセンボン_double_power_when_poisoned`
  を新規実装（`ベノムショック_double_power_when_poisoned` と同一ロジック、
  `apply_fixed_modifier(value, 8192)` で威力2倍を表現）。
- `src/jpoke/data/move.py` の `どくばりセンボン` エントリに
  `Event.ON_CALC_POWER_MODIFIER: h.MoveHandler(ha.どくばりセンボン_double_power_when_poisoned)`
  を追加。
- `scripts/sort_handlers.py src/jpoke/handlers/move_attack.py` と
  `scripts/sort_data/sort_moves.py` を実行して並び替えを反映。

## リーサル計算
- 対象外（`docs/progress/move.md` の「リーサル実装」列は `n/a`）。
  どく付与はターン経過依存の追加効果でありリーサル対象外。威力2倍化も
  ベノムショック同様の既存方針に合わせて対象外とした。
