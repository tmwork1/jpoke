# ひかりのかべ レビュー結果

## 仕様書

- ウォール（壁）設置技のため、既存の慣例（`リフレクター`等と同様）に合わせて技固有の仕様は
  `docs/spec/fields/ひかりのかべ.md`（既存・場の状態としての仕様書）に一本化した
  - `docs/spec/moves/` に技単体の仕様書を新規作成する案も検討したが、`リフレクター`・`おいかぜ`等
    同種の場設置技はいずれも `docs/spec/moves/` に個別ファイルを持たず `docs/spec/fields/` のみで
    管理されているため、その慣例に合わせて `docs/spec/moves/ひかりのかべ.md` は作成しなかった
  - `docs/spec/fields/ひかりのかべ.md` の内容を一次情報（wiki）と照合した結果、シングルバトルでの
    1/2軽減・急所無効・すりぬけ貫通・ひかりのねんどによる8ターン延長・オーロラベールとの非重複など
    すべて一致しており、修正は不要だった

## 実装計画書

- `docs/plan/moves/ひかりのかべ.md`（未実装前提の古い計画書で、技側ハンドラを「未実装」と誤記していた）
  を削除した。実際には技・フィールド両方のハンドラがすでに実装済みであり、正しい実装計画は
  `docs/plan/fields/ひかりのかべ.md` に既に存在していたため、`moves/` 側は重複かつ内容が古く誤りだった
  （リフレクターも `plan/fields/` のみを持ち `plan/moves/` は存在しない慣例と揃えた）
- `docs/plan/fields/ひかりのかべ.md` 内のターン終了ハンドラ名を実装と一致するよう修正
  （誤: `ひかりのかべ_tick_side_field` / `tick_side_field(...)` → 正: `ひかりのかべ_tick` / `_tick_side_field(...)`）
