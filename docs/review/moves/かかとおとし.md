    （いしあたま／すてみ／ちからずく）・とびひざげりとの違いをまとめた。
  - 本技のこんらん効果は **3〜5ターン継続**（通常のこんらん付与技は2〜5ターン）という固有仕様であることを明記。
  - まもる・タイプ無効化時にも失敗反動を受ける正式仕様を記載しつつ、現状の実装はON_MISS（命中判定失敗）のみに

## 実装（handlers/ / data/）
- `src/jpoke/data/move.py`: かかとおとしの `flags` に `"recoil"` を追加（`{"contact", "recoil", "secondary_effect"}`）。
  - すてみ特性は `ctx.move.has_flag("recoil")` を判定条件にしており（`handlers/ability.py` の `すてみ_boost_power`）、
    このフラグが無いと wiki記載の「すてみ所持時に威力1.2倍」が発動しなかった。
  - `recoil` フラグは いしあたま が参照する `HPChangeReason`（`ctx.hp_change_reason == "recoil"`）とは別軸のため、
    追加してもいしあたまによる反動無効化には影響しない（本技の反動は `reason="self_cost"` で実装済み）。
- `src/jpoke/handlers/move_attack.py`: `かかとおとし_apply_confusion` を修正。
  - 従来は汎用ヘルパー `apply_confusion_to_defender`（2〜5ターン固定）を使用していたが、
    本技固有の3〜5ターン仕様に合わせて `battle.volatile_manager.apply(..., count=battle.random.randint(3, 5))`
    を直接呼ぶ専用実装に変更した（`resolve_secondary_chance` によるちからずく無効化の挙動は維持）。
- `python scripts/sort_handlers.py src/jpoke/handlers/move_attack.py` と
  `python scripts/sort_data/sort_moves.py` を実行し、五十音順を維持した。
