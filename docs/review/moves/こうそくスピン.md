# こうそくスピン レビュー結果

## 仕様書

- 既存の `docs/spec/moves/こうそくスピン.md` に以下を追記・修正した。
  - **やどりぎのタネ解除**が命中時の共通効果から欠落していたため追加した
    （一次情報 `docs/wiki/moves/こうそくスピン.html` の Champions 説明文
    「自分にかかっているバインド やどりぎのタネ状態と…解除する」より）。
  - 命中時の解除効果は `Event.ON_HIT`（priority=100）で発動すること、
    `Event.ON_DAMAGE_HIT` を採用しない理由（みがわり被弾時・実ダメージ0のとき発火しない）を明記した。
  - りんぷん・おんみつマントの影響を受けない一方、使用者がちからずくの場合は
    （威力1.3倍化と引き換えに）解除・素早さ上昇のいずれも発動しないという
    第9世代の仕様を明記した（一次情報の「技の仕様」節より）。
  - キラースピンの追加効果（どく付与）の確率を誤記の「20%」から「100%」に修正した。

- `docs/plan/moves/こうそくスピン.md` が存在しなかったため新規作成した。

  扱い・キラースピンとの実装共有）を記載した。

## 実装（handlers/ / data/）

- `src/jpoke/handlers/move_attack.py` の `_clear_spin_effects`（こうそくスピン・キラースピン共通）に
  **やどりぎのタネの解除が欠落していた**バグを修正した（`battle.volatile_manager.remove(ctx.attacker, "やどりぎのタネ")` を追加）。
- 同関数と `こうそくスピン_boost_attacker_spe` が `modify_attacker_stats`（`resolve_secondary_chance` 経由）を
  使っていたため、相手が **おんみつマント／りんぷん** を持つと解除・素早さ上昇が誤って無効化される
  バグを修正した。`_is_sheer_force_blocked` ヘルパーを追加し、`ctx.attacker.ability.name == "ちからずく"`
  を直接判定することで、りんぷん・おんみつマントの影響を受けず、ちからずくの場合のみ発動しない
  仕様（第9世代）に合わせた（`だっしゅつボタン_reserve_switch` 等の既存パターンを踏襲）。
- `python scripts/sort_handlers.py src/jpoke/handlers/move_attack.py` を実行済み。
- `data/move.py` の変更なし（PP=40 は元々正しかった）。
