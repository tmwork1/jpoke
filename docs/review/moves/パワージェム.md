# パワージェム レビュー結果

## 仕様書
- 対象外（新規作成しなかった）。パワージェムはタイプ・分類・威力・命中率・PP以外に固有効果を持たない
  純粋なダメージ技であり、他の同種の技（例: 10まんばりき）と同様に仕様書・計画書を作成しない方針
  に合わせた。一次情報（docs/wiki/moves/パワージェム.html）で以下を確認済み：
  - タイプ: いわ / 分類: 特殊 / 威力: 80（第六世代以降。第五世代までは70。当プロジェクトは最新仕様
    準拠のため80が正しい） / 命中率: 100 / PP: 20（本家基準） / 直接攻撃: なし（非接触）/ 追加効果なし
