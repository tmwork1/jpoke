## 実装（handlers/ / data/）
- `src/jpoke/handlers/move_status.py` の `ハバネロエキス_apply` を修正。
  - 修正前: `battle.modify_stats` を atk・def それぞれ個別に2回呼んでおり、
    (1) 両方とも境界（こうげき+6かつぼうぎょ-6）で変化不可の場合でも技が失敗として扱われない
    （常に `HandlerReturn(value=value)` で元の `True` を返すため）、
    (2) しろいハーブ等の「ランク変化を1回のイベントとして検知する」効果と噛み合わない、
    という2つの問題があった。
  - 修正後: 既存の `modify_defender_stats` ユーティリティ（`おたけび`・`おきみやげ`等と同じ
    パターン）を使い、`{"atk": 2, "def": -2}` を1回の `modify_stats` 呼び出しにまとめた。
    これにより両方とも変化不可の場合に戻り値が空dict（偽値）となり、
    `move_executor._execute_status_hit` が技全体を失敗と正しく判定するようになった。
- `src/jpoke/data/moves/move_ha.py` の `"ハバネロエキス"` エントリを修正。
  - PP を `15`→`16` に修正（`docs/champions/move_list.txt` の値に合わせた。Gen9本家値からの
    移行漏れバグ）。
  - `accuracy=100` を削除。ハバネロエキスは必中技であり、`accuracy` 省略時（`None`）に
    命中判定がバイパスされる実装（`あくび`等と同じパターン）に合わせた。
