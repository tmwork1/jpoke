# ニードルアーム レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/ニードルアーム.html` を確認したところ、本技はカテゴリ
「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」に分類されており、本文中の技データ表にも
赤字で「ソード・シールドで使用不可」「BDSPで使用不可」「SVで使用不可」と明記されている
（第八世代の説明文にも「このわざは つかえません」と明記）。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当し、`src/jpoke/data/ps-champ-ja/moves.json`（ポケモンチャンピオンズの技一覧、PPの正とすべき一次情報）にも
項目が存在しないことを確認した。よって **本技は実装対象外と判断した**。以降の項目はこの判断に基づく
対応内容を記載する。

なお、この判断は同計画書（`docs/plan/moves/攻撃技_ひるみ系.md`）に基づき過去に実装され、既に対象範囲外と
判定・削除済みの「ひっさつまえば」「ふわふわフォール」「まわしげり」「ダブルパンツァー」と同様の経緯である。
