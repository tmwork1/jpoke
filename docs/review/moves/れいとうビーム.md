  でも12となっている。
- `src/jpoke/handlers/move_attack.py` の `れいとうビーム_apply_freeze_to_defender` は
  共通ヘルパー `apply_ailment_to_defender(..., ailment="こおり", chance=0.1)` を正しく利用しており誤りなし。
  type="こおり", category="special", power=90, accuracy=100, flags={"secondary_effect"},
  Event.ON_DAMAGE_HIT は仕様と一致。
