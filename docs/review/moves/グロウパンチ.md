  効果（『こうげき』+1）は、他の同種の自分ランク上昇技（あやしいかぜ、
  げんしのちから、くさわけ 等）と同様に `docs/spec/moves/_move_secondary_1.md`
  の一覧表（63行目、`グロウパンチ attacker 1 1 0 0 0 0 ...`）で管理する方針
  であり、一次情報（Wiki）と照合した結果、効果自体は正しく記載されていた。
  - タイプ: かくとう、分類: 物理、威力: 40、命中率: 100、PP: 20（第六〜第八
    世代時点の公式値）、直接攻撃: ○、追加効果: 100%の確率で自分の
    『こうげき』ランクを1段階上げる（一次情報のインフォボックスと一致）。

## 実装（handlers/ / data/）

- **不備を発見し修正した**: `src/jpoke/data/moves/move_ka.py` の
  `"グロウパンチ"` の `pp` が `40` になっていたが、一次情報（Wiki）による
  公式PPは `20` であり、`src/jpoke/data/ps-champ-ja/moves.json` にも本技の記載が
  ない（チャンピオンズ固有の上書き値が存在しない）ため、Wiki記載の`20`を
  正として `pp=20` に修正した。`40` という値を裏付ける情報源はなく、
  移行時の誤記と判断した。
- `type="かくとう"`、`category="physical"`、`power=40`、`accuracy=100`は
  一次情報と一致しており妥当。変更なし。
- `flags={"contact", "punch", "secondary_effect"}` は一次情報の
  「直接攻撃: ○」「パンチ技」「追加効果あり（ちからずくで威力1.3倍・
  ランク上昇消失）」といずれも一致しており妥当。変更なし。
- `src/jpoke/handlers/move_attack.py` の
  `グロウパンチ_boost_attacker_A`（`modify_attacker_stats(..., stats={"atk": 1})`）
  は、他の同系統ハンドラ（`コメットパンチ_boost_attacker_A` 等）と同じ
  共通実装パターンで、`Event.ON_HIT` で命中後にのみ発火するため
  「相性などに技が無効化されたときや、命中しなかったときは攻撃は上がらない」
  という一次情報の記述とも整合している。変更なし。
- てつのこぶし（punchフラグで威力1.2倍）・ちからずく（secondary_effectフラグで
  威力1.3倍・ランク上昇無効化）は既存の特性側ハンドラで汎用的に処理される
  仕組みであり、グロウパンチ固有の対応は不要。変更なし。
