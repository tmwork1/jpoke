## 実装（handlers/ / data/）
- `src/jpoke/data/move.py` の `"かえんだん"` エントリに誤りがあったため修正した:
  - `pp`: 15 → 5（一次情報のGen5基本値と不一致だった）
  - `power`: 75 → 100（同上）
  - `flags`: `{"secondary_effect"}` → `{"bullet", "secondary_effect"}`
    （弾のわざであるにもかかわらず `bullet` フラグが欠落しており、特性ぼうだんで
    無効化されない不具合があった）
  - `handlers/move_attack.py` の `かえんだん_apply_burn_to_defender`（30%やけど付与）は
    仕様と一致しており変更なし。
  - `data/move.py` 修正に伴い `python scripts/sort_data/sort_moves.py` を実行済み。
