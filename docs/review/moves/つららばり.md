# つららばり レビュー結果

## 仕様書
- 変更なし。つららばりは追加効果を持たない汎用の2～5回連続攻撃技であり、
  他の同種技（ミサイルばり・ロックブラスト・タネマシンガン・みだれづき等）と同様に
  `docs/spec/moves/_multi_hit.md`（連続攻撃技の共通仕様）でカバーされる対象であるため、
  技専用の仕様書ファイルは作成しない（既存の同種技も個別ファイルなし、という慣例と一致）。

## 実装計画書
- 変更なし。上記と同じ理由で、つららばり専用の実装計画書は不要（既存の同種技にも計画書は存在しない）。

## 実装（handlers/ / data/）
- 変更なし。`src/jpoke/data/move.py` の `つららばり` エントリ（type=こおり, category=physical,
  pp=30, power=25, accuracy=100, multi_hit={min:2, max:5, check_hit_each_time:False,
  power_sequence:()}, handlers={}）は一次情報（PokemonWiki）および `_multi_hit.md` の
  「2-5回攻撃技」区分と一致しており、追加効果もないため修正不要と判断した。

## リーサル計算
- 対象外。`docs/progress/move.md` のリーサル実装列が `n/a` であり、同種の連続攻撃技
  （ミサイルばり・ロックブラスト等）にも技固有のリーサルハンドラは存在しない。

## テスト
- 変更なし。`tests/moves_attack/test_move_ta.py` の
  `test_つららばり_複数ヒットする` が既存の同種技のテスト
  （`test_ミサイルばり_複数ヒットする` 等）と同一パターンでヒット回数(2~5)を検証しており、
  過不足なし。

## 進捗ファイルの修正
- `docs/progress/move.md` の つららばり 行に誤りがあったため修正した。
  - PP列: `20` → `30`（一次情報・実装ともに PP=30）
  - レビュー列: `-` → `✓`
