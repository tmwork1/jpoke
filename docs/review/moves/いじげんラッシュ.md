## 実装（handlers/ / data/）

以下の誤りを修正した。

- `data/move.py`: `accuracy=100` → `accuracy=None`（必中技のため。回避ランクの影響を受けてしまう誤りだった）。
- `data/move.py`: `flags={"secondary_effect"}` → `flags={"unprotectable", "bypass_substitute"}`
  - `unprotectable`・`bypass_substitute` が欠落しており、まもる・みがわりに通常通り防がれてしまう誤りだった。
  - `secondary_effect` は誤りで、自分自身のランクダウンはちからずく（Sheer Force）の対象外
    （Bulbapedia記載）にもかかわらず、このフラグがあるとちからずく所持者使用時に威力が誤って1.3倍され、
    防御低下が無効化されてしまう不具合があった。
- `data/move.py`・`handlers/move_attack.py`: まもる系揮発性状態を解除する処理が欠落していたため、
  `いじげんラッシュ_remove_protect`（`いじげんホール_remove_protect`と同一パターン、
  `_FAINT_REMOVE_VOLATILES` を使用）を新規実装し、ON_HITハンドラに追加登録した。
  （`unprotectable` フラグはダメージが防がれないことのみを保証し、まもる状態自体の解除は別処理のため）
- `python scripts/sort_handlers.py src/jpoke/handlers/move_attack.py` 実行済み。
- `python scripts/sort_data/sort_moves.py` 実行済み（並び順に変化なし）。
