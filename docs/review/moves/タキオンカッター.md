## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ta.py` の `MOVES_TA["タキオンカッター"]` を修正した。
  - `accuracy=100` → `accuracy=None`（必中）に修正。100%命中の技として扱われていたため、
    命中ランク補正・回避率上昇の影響を受けてしまう誤りだった。
  - `multi_hit={"min": 2, "max": 2, "check_hit_each_time": False, "power_sequence": ()}` を追加。
    2回連続攻撃（連続攻撃技）が未実装で、通常の1回攻撃技として動作していた欠落を修正した。
  - `flags={"slash"}` を追加。切る技（特性『きれあじ』の対象）である旨が未反映だった欠落を修正した。
  - 固有ハンドラの追加・修正なし（`handlers={}` のまま。上記はすべて `core/move_executor.py` の
    汎用機構で処理される）。
