  `src/jpoke/data/ps-champ-ja/moves.json` ではヘドロばくだんのPPは12（champions全技PP4段階統一
  8/12/16/20 のひとつ）であり、実装がGen9本家基準のPP10のままになっていた移行漏れバグ。
  それ以外（type="どく", category="special", power=90, accuracy=100,
  flags={"bullet", "secondary_effect"}, `Event.ON_DAMAGE_HIT` での30%どく付与）は
  仕様書と一致しており修正不要だった。
- `src/jpoke/handlers/move_attack.py`: 変更なし。
  `apply_ailment_to_defender(battle, ctx, value, ailment="どく", chance=0.3)` は
  共通ヘルパー（`handlers/move.py`）を使用しており、確率解決・どく/はがねタイプ無効化・
  めんえき無効化はすべて `battle.resolve_secondary_chance` / `battle.ailment_manager.apply`
  内部で処理されるため、本技固有の重複ガードは不要かつ存在しない。
