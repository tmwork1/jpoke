# からをやぶる レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/からをやぶる.md`）。
  - 一次情報（docs/wiki/moves/からをやぶる.html）の「技の仕様」節に基づき、以下を明記した。
    - 能力低下は使用者自身が引き起こしたものとして扱われるため、しろいきり／クリアボディ等の
      能力低下ブロック効果や、使用者自身のまけんき／かちきの影響を受けないこと
    - あまのじゃく・たんじゅん適用時のランク変化量
    - ぼうぎょ・とくぼうが最低（あるいはこうげき・とくこう・すばやさが最高）でも、
      片方だけは通常通り変化し、両方の条件が重なった場合のみ技自体は失敗しないままランク変化なしになること
    - しろいハーブ／だっしゅつパック、相手のびんじょう／ものまねハーブが技の完了後に発動すること
  - PP は champions 基準の 16（`docs/champions/move_list.txt`）を明記した。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ka.py` の `からをやぶる` エントリの `pp` を 15 → 16 に修正した
  （champions 基準の PP 値。Gen9 本家基準の値が移行漏れとして残っていた）。
- `src/jpoke/handlers/move_status.py` の `からをやぶる_modify_attacker_stats` はランク変化量
  （B-1, D-1, A+2, C+2, S+2）・`subject_spec`・event priority とも仕様書・計画書と一致しており、
  修正なし。
- `modify_attacker_stats` は `source=ctx.attacker` を渡しているため、`_block_stat_drop_by_foe` /
  まけんき等の「相手由来か」判定（`ctx.source != ctx.target`）により、自己発生の能力低下として
  正しく特性・場の効果をバイパスすることを確認した（既存の汎用ロジックで担保されており、
  修正不要）。
