## 実装（handlers/ / data/）
- `src/jpoke/data/move.py` の `"だいもんじ"` エントリを修正:
  - `flags={"secondary_effect"}` → `flags={"secondary_effect", "thaw"}` に変更。
    だいもんじはほのおタイプの攻撃技であり、被弾した相手がこおり状態なら解凍するべきだが
    （`docs/spec/ailments/こおり.md` 31-32行目）、この処理を担う `こおり_cure_by_thaw_move`
    （`src/jpoke/handlers/ailment.py`, `Event.ON_DAMAGE_HIT`）は `ctx.move.has_flag("thaw")`
    で判定するため、フラグが欠落していると解凍処理が発動しない不具合があった。
    既存の同種特殊ほのお技 `かえんほうしゃ` は `"thaw"` を持っており、だいもんじも同様に
    修正する必要があると判断した（使用者自身の無条件解凍は対象外のリストのため追加しない）。
  - `pp=5, power=110, accuracy=85, category="special"` および
    `だいもんじ_apply_burn_to_defender`（10%やけど付与）は仕様通りで変更なし。
  - `docs/spec/moves/_move_labels.md` の `thaw` ラベル一覧にだいもんじを五十音順で追加。
