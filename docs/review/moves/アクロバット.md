## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_a.py` の `"アクロバット"` エントリを修正した。
  - `pp` を `15`（本家基準）から `16`（`docs/champions/move_list.txt` の値）に修正した。
  - `Event.ON_CALC_POWER_MODIFIER` に `アクロバット_double_power_when_no_item` を登録し、
    「道具を持っていないとき威力2倍」の効果を実装した（未実装だった）。
- `src/jpoke/handlers/move_attack.py` に `アクロバット_double_power_when_no_item` を追加した。
  - `ctx.attacker.has_item()`（デフォルト `consider_enabled=False`）が `False` のときのみ
    `apply_fixed_modifier(value, 8192)` で威力を2倍にする。ぶきよう・さしおさえ・マジックルームで
    道具の効果が無効化されているだけの場合は `has_item()` が `True` を返すため、威力2倍にならない
    （一次情報の「技の仕様」節の記載と一致）。
- 既知の課題（今回のタスクスコープ外として対応せず）:
  - `テクニシャン_boost_power`（`src/jpoke/handlers/ability.py`）は `ctx.move.data.power`
    （宣言時の固定威力55）で判定しており、アクロバットの道具有無による威力変動を考慮していない。
    `docs/spec/abilities/テクニシャン.md`（8行目）が明記する「変動後の威力で判定する」仕様とは
    厳密には一致せず、道具を持っていない場合（威力110）でもテクニシャンの1.5倍が適用され得る。
    これはアクロバット固有の問題ではなく、めざめるパワー・かぜおこし・おいうちなど技自身の条件で
    威力が変動する技全般に共通するテクニシャン側の実装課題のため、本タスクでは修正せず
    `docs/plan/moves/アクロバット.md` に記録するに留めた。
