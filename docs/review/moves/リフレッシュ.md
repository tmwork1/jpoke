# リフレッシュ レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/リフレッシュ.html` を確認したところ、本技は「ソード・シールド」
「BDSP（ブリリアントダイヤモンド・シャイニングパール）」「スカーレット・バイオレット」
のいずれでも**使用不可能なわざ**に分類されている（技情報ボックスに「ソード・シールドで
使用不可」「BDSPで使用不可」「SVで使用不可」と明記され、`wgCategories` にも
「ソード・シールドで使用不可能なわざ」等が含まれる）。第八世代以降、本技を習得できる
ポケモンが存在しないため実質的に使用不可能であり、第9世代でも復活していない。
`src/jpoke/data/ps-champ-ja/moves.json` にも「リフレッシュ」の項目が存在しないことを確認し、
ポケモンチャンピオンズ側でも採用されていないという二重の根拠を得た。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは
実装しない」）に該当するため、**本技は実装対象外と判断した**
（前例: `docs/review/moves/オクタンほう.md`, `docs/review/moves/シグナルビーム.md`,
`docs/review/moves/ふわふわフォール.md` 等と同じ判断基準）。

さらに、対象範囲外の判断とは別に、削除前の既存実装には**回復対象の誤り**があったことも
判明した。旧実装（`リフレッシュ_cure_ailment`）は `battle.ailment_manager.remove(mon)` を
無条件に呼び出しており、現在の状態異常が何であれ回復させていた。しかし一次情報によれば
リフレッシュが回復するのは「どく・もうどく・まひ・やけど」の3系統のみであり、「こおり」
「ねむり」は対象外（第三世代〜第七世代のいずれの説明文でも「どく まひ やけどの 状態異常」
としか記載されていない）。したがって旧実装は本来「こおり」「ねむり」中に使用しても
回復させてしまう誤りを含んでいた。今回は対象範囲外の判断により修正ではなく削除で対応した。

以降の項目はこの判断に基づく対応内容を記載する。
