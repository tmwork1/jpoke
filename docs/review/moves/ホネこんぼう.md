# ホネこんぼう レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/ホネこんぼう.html` を確認したところ、本技はカテゴリ
「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」に分類されており、本文中の技データ表にも
「ソード・シールドで使用不可」「BDSPで使用不可」「SVで使用不可」と明記されている。
第八世代の説明文も「このわざは つかえません」となっている。`docs/champions/move_list.txt`
（ポケモンチャンピオンズの技一覧）にも本技のエントリは存在しない。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。以降の項目はこの判断に基づく対応内容を記載する
（本タスクの依頼文には「ホネこんぼうは2回連続で攻撃する技です」とあったが、これは誤りである。
2回以上連続で攻撃する「ボーンラッシュ」との混同と思われる。実際のホネこんぼうは単発攻撃技で、
10%の確率で相手をひるませる追加効果を持つのみである）。

なお、削除前の既存実装を確認したところ `MoveData(power=100, pp=10, flags={"contact", "secondary_effect"})`
という値が登録されていたが、実際のゲームでの本技の基礎値（第1〜7世代時点）は威力65・命中85・PP20・
非接触技であり、この点でも誤りがあった（対象範囲外である以前に、仮に実装対象だったとしても値が誤っていた）。
