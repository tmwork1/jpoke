## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_sa.py`（サンダーダイブ登録箇所）
  - PP を `15`（Gen9本家値、champions移行漏れのバグ）→ `16`（`src/jpoke/data/ps-champ-ja/moves.json` の値）に修正。
  - `flags` に `"recoil"` を追加（`{"minimize", "contact"}` → `{"minimize", "contact", "recoil"}`）。
    すてみ特性の威力1.2倍補正はこのフラグを判定条件にしているため、欠落していると発動しなかった
    （とびげり・とびひざげりレビューで判明した同種のバグパターン）。
- `src/jpoke/handlers/move_attack.py`（`サンダーダイブ_crash`）
  - 失敗反動ダメージの `reason` を `"self_cost"` → `"fixed_recoil"` に修正。
    `"self_cost"` は `マジックガード_ignore_damage`（`handlers/ability.py`）の除外リストに含まれており、
    マジックガードで防げない扱いになっていたバグ。wiki記載（マジックガード所持時は反動を受けない）と
    `docs/plan/moves/失敗反動技.md` 注記2の既知パターンに基づき修正。`fixed_recoil` は
    `reason == "recoil"` のみを判定するいしあたまでは引き続き防げない（仕様通り）。
- `docs/plan/moves/失敗反動技.md`（共通計画書）
  - サンダーダイブの行を「修正済み」状態に更新し、注記2の記述を「残る1技（かかとおとし）」に更新。
