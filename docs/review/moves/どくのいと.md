# どくのいと レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/どくどく.md` の形式に準拠）
- 一次情報（`docs/wiki/moves/どくのいと.html`）で判明した重要事項を明記:
  - **Championsではすばやさ低下量が1段階→2段階に変更されている**（`docs/champions/changes_from_sv.md`・
    `docs/champions/move_list.txt` とも整合。実装は従来1段階のままだった重大なバグ）
  - すばやさ低下とどく付与は独立効果で、すばやさ低下が先に発動しどく付与が後に発動する
  - どちらの効果も失敗した場合のみ技全体が失敗する
  - どく・はがねタイプには「どく」付与のみ無効（すばやさ低下effectは通常通り発動）
  - みがわり状態には無効（特性『すりぬけ』所持者は無視可能）

  - すばやさ低下量を `-1` → `-2` に修正（Championsでの仕様変更に追従していなかったバグ）
  - 効果の発動順序を「どく付与→すばやさ低下」から「すばやさ低下→どく付与」に修正（一次情報の
    「すばやさランクを下げる効果の後にどく状態にする効果が発動する」という記述に合わせた）

    （一次情報の「どちらの効果も失敗した場合のみ技全体が失敗する」という記述に合わせた）

  一次情報・`docs/champions/move_list.txt` と一致しており変更なし
