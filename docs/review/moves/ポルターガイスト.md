## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ha.py` の `"ポルターガイスト"` の `pp` を `5`（Gen9本家値）から
  `8`（`src/jpoke/data/ps-champ-ja/moves.json` 準拠）に修正した。champions側は全技PPが8/12/16/20の
  4段階に統一されているため、5という値はGen9基準からの移行漏れバグと判断した。
- `handlers/move_attack.py` の `ポルターガイスト_check_item` はロジック自体（`ctx.defender.has_item()`
  による所持チェック、失敗時のログ出力、`stop_event=True`）に問題なし。変更なし。
