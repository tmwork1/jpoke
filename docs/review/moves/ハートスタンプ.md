# ハートスタンプ レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/ハートスタンプ.html` を確認したところ、本技はカテゴリ
「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」に分類されており、第八世代の説明文も
「このわざは つかえません おもいだすことが できなくなりますが わざを わすれることを おすすめします」
となっている。`src/jpoke/data/ps-champ-ja/moves.json` にも本技のエントリは存在しない
（`ヒートスタンプ` との混同がないことは個別に確認済み）。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。以降の項目はこの判断に基づく対応内容を記載する。

なお、削除前の既存実装のスペック自体（タイプ:エスパー、物理、威力60、命中100、PP25、追加効果30%ひるみ）は
一次情報と一致しており値自体に誤りはなかったが、対象範囲外である以上いずれにせよ削除が必要だった。

  - `src/jpoke/handlers/move_attack.py`: `ハートスタンプ_apply_flinch` ハンドラを削除

    重複判定用リスト）から `"ハートスタンプ"` を削除（実体が存在しない技名の混入を解消）

    MOVES辞書（721件）と整合させた

    保持する参照ファイルであり、`まわしげり` 等の既存の対象範囲外技についても記載が残されたままになっている
