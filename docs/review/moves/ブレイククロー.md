# ブレイククロー レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/ブレイククロー.md`）。
  - 一次情報（`docs/wiki/moves/ブレイククロー.html`）・`docs/spec/moves/_move_secondary_1.md`・
    `docs/champions/move_list.txt`・`docs/champions/changes_from_sv.md` を参照し、
    タイプ・分類・威力・命中・PP・追加効果確率（50%・ぼうぎょ-1）・切る技追加を記載した。
  - なお指示文には「30%の確率」と記載されていたが、一次情報（Wiki本文・Champions説明文とも
    「50%の確率で相手の防御を1段階下げる」）および `docs/spec/moves/_move_secondary_1.md` の
    既存データ（`ブレイククロー defender 0.5 0 -1 ...`）と照合した結果、正しい確率は **50%** であることを
    確認した。30%はおそらく同じ「切る技追加」対象のフェイタルクロー（Championsで状態異常付与確率が
    50%→30%に変更）と混同したものと考えられる。仕様書・実装は一次情報に基づき50%のまま維持した。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ha.py` の `MOVES["ブレイククロー"]` を修正した。
  - `pp=10` → `pp=12`（`docs/champions/move_list.txt` 準拠。本家基準PP10からの移行漏れバグ）。
  - `flags={"contact"}` → `flags={"contact", "secondary_effect", "slash"}`。
    - `secondary_effect` 欠落により、特性ちからずくとの相互作用（威力1.3倍・追加効果無効化）が
      発動しない不具合があった（`handlers/ability.py` の `ちからずく_boost` /
      `ちからずく_disable_secondary_effect` は `ctx.move.has_flag("secondary_effect")` で判定するため）。
    - `slash` 欠落により、Championsで新たに「切る技」に追加された効果（特性きれあじで威力1.5倍）が
      発動しない不具合があった（`docs/champions/changes_from_sv.md` に
      「ブレイククロー 切る技(切り技)に追加」と明記。同様に追加されたドラゴンクロー・シャドークロー・
      フェイタルクローには既に `slash` フラグが付与済みで、ブレイククローだけ移行漏れだった）。
  - `handlers/move_attack.py` の `ブレイククロー_lower_defender_def`（`modify_defender_stats` 経由で
    `chance=0.5` ・ `stats={"def": -1}`）は仕様通りで変更なし。
