# コーチング レビュー結果

## 対象範囲について（重要）

一次情報（`docs/wiki/moves/コーチング.html`）の「技の仕様」節に
「シングルバトルのときや味方がいないときに使うと失敗する。」と明記されており、
「備考」節にも「自身は効果対象外であるため、ダブルバトル以上の場合の専用技と言える。」とある。
効果も「自分以外の味方全員」の『こうげき』『ぼうぎょ』ランクを1段階ずつ上げるというもので、
シングルバトルでは効果対象が存在しないため必ず失敗する。

README.md の対象範囲定義「対象はポケモンチャンピオンズのシングルバトルのみ（ダブルバトル等は対象外）」
に基づき、本技は**実装対象外**と判断した。`docs/progress/move.md` では既に本技の全列
（仕様書・計画書・実装・テスト・リーサル実装・リーサルテスト・レビュー）が「ダブル専用」として
正しく記載されている。同様の扱いを受けている先例（`てだすけ`, `おさきにどうぞ`, `いかりのこな`,
`アロマミスト` など14技）と同様、`docs/spec/moves/`・`docs/plan/moves/`・`src/jpoke/data/move.py`・
`src/jpoke/handlers/` のいずれにもエントリを作成しない方針を維持する。

PP値についても `docs/champions/move_list.txt`（12）と `docs/progress/move.md` の記載（12）が
一致しており、修正は不要だった。
