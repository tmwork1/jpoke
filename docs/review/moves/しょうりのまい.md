## 実装（handlers/ / data/）
- **バグ修正**: `src/jpoke/data/moves/move_sa.py` の `しょうりのまい` エントリが
  `target="self"` を指定しておらず、かつ `handlers={}`（「追加効果なし」コメント付き）と
  なっていた。しょうりのまいの効果（こうげき・ぼうぎょ・すばやさを1段階ずつ上げる）は
  技の主効果であり「追加効果なし」は誤り。`MoveData.target` のデフォルトは `"foe"` のため、
  修正前は相手を対象とする技として扱われてしまう不具合があった。
  - `target="self"` を追加
  - `Event.ON_STATUS_HIT` に `しょうりのまい_modify_attacker_stats` ハンドラを登録
- `src/jpoke/handlers/move_status.py` に `しょうりのまい_modify_attacker_stats` を新規実装
  （`しびれごな_apply_ailment_to_defender` と `しんぴのまもり_set_side_field` の間、五十音順の
  暫定位置。最終整形は `sort_handlers.py` に委ねる）。既存の `modify_attacker_stats`
  ヘルパー（`stats: dict[Stat, int]` 対応）を利用し、`{"atk": 1, "def": 1, "spe": 1}` を渡す
  （`からをやぶる_modify_attacker_stats` と同様の複数ステータス同時変化パターン）。
