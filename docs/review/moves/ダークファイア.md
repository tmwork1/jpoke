# ダークファイア レビュー結果

## 一次情報の確認結果（重要: 実装対象外と判定）

一次情報（`docs/wiki/moves/ダークファイア.html`）を確認した結果、ダークファイアは
**第三世代『ポケモンXD 闇の旋風ダーク・ルギア』にのみ登場するダークタイプの専用技**
（ファイヤー(XD)専用技）であり、以降の世代（第4〜9世代を含む）には一切登場しない。

README.md の対象範囲規定「第9世代で実装されていない技・アイテム・特性・ポケモンなどは
実装しない」に該当するため、本技は実装対象外と判定した。
また `docs/champions/move_list.txt` にも該当技は存在しない。

一次情報上の実データ（参考）:
- タイプ: あく（ダークわざ）
- 分類: 特殊
- 威力: 75
- 命中率: 100
- PP: 記載なし（"-"。XDのシャドーわざ特有の仕様で無制限PPの可能性）
- 範囲: 1体選択
- 優先度: 0
- 直接攻撃: しない
- 追加効果: 10%の確率で相手をやけど状態にする
- 判定: おうじゃのしるし○ / マジックコート× / よこどり× / オウムがえし○

既存実装は上記の一次情報と全く異なる（誤った）データ
（タイプ=ゴースト、威力=135、命中率=85、PP=5）で登録されていた。
これは第9世代の実在技のデータを参照した形跡もなく、明らかな誤実装（存在しない技の捏造データ）
と判断し、実装・計画書・テストを削除した。

## 仕様書
- 対象外のため新規作成しない（`docs/spec/moves/ダークファイア.md` は作成しない）。
  第9世代に存在しない技の仕様書を作成すると CLAUDE.md/README.md の対象範囲規定に反するため。

## 実装計画書
- 独立した `docs/plan/moves/ダークファイア.md` は作成しない（対象外のため）。
- ただし `docs/plan/moves/攻撃技_やけど系.md`（やけど付与技の一括計画書）に、誤って
  ダークファイアが「第9世代公式データに基づく」として対象技一覧・実装状況表・ハンドラコード例・
  MoveData 追加例に含まれていたため、該当箇所をすべて削除し、「対象外（実装しない）」の
  注記として一次情報に基づく判定根拠を追記した。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ta.py` から誤った `"ダークファイア": MoveData(...)` エントリを削除した。
- `src/jpoke/handlers/move_attack.py` から誤った `ダークファイア_apply_burn_to_defender` ハンドラ関数を削除した。
- `MoveName` Literal 型は `data/move.py`（`MOVES` 辞書）から
  `scripts/generate_literals/generate_move_literal.py` で自動生成されるため、
  上記削除に伴い `python scripts/generate_literals/generate_move_literal.py` を実行し、
  `src/jpoke/types/move.py` から `"ダークファイア"` を除去した（対象外技のため技名として選択不可になる）。

## リーサル計算
