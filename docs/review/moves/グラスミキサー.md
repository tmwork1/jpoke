# グラスミキサー レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/グラスミキサー.html` を確認したところ、本技はインフォボックスに赤字で
「SVで使用不可」と明記されており、カテゴリにも「スカーレット・バイオレットで使用不可能なわざ」と
分類されている（第5〜8世代では使用可能だった技）。`src/jpoke/data/ps-champ-ja/moves.json`（ポケモンチャンピオンズの
技一覧）にも該当項目は存在しない。WebSearch でも "Leaf Tornado is not a usable move in Pokémon Scarlet
and Violet" と独立に確認できた。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。以降の項目はこの判断に基づく対応内容を記載する
（`docs/review/moves/オクタンほう.md` の前例と同じ扱い）。

## 仕様書
- 新規作成（`docs/spec/moves/グラスミキサー.md`）。
- 対象範囲外である旨と根拠（第8世代までは使用可能だがSVでは使用不可、ポケモンチャンピオンズにも項目なし）を明記した。
- 参考として第5〜8世代時点の技データ（タイプ:くさ、特殊、威力65、命中90、PP10、
  追加効果50%命中率-1段階、直接攻撃なし）を記録した。
- 過去に誤って登録されていた `MoveData.power=60, accuracy=55` が一次情報（威力65・命中90）と
  一致しないことも記録した。

## 実装計画書
- 新規作成（`docs/plan/moves/グラスミキサー.md`）。
- 「実装しない」という結論と、過去の一括実装（`docs/plan/moves/攻撃技_防御側ランク変化系.md`）で
  対象範囲確認なしに誤って登録されていた経緯を記録した。
- `docs/plan/moves/攻撃技_防御側ランク変化系.md` の冒頭「追記（再レビュー後の修正）」節に
  グラスミキサーに関する追記セクションを追加し、本文中の記載（威力60・命中55等）が誤った過去の
  登録内容であり削除済みであることを明記した（本文中のコード断片自体はオクタンほうの前例に倣い
  そのまま残し、削除済みである旨のみ追記した）。

## 実装（handlers/ / data/）
- 誤って登録されていた実装を削除した。
  - `src/jpoke/data/moves/move_ka.py`: `"グラスミキサー": MoveData(...)` エントリを削除
    （一次情報では基礎威力65・命中90だが、威力60・命中55という誤った値で登録されていた）
  - `src/jpoke/handlers/move_attack.py`: `グラスミキサー_lower_acc` ハンドラを削除
    （ハンドラ本体の効果 `modify_defender_stats(..., stats={"accuracy": -1}, chance=0.5)` 自体は
    一次情報の追加効果と一致していた）
  - `src/jpoke/types/move.py`: `MoveName` Literal から `"グラスミキサー"` を削除し、
    `python scripts/generate_literals/generate_move_literal.py` を実行して MOVES辞書と整合することを確認した
    （実行結果は「変更なし」で手動削除内容と一致）
  - 五十音ソート（`sort_handlers.py` / `sort_data/sort_moves.py`）はこのレビュー段階では実行しない
    （マージ後に一括整形するため。指示書の方針に従った）
  - `_INNATE_FLINCH_MOVES` 等、他ハンドラ内の技名リストへの「グラスミキサー」参照は見つからなかった
