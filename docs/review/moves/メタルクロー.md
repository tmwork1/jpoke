## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ma.py` の `メタルクロー` エントリで `flags={"contact"}` のみが
  設定されており、`secondary_effect` フラグが欠落していた。同種の追加効果あり技
  （アイアンヘッド・あやしいかぜ・グロウパンチ・げんしのちから・コメットパンチ・はがねのつばさ）は
  すべて `secondary_effect` フラグを持っており、ちからずく特性（`ちからずく_boost` /
  `ちからずく_disable_secondary_effect`、いずれも `ctx.move.has_flag("secondary_effect")` で判定）
  との連携に必要なため、`flags={"contact", "secondary_effect"}` に修正した。
- `src/jpoke/handlers/move_attack.py` の `メタルクロー_boost_attacker_A`
  （`modify_attacker_stats(battle, ctx, value, stats={"atk": 1}, chance=0.1)`）は
  一次情報・共有テーブルと一致しており問題なし。
- PP=35（Gen9本家基準）は `docs/champions/move_list.txt` に メタルクロー の記載が存在しないため
  そのまま採用する方針（過去の ずつき レビュー前例）に従い、変更なし。
- `power=50` `accuracy=95` `category="physical"` `type="はがね"` `target`（デフォルト`foe`）
  `priority`（デフォルト0）はすべて一次情報と一致しており問題なし。
