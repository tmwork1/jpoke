  （既存の `{"contact"}` はそのまま維持）。これが欠けていたため、`ちからずく_boost` による威力1.3倍化と
  `ちからずく_disable_secondary_effect` によるすばやさ低下無効化のいずれも発動しない状態だった。
- `docs/spec/moves/_move_secondary_1.md` に「ローキック defender 1 0 0 0 0 -1 ...」（確率1・S-1）と
  記載されており、修正後の実装と一致することを確認した。
- PP は元々 `pp=20` で `docs/champions/move_list.txt` の値と一致していた（コード側の修正不要）。
- `handlers/move_attack.py` の `ローキック_lower_defender_spd`（`modify_defender_stats` 経由、
  chance省略時デフォルト100%）は仕様と一致しており修正不要。

## リーサル計算
- 対象外（n/a）。本技の追加効果は相手の『すばやさ』ランクを下げるのみでダメージ計算に直接介入する
