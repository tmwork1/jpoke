# ミストボール レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/ミストボール.md`）。`docs/wiki/moves/ミストボール.html` を一次情報として、
  タイプ・分類・威力・命中・PP・対象・優先度・追加効果（50%でとくこう-1）・判定フラグ・
  弾のわざ（ぼうだん無効化）・ラティアス専用技である旨を記載した。
  PPは `docs/champions/move_list.txt` に本技が存在しないため、方針どおりGen9本家の5をそのまま採用した。

## 実装計画書
- 新規作成（`docs/plan/moves/ミストボール.md`）。既存実装の `flags={"bullet"}` に
  `"secondary_effect"` が欠落していた問題を指摘し、`アイアンテール`・`がんせきふうじ` と同様の
  「ダメージ+確率発動ランク低下」技のパターンに合わせる修正内容を記載した。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ma.py` の `"ミストボール"` エントリの `flags` に `"secondary_effect"` を
  追加した（修正前は `{"bullet"}` のみで、ちからずくとの相互作用（威力1.3倍化・追加効果無効化）が
  発動しないバグがあった）。
- `src/jpoke/handlers/move_attack.py` の `ミストボール_lower_spa_C`（`modify_defender_stats` で
  spa -1・chance 0.5）は仕様と一致しており修正不要。

## リーサル計算
- 対象外（`docs/progress/move.md` の「リーサル実装」列が `n/a`）。本技の追加効果はとくこうランク低下の
  みでHPに影響しないため、`handlers/lethal.py` への実装は不要。

## テスト
- `tests/moves_attack/test_move_ma.py` に以下を追加した:
  - `test_ミストボール_とくこう低下が発動する`: `secondary_chance=1.0` でとくこうが1段階下がることを確認。
  - `test_ミストボール_ちからずくで威力上昇しとくこう低下は発動しない`: ちからずく所持時に
    `power_modifier == 5325`（1.3倍相当）かつとくこう低下が発動しないことを確認
    （`flags={"secondary_effect"}` 追加により初めて正しく動作する）。
  - `test_ミストボール_ぼうだん持ちには技が無効化される`: 特性『ぼうだん』所持の相手に技自体が
    無効化される（汎用ハンドラで処理済みであることの確認）ことを確認。
