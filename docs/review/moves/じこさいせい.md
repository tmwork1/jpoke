    残っていたバグ）

    `"foe_side"` を前提に判定しているため、修正前は誤って「まもるで防がれる」「マジックコートで
    跳ね返される」動作になっていた不具合を修正した。
  - `handlers/move_status.py` の `じこさいせい_heal_self` 本体（HP満タン判定・
    `battle.modify_hp(mon, r=1/2)` による回復）は仕様通りで変更なし。
