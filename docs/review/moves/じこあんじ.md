# じこあんじ レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/じこあんじ.md`）。`docs/wiki/moves/じこあんじ.html` の
  「技の仕様」節を一次情報として、以下を明記した。
  - 能力ランク（A/B/C/D/S/命中/回避）の完全上書きコピー（片方向）
  - 第六世代以降は急所ランクに関する効果（本プロジェクトでは`きゅうしょアップ`
    ＝`critical_rank`）もコピー対象
  - `たんじゅん`・`あまのじゃく`・`クリアボディ`・`だっしゅつパック`・`かちき`・
    `まけんき`・`びんじょう`・`ものまねハーブ`は反応しない一方、`しろいハーブ`のみ
    コピー直後に負のランクを打ち消す特例がある
  - `まもる`無視（`unprotectable`）・`マジックコート`無効（`unreflectable`）・
    `みがわり`貫通（`bypass_substitute`）
  - `おうごんのからだ`の相手には失敗（既存実装のみで対応済み、追加実装不要）
  - `よこどり`・`オウムがえし`は本プロジェクト未実装のため対象外

## 実装計画書
- 新規作成（実質的に上書き。既存ファイルは旧世代のスタット表記（A/B/C/D/S/ACC/EVA）や
  未整理の検討メモが残っており、現行アーキテクチャに合わせて全面的に書き直した）。
- PP・命中率・flags の修正内容、急所ランクコピー・しろいハーブ特例のハンドラ実装方針を明記。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_sa.py`: `pp=10` → `pp=12`（`src/jpoke/data/ps-champ-ja/moves.json`の
  実値に合わせて修正。移行漏れのバグだった）。`accuracy=100` → `accuracy=None`（必中・
  命中判定なしの仕様に合わせて修正。従来はアクセスランク補正の影響を受けてしまう誤りだった）。
  `flags={"unprotectable", "unreflectable", "bypass_substitute"}` を追加
  （まもる無視・マジックコート無効・みがわり貫通、いずれも未設定だった）。
- `src/jpoke/handlers/move_status.py` の `じこあんじ_copy_ranks`:
  - 急所ランク（`critical_rank`）のコピー処理を追加（`attacker.critical_rank =
    defender.critical_rank`）。第六世代以降の仕様として欠落していた。
  - `しろいハーブ`の特例発動処理を追加（コピー結果に負のランクが残る場合、当該ランクを
    0に戻してアイテムを消費する）。従来は`battle.modify_stats`を経由しない直接代入の
    ため`ON_MODIFY_STAT`が発火せず、しろいハーブが一切発動しない誤りだった。
  - `ItemPayload` のimportを追加。

## テスト
- `tests/moves_status/test_move_sa.py` に以下を追加。
  - 急所ランクのコピー（きゅうしょアップ付与→コピーされる／相手が0ならクリアされる）
  - しろいハーブの特例発動（負のランクを0に戻しアイテムを消費する）
  - だっしゅつパックが発動しないこと
  - `まもる`状態の相手にも効果が発動すること（`unprotectable`）
  - `みがわり`状態の相手にも効果が発動すること（`bypass_substitute`）
  - 命中判定を行わず必ず命中すること（`move_executor.accuracy is None`）
- `tests/volatiles/test_volatile_hamayarawa.py` に、`マジックコート`でじこあんじが
  跳ね返されないこと（`unreflectable`）を確認するテストを追加。
- `python -m pytest tests/ -v` を実行し、4184 passed, 1 skipped で全件パスを確認
  （結果は `.loop/test_logs/じこあんじ.log` に保存）。
  ※ 本レビューとは無関係に、乱数シードを固定していない一部テスト
  （`test_だっしゅつボタン_ちからずくの効果が発動した技を受けたときは発動しない` 等）が
  実行順・実行タイミングにより稀に失敗することを確認したが、再実行すると全件パスする
  既知の非決定性であり、じこあんじの変更とは無関係。
