## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ra.py`: `らいめいげり` の `flags` に `"secondary_effect"` を追加。
  - 修正理由: 本技は「追加効果として100%の確率で相手のぼうぎょを1段階下げる」効果を持ち、
    `docs/wiki/abilities/ちからずく.html` に記載の「相手のランク補正を下げる技」に該当するため、
    ちからずく所持時は威力1.3倍・追加効果無効化の対象となるべきだが、`secondary_effect` フラグが
    欠落しており対象外になっていた。同種の100%相手ランク低下技（`うらみつらみ` `ほのおのムチ`）は
    いずれも `secondary_effect` フラグを保持しており、扱いを揃えた。
- `src/jpoke/handlers/move_attack.py` の `らいめいげり_lower_defender_def`: 変更なし
  （`modify_defender_stats(battle, ctx, value, stats={"def": -1})` は仕様と一致）。
