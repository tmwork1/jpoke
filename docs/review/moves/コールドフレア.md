  こおり/特殊/威力140/命中90/PP5/対象1体/優先度0/非接触、効果（1ターン目に溜め2ターン目で攻撃・
  30%でやけど付与）、判定フラグ（まもる○・マジックコート×・よこどり×・オウムがえし○ 等）を記載した。
- PPについて `src/jpoke/data/ps-champ-ja/moves.json` に本技の記載が無いことを確認した（`コ` で始まる技のうち
  本技と `コートチェンジ` のみ欠落）。既存レビュー前例（`ねんりき.md`）に倣い、本家（Wiki）の値である
  PP5をそのまま採用する方針を明記した。

## 実装計画書
- 新規作成（`docs/plan/moves/コールドフレア.md`）。実装調査の結果、既存実装には「30%でやけど付与」
  ハンドラ（`ON_DAMAGE_HIT`）のみが登録されており、**技の中核である「1ターン目に溜め、2ターン目で
  攻撃する」2ターン構造が全く実装されていなかった**（`Event.ON_MOVE_CHARGE` ハンドラ未登録、
  対応する揮発状態 `VolatileData` 未登録）ことが判明したため、これを実装する計画を記載した。
  - 姿を隠さない・天候スキップが無い最もシンプルな溜め技であるため、`あなをほる` の
    `ON_MOVE_CHARGE` 登録スタイル（`charge_into_volatile` 共通ヘルパー）と、
    `ソーラービーム`/`メテオビーム`/`エレクトロビーム` の VolatileData 構成
    （`forced=True` + `ON_MODIFY_COMMAND_OPTIONS`(force_command) + `ON_HIT`(remove_volatile)のみ、
    回避判定・交代禁止は持たない）を組み合わせる方針とし、priorityの根拠を明記した。
