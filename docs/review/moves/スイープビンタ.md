# スイープビンタ レビュー結果

## 仕様書
- 対象外（変更なし）: スイープビンタは追加効果を持たない汎用の2～5回連続攻撃技であり、
  `docs/spec/moves/連続技.md`（9.3 2-5回攻撃技の表に既に記載あり）でカバーされる。
  個別の仕様書は不要と判断し、新規作成しなかった（`つっぱり`・`みだれづき`等の同種の技と同様の扱い）。
  一次情報（`docs/wiki/moves/スイープビンタ.html`）と照合し、タイプ=ノーマル、分類=物理、
  威力=25、命中=85、範囲=1体選択、優先度=0、直接攻撃=○、2～5回連続攻撃、
  急所判定は各回、命中判定は初回のみ、追加効果なし、まもる有効、おうじゃのしるし対象（第五世代以降）、
  マジックコート×、よこどり×、オウムがえし○であることを確認した（PPはchampions側で12に変更されているため後述）。

## 実装計画書
- 対象外（変更なし）: 固有ハンドラを持たないため、個別の実装計画書は不要と判断した
  （`つっぱり`・`みだれひっかき`等の同種の技と同様）。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_sa.py` の `スイープビンタ` エントリで `pp=10`（Gen9本家基準の値）に
  なっていたバグを発見し、`src/jpoke/data/ps-champ-ja/moves.json`（`スイープビンタ ノーマル 物理 25 85 12 接○ 守○`）
  および `docs/progress/move.md` の該当行（PP列=12）に合わせて `pp=12` に修正した。
  その他の項目（type, category, power=25, accuracy=85, flags={"contact"},
  multi_hit={min:2, max:5, check_hit_each_time:False, power_sequence:()}）は
  一次情報・既存の同種連続攻撃技（`みだれづき`, `みだれひっかき`）と一致しており修正不要だった。

## リーサル計算
- 対象外: `docs/progress/move.md` の該当行は「リーサル実装」「リーサルテスト」ともに `n/a`。
  連続攻撃技は仕様上ヒット回数が乱数依存のためリーサル計算の対象外として扱われている。
