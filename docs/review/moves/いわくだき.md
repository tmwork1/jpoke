# いわくだき レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/いわくだき.md`）。一次情報（ポケモンWiki）から以下を確認：
  - かくとうタイプ / 物理 / 威力40（第四世代以降）/ 命中100 / PP15 / 接触あり
  - 追加効果: 50%の確率で相手の『ぼうぎょ』ランクを1段階下げる
  - 技の説明文は「パンチで攻撃する」だが、一次情報の備考欄に明記の通り**パンチ技には分類されない**
    （てつのこぶし・パンチグローブの対象外）

## 実装計画書
- 新規作成（`docs/plan/moves/いわくだき.md`）。実装済みの `data/move.py` の記述に対し、
  `flags` に `"secondary_effect"` が欠落しているバグを指摘し、修正内容を明記した。

## 実装（handlers/ / data/）
- `src/jpoke/data/move.py` の `"いわくだき"` エントリの `flags` を
  `{"contact"}` → `{"contact", "secondary_effect"}` に修正。
  - 本技は追加効果（50%ぼうぎょ-1）を持つため、ちからずく特性
    （`ちからずく_boost` / `ちからずく_disable_secondary_effect`、`handlers/ability.py`）が
    `ctx.move.has_flag("secondary_effect")` で判定する対象に含める必要があった。
    欠落したままだと、ちからずく所持ポケモンが使用しても威力1.3倍化・追加効果無効化のどちらも
    発生しないバグとなっていた。
  - `"punch"` フラグは一次情報により対象外と確認済みのため追加していない（実装は元々含んでおらず正しい）。
- `handlers/move_attack.py` の `いわくだき_lower_defender_def`（`modify_defender_stats` 呼び出し）は
  仕様と一致しており、修正なし。
- `python scripts/sort_data/sort_moves.py` を実行し、`data/move.py` の並び順を維持。
