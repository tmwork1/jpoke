## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ra.py` の `"りゅうのはどう"` エントリで `pp=12` と
  なっていたのを `pp=10` に修正した。
  - Gen9基準（本家）のりゅうのはどうのPPは10だが、`docs/champions/move_list.txt` の
    PP欄は12となっていた。ただし同ファイルの他の技（りゅうせいぐん・ラスターカノン・
    リーフストーム等）についても champions move_list.txt のPP欄はGen9基準PPと
    乖離しており、既存実装はいずれもchampions欄ではなくGen9基準PPを採用している。
    この慣例に合わせて `pp=10` に修正した。
- その他（type・category・power・accuracy・flags={"pulse"}・handlers空）は
  仕様と一致しており変更なし。
