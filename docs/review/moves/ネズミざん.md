# ネズミざん レビュー結果

## 一次情報との照合メモ

タスク指示には「ネズミざんは味方の数に応じて連続で繰り出される技です」とあったが、
`docs/wiki/moves/ネズミざん.html`（ポケモンWiki一次情報）を確認したところ、
ネズミざんは通常の連続攻撃技（外れるまで最大10回、1発ごとに命中判定）であり、
味方の数に応じてヒット回数が決まる効果は存在しない
（「1-10回の間 連続で あたる」との記載のみ）。
味方の数に応じて攻撃回数が決まる技は「ふくろだたき」であり
（`docs/spec/moves/連続技.md` の「9.6 味方人数依存の連続攻撃技」参照）、
タスク指示はこの技と混同したものと判断し、一次情報に基づいて通常の
連続攻撃技として実装・レビューした。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_na.py` の `ネズミざん` の `pp` が `10`
  （第9世代本家基準）になっていたバグを修正し、`src/jpoke/data/ps-champ-ja/moves.json`
  準拠の `12` に変更した。
  - `docs/progress/move.md` の該当行は PP=12 と記載されていたため、
    移行漏れバグと判断し実装側を修正した（進捗表は変更していない）。
- `multi_hit`（`min=10, max=10, check_hit_each_time=True, power_sequence=()`）、
  `flags`（`contact`, `slash`, `check_hit_each_time`）は仕様通りで、
  `core/move_executor.py` の汎用連続攻撃機構（`_resolve_hit_count` /
  `_resolve_hit_power` / ヒットごと命中判定ループ）で正しく処理されることを
  トリプルアクセル・トリプルキックの既存実装と比較して確認した。
- スキルリンク（`スキルリンク_modify_hit_check_each_time`）・いかさまダイス
  （`いかさまダイス_modify_hit_check_each_time`）・きれあじ
  （`きれあじ_modify_power`、`slash` フラグ判定）はいずれも技名ハードコードでは
  なく汎用フラグ／イベントベースの実装になっており、ネズミざんにも正しく適用
  されることを確認した。修正不要。
