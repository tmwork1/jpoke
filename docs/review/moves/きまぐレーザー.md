# きまぐレーザー レビュー結果

## 仕様書
- 変更なし（新規作成せず）。きまぐレーザーは特殊・ドラゴン・威力80・命中100・PP5・非直接攻撃技であり、
  「30%の確率で威力が2倍になる（基礎威力ではなく威力補正として2倍）」という単一の効果のみを持つ。
  同種の単純な威力変動技（DDラリアット・せいなるつるぎ等）は進捗表で仕様書列を「-」として扱う運用に
  統一されているため、きまぐレーザーについても専用の仕様書ファイルは作成しない。

## 実装計画書
- 変更なし（新規作成せず）。上記と同じ理由により専用の計画書ファイルは作成しない。
  `docs/plan/moves/攻撃技_状態依存系.md` に「きまぐレーザー」のハンドラ構成
  （`ON_CALC_POWER_MODIFIER` / `attacker:self` / 30%確率で威力2倍）と
  priority・subject_spec の根拠が既に記載されており、一次情報（`docs/wiki/moves/きまぐレーザー.html`）と
  照合して内容に誤りがないことを確認した。

## 実装（handlers/ / data/）
- 変更なし。`src/jpoke/data/move.py` の `"きまぐレーザー"` エントリ
  （type="ドラゴン", category="special", pp=5, power=80, accuracy=100, flags未設定＝直接攻撃×）は
  一次情報（タイプ ドラゴン / 分類 特殊 / 威力80 / 命中率100 / PP5 / 直接攻撃×）と一致している。
- `handlers/move_attack.py` の `きまぐレーザー_maybe_double_power` は
  `battle.random.random() < 0.3` を判定して `apply_fixed_modifier(value, 8192)`（2倍）を返す実装であり、
  一次情報の「30%の確率で威力が2倍になる」と一致している。`subject_spec="attacker:self"` も
  使用者自身の確率判定として妥当。
- 修正不要と判断した。

## リーサル計算

## テスト
- `tests/moves_attack/test_move_ka.py` に以下の既存テストがあり、実装と整合していることを確認した。
  - `test_きまぐレーザー_30パーセントで威力2倍`（`battle.random.random` を `0.29` に固定し `power_modifier == 8192` を確認）
  - `test_きまぐレーザー_70パーセントで威力等倍`（`battle.random.random` を境界値 `0.3` に固定し `power_modifier == 4096` を確認）
- 境界値（`< 0.3` の真偽）を両方カバーしており、追加のテストは不要と判断した。
- `python -m pytest tests/ -v` を実行し、3293 passed, 1 skipped で全件成功を確認した
  （ログ: `.loop/test_logs/きまぐレーザー.log`）。
  ※実行中に `test_オーロラベール_ひかりのかべとは重複しない` が急所判定の乱数依存で
  1回だけ失敗したが、単体再実行では成功しており本タスクの変更とは無関係のフレーキーテストと判断した。
