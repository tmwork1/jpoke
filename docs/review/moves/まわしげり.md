# まわしげり レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/まわしげり.html` を確認したところ、本技はカテゴリ
「ソード・シールドで使用不可能なわざ」「ブリリアントダイヤモンド・シャイニングパールで使用不可能なわざ」
「スカーレット・バイオレットで使用不可能なわざ」に分類されており、第八世代の説明文も
「このわざは つかえません」となっている。`docs/champions/move_list.txt` にも本技のエントリは存在しない。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。以降の項目はこの判断に基づく対応内容を記載する。

さらに、削除前の既存実装を確認したところ `MoveData(power=85, accuracy=90, pp=10)` という値が
登録されていたが、実際のゲームでの本技の基礎値は威力60・命中85・PP15であり、この点でも誤りがあった
（対象範囲外である以前に、仮に実装対象だったとしても値が誤っていた）。
