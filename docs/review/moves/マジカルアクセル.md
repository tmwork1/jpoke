## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ma.py` の `"マジカルアクセル"` エントリを修正:
  - `type`: `"エスパー"` → `"フェアリー"`（一次情報のタイプはフェアリー。誤って別技のタイプ
    が使われていた）
  - `category`: `"special"` → `"physical"`（一次情報の分類は物理。特殊として誤登録されていた）
  - `pp`: `10` → `12`（Championsの8/12/16/20の4段階統一ルールに従い、本家PP10からの
    変換前例（バーンアクセル等）に倣って修正）
  - `flags`: `"non_encore"` を追加（一次情報でアンコール失敗が明記されているが未実装だった。
    `"non_copycat"` は既存実装済みで変更不要）
- `handlers/move_attack.py` の `マジカルアクセル_apply_confusion_to_defender`
  （`apply_confusion_to_defender(battle, ctx, value, chance=0.3)` を呼ぶのみ）は仕様と一致しており
  修正不要。
