## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_a.py` の `インファイト` エントリを修正:
  - `pp=5` → `pp=8`（`docs/champions/move_list.txt` 準拠。Championsの技PPは
    8/12/16/20の4段階に統一されているため）。
  - `flags={"contact", "secondary_effect"}` → `flags={"contact"}`
    （自分自身のランクを下げる確定効果はちからずくの対象外のため。既知の指摘通り、
    このフラグが残っているとちからずく所持時に威力1.3倍・ランクダウン無効化が
    誤発動するバグになっていた）。
  - ハンドラ参照を `ha.インファイト_lower_attacker_stats` に修正（改名に合わせる）。
- `src/jpoke/handlers/move_attack.py` の `インファイト_lower_defender_stats` を
  `インファイト_lower_attacker_stats` に改名（実装内容は `modify_attacker_stats` による
  自分自身のランクダウンで元々正しかったが、関数名が「defender」となっており
  実装内容と矛盾していた命名バグを修正）。docstring を追加。
