## 実装（handlers/ / data/）
- `src/jpoke/data/move.py` の `おはかまいり` エントリから誤った `flags={"contact"}` を削除した。
  - 一次情報の「直接攻撃: ×」に反し、接触技として登録されていたバグ。
  - `core/query.py` の `is_contact()` / `is_contact_reaction()` が `MoveData.flags` を参照するため、
    修正前はさめはだ・てつのトゲ・ゴツゴツメット等の「相手の直接攻撃を受けたときに反応する」効果が
    誤発動する状態だった。
  - `handlers/move_attack.py` の `おはかまいり_calc_power`（威力計算ロジック）自体には誤りがなく、
    変更なし。
