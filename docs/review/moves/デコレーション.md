## 実装（handlers/ / data/）
- 既存の `src/jpoke/data/moves/move_ta.py` の `"デコレーション"` エントリに以下の誤り・欠落が
  あったため修正した。
  - PP誤り: `15`（Gen9本家基準のまま） → `16`（`src/jpoke/data/ps-champ-ja/moves.json` L522 準拠）。
  - 命中率誤り: `accuracy=100` が明示指定されており通常の命中判定が行われる実装になっていた
    → 必中のため `accuracy` 指定を削除（デフォルト `None` で命中判定自体を行わない）。
  - 効果欠落: `handlers={}` のままで能力ランク変化の効果が一切実装されていなかった
    → `Event.ON_STATUS_HIT` に `デコレーション_modify_defender_stats` を追加。
  - フラグ欠落: `flags` が未設定で、まもる・ダイウォール・マジックコートで防がれてしまう実装
    になっていた → `flags={"unprotectable", "unreflectable"}` を追加。
- `src/jpoke/handlers/move_status.py` に `デコレーション_modify_defender_stats`
  （相手のこうげき・とくこうを2段階ずつ上げる）を新規実装した。既存の共通ヘルパー
  `modify_defender_stats`（`handlers/move.py`）を利用し、こうげき・とくこう両方とも
  変化しなかった場合（両方+6で頭打ち等）は `move_executor` 側の汎用判定で技全体が
  自動的に失敗扱いになる（`つぶらなひとみ`・`ちょうのまい` と同型）。
- 挿入位置: `handlers/move_status.py` は `でんじふゆう_check_can_use` の後・`とおせんぼう_apply`
  の前（「で」グループ相当の位置）。`data/moves/move_ta.py` は既存エントリの位置をそのまま維持。
- 五十音ソート（`sort_handlers.py` / `sort_data/sort_moves.py`）は本レビューでは実行していない
  （指示により、マージ後に一括整形）。
