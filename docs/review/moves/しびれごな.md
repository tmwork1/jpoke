## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_sa.py` の `しびれごな` の PP を 30 → 20 に修正
  （Pokémon Champions基準の `src/jpoke/data/ps-champ-ja/moves.json` では PP 20。Gen9本家基準の
  30が移行漏れとして残っていたバグ）。
- `handlers/move_status.py` の `しびれごな_apply_ailment_to_defender` はまひ付与のみを行う
  シンプルな実装で、`でんじは_apply_ailment_to_defender` / `へびにらみ_apply_ailment_to_defender`
  と同様のパターンになっており、修正不要と判断した。
- でんき/くさタイプ無効・ぼうじん・ぼうじんゴーグルによる粉技無効化はフレームワーク側
  （`move_executor.py` の `_check_grass_type_powder_immunity`、`ぼうじん_block_powder`、
  `ぼうじんゴーグル_block_powder_move`）が `powder` フラグから汎用的に処理しており、
  しびれごな側の実装に欠落はなかった。
- でんきタイプへのまひ無効・特性による無効（じゅうなん等）は `ailment_manager` および各特性
  ハンドラが担保しており、しびれごな固有の実装は不要（欠落なし）。
