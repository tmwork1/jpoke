# ガリョウテンセイ レビュー結果

## 仕様書
- 新規作成（`docs/spec/moves/ガリョウテンセイ.md`）。
  - タイプ/分類/威力/命中/PP/対象/優先度/接触の基本情報を明記。
  - 効果: 命中して技が成功した後、確定で自分の『ぼうぎょ』『とくぼう』ランクが各1段階下がる
    （インファイトのひこうタイプ版）。
  - PPはチャンピオンズ`src/jpoke/data/ps-champ-ja/moves.json`に本技自体が未収録のため、
    本家PP5が同じインファイト・だいもんじ（チャンピオンズPP=8）と同じ変換規則を適用し、
    PP=8として採用（実装は誤って本家基準PP=5のままだった）。
  - レックウザの技依存メガシンカ（メガストーン不要）は本プロジェクトのメガシンカ実装
    （持ち物トリガーのみ）の対象外である旨を明記。

## 実装（handlers/ / data/）

  `modify_attacker_stats` で自分のランクを正しく下げており、誤りは命名のみ）。
  docstringを追加。
- `src/jpoke/data/moves/move_ka.py`:
  - `pp`: 5 → 8（チャンピオンズPP統一規則の適用漏れを修正）。
  - `flags`: `{"contact", "secondary_effect"}` → `{"contact"}`
    （自分自身への確定ランク低下は「追加効果」に該当せず、`secondary_effect`が
    付いているとちからずく所持時に誤って発動しなくなるバグだったため削除）。
  - ハンドラ参照をリネーム後の関数名に更新。
