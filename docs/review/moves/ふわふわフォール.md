# ふわふわフォール レビュー結果

## 一次情報確認の結果（重要）

`docs/wiki/moves/ふわふわフォール.html` を確認したところ、本技は
**第七世代（`ポケットモンスター Let's Go! ピカチュウ・Let's Go! イーブイ`）専用の「相棒わざ」** であり、
相棒ピカチュウのみが使用できる専用技として追加されたもの（カテゴリ「専用技」「相棒わざ」）。
ソード・シールド以降（BDSP・SVを含む）には一度も実装されていない。

README.md の「対象範囲」定義（「第9世代で実装されていない技・アイテム・特性・ポケモンなどは実装しない」）
に該当するため、**本技は実装対象外と判断した**。さらに `src/jpoke/data/ps-champ-ja/moves.json`
（ポケモンチャンピオンズの全技一覧）にも「ふわふわフォール」の項目が存在しないことを確認し、
ポケモンチャンピオンズ側でも採用されていないという二重の根拠を得た。

以降の項目はこの判断に基づく対応内容を記載する。

## 仕様書
- 新規作成（`docs/spec/moves/ふわふわフォール.md`）。
- 対象範囲外である旨と根拠（Let's Go! 専用の相棒わざであり、SV・ポケモンチャンピオンズいずれにも
  存在しない）を明記した。
- 参考として第七世代時点の技データ（タイプ:ひこう、物理、威力90、命中95、PP15、追加効果30%ひるみ、
  直接攻撃）を記録した。

## 実装計画書
