- 変更なし。
- あくうせつだんはダメージ計算・追加効果ともに単純な「急所ランク+1」の技であり、
  `docs/spec/critical.md` の「急所ランク+1」技一覧に既に記載済みのため、専用の
  仕様書（`docs/spec/moves/あくうせつだん.md`）は新規作成しなかった
  （3ぼんのや・エアカッターなど他の単純な急所ランク+1技も同様に専用仕様書を持たない運用と一致）。
- 一次情報（`docs/wiki/moves/あくうせつだん.html`）で以下を確認し、実装と整合していることを確認した。
  - タイプ: ドラゴン / 分類: 特殊 / 威力: 100 / 命中率: 95 / PP: 5 / 範囲: 1体選択 / 優先度: 0

  - `test_あくうせつだん_急所ランクが1_乱数大で急所なし`: 急所閾値以上の乱数で急所にならないことを確認

- `python scripts/sort_tests.py tests/moves_attack/test_move_a.py` でソート、
  `python scripts/generate_test_list.py` でテスト一覧を更新した。
- `python -m pytest tests/ -v` を実行し、新規追加した3件はすべてPASSしたことを確認した

  既存不具合）が観測されたが、本タスクの変更とは無関係の既存問題であるため対象外とした。
