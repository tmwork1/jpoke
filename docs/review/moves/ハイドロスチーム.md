  - 一次情報（`docs/wiki/moves/ハイドロスチーム.html`）をもとに、基本情報（みず/特殊/威力80/
    命中100/PP16/相手1体/優先度0/接触なし）と効果（にほんばれ下1.5倍・こおり自己解凍・
    相手のこおり解除）を記載。
  - PPはGen9本家15だが、`src/jpoke/data/ps-champ-ja/moves.json` に直接の記載がないため、同じく
    Gen9本家PP15だった特殊技（あくのはどう・シャドーボール等）がいずれもchampions PP16に
    統一されていることから、championsのPPは16であると推定して記載。
  - `docs/spec/abilities/ちからずく.md`（13行目・39行目）のちからずく無効化リストに
    ハイドロスチームが含まれない（＝ちからずく対象技ではない）ことを明記。

## 実装（handlers/ / data/）
- `src/jpoke/data/moves/move_ha.py`
  - `"ハイドロスチーム"` の `pp` を Gen9本家基準の `15` から championsの `16` に修正。
  - `Event.ON_TRY_ACTION`（priority=5）に `ハイドロスチーム_thaw_attacker` を追加登録
    （既存実装は威力補正のみで、使用者自身のこおり自己解凍処理が欠落していた）。
- `src/jpoke/handlers/move_attack.py`
  - `ハイドロスチーム_thaw_attacker` を新規実装（`ねっとう_thaw_attacker` と同一パターン）。
  - `ハイドロスチーム_power_modifier`（既存の晴れ威力補正ハンドラ）は仕様と照合した結果、
    攻撃側・防御側それぞれのばんのうがさ判定・ノーてんき判定を含めて正確に実装されており
    修正不要と判断。
- `src/jpoke/handlers/ailment.py`
  - `こおり_cure_by_thaw_move` のちからずく無効化条件が「技タイプがほのおでない」のみを
    基準にしており、ハイドロスチーム（ちからずく対象外の技）まで誤って無効化していたバグを
    修正。条件に `ctx.move.has_flag("secondary_effect")` を追加し、ちからずくの実際の対象技
    （＝secondary_effectフラグを持つ非ほのおタイプのthaw技）のみを無効化対象にするよう変更。
    これによりハイドロスチームは常に相手のこおりを解除できるようになった。
