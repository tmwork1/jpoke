# ツインビーム レビュー結果

## 仕様書
- 変更なし。
- ツインビームは「固定2回攻撃技」であり、`docs/spec/moves/連続技.md`（9.1節）に
  タイプ・分類・威力・命中・PP（第9世代本家基準）・直接攻撃の有無・効果が
  一覧として記載済み。同じ扱いの兄弟技（ダブルウイング等）も個別仕様書を
  持たない運用のため、個別ファイルは新規作成せず既存の共通仕様書を正とした。
- 一次情報（`docs/wiki/moves/ツインビーム.html`）と照合し、タイプ=エスパー、
  分類=特殊、威力=40、命中=100、直接攻撃=×（非接触）、効果=2回連続攻撃
  （連続攻撃技、ゆびをふるでは選ばれない）で共通仕様書の記載と整合している
  ことを確認した。
