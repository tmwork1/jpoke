# つっぱり レビュー結果

## 仕様書
- 対象外（変更なし）: つっぱりは追加効果を持たない汎用の2～5回連続攻撃技であり、
  `docs/spec/moves/連続技.md`（9.3 2-5回攻撃技の表に既に記載あり）でカバーされる。
  個別の仕様書は不要と判断し、新規作成しなかった（同種の他の連続攻撃技と同様の扱い）。

## 実装計画書
- 対象外（変更なし）: 固有ハンドラを持たないため、個別の実装計画書は不要と判断した
  （みだれづき・つららばり等の同種の技と同様）。

## 実装（handlers/ / data/）
- `src/jpoke/data/move.py` の `つっぱり` エントリに `multi_hit`（min=2, max=5,
  check_hit_each_time=False, power_sequence=()）が欠落していたため追加した。
  一次情報（`docs/wiki/moves/つっぱり.html`）で「2~5回連続で攻撃する」「命中判定は初回のみ」
  であることを確認し、既存の同種2-5回技（スイープビンタ等）と同じ設定に揃えた。

## リーサル計算
- 対象外（`docs/progress/move.md` の該当行は n/a / n/a のまま）。連続攻撃技はリーサル計算の
  対象外として扱われており、`handlers/lethal.py` に多段ヒット技全般を扱う専用ロジックは無い。

## テスト
- `tests/moves_attack/test_move_ta.py` に `test_つっぱり_複数ヒットする` を追加した
  （既存の `test_つららばり_複数ヒットする` 等と同じパターンで `_resolve_hit_count` が
  2～5の範囲に収まることを確認）。
- `python scripts/sort_tests.py tests/moves_attack/test_move_ta.py` でソート、
  `python scripts/generate_test_list.py` で `docs/tests/` を更新、
  `python -m pytest tests/ -v` で全件パス（3426 passed, 1 skipped）を確認した。
