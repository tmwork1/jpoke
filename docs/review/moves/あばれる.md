# あばれる レビュー結果

## 仕様書

- `docs/spec/moves/あばれる.md` は存在しない。あばれる技固有の効果（強制連続
  攻撃・使用後こんらん）は、あばれる・げきりん・はなびらのまい・だいふんげき
  で共用の揮発性状態として `docs/spec/volatiles/あばれる.md` に既に詳細な
  仕様書（付与条件・消滅条件・こんらん付与の特例・ターン処理優先度）が
  存在しており、これが技レベルの効果を包含している。
  `Vジェネレート`・`Gのちから`・`3ぼんのや` 等のレビュー実績と同様、共用の
  仕様書がある場合は技単体の個別ファイルを新規作成しない方針を踏襲し、
  新規作成は行わなかった。
- 一次情報（`docs/wiki/moves/あばれる.html`）と実装を照合し、以下を確認した：
  タイプ＝ノーマル、分類＝物理、威力＝120（第五世代以降）、命中率＝100、
  PP＝10（第五世代以降）、直接攻撃＝○、追加効果＝×、急所／命中判定＝各回、
  まもる＝○、マジックコート／よこどり＝×、オウムがえし＝○。いずれも
  実装（`data/move.py`）と一致していた。

## 実装計画書

- `docs/plan/moves/あばれる.md` は存在しない。同様の理由で
  `docs/plan/volatiles/あばれる.md`（あばれる_apply・あばれる_tick・強制
  コマンドの実装計画を含む）が既に技の実装内容をカバーしているため、
  個別の新規作成は行わなかった。

## 実装（handlers/ / data/）

- `src/jpoke/data/move.py` の `"あばれる"`: `type="ノーマル"`,
  `category="physical"`, `pp=10`, `power=120`, `accuracy=100`,
  `flags={"contact"}`, `Event.ON_HIT` に `ha.あばれる_apply` を登録。
  いずれも一次情報と一致しており妥当。変更なし。
- `src/jpoke/handlers/move_attack.py` の `あばれる_apply`
  （`ON_HIT`, `subject_spec="attacker:self"`）: 初回命中時にのみランダム
  2〜3ターンの揮発性状態を付与し、既に付与済み（強制行動2ターン目以降）の
  場合は何もしない実装で仕様と一致。変更なし。
- `src/jpoke/data/volatile.py` の `"あばれる"`（`forced=True`,
  `ON_MODIFY_COMMAND_OPTIONS` で `force_command`, `ON_DAMAGE_HIT`
  priority=180 で `あばれる_tick`）、`src/jpoke/handlers/volatile.py` の
  `あばれる_tick`（ターン経過後カウント0でこんらん付与）は
  `docs/spec/volatiles/あばれる.md` の仕様通りで既存レビュー
  （`docs/review/volatiles/あばれる.md`）済み。変更なし。
- ミス・まもる等による無効化時やねむり等の行動不能時に「あばれる」状態が
  正しく解除されるか（`docs/spec/volatiles/あばれる.md` に記載の消滅条件
  2〜4, 7）は、同仕様書内で「将来対応」と明記された既知の未実装事項であり、
  げきりん・はなびらのまい・だいふんげきと共用の揮発性状態レベルの課題で
  ある。本レビューは技（あばれる）単体のスコープであり、共用の揮発性状態は
  別途 `docs/review/volatiles/あばれる.md` でレビュー済みのため、今回は
  現状維持とした。

## テスト

- `tests/moves_attack/test_move_a.py` に既存の3件
  （`test_あばれる_初回命中時にあばれる揮発状態が付与される`,
  `test_あばれる_カウント終了後にこんらんが付与される`,
  `test_あばれる_あばれる状態中はON_HITで揮発状態を再付与しない`）で
  ON_HIT経由の初回付与・再付与抑制・こんらん移行を確認済み。
  `tests/volatiles/test_volatile_a.py` に既存の4件でカウント進行・強制行動
  （PP不消費）を確認済み。実装と照合し過不足なし。変更なし。
- `python scripts/sort_tests.py tests/moves_attack/test_move_a.py` を実行
  したが差分なし（既に五十音順）。
- `python scripts/generate_test_list.py` を実行したが差分なし
  （既に最新）。
- `python -m pytest tests/ -v` を実行し、あばれる関連の7件
  （`tests/moves_attack/test_move_a.py`・`tests/volatiles/test_volatile_a.py`
  内の該当テスト）はすべて成功。結果は `.loop/test_logs/あばれる.log` に
  保存した。全体では `tests/test_megaevol.py` の2件
  （`test_メガシンカ_フォルムが変わる`）が
  `KeyError: 'ほのおのたてがみ'` で失敗しているが、これは同時実行中の
  他の並列レビュー作業による `src/jpoke/data/pokedex.json` /
  ability 関連データの編集途中状態に起因するものであり、あばれるの変更
  （今回は無変更）とは無関係であることを、あばれる関連テストのみを
  対象にした単独実行（7 passed）で確認した。
