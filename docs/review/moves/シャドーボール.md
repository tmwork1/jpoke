## 実装（handlers/ / data/）
- **PP誤り修正**: `src/jpoke/data/moves/move_sa.py` の `pp=15`（Gen9本家値）を
  `pp=16`（champions/move_list.txt 準拠）に修正。
- **`secondary_effect` フラグの欠落を修正**: シャドーボールは相手のとくぼうランクを
  下げる追加効果を持つため、`docs/spec/abilities/ちからずく.md` の「相手のランク補正を
  下げる技」に該当しちからずく対象技のはずだが、`flags={"bullet"}` のみが設定されており
  `secondary_effect` が欠落していた。このため、ちからずくを持つポケモンが使用しても
  威力1.3倍化・追加効果無効化が発動しないバグになっていた。
  → `flags={"bullet", "secondary_effect"}` に修正。
- `シャドーボール_lower_defender_spd`（`handlers/move_attack.py`）のロジック自体
  （`modify_defender_stats` で20%の確率でとくぼうランク-1）は正しく、変更なし。
