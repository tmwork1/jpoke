# するどいくちばし レビュー結果

## 仕様書
- 新規作成（`docs/spec/items/するどいくちばし.md`）
  - 効果: ひこうタイプ技の威力を1.2倍にする（第四世代以降）、なげつける威力50
  - 詳細仕様: タイプが変わる技（ノーマルスキン・スカイスキン等）は変更後のタイプがひこうタイプであれば効果あり
  - フライングプレスはかくとうタイプ扱い（相性計算のみひこう追加）のため効果なし、という一次情報の注記を追加

## 実装計画書
- 既存の `docs/plan/items/するどいくちばし.md` を、仕様書整備済みの他アイテム（`くろいメガネ.md`）と同じ形式に全面更新
  - イベント・subject_spec・priority・ハンドラロジックを明記
  - フライングプレスが `move.type` を変更しない（かくとうのまま）ため、追加実装なしで自動的に効果対象外になる旨を明記

## 実装（handlers/ / data/）
- 変更なし
  - `src/jpoke/handlers/item.py` の `するどいくちばし_modify_power_by_type` は `_modify_power_by_type(ctx.move, value, type_="ひこう", modifier=4915)` を呼び出しており、仕様通り威力1.2倍（4915/4096）
  - `src/jpoke/data/item.py` の `ItemData` は `fling_power=50` で一次情報の「なげつける:威力50」と一致
  - `Event.ON_CALC_POWER_MODIFIER` / `subject_spec="attacker:self"` は他のタイプ強化アイテムと同一パターンで問題なし

## テスト
- 変更なし
  - `tests/test_item.py` の `test_タイプ強化アイテム`（パラメータ化テスト）に `("するどいくちばし", "ひこう")` が既に含まれており、威力補正倍率（4915）を検証済み
  - `python -m pytest tests/ -v` で3142件成功・1件スキップ（既存のスキップで本アイテムと無関係）を確認
