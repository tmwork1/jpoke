# ぼうごパット レビュー結果

## 仕様書
- 一次情報（`docs/wiki/items/ぼうごパット.html`）と照合し、空欄だった「効果」節を補充した。
  - 直接攻撃に対して発動する技・特性・持ち物の効果を防ぐこと（非消費）
  - もふもふ/わるいてぐせを除き、直接攻撃に反応する特性の効果を防げること（メッセージの出方の分類）
  - キングシールド等のまもる系反撃効果・くちばしキャノンの反撃やけど・ゴツゴツメットのダメージを防ぐこと（メッセージなし）
  - なげつける威力30、ウッウロボの素材情報
- 「詳細な仕様」節は既に一次情報と一致していたため変更なし。

## 実装計画書
- 既存の計画書は「自分が使う技の接触判定自体を無くす」という誤った設計（パンチグローブ/えんかくと同じ`ON_CHECK_CONTACT`を強制Falseにする方式）になっており、
  一次情報の「所持者の技が直接攻撃でなくなるわけではない…所持者の特性かたいツメ/どくしゅ/ふかしのこぶしの効果は発動する」という記述と矛盾していたため、全面的に書き直した。
- 新設イベント`Event.ON_CHECK_CONTACT_REACTION`と`PokemonQuery.is_contact_reaction()`を用いる設計に変更し、
  「相手が接触を受けたことに反応する効果」と「自分の技が接触技であることに由来する効果」を区別する方針を明記した。
- turn.md未掲載イベントであることの根拠（既存の`ON_CHECK_CONTACT`に準拠しデフォルト優先度100）を明記した。

## 実装（handlers/ / data/）
- **重大な実装誤りを修正**: 従来の実装は`Event.ON_CHECK_CONTACT`をサブジェクト`attacker:self`で強制的にFalseにしており、
  これは「自分の技を接触技でなくす」パンチグローブ/えんかくと同じ効果になってしまっていた。
  この実装では以下が本来の仕様と異なっていた：
  - かたいツメ/どくしゅ/ふかしのこぶし（自分の技が接触技であることに由来する効果）が誤って無効化されていた
  - もふもふ（相手の被ダメージ半減）が誤って無効化されていた
  - わるいてぐせ（相手によるアイテム強奪）が誤って無効化されていた
- 修正内容:
  - `src/jpoke/enums/event.py`: `Event.ON_CHECK_CONTACT_REACTION`を新設
  - `src/jpoke/core/query.py`: `PokemonQuery.is_contact_reaction(ctx)`を新設（`is_contact(ctx)`を初期値として`ON_CHECK_CONTACT_REACTION`を発火）
  - `src/jpoke/handlers/item.py`: `ぼうごパット_negate_contact`（`ON_CHECK_CONTACT`購読）を`ぼうごパット_block_contact_reaction`（`ON_CHECK_CONTACT_REACTION`購読）に変更
  - `src/jpoke/data/item.py`: ぼうごパットのハンドラ登録イベントを`ON_CHECK_CONTACT_REACTION`に変更
  - 「相手が接触を受けたことに反応する効果」を判定している以下の箇所を`is_contact(ctx)`から`is_contact_reaction(ctx)`に変更（ぼうごパットの影響を受けるようにした）:
    - `handlers/ability.py`: `_apply_contact_counter_ailment` / `_apply_contact_counter_chip`（さめはだ・てつのトゲ・どくのトゲ・せいでんき・ほのおのからだ）、
      カーリーヘアー、さまようたましい、ぬめぬめ、のろわれボディ、ほうし、ほろびのボディ、`_overwrite_ability_on_contact`（ミイラ・とれないにおい）、メロメロボディ、ゆうばく
    - `handlers/item.py`: `_apply_contact_item_chip`（ゴツゴツメット）
    - `handlers/volatile.py`: くちばしキャノンの反撃やけど、`_run_protect`（キングシールド・スレッドトラップ・トーチカ・ニードルガード・ブロッキング）
    - `core/move_executor.py`: `contact_hitter`記録（くちばしキャノンの遅延判定用）
  - かたいツメ・どくしゅ・ふかしのこぶし（自分の技が接触技であることに由来する効果）と、もふもふ・わるいてぐせ（ぼうごパット対象外と明記された効果）は
    引き続き`is_contact(ctx)`を使用し、ぼうごパットの影響を受けないことを維持した。
- `python scripts/sort_handlers.py src/jpoke/handlers/item.py src/jpoke/handlers/ability.py src/jpoke/handlers/volatile.py` を実行済み。
- `python scripts/sort_data/sort_items.py` 相当の並び替えは、既存の五十音順配置を変更していないため対象なし（イベント種別・関数名のみ変更）。
