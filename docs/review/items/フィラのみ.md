  - 修正前は共通関数 `heal_on_quarter_hp`（こんらん未対応）を使用しており、HP回復のみでこんらんが一切発生しない誤った実装だった。
  - `handlers/move_attack.py` の `なげつける_apply_item_effect` 内のフィラのみ処理（既存実装）と同一の性格リストであることを確認済み（相互に整合）。

## 参考: スコープ外で発見した既存の疑わしい点（未修正）
- `handlers/move_attack.py` の `なげつける_apply_item_effect` 内、バンジのみの分岐で `("やんちゃ", "のうてんき", "うっかりや", "なまいき")` となっているが、`なまいき`（せいかく的には無補正の性格）ではなく `むじゃき`（とくぼうが下がるすばやさ↑性格）が正しいはずである（`handlers/item.py` の `バンジのみ_heal_on_quarter_hp` は正しく `むじゃき` を使用しており、なげつける側とだけ食い違っている）。本レビューはフィラのみが対象範囲のため修正していない。バンジのみの再レビュー時に確認・修正することを推奨する。
