# こんごうだま レビュー結果

## 仕様書
- 変更なし。一次情報（docs/wiki/items/こんごうだま.html）と照合した結果、既存の
  `docs/spec/items/こんごうだま.md` の記述（効果・なげつける威力60・バトル以外の効果等）は
  一次情報と一致しており誤り・欠落はなかった。

## 実装計画書
- 修正した:
  - 登録コード例から誤った `removable=False` / `fling_power=0` を削除し、`fling_power=60` に修正。
  - こんごうだまが「奪う/奪われることが可能な通常アイテム」であり、フォルムチェンジを伴う
    `だいこんごうだま`（`removable=False`, `fling_power=0`）とは区別される旨の注記を追加。

## 実装（handlers/ / data/）
- 修正した:
  - `src/jpoke/data/item.py` の `こんごうだま` エントリで誤って設定されていた
    `removable=False` を削除し（既定値 `True` に戻す）、`fling_power=0` を仕様書通り `fling_power=60` に修正。
    こんごうだまはフォルムチェンジを伴わない通常の専用アイテムであり、はたきおとす・トリック等で
    奪う/奪われることができ、なげつけるも威力60で発動できる（一次情報に「なげつける: 威力60」と
    明記されているにも関わらず、従来実装では `だいこんごうだま`（フォルムチェンジ版、奪われない・
    なげつける失敗）と同じ設定が誤って流用されていた）。
  - `handlers/item.py` の `こんごうだま_modify_power`（ディアルガ・ディアルガ(オリジン)のドラゴン/はがね技を
    1.2倍にする）はロジック自体に誤りはなく変更なし。

    こんごうだまは fling_power=60 になったため、真に「なげつける失敗」の実装である
