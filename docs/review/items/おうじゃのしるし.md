# おうじゃのしるし レビュー結果

## 仕様書
- 変更なし（`docs/wiki/items/おうじゃのしるし.html` の内容と照合し、誤り・欠落なしと確認）

## 実装計画書
- 元々ひるみ効果がある技（アイアンヘッド等）への非重複ルール（第五世代以降の仕様）を明記
- 一撃必殺技には効果が無いことを明記
- りんぷん・てんのめぐみとの連携（`battle.resolve_secondary_chance` 経由）を明記
- なげつける使用時に100%の確率でひるませる仕様を追記

## 実装（handlers/ / data/）
- `handlers/item.py` の `flinch_on_hit_10pct`:
  - `battle.resolve_secondary_chance(ctx, 0.1)` を経由するよう修正（りんぷんによる無効化・てんのめぐみによる確率2倍が反映されていなかった）
  - 一撃必殺技（`ctx.move.has_flag("ohko")`）を除外
  - 元々ひるみの追加効果を持つ技（`_INNATE_FLINCH_MOVES`、アイアンヘッド・あくのはどう・エアスラッシュ等32技）を除外し、第五世代以降の非重複仕様を反映
- `handlers/move_attack.py` の `なげつける_apply_item_effect`:
  - `おうじゃのしるし`・`するどいキバ` を投げた場合に100%の確率でひるみを付与する分岐を追加（既存の `TODO` コメントを解消）
- `data/item.py` の `おうじゃのしるし` エントリ自体（`Event.ON_DAMAGE_HIT` / `subject_spec="attacker:self"` / `fling_power=30`）は変更なし

## テスト
- 既存2件（ひるまない確率／ひるみ付与）は変更なし
- 追加したテスト（`おうじゃのしるし` / `するどいキバ` でパラメータ化）:
  - 変化技では発動しないこと
  - 一撃必殺技には効果が無いこと（乱数消費回数の差分比較で確認。きあいのタスキで耐えたケースでの直接確認は下記「発見した既知の問題」により断念）
  - 元々ひるみ効果がある技（エアスラッシュ）には重複しないこと（乱数消費回数の差分比較で確認）
  - りんぷんにより無効化されること
  - てんのめぐみにより確率が2倍（20%）になること
  - なげつけるで使用すると100%の確率でひるませること

## 発見した既知の問題（本レビューの対象外・未修正）
- `きあいのタスキ` と一撃必殺技（例: じわれ）を組み合わせた場合、`docs/spec/items/きあいのタスキ.md` の
  「ダメージ固定技や一撃必殺技のダメージも耐える」という仕様に反し、HP1で耐えずそのままひんしになる不具合を確認した。
  `おうじゃのしるし` 自体の実装には影響しないため本レビューでは修正していない。別途 `きあいのタスキ` のレビュー時に対応が必要。
