    （ダメージ固定技はタイプ相性上抜群でも発動しない仕様に対応）
  - 不要だった `mon.alive` の重複ガードを削除（他の同種ハンドラと同様、`changes` の有無で判定）

- `data/move.py`: いかりのまえば・カタストロフィ・ちきゅうなげ・ナイトヘッド（非一撃必殺のダメージ固定技）に
  `"fixed_damage"` フラグを付与（じわれ等の一撃必殺技は対象外のままとし、タイプ相性判定を維持）
- `data/volatile.py`: `こらえる` の `ON_MODIFY_MOVE_DAMAGE` ハンドラの priority を `60` → `95` に修正
  - 従来は一撃必殺技（`ohko_damage`, priority=90）より先に評価されており、こらえるによるHP1補正が
    一撃必殺技の固定ダメージ（HP全部）で上書きされてしまい、こらえるが一撃必殺技に対して機能していなかった
    （既存の他の生存系ハンドラ：がんじょう・きあいのタスキはデフォルトpriority=100で一撃必殺技より後に
    評価されており正しく機能していたため、こらえるのみ整合していなかった）
  - この修正はじゃくてんほけんの「弱点タイプの一撃必殺技をこらえるで耐えた場合は発動する」の
    仕様確認テストのために必要だったため、あわせて修正した
