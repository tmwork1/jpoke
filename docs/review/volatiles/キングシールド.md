## 実装（handlers/ / data/）

- `src/jpoke/handlers/volatile.py` の `キングシールド_protect` を修正
  - 修正前: `_run_protect(battle, ctx, value, stats_change_on_contact={"A": -1})`
    → `protect_non_attack=True`（デフォルト）のため変化技も防いでいた（仕様違反）
  - 修正後: `_run_protect(battle, ctx, value, stats_change_on_contact={"A": -1}, protect_non_attack=False)`
    → 変化技は防がない（仕様: 「変化技を防ぐことはできない」）
- `src/jpoke/handlers/volatile.py` の `スレッドトラップ_protect` も同様に修正
  - Wiki仕様「変化技は防げない」に基づき `protect_non_attack=False` を追加
