# みちづれ レビュー結果

## 仕様書

**新規作成**: `docs/spec/volatiles/みちづれ.md`

仕様書が存在しなかったため、一次情報（`docs/wiki/volatiles/みちづれ_(状態変化).html`）をもとに新規作成した。

主要仕様:
- 攻撃技を受けてひんしになったとき、攻撃者もひんしにする
- マジックガード・きあいのタスキ・きあいのハチマキ・がんじょうを無視してひんしにできる（reason="perish"）
- みちづれ状態は自身が行動すると解除される（ON_TRY_ACTION priority=10）
- 攻撃技以外（状態異常・場の状態等）でひんしになっても発動しない
