# そらをとぶ レビュー結果

## 仕様書

- 新規作成（docs/spec/volatiles/そらをとぶ.md）
- 主な内容:
  - 発生条件: 技「そらをとぶ」「とびはねる」の1ターン目に発生
  - 消滅条件: 2ターン目攻撃完了時、または攻撃失敗時
  - 回避: ほとんどの技が命中しない。命中する技はかぜおこし・かみなり・たつまき・ぼうふう（第五世代以降）・うちおとすなど
  - 威力2倍: かぜおこし・たつまきは威力2倍（第三世代以降）
  - 行動固定・交代禁止（ゴーストタイプ除外）
