# みがわり レビュー結果

## 仕様書

**変更あり**: 新規作成（`docs/spec/volatiles/みがわり.md`）

仕様書が存在しなかったため、wiki情報（`docs/wiki/volatiles/みがわり_(状態変化).html`）をもとに作成した。

主な仕様ポイント：
- みがわりHPは最大HPの1/4（切り捨て）
- 攻撃技はみがわりHPを削り、本体HPに影響しない
- 変化技はみがわりで無効化される
- **音技（soundラベル）はみがわりを貫通する**（第六世代以降の仕様）
- bypass_substituteラベルの技もみがわりを貫通する
- すりぬけ特性はON_CHECK_HIT_SUBSTITUTEイベントで制御

## 実装（handlers/ / data/）

### src/jpoke/core/move_executor.py

**変更あり**: `check_hit_substitute` に音技の貫通チェックを追加

修正前:
```python
def check_hit_substitute(self, ctx: AttackContext) -> bool:
    if ctx.move.target != "foe":
        return False
    if ctx.move.has_label("bypass_substitute"):
        return False
    return self._events.emit(Event.ON_CHECK_HIT_SUBSTITUTE, ctx, True)
```

修正後:
```python
def check_hit_substitute(self, ctx: AttackContext) -> bool:
    if ctx.move.target != "foe":
        return False
    if ctx.move.has_label("bypass_substitute"):
        return False
    # 音技はみがわりを貫通する（第六世代以降の仕様）
    if ctx.move.has_label("sound"):
        return False
    return self._events.emit(Event.ON_CHECK_HIT_SUBSTITUTE, ctx, True)
```
