## テスト
### 【全般】ダメージ計算の検証方法
実際の動作条件に近い状態で検証したい。
具体的には、
'''
assert 5325 == t.calc_damage_modifier(battle_contact, Event.ON_CALC_POWER_MODIFIER)
'''
ではなく、
'''
battle.run_move(attacker, attacker.moves[0])
assert 5325 == battle.damage_calculator.damage_modifier
'''
のようにrun_move()したあとのデバッグ用変数を確認する。

すなわち、t.calc_damage_modifier()メソッドは使わない (定義削除済み)。

### test_utils.py
汎用関数の引数の atk_idx を省略不可にして明示させる

### 【全般】補正係数の検証では、数値を左辺に置く
'''
4096 == assert t.calc_damage_modifier(battle, Event.ON_CALC_DAMAGE_MODIFIER)
'''

### 【全般】attacker, defender の定義を一行にまとめる
'''
    defender = battle.actives[0]
    attacker = battle.actives[1]
    ->
    defender, attacker = battle.actives
'''
